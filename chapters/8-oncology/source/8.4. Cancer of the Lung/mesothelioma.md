# Mesothelioma

> [!tip] **FCPS/MRCP Priority: HIGH**
> **Mesothelioma = Malignant pleural/peritoneal neoplasm**; **Asbestos exposure** (latency 20-50yr); **Pleural (80%) vs Peritoneal (20%)**; **Histology: Epithelioid (60%, best), Sarcomatoid (10%, worst), Biphasic (30%)**; **IHC: Calretinin+, WT1+, D2-40+, CK5/6+, CEA-, BerEP4-, MOC-31-, TTF-1-**; **Staging: TNM 8th (pleural)**; **1L: Cisplatin + Pemetrexed ± Bevacizumab (MAPS)**; **2L: Nivolumab + Ipilimumab (CheckMate 743) — Superior to chemo**; **Surgery: P/D preferred over EPP (MARS 2)**; **RT: Prophylactic tract RT (PREVENT), Palliative**.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Link **asbestos exposure** to mesothelioma (latency, types: crocidolite > amosite > chrysotile)
- [ ] Apply **IHC panel** for diagnosis vs adenocarcinoma (Calretinin/WT1/D2-40 vs CEA/BerEP4)
- [ ] Classify **histological subtypes** (Epithelioid, Sarcomatoid, Biphasic) and prognostic impact
- [ ] Stage **pleural mesothelioma** using TNM 8th Edition
- [ ] Prescribe **1L**: Cisplatin + Pemetrexed **± Bevacizumab** (MAPS trial)
- [ ] Prescribe **2L**: **Nivolumab + Ipilimumab** (CheckMate 743) — **Standard 2L**
- [ ] Compare **EPP vs P/D** (MARS 2: P/D non-inferior, less morbidity)
- [ ] Indicate **RT**: Prophylactic tract RT (PREVENT trial — negative), Palliative RT

## 2. Definition & Epidemiology

| Feature | Detail |
|---------|--------|
| **Definition** | Malignant neoplasm of mesothelial cells; **Pleura (80%)**, **Peritoneum (20%)**, Pericardium/Tunica vaginalis (rare) |
| **Incidence** | UK: ~2,700/year; **Peak 2010-2020** (asbestos legacy); Male:Female = 5:1 (occupational) |
| **Prevalence** | **Median OS 12-18 months**; 5-year OS ~10% (improving with IO) |
| **Peak Age** | 70-80 years |
| **Latency** | **20-50 years** post-exposure |
| **Risk Factors** | **Asbestos** (crocidolite > amosite > chrysotile); **Occupational**: Shipbuilding, construction, insulation, brake linings; **Para-occupational** (family laundry); **Environmental** (vicinity to mines); **Radiation** (therapeutic); **Simian Virus 40** (controversial); **BAP1 germline** (predisposition) |

## 3. Aetiology & Pathophysiology

```mermaid
flowchart LR
    A[Asbestos Fibres] --> B[Phagocytosis by Mesothelial Cells/Macrophages]
    B --> C[Frustrated Phagocytosis]
    C --> D[ROS/RNS Generation]
    D --> E[DNA Damage]
    E --> F[Chronic Inflammation]
    F --> G[Cytokine Release (TNF-α, IL-1β, TGF-β)]
    G --> H[Genomic Alterations]
    H --> H1[BAP1 Loss (60%) - Tumour Suppressor]
    H --> H2[CDKN2A/p16 Loss (70%) - Cell Cycle]
    H --> H3[NF2/Merlin Loss (40%) - Contact Inhibition]
    H --> H4[TP53 Mut (15-20%)]
    H --> H5[SETD2, PBRM1 Mut]
    H1 --> I[Mesothelioma Development]
    H2 --> I
    H3 --> I
    H4 --> I
    H5 --> I
```

### Histological Subtypes (WHO 2021)

| Subtype | Frequency | Prognosis | Key Features |
|---------|-----------|-----------|--------------|
| **Epithelioid** | **60%** | **Best** (Median OS 18-24mo) | Tubulopapillary, solid, deciduoid; **Calretinin+, WT1+** |
| **Sarcomatoid** | **10%** | **Worst** (Median OS 6-10mo) | Spindle cells, desmoplastic; **Resistant to chemo**; **Vimentin+, Calretinin variable** |
| **Biphasic** | **30%** | **Intermediate** | Mixed epithelioid + sarcomatoid; **Prognosis by % epithelioid** |

## 6. Diagnosis & Investigations

| Investigation | Role | Key Details |
|---------------|------|-------------|
| **Pleural Fluid Cytology** | Initial | **Low sensitivity (30%)**; **Cell block for IHC** |
| **Thoracoscopic Biopsy (VATS)** | **Gold standard** | **Multiple biopsies**; **Adequate tissue for IHC + Molecular** |
| **Image-guided Core Biopsy** | If VATS not feasible | Lower yield |
| **IHC Panel (Mesothelial vs Adenocarcinoma)** | **Diagnostic** | **Mesothelial markers**: **Calretinin+, WT1+, D2-40 (Podoplanin)+, CK5/6+**<br/>**Adenocarcinoma markers**: **CEA+, BerEP4+, MOC-31+, TTF-1+ (lung), Napsin A+ (lung)** |
| **CT Chest/Abdomen (Pleural/Peritoneal)** | Staging | **Pleural thickening, nodularity, effusion, interlobar fissure** |
| **PET-CT** | Staging, Response | **SUVmax correlates with histology** (Epitheliod higher) |
| **MRI** | Chest wall/diaphragm invasion | Superior soft tissue |
| **BAP1 IHC / Germline** | Prognosis, Predisposition | **Loss = worse prognosis**; Germline testing if young/family history |

## 7. Differential Diagnosis

| Condition | Distinguishing Features |
|-----------|-------------------------|
| **Metastatic Adenocarcinoma (Lung, Breast, Ovary, GI)** | **CEA+, BerEP4+, MOC-31+, TTF-1+ (lung)**; **Calretinin-/WT1-** |
| **Reactive Mesothelial Hyperplasia** | **BAP1 retained**, No invasion, Uniform cells; **p53 normal** |
| **Pleural Plaques / Fibrosis** | Benign, calcified, parietal; **No nodules/effusion** |
| **Solitary Fibrous Tumour** | **STAT6+, CD34+, BCL2+**; **Calretinin-** |
| **Lymphoma** | **CD45+, CD20+** (B-cell); Pleural effusion cytology |
| **Peritoneal Carcinomatosis** | Adenocarcinoma markers+, WT1+ (ovarian) — **PAX8+ for ovarian** |

## 8. Management

### First-Line Systemic Therapy

```mermaid
flowchart TD
    A[Unresectable Mesothelioma] --> B{Performance Status / Fitness}
    B -->|**Fit (ECOG 0-1)**| C[**Cisplatin 75mg/m² d1 + Pemetrexed 500mg/m² d1 q3w ×4-6 cycles**<br/>**+ Bevacizumab 15mg/kg d1** (MAPS trial) — **Preferred if eligible**<br/>**OS 18.8 vs 16.1 mo (HR 0.77)**; **PFS 9.2 vs 7.3 mo**]
    B -->|**Unfit / Cisplatin contraindicated**| D[**Carboplatin AUC5 d1 + Pemetrexed 500mg/m² d1 q3w ×4-6 cycles**<br/>**Non-inferior to Cisplatin** (IFCT-GFPC 0801)<br/>**No Bev with Carbo** (not studied)]
    C --> E[**Maintenance Pemetrexed** (if response/SD, standard NSCLC practice)]
    D --> E
    E --> F[**Folic Acid 400mcg daily + B12 1mg IM q3 cycles**<br/>**Dexamethasone 8mg BID d-1,0,1** (premedication for Pem)]
```

### Second-Line / Subsequent Therapy

```mermaid
flowchart TD
    A[Progression post Platinum+Pemetrexed] --> B{Histology / Biomarkers}
    B -->|**All subtypes**| C[**Nivolumab 360mg q3w + Ipilimumab 1mg/kg q6w** (CheckMate 743)<br/>**OS HR 0.74**, mOS 18.1 vs 14.1 mo<br/>**Superior to Chemo (Gem/Vinorelbine)**<br/>**Standard 2L regardless of histology**]
    B -->|**Epithelioid**| D[**Pembrolizumab** (KEYNOTE-158) — ORR ~20%<br/>**Single-agent IO** — less effective than Nivo+Ipi]
    C --> E[**3L+**: Gemcitabine / Vinorelbine / Clinical Trial / Re-challenge Pem (if long interval)]
```

### Surgery (Selected Early Stage)

| Procedure | Description | Evidence |
|-----------|-------------|----------|
| **EPP (Extrapleural Pneumonectomy)** | **En bloc**: Lung, parietal/visceral pleura, pericardium, diaphragm → **Gortex reconstruction** | **MARS 1**: EPP + Chemo + RT vs Chemo + RT → **No OS benefit, high mortality** |
| **P/D (Pleurectomy/Decortication)** | **Parietal + Visceral pleurectomy** (lung-sparing) | **MARS 2**: **P/D non-inferior to EPP**, **Lower mortality (1.7% vs 4.5%)**, Less morbidity **→ Preferred** |
| **Cytoreductive Surgery + HIPEC** | Peritoneal mesothelioma | **Selected centres**, Epithelioid, Complete cytoreduction (CC-0/1) |

### Radiation Therapy

| Indication | Details |
|------------|---------|
| **Prophylactic Tract RT** | **PREVENT trial**: 21Gy/3fx to intervention tracts **NO benefit** for procedure tract metastases — **Not routine** |
| **Palliative RT** | **30Gy/10fx** or **20Gy/5fx** for pain, SVC obstruction, nerve root compression |
| **Adjuvant RT (Post-EPP)** | **Hemithorax 50-54Gy** (historical, MARS era) — **Not standard post-P/D** |

## 11. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Mesothelioma vs Adenocarcinoma (Lung) IHC** | **Calretinin/WT1/D2-40/CK5/6 = Mesothelioma**; **CEA/BerEP4/MOC-31/TTF-1 = Lung Adeno**; **Use PANEL, not single marker** |
| **EPP vs P/D** | **EPP** = Lung resection + Pleurectomy + Pericardium + Diaphragm; **P/D** = **Pleurectomy only (lung-sparing)**; **MARS 2: P/D preferred** |
| **Bevacizumab in MAPS** | **Only with Cisplatin** (not studied with Carbo); **Contraindicated**: Haemoptysis, recent surgery, anticoagulation, brain mets |
| **CheckMate 743 — Nivo+Ipi vs Chemo** | **Nivo+Ipi superior OS in ALL histologies** (Epithelioid HR 0.86, Non-epitheliod HR 0.46); **No PD-L1 selection** |
| **PREVENT trial** | **Prophylactic RT to intervention tracts (21Gy/3fx) — NEGATIVE** — Do not routinely irradiate biopsy/drain tracts |
| **Peritoneal vs Pleural staging** | **Pleural**: TNM 8th; **Peritoneal**: **PCI (Peritoneal Cancer Index)** + Simplified Staging |

**Mnemonic: MESOTHELIOMA**
- **M**esothelial IHC: **Calretinin, WT1, D2-40, CK5/6+**
- **E**pithelioid best, **S**arcomatoid worst, **O**utcome by histology
- **T**NM 8th (Pleural), **H**IPEC (Peritoneal)
- **E**xposure: **Asbestos** (Crocidolite > Amosite > Chrysotile), Latency 20-50yr
- **L**ung Adeno markers: **CEA, BerEP4, MOC-31, TTF-1** (Negative in Meso)
- **I**pilimumab + **N**ivolumab = **2L Standard** (CheckMate 743)
- **O**riginal 1L: **Cis+Pem ± Bev** (MAPS)
- **M**ARS 2: **P/D > EPP** (Lung-sparing, less mortality)
- **A**djuvant RT: **PREVENT NEGATIVE** (No prophylactic tract RT)

   - **Answer: A** — follow-up: **Loss = worse prognosis**; **Germline BAP1** = Cancer predisposition syndrome (mesothelioma, uveal melanoma, RCC)

   - A. **CRS + HIPEC** (Epithelioid, CC-0/1)

   - C. Empirical broad approach
   - D. No intervention required

## 19. Flashcards

**Q1:** Asbestos?
**A1:** Crocidolite (blue) > Amosite (brown) > Chrysotile (white); Latency 20-50yr; All types carcinogenic

**Q2:** IHC?
**A2:** Mesothelioma: Calretinin+/WT1+/D2-40+/CK5/6+; Adenocarcinoma: CEA+/BerEP4+/MOC-31+/TTF-1+

**Q3:** Histology?
**A3:** Epithelioid (60%) best, Sarcomatoid (10%) worst, Biphasic (30%) intermediate

**Q4:** Staging?
**A4:** TNM 8th (Pleural); Peritoneal: PCI (Peritoneal Cancer Index)

**Q5:** 1L?
**A5:** Cisplatin + Pemetrexed ± Bevacizumab (MAPS); Carbo+Pem if Cis unfit; Folate/B12 + Dex premed

**Q6:** 2L?
**A6:** Nivolumab + Ipilimumab (CheckMate 743) — OS HR 0.74, Standard 2L

**Q7:** Surgery?
**A7:** P/D > EPP (MARS 2: non-inferior, less mortality); Trimodality (Chemo → P/D → RT) selected centres

**Q8:** RT?
**A8:** Prophylactic tract RT: PREVENT trial NEGATIVE — Not routine; Palliative RT for symptoms

# Inherited Cancer Syndromes

> [!tip] **FCPS/MRCP Priority: HIGH**
> **Hereditary Cancer = 5-10% of All Cancers**; **Major Syndromes**: **Lynch (MMR: MLH1/2/6/PMS2/EPCAM)**, **FAP/AFAP (APC)**, **MAP (MUTYH)**, **HBOC (BRCA1/2, PALB2)**, **Li-Fraumeni (TP53)**, **Peutz-Jeghers (STK11)**, **Juvenile Polyposis (SMAD4/BMPR1A)**, **PTEN Hamartoma**, **VHL**, **MEN1/2**, **FAMMM**; **Diagnostic Criteria**: Amsterdam II (Clinical), Revised Bethesda (Testing); **Surveillance**: Syndrome-specific; **Genetic Testing**: Universal for CRC <50, Ovarian, Pancreatic, Metastatic Prostate; **Cascade Testing**; **Risk-Reducing Surgery** (RRSO, Colectomy).

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Identify **major hereditary cancer syndromes** and their genetic basis
- [ ] Apply **Amsterdam II** and **Revised Bethesda** criteria for Lynch syndrome
- [ ] Recognise **BRCA1/2**, **Li-Fraumeni**, **FAP**, **MAP**, **Peutz-Jeghers**, **VHL**, **MEN1/2**
- [ ] Design **surveillance protocols** for each syndrome
- [ ] Counsel on **genetic testing** and **cascade testing**
- [ ] Indicate **risk-reducing surgery** (RRSO, Colectomy, Prophylactic Mastectomy)

## 2. Lynch Syndrome (HNPCC)

### Genetics
| Gene | Protein | Frequency |
|------|---------|-----------|
| **MLH1** | MutLα | ~40% |
| **MSH2** | MutSα | ~40% |
| **MSH6** | MutSα | ~10-15% |
| **PMS2** | MutLα | ~5-10% |
| **EPCAM** | EpCAM (MSH2 silencing) | <5% |

### Cancer Risks (Cumulative to Age 70)
| Cancer | MLH1/MSH2 | MSH6 | PMS2 |
|--------|-----------|------|------|
| **Colorectal** | 40-50% | 10-20% | 15-20% |
| **Endometrial** | 40-60% | 20-30% | 10-15% |
| **Ovarian** | 8-10% | 1-5% | <5% |
| **Gastric** | 5-10% | <5% | <5% |
| **Small Bowel** | 4-7% | <5% | <5% |
| **Urothelial** | 4-8% | <5% | <5% |
| **Pancreatic** | 2-5% | <5% | <5% |

## 3. Diagnostic Criteria

### Amsterdam II Criteria (Clinical Diagnosis)
**All of the following:**
1. **≥3 Relatives** with HNPCC-associated cancer (CRC, Endometrial, Small Bowel, Ureter, Renal Pelvis)
2. **One is First-Degree Relative** of the other two
3. **≥2 Successive Generations** affected
4. **≥1 Diagnosed <50 years**
5. **FAP Excluded**
6. **Tumours Verified** by Histology

### Revised Bethesda Criteria (Testing Indication)
**Test Tumour for MSI/MMR if ANY:**
1. **CRC <50 years**
2. **Synchronous/Metachronous LS Cancers**
3. **CRC with MSI-H Histology <60 years** (TILs, Mucinous, Signet Ring, Crohn's-like)
4. **CRC with 1+ FDR with LS Cancer <50yr**
5. **CRC with 2+ FDR/SDR with LS Cancer Any Age**

## 4. Familial Adenomatous Polyposis (FAP)

### Genetics
| Gene | Inheritance | Frequency |
|------|-------------|-----------|
| **APC** (Chr 5q21) | **Autosomal Dominant** | **1:10,000-15,000** |

### Variants
| Variant | Polyp Count | Features |
|---------|-------------|----------|
| **Classic FAP** | **>100** | Classic |
| **Attenuated FAP (AFAP)** | **10-100** | Later onset, Proximal |
| **Gardner Syndrome** | FAP + | Osteomas, Desmoids, Epidermoid Cysts |
| **Turcot Syndrome** | FAP + | CNS Tumours (Medulloblastoma) |

### Cancer Risks
| Cancer | Risk |
|--------|------|
| Colorectal | **~100%** (if untreated) |
| Duodenal/Periampullary | 5-10% |
| Desmoid Tumours | 10-15% |
| Thyroid (Papillary) | 2-5% (Female) |
| CNS (Medulloblastoma) | <1% (Turcot) |

## 6. Genetic Testing & Cascade Testing

### Indications for Germline Testing
| Criteria | Action |
|----------|--------|
| **CRC <50yr** | Test Lynch, FAP, MAP, PPAP, PJS, JPS, PTEN |
| **≥10 Colorectal Adenomas** | FAP, AFAP, MAP, PJS, JPS, PTEN |
| **Endometrial <50yr** | Lynch |
| **Ovarian Cancer** | Lynch, BRCA1/2 |
| **FH of CRC/EC/OC/GC <50yr** | Lynch, FAP, MAP, BRCA1/2 |
| **Known Family Mutation** | Predictive Testing |

### Cascade Testing Pathway
| Step | Action |
|------|--------|
| 1. Index Case | Comprehensive Germline Testing (NGS Panel) |
| 2. Identify Variant | Class 4/5 Pathogenic |
| 3. Inform Relatives | Genetic Counselling |
| 4. Predictive Testing | At-Risk Relatives (50% AD, 25% AR) |

## 7. Risk-Reducing Surgery

| Syndrome | Procedure | Timing |
|----------|-----------|--------|
| **FAP** | Proctocolectomy + IPAA | Late Teens/Early 20s |
| **Lynch (CRC)** | Colectomy + IRA | At CRC Diagnosis |
| **Lynch (Endometrial)** | RRSO (BSO + Hysterectomy) | Age 40-45, Post-Childbearing |
| **BRCA1/2** | RRSO (BSO) | 35-45 (BRCA1), 40-45 (BRCA2) |
| **BRCA1/2** | Bilateral Mastectomy | Individualised |

## 9. Viva Questions (MRCP PACES / FCPS)

| Question | Expected Answer |
|----------|-----------------|
| **Lynch Syndrome — Genes, Cancer Risks?** | MLH1/2/6/PMS2/EPCAM; CRC 40-50%, Endometrial 40-60%, Ovarian 10%, Gastric 10%. |
| **Amsterdam II vs Bethesda Criteria?** | Amsterdam II: Clinical Dx (3 Relatives, 2 Generations, 1 <50, FAP Excluded); Bethesda: Testing Indication (CRC <50, MSI-H, FH). |
| **FAP — Gene, Polyp Count, Surveillance?** | APC, >100 Adenomas, Colonoscopy Annual from 10-15yr, OGD 1-3yr. |
| **MAP — Inheritance, Gene, Surveillance?** | AR, MUTYH, Colonoscopy 1-2yr from 20-25, OGD 2-3yr. |
| **Li-Fraumeni — Gene, Cancers, Surveillance?** | TP53, Sarcoma/Brain/Adrenal/Leukaemia/BC, WBMRI Annual + Breast MRI + Brain MRI. |
| **BRCA1 vs BRCA2 — Differences?** | BRCA1: TNBC, Ovarian, Higher Penetrance; BRCA2: Breast, Ovarian, Prostate, Pancreatic, Male Breast. |
| **Lynch — MLH1 Loss with BRAF Mut / Methylation = Sporadic?** | MLH1 Loss + BRAF V600E or MLH1 Methylation = Sporadic; Germline only if both negative. |
| **FAP Desmoids — Management?** | Post-Surgical, Avoid Unnecessary Surgery, Medical (NSAID, Hormonal, TKI). |

## 10. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Lynch vs FAP** | Lynch: MMR Genes, MSI-H, Fewer Polyps, Extra-Colonic; FAP: APC, 100s Polyps, Desmoids, Osteomas |
| **MAP vs FAP** | MAP: Recessive (MUTYH), 20-50 Polyps; FAP: Dominant (APC), 100+ Polyps |
| **AFAP vs Classic FAP** | AFAP: 10-100 Polyps, Proximal, Later Onset; Classic: 100+, Teen/20s, Whole Colon |
| **Lynch MLH1 Loss — Sporadic vs Germline** | MLH1 Loss + BRAF V600E or MLH1 Methylation = Sporadic; Both Negative = Germline |
| **PJS vs JPS Polyps** | PJS: Hamartoma with Arborizing Smooth Muscle, Pigmentation; JPS: Hamartoma, No Smooth Muscle Core |

**Mnemonic: HEREDITARY-CRC**
- **H**ereditary CRC: 5-10% of CRC
- **E**pigenetic: MLH1 Methylation = Sporadic Lynch
- **R**isk Genes: MLH1/2/6/PMS2/EPCAM (Lynch), APC (FAP), MUTYH (MAP), STK11 (PJS), SMAD4/BMPR1A (JPS)
- **E**xtracolonic: Lynch (Endo, Ovarian, Gastric), FAP (Desmoids, Osteomas), PJS (Pigmentation, Breast, Pancreas), PTEN (Breast, Thyroid)
- **I**nheritance: Lynch/FAP/APC/PPAP = AD; MAP = AR; PJS/JPS/PTEN = AD
- **T**umour Testing: MSI/IHC First → Germline if Deficient
- **A**FAP: 10-100 Polyps, Proximal, Later Onset
- **R**ecessive MAP: MUTYH, AR, 20-50 Polyps
- **Y**oung Onset: CRC <50 → Test Lynch/FAP/MAP/PPAP
- **C**ascade Testing: Index → Germline → Relatives (50% AD)
- **R**isk-Reducing Surgery: FAP → Colectomy+IPAA, Lynch → Colectomy/RRSO
- **C**riteria: Amsterdam II (Clinical), Bethesda (Testing)

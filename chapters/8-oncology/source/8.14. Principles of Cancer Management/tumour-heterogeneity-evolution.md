# Tumour Heterogeneity & Evolution

> [!tip] **FCPS/MRCP Priority: HIGH**
> **Tumour Heterogeneity = Spatial + Temporal Variation**; **Spatial**: Intra-tumour (ITH), Inter-tumour; **Temporal**: Clonal Evolution (Linear, Branching, Neutral), Punctuated Equilibrium; **Mechanisms**: CIN, MSI, Chromothripsis, Katagiri, APOBEC; **Clinical Impact**: Therapy Resistance, Metastasis, Sampling Bias, Liquid Biopsy; **TRACERx/PEACE**: Phylogenetic Trees, Parallel/Convergent Evolution.

## 2. Types of Tumour Heterogeneity

| Dimension | Type | Description |
|-----------|------|-------------|
| **Spatial** | **Intra-tumour (ITH)** | Genetic/phenotypic differences within same tumour (primary vs regions) |
| | **Inter-tumour** | Differences between primary and metastatic sites, or between patients |
| **Temporal** | **Clonal Evolution** | **Linear** (stepwise), **Branching** (parallel subclones), **Neutral** (no selection) |
| | **Punctuated Equilibrium** | **Bursts of genomic change** (Chromothripsis, Kataegis) followed by stability |

---

## 3. Mechanisms of Heterogeneity

### Genomic Instability

| Mechanism | Description | Associated Cancers |
|-----------|-------------|-------------------|
| **CIN** (Chromosomal Instability) | **Aneuploidy**, Large-scale CNV, Chromothripsis | Most solid tumours (CRC, Breast, Ovarian, Lung) |
| **MSI** (Microsatellite Instability) | **MMR Deficiency**, Hypermutation, Indels | Lynch, CRC (15%), Endometrial, Gastric |
| **Chromothripsis** | **Catastrophic Chromosome Shattering** (1-2 events) | Medulloblastoma, Neuroblastoma, Bone Sarcomas, 2-3% Cancers |
| **Kataegis** | **Localised Hypermutation** (APOBEC) | Breast, Cervical, Bladder, Head & Neck |
| **APOBEC Signatures** | **Cytosine Deamination** (SBS2, SBS13) | Breast, Cervical, Head & Neck, Bladder |
| **HRD** (Homologous Recombination Deficiency) | **BRCA1/2, PALB2, RAD51** Loss | Ovarian, Breast, Prostate, Pancreatic |

### Mutational Signatures (COSMIC)

| Signature | Aetiology | Associated Cancers |
|-----------|-----------|-------------------|
| **SBS1** | Ageing (Spontaneous deamination) | Universal |
| **SBS2/13** | **APOBEC** (Cytosine Deamination) | Breast, Cervical, Bladder, Head & Neck |
| **SBS3** | **HRD** (BRCA1/2) | Ovarian, Breast, Pancreatic |
| **SBS4** | **Tobacco** (Benzo[a]pyrene) | Lung, Head & Neck, Liver |
| **SBS7** | **UV Radiation** | Melanoma, Skin SCC |
| **SBS6** | **MMR Deficiency** (MSI) | Lynch, CRC, Endometrial |
| **SBS17** | Unknown | Gastric, Oesophageal |

## 4. Clonal Evolution Models

### Linear vs Branching Evolution

| Model | Description | Pattern |
|-------|-------------|---------|
| **Linear** | Sequential acquisition of mutations in single lineage | **Sequential**, Stepwise |
| **Branching** | **Parallel subclones** diverge from common ancestor | **Parallel Subclones**, Coexistence |
| **Neutral** | No selection, drift dominates | **Many Low-Frequency Subclones** |
| **Punctuated Equilibrium** | **Bursts** (Chromothripsis, Kataegis) + **Stasis** | **Burst-Stasis-Burst** |

### Phylogenetic Tree Interpretation

| Tree Pattern | Evolutionary Mode | Clinical Implication |
|--------------|-------------------|---------------------|
| **Star-like** | **Star-like** | **Rapid expansion**, Single dominant clone |
| **Branching** | **Multiple subclones** | **Heterogeneity**, Therapy Resistance |
| **Linear** | **Sequential** | **Sequential Treatments** possible |
| **Punctuated** | **Bursts + Stasis** | **Rapid Evolution**, Early Metastasis |

## 5. Clinical Impact of Heterogeneity

### Therapy Resistance

| Mechanism | Example |
|-----------|---------|
| **Pre-existing Resistant Subclone** | **EGFR T790M** in NSCLC pre-exists at low frequency |
| **Adaptive Evolution** | **EGFR C797S** emerges under Osimertinib pressure |
| **Parallel Resistance** | **Multiple subclones** with different resistance mechanisms (MET amp, HER2 amp, SCLC transformation) |
| **Lineage Plasticity** | **Lineage Switch** (Adenocarcinoma → SCLC under EGFR TKI) |

### Metastasis

| Concept | Description |
|-----------|-------------|
| **Parallel Progression** | **Metastases seed early**, evolve in parallel with primary |
| **Linear Progression** | **Metastasis from dominant subclone** late in evolution |
| **Self-Seeding** | **Metastases re-seed primary** tumour |

### Sampling Bias & Liquid Biopsy

| Issue | Impact |
|-------|--------|
| **Single Biopsy** | **Underestimates ITH**, Misses subclones |
| **Multi-region Sequencing** | **Better ITH capture**, Impractical clinically |
| **Liquid Biopsy (ctDNA)** | **Captures global heterogeneity**, Detects minor subclones |

---

## 6. Landmark Studies

### TRACERx (Non-Small Cell Lung Cancer)
| Finding | Significance |
|--------|--------------|
| **High ITH** | **Branching evolution** dominant, Parallel subclones |
| **Metastasis** | **Parallel dissemination** from primary, not linear |
| **Driver Events** | **Early** (EGFR, KRAS, TP53), **Late** (MYC, CCNE1 amp) |
| **Resistance** | **Parallel evolution** of resistance in metastases |

### PEACE (Post-mortem Cancer Evolution)
| Finding | Significance |
|--------|--------------|
| **Widespread Metastasis** | **Polyclonal seeding**, Multiple waves |
| **Driver Mutations** | **Truncal** (TP53, KRAS), **Branch** (APC, PIK3CA) |
| **Metastasis Seeding** | **Polyclonal**, Multiple primary clones |

---

## 7. Parallel vs Convergent Evolution

| Type | Definition | Example |
|------|------------|---------|
| **Parallel Evolution** | **Independent acquisition** of same mutation in different subclones | **EGFR T790M** arising independently in multiple subclones under EGFR TKI pressure |
| **Convergent Evolution** | **Different mutations** in same gene/pathway converging on same phenotype | **EGFR C797S** (TKI resistance) + **MET Amp** (Bypass) → Both cause EGFR TKI resistance |

## 8. Clinical Applications

### Liquid Biopsy & Heterogeneity
| Application | Advantage |
|----------|-----------|
| **ctDNA MRD** | Captures **global heterogeneity** (vs single biopsy) |
| **Resistance Tracking** | **Early detection** of resistance mutations (EGFR T790M, ESR1, KRAS) |
| **Tumour Fraction** | **Prognostic**, **Response monitoring**, **MRD** |
| **Phylogenetic Inference** | **ctDNA phylogeny** → Tumour evolutionary history |

### Clinical Decision-Making
| Scenario | Heterogeneity-Informed Decision |
|--------|--------------------------------|
| **Targeted Therapy** | **Target truncal drivers** (present in all subclones) |
| **Combination Therapy** | **Target multiple pathways** to prevent resistance |
| **Adaptive Therapy** | **Dose modulation** to maintain sensitive cells suppressing resistant ones |
| **Surveillance** | **ctDNA serial monitoring** for emerging resistance |

## 11. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **ITH vs Inter-tumour** | **ITH**: Within single tumour; **Inter-tumour**: Between different tumours (primary vs met, or between patients) |
| **Linear vs Branching** | **Linear**: One clone → Next clone; **Branching**: Multiple subclones coexist from common ancestor |
| **Punctuated Equilibrium** | **Not continuous** — **Bursts (Chromothripsis/Kataegis)** followed by **Stasis** |
| **Parallel vs Convergent** | **Parallel**: Same mutation, independent; **Convergent**: Different mutations, same pathway/output |
| **TRACERx vs PEACE** | **TRACERx**: NSCLC, prospective, longitudinal; **PEACE**: Multi-cancer, post-mortem, deep sequencing |
| **Parallel vs Linear Metastasis** | **Parallel**: Early dissemination, parallel evolution; **Linear**: Late, from dominant clone |
| **CHIP vs Tumour Heterogeneity** | **CHIP**: Blood-derived, age-related; **Tumour**: Tissue-derived, cancer-specific |
| **Parallel Evolution vs Clonal Expansion** | **Parallel**: Same mutation in parallel branches; **Clonal Expansion**: Single clone expands |

**Mnemonic: TUMOUR-EVOLUTION**
- **T**umour **H**eterogeneity: **Spatial (Intra/Inter) + Temporal (Evolution)**
- **U**niformity: **Not Uniform** — Heterogeneity is Universal
- **M**echanisms: **CIN, MSI, Chromothripsis, Kataegis, APOBEC, HRD**
- **O**rigin: **Clonal Evolution** (Linear, Branching, Punctuated)
- **U**niform vs **B**ranching: **Linear vs Branching** Evolution
- **R**esistance: **Pre-existing + Adaptive** (Parallel + Convergent)
- **E**volution Models: **Linear, Branching, Neutral, Punctuated**
- **P**arallel vs Convergent: **Same Mutation vs Same Pathway**
- **H**eterogeneity Impact: **Resistance, Metastasis, Sampling Bias**
- **E**volutionary Studies: **TRACERx (NSCLC), PEACE (Post-mortem)**
- **T**emporal: **Clonal Evolution over Time**
- **I**TH vs Inter: **Intra-tumour vs Inter-tumour**
- **O**n-Treatment: **ctDNA Captures Evolution**, **MRD, Resistance**
- **N**eutral Evolution: **Drift Dominates, No Selection**

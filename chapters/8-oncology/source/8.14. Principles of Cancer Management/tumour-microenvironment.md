# Tumour Microenvironment (TME)

> [!tip] **FCPS/MRCP Priority: HIGH**
> **TME = Non-Cancerous Cells + ECM + Signalling Molecules** in tumour; **Key Cellular Components**: **CAFs** (FAP+, α-SMA+, Collagen, TGF-β, SDF-1), **TAMs** (M1: Anti-tumour, M2: Pro-tumour, CD163+, CD206+), **MDSCs** (PMN-MDSC, M-MDSC, Arg1, iNOS, ROS), **Tregs** (FoxP3+, CTLA-4, IL-10, TGF-β), **TILs** (CD8+, CD4+, γδ T, NK, B Cells), **Endothelial Cells**, **Pericytes**; **ECM**: Collagen, HA, MMPs, LOX; **Angiogenesis**: VEGF, FGF, Angiopoietins; **Immune Checkpoints**: PD-1/PD-L1, CTLA-4, LAG-3, TIM-3, TIGIT; **Cytokines**: TGF-β, IL-6, IL-10, VEGF, CXCL12.

## 2. Cellular Components of TME

### Cancer-Associated Fibroblasts (CAFs)

| Feature | Detail |
|--------|--------|
| **Markers** | **FAP (Fibroblast Activation Protein)**, **α-SMA**, **PDGFRα/β**, **FSP1/S100A4**, **Vimentin** |
| **Origins** | **Resident Fibroblasts**, **Bone Marrow MSCs**, **Endothelial-Mesenchymal Transition (EndMT)**, **Epithelial-Mesenchymal Transition (EMT)**, **Pericytes** |
| **Functions** | **ECM Production/Remodelling** (Collagen I/III, Fibronectin, Hyaluronan), **LOX/LOXL2** (Collagen Cross-linking), **Growth Factors** (HGF, FGF, IGF, SDF-1/CXCL12), **Cytokines** (TGF-β, IL-6, SDF-1), **Immune Exclusion** (FAP+ CAFs → CXCL12 → T-cell Exclusion), **Therapy Resistance** (Chemoprotection, Anti-angiogenic Resistance) |
| **Subtypes** | **Inflammatory CAFs (iCAFs)**: IL-6, CXCL1, LIF; **Myofibroblastic CAFs (myCAFs)**: α-SMA, Contractility; **Antigen-Presenting CAFs (apCAFs)**: MHC-II, CD74 |

### Tumour-Associated Macrophages (TAMs)

| Polarisation | Markers | Cytokines | Function |
|--------------|---------|-----------|----------|
| **M1 (Classical/Pro-inflammatory)** | **CD80, CD86, MHC-II, iNOS** | **IL-12, TNF-α, IL-1β, IL-23, CXCL9/10/11** | **Anti-tumour**: Phagocytosis, ADCC, Th1 Activation, Tumour Cell Killing |
| **M2 (Alternative/Pro-tumour)** | **CD163, CD206, CD204, Arg1, Ym1** | **IL-10, TGF-β, VEGF, EGF, CCL17/22** | **Pro-tumour**: Angiogenesis, Matrix Remodelling, Immune Suppression (Treg Recruitment), Tissue Repair |

> **TAM Polarisation**: **M1 → M2 Shift** driven by **Tumour-Derived Factors** (IL-4, IL-10, IL-13, M-CSF, TGF-β, PGE2, Lactate, Hypoxia/HIF-1α)
> **Prognosis**: **High M2/M1 Ratio = Poor Prognosis**

### Myeloid-Derived Suppressor Cells (MDSCs)

| Subset | Markers (Human) | Mechanism of Suppression |
|--------|-----------------|--------------------------|
| **PMN-MDSC (Polymorphonuclear)** | **CD11b+ CD14- CD15+ CD66b+ LOX-1+** | **ARG1, ROS, Peroxynitrite, PGE2** |
| **M-MDSC (Monocytic)** | **CD11b+ CD14+ HLA-DRlow/- CD15-** | **ARG1, iNOS, NO, IL-10, TGF-β, PD-L1** |
| **eMDSC (Early)** | **CD11b+ CD14- CD15- HLA-DR-** | **Less Characterised** |

> **Recruitment**: **GM-CSF, G-CSF, IL-6, VEGF, CXCL1/2/5/8, CCL2, S100A8/A9, Stem Cell Factor (SCF)**
> **Clinical**: High MDSC = Poor Prognosis, Resistance to ICIs

### Regulatory T Cells (Tregs)

| Feature | Detail |
|--------|--------|
| **Definition** | **CD4+ CD25high FoxP3+** (Lineage: Thymic tTreg vs Peripheral pTreg) |
| **Suppressive Mechanisms** | **CTLA-4** (Trans-endocytosis of CD80/86 on DCs), **IL-10, TGF-β, IL-35**, **cAMP**, **CD39/CD73 → Adenosine**, **Metabolic Disruption (IL-2 Consumption)** |
| **Recruitment/Expansion** | **CCL22/CCR4**, **TGF-β, IL-2, Retinoic Acid**, **IDO** (Kynurenine/AhR) |
| **Prognosis** | **High Intratumoural Tregs = Poor Prognosis** (Most cancers); **Exception**: Follicular Lymphoma, Hodgkin (Good) |

### Tumour-Infiltrating Lymphocytes (TILs)

| Population | Significance |
|------------|--------------|
| **CD8+ Cytotoxic T Cells** | **Primary Anti-tumour Effectors**; **High CD8+ TILs = Good Prognosis** (Most solid tumours); **Exhaustion** (PD-1, TIM-3, LAG-3, TIGIT, TOX) |
| **CD4+ Helper T Cells** | **Th1 (IFN-γ, Good)**, **Th17 (IL-17, Context-Dependent)**, **Tfh (B Cell Help, TLS)** |
| **γδ T Cells** | **Vδ1 (Tissue-Resident, Cytotoxic)**, **Vδ2 (Circulating, Phosphoantigen-Responsive)** |
| **NK Cells** | **CD56bright (Cytokine)**, **CD56dim (Cytotoxic)**; **Missing-Self Recognition (MHC-I Loss)** |
| **B Cells / Plasma Cells** | **Tertiary Lymphoid Structures (TLS)**; **Antibody Production, Antigen Presentation**; **TLS = Good Prognosis, ICI Response** |

## 3. Extracellular Matrix (ECM) & Remodelling

| Component | Function in TME |
|-----------|-----------------|
| **Collagen I/III** | **Structural Scaffold**, **Stiffness** → Mechanotransduction (YAP/TAZ, Integrin/FAK), **Fibrosis** |
| **Hyaluronan (HA)** | **Hydration, Volume**, **CD44/RHAMM Binding** → PI3K/AKT, EMT, Chemoresistance; **PEGPH20** (PEGylated Hyaluronidase) Trials |
| **MMPs (Matrix Metalloproteinases)** | **ECM Degradation** → Invasion, Angiogenesis (VEGF Release), Growth Factor Activation; **MMP2/9 (Gelatinases)**, **MMP14 (MT1-MMP)** |
| **TIMPs** | **MMP Inhibitors**; **TIMP1-4**; **Imbalance MMP/TIMP = Invasion** |
| **LOX/LOXL2** | **Collagen Cross-linking** → Stiffness, Pre-metastatic Niche, Chemoresistance; **LOX Inhibitors** (Simtuzumab - Failed) |

## 4. Tumour Vasculature & Angiogenesis

### Angiogenic Switch

| Phase | Description |
|-------|-------------|
| **Pre-switch** | **Pro-angiogenic ≈ Anti-angiogenic**; Tumour ≤1-2mm, Dormant |
| **Switch ON** | **Pro-angiogenic > Anti-angiogenic**; **Hypoxia/HIF-1α**, **Oncogenes (RAS, MYC)**, **Loss of TSGs (TP53, VHL)** |
| **Result** | **Sprouting Angiogenesis**, **Vessel Co-option**, **Vasculogenic Mimicry**, **Vasculogenesis (BM-Derived EPCs)** |

### Tumour Vasculature Abnormalities

| Feature | Consequence |
|---------|-------------|
| **Chaotic Architecture** | **Heterogeneous Perfusion**, **Hypoxic Regions**, **Drug Delivery Impairment** |
| **Leaky Vessels** | **High Interstitial Pressure**, **Oedema**, **Metastatic Intravasation** |
| **Poor Pericyte Coverage** | **Vessel Instability**, **Increased Permeability** |
| **Abnormal Basement Membrane** | **Impaired T-cell Extravasation** |

### Anti-Angiogenic Therapy

| Agent | Target | Resistance Mechanisms |
|-------|--------|----------------------|
| **Bevacizumab** | **VEGF-A** | **Alternative Pro-angiogenic Factors** (FGF, PDGF, Ang2), **Vessel Co-option**, **Invasive Switch**, **Myeloid Cell Recruitment** |
| **Aflibercept** | **VEGF-A, VEGF-B, PlGF** | Similar |
| **Ramucirumab** | **VEGFR2** | Similar |
| **TKIs (Sunitinib, Pazopanib, Axitinib, Lenvatinib)** | **VEGFR1-3, PDGFR, FGFR, c-KIT** | **Off-target Effects**, **Alternative Pathways** |

## 5. Immune Checkpoints in TME

| Checkpoint | Ligand/Receptor | Expression | Mechanism | Clinical |
|------------|----------------|------------|-----------|----------|
| **PD-1 / PD-L1** | **PD-1 (T cells) ↔ PD-L1/PD-L2 (Tumour, APCs, Macrophages)** | T cell exhaustion | **Inhibits TCR/CD28 Signalling** (SHP-2 → PI3K/AKT, RAS/ERK) | **Pembrolizumab, Nivolumab, Atezolizumab, Durvalumab, Cemiplimab** |
| **CTLA-4** | **CTLA-4 (Tregs, Activated T) ↔ CD80/CD86 (APCs)** | Early activation | **Outcompetes CD28** (Higher Affinity), **Trans-endocytosis of CD80/86** | **Ipilimumab, Tremelimumab** |
| **LAG-3** | **LAG-3 (Exhausted T) ↔ MHC-II (APCs)** | Exhaustion | **Inhibits TCR Signalling**, **Treg Function** | **Relatlimab (Fixed-Dose with Nivolumab)** |
| **TIM-3** | **TIM-3 (Exhausted T, Tregs, DCs) ↔ Galectin-9, CEACAM1, HMGB1** | Exhaustion | **Inhibits TCR Signalling**, **DC Dysfunction** | **Sabatalimab, TSR-022** |
| **TIGIT** | **TIGIT (T, NK) ↔ CD155/PVR (Tumour, APCs)** | Exhaustion | **Competes with CD226 (DNAM-1)**, **Inhibits NK/T Cytotoxicity** | **Tiragolumab, Domvanalimab** |
| **VISTA** | **VISTA (Myeloid, T) ↔ VSIG3, PSGL-1** | Myeloid/T cell suppression | **pH-Dependent**, **Maintains Peripheral Tolerance** | **CA-170, HMBD-002** |

## 6. Metastatic Niche & Tertiary Lymphoid Structures

### Pre-Metastatic Niche Formation

| Step | Mechanism |
|------|-----------|
| **1. Primary Tumour Secretion** | **Exosomes** (Integrins, miRNAs, Proteins), **VEGF, TGF-β, LOX, S100A8/A9**, **MMPs** |
| **2. Bone Marrow Mobilisation** | **VEGFR1+ HPCs, MDSCs, Fibrocytes** → Circulation |
| **3. Target Organ Remodelling** | **Fibronectin Deposition** (by Fibrocytes), **Integrin Binding** (Exosome Integrins → Organotropism), **Inflammatory Cytokines** (TNF-α, IL-1β, IL-6) |
| **4. Immune Suppression** | **MDSC Recruitment**, **Treg Expansion**, **NK Cell Inhibition** |
| **5. Metastatic Cell Arrival** | **Adhesion, Extravasation, Colonisation** |

> **Organotropism**: **Exosome Integrins** (α6β4/α6β1 → Lung; αvβ5 → Liver; αvβ3 → Brain; α2β1 → Bone)

### Tertiary Lymphoid Structures (TLS)

| Feature | Detail |
|--------|--------|
| **Structure** | **Ectopic Lymphoid Organs** in TME: **T cell Zone**, **B cell Follicles (GCs)**, **HEVs, FDCs, DCs** |
| **Formation** | **LTα1β2, TNF, CXCL13, CCL19/21, IL-7, IL-22**; **Stromal Cell Differentiation** |
| **Function** | **Local Immune Priming**, **T/B Cell Activation**, **Antibody Production**, **Tfh/GC Reaction** |
| **Prognosis** | **TLS Presence = Good Prognosis**, **Predicts ICI Response** (Melanoma, Sarcoma, NSCLC, Breast, CRC) |
| **Clinical** | **TLS Signature** → Biomarker for ICI Benefit |

## 9. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **CAFs vs Normal Fibroblasts** | **CAFs**: Activated (FAP+, α-SMA+), Tumour-Promoting; **Normal**: Quiescent, Homeostatic |
| **M1 vs M2 Macrophages** | **M1**: Pro-inflammatory (IL-12, TNF-α, iNOS); **M2**: Pro-tumour (CD163, Arg1, IL-10, VEGF); **TAMs = M2 Dominant in TME** |
| **MDSCs vs Neutrophils/Monocytes** | **MDSCs**: Immature, Immunosuppressive (ARG1, iNOS, ROS); **Normal Neutrophils/Monocytes**: Mature, Anti-microbial |
| **Tregs vs Exhausted T Cells** | **Tregs**: FoxP3+, Suppressive by Design; **Exhausted T Cells**: PD-1+, TIM-3+, LAG-3+, Dysfunctional Effectors |
| **TLS vs Lymph Node** | **TLS**: Ectopic, No Capsule, In TME, Inducible; **Lymph Node**: Anatomy, Capsule, Constitutional |
| **Pre-Metastatic vs Metastatic Niche** | **Pre-Metastatic**: Primary Tumour Prepares Distant Organ BEFORE Arrival; **Metastatic**: After Tumour Cell Arrival |

**Mnemonic: TME-COMPONENTS**
- **C**AFs: **FAP+, α-SMA**, **ECM, Immune Exclusion (CXCL12)**
- **O**rigin of CAFs: **Resident Fibroblasts, MSCs, EndMT, EMT, Pericytes**
- **M**acrophages: **M1 (Anti) vs M2 (Pro)** — **CD163/CD206 = M2**
- **P**olarisation: **Tumour Factors (IL-4, IL-10, TGF-β) → M2 Shift**
- **O**ther Myeloid: **MDSCs (PMN/M-MDSC), ARG1, iNOS, ROS**
- **N**eutrophils: **PMN-MDSC (CD15+, LOX-1)**, **NETs**
- **E**ndothelial: **Abnormal Vasculature, Leaky, Hypoxic**
- **N**K Cells: **Missing-Self (MHC-I Loss)**
- **N**ew Checkpoints: **CTLA-4, LAG-3, TIM-3, TIGIT, VISTA**
- **N**K/T Exhaustion: **PD-1, TIM-3, LAG-3, TIGIT, TOX**
- **T**regs: **FoxP3+, CTLA-4, IL-10, Adenosine**
- **S**ubject TILs: **CD8+ = Good Prognosis**, **TLS = Good Prognosis**
- **E**CM: **Collagen, HA, MMPs, LOX, Stiffness, HA/PEGH20**

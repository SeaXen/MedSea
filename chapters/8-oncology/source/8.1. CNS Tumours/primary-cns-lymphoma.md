# Primary CNS Lymphoma (PCNSL)

> [!tip] **FCPS/MRCP Priority: HIGH**
> **PCNSL = Extranodal DLBCL (90%) Confined to CNS**; **Immunocompetent (60%) vs Immunocompromised (HIV, Post-Transplant — 40%)**; **EBV+ in Immunocompromised**; **MRI: Homogeneous Enhancement, Periventricular, Corpus Callosum **; **Diagnosis: Stereotactic Biopsy (AVOID Steroids Pre-Biopsy — Causes Lysis)**; **Treatment: HD-MTX Based (MATRix, R-MPV, De Angelis)**; **Consolidation: WBRT (Neurotoxicity >60yo), HD-AraC, ASCT**; **Elderly: HD-MTX + RT (NORDIC)**; **Relapse: HD-AraC, CAR-T, Ibrutinib, Lenalidomide**; **Ommaya for Intrathecal**.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise **Typical MRI Features** (Homogeneous, Periventricular, Corpus Callosum, "Butterfly")
- [ ] Understand **Contraindication of Steroids Before Biopsy** (Rapid Lysis → False Negative)
- [ ] Select **Induction Regimen**: **MATRix (Rituximab, MTX, Cytarabine, Thiotepa)**, **R-MPV**, **De Angelis (MTX-based)**
- [ ] Choose **Consolidation Strategy**: **WBRT (Neurotoxicity Risk >60yo)**, **HD-AraC**, **ASCT**
- [ ] Manage **Elderly PCNSL** (NORDIC: HD-MTX + RT)
- [ ] Treat **Immunocompromised PCNSL** (HIV: ART + Chemo; Post-Tx: Reduce Immunosuppression)
- [ ] Sequence **Relapse Therapy**: **HD-AraC, CAR-T, Ibrutinib, Lenalidomide, Clinical Trials**
- [ ] Know **Ommaya Reservoir** for Intrathecal Therapy/CSF Sampling

## 2. Definition & Epidemiology

| Feature | Detail |
|---------|--------|
| **Definition** | **Extranodal Diffuse Large B-Cell Lymphoma (DLBCL)** confined to CNS (Brain, Eyes, Leptomeninges, Spinal Cord) **Without Systemic Disease** at Diagnosis |
| **Incidence** | **4-5% of Primary Brain Tumours**; ~0.5-1/100,000/year; Rising (Aging, Immunosuppression) |
| **Prevalence** | Median OS **3-5 years** (Modern HD-MTX Protocols); 5-yr OS **40-50%** |
| **Peak Age** | **60-70 years** (Immunocompetent); **30-40** (HIV) |
| **Immunocompetent vs Immunocompromised** | **60% Immunocompetent**; **40% Immunocompromised** (HIV, Post-Transplant, Autoimmune on Immunosuppression) |
| **Risk Factors** | **Immunosuppression** (HIV/AIDS, Post-Transplant, Chronic), **Autoimmune Diseases**, **Age** |

## 3. Aetiology & Pathophysiology

```mermaid
flowchart LR
    A[B-Cell in CNS] --> B[Oncogenic Transformation]
    B --> C1[**Immunocompetent**<br/>**MYD88 L265P (70%)**<br/>**CD79B Mut (20%)**<br/>**PIM1, BTG1, CDKN2A Del**<br/>**NF-κB Pathway Activation**]
    B --> C2[**Immunocompromised (HIV/Post-Tx)**<br/>**EBV+ (Latency III: LMP1, EBNA2)**<br/>**EBV-Driven Proliferation**<br/>**Similar Mutations + EBV**]
    C1 --> D[**DLBCL Phenotype**<br/>**CD20+, BCL6+, MUM1+, BCL2+**]
    C2 --> D
```

### Key Mutations (Immunocompetent PCNSL)

| Gene | Frequency | Pathway |
|------|-----------|---------|
| **MYD88 L265P** | **70%** | **NF-κB Activation** (BTK Target) |
| **CD79B** | **20%** | **BCR Signalling** (BTK Target) |
| **PIM1** | **30%** | **Cell Survival** |
| **BTG1** | **15%** | **Cell Cycle** |
| **CDKN2A Deletion** | **40%** | **Cell Cycle** |
| **BCL6 Translocation** | **30%** | **Germinal Centre Origin** |

## 4. Clinical Features

| Feature | Description |
|---------|-------------|
| **Focal Neurological Deficits** | Hemiparesis, Aphasia, Visual Field Defects (Common) |
| **Neurocognitive/Behavioural** | Confusion, Personality Change, Memory Loss |
| **Headache** | Raised ICP (Hydrocephalus) |
| **Seizures** | **10-20%** |
| **Cranial Nerve Palsies** | Leptomeningeal Spread |
| **Ocular Involvement** | **Vitreoretinal Lymphoma (15-25%)** — Floaters, Blurred Vision, Uveitis |
| **Immunocompromised** | Rapid Progression, Multiple Lesions, EBV+ |

---

## 5. MRI Features (Typical)

| Feature | Description |
|---------|-------------|
| **Enhancement** | **Homogeneous, Intense** ("Lightbulb") on T1 Post-Gadolinium |
| **Location** | **Periventricular (60%)**, **Corpus Callosum (30-40%)**, **Basal Ganglia, Thalamus, Cerebellum** |
| **"Butterfly" Lesion** | **Crosses Corpus Callosum** (Bihemispheric) |
| **Oedema** | **Variable (Often Less than GBM)** |
| **Necrosis** | **Rare** (vs GBM) |
| **Leptomeningeal Enhancement** | **10-20%** |
| **Eye Involvement** | **Vitreoretinal** (Chorioretinal Lesions, Vitritis) |

> **Differential:** GBM (Ring Enhancement, Necrosis), Metastases (Multiple, Junction), Tumefactive Demyelination (Open Ring, Clinical Context), Abscess (Rim Enhancement, DWI Restriction)

## 6. Diagnosis & Investigations

| Investigation | Role | Key Details |
|---------------|------|-------------|
| **MRI Brain + Orbits (Gold Standard)** | **Diagnosis, Extent, Baseline** | **Homogeneous Enhancement**, Periventricular, Corpus Callosum, **DWI Restriction** (High Cellularity) |
| **Stereotactic Biopsy** | **Definitive Diagnosis** | **AVOID STEROIDS ≥7-10 Days Pre-Biopsy** (Causes Rapid Lysis → False Negative Rate ↑) |
| **CSF Analysis** | **Leptomeningeal / Vitreoretinal** | **Cytology (Low Sensitivity)**, **Flow Cytometry**, **EBV PCR**, **IL-10 > IL-6 (Suggests Lymphoma)**, **ctDNA** |
| **Vitreous Biopsy** | **Ocular Involvement** | If Ocular Symptoms / MRI Negative |
| **Slit-Lamp Exam** | **Screening for Ocular Involvement** | Mandatory at Diagnosis |
| **HIV Serology** | **Immunocompromised Workup** | CD4 Count, Viral Load |
| **EBV PCR (CSF/Blood)** | **Immunocompromised PCNSL** | EBV+ in HIV/Post-Tx |
| **PET-CT Body** | **Exclude Systemic Disease** (Required for "Primary" CNS) | **Essential** Before Diagnosis |
| **Testicular Ultrasound** | **Occult Testicular Primary** (Male >60) | If No Other Systemic Site |

## 7. Treatment — Induction (HD-MTX Based)

### Key Principle: **High-Dose Methotrexate (HD-MTX) is Backbone** (Crosses BBB)

| Regimen | Components | Key Trial / Use |
|---------|------------|-----------------|
| **MATRix** | **Rituximab 375mg/m² d0** + **MTX 3.5g/m² d1** + **Cytarabine 2g/m² d2** + **Thiotepa 20mg/m² d2** q21d ×4-6 cycles | **IELSG32 / MATRix Trial**: **CR 70%, PFS Superior to R-MPV** |
| **R-MPV** | **Rituximab + MTX 3.5g/m² + Procarbazine 100mg/m² d1-5 + Vincristine 1.4mg/m²** q21d | **Standard (Historical)** |
| **De Angelis** | **MTX 3.5g/m² d1, 15, 29** + **Cytarabine 1g/m² d2, 16, 30** + **Thiotepa 20mg/m² d2** q28d ×3-4 cycles | **Italian Standard** |
| **NORDIC (Elderly)** | **HD-MTX 3.5g/m² q2w ×4-6** + **RT 45Gy (if CR/PR)** | **Age >70 / Frail** |

### HD-MTX Administration Protocol

| Component | Details |
|-----------|---------|
| **Dose** | **3.5-8 g/m²** (Typically 3.5-4g/m²) IV over 2-4 Hours |
| **Leucovorin Rescue** | **Start 24h Post-MTX**, **Dose Based on MTX Levels** (e.g., 15mg q6h until MTX <0.05µM) |
| **Hydration** | **2.5-3L/m²/24h** (Urine Output >2L/d, Alkalinisation pH>7) |
| **Monitoring** | **MTX Levels at 24, 48, 72h** (Extend Leucovorin if Delayed Clearance) |
| **Toxicity** | **Renal (Crystallisation)**, **Hepatic**, **Myelosuppression**, **Mucositis**, **Neurotoxicity (Acute: Confusion, Seizures; Chronic: Leukoencephalopathy)** |

## 8. Consolidation Strategies

| Strategy | Indication | Key Evidence |
|----------|------------|--------------|
| **WBRT (Whole Brain RT)** | **Historical Standard** | **CNS Relapse ↓; BUT Neurotoxicity (Dementia, Ataxia) >60yo** |
| **HD-AraC (High-Dose Cytarabine)** | **Non-RT Consolidation** | **3g/m² q12h ×4 doses q2-4wk ×3-4 cycles**; **Less Neurotoxicity** |
| **Autologous Stem Cell Transplant (ASCT)** | **Young, Fit, CR/PR** | **BEAM / TBI Conditioning**; **Long-Term Remission** |
| **Observation** | **Elderly / Frail / CR after Induction** | **Close Surveillance** |

### Consolidation Decision Algorithm

```mermaid
flowchart TD
    A[Post-Induction (CR/PR)] --> B{Age / Fitness}
    B -->|**<60yo, Fit, CR**| C[**ASCT (BEAM/TBI)** — **Preferred**<br/>**Longest PFS/OS**]
    B -->|**60-70yo, Fit**| D[**HD-AraC Consolidation** (3g/m² q12h ×4 q2-4wk ×3-4 cycles)<br/>**OR WBRT (45Gy/25fx) if No ASCT**]
    B -->|**>70 / Frail**| E[**WBRT 45Gy/25fx** (NORDIC) **OR Observation**<br/>**Neurotoxicity Risk High with WBRT**]
    C --> F[**Surveillance MRI q3mo ×2yr, q6mo ×3yr**]
    D --> F
    E --> F
```

## 9. Special Populations

### Immunocompromised PCNSL (HIV / Post-Transplant)

| Population | Management |
|------------|------------|
| **HIV-Associated** | **ART Initiation/Optimisation (CD4↑, VL↓)** + **HD-MTX Based Chemo** (Tolerated if CD4>50); **EBV+** |
| **Post-Transplant (PTLD)** | **Reduce Immunosuppression (RIS)** — **First Step**<br/>**Rituximab ± HD-MTX** if No Response to RIS<br/>**EBV-Driven** |

### Elderly (>70yo / Frail)

| Approach | Details |
|----------|---------|
| **NORDIC Protocol** | **HD-MTX 3.5g/m² q2w ×4-6** + **RT 45Gy (if CR/PR)** |
| **Reduced Dose MTX** | **1.5-3g/m²** if Renal Impairment |
| **Avoid WBRT** | **Severe Neurotoxicity** |

## 10. Relapsed/Refractory PCNSL

```mermaid
flowchart TD
    A[Relapsed/Refractory PCNSL] --> B{Timing / Prior Therapy}
    B -->|**Late Relapse (>12mo), HD-MTX Naive**| C[**Re-induction HD-MTX Based (MATRix/R-MPV)**]
    B -->|**Early Relapse, Post-HD-MTX**| D[**HD-AraC (3g/m² q12h ×4 q2-4wk ×3-4 cycles)**<br/>**Standard Salvage**]
    D --> E[**Novel Agents**]
    E --> E1[**Ibrutinib 560mg Daily** (BTK Inhibitor — **MYD88/CD79B Mut**)]<br/>**Phase 2: ORR 50-60%**]
    E --> E2[**Lenalidomide 25mg d1-21 q28d**<br/>**Immunomodulatory, CNS Penetration**]
    E --> E3[**CAR-T (CD19)**<br/>**Autologous / Allogeneic**<br/>**Neurotoxicity (ICANS) Risk**]
    E --> E4[**Nivolumab/Pembrolizumab** (ICI)<br/>**Phase 2: Modest Activity**]
    E --> E5[**Ommaya Reservoir** — **Intrathecal MTX/AraC/ Rituximab**<br/>**For Leptomeningeal / Recurrent**]
```

## 11. Ommaya Reservoir

| Indication | Details |
|------------|---------|
| **Leptomeningeal Disease** | **Intrathecal MTX 12mg / Cytarabine 50mg / Rituximab 25mg** (Weekly → Monthly) |
| **CSF Sampling** | **Serial Cytology/Flow/PCR** Without Repeated LPs |
| **Placement** | **Frontal Horn of Lateral Ventricle** (Stereotactic/Endoscopic) |
| **Complications** | **Infection (5-10%)**, **Obstruction**, **Haemorrhage**, **Seizure** |

## 14. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Steroids Pre-Biopsy** | **ABSOLUTELY CONTRAINDICATED** — **Causes Apoptosis/Lysis Within Hours** — **False Negative Rate ~50% if Given** |
| **HD-MTX vs Standard MTX** | **HD-MTX: 3.5-8 g/m²** (Crosses BBB); **Standard: <1 g/m²** (Does Not Cross) |
| **WBRT Neurotoxicity** | **Dementia, Ataxia, Incontinence**; **Delayed (Months-Years)**; **>60yo High Risk** — **Avoid in Elderly** |
| **MATRix vs R-MPV** | **MATRix: Ritux + MTX + Ara-C + Thiotepa** (IELSG32: Superior PFS); **R-MPV: Ritux + MTX + Procarbazine + Vincristine** (Older) |
| **ASCT Conditioning** | **BEAM (BCNU, Etoposide, Ara-C, Melphalan)** OR **TBI-Based** — **No TBI Preferred (Neurotoxicity)** |
| **HD-AraC Dose** | **3g/m² q12h ×4 doses** = **12g/m² per Cycle** (High-Dose, Crosses BBB) |
| **Ommaya vs LP** | **Ommaya: Repeated Access, Intrathecal Therapy, Less Trauma**; **LP: Diagnostic, Single Access** |

**Mnemonic: PCNSL-HDMTX**
- **P**CNSL: **DLBCL (90%)**, CNS Only, **No Systemic** (PET-CT Body)
- **C**NS Lymphoma: **Homogeneous Enhancement**, Periventricular, Corpus Callosum, Butterfly
- **N**o Steroids Pre-Biopsy (Lysis → False Negative)
- **S**tereotactic Biopsy (Gold Standard)
- **L**uminal: MYD88 L265P (70%), CD79B (20%) → BTK Target
- **H**D-MTX Backbone: **3.5-8g/m²** (MATRix, R-MPV, De Angelis, NORDIC)
- **D**iagnosis: Biopsy + CSF (IL-10>IL-6) + HIV/EBV + PET-CT Body
- **M**ATRix: **Ritux + MTX + Ara-C + Thiotepa** (Best Induction)
- **T**reatment Consolidation: **ASCT <60**, **HD-AraC 60-70**, **WBRT >70** (Neurotox)
- **X** Relapse: **HD-AraC**, **Ibrutinib (MYD88)**, **CAR-T**, **Lenalidomide**, **Ommaya**

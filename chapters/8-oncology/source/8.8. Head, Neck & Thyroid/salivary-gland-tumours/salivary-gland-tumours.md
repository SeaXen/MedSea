# Salivary Gland Tumours

> [!tip] **FCPS/MRCP Priority: MEDIUM**
> **Salivary Gland Tumours**: **Parotid 80%** (Benign 80%), **Submandibular 10-15%** (Malignant 50%), **Minor Glands 5-10%** (Malignant 70-80%); **Malignant Histologies**: **Mucoepidermoid (30%)**, **Adenoid Cystic (AdCC) 15-20%**, **Acinic Cell (10%)**, **Salivary Duct Carcinoma**, **Carcinoma Ex Pleomorphic Adenoma**, **Epithelial-Myoepithelial**, **Myoepithelial**; **Surgery: Facial Nerve Preservation (Parotid)** — **Superficial Parotidectomy (Low-grade, Confined) vs Total Parotidectomy (High-grade, Deep Lobe, Facial Nerve Involvement)**; **Adjuvant RT**: High-grade, Positive Margins, PNI, T3-4, N+; **AdCC**: **Perineural Invasion (PNI) Hallmark**, **Late Distant Mets (Lung)**, **Long Natural History**; **Metastatic**: Clinical Trials, Platinum-based, **TKIs (HER2, NTRK, RET, TRK)**.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Classify **Salivary Gland Malignancies** by histology and grade
- [ ] Apply **Surgery Principles**: **Facial Nerve Preservation** (Parotid) — **Superficial vs Total Parotidectomy**
- [ ] Indicate **Adjuvant RT**: High-grade, Margins+, PNI, T3-4, N+
- [ ] Recognise **Adenoid Cystic Carcinoma (AdCC)**: PNI, Late Lung Mets, Long Course
- [ ] Manage **Mucoepidermoid Carcinoma** by Grade (Low/Intermediate/High)
- [ ] Sequence **Metastatic Therapy**: Clinical Trials, Platinum-based, Targeted (HER2, NTRK, RET)

## 2. Epidemiology & Anatomy

| Gland | Frequency | Benign vs Malignant | Common Benign | Common Malignant |
|-------|-----------|---------------------|---------------|------------------|
| **Parotid** | **80%** | **Benign 80%** (Pleomorphic Adenoma, Warthin) | Pleomorphic Adenoma, Warthin | **Mucoepidermoid, AdCC, Acinic Cell** |
| **Submandibular** | **10-15%** | **Malignant 50%** | Pleomorphic Adenoma | **AdCC, Mucoepidermoid** |
| **Sublingual** | <1% | **Malignant 80%** | Rare | **AdCC, Mucoepidermoid** |
| **Minor Glands** | **5-10%** | **Malignant 70-80%** | Rare | **AdCC, Mucoepidermoid** |

## 3. Histological Classification (WHO 2017/2022)

### Malignant Tumours

| Histology | Frequency | Grade | Key Features | Prognosis |
|-----------|-----------|-------|--------------|-----------|
| **Mucoepidermoid Carcinoma (MEC)** | **30%** | **Low / Intermediate / High** | **Mucous, Epidermoid, Intermediate Cells**, **MAML2 Fusion (CRTC1-MAML2)** | **Low: Excellent; High: Poor** |
| **Adenoid Cystic Carcinoma (AdCC)** | **15-20%** | **Always High-Grade Behaviour** | **Cribriform, Tubular, Solid**, **MYB-NFIB Fusion**, **Perineural Invasion (Hallmark)** | **Long Natural History, Late Lung Mets** |
| **Acinic Cell Carcinoma** | **10%** | **Low-Intermediate** | **Serous Acinar Differentiation**, **NRAS/HRAS Mut**, **NRF2** | **Good** |
| **Salivary Duct Carcinoma** | **5-10%** | **High** | **Apocrine Differentiation**, **AR+, HER2+, GATA3+**, **TP53, PIK3CA** | **Poor (Aggressive)** |
| **Carcinoma Ex Pleomorphic Adenoma (CXPA)** | **5-10%** | **Variable (Depends on Invasive Component)** | **Arising in Pleomorphic Adenoma**, **High-Grade** | **Variable** |
| **Epithelial-Myoepithelial Carcinoma** | **<5%** | **Low-Intermediate** | **Biphasic (Epithelial + Myoepithelial)**, **HRAS** | **Intermediate** |
| **Myoepithelial Carcinoma** | **<5%** | **High** | **Myoepithelial Differentiation**, **PLAG1, HMGA2** | **Variable** |
| **Secretory Carcinoma** | **Rare** | **Low-Intermediate** | **ETV6-NTRK3 Fusion** | **Good** |
| **Hyalinizing Clear Cell Carcinoma** | **Rare** | **Low** | **Clear Cells, Hyaline Stroma**, **ETV6-FUS** | **Good** |

## 4. Clinical Features

| Feature | Description |
|---------|-------------|
| **Painless Mass** | **Most Common** (Parotid: Preauricular/Posterior to mandible) |
| **Facial Nerve Palsy** | **Sign of Malignancy** (Invasion) — **Urgent Assessment** |
| **Pain** | **Late**, Perineural Invasion (AdCC) |
| **Trismus** | Medial Pterygoid Involvement |
| **Skin Involvement** | Advanced (Fixation, Ulceration) |
| **Lymph Nodes** | **High-Grade / AdCC / Salivary Duct** — Early Nodal Spread |

## 5. Staging (AJCC 8th Edition)

| T Stage | Description |
|---------|-------------|
| T1 | ≤2cm, No Extraparenchymal Extension |
| T2 | >2-4cm, No Extraparenchymal Extension |
| T3 | >4cm **OR** Extraparenchymal Extension (Soft Tissue, Skin, Mandible, Facial Nerve) |
| T4a | **Skin, Mandible, Maxilla, External Auditory Canal** |
| T4b | **Skull Base, Pterygoid Plates, Carotid Encasing** |

| N Stage | Description |
|---------|-------------|
| N1 | Single Ipsilateral ≤3cm |
| N2a | Single Ipsilateral 3-6cm |
| N2b | Multiple Ipsilateral ≤6cm |
| N2c | Bilateral/Contralateral ≤6cm |
| N3 | >6cm |

## 6. Diagnosis & Investigations

| Investigation | Role | Key Details |
|---------------|------|-------------|
| **Ultrasound + FNAC** | **First-Line** | **FNAC: Benign vs Malignant, Grade Low/High** — **May Not Subtype** |
| **MRI (Parotid/Submandibular)** | **Gold Standard Local Staging** | **Facial Nerve Relationship**, **Deep Lobe**, **Perineural Spread**, **Bone Invasion** |
| **CT Neck** | **Alternative / Nodal Staging** | **Cortical Bone**, **Nodes** |
| **PET-CT** | **High-Grade / Advanced / Recurrence** | **FDG-Avid** (Low-Grade Less Avid) |
| **Core Needle Biopsy** | **If FNAC Non-Diagnostic** | **Better Architecture for Subtyping** |
| **Molecular Testing** | **Targeted Therapy** | **HER2, NTRK, RET, MAML2, MYB, ETV6-NTRK3** |

## 7. Management

### Surgery (Parotid)

```mermaid
flowchart TD
    A[Parotid Malignancy] --> B{Facial Nerve Involvement}
    B -->|**No (Nerve Intact)**| C{Grade / Extent}
    C -->|**Low-Grade, Confined to Superficial Lobe**| D[**Superficial Parotidectomy** (Facial Nerve Dissection + Preservation)]
    C -->|**High-Grade, Deep Lobe, >4cm, Extrapharyngeal**| E[**Total Parotidectomy** (Superficial + Deep Lobe, Facial Nerve Dissection)]
    B -->|**Yes (Nerve Invaded)**| F[**Total Parotidectomy + Facial Nerve Resection (Segmental)**<br/>**Immediate Reconstruction: Cable Graft (Sural) / Hypoglossal-Facial Anastomosis**]
    D --> G[**Neck Dissection**]
    E --> G
    F --> G
    G -->|**cN0, Low-Grade**| H[**Observation** (No Elective ND)]
    G -->|**cN0, High-Grade / AdCC / Salivary Duct**| I[**Elective Selective ND (I-III)**]
    G -->|**cN+**| J[**Therapeutic Neck Dissection (I-V)**]
```

### Adjuvant Radiotherapy

| Indication | Dose |
|------------|------|
| **High-Grade (MEC High, AdCC, Salivary Duct, CXPA, Myoepithelial)** | **60-66Gy/30-33fx** to Primary Bed + Neck (if indicated) |
| **Positive Margins (R1)** | **60-66Gy** (+ Boost to Margin) |
| **Close Margins (<2mm)** | **60-66Gy** |
| **Perineural Invasion (PNI)** | **60-66Gy** (Include Neural Pathways) |
| **T3-4 (Size >4cm / Extrapharyngeal)** | **60-66Gy** |
| **N+ (Nodal Metastases)** | **60-66Gy** to Nodal Levels (Ipsilateral/Bilateral) |
| **Lymphovascular Invasion (LVI)** | **Consider** |

**AdCC Specific:** **RT for PNI** — **Include Skull Base/Foramina if PNI Proximal**.

### Submandibular / Minor Glands

| Gland | Surgery | Adjuvant RT |
|-------|---------|-------------|
| **Submandibular** | **Submandibular Gland Excision** + **Facial Nerve (Marginal Mandibular Branch) Preservation** | **Same Indications** (High-Grade, Margins, PNI, N+) |
| **Sublingual/Minor** | **Wide Local Excision** (Adequate Margins) | **Same Indications** |

## 8. Histology-Specific Management

### Mucoepidermoid Carcinoma (MEC)

| Grade | Features | Surgery | Adjuvant RT |
|-------|----------|---------|-------------|
| **Low** | **Mucous Cells Predominant**, **MAML2+**, **No PNI** | **Superficial Parotidectomy** (if confined) | **None** (if R0, N0) |
| **Intermediate** | Mixed | **Total Parotidectomy** (if Deep Lobe/Extension) | **Consider** (if Margins Close/PNI) |
| **High** | **Solid/Epidermoid Predominant**, **High Mitoses**, **PNI+** | **Total Parotidectomy** + **Selective ND** | **Yes (60-66Gy)** |

### Adenoid Cystic Carcinoma (AdCC)

| Feature | Management |
|---------|------------|
| **PNI (Hallmark)** | **RT Mandatory** (Include Neural Pathways to Skull Base) |
| **Late Distant Mets** | **Lung (Most Common)**, **Bone, Liver** — **Can Occur 10-20 Years Later** |
| **Long Natural History** | **Indolent but Relentless** — **Median Survival 10-15 Years** |
| **Surgery** | **Total Parotidectomy** (PNI Often Deep) + **Selective ND** |
| **RT** | **66Gy** to Primary + **Neural Pathways** (Mandibular, Maxillary Divisions) |
| **Systemic** | **Clinical Trials** (TKIs: Lenvatinib, Sunitinib), **Platinum-Based** (Poor Response) |

### Acinic Cell Carcinoma

| Feature | Management |
|---------|------------|
| **Low-Intermediate Grade** | **Superficial Parotidectomy** (if Confined) |
| **Rare PNI** | **RT if PNI+/Margins+** |
| **Late Recurrence** | **Surveillance Long-Term** |
| **NRAS/HRAS Mut** | **Potential MEK Inhibitor Target** (Investigational) |

### Salivary Duct Carcinoma (SDC)

| Feature | Management |
|---------|------------|
| **High-Grade, Apocrine, AR+, HER2+** | **Total Parotidectomy + Neck Dissection** |
| **HER2+ (20-30%)** | **Trastuzumab-based** (Trastuzumab + Docetaxel, T-DXd) — **Investigational** |
| **Androgen Receptor (AR+)** | **Anti-Androgen (Bicalutamide/Enzalutamide)** — **Case Reports** |
| **RT** | **Yes (High-Grade)** |

## 12. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Superficial vs Total Parotidectomy** | **Superficial**: Facial Nerve Dissection, Remove Superficial Lobe Only (Low-Grade, Confined); **Total**: Remove Superficial + Deep Lobe (High-Grade, Deep Lobe, >4cm, Nerve Involvement) |
| **Facial Nerve — Preserve vs Resect** | **Preserve if No Gross Invasion** (Dissect Off Tumour); **Resect Segmentally if Gross Invasion** → **Immediate Reconstruction** |
| **AdCC PNI — RT Field** | **Include Neural Pathways** (Mandibular V3, Maxillary V2, Skull Base Foramina) — **Not Just Primary Bed** |
| **MEC Grading** | **Low: Mucous >90%, No PNI**; **Intermediate: Mixed**; **High: Solid/Epidermoid >50%, High Mitoses, PNI+** |
| **Salivary Duct vs MEC** | **SDC: Apocrine, AR+, HER2+, High-Grade Always**; **MEC: Mucous/Epidermoid/Intermediate, Variable Grade** |
| **Minor Gland Malignancy** | **Higher Malignant % (70-80%)** — **Wide Excision + RT** (Same Principles) |

**Mnemonic: SALIVARY**
- **S**alivary: **Parotid 80% (Benign 80%)**, **Submandibular 10-15% (Mal 50%)**, **Minor 5-10% (Mal 70%)**
- **A**denoid Cystic (AdCC): **PNI Hallmark**, **MYB-NFIB**, **Late Lung Mets**, **Long History**
- **L**ow vs High Grade: **MEC Low/Int/High**, **AdCC Always High Behaviour**
- **I**nvasion: **Facial Nerve Preserve** (Dissect) vs **Resect** (Gross Invasion) → **Reconstruct**
- **V**isualise (MRI): **Gold Standard** — Facial Nerve, Deep Lobe, PNI, Bone
- **A**djuvant RT: **High-Grade, Margins+, PNI, T3-4, N+** → **60-66Gy**
- **R**ecurrence: **AdCC Late (10-20yr Lung)**; **Others Earlier**
- **Y**ield: **Surgery Mainstay**, **RT Adjuvant**, **Systemic Limited (HER2, NTRK, RET, AR)**

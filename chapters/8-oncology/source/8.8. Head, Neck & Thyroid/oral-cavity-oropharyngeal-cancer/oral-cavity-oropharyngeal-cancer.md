# Oral Cavity & Oropharyngeal Cancer

> [!tip] **FCPS/MRCP Priority: HIGH**
> **Oropharyngeal Cancer (OPC)**: **HPV+ (p16+) 70%** (Tonsil, Base of Tongue) — **Better prognosis**; **Oral Cavity Cancer (OCC)**: **HPV-** (Tobacco/Alcohol) — **Worse prognosis**; **AJCC 8th Staging**: **Separate staging for HPV+ OPC** (Clinical + Pathological); **Early (T1-2 N0)**: **Surgery (TORS for OPC) OR RT** — Equivalent; **Locally Advanced**: **CCRT (Cisplatin 100mg/m² q3w ×3)** — **Standard**; **HPV+ De-escalation**: **PATHOS (Adjuvant)**, **ECOG 3311 (Transoral → Risk-adapted)**, **De-ESCALaTE HPV (Cisplatin vs Cetuximab)** — **Cetuximab INFERIOR**; **Neck: Selective ND (I-III) for cN0, Comprehensive for cN+**; **Sentinel Node for OCC cN0**

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Distinguish **HPV+ Oropharyngeal** vs **HPV- Oral Cavity** by anatomy, aetiology, prognosis
- [ ] Apply **AJCC 8th Staging** — **Separate HPV+ OPC staging** (Clinical/Pathological)
- [ ] Select **primary treatment**: **TORS vs RT** for early; **CCRT (Cisplatin) vs Surgery+Adjuvant** for advanced
- [ ] Interpret **de-escalation trial results** (Cetuximab inferior, RT dose reduction ongoing)
- [ ] Perform **neck management**: **Selective ND (I-III)** vs **Comprehensive**; **SLN for oral cavity**
- [ ] Stage **p16+ OPC** correctly — **Clinical vs Pathological staging differ**

## 2. Definition & Epidemiology

| Feature | Oropharyngeal (OPC) | Oral Cavity (OCC) |
|---------|---------------------|-------------------|
| **Subsites** | **Tonsil (Palatine)**, **Base of Tongue (BOT)**, Soft Palate, Pharyngeal Wall | **Mobile Tongue**, **Floor of Mouth**, **Buccal Mucosa**, **Retromolar Trigone**, **Hard Palate**, **Upper/Lower Alveolus** |
| **Histology** | **SCC >95%** | **SCC >95%** |
| **HPV/p16** | **70-80% Positive** (HPV16) | **<5% Positive** |
| **Aetiology** | **HPV16 (Sexual transmission)** | **Tobacco + Alcohol (Synergistic)** |
| **Incidence** | **Rising** (HPV+) | **Stable/Declining** (Tobacco control) |
| **Peak Age** | 50-60 (HPV+) | 60-70 |
| **Prognosis** | **HPV+: Excellent (5-yr OS 85-90%)** | **5-yr OS ~50-60%** |

## 3. Aetiology & Pathophysiology

```mermaid
flowchart LR
    A[HPV16 Infection] --> B[E6/E7 Oncoproteins]
    B --> C1[E6 → p53 Degradation]
    B --> C2[E7 → pRb Degradation → p16 Overexpression]

    G --> H[TP53 Mutations (80%)]
    H --> I[**HPV- OCC: Genomic instability, Poor differentiation, Immunologically cold**]
```

## 4. Staging (AJCC 8th Edition)

### HPV+ Oropharyngeal Cancer — **Separate Staging System**

| Clinical Stage | T | N |
|----------------|---|---|
| **I** | T1-2 | N0-1 (≤6cm) |
| **II** | T1-2 | N2 (≤6cm, bilateral/contralateral) |
| | T3 | N0-2 |
| **III** | T4 | N0-2 |
| | Any T | N3 (>6cm) |
| **IV** | M1 | - |

**Pathological Stage** (Post-surgery):
| Stage | T | N |
|-------|---|---|
| **I** | T1-2 | N0 |
| **II** | T1-2 | N1 (≤6cm, ipsilateral) |
| | T3 | N0 |
| **III** | T1-2 | N2 (bilateral/contralateral ≤6cm) |
| | T3 | N1-2 |
| | T4 | N0-2 |
| **IV** | Any T | N3 (>6cm) OR M1 |

### HPV- Oral Cavity / Oropharyngeal — **Standard Staging**

| Stage | T | N |
|-------|---|---|
| **I** | T1 | N0 |
| **II** | T2 | N0 |
| **III** | T3 | N0 |
| | T1-3 | N1 |
| **IVA** | T4a | N0-1 |
| | T1-4a | N2 |
| **IVB** | Any T | N3 |
| | T4b | Any N |
| **IVC** | M1 | - |

**Key T Definitions (Oral Cavity):**
- T1: ≤2cm; T2: >2-4cm; T3: >4cm; T4a: Cortical bone, Deep tongue, Maxilla, Skin; T4b: Pterygoid plates, Skull base, Carotid

**Key N Definitions (Both):**
- N1: Single ipsilateral ≤3cm; N2a: Single ipsilateral 3-6cm; N2b: Multiple ipsilateral ≤6cm; N2c: Bilateral/contralateral ≤6cm; N3: >6cm

| Investigation | Role | Key Details |
|---------------|------|-------------|
| **Panendoscopy + Biopsy** | **Diagnosis** | **p16 IHC MANDATORY for OPC** (HPV surrogate) |
| **MRI Head & Neck** | **Local Staging** (T) | Deep invasion, Perineural, Bone, Nodes |
| **CT Chest/Abd/Pelvis** | Distant Mets | |

| **US + FNAC** | Nodal Assessment | **cN0 vs cN+** |
| **HPV/p16 Testing** | **OPC: p16 IHC (surrogate); HPV PCR/ISH confirmatory** | **p16+ = HPV-driven staging** |

## 6. Management

### Early Stage (T1-2 N0)

```mermaid
flowchart TD
    A[Early Stage T1-2 N0] --> B{Subsite}
    B -->|**Oropharyngeal (HPV+)**| C[**TORS (Transoral Robotic Surgery)**<br/>**Preferred** (Function preservation)<br/>**Pathological Risk → Adjuvant**<br/>**Low risk (T1-2, N0, ≤1cm margins, no LVI/PNI) → Obs**<br/>**Intermediate → Adjuvant RT (50-54Gy)**<br/>**High (Positive margins, ENE, ≥2 LNs+) → ChemoRT**]
    B -->|**Oropharyngeal (HPV+)**| D[**Primary RT (IMRT/VMAT) 70Gy/35fx**<br/>**Equivalent OS to Surgery**<br/>**Better swallowing outcomes debated**]
    B -->|**Oral Cavity (HPV-)**| E[**Surgery (Wide Local Excision + Neck Management)**<br/>**SLN Biopsy (cN0) → If +ve → Completion ND**<br/>**Adjuvant: RT (T3-4, Close margins, LVI/PNI) ± Chemo (Positive margins, ENE)**]
```

### Locally Advanced (T3-4, N+)

```mermaid
flowchart TD
    A[Locally Advanced HNC] --> B{HPV Status / Subsite}
    B -->|**HPV+ Oropharyngeal**| C[**CCRT: Cisplatin 100mg/m² q3w ×3 + IMRT 70Gy/35fx**<br/>**Standard (RTOG 0129, 0522)**<br/>**De-escalation Trials Ongoing**]
    B -->|**HPV- Oropharyngeal / Oral Cavity**| D[**Surgery (Composite Resection + Neck Dissection)**<br/>**Adjuvant RT ± Chemo**<br/>**INDICATIONS for ChemoRT**: Positive margins, ENE<br/>**INDICATIONS for RT alone**: Close margins, LVI, PNI, T3-4, N2-3, Perineural]
    C --> E[**Neck Dissection Post-CCRT** if Residual N+ (PET-CT at 12wk)]
    D --> F[**Planned Neck Dissection** for cN+]
```

### De-Escalation Trials (HPV+ OPC)

| Trial | Design | Key Result |
|-------|--------|------------|
| **De-ESCALaTE HPV** | CCRT: **Cisplatin vs Cetuximab** + RT | **Cetuximab INFERIOR** (OS HR 2.0, PFS HR 1.5) — **Do NOT use Cetuximab** |
| **RTOG 1016** | CCRT: **Cisplatin vs Cetuximab** + RT | **Cetuximab INFERIOR** (OS HR 1.45) |
| **ECOG 3311** | TORS → Risk-adapted Adjuvant: **Low (Obs), Int (RT 50Gy), High (ChemoRT)** | **Feasible, Low morbidity**, **Ongoing survival analysis** |
| **PATHOS** | TORS → Adjuvant: **Obs vs RT 50Gy vs RT 60Gy vs ChemoRT** | **Recruiting**, **Risk-stratified** |
| **Quarterback** | CCRT: **Reduced dose Cisplatin (60mg/m²) vs Standard** | **Ongoing** |

### Neck Management

| Scenario | Management |
|----------|------------|
| **OCC cN0** | **Sentinel Node Biopsy (SLNB)** — **If +ve → Completion ND (I-III)** |
| **OCC cN+** | **Comprehensive Neck Dissection (I-V)** |
| **OPC cN0** | **Ipsilateral Selective ND (I-III)** OR **RT to Neck** (if primary RT) |
| **OPC cN+** | **Comprehensive ND** (if surgery) OR **CCRT (+ Planned ND if residual)** |
| **Post-CCRT Residual Node** | **PET-CT at 12 weeks**; **If +ve → Neck Dissection** |

## 8. Viva Questions (MRCP PACES / FCPS)

| Question | Expected Answer |
|----------|-----------------|
| **55M, Tonsil mass, p16+, T2 N1 (ipsilateral 2cm). Clinical Stage?** | **HPV+ OPC Clinical Stage I** (T1-2 N0-1 ≤6cm = Stage I). |
| **Same patient, post-TORS: pT2 pN1 (2 nodes, 1 with ENE). Pathological Stage?** | **HPV+ OPC Pathological Stage III** (T1-2 N2 bilateral/contralateral OR ENE = Stage III). |
| **HPV+ OPC T2 N0 — TORS vs RT?** | **Both equivalent OS**; **TORS: Better pathology staging, Risk-adapted adjuvant (ECOG 3311)**; **RT: Non-invasive, Better swallowing (some data)**. |

| **Positive Margins / ENE post-surgery — Adjuvant?** | **ChemoRT (Cisplatin + RT 60-66Gy)** — **Standard for ENE/Margins+**. |
| **Post-CCRT Neck PET-CT — timing, action?** | **12 weeks post-RT**; **If SUV+ → Neck Dissection**; **If SUV- → Surveillance**. |

## 9. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Clinical vs Pathological Staging (HPV+ OPC)** | **Clinical**: Pre-treatment (Imaging/Exam) — **Uses N1-2 ≤6cm**; **Pathological**: Post-surgery — **Uses N1 (ipsi ≤6cm), N2 (bilat/contra ≤6cm), N3 (>6cm)** |
| **p16+ vs HPV PCR** | **p16 IHC = Surrogate** (Overexpression = HPV-driven); **HPV PCR/ISH = Confirmatory**; **p16+ ≠ always transcriptionally active HPV** (rare) |
| **De-escalation: Cetuximab vs RT Dose Reduction** | **Cetuximab REPLACES Cisplatin — FAILED**; **Current: RT dose reduction (60Gy), Reduced Cisplatin, Induction chemo → RT** |
| **TORS vs RT for Early OPC** | **No RCT OS difference**; **TORS: Staging, Avoids RT morbidity (xerostomia), Enables risk-adapted adjuvant**; **RT: Avoids surgery, Similar function** |
| **Selective vs Comprehensive ND** | **Selective (I-III)**: cN0 OPC, cN0 OCC (if SLNB -ve); **Comprehensive (I-V)**: cN+ OPC/OCC, ENE |
| **Post-CCRT Neck Dissection** | **PET-CT at 12 weeks** (not earlier — inflammation); **SUV+ → Dissection**; **SUV- → Observation** |

**Mnemonic: ORAL-ORO-PHARYNGEAL**
- **O**ropharyngeal: **HPV+ 70%** (Tonsil, BOT) — **p16+**, **Better OS**
- **R**isk: **Tobacco/Alcohol** (OCC) vs **HPV16** (OPC)
- **A**JCC 8th: **Separate HPV+ Staging** (Clinical + Pathological)
- **L**aryngeal: Separate (Larynx preservation)
- **O**ral Cavity: **HPV-**, **Tongue/FOM/Buccal** — **Surgery + SLNB**
- **R**esection: **TORS** (OPC Early) — **Function Preservation**
- **O**ngoing De-escalation: **Cetuximab FAILED** (De-ESCALaTE), **ECOG 3311 (TORS Risk-adapted)**, **PATHOS**
- **P**ost-CCRT Neck: **PET-CT 12wk → Dissection if +ve**
- **H**PV+ Staging: **Clinical I/II/III/IV**, **Path I/II/III/IV** (Different N categories)
- **A**djuvant ChemoRT: **Margins+ / ENE** → **Cisplatin + RT**
- **R**T Early OPC: **TORS vs RT** — **Equivalent OS**
- **Y**ounger: **HPV+ OPC** (50-60); **Older: OCC** (60-70)
- **N**eck: **OCC cN0 = SLNB**, **OPC cN0 = Selective ND/RT**
- **G**lobally: **HPV+ Rising**, **HPV- Declining**
- **E**GFR/Cetuximab: **NOT Standard** (Inferior to Cisplatin)
- **A**lcohol/Tobacco: **Synergistic** for OCC
- **L**ymph Nodes: **Levels I-III Selective**, **I-V Comprehensive**

**A3:** Separate Clinical/Pathological Staging; Clinical I/II/III/IV; Path I/II/III/IV

**Q4:** Early T1-2 N0?
**A4:** OPC: TORS (Preferred) OR RT; OCC: Surgery + SLNB

**Q5:** Locally Advanced?
**A5:** CCRT: Cisplatin 100mg/m² q3w ×3 + RT 70Gy — Standard

**Q6:** De-ESCALaTE HPV?
**A6:** Cetuximab vs Cisplatin + RT → Cetuximab INFERIOR (OS HR 2.0) — NO Cetuximab

**Q7:** ECOG 3311?
**A7:** TORS → Risk-adapted Adjuvant (Low: Obs, Int: RT 50Gy, High: ChemoRT)

**Q8:** Neck?
**A8:** OCC cN0: SLNB; OCC cN+: Comp ND; OPC cN0: Selective ND I-III/RT; OPC cN+: Comp ND

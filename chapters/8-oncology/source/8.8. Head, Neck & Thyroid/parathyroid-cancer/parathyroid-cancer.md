# Parathyroid Cancer

> [!tip] **FCPS/MRCP Priority: MEDIUM**
> **Parathyroid Cancer = Rare (<1% of Primary Hyperparathyroidism)**; **Functional (90% — Hyperparathyroidism) vs Non-functional**; **Genetic: MEN1, MEN2A, HPT-JT (CDC73/HRPT2), FIHP**; **Diagnosis: Severe Hypercalcaemia (Ca>3.0), PTH ↑↑, Neck Mass**; **Imaging: US, Sestamibi, 4D-CT, PET-CT (Choline/Fluorocholine)**; **Surgery: En Bloc Resection (Thyroid Lobectomy + Parathyroid + Surrounding Soft Tissue) + Central ND** — **No Capsule Rupture**; **Adjuvant RT for R1/R2/Close Margins**; **Metastatic: Cinacalcet (Calcimimetic), Denosumab, TKIs (Sunitinib, Lenvatinib) — Limited Efficacy**; **Recurrence 20-50% (Late, Local > Distant)**.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise **clinical presentation**: Severe Hypercalcaemia, PTH↑↑, Neck Mass
- [ ] Apply **imaging**: US, Sestamibi, 4D-CT, PET-CT (Choline)
- [ ] Perform **En Bloc Resection** (Thyroid + Parathyroid + Surrounding Tissue) + Central ND
- [ ] Indicate **Adjuvant RT** for Positive/Close Margins
- [ ] Manage **Metastatic Disease**: Cinacalcet, Denosumab, TKIs
- [ ] Screen for **Genetic Syndromes**: MEN1, MEN2A, HPT-JT (CDC73), FIHP
- [ ] Monitor **Recurrence**: PTH, Calcium, Imaging (Late, Local > Distant)

## 2. Definition & Epidemiology

| Feature | Detail |
|---------|--------|
| **Definition** | Malignant neoplasm of parathyroid chief cells; **Functional (Secretes PTH) 90%** |
| **Incidence** | **<1% of Primary Hyperparathyroidism**; ~1-2/million/year |
| **Prevalence** | 5-year OS ~85% (Localised), ~50% (Metastatic); **Recurrence 20-50%** (Late, 5-20 years) |
| **Peak Age** | 45-55 years |
| **Sex Ratio** | F:M = 1:1 (vs Adenoma F>M) |
| **Genetic** | **MEN1 (MEN1 gene)**, **MEN2A (RET)**, **HPT-JT (CDC73/HRPT2)**, **FIHP (Familial Isolated HPT)** |

---

## 3. Clinical Features

| Feature | Description |
|---------|-------------|
| **Severe Hypercalcaemia** | **Ca >3.0 mmol/L** (Often >3.5) — **Classic** |
| **PTH** | **Markedly Elevated** (Often >5x ULN) — **Inappropriate for Calcium** |
| **Neck Mass** | **Palpable, Hard, Fixed** (vs Adenoma: Soft, Mobile) |
| **Renal** | Nephrolithiasis, Nephrocalcinosis, Renal Impairment |
| **Bone** | Osteitis Fibrosa Cystica (Brown Tumours), Pathological Fractures |
| **GI** | Pancreatitis, Peptic Ulcers, Constipation |
| **Neuromuscular** | Weakness, Fatigue, Depression, Confusion (Hypercalcaemic Crisis) |
| **Non-functional** | Normocalcaemic, PTH Normal/↑, Neck Mass — **Rare** |

---

## 4. Diagnosis & Investigations

| Investigation | Role | Key Details |
|---------------|------|-------------|
| **Serum Calcium (Corrected)** | **Diagnostic** | **>2.8 mmol/L (Severe)** — **Prompts PTH Check** |
| **PTH (Intact)** | **Diagnostic** | **↑↑ (Often 5-10x ULN)** — **Inappropriate for Hypercalcaemia** |
| **Phosphate** | Low (PTH effect) | |
| **ALP** | ↑ (Bone Turnover) | |
| **Renal Function** | Assess Impairment | |
| **Neck Ultrasound** | **First-Line** | **Hypoechoic, Heterogeneous, Irregular, Vascular** |
| **Sestamibi Scan (99mTc-MIBI)** | **Functional Imaging** | **Dual Phase (Early/Delayed)** — **Retention in Parathyroid Cancer** |
| **4D-CT (Multiphase CT)** | **Anatomical + Functional** | **Arterial Phase Enhancement → Washout** — **High Sensitivity** |
| **PET-CT (Choline / Fluorocholine / FDG)** | **Metastatic Workup / Recurrence** | **Choline PET Superior for Parathyroid** |
| **Biopsy (FNA)** | **Generally Avoided** | **Risk of Seeding, Capsule Rupture** — **Diagnosis Usually Surgical** |

---

## 5. Differential Diagnosis

| Condition | Distinguishing Features |
|-----------|-------------------------|
| **Parathyroid Adenoma** | **Ca 2.6-3.0**, **PTH 2-3x ULN**, **Soft, Mobile**, **No Invasion** |
| **Primary Hyperplasia (4-Gland)** | **Ca Mild-Moderate**, **PTH ↑**, **Genetic (MEN1/2A)**, **All 4 Glands Enlarged** |
| **Tertiary Hyperparathyroidism** | **CKD Background**, **Autonomous**, **All 4 Glands** |
| **Familial Hypocalciuric Hypercalcaemia (FHH)** | **Ca Mild**, **PTH Normal/↑**, **Low Urinary Calcium (UCa/Cr <0.01)**, **CASR Mutation** |
| **Metastatic Cancer to Parathyroid** | **Known Primary**, **PTH Normal**, **Biopsy: Non-Parathyroid Markers** |
| **Thyroid Cancer (Medullary/Anaplastic)** | **Calcitonin/CEA (MTC)**, **Aggressive**, **Different IHC** |

## 6. Management

### Surgery (Curative Intent)

```mermaid
flowchart TD
    A[Parathyroid Cancer Suspected] --> B[**En Bloc Resection**]
    B --> B1[**Ipsilateral Thyroid Lobectomy + Isthmusectomy**]
    B --> B2[**Parathyroid Tumour + Surrounding Soft Tissue** (Fat, Muscle, Nerve if invaded)]
    B --> B3[**CENTRAL NECK DISSECTION (Level VI)** — **Routine**]
    B --> B4[**NO CAPSULE RUPTURE** — **Critical** (Seeding → Recurrence)]
    B1 --> C[**Frozen Section** (Confirm Parathyroid vs Thyroid)]
    C --> D[**Intraoperative PTH (ioPTH)** — **Drop >50% at 10min = Complete Resection**]
    D --> E{Pathology Margins}
    E -->|**R0 (Negative)**| F[**Surveillance**]
    E -->|**R1/R2 (Close/Positive)**| G[**Adjuvant Radiotherapy**]
```

### Adjuvant Radiotherapy

| Indication | Dose/Fractionation |
|------------|-------------------|
| **Positive Margins (R1)** | **50-54Gy/25-30fx** to Tumour Bed + Central Neck |
| **Close Margins (<2mm)** | **50-54Gy/25-30fx** |
| **Gross Residual (R2)** | **60-66Gy/30-33fx** (Definitive) |
| **Lymph Node Positive** | **50-54Gy** to Involved Levels |

### Medical Management (Metastatic / Unresectable / Recurrent)

| Agent | Mechanism | Indication |
|-------|-----------|------------|
| **Cinacalcet** | **Calcimimetic** (CaSR Allosteric Modulator) | **Hypercalcaemia Control** — **First-Line** (30-90mg/day, titrate to Ca) |
| **Denosumab** | **RANKL Inhibitor** | **Hypercalcaemia Refractory to Cinacalcet** / **Bone Mets** (120mg SC q4wk + Loading) |
| **Bisphosphonates (Zoledronic Acid)** | **Osteoclast Inhibition** | **Bone Mets / Hypercalcaemia** (4mg IV q3-4wk) |
| **Sunitinib** | **Multi-TKI (VEGFR, PDGFR, c-KIT)** | **Case Reports / Small Series** — **Partial Responses** |
| **Lenvatinib** | **VEGFR, FGFR, RET, KIT** | **Case Reports** — **Activity in Thyroid/Parathyroid** |
| **Clinical Trials** | — | **Preferred** (Rare Disease) |

### Genetic Screening & Surveillance

| Syndrome | Gene | Screening |
|----------|------|-----------|
| **MEN1** | **MEN1** | **Annual Calcium, PTH, Prolactin, MRI Pituitary, Pancreatic US/CT** |
| **MEN2A** | **RET** | **Annual Calcitonin, Calcium, PTH, Adrenal Imaging** |
| **HPT-JT** | **CDC73 (HRPT2)** | **Annual Ca, PTH, Renal US (Ossifying Fibroma), Jaw Imaging** |
| **FIHP** | **Undefined** | **Annual Ca, PTH, Family Cascade Testing** |

### Post-Operative Surveillance

| Modality | Schedule |
|----------|----------|
| **Serum Calcium + PTH** | **q3mo ×2yr, q6mo ×3yr, then Annually** (Late Recurrence) |
| **Neck US** | **q6mo ×2yr, then Annually** |
| **Sestamibi / 4D-CT / Choline PET** | **If Biochemical Recurrence (PTH↑, Ca↑)** |
| **Chest/Abdomen CT** | **If Distant Mets Suspected** (Lung, Liver, Bone) |

## 9. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Parathyroid Cancer vs Adenoma — PTH Level** | **Adenoma**: PTH 2-3x ULN; **Cancer**: PTH **5-10x ULN** (Markedly Elevated) |
| **En Bloc Resection — What Structures?** | **Ipsilateral Thyroid Lobectomy + Isthmus + Parathyroid Tumour + Surrounding Fat/Muscle/Nerve (if invaded) + Central ND (Level VI)** |
| **Capsule Rupture — Consequence?** | **Seeding → Local Recurrence** — **ABSOLUTELY AVOID** during dissection |
| **ioPTH Timing** | **Baseline (Pre-excision)** → **10 Minutes Post-excision** → **>50% Drop = Complete** |
| **Adjuvant RT Dose** | **50-54Gy/25-30fx** (Post-op Bed + Central Neck); **60-66Gy** if Gross Residual |
| **Cinacalcet vs Denosumab** | **Cinacalcet: Calcimimetic (Lowers PTH → Lowers Ca)** — **First-Line**; **Denosumab: RANKL Inhibitor (Direct Bone Effect)** — **Refractory/Cinacalcet Failure** |
| **Recurrence — Local vs Distant** | **Local/Regional > Distant**; **Late Recurrence (10-20 Years Common)** — **Lifelong Surveillance** |

**Mnemonic: PARATHYROID-CA**
- **P**arathyroid Cancer: **<1% pHPT**, **Functional 90% (Hypercalcaemia Ca>3, PTH↑↑)**
- **A**denoma vs Cancer: **PTH 2-3x vs 5-10x ULN**, **Soft vs Hard**, **Mobile vs Fixed**
- **R**esection: **En Bloc** — **Thyroid Lobectomy + Parathyroid + Soft Tissue + Central ND** — **NO CAPSULE RUPTURE**
- **A**dvjuvant RT: **R1/R2/Close Margins** → **50-54Gy to Bed + Central Neck**
- **T**H: **PTH 5-10x ULN**, **Ca >3.0** — **Severe Hypercalcaemia**
- **H**ypercalcaemia Rx: **Cinacalcet (Calcimimetic) → Denosumab (RANKL) → Bisphosphonates**
- **Y** Genetic: **MEN1, MEN2A, HPT-JT (CDC73), FIHP** — **Cascade Testing**
- **R**ecurrence: **20-50%**, **Late (5-20yr)**, **Local > Distant**
- **O**perative ioPTH: **>50% Drop at 10min = Complete**
- **I**maging: **US, Sestamibi, 4D-CT, Choline PET** — **Biopsy Avoided**
- **D**enosumab: **Refractory Hypercalcaemia** (RANKL Inhibitor)

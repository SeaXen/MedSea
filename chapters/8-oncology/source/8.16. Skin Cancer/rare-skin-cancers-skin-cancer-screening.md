## 2. Rare Skin Cancers

### Merkel Cell Carcinoma (MCC)

| Feature | Detail |
|---------|--------|
| **Frequency** | **<1% of Skin Cancers**; Rare, Aggressive |
| **Incidence** | ~500/year UK; Rising (Immunosuppression, Aging) |
| **Peak Age** | 70-80 years |
| **MCPyV** | **80% Positive** (Merkel Cell Polyomavirus) — **Viral T-Antigens (LT/ST)** |
| **MCPyV-Negative** | **UV Signature (TP53, RB1)** — **Higher Mutational Burden** |
| **Cell of Origin** | **Merkel Cells (Mechanoreceptors)** — **Neuroendocrine** |

### Clinical Features

| Feature | Description |
|---------|-------------|
| **Presentation** | **Rapidly Growing, Painless, Firm, Red/Purple Nodule** (AEIOU: Asymptomatic, Expanding, Immunosuppressed, Older, UV-exposed) |
| **Location** | **Head/Neck (50%)**, **Extremities (40%)**, **Trunk (10%)** |
| **Lymph Nodes** | **Early Spread** — **SLNB / Clinical N+ Common** |
| **Immunosuppression** | **↑ Risk (Tx, HIV, CLL)** |

### Staging & Treatment (Summary)
| Stage | Treatment |
|-------|-----------|
| **I-II** | **Wide Local Excision (1-2cm Margins) + SLNB + Adjuvant RT** |
| **III** | **Surgery + Completion ND + Adjuvant RT** |
| **IV** | **Avelumab 10mg/kg q2w** (JAVELIN Merkel 200) — **ORR ~62%** / **Pembrolizumab 200mg q3w** (KEYNOTE-017/158) — **ORR ~56%** |

## 3. Kaposi Sarcoma (KS)

| Feature | Detail |
|---------|--------|
| **Aetiology** | **HHV-8 (KSHV)** |
| **Clinical Forms** | **Classic (Mediterranean Elderly)**, **Endemic (African)**, **Iatrogenic (Post-Transplant)**, **Epidemic (HIV/AIDS)** |
| **IHC** | **HHV-8 LANA+**, **CD31+, ERG+, CD34+** |
| **Staging (AIDS Clinical Trials Group)** | **T0 (Localised Cutaneous)**, **T1 (Extensive/Visceral)**, **I0 (CD4≥200)**, **I1 (CD4<200)** |
| **Treatment** | **ART (HIV) → Regression**, **RT (Localised)**, **Paclitaxel (Systemic)**, **Pomalidomide (Relapsed)** |
| **Key IHC** | **LANA+ (HHV-8), CD31+, ERG+, CD34+** |

## 4. Dermatofibrosarcoma Protuberans (DFSP)

| Feature | Detail |
|---------|--------|
| **Translocation** | **t(17;22)(q22;q13): COL1A1-PDGFB** |
| **Age** | **Young Adults (20-50)** |
| **Site** | **Trunk, Proximal Extremities** |
| **IHC** | **CD34+ (Diffuse Strong), CD117-, S100-, Factor XIIIa+** |
| **Molecular** | **COL1A1-PDGFB Fusion** → **PDGFRB Autocrine Activation** |
| **Behaviour** | **Locally Aggressive**, **Rare Mets (<5%)**, **High Local Recurrence** |
| **Surgery** | **Wide Excision (2-3cm Margins) / Mohs Micrographic Surgery** |
| **Adjuvant** | **RT (Close/Positive Margins)** |
| **Metastatic/Unresectable** | **Imatinib 400mg Daily** (PDGFRB Inhibition) — **High Response Rate** |

## 5. Angiosarcoma

| Feature | Detail |
|---------|--------|
| **Subtypes** | **Primary (Cutaneous/Soft Tissue)**, **Secondary (Radiation-Associated, Post-Mastectomy, Chronic Lymphoedema)** |
| **Molecular** | **MYC Amplification** (Radiation-Associated, >90%), **KDR/VEGFR2 Mut**, **PTPRB, PLCG1** |
| **IHC** | **CD31+, ERG+, FLI1+, CD34+**, **Factor VIII+** |
| **Behaviour** | **Aggressive**, **Multifocal**, **Early Haematogenous Spread (Liver, Lung)** |
| **Primary Treatment** | **Surgery R0 ± RT** (Adjuvant) |
| **Metastatic** | **Paclitaxel (Weekly) + Bevacizumab** — **Best Response**, **Doxorubicin-based**, **Pembrolizumab** (Immunotherapy), **Sorafenib/Pazopanib** (VEGFR TKI) |

> **Radiation-Associated**: **Latency 10-20yr**, **MYC Amp**, **Poor Prognosis**

## 7. Skin Cancer Screening

> [!tip] **No Population Screening Programme for Skin Cancer in UK**

### High-Risk Individuals Warranting Surveillance

| Risk Factor | Surveillance |
|-----------|--------------|
| **Organ Transplant Recipients** | **6-12 Monthly Dermatology Review** (SCC Risk ↑65-250x) |
| **Strong Family History** (Melanoma, SCC, BCC) | **Annual Dermatology Review** |
| **Personal History** of Skin Cancer | **6-12 Monthly Dermatology Review** |
| **Fair Skin (Fitzpatrick I-II)** + **High UV Exposure** | **Annual Dermatology Review** |
| **Genetic Syndromes** (XP, Gorlin, FAMMM) | **Specialised Surveillance Protocols** |
| **Immunosuppression** (HIV, Tx, CLL) | **6-12 Monthly Dermatology Review** |
| **Multiple Atypical Nevi (>50)** | **Annual Dermoscopy + Photography** |

## 8. Mole Screening (ABCDE Rule)

| Letter | Feature | Concerning Sign |
|--------|---------|-----------------|
| **A** | **Asymmetry** | One half ≠ other half |
| **B** | **Border Irregularity** | Irregular, Notched, Blurred |
| **C** | **Colour Variation** | Multiple Shades (Brown, Black, Red, White, Blue) |
| **D** | **Diameter** | **>6mm** (But Melanomas Can Be Smaller) |
| **E** | **Evolution** | **Change in Size, Shape, Colour, Elevation, Symptoms** |

> **E = Evolution (Most Important)** — **Change Over Time = Red Flag**

## 9. Dermoscopy

| Feature | Benign | Malignant (Melanoma) |
|---------|--------|----------------------|
| **Pattern** | **Regular, Symmetric** | **Asymmetric, Chaotic** |
| **Network** | **Regular, Uniform** | **Atypical Network, Irregular** |
| **Globules** | **Regular, Peripheral** | **Irregular, Central, Amorphous** |
| **Vessels** | **Comma, Hairpin (Benign)** | **Irregular, Dotted, Linear (Malignant)** |
| **Regression** | **Absent** | **White Scar-Like Areas, Peppering** |
| **Colours** | **1-2 Colours** | **≥3 Colours (Black, Brown, Red, White, Blue)** |

### 7-Point Checklist (Weighted)
| Major Criteria (2 Points Each) | Minor Criteria (1 Point Each) |
|--------------------------------|-------------------------------|
| **Atypical Network** | **Irregular Streaks** |
| **Irregular Streaks** | **Irregular Pigmentation** |
| **Regression Structures** | **Blue-White Veil** |
| | **Atypical Vascular Pattern** |

**Score ≥3 = Suspicious → Refer for Excision/Biopsy**

## 13. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Skin Cancer Screening — Programme or Surveillance?** | **No Population Programme** — **High-Risk Surveillance Only** |
| **ABCDE — E = Evolution** | **Most Important Criterion** — **Change in Size, Shape, Colour, Elevation, Symptoms** |
| **Dermoscopy — Benign vs Malignant** | **Benign**: Regular Network, Uniform Globules; **Malignant**: Atypical Network, Irregular Streaks, Regression, Blue-White Veil |
| **7-Point Checklist** | **Major (2pts): Atypical Network, Irregular Streaks, Regression**; **Minor (1pt): Irregular Pigmentation, Blue-White Veil, Vascular Pattern** |
| **Melanoma vs NMSC — Screening** | **Melanoma: High-Risk Surveillance**; **NMSC: High-Risk Surveillance (Transplant, Prior CA)** — **No Population Programme for Either** |
| **Total Body Photography vs Sequential Dermoscopy** | **TB Photography: Baseline for High-Risk**; **Sequential Dermoscopy: Monitoring Individual Lesions** |
| **Kaposi vs Angiosarcoma** | **KS: HHV-8, LANA+, Art/RT/Chemo**; **Angio: MYC Amp (Rad-Assoc), Paclitaxel+Bev, Pembro** |
| **DFSP vs SFT** | **DFSP: COL1A1-PDGFB, CD34+, Imatinib**; **SFT: NAB2-STAT6, STAT6 Nuclear+, Pazopanib/Bev** |

**Mnemonic: RARE-SKIN-SCREEN**
- **R**are: **Merkel (MCPyV), Kaposi (HHV-8), DFSP (COL1A1-PDGFB), Angio (MYC Amp)**
- **A**BCDE: **A-Asymmetry, B-Border, C-Colour, D-Diameter>6mm, E-Evolution (Most Important)**
- **R**isk: **Transplant (SCC ↑65-250x), FH, Fair Skin, Prior CA, Genetic, Immunosuppression**
- **E**volution (E in ABCDE): **Change = Red Flag, Most Important**
- **S**creening: **No Population Programme, High-Risk Surveillance Only**
- **K**aposi: **HHV-8, LANA+, ART + Chemo/RT**
- **I**ntervention: **ABCDE, Dermoscopy, 7-Point Checklist, Total Body Photography**
- **N**odules: **Merkel (MCPyV, Neuroendocrine, ICI), DFSP (COL1A1-PDGFB, Imatinib), Angio (MYC Amp, Paclitaxel+Bev)**
- **C**ancer Screening: **No Population Programme, High-Risk Surveillance Only**
- **R**are Skin Cancers: **Merkel, Kaposi, DFSP, Angio, EMC, Atypical Fibroxanthoma**
- **E**arly Detection: **ABCDE, Dermoscopy (7-Point), Total Body Photography**
- **E**pidemiology: **Melanoma 5th Common, BCC/SCC Most Common Cancers**
- **E**arly: **ABCDE, Dermoscopy (7-Point), Photography**
- **N**o Population Programme: **Skin Cancer Screening**

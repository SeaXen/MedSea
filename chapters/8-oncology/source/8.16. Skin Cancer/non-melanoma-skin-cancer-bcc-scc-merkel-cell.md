# Non-Melanoma Skin Cancer (NMSC)

> [!tip] **FCPS/MRCP Priority: MEDIUM**
> **NMSC = Most Common Cancers (BCC > SCC)**; **BCC 75-80%**, **SCC 20-25%**, **Merkel Cell <1%**; **UVR Primary Risk**; **BCC: PTCH1/SMO (Hedgehog), Local Destruction, Surgery (Mohs), Vismodegib/Sonidegib (Advanced)**; **SCC: UVR, Immunosuppression, TP53, Surgery (Mohs), RT, Cemiplimab (Advanced)**; **Merkel Cell: MCPyV, Neuroendocrine, Aggressive, Surgery + RT, ICI (Avelumab/Pembro)**; **High-Risk Features: Size, Depth, PNI, Poor Differentiation, Immunosuppression, Location (H-zone)**

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Distinguish **BCC, SCC, Merkel Cell** by clinical features, histology, behavior
- [ ] Apply **High-Risk Criteria** (Size, Depth, PNI, Differentiation, Immunosuppression, Location, Recurrence)
- [ ] Perform **Surgery**: **Mohs Micrographic Surgery** (Gold Standard for H-zone/Recurrent/High-Risk)
- [ ] Indicate **Radiotherapy**: Adjuvant (High-Risk), Primary (Unfit/Refusal), Palliative
- [ ] Prescribe **Systemic Therapy**: **Vismodegib/Sonidegib** (Advanced BCC), **Cemiplimab** (Advanced SCC), **Avelumab/Pembrolizumab** (Merkel Cell)
- [ ] Recognise **Immunosuppression** (Organ Transplant, HIV) → **↑SCC Risk 65-250x**
- [ ] Manage **Merkel Cell Carcinoma**: **MCPyV+, Neuroendocrine**, **Surgery + RT + ICI (Avelumab/Pembro)**

## 2. Basal Cell Carcinoma (BCC)

### Epidemiology & Risk

| Feature | Detail |
|---------|--------|
| **Frequency** | **75-80% of NMSC**; Most Common Human Cancer |
| **Incidence** | ~100,000/year UK (Incomplete Registration) |
| **Risk Factors** | **UVR (Chronic + Intermittent)**, **Fair Skin**, **Age**, **Genetic (Gorlin Syndrome: PTCH1)**, **Immunosuppression (Mild ↑)** |
| **Metastasis** | **Extremely Rare (<0.1%)** — **Locally Destructive** |

### Clinical Subtypes

| Subtype | Clinical Appearance | Behavior |
|---------|---------------------|----------|
| **Nodular (Classic)** | **Pearly Nodule, Telangiectasia, Rolled Borders, Central Ulceration (Rodent Ulcer)** | Common, Well-Defined |
| **Superficial** | **Erythematous Scaly Plaque**, Slow-Growing, Trunk/Limbs | **Low-Risk**, Multiple |
| **Morpheaform (Sclerosing/Infiltrative)** | **Waxy, Scar-Like, Indistinct Borders**, Firm | **High-Risk**, Deep Invasion, Recurrence |
| **Basosquamous** | Mixed BCC + SCC Features | **Aggressive**, Metastatic Potential |
| **Pigmented** | Brown/Black Pigmentation | Mimics Melanoma |

### Hedgehog Pathway

```mermaid
flowchart LR
    A[PTCH1 Mutation (90%) / SMO Mutation (10%)] --> B[Loss of PTCH1 Inhibition]
    B --> C[SMO Activation] --> D[GLI Transcription Factors → Target Genes]
    D --> E[Cell Proliferation, Survival]
    E --> F[BCC Development]
    F --> G[**Hedgehog Inhibitors: Vismodegib (SMO), Sonidegib (SMO)**]
```

### Treatment

| Setting | Treatment |
|---------|-----------|
| **Low-Risk (Small, Well-Defined, Non-H-zone)** | **Surgical Excision (3-4mm Margins)**, **Curettage + Cautery**, **Cryotherapy**, **Topical (Imiquimod 5%, 5-FU 5%)**, **Photodynamic Therapy (PDT)** |
| **High-Risk / H-zone / Recurrent** | **Mohs Micrographic Surgery (Gold Standard)** — **Tissue Sparing, Highest Cure Rate** |
| **Advanced / Metastatic (Unresectable)** | **Vismodegib 150mg OD** (ERIVANCE) — **ORR ~43%**<br/>**Sonidegib 200mg OD** (BOLT) — **ORR ~47%** |
| **Radiotherapy** | **Primary (Unfit/Refusal)**, **Adjuvant (Positive Margins, PNI)**, **Palliative** |

## 3. Squamous Cell Carcinoma (SCC)

### Epidemiology & Risk

| Feature | Detail |
|---------|--------|
| **Frequency** | **20-25% of NMSC** |
| **Risk Factors** | **UVR (Chronic)**, **Immunosuppression (Tx: ↑65-250x)**, **Fair Skin**, **Chronic Inflammation/Scars (Marjolin's Ulcer)**, **HPV (Genital/Perianal)**, **Arsenic**, **Xeroderma Pigmentosum** |
| **Metastasis** | **2-5% Overall**, **Higher if High-Risk** |

### High-Risk Features (BCC & SCC)

| Feature | Threshold |
|---------|-----------|
| **Size** | **>2cm** (BCC), **>2cm** (SCC) |
| **Depth** | **>2mm** (Clark Level IV/V), **>6mm** (Breslow) |
| **Perineural Invasion (PNI)** | **Clinical (Symptomatic) OR Microscopic (>0.1mm / Large Nerve)** |
| **Differentiation** | **Poor / Undifferentiated** |
| **Immunosuppression** | **Organ Transplant, HIV, Lymphoma** |
| **Location (H-zone)** | **Central Face, Ears, Periorbital, Nose, Lips, Chin, Mandible, Temple, Pre/Post-Auricular** |
| **Recurrent** | **Previously Treated** |

### Clinical Subtypes (SCC)

| Subtype | Features |
|---------|----------|
| **In-Situ (Bowen's Disease)** | **Erythematous Scaly Plaque**, **Full-Thickness Epidermal Dysplasia**, **No Invasion** |
| **Actinic Keratosis (AK)** | **Precursor**, **Scaly Rough Papule**, **Field Cancerization** |
| **Keratoacanthoma (KA)** | **Rapid Growth (Weeks), Central Keratin Plug**, **Self-Limiting (May Regress)**, **SCC Variant** |
| **Invasive SCC** | **Ulcerated Nodule/Plaque**, **Induration**, **Variable Differentiation** |
| **Verrucous Carcinoma** | **Low-Grade, Exophytic, Cauliflower**, **Rare Mets** |

### Treatment

| Setting | Treatment |
|---------|-----------|
| **In-Situ (Bowen's)** | **Topical (5-FU, Imiquimod, Ingenol Mebutate)**, **PDT**, **Cryotherapy**, **Excision** |
| **Low-Risk Invasive** | **Surgical Excision (4-6mm Margins)**, **Mohs (H-zone)** |
| **High-Risk** | **Mohs Surgery (Gold Standard)** OR **Wide Excision + Adjuvant RT** |
| **Lymph Nodes** | **Clinical N+ → Neck Dissection / RT** |
| **Advanced / Metastatic** | **Cemiplimab 350mg q3w** (EMPOWER-CSCC) — **Anti-PD-1, ORR ~50%** |
| **Radiotherapy** | **Primary (Unfit/Refusal)**, **Adjuvant (Positive Margins, PNI, High-Grade, N+)**, **Palliative** |

## 4. Merkel Cell Carcinoma (MCC)

### Epidemiology & Pathogenesis

| Feature | Detail |
|---------|--------|
| **Frequency** | **<1% of Skin Cancers**; Rare, Aggressive |
| **Incidence** | ~500/year UK; Rising (Immunosuppression, Aging) |
| **Peak Age** | 70-80 years |
| **MCPyV** | **80% Positive** (Merkel Cell Polyomavirus) — **Viral T-Antigens (LT/ST)** |
| **MCPyV-Negative** | **UV Signature (TP53, RB1)** — **Higher Mutational Burden** |
| **Cell of Origin** | **Merkel Cells (Mechanoreceptors)** — **Neuroendocrine** |

### Clinical Features

| Feature | Description |
|---------|-------------|
| **Presentation** | **Rapidly Growing, Painless, Firm, Red/Purple Nodule** (AEIOU: Asymptomatic, Expanding, Immunosuppressed, Older, UV-exposed) |
| **Location** | **Head/Neck (50%)**, **Extremities (40%)**, **Trunk (10%)** |
| **Lymph Nodes** | **Early Spread** — **SLNB / Clinical N+ Common** |
| **Immunosuppression** | **↑ Risk (Tx, HIV, CLL)** |

### Staging (AJCC 8th)

| Stage | Description |
|-------|-------------|
| **I** | **Primary ≤2cm**, No Nodes (SLNB-) |
| **II** | **Primary >2cm**, No Nodes (SLNB-) |
| **III** | **Node Positive** (SLNB+ or Clinical N+) |
| **IV** | **Distant Mets** |

### Treatment

```mermaid
flowchart TD
    A[Merkel Cell Carcinoma] --> B{Stage}
    B -->|**Stage I-II (Local)**| C[**Wide Local Excision (1-2cm Margins)**<br/>**+ Sentinel Node Biopsy (SLNB)**<br/>**+ Adjuvant RT to Primary Bed (50-56Gy)**<br/>**If SLNB+ → Completion ND + RT to Nodal Basin**]
    B -->|**Stage III (Node+)**| D[**Surgery (Excision + Completion ND)**<br/>**Adjuvant RT to Primary + Nodal Basin (50-56Gy)**<br/>**Consider Adjuvant Chemo (Carbo/Etoposide) — Controversial**]
    B -->|**Stage IV (Metastatic)**| E[**Immune Checkpoint Inhibitor**<br/>**Avelumab 10mg/kg q2w** (JAVELIN Merkel 200) — **ORR ~62%, Durable**<br/>**Pembrolizumab 200mg q3w** (KEYNOTE-017/158) — **ORR ~56%**<br/>**Chemotherapy (Carbo/Etoposide) — Transient, Not Preferred**]
```

### Chemoradiation (Adjuvant)

| Setting | Regimen |
|---------|---------|
| **Primary RT (Unfit for Surgery)** | **RT 50-56Gy to Primary + Nodal Basin** |
| **Adjuvant RT (Post-Surgery)** | **50-56Gy/25-28fx** to Primary Bed ± Nodal Basin |
| **Concurrent Chemo (Carbo/Etoposide)** | **Controversial** — **Not Standard** |

| **BCC Advanced — Systemic Therapy?** | **Vismodegib 150mg OD** (SMO Inhibitor) OR **Sonidegib 200mg OD** — **Hedgehog Pathway Inhibition (PTCH1/SMO)**. |
| **SCC Advanced — Systemic Therapy?** | **Cemiplimab 350mg q3w** (Anti-PD-1) — **EMPOWER-CSCC: ORR ~50%** |
| **Merkel Cell Carcinoma — MCPyV, Treatment?** | **MCPyV+ 80%**; **Stage I-II: Excision + SLNB + Adjuvant RT**; **Stage IV: Avelumab 10mg/kg q2w / Pembrolizumab 200mg q3w** |
| **Immunosuppression — SCC Risk?** | **Organ Transplant: ↑65-250x**; **SCC:BCC Ratio Reverses (SCC > BCC in Tx)**; **Aggressive, Multiple, Early Mets** |
| **Mohs Surgery — Indications?** | **H-zone, Recurrent, High-Risk (PNI, >2cm, Aggressive Histology), Cosmetically Sensitive, Poorly Defined Borders** — **Gold Standard**. |
| **Adjuvant RT for SCC/BCC — Indications?** | **PNI (Clinical/Microscopic), Positive Margins, High-Grade, Size>2cm/Depth>2mm, N+, Immunosuppression, Recurrent** |
| **Merkel Cell — SLNB, RT?** | **SLNB for Staging (All Stages)**; **Adjuvant RT to Primary Bed + Nodal Basin (50-56Gy)**. |
| **Bowen's Disease (SCC In-Situ) — Treatment?** | **Topical (5-FU, Imiquimod, Ingenol Mebutate), PDT, Cryotherapy, Excision** — **No RT Usually Needed**. |

## 7. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **BCC vs SCC Metastasis** | **BCC: Extremely Rare (<0.1%)**; **SCC: 2-5% (Higher if High-Risk/Immunosuppression)** |
| **Morpheaform BCC vs SCC** | **Morpheaform: Waxy, Scar-Like, Indistinct Borders**; **SCC: Ulcerated, Indurated, Keratotic** |
| **PNI Definition — Clinical vs Microscopic** | **Clinical PNI: Symptomatic (Pain, Paresthesia, Weakness)**; **Microscopic PNI: >0.1mm Nerve Involvement, ≥3 Foci, Nerve ≥0.1mm** |
| **Immunosuppression — BCC vs SCC Ratio** | **Normal: BCC:SCC ≈ 4:1**; **Transplant: SCC:BCC >1 (SCC Increased 65-250x)** |
| **Mohs vs Standard Excision** | **Mohs: 100% Margin Assessment, Tissue Sparing, Staged** — **Standard: 3-4mm (BCC), 4-6mm (SCC) Margins, Bread-Loaf** |
| **Merkel Cell — SLNB Timing** | **SLNB at Time of Primary Excision** — **If +ve → Completion ND + RT** |
| **Cemiplimab vs Avelumab vs Pembrolizumab** | **Cemiplimab: SCC (Anti-PD-1)**; **Avelumab: Merkel (Anti-PD-L1)**; **Pembrolizumab: Merkel/Melanoma (Anti-PD-1)** |

**Mnemonic: NMSC-HIGH-RISK**
- **N**MSC: **BCC 80% (Hedgehog)**, **SCC 20% (TP53)**, **Merkel (MCPyV)**
- **M**etastasis: **BCC Rare**, **SCC 2-5%**, **Merkel High**
- **S**CC Immunosuppression: **↑65-250x (Tx)**, **SCC > BCC in Tx**
- **C**linical PNI: **Pain/Numbness/Weakness**; **Micro PNI: >0.1mm Nerve, ≥3 Foci**
- **H**-zone: **Central Face, Ears, Periorbital, Nose, Lips, Chin, Mandible, Temple, Auricular**
- **H**igh-Risk Features: **Size>2cm, Depth>2mm, PNI, Poor Diff, Immunosuppression, H-zone, Recurrent**
- **R**T Adjuvant: **PNI, Margins+, High-Grade, >2cm, N+, Immunosuppression**
- **I**mmunosuppression: **SCC ↑65-250x (Tx)**, **SCC:BCC Reverses**
- **A**dvanced Systemic: **BCC: Vismodegib/Sonidegib (SMO)**; **SCC: Cemiplimab (PD-1)**; **Merkel: Avelumab/Pembro (PD-1/L1)**
- **L**ocally Advanced: **Mohs Surgery (Gold Standard H-zone/Recurrent/High-Risk)**
- **M**erkel: **MCPyV 80%**, **Neuroendocrine**, **ICI (Avelumab/Pembro)**
- **O**rgan Transplant: **SCC Risk ↑65-250x**, **SCC:BCC Reverses**
- **H**edgehog: **PTCH1/SMO → Vismodegib/Sonidegib (BCC)**
- **Y**early Screening: **Transplant Patients (Skin Checks q6-12mo)**

# Endometrial Cancer

> [!tip] **FCPS/MRCP Priority: HIGH**
> **Endometrial Cancer = Most common gynaecological cancer in developed countries**; **Molecular Classification (TCGA/WHO 2020) replaces histology**: **POLE ultra-mutated (10%)**, **MMRd/MSI-H (30%)**, **p53abn (10%)**, **NSMP (50%)**; **FIGO 2009/2023 Staging**; **Surgery: Total Hysterectomy + BSO + SLN Mapping** (preferred over systematic lymphadenectomy); **Adjuvant**: **NSMP**: Vaginal Brachytherapy; **MMRd**: Vaginal Brachy ± EBRT; **p53abn**: Chemo (Carbo/Pac) + EBRT; **POLE*: Observation (excellent prognosis)**; **Recurrence**: **MMRd/MSI-H → ICI (Pembrolizumab/Dostarlib)**; **NSMP → Hormonal (Letrozole) / PARPi (ATHENA/ENGOT)**.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Apply **TCGA/WHO 2020 Molecular Classification**: POLE, MMRd, p53abn, NSMP
- [ ] Stage using **FIGO 2009/2023** (2023: Stage I-IIIA substaging by molecular)
- [ ] Perform **SLN Mapping** (Indocyanine Green) — preferred over systematic LND
- [ ] Select **adjuvant therapy** by molecular subtype + stage
- [ ] Manage **recurrent disease**: MMRd → ICI; NSMP → Hormonal/PARPi; p53abn → Chemo
- [ ] Recognise **Lynch Syndrome** (MMRd) → Cascade testing, surveillance

## 2. Definition & Epidemiology

| Feature | Detail |
|---------|--------|
| **Definition** | Malignant tumour of endometrium; **Endometrioid 80%** (Type I, oestrogen-driven); **Non-endometrioid 20%** (Type II, serous, clear cell, carcinosarcoma) |
| **Incidence** | UK: ~9,700/year; **4th most common female cancer**; Rising (obesity) |
| **Prevalence** | 5-year OS: Stage I ~90%, Stage III ~50%, Stage IV ~15% |
| **Peak Age** | 60-70 years (post-menopausal) |
| **Risk Factors** | **Obesity** (↑ oestrogen), **Nulliparity**, **Early menarche/Late menopause**, **PCOS**, **Tamoxifen**, **HRT (unopposed oestrogen)**, **Lynch Syndrome** (MMR genes), **Diabetes**, **Hypertension** |

## 3. Aetiology & Molecular Classification (TCGA/WHO 2020)

```mermaid
flowchart LR
    A[Endometrial Cancer] --> B[Molecular Classification]
    B --> B1[**POLE Ultra-mutated (10%)**<br/>POLE exonuclease domain mut<br/>**Excellent prognosis**<br/>**Observation** even if high-risk histology]
    B --> B2[**MMRd/MSI-H (30%)**<br/>MLH1/PMS2/MSH2/MSH6 loss<br/>**Lynch Syndrome (germline)** or MLH1 methylation<br/>**ICI responsive**]
    B --> B3[**p53abn (10%)**<br/>TP53 mutation<br/>**Serous-like, aggressive**<br/>**Worst prognosis**]
    B --> B4[**NSMP (50%)**<br/>No Specific Molecular Profile<br/>**Endometrioid, hormone-driven**<br/>**Hormonal therapy responsive**]
```

### Molecular Subtypes & Management Implications

| Subtype | Frequency | Key Features | Prognosis | Adjuvant Approach |
|---------|-----------|--------------|-----------|-------------------|
| **POLE** | 10% | **Ultra-high TMB** (>100 mut/Mb), POLE P286R/V411L | **Excellent** | **De-escalation: Observation** (even Stage III) |
| **MMRd** | 30% | **MSI-H**, MMR protein loss (IHC), MLH1 methylation (sporadic) vs germline (Lynch) | **Intermediate** | Vaginal Brachy ± EBRT; **ICI for recurrence** |
| **p53abn** | 10% | **TP53 mut**, Serous/clear cell/carcinosarcoma | **Poor** | **Chemo (Carbo/Pac) + EBRT** |
| **NSMP** | 50% | No defining mutation, ER/PR+, **CTNNB1, PIK3CA, ARID1A** | **Favourable** | Vaginal Brachy; **Hormonal for recurrence** |

## 4. Clinical Features

| Feature | Description |
|---------|-------------|
| **Post-menopausal Bleeding (PMB)** | **90%** — **Red flag** |
| **Abnormal Discharge** | Watery, blood-tinged |
| **Pelvic Pain** | Late (uterine distension, invasion) |
| **Weight Loss** | Advanced |
| **Incidental** | Asymptomatic on imaging (increasing) |

## 5. Staging (FIGO 2009 / 2023)

| Stage | FIGO 2009 | FIGO 2023 (Molecular-Integrated) |
|-------|-----------|----------------------------------|
| **I** | Confined to uterus | IA: <50% myoinvasion; IB: ≥50%<br/>**IA1/IA2/IA3 by molecular** |
| **II** | Cervical stromal invasion | **II** (unchanged) |
| **III** | Local/regional spread | **IIIA**: Serosa/adnexa; **IIIB**: Vaginal/parametrial; **IIIC1**: Pelvic nodes; **IIIC2**: Para-aortic nodes |
| **IV** | Distant mets | **IVA**: Bladder/bowel mucosa; **IVB**: Distant |

**2023 Key Change**: Stage I-IIIA substaged by molecular subtype (e.g., Stage IB POLE vs p53abn different adjuvant)

## 6. Diagnosis & Investigations

| Investigation | Role | Key Details |
|---------------|------|-------------|
| **Transvaginal US** | First-line | **Endometrial thickness >4mm (post-menopausal) = biopsy** |
| **Pipelle Biopsy / Hysteroscopy** | **Diagnosis** | **Gold standard**; Sufficient for molecular |
| **MRI Pelvis** | **Local staging** | Myoinvasion depth, cervical stromal invasion, nodal |
| **CT Chest/Abd/Pelvis** | Distant staging | Stage III/IV |
| **Molecular Testing (MANDATORY)** | **POLE sequencing, MMR IHC (MLH1, PMS2, MSH2, MSH6), p53 IHC** | **All 3 required** for classification |
| **Germline Testing** | If MMRd (Lynch) | **Cascade testing** |
| **CA125** | Baseline, monitoring | Non-specific |

---

## 7. Differential Diagnosis

| Condition | Distinguishing Features |
|-----------|-------------------------|
| **Endometrial Hyperplasia (Atypical)** | **EIN (Endometrial Intraepithelial Neoplasia)**; Precursor; Progression to cancer ~30% |
| **Endometrial Polyp** | Focal, benign; Hysteroscopic resection |
| **Cervical Cancer** | Cervical origin, HPV-driven, different staging |
| **Ovarian Cancer** | Adnexal mass, CA125↑, different molecular |
| **Uterine Sarcoma** | Leiomyosarcoma, ESS; Rapid growth, different IHC |
| **Metastasis to Uterus** | Known primary; Rare |

## 8. Management

### Surgery

```mermaid
flowchart TD
    A[Endometrial Cancer] --> B[**Total Hysterectomy + BSO**]
    B --> C[**Sentinel Lymph Node Mapping (SLN)**<br/>**Indocyanine Green (ICG) injection**<br/>**Preferred over systematic LND** (FIRES, ENDO-3)<br/>**If SLN not mapped → Side-specific LND**]
    C --> D{Stage + Molecular}
    D -->|**Stage IA, POLE or NSMP, Low-grade**| E[**Surgery Only** (No adjuvant)]
    D -->|**Stage IB/II/III, NSMP/MMRd**| F[**Adjuvant Vaginal Brachytherapy**<br/>**PORTEC-2**: Vaginal Brachy = EBRT for VFS, less toxicity]
    D -->|**Stage IB/II/III, p53abn**| G[**Chemo (Carbo/Pac ×6) + EBRT**<br/>**PORTEC-3**: Chemo+RT > RT alone for high-risk]
    D -->|**Stage IIIC (Node+)**| H[**Chemo (Carbo/Pac ×6) ± EBRT**<br/>**Molecular: p53abn/MMRd/NSMP all benefit**]
```

### Adjuvant Therapy by Molecular Subtype

| Molecular | Stage IA | Stage IB | Stage II | Stage III |
|-----------|----------|----------|----------|-----------|
| **POLE** | **Obs** | **Obs** | **Obs/Vag Brachy** | **Obs/Vag Brachy** (de-escalation trials) |
| **MMRd** | Obs | Vag Brachy | Vag Brachy ± EBRT | Chemo+EBRT or Vag Brachy+Chemo |
| **p53abn** | Vag Brachy | Chemo+EBRT | Chemo+EBRT | **Chemo+EBRT** |
| **NSMP** | Obs | Vag Brachy | Vag Brachy ± EBRT | Vag Brachy + Chemo (if high-risk) |

**Key Trials:**
- **PORTEC-1/2**: Vaginal Brachytherapy non-inferior to EBRT for VFS, less toxicity
- **PORTEC-3**: Chemo (Carbo/Pac) + EBRT vs EBRT alone for high-risk → **OS benefit for Serous/p53abn**
- **GOG-249**: Vaginal Brachy + Chemo vs EBRT alone
- **RAINBO Trials** (ongoing): Molecular-guided adjuvant

### Recurrent/Metastatic Disease

```mermaid
flowchart TD
    A[Recurrent Endometrial] --> B{Molecular Subtype}
    B -->|**MMRd / MSI-H**| C[**ICI: Pembrolizumab** (KEYNOTE-158) **OR Dostarlimab** (GARNET)<br/>**ORR ~40-50%, Durable**<br/>**1L for recurrent (RUBY: Dostarlib + Carbo/Pac → Dostarlib maint)**]
    B -->|**NSMP / POLE**| D[**Hormonal Therapy**<br/>**Letrozole 2.5mg OD** (or Megestrol, Medroxyprogesterone)<br/>**OR Aromatase Inhibitor + CDK4/6i** (PALbOc: Palbociclib + Letrozole)<br/>**PARPi: Niraparib** (ATHENA/ENGOT) **if HRD+ / BRCA+**]
    B -->|**p53abn**| E[**Chemotherapy**<br/>**Carbo/Pac** (1L recurrent)<br/>**PLD, Topotecan, Gemcitabine** (2L+)<br/>**Bevacizumab + Chemo** (MITO END-3)<br/>**ICI + Chemo** (RUBY/ENGOT-EN6: Dostarlib/Pembro + Carbo/Pac)]
    C --> F[**Clinical Trials**]
    D --> F
    E --> F
```

### RUBY / ENGOT-EN6 / KEYNOTE-868 (1L Recurrent/Advanced)
- **Dostarlimab/Pembrolizumab + Carboplatin/Paclitaxel → ICI Maintenance**
- **Regardless of MMR status** (but MMRd derives most benefit)
- **New standard for 1L advanced/recurrent**

## 11. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Molecular vs Histology** | **Molecular (TCGA) REPLACES histology** for adjuvant decisions; **Histology still reported** but molecular guides Rx |
| **POLE — always observe?** | **POLE pathogenic mut (exonuclease domain)** → **Excellent outcome**; **Non-pathogenic POLE = NSMP**; **De-escalation trials ongoing for Stage III** |
| **MMRd — which IHC panel?** | **MLH1, PMS2, MSH2, MSH6** — **Loss of ANY = MMRd**; **MLH1 loss + MLH1 methylation = Sporadic**; **No methylation = Lynch** |
| **NSMP — no mutations?** | **NSMP = No POLE, No MMRd, No p53abn**; **Has CTNNB1, PIK3CA, ARID1A, PTEN** but not defining |
| **Adjuvant Chemo — when?** | **p53abn (any stage ≥IB)**; **Stage IIIC (node+)**; **Serous/Clear cell/Carcinosarcoma** (usually p53abn) |
| **RUBY vs KEYNOTE-868 vs ENGOT-EN6** | **RUBY**: Dostarlib + Carbo/Pac → Dostarlib maint; **KEYNOTE-868**: Pembro + Carbo/Pac → Pembro maint; **ENGOT-EN6**: Dostarlib + Carbo/Pac → Dostarlib maint — **All 1L Advanced/Recurrent** |

**Mnemonic: ENDOMETRIAL**
- **E**ndometrioid 80% (Type I), **Non-endo 20%** (Type II)
- **N**ew Molecular: **POLE, MMRd, p53abn, NSMP** (TCGA/WHO 2020)
- **D**iagnosis: **PMB = Biopsy** (Thickness >4mm post-meno)
- **O**bs for **POLE** (even Stage IB/II)
- **M**MRd → **ICI** (Pembro/Dostarlib) for recurrence
- **E**BRT + Chemo for **p53abn** (PORTEC-3)
- **T**CGAsubtypes replace histology
- **R**ecurrent: **MMRd=ICI, NSMP=Hormonal/PARPi, p53abn=Chemo**
- **I**CG for **SLN Mapping** (Preferred over LND)
- **A**djuvant: **Vag Brachy** (PORTEC-2) for NSMP/MMRd Stage IB
- **L**ynch if **MMRd germline** → Cascade testing

**Q7:** Recurrent NSMP?
**A7:** Letrozole, AI + CDK4/6i, PARPi (if HRD+)

**Q8:** 1L Advanced?
**A8:** ICI + Carbo/Pac → ICI Maint (RUBY, KEYNOTE-868, ENGOT-EN6)

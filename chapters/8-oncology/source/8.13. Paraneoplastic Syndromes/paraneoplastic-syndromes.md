# Paraneoplastic Syndromes

> [!tip] **FCPS/MRCP Priority: HIGH**
> **Paraneoplastic Syndromes = Remote Effects of Cancer, Not Direct Invasion/Metastasis**; **Immune-Mediated (Onconeural Antibodies) or Hormonal/Cytokine Mediated**; **Endocrine: SIADH (SCLC), Ectopic ACTH (SCLC, Carcinoid), Hypercalcaemia/PTHrP (SCC, Breast), Hypoglycaemia/IGF-2 (Mesenchymal), Carcinoid/Serotonin (NET)**; **Neurological: Lambert-Eaton/VGCC (SCLC), PCD/CDR2 (Gynae, SCLC), Anti-Hu/ANNA-1 (SCLC, Encephalomyelitis), Anti-Yo/PCA1 (Gynae, PCD), Anti-Ri/ANNA-2 (Breast, Opsoclonus), Anti-Ma2 (Testicular, Limbic Encephalitis), Anti-CV2/CRMP5 (SCLC, Neuropathy), Anti-NMDA (Ovarian Teratoma)**; **Dermatological: Acanthosis Nigricans, Dermatomyositis, Sweet's Syndrome, Bazex, Leser-Trélat**; **Haematological: TTP, DIC, Erythrocytosis, Trousseau's**; **Rheumatological: HPOA, RS3PE**.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Classify **Paraneoplastic Syndromes** by System (Endocrine, Neurological, Dermatological, Haematological, Rheumatological, Renal)
- [ ] Recognise **Endocrine Syndromes**: **SIADH (SCLC)**, **Ectopic ACTH (SCLC, Carcinoid)**, **Hypercalcaemia/PTHrP (SCC, Breast)**, **Hypoglycaemia/IGF-2 (Mesenchymal)**, **Carcinoid/Serotonin (NET)**
- [ ] Identify **Neurological Syndromes** and their **Onconeural Antibodies**: **Lambert-Eaton/VGCC (SCLC)**, **PCD/CDR2 (Gynae)**, **Anti-Hu (SCLC)**, **Anti-Yo (Gynae)**, **Anti-Ri (Breast)**, **Anti-Ma2 (Testicular)**, **Anti-CV2/CRMP5 (SCLC)**, **Anti-NMDA (Ovarian Teratoma)**
- [ ] Recognise **Dermatological Signs**: **Acanthosis Nigricans, Dermatomyositis, Sweet's Syndrome, Bazex, Leser-Trélat**
- [ ] Identify **Haematological Syndromes**: **TTP, DIC, Erythrocytosis, Trousseau's**
- [ ] Link **Specific Antibodies** to **Associated Cancers** for Targeted Investigation

## 2. Endocrine Paraneoplastic Syndromes

| Syndrome | Mechanism | Associated Cancers | Key Features |
|----------|-----------|-------------------|--------------|
| **SIADH** | **Ectopic ADH (AVP)** | **SCLC (Most Common), NSCLC, HL, CNS Tumours, Drugs** | **Hyponatraemia, Euvolemia, Urine Osm >100, Urine Na >40** |
| **Ectopic ACTH / Cushing's** | **Ectopic ACTH (PRO-ACTH/POMC)** | **SCLC (Most Common), Carcinoid, NET, Thymic, Medullary Thyroid** | **Hypokalaemia, Metabolic Alkalosis, Hyperglycaemia, Hypertension, Skin Changes** |
| **Hypercalcaemia** | **PTHrP (Humoral), Osteolytic Mets, Vit D (Lymphoma)** | **SCC (Lung, H&N, Cervix), Breast, Myeloma, RCC, Lymphoma** | **Polyuria, Nausea, Confusion, Renal Stones, Short QT** |
| **Hypoglycaemia** | **IGF-2 (Big IGF-2), Insulinoma, Glycogenolysis** | **Mesenchymal (Solitary Fibrous, Sarcoma), HCC, Adrenal, Insulinoma** | **Whipple's Triad, Low Glucose, High Insulin/IGF-2, Suppressed C-Peptide** |
| **Carcinoid Syndrome** | **Serotonin, Kallikrein, Bradykinin, Prostaglandins** | **Midgut NET (Ileum, Appendix), Bronchial NET** | **Flushing, Diarrhoea, Valvular Heart Disease (Right Sided), Bronchospasm** |
| **Hyponatraemia (CSW)** | **Cerebral Salt Wasting (Brain Natriuretic Peptide)** | **CNS Tumours, SAH, TBI** | **Hypovolaemia, High Urine Na, High Urine Output, Hyponatraemia** |

## 3. Neurological Paraneoplastic Syndromes

### Classic Onconeural Antibodies & Associated Cancers

| Antibody | Target | Syndrome | Strongest Cancer Association |
|----------|--------|----------|------------------------------|
| **Anti-Hu (ANNA-1)** | **HuD (RNA-Binding)** | **Encephalomyelitis, Sensory Neuropathy, GI Dysmotility** | **SCLC (Most Common)** |
| **Anti-Yo (PCA-1)** | **CDR2 (Purkinje Cell)** | **Paraneoplastic Cerebellar Degeneration (PCD)** | **Gynaecological (Ovarian, Uterine), Breast** |
| **Anti-Ri (ANNA-2)** | **NOVA-1/2 (RNA-Binding)** | **Opsoclonus-Myoclonus Syndrome, Brainstem Encephalitis** | **Breast, Gynaecological, SCLC** |
| **Anti-Ma2** | **Ma2 (Neuronal)** | **Limbic Encephalitis, Brainstem Encephalitis, Hypersomnia** | **Testicular Germ Cell (Most Common), Lung** |
| **Anti-CV2 / CRMP-5** | **CRMP-5 (Collapsin Response Mediator)** | **Encephalomyelitis, Optic Neuritis, Chorea, Neuropathy** | **SCLC, Thymoma** |
| **Anti-NMDA** | **NR1/NR2 (NMDA Receptor)** | **Anti-NMDA Receptor Encephalitis** | **Ovarian Teratoma (50%), SCLC, Other** |
| **Anti-LGI1** | **LGI1 (Synaptic)** | **Limbic Encephalitis, Faciobrachial Seizures** | **Thymoma, SCLC (Rare)** |
| **Anti-CASPR2** | **CASPR2 (VGKC Complex)** | **Morvan's, Limbic Encephalitis, Neuromyotonia** | **Thymoma, SCLC** |
| **Anti-Amphiphysin** | **Amphiphysin (Synaptic Vesicle)** | **Stiff Person Syndrome, Encephalomyelitis** | **Breast, SCLC** |
| **Anti-GAD65** | **GAD65 (GABA Synthesis)** | **Stiff Person Syndrome, Cerebellar Ataxia, Epilepsy** | **Thymoma, SCLC, T1DM** |
| **VGCC (P/Q-Type)** | **VGCC (P/Q-Type Calcium Channel)** | **Lambert-Eaton Myasthenic Syndrome (LEMS)** | **SCLC (60% LEMS)** |

### Lambert-Eaton Myasthenic Syndrome (LEMS)

| Feature | Detail |
|---------|--------|
| **Antibody** | **VGCC (P/Q-Type) Antibody** (Calcium Channel) |
| **Cancer Association** | **SCLC (60%)**, **Autonomous (40%)** |
| **Clinical** | **Proximal Weakness (Hip Girdle), Dry Mouth, Impotence, Areflexia** |
| **Key Sign** | **Facilitation on Repetitive Nerve Stimulation (Increase CMAP >100%)** |
| **EMG/NCS** | **Low Baseline CMAP, Facilitation Post-Exercise/High-Freq RNS** |
| **Autonomic** | **Dry Mouth, Constipation, Impotence, Orthostatic Hypotension** |
| **Treatment** | **3,4-DAP (Amifampridine) — K Channel Blocker**, **IVIG/Plasma Exchange**, **Treat SCLC** |

### Paraneoplastic Cerebellar Degeneration (PCD)

| Antibody | Cancer | Features |
|----------|--------|----------|
| **Anti-Yo (PCA-1)** | **Ovarian, Uterine, Breast** | **Subacute Cerebellar Ataxia, Dysarthria, Nystagmus** |
| **Anti-Hu** | **SCLC** | **Cerebellar + Brainstem + Spinal + Neuropathy** |
| **Anti-Tr (Delta/Notch-Like)** | **Hodgkin Lymphoma** | **Cerebellar Ataxia** |
| **Anti-mGluR1** | **Hodgkin Lymphoma** | **Cerebellar Ataxia** |

## 4. Dermatological Paraneoplastic Syndromes

| Syndrome | Features | Associated Cancers |
|----------|----------|-------------------|
| **Acanthosis Nigricans** | **Velvety Hyperpigmentation (Axilla, Neck, Groin), Skin Tags** | **Gastric (Most Common), Lung, Uterine, Ovarian, Pancreatic** |
| **Dermatomyositis** | **Heliotrope Rash, Gottron's Papules, Proximal Weakness, Photosensitivity** | **Ovarian (Strongest), Lung, Pancreatic, Gastric, Breast** |
| **Sweet's Syndrome** | **Fever, Neutrophilia, Tender Erythematous Plaques (Pseudo-Abscess)** | **AML (Most Common), MDS, Solid Tumours (Breast, GI, GU)** |
| **Bazex Syndrome (Acrokeratosis Paraneoplastica)** | **Psoriasiform Eruption (Hands, Feet, Ears, Nose), Nail Dystrophy** | **H&N SCC (Most Common), Lung, Oesophageal** |
| **Leser-Trélat Sign** | **Sudden Eruption of Multiple Seborrhoeic Keratoses** | **Gastric (Most Common), Lung, Uterine, Breast, Lymphoma** |
| **Erythema Gyratum Repens** | **Concentric Erythematous Rings (Wood Grain)** | **Lung (Most Common), GI, Breast** |
| **Necrolytic Migratory Erythema** | **Annular Erythema, Crusting, Perioral/Genital, Glucagonoma** | **Glucagonoma (Pancreatic NET)** |
| **Pyoderma Gangrenosum** | **Painful Ulcers (Violaceous Borders), Pathergy** | **AML, MDS, CML, Lymphoma, Solid Tumours** |
| **Paraneoplastic Pemphigus** | **Severe Mucocutaneous Erosions, Polymorphic, Bronchiolitis Obliterans** | **CLL, NHL, Castleman Disease, Thymoma** |

## 5. Haematological Paraneoplastic Syndromes

| Syndrome | Mechanism | Associated Cancers | Key Features |
|----------|-----------|-------------------|--------------|
| **TTP (Thrombotic Thrombocytopenic Purpura)** | **ADAMTS13 Deficiency (Autoantibody)** | **Cancer-Related (Secondary), Drugs (Gemcitabine, Ticlopidine)** | **MAHA, Thrombocytopenia, Fever, Neuro, Renal (Pentad)** |
| **DIC** | **Tissue Factor Release, Protease Activation** | **APL (Most Common), Adenocarcinoma (Pancreas, Lung, Gastric), Sepsis** | **Bleeding/Thrombosis, ↑PT/APTT, ↓Fibrinogen, ↑D-Dimer, Schistocytes** |
| **Erythrocytosis** | **EPO Production (Renal), Autonomous** | **RCC (Most Common), HCC, Cerebellar Haemangioblastoma, Pheochromocytoma** | **Hb >18.5 (M) / >16.5 (F), Thrombosis Risk** |
| **Trousseau's Syndrome** | **Mucin-Induced Hypercoagulability, TF Release** | **Mucinous Adenocarcinoma (Pancreas, Lung, Gastric, Ovarian, Colon)** | **Migratory Thrombophlebitis, DVT/PE, Arterial Thrombosis** |
| **Paraneoplastic Neutrophilia** | **G-CSF, IL-6, GM-CSF** | **Various** | **Leukemoid Reaction (>50×10⁹/L)** |
| **Microangiopathic Haemolytic Anaemia (MAHA)** | **Fibrin Strands, Mechanical RBC Fragmentation** | **DIC, TTP, HUS, Cancer (Mucinous Adeno, Metastatic)** | **Schistocytes, ↑LDH, ↓Haptoglobin, ↑Bilirubin** |

## 6. Rheumatological Paraneoplastic Syndromes

| Syndrome | Features | Associated Cancers |
|----------|----------|-------------------|
| **HPOA (Hypertrophic Pulmonary Osteoarthropathy)** | **Clubbing, Periostitis (Tibia, Radius, Ulna), Arthralgia, Vascular Proliferation** | **Lung (SCC, Adenocarcinoma), Mesothelioma, Pleural Mets** |
| **RS3PE (Remitting Seronegative Symmetrical Synovitis with Pitting Oedema)** | **Pitting Oedema (Hands/Feet), Synovitis, Negative RF/CCP** | **Solid Tumours (Prostate, Lung, GI), Lymphoma** |
| **Paraneoplastic Polyarthritis** | **Migratory, Seronegative** | **Lung, Breast, Pancreatic** |
| **Polymyalgia Rheumatica-Like** | **Proximal Stiffness/Pain, Elevated ESR/CRP** | **Lymphoma, Solid Tumours** |

## 7. Renal Paraneoplastic Syndromes

| Syndrome | Mechanism | Associated Cancers |
|----------|-----------|-------------------|
| **Nephrotic Syndrome (Membranous > Minimal Change)** | **Immune Complex Deposition (Tumour Antigens)** | **Lung, Gastric, Colon, Breast, Lymphoma, CLL** |
| **AA Amyloidosis** | **Chronic Inflammation → SAA → Amyloid A** | **Renal Cell Carcinoma, HCC, Lymphoma, Castleman, Chronic Infections** |
| **Tumour Lysis Nephropathy** | **Uric Acid/Phosphate Crystals** | **High-Grade Lymphoma, ALL, AML, Burkitt** |
| **Obstructive Uropathy** | **Retroperitoneal Mets, Direct Invasion** | **Lymphoma, Testicular, Gynaecological, Prostate, Colorectal** |

```mermaid
flowchart TD

    C -->|**Neurological**| E[**LEMS, PCD, Encephalitis, Neuropathy, Stiff Person**]

    C -->|**Rheumatological**| H[**HPOA, RS3PE**]
    C -->|**Renal**| I[**Nephrotic, AA Amyloid, Obstructive**]
    D --> J[**Targeted Cancer Search**]
    E --> J
    F --> J
    G --> J
    H --> J
    I --> J
    J --> K[**Imaging: CT Chest/Abd/Pelvis, PET-CT**]
    J --> L[**Serology: Onconeural Antibody Panel (Hu, Yo, Ri, Ma2, CV2, NMDA, VGCC, Amphiphysin, GAD, LGI1, CASPR2)**]
    J --> M[**Tumour Markers (CEA, CA19-9, CA125, AFP, PSA, NSE, Chromogranin, 5-HIAA)**]
    L --> N[**If Cancer Found: Treat Cancer + Symptomatic Paraneoplastic Management**]
    M --> N
    N --> O[**Immunotherapy (IVIG, PLEX, Rituximab, Steroids) for Immune-Mediated**]
```

## 9. Treatment Principles

| Syndrome Category | Primary Treatment | Symptomatic/Immunotherapy |
|-------------------|-------------------|---------------------------|
| **Endocrine** | **Treat Underlying Cancer** | **SIADH: Fluid Restrict, Tolvaptan**; **Ectopic ACTH: Ketoconazole, Metyrapone**; **HyperCa: Bisphosphonates, Denosumab** |
| **Neurological (Immune-Mediated)** | **Treat Cancer + Immunotherapy** | **IVIG (2g/kg), Plasma Exchange, Rituximab, Corticosteroids, Cyclophosphamide** |
| **LEMS** | **Treat SCLC** | **3,4-DAP (Amifampridine), Pyridostigmine, IVIG, PLEX** |
| **Dermatological** | **Treat Cancer** | **Topical Steroids, Systemic Steroids, Rituximab (Pemphigus)** |
| **Haematological (TTP)** | **Plasma Exchange (Daily) + Corticosteroids** | **Rituximab, Caplacizumab (Anti-vWF)** |
| **DIC** | **Treat Underlying Cause (APL: ATRA), Replace Factors** | **Heparin (Controversial), Antithrombin III** |
| **Rheumatological** | **Treat Cancer** | **NSAIDs, Corticosteroids, Bisphosphonates (HPOA)** |

## 12. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **LEMS vs Myasthenia Gravis** | **LEMS: VGCC Ab, Facilitation on RNS, Proximal Weakness, Autonomic, SCLC**; **MG: AChR Ab, Decrement on RNS, Ocular/Bulbar, Thymoma** |
| **Anti-Hu vs Anti-Yo** | **Anti-Hu: SCLC, Encephalomyelitis + Neuropathy**; **Anti-Yo: Gynae, Pure Cerebellar Degeneration** |
| **Anti-Ri vs Anti-Ma2** | **Anti-Ri: Breast/Gynae, Opsoclonus-Myoclonus**; **Anti-Ma2: Testicular, Limbic Encephalitis** |
| **SIADH vs CSW** | **SIADH: Euvolemic, Urine Na>40, Urine Osm>100**; **CSW: Hypovolemic, High Urine Output, High Urine Na** |
| **TTP vs DIC** | **TTP: MAHA + Thrombocytopenia, Normal PT/APTT, ADAMTS13 Def**; **DIC: ↑PT/APTT, ↓Fibrinogen, ↑D-Dimer, Consumptive** |
| **Acanthosis Nigricans vs Melanoma** | **Acanthosis: Velvety, Symmetric, Axilla/Neck**; **Melanoma: Asymmetric, Irregular Borders, Colour Variegation** |
| **Sweet's vs Erythema Nodosum** | **Sweet's: Neutrophilia, Fever, Plaques (Neutrophils), AML/MDS**; **EN: Tender Nodules (Septal Panniculitis), TB, Sarcoid, Strep, Drugs** |
| **HPOA vs Clubbing Alone** | **HPOA: Clubbing + Periostitis (Bone) + Arthralgia + Vascular**; **Clubbing Alone: Cyanotic Heart, Lung Fibrosis, GI, Liver** |

**Mnemonic: PARANEOPLASTIC**
- **P**araneoplastic: **Remote Effects, Not Direct Invasion**
- **A**nibody-Mediated: **Onconeural (Hu, Yo, Ri, Ma2, CV2, NMDA, VGCC, Amphiphysin, GAD, LGI1, CASPR2)**
- **R**emote **E**ffects: **Hormonal (SIADH, ACTH, PTHrP, IGF-2, Serotonin)**
- **A**nibody-Neurological: **Hu (SCLC, Encephalomyelitis), Yo (Gynae, PCD), Ri (Breast, Opsoclonus), Ma2 (Testicular, Limbic), CV2 (SCLC), NMDA (Ovarian Teratoma)**
- **N**euro **L**ambert-Eaton: **VGCC Ab, SCLC, Facilitation, 3,4-DAP**
- **E**ndocrine: **SIADH (SCLC), ACTH (SCLC/Carcinoid), PTHrP (SCC/Breast), IGF-2 (Mesenchymal), Carcinoid (NET)**
- **O**nconeural **P**araneoplastic: **PCD (Anti-Yo), LEMS (VGCC), Encephalitis (Hu, Ma2, NMDA), Neuropathy (Hu, CV2)**
- **L**ambert-Eaton: **VGCC, Facilitation, SCLC, 3,4-DAP**
- **A**canthosis **N**igricans: **Gastric, Lung, Velvety Hyperpigmentation**
- **A**utoimmune **S**kin: **Dermatomyositis (Ovarian/Lung), Sweet's (AML), Bazex (H&N), Leser-Trélat (Gastric)**
- **A**utoimmune **S**yndromes: **TTP (ADAMTS13, PLEX), DIC (APL, Adeno), Trousseau's (Mucinous Adeno)**
- **T**TP: **ADAMTS13 Def, MAHA, Thrombocytopenia, PLEX + Steroids**
- **I**mmunotherapy: **IVIG, PLEX, Rituximab, Steroids, Cyclophosphamide**
- **C**ancer Search: **CT/PET-CT, Tumour Markers, Onconeural Panel**
- **T**argeted Treatment: **Treat Cancer = Best for Paraneoplastic**

> [!tip] **FCPS/MRCP Priority: HIGH**
> **NPC = EBV-associated (Type II/III Non-keratinising/Undifferentiated)**; **Endemic: SE Asia, North Africa, Arctic**; **WHO Histology**: **Type I (Keratinising SCC, EBV-)**, **Type II (Non-keratinising, EBV+)**, **Type III (Undifferentiated, EBV+)** — **Type II/III = 90% endemic**; **AJCC 8th Staging**; **Radiosensitive** — **RT Primary**; **Early (Stage I): RT Alone (IMRT)**; **Locally Advanced (II-IVA): CCRT (Cisplatin 100mg/m² q3w ×3) ± Induction (TPF/GemCis)**; **Adjuvant Chemo controversial**; **Metastatic: Gem/Cis 1L** → **ICI (Pembro/Nivo/Tislelizumab) 2L**; **EBV DNA Monitoring**: **Plasma EBV DNA = Tumour Marker** for Dx, Response, Surveillance.

## 2. 2. Definition & Epidemiology

| Feature | Detail |
|---------|--------|
| **Definition** | Malignant tumour of nasopharyngeal epithelium; **Strong EBV Association** (Type II/III) |
| **Incidence** | **Endemic**: SE Asia (Southern China, Hong Kong), North Africa, Arctic; **UK: ~250/year** (Rare, migrant populations) |
| **Prevalence** | 5-year OS: Stage I ~90%, Stage II ~80%, Stage III ~70%, Stage IV ~50% |
| **Peak Age** | **Bimodal**: 15-25 (EBV-driven), 50-60 (Environmental) |
| **Sex Ratio** | **M:F = 2-3:1** |
| **Risk Factors** | **EBV Infection** (Necessary but not sufficient), **Genetic** (HLA-A*0207, GWAS loci), **Diet** (Salted fish, Nitrosamines), **Smoking**, **Occupational** (Formaldehyde, Wood dust) |

## 3. 3. Aetiology & Pathophysiology

```mermaid
flowchart LR
    A[EBV Infection (Latency II)] --> B[EBV Proteins Expressed]
    B --> B1[**LMP1** (Latent Membrane Protein 1) — **Constitutive CD40 Signaling** → NF-κB, JAK/STAT, PI3K/AKT]
    B --> B2[**LMP2A** — **BCR Mimic** → PI3K/AKT, Survival]
    B --> B3[**EBNA1** — **Episome Maintenance**, Immune Evasion]
    B1 --> C[**Oncogenic Pathways**]
    B2 --> C
    B3 --> C
    C --> D1[Cell Proliferation, Anti-Apoptosis]
    C --> D2[Angiogenesis (VEGF)]
    C --> D3[Invasion, Metastasis (MMPs)]
    C --> D4[Immune Evasion (PD-L1↑, HLA↓)]
```

### 1. WHO Histological Classification

| Type | Description | EBV | Frequency (Endemic) | Prognosis |
|------|-------------|-----|---------------------|-----------|
| **Type I: Keratinising SCC** | Well-differentiated, Keratin pearls, Intercellular bridges | **Negative** | **<5%** | Worse (Radioresistant) |
| **Type II: Non-keratinising** | Differentiated, No keratin, Syncytial | **Positive** | **~15-20%** | Good |
| **Type III: Undifferentiated** | **Lymphoepithelioma**, Sheets of cells, Dense lymphoid stroma | **Positive** | **~75-80%** | **Best (Most Radiosensitive)** |

**Key: Type II/III = "WHO Type II/III" = Non-keratinising = EBV+ = Radiosensitive**

## 4. 4. Staging (AJCC 8th Edition)

| T Stage | Description |
|---------|-------------|
| T1 | **Tumour confined to Nasopharynx** (Mucosa, Soft Palate) |
| T2 | **Parapharyngeal Space** Extension |
| T3 | **Bony Invasion** (Clivus, Pterygoid Plates, Sphenoid, Cervical Vertebrae) |
| T4 | **Intracranial Extension** (Cranial Nerves, Cavernous Sinus, Brain) / **Infratemporal Fossa** / **Hypopharynx** / **Orbit** |

| N Stage | Description |
|---------|-------------|
| N1 | **Unilateral** Cervical LN ≤6cm (Level II, III, Retropharyngeal) |
| N2 | **Bilateral** Cervical LN ≤6cm |
| N3 | **LN >6cm** OR **Supraclavicular Fossa** (Level IV, V) |

| Stage Grouping | |
|----------------|---|

| **II** | T1 N1 M0 / T2 N0-1 M0 |

### 1. Stage I (T1 N0)

```mermaid
flowchart TD
```

### 4. EBV DNA Monitoring

| Timepoint | Clinical Utility |
|-----------|-----------------|
| **Pre-treatment** | **Prognostic**: High EBV DNA → Worse OS; **Staging**: Detects Occult Mets |
| **Mid-RT** | **Early Response**: ↓EBV DNA predicts CR |
| **Post-RT (3mo)** | **Remission Confirmation**: Undetectable = CR |
| **Surveillance** | **q3mo ×2yr, q6mo ×3yr**; **Rise → Early Relapse Detection** |
| **Post-Salvage** | **Monitor Response** |

## 9. 9. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Adjuvant Chemo — Give or Not?** | **Intergroup 0099 (Old RT): YES**; **Modern IMRT Trials (RTOG 0225, NPC-9902): NO clear benefit**; **Current Practice: Often Omitted or Individualised** |
| **Induction Chemo — TPF vs GemCis** | **TPF: More Toxic (Neutropenia, Diarrhoea)**; **GemCis: Better Tolerated, Similar Efficacy** — **GemCis Preferred in Many Centres** |
| **EBV DNA vs Clinical Staging** | **EBV DNA Detects Occult Mets** (Upstages ~10-15%); **Used for Prognosis, Not Formal Staging (AJCC)** |
| **Type I vs II/III RT Sensitivity** | **Type I: Keratinising, EBV-, Radioresistant, Higher Dose/Boost Needed**; **Type II/III: Non-keratinising, EBV+, Highly Radiosensitive** |
| **Cisplatin Schedule in CCRT** | **100mg/m² q3w ×3 (Days 1, 22, 43)** — **Standard**; **Weekly Cisplatin 40mg/m²** used in some protocols (less nephrotoxicity) |
| **Parotid Sparing IMRT** | **Mean Dose <26Gy** → **Significantly Reduces Xerostomia** — **Critical Quality Metric** |

**Mnemonic: NPC-EBV**
- **N**asopharyngeal: **EBV-Associated** (Latency II: LMP1/2A, EBNA1)
- **P**athology: **WHO I (Keratinising, EBV-)**, **II (Non-keratinising, EBV+)**, **III (Undifferentiated, EBV+)**
- **C**CRT: **Cisplatin 100mg/m² q3w ×3 + IMRT 70Gy** (Stage II-IVA)
- **E**arly Stage I: **RT Alone (IMRT)** — No Chemo
- **B**iopsy: **EBER ISH Confirms EBV** (Type II/III)
- **V**iral DNA: **Plasma EBV DNA** — Dx, Staging, Response, Surveillance
- **I**nduction: **TPF / GemCis** for High Volume/N3/T4 (NARCTEC)
- **D**istrict: **Endemic SE Asia, N Africa**
- **A**djuvant Chemo: **Controversial** (Intergroup 0099 YES, IMRT NO)
- **M**etastatic: **Gem/Cis 1L** → **PD-1 Inhibitors 2L**
- **R**T Technique: **IMRT Parotid Sparing (<26Gy Mean)**
- **T**oxicities: **Xerostomia, Trismus, Cranial Neuropathies, Hearing, Hypothyroidism**

   - **Answer: A** — palliative: **Plasma EBV DNA = Tumour Marker** — **Dx, Staging, Response, Surveillance**


## 17. 17. Flashcards

**Q1:** EBV Association?
**A1:** Latency II: LMP1, LMP2A, EBNA1 → NF-κB, PI3K/AKT, Immune Evasion

**Q2:** WHO Types?
**A2:** Type I: Keratinising (EBV-), Type II: Non-keratinising (EBV+), Type III: Undifferentiated (EBV+) — II/III = 90% Endemic

**Q3:** Staging?
**A3:** T1: Nasopharynx, T2: Parapharyngeal, T3: Bony, T4: Intracranial; N1: Unilateral, N2: Bilateral, N3: >6cm/SCF

**Q4:** Stage I?
**A4:** RT Alone (IMRT) — 66-70Gy; Parotid Sparing

**Q5:** Stage II-IVA?
**A5:** CCRT: Cisplatin 100mg/m² q3w ×3 + IMRT 70Gy — Standard

**Q6:** Induction Chemo?
**A6:** TPF / GemCis for High Volume / N3 / T4 — NARCTEC/ZHONGSHAN: Benefit

**Q7:** Adjuvant Chemo?
**A7:** Cisplatin + 5-FU ×3 cycles — Intergroup 0099 Benefit BUT Modern RT (IMRT) Diminishes Benefit (RTOG 0225, NPC-9902)

**Q8:** Metastatic 1L?
**A8:** Gemcitabine + Cisplatin — Standard

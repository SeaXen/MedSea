> **Glioblastoma = Most Common Primary Malignant Brain Tumour (WHO CNS5 Grade 4)**; **IDH-wildtype (90%, older) vs IDH-mutant (10%, younger, better prognosis)**; **Molecular: TERT promoter mut (80%), EGFR amp (40%), PTEN loss (40%), +7/-10 chr (80%)**; **Standard: Maximal Safe Resection → Stupp Protocol (RT 60Gy/30fx + Concurrent TMZ 75mg/m² daily → Adjuvant TMZ 150-200mg/m² d1-5 q28d ×6-12 cycles)**; **MGMT Promoter Methylation = Predicts TMZ Benefit**; **Recurrence: Re-resection, Re-RT, TTFields (Optune), Lomustine, Bevacizumab, Clinical Trials**.

## 2. 2. Definition & Epidemiology

| Feature | Detail |
|---------|--------|
| **Definition** | **WHO CNS5 Grade 4** diffuse astrocytic glioma; **IDH-wildtype (90%)** or **IDH-mutant (10%)** |
| **Incidence** | **3-4/100,000/year**; **Most common primary malignant brain tumour** (50% of gliomas) |
| **Prevalence** | Median OS **15-18 months**; 5-year OS **~5-10%** |
| **Peak Age** | **IDH-wt: 60-70**; **IDH-mut: 40-50** |
| **Sex Ratio** | **M:F = 1.5:1** |
| **Risk Factors** | **Ionising Radiation** (prior RT), **Genetic Syndromes** (Li-Fraumeni - TP53, Neurofibromatosis Type 1, Turcot/Lynch), **No proven environmental link** |

### 1. Key Molecular Markers

| Marker | Frequency | Significance |
|--------|-----------|--------------|
| **IDH1/2 Mutation** | **10% (mutant)** | **Defines Secondary GBM**; Better prognosis; Younger |
| **TERT Promoter Mutation** | **80% (IDH-wt)** | Telomerase activation; Poor prognosis |
| **EGFR Amplification** | **40%** | Often with EGFRvIII variant; Target for trials |
| **PTEN Loss/Mutation** | **40%** | PI3K/AKT pathway activation |
| **+7/-10 Chromosome** | **80%** | **Hallmark of IDH-wt GBM**; EGFR on 7, PTEN on 10 |
| **MGMT Promoter Methylation** | **~40%** | **Silences DNA Repair → TMZ Sensitivity**; **Predicts Benefit** |
| **H3F3A K27M** | **0% (Excludes)** | If present → **Diffuse Midline Glioma H3 K27-altered** |

## 4. 4. Clinical Features

| Feature | Description |
|---------|-------------|
| **Headache** | **Most Common (50%)**; Progressive, Morning, Valsalva-aggravated |
| **Seizures** | **25-40%** (Focal > Generalised); Often presenting symptom |
| **Focal Neurological Deficits** | Hemiparesis, Aphasia, Visual Field Defects, Cognitive Decline |
| **Personality/Behavioural Change** | Frontal lobe involvement |
| **Raised ICP Signs** | Papilloedema, Nausea/Vomiting, Cranial Nerve Palsies (III, VI) |
| **Rapid Progression** | Weeks to Months |

## 5. 5. Diagnosis & Investigations

| Investigation | Role | Key Details |
|---------------|------|-------------|
| **MRI Brain (Gold Standard)** | **Diagnosis, Extent, Surgical Planning** | **T1 Post-Contrast: Ring-Enhancing Lesion with Central Necrosis**; **T2/FLAIR: Vasogenic Oedema**; **Crosses Corpus Callosum (Butterfly Glioma)**; **Perfusion (rCBV ↑), Spectroscopy (Cho/NAA ↑, Lipids/Lactate ↑)** |
| **CT Brain** | Emergency, Surgical Planning | Haemorrhage, Calcification, Mass Effect |
| **Biopsy / Resection** | **Histology + Molecular** | **IDH IHC (R132H), ATRX, p53, Ki-67, MGMT Methylation (Pyrosequencing/MSP)** |
| **Liquid Biopsy (ctDNA)** | Emerging | Not Standard |
| **PET-CT (FET/DOPA)** | Recurrence vs Radionecrosis | Not Routine |

## 6. 6. Staging & Classification

**No Formal TNM Staging for Primary Brain Tumours**
- **WHO CNS5 Grade 4** (by Histology + Molecular)
- **ECOG/Karnofsky Performance Status** (Critical for Treatment Decisions)
- **RPA Class (Recursive Partitioning Analysis)** / **EORTC Prognostic Factors**

### 1. EORTC Prognostic Factors (Newly Diagnosed)
1. **Age** (<50 vs ≥50)
2. **KPS** (>80 vs ≤80)
3. **Extent of Resection** (GTR vs STR vs Biopsy)
4. **MGMT Methylation** (Yes vs No)
5. **IDH Status** (Mutant vs Wildtype)

## 7. 7. Management

### 1. Newly Diagnosed GBM — Standard (Stupp Protocol)

```mermaid
flowchart TD
    A[Newly Diagnosed GBM] --> B[**Maximal Safe Resection**
**Gross Total Resection (GTR) Preferred**
**5-ALA Fluorescence Guidance**
**Awake Craniotomy if Eloquent**]
    B --> C[**Recovery + Baseline MRI (24-48h Post-Op)**]
    C --> D[**Stupp Protocol: Concurrent Chemoradiation**]
    D --> D1[**RT: 60Gy / 30 Fractions / 6 Weeks**
**Involved Field (GTV + 2cm Margin = CTV)**]
    D --> D2[**Concurrent Temozolomide (TMZ): 75mg/m² Daily ×42 Days**
**During RT**]
    D1 --> E[**4-Week Break**]
    D2 --> E
    E --> F[**Adjuvant Temozolomide**]
    F --> F1[**TMZ 150-200mg/m² Day 1-5 Every 28 Days**
**×6-12 Cycles** (Standard 6, Extended 12 if Tolerated/Responding)]
    F1 --> G[**Surveillance: MRI q2-3mo ×2yr, q3-4mo ×3yr, then q6mo**]
```

### 2. MGMT Promoter Methylation — Predictive Biomarker

| MGMT Status | TMZ Benefit | Management Consideration |
|-------------|-------------|--------------------------|
| **Methylated (~40%)** | **Significant OS Benefit** (HR ~0.5) | **Standard Stupp Protocol** |
| **Unmethylated (~60%)** | **Limited Benefit** | **Still Standard** (RT + TMZ); **Consider Clinical Trial** |
| **Elderly/Frail (>70, KPS<70)** | **Short-Course RT (40Gy/15fx) + TMZ** (Nordic Trial) **OR TMZ Alone if Methylated** |

### 3. Tumour Treating Fields (TTFields / Optune)

| Indication | Details |
|------------|---------|
| **Newly Diagnosed** | **EF-14 Trial**: TTFields + TMZ Adjuvant vs TMZ Alone → **OS HR 0.63, Median OS 20.9 vs 16.0mo** |
| **Recurrent** | **EF-11 Trial**: Non-Inferior to Chemo, Better QoL |
| **Technique** | **Transducer Arrays on Shaved Scalp**, **200 kHz**, **≥18hrs/day** |
| **Side Effects** | **Skin Irritation (Grade 1-2)**, **Logistical Burden** |

```mermaid
flowchart TD

    C --> E[**Post-Op Options**]
    D --> E
    E --> E1[**Re-RT (If >6mo Prior, Limited Volume)**
**SRS / Hypofractionated (25Gy/5fx, 35Gy/10fx)**]
    E --> E2[**TTFields (Optune) + TMZ / Lomustine**]
    E --> E3[**Lomustine (CCNU) 110-130mg/m² q6-8wk**
**Standard Single-Agent**]
    E --> E4[**Bevacizumab 10mg/kg q2wk**
**Anti-VEGF, Reduces Oedema, Improves PFS (Not OS)**]
    E --> E5[**Clinical Trials** (EGFRvIII CAR-T, Oncolytic Virus, Immunotherapy, PARPi)]
    E --> E6[**Best Supportive Care** (Dexamethasone, Antiepileptics, VTE Prophylaxis)]
```

### 1. Pseudoprogression vs True Progression (iRANO)

| Feature | **Pseudoprogression** | **True Progression** |
|---------|----------------------|---------------------|
| **Timing** | **<3-6 Months Post-RT** | **>6 Months** or **During Adjuvant TMZ** |
| **MRI** | New/Enhancing Lesion, **Often Increases Then Stabilises/Regresses** | **Progressive Enhancement, Mass Effect, New Lesions** |
| **Clinical** | **Stable/Improving** | **Worsening Deficits** |
| **Management** | **Continue TMZ, Observe, Follow-Up MRI 4-8 Weeks** | **Change Therapy** |
| **Confirmation** | **Biopsy** (Rare) / **Advanced Imaging (PET, Perfusion, Spectroscopy)** | **Biopsy / Clinical Deterioration** |

## 9. 9. Supportive Care

| Issue | Management |
|-------|------------|
| **Seizures** | **Levetiracetam Preferred** (No Enzyme Induction); **Avoid Phenytoin/Carbamazepine (CYP450 Induction → ↓TMZ Levels)** |
| **Dexamethasone** | **Minimise Dose** (Myopathy, Hyperglycaemia, Immune Suppression); **Taper ASAP Post-RT** |
| **VTE Prophylaxis** | **LMWH** (High Risk: Immobility, Surgery, Tumour); **DOACs if No Brain Mets** |
| **Antiemesis** | **Ondansetron/Dexamethasone** for TMZ |
| **Fatigue/QoL** | **Exercise, Psychosocial Support, Palliative Care Early** |

## 12. 12. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **IDH-wt vs IDH-mut GBM** | **IDH-wt = Primary (90%), Older, TERTp, EGFR, +7/-10, Poor**; **IDH-mut = Secondary (10%), Younger, TP53, ATRX, Better** |
| **MGMT Methylation = Good or Bad?** | **Methylated = GOOD** (Gene Silenced → No DNA Repair → TMZ Works); **Unmethylated = BAD for TMZ** |
| **Stupp vs Short-Course RT** | **Standard: 60Gy/30fx**; **Elderly/Frail: 40Gy/15fx + TMZ (Nordic)** |
| **Pseudoprogression vs Radionecrosis** | **Pseudo: <6mo, TMZ-Related, Regresses**; **Radionecrosis: Late (>6mo), RT-Related, May Need Surgery/Bev** |
| **Bevacizumab in GBM** | **PFS Benefit Only, NO OS Benefit**; **Used for Oedema/Symptom Control**; **Not Curative** |
| **TTFields Compliance** | **≥18hrs/day Critical**; **Skin Toxicity Main Limitation**; **Shave Scalp Monthly** |

**Mnemonic: GBM-STUPP**
- **G**BM: **IDH-wt 90%** (Primary), **IDH-mut 10%** (Secondary)
- **B**rain MRI: **Ring-Enhancing, Necrosis, Oedema, Butterfly**
- **M**olecular: **TERTp, EGFR, PTEN, +7/-10, MGMT**
- **S**tupp: **60Gy/30fx RT + TMZ 75mg/d Concurrent → TMZ 6-12 Cycles Adjuvant**
- **T**TFields: **Optune, EF-14: OS 20.9mo, ≥18hrs/day**
- **U**n methylated MGMT: **Poor TMZ Response**; **Methylated: Good**
- **P**seudoprogression: **<6mo Post-RT, Observe, Don't Switch**
- **P**rognosis: **Median OS 15-18mo**; **MGMT Met 20-24mo**; **IDH-mut 30+mo**

> [!tip] **FCPS/MRCP Priority: HIGH**
> **Metastatic Breast Cancer = Incurable but Treatable**; **Goal: Prolong Life, Palliate Symptoms, Maintain QOL**; **Subtype-Directed**: **HR+/HER2-**: CDK4/6i + AI 1L → Fulvestrant + CDK4/6i 2L (ESR1 → Fulvestrant + CDK4/6i), PIK3CA mut → Alpelisib + Fulvestrant; **HER2+**: Trastuzumab + Pertuzumab + Taxane 1L → T-DM1 2L → T-DXd 2L+ → Tucatinib 3L+; **TNBC**: PD-L1+ → Pembrolizumab + Chemo 1L → Sacituzumab Govitecan 2L+ / T-DXd / PARPi (BRCA+); **Bone Mets**: Denosumab/Zoledronic; **Brain Mets**: SRS, T-DXd, Tucatinib; **Visceral Crisis**: Chemo (Eribulin, Capecitabine, Gemcitabine); **Supportive**: Bone agents, VTE prophylaxis, Palliative care.

## 1. 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Select **1L therapy for HR+/HER2-** using CDK4/6i + AI, and sequence subsequent lines
- [ ] Manage **HER2+ MBC** with Trastuzumab/Pertuzumab → T-DM1 → T-DXd → Tucatinib
- [ ] Treat **TNBC** with immunotherapy (PD-L1+) and Sacituzumab Govitecan/T-DXd
- [ ] Manage **visceral crisis** with aggressive chemotherapy
- [ ] Treat **bone metastases** with Denosumab/Zoledronic acid
- [ ] Manage **brain metastases** with SRS, T-DXd, Tucatinib
- [ ] Apply **supportive care**: Bone health, VTE prophylaxis, palliative care integration

## 2. 2. HR+/HER2- Metastatic Breast Cancer

### 1. First-Line
| Regimen | Key Trial | Population |
|---------|-----------|------------|
| **CDK4/6i + AI** (Palbociclib/Ribociclib/Abemaciclib + Letrozole/Anastrozole) | **PALOMA-2, MONALEESA-2, MONARCH-3** | **1L Standard**, Postmeno, No visceral crisis |
| **Ribociclib + AI** | **MONALEESA-3** | **1L**, Including premeno + OFS |
| **Abemaciclib + AI** | **MONARCH-3** | **1L**, Good CNS penetration |

> **Premenopausal**: Add **OFS (Goserelin/Leuprorelin)** to all CDK4/6i + AI regimens.

### 2. Subsequent Lines (HR+/HER2-)
| Line | Regimen | Key Biomarker/Trial |
|------|---------|---------------------|
| **2L (Post CDK4/6i + AI)** | **Fulvestrant 500mg + CDK4/6i** | **PALOMA-3, MONALEESA-3, MONARCH-2** |
| **ESR1 Mutation** (Post AI) | **Fulvestrant + CDK4/6i** OR **Elacestrant** (EMERALD) | **ESR1 mut → ORR 40-50%** |
| **PIK3CA Mutation** | **Alpelisib 300mg + Fulvestrant** (SOLAR-1) | **PIK3CA mut → PFS HR 0.65** |
| **Post CDK4/6i + Fulvestrant** | **Everolimus + Exemestane** (BOLERO-2) | **mTOR inhibition** |
| **Later** | **Chemotherapy** (Capecitabine, Eribulin, Gemcitabine, Vinorelbine) | **Sequential single agents** |

### 3. CDK4/6 Inhibitor Comparison
| Agent | Dose | Key SE | Unique Feature |
|-------|------|--------|----------------|
| **Palbociclib** | 125mg d1-21/28 | Neutropenia (70%), Fatigue | First approved |
| **Ribociclib** | 600mg d1-21/28 | Neutropenia, QTc prolongation, Hepatotoxicity | **Overall OS benefit (MONALEESA-3,7)** |
| **Abemaciclib** | 150mg BD (continuous) | Diarrhoea (80%), Neutropenia (low) | **Continuous dosing**, **CNS penetration**, **MonarchE adjuvant** |

## 5. 5. Endocrine Therapy Resistance Mechanisms (HR+)

| Mechanism | Biomarker | Targeted Therapy |
|-----------|-----------|------------------|
| **ESR1 Mutation** | **Y537S, D538G** (ctDNA) | **Elacestrant (SERD)**, **Fulvestrant + CDK4/6i** |
| **PIK3CA Mutation** | **PIK3CA** (ctDNA/Tissue) | **Alpelisib + Fulvestrant** (SOLAR-1) |
| **CCND1 Amplification** | **FISH/NGS** | **CDK4/6i** (Intrinsic resistance) |
| **FGFR1 Amplification** | **FISH** | **FGFR Inhibitors** (Trials) |
| **RB1 Loss** | **IHC/NGS** | **CDK4/6i Resistance** |
| **HER2 Amplification** | **IHC/FISH** | **Add HER2-Targeted** |

## 6. 6. Visceral Crisis

| Definition | Management |
|------------|------------|
| **Rapidly progressive visceral mets** (Liver, Lung, CNS) causing organ dysfunction | **Aggressive Combination Chemotherapy** (Not endocrine/targeted alone) |
| **Preferred Regimens** | **FEC/FAC**, **Docetaxel + Capecitabine**, **Gemcitabine + Carboplatin**, **Eribulin** |
| **Duration** | **3-6 cycles** → Switch to maintenance endocrine/targeted if response |
| **Goal** | **Rapid Disease Control** → Transition to less toxic maintenance |

## 7. 7. Bone Metastases

| Agent | Dose | Key Trial |
|-------|------|-----------|
| **Denosumab** | **120mg SC q4wk** (Loading: d1,8,15,29 then q4wk) | **Superior to Zoledronic Acid** (Delay SRE HR 0.82) |
| **Zoledronic Acid** | **4mg IV q4wk** (Renal adjust) | **Standard** |
| **Duration** | **Continue until progression/toxicity** | **SRE reduction ~30-40%** |

> **Prevention**: **Calcium 1200mg + Vitamin D 800-1000IU daily**; **Dental exam pre-treatment** (ONJ risk).

## 8. 8. Brain Metastases

### 1. Management by Subtype
| Subtype | Preferred CNS Therapy |
|-------|----------------------|
| **HER2+** | **T-DXd, Tucatinib + Trastuzumab + Capecitabine, SRS** |
| **TNBC** | **SRS ± Pembrolizumab, Sacituzumab Govitecan, T-DXd (HER2-low)** |
| **HR+/HER2-** | **SRS, Consider CDK4/6i (Abemaciclib CNS penetration), Chemo** |

### 2. Local Therapy
| Scenario | Treatment |
|--------|---------|
| **1-3 Mets** | **SRS (Stereotactic Radiosurgery)** — **Standard** |
| **>3 Mets** | **WBRT (30Gy/10fx) OR Hippocampal-Avoidance WBRT + Memantine** |
| **Symptomatic Single Large** | **Surgery + SRS** |
| **Leptomeningeal** | **Intrathecal Chemo (MTX/AraC), Systemic (T-DXd, Tucatinib)** |

## 9. 9. Supportive Care

| Issue | Management |
|-------|------------|
| **Bone Health** | **Denosumab/Zoledronic Acid + Calcium/Vit D** |
| **VTE Prophylaxis** | **LMWH** if high risk (Immobility, Metastatic, Prior VTE) — **Not routine** |
| **Cardiotoxicity (CDK4/6i, HER2)** | **Baseline Echo, q3mo LVEF**; **Abemaciclib: Monitor Diarrhoea, Creatinine** |
| **Nausea (CDK4/6i)** | **Take with food, Antiemetic PRN** |
| **Diarrhoea (Abemaciclib)** | **Loperamide, Dose reduce, Hydration** |
| **QT Prolongation (Ribociclib)** | **ECG Baseline/q2wk×2, then q4wk; Correct K/Mg** |
| **Hepatotoxicity (Ribociclib/Abemaciclib)** | **LFT Baseline q2wk×2mo, then q4wk**; **Hold if ALT/AST >5xULN** |
| **Neutropenia (Palbociclib/Ribociclib)** | **CBC q2wk×2mo, then q4wk**; **Dose reduce if <1.0** |

## 12. 12. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **CDK4/6i 1L vs 2L** | **1L: + AI**; **2L: + Fulvestrant** (If progressed on AI) |
| **ESR1 mut vs PIK3CA mut** | **ESR1: AI resistance → SERD/Elacestrant**; **PIK3CA: PI3K pathway → Alpelisib** |
| **T-DM1 vs T-DXd** | **T-DM1: Trastuzumab + DM1 (Maytansine)**, 2L standard; **T-DXd: Trastuzumab + Deruxtecan (Topo I inhibitor)**, 3L+, Higher potency, CNS activity |
| **Tucatinib vs T-DXd** | **Tucatinib: TKI (HER2 selective), HER2CLIMB CNS benefit**; **T-DXd: ADC, DESTINY-Breast03, Broad CNS activity** |
| **Sacituzumab Govitecan vs T-DXd** | **Sacituzumab: TROP-2 ADC (TNBC)**; **T-DXd: HER2 ADC (HER2+, HER2-low, TNBC HER2-low)** |
| **CDK4/6i - Palbo vs Abe Neutropenia** | **Palbo: 60-70% Gr3-4 Neutropenia**; **Abe: <20% Neutropenia, Mainly Diarrhoea** |
| **Alpelisib Hyperglycaemia** | **Monitor Glucose q1wk×2mo then q4wk; Metformin prophylaxis often used** |

**Mnemonic: MBC-TREATMENT**
- **M**BC Subtypes: **HR+/HER2-**, **HER2+**, **TNBC** → Different Algorithms
- **B**reast HR+ 1L: **CDK4/6i + AI** (Palbo/Ribo/Abe + Letrozole/Anastrozole)
- **C**DK4/6i SE: **Palbo: Neutropenia; Ribo: QTc/Hepato; Abe: Diarrhoea, CNS+**
- **E**SR1 mut: **Elacestrant / Fulvestrant+CDK4/6i** (EMERALD)
- **A**pelisib: **PIK3CA mut → Alpelisib + Fulvestrant** (SOLAR-1)
- **T**NBC 1L: **PD-L1 CPS≥10 → Pembro + Chemo** (KEYNOTE-355)
- **R**ecurrent HR+: **Fulvestrant + CDK4/6i** (PALOMA-3, MONALEESA-3)
- **T**-DXd: **HER2+ 3L, HER2-low, TNBC** (DESTINY-Breast03/04)
- **E**verolimus: **Post-CDK4/6i + Fulvestrant** (BOLERO-2)
- **A**lectinib: **NOT for Breast** (ALCL/ALK+ NSCLC) — **Trastuzumab Deruxtecan = T-DXd**
- **T**ucatinib: **HER2CLIMB** (CNS mets, +Tras+Cape)
- **M**etastatic Bone: **Denosumab > Zoledronic** (SRE delay)
- **E**verolimus/proliferation: **mTOR inhibitor**
- **N**euro/Brain Mets: **HER2+: T-DXd/Tucatinib+SRS; TNBC: SRS+Systemic**
- **T**reatment Sequencing: **1L → 2L → 3L+** (Subtype specific)

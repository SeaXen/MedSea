> [!tip] **FCPS/MRCP Priority: HIGH**
> **TLS = Life-Threatening Metabolic Emergency**; **Cairo-Bishop Criteria**: Lab TLS (≥2: Uric Acid ≥476 μmol/L, K ≥6.0, Phos ≥2.1, Ca ≤1.75); Clinical TLS (Lab TLS + Creatinine >1.5x ULN, Arrhythmia, Seizure, Death); **Risk**: Burkitt, ALL (High WBC), AML (High WBC), High-Grade NHL, SCLC; **Management**: Hydration 3L/m²/d → Rasburicase (High Risk/Established) / Allopurinol (Prophylaxis); Monitor q4-6h.

## 2. 2. Definition & Pathophysiology

| Event | Mechanism |
|-------|-----------|
| **Massive Tumour Cell Lysis** | Spontaneous or Post-Chemotherapy → Intracellular Contents Released |
| **Potassium Release** | **Intracellular K⁺ (150 mmol/L)** → Hyperkalaemia |
| **Phosphate Release** | **Intracellular Phosphate** → Hyperphosphataemia |
| **Nucleic Acid Catabolism** | **Purines → Uric Acid** (via Xanthine Oxidase) → Hyperuricaemia |
| **Calcium Phosphate Precipitation** | **Hyperphosphataemia + Hypocalcaemia** → CaPO₄ Crystals in Renal Tubules |
| **Uric Acid Crystallisation** | **Low Solubility at pH<5.5** → Renal Tubular Obstruction → AKI |

---

## 3. 3. Cairo-Bishop Criteria (2004)

### 1. Laboratory TLS (≥2 of the following within 3 days pre- / 7 days post-chemotherapy)

| Parameter | Threshold |
|-----------|-----------|
| **Uric Acid** | **≥476 μmol/L (8 mg/dL)** |
| **Potassium** | **≥6.0 mmol/L** |
| **Phosphate** | **≥2.1 mmol/L (6.5 mg/dL)** (Adults) / **≥2.9 mmol/L** (Children) |
| **Calcium** | **≤1.75 mmol/L (7 mg/dL)** |

### 2. Clinical TLS (Laboratory TLS + ≥1 of)

| Complication | Definition |
|--------------|------------|
| **Renal** | Creatinine >1.5x ULN |
| **Cardiac** | Arrhythmia / Sudden Death |
| **Neurological** | Seizure |
| **Fatal** | Death |

---

## 4. 4. Risk Stratification

| Risk Group | Tumour Types | TLS Incidence | Prophylaxis |
|------------|--------------|---------------|-------------|
| **High** | **Burkitt Lymphoma**, **ALL (WBC >50-100×10⁹/L)**, **AML (WBC >50×10⁹/L)**, **High-Grade NHL (Lymphoblastic, DLBCL High Tumour Burden)**, **SCLC** | **5-25%** | **Rasburicase + Aggressive Hydration** |
| **Intermediate** | **DLBCL (Bulky)**, **Other NHL**, **Multiple Myeloma**, **Acute Leukaemias (Lower WBC)**, **Solid Tumours (High Turnover)** | **1-5%** | **Allopurinol + Hydration** |
| **Low** | **Solid Tumours (Low Turnover)**, **Indolent NHL (CLL, Follicular)**, **CML Chronic Phase** | **<1%** | **Hydration + Allopurinol PRN** |

## 5. 5. Management Algorithm

```mermaid
flowchart TD
    A[TLS Suspected / High Risk] --> B[**Aggressive Hydration: 3L/m²/d IV 0.9% Saline** (Target Urine >100mL/h)]
    B --> C[**Rasburicase 0.2mg/kg IV Daily** (Until Uric Acid Normal) — **Gold Standard**]
    C --> D[**Allopurinol 300mg OD** (If Rasburicase Unavailable / Prophylaxis)]
    D --> E[**Monitor: Uric Acid, K, Phos, Ca, Creatinine, ECG q4-6h**]
    E --> F{Uric Acid / Electrolytes}
    F -->|**K ≥6.0 / ECG Changes**| G[**Calcium Gluconate 10mL 10% IV**, **Insulin/Dextrose**, **Salbutamol**, **Dialysis if Refractory**]
    F -->|**Phos ≥2.1 / Ca ≤1.75**| H[**Phosphate Binders (Sevelamer), Calcium Supplement (If Hypocalcaemic)**]
    F -->|**AKI / Oliguria**| I[**Renal Replacement Therapy (Dialysis)**]
```

## 6. 6. Rasburicase vs Allopurinol

| Feature | **Rasburicase** | **Allopurinol** |
|---------|-----------------|-----------------|
| **Mechanism** | **Recombinant Urate Oxidase** → Converts Uric Acid → **Allantoin** (5-10x More Soluble) | **Xanthine Oxidase Inhibitor** → Prevents Uric Acid Formation |
| **Onset** | **Hours** (Rapid Uric Acid Reduction) | **Days** (Prevents New Formation Only) |
| **Indication** | **Established TLS / High Risk** | **Prophylaxis / Intermediate Risk** |
| **Contraindication** | **G6PD Deficiency** (Haemolysis, Methaemoglobinaemia) | **Hypersensitivity** (Stevens-Johnson, DRESS) |
| **Cost** | **High** | **Low** |
| **Dose** | **0.2mg/kg IV Daily** (1-5 Days) | **300mg OD** (Renal Adjust if CrCl<30) |
| **Monitoring** | **Uric Acid q4-6h**, **Methaemoglobin** | **Uric Acid Daily**, **LFTs** |

## 7. 7. G6PD Deficiency & Rasburicase

| Population | G6PD Deficiency Prevalence | Action |
|------------|---------------------------|--------|
| **African, Mediterranean, Middle Eastern, Asian** | **5-25%** | **Screen Before Rasburicase** if High Prevalence |
| **Test** | **Quantitative G6PD Assay** (Not Qualitative) | **If Deficient: Avoid Rasburicase, Use Allopurinol** |

## 8. 8. Complication Management

### 1. Hyperkalaemia (K⁺ ≥6.0 mmol/L or ECG Changes)

| Step | Treatment |
|------|-----------|
| **1. Cardiac Stabilisation** | **Calcium Gluconate 10mL 10% IV** (Over 5-10min) |
| **2. Shift K⁺ Intracellular** | **Insulin 10U + Dextrose 50mL 50% IV**; **Salbutamol 5mg Nebulised** |
| **3. Enhance Excretion** | **Furosemide 40-80mg IV** (If Urine Output Adequate); **Sodium Polystyrene Sulfonate** |
| **4. Definitive** | **Dialysis** (Refractory, Severe AKI, ECG Changes Persist) |

### 2. Hypocalcaemia (Corrected Ca ≤1.75 mmol/L)

| Scenario | Treatment |
|----------|-----------|
| **Symptomatic (Tetany, Seizures, QT Prolongation)** | **Calcium Gluconate 10mL 10% IV** (Repeat PRN) |
| **Asymptomatic** | **Oral Calcium + Vitamin D**; **Avoid IV if Phosphate High** (Risk CaPO₄ Precipitation) |

### 3. Acute Kidney Injury

| Indication for Dialysis | Modality |
|-------------------------|----------|
| **Refractory Hyperkalaemia** | **Intermittent HD / CRRT** |
| **Severe Metabolic Acidosis** | **CRRT Preferred** |
| **Fluid Overload / Pulmonary Oedema** | **CRRT / HD** |
| **Uric Acid >1000 μmol/L Refractory** | **HD** |

## 12. 12. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Rasburicase vs Allopurinol** | **Rasburicase**: Degrades Existing UA (Urate Oxidase); **Allopurinol**: Prevents New UA (XO Inhibitor) |
| **Lab TLS vs Clinical TLS** | **Lab**: ≥2 Biochemical Abnormalities; **Clinical**: Lab TLS + Organ Dysfunction |
| **Rasburicase in G6PD** | **Contraindicated** (Haemolysis); **Allopurinol Safe** |
| **Phosphate in TLS** | **↑** (Intracellular Release) → **Hypocalcaemia** (CaPO₄ Precipitation) |
| **Hypocalcaemia Correction** | **Do Not Overcorrect** → Risk CaPO₄ Precipitation; Only Symptomatic |
| **TLS vs Hypercalcaemia — Phosphate** | **TLS: Phos ↑**; **Hypercalcaemia: Phos Normal/↓** (PTHrP → Phos ↓) |
| **Allopurinol in Renal Impairment** | **Dose Reduce** (CrCl<30: 100mg OD/Alternate Day); **Rasburicase: No Renal Adjustment** |

**Mnemonic: TLS-CAIRO-BISHOP**
- **T**umour **L**ysis **S**yndrome
- **C**AIRO: **Ca** (Hypocalcaemia ≤1.75), **I** (UA ≥476), **R** (K≥6.0), **O** (Phos ≥2.1)
- **B**ishop: **Clinical = Lab + Organ Dysfunction** (Creatinine, Arrhythmia, Seizure, Death)
- **R**isk: **Burkitt, ALL, AML (High WBC), High-Grade NHL**
- **A**llopurinol: **Prophylaxis/Intermediate** (XO Inhibitor)
- **S**creen G6PD: **Rasburicase Contraindicated**
- **H**ydration: **3L/m²/d Saline**, Urine >100mL/h
- **I**mmediate: **Rasburicase 0.2mg/kg IV** (Urate Oxidase → Allantoin)
- **O**rgan Dysfunction: **AKI, Arrhythmia, Seizure**
- **P**hosphate: **↑ → Hypocalcaemia** (CaPO₄ Precipitation)

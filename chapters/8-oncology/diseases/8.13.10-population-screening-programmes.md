## 2. 2. UK Breast Screening Programme (NHS BSP)

| Parameter | Detail |
|-----------|--------|
| **Age Range** | **50-70 years** (Extension to **47-73** in progress) |
| **Interval** | **Every 3 years** |
| **Test** | **Digital Mammography** (2-view: CC + MLO) |
| **Coverage** | **~75%** of eligible women |
| **Recall Rate** | **~4-5%** |
| **Cancer Detection Rate** | **~8-9 per 1000 screened** |
| **PPV** | **~15-20%** (Of recalled) |
| **Mortality Reduction** | **~20-25%** |
| **Overdiagnosis** | **~10-20%** |

## 3. 3. UK Cervical Screening Programme (NHS CSP)

| Parameter | Detail |
|-----------|--------|
| **Age Range** | **25-64 years** |
| **Interval** | **25-49: 3 years**, **50-64: 5 years** |
| **Primary Test** | **HPV Primary Screening** (HR-HPV PCR) |
| **Triage** | **HPV+ → Cytology (LBC)**; **Cytology+ → Colposcopy**; **HPV- → Routine Recall** |
| **HPV Vaccination** | **9-valent (6,11,16,18,31,33,45,52,58)** → **~90% prevention** |
| **Self-Sampling** | **Pilot** (Increase uptake in non-attenders) |

---

## 4. 4. UK Colorectal Cancer Screening (BCSP)

| Parameter | Detail |
|-----------|--------|
| **Age Range** | **60-74 years** (Extension to **50-59** in progress) |
| **Interval** | **Every 2 years** |
| **Test** | **FIT (Faecal Immunochemical Test)** — Quantitative, Single Sample |
| **Threshold** | **≥120 µg Hb/g faeces** |
| **Colonoscopy Referral** | **FIT+ve → Colonoscopy** |
| **Mortality Reduction** | **~15-25%** |
| **FSIG (Bowel Scope)** | **Discontinued in England (2021)** |

---

## 5. 5. UK Lung Cancer Screening (Pilot)

| Parameter | Detail |
|-----------|--------|
| **Age Range** | **55-74 years** |
| **Eligibility** | **≥20 pack-year smoking history** (Current/Former <15yr quit) |
| **Test** | **Low-Dose CT (LDCT)** |
| **Interval** | **Annual** |
| **Key Trials** | **NLST (20% Mortality Reduction)**, **NELSON (24-26% in men)** |
| **Nodule Management** | **Lung-RADS** (Category 1-4) |
| **False Positives** | **High (~20-25%)** |
| **Overdiagnosis** | **~10-20%** |

## 6. 6. PSA Testing (Prostate) — Not Population Programme

| Aspect | Detail |
|--------|--------|
| **Status** | **Informed Choice Programme** (Men >50 can request PSA) |
| **Test** | **Serum PSA** (Age-specific cutoffs) |
| **Limitations** | **Low Specificity** (BPH, Prostatitis), **Overdiagnosis** (Gleason 6) |
| **ERSPC Trial** | **20% Mortality Reduction** (13yr FU), **NNS 570, NNT 18** |
| **PLCO Trial** | **No Mortality Benefit** (Contamination) |
| **MRI Pathway** | **mpMRI before Biopsy** (PI-RADS) → Reduces Unnecessary Biopsies |

## 9. 9. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **FIT vs FOBT** | **FIT**: Quantitative, Antibody-Based, Specific for Human Hb, Single Sample; **FOBT**: Guaiac, Qualitative, Non-Specific, 3 Samples |
| **Cervical HPV Primary vs Cytology** | **HPV Primary**: More Sensitive, Detects Glandular Lesions; **Cytology Triage** of HPV+ Reduces Colposcopies |
| **FIT Threshold** | **Lower Threshold** = More Sensitivity, More Colonoscopies; **Higher** = More Specificity, Missed Cancers |
| **PSA vs Organised Screening** | **PSA**: Not Population Programme; **Overdiagnosis**, **Lead Time Bias**, **No Mortality Benefit in PLCO** |
| **FSIG Discontinued** | **One-off FSIG at 55** (Discontinued 2021); **FIT Superior** (Whole Colon, Non-Invasive) |

**Mnemonic: POP-SCREEN**
- **P**opulation Breast: **50-70, Mammo 3yr**
- **O**ropharyngeal/Cervical: **25-64, HPV Primary, 3/5yr**
- **P**ulmonary/Lung: **55-74, 20pk-yr, LDCT Annual**
- **S**tool/Colorectal: **60-74, FIT 2yr, 120µg/g**
- **C**hoice PSA: **Informed Choice, Not Population**
- **R**ecall Rate: **Breast 4-5%, Cervical HPV+**
- **E**xtension: **Breast 47-73, Colorectal 50-59**
- **E**vidence: **NLST 20% ↓, NELSON 24% ↓**
- **E**valuation: **Sens/Spec/PPV/NPV**
- **N**ot PSA: **Informed Choice, Overdiagnosis**

## 2. 2. Hypercalcaemia of Malignancy

### 1. Pathophysiology & Aetiology

| Mechanism | Tumours | Mediator |
|-----------|---------|----------|
| **Humoral (PTHrP)** | **SCC (Lung, H&N, Cervix), Breast, RCC, Ovarian** | **PTHrP (PTH-related Peptide)** |
| **Osteolytic Mets** | **Breast, Myeloma, Prostate, Lung** | **RANKL, IL-6, TNF-α** |
| **Vitamin D (1,25-OH2D)** | **Lymphoma (HL, NHL), Sarcoidosis** | **1,25-(OH)2D (Macrophage)** |
| **Ectopic PTH** | **Rare (Parathyroid Ca, Lung)** | **PTH** |

### 2. Clinical Features

| Feature | Description |
|---------|-------------|
| **Polyuria/Polydipsia** | **Nephrogenic DI (Ca Impairs AQP2)** |
| **Nausea/Vomiting/Constipation** | **GI Atony** |
| **Abdominal Pain** | **Pancreatitis, Peptic Ulcer** |
| **Confusion/Lethargy/Coma** | **Neurotoxicity (Severe >3.5mmol/L)** |
| **Cardiac** | **Shortened QT, Arrhythmias (Brady/AV Block)** |
| **Renal** | **Stones, Nephrocalcinosis, AKI** |

### 3. Diagnostic Criteria

| Parameter | Threshold |
|-----------|-----------|
| **Corrected Calcium** | **>2.6 mmol/L** (or **>2.8 mmol/L** for Clinical Significance) |
| **Albumin Correction** | **Corrected Ca = Measured Ca + 0.02 × (40 - Albumin)** |
| **Ionised Calcium** | **>1.3 mmol/L** (Gold Standard) |
| **PTH** | **Suppressed (<10 pg/mL)** — **Confirms Malignancy-Related** |
| **PTHrP** | **Elevated** (If Available) |
| **Vitamin D Metabolites** | **1,25-(OH)2D Elevated (Lymphoma)** |

### 4. Management Algorithm

```mermaid
flowchart TD
    A[Hypercalcaemia >2.8 mmol/L or Symptomatic] --> B[**IV 0.9% Saline 1-3L/24h** (Target Urine Output >100mL/h)]
    B --> C[**Bisphosphonate: Zoledronic Acid 4mg IV over 15min** (Onset 24-48h, Peak 2-4d, Duration 2-4wk)]
    C --> D{Severity / Response}
    D -->|**Severe (>3.5 mmol/L) / Refractory**| E[**Calcitonin 4-8IU/kg SC/IV q6-12h** (Rapid Onset 2-4h, Tachyphylaxis 48h)]
    D -->|**Refractory / Renal Impairment**| F[**Denosumab 120mg SC** (Onset 2-4d, No Renal Adjustment)]
    D -->|**Vitamin D Mediated (Lymphoma)**| G[**Corticosteroids (Prednisolone 40-60mg)**]
    E --> H[**Monitor Ca q6-12h, Renal Function, ECG**]
    F --> H
    G --> H
    H --> I[**Treat Underlying Malignancy**]
```

### 5. Bisphosphonate vs Denosumab

| Feature | **Zoledronic Acid** | **Denosumab** |
|---------|---------------------|---------------|
| **Onset** | 24-48h | 2-4 days |
| **Duration** | 2-4 weeks | 4+ weeks |
| **Renal Impairment** | **CrCl <30: Dose Reduce/Avoid** | **No Adjustment** (Preferred) |
| **Osteonecrosis Jaw** | **Risk (Dental Exam Pre-Tx)** | **Similar Risk** |
| **Hypocalcaemia** | **Monitor Ca, Vit D, Phosphate** | **Higher Risk (Supplement Ca/Vit D)** |

## 3. 3. Tumour Lysis Syndrome (TLS)

### 1. Cairo-Bishop Criteria

| Laboratory TLS (≥2 within 3d pre / 7d post chemo) | Clinical TLS |
|--------------------------------------------------|--------------|
| **Uric Acid ≥476 μmol/L (8 mg/dL)** | **Lab TLS +** |
| **Potassium ≥6.0 mmol/L** | **Creatinine >1.5x ULN** |
| **Phosphate ≥2.1 mmol/L (6.5 mg/dL)** | **Cardiac Arrhythmia** |
| **Calcium ≤1.75 mmol/L (7 mg/dL)** | **Seizure** |
| | **Sudden Death** |

### 2. Risk Stratification

| Risk Group | Tumours | Prophylaxis |
|------------|---------|-------------|
| **High** | **Burkitt, ALL (High WBC), AML (High WBC), NHL (High Grade), SCLC** | **Rasburicase + Aggressive Hydration (3L/m²/d)** |
| **Intermediate** | **DLBCL, Other NHL, Multiple Myeloma, Solid Tumours (High Turnover)** | **Allopurinol + Hydration (2-3L/d)** |
| **Low** | **Solid Tumours (Low Turnover), Indolent NHL** | **Hydration + Allopurinol PRN** |

### 3. Management

```mermaid
flowchart TD
    A[TLS Suspected / High Risk] --> B[**Aggressive Hydration: 3L/m²/d IV 0.9% Saline** (Target Urine >100mL/h)]
    B --> C[**Rasburicase 0.2mg/kg IV Daily** (Until Uric Acid Normal) — **Gold Standard**]
    C --> D[**Allopurinol 300mg OD** (If Rasburicase Unavailable / Prophylaxis)]
    D --> E[**Monitor: Uric Acid, K, Phos, Ca, Creatinine, ECG q4-6h**]
    E --> F{Uric Acid / Electrolytes}
    F -->|**K ≥6.0 / ECG Changes**| G[**Calcium Gluconate 10mL 10% IV**, **Insulin/Dextrose**, **Salbutamol**, **Dialysis if Refractory**]
    F -->|**Phos ≥2.1 / Ca ≤1.75**| H[**Phosphate Binders (Sevelamer), Calcium Supplement (If Hypocalcaemic)**]
    F -->|**AKI / Oliguria**| I[**Renal Replacement Therapy (Dialysis)**]
```

### 4. Rasburicase vs Allopurinol

| Feature | **Rasburicase** | **Allopurinol** |
|---------|-----------------|-----------------|
| **Mechanism** | **Recombinant Urate Oxidase (Converts Uric Acid → Allantoin)** | **Xanthine Oxidase Inhibitor (Prevents Uric Acid Formation)** |
| **Onset** | **Hours** | **Days** |
| **Indication** | **Established TLS / High Risk** | **Prophylaxis / Intermediate Risk** |
| **Contraindication** | **G6PD Deficiency (Haemolysis)** | **Hypersensitivity** |
| **Cost** | **High** | **Low** |

## 4. 4. SIADH (Syndrome of Inappropriate ADH Secretion)

### 1. Aetiology in Malignancy

| Tumour | Association |
|--------|-------------|
| **SCLC** | **Most Common (10-15%)** — **Ectopic ADH** |
| **NSCLC** | **Adenocarcinoma** |
| **Head & Neck** | **SCC** |
| **Lymphoma** | **HL, NHL** |
| **CNS Tumours** | **Direct Hypothalamic Effect** |
| **Drugs** | **Cyclophosphamide, Vincristine, Cisplatin, Opioids, SSRIs** |

### 2. Diagnostic Criteria (Schwartz-Bartter)

| Criterion | Required |
|-----------|----------|
| **Hyponatraemia** | **Na <135 mmol/L** |
| **Serum Osmolality** | **<275 mOsm/kg** |
| **Urine Osmolality** | **>100 mOsm/kg (Inappropriately Concentrated)** |
| **Urine Sodium** | **>40 mmol/L** |
| **Euvolemia** | **No Oedema, No Dehydration, Normal Skin Turgor** |
| **Normal Thyroid/Adrenal/Renal Function** | **Exclude Hypothyroidism, Adrenal Insufficiency, Renal Failure** |
| **No Diuretic Use** | **Recent** |

### 3. Management Algorithm

```mermaid
flowchart TD
    A[Hyponatraemia <130 mmol/L or Symptomatic] --> B{Severity}
    B -->|**Severe (<120 / Seizures / Coma)**| C[**Hypertonic Saline 3% 1-2mL/kg/h** (Target Na Rise 4-6 mmol/L in 24h, Max 8-10 mmol/L/24h)]
    B -->|**Moderate (120-129) / Asymptomatic**| D[**Fluid Restriction 800-1000mL/d** (First-Line)]
    D --> E{Response}
    E -->|**Na Not Rising**| F[**Demeclocycline 600-1200mg/d** (V2 Antagonist, Onset 2-4d, Photosensitivity)]
    E -->|**Still Not Rising**| G[**Tolvaptan 15-30mg OD** (V2 Antagonist, Aquaretic, Avoid >24h, Monitor Liver)]
    F --> H[**Monitor Na q6-12h, Urine Output, Osmolality**]
    G --> H
    C --> H
    H --> I[**Treat Underlying Cause (Chemo for SCLC, etc.)**]
```

### 4. Correction Rate Limits

| Scenario | Max Rate |
|----------|----------|
| **Acute (<48h)** | **8-10 mmol/L/24h** |
| **Chronic (>48h)** | **6-8 mmol/L/24h** (Max 12-14 mmol/L/48h) |
| **Risk** | **Osmotic Demyelination (Central Pontine Myelinolysis) if Over-Correction** |

## 5. 5. Neutropenic Sepsis

### 1. Definition

| Criterion | Threshold |
|-----------|-----------|
| **Neutropenia** | **ANC <0.5 ×10⁹/L** (or **<1.0 ×10⁹/L** Predicted to Fall) |
| **Fever** | **≥38.3°C (Single) OR ≥38.0°C Sustained >1h** |

### 2. MASCC Risk Index (Outpatient vs Inpatient)

| Factor | Score |
|--------|-------|
| **Burden of Illness** | **No/Mild (5), Moderate (3), Severe (0)** |
| **No Hypotension** | **5** |
| **No COPD** | **4** |
| **Solid Tumour / No Prior Fungal** | **4** |
| **No Dehydration** | **3** |
| **Outpatient Status** | **3** |
| **Age <60** | **2** |

| Total Score | Risk | Management |
|-------------|------|------------|
| **≥21** | **Low** | **Oral Antibiotics (Ciprofloxacin + Amoxicillin/Clavulanate) ×5-7d, Outpatient** |
| **<21** | **High** | **IV Antibiotics, Inpatient** |

### 3. Empiric IV Antibiotics

| Regimen | Coverage |
|---------|----------|
| **Piperacillin-Tazobactam 4.5g IV q6h** | **Pseudomonas, ESBL, Anaerobes** (Preferred Monotherapy) |
| **Meropenem 1g IV q8h** | **Carbapenem Alternative (If Beta-Lactam Allergy / ESBL High Prevalence)** |
| **Add Vancomycin 15-20mg/kg IV q12h** | **If: Hypotension, Catheter-Related, Skin/Soft Tissue, MRSA Suspected, Penicillin Allergy** |
| **Add Antifungal (Caspofungin/Amphotericin B)** | **If: Persistent Fever >4-7d, Clinical Deterioration, High-Risk** |

### 4. G-CSF Prophylaxis

| Indication | Agent |
|------------|-------|
| **Primary Prophylaxis** | **Febrile Neutropenia Risk >20%** (e.g., Dose-Dense Chemo, Age>65, Comorbidities) |
| **Secondary Prophylaxis** | **Prior Episode of FN** |
| **Agent** | **Filgrastim 5mcg/kg SC Daily** OR **Pegfilgrastim 6mg SC Single Dose (Day 2 Post-Chemo)** |

## 6. 6. Superior Vena Cava Obstruction (SVCO)

### 1. Aetiology & Presentation

| Cause | Frequency |
|-------|-----------|
| **Lung Cancer (SCLC, NSCLC)** | **85%** |
| **Lymphoma (HL, NHL)** | **10%** |
| **Mediastinal Tumours (Thymoma, Germ Cell)** | **5%** |
| **Benign (Fibrosis, Thrombosis, Catheter)** | **Rare** |

| Symptom | Frequency |
|---------|-----------|
| **Dyspnoea** | **60-80%** |
| **Facial/Neck/Arm Swelling** | **50-70%** |
| **Cough** | **40-50%** |
| **Distended Neck/Chest Veins** | **40-60%** |
| **Headache/Confusion** | **Cerebral Oedema (Late)** |

### 2. Management Algorithm

```mermaid
flowchart TD
    A[Suspected SVCO] --> B[**Dexamethasone 16mg IV Daily** (Reduce Oedema, Symptom Relief)]
    B --> C[**Imaging: CT Chest (Contrast) / Venography** — **Confirm Diagnosis, Assess Thrombus**]
    C --> D[**Oncology Referral: Tissue Diagnosis (If Not Known)**]
    D --> E{Treatment Approach}
    E -->|**SCLC / Lymphoma / Chemosensitive**| F[**Chemotherapy (Urgent)** — **Rapid Response**]
    E -->|**NSCLC / Radiosensitive / Symptomatic**| G[**RT: 30Gy/10fx or 20Gy/5fx (Rapid Palliation)**]
    E -->|**Severe / Life-Threatening / Thrombus**| H[**Stenting (Endovascular) — Immediate Relief**]
    E -->|**Thrombus Confirmed**| I[**Anticoagulation (LMWH) ± Thrombolysis/Thrombectomy**]
    F --> J[**Monitor: Symptoms, Venous Flow, Tumour Response**]
    G --> J
    H --> J
    I --> J
```

### 3. RT vs Stenting

| Approach | Indication | Speed of Relief |
|----------|------------|-----------------|
| **RT** | **NSCLC, Radiosensitive, Not Immediately Life-Threatening** | **Days (2-3 Fractions)** |
| **Stenting** | **Life-Threatening, Thrombus, Failed RT, NSCLC** | **Immediate (Hours)** |
| **Chemo** | **SCLC, Lymphoma, Chemosensitive** | **Days-Weeks** |

## 7. 7. Malignant Spinal Cord Compression (MSCC)

### 1. Definition & Aetiology

| Cause | Frequency |
|-------|-----------|
| **Metastatic Deposits (Vertebral Body → Epidural Space)** | **>95%** |
| **Common Primaries** | **Prostate, Breast, Lung, Myeloma, Lymphoma, RCC** |
| **Direct Extension (Paravertebral Mets)** | **<5%** |

### 2. Clinical Presentation

| Feature | Frequency |
|---------|-----------|
| **Back Pain (Local, Nocturnal, Worse Lying Down)** | **90-95%** |
| **Motor Weakness** | **60-80%** |
| **Sensory Loss (Level)** | **50-70%** |
| **Sphincter Dysfunction (Urinary Retention, Constipation)** | **40-60%** |
| **Lhermitte's Sign (Neck Flexion → Electric Shock)** | **Classic** |

### 3. MSCC Clinical Grading (Bilsky / Tomita)

| Grade | Features |
|-------|----------|
| **1** | **Vertebral Body Only** |
| **2** | **Epidural Impingement, No Cord Compression** |
| **3** | **Cord Compression (No CSF Space)** |
| **4** | **Cord Displacement/Deformation** |

### 4. Management Algorithm

```mermaid
flowchart TD
    A[Suspected MSCC] --> B[**Dexamethasone 16mg IV Stat, Then 4mg q6h** (Reduce Oedema, Preserve Function)]
    B --> C[**Urgent MRI Whole Spine (Gold Standard)** — **Confirm Level, Extent, Number**]
    C --> D{Neurological Status}
    D -->|**Ambulatory (Grade 1-2)**| E[**Urgent RT: 20Gy/5fx or 30Gy/10fx (Within 24h)**]
    D -->|**Non-Ambulatory <48h (Grade 3-4)**| F[**Decompressive Surgery + RT (If: Single Level, Good Prognosis, Life Expectancy >3mo)**]
    D -->|**Non-Ambulatory >48h / Complete Paraplegia**| G[**RT Alone (Surgery Futile)**]
    E --> H[**Monitor: Neuro Checks q4h, DVT Prophylaxis, Bladder Care**]
    F --> H
    G --> H
```

### 5. Surgery vs RT

| Scenario | Preferred |
|----------|-----------|
| **Single Level, Good PS, Life Exp >3mo, Radioresistant (RCC, Melanoma, Sarcoma)** | **Surgery + RT** |
| **Multiple Levels, Chemosensitive (Lymphoma, SCLC), Radiosensitive** | **RT Alone** |
| **Paraplegia >48h** | **RT Alone (Surgery Futile)** |

## 8. 8. Airway Obstruction (Malignant)

### 1. Causes & Presentation

| Cause | Tumours |
|-------|---------|
| **Endobronchial Tumour** | **Lung (NSCLC, SCLC), Metastatic (Breast, Colon, Melanoma)** |
| **Extrinsic Compression** | **Mediastinal Lymphoma, Thymoma, Thyroid, Lymph Nodes** |
| **Tracheal Invasion** | **Thyroid, Lung, Oesophageal** |

| Symptom | Urgency |
|---------|---------|
| **Stridor** | **Emergency** |
| **Dyspnoea at Rest** | **Emergency** |
| **Hoarseness** | **Urgent** |
| **Cough/Haemoptysis** | **Urgent** |

### 2. Management

| Intervention | Indication |
|--------------|------------|
| **Bronchoscopy (Rigid/Flexible)** | **Diagnosis, Debulking, Stent Placement** |

| **RT** | **Primary (If Unresectable), Post-Stent/Debulking** |

## 9. 9. Raised Intracranial Pressure (ICP)

### 1. Aetiology in Malignancy

| Cause | Examples |
|-------|----------|
| **Brain Mets** | **Lung, Breast, Melanoma, RCC** |
| **Primary Brain Tumour** | **GBM, Meningioma, Medulloblastoma** |
| **Cerebral Oedema (RT, Chemo)** | **RT-Induced, Chemo-Induced (Methotrexate)** |
| **Venous Sinus Thrombosis** | **Hypercoagulable, Tumour Invasion** |
| **Carbamazepine/Opioids** | **Drug-Induced** |

### 2. Management

| Intervention | Dose/Details |
|--------------|--------------|
| **Dexamethasone** | **16mg IV Stat → 4mg q6h** (Reduce Vasogenic Oedema) |
| **Mannitol 20%** | **0.5-1g/kg IV over 30min** (Osmotic Diuresis, q4-6h) |
| **Hypertonic Saline 3%** | **2-5mL/kg/h** (Alternative to Mannitol) |
| **Hyperventilation** | **PaCO2 30-35 mmHg** (Temporary, <24h, Cerebral Vasoconstriction) |
| **EVD** | **CSF Diversion (Hydrocephalus, Raised ICP Refractory)** |
| **Head Elevation 30°** | **Promotes Venous Drainage** |
| **Avoid Hypotension/Hypoxia** | **Maintain CPP >70 mmHg** |

## 12. 12. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Corrected Calcium Formula** | **Corrected Ca = Measured Ca + 0.02 × (40 - Albumin)** (Albumin in g/L) |
| **Zoledronic vs Denosumab in Renal Failure** | **Denosumab Preferred (No Renal Adjustment)**; **Zoledronic: CrCl <30 Avoid/Reduce** |
| **Rasburicase in G6PD** | **Contraindicated (Haemolysis)** — **Screen G6PD if High Prevalence Area** |
| **Hypertonic Saline vs Mannitol** | **Hypertonic Saline: Better Volume Status, Less Diuresis**; **Mannitol: Osmotic Diuresis, Monitor Volume** |
| **SIADH vs CSW (Cerebral Salt Wasting)** | **SIADH: Euvolemic, Urine Na>40**; **CSW: Hypovolemic, Urine Na>40, High Urine Output** |
| **MASCC Score Components** | **Burden (5), No Hypotension (5), No COPD (4), Solid Tumour (4), No Dehydration (3), Outpatient (3), Age<60 (2)** |
| **SVCO Stent vs RT** | **Stent: Immediate Relief, Life-Threatening, Thrombus**; **RT: Days, Radiosensitive, Not Imminent** |
| **MSCC RT vs Surgery** | **Surgery: Single Level, Radioresistant (RCC, Melanoma), Good PS, <48h Paraplegia**; **RT: Multiple Levels, Radiosensitive, Paraplegia >48h** |
| **Correction Rate Hyponatraemia** | **Acute: 8-10 mmol/L/24h**; **Chronic: 6-8 mmol/L/24h** — **Over-Correction → Central Pontine Myelinolysis** |

**Mnemonic: ONCO-EMERGENCY**
- **O**ncologic **H**ypercalcaemia: **Saline → Zoledronic → Denosumab/Calcitonin**
- **N**eutropenic **S**epsis: **MASCC ≥21 Oral, <21 IV Pip-Tazo**
- **C**O **T**umour **L**ysis: **Cairo-Bishop (K, Phos, UA, Ca) → Rasburicase**
- **S**IADH: **Fluid Restrict 800-1000mL → Hypertonic Saline (Severe) → Tolvaptan**
- **E**mergency **S**VCO: **Dex 16mg → RT/Stent/Chemo**
- **M**RCC: **Dex 16mg → MRI Spine → RT 20/5 (Ambulatory) / Surgery (Radioresistant)**
- **A**irway: **Bronchoscopy → Stent/Debulk/RT**
- **R**aised **I**CP: **Dex 16mg → Mannitol/Hypertonic → EVD**
- **N**eutropenia **G**-CSF: **Primary >20% Risk, Secondary Prior FN**
- **E**lectrolytes **R**eplacement: **K (Insulin/Glucose), Phos (Binders), Ca (Gluconate)**
- **M**onitoring: **Na q6-12h (SIADH), Ca q6-12h (HyperCa), K/Phos/UA q4-6h (TLS)**
- **D**examethasone: **Universal 16mg Stat → 4mg q6h (SVCO, MSCC, ICP, Brain Mets)**
- **E**arly **R**ecognition: **Key to Survival in All**

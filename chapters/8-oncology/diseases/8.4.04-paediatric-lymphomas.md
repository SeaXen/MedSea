> [!tip] **FCPS/MRCP Priority: HIGH**
> **Paediatric Lymphoma = 3rd Most Common Childhood Cancer**; **Hodgkin Lymphoma (HL) 40%, NHL 60%**; **HL: cHL (NS, MC, LD, LR) + NLPHL**; **Staging: Ann Arbor (Cotswolds)**; **Risk Adapted (OEPA/German)**: **Early Favourable: 2x OEPA → PET → RT if PET+**; **Advanced: OEPA/OECO → PET → RT if PET+**; **NHL**: **Burkitt (BL) t(8;14) MYC-IGH**, **BFM/COG (CYVE, COPADM)**; **DLBCL: R-CHOP/R-EPOCH**; **ALCL: ALK+ (t(2;5) NPM-ALK) 80%**, **ALK-**; **Primary Mediastinal B-Cell (PMBCL)**: **R-CHOP + RT**; **Lymphoblastic: T-ALL Like**.

## 2. 2. Hodgkin Lymphoma (HL)

### 1. Epidemiology & Classification

| Feature | Detail |
|---------|--------|
| **Incidence** | **~15-20% Childhood Cancer**; **Peak 15-19yr** (Adolescent); **Rare <5yr** |
| **Male:Female** | **1:1 (Adolescent)**; **M>F <10yr** |
| **WHO Classification** | **cHL (95%)**: **NS (70%), MC (20-25%), LD (5%), LR (1%)**; **NLPHL (5%)** |
| **RS Cells** | **cHL: CD30+, CD15+, PAX5+, CD20-/+ (Weak), ALK-, EBV+ (40%)** |
| **LP Cells** | **NLPHL: CD20+, CD79a+, BCL6+, CD15-, CD30-, EMA+, EBV-** |

### 2. Ann Arbor Staging (Cotswolds Modification)

| Stage | Definition |
|-------|------------|
| **I** | Single Lymph Node Region (I) or Single Extralymphatic Site (IE) |
| **II** | ≥2 Lymph Node Regions **Same Side of Diaphragm** (II) or Local Extralymphatic Extension (IIE) |
| **III** | Lymph Node Regions **Both Sides of Diaphragm** (III) ± Splenic (IIIS) ± Extralymphatic (IIIE) |
| **IV** | **Diffuse Extralymphatic Involvement** (Liver, Bone Marrow, Lung, CNS) |

**Bulky Disease**: **Mediastinal Mass Ratio >0.33** (Max Mediastinal Width / Thoracic Width) OR **Node ≥6cm** (cHL) / **≥5cm** (NHL)

**B Symptoms**: **Fever >38°C, Drenching Night Sweats, Weight Loss >10% 6mo**

---

## 3. 3. Risk Stratification & Treatment (OEPA / German HL Protocols)

### 1. Early Stage (I-II) — Favourable vs Unfavourable

| Risk Group | Criteria (OEPA) | Treatment |
|------------|-----------------|-----------|
| **Early Favourable** | **Stage IA-IIA, No B Symptoms, No Bulky, ESR<50** | **2x OEPA → PET (Deauville) → RT 19.8Gy ONLY if PET+ (Deauville 4-5)** |
| **Early Unfavourable** | **Stage I-II with B Symptoms OR Bulky OR ESR≥50** | **4x OEPA → PET → RT 30Gy if PET+** |

**OEPA Regimen (1 Cycle = 28 Days):**
- **Vincristine 1.5mg/m² (Max 2mg) d1, 8, 15**
- **Etoposide 150mg/m² IV d1-3**
- **Prednisone 60mg/m² PO d1-14**
- **Doxorubicin 30mg/m² IV d1, 15**

### 2. Advanced Stage (III-IV)

| Risk Group | Treatment |
|------------|-----------|
| **Advanced** | **OEPA (2 cycles) → OECO (2-4 cycles) → PET → RT 30Gy to Initial Bulky/Residual PET+ Sites** |

**OECO Regimen (1 Cycle = 28 Days):**
- **Vincristine 1.5mg/m² d1, 8, 15**
- **Etoposide 150mg/m² d1-3**
- **Cyclophosphamide 600mg/m² d1**
- **Prednisone 60mg/m² d1-14**
- **Doxorubicin 30mg/m² d1, 15** (Same as OEPA)

### 3. PET Response Assessment (Deauville 5-Point Scale)

| Score | Definition | Action |
|-------|------------|--------|
| **1-2** | **Complete Metabolic Response** (Uptake ≤Mediastinum/Liver) | **No RT** (Early Favourable) / **No RT** (Advanced) |
| **3** | **Uptake >Mediastinum ≤Liver** | **No RT** (Usually) |
| **4-5** | **Uptake >Liver / New Lesions** | **RT 19.8Gy (Early Fav) / 30Gy (Early Unfav/Advanced)** |

> **Key: PET-Adapted RT Omission = Standard** — **Reduces Late Toxicity (Secondary Cancers, Cardiotoxicity, Hypothyroidism)**

## 4. 4. Nodular Lymphocyte-Predominant HL (NLPHL)

| Feature | Detail |
|---------|--------|
| **Age** | **Younger (Median 10-15yr)**, **Male Predominance** |
| **Stage** | **Often Early (I-II)** |
| **Cells** | **LP Cells (Popcorn): CD20+, CD79a+, BCL6+, CD15-, CD30-, EMA+, EBV-** |
| **Background** | **Nodular, B-Cell Rich** |
| **Treatment** | **Stage IA: Excision + Observation** (No Chemo/RT) / **Advanced: ABVD-like / Rituximab + Chemo** |
| **Prognosis** | **Excellent** (But Late Relapse, Transformation to DLBCL ~10%) |

## 5. 5. Non-Hodgkin Lymphoma (NHL) — Paediatric

### 1. Common Subtypes

| Subtype | Frequency | Key Features | Treatment |
|---------|-----------|--------------|-----------|
| **Burkitt Lymphoma (BL)** | **40-50% NHL** | **t(8;14) MYC-IGH**, **CD20+, CD10+, BCL6+, Ki-67 ~100%**, **EBV+ (Endemic)** | **BFM/COG: CYVE/COPADM (Intensive, Short)** |
| **DLBCL** | **20-30% NHL** | **Germinal Centre / Activated B-Cell**, **CD20+, BCL2+, BCL6+** | **R-CHOP / R-EPOCH** (Rituximab Added) |
| **ALCL (ALK+)** | **10-15% NHL** | **t(2;5) NPM-ALK**, **CD30+, ALK+, EMA+**, **CD3-/CD4+**, **Skin/Nodes/Lung** | **ALCL99: Vincristine/Pred → VPA (Vinblastine, Pred, Adriamycin)** |
| **ALCL (ALK-)** | **5% NHL** | **DUSP22-IRF4, TP63**, **Poorer Prognosis** | **ALCL99 + Brentuximab (CD30)** |
| **PMBCL** | **5-10% NHL** | **Mediastinal, CD20+, CD30+, MAL+, PD-L1+** | **R-CHOP + RT (DA-EPOCH-R Alternative)** |
| **Lymphoblastic Lymphoma (T-LBL)** | **20-25% NHL** | **T-Cell Precursors**, **CD3+, TdT+, CD1a+**, **Mediastinal Mass** | **ALL-Type (BFM/COG): High-Dose MTX, Ara-C, Asparaginase** |

## 6. 6. Burkitt Lymphoma (BL) — BFM / COG Protocols

### 1. Key Features
- **t(8;14) MYC-IGH (85%)**, **t(2;8), t(8;22) Variants**
- **CD20+, CD10+, BCL6+, Ki-67 ~100%**
- **"Starry Sky" Histology** (Tingible Body Macrophages)
- **Rapid Proliferation (Doubling Time ~24h)** — **TLS Risk Very High**

### 2. Risk Stratification (BFM)

| Group | Criteria | Treatment |
|-------|----------|-----------|
| **R1 (Low)** | **Resected Stage I/II, No CNS/BM** | **CYVE (Reduced)** |
| **R2 (Standard)** | **Stage III, No CNS/BM** | **CYVE (Standard)** |
| **R3 (High)** | **Stage IV, CNS/BM+** | **COPADM (Intensive CNS-Directed)** |

### 3. CYVE Regimen (Cyclophosphamide, Vincristine, Etoposide, VP-16, Doxorubicin, Methotrexate, Cytarabine)

| Phase | Key Drugs |
|-------|-----------|
| **Prophase** | **Cyclophosphamide, Vincristine, Prednisone, Intrathecal MTX** (Cytoreduction, TLS Prevention) |
| **Cytoreduction** | **Vincristine, Prednisone, IT MTX** |
| **Induction** | **Cyclophosphamide, Doxorubicin, Vincristine, IT MTX** |
| **Consolidation** | **High-Dose MTX (3g/m²) + IT MTX**, **Cytarabine, Etoposide** |
| **Re-induction** | **Similar to Induction** |
| **Maintenance** | **Oral MTX/MP, IT MTX** |

## 7. 7. ALCL (Anaplastic Large Cell Lymphoma)

### 1. ALK+ vs ALK-

| Feature | **ALK+ (80%)** | **ALK- (20%)** |
|---------|----------------|----------------|
| **Genetics** | **t(2;5) NPM-ALK** | **DUSP22-IRF4 (30%), TP63 (30%)** |
| **Age** | **Children/Adolescents** | **Older Adolescents/Young Adults** |
| **IHC** | **CD30+, ALK+, EMA+, CD3-/CD4+** | **CD30+, ALK-, EMA Variable** |
| **Prognosis** | **Excellent (85-90% OS)** | **Intermediate** |
| **Treatment** | **ALCL99: Vincristine/Prednisone → VPA** | **ALCL99 + Brentuximab Vedotin (CD30)** |

### 2. ALCL99 Protocol

| Phase | Regimen |
|-------|---------|
| **Prephase** | **Vincristine 1.5mg/m² weekly ×2**, **Prednisone 60mg/m² d1-14** |
| **VPA** | **Vinblastine 6mg/m² d1, 8, 15**, **Prednisone 60mg/m² d1-14**, **Doxorubicin 30mg/m² d1, 15** (×3-6 cycles) |
| **Maintenance** | **Vincristine/Prednisone pulses** |

## 11. 11. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **cHL vs NLPHL** | **cHL: RS Cells (CD30+, CD15+, PAX5+, CD20-)**; **NLPHL: LP Cells (CD20+, CD79a+, BCL6+, CD15-, CD30-)** |
| **OEPA vs OECO** | **OEPA: Vincristine, Etoposide, Prednisone, Doxorubicin**; **OECO: Vincristine, Etoposide, Cyclophosphamide, Prednisone, Doxorubicin** (Cyclo replaces Vincristine weekly) |
| **Early Favourable vs Unfavourable HL** | **Fav: IA-IIA, No B, No Bulky, ESR<50 → 2xOEPA + PET**; **Unfav: B Symptoms OR Bulky OR ESR≥50 → 4xOEPA + PET + RT** |
| **BL vs DLBCL — Ki-67** | **BL: Ki-67 ~100%**; **DLBCL: Ki-67 Variable (40-90%)** |
| **ALCL ALK+ vs ALK-** | **ALK+: t(2;5) NPM-ALK, Excellent Prognosis, ALCL99 VPA**; **ALK-: DUSP22/TP63, Poorer, Add Brentuximab** |
| **PMBCL vs DLBCL** | **PMBCL: Mediastinal, CD30+, MAL+, PD-L1+, Young Women**; **DLBCL: Nodal, Various Sites** |
| **T-LBL vs T-ALL** | **T-LBL: Lymphadenopathy/Mediastinal Mass, <25% BM Blasts**; **T-ALL: >25% BM Blasts**; **Treatment Identical (ALL-Type)** |

**Mnemonic: PAED-LYMPHOMA**
- **P**aediatric: **HL 40%, NHL 60%**, **Bimodal HL (Adolescent), Peak NHL (5-10yr)**
- **A**nn Arbor: **I, II (Same Diaphragm), III (Both), IV (Extralymphatic)**
- **E**arly HL: **OEPA 2x (Fav) vs 4x (Unfav)** → **PET → RT if 4-5**
- **D**LBCL: **R-CHOP/R-EPOCH**, **PMBCL: R-CHOP + RT**
- **L**ymphoma NHL: **BL (t(8;14) MYC, BFM CYVE), ALCL ALK+ (t(2;5) NPM-ALK, VPA), DLBCL, T-LBL (ALL-Type)**
- **Y**oung NLPHL: **LP Cells (CD20+, CD15-, CD30-)**, **Excellent, Excision/Obs**
- **P**ET-Adapted: **Deauville 1-3 No RT, 4-5 RT** (19.8/30Gy)
- **M**YC Translocation: **BL t(8;14) MYC-IGH**, **Ki-67 100%**, **TLS Risk**
- **A**LK+ ALCL: **t(2;5) NPM-ALK**, **VPA (Vinblastine/Pred/Doxo)**, **Excellent**
- **O**EPA: **Vincristine, Etoposide, Pred, Doxo**; **OECO: Add Cyclophosphamide**
- **N**LPHL: **LP Cells (Popcorn), CD20+, CD15-, CD30-**
- **O**missions RT: **Deauville 1-3**, **Reduces Late Toxicity (Cancers, Heart, Thyroid)**

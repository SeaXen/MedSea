> [!tip] **FCPS/MRCP Priority: MEDIUM**
> **Vulval Cancer**: Rare (4% gynae cancers); **SCC 90%**; **Two pathways**: **HPV+ (40%, young, basaloid/warty, better)** vs **HPV- (60%, old, lichen sclerosus, differentiated, worse)**; **FIGO 2018**; **Surgery: Wide Local Excision + SLN (GROINSS-V II)** for T1-2 <4cm; **Inguinal LND** if SLN+ or large; **Adjuvant RT** for margins <8mm, nodes+, ENE; **Vaginal Cancer**: Rare (1-2% gynae); **SCC 85%**; **Primary RT (EBRT + Brachy)** standard; **Surgery** for selected Stage I/recurrent; **HPV Vaccination prevents both**.

## 1. 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Distinguish **HPV+ vs HPV- vulval cancer** pathways (age, histology, prognosis)
- [ ] Apply **FIGO 2018 Staging** for vulval and vaginal cancer
- [ ] Select **SLN mapping (GROINSS-V II)** vs **Inguinal LND** for vulval cancer
- [ ] Indicate **adjuvant RT** for vulval cancer (margins, nodes, ENE)
- [ ] Manage **vaginal cancer** with **primary RT (EBRT + Brachy)**
- [ ] Recognise **Lichen Sclerosus** as precursor for differentiated VIN/vulval cancer
- [ ] Know **HPV Vaccination** prevents vulval/vaginal HPV+ cancers

## 2. 2. Vulval Cancer

### 1. Epidemiology & Aetiology

| Feature | HPV+ Pathway (40%) | HPV- Pathway (60%) |
|---------|-------------------|-------------------|
| **Age** | **Younger** (35-55) | **Older** (65-85) |
| **Histology** | Basaloid, Warty, Usual SCC | **Differentiated (VIN-d)**, Keratinising SCC |
| **Precursor** | **HSIL (VIN 2/3)** | **dVIN (Differentiated VIN)** → **Lichen Sclerosus** |
| **HPV** | **16/18** (p16+) | **Negative** (p16-) |
| **Prognosis** | **Better** | **Worse** (local recurrence, nodal mets) |
| **Associated** | CIN, Cervical cancer | **Lichen Sclerosus (LS)**, Autoimmune |

### 2. FIGO 2018 Staging (Vulval)

| Stage | Description |
|-------|-------------|
| **I** | Confined to vulva/perineum |
| IA | **≤2cm**, **Stromal invasion ≤1mm**, No nodes |
| IB | **>2cm** OR **Stromal invasion >1mm**, No nodes |
| **II** | Any size, **Extension to adjacent** (lower urethra/vagina/anus), No nodes |
| **III** | **Node positive** |
| IIIA | 1-2 nodes **<5mm** OR 1 node **≥5mm** |
| IIIB | ≥3 nodes **<5mm** OR ≥2 nodes **≥5mm** |
| IIIC | **ENE (Extranodal Extension)** OR **Fixed/matted nodes** |
| **IV** | Distant mets / Bladder/Rectal mucosa / Pelvic bone / Fixed nodes |

## 3. 3. Vaginal Cancer

### 1. Epidemiology

| Feature | Detail |
|---------|--------|
| **Incidence** | **1-2% of gynaecological cancers**; ~250/year UK |
| **Histology** | **SCC 85%**, Adenocarcinoma (DES-associated clear cell), Melanoma, Sarcoma |
| **Peak Age** | 60-70 years |
| **Risk Factors** | **HPV 16/18 (60-70%)**, **Prior cervical cancer/CIN**, **DES exposure (Clear cell)**, **Smoking**, **Immunosuppression** |
| **Location** | **Upper third (60%), Posterior wall** |

### 2. FIGO 2018 Staging (Vaginal)

| Stage | Description |
|-------|-------------|
| **I** | Confined to vaginal wall |
| **II** | Paravaginal tissue invasion, **No** pelvic wall |
| **III** | Pelvic wall invasion **OR** Nodal mets |
| IIIA | Pelvic wall |
| IIIB | Hydronephrosis/Non-functioning kidney |
| IIIC1 | Pelvic LN+ |
| IIIC2 | Para-aortic LN+ |
| **IV** | Beyond true pelvis / Bladder/Rectal mucosa / Distant mets |

## 4. 4. Diagnosis & Investigations

| Investigation | Vulval | Vaginal |
|---------------|--------|---------|
| **Clinical Exam** | **Vulvoscopy + Biopsy** | **Colposcopy + Biopsy** (speculum) |
| **MRI Pelvis** | **Local staging** (depth, adjacent organs) | **Primary staging** |
| **CT/PET-CT** | Nodal/Distant (Stage IB+) | Nodal/Distant |
| **SLN Mapping** | **ICG/Technetium** for inguinal nodes | Not standard |
| **Cystoscopy/Proctoscopy** | If advanced (Stage IVA) | If advanced |
| **HPV/p16 Testing** | Prognostic (HPV+ better) | Prognostic |

## 5. 5. Management

### 1. Vulval Cancer

```mermaid
flowchart TD
    A[Vulval Cancer] --> B{Stage}
    B -->|**Stage IA (≤2cm, ≤1mm invasion)**| C[**Wide Local Excision (1cm margin)**
**No inguinal LND / SLN** (risk <1%)
**Surveillance**]
    B -->|**Stage IB (>2cm or >1mm invasion)**| D[**Wide Local Excision + SLN Mapping**
**GROINSS-V II**: SLN (ICG + Technetium)
**If SLN- → No LND**; **If SLN+ → Inguinal LND**
**If SLN not feasible → Inguinal LND**]
    B -->|**Stage II (Adjacent organ)**| E[**Wide Local Excision + Inguinal LND** (bilateral)
**Neoadj RT/ChemoRT** considered for bulky]
    B -->|**Stage III (Node+)**| F[**Inguinal LND + Pelvic LND** (if inguinal+)
**Adjuvant RT to Groin/Pelvis**
**ChemoRT (Cisplatin) if ENE/multiple nodes**]
    D --> G[**Adjuvant RT Indications**]
    E --> G
    F --> G
```

### 3. GROINSS-V II Trial (SLN for Vulval)

| Feature | Detail |
|---------|--------|
| **Population** | T1-2, <4cm, clinically node-negative |
| **SLN Technique** | **Technetium-99m + Patent Blue / ICG** |
| **Result** | **SLN negative → No LND** (Groin recurrence 2.3% vs 0.9% LND); **Less morbidity** (lymphedema, wound infection) |
| **SLN Positive** | **Completion Inguinal LND** → Adjuvant RT |

## 6. 6. Adjuvant RT for Vulval Cancer

| Indication | RT Field | Dose |
|------------|----------|------|
| **Positive Margins (<8mm)** | Primary site (vulva) | 50-54Gy/25-30fx |
| **Node+ (1-2 micro/macro)** | Ipsilateral Inguinal + Pelvic | 50-54Gy |
| **ENE / ≥3 Nodes / Fixed** | Bilateral Inguinal + Pelvic | 50-54Gy + Boost to gross nodes |
| **ChemoRT** | **Cisplatin 40mg/m² weekly** concurrent | If ENE / bulky nodes |

## 9. 9. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Vulval vs Vaginal Staging** | **Both FIGO 2018**; **Vulval: IA ≤2cm ≤1mm**; **Vaginal: I = Vaginal wall only** |
| **SLN vs LND for Vulval** | **SLN (GROINSS-V II) for T1-2 <4cm cN0**; **LND for SLN+, large, cN+, or SLN failed** |
| **Vaginal Primary RT vs Surgery** | **RT Standard** (preserves bladder/rectal/sexual function); **Surgery for selected early** |
| **Radiation Fields** | **Vulval**: Primary site if margins <8mm; **Inguinal/Pelvic** if nodes+; **Vaginal**: **Pelvis + Brachy** |
| **dVIN vs HSIL** | **dVIN (HPV-, Lichen Sclerosus background)** → Differentiated SCC; **HSIL (HPV+, VIN 2/3)** → Usual/Basaloid SCC |
| **HPV+ vs HPV- Prognosis** | **HPV+ Better** (young, immune-responsive); **HPV- Worse** (old, local recurrence, nodal mets) |

**Mnemonic: VULVAL-VAGINAL**
- **V**ulval: **SCC 90%**, **HPV+ 40% (Young, Better)** vs **HPV- 60% (Old, LS, Worse)**
- **U**pper Vagina: Most common site (60%)
- **L**ichen Sclerosus → **dVIN** → **Differentiated SCC** (HPV-)
- **V**IN: **HSIL (HPV+)** vs **dVIN (HPV-)**
- **A**djuvant RT: **Margins<8mm, Node+, ENE** → Groin±Pelvis
- **L**ocal Excision: **Wide 1cm margin** + **SLN (GROINSS-V II)**
- **V**aginal: **RT Standard** (EBRT + Brachy); Surgery selected
- **A**natomy: **Vulva** (external) vs **Vagina** (canal)
- **G**RIONSS-V II: **SLN for T1-2 <4cm**
- **I**nguinal LND: **SLN+ / Failed SLN / cN+**
- **N**odal ENE: **ChemoRT (Cisplatin)**
- **A**ll: **HPV Vaccine Prevents** (Vulval 40%, Vaginal 60-70%)
- **L**S Surveillance: **Lifelong** for Lichen Sclerosus

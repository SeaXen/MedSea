> [!tip] **FCPS/MRCP Priority: HIGH**
> **Hypercalcaemia = Most Common Metabolic Emergency in Cancer**; **PTHrP (Humoral) 80%** (SCC Lung/H&N/Cervix, Breast, RCC), **Osteolytic Mets** (Breast, Myeloma, Prostate), **Vitamin D (Lymphoma)**; **IV 0.9% Saline → Zoledronic Acid 4mg IV** (Onset 24-48h) **OR Denosumab 120mg SC** (Renal Impairment, Onset 2-4d); **Calcitonin 4-8IU/kg** (Rapid Onset 2-4h, Tachyphylaxis 48h); **Treat Underlying Cancer**.

---

## 1. 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Identify **causes**: PTHrP (Humoral), Osteolytic Metastases, Vitamin D (Lymphoma)
- [ ] Recognise **clinical features**: Polyuria, Nausea, Confusion, Cardiac Arrhythmias
- [ ] Apply **management algorithm**: IV Fluids → Bisphosphonate/Denosumab → Calcitonin (Severe)
- [ ] Select **bisphosphonate vs denosumab** based on renal function
- [ ] Know **calcitonin role** for rapid correction in severe cases
- [ ] Understand **PTH suppression** confirms malignancy-related aetiology

## 2. 2. Aetiology & Pathophysiology

| Mechanism | Mediator | Associated Cancers |
|-----------|----------|-------------------|
| **Humoral (PTHrP)** | **PTHrP** (PTH-related Peptide) | **SCC** (Lung, H&N, Cervix, Oesophagus), **Breast**, **RCC**, **Ovarian**, **Bladder** |
| **Osteolytic Metastases** | **RANKL, IL-6, TNF-α, IL-1** | **Breast**, **Multiple Myeloma**, **Prostate**, **Lung**, **Thyroid** |
| **Vitamin D (1,25-OH2D)** | **1,25-(OH)2D** (Macrophage) | **Lymphoma** (HL, NHL), **Sarcoidosis** |
| **Ectopic PTH** | **PTH** (Rare) | **Parathyroid Carcinoma**, **Lung** |

---

## 3. 3. Clinical Features

| System | Features |
|--------|----------|
| **Renal** | **Polyuria/Polydipsia** (Nephrogenic DI), **Nephrolithiasis**, **Nephrocalcinosis**, **AKI** |
| **GI** | **Nausea, Vomiting, Constipation, Anorexia**, **Pancreatitis**, **Peptic Ulcer** |
| **Neurological** | **Confusion, Lethargy, Coma** (Severe >3.5 mmol/L), **Weakness, Fatigue** |
| **Cardiac** | **Shortened QT**, **Arrhythmias** (Bradycardia, AV Block, VF), **Hypertension** |
| **Musculoskeletal** | **Bone Pain**, **Pathological Fractures**, **Proximal Myopathy** |

---

## 4. 4. Diagnostic Criteria

| Parameter | Threshold |
|-----------|-----------|
| **Corrected Calcium** | **>2.6 mmol/L** (Clinical significance >2.8 mmol/L) |
| **Albumin Correction** | **Corrected Ca = Measured Ca + 0.02 × (40 - Albumin)** (Albumin in g/L) |
| **Ionised Calcium** | **>1.3 mmol/L** (Gold Standard) |
| **PTH** | **Suppressed (<10 pg/mL)** — **Confirms Malignancy-Related** |
| **PTHrP** | **Elevated** (If Available) |
| **Vitamin D Metabolites** | **1,25-(OH)2D Elevated** (Lymphoma) |

## 5. 5. Management Algorithm

```mermaid
flowchart TD
    A[Hypercalcaemia >2.8 mmol/L or Symptomatic] --> B[**IV 0.9% Saline 1-3L/24h** (Target Urine Output >100mL/h)]
    B --> C[**Bisphosphonate: Zoledronic Acid 4mg IV over 15min** (Onset 24-48h, Peak 2-4d, Duration 2-4wk)]
    C --> D{Severity / Response}
    D -->|**Severe (>3.5 mmol/L) / Refractory**| E[**Calcitonin 4-8IU/kg SC/IV q6-12h** (Rapid Onset 2-4h, Tachyphylaxis 48h)]
    D -->|**Refractory / Renal Impairment**| F[**Denosumab 120mg SC** (Onset 2-4d, No Renal Adjustment)]
    D -->|**Vitamin D Mediated (Lymphoma)**| G[**Corticosteroids (Prednisolone 40-60mg)**]
    E --> H[**Monitor Ca q6-12h, Renal Function, ECG**]
    F --> H
    G --> H
    H --> I[**Treat Underlying Cancer**]
```

## 6. 6. Bisphosphonate vs Denosumab Comparison

| Feature | **Zoledronic Acid** | **Denosumab** |
|---------|---------------------|---------------|
| **Mechanism** | Pyrophosphate Analogue → Osteoclast Apoptosis | Anti-RANKL mAb → Inhibits Osteoclast Formation |
| **Onset** | 24-48h | 2-4 days |
| **Duration** | 2-4 weeks | 4+ weeks |
| **Renal Impairment** | **CrCl <30: Dose Reduce/Avoid** | **No Adjustment** (Preferred in CKD) |
| **Osteonecrosis Jaw** | Risk (Dental Exam Pre-Tx) | Similar Risk |
| **Hypocalcaemia** | Monitor Ca, Vit D, Phosphate | **Higher Risk** (Supplement Ca/Vit D Mandatory) |
| **Contraindication** | Severe Renal Impairment | Hypocalcaemia (Must Correct First) |

---

## 7. 7. Calcitonin

| Feature | Detail |
|---------|--------|
| **Mechanism** | Direct Osteoclast Inhibition, Renal Calcium Excretion ↑ |
| **Onset** | **2-4 hours** (Rapid) |
| **Duration** | **Tachyphylaxis by 48h** (Downregulation of Receptors) |
| **Dose** | **4-8 IU/kg SC/IV q6-12h** |
| **Role** | **Bridge Therapy** in Severe/Refractory while Bisphosphonate/Denosumab Takes Effect |
| **Side Effects** | Nausea, Flushing, Hypersensitivity |

---

## 8. 8. Corticosteroids

| Indication | Mechanism | Dose |
|------------|-----------|------|
| **Vitamin D-Mediated (Lymphoma)** | Inhibits 1α-Hydroxylase → ↓1,25-(OH)2D | **Prednisolone 40-60mg OD** |
| **Haematological Malignancies** | Direct Anti-lymphoma Effect | **Dexamethasone 16-20mg OD** |
| **PTHrP-Mediated (Some)** | Reduces PTHrP Production | Variable Response |

## 9. 9. Monitoring & Supportive Care

| Parameter | Frequency |
|-----------|-----------|
| **Corrected/Ionised Calcium** | **q6-12h** until Normal, then Daily |
| **Renal Function (U&E, Creatinine)** | **Daily** |
| **ECG** | **Continuous** (QT Interval, Arrhythmias) |
| **Fluid Balance** | **Strict I/O**, Target Urine >100mL/h |
| **Phosphate, Magnesium** | **Daily** (Hypophosphataemia Common Post-Treatment) |

## 12. 12. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Corrected Calcium Formula** | **Corrected Ca = Measured Ca + 0.02 × (40 - Albumin)** (Albumin in g/L) |
| **Zoledronic vs Denosumab in Renal Failure** | **Denosumab Preferred (No Renal Adjustment)**; **Zoledronic: CrCl <30 Avoid/Reduce** |
| **Calcitonin Tachyphylaxis** | **Receptors Downregulate by 48h** — **Use Only as Bridge** |
| **Hypercalcaemia Phosphate** | **PTHrP → Phosphate ↓** (Unlike TLS where Phosphate ↑) |
| **Corticosteroids in Lymphoma** | **Vitamin D-Mediated** → **Inhibits 1α-Hydroxylase** → **↓1,25-(OH)2D** |
| **Corrected vs Ionised Calcium** | **Ionised = Gold Standard** (Protein-Independent); **Corrected = Surrogate** (If Ionised Unavailable) |

**Mnemonic: HYPERCALCAEMIA**
- **H**ypercalcaemia: **PTHrP (80%)**, Osteolytic, Vit D
- **Y** = **y = PTH Suppressed** (Malignancy Confirmed)
- **P**THrP: **Humoral (SCC, Breast, RCC)**
- **E**xtravasation: **Osteolytic Mets** (Breast, Myeloma)
- **R**enal: **Polyuria, Stones, AKI**
- **C**alcitonin: **Rapid Bridge (2-4h)**
- **A**lcohol/Albumin: **Correct Ca Formula**
- **L**ymphoma: **Vit D (Corticosteroids)**
- **A**cidification: **Saline Hydration 1st**
- **E**GFR: **Denosumab if Renal Impairment**
- **M**alignancy Treatment: **Definitive**
- **I**onised Calcium: **Gold Standard**
- **A**symptomatic >2.8 Treat

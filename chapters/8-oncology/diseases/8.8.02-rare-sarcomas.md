## 1. 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Identify **key molecular drivers** for each rare sarcoma subtype
- [ ] Select **appropriate systemic therapy** based on molecular profile
- [ ] Apply **surgical principles**: R0 resection, organ preservation
- [ ] Use **targeted therapies**: Imatinib (DFSP), Pazopanib (SFT), Tazemetostat (Epithelioid), Pazopanib/Bevacizumab (Angiosarcoma), Pomaldomide (KS)
- [ ] Know **adjuvant/neoadjuvant** strategies per subtype
- [ ] Apply **metastatic sequencing** per sarcoma type

## 2. 2. Synovial Sarcoma

### 1. Molecular & Clinical Features
| Feature | Detail |
|---------|--------|
| **Translocation** | **t(X;18)(p11;q11): SS18-SSX1/2/4** (Pathognomonic) |
| **Age** | **Young Adults** (15-40) |
| **Site** | **Extremities** (Peri-articular), Trunk, Head & Neck |
| **Histology** | **Biphasic** (Epithelial + Spindle) or **Monophasic** (Spindle) |
| **IHC** | **TLE1+**, **EMA+, CK+, CD99+**, CD34-, S100- |
| **Molecular** | **SS18-SSX1/2/4** (SSX1 = Worse Prognosis) |

### 2. Treatment
| Setting | Regimen |
|---------|---------|
| **Localised** | **Surgery (R0) ± RT** (Pre-op 50Gy/25fx or Post-op 60-66Gy) |
| **Adjuvant Chemo** | **Ifosfamide-based** (VAC/IE or AI) — **Ifosfamide High Activity** |
| **Metastatic** | **Ifosfamide + Doxorubicin (AI)** → **Pazopanib 2L** / **Trabectedin** / **Immunotherapy** (Trials) |
| **Targeted** | **NY-ESO-1 TCR Therapy** (Clinical Trials), **Cediranib** (VEGFR) |

## 3. 3. Alveolar Soft Part Sarcoma (ASPS)

| Feature | Detail |
|---------|--------|
| **Translocation** | **t(X;17)(p11;q25): ASPSCR1-TFE3** |
| **Age** | **Adolescents/Young Adults** (15-40) |
| **Site** | **Lower Extremity** (Thigh, Buttock) |
| **IHC** | **TFE3+ (Nuclear)**, **CD68+**, **S100-**, **Desmin-** |
| **Behaviour** | **Indolent but Relentless**, **Late Lung Mets** (Years-Decades) |
| **Treatment** | **Surgery R0** (If Resectable) ± **Adjuvant RT** |
| **Metastatic** | **Sunitinib** (VEGFR) — **Best Response**, **Axelitinib**, **Pazopanib**, **Cediranib**; **Immunotherapy** (Pembro/Nivo - Trials) |

> **Key**: **ASPSCR1-TFE3** → **TFE3 Nuclear IHC** = Diagnostic

## 4. 4. Clear Cell Sarcoma (CCS) / Melanoma of Soft Parts

| Feature | Detail |
|---------|--------|
| **Translocation** | **t(12;22)(q13;q12): EWSR1-ATF1** |
| **Age** | **Young Adults** (20-40) |
| **Site** | **Tendons/Aponeuroses** (Lower Extremity, Foot/Ankle) |
| **IHC** | **S100+, HMB45+, Melan-A+, MITF+** (Melanocytic) |
| **Molecular** | **EWSR1-ATF1 Fusion** |
| **Behaviour** | **Aggressive**, **Lung/Lymph Node Mets** |
| **Treatment** | **Surgery R0 ± RT** (Adjuvant) |
| **Metastatic** | **Immunotherapy (Pembro/Nivo)** — **High Response**, **Clinical Trials** (Targeted) |

> **Distinction from Melanoma**: **EWSR1-ATF1 Fusion**, **Deep Soft Tissue**, **Younger Age**

| **Age** | **Adolescents/Young Adults** (Male Predominance) |

## 7. 7. Epithelioid Sarcoma

| Feature | Detail |
|---------|--------|
| **Molecular** | **SMARCB1/INI1 Loss** (>90%) |
| **Age** | **Young Adults** (20-40) |
| **Site** | **Distal Extremities** (Hand, Forearm, Foot) |
| **IHC** | **INI1 (SMARCB1) Loss**, **Cytokeratin+, EMA+, CD34+, Vimentin+**, **CD34+** |
| **Molecular** | **SMARCB1/INI1 Loss (Deletion/Mutation)** |
| **Behaviour** | **Aggressive**, **Lymph Node Mets Common (20-40%)**, **Lung Mets** |
| **Surgery** | **Wide Excision ± RT** (Adjuvant) |
| **Metastatic** | **Tazemetostat (EZH2 Inhibitor)** — **FDA Approved** (ORR ~15%), **Chemo (Doxorubicin/Ifosfamide)**, **Immunotherapy (Trials)** |

> **Key**: **INI1 Loss** = Pathognomonic, **Distinguish from Epithelioid MPNST, Carcinoma**

## 8. 8. Angiosarcoma

| Feature | Detail |
|---------|--------|
| **Subtypes** | **Primary (Cutaneous/Soft Tissue)**, **Secondary (Radiation-Associated, Post-Mastectomy, Chronic Lymphoedema)** |
| **Molecular** | **MYC Amplification** (Radiation-Associated, >90%), **KDR/VEGFR2 Mut**, **PTPRB, PLCG1** |
| **IHC** | **CD31+, ERG+, FLI1+, CD34+**, **Factor VIII+** |
| **Behaviour** | **Aggressive**, **Multifocal**, **Early Haematogenous Spread (Liver, Lung)** |
| **Primary Treatment** | **Surgery R0 ± RT** (Adjuvant) |
| **Metastatic** | **Paclitaxel (Weekly) + Bevacizumab** — **Best Response**, **Doxorubicin-based**, **Pembrolizumab** (Immunotherapy), **Sorafenib/Pazopanib** (VEGFR TKI) |

> **Radiation-Associated**: **Latency 10-20yr**, **MYC Amp**, **Poor Prognosis**

## 9. 9. Kaposi Sarcoma (KS)

| Feature | Detail |
|---------|--------|
| **Aetiology** | **HHV-8 (KSHV)** |
| **Clinical Forms** | **Classic (Mediterranean Elderly)**, **Endemic (African)**, **Iatrogenic (Post-Transplant)**, **Epidemic (HIV/AIDS)** |
| **IHC** | **HHV-8 LANA+**, **CD31+, ERG+, D2-40+** |
| **Staging (AIDS Clinical Trials Group)** | **T0 (Localised Cutaneous)**, **T1 (Extensive/Visceral)**, **I0 (CD4≥200)**, **I1 (CD4<200)** |
| **Treatment** | **ART (HIV) → Regression**, **Local RT (Localised)**, **Paclitaxel (Systemic)**, **Liposomal Doxorubicin**, **Pomalidomide (Relapsed)** |

## 10. 10. Dermatofibrosarcoma Protuberans (DFSP)

| Feature | Detail |
|---------|--------|
| **Translocation** | **t(17;22)(q22;q13): COL1A1-PDGFB** |
| **Age** | **Young Adults** (20-50) |
| **Site** | **Trunk, Proximal Extremities** |
| **IHC** | **CD34+ (Diffuse Strong), CD117-, S100-, Factor XIIIa+** |
| **Molecular** | **COL1A1-PDGFB Fusion** → **PDGFRB Autocrine Activation** |
| **Behaviour** | **Locally Aggressive**, **Rare Mets (<5%)**, **High Local Recurrence** |
| **Surgery** | **Wide Excision (2-3cm Margins) / Mohs Micrographic Surgery** |
| **Adjuvant** | **RT (Close/Positive Margins)** |
| **Metastatic/Unresectable** | **Imatinib 400mg Daily** (PDGFRB Inhibition) — **High Response Rate** |

> **Key**: **CD34+ Diffuse**, **COL1A1-PDGFB**, **Imatinib Responsive**

## 11. 11. Solitary Fibrous Tumour (SFT)

| Feature | Detail |
|---------|--------|
| **Translocation** | **NAB2-STAT6 Fusion (Inversion 12q13)** |
| **Age** | **Adults** (40-60) |
| **Site** | **Pleura (50%), Soft Tissue (Meninges, Paranasal, Orbit, Deep)** |
| **IHC** | **STAT6 (Nuclear)+**, **CD34+, BCL2+**, **CD117-, S100-, Desmin-** |
| **Molecular** | **NAB2-STAT6 Fusion (Inversion 12q13)** |
| **Grading** | **WHO Grade 1-3** (Mitotic Count, Necrosis, Cellularity) |
| **Risk Stratification** | **Demetri Criteria**: Size >10cm, Mitoses >4/10 HPF, Necrosis, Age >55 |
| **Surgery** | **R0 Resection** (En Bloc) |
| **Adjuvant RT** | **Close/Positive Margins, High Grade, Size >10cm** |
| **Metastatic** | **Pazopanib** (Preferred), **Bevacizumab + Chemo**, **Sunitinib**, **Temozolomide/Bevacizumab** |

## 15. 15. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **ASPS vs Clear Cell Sarcoma** | **ASPS: ASPSCR1-TFE3, TFE3 IHC+, Young, Thigh**; **CCS: EWSR1-ATF1, S100/HMB45/Melan-A+, Tendon/Aponeurosis** |
| **DSRCT vs Ewing** | **DSRCT: EWSR1-WT1, Peritoneal, WT1+, Desmin Dot+**; **Ewing: EWSR1-FLI1/ERG, Bone/Soft Tissue, CD99+, FLI1+** |
| **Epithelioid Sarcoma vs Carcinoma** | **Epithelioid Sarcoma: INI1 Loss, CK+/Vim+, CD34+**; **Carcinoma: INI1 Retained, CK+, Vimentin-** |
| **Angiosarcoma Primary vs Radiation-Associated** | **Primary: Cutaneous/Soft Tissue, MYC WT**; **Rad-Assoc: Post-RT (10-20yr), MYC Amp, Poorer Prognosis** |
| **DFSP vs SFT** | **DFSP: COL1A1-PDGFB, CD34+ Diffuse, Imatinib Responsive**; **SFT: NAB2-STAT6, STAT6 Nuclear+, Pazopanib/Bev** |
| **SFT Benign vs Malignant** | **Grading: Size >10cm, Mitoses >4/10 HPF, Necrosis, Cellularity** → **Malignant** |
| **Synovial Sarcoma Subtypes** | **SS18-SSX1 (Worse)** vs **SS18-SSX2/4 (Better)** |

**Mnemonic: RARE-SARCOMA**
- **R**are: **Synovial, ASPS, Clear Cell, DSRCT, EMC, Epithelioid, Angio, Kaposi, DFSP, SFT**
- **A**PSPS: **ASPSCR1-TFE3**, **Sunitinib**
- **R**habdo? No, **Clear Cell: EWSR1-ATF1**, **Pembro/Nivo**
- **E**wing-like: **DSRCT (EWSR1-WT1)**, **VAC/IE + CRS+HIPEC**
- **D**etailed Molecular: **SYT-SSX (Synovial), ASPSCR1-TFE3 (ASPS), EWSR1-ATF1 (CCS), WT1 (DSRCT), EWSR1-NR4A3 (EMC), INI1 Loss (Epithelioid), MYC Amp (Angio), HHV-8 (KS), COL1A1-PDGFB (DFSP), NAB2-STAT6 (SFT)**
- **S**ynovial: **t(X;18) SYT-SSX, TLE1+, Ifosfamide Active**
- **A**ngiosarcoma: **MYC Amp (Rad), Paclitaxel+Bev, Pembro**
- **R**habdomyosarcoma: **NOT IN THIS LIST** (Paediatric, Separate)
- **C**apitosarcoma: **KS = HHV-8, ART + Paclitaxel + Pomalidomide**
- **O**steosarcoma: **Separate (Bone Sarcoma)**
- **M**etastatic Sequencing: **Surgery → Targeted (Imatinib DFSP, Sunitinib ASPS, Tazemetostat Epithelioid, Pazopanib SFT) → Immunotherapy → Chemo**
- **A**djuvant: **Ifosfamide (Synovial), RT (Local Control)**

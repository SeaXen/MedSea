## 2. 2. Cervical Cancer Screening — Only Organised Programme

| Parameter | Detail |
|-----------|--------|
| **Programme** | **NHS Cervical Screening Programme (NHS CSP)** |
| **Age Range** | **25-64 years** |
| **Interval** | **25-49: 3 years**, **50-64: 5 years** |
| **Primary Test** | **HPV Primary Screening** (hrHPV PCR) |
| **Triage** | **HPV+ → Cytology (LBC)**; **Cytology+ → Colposcopy**; **HPV- → Routine Recall** |
| **HPV Vaccination** | **9-valent (6,11,16,18,31,33,45,52,58)** → **~90% prevention** |
| **Mortality Reduction** | **~70%** since programme inception |

> **Only Organised Gynaecological Cancer Screening Programme in UK**

## 3. 3. Ovarian Cancer — No Population Screening

| Aspect | Detail |
|-------|--------|
| **Population Screening** | **Not Recommended** (UKCTOCS Trial: No Mortality Benefit) |
| **UKCTOCS Trial** | **200,000 women, Multimodal (CA125 + TVUS) vs US vs Control** → **No Mortality Benefit** |
| **High-Risk Surveillance** | **BRCA1/2**, **Lynch**, **Strong FH** → **CA125 + TVUS q6-12mo** |
| **Ovarian Cancer** | **CA125 + TVUS q6-12mo** (High-Risk Only) |
| **Genetic Counselling** | **BRCA1/2, Lynch** → **Cascade Testing, RRSO** |

### 1. UKCTOCS Trial Summary
| Feature | Detail |
|--------|--------|
| **Design** | **RCT, 200,000 women**, Multimodal (CA125+TVUS) vs US vs Control |
| **Primary Outcome** | **Ovarian Cancer Mortality** |
| **Result** | **No Significant Mortality Reduction** (HR 0.93, NS) |
| **Mode Shift** | **Earlier Stage Detection** (Stage I/II ↑) but **No Mortality Benefit** |
| **Conclusion** | **No Population Screening Recommended** |

## 4. 4. Endometrial Cancer — High-Risk Surveillance

| Risk Factor | Surveillance |
|-------------|--------------|
| **Lynch Syndrome** | **Annual Endometrial Sampling / TVUS** (From 30-35yr) |
| **Obesity (BMI >30)** | **Annual Endometrial Sampling / TVUS** (From 35-40yr) |
| **Tamoxifen Use** | **Annual TVUS** (During + 2-5yr Post-Treatment) |
| **Unopposed Oestrogen** (PCOS, HRT) | **Consider Annual Surveillance** |
| **Genetic Testing** | **If <50yr, FH, Obesity** → **Lynch/PTEN Testing** |

### 1. Lynch Syndrome Gynaecological Surveillance
| Surveillance | Frequency | Start Age |
|--------------|-----------|-----------|
| **Endometrial Sampling / TVUS** | **Annual** | **30-35 years** |
| **Ovarian Surveillance** | **CA125 + TVUS q6-12mo** | **30-35 years** |
| **Gastric/SB/Urinary** | **OGD q2-3yr, Urinalysis** | **25-30 years** |

## 5. 5. Vulval & Vaginal Cancer — High-Risk Surveillance

| Condition | Surveillance |
|---------|--------------|
| **Lichen Sclerosus (LS)** | **Annual Vulval Exam**, **Biopsy if Change**, **Topical Steroids** |
| **HPV Infection** (VIN) | **Colposcopy + Biopsy**, **Vaccination Preventive** |
| **Immunosuppression** (HIV, Post-Tx) | **Annual Vulval Exam**, **Colposcopy if Abnormal** |
| **Prior Cervical/Vaginal CIN/CA** | **Annual Vulval Exam + Colposcopy** |
| **VIN (Vulval Intraepithelial Neoplasia)** | **Surveillance q6-12mo**, **Treatment if Progression** |

| **Cervical vs Ovarian Screening** | **Cervical: Organised Programme (HPV Primary)**; **Ovarian: No Programme (UKCTOCS Negative)** |
| **Endometrial vs Ovarian Surveillance** | **Endometrial: High-Risk (Lynch, Obesity, Tamoxifen) → Annual Sampling/TVUS**; **Ovarian: High-Risk (BRCA/Lynch) → CA125+TVUS** |

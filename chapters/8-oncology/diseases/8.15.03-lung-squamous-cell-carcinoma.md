> [!tip] **FCPS/MRCP Priority: HIGH**
> **Squamous NSCLC = 25-30% of NSCLC**; **Central, cavitary**; **Strong smoking association**; **IHC: p40+, p63+, TTF-1-, Napsin A-**; **No high-frequency actionable drivers** (FGFR1 amp 10-20%, PIK3CA 10%, DDR2 3-4%, PTEN loss); **Chemo: Cisplatin + Gemcitabine** (squamous preferred regimen); **Avoid Pemetrexed (ineffective) & Bevacizumab (pulmonary haemorrhage risk)**; **Metastatic 1L**: PD-L1≥50% Pembro alone; PD-L1 1-49% Pembro+Chemo; PD-L1<1% Pembro+Chemo (KEYNOTE-407); **Adjuvant IO** post-chemo for PD-L1≥1% (IMpower010/KEYNOTE-091).

## 1. 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Distinguish squamous from adenocarcinoma by IHC (p40/p63+/TTF-1-) and location (central)
- [ ] Know **no routine actionable driver mutations** (contrast with adenocarcinoma)
- [ ] Select **squamous-preferred chemotherapy**: Cisplatin + Gemcitabine; **Avoid Pem/Bev**
- [ ] Apply **PD-L1 stratified 1L metastatic therapy**: Pembro alone (≥50%), Pembro+Chemo (1-49%, <1%)
- [ ] Recognise **Necitumumab + Chemo** (SQUIRE) — historical, not current standard
- [ ] Manage **cavitary lesions** and **haemoptysis** risk
- [ ] Sequence **adjuvant IO** for PD-L1≥1% Stage II-IIIA post-chemo

## 2. 2. Definition & Epidemiology

| Feature | Detail |
|---------|--------|
| **Definition** | Malignant squamous epithelial tumour of lung; **NSCLC subtype**; **Central, often cavitary**; **Keratinisation, intercellular bridges** |
| **Incidence** | **25-30% of NSCLC**; Declining incidence (smoking reduction) |
| **Prevalence** | 5-year OS similar to adenocarcinoma by stage; **Worse if no IO access** |
| **Peak Age** | 65-75 years |
| **Sex Ratio** | **Male > Female** (3:1); **Strong smoking association** |
| **Risk Factors** | **Smoking** (primary, dose-dependent), **Radon**, **Asbestos**, **Occupational** (arsenic, chromium, nickel), **Prior RT**, **COPD**, **IPF** |

### 1. Molecular Profile

| Alteration | Frequency | Significance |
|------------|-----------|--------------|
| **TP53** | **90%** | Universal; Not directly targetable |
| **CDKN2A/p16** | 70% | Cell cycle; Homozygous deletion |
| **PTEN** | 30% | PI3K/AKT activation |
| **PIK3CA** | 10% | PI3K pathway; Investigational PI3K inhibitors |
| **FGFR1 amp** | 10-20% | FGFR inhibitors (erdafitinib, futibatinib) — trials |
| **DDR2** | 3-4% | Dasatinib trials — limited activity |

| **KEAP1/NFE2L2** | 20-30% | Oxidative stress; Associated with IO resistance |

| **PD-L1** | Variable | **TPS guides IO** |

## 4. 4. Clinical Features

| Feature | Description |
|---------|-------------|
| **Central Location** | **Hilar/perihilar** mass; **Endobronchial** growth → Obstruction, atelectasis, post-obstructive pneumonia |

| Condition | Distinguishing Features |
|-----------|-------------------------|
| **Adenocarcinoma** | Peripheral, **TTF-1+, Napsin A+**, glandular formation; Driver mutations common |

| **Small Cell Lung Cancer** | Neuroendocrine markers+, **Ki-67 high**, TTF-1+, central |

| **Lung Abscess** | Air-fluid level, fever, neutrophilia; Responds to antibiotics |

### 3. Key Regimen Differences vs Adenocarcinoma

| Component | Adenocarcinoma | **Squamous** |
|-----------|----------------|--------------|
| **Platinum Partner** | **Pemetrexed** (non-squamous) | **Gemcitabine** OR **Taxane (Paclitaxel/Nab-Paclitaxel)** |
| **Bevacizumab** | **Allowed** (non-squamous) | **CONTRAINDICATED** (↑ pulmonary haemorrhage) |
| **Pemetrexed Maintenance** | Standard | **Avoid** |
| **Necitumumab + Chemo** | Not used | **SQUIRE trial**: Necitumumab + Cis+Gem → OS HR 0.85 (modest) — **Not standard** |

### 4. Subsequent Lines

| Line | Options |
|------|---------|
| **Post IO+Chemo** | **Docetaxel ± Ramucirumab** (REVEL); **Nab-Paclitaxel**; **Gemcitabine** (if not used 1L); **Nivolumab** (CheckMate 017 — historical, pre-chemo-IO era); **Clinical Trial** |
| **IO-naive (rare now)** | **Nivolumab / Pembrolizumab / Atezolizumab** (PD-L1 guided) |
| **Brain Mets** | **SRS** (oligo); **WBRT** (multiple); **IO less CNS penetrant** |

## 11. 11. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Squamous vs Adeno chemo** | **Squamous**: Cis+Gem / Carbo+Taxane; **Adeno**: Cis+Pem / Carbo+Pem; **Pem contraindicated in squamous** |
| **Bevacizumab contraindication** | **Squamous + Cavitation + Central** = **High haemorrhage risk** → **Absolute contraindication** |
| **KEYNOTE-024 vs 189 vs 407** | **K-024**: Pembro alone PD-L1≥50% (both histologies); **K-189**: Pembro+Pem+Plat (non-squam); **K-407**: Pembro+Carbo+Taxane (squamous) |
| **Adjuvant IO PD-L1 cutoff** | **IMpower010**: PD-L1≥1% (TC≥1% or IC≥1%); **KEYNOTE-091**: PD-L1≥1% (TPS); **Both post-chemo** |
| **Necitumumab** | **Anti-EGFR mAb**; **Only squamous** (EGFR expressed); **Modest benefit, added toxicity** — Not NICE/ESMO recommended |
| **FGFR1 in squamous** | **Amplification 10-20%**; **Erdafitinib/Futibatinib trials** — **Not approved** for lung (approved in urothelial) |

**Mnemonic: SQUAMOUS**
- **S**moking related (90%+)
- **Q**uality IHC: **p40+/p63+/TTF-1-**
- **U**nctionable drivers: **NONE approved** (FGFR1, PIK3CA, DDR2 investigational)
- **A**void: **Pemetrexed, Bevacizumab**
- **M**etastatic 1L: **KEYNOTE-407** (Pembro+Carbo+Taxane all PD-L1)
- **O**simertinib/Targeted: **NOT used**
- **U**pfront adjuvant: **Cis+Gem ×4**; **IO post-chemo if PD-L1≥1%**
- **S**QUARE: Necitumumab (historical, not standard)

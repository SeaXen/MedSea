> [!tip] **FCPS/MRCP Priority: HIGH**
> **Locally Advanced Breast Cancer = Stage IIIA-IIIC (T3/T4, N2-3, Inflammatory T4d)**; **Standard: Neoadjuvant Systemic Therapy → Surgery → Adjuvant Therapy → RT**; **Neoadjuvant**: Anthracycline/Taxane backbone; **HER2+**: Add Trastuzumab ± Pertuzumab; **TNBC**: Add Carboplatin ± Pembrolizumab; **Surgery**: BCS if downstaged/resectable, else Mastectomy + ALND; **pCR (ypT0/is ypN0)** = Excellent prognosis; **Adjuvant**: HER2+ residual → T-DM1 (KATHERINE); TNBC residual → Capecitabine (CREATE-X) / Olaparib if BRCA+ (OlympiA); **Inflammatory (T4d)**: Neoadj Chemo → Mastectomy + RT (No BCS); **Locoregional Recurrence**: Multidisciplinary, surgery if resectable + RT ± Systemic.

## 1. 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Define **locally advanced breast cancer** (Stage IIIA-IIIB, Inflammatory)
- [ ] Prescribe **neoadjuvant chemotherapy** regimens by subtype
- [ ] Define **pathological complete response (pCR)** and its prognostic value
- [ ] Select **surgery post-neoadjuvant**: BCS vs Mastectomy, ALND vs SLNB
- [ ] Prescribe **adjuvant therapy post-neoadjuvant** by subtype and response
- [ ] Manage **Inflammatory Breast Cancer** (T4d) with multimodal approach
- [ ] Apply **radiotherapy indications** post-neoadjuvant surgery

## 2. 2. Definition & Staging

| Stage | TNM Criteria | Subtypes |
|-------|--------------|----------|
| **IIIA** | T0-2 N2, T3 N1-2, T4 N0-1 | Any receptor status |
| **IIIB** | T4a-c N0-2 | Chest wall/skin involvement |
| **IIIC** | Any T N3 | Ipsilateral supraclavicular/IMN |
| **Inflammatory (T4d)** | **Clinical**: Rapid onset, peau d'orange, erythema >1/3 breast, warmth, ridge | **Dermal lymphatic invasion** (Pathology) |

> **Key**: **No distant mets (M0)**. **Multimodal treatment mandatory**.

## 3. 3. Neoadjuvant Systemic Therapy

### 1. Regimens by Subtype
| Subtype | Preferred Regimen | Key Trials |
|---------|-------------------|------------|
| **HR+/HER2-** (Luminal) | **Anthracycline/Taxane** (AC → T, FEC → T, DD AC → T) | **NSABP B-18, B-27, GeparTrio** |
| **HER2+** | **TCHP** (Docetaxel + Carboplatin + Trastuzumab + Pertuzumab) OR **THP** | **KRISTINE, TRYPHAENA, NeoSphere** |
| **TNBC** | **Carboplatin + Paclitaxel ×12 → Dose-Dense AC ×4** ± **Pembrolizumab** | **KEYNOTE-522, GeparSixto, BrighTNess** |
| **HR+/HER2- (High Risk)** | **Anthracycline/Taxane ± Capecitabine** | **GeparTrio, GEICAM** |

### 2. Dose-Dense vs Standard
| Schedule | Regimen | Advantage |
|----------|---------|-----------|
| **Dose-Dense** (q2w + G-CSF) | **AC q2w ×4 → Paclitaxel q2w ×4** | **CALGB 9741**: Improved DFS/OS |
| **Standard** (q3w) | **AC q3w ×4 → Paclitaxel q3w ×4** | Less myelosuppression, standard |

## 4. 4. Pathological Complete Response (pCR)

### 1. Definition
| Term | Definition |
|------|------------|
| **pCR (ypT0/is ypN0)** | **No invasive cancer in breast AND nodes** (DCIS allowed) |
| **RCB (Residual Cancer Burden)** | **RCB-0 = pCR**; RCB-I/II/III = Residual disease |

### 2. Prognostic Value by Subtype
| Subtype | pCR Rate | Prognostic Value if pCR |
|---------|----------|------------------------|
| **TNBC** | **40-65%** | **Excellent** (DFS >90%) |
| **HER2+ (Non-pCR)** | **30-60%** | **Excellent** (DFS >90%) |
| **HER2+ (Non-pCR)** | — | **Poorer** (Needs T-DM1) |
| **HR+/HER2-** | **10-20%** | **Good but less predictive** |

## 5. 5. Surgery Post-Neoadjuvant

### 1. Breast Surgery
| Pre-NAC Stage | Post-NAC Response | Surgery |
|---------------|-------------------|---------|
| **cT1-2, cN0** | **Downstaged to ≤2cm, cN0** | **BCS + SLNB** (if feasible) |
| **cT2-3, cN0** | **Downstaged, N0** | **BCS + SLNB** (if margins clear) |
| **cT3-4, cN0** | **Significant response** | **BCS** (if margins/breast ratio ok) |
| **Any cN+** | **cN+ → ypN0** | **Mastectomy + ALND** (or TAD if ypN0) |

### 2. Axilla Post-Neoadjuvant
| Pre-NAC | Post-NAC | Surgery |
|---------|----------|---------|
| **cN0** | **ycN0** | **SLNB** (Standard) |
| **cN1** | **ypN0** | **TAD (Preferred) OR SLNB** (SENTINA, ACOSOG Z1071) |
| **cN1-2** | **ypN+** | **ALND** |

> **TAD (Targeted Axillary Dissection)**: Clip placed in positive node pre-NAC → SLNB + clip retrieval → **If clip+SLN negative → No ALND**

## 6. 6. Adjuvant Therapy Post-Neoadjuvant

### 1. HER2+ Disease
| Response | Adjuvant Therapy | Trial |
|----------|------------------|-------|
| **pCR (ypT0/is ypN0)** | **Continue Trastuzumab ± Pertuzumab to complete 1yr** | **KRISTINE, APHINITY** |
| **Residual Disease (ypT/ypN+)** | **T-DM1 (Kadcyla) ×14 cycles** | **KATHERINE: DFS HR 0.50** |
| **Residual + Pertuzumab-naive** | **Trastuzumab + Pertuzumab** | APHINITY subset |

### 2. TNBC
| Response | Adjuvant Therapy | Trial |
|----------|------------------|-------|
| **pCR** | **Observation** (No further chemo) | **KEYNOTE-522, CREATE-X** |
| **Residual Disease** | **Capecitabine 6-8 cycles** (1250mg/m² bid d1-14 q3w) | **CREATE-X: DFS HR 0.70** |
| **Residual + BRCA1/2 mutation** | **Olaparib 1yr** | **OlympiA: DFS HR 0.58** |

### 3. HR+/HER2-
| Response | Adjuvant Therapy |
|----------|------------------|
| **Any** | **Continue Endocrine Therapy** (AI/Tamoxifen ± OFS per menopausal status) |
| **High Risk (Residual, Node+)** | **Consider Chemo** (if not given neoadjuvant) |

## 7. 7. Radiotherapy Post-Neoadjuvant

| Surgery | RT Indication |
|---------|---------------|
| **BCS** | **Always** (Whole breast ± boost) |
| **Mastectomy** | **Pre-NAC T3/T4 or N+** → **Chest wall + Regional Nodes** |
| **Post-NAC ypN+** | **Chest wall + Regional Nodes (Supraclav/IMN)** |
| **Post-NAC ypN0 but pre-NAC N+** | **Consider RT to Regional Nodes** (NSABP B-18/NSABP B-27) |
| **Inflammatory** | **Mastectomy + Chest Wall + Regional Nodes RT** |

> **Key**: **RT fields based on PRE-NAC stage** (Not post-NAC pathological stage)

## 8. 8. Inflammatory Breast Cancer (IBC)

| Feature | Detail |
|---------|--------|
| **Definition** | **T4d**: Rapid onset (<6mo), peau d'orange >1/3, erythema, warmth, ridging, dermal lymphatic invasion |
| **Stage** | **Stage IIIB (T4d N0-2) or IIIC (N3)** |
| **Biology** | **Aggressive**, High grade, Often HER2+/TNBC, Young age |
| **Standard Approach** | **Neoadjuvant Chemo → Modified Radical Mastectomy → Chest Wall + Regional Nodal RT** |
| **Neoadjuvant Regimen** | **Anthracycline/Taxane** (Dose-Dense) ± **Pembrolizumab (TNBC)** / **Trastuzumab±Pertuzumab (HER2+)** |
| **Surgery** | **Modified Radical Mastectomy** (BCS contraindicated) |
| **RT** | **Chest Wall + Supraclavicular + IMN** (50-60Gy) |
| **Prognosis** | **5-yr OS ~40-50%** (Improved with multimodal) |

---

## 9. 9. Locoregional Recurrence (LRR)

| Scenario | Management |
|----------|------------|
| **Isolated Chest Wall** | **Surgery (Wide Excision) + RT** (if not previously irradiated) |
| **Isolated Nodal (Axilla/SCF/IMN)** | **Surgery (Lymphadenectomy) + RT** (if not previously irradiated) |
| **Combined Chest Wall + Nodal** | **Surgery + RT** (if not previously irradiated) |
| **Previously Irradiated** | **Surgery ± Re-RT (Re-irradiation)** / **Systemic Therapy** |
| **Distant Mets + LRR** | **Systemic Therapy** (Local therapy palliative) |
| **Systemic Therapy** | **Based on receptor status** (Endocrine, Chemo, HER2, TNBC protocols) |

## 12. 12. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **pCR vs RCB-0** | **Synonymous**; **pCR = RCB-0** (No invasive cancer breast/nodes) |
| **cN+ vs ycN+** | **cN+**: Clinical node positive pre-NAC; **ycN+**: Pathological node positive post-NAC |
| **TAD vs SLNB** | **TAD**: Clip placed pre-NAC in positive node → Retrieve clip + SLNB post-NAC; **Higher accuracy than SLNB alone** |
| **pCR in Luminal** | **Low rate (10-20%)**, **Not as prognostically discriminatory** as in TNBC/HER2+ |
| **Post-Mastectomy RT in Inflammatory** | **Mandatory: Chest wall + Supraclavicular + IMN (50-60Gy)**; **No BCS** |
| **Locoregional Recurrence Post-RT** | **Surgery ± Re-RT**; **If distant mets → Systemic therapy only** |

**Mnemonic: LADC-LOCAL-ADV**
- **L**ocally Advanced: **Stage IIIA-IIIC, T4d Inflammatory**
- **A**djuvant Neoadj: **Anthracycline/Taxane** backbone
- **D**ownstaging: **pCR = ypT0/is ypN0** (Best in TNBC/HER2+)
- **C**hemo HER2+: **TCHP (Doc+Cis+Tras+Per)** or **THP**
- **L**ymph Nodes Post-NAC: **cN0→SLNB; cN+ ypN0→TAD; cN+ ypN+→ALND**
- **O**utcome pCR: **TNBC/HER2+ Excellent**; **Luminal Low pCR rate**
- **A**djuvant Post-NAC: **HER2+ Residual→T-DM1; TNBC Residual→Capecitabine/Olaparib**
- **L**ymph Node RT: **PRE-NAC Stage Based** (N+ → Chest Wall + Regional)
- **I**nflammatory: **Neoadj Chemo → Mastectomy (No BCS) → RT Chest Wall + SCF + IMN**
- **C**locoregional Recurrence: **Surgery + RT (If not previously irradiated)**

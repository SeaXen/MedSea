> [!tip] **FCPS/MRCP Priority: HIGH**
> **Ovarian Cancer = Most lethal gynaecological cancer**; **Epithelial 90%** (High-grade serous 70%, Endometrioid 10%, Clear cell 10%, Mucinous 3%, Low-grade serous 5%); **FIGO 2014 Staging**; **HRD (Homologous Recombination Deficiency) = BRCA1/2 mut (germline/somatic 20-25%) + HRR gene mut** → **PARP Inhibitor eligibility**; **1L: Carboplatin AUC5-6 + Paclitaxel 175mg/m² q3w ×6 cycles ± Bevacizumab 15mg/kg**; **Maintenance PARPi**: Olaparib (SOLO-1 BRCA+, PAOLA-1 HRD+), Niraparib (PRIMA all-comers), Rucaparib (ATHENA); **Bev maintenance** if Bev used 1L (GOG-0218, ICON7); **Recurrence**: Platinum-sensitive (PFI>6mo) → Plat-doublet + PARPi; Platinum-resistant (PFI<6mo) → Non-plat ± Bev.

## 1. 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Classify **epithelial subtypes** and their molecular features (HGSC: TP53 96%, HRD 50%; Endometrioid: ARID1A, PIK3CA, PTEN; Clear cell: ARID1A, PIK3CA; Mucinous: KRAS)
- [ ] Apply **FIGO 2014 staging** for ovarian/fallopian tube/primary peritoneal
- [ ] Define **HRD** (BRCA1/2, HRR genes: RAD51C/D, BRIP1, PALB2, ATM, etc.) and **PARPi eligibility**
- [ ] Prescribe **1L chemo ± Bev** and select **maintenance** (PARPi vs Bev vs None)
- [ ] Distinguish **Platinum-sensitive (PFI>6mo)** vs **Platinum-resistant (PFI<6mo)** recurrence management
- [ ] Sequence **PARP inhibitors**: Olaparib, Niraparib, Rucaparib, Talazoparib — indications, toxicity (anaemia, thrombocytopenia, fatigue, GI, MDS/AML risk)
- [ ] Recognise **Germline BRCA testing** for all patients (cascade testing)

## 2. 2. Definition & Epidemiology

| Feature | Detail |
|---------|--------|
| **Definition** | Malignant tumour of ovarian surface epithelium / fallopian tube / primary peritoneum (same disease continuum) |
| **Incidence** | UK: ~7,500/year; 6th most common female cancer; **Most lethal gynaecological cancer** |
| **Prevalence** | 5-year OS: Stage I ~90%, Stage III ~40%, Stage IV ~20% |
| **Peak Age** | 60-70 years (HGSC); Younger for germ cell/sex cord-stromal |
| **Risk Factors** | **Nulliparity**, **Early menarche/Late menopause**, **Endometriosis** (Clear cell/Endometrioid), **Obesity**, **HRT**, **Family history**, **BRCA1/2** (↑40-60% lifetime), **Lynch Syndrome** |
| **Protective** | **OCP** (↓50%), **Multiparity**, **Breastfeeding**, **Tubal ligation**, **Salpingectomy** |

## 3. 3. Aetiology & Pathophysiology

```mermaid
flowchart LR
    A[Ovarian Surface Epithelium / Fallopian Tube Fimbria] --> B[Cell of Origin Debate]
    B --> C1[**HGSC: Fallopian Tube (STIC lesions)** → TP53 mut → Genomic instability]
    B --> C2[**Endometrioid/Clear Cell: Endometriosis** → ARID1A, PIK3CA, PTEN]
    B --> C3[**Mucinous: Intestinal metaplasia** → KRAS, HER2]
    B --> C4[**Low-grade Serous: MAPK pathway** → KRAS, NRAS, BRAF]
    C1 --> D[**HRD: BRCA1/2, HRR genes** → PARPi sensitivity]
    C2 --> E[**MMRd/MSI-H** → ICI potential]
    C3 --> F[**KRAS, HER2** → Targeted therapy]
```

### 1. Histological Subtypes & Molecular Profile

| Subtype | Frequency | Key Mutations | HRD/MSI | Prognosis |
|---------|-----------|---------------|---------|-----------|
| **High-Grade Serous (HGSC)** | **70%** | **TP53 (96%)**, **BRCA1/2 (20-25%)**, CCNE1 amp, RB1 loss | **HRD 50%** | Aggressive, Chemo-sensitive |
| **Endometrioid** | 10% | ARID1A, PIK3CA, PTEN, CTNNB1, **MMRd 20-30%** | MSI-H subset | Better |
| **Clear Cell** | 10% | **ARID1A (50%)**, PIK3CA, TP53 (rare), TERT | Rare | Chemo-resistant |
| **Mucinous** | 3% | **KRAS (60%)**, HER2 amp, TP53, CDKN2A | Rare | Chemo-resistant if advanced |
| **Low-Grade Serous** | 5% | **KRAS/NRAS/BRAF (MAPK)**, **No TP53**, ER+ | No | Indolent, Chemo-resistant |
| **Carcinosarcoma (MMMT)** | 2-5% | TP53, PIK3CA | Variable | Aggressive |

## 4. 4. Clinical Features

| Feature | Description |
|---------|-------------|
| **Asymptomatic** | Early stage often incidental |
| **Abdominal Distension** | **Ascites** (most common presentation) |
| **Bloating, Early Satiety** | GI symptoms |
| **Pelvic/Abdominal Pain** | Vague, chronic |
| **Urinary Frequency** | Bladder irritation |
| **Abnormal Vaginal Bleeding** | Post-menopausal |
| **Constitutional** | Weight loss, fatigue (advanced) |
| **Paraneoplastic** | **Trousseau's** (thrombosis), **Dermatomyositis** |

## 5. 5. Staging (FIGO 2014)

| Stage | Description |
|-------|-------------|
| **I** | Confined to ovaries/tubes |
| IA | One ovary/tube, intact capsule, no surface tumour, negative washings |
| IB | Both ovaries/tubes, intact capsules, no surface tumour, negative washings |
| IC | **IC1**: Intraoperative spillage; **IC2**: Capsule rupture pre-op / surface tumour; **IC3**: Malignant cells in ascites/washings |
| **II** | Pelvic extension |
| IIA | Uterus/Fallopian tubes |
| IIB | Other pelvic organs |
| **III** | **Microscopic (/Macroscopic) peritoneal mets outside pelvis** |
| IIIA1 | Retroperitoneal nodes only (IIIA1i ≤10mm, IIIA1ii >10mm) |
| IIIA2 | Microscopic peritoneal mets ± nodes |
| IIIB | Macroscopic peritoneal mets ≤2cm |
| IIIC | Macroscopic peritoneal mets >2cm |
| **IV** | Distant mets |
| IVA | Pleural effusion (positive cytology) |
| IVB | Parenchymal mets (liver, lung, spleen, nodes outside abdomen) |

## 6. 6. Diagnosis & Investigations

| Investigation | Role | Key Details |
|---------------|------|-------------|
| **CA125** | **Tumour marker** (not diagnostic alone) | ↑ in 80% epithelial; **Premenopausal: ↑ in endometriosis, menstruation, PID**; **Postmenopausal: ↑ specificity**; **RMI** (Risk of Malignancy Index) = CA125 × Menopausal score × US score |
| **HE4 + ROMA** | Adjunct to CA125 | Better specificity |
| **Transvaginal US** | First-line imaging | **Complex cystic/solid mass**, **Papillary projections**, **Ascites** |
| **CT Chest/Abdomen/Pelvis** | **Staging** | Peritoneal carcinomatosis, omental cake, nodes, liver/spleen surface mets, pleural effusion |
| **MRI Pelvis** | Local staging (uterine invasion) | Better soft tissue |
| **PET-CT** | Not routine; Recurrence, equivocal | |
| **Diagnostic Laparoscopy** | Staging, biopsy if neoadjuvant planned | |
| **Germline BRCA1/2 Testing** | **ALL patients** (NCCN/ESMO) | Cascade testing; PARPi eligibility |
| **Somatic HRD Testing** | Tumour BRCA + HRR panel + Genomic instability score (Myriad MyChoice, FoundationOne) | PARPi eligibility (HRD+) |
| **Biopsy** | Image-guided / Laparoscopic / Ascites cytology | Confirms histology; **Avoid tumour spillage if resectable** |

## 7. 7. Differential Diagnosis

| Condition | Distinguishing Features |
|-----------|-------------------------|
| **Benign Ovarian Cyst** | Simple, thin-walled, no solid components, no ascites, CA125 normal/mildly ↑ |
| **Borderline Tumour (Low Malignant Potential)** | **Serous/Mucinous**, Young, Stage I, **No stromal invasion**, Fertility-sparing surgery |
| **Fallopian Tube Cancer** | **STIC lesions**, Same treatment as ovarian |
| **Primary Peritoneal Cancer** | **Ovaries normal/minimal**, Same treatment |
| **Metastatic Cancer (Krukenberg)** | **Signet ring cells**, Stomach/Colon/Appendix/Breast primary; Bilateral |
| **Endometriosis** | Cystic (endometrioma), **Ground glass US**, CA125 ↑ (premenopausal) |
| **Pelvic Inflammatory Disease** | Acute pain, fever, ↑WBC, adnexal tenderness |

## 8. 8. Management

### 1. Primary Treatment Algorithm

```mermaid
flowchart TD
    A[Newly Diagnosed Ovarian Cancer] --> B{Stage}
    B -->|**Stage I (IA/IB Low-grade)**| C[**Fertility-sparing surgery** (if desired) + **Staging**
**No adjuvant chemo** (IA/IB G1-2)
**IC / High-grade: Adjuvant Carbo/Pac ×3-6 cycles**]
    B -->|**Stage I (IC, High-grade) / II**| D[**Primary Debulking Surgery (PDS)** → **Adjuvant Carbo/Pac ×6 cycles**
**± Bevacizumab** (if high-risk: Stage IC/II, clear cell, suboptimal debulking)]
    B -->|**Stage III/IV**| E{Resectability / Fitness}
    E -->|**Resectable to R0 / Fit**| F[**Primary Debulking Surgery (PDS)** → **Adjuvant Chemo ± Bev**]
    E -->|**Unresectable / Unfit / High tumour burden**| G[**Neoadjuvant Chemotherapy (NACT)**
**Carbo/Pac ×3 cycles** → **Interval Debulking Surgery (IDS)** → **Carbo/Pac ×3 cycles ± Bev**]
    F --> H[**Maintenance Therapy**]
    G --> H
```

### 2. 1L Chemotherapy Regimens

| Regimen | Dose | Cycles | Indication |
|---------|------|--------|------------|
| **Carboplatin + Paclitaxel** | Carbo AUC5-6 + Pac 175mg/m² q3w | 6 (3+3 if NACT) | **Standard 1L** |
| **Carbo/Pac + Bevacizumab** | + Bev 15mg/kg q3w (with chemo) → Bev maint 15mg/kg q3w ×15mo | 6 + maint | **High-risk / Stage IV / Suboptimal debulking** (GOG-0218, ICON7) |
| **Dose-dense Weekly Paclitaxel** | Carbo AUC5-6 q3w + Pac 80mg/m² weekly | 6 | **JGOG-3016/GOG-262**: PFS benefit (esp. no Bev) |

### 3. Maintenance Therapy (Post-1L Response)

```mermaid
flowchart TD
    A[Post-1L Chemo (CR/PR)] --> B{HRD / BRCA Status}
    B -->|**g/s BRCA1/2 mut**| C[**Olaparib 300mg BD** (SOLO-1) — **7-yr PFS benefit, OS trend**
**OR Niraparib 300mg OD** (PRIMA) — **All-comers, BRCA+ HR 0.27**
**OR Rucaparib 600mg BD** (ATHENA) — **BRCA+ve**]
    B -->|**HRD+ (BRCA wt, HRR mut or GIS high)**| D[**Olaparib + Bevacizumab** (PAOLA-1) — **HRD+ PFS HR 0.33**
**OR Niraparib** (PRIMA) — **HRD+ PFS HR 0.38**]
    B -->|**HRD- (HR-proficient)**| E[**Niraparib** (PRIMA) — **HR-proficient PFS HR 0.68** (modest)
**OR Bevacizumab maintenance** (if Bev used 1L: GOG-0218/ICO7)
**OR Observation**]
    C --> F[**PARPi Toxicity Management**]
    D --> F
    E --> F
```

### 4. PARP Inhibitor Comparison

| PARPi | Key Trial | Indication | Dose | Key Toxicity |
|-------|-----------|------------|------|--------------|
| **Olaparib** | SOLO-1 (gBRCA), PAOLA-1 (HRD+ + Bev) | **1L Maint: g/s BRCA+ (SOLO-1); HRD+ + Bev (PAOLA-1)** | 300mg BD | **Anaemia (20-30%), Nausea, Fatigue, ↑MDS/AML (1-2%)** |
| **Niraparib** | PRIMA (All-comers), NOVA (Recurrent) | **1L Maint: All-comers (PRIMA); Recurrent: BRCA+/HRD+** | 300mg OD (200mg if <77kg or Plt<150) | **Thrombocytopenia (30-40% Gr3-4), Anaemia, Neutropenia, HTN** |
| **Rucaparib** | ATHENA (1L), ARIEL (Recurrent) | **1L Maint: BRCA+ (ATHENA); Recurrent: BRCA+/HRD+** | 600mg BD | **Anaemia, Nausea, AST/ALT↑, Photosensitivity** |
| **Talazoparib** | TALA | Emerging | 1mg OD | Similar |

### 5. Recurrent Disease Management

| Setting | Definition | Treatment |
|---------|------------|-----------|
| **Platinum-Sensitive** | **PFI >6 months** (Platinum-Free Interval) | **Platinum Doublet Re-challenge** (Carbo/Pac, Carbo/Gem, Carbo/PLD) **+ PARPi Maintenance** (NOVA: Niraparib; ARIEL3: Rucaparib; ENGOT-OV16: Olaparib) |
| **Partially Sensitive** | PFI 6-12 months | Same as sensitive |
| **Platinum-Resistant** | **PFI <6 months** | **Non-Platinum Single Agent** ± Bevacizumab:
• **PLD (PEGylated Liposomal Doxorubicin)** — **Standard**
• **Weekly Paclitaxel**
• **Topotecan** (IV/weekly/oral)
• **Gemcitabine**
• **Trabectedin + PLD** (OVA-301) if PFI 6-12mo
• **Bevacizumab + Chemo** (AURELIA: Bev+Chemo vs Chemo alone — PFS HR 0.48) |
| **Platinum-Refractory** | Progression **on** platinum | Same as resistant; **Clinical Trial** priority |

## 10. 10. Viva Questions (MRCP PACES / FCPS)

| Question | Expected Answer |
|----------|-----------------|
| **55F, pelvic mass, ascites, CA125 1200. CT: omental cake, peritoneal deposits. FIGO Stage?** | **Stage IIIC** (Macroscopic peritoneal mets >2cm). |
| **Same patient, germline BRCA2 mut. 1L chemo + maintenance?** | **Carbo/Pac + Bev (if high-risk) ×6 cycles → Olaparib 300mg BD maintenance (SOLO-1)** — **Preferred for gBRCA+**. |
| **HRD definition — what genes?** | **BRCA1/2 (germline/somatic)** + **HRR genes: RAD51C, RAD51D, BRIP1, PALB2, ATM, CHEK2, FANC genes, etc.** + **Genomic Instability Score (LOH, TAI, LST)**. |
| **SOLO-1 vs PAOLA-1 vs PRIMA — key differences?** | **SOLO-1**: **gBRCAm**, Olaparib vs Placebo → **7-yr PFS 48% vs 21%**; **PAOLA-1**: **HRD+**, Olaparib+Bev vs Bev → **PFS HR 0.33**; **PRIMA**: **All-comers**, Niraparib vs Placebo → **BRCA+ HR 0.27, HRD+ HR 0.38, HR-proficient HR 0.68**. |
| **Platinum-Free Interval (PFI) — sensitive vs resistant?** | **Sensitive: PFI >6 months** → Re-challenge platinum + PARPi maint; **Resistant: PFI <6 months** → Non-platinum chemo ± Bev. |
| **Platinum-resistant recurrence — standard options?** | **PLD, Weekly Paclitaxel, Topotecan, Gemcitabine** ± **Bevacizumab** (AURELIA: Bev+Chemo PFS HR 0.48). |
| **Niraparib dosing — weight/plt adjustment?** | **300mg OD**; **If <77kg OR Plt<150 → 200mg OD**; **Monitor CBC q2w×1mo, monthly×11mo, then q3mo** (thrombocytopenia risk). |
| **Olaparib vs Niraparib vs Rucaparib — key toxicity diff?** | **Olaparib**: Anaemia; **Niraparib**: **Thrombocytopenia** (dose-adjusted); **Rucaparib**: AST/ALT↑, Photosensitivity, Dysgeusia. |
| **Borderline ovarian tumour — management?** | **Fertility-sparing surgery** (unilateral salpingo-oophorectomy + staging), **No adjuvant chemo** if Stage I; **Complete surgery** if childbearing complete. |
| **Germline BRCA testing — when?** | **At diagnosis for ALL ovarian cancer patients** (NCCN/ESMO) — guides PARPi, cascade testing, risk-reducing surgery. |

## 11. 11. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **HRD vs BRCA** | **BRCA+ = HRD+**; **HRD+ includes BRCA+ + HRR gene mut + GIS high**; **Test: Germline BRCA → Somatic HRD panel** |
| **PARPi Line of Therapy** | **1L Maintenance** (SOLO-1, PAOLA-1, PRIMA, ATHENA) — **Now standard**; **Recurrent Maintenance** (NOVA, ARIEL3, ENGOT-OV16) — **Historical** |
| **Bevacizumab 1L + PARPi Maint** | **PAOLA-1**: Olaparib + Bev maint for **HRD+**; **GOG-0218/ICON7**: Bev maint alone (no PARPi) — **Can combine if HRD+** |
| **Dose-dense Paclitaxel vs Standard** | **Weekly 80mg/m² + Carbo q3w** (JGOG-3016): PFS/OS benefit **esp. if no Bev**; **Not standard with Bev** (GOG-262) |
| **PFI Calculation** | **From last platinum dose to progression**; **Not from diagnosis**; **>6mo = Sensitive** |
| **Olaparib + Bev in PAOLA-1** | **Only HRD+**; **HRD- did NOT benefit** (HR 0.92); **Bev given 15mg/kg q3w ×15mo total** |
| **MDS/AML Risk with PARPi** | **~1-2%**; **Higher with prior chemo/RT**; **Monitor CBC**; **Discontinue if MDS/AML confirmed** |

**Mnemonic: OVARIAN-PARP**
- **O**varian: **HGSC 70%** (TP53, BRCA, HRD 50%)
- **V**ARIANTS: Endo (MMRd), Clear (ARID1A), Muc (KRAS), LGS (MAPK)
- **A**SCO/FIGO: I (ovary), II (pelvis), III (peritoneal), IV (distant)
- **R**ecurrence: **PFI>6mo Sensitive** (Plat+PARPi), **PFI<6mo Resistant** (Non-plat±Bev)
- **I**nitial: Carbo/Pac ± Bev → **PARPi Maint** (BRCA: Olaparib; HRD+: Olaparib+Bev/Nira; HRD-: Nira/Bev/Obs)
- **A**ll patients: **Germline BRCA + Somatic HRD**
- **N**iraparib: **Plt/Weight dose adjust** (Thrombocytopenia); **Olaparib**: Anaemia; **Rucaparib**: LFT/Photosensitivity
- **P**ARPi 1L Maint: **SOLO-1, PAOLA-1, PRIMA, ATHENA** — Now standard

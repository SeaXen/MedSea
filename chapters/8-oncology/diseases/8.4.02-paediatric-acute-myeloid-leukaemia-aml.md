> **Paediatric AML = 20% Paediatric Leukaemia**; **FAB M0-M7 / WHO 2022**; **Risk Stratification**: **Favourable**: **t(8;21) RUNX1-RUNX1T1 (CBF), inv(16) CBFB-MYH11 (CBF), CEBPA bZIP** → **Good Prognosis**; **Unfavourable**: **FLT3-ITD High Allelic Ratio**, **KMT2A-r**, **Complex Karyotype**, **-7/-7q, -5/-5q, Hypodiploidy**, **TP53**; **Intermediate**: **Rest**; **Induction: DA (Daunorubicin + Cytarabine) ×2** ± **Gemtuzumab Ozogamicin**; **Consolidation: HD-Ara-C ×3-4** ± **Gemtuzumab**; **FLT3-ITD → Midostaurin (1L) / Sorafenib/Gilteritinib (Relapse)**; **Allo-HSCT CR1 for HR/IR**; **CBF AML Excellent Prognosis**; **Down Syndrome AML (GATA1) → Reduced Intensity**; **APL Separate Protocol**.

## 2. 2. Definition & Epidemiology

| Feature | Detail |
|---------|--------|
| **Definition** | Clonal proliferation of **Myeloid Precursors** (>20% Blasts in BM/Blood) |
| **Incidence** | **~20% Paediatric Leukaemia**; ~70-80/year UK |
| **Prevalence** | **5-yr OS ~70%** (Improving); **Favourable Risk >85%**, **Unfavourable ~40-50%** |
| **Peak Age** | **Bimodal**: **Infants <1yr**, **Adolescents 15-18yr** |
| **Sex Ratio** | M:F ≈ 1:1 |
| **Risk Factors** | **Prior Chemo/RT (t-AML)**, **Down Syndrome (GATA1) → 150x Risk**, **Fanconi Anaemia, Bloom, Kostmann, Neurofibromatosis Type 1**, **Li-Fraumeni (TP53)** |

## 3. 3. Risk Stratification (COG / MRC / NOPHO / ELN Adapted)

### 1. Favourable Risk (Good Prognosis — Chemo Only, No Allo-HSCT CR1)

| Genetic Lesion | FAB/WHO | Prognosis |
|----------------|---------|-----------|
| **t(8;21)(q22;q22) RUNX1-RUNX1T1** | M2 / CBF-AML | **Excellent (>90% OS)** |
| **inv(16)(p13.1q22) CBFB-MYH11** | M4eo / CBF-AML | **Excellent (>90% OS)** |
| **CEBPA bZIP In-Frame Mutation** | M1/M2 | **Favourable (if Double Mut)** |

### 2. Unfavourable Risk (Poor Prognosis — Allo-HSCT CR1 Indicated)

| Genetic Lesion | Significance |
|----------------|--------------|
| **FLT3-ITD High Allelic Ratio (>0.5)** | **Poor**; **Midostaurin Benefit** |
| **KMT2A-r (11q23) / MLL-r** | **Poor**; **Infant Especially** |
| **Complex Karyotype (≥3 Abnormalities)** | **Very Poor** |
| **-7 / -7q, -5 / -5q** | **Poor** |
| **Hypodiploidy (<44 chr) / Near Haploidy** | **Very Poor** |
| **TP53 Mutation** | **Very Poor** (t-AML, Li-Fraumeni) |
| **KMT2A-PTD / MLL-PTD** | **Unfavourable** |
| **NUP98-NSD1** | **Unfavourable** |
| **RUNX1 Mutation (Non-CBF)** | **Unfavourable** |
| **ASXL1 Mutation** | **Unfavourable** |

### 3. Intermediate Risk (Standard — Allo-HSCT CR1 Debated/Institutional)

| Category | Examples |
|----------|----------|
| **Normal Karyotype (CN-AML)** | **NPM1mut/FLT3-WT** (Favourable-ish), **NPM1-WT/FLT3-WT** (True Intermediate) |
| **NPM1 Mutation (No FLT3-ITD)** | **Favourable** (if FLT3-WT) — **ELN Favourable** |
| **Other Non-Defining Abnormalities** | **+8, +21, +13, del(9q), etc.** |

## 5. 5. Induction Therapy (Standard = DA ×2)

```mermaid
flowchart TD
    A[Newly Diagnosed Paediatric AML] --> B[**Induction 1: DA**]
    B --> B1[**Daunorubicin 50mg/m² IV d1-3** (or Idarubicin 12mg/m² d1-3)]
    B --> B2[**Cytarabine (Ara-C) 100mg/m² IV Continuous Infusion d1-10** (or 3g/m² q12h ×4 doses for HD-Ara-C Induction)]
    B1 --> C[**Recovery + Marrow Assessment (Day 14-21)**]
    B2 --> C
    C --> D{Response}
    D -->|**M1 (<5% Blasts)**| E[**Induction 2: DA (Same)**]
    D -->|**M2 (5-25% Blasts)**| F[**Induction 2: DA + Gemtuzumab (if CD33+)** OR **FLAG-IDA**]
    D -->|**M3 (>25% Blasts / Persistent)**| G[**Alternative: FLAG-IDA / FLAMSA / Clinical Trial**]
    E --> H[**MRD Assessment (Flow/PCR) → Risk Stratification**]
    F --> H
    G --> H
```

### 1. Gemtuzumab Ozogamicin (GO) — CD33 Targeted

| Setting | Dose/Schedule |
|---------|---------------|
| **Induction** | **3mg/m² d1, 4, 7** (with DA) — **AAML 0531: Improved EFS** |
| **Consolidation** | **3mg/m² d1** (with HD-Ara-C) — **AAML 0531** |
| **Toxicity** | **VOD/SOS (Hepatic)**, **Myelosuppression**, **Infection** |

## 7. 7. Allogeneic HSCT in CR1

| Risk Group | Allo-HSCT CR1 |
|------------|---------------|
| **Favourable (CBF, CEBPA)** | **NO** — **Chemo Only** (Excellent Outcome >90% OS) |
| **Unfavourable** | **YES** — **Standard** (MFDS/MRD Matched Sibling > MUD > Haplo) |
| **Intermediate** | **Debated** — **Institutional Policy** (MRD-Guided: MRD+ → Allo-HSCT) |
| **Donor Preference** | **MRD > MUD (10/10) > Haploidentical (PT-Cy)** |
| **Conditioning** | **Busulfan-Based (BU/Cy, BU/Flu)** — **Reduced Intensity for Young/Comorbid** |

## 8. 8. MRD Monitoring (Flow / PCR)

| Timepoint | Significance |
|-----------|--------------|
| **Post-Induction 1** | **MRD+ → Higher Relapse Risk**; **Intensify/Allo-HSCT** |
| **Post-Induction 2 / Pre-Consolidation** | **MRD+ → Consider Allo-HSCT CR1** |
| **Post-Consolidation** | **MRD+ → High Relapse Risk** |
| **Methods** | **Flow (LAIP) Sensitivity 10⁻⁴**; **PCR (WT1, NPM1, CBF, KMT2A, FLT3-ITD)** |

## 9. 9. Special Subtypes

### 1. Core Binding Factor (CBF) AML — t(8;21) / inv(16)

| Feature | Detail |
|-------|--------|
| **Prognosis** | **Excellent (>90% OS)** — **Chemo Only, No Allo-HSCT CR1** |
| **Relapse Risk** | **CNS Relapse Higher (inv(16))** — **CNS Prophylaxis: IT MTX/AraC + HD-Ara-C** |
| **KIT Mutation** | **~30%** (Associated with Higher Relapse) — **Dasatinib Trials** |
| **FLT3-ITD** | **Rare in CBF** |

### 2. Down Syndrome AML (GATA1 Mutation)

| Feature | Detail |
|-------|--------|
| **Age** | **<4 years** (Peak 1-2yr) |
| **Genetics** | **GATA1 Mutation (Truncating)** — **Transient Abnormal Myelopoiesis (TAM) → AMKL** |
| **Biology** | **Megakaryoblastic (M7)**; **Normal Karyotype (No t-AML)** |
| **Treatment** | **Reduced Intensity (CCG 2891 / COG AAML 0431)** |
| **Regimen** | **Low-Dose Ara-C + Daunorubicin** (Reduced Anthracycline) |
| **Outcome** | **Excellent (>90% EFS)** — **No Allo-HSCT CR1** |
| **TAM (Transient Abnormal Myelopoiesis)** | **Spontaneous Regression (80%)** — **Observe; If Persistent/Symptomatic → Low-Dose Ara-C** |

### 3. DS-AMKL / Transient Myeloproliferative Disorder (TMD)

| Phase | Management |
|-------|------------|
| **TMD (Neonatal)** | **Observe**; **If Severe (Hepatic Fibrosis, Effusions, High WBC) → Low-Dose Ara-C 10-20mg/m² ×5-12 days** |
| **AMKL (Progression)** | **Reduced Intensity Chemo** (As Above) |

### 4. Infant AML (KMT2A-r)

| Feature | Detail |
|-------|--------|
| **KMT2A-r** | **~50% Infant AML** (AFF1, MLLT1, MLLT3, ELL) |
| **Protocol** | **Interfant / COG AAML 1031** — **Intensive (High-Dose Cytarabine, Daunorubicin, Etoposide, MTX)** |
| **Allo-HSCT** | **CR1 for KMT2A-r** (Controversial, Trial Dependent) |

## 10. 10. Relapsed/Refractory AML

| Setting | Treatment |
|---------|-----------|
| **Early Relapse (<6mo)** | **FLAG-IDA / FLAMSA / CLIA** → **Allo-HSCT** |
| **Late Relapse (>6mo)** | **Re-induction (DA/FLAG-IDA)** → **Allo-HSCT CR2** |
| **Refractory** | **FLAG-IDA / FLAMSA / CLIA / CPX-351 (Vyxeos)** → **Allo-HSCT** |
| **FLT3-ITD Relapse** | **Sorafenib / Gilteritinib / Quizartinib** + Chemo → **Allo-HSCT** |
| **Isolated CNS/Testicular** | **Local RT + Systemic Re-induction** |
| **Gemtuozumab** | **CD33+ Relapse** — **Single Agent / Combo** |
| **CPX-351 (Vyxeos)** | **Liposomal Doxorubicin:Cytarabine 5:1** — **t-AML, Secondary AML** |

## 14. 14. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **FLT3-ITD Allelic Ratio** | **High AR >0.5 = Unfavourable**; **Low AR ≤0.5 = Intermediate** (TKI Benefit Less Clear) |
| **CBF AML Allo-HSCT** | **NO Allo-HSCT CR1** (Excellent Chemo Outcome); **Relapse → Allo-HSCT** |
| **FLT3-ITD vs TKD** | **ITD = Internal Tandem Duplication (JM Domain)** → **High AR Unfavourable**; **TKD = Tyrosine Kinase Domain (D835)** → **No Prognostic Impact** |
| **Gemuzumab Timing** | **Induction 1 (d1,4,7) + Induction 2 (d1) + Consolidation (d1 with HD-Ara-C)** — **AAML 0531 Schedule** |
| **Down Syndrome AML vs t-AML** | **DS-AML: GATA1 mut, Normal Karyotype, Megakaryoblastic**; **t-AML: Prior Chemo/RT, Complex Karyotype, TP53 mut** |
| **MRD Methods** | **Flow (10⁻⁴) Faster, Broader**; **PCR (10⁻⁴-10⁻⁶) Specific (NPM1, CBF, WT1, FLT3-ITD)** |

**Mnemonic: PAED-AML**
- **P**aediatric: **20% Leukaemia**, Bimodal (Infant, Teen)
- **A**ML Risk: **Favourable (CBF t(8;21)/inv16, CEBPA)**, **Unfavourable (FLT3-ITD High AR, KMT2A-r, Complex, -7/-5, TP53)**
- **E**xcellent CBF: **t(8;21), inv(16)** → **Chemo Only, NO Allo-HSCT CR1** (>90% OS)
- **D**own Syndrome: **GATA1 mut, AMKL**, **Reduced Intensity**, **Excellent Outcome**
- **A**ML Induction: **DA ×2** (Dauno + Ara-C) ± **Gemtuzumab**
- **M**idostaurin: **FLT3-ITD 1L** (50mg BD d8-21), **Sorafenib/Gilteritinib Relapse**
- **L**ow Syndrome AML: **GATA1**, **TAM Regresses (80%)**
- **K**MT2A-r Infant: **Intensive (Interfant)**, **Allo-HSCT CR1 Considered**
- **R**efractory/Relapse: **FLAG-IDA / FLAMSA / CPX-351** → **Allo-HSCT**
- **M**RD: **Flow/PCR Post-Induction/Consol** → **Guides Allo-HSCT**
- **C**NS Prophylaxis: **IT MTX/AraC + HD-Ara-C** (esp. CBF/inv16)
- **C**onditioning: **BU/Cy > BU/Flu** (Allo-HSCT)

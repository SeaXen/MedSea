> [!tip] **FCPS/MRCP Priority: HIGH**
> **Prostate cancer = most common male cancer**; **Adenocarcinoma >95%** (acinar). **Risk Stratification**: NCCN (Very Low/Low/Int/High/Very High) / EAU (Low/Intermediate/High). **Localised**: Active Surveillance (Low), Surgery/RT (Int/High). **mHSPC**: **ADT + ARSI (Abira/Enza/Apa/Daro) OR Docetaxel** (STAMPEDE, LATITUDE, ENZAMET, ARASENS). **mCRPC**: **ARSI sequencing** (Abira→Enza or Enza→Abira), **PARPi** (Olaparib/Talazoparib - BRCA), **Radium-223** (bone-only, symptomatic), **PSMA-Lutetium-177** (VISION - PSMA+), **Cabazitaxel**. **PSMA-PET** = staging/restaging game-changer.

## 2. 2. Definition & Epidemiology

| Feature | Detail |
|---------|--------|
| **Definition** | Malignant tumour of prostate epithelium; **Acinar Adenocarcinoma >95%**; Variants: Ductal, Small cell, Neuroendocrine, Squamous |
| **Incidence** | UK: ~52,000/year; **Most common male cancer**; 2nd cause of cancer death (M) |
| **Prevalence** | 5-year OS ~95% (localised), ~30% (metastatic) |
| **Peak Age** | 65-75 years; Rare <50 |
| **Sex Ratio** | Male only |
| **Risk Factors** | **Age**, **Family history** (↑2-3x if 1st degree), **BRCA2** (↑5-8x, earlier onset), **BRCA1**, **HOXB13**, **Lynch**, **African ancestry** (↑ incidence & mortality), **Obesity** (aggressive disease), **Diet** (high dairy, processed meat) |

## 3. 3. Aetiology & Pathophysiology

```mermaid
flowchart LR
    A[Initiation] --> B[Androgen Receptor (AR) Pathway]
    B --> C[Luminal Cell Differentiation]
    C --> D[Early Events]
    D --> D1[TMPRSS2-ERG Fusion (50%)]
    D --> D2[SPINK1 Overexpression (10%)]
    D --> D3[SPOP Mutation (10%)]
    D --> D4[FOXA1 Mutation]
    C --> E[Progression to CRPC]
    E --> E1[AR Amplification/Mutation (T878A, L702H)]
    E --> E2[AR Splice Variants (AR-V7)]
    E --> E3[Glucocorticoid Receptor Activation]
    E --> E4[Neuroendocrine Differentiation (RB1/TP53 loss)]
    E --> E5[BRCA1/2, ATM, PALB2 Loss (HRD)]
```

### 1. Molecular Subtypes

| Subtype | Frequency | Key Features | Therapeutic Implications |
|---------|-----------|--------------|-------------------------|
| **TMPRSS2-ERG** | 50% | Androgen-driven fusion | Early event; Not directly targetable |
| **SPINK1** | 10% | Mutually exclusive with ERG | Aggressive |
| **SPOP** | 10% | Chromatin remodelling | Better prognosis |
| **BRCA2/ATM/PALB2 (HRD)** | 5-10% (germline+somatic) | **Homologous Recombination Deficiency** | **PARPi (Olaparib, Talazoparib)**, Platinum sensitivity |
| **MSI-H/dMMR** | 3-5% | Hypermutation | **ICI (Pembrolizumab)** |
| **Neuroendocrine (NEPC)** | 1-2% (↑ post-ARSI) | **RB1/TP53 loss**, ASCL1, NE markers | Platinum+Etoposide; Poor prognosis |

## 4. 4. Clinical Features

| Feature | Description |
|---------|-------------|
| **Localised** | Often **asymptomatic**; Detected on PSA screening; **LUTS** (frequency, nocturia, weak stream) — usually BPH |
| **Locally Advanced** | Urinary obstruction, haematuria, bladder outlet obstruction |
| **Metastatic** | **Bone pain** (axial skeleton, pelvis, femur), Spinal cord compression, Pathological fractures, Lymphadenopathy, Visceral mets (late) |
| **Paraneoplastic** | Rare; Hypertrophic pulmonary osteoarthropathy (HPOA) |

| System | Detail |

| **M Stage** | M1a: Non-regional nodes; M1b: Bone; M1c: Visceral |

| System | Categories |

## 6. 6. Diagnosis & Investigations

| Investigation | Role | Key Findings |
|---------------|------|--------------|
| **PSA** | Screening, Monitoring | **Age-specific ref ranges**; ↑ in cancer, BPH, prostatitis, UTI, post-TURP, post-RT; **Free:Total PSA ratio** (↑ specificity if <10-15%) |
| **DRE** | Clinical staging | Asymmetry, induration, nodule; cT stage |
| **MRI (mpMRI)** | **Pre-biopsy** (PI-RADS v2.1) — Targeted biopsy | **PI-RADS 3-5** → Targeted + Systematic biopsy; Avoids overdiagnosis of GG1 |
| **Transperineal Biopsy** | **Preferred over TRUS** — Lower sepsis risk, better anterior sampling | Template or MRI-targeted |
| **PSMA-PET/CT (Ga-68 / F-18)** | **Staging (High risk)**, **Restaging (BCR)**, **Lu177 eligibility** | Superior to CT/Bone scan; Detects oligometastatic disease |
| **CT Chest/Abd/Pelvis** | Conventional staging (if PSMA-PET unavailable) | Nodes >1cm, Visceral mets |
| **Bone Scan (Tc-99m)** | Conventional bone staging | Superseded by PSMA-PET |
| **Germline Testing** | **All metastatic**; High-risk localised; Family history | **BRCA1/2, ATM, PALB2, CHEK2, MMR, HOXB13** |
| **Somatic NGS** | mCRPC | HRD, MSI, AR-V7 (blood), TMB |

## 7. 7. Differential Diagnosis

| Condition | Distinguishing Features |
|-----------|-------------------------|
| **Benign Prostatic Hyperplasia (BPH)** | LUTS, symmetrically enlarged prostate, PSA ↑ (usually <10), **No nodules on DRE** |
| **Prostatitis** | Acute: Fever, dysuria, tender prostate; Chronic: Pelvic pain, PSA ↑; **Resolves with antibiotics** |
| **Neuroendocrine Prostate Cancer** | Rapid progression, **low PSA**, visceral mets, elevated NSE/Chromogranin; **RB1/TP53 loss** |
| **Small Cell Carcinoma** | Aggressive, paraneoplastic (SIADH, Lambert-Eaton), **PSA often low**, Platinum+Etoposide |
| **Metastasis to Prostate** | Rare; Bladder, RCC, Lung, Melanoma; History of primary |

## 11. 11. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Gleason vs Grade Groups** | **GG1=3+3=6**, **GG2=3+4=7**, **GG3=4+3=7**, **GG4=8 (4+4, 3+5, 5+3)**, **GG5=9-10** |
| **Abiraterone vs Enzalutamide — when which?** | **Abira**: High-volume, symptomatic (LATITUDE); **Enza**: Low-volume, asymptomatic, **no seizure hx**; **Apa**: Similar to Enza; **Daro**: No seizure risk, **no CYP3A4 interaction** (ARASENS) |
| **ARSI switch — efficacy?** | **Cross-resistance ~30-50%**; Response to 2nd ARSI lower than 1st |
| **Radium-223 + ARSI** | **CONTRAINDICATED** (ERA-223: ↑ fractures, no OS benefit); Sequential OK |
| **PSMA-PET vs Bone Scan** | **PSMA-PET superior**: Higher sensitivity/specificity; Detects nodal/visceral; **Standard for high-risk staging & BCR** |
| **Oligometastatic definition** | **≤3-5 mets** on PSMA-PET; **SBRT to all mets + ADT** (STOMP, ORIOLE) |

**Mnemonic: PROSTATE**
- **P**SA + DRE + mpMRI (PI-RADS) → Diagnosis
- **R**isk: NCCN (VLow/Low/IntF/IntU/High/VHigh) / EAU (Low/Int/High)
- **O**ligometastatic: ≤5 mets → SBRT + ADT
- **S**creening: Age 50-70 (earlier FH/Black)
- **T**reatment Localised: AS (VLow/Low) / RP / RT±ADT (Int/High)
- **A**DT + **ARSI** (Abira/Enza/Apa/Daro) or **Docetaxel** = mHSPC 1L
- **T**axane (Docetaxel → Cabazitaxel) = mCRPC chemo
- **E**NHANCE: PARPi (BRCA), Ra-223 (bone), Lu-177 (PSMA+)

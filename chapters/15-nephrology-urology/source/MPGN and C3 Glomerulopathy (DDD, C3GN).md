# MPGN & C3 Glomerulopathy (DDD, C3GN)

**Related:** [[Glomerular Diseases — Overview and Classification]], [[Secondary Glomerulonephritides — Lupus Nephritis]], [[Secondary Glomerulonephritides — ANCA-Associated Vasculitis (GPA, MPA, EGPA)]], [[Cryoglobulinaemic Glomerulonephritis]], [[Acute Kidney Injury (AKI)]], [[Nephrology and Urology MOC]]

> [!important]
> **MPGN = Mesangiocapillary GN. "Tram-track" GBM double contours on LM. Reclassified by IF: Immune-complex MPGN (IC-MPGN) vs C3 Glomerulopathy (C3G). C3G = C3 dominant (no Ig) → DDD (intramembranous dense deposits) or C3GN (subendothelial/mesangial). Driven by alternative complement pathway dysregulation (C3NeF, CFH/CFI/C3/CFB mutations).**

---

## Learning Objectives
- Classify MPGN by immunofluorescence (IC-MPGN vs C3G)
- Differentiate DDD vs C3GN on EM
- Identify underlying causes (infection, autoimmune, monoclonal gammopathy, complement mutations)
- Apply complement studies (C3, C4, C3NeF, CFH, CFI, C3, CFB genetics)
- Manage based on aetiology (immunosuppression, complement inhibition, plasma exchange)
- Apply to FCPS/MRCP vignettes

## Classification (Modern — by IF)

| Category | IF Pattern | Light Microscopy | Electron Microscopy | Key Drivers |
|----------|------------|------------------|---------------------|-------------|
| **IC-MPGN** | **IgG/IgM + C3** (granular) | MPGN pattern | Subendothelial ± mesangial deposits | **Immune complexes** (infections, autoimmune, MGUS/myeloma) |
| **C3 Glomerulopathy (C3G)** | **C3 dominant** (≥2+ > Ig) | MPGN pattern | **DDD**: Intramembranous dense deposits (osmiophilic ribbon); **C3GN**: Subendothelial/mesangial deposits | **Alternative pathway dysregulation** (C3NeF, CFH/CFI/C3/CFB mutations) |

> **Old classification** (I, II, III) **obsolete**. Use **IF-based classification** (IC-MPGN vs C3G).

## Immune-Complex MPGN (IC-MPGN)

### Aetiology
| Cause | Mechanism | Typical Features |
|-------|-----------|------------------|
| **Chronic infections** | Immune complex deposition | **HCV** (cryoglobulinaemic), HBV, Endocarditis (IE), Shunt nephritis, Malaria, Schistosomiasis |
| **Autoimmune** | IC deposition | **SLE** (full-house IF), Cryoglobulinaemia, Sjögren's, IgA vasculitis (Henoch-Schönlein) |
| **Monoclonal gammopathy** | Monoclonal IC | **MGUS**, Myeloma, Lymphoproliferative — **check serum/urine immunofixation** |
| **Drug-induced** | Hapten/IC | Interferon-α, gemcitabine, heroin |

### Clinical Presentation
- **Nephritic syndrome** (haematuria, RBC casts, hypertension, AKI) — most common
- **Nephrotic syndrome** (20–30%)
- **Mixed nephrotic-nephritic**
- **Asymptomatic proteinuria/haematuria**
- Low C3, **normal C4** (alternative pathway) — except cryoglobulinaemia (low C3 + low C4)

## C3 Glomerulopathy (C3G)

### Pathogenesis
| Factor | Role |
|--------|------|
| **C3 Nephritic Factor (C3NeF)** | Autoantibody stabilising C3 convertase (C3bBb) → **uncontrolled C3 cleavage** — 80% DDD, 50% C3GN |
| **CFH mutation/autoantibody** | Loss of C3b regulation → ↑ alternative pathway |
| **CFI mutation** | Impaired C3b inactivation |
| **C3 mutation** | Gain-of-function → resistant to inactivation |
| **CFB mutation** | Gain-of-function → ↑ C3 convertase formation |
| **Thrombomodulin (THBD) mutation** | Impaired C3b inactivation on endothelium |

> **C3NeF** = IgG autoantibody binding C3bBb → prevents decay → persistent C3 activation.

---

### Dense Deposit Disease (DDD)
| Feature | Detail |
|---------|--------|
| **Age** | Children/young adults (5–30yr) |
| **Presentation** | Nephritic (50%), Nephrotic (30%), Mixed |
| **EM** | **Intramembranous dense deposits** (highly osmiophilic, "ribbon-like" transformation of GBM) |
| **IF** | C3 dominant (linear/granular capillary wall), no significant Ig |
| **Serum C3** | **Low** (consumed), C4 normal |
| **C3NeF** | **80–90% positive** |
| **Associations** | **Drusen** (retinal, ↑ AMD risk), **Partial lipodystrophy** (acquired, face/arms) |
| **Prognosis** | **Poor** — 50% ESRD at 10yr; recurrence post-transplant >50% |

---

### C3 Glomerulonephritis (C3GN)
| Feature | Detail |
|---------|--------|
| **Age** | Broader (children → elderly) |
| **Presentation** | Variable (nephritic, nephrotic, asymptomatic) |
| **EM** | **Subendothelial &/or mesangial deposits** (NOT intramembranous dense deposits) |
| **IF** | C3 dominant (mesangial + capillary wall), no Ig |
| **Serum C3** | Low (60–70%), C4 normal |
| **C3NeF** | 40–50% positive |
| **Associations** | Monoclonal gammopathy (screen!), CFH/CFI/C3/CFB mutations |
| **Prognosis** | Variable — better than DDD but still significant ESRD risk |

---

## Complement Workup (Essential)
| Test | Interpretation |
|------|----------------|
| **C3** | Low in active disease (both IC-MPGN & C3G) |
| **C4** | Normal (alternative pathway) — **low if classical pathway** (SLE, cryoglobulinaemia) |
| **C3NeF** | Positive in ~80% DDD, 50% C3G |
| **CFH, CFI, C3, CFB levels** | Low if consumption; genetic testing if familial |
| **CFH autoantibody** | Causes acquired CFH deficiency |
| **Serum/urine immunofixation** | **Mandatory** — exclude monoclonal gammopathy (MGUS/myeloma) |
| **Genetic panel** | CFH, CFI, CFB, C3, THBD, DGKE (if familial/early onset) |

## Management

### IC-MPGN
| Aetiology | Treatment |
|-----------|-----------|
| **HCV-associated** | **DAA therapy** (cure HCV → GN remission) ± Rituximab if cryoglobulinaemic |
| **HBV** | Antivirals (entecavir/tenofovir) |
| **IE/Endocarditis** | Antibiotics (+ valve surgery if needed) |
| **SLE** | Standard lupus nephritis protocol (MMF/CyC + steroids) |
| **Monoclonal gammopathy** | **Treat underlying clone** (bortezomib/dex ± rituximab ± daratumumab) |
| **Idiopathic** | **MMF + steroids** (KDIGO: 6mo MMF + steroids); consider CNI if refractory |

### C3 Glomerulopathy
| Approach | Details |
|----------|---------|
| **Supportive** | ACEi/ARB (↓ proteinuria, ↓ BP), SGLT2i, BP control <130/80, statin |
| **Immunosuppression** | **MMF + steroids** (some benefit); CNI (tacrolimus) — evidence limited |
| **Eculizumab** (anti-C5) | **Case reports/series** — benefit in C3NeF+ with progressive CKD; not standard |
| **Iptacopan** (factor B inhibitor) | **Emerging** — APPEAR-C3G trial (phase 3) shows proteinuria reduction |
| **Pegcetacoplan** (C3 inhibitor) | **Emerging** — phase 2/3 trials ongoing |
| **Plasma exchange** | For CFH autoantibody (removes autoantibody) + immunosuppression |
| **Liver transplant** | For **CFH mutation** (CFH made in liver) — curative |

> **Transplant recurrence**: High in C3G (DDD >50%, C3GN ~30%), lower in IC-MPGN if aetiology treated.

## High-Yield FCPS/MRCP Points

> [!important]
> - **MPGN reclassified by IF**: IC-MPGN (Ig+C3) vs C3G (C3 dominant).
> - **DDD**: Intramembranous dense deposits (EM), C3NeF 80–90%, drusen, lipodystrophy, poor prognosis.
> - **C3GN**: Subendothelial/mesangial deposits, C3NeF 40–50%, screen for monoclonal gammopathy.
> - **Low C3 + normal C4** = alternative pathway (C3G, IC-MPGN non-cryo). **Low C3 + low C4** = classical pathway (SLE, cryoglobulinaemia).
> - **C3NeF** = stabilises C3bBb → uncontrolled alternative pathway.
> - **Always check serum/urine immunofixation** in C3G (exclude MGUS/myeloma).
> - **Genetic testing**: CFH, CFI, C3, CFB, THBD if familial/early onset.
> - **Eculizumab** considered in progressive C3G with C3NeF+.
> - **HCV-MPGN**: Treat with DAAs (not immunosuppression).
> - **Transplant recurrence high** in C3G (especially DDD).

## Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **MPGN types I/II/III still used** | Obsolete — use **IF classification** (IC-MPGN vs C3G) |
| **DDD = MPGN Type II** | DDD = **C3G subtype** (C3 dominant, intramembranous deposits) |
| **All MPGN = immune complex** | **C3G = no immune complexes** (C3 dominant, alternative pathway) |
| **Low C3 = always MPGN** | Also post-strep GN, SLE, cryoglobulinaemia, IE, C3G |
| **C3NeF = only in DDD** | Also in C3GN (~50%), some IC-MPGN |
| **C3GN = no monoclonal gammopathy** | **Screen mandatory** — MGUS/myeloma can mimic C3GN |
| **Eculizumab = first-line C3G** | Not standard — supportive + MMF/steroids first; eculizumab case-by-case |
| **DDD = drusen = eye only** | Drusen = retinal deposits → ↑ AMD risk (ophthalmology referral) |
| **IC-MPGN = never recur post-Tx** | Recurs if aetiology untreated (e.g., untreated HCV, MGUS) |

## Mnemonics
- **MPGN classification**: **I**C-MPGN = **I**mmune **C**omplex (Ig+C3); **C**3G = **C**3 dominant (no Ig)
- **C3G subtypes**: **D**DD = **D**ense **D**eposits (intramembranous); **C**3GN = **C**3 deposits (subendothelial/mesangial)
- **Complement pathway**: **Low C3 + Normal C4** = **Alternative** (C3G, IC-MPGN); **Low C3 + Low C4** = **Classical** (SLE, cryo)
- **C3NeF**: **C**3 **N**ephritic **F**actor = **C**onverts **C**3 **F**orever (stabilises C3bBb)
- **DDD associations**: **D**rusen (eyes), **L**ipodystrophy (face/arms) = **DDL**

## Summary
**MPGN & C3 Glomerulopathy** is a **Must Know 🔴** topic.  
**Classification**: IC-MPGN (immune complexes: Ig+C3) vs C3G (C3 dominant, no Ig).  
**DDD**: Intramembranous dense deposits, C3NeF 80–90%, drusen/lipodystrophy, poor prognosis.  
**C3GN**: Subendothelial/mesangial deposits, C3NeF 40–50%, **screen for monoclonal gammopathy**.  
**Complement**: Low C3 + Normal C4 = alternative pathway. C3NeF = stabilises C3bBb.  
**Management**: Treat cause (DAAs for HCV, clone-directed for MGUS). Supportive: ACEi/ARB, SGLT2i. C3G: MMF+steroids ± CNI; eculizumab/iptacopan emerging for progressive C3NeF+.  
**Transplant**: High recurrence in C3G (DDD > C3GN). CFH mutation → liver-kidney transplant.  
**Exam focus**: IF classification, EM distinction DDD vs C3GN, complement profile, immunofixation mandate, C3NeF, HCV treatment, transplant recurrence.

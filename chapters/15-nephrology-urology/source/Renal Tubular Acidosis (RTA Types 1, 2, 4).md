# Renal Tubular Acidosis (RTA Types 1, 2, 4)

**Related:** [[Functional Anatomy and Physiology of the Kidney and Urinary Tract]], [[Chronic Kidney Disease (CKD)]], [[Blood Tests in Renal Disease]], [[Electrolyte, Water, and Acid-Base Disorders]], [[Nephrology and Urology MOC]]

> [!important]
> **RTA = failure of renal tubular acidification → normal anion gap metabolic acidosis (NAGMA). Three main types: Type 1 (Distal) = H+ secretion defect; Type 2 (Proximal) = HCO3- reabsorption defect; Type 4 (Hypoaldosteronism) = aldosterone deficiency/resistance. Type 3 (combined) = historical/rare.**

## Overview: RTA Classification

| Type | Site | Primary Defect | Urine pH | K+ | Urine Anion Gap |
|------|------|----------------|----------|-----|-----------------|
| **Type 1 (Distal)** | α-Intercalated CD | **H+ ATPase defect** — cannot acidify urine | **>5.5** (inappropriately alkaline) | **Low** (hypokalaemia) | **Positive** |
| **Type 2 (Proximal)** | Proximal tubule | **NHE3/CA defect** — impaired HCO3- reabsorption | **<5.5** (can acidify if HCO3- low) | **Low** (hypokalaemia) | **Positive** |
| **Type 4 (Hypoaldo)** | Principal CD | **Aldosterone deficiency/resistance** — ↓ ENaC → ↓ H+/K+ secretion | **<5.5** (can acidify) | **High** (hyperkalaemia) | **Positive** |

---

## Type 1 (Distal) RTA

### Pathophysiology
- **α-Intercalated cell defect**: H+ ATPase (or H+/K+ ATPase) dysfunction → cannot secrete H+ into lumen
- **Result**: Failure to generate new HCO3- → NAGMA; **urine pH >5.5** despite systemic acidosis
- **Hypokalaemia**: K+ secretion ↑ (luminal negativity maintained by H+ secretion failure)

### Aetiology
| Category | Causes |
|----------|--------|
| **Primary/Genetic** | **ATP6V1B1**, **ATP6V0A4** (autosomal recessive, sensorineural hearing loss), **SLC4A1** (AE1, autosomal dominant, Southeast Asian ovalocytosis) |
| **Autoimmune** | **Sjögren's syndrome** (most common adult cause), SLE, RA |
| **Drug-induced** | Amphotericin B, lithium, ifosfamide, tenofovir, toluene (glue sniffing) |
| **Obstructive/Nephrocalcinosis** | Medullary sponge kidney, hypercalciuria, nephrocalcinosis |
| **Other** | Transplant rejection, sickle cell, hypergammaglobulinaemia |

### Clinical Features
| Feature | Detail |
|---------|--------|
| **Presentation** | NAGMA, **hypokalaemia** (weakness, paralysis), **nephrocalcinosis**, renal stones (CaP) |
| **Children** | Failure to thrive, rickets (vit D resistance), polyuria |
| **Hearing loss** | ATP6V1B1/ATP6V0A4 mutations |

### Diagnosis
| Test | Type 1 Finding |
|------|----------------|
| **Urine pH** | **>5.5** (cannot acidify despite acidosis) |
| **Urine Anion Gap** | **Positive** (UAG = UNa + UK - UCl > 0) — ↓ NH4+ excretion |
| **Serum K+** | **Low** (hypokalaemia) |
| **Bicarbonate loading** | ↑ Urine HCO3- excretion (distal defect) |
| **NH4+Cl load test** | Urine pH fails to drop <5.5 |

## Type 2 (Proximal) RTA

### Pathophysiology
- **Proximal tubule defect**: Impaired HCO3- reabsorption (NHE3, carbonic anhydrase) → **HCO3- wasting**
- **Threshold phenomenon**: At low serum HCO3- (<15–18), distal reabsorption compensates → urine **can acidify (pH <5.5)**
- **Fanconi syndrome** = generalised proximal dysfunction (glucose, AA, PO4, uric acid, HCO3-)

### Aetiology
| Category | Causes |
|----------|--------|
| **Isolated** | **SLC4A4** (NBCe1), **CA2** (carbonic anhydrase II — also osteopetrosis, cerebral calcification) |
| **Fanconi Syndrome** | **Cystinosis** (children), Wilson's, Lowe syndrome, Tyrosinaemia, Galactosaemia, Fructose intolerance |
| **Drugs/Toxins** | **Tenofovir**, ifosfamide, cisplatin, aminoglycosides, lead, cadmium, outdated tetracycline |
| **Autoimmune/Infiltrative** | Sjögren's, SLE, myeloma (light chains), amyloidosis |
| **Transplant** | Calcineurin inhibitor toxicity |

### Clinical Features
| Feature | Detail |
|---------|--------|
| **Presentation** | NAGMA, **hypokalaemia**, **renal HCO3- wasting** (fractional excretion HCO3- >15%) |
| **Fanconi** | Rickets/osteomalacia (PO4 wasting), glycosuria (normal glucose), aminoaciduria, hypouricaemia |
| **Children (Cystinosis)** | Failure to thrive, photophobia (corneal crystals), Fanconi syndrome |

### Diagnosis
| Test | Type 2 Finding |
|------|----------------|
| **Urine pH** | **<5.5** (if serum HCO3- low; >7 if HCO3- infused) |
| **Fractional Excretion HCO3-** | **>15%** (at normal serum HCO3-) |
| **Urine Anion Gap** | Positive |
| **Serum K+** | Low |
| **Fanconi screen** | Glucose, PO4, uric acid, amino acids in urine |

## Type 4 (Hypoaldosteronism) RTA

### Pathophysiology
- **Aldosterone deficiency or resistance** → ↓ ENaC activity in CD → ↓ lumen-negative PD → ↓ H+ & K+ secretion
- **Result**: **Hyperkalaemia** (hallmark), NAGMA, urine pH **<5.5** (can acidify if H+ gradient intact)
- **"Hyperkalaemic distal RTA"** — most common RTA in adults

### Aetiology
| Category | Causes |
|----------|--------|
| **Hypoaldosteronism** | **Hyporeninaemic hypoaldosteronism** (diabetic nephropathy, interstitial disease, NSAIDs, ACEi/ARB, heparin) |
| **Aldosterone Resistance** | **Obstructive uropathy**, sickle cell, lupus, amyloidosis, transplant, **K+-sparing diuretics** (spiro, amiloride, triamterene), trimethoprim, pentamidine |
| **Pseudohypoaldosteronism** | **Type 1** (ENaC mutation — severe, neonatal), **Type 2** (WNK1/4 mutation — hypertension, hyperkalaemia = Gordon's syndrome) |

### Clinical Features
| Feature | Detail |
|---------|--------|
| **Presentation** | **Hyperkalaemia** (often asymptomatic), mild NAGMA, usually older adults with diabetes/CKD |
| **Urine pH** | **<5.5** (distal acidification intact) |
| **Associated** | Diabetic nephropathy (most common), CKD, ACEi/ARB, NSAIDs |

### Diagnosis
| Test | Type 4 Finding |
|------|----------------|
| **Serum K+** | **High** (hyperkalaemia) |
| **Urine pH** | **<5.5** (can acidify) |
| **Urine Anion Gap** | Positive |
| **Aldosterone** | Low (if hypoaldosteronism) or high (if resistance) |
| **Renin** | Low (hyporeninaemic) or high (resistance) |
| **Acute K+ load test** | Impaired K+ excretion (TTKG <7) |

## Diagnostic Algorithm (NAGMA)
```
NAGMA (Normal AG Metabolic Acidosis)
    │
    ├─ Urine pH >5.5 → **Type 1 (Distal)**
    │     └─ Hypokalaemia → Genetic (ATP6V1B1, SLC4A1), Sjögren's, Drugs, Obstructive
    │
    ├─ Urine pH <5.5 + Hypokalaemia → **Type 2 (Proximal)**
    │     └─ Fanconi screen → Cystinosis, Myeloma, Drugs, CA2 mutation
    │
    ├─ Urine pH <5.5 + Hyperkalaemia → **Type 4 (Hypoaldo)**
    │     └─ Check renin/aldosterone, review drugs (ACEi/ARB, NSAIDs, K+-sparing), DM
    │
    └─ Urine Anion Gap = UNa + UK - UCl
          Positive (>0) → Renal cause (RTA)
          Negative (<-20) → GI loss (diarrhoea)
```

## Management Summary

| Type | Alkali Therapy | Specific Therapy |
|------|----------------|------------------|
| **Type 1** | **NaHCO3 / NaCitrate / KCl-Citrate** 1–3 mEq/kg/day — **large doses** (10–20 mEq/kg) due to urinary losses | **K+ replacement** (hypokalaemia); treat underlying (Sjögren's, stop drugs); thiazide ↑ proximal reabsorption |
| **Type 2** | **NaHCO3 / NaCitrate** 5–15 mEq/kg/day — **very large doses** (HCO3- wasting) | **Thiazide** (induces mild volume contraction → ↑ proximal HCO3- reabsorption); **K+ replacement**; treat Fanconi cause |
| **Type 4** | **NaHCO3** 1–2 mEq/kg/day (milder acidosis) | **Stop offending drugs** (ACEi/ARB, NSAIDs, K+-sparing); **fludrocortisone 0.1–0.5mg/day** (if hypoaldo); dietary K+ restriction; loop diuretic (↑ K+ excretion); patiromer/sodium zirconium cyclosilicate |

> **Na-Citrate** preferred (also provides K+ citrate for stones in Type 1). **Avoid NaHCO3 in heart failure/Na+ overload**.

## High-Yield FCPS/MRCP Points

> [!important]
> - **RTA = NAGMA + Positive Urine Anion Gap** (vs diarrhoea = negative UAG).
> - **Type 1**: Urine pH >5.5, **hypokalaemia**, nephrocalcinosis, stones. Causes: Sjögren's, genetic (ATP6V1B1/0A4/SLC4A1), amphotericin, lithium.
> - **Type 2**: Urine pH <5.5 (if HCO3- low), **hypokalaemia**, **FE HCO3- >15%**, Fanconi syndrome. Causes: Cystinosis, tenofovir, myeloma, CA2 mutation.
> - **Type 4**: Urine pH <5.5, **hyperkalaemia** (hallmark). Most common adult RTA. Causes: Diabetic nephropathy, ACEi/ARB, NSAIDs, K+-sparing diuretics, obstructive uropathy.
> - **Urine Anion Gap** = UNa + UK - UCl. Positive = renal (RTA); Negative = GI (diarrhoea).
> - **Type 1 stones = CaP** (alkaline urine); **Type 4 = no stones**.
> - **Sjögren's** = Type 1 (distal) + Type 2 (proximal) both possible.
> - **Cystinosis** = childhood Fanconi (Type 2) + corneal crystals.

## Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **All RTA = hypokalaemia** | **Type 4 = HYPERKALAEMIA** (most common adult RTA) |
| **Urine pH <5.5 = not RTA** | Type 2 & 4 **can acidify** (pH <5.5) — use UAG, K+, FE HCO3- |
| **Type 1 = only genetic** | **Sjögren's = most common adult cause**; drugs (amphotericin, lithium) common |
| **Fanconi = only cystinosis** | Also Wilson's, Lowe, myeloma, amyloidosis, drugs (tenofovir, ifosfamide) |
| **Type 4 = aldosterone deficiency only** | Also **aldosterone resistance** (obstructive uropathy, drugs, Gordon's syndrome) |
| **ACEi/ARB cause Type 1** | ACEi/ARB → **Type 4** (hyporeninaemic hypoaldosteronism) |
| **NaHCO3 dose same for all** | **Type 2 needs massive doses** (5–15 mEq/kg/day); Type 4 mild (1–2) |
| **Fludrocortisone for all** | Only **Type 4** (hypoaldosteronism) |
| **UAG negative = RTA** | UAG **positive** = RTA; **negative** = diarrhoea (GI loss) |
| **Nephrocalcinosis = all RTA** | **Type 1 only** (alkaline urine + hypercalciuria + hypocitraturia) |

## Mnemonics
- **RTA Types**: **1** = **D**istal (H+); **2** = **P**roximal (HCO3-); **4** = **A**ldosterone (HyperK) = **DPA**
- **Urine pH**: Type 1 **>5.5** (can't acidify); Types 2/4 **<5.5** (can acidify)
- **Potassium**: Types 1/2 = **LOW** (hypoK); Type 4 = **HIGH** (hyperK)
- **Urine Anion Gap**: **P**ositive = **R**enal (RTA); **N**egative = **G**I (diarrhoea) = **PRNG**
- **Type 1 stones**: **CaP** (alkaline urine) = **C**alcium **P**hosphate
- **Sjögren's RTA**: **S**S = **S**piping (Type 1 distal) + **S**ieving (Type 2 proximal)
- **Fanconi mnemonic**: **G**lucose, **A**mino acids, **P**hosphate, **U**ric acid, **H**CO3- = **GAPUH**

## Summary
**Renal Tubular Acidosis** is a **Must Know 🔴** topic.  
**Classification**: NAGMA + Positive UAG = RTA. Three types by site, pH, K+.  
**Type 1 (Distal)**: α-intercalated H+ ATPase defect → urine pH >5.5, **hypokalaemia**, nephrocalcinosis, CaP stones. Causes: Sjögren's, genetic (ATP6V1B1/0A4, SLC4A1), amphotericin, lithium. Rx: High-dose alkali, K+ replacement.  
**Type 2 (Proximal)**: NHE3/CA defect → HCO3- wasting, FE HCO3- >15%, urine pH <5.5 if HCO3- low, **hypokalaemia**. Fanconi = generalised proximal loss. Causes: Cystinosis, CA2, tenofovir, myeloma. Rx: Massive alkali, thiazide, K+ replacement.  
**Type 4 (Hypoaldosteronism)**: Aldosterone deficiency/resistance → ↓ ENaC → **hyperkalaemia** (hallmark), urine pH <5.5, mild NAGMA. Most common adult RTA. Causes: Diabetic nephropathy, ACEi/ARB, NSAIDs, K+-sparing diuretics, obstructive uropathy. Rx: Stop drugs, fludrocortisone, loop diuretic, K+ binders, mild alkali.  
**Diagnostic key**: UAG (UNa+UK-UCl) positive = renal; negative = GI. Urine pH >5.5 = Type 1; <5.5 + hypoK = Type 2; <5.5 + hyperK = Type 4.

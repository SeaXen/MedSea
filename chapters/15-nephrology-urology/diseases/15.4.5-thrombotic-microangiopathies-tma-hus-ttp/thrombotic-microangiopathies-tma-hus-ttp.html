Vascular Diseases of the Kidney — Thrombotic Microangiopathies (TMA, HUS, TTP)
1 Learning Objectives
  • Differentiate TTP, typical HUS (STEC), and atypical HUS (aHUS) by ADAMTS13, clinical features, triggers
  • Recognise MAHA (schistocytes, ↑ LDH, ↓ haptoglobin, ↓ Hb) + thrombocytopenia
  • Apply urgent management: TTP = plasma exchange + steroids + caplacizumab; HUS = supportive; aHUS = complement inhibitors
  • Understand complement pathway and genetic testing in aHUS

5 Clinical Presentation
TTP
  • Acute/subacute onset
  • Neurological: confusion, headache, stroke, seizure, visual disturbances, coma
  • Renal: mild-moderate AKI (Cr usually <250–300)
  • Fever, petechiae/purpura
  • No diarrhoea (prodrome)
  • Women 20–50yo predominant
Typical HUS (STEC)
  • Children (peak 6mo–5yo)
  • Prodrome: bloody diarrhoea (3–7 days) → HUS
  • Severe AKI (oliguric, often dialysis)
  • Neurological: rare/mild
  • E. coli O157:H7 (hamburger, unpasteurised milk, petting zoos)
aHUS
  • All ages; familial or sporadic
  • Severe AKI (often dialysis)
  • Hypertension common
  • Neurological: variable
  • No diarrhoea (or non-infectious)
  • Trigger: pregnancy, infection, drugs, surgery
6 Management
TTP — Medical Emergency
InterventionDetails
-----------------------
**Plasma Exchange (PLEX)****URGENT — start within 4–8 hours**; 1–1.5 plasma volumes daily until platelets >150×10⁹/L × 2 days
**Corticosteroids****High-dose** (methylprednisolone 1g × 3d → pred 1mg/kg) + PLEX
**Caplacizumab****Anti-VWF nanobody**; blocks VWF-platelet interaction; **early addition to PLEX + steroids** (HERCULES trial)
**Rituximab****Acquired TTP**: anti-CD20; reduces relapse (40–50% relapse without)
**Splenectomy**Historical; now rarely needed (rituximab)
**Folic Acid**5mg daily (haemolysis)
**Platelet Transfusion****CONTRAINDICATED** (worsens thrombosis) — only if life-threatening bleeding
**FFP Transfusion**If PLEX delayed (>4–8h); bridging
[!key] TTP = PLEX + steroids + caplacizumab URGENT. Platelet transfusion CONTRAINDICATED.
Typical HUS (STEC) — Supportive
InterventionDetails
-----------------------
**Supportive**Fluids, electrolytes, dialysis if needed
**Antibiotics****NOT recommended** (may ↑ Shiga toxin release)
**Antimotility****Avoid** (loperamide)
**Eculizumab****Not standard**; case reports in severe/neurological
**Monitor**Neurological, dialysis, haemolysis resolution
Atypical HUS (aHUS) — Complement Inhibition
InterventionDetails
-----------------------
**Eculizumab****Anti-C5 monoclonal antibody**; blocks terminal complement; **transformed outcomes**
**Dosing**: 900mg weekly × 4 → 1200mg q2wk (adults); weight-based paeds
**Ravulizumab****Long-acting anti-C5**; q8wk dosing (adults)
**Meningococcal Vaccination****Mandatory pre-eculizumab/ravulizumab** (MenACWY + MenB); antibiotic prophylaxis
**Plasma Exchange/Infusion**Bridge to eculizumab; not definitive
**Genetic Testing**CFH, CFI, MCP (CD46), C3, CFB, THBD, DGKE; guides prognosis/family screening
**Duration****Lifelong** (if complement mutation); consider trial off if no mutation + remission
[!key] aHUS = eculizumab/ravulizumab (anti-C5) + meningococcal vaccination. PLEX not definitive for aHUS (complement factors replenished).
Secondary TMA
CauseManagement
-------------------
**Malignant HTN****BP control** (IV labetalol, nicardipine); treat TMA if persistent
**Scleroderma Renal Crisis****ACEi (captopril) aggressive**; target BP; dialysis if needed
**CNI (Tac/CsA)****Stop/reduce CNI**; switch to mTOR/belatacept; PLEX if severe
**Pregnancy (HELLP/aHUS/TTP)**Multidisciplinary; delivery if term; PLEX/eculizumab per type
**Drug-induced**Stop offending drug
7 ADAMTS13 Interpretation
ADAMTS13 ActivityInterpretation
-----------------------------------
**<10%****TTP** (acquired if inhibitor+; congenital if no inhibitor, early onset)
**10–20%**Borderline; consider TTP if clinical fit
**>20% (Normal)****NOT TTP** → HUS/aHUS/secondary TMA
[!key] Send ADAMTS13 BEFORE PLEX (PLEX replenishes ADAMTS13 → false normal).

10 Common Confusions / Exam Traps
TrapCorrection
------------------
**All TMA = TTP**TTP = ADAMTS13 <10%; HUS/aHUS = normal ADAMTS13
**Platelets low = transfuse****Platelet transfusion CONTRAINDICATED in TTP** (worsens thrombosis)
**PLEX before ADAMTS13****Send ADAMTS13 FIRST** (PLEX replenishes → false normal)
**Typical HUS = antibiotics****Antibiotics NOT recommended** (↑ Shiga toxin release)
**aHUS = PLEX definitive****PLEX not definitive** (complement replenished); need eculizumab
**Eculizumab = no vaccination****Meningococcal vaccination MANDATORY** (MenACWY + MenB)
**HUS only in children**Typical HUS = children; aHUS = all ages
**TTP = always pentad**Pentad = classic but NOT required; MAHA + thrombocytopenia enough

**DIC = TMA**DIC = abnormal PT/APTT/fibrinogen/D-dimer; TMA = normal coagulation
11 Mnemonics
  • MAHA: Schistocytes, LDH↑, Haptoglobin↓, Hb↓, Retic↑ = SLHHR
  • TTP Pentad: MAHA, Thrombocytopenia, Neuro, Renal, Fever = MTNRF
  • ADAMTS13: <10% = TTP; Normal = HUS/aHUS = <10 TTP, N HUS
  • TTP Tx: PLEX + Steroids + Caplacizumab = PSC (urgent)
  • TTP Contraindicated: Platelet Transfusion = PT (worsens)
  • Typical HUS: STEC, Children, Diarrhoea (bloody), Severe AKI = SCDS
  • aHUS: Complement dysregulation, Eculizumab/Ravulizumab, Meningococcal vaccine = CERM
  • TMA vs DIC: PT/APTT/Fibrinogen Normal = TMA; Abnormal = DIC = P/FN vs ABN

20 Summary

Thrombotic Microangiopathies is a Must Know 🔴 topic. Key takeaway: TMA = MAHA (schistocytes, ↑LDH, ↓haptoglobin) + thrombocytopenia + organ dysfunction. TTP = ADAMTS13 <10% (pentad: MAHA, thrombocytopenia, neuro, renal, fever). Typical HUS = STEC O157:H7, children, bloody diarrhoea, severe AKI, normal ADAMTS13. aHUS = complement dysregulation, normal ADAMTS13, severe AKI. TTP urgent: PLEX + steroids + caplacizumab (within 4–8h); platelet transfusion CONTRAINDICATED. ADAMTS13 send BEFORE PLEX. Typical HUS: supportive only, NO antibiotics. aHUS: eculizumab/ravulizumab (anti-C5) + meningococcal vaccine mandatory. Secondary TMA: malignant HTN (BP), scleroderma (ACEi), CNI (stop), pregnancy. TMA vs DIC: coagulation normal in TMA. Exam focus: ADAMTS13 interpretation, TTP vs HUS vs aHUS, urgent TTP management, caplacizumab/rituximab, eculizumab vaccination, ADAMTS13 timing, TMA vs DIC, secondary TMA management. Clinical relevance: TTP is a true emergency; delay in PLEX increases mortality; aHUS transformed by complement inhibition.

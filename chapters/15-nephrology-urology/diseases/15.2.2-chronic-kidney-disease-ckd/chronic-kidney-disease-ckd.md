> **CKD = eGFR <60 mL/min/1.73m² for ≥3 months OR markers of kidney damage (ACR ≥3 mg/mmol, haematuria, structural abnormalities) for ≥3 months. KDIGO staging: G1–G5 by eGFR + A1–A3 by ACR. ## 2. Definition & Diagnosis

| Criterion | Definition |
|-----------|------------|
| **CKD** | eGFR <60 mL/min/1.73m² **for ≥3 months** **OR** markers of kidney damage (ACR ≥3 mg/mmol, haematuria, structural abnormalities) for ≥3 months |
| **Markers of Kidney Damage** | ACR ≥3 mg/mmol (or AER ≥30 mg/24h), haematuria (non-urological), structural abnormalities (imaging/histology), transplant recipient |

> [!key]
> **CKD requires persistence ≥3 months.** Single eGFR <60 or ACR ≥3 is not enough — must be confirmed on ≥2 occasions ≥90 days apart.

## 3. KDIGO Staging (2012)

### GFR Categories (G)

| Category | eGFR (mL/min/1.73m²) | Description |
|----------|----------------------|-------------|
| **G1** | ≥90 | Normal or high |
| **G2** | 60–89 | Mildly decreased |
| **G3a** | 45–59 | Mildly to moderately decreased |
| **G3b** | 30–44 | Moderately to severely decreased |
| **G4** | 15–29 | Severely decreased |
| **G5** | <15 | Kidney failure |

### Albuminuria Categories (A)

| Category | ACR (mg/mmol) | AER (mg/24h) | Description |
|----------|---------------|--------------|-------------|
| **A1** | <3 | <30 | Normal to mildly increased |
| **A2** | 3–30 | 30–300 | Moderately increased (microalbuminuria) |
| **A3** | >30 | >300 | Severely increased (macroalbuminuria) |

### Prognosis by Combined G + A (Heat Map)

| | A1 (<3) | A2 (3–30) | A3 (>30) |
|---|---------|-----------|----------|
| **G1 (≥90)** | Low | Moderate | High |
| **G2 (60–89)** | Low | Moderate | High |
| **G3a (45–59)** | Moderate | High | Very High |
| **G3b (30–44)** | High | Very High | Very High |
| **G4 (15–29)** | Very High | Very High | Very High |
| **G5 (<15)** | Very High | Very High | Very High |

> [!key]
> **G3aA1 = Moderate risk; G3bA2 = Very High risk.** Low eGFR + high albuminuria = worst prognosis.

## 4. Aetiology (Common Causes)

| Cause | % in Adults | Notes |
|-------|-------------|-------|
| **Diabetic Kidney Disease** | 35–45% | Commonest cause worldwide |
| **Hypertensive Nephrosclerosis** | 20–25% | Often overdiagnosed; needs biopsy if atypical |
| **Glomerulonephritis** | 15–20% | IgA, FSGS, MN, etc. |
| **Obstructive Uropathy** | 5–10% | BPH, stones, strictures |
| **ADPKD** | 5% | Autosomal dominant |
| **Tubulointerstitial** | 5% | Analgesics, lithium, reflux |
| **Unknown** | 5–10% | Late presentation, small kidneys |

## 5. Clinical Presentation

| Stage | Typical Features |
|-------|------------------|
| **G1–G2** | Asymptomatic; detected on screening (ACR, BP) |
| **G3a–G3b** | Fatigue, nocturia, mild anaemia, early CKD-MBD |
| **G4** | Anaemia, bone disease, acidosis, volume overload, drug accumulation |
| **G5** | Uraemic symptoms: pericarditis, encephalopathy, neuropathy, bleeding, severe volume overload, refractory hyperkalaemia |

## 6. Management Principles

### 1. Cardiovascular Risk Reduction (Major Cause of Death)
| Intervention | Target |
|--------------|--------|
| **Statin** | All CKD ≥50yo; <50yo if CVD/diabetes/10yr risk >10% |
| **Antiplatelet** | Secondary prevention only (not primary) |
| **BP Control** | **<130/80** (KDIGO 2021) |
| **SGLT2 Inhibitor** | **DAPA-CKD/EMPA-KIDNEY**: eGFR ≥20, UACR ≥200 — **↓ ESRD/CV death 30–40%** |
| **Finerenone** | T2DM + DKD on ACEi/ARB + SGLT2i, eGFR ≥25, K+ ≤4.8 — **↓ CV/renal events** |

### 2. Blood Pressure
| Agent | Preference |
|-------|------------|
| **ACEi/ARB** | **1st line** (especially if A2/A3); max tolerated dose |
| **Add-on** | Thiazide (G1–G3), loop diuretic (G4–G5), CCB, β-blocker |
| **Dual RAAS Blockade** | **CONTRAINDICATED** (↑ hyperkalaemia, AKI) |

### 3. CKD-Mineral and Bone Disorder (CKD-MBD)

| Parameter | Target (G3a–G5) | Treatment |
|-----------|-----------------|-----------|
| **Phosphate** | 0.9–1.3 mmol/L (G3–G5) | Dietary restriction, **phosphate binders** (Ca-based, sevelamer, lanthanum, sucroferric oxyhydroxide) |
| **Calcium** | 2.1–2.5 mmol/L (corrected) | Avoid Ca overload; limit Ca-based binders if vascular calcification |
| **PTH** | **2–9× ULN** (G3–G5) | **Vitamin D analogues** (alfacalcidol, calcitriol) if PTH rising + low/normal Ca + normal/high PO4; **Cinacalcet** if PTH very high + high Ca/PO4 |
| **Vitamin D (25-OH)** | >30 ng/mL | Supplement if deficient (cholecalciferol) |

> [!key]
> **CKD-MBD triad**: ↑ PO4, ↓ Ca, ↑ PTH → vascular calcification + bone disease. **KDIGO: PTH 2–9× ULN.**

### 4. Anaemia of CKD

| Parameter | Target / Action |
|-----------|-----------------|
| **Hb Target** | **100–115 g/L** (avoid >130 g/L — ↑ stroke/thrombosis risk) |
| **Iron** | TSAT ≥20%, Ferritin ≥100 ng/mL (or ≥200 if on ESA) |
| **ESA** | Darbepoetin/Epoetin alfa; start if Hb <100 g/L + iron replete; individualise |
| **Workup** | Exclude B12/folate deficiency, blood loss, haemolysis, aluminium (historical) |

### 5. Metabolic Acidosis
| Threshold | Treatment |
|-----------|-----------|
| **HCO₃⁻ <22 mmol/L** | **Oral sodium bicarbonate** 500–1000 mg TDS (target HCO₃⁻ ≥22) |
| **Benefit** | Slows CKD progression, improves muscle, bone, nutrition |

### 6. Volume Management
| Stage | Approach |
|-------|----------|
| **G1–G3** | Salt restriction <5g/day; thiazide if needed |
| **G4–G5** | Loop diuretic (furosemide/b bumetanide); ultrafiltration if on dialysis |

### 7. Nutrition
| Parameter | Recommendation |
|-----------|----------------|
| **Protein** | 0.8 g/kg/day (G1–G4); 1.0–1.2 g/kg/day (G5 on dialysis) |
| **Salt** | <2g Na/day (<5g salt) |
| **K+** | Restrict if hyperkalaemic; low-K+ diet if on dialysis |
| **Calories** | 30–35 kcal/kg/day |

### 8. Drug Dosing in CKD
| Principle | Application |
|-----------|-------------|
| **eGFR-based dosing** | Use Cockcroft-Gault (CrCl) for drug dosing, not CKD-EPI eGFR |
| **Avoid nephrotoxins** | NSAIDs, aminoglycosides, IV contrast (if possible), tick-dose adjustments |
| **Common adjustments** | Antibiotics, DOACs, metformin, gabapentin, digoxin, allopurinol |

---

## 7. Complications & Acute Management

| Complication | Emergency Threshold | Management |
|--------------|---------------------|------------|
| **Hyperkalaemia** | K+ >6.5 or ECG changes | Ca gluconate, insulin/dextrose, salbutamol, binder, **dialysis** |
| **Fluid Overload** | Pulmonary oedema | IV loop diuretic, fluid restrict, **dialysis** if refractory |
| **Acidosis** | pH <7.2, HCO₃⁻ <12 | IV bicarbonate, **dialysis** |
| **Uraemic Pericarditis** | Friction rub, effusion | **Dialysis** (intensive) |
| **Uraemic Encephalopathy** | Confusion, asterixis, seizures | **Dialysis** |

---

## 8. Preparation for RRT (G4–G5)

| Timing | Action |
|--------|--------|
| **G4 (eGFR 15–29)** | **Refer to nephrology**; **AV fistula creation** (6mo maturation); transplant workup; patient education (modalities); conservative care discussion |
| **G5 (eGFR <15)** | **Start RRT** (AEIOU); pre-emptive transplant if living donor; dialysis access ready |

### Vascular Access
| Type | Preference | Maturation |
|------|------------|------------|
| **AVF (Arteriovenous Fistula)** | **Gold standard** (radiocephalic > brachiocephalic) | 6–8 weeks |
| **AVG (Arteriovenous Graft)** | If AVF not feasible | 2–3 weeks |
| **CVC (Central Venous Catheter)** | **Temporary only** (high infection/stent risk) | Immediate |

---

## 9. Conservative (Non-Dialytic) Management
- For frail elderly with multiple comorbidities, poor functional status
- Focus: symptom control, volume, anaemia, acidosis, BP, advance care planning
- Survival similar to dialysis in selected >75yo with high comorbidity

## 10. High-Yield FCPS/MRCP Points

> [!important]
> - **CKD = eGFR <60 or ACR ≥3 for ≥3 months**
> - **KDIGO G1–G5 + A1–A3**; heat map for prognosis
> - **Commonest cause: Diabetic kidney disease (~40%)**
> - **BP <130/80**; **ACEi/ARB 1st line**; **NO dual RAAS blockade**
> - **SGLT2i (dapagliflozin/empagliflozin): eGFR ≥20, UACR ≥200 — game changer**

> - **CKD-MBD: PO4 0.9–1.3, Ca 2.1–2.5, PTH 2–9× ULN**

> - **Acidosis: HCO₃⁻ <22 → NaHCO₃**
> - **G4: refer nephrology, AV fistula planning, transplant workup**
> - **G5: RRT (AEIOU) or conservative care**
> - **Drug dosing: use CrCl (Cockcroft-Gault), not eGFR**
> - **Avoid NSAIDs, aminoglycosides, contrast**

## 11. Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **eGFR <60 once = CKD** | Needs **≥3 months** persistence |
| **ACEi + ARB = better renoprotection** | **Contraindicated** (ONTARGET, VA NEPHRON) |
| **Statin only if LDL high** | **All CKD ≥50yo** (regardless of LDL) |
| **Hb target >120 g/L** | **100–115 g/L**; >130 ↑ stroke/thrombosis (CHOIR, TREAT) |
| **PTH target = normal** | **2–9× ULN** (suppressed PTH = adynamic bone) |
| **SGLT2i contraindicated in CKD** | **Indicated eGFR ≥20** (DAPA-CKD, EMPA-KIDNEY) |
| **Ca-based binders always 1st line** | Limit if vascular calcification/ hypercalcaemia; sevelamer/lanthanum alternatives |
| **Dialysis = always better than conservative** | Conservative non-dialytic management valid for frail elderly |
| **AVF = ready immediately** | **6–8 weeks maturation**; plan in G4 |
| **Drug dosing by eGFR** | Use **CrCl (Cockcroft-Gault)** for drug dosing |

## 21. Summary

**Chronic Kidney Disease** is a **Must Know 🔴** topic.
**Key takeaway:** CKD = eGFR <60 or ACR ≥3 for ≥3 months. **KDIGO G1–G5 + A1–A3** heat map for prognosis. Commonest cause: **diabetic kidney disease**. BP **<130/80**; **ACEi/ARB 1st line; NO dual blockade**. **SGLT2i (eGFR ≥20, UACR ≥200) game-changer**. Statin all ≥50yo. **CKD-MBD: PO4 0.9–1.3, Ca 2.1–2.5, PTH 2–9× ULN**. Anaemia: **Hb 100–115 g/L**. Acidosis: **HCO₃⁻ <22 → NaHCO₃**. **G4: refer, AV fistula (6wk maturation), transplant workup**. **G5: AEIOU for RRT**. Drug dosing: **CrCl (Cockcroft-Gault)**. Avoid nephrotoxins.
**Exam focus:** Definition, staging, diabetic kidney disease, BP/ACEi/ARB/SGLT2i/statin, CKD-MBD targets, anaemia, acidosis, G4/G5 referral, drug dosing, conservative management.
**Clinical relevance:** Early SGLT2i + ACEi/ARB dramatically slows progression; comprehensive multifactorial management reduces CV events (major cause of death in CKD).

> [!important]
> **TMA = microangiopathic haemolytic anaemia (MAHA) + thrombocytopenia + organ dysfunction. ADAMTS13 activity distinguishes: TTP = severe deficiency (<10%); Typical HUS (STEC) = normal ADAMTS13; aHUS = complement dysregulation (normal ADAMTS13). TTP = neurological + renal + fever + MAHA + thrombocytopenia (pentad) → PLASMA EXCHANGE urgent. Typical HUS = diarrhoea (STEC) → supportive. aHUS = complement inhibitors (eculizumab/ravulizumab).**

---

## 1. Learning Objectives
- Differentiate TTP, typical HUS (STEC), and atypical HUS (aHUS) by ADAMTS13, clinical features, triggers
- Recognise MAHA (schistocytes, ↑ LDH, ↓ haptoglobin, ↓ Hb) + thrombocytopenia
- Apply urgent management: TTP = plasma exchange + steroids + caplacizumab; HUS = supportive; aHUS = complement inhibitors
- Understand complement pathway and genetic testing in aHUS

---

## 2. Classification & Pathophysiology

| TMA Type | ADAMTS13 Activity | Pathophysiology | Typical Trigger |
|----------|-------------------|-----------------|-----------------|
| **TTP** (Thrombotic Thrombocytopenic Purpura) | **<10%** (severe deficiency) | **Autoantibody** (acquired) or **mutation** (congenital/Upshaw-Schulman) → ↑ ultra-large VWF multimers → platelet aggregation | Idiopathic, pregnancy, drugs, infection, autoimmune |
| **Typical HUS** (STEC-HUS) | **Normal** (50–150%) | **Shiga toxin** (STEC O157:H7) → endothelial injury → platelet activation | **E. coli O157:H7** (bloody diarrhoea); Shigella dysenteriae |
| **Atypical HUS** (aHUS) | **Normal** | **Complement dysregulation** (alternative pathway) → uncontrolled C3 convertase → endothelial damage | Genetic (CFH, CFI, MCP, C3, CFB, THBD, DGKE); triggers: pregnancy, infection, drugs |
| **Secondary TMA** | Variable | Underlying disease | Malignant HTN, Scleroderma, SLE, APS, drugs (CNI, quinine, gemcitabine), transplantation, cancer |

---

## 3. Diagnostic Criteria (MAHA + Thrombocytopenia)

| Feature | TTP | Typical HUS (STEC) | aHUS |
|---------|-----|-------------------|------|
| **ADAMTS13** | **<10%** | Normal | Normal |
| **Neurological** | **Prominent** (confusion, stroke, seizure, headache) | Rare/mild | Variable |
| **Renal** | Mild-moderate AKI | **Severe AKI** (oliguric) | **Severe AKI** |
| **Diarrhoea** | No | **Bloody diarrhoea (prodrome)** | No (or non-bloody) |
| **Fever** | Common | Common | Variable |
| **Age** | Adults (20–50), women > men | **Children** (<5yo) | All ages |
| **Triggers** | Pregnancy, drugs, infection | STEC infection (foodborne) | Complement mutations + trigger |

> [!key]
> **Pentad (TTP)**: MAHA + Thrombocytopenia + Neurological + Renal + Fever (classic but NOT all needed for diagnosis).
> **Diagnosis = MAHA + Thrombocytopenia + no other cause** → check ADAMTS13 urgently.

## 4. Laboratory Features (MAHA)

| Test | Finding | Significance |
|------|---------|--------------|
| **Schistocytes** | **≥1% (or >2/HPF)** on peripheral smear | **Hallmark of MAHA** |
| **LDH** | **Markedly elevated** (>1000 U/L) | Intravascular haemolysis |
| **Haptoglobin** | **Low/undetectable** | Consumed by free Hb |
| **Bilirubin** | ↑ Indirect (unconjugated) | Haemolysis |
| **Haemoglobin** | ↓ (anaemia) | Haemolysis |
| **Reticulocytes** | ↑ (reticulocytosis) | Compensatory |
| **Platelets** | **<100×10⁹/L** (often <20×10⁹/L in TTP) | Consumption |
| **Coagulation (PT/APTT/Fibrinogen)** | **Normal** (distinguishes from DIC) | **TMA ≠ DIC** |
| **D-dimer** | Normal/mildly elevated | Not DIC |
| **ADAMTS13 Activity** | **<10% = TTP**; Normal = HUS/aHUS | **Urgent send** (result guides therapy) |
| **ADAMTS13 Inhibitor** | Present in acquired TTP | Autoimmune TTP |
| **Complement Studies** | C3, C4, CFH, CFI, MCP, C3, CFB, genetic panel | aHUS workup |

## 5. Clinical Presentation

### TTP
- Acute/subacute onset
- **Neurological**: confusion, headache, stroke, seizure, visual disturbances, coma
- **Renal**: mild-moderate AKI (Cr usually <250–300)
- **Fever**, **petechiae/purpura**
- **No diarrhoea** (prodrome)
- Women 20–50yo predominant

### Typical HUS (STEC)
- **Children** (peak 6mo–5yo)
- **Prodrome**: bloody diarrhoea (3–7 days) → HUS
- **Severe AKI** (oliguric, often dialysis)
- Neurological: rare/mild
- **E. coli O157:H7** (hamburger, unpasteurised milk, petting zoos)

### aHUS
- All ages; familial or sporadic
- **Severe AKI** (often dialysis)
- Hypertension common
- Neurological: variable
- **No diarrhoea** (or non-infectious)
- **Trigger**: pregnancy, infection, drugs, surgery

## 6. Management

### TTP — Medical Emergency

| Intervention | Details |
|--------------|---------|
| **Plasma Exchange (PLEX)** | **URGENT — start within 4–8 hours**; 1–1.5 plasma volumes daily until platelets >150×10⁹/L × 2 days |
| **Corticosteroids** | **High-dose** (methylprednisolone 1g × 3d → pred 1mg/kg) + PLEX |
| **Caplacizumab** | **Anti-VWF nanobody**; blocks VWF-platelet interaction; **early addition to PLEX + steroids** (HERCULES trial) |
| **Rituximab** | **Acquired TTP**: anti-CD20; reduces relapse (40–50% relapse without) |
| **Splenectomy** | Historical; now rarely needed (rituximab) |
| **Folic Acid** | 5mg daily (haemolysis) |
| **Platelet Transfusion** | **CONTRAINDICATED** (worsens thrombosis) — only if life-threatening bleeding |
| **FFP Transfusion** | If PLEX delayed (>4–8h); bridging |

> [!key]
> **TTP = PLEX + steroids + caplacizumab URGENT. Platelet transfusion CONTRAINDICATED.**

### Typical HUS (STEC) — Supportive

| Intervention | Details |
|--------------|---------|
| **Supportive** | Fluids, electrolytes, dialysis if needed |
| **Antibiotics** | **NOT recommended** (may ↑ Shiga toxin release) |
| **Antimotility** | **Avoid** (loperamide) |
| **Eculizumab** | **Not standard**; case reports in severe/neurological |
| **Monitor** | Neurological, dialysis, haemolysis resolution |

### Atypical HUS (aHUS) — Complement Inhibition

| Intervention | Details |
|--------------|---------|
| **Eculizumab** | **Anti-C5 monoclonal antibody**; blocks terminal complement; **transformed outcomes** |
| | **Dosing**: 900mg weekly × 4 → 1200mg q2wk (adults); weight-based paeds |
| **Ravulizumab** | **Long-acting anti-C5**; q8wk dosing (adults) |
| **Meningococcal Vaccination** | **Mandatory pre-eculizumab/ravulizumab** (MenACWY + MenB); antibiotic prophylaxis |
| **Plasma Exchange/Infusion** | Bridge to eculizumab; not definitive |
| **Genetic Testing** | CFH, CFI, MCP (CD46), C3, CFB, THBD, DGKE; guides prognosis/family screening |
| **Duration** | **Lifelong** (if complement mutation); consider trial off if no mutation + remission |

> [!key]
> **aHUS = eculizumab/ravulizumab (anti-C5) + meningococcal vaccination.**
> **PLEX not definitive for aHUS** (complement factors replenished).

### Secondary TMA

| Cause | Management |
|-------|------------|
| **Malignant HTN** | **BP control** (IV labetalol, nicardipine); treat TMA if persistent |
| **Scleroderma Renal Crisis** | **ACEi (captopril) aggressive**; target BP; dialysis if needed |
| **CNI (Tac/CsA)** | **Stop/reduce CNI**; switch to mTOR/belatacept; PLEX if severe |
| **Pregnancy (HELLP/aHUS/TTP)** | Multidisciplinary; delivery if term; PLEX/eculizumab per type |
| **Drug-induced** | Stop offending drug |

## 7. ADAMTS13 Interpretation

| ADAMTS13 Activity | Interpretation |
|-------------------|----------------|
| **<10%** | **TTP** (acquired if inhibitor+; congenital if no inhibitor, early onset) |
| **10–20%** | Borderline; consider TTP if clinical fit |
| **>20% (Normal)** | **NOT TTP** → HUS/aHUS/secondary TMA |

> [!key]
> **Send ADAMTS13 BEFORE PLEX** (PLEX replenishes ADAMTS13 → false normal).

## 8. Differential: TMA vs DIC

| Feature | TMA | DIC |
|---------|-----|-----|
| **Platelets** | Low | Low |
| **Schistocytes** | **Prominent** | Few/absent |
| **PT/APTT** | **Normal** | **Prolonged** |
| **Fibrinogen** | **Normal** | **Low** |
| **D-dimer** | Normal/mild | **High** |
| **Coagulation** | Not consumed | Consumed |
| **Trigger** | VWF/complement/Shiga toxin | Sepsis, trauma, obstetric, cancer |

## 9. High-Yield FCPS/MRCP Points

> [!important]
> - **TMA = MAHA (schistocytes, ↑LDH, ↓haptoglobin) + thrombocytopenia + organ dysfunction**
> - **TTP = ADAMTS13 <10%**: neuro + renal + fever + MAHA + thrombocytopenia (pentad)
> - **Typical HUS = STEC O157:H7**: children, bloody diarrhoea → severe AKI, normal ADAMTS13
> - **aHUS = complement dysregulation**: normal ADAMTS13, severe AKI, **eculizumab/ravulizumab**
> - **TTP: PLEX + steroids + caplacizumab URGENT** (within 4–8h); platelet transfusion CONTRAINDICATED
> - **ADAMTS13 send BEFORE PLEX** (PLEX replenishes it)
> - **Typical HUS: supportive only**; antibiotics/antimotility NOT recommended
> - **aHUS: eculizumab/ravulizumab (anti-C5) + MenACWY/MenB vaccination**
> - **Secondary TMA**: malignant HTN (BP control), scleroderma (ACEi), CNI (stop/reduce), pregnancy
> - **TMA vs DIC: PT/APTT/Fibrinogen NORMAL in TMA; abnormal in DIC**
> - **Caplacizumab**: anti-VWF nanobody; early addition to PLEX+steroids (HERCULES)
> - **Rituximab**: acquired TTP relapse prevention

## 10. Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **All TMA = TTP** | TTP = ADAMTS13 <10%; HUS/aHUS = normal ADAMTS13 |
| **Platelets low = transfuse** | **Platelet transfusion CONTRAINDICATED in TTP** (worsens thrombosis) |
| **PLEX before ADAMTS13** | **Send ADAMTS13 FIRST** (PLEX replenishes → false normal) |
| **Typical HUS = antibiotics** | **Antibiotics NOT recommended** (↑ Shiga toxin release) |
| **aHUS = PLEX definitive** | **PLEX not definitive** (complement replenished); need eculizumab |
| **Eculizumab = no vaccination** | **Meningococcal vaccination MANDATORY** (MenACWY + MenB) |
| **HUS only in children** | Typical HUS = children; aHUS = all ages |
| **TTP = always pentad** | Pentad = classic but NOT required; MAHA + thrombocytopenia enough |
| **ADAMTS13 <10% = always acquired** | Can be congenital (Upshaw-Schulman) if early onset, no inhibitor |
| **DIC = TMA** | DIC = abnormal PT/APTT/fibrinogen/D-dimer; TMA = normal coagulation |

## 11. Mnemonics

- **MAHA**: **S**chistocytes, **L**DH↑, **H**aptoglobin↓, **H**b↓, **R**etic↑ = **SLHHR**
- **TTP Pentad**: **M**AHA, **T**hrombocytopenia, **N**euro, **R**enal, **F**ever = **MTNRF**
- **ADAMTS13**: **<10% = TTP**; **Normal = HUS/aHUS** = **<10 TTP, N HUS**
- **TTP Tx**: **P**LEX + **S**teroids + **C**aplacizumab = **PSC** (urgent)
- **TTP Contraindicated**: **P**latelet **T**ransfusion = **PT** (worsens)
- **Typical HUS**: **S**TEC, **C**hildren, **D**iarrhoea (bloody), **S**evere AKI = **SCDS**
- **aHUS**: **C**omplement dysregulation, **E**culizumab/**R**avulizumab, **M**eningococcal vaccine = **CERM**
- **TMA vs DIC**: **P**T/APTT/**F**ibrinogen **N**ormal = TMA; **A**bnormal = **DIC** = **P/FN vs ABN**

## 20. Summary

**Thrombotic Microangiopathies** is a **Must Know 🔴** topic.
**Key takeaway:** TMA = MAHA (schistocytes, ↑LDH, ↓haptoglobin) + thrombocytopenia + organ dysfunction. **TTP = ADAMTS13 <10%** (pentad: MAHA, thrombocytopenia, neuro, renal, fever). **Typical HUS = STEC O157:H7**, children, bloody diarrhoea, severe AKI, normal ADAMTS13. **aHUS = complement dysregulation**, normal ADAMTS13, severe AKI. **TTP urgent: PLEX + steroids + caplacizumab (within 4–8h); platelet transfusion CONTRAINDICATED.** **ADAMTS13 send BEFORE PLEX.** Typical HUS: supportive only, NO antibiotics. **aHUS: eculizumab/ravulizumab (anti-C5) + meningococcal vaccine mandatory.** Secondary TMA: malignant HTN (BP), scleroderma (ACEi), CNI (stop), pregnancy. **TMA vs DIC: coagulation normal in TMA.**
**Exam focus:** ADAMTS13 interpretation, TTP vs HUS vs aHUS, urgent TTP management, caplacizumab/rituximab, eculizumab vaccination, ADAMTS13 timing, TMA vs DIC, secondary TMA management.
**Clinical relevance:** TTP is a true emergency; delay in PLEX increases mortality; aHUS transformed by complement inhibition.

Cryoglobulinaemic Glomerulonephritis
1 Learning Objectives
  • Classify cryoglobulinaemia (Types I, II, III)
  • Recognise clinical triad (purpura, arthralgia, weakness) + renal involvement
  • Interpret complement profile (Low C4, Low/Normal C3, Cryoglobulins+, RF+)
  • Manage based on type and severity (DAAs for HCV, rituximab, plasma exchange)
  • Differentiate from other MPGN causes
  • Apply to FCPS/MRCP immune-complex GN vignettes
2 Classification (Brouet)
TypeCompositionAssociationPrevalence
--------------------------------------------
**Type I****Monoclonal** (IgM/IgG/IgA κ/λ)**Lymphoproliferative** (Waldenström, myeloma, CLL, lymphoma)10–15%
**Type II****Mixed**: Monoclonal IgM (κ) + Polyclonal IgG**HCV (80–90%)**, also lymphoproliferative**50–60%**
**Type III****Mixed**: Polyclonal IgM + Polyclonal IgG**SLE, RA, Sjögren's**, other autoimmune25–30%
Type II (HCV) = most common mixed cryoglobulinaemia. Type I = malignancy. Type III = autoimmune.
3 Pathogenesis
  • Cryoglobulins = immunoglobulins that precipitate at <37°C and dissolve on rewarming
  • Type II/III: IgM (with rheumatoid factor activity) binds polyclonal IgG → immune complexes
  • Cold exposure → precipitation in capillaries (skin, kidneys, nerves) → vasculitis
  • Renal: Subendothelial deposits → MPGN pattern (mesangial expansion, GBM duplication)

6 Renal Histology
FindingDescription
----------------------
**LM****MPGN pattern** (mesangial expansion, GBM double contours "tram-track")
**IF****Granular IgM, IgG, C3** (capillary wall + mesangium)

7 Management
Type I (Monoclonal) — Treat Underlying Malignancy
ApproachDetail
------------------
**Primary**Treat lymphoproliferative disorder (rituximab-based chemo, BTKi, etc.)
**Renal**Plasmapheresis if severe RPGN/acute
**Rituximab**Often effective (targets B-cell clone)
Type II (HCV-Associated) — **Treat HCV First**
SeverityRegimen
-------------------
**Mild (proteinuria <1g, preserved eGFR)****DAAs alone** (sofosbuvir/velpatasvir × 12wk) — cryoglobulins clear with HCV cure
**Moderate (proteinuria 1–3.5g, eGFR >30)****DAAs + Rituximab** (375mg/m² weekly × 4)
**Severe (RPGN, nephrotic, AKI, eGFR <30)****Plasma Exchange + Rituximab + Steroids → then DAAs**
DAAs cure HCV → cryoglobulins disappear in 90%. Rituximab for persistent/severe vasculitis.
Type III (Autoimmune) — Treat Underlying Disease
DiseaseManagement
---------------------
**SLE**Standard lupus nephritis protocol (MMF/CyC + steroids)
**RA / Sjögren's**Rituximab + steroids ± immunosuppressants
**Severe**Plasma exchange + rituximab + steroids
8 Plasma Exchange (PLEX) Indications
IndicationProtocol
----------------------
**RPGN** (rapidly rising Cr)1–1.5 plasma vol daily × 7–14 days
**Severe neuropathy** (mononeuritis multiplex)Same
**Severe skin ulcers/gangrene**Same
**Hyperviscosity** (Type I, high monoclonal)Same
PLEX removes cryoglobulins acutely; bridge to immunosuppression/DAAs.

10 High-Yield FCPS/MRCP Points
[!important] - Cryoglobulinaemic GN = MPGN from cryoglobulin precipitation. - Types: I (monoclonal, malignancy), II (mixed monoclonal+polyclonal, HCV 80–90%), III (polyclonal, SLE/RA). - Hallmark labs: Low C4, High RF, Cryoglobulins+, HCV RNA+ (Type II). - Renal: MPGN pattern, subendothelial deposits + microtubules (30–50nm) on EM. - Clinical: Purpura, arthralgia, weakness + neuropathy. - Type II (HCV): DAAs first-line (cure HCV → cryoglobulins clear); add rituximab if moderate/severe. - Type I: Treat malignancy (rituximab-chemo). - Type III: Treat autoimmune (SLE/RA protocol). - Severe: Plasma exchange + rituximab + steroids. - Differential: MPGN (C3G, IC-MPGN), SLE, ANCA, anti-GBM, IE.
11 Common Confusions / Exam Traps
TrapCorrection
------------------
**All cryoglobulinaemia = HCV****Type I = malignancy**; Type III = SLE/RA
**Low C3 + Low C4 = always SLE****Cryoglobulinaemia also Low C3 + Low C4** (classical pathway)
**RF = only RA****High RF in cryoglobulinaemia** (monoclonal IgM has RF activity)
**DAAs alone for all HCV cryo****Severe (RPGN/nephrotic) = PLEX + rituximab + steroids first**
**Cryoglobulins = send cold tube****Send WARM tube (37°C)** — cold precipitates in tube, false negative
**Type I = MPGN**Type I = usually **no renal** or minimal; Type II/III = MPGN
**Cryoglobulins = only renal****Skin, joints, nerves, liver** all involved
**ANCA positive in cryo****Usually ANCA negative** — differentiates from ANCA vasculitis
12 Mnemonics
  • Types: I = Immune malignancy (monoclonal); II = II = HCV (mixed); III = III = Autoimmune (polyclonal)
  • Type II: HCV + Mixed = HM (80–90%)
  • Lab triad: Low C4, High RF, Cryoglobulins+ = LHC
  • EM: Microtubules 30–50nm = M3
  • Treatment: HCV = DAAs; Malignancy = Rituximab-chemo; Autoimmune = Rituximab/immunosuppression

21 Summary

Cryoglobulinaemic Glomerulonephritis is a Must Know 🔴 immune-complex GN topic. Classification: Type I (monoclonal, malignancy), Type II (mixed monoclonal+polyclonal, HCV 80–90%), Type III (polyclonal, SLE/RA). Labs: Low C4 (hallmark), High RF, Cryoglobulins+ (send warm tube), HCV RNA+ (Type II). Renal: MPGN pattern, subendothelial microtubules (30–50nm) on EM. Clinical: Meltzer's triad (purpura, arthralgia, weakness) + neuropathy, skin ulcers. Management: Type II (HCV) — DAAs first; add rituximab if moderate/severe; PLEX + Rituximab + Steroids for RPGN/severe. Type I — treat malignancy. Type III — treat autoimmune. PLEX indications: RPGN, severe neuropathy, hyperviscosity, alveolar haemorrhage. Exam focus: Three types, Low C4/High RF, microtubules EM, warm tube, DAAs for HCV, PLEX+Rituximab for severe, Meltzer's triad.

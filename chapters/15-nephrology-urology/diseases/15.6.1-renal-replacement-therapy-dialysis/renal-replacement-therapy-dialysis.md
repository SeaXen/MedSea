# Renal Replacement Therapy — Dialysis (HD, PD), Indications, Complications

**Related:** [[Acute Kidney Injury (AKI)]], [[Chronic Kidney Disease (CKD)]], [[Kidney Transplantation (Indications, Workup, Immunosuppression, Complications)]], [[Blood Tests in Renal Disease]], [[Nephrology and Urology MOC]]

> [!important]
> **RRT = life-sustaining therapy for ESRD (G5) or severe AKI (AEIOU). Modalities: Intermittent Haemodialysis (IHD), Continuous Renal Replacement Therapy (CRRT), Peritoneal Dialysis (PD). Indications: AEIOU. Vascular access: AVF (gold standard), AVG, CVC. Complications: access-related, metabolic, infections, cardiovascular.**

---

## Learning Objectives
- Identify indications for RRT (AEIOU mnemonic)
- Compare HD, CRRT, PD modalities
- Understand vascular access types and complications
- Manage dialysis complications (intradialytic hypotension, disequilibrium, access issues)
- Apply to FCPS/MRCP clinical scenarios

## Indications for RRT (AEIOU Mnemonic)

| Letter | Indication | Details |
|--------|------------|---------|
| **A** | **Acidosis** | pH <7.2, HCO₃⁻ <12–15 mmol/L refractory to medical therapy |
| **E** | **Electrolytes** | K+ >6.5 mmol/L or ECG changes refractory to medical therapy |
| **I** | **Ingestion** | Toxic alcohols (methanol, ethylene glycol), lithium, salicylates, metformin |
| **O** | **Overload** | Pulmonary oedema refractory to diuretics; pericardial effusion/tamponade |
| **U** | **Uraemia** | Pericarditis, encephalopathy, neuropathy, bleeding diathesis, refractory nausea/vomiting |

> [!key]
> **AEIOU = absolute indications for urgent RRT.** Chronic ESRD (G5) = planned RRT start.

## Dialysis Modalities Comparison

| Feature | **IHD** (Intermittent HD) | **CRRT** (CVVH/CVVHDF) | **PD** (Peritoneal Dialysis) |
|---------|---------------------------|------------------------|------------------------------|
| **Schedule** | 3–4h × 3/week | Continuous 24/7 | Continuous (CAPD) or nightly (APD) |
| **Setting** | Hospital/satellite unit | ICU | Home |
| **Haemodynamics** | Large fluid shifts → hypotension risk | **Stable** (slow fluid removal) | Stable |
| **Clearance** | High (intermittent) | Lower (continuous) | Lowest (continuous) |
| **Anticoagulation** | Heparin/citrate (circuit) | Citrate (regional) / heparin | None |
| **Solute Removal** | Diffusion dominant | Convection (CVVH) / Mixed (CVVHDF) | Diffusion |
| **Fluid Removal** | Ultrafiltration (UF) | UF (controlled) | UF (glucose gradient) |
| **Indications** | Stable ESRD, AKI (if stable) | **Haemodynamically unstable AKI**, sepsis, fluid overload | **ESRD (patient choice), children, vascular access failure** |
| **Access** | AVF/AVG/CVC | CVC (large bore) | PD catheter (Tenckhoff) |

## Haemodialysis (IHD) — Detailed

### Vascular Access
| Type | Description | Maturation | Longevity | Complications |
|------|-------------|------------|-----------|---------------|
| **AVF** (Arteriovenous Fistula) | **Radiocephalic (wrist) → Brachiocephalic (elbow) → Brachiobasilic (upper arm)** | 6–8 weeks | **Years (gold standard)** | Stenosis, thrombosis, aneurysm, steal syndrome |
| **AVG** (Arteriovenous Graft) | PTFE loop (usually upper arm) | 2–3 weeks | 1–3 years | Higher infection/thrombosis than AVF |
| **CVC** (Central Venous Catheter) | **Tunnelled** (Permanent) or **Non-tunnelled** (Temporary) | Immediate | Weeks–months | **Infection (CRBSI), thrombosis, stenosis, fibrin sheath** |

> [!key]
> **AVF = gold standard (radiocephalic first).** **CVC = last resort (high infection).** **Fistula First initiative.**

### HD Prescription (Typical)
| Parameter | Typical Value |
|-----------|---------------|
| **Blood Flow (Qb)** | 300–400 mL/min |
| **Dialysate Flow (Qd)** | 500–800 mL/min |
| **Duration** | 3–4 hours × 3/week |
| **Dialyser** | High-flux (β2-microglobulin clearance) |
| **Anticoagulation** | Unfractionated heparin (bolus + infusion) or citrate (regional) |
| **Dialysate** | K+ 1–2 mmol/L (adjust), Ca 1.25–1.5 mmol/L, Bicarbonate 32–35 mmol/L, Glucose 5.5 mmol/L |

### HD Adequacy
| Measure | Target |
|---------|--------|
| **spKt/V (single-pool)** | **≥1.2** per session (minimum); **≥1.4** preferred |
| **URR (Urea Reduction Ratio)** | **≥65%** (preferably ≥70%) |
| **eKt/V** | ≥1.0 (equilibrated) |
| **Frequency** | Monthly |

## Continuous Renal Replacement Therapy (CRRT)

### Modalities
| Modality | Mechanism | Best For |
|----------|-----------|----------|
| **CVVH** (Continuous Veno-Venous Haemofiltration) | **Convection** (solvent drag) | Middle molecules, fluid removal |
| **CVVHD** (Continuous Veno-Venous Haemodialysis) | **Diffusion** | Small molecules |
| **CVVHDF** (Continuous Veno-Venous Haemodiafiltration) | **Convection + Diffusion** | **Most common** — balanced clearance |

### Prescription
| Parameter | Typical Value |
|-----------|---------------|
| **Blood Flow** | 100–200 mL/min |
| **Effluent Flow (Dose)** | **20–25 mL/kg/h** (KDIGO: 20–25 mL/kg/h) |
| **Anticoagulation** | **Regional citrate** (preferred) — target post-filter ionised Ca 0.25–0.35 mmol/L; Systemic heparin if citrate contraindicated |
| **Filter Life** | 24–72 hours (goal >24h) |

### Indications for CRRT over IHD
- Haemodynamic instability (vasopressors, shock)
- Severe fluid overload (need precise UF control)
- Raised ICP (avoid rapid osmolar shifts)
- Sepsis (cytokine clearance — evidence limited)
- Lack of vascular access for IHD (large CVC for CRRT)

## Peritoneal Dialysis (PD)

### Types
| Type | Description |
|------|-------------|
| **CAPD** (Continuous Ambulatory PD) | **4 exchanges/day** (manual); dwell 4–6h day, 8–10h night |
| **APD** (Automated PD) | **Cycler at night** (8–10h); 1–2 daytime dwells (last fill) |

### PD Catheter & Solutions
| Component | Details |
|-----------|---------|
| **Catheter** | Tenckhoff (double cuff); inserted laparoscopically/open/percutaneously; **2-week break-in** before full use |
| **Solutions** | Glucose-based (1.36%, 2.27%, 3.86%) — osmotic agent; Icodextrin (7.5%) — long dwell (colloid osmotic); Bicarbonate/lactate buffer |
| **Dwell Phases** | Fill → Dwell → Drain |

### PD Adequacy
| Measure | Target |
|---------|--------|
| **Weekly Kt/V** | **≥1.7** |
| **Weekly Creatinine Clearance** | **≥45 L/week/1.73m²** |
| **Residual Renal Function** | Critical — preserves urine output, volume control, middle molecule clearance |

### PD Complications
| Complication | Management |
|--------------|------------|
| **Peritonitis** | **Cloudy effluent + abdominal pain**; Cell count >100/μL (neutrophils >50%); **Empiric IP antibiotics** (vancomycin + ceftazidime/aminoglycoside); Culture-directed; Catheter removal if refractory/fungal/multiple |
| **Exit-site/Tunnel Infection** | Topical + systemic antibiotics; catheter removal if tunnel |
| **Hernia** | Inguinal, umbilical, incisional; surgical repair |
| **Ultrafiltration Failure** | High transporter (fast glucose absorption); icodextrin for long dwell; consider HD |
| **Encapsulating Peritoneal Sclerosis (EPS)** | Rare, late; bowel obstruction; high mortality; stop PD |

## Dialysis Complications

### Intradialytic Hypotension (IDH)
| Feature | Management |
|---------|------------|
| **Definition** | SBP drop >20 mmHg or MAP drop >10 mmHg + symptoms |
| **Causes** | Rapid UF, autonomic dysfunction, antihypertensives, low dialysate Ca, acetate buffer (historical) |
| **Prevention** | Sodium profiling, cooled dialysate (35–35.5°C), midodrine, reduce UF rate, avoid pre-dialysis antihypertensives |
| **Treatment** | Stop UF, Trendelenburg, IV saline bolus (100–200mL) |

### Dialysis Disequilibrium Syndrome (DDS)
| Feature | Management |
|---------|------------|
| **Mechanism** | Rapid urea clearance → cerebral oedema (osmotic gradient) |
| **Risk** | First HD, severe uraemia (Cr >800), elderly, children |
| **Symptoms** | Headache, nausea, confusion, seizures, coma |
| **Prevention** | **Gentle first HD** (low Qb, short duration, high dialysate Na, mannitol) |
| **Treatment** | Stop HD, hypertonic saline, mannitol, airway protection |

### Access Complications
| Complication | AVF/AVG | CVC |
|--------------|---------|-----|
| **Thrombosis** | Thrombectomy, thrombolysis, revision | Thrombolysis (alteplase), exchange |
| **Stenosis** | Angioplasty (balloon), stenting | Fibrin sheath stripping, exchange |
| **Infection** | Rare (superficial) | **CRBSI** (blood cultures ×2, catheter-tip); remove if refractory/Staph aureus/fungal |
| **Aneurysm** | Surgical repair if large/painful | N/A |
| **Steal Syndrome** | Ischaemic hand; banding, revision | N/A |

### Metabolic & Other
| Complication | Management |
|--------------|------------|
| **Hyperkalaemia** | Low-K+ dialysate (1mmol/L), insulin/dextrose pre-HD, avoid high-K+ foods |
| **Hypokalaemia** | Higher K+ dialysate (2–3mmol/L), supplement |
| **Dialysis Amyloidosis** | β2-microglobulin deposition (carpal tunnel, cysts, arthropathy); high-flux dialysers reduce |
| **Cramps** | Reduce UF rate, hypertonic saline, quinine (prophylactic) |
| **Itch (Pruritus)** | Optimise PO4, Ca; gabapentin, UVB, montelukast, difelikefalin (KOR agonist) |

## HD vs PD — Patient Factors

| Factor | Favours HD | Favours PD |
|--------|------------|------------|
| **Vascular Access** | Good vessels for AVF | Failed access / no vessels |
| **Haemodynamics** | Stable | Unstable (but PD also ok) |
| **Residual Renal Function** | Less critical | **Critical** (preserves RRF) |
| **Home Environment** | Not suitable | Suitable for home |
| **Patient Preference** | Unit-based, professional care | **Autonomy, flexibility, travel** |
| **Abdominal Surgery** | No issue | Adhesions → contraindicated |
| **Obesity** | HD often easier | PD possible but UF harder |
| **Heart Failure** | HD may worsen | PD better volume control |

## High-Yield FCPS/MRCP Points

> [!important]
> - **AEIOU = absolute RRT indications** (Acidosis, Electrolytes, Ingestion, Overload, Uraemia)
> - **IHD = 3x/week, 3–4h; AVF gold standard (radiocephalic); CVC high infection**
> - **CRRT = haemodynamically unstable; citrate anticoagulation preferred; dose 20–25 mL/kg/h**
> - **PD = CAPD (4 exchanges) or APD (cycler); Tenckhoff catheter; peritonitis = cloudy effluent + pain**
> - **Adequacy**: IHD spKt/V ≥1.2, URR ≥65%; PD weekly Kt/V ≥1.7, CrCl ≥45 L/week
> - **IDH**: prevent with sodium profiling, cool dialysate, midodrine; treat: stop UF, IV saline
> - **DDS**: first HD, severe uraemia; prevent: gentle HD (low Qb, short, high Na dialysate)
> - **Peritonitis (PD)**: cloudy effluent + pain; cell count >100/μL; empiric IP vancomycin + ceftazidime
> - **Access**: AVF > AVG > CVC (Fistula First); CVC = CRBSI risk
> - **Anticoagulation**: HD = heparin/citrate; CRRT = citrate preferred; PD = none
> - **PD preserves residual renal function** — key advantage

## Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **CRRT = only for AKI** | Also for ESRD with instability; fluid overload |
| **PD = only for young/healthy** | Also for elderly; better QoL for some |
| **CVC = acceptable long-term access** | **CVC = last resort**; high infection, stenosis; aim for AVF |
| **IDH = just give saline** | **Prevention better**: sodium profiling, cool dialysate, midodrine, reduce UF |
| **DDS = only first HD** | Can occur in any session if aggressive; prevent with gentle HD |
| **Citrate anticoagulation = no monitoring** | **Monitor post-filter ionised Ca (0.25–0.35), acid-base (metabolic alkalosis risk)** |
| **PD adequacy = only Kt/V** | **Residual renal function critical** — include in total Kt/V |
| **Peritonitis = always remove catheter** | Remove if refractory, fungal, Staph aureus, multiple episodes |
| **AVF ready immediately** | **6–8 weeks maturation**; use CVC meanwhile |
| **All patients need same HD dose** | Larger patients need more; spKt/V ≥1.2 per session |

## Summary

**Renal Replacement Therapy** is a **Must Know 🔴** topic.
**Key takeaway:** **AEIOU** = urgent RRT indications. **HD**: 3x/week, **AVF gold standard** (radiocephalic), CVC last resort; **spKt/V ≥1.2**. **CRRT**: haemodynamically unstable; **citrate anticoagulation**; 20–25 mL/kg/h. **PD**: CAPD/APD; **Tenckhoff catheter**; preserves RRF; weekly Kt/V ≥1.7; **peritonitis** = cloudy effluent + pain + >100 WBC; IP vancomycin + ceftazidime. **IDH prevention**: sodium profiling, cool dialysate, midodrine. **DDS prevention**: gentle first HD. **Access**: AVF > AVG > CVC. **Citrate**: post-filter iCa 0.25–0.35.
**Exam focus:** AEIOU, modality selection, AVF hierarchy, adequacy targets, IDH/DDS, PD peritonitis, citrate anticoagulation, access complications, steal syndrome, icodextrin for high transporters.
**Clinical relevance:** Modality choice impacts survival, QoL, healthcare costs; Fistula First reduces CVC complications; PD preserves residual function.

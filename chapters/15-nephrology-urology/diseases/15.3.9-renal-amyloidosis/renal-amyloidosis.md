# Amyloidosis (Renal Amyloidosis)

**Related:** [[Glomerular Diseases — Overview and Classification]], [[Secondary Glomerulonephritides — Lupus Nephritis]], [[Secondary Glomerulonephritides — ANCA-Associated Vasculitis (GPA, MPA, EGPA)]], [[Secondary Glomerulonephritides — Diabetic Nephropathy]], [[Nephrology and Urology MOC]]

> [!important]
> **Renal Amyloidosis = extracellular deposition of misfolded protein fibrils (β-pleated sheets). AL (light chain) = commonest renal; AA (amyloid A) = chronic inflammation; Hereditary (ATTR, AFib, etc.). Renal: nephrotic syndrome, CKD. Diagnosis: biopsy + Congo red + apple-green birefringence + typing (immunohistochemistry/mass spec). Treat underlying cause.**

---

## Learning Objectives
- Classify amyloid types (AL, AA, hereditary, dialysis-related)
- Recognise renal and systemic manifestations
- Apply diagnostic algorithm (biopsy, Congo red, typing)
- Manage per type (AL: chemotherapy; AA: treat inflammation; hereditary: TTR stabilisers/silencers)

---

## Classification & Aetiology

| Type | Precursor Protein | Source | Clinical Context |
|------|-------------------|--------|------------------|
| **AL** (Amyloid Light Chain) | **Immunoglobulin light chain** (κ or λ) | **Plasma cell dyscrasia** (MGUS, myeloma, Waldenström's) | **Commonest renal amyloidosis**; median age 60–70; λ > κ |
| **AA** (Amyloid A) | **Serum amyloid A (SAA)** | Chronic inflammation (RA, JIA, IBD, FMF, TB, bronchiectasis) | Secondary to chronic inflammatory disease |
| **ATTR** (Transthyretin) | **Transthyretin** | **Hereditary** (ATTRv: Val30Met, etc.) or **Wild-type** (ATTRwt: elderly, cardiac) | Cardiac + neuropathy ± renal |
| **AFib** (Fibrinogen Aα) | **Fibrinogen Aα-chain** | Hereditary | Renal dominant (amyloid nephropathy) |
| **Aβ2M** (β2-Microglobulin) | **β2-Microglobulin** | **Dialysis-related** (long-term HD >10–15yr) | Carpal tunnel, arthropathy, bone cysts; renal not affected (already ESRD) |

---

## Renal Manifestations

| Feature | AL Amyloidosis | AA Amyloidosis | Hereditary (AFib, ATTR) |
|---------|----------------|----------------|------------------------|
| **Proteinuria** | Nephrotic range common | Nephrotic range common | Variable |
| **Nephrotic Syndrome** | ~70% | ~50% | Common (AFib) |
| **Renal Impairment** | Common at presentation | Variable | Progressive CKD |
| **Haematuria** | Rare | Rare | Rare |
| **Hypertension** | Uncommon (vs diabetic) | Variable | Variable |
| **Kidney Size** | **Normal/Large** (infiltration) | Normal/Large | Normal/Small late |

> [!key]
> **Normal/Large kidneys + nephrotic syndrome + no retinopathy + low BP = think amyloidosis.**

---

## Systemic Manifestations (AL)

| System | Features |
|--------|----------|
| **Heart** | **Restrictive cardiomyopathy** (thickened walls, preserved EF, low voltage ECG), arrhythmias, HFpEF |
| **Nerves** | Peripheral neuropathy (painful, autonomic), carpal tunnel |
| **GI** | Macroglossia (pathognomonic), GI dysmotility, bleeding, malabsorption |
| **Liver** | Hepatomegaly, elevated ALP |
| **Skin** | Purpura (periorbital, "raccoon eyes"), waxy plaques, nail changes |
| **Soft Tissue** | Shoulder pads (arthropathy), jaw claudication |
| **Bleeding** | Factor X deficiency (adsorption to amyloid) → coagulopathy |

## Diagnosis

### 1. Suspicion
- Nephrotic syndrome + **normal/large kidneys** + **no hypertension** + **systemic features** (heart, nerve, GI, skin)
- **AL**: Age >50, monoclonal protein, cardiac involvement
- **AA**: Chronic inflammatory disease
- **Hereditary**: Family history, neuropathy/cardiac (ATTR), renal dominant (AFib)

### 2. Biopsy (Gold Standard)
| Stain | Finding |
|-------|---------|
| **Congo Red** | **Apple-green birefringence** under polarised light (β-pleated sheets) |
| **Thioflavin T** | Fluorescence (sensitive) |
| **Electron Microscopy** | **Non-branching fibrils 7–10 nm** (random orientation) |

### 3. Typing (Essential for Treatment)
| Method | Detail |
|--------|--------|
| **Immunohistochemistry** | κ, λ, AA, TTR, fibrinogen — widely available, cross-reactivity possible |
| **Laser Microdissection + Mass Spectrometry (LMD-MS)** | **Gold standard** — definitive typing |
| **Serum Free Light Chains (sFLC)** | **AL: abnormal κ/λ ratio + involved FLC elevated** |
| **Serum/Urinary Immunofixation** | Detect monoclonal protein |
| **Bone Marrow Biopsy** | Plasma cell % (MGUS <10%, myeloma ≥10%) |
| **SAA Level** | AA: elevated in active inflammation |

> [!key]
> **Typing MUST be done before treatment** — AL vs AA vs hereditary = completely different management.

## AL Amyloidosis — Staging & Treatment

### Staging (Mayo 2012)

| Stage | Criteria | Median OS |
|-------|----------|-----------|
| **I** | NT-proBNP <1800, Troponin T <0.025, Involved FLC <180 | ~94 months |
| **II** | 1 criterion elevated | ~40 months |
| **III** | 2 criteria elevated | ~14 months |
| **IV** | 3 criteria elevated | ~6 months |

### Treatment (Plasma Cell Directed)

| Regimen | Indication |
|---------|------------|
| **Daratumumab + CyBorD (Dara-CyBorD)** | **New standard 1st line** (ANDROMEDA trial: ↑ haematologic CR, ↑ organ response) |
| **CyBorD** (Cyclophosphamide + Bortezomib + Dexamethasone) | 1st line if daratumumab unavailable |
| **Bortezomib-based** | Renal impairment (bortezomib not renally cleared) |
| **ASCT** (Autologous Stem Cell Transplant) | **Eligible patients** (age <70, stage I–II, adequate organ function) — consolidation post-induction |
| **Venetoclax** | t(11;14) translocation (BCL2 high) |

### Response Criteria (Haematologic)
| Response | Criteria |
|----------|----------|
| **CR** | Involved FLC normal + normal κ/λ ratio + negative immunofixation + <5% marrow plasma cells |
| **VGPR** | Involved FLC <40 mg/L |
| **PR** | Involved FLC ↓ >50% |
| **Organ Response** | ↓ Proteinuria >50% (renal), ↓ NT-proBNP >30% (cardiac), ↓ liver span/ALP (hepatic) |

---

## AA Amyloidosis — Treatment

| Principle | Detail |
|-----------|--------|
| **Treat underlying inflammation** | **Control SAA <10 mg/L** (target) → halts amyloid deposition, may regress |
| **Anti-inflammatory** | DMARDs (methotrexate, sulfasalazine), biologics (anti-TNF, anti-IL-1, anti-IL-6), colchicine (FMF) |
| **Colchicine** | **FMF: lifelong prophylaxis** (prevents amyloidosis) |
| **No chemotherapy** | Not plasma cell disorder |

---

## Hereditary Amyloidosis — Treatment

| Type | Treatment |
|------|-----------|
| **ATTRv** (Hereditary TTR) | **TTR stabilisers**: Tafamidis, Diflunisal; **Gene silencers**: Patisiran (siRNA), Inotersen (ASO), Vutrisiran (siRNA) |
| **ATTRwt** (Wild-type) | Tafamidis (cardiomyopathy); supportive |
| **AFib** | **Liver transplant** (source of fibrinogen) ± kidney transplant; TTR silencers not effective |
| **Other** | Organ-specific; genetic counselling |

---

## Dialysis-Related Amyloidosis (Aβ2M)

| Feature | Detail |
|---------|--------|
| **Cause** | β2-microglobulin retention (long-term HD >10–15yr; low-flux membranes) |
| **Manifestations** | Carpal tunnel syndrome, destructive arthropathy (shoulders, hips), bone cysts, pathological fractures, flexor tenosynovitis |
| **Prevention** | **High-flux dialysers**, longer dialysis, convective therapies (HDF) |
| **Treatment** | Symptomatic; kidney transplant (resolves) |

## High-Yield FCPS/MRCP Points

> [!important]
> - **Renal amyloidosis**: AL (commonest renal), AA (chronic inflammation), Hereditary (ATTR, AFib), Aβ2M (dialysis)
> - **AL = plasma cell dyscrasia** (MGUS/myeloma); λ > κ
> - **Renal**: Nephrotic syndrome, **normal/large kidneys**, **no hypertension**
> - **AL Systemic**: Heart (restrictive CM, low voltage ECG), GI (macroglossia), nerves, liver, skin (purpura), Factor X deficiency
> - **Diagnosis**: Biopsy + **Congo red + apple-green birefringence** + **TYPING** (IHC/LMD-MS)
> - **AL Typing**: sFLC (abnormal κ/λ ratio), immunofixation, bone marrow
> - **AL Staging**: Mayo 2012 (NT-proBNP, Troponin, involved FLC)
> - **AL Treatment**: **Dara-CyBorD** 1st line (ANDROMEDA); ASCT eligible; Bortezomib-based if renal impairment
> - **AA Treatment**: **Treat inflammation → SAA <10 mg/L**; colchicine for FMF
> - **ATTR**: Tafamidis (stabiliser), Patisiran/Inotersen/Vutrisiran (silencers)
> - **AFib**: Liver transplant ± kidney transplant
> - **Aβ2M**: Long-term HD; carpal tunnel, arthropathy; high-flux dialysers prevent

## Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **All amyloid = AL** | AA, hereditary, dialysis-related = different treatment |
| **Congo red = diagnosis complete** | **Typing essential** — AL vs AA vs hereditary = different treatment |
| **AL = myeloma** | AL = plasma cell dyscrasia (MGUS, smouldering, myeloma); most have MGUS |
| **AA = only RA** | Any chronic inflammation (IBD, FMF, TB, bronchiectasis, etc.) |
| **Amyloid kidneys = small** | **Normal/LARGE** (infiltration) — distinguishes from diabetic/chronic |
| **AL treatment = only ASCT** | **Dara-CyBorD 1st line** (ANDROMEDA); ASCT = consolidation for eligible |
| **SAA = only for AA diagnosis** | **SAA <10 = treatment target** for AA |
| **ATTR = only cardiac** | ATTRv = neuropathy + cardiac + renal; ATTRwt = cardiac dominant |
| **Aβ2M = renal amyloidosis** | Kidneys already ESRD; deposits in joints/tendons/bone |
| **Factor X deficiency = only AL** | Can occur in any amyloid (adsorption to fibrils) but commonest in AL |

## Summary

**Amyloidosis** is a **Must Know 🔴** topic.
**Key takeaway:** Types: **AL** (plasma cell dyscrasia, λ>κ, commonest renal), **AA** (chronic inflammation, SAA), **Hereditary** (ATTRv/wt, AFib), **Aβ2M** (long-term HD). Renal: **nephrotic + normal/large kidneys + no HT**. AL systemic: heart (restrictive CM, low voltage), macroglossia, neuropathy, Factor X deficiency. **Diagnosis: Biopsy + Congo red (apple-green) + TYPING (essential)**. AL: sFLC ratio, immunofixation, marrow. **Mayo staging**: NT-proBNP, Troponin, involved FLC. **AL Tx: Dara-CyBorD 1st line** (ANDROMEDA); ASCT eligible. **AA: SAA <10 target**. **ATTR: tafamidis, patisiran/inotersen/vutrisiran**. **AFib: liver transplant**. **Aβ2M: high-flux dialysers**.
**Exam focus:** Types, renal triad, AL systemic features, Congo red, typing essential, AL staging, Dara-CyBorD, AA SAA target, ATTR treatments, AFib liver transplant, Aβ2M prevention.
**Clinical relevance:** Early typing directs life-saving therapy; Dara-CyBorD improved AL outcomes; SAA control halts AA progression.

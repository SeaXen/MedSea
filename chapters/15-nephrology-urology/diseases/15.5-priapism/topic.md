> [!important]
> **Priapism = prolonged penile erection (>4 hours) unrelated to sexual stimulation. UROLOGICAL EMERGENCY. Types: Ischaemic (low-flow, 95%) = corporal hypoxia → urgent aspiration/irrigation ± shunting; Non-ischaemic (high-flow, 5%) = arterial fistula → conservative/embolisation. Stuttering = recurrent ischaemic episodes. Sickle cell disease = most common cause in children/young men. PDE5 inhibitors, anticoagulants, antipsychotics = drug causes.**

## 2. Classification

| Type | Mechanism | Blood Flow | Age | Urgency |
|------|-----------|------------|-----|---------|
| **Ischaemic (Low-flow)** | Venous occlusion → corporal hypoxia/acidosis | **Low/Stagnant** | All (peak 30–40, sickle cell: children) | **Emergency (>4hr = permanent damage)** |
| **Non-ischaemic (High-flow)** | Arterial-lacunar fistula (trauma) → unregulated arterial inflow | **High** | Post-perineal trauma | Semi-elective |
| **Stuttering (Recurrent Ischaemic)** | Recurrent self-limiting ischaemic episodes | Intermittent | Sickle cell, idiopathic | Prevent progression |

## 3. Ischaemic Priapism (Low-Flow) — Emergency

### Aetiology
| Category | Causes |
|----------|--------|
| **Idiopathic** | ~30–50% |
| **Sickle Cell Disease** | **Most common in children/adolescents** (HbSS, HbSC, HbSβ-thal) |
| **Drug-Induced** | **PDE5 inhibitors** (sildenafil, tadalafil), **Anticoagulants** (warfarin, heparin, DOACs), **Antipsychotics** (risperidone, olanzapine, chlorpromazine), **Antidepressants** (trazodone, SSRIs), **Alpha-blockers** (tamsulosin), **Cocaine, Alcohol** |
| **Malignancy** | Leukaemia, metastatic infiltration |
| **Neurological** | Spinal cord injury, herniated disc, tumours |
| **Infections** | Sepsis, UTI |

---

### Clinical Presentation
| Feature | Ischaemic | Non-ischaemic |
|---------|-----------|---------------|
| **Pain** | **Severe, progressive** | Painless or mild |
| **Erection** | **Rigid corporal bodies**, glans **soft** (glans not involved) | **Non-rigid**, tumescent, glans may be engorged |
| **Duration** | >4 hours (often hours–days) | Hours to days |
| **Corpus spongiosum/glans** | **Soft** (not involved) | May be engorged |
| **Penile Doppler** | **Low/absent cavernosal flow** | **High arterial flow** |
| **Cavernosal Blood Gas** | **pH <7.25, pO₂ <30, pCO₂ >60** (acidosis) | Normal arterial values |

> **Glans soft + rigid corpora = ischaemic priapism** (corpus spongiosum spared).

---

### Cavernosal Blood Gas (Diagnostic)
| Parameter | Ischaemic | Normal Arterial |
|-----------|-----------|-----------------|
| **pH** | **<7.25** (acidosis) | 7.35–7.45 |
| **pO₂** | **<30 mmHg** (hypoxia) | 80–100 mmHg |
| **pCO₂** | **>60 mmHg** (hypercapnia) | 35–45 mmHg |

## 4. Management — Ischaemic Priapism (Algorithm)

### Stepwise Approach (Time-Critical)
| Time | Intervention |
|------|--------------|
| **0–4 hours** | Conservative: analgesia, ice packs, gentle exercise, oral terbutaline/sudafed (α-agonist) |
| **>4 hours** | **EMERGENCY: Corporal aspiration + irrigation + intracavernosal phenylephrine** |
| **Failed aspiration** | **Surgical shunt (distal → proximal)** |
| **Sickle Cell** | **Hydration, oxygenation, analgesia, exchange transfusion** ± aspiration/shunt |

---

### 1. Corporal Aspiration & Irrigation
| Step | Detail |
|------|--------|
| **Anaesthesia** | Penile block (dorsal nerve) or local infiltration |
| **Aspiration** | 19G butterfly needle at 2 & 10 o'clock (corpora cavernosa) — aspirate 20–50mL dark blood |
| **Irrigation** | Normal saline 50–100mL per side until effluent clear |
| **Phenylephrine** | **100–500 μg/mL** (dilute 1mg in 100–200mL NS) — inject 1mL q3–5min × max 1mg (monitor BP/HR) |
| **Repeat** | Aspirate/irrigate q10–15min if detumescence incomplete |

> **Phenylephrine**: α₁-agonist → corporal smooth muscle contraction. **Monitor BP/HR** (hypertension, bradycardia, arrhythmia). Contraindicated in severe CAD, uncontrolled HTN.

---

### 2. Surgical Shunt (If Aspiration Fails @ 1 hour)
| Shunt Type | Description | Success Rate |
|------------|-------------|--------------|
| **Distal (Glans-Corporal)** | **Winter** (biopsy punch 3–5mm at glans-corpus junction), **Ebbehøj** (incision) | 60–80% |
| | **Al-Ghorab** (excision of tunica at glans-corpus) | 70–85% |
| **Proximal (Corporospongiosal / Cavernosospongiosal)** | **Quackels** (corpus-spongiosum fistula), **Grayhack** (corpus-spongiosum with saphenous vein graft) | 70–90% |
| | **Eberhard** (cavernosospongiosal with graft) | 70–85% |

> **Sequence**: Distal first (Winter/Ebbehøj/Al-Ghorab) → Proximal if distal fails.

---

### 3. Sickle Cell Priapism Specifics
| Intervention | Detail |
|--------------|--------|
| **Hydration** | IV fluids (1.5–2x maintenance) |
| **Oxygen** | 100% O₂ (reduce sickling) |
| **Analgesia** | Opioids (avoid NSAIDs if renal impairment) |
| **Exchange Transfusion** | **Target HbS <30%, Hb >10g/dL** — if aspiration fails or recurrent |
| **Hydroxyurea** | Prophylaxis for recurrent/stuttering |
| **Aspiration/Shunt** | Same as above; lower threshold for exchange |

---

## 5. Non-Ischaemic Priapism (High-Flow)

### Aetiology
- **Perineal/penile trauma** (straddle injury, pelvic fracture) → **arterial-lacunar fistula**
- **Iatrogenic** (penile injection, surgery)

### Clinical
- Painless/tender, **non-rigid** tumescence
- Often post-trauma history
- Doppler: **High arterial flow** (arterialised blood gas)

### Management
| Approach | Detail |
|----------|--------|
| **Observation** | 25–50% spontaneous resolution |
| **Selective Arterial Embolisation** | **Definitive** (superselective angiography + coils/particles) — 90% success |
| **Surgical Ligation** | If embolisation fails/unavailable |
| **Compression** | Perineal pressure (temporary) |

---

## 6. Stuttering (Recurrent Ischaemic) Priapism
- **Recurrent self-limiting episodes** (<4hr each)
- **High risk of progression** to fulminant ischaemic priapism
- **Aetiology**: Sickle cell (most common), idiopathic
- **Prophylaxis**: **Hydroxyurea** (sickle cell), **PDE5 inhibitors** (some evidence), ** GnRH agonists**, **Baclofen**, **Terbutaline**

---

## 7. High-Yield FCPS/MRCP Points

> [!important]
> - **Priapism >4hr = urological emergency**.
> - **Ischaemic (95%)** = rigid corpora, soft glans, **painful**, **low flow**, **acidic blood gas** (pH<7.25, pO₂<30).
> - **Non-ischaemic (5%)** = painless, non-rigid, **high flow**, post-trauma.
> - **Ischaemic >4hr**: **Aspiration + irrigation + phenylephrine** (emergency).
> - **Phenylephrine**: 100–500 μg/mL, 1mL q3–5min, max 1mg; **monitor BP/HR**.
> - **Failed aspiration → shunt**: Distal (Winter/Al-Ghorab) → Proximal.
> - **Sickle cell**: Hydration, O₂, analgesia, **exchange transfusion** (HbS<30%), hydroxyurea prophylaxis.
> - **Non-ischaemic**: Post-trauma, painless, non-rigid → **embolisation** (90% success).
> - **Stuttering**: Recurrent ischaemic → hydroxyurea prophylaxis.

## 8. Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **All priapism = ischaemic** | **5% = non-ischaemic** (high flow, post-trauma, painless) |
| **Glans rigid in ischaemic** | **Glans SOFT in ischaemic** (corpus spongiosum spared); glans rigid in non-ischaemic |
| **Ice packs first for ischaemic** | Ice OK adjunct but **aspiration/irrigation is definitive** after 4hr |
| **Phenylephrine = no monitoring** | **Monitor BP/HR** (hypertension, bradycardia, arrhythmia) |
| **Sickle cell = same as idiopathic** | **Lower threshold for exchange transfusion**; hydroxyurea prophylaxis |
| **Non-ischaemic = emergency surgery** | **Observation/embolisation** first; surgery if embolisation fails |
| **All priapism >4hr = shunt** | **Aspiration/irrigation/phenylephrine FIRST** (60–80% success) |
| **Oral terbutaline = definitive** | Only adjunct <4hr; >4hr = aspiration |
| **Glans involvement = ischaemic** | **Glans soft = ischaemic**; glans engorged = non-ischaemic |
| **Priapism in children = only sickle cell** | Also idiopathic, trauma, drugs, leukaemia |

## 9. Mnemonics
- **Ischaemic features**: **R**igid corpus, **S**oft glans, **P**ainful, **A**cidic blood = **RSPA**
- **Blood gas ischaemic**: **p**H **L**ow, **p**O₂ **L**ow, **p**CO₂ **H**igh = **Low Low High**
- **Ischaemic stepwise**: **A**spiration → **I**rrigation → **P**henylephrine → **S**hunt = **AIPS**
- **Phenylephrine**: **P**henylephrine = **P**ressure monitor (BP/HR)
- **Shunt sequence**: **D**istal (Winter/Al-Ghorab) → **P**roximal = **DP**
- **Sickle cell**: **H**ydration, **O**xygen, **A**nalgesia, **E**xchange = **HOAE**

## 18. Summary
**Priapism** is a **Must Know 🔴** urological emergency.  
**Ischaemic (Low-flow, 95%)**: Rigid corpora, **soft glans**, painful, **blood gas pH<7.25, pO₂<30, pCO₂>60**. Causes: sickle cell, drugs (PDE5i, antipsychotics, anticoagulants), idiopathic. **>4hr = emergency**.  
**Stepwise Rx**: **Aspiration + irrigation + phenylephrine** (100–500 μg/mL, q3–5min, max 1mg, **monitor BP/HR**) → if fails → **Distal shunt (Winter/Al-Ghorab)** → **Proximal shunt**.  
**Sickle Cell**: Hydration, O₂, analgesia, **exchange transfusion (HbS<30%)**, hydroxyurea prophylaxis for stuttering.  
**Non-ischaemic (High-flow, 5%)**: Post-trauma, painless, non-rigid, **high flow** → **embolisation** (90% success).  
**Stuttering**: Recurrent ischaemic episodes → **hydroxyurea prophylaxis**.  
**Exam focus**: Ischaemic vs non-ischaemic differentiation, >4hr emergency, aspiration/irrigation/phenylephrine protocol, phenylephrine monitoring, shunt sequence, sickle cell exchange target, non-ischaemic embolisation, glans soft in ischaemic.

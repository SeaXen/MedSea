> [!important]
> **Cystinosis = autosomal recessive lysosomal cystine storage disorder (CTNS gene, chr17p13). Cystine accumulates in lysosomes → multi-organ damage. Three forms: Nephropathic infantile (95%, severe), Late-onset juvenile/adult (milder, no Fanconi), Ocular (corneal crystals only). Most common cause of childhood Fanconi syndrome.**

---

## 1. Learning Objectives
- Recognise classic infantile nephropathic cystinosis presentation (Fanconi, rickets, photophobia)
- Differentiate three clinical forms
- Diagnose via leukocyte cystine assay, genetic testing, corneal slit-lamp
- Manage with cysteamine (cystine-depleting), supportive Fanconi care, transplant
- Monitor for extra-renal complications (thyroid, diabetes, myopathy, CNS)
- Apply to FCPS/MRCP paediatric nephrology vignettes

## 2. Genetics & Pathogenesis
| Gene | Protein | Function | Mutation Effect |
|------|---------|----------|-----------------|
| **CTNS** (chr17p13) | **Cystinosin** | Lysosomal cystine/H+ symporter (exports cystine from lysosome) | Loss-of-function → cystine accumulation in lysosomes → crystallisation → cellular damage |

- **Autosomal recessive**
- Carrier frequency ~1:100 (higher in French Canadians)
- **Cystine** = dimer of cysteine; poorly soluble → crystallises in lysosomes

---

## 3. Clinical Forms

| Form | Age Onset | Key Features | Prognosis |
|------|-----------|--------------|-----------|
| **Nephropathic (Infantile)** | **6–12 months** | **Fanconi syndrome**, rickets, polyuria, failure to thrive, photophobia | ESRD ~10yr if untreated; transplant → long survival |
| **Late-onset (Juvenile/Adult)** | **>12 years** | **Proteinuria/CKD** (no/focal Fanconi), milder | ESRD 20–40yr |
| **Ocular (Benign)** | Any age | **Corneal cystine crystals only** (photophobia) | No renal disease |

---

## 4. Nephropathic Infantile Cystinosis (Classic)

### Presentation (6–18 months)
| System | Features |
|--------|----------|
| **Renal** | **Full Fanconi syndrome**: polyuria, polydipsia, dehydration, glycosuria (normal glucose), aminoaciduria, phosphaturia (rickets), bicarbonaturia (RTA Type 2), hypouricaemia, low-molecular-weight proteinuria |
| **Growth** | **Failure to thrive**, short stature |
| **Bone** | **Rickets** (phosphate wasting, 1,25-Vit D deficiency) — resistant to Vit D |
| **Eyes** | **Photophobia**, blepharospasm (corneal cystine crystals) — by age 2–3yr |
| **Thyroid** | Hypothyroidism (later) |
| **Pancreas** | Diabetes mellitus (insulin-deficient, later) |
| **Muscle** | Myopathy, swallowing difficulty (later) |
| **CNS** | Neurocognitive impairment (rare, late) |

> **Fanconi + photophobia + rickets in infant** = cystinosis until proven otherwise.

---

## 5. Late-Onset Cystinosis
- Onset >12 years (often 15–30yr)
- **Proteinuria/CKD** predominant
- **No/mild Fanconi**, no rickets
- Corneal crystals present
- ESRD ~25–40yr

---

## 6. Ocular (Benign) Cystinosis
- **Corneal crystals only** (slit-lamp)
- Photophobia
- **No renal involvement**, normal kidney function
- Adult diagnosis common

---

## 7. Diagnosis

| Test | Finding |
|------|---------|
| **Leukocyte cystine assay** | **Gold standard** — elevated cystine (nmol ½ cystine/mg protein): Infantile >500, Juvenile 150–500, Carrier 50–150, Normal <50 |
| **Genetic testing (CTNS)** | Confirms; 57kb founder deletion common in Northern Europeans |
| **Slit-lamp exam** | **Corneal cystine crystals** (needle-shaped, refractile) — pathognomonic |
| **Urinalysis** | Fanconi pattern (glycosuria, phosphaturia, aminoaciduria, bicarbonaturia) |
| **Renal biopsy** | Not needed; shows cystine crystals in tubular cells (electron-dense) |

> **Leukocyte cystine assay** = diagnostic standard. **Slit-lamp** = quick screen.

---

## 8. Management

### Cysteamine (Cystine-Depleting Agent)
| Form | Dose | Notes |
|------|------|-------|
| **Cysteamine bitartrate (Cystagon)** | **1.3–1.95 g/m²/day** divided q6h | **Start ASAP** — delays ESRD by 10+ years |
| **Enteric-coated (Procysbi)** | Equivalent dose q12h | Better GI tolerability |
| **Ophthalmic drops (0.55% / 0.1%)** | q1h waking hours | **Dissolves corneal crystals**, prevents photophobia |

> **Cysteamine** enters lysosome → reacts with cystine → forms cysteine + cysteine-cysteamine mixed disulfide → exits via cationic amino acid transporter.

### Supportive Fanconi Management
| Issue | Treatment |
|-------|-----------|
| **RTA Type 2 (bicarbonaturia)** | NaHCO₃/NaCitrate 5–15 mEq/kg/day (high dose) |
| **Hypophosphataemia/Rickets** | Phosphate supplements + 1,25-Vit D (calcitriol) |
| **Hypokalaemia** | KCl supplements |
| **Dehydration/Polyuria** | Free water access, avoid dehydration |
| **Growth** | GH therapy (if growth failure despite metabolic control) |
| **Thyroid** | Levothyroxine if hypothyroid |
| **Diabetes** | Insulin (insulin-deficient) |

### Transplant
- **Kidney transplant** = definitive for ESRD
- **Cysteamine continues post-transplant** → prevents extra-renal complications
- **No recurrence** in allograft (genetic defect in host, not kidney)
- Combined liver-kidney not needed (liver not major site)

---

## 9. Extra-Renal Complications (Monitor Post-Transplant)

| System | Complication | Screening |
|--------|--------------|-----------|
| **Eyes** | Corneal crystals, band keratopathy, retinopathy | Annual slit-lamp |
| **Thyroid** | Hypothyroidism (50% by age 30) | Annual TSH |
| **Pancreas** | Diabetes mellitus (insulin-deficient) | Annual HbA1c |
| **Muscle** | Distal myopathy, swallowing dysfunction | Clinical review |
| **CNS** | Neurocognitive decline, calcifications | MRI if symptomatic |
| **Gonadal** | Male infertility (obstructive azoospermia) | Semen analysis |
| **Skin** | Photosensitivity, hyperpigmentation | — |

## 11. Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **Cystinosis = only kidneys** | **Multi-system**: eyes, thyroid, pancreas, muscle, CNS, gonads |
| **All cystinosis = Fanconi** | **Late-onset = proteinuria/CKD, no Fanconi**; Ocular = eyes only |
| **Cysteamine cures everything** | Cysteamine **delays ESRD, dissolves corneal crystals** — but extra-renal complications still need monitoring |
| **Stop cysteamine post-transplant** | **Continue lifelong** — prevents extra-renal complications |
| **Corneal crystals = only cystinosis** | Also in **tyrosinaemia, hypercholesterolaemia, drug-induced (chloroquine, amiodarone), Fabry** |
| **Cystinosis = only children** | Late-onset presents in teens/adults with proteinuria |
| **Leukocyte cystine = urinary cystine** | **Leukocyte cystine** = diagnostic; urinary cystine = screening only |
| **All Fanconi = cystinosis** | Also Wilson's, Lowe, tyrosinaemia, galactosaemia, drugs (tenofovir, ifosfamide), myeloma |

## 21. Summary
**Cystinosis** is a **Must Know 🔴** paediatric nephrology topic.  
**Genetics**: CTNS (chr17p13) → cystinosin (lysosomal cystine transporter) defect → cystine accumulation. AR.  
**Forms**: **Infantile (95%)** = Fanconi + rickets + photophobia (6–18mo); **Late-onset** = proteinuria/CKD, no Fanconi; **Ocular** = corneal crystals only.  
**Diagnosis**: **Leukocyte cystine assay** (gold standard), **slit-lamp** (corneal needle-shaped crystals), CTNS sequencing.  
**Treatment**: **Cysteamine** (Cystagon q6h / Procysbi q12h) + **ophthalmic drops** (q1h). Start early, continue lifelong. Supportive Fanconi care (alkali, phosphate, Vit D, K+).  
**Transplant**: Curative for ESRD; **continue cysteamine** for extra-renal protection. No recurrence.  
**Extra-renal**: Hypothyroidism, diabetes, myopathy, CNS, male infertility — monitor annually.  
**Exam focus**: Infantile triad, leukocyte cystine assay, slit-lamp, cysteamine (oral + eye drops), three forms, transplant + continue cysteamine, extra-renal monitoring.

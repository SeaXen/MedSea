# Craniopharyngioma

> [!info]
> **Craniopharyngioma = Benign Suprasellar Tumour** arising from Rathke's Pouch Remnants. **Bimodal Age Distribution** (Children 5-14, Adults 50-70). **Most Common Paediatric Suprasellar Tumour**. **Calcifications + Cystic/Solid** on Imaging. High Recurrence Rate.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Describe embryology, pathology, and bimodal age distribution
- [ ] Recognise clinical presentation (visual, endocrine, hypothalamic)
- [ ] Interpret imaging findings (CT calcifications, MRI cystic/solid)
- [ ] Apply management algorithms (surgery ± radiotherapy)
- [ ] Understand long-term complications (hypothalamic obesity, DI, panhypopituitarism)

## 1. Definition & Epidemiology

| Feature | Details |
|---------|---------|
| **Origin** | **Rathke's Pouch Remnants** (Embryological Remnant of Anterior Pituitary Development) |
| **WHO Grade** | **Grade 1** (Benign) — But Locally Aggressive |
| **Age Distribution** | **Bimodal**: **Peak 1: 5-14 years** (Paediatric); **Peak 2: 50-74 years** (Adult) |
| **Incidence** | 0.5-2.5 per million/year; **Most Common Paediatric Suprasellar Tumour** |
| **Sex** | Equal |
| **Location** | **Suprasellar** (95%); Intrasellar (5%); Often Extends to Third Ventricle, Hypothalamus, Optic Chiasm |
| **WHO Grade** | **Grade 1** (Benign Histology) — But **Locally Invasive**, High Recurrence |

---

## 2. Histopathology

| Subtype | Histology | Frequency | Calcifications |
|---------|-----------|-----------|----------------|
| **Adamantinomatous** | **Wet Keratin** (Squamous Epithelium), **Cholesterol Clefts**, **Calcifications**, Cystic Fluid ("Motor Oil"), Palisading Nuclei | **90-95% (Children)** | **90%** (Coarse/Chunky) |
| **Papillary** | Papillary Architecture, Squamous Epithelium, No Wet Keratin | **5-10% (Adults)** | **Rare** |

---

## 2. Clinical Presentation

### By Tumour Location / Compression

| Compressed Structure | Clinical Features |
|---------------------|-------------------|
| **Optic Chiasm** | **Bi-temporal Hemianopia** (Classic), Visual Acuity Loss, Colour Vision Defect |
| **Optic Nerves** | Unilateral Visual Loss, Visual Field Defects |
| **Pituitary Stalk/Pituitary** | **Hypopituitarism** (GH > Gonad > TSH > ACTH), Growth Failure (Children), Delayed Puberty |
| **Hypothalamus** | **Hypothalamic Obesity** (Rapid Weight Gain), **Diabetes Insipidus**, Temperature Dysregulation, Sleep Disturbance, Behavioural Changes |
| **Third Ventricle** | **Obstructive Hydrocephalus** → Headache, Nausea, Vomiting, Papilloedema |
| **Optic Nerves** | Unilateral Visual Loss, Optic Atrophy |

### Age-Based Presentation
| Feature | **Children (5-14y)** | **Adults (50-74y)** |
|---------|----------------------|---------------------|
| **Commonest Presentation** | **Visual Loss** (Bi-temporal Hemianopia) | **Visual Loss / Headache** |
| **Endocrine** | **Growth Failure** (GH Deficiency), **Delayed Puberty** | **Hypogonadism**, Hypopituitarism |
| **Hypothalamic** | Obesity, DI, Temperature Dysregulation | Obesity, DI, Temperature Dysregulation |
| **Hydrocephalus** | Common (Third Ventricle Obstruction) | Less Common |
| **Calcifications on CT** | **>90%** (Adamantinous) | Variable (Papillary Rare) |

## 3. Imaging — Key Diagnostic Features

### CT Findings
| Feature | Adamantinous | Papillary |
|---------|--------------|-----------|
| **Calcifications** | **>90%** (Coarse, Chunky, "Starry Sky") | Rare |
| **Cystic Component** | **Prominent** ("Motor Oil" Fluid, High Protein) | Less Prominent |
| **Solid Component** | Enhancing Nodules | Solid Enhancing Mass |
| **Suprasellar Extension** | **Yes (Almost Always)** | Yes |

### MRI Findings
| Sequence | Findings |
|----------|----------|
| **T1** | **Cyst Fluid: Variable** (High Protein → High T1; "Motor Oil" Appearance), Solid: Isointense |
| **T2** | Cyst Fluid: Hyperintense; Solid: Heterogeneous |
| **Post-Gadolinium** | **Rim Enhancement** (Cyst Wall), **Solid Nodule Enhancement** |
| **DWI** | Restricted Diffusion in Cyst (High Protein) |
| **Suprasellar Extension** | Universal (Optic Chiasm, Third Ventricle, Hypothalamus) |
| **Calcifications** | **Blooming on SWI/T2*** (Adamantinous) |

### Differential Diagnosis on Imaging
| Feature | **Craniopharyngioma** | **Pituitary Adenoma** | **Arachnoid Cyst** | **Rathke's Cleft Cyst** | **Hypothalamic Glioma** |
|---------|----------------------|-----------------------|-------------------|------------------------|------------------------|
| **Age** | Bimodal (Child/Adult) | Adult Predominant | Any | Any | Child/Young Adult |
| **Calcifications** | **>90% (Adamantinous)** | Rare | None | Rare | None |
| **Cystic/Solid** | **Cystic + Solid (Enhancing Nodules)** | Mostly Solid (Enhancing) | **Pure Cyst** | **Pure Cyst** | Mostly Solid |
| **Location** | **Suprasellar (95%)** | Intrasellar +/- Suprasellar | Suprasellar/Pontine | **Intersellar (Ant/Post Pituitary)** | Hypothalamic/Suprasellar |
| **Calcifications** | **Chunky/Coarse** | None | None | None | None |
| **Enhancement** | Rim + Nodular | Homogeneous | None | None | Heterogeneous |
| **Calcifications on CT** | **>90%** | <5% | 0% | <5% | 0% |

## 3. Clinical Syndromes

### Visual Pathway Compression
| Feature | Details |
|---------|---------|
| **Classic** | **Bi-temporal Hemianopia** (Chiasmal Compression) |
| **Early** | Superior Quadrantanopia (Superior Chiasm Compression) |
| **Advanced** | Central Visual Acuity Loss, Optic Atrophy, Colour Vision Defect |
| **Unilateral** | Optic Nerve Compression (Rare) |

### Hypothalamic Syndrome
| Feature | Details |
|---------|---------|
| **Obesity** | **Rapid, Severe, Refractory** (Ventromedial Nucleus Damage); Insatiable Appetite |
| **Diabetes Insipidus** | **Central DI** (Supraoptic Nucleus/Stalk Damage); 30-50% at Presentation, ↑ Post-Surgery |
| **Temperature Dysregulation** | Poikilothermia (Anterior Hypothalamus), Hyperthermia (Posterior) |
| **Sleep Disturbance** | Narcolepsy-like, Circadian Rhythm Disruption (SCN Damage) |
| **Behavioural/Psychiatric** | Apathy, Aggression, Cognitive Impairment, Memory Deficits (Mammillary Bodies) |

### Endocrine Deficiencies (Order of Loss)
| Order | Axis | Frequency |
|-------|------|---------|
| **1st** | **GH** | **>90% (Children) — Growth Failure** |
| **2nd** | **Gonadotrophs** | **80-90% — Delayed Puberty / Hypogonadism** |
| **3rd** | **TSH** | **50-70% — Central Hypothyroidism** |
| **4th** | **ACTH** | **30-50% — Secondary Adrenal Insufficiency** |
| **Common** | **DI** | **30-50% (Hypothalamic/Stalk Damage)** |

## 4. Management Algorithm

```
Craniopharyngioma Suspected (Imaging + Clinical)
         │
         ▼
MULTIDISCIPLINARY TEAM (Neurosurgery, Endocrinology, Ophthalmology, Radiation Oncology, Paediatrics)
         │
         ▼
STAGING & ASSESSMENT
         │
         ├── MRI Brain (3mm, Gadolinium) — Extent, Hypothalamic Involvement, Vessels
         ├── CT Head (Calcifications) — Adamantinous vs Papillary
         ├── Visual Fields (Goldmann/Humphrey) — Baseline
         ├── Full Hormone Panel (GH/IGF-1, LH/FSH, TSH/fT4, Cortisol/ACTH, PRL, Electrolytes)
         ├── Visual Acuity + Fundoscopy
         └── Genetic Counselling (If Familial/Syndromic)
         │
         ▼
MANAGEMENT DECISION
         │
         ├── **GROSS TOTAL RESECTION (GTR) IF FEASIBLE**
         │       (No Hypothalamic Involvement; Tumour Accessible)
         │       → Transsphenoidal (If Intrasellar) / Transcranial (Suprasellar)
         │       → **Adjuvant RT if GTR** (To Reduce Recurrence)
         │
         ├── **SUBTOTAL RESECTION (STR) + ADJUVANT RADIOTHERAPY**
         │       (Hypothalamic Involvement / Vascular Encasing / High Risk GTR)
         │       → **Stereotactic Radiosurgery (SRS)** or **Fractionated RT** (54 Gy/30 fractions)
         │
         ├── **CYSTIC DISEASE DOMINANT**
         │       → **Cyst Aspiration / Cystoperitoneal Shunt / Intracystic Bleomycin/Interferon/¹⁸⁹Re**
         │       → RT if Residual Solid Component
         │
         └── **PALLIATIVE / BEST SUPPORTIVE CARE**
                 (Elderly / Comorbid / Advanced Disease / Patient Choice)
                 → Symptom Control (DI, Hormone Replacement, Obesity Management)
         │
         ▼
POST-OP MANAGEMENT
         │
         ├── IMMEDIATE: DI (Desmopressin), Hormone Replacement, Visual Fields, Electrolytes
         ├── ENDOCRINE REPLACEMENT (GH, Thyroxine, Sex Steroids, Hydrocortisone)
         ├── DI MANAGEMENT (Desmopressin; Monitor Sodium)
         ├── HYPOTHALAMIC OBESITY (GLP-1 Agonists, Lifestyle, Bariatric Surgery Consideration)
         └── SURVEILLANCE (MRI 3mo, 6mo, 1yr then Yearly; Endocrine q6mo; Visual Fields q6-12mo)
```

## 3. Surgical Approaches

| Approach | Indication | Advantages |
|---------|------------|------------|
| **Endoscopic Endonasal (Transsphenoidal)** | **Intrasellar / Small Suprasellar** | Minimally Invasive; No Brain Retraction; Good for Cystic Component |
| **Transcranial (Pterional / Subfrontal / Interhemispheric)** | **Large Suprasellar / Third Ventricle / Hypothalamic Involvement** | Better Access to Suprasellar / Third Ventricle / Vessels |
| **Combined (Endonasal + Transcranial)** | Giant / Complex / Extensive Vascular Involvement | Maximum Exposure |

### Extent of Resection
| Goal | Criteria | Recurrence Risk |
|------|---------|-----------------|
| **Gross Total Resection (GTR)** | No Residual Tumour on Post-op MRI | **Lowest** (10-20% at 10yr) |
| **Near-Total (>95%)** | Minimal Residual | Low (20-30% at 10yr) |
| **Subtotal (STR)** | Significant Residual | **High** (50-70% at 10yr) → **Requires Adjuvant RT** |

---

## 4. Adjuvant Radiotherapy

| Indication | Modality | Dose / Schedule |
|------------|----------|-----------------|
| **Post-STR** | **Fractionated RT** (Conformal/IMRT) | **54 Gy / 30 Fractions** (1.8 Gy/Fx) |
| **Small Residual <3cm** | **Stereotactic Radiosurgery (SRS)** | 12-14 Gy (Single Fraction) |
| **Recurrence / Inoperable** | SRS or Fractionated RT | Per Protocol |

| Complication | Incidence | Mitigation |
|--------------|-----------|------------|
| **Hypopituitarism** | **30-50% at 10-15yr** | Conformal Techniques; Dose Constraints to Pituitary/Chiasm |
| **Optic Neuropathy** | <1% (Fractionated); 1-3% (SRS) | Dose Constraint to Optic Chiasm/Nerves |
| **Cognitive Decline** | Variable | Modern Conformal Techniques |
| **Secondary Malignancy** | <1% (Long-term) | Modern Techniques |
| **Cerebral Vasculopathy** | Moyamoya-like (Rare) | Vascular Dose Constraints |

---

## 4. Complications & Long-term Management

### Diabetes Insipidus (DI)
| Timing | Frequency | Management |
|--------|-----------|------------|
| **Pre-op** | 30-50% | Baseline Assessment |
| **Post-op (Acute)** | 60-80% (Transient) | **Desmopressin** (Intranasal/Oral/SC) + Fluid Balance |
| **Permanent** | 10-20% | **Lifelong Desmopressin** + Fluid Management |

### Hypothalamic Obesity
| Feature | Management |
|---------|-----------|
| **Mechanism** | Ventromedial Hypothalamus Damage → Leptin Resistance + Hyperphagia + ↓ Energy Expenditure |
| **Medical** | **GLP-1 RA (Liraglutide/Semaglutide)**; Metformin; Topiramate (Adjunct) |
| **Lifestyle** | Structured Diet, Exercise, Behavioural Therapy |
| **Surgical** | **Bariatric Surgery** (RYGB/Sleeve) — Consider in Severe Refractory Cases |

### Endocrine Deficiencies — Replacement
| Deficiency | Replacement |
|------------|-------------|
| **GH** | Children: RH-GH 0.025-0.035 mg/kg/day SC; Adults: 0.1-0.3 mg/day SC (if Symptomatic) |
| **TSH/fT4** | **Levothyroxine** (Start BEFORE Glucocorticoid) |
| **ACTH/Cortisol** | **Hydrocortisone 15-20mg AM + 5-10mg PM** |
| **Gonadotrophs** | Children: **Puberty Induction** (Low Dose → Adult Dose); Adults: **Testosterone / HRT** |
| **DI** | Desmopressin (Intranasal/Oral/SC) + Fluid Balance |

## 5. Recurrence & Surveillance

| Extent of Resection | 5-Year Recurrence | 10-Year Recurrence | Adjuvant RT |
|-------------------|-------------------|--------------------|-------------|
| **GTR** | 10-20% | 20-30% | Optional (Low Risk) |
| **Near-Total (>95%)** | 20-30% | 30-40% | Often Recommended |
| **STR** | 50-60% | 60-70% | **Mandatory** |

### Surveillance Schedule
| Timepoint | Investigations |
|-----------|---------------|
| **3 Months** | MRI Brain (Contrast), Visual Fields, Full Hormones |
| **6 Months** | MRI Brain, Visual Fields, Hormones (GH/IGF-1, TSH/fT4, Cortisol, LH/FSH) |
| **1 Year** | MRI Brain (Contrast), Visual Fields, Full Hormones |
| **Annually (Years 2-5)** | MRI Brain (Contrast), Visual Fields, Hormones |
| **Years 5-10** | MRI + Hormones + Visual Fields (Every 1-2 Years) |
| **>10 Years** | 2-3 Yearly MRI + Hormones + Visual Fields |

## 6. Special Complications

### Hypothalamic Obesity
| Feature | Details |
|---------|---------|
| **Pathophysiology** | Ventromedial Hypothalamus Damage → Leptin Resistance + Hyperphagia + ↓ Energy Expenditure |
| **Onset** | Rapid (Months Post-Surgery/RT) |
| **Management** | GLP-1 RA (Liraglutide/Semaglutide), Metformin, Topiramate, Bariatric Surgery (Refractory) |

### Vasculopathy
| Type | Features |
|------|----------|
| **Moyamoya-like Vasculopathy** | Post-RT (5-10yr); Stenosis of Circle of Willis; Ischaemic Strokes |
| **Cavernous Sinus / Vascular Injury** | Post-Surgical (Rare) |

### Secondary Malignancy
| Risk | Timeline |
|------|----------|
| **Secondary Glioma / Sarcoma** | **10-20 Years Post-RT**; <1% Cumulative Incidence |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Craniopharyngioma vs Pituitary Adenoma** | Craniopharyngioma = Suprasellar, Calcified, Cystic; Adenoma = Intrasellar, Enhancing, No Calcification |
| **Craniopharyngioma vs Rathke's Cyst** | Cranio = Solid+Cystic + Calcifications; Rathke's = Pure Cystic, Midline, No Calcification |
| **Adamantinous vs Papillary** | Adamantinous = Child + Calcifications + Wet Keratin; Papillary = Adult + No Calcification + Papillary Architecture |
| **Craniopharyngioma vs Hypothalamic Glioma** | Cranio = Cystic + Calcifications + Enhancing Nodules; Glioma = Solid + Infiltrative + No Calcification |
| **GTR vs STR** | GTR = Curative Intent (10-20% Recurrence); STR = Requires RT (50-70% Recurrence without RT) |
| **Post-op DI** | Transient in 60-80%; Permanent 10-20% → Lifelong Desmopressin |

**Mnemonic: CRANIO**
- **C**alcifications (>90% Adamantinous)
- **R**athke's Pouch Origin
- **A**damantinous (Child) vs **P**apillary (Adult)
- **N**ear Optic Chiasm → Bi-temporal Hemianopia
- **I**ntracystic "Motor Oil" Fluid (High Protein)
- **O**besity (Hypothalamic) + **D**iabetes Insipidus + Growth Failure

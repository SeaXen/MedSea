# Phaeochromocytoma & Paraganglioma

> [!info]
> **Phaeochromocytoma = Catecholamine-Secreting Tumour of Adrenal Medulla.** **Paraganglioma = Extra-Adrenal (Sympathetic/Parasympathetic).** "Rule of 10s" (Classic Teaching). **Alpha-Blockade BEFORE Beta-Blockade** — Mandatory Pre-Op Prep. **Genetic Testing Mandatory** (30-40% Hereditary).

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise clinical presentation (paroxysmal hypertension, headaches, sweating, palpitations)
- [ ] Apply diagnostic criteria (plasma free metanephrines, 24h urine metanephrines)
- [ ] Apply localisation imaging (CT/MRI, ⁶⁸Ga-DOTATATE PET, ¹²³I-MIBG)
- [ ] Execute pre-operative alpha-blockade protocol (phenoxybenzamine)
- [ ] Indicate genetic testing and recognise hereditary syndromes (MEN2, VHL, NF1, SDHx)
- [ ] Differentiate phaeochromocytoma vs paraganglioma

## 2. Epidemiology & "Rule of 10s" (Classic Teaching)

| Rule of 10s | Detail |
|-------------|--------|
| **10% Bilateral** | Adrenal Phaeochromocytoma |
| **10% Malignant** | Phaeochromocytoma; Paraganglioma Higher (30-50%) |
| **10% Extra-Adrenal** | Paraganglioma (Sympathetic Chain, Organ of Zuckerkandl) |
| **10% Familial** | **Actually 30-40%** (MEN2, VHL, NF1, SDHx, TMEM127, MAX) |
| **10% Pediatric** | <18 Years |
| **10% Extra-Abdominal** | Paraganglioma (Head/Neck) |

> **Modern View**: "Rule of 10s" is an Approximation; **Heritability 30-40%** is Key Takeaway.

## 2. Clinical Presentation

### Classic Triad (Episodic)
| Symptom | Frequency | Mechanism |
|---------|-----------|-----------|
| **Headache** | **90%** | Severe, Throbbing, Often Precipitated |
| **Sweating** | **70-80%** | Profuse, Generalised |
| **Palpitations** | **60-70%** | Tachycardia, Arrhythmias |

### Paroxysmal vs Sustained Hypertension
| Pattern | Frequency | Features |
|---------|-----------|----------|
| **Paroxysmal** | **50-60%** |Episodic SBP >200, Duration Minutes-Hours |
| **Sustained** | 30-40% | Persistent HTN, Paroxysms Superimposed |
| **Normotensive** | 5-10% | Normotensive Between Spells |

### Additional Features
| System | Features |
|--------|----------|
| **Neurological** | Tremor, Anxiety, Panic Attacks, Tremor |
| **GI** | Nausea, Vomiting, Abdominal Pain, Constipation |
| **Dermatological** | Flushing, Pallor (Post-Attack) |
| **Ophthalmic** | Blurred Vision, Hypertensive Retinopathy |
| **Metabolic** | **Hypoglycaemia** (Rare), Hyperglycaemia, Weight Loss |

## 3. Diagnosis — Biochemical Confirmation

### Gold Standard: Plasma Free Metanephrines
| Test | Sensitivity | Specificity | Cut-off |
|------|-------------|-------------|---------|
| **Plasma Free Metanephrines** | **99%** | **89%** | **Metanephrine ≥ Upper Limit** OR **Normetanephrine ≥ Upper Limit** |
| **24h Urine Metanephrines** | 95% | 95% | Metanephrine + Normetanephrine >ULN |

### Sample Collection Protocol (Critical)
| Requirement | Details |
|-------------|---------|
| **Fasting** | **≥8 Hours** (Preferably Morning) |
| **Position** | **Supine ≥30 Minutes** Prior |
| **Medications to Avoid** | **L-DOPA, SSRIs, TCAs, MAOIs, Cocaine, Amphetamines, Sympathomimetics** (≥7 Days) |
| **Drugs Causing False Positives** | L-DOPA, Metoclopramide, Antidepressants (TCA, SSRI), Cocaine, Amphetamines, Buspirone, Reserpine |

### 24h Urine Metanephrines (Alternative)
| Collection | 24h Urine + Acid Preservative (HCl) |
|------------|-------------------------------------|
| **Advantages** | Less Affected by Acute Stress; Integrates Over 24h |
| **False Negatives** | Incomplete Collection, Acidification Failure |

## 3. Localisation Imaging

### First-Line: CT/MRI Adrenal
| Modality | Protocol |
|----------|---------|
| **CT (Adrenal Protocol)** | 1mm Slices, Arterial + Venous Phase, Non-Contrast for HU |
| **MRI (Adrenal Protocol)** | T1/T2, In/Out Phase (Chemical Shift), Post-Gadolinium |

### Functional Imaging (Gold Standard for Localisation)
| Modality | Sensitivity | Specificity | Indication |
|----------|-------------|-------------|------------|
| **⁶⁸Ga-DOTATATE PET/CT** | **90-95%** | **90%+** | **First-Line** (SSTR2+) |
| **¹²³I-MIBG Scintigraphy** | 80-90% | 80-90% | If DOTATATE Unavailable; MIBG-Avid Tumours |
| **¹⁸F-FDG PET** | High (Malignant) | Lower | SCLC, Poorly Differentiated NET |
| **Octreoscan (¹¹¹In-Octreotide)** | 80-85% | 80% | If DOTATATE/MIBG Unavailable |

## 4. Genetic Testing — Mandatory (30-40% Hereditary)

| Syndrome | Gene | Phaeo Features | Other Features |
|----------|------|----------------|----------------|
| **MEN2A** | **RET** (Codon 634) | **Phaeo 50%** (Often Bilateral), MTC 95%, pHPT 20-30% | |
| **MEN2B** | **RET M918T** (95%) | Phaeo 50%, MTC 100% | Marfanoid, Mucosal Neuromas, Café-au-lait |
| **VHL** | **VHL** | Phaeo (Often Bilateral), RCC, Haemangioblastomas (CNS/Retina) | |
| **NF1** | **NF1** | Phaeo (Unilateral), Café-au-lait Spots, Neurofibromas | |
| **SDHB** | **SDHB** | **PGL (Extra-Adrenal), High Malignancy (30-50%)**, GIST, RCC | |
| **SDHD** | **SDHD** | **HNPGL (Head/Neck PGL), Phaeo** | |
| **SDHA/SDHC/SDHC** | **SDHA/C** | Variable | |
| **TMEM127/MAX** | **TMEM127/MAX** | Phaeo/PGL | |

### Genetic Testing Indications
| Indication | Action |
|-----------|--------|
| **All Phaeo/PGL Patients** | **Mandatory** (30-40% Hereditary) |
| **Bilateral/Multifocal** | High Probability Genetic |
| **Family History** | Strong Indication |
| **Young Age (<40)** | Higher Genetic Yield |
| **Malignant Phaeo** | SDHB Suspicion High |

## 4. Pre-Operative Preparation — Mandatory Protocol

### Golden Rule: **ALPHA-BLOCKADE BEFORE BETA-BLOCKADE**

### Alpha-Blockade Protocol (Phenoxybenzamine)
| Step | Dose | Target |
|------|------|--------|
| **Starting** | **10mg BD** (Oral) | — |
| **Titration** | Increase by **10mg BD every 2-3 Days** | SBP **<130/80 Supine**, **<120/70 Standing** |
| **Maintenance** | **20-80mg/day** (Divided BD/TDS) | HR **60-80 bpm** |
| **Duration** | **≥2 Weeks** (Ideally 3-4 Weeks) | Before Surgery |

### Beta-Blockade (ONLY After Alpha-Blockade Established)
| Agent | Dose | Timing |
|-------|------|--------|
| **Propranolol** | 20-40mg TDS | **AFTER** Alpha-Blockade Established |
| **Atenolol/Metoprolol** | 50-100mg OD/BD | **After** Alpha-Blockade Adequate |

### Volume Expansion
| Intervention | Details |
|------------|---------|
| **IV Fluids** | **2-3L 0.9% NaCl/Day** (Start with Alpha-Blockade) |
| **Goal** | **CVP 8-12 mmHg**, UOP >0.5 mL/kg/h |
| **Monitor** | Na+, K+, Glucose, CVP, UOP q1-2h |

## 4. Paraganglioma (Extra-Adrenal) — Key Differences

| Feature | **Phaeochromocytoma (Adrenal)** | **Paraganglioma (Extra-Adrenal)** |
|---------|----------------------------------|-----------------------------------|
| **Location** | Adrenal Medulla | **Sympathetic Chain** (Organ of Zuckerkandl — Most Common) |
| **Malignancy Rate** | **~10%** | **30-50%** (SDHB Highest) |
| **Malignancy Definition** | **Metastases** (Non-Chromaffin Tissue) | **Metastases** (Bone, Liver, Lung, Lymph Nodes) |
| **Genetics** | MEN2, VHL, NF1, SDHx | **SDHB (High Malignancy), SDHD (HNPGL)** |
| **Imaging** | Adrenal CT/MRI | **Whole Body** (Neck to Pelvis) |

### Head & Neck Paraganglioma (HNPGL)
| Feature | Details |
|---------|---------|
| **Location** | Carotid Body, Jugulotympanic, Vagal, Laryngeal |
| **Gene** | **SDHD** (Maternal Imprinting — Paternal Transmission), **SDHC**, **SDHAF2** |
| **Malignancy** | **Rare (<5%)** |
| **Symptoms** | Mass Effect (Cranial Nerve Palsies, Tinnitus, Dysphagia) |
| **Treatment** | Surgery ± Radiotherapy (Stereotactic) |

## 4. Pre-Operative Optimisation Checklist

| Parameter | Target | Timeline |
|-----------|--------|----------|
| **Alpha-Blockade** | Phenoxybenzamine 20-80mg/day | ≥2 Weeks (Ideally 3-4 Weeks) |
| **BP Control** | Supine <130/80; Standing <120/70 | Achieved Before Surgery |
| **Heart Rate** | 60-80 bpm (Add Beta-Blocker After Alpha) | After Alpha-Blockade |
| **Volume Status** | Euvolaemic (CVP 8-12, UOP >0.5 mL/kg/h) | At Surgery |
| **Glucose** | Normoglycaemia (Insulin Sliding Scale if Diabetic) | Peri-operative |
| **Electrolytes** | K+ 3.5-5.0 (Correct Hypokalaemia Pre-Op) | Pre-Op |

## 5. Surgical Management

### Laparoscopic Adrenalectomy — Gold Standard
| Approach | Indication |
|----------|------------|
| **Laparoscopic Transabdominal** | **Standard** (<8-10cm, No Invasion) |
| **Posterior Retroperitoneoscopic (PRA)** | Small Tumours, Prior Abdominal Surgery |
| **Open Adrenalectomy** | >10cm, Suspected Carcinoma, Invasion, Prior Surgery |

### Intraoperative Management
| Challenge | Management |
|-----------|------------|
| **Tumour Manipulation** | **Catecholamine Surge** → Deepen Anaesthesia, Esmolol/Nitroprusside |
| **Venous Sampling** | Adrenal Vein Sampling (If Bilateral/Uncertain Laterality) |
| **Bilateral Phaeo** | Staged (6-8 Weeks Apart) or Simultaneous (Experienced Centre) |

### Post-Operative Care
| Complication | Monitoring |
|-------------|------------|
| **Hypotension** | Vasopressors (Norepinephrine), Fluid Resuscitation |
| **Hypoglycaemia** | Glucose Monitoring q1-2h (First 24h) |
| **Adrenal Insufficiency** | Hydrocortisone 50-100mg IV q6h (If Bilateral/Contra-lateral Suppressed) |

## 5. Malignancy & Follow-up

### Malignancy Criteria (No Histological Benign/Malignant)
| Criterion | Definition |
|-----------|------------|
| **Metastases** | Non-Chromaffin Tissue (Bone, Liver, Lung, Lymph Nodes) |
| **Local Invasion** | Adjacent Organ Invasion |
| **Recurrence** | Local/Distant Recurrence Post-Resection |

### SDHB Mutation Carriers — High Malignancy Risk
| Feature | Details |
|--------|---------|
| **PGL (Extra-Adrenal)** | **Most Common** in SDHB Carriers |
| **Malignancy Rate** | **30-50%** (Highest of All SDHx) |
| **Surveillance** | **Annual ⁶⁸Ga-DOTATATE PET/CT** + Biochemical |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Phaeo vs Paraganglioma** | Phaeo = Adrenal; PGL = Extra-Adrenal (Sympathetic Chain/Organ of Zuckerkandl) |
| **Malignancy** | **Metastases to Non-Chromaffin Tissue** = Malignant (Not Size/Histology) |
| **Alpha vs Beta Blockade** | **Alpha FIRST** (Phentolamine/Phenoxybenzamine); Beta ONLY After Alpha |
| **Beta-Blocker First** | **DANGEROUS** → Unopposed Alpha → Hypertensive Crisis |
| **SDHB vs SDHD** | SDHB = Malignant PGL (Abdomen/Pelvis); SDHD = HNPGL (Head/Neck) |
| **MEN2A vs MEN2B** | MEN2A: MTC + Phaeo + pHPT; MEN2B: MTC + Phaeo + Marfanoid/Neuromas |
| **MIBG vs DOTATATE** | DOTATATE Superior Sensitivity (90-95% vs 80-90%); MIBG for MIBG-Avid Tumours |
| **Genetic Testing** | **Mandatory for ALL** (30-40% Hereditary) — Not Just Syndromic |

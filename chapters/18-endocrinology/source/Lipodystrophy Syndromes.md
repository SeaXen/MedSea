# Lipodystrophy Syndromes

> [!info]
> **Lipodystrophy = Heterogeneous Group of Disorders Characterised by Selective Loss of Adipose Tissue.** **Congenital (Genetic) vs Acquired; Generalised vs Partial.** **Severe Metabolic Complications: Insulin Resistance, Diabetes, Hypertriglyceridaemia, Hepatic Steatosis, Pancreatitis, Cardiomyopathy.** **Leptin Replacement (Metreleptin) = Disease-Modifying Therapy for Generalised Forms.**

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Classify lipodystrophy (Generalised vs Partial, Congenital vs Acquired)
- [ ] Recognise clinical features and metabolic complications
- [ ] Apply metabolic monitoring and management strategies
- [ ] Prescribe metreleptin (leptin replacement) for eligible patients
- [ ] Counsel on genetic testing and family screening

## 2. Classification

| Category | Subtypes |
|----------|----------|
| **Congenital Generalised Lipodystrophy (CGL)** | **CGL1 (AGPAT2)**, **CGL2 (BSCL2)**, **CGL3 (CAV1)**, **CGL4 (PTRF/CAVIN1)** |
| **Acquired Generalised Lipodystrophy (AGL)** | **Lawrence Syndrome** (Autoimmune, Post-Infectious, Idiopathic) |
| **Familial Partial Lipodystrophy (FPLD)** | **FPLD1 (Unknown)**, **FPLD2 (LMNA)**, **FPLD3 (PPARG)**, **FPLD4 (PLIN1)**, **FPLD5 (CIDEC)**, **FPLD6 (LIPE)** |
| **Acquired Partial Lipodystrophy (APL)** | **Barraquer-Simons Syndrome** (Complement 3 Nephritic Factor) |
| **HIV-Associated Lipodystrophy** | ART-Related (NRTI, PI) — Lipoatrophy (Face, Limbs) ± Lipohypertrophy (Visceral) |

## 2. Major Subtypes — Clinical Features

### Congenital Generalised Lipodystrophy (CGL) — Berardinelli-Seip Syndrome
| Feature | CGL1 (AGPAT2) | CGL2 (BSCL2) | CGL3 (CAV1) | CGL4 (PTRF) |
|---------|--------------|--------------|-------------|-------------|
| **Adipose Loss** | **Near-Total (Birth/Infancy)** | Near-Total | Near-Total | Near-Total |
| **Metabolic Severity** | **Severe** | **Most Severe** | Severe | Severe |
| **Additional Features** | Hepatomegaly, Acromegaloid | **Mental Retardation**, Cardiomyopathy | Cardiac, Rhabdomyolysis | Myopathy, Pyloric Stenosis |
| **Leptin** | **Very Low/Undetectable** | Very Low | Low | Low |

### Familial Partial Lipodystrophy Type 2 (FPLD2, Dunnigan Variety)
| Feature | Details |
|---------|---------|
| **Gene** | **LMNA** (Lamin A/C) — **AD** |
| **Fat Loss** | **Limbs, Gluteal, Truncal** (Puberty Onset) |
| **Fat Accumulation** | **Face, Neck, Intra-Abdominal (Cushingoid)** |
| **Metabolic** | **Severe Insulin Resistance**, Diabetes, HyperTG, PCOS (Women) |
| **Cardiac** | **Conduction Defects, Cardiomyopathy** (LMNA Cardiac Phenotype) |
| **Muscular** | Muscular Dystrophy (Some) |

### Familial Partial Lipodystrophy Type 3 (FPLD3)
| Feature | Details |
|---------|---------|
| **Gene** | **PPARG** (Peroxisome Proliferator-Activated Receptor Gamma) — **AD** |
| **Fat Loss** | Limbs, Gluteal |
| **Metabolic** | Early-Onset Diabetes, Severe IR, Hypertension |
| **Other** | Acanthosis Nigricans, PCOS |

### Acquired Generalised Lipodystrophy (AGL) — Lawrence Syndrome
| Feature | Details |
|---------|---------|
| **Onset** | Childhood/Adolescence (After Normal Infancy) |
| **Trigger** | **Autoimmune** (Panniculitis), **Infections** (Measles, Varicella), **Idiopathic** |
| **Fat Loss** | **Progressive, Generalised** (Including Face, Palms, Soles) |
| **Autoimmune** | **High Association** (SLE, RA, Autoimmune Hepatitis, Low C3/C4) |
| **Metabolic** | Severe IR, Diabetes, HyperTG, Hepatic Steatosis |

### Acquired Partial Lipodystrophy (APL) — Barraquer-Simons Syndrome
| Feature | Details |
|---------|---------|
| **Onset** | Childhood/Adolescence |
| **Fat Loss** | **Cephalocaudal** (Face, Neck, Arms, Trunk) → **Lower Body Spared** |
| **Gender** | **Female > Male (4:1)** |
| **Complement** | **C3 Nephritic Factor (C3NeF)** → Low C3, Normal C4 |
| **Renal** | **Membranoproliferative GN (MPGN) Type 1** (20-25%) |
| **Autoimmune** | Associated Autoimmunity (Precedes MPGN) |

### HIV-Associated Lipodystrophy
| Feature | Details |
|---------|---------|
| **Lipoatrophy** | Face, Limbs, Buttocks (NRTI: Stavudine, Zidovudine, Didanosine) |
| **Lipohypertrophy** | Visceral Abdomen, Dorsocervical Fat Pad (Buffalo Hump), Breasts (PI: Lopinavir, Indinavir) |
| **Metabolic** | Insulin Resistance, Diabetes, HyperTG, Low HDL |
| **Management** | Switch ART (Avoid Stavudine/Zidovudine); Tesamorelin (GH-RH Analogue) for Visceral Fat |

## 3. Metabolic Complications (Universal)

| Complication | Mechanism | Management |
|-------------|-----------|------------|
| **Severe Insulin Resistance** | ↓ Adipose Tissue → ↓ Leptin → ↓ Insulin Sensitivity | **Metreleptin** (If Generalised); High-Dose Insulin; Metformin; TZDs (Cautious) |
| **Diabetes Mellitus** | Insulin Resistance + β-Cell Failure | **High-Dose Insulin** (Often >2 U/kg/day); Metreleptin Improves Glycaemia |
| **Hypertriglyceridaemia** | Impaired Clearance + Hepatic Overproduction | **Fibrates** (Fenofibrate), Omega-3, **Metreleptin** ↓ TG; Avoid Alcohol/Oestrogen |
| **Acute Pancreatitis** | Severe HyperTG (>10-20 mmol/L) | **Aggressive TG Lowering**; Insulin Infusion; Plasmapheresis (Emergency) |
| **Hepatic Steatosis / NASH** | Ectopic Fat Deposition | **Weight Loss** (If Partial); Metreleptin ↓ Liver Fat; Pioglitazone (Cautious) |
| **Cardiomyopathy** | Lipotoxicity, Metabolic Stress | Echo Surveillance; Metreleptin May Improve; Standard HF Therapy |
| **PCOS / Hyperandrogenism (Women)** | Insulin Resistance → Ovarian Theca Cell Stimulation | COCP, Metformin, Weight Loss (If Partial) |
| **Acanthosis Nigricans** | Insulin → IGF-1 Receptor Cross-Activation | Marker of Severe IR |

## 3. Leptin Replacement — Metreleptin

| Aspect | Details |
|--------|---------|
| **Indication** | **Generalised Lipodystrophy** (CGL, AGL) with **Metabolic Complications** |
| **Mechanism** | **Recombinant Human Leptin** → Restores Leptin Signalling → ↑ Insulin Sensitivity, ↓ Hepatic Gluconeogenesis, ↓ Lipogenesis |
| **Dose** | **0.06-0.12 mg/kg/day SC** (Start Low, Titrate by 0.02 mg/kg q2-4wk) |
| **Max Dose** | 10mg/day (Women), 5mg/day (Men) |
| **Efficacy** | **HbA1c ↓ 1.5-2.5%**; **TG ↓ 30-60%**; **Liver Fat ↓ 30-50%** |
| **Monitoring** | HbA1c, TG, Liver Fat (MRI/MRE), Weight, Anti-Leptin Antibodies |
| **Adverse Effects** | **Anti-Leptin Antibodies** (Rare Neutralising), Injection Site Reactions, Hypoglycaemia (With Insulin), T-Cell Lymphoma Risk (Theoretical) |
| **Contraindications** | **Generalised Obesity** (Not Indicated); Pregnancy (Category C) |

## 4. Metabolic Management Algorithm

```
Lipodystrophy Diagnosed
         │
         ▼
ASSESS: Type (Generalised vs Partial), Metabolic Severity
         │
         ├── GENERALISED (CGL, AGL)
         │       ├── METRELEPTIN INDICATED
         │       │       0.06 mg/kg/day SC → Titrate
         │       │       Target: HbA1c <53, TG <2.3, Liver Fat ↓
         │       │
         │       ├── DIABETES
         │       │       High-Dose Insulin (Often 1-2+ U/kg/day)
         │       │       Metformin (Adjunct)
         │       │       TZDs (Cautious — Weight Gain, HF Risk)
         │       │
         │       ├── HYPERTRIGLYCERIDAEMIA
         │       │       Fenofibrate 145-200mg/day
         │       │       Omega-3 (2-4g/day)
         │       │       Metreleptin (Primary)
         │       │       Emergency: TG >20 mmol/L → Insulin Infusion ± Plasmapheresis
         │       │
         │       └── HEPATIC STEATOSIS
         │               Ultrasound/MRI q1-2yr; Metreleptin; Pioglitazone (If Indicated)
         │
         └── PARTIAL (FPLD, APL)
                 ├── LIFESTYLE (Weight Maintenance/Reduction)
                 ├── METFORMIN (1st Line for IR)
                 ├── TZDs (Pioglitazone) — **If No HF** (↑ Adipogenesis in Remaining Depots)
                 ├── SGLT2i / GLP-1 RA (Cardio-Renal Benefit)
                 ├── INSULIN (If Uncontrolled)
                 └── METRELEPTIN — **Not Routine** (Consider if Severe Metabolic Despite Above)
```

## 5. Monitoring & Surveillance

| Parameter | Frequency | Target |
|-----------|-----------|--------|
| **HbA1c** | **q3 Months** | **<48-53 mmol/mol (6.5-7.0%)** |
| **Fasting Lipids (TG, HDL, LDL)** | **q3-6 Months** | **TG <2.3 mmol/L**; HDL >1.0 |
| **Liver Function (ALT, AST, GGT)** | **q3-6 Months** | Normal |
| **Liver Fat (MRI-PDFF / FibroScan)** | **Annual** | ↓ Trend |
| **Cardiac Echo** | **Annual (CGL/FPLD2)** | LV Function; Conduction (FPLD2) |
| **Renal Function** | **q6 Months** | Normal (APL: Monitor for MPGN) |
| **Complement (C3, C4, C3NeF)** | **q6-12 Months (APL)** | C3 Normalisation |
| **Anti-Leptin Antibodies** | **q6-12 Months (On Metreleptin)** | Non-Neutralising |
| **Bone Density (DEXA)** | **q2-3 Years** | T-Score >-2.5 |

## 5. Special Considerations

### Pregnancy in Lipodystrophy
| Aspect | Management |
|--------|------------|
| **Metreleptin** | **Contraindicated** (Category C); Stop Pre-Conception |
| **Diabetes** | **Insulin Intensification** (Basal-Bolus); Targets: FBG <5.3, 1h <7.8, 2h <6.4 |
| **HyperTG** | **Fenofibrate** (Stop if Pregnant); Omega-3; **Insulin Infusion if TG >10 mmol/L** |
| **Delivery** | Multidisciplinary (Endo, OB, Cardiology if Cardiac Involvement) |

### Paediatric Considerations
| Aspect | Management |
|--------|------------|
| **Growth** | Monitor Height Velocity; Metreleptin May ↑ Growth Velocity |
| **Puberty** | Delayed in CGL; Monitor LH/FSH/Sex Steroids |
| **Metreleptin Dosing** | Weight-Based (0.06-0.12 mg/kg); Adjust q3mo for Growth |
| **Cardiac** | Echo q6-12mo (CGL2/BSCL2 High Cardiac Risk) |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Generalised vs Partial** | Generalised = Near-Total Fat Loss (CGL, AGL); Partial = Regional (FPLD, APL) |
| **CGL vs AGL** | CGL = Congenital (Birth); AGL = Acquired (Childhood/Adolescence) |
| **FPLD2 vs FPLD3** | FPLD2 = LMNA (Cardiac Risk); FPLD3 = PPARG (Early DM/HTN) |
| **APL vs FPLD** | APL = Acquired, Cephalocaudal, Female Predominant, C3NeF; FPLD = Genetic, AD |
| **Metreleptin Indication** | **Generalised Lipodystrophy ONLY** (Not Partial, Not Obesity) |
| **Metreleptin in Obesity** | **Contraindicated** (Leptin Resistance, Not Deficiency) |
| **FPLD2 Cardiac** | **LMNA** → Conduction Defects, Cardiomyopathy (Not in Other FPLD) |
| **APL Renal** | **C3NeF + Low C3** → **MPGN (20-25%)** |
| **HIV Lipodystrophy** | NRTI (Stavudine) → Lipoatrophy; PI → Lipohypertrophy |
| **Leptin Deficiency vs Lipodystrophy** | Leptin Deficiency = **Monogenic (LEP)**; Lipodystrophy = **Fat Loss → Low Leptin** |
| **CGL2 (BSCL2)** | **Neurodevelopmental Delay + Cardiomyopathy** (Unique) |
| **ACL vs APL** | AGL = Generalised; APL = Partial (Cephalocaudal) |
| **Metreleptin Max Dose** | 10mg/day (Women), 5mg/day (Men) |

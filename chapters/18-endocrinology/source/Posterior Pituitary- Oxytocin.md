# Posterior Pituitary: Oxytocin

> [!info]
> **Oxytocin = Hormone of parturition and lactation.** Synthesised in hypothalamic paraventricular nucleus, released from posterior pituitary. Unique for its **positive feedback loop** during labour. Synthetic oxytocin (Syntocinon) widely used in obstetrics.

## 1. Oxytocin Physiology

### Synthesis & Secretion
| Feature | Details |
|---------|---------|
| **Synthesis** | Hypothalamic **Paraventricular Nucleus (PVN)** (also some in Supraoptic) |
| **Transport** | Axoplasmic transport → Posterior Pituitary (Neurohypophysis) |
| **Structure** | **9 amino acid cyclic peptide** (Cys-Tyr-Ile-Gln-Asn-Cys-Pro-Leu-Gly) with disulphide bond |
| **Half-life** | **3-5 minutes** (very short) |
| **Main Stimuli** | **Cervical stretch** (Ferguson reflex), **Suckling** (Milk ejection reflex), Sexual activity, Stress |

### Oxytocin Receptor (OTR)
| Feature | Details |
|---------|---------|
| **Type** | Gq/11-coupled receptor (Oxytocin Receptor, OTR) |
| **Signalling** | Gq → PLC → IP3/DAG → ↑ Intracellular Ca²⁺ → Contraction |
| **Main Targets** | **Uterus** (Myometrial contraction), **Breast** (Myoepithelial cells → Milk ejection), Brain (Social bonding, Trust, Anxiety modulation) |
| **Upregulation** | **Oestrogen** ↑ OTR expression (Late pregnancy); **Progesterone** ↓ OTR |

## 2. Oxytocin in Labour — Positive Feedback Loop

### The Ferguson Reflex (Positive Feedback)
```
Cervical Stretch (Fetal Descent)
         ↓
Afferent Signals → Spinal Cord → Brainstem → Hypothalamus (PVN)
         ↓
**Oxytocin Release** (Posterior Pituitary)
         ↓
Uterine Contraction (via OTR → ↑ Intracellular Ca²⁺)
         ↓
**Further Cervical Stretch** → Positive Feedback Loop
         ↓
**Exponential Rise** in Oxytocin & Contraction Frequency/Intensity
         ↓
**Delivery** → Stretch Ceases → Feedback Loop Breaks
```

### Key Features of Parturition Oxytocin
| Feature | Details |
|---------|---------|
| **Trigger** | Cervical stretch (Fetal head descent) |
| **Release Pattern** | **Pulsatile**, increasing frequency & amplitude |
| **OTR Upregulation** | **Oestrogen** ↑ OTR expression (Late pregnancy); Progesterone withdrawal removes inhibition |
| **Positive Feedback** | **Rare example** of physiological positive feedback; Drives labour to completion |
| **Termination** | Delivery → Stretch removed → Afferent input stops → Oxytocin falls |

## 2. Oxytocin in Lactation — Milk Ejection Reflex

### Milk Ejection Reflex (Let-down)
```
Suckling (Nipple Stimulation)
         ↓
Afferent → Spinal Cord → Brainstem → Hypothalamus (PVN)
         ↓
**Oxytocin Release** (Posterior Pituitary)
         ↓
**Myoepithelial Cell Contraction** (Breast Alveoli/Ducts)
         ↓
**Milk Ejection** (Milk flows from alveoli → Ducts → Nipple)
         ↓
**Sustained Suckling** → Pulsatile Oxytocin → Multiple Let-downs per Feed
```

### Key Features of Lactation Oxytocin
| Feature | Details |
|---------|---------|
| **Trigger** | **Suckling** (Nipple stimulation) → Afferent via sympathoadrenal pathway |
| **Release Pattern** | **Pulsatile** (Multiple let-downs per feed) |
| **Action** | **Myoepithelial Cell Contraction** (Alveoli/Ducts) → Milk ejection |
| **Prolactin Synergy** | Prolactin = Milk **Synthesis** (Lactogenesis); Oxytocin = Milk **Ejection** (Galactopoiesis) |
| **Conditioned Reflex** | Mother's thought/baby's cry → Oxytocin release (Learned) |

## 3. Synthetic Oxytocin (Syntocinon) — Clinical Use

### Indications
| Indication | Regimen | Monitoring |
|------------|---------|------------|
| **Induction of Labour** | Start **0.5-2 mU/min** IV → Increase by 1-2 mU/min q30-60min | **CTG Continuous**; Uterine Activity (IUPC); Max 20 mU/min |
| **Augmentation of Labour** | Start **1-2 mU/min** → Titrate to **3-4 contractions/10min** | Same as Induction |
| **Postpartum Haemorrhage (PPH)** | **5-10 IU IV Bolus** → **10-40 IU in 500mL NS** at 125 mL/h | Uterine Tone; Vaginal Bleeding; Vitals |
| **Retained Placenta** | 10-20 IU IV/IM | Uterine Contraction |
| **Incomplete Abortion** | 5-10 IU IV | Uterine Evacuation |

### Syntocinon Dosing Protocol (IV Infusion)
| Step | Rate | Titration |
|------|------|-----------|
| **Start** | **0.5-2 mU/min** | - |
| **Increment** | **1-2 mU/min every 30-60 min** | Until 3-4 contractions/10min |
| **Maintenance** | **8-16 mU/min** (Typical) | Max **20 mU/min** |
| **Max Rate** | **20 mU/min** | Higher → Hyperstimulation Risk |

### Oxytocin Dilution (Standard)
| Preparation | Concentration |
|-------------|---------------|
| **10 IU in 500mL Normal Saline** | **20 mU/mL** (1 mL/min = 20 mU/min) |
| **5 IU in 500mL Normal Saline** | **10 mU/mL** |

## 3. Oxytocin — Adverse Effects & Contraindications

| Adverse Effect | Mechanism | Management |
|--------------|-----------|------------|
| **Uterine Hyperstimulation** | Excessive contractions (>5/10min) + ↓ Resting Tone | **Stop/Reduce Oxytocin**; Tocolytic (Terbutaline/Atosiban) |
| **Fetal Distress** | Uteroplacental insufficiency | Stop Oxytocin; Left lateral; O₂; Consider C-section |
| **Water Intoxication / Hyponatraemia** | Oxytocin V2 effect (ADH-like) + Large fluid volumes | Limit fluids; Monitor Na+; Stop Oxytocin if severe |
| **Postpartum Hypertension** | Vasoconstriction (V1-like at high doses) | Monitor BP; Antihypertensives if needed |
| **Amniotic Fluid Embolism** | Rare, catastrophic | Emergency C-section, Supportive care |

### Contraindications to Oxytocin
| Absolute | Relative |
|----------|---------|
| **Cephalopelvic Disproportion** | Grand Multiparity (>5) |
| **Malpresentation** (Transverse, Brow) | Previous Uterine Surgery (Previous C-section — relative) |
| **Placenta Praevia** | **Multiple Pregnancy** |
| **Active Genital Herpes** | **Previous Uterine Rupture** |
| **Fetal Distress** (Pre-existing) | **Grand Multiparity** |

## 4. Atosiban — Oxytocin Receptor Antagonist (Tocolytic)

| Feature | Details |
|---------|---------|
| **Mechanism** | **Competitive Oxytocin Receptor Antagonist** (OTR) |
| **Indication** | **Acute Tocolysis** for Preterm Labour (24-33 weeks) |
| **Dosing** | **Loading**: 6.75mg IV over 15 min → **Maintenance**: 18mg/h IV for 3h → 6mg/h IV up to 45h (Max 72h total) |
| **Advantages** | No β-agonist side effects (Tachycardia, Hypokalaemia, Hyperglycaemia); No cardiac contraindications |
| **Advantages vs β-agonists** | Safe in cardiac disease, Diabetes, Hyperthyroidism |
| **Contraindications** | >33 weeks, Placental Abruption, Chorioamnionitis, Severe Pre-eclampsia, Fetal Distress |

### Atosiban vs β-agonists (Salbutamol/Terbutaline)
| Feature | Atosiban | Salbutamol/Terbutaline |
|---------|----------|------------------------|
| **Mechanism** | OTR Antagonist | β₂-Agonist |
| **Tachycardia** | **No** | **Yes** (Dose-limiting) |
| **Hypokalaemia** | **No** | **Yes** |
| **Hyperglycaemia** | **No** | **Yes** |
| **Maternal HR** | No change | ↑↑ |
| **Cardiac Contraindications** | None | IHD, Arrhythmias, HTN |

## 4. Oxytocin in Postpartum Haemorrhage (PPH)

### WHO/FIGO Recommendations
| Scenario | Regimen |
|---------|---------|
| **Prophylaxis (Active Management 3rd Stage)** | **10 IU IM** (or **5 IU IV**) immediately after delivery |
| **Treatment (Uterine Atony PPH)** | **10 IU IV Bolus** → **20-40 IU in 500-1000mL NS** at 125-250 mL/h |
| **Refractory PPH** | Add Ergometrine (0.2-0.5mg IM/IV) / Carboprost (250µg IM) / Misoprostol (800µg PR/SL) / Tranexamic Acid 1g IV |

### WHO PPH Bundle (2018)
1. **Oxytocin 10 IU IM/IV** immediately after birth
2. **Controlled Cord Traction**
3. **Uterine Massage**
4. **Monitor Uterine Tone & Bleeding** for 24h

|----------|----------|

## 5. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **Oxytocin Synthesis** | Hypothalamic **Paraventricular Nucleus (PVN)**; Posterior Pituitary release |
| **Positive Feedback** | **Only hormonal positive feedback in humans** = Oxytocin in Labour (Ferguson Reflex) |
| **Labour Trigger** | **Cervical Stretch** → Ferguson Reflex → Oxytocin Pulses → Contractions |
| **Lactation** | **Suckling → Oxytocin → Myoepithelial Contraction → Milk Ejection** (Prolactin = Milk Synthesis) |
| **Syntocinon Induction** | Start 0.5-2 mU/min → Max 20 mU/min; CTG Continuous |
| **Oxytocin in PPH** | **10 IU IV Bolus** → 20-40 IU in 500-1000mL NS @ 125-250 mL/h |
| **Hyperstimulation** | >5 Contractions/10min OR Resting Tone >20mmHg → Stop Oxytocin |
| **Water Intoxication** | Oxytocin V2 activity + Fluids → **Hyponatraemia** (Limit fluids, Monitor Na+) |
| **Atosiban** | **OTR Antagonist**; Tocolytic 24-33+6 wks; No β-side effects; Loading 6.75mg/15min → 18mg/h → 6mg/h |
| **Carbetocin** | Long-acting Oxytocin Analog (t½ 40-80min); Single 100µg IM/IV for PPH Prophylaxis |
| **Ergometrine** | 0.2-0.5mg IM/IV; Second-line PPH; Contraindicated in HTN/Pre-eclampsia/Heart Disease |

## 7. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Oxytocin vs ADH** | ADH = V2 Water Retention; Oxytocin = Uterine Contraction + Milk Ejection |
| **Oxytocin vs Prostaglandins** | Oxytocin = Induction/Augmentation; PGE2/Misoprostol = Cervical Ripening |
| **Oxytocin in PPH** | **1st Line** (10 IU IV); Ergometrine/Carboprost = 2nd/3rd Line |
| **Syntocinon Dilution** | 10 IU in 500mL = 20 mU/mL; 1 mL/min = 20 mU/min |
| **Atosiban vs β-agonists** | Atosiban = No tachycardia/hypokalaemia; Safe in Heart Disease/Diabetes |
| **Oxytocin V2 Effect** | Cross-reactivity with V2 → Water Retention → Hyponatraemia (Limit Fluids) |
| **Ergometrine Contraindications** | HTN, Pre-eclampsia, IHD, Sepsis, Sepsis |

**Mnemonic: OXYTOCIN IN LABOUR**
- **O**xytocin = **O**nly positive feedback hormone
- **X** = **X**-tra contractions (Ferguson Reflex)
- **Y** = **Y**ields uterine contraction
- **T** = **T**rigger = Cervical Stretch
- **O** = **O**xycontin... no, **O**xytocin release from PVN
- **C** = **C**ontraction frequency ↑
- **I** = **I**nduction/Augmentation/PPH
- **N** = **N**o V1 pressor at low dose (V2 water retention only)

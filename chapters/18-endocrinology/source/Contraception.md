# Contraception

> [!info]
> **Contraception = Prevention of Pregnancy.** **UKMEC (Medical Eligibility Criteria)** guides safe prescribing. **LARC (Long-Acting Reversible Contraception) = First-Line** for Most Women. **Tiered Efficacy**: LARC > COCP/POP > Barrier > Natural Methods.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Apply UKMEC categories for contraceptive eligibility
- [ ] Compare efficacy, mechanisms, and side effects of all methods
- [ ] Select appropriate method based on patient factors (age, BMI, medical history, preferences)
- [ ] Manage emergency contraception
- [ ] Counsel on non-contraceptive benefits and side effects

## 2. Contraceptive Methods — Quick Reference

### Long-Acting Reversible Contraception (LARC)

| Method | Mechanism | Duration | Pearl Index (Typical) | Key Features |
|------|----------|--------|----------------------|--------------|
| **Copper IUD** | Cu²⁺ Toxic to Sperm/Ova; Endometrial Inflammation | **5-10 Years** | **0.6-0.8%** | Non-Hormonal; Heavy Periods; Emergency Contraception |
| **LNG-IUS (52mg)** | LNG Release → Endometrial Suppression | **5 Years** | **0.2%** | ↓ Menstrual Bleeding; Local Progestogen; LNG 20µg/day |
| **LNG-IUS (19.5mg/13.5mg)** | Lower Dose | 3-5 Years | 0.2% | Smaller Frame; Less Bleeding |
| **Implant (ENG 68mg)** | ENG Release → Oestrogen Suppression | **3 Years** | 0.05% | Subdermal; Predictable Bleeding |
| **DMPA (Injectable)** | MPA → Ovulation Suppression; Endometrial Thinning | **12-13 Weeks** | 3-6% | Weight Gain; Delayed Fertility Return (9-10mo) |

## 3. Combined Oral Contraceptive Pill (COCP)

| Aspect | Details |
|------|---------|
| **Composition** | **EE 20-35µg** + **Progestogen** (Levonorgestrel, Norethisterone, Desogestrel, Gestodene, Drospirenone, Dienogest) |
| **Regimen** | **21 Days On / 7 Days Off** (Withdrawal Bleed) |
| **Extended Regimens** | Tricycling (63/7), Continuous (No Break) |
| **Mechanism** | **Suppress Ovulation** (E2 → ↓ FSH/LH); Endometrial Thinning; Cervical Mucus Thickening |
| **Failure Rate** | 7-9% (Typical); 0.3% (Perfect) |

### COCP — Non-Contraceptive Benefits
| Benefit | Evidence |
|--------|----------|
| **Menstrual Regulation** | Regular, Lighter, Less Painful Periods |
| **Acne / Hirsutism** | **Drospirenone/Dienogest/Cyproterone** (Anti-Androgenic) |
| **Endometrial Cancer** | **↓ 50%** (Long-Term) |
| **Ovarian Cancer** | **↓ 40-50%** (Long-Term) |
| **Colorectal Cancer** | **↓ 20%** |
| **PID** | ↓ Risk |
| **Functional Ovarian Cysts** | ↓ Incidence |

### COCP — Risks (UKMEC 3/4)
| Risk | Magnitude | Mitigation |
|--------|-----------|------------|
| **VTE** | **3-6x** (Baseline 5-10/100,000 → 20-40/100,000) | **Transdermal Preferred** if Risk Factors |
| **Stroke** | **1.5-2x** (Ischaemic) | Avoid if Migraine with Aura |
| **MI** | Slight ↑ | Avoid if CVD |
| **Breast Cancer** | **Slight ↑** (Current/Recent Use) | ↓ After Stopping |
| **Cervical Cancer** | Slight ↑ (HPV Co-Factor) | HPV Vaccination |
| **Benefits vs Risks** | **Net Benefit** for Most Women <35, Non-Smoker |

### COCP Contraindications (UKMEC 4)
| Condition | Rationale |
|-----------|-----------|
| **Current/Recent Breast Cancer** | Oestrogen-Dependent Tumour |
| **VTE (Current/History)** | ↑ Thrombotic Risk |
| **Stroke / TIA / MI** | Thrombotic Risk |
| **Uncontrolled HTN** (≥160/100) | ↑ Stroke/MI Risk |
| **Migraine with Aura** | ↑ Stroke Risk (2-4x) |
| **Smoker ≥35y, >15/day** | Synergistic VTE Risk |
| **Major Surgery (Immobility)** | Post-Op VTE Risk |
| **Major Surgery (Planned)** | Stop 4 Weeks Pre-Op |
| **Liver Tumour / Severe Liver Disease** | Metabolism Impaired |

## 4. Progestogen-Only Pill (POP)

| Feature | Details |
|---------|---------|
| **Types** | **Desogestrel 75µg** (Newer, Ovulation Inhibition); **Traditional** (Norethisterone 350µg, Levonorgestrel 30µg) — Cervical Mucus Effect |
| **Mechanism** | **Desogestrel**: Ovulation Inhibition (97%); **Traditional**: Cervical Mucus + Endometrial Thinning |
| **Regimen** | **Strict Timing** (Same Time Daily; Desogestrel 12h Window) |
| **Bleeding Pattern** | Unpredictable (Amenorrhoea, Spotting, Prolonged Bleeding) |
| **UKMEC 4** | Breast Cancer, Undiagnosed Bleeding, Severe Liver Disease |

## 4. Emergency Contraception (EC)

| Method | Window | Efficacy | Details |
|--------|--------|----------|---------|
| **Cu-IUD** | **≤5 Days (120h)** | **>99%** (Most Effective) | **First-Line**; Also Ongoing Contraception |
| **Ulipristal Acetate (UPA 30mg)** | **≤120h (5 Days)** | **~98%** (More Effective Than LNG) | Prescription; Delay Ovulation |
| **Levonorgestrel (LNG 1.5mg)** | **≤72h (3 Days)** | **~95%** (Within 24h) | OTC; Less Effective After 72h |

### EC Decision Algorithm
```
Unprotected Intercourse / Contraceptive Failure
         │
         ▼
Time Since Intercourse
         │
         ├── ≤120h (5 Days) → **Cu-IUD** (1st Line) → If Unsuitable/Unavailable → UPA 30mg
         │
         └── ≤72h (3 Days) → **LNG 1.5mg** (If UPA Unavailable/Contraindicated)
         │
         ▼
Post-EC Care
         │
         ├── Pregnancy Test if Next Period Delayed >7 Days
         ├── Start/Resume Regular Contraception Immediately
         ├── Cu-IUD → Ongoing Contraception
         └── STI Screen if Indicated
```

## 4. Quick Reference — Method Selection

| Patient Profile | First-Line Choice | Alternative |
|------------------|-------------------|-------------|
| **Nulliparous, No Risk Factors** | **Implant** or **IUS** | Cu-IUD |
| **Postpartum (Breastfeeding)** | **Implant** or **POP** or **Cu-IUD** (After 4wks) | POP |
| **Postpartum (Non-Breastfeeding)** | **Implant**, **Cu-IUD**, **DMPA** | POP |
| **Post-Abortion** | **All Methods** (Including IUD Immediate) | — |
| **Postpartum <21 Days (Non-BF)** | **Cu-IUD**, Implant, DMPA, POP | COCP (UKMEC 4) |
| **Postpartum <21 Days (BF)** | **Implant, POP, Cu-IUD** | DMPA (UKMEC 2) |

| **Adolescent** | **Implant, Cu-IUD, IUS** | POP, Condoms |
| **HIV (On ART)** | Most Methods OK; **Check Interactions** (Rifampicin, EFV) | POP if EFV |
| **Breast Cancer** | **Cu-IUD, POP, LNG-IUS (Caution)** | |
| **VTE History** | **Cu-IUD, POP, LNG-IUS (Caution)** | COCP, DMPA (UKMEC 3-4) |
| **Smoker >35, >15/day** | **Cu-IUD, POP, Implant, LNG-IUS** | COCP (UKMEC 4) |

## 4. Follow-Up & Counselling

### Initiation Counselling
| Topic | Key Points |
|--------|------------|
| **Mechanism** | How It Works |
| **Effectiveness** | Pearl Index (Typical vs Perfect) |
| **Side Effects** | Common (Bleeding Changes, Weight, Mood, Headache) |
| **Warning Signs** | When to Seek Help (Severe Pain, Heavy Bleeding, Signs of VTE) |
| **Follow-Up** | 3-6 Month Review (Bleeding Pattern, Side Effects, Satisfaction) |
| **STI Protection** | **Condoms Only** (Dual Protection) |

### Common Side Effects & Management
| Side Effect | Method | Management |
|-------------|----------|------------|
| **Irregular Bleeding** | POP, Implant, IUS, DMPA | Reassurance; Mefenamic Acid (IUS); Switch if Persistent >6mo |
| **Weight Gain** | DMPA (Mean +2-3kg) | Counselling; Switch if >5kg |
| **Mood Changes** | COCP, POP, DMPA | Switch Method; SSRI if Depressive |
| **Acne** | COCP (Anti-Androgenic Progestogen) | Switch Progestogen |
| **Amenorrhoea** | IUS, Implant, DMPA | Reassurance (Physiological) |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **LARC vs Non-LARC** | LARC = Implant, IUS, Cu-IUD; Set-and-Forget; Non-LARC = User-Dependent |
| **COCP vs POP** | COCP = Oestrogen + Progestogen (Suppress Ovulation); POP = Progestogen Only (Cervical Mucus/Endometrium) |
| **Emergency Contraception** | Cu-IUD = **Most Effective** (≤5 Days); UPA = 120h; LNG = 72h |
| **DMPA Delay** | **Fertility Return 9-10 Months** (Not Immediate) |
| **IUD vs IUS** | **Cu-IUD** = Copper; **LNG-IUS** = Levonorgestrel-Releasing |
| **POP Timing** | **Desogestrel**: 12h Window; **Traditional POP**: 3h Window |
| **UKMEC Categories** | 1=Use, 2=Use, 3=Avoid, 4=Do Not Use |
| **EC Window** | Cu-IUD ≤5 Days; UPA ≤5 Days; LNG ≤72h (Best ≤24h) |
| **DMPA Delay** | **Return to Fertility 9-10 Months** (Plan Ahead) |
| **Breastfeeding + COCP** | Suppresses Lactation; **Avoid <6 Weeks** |
| **Vasectomy** | >99% Effective; Reversal 50-70% (Not Guaranteed) |

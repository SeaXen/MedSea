# Polycystic Ovary Syndrome (PCOS)

> [!info]
> **PCOS = Most Common Endocrine Disorder in Women of Reproductive Age (10-15%).** **Rotterdam Criteria (2/3): Oligo/Anovulation + Hyperandrogenism + Polycystic Ovaries.** **Insulin Resistance = Core Pathophysiology.** **Long-term Risks: T2DM, CVD, Endometrial Cancer, Infertility.**

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Apply Rotterdam diagnostic criteria (2/3) and phenotypic classification
- [ ] Differentiate PCOS from other hyperandrogenic states
- [ ] Assess metabolic risk and screen for long-term complications
- [ ] Apply management algorithms (lifestyle, COCP, metformin, letrozole, IVF)
- [ ] Counsel on long-term health risks (T2DM, CVD, endometrial cancer)

## 2. Diagnostic Criteria — Rotterdam (2/3 Required)

| Criterion | Definition |
|-----------|------------|
| **1. Oligo/Anovulation** | **<8 Cycles/Year** or Cycle Length **>35 Days** |
| **2. Hyperandrogenism** | **Clinical** (Hirsutism mFG >8, Acne, Alopecia) **OR Biochemical** (Total Testosterone >2.5-3 nmol/L, FAI >5) |
| **3. Polycystic Ovarian Morphology (PCOM)** | **≥20 Follicles (2-9mm) Per Ovary** OR **Ovarian Volume >10 mL** (US) |

**Diagnosis = 2 of 3 Criteria** (After Excluding Other Aetiologies)

### Phenotypic Classification (4 Phenotypes)
| Phenotype | Criteria | Metabolic Risk |
|----------|----------|----------------|
| **A (Classic/Full)** | **Oligo/Anovulation + Hyperandrogenism + PCOM** | **Highest** |
| **B (Classic Non-PCO)** | **Oligo/Anovulation + Hyperandrogenism** | High |
| **C (Ovulatory PCOS)** | **Hyperandrogenism + PCOM** | Moderate |
| **D (Non-Hyperandrogenic)** | **Oligo/Anovulation + PCOM** | **Lowest** |

---

## 2. Differential Diagnosis — Exclusion of Mimics

| Condition | Key Differentiators |
|-----------|---------------------|
| **NCCAH (21-OH Deficiency)** | ↑ 17-OHP (Basal >200, Stim >1000 ng/dL); Genetic Testing (CYP21A2) |
| **Cushing Syndrome** | Cortisol Excess (Dex Suppression Test); Striae, Moon Face, HTN |
| **Androgen-Secreting Tumour** | **Rapid Onset**, Very High Testosterone (>5 nmol/L); DHEAS ↑; Imaging |
| **Thyroid Dysfunction** | TSH Abnormal; Prolactin Normal |
| **Hyperprolactinaemia** | Prolactin ↑; Galactorrhoea; MRI Pituitary |
| **Drug-Induced** | Anabolic Steroids, Danazol, Valproate, CYP21A2 Inhibitors |
| **Acromegaly** | GH/IGF-1 ↑; OGTT GH Not Suppressed |
| **Congenital Adrenal Hyperplasia** | 17-OHP ↑; 21-Hydroxylase Deficiency |

## 3. Clinical Features & Phenotypes

| Feature | Frequency | Details |
|---------|-----------|---------|
| **Oligomenorrhoea/Amenorrhoea** | **70-80%** | <8 Cycles/Year or >35 Day Cycles |
| **Hirsutism (mFG >8)** | **60-70%** | Ferriman-Gallwey Score; Chin, Upper Lip, Chest, Abdomen, Thighs |
| **Acne** | 40-50% | Jawline, Jaw, Back |
| **Alopecia** | 10-20% | Female Pattern (Ludwig) |
| **Acanthosis Nigricans** | 30-40% | Neck, Axillae, Groins (Insulin Resistance Marker) |
| **Obesity** | **40-80%** (Central) | BMI >30 in 40-50%; Waist >88cm |
| **Infertility** | **70-80%** | Anovulation; ↑ Miscarriage Risk |
| **Metabolic** | **Insulin Resistance** (70-80%); Dyslipidaemia; ↑ T2DM Risk |

## 3. Diagnostic Algorithm

```
Clinical Suspicion (Oligomenorrhoea + Hyperandrogenism)
         │
         ▼
EXCLUDE: Pregnancy, Thyroid, Prolactin, Cushing, NCCAH, Androgen-Secreting Tumour
         │
         ▼
APPLY ROTTERDAM CRITERIA (2/3)
         │
         ├── Oligo/Anovulation (<8 cycles/yr or >35d cycle)
         ├── Hyperandrogenism (Clinical mFG>8 OR Biochemical T >2.5-3 nmol/L)
         └── PCOM on US (≥20 Follicles 2-9mm OR Volume >10mL)
         │
         └── 2/3 CRITERIA MET → DIAGNOSE PCOS
         │
         ▼
PHENOTYPING (A/B/C/D) → Guides Metabolic Risk & Management
```

## 3. Investigations

### Hormonal Profile
| Test | Timing | Expected in PCOS |
|------|--------|------------------|
| **Total Testosterone** | Day 2-4 (or Random) | **>2.5-3 nmol/L** (or FAI >5) |
| **Free Androgen Index (FAI)** | (Total T × 100) / SHBG | **>5** |
| **SHBG** | Low (Insulin ↓ SHBG Synthesis) | |
| **LH/FSH** | Day 2-4 | **LH:FSH >2:1** (Classic but Not Diagnostic) |
| **17-OHP** | Day 2-4 | **Exclude NCCAH** (Basal <200 ng/dL; Stim >1000 ng/dL) |
| **TSH, Prolactin** | | Exclude Thyroid / Prolactinoma |
| **17-OHP (Basal)** | | **<200 ng/dL** (Exclude NCCAH) |
| **DHEAS** | Mildly ↑ Often | Exclude Adrenal Tumour if Markedly ↑ |

### Metabolic Workup
| Test | Target / Threshold |
|------|--------------------|
| **Fasting Glucose / HbA1c** | <5.6 mmol/L / <5.7% |
| **OGTT (75g)** | **120min <7.8 mmol/L** (Normal); **7.8-11.0 = IGT**; **≥11.1 = DM** |
| **Fasting Insulin / HOMA-IR** | HOMA-IR >2.5 = Insulin Resistance |
| **Lipid Profile** | TG >1.7, HDL <1.3 (Women) |
| **Liver Function** | NAFLD Common (ALT ↑) |

### Pelvic Ultrasound
| Finding | PCOM Criteria |
|---------|---------------|
| **Follicle Count** | **≥20 Follicles (2-9mm) Per Ovary** |
| **Ovarian Volume** | **>10 mL** |
| **Stroma** | Increased Echogenicity |

## 3. Long-Term Health Risks

| Complication | Relative Risk | Screening |
|------------|---------------|-----------|
| **Type 2 Diabetes** | **4-7x** | OGTT q1-3yr (Age >30 / BMI >25 / Prior GDM) |
| **Cardiovascular Disease** | **2-3x** | BP, Lipids, HbA1c Annually |
| **Endometrial Cancer** | **3-4x** (Unopposed Oestrogen) | Ultrasound if Abnormal Bleeding; Progestogen Challenge |
| **Obstructive Sleep Apnoea** | 5-10x | STOP-BANG; Sleep Study if Symptomatic |
| **Mood Disorders** | 2-3x (Depression/Anxiety) | PHQ-9/GAD-7 Screening |
| **NAFLD** | 2-3x | LFTs, Ultrasound Liver |

---

## 4. Management — Phenotype-Guided

### Universal First-Line: Lifestyle Modification
| Target | Intervention |
|--------|--------------|
| **Weight Loss** | **5-10% Body Weight** → Restores Ovulation in 50-60% |
| **Diet** | Low GI, High Fibre, Moderate Protein, Mediterranean |
| **Exercise** | **150 min/week Moderate** + Resistance 2x/week |
| **Behavioural** | CBT, Self-Monitoring, Stress Management |

### Pharmacotherapy — By Goal

| Goal | First-Line | Second-Line | Third-Line |
|------|-----------|-------------|------------|
| **Menstrual Regulation** | **COCP** (Low-Dose, EE 20-30µg + Anti-Androgen Progestogen) | Cyclical Progestogen (MPA 10mg Days 1-12) | — |
| **Hirsutism/Acne** | **COCP** (Cyproterone/Drospirenone) | **Anti-Androgens** (Spironolactone 50-100mg, Finasteride, Flutamide) | Laser/Electrolysis |
| **Ovulation Induction** | **Letrozole 2.5-5mg Days 2-6** (1st Line) | **Clomifene 50-150mg Days 2-6** | Gonadotrophins (rFSH/hMG) → IVF |
| **Metabolic/IR** | **Metformin 500mg BD → 1g BD** (If BMI ≥25 / IGT) | Pioglitazone (If Metformin Intolerant) | GLP-1 RA (Semaglutide/Tirzepatide) if Obese |
| **Weight Loss (BMI ≥25)** | **Lifestyle 5-10% Weight Loss** | **GLP-1 RA** (Semaglutide 2.4mg / Tirzepatide 15mg) | **Bariatric Surgery** (BMI ≥35/40) |

## 4. Ovulation Induction — Algorithm

```
Anovulatory PCOS Desiring Pregnancy
         │
         ▼
PRE-CONCEPTION: Folic Acid 5mg/day; Rubella/Varicella Immune; TSH/TSH/HbA1c
         │
         └── LIFESTYLE OPTIMISATION (3-6 Months)
                 │
                 ├────────────────┬─────────────────────┐
                 │                │                     │
          LETROZOLE 2.5-5mg    CLOMIFENE 50-150mg      GONADOTROPHINS
          Days 2-6 (1st Line)   Days 2-6 (If Letrozole Fail)   rFSH/hMG (If Oral Fail)
          (↑ Live Birth, ↓       (Traditional 1st Line)         (Monitor: US + E2)
           Multiples, ↓ OHSS)
                 │                │                      │
                 └────────────────┴──────────────────────┘
                              │
                              ▼
                        MONITORING: US Follicular Tracking (Day 10-12)
                        TRIGGER: hCG 5000-10000 IU (When Lead Follicle ≥18mm)
                        TIMING: IUI 36h Post-hCG / Timed Intercourse
                        LUTEAL SUPPORT: Progesterone (If IUI/IVF)
```

## 4. Long-Term Health Surveillance

| Complication | Screening Interval | Threshold for Action |
|-------------|-------------------|----------------------|
| **Type 2 Diabetes** | OGTT **q1-3yr** (Age >30 / BMI >25 / Prior GDM) | HbA1c ≥5.7% → Preventive Metformin |
| **Cardiovascular** | BP, Lipids **Annually**; Carotid IMT if High Risk | LDL >3.0 → Statin |
| **Endometrial Cancer** | **Ultrasound** if Abnormal Bleeding; **Progestogen Challenge** q1-2yr if Amenorrhoeic | Endometrial Thickness >5mm → Biopsy |
| **Sleep Apnoea** | STOP-BANG Questionnaire; Sleep Study if High Risk | CPAP if AHI >15 |
| **Mood Disorders** | PHQ-9/GAD-7 Annually | Refer if Positive |
| **Bone Health** | DEXA if Amenorrhoeic >1yr / Steroid Use | T-score ≤-2.5 → Bisphosphonate |

## 5. Special Situations

### PCOS in Adolescence
| Challenge | Approach |
|-----------|---------|
| **Diagnosis Delay** | Wait 2 Years Post-Menarche; **PCOM Common in Normal Teens** |
| **Diagnosis** | Oligo/Anovulation **≥2 Years Post-Menarche** + Hyperandrogenism |
| **Management** | COCP (1st Line); Metformin if IGT; Lifestyle Emphasis |

### Pregnancy in PCOS
| Risk | Management |
|--------|------------|
| **Miscarriage** | **1.5-2x Higher** | Progestogen Support (If Prior Loss) |
| **GDM** | **2-3x Risk** | Early OGTT (16-18wk); Metformin if IGT |
| **Pre-eclampsia** | 2-3x Risk | Aspirin 75-150mg OD from 12w (If >1 Risk Factor) |
| **Preterm Birth** | ↑ Risk | Cervical Length Monitoring |

| **GLP-1 RA** | Semaglutide 2.4mg / Tirzepatide 15mg for Weight Loss in Obese PCOS |

| **Letrozole vs Clomifene** | **Letrozole Superior** (↑ Live Birth, ↓ Multiples, ↓ OHSS) |
| **Metformin in Pregnancy** | Continue if Prior GDM/IR; Not for Ovulation Induction Alone |
| **Vitamin D** | Deficiency Common in PCOS; Supplement if <30 ng/mL |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **PCOS vs NCCAH** | NCCAH: 17-OHP >200 (Basal) / >1000 (Stim); Genetic CYP21A2 |
| **PCOS vs Cushing** | Cushing: Striae, Moon Face, Hypertension, Easy Bruising; Dex Suppression Test |
| **PCOS vs Androgen-Secreting Tumour** | Tumour: Rapid Onset, Very High Testosterone (>5 nmol/L), DHEAS ↑↑ |
| **Letrozole vs Clomifene** | Letrozole: ↑ Live Birth, ↓ Multiples, ↓ OHSS; Preferred 1st Line |
| **Metformin in PCOS** | BMI ≥25 + IGT → Metformin; Not for All PCOS |
| **Ovulation Induction** | Letrozole 1st Line; Clomifene 2nd; Gonadotrophins 3rd |
| **Endometrial Cancer** | 3-4x Risk → Progestogen Challenge q1-2yr if Oligomenorrhoea |
| **Insulin Resistance** | HOMA-IR >2.5; Central Obesity + Acanthosis Nigricans |
| **Acanthosis Nigricans** | Insulin Resistance Marker; Neck, Axillae, Groins |

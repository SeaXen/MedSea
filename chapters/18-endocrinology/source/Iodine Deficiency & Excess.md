# Iodine Deficiency & Excess

> [!info]
> **Iodine = Essential Micronutrient for Thyroid Hormone Synthesis.** **Deficiency = World's Leading Cause of Preventable Brain Damage.** **Excess (Amiodarone, Contrast) → Wolff-Chaikoff Effect & Jod-Basedow Phenomenon.** WHO/ICCIDD/UNICEF monitor global IDD status.

---

## 2. Iodine Deficiency Disorders (IDD) — Clinical Spectrum

| Condition | Population | Key Features |
|-----------|------------|--------------|
| **Congenital Hypothyroidism / Endemic Cretinism** | Neonates | **Mental Retardation**, Deaf-Mutism, Spastic Diplegia, Growth Failure |
| **Neonatal Hypothyroidism** | Neonates | Prolonged Jaundice, Poor Feeding, Constipation, Macroglossia |
| **Childhood/Adolescent Goitre** | Children/Adolescents | Diffuse/Multinodular Goitre; Learning Difficulties |
| **Adult Goitre** | Adults | Compressive Symptoms, Subclinical/Overt Hypothyroidism |
| **Hypothyroidism** | All Ages | Fatigue, Weight Gain, Cold Intolerance, Myxoedema |
| **Impaired Mental Function** | Children | **IQ Loss 10-15 Points** (Even Mild Deficiency) |
| **Pregnancy Complications** | Pregnant Women | Miscarriage, Stillbirth, Cretinism, Low Birth Weight |

## 3. Iodine Excess — Mechanisms & Clinical Effects

### Sources of Excess Iodine
| Source | Typical Iodine Load |
|--------|---------------------|
| **Amiodarone** | **75 mg/day** (200mg tablet = 37% Iodine = 75 mg) — **500x RDA** |
| **Iodinated Contrast Media** | **10-50 g Iodine** per Study (IV) |
| **Iodinated Antiseptics** (Povidone-Iodine) | Topical Absorption (Burns, Wounds) |
| **Iodine Supplements / Kelp** | Variable (Often >500 µg/dose) |
| **Dietary** (Seaweed/Kelp in East Asian Diets) | Up to 1-2 mg/day |

## 3. Wolff-Chaikoff Effect & Escape Phenomenon

| Phase | Mechanism | Clinical Correlate |
|-------|-----------|-------------------|
| **Acute Wolff-Chaikoff** (24-48h) | **Excess Iodide → Inhibits TPO** → ↓ Organification → ↓ Hormone Synthesis | Transient ↓ T4/T3, ↑ TSH |
| **Escape Phenomenon** (10-14 days) | **Downregulation of NIS** → ↓ Iodide Uptake → ↓ Intracellular Iodide → Organification Resumes | TSH Normalises, T4/T3 Recover |
| **Failed Escape** | **Failure to Downregulate NIS** → Persistent Organification Block → **Hypothyroidism** | **Amiodarone Hypothyroidism; Iodine-Induced Hypothyroidism** |

**Key**: **Normal Thyroid Escapes in 10-14 Days**. **Failure to Escape → Hypothyroidism** (Underlying Thyroid Autoimmunity/Defect).

---

## 3. Jod-Basedow Phenomenon (Iodine-Induced Hyperthyroidism)

| Feature | Details |
|---------|---------|
| **Definition** | **Iodine-Induced Thyrotoxicosis** in Underlying Autonomous Nodules / Latent Graves |
| **Mechanism** | Excess Iodine Substrate → ↑ Hormone Synthesis in Autonomous Tissue |
| **Risk Groups** | **Multinodular Goitre**, **Latent Graves**, **Iodine-Deficient Areas** suddenly Iodised |
| **Lab Findings** | **TSH Suppressed**, fT4/fT3 ↑, **RAI Uptake Normal/High** |
| **Treatment** | **Thionamides (Carbimazole/PTU) + Perchlorate** (Blocks Iodide Uptake) |

---

## 3. Amiodarone-Induced Thyroid Dysfunction (AIT) — Recap

| Type | Mechanism | RAI Uptake | Treatment |
|------|-----------|------------|-----------|
| **Type 1 (Jod-Basedow)** | Iodine Load → Autonomous Nodules | **High/Normal** | **Thionamides (CBZ/PTU) + Perchlorate** |
| **Type 2 (Destructive)** | Cytotoxicity → Thyroiditis | **Low/Absent** | **Steroids (Pred 30-40mg/day)** |
| **Mixed** | Both Mechanisms | Variable | **Thionamides + Steroids** |
| **Amiodarone Hypothyroidism** | Wolff-Chaikoff Failure to Escape | — | **Levothyroxine** (Continue Amiodarone) |

---

## 3. Iodinated Contrast Media

| Feature | Details |
|--------|---------|
| **Iodine Load** | **10-50 g Iodine** per IV Study (vs 150 µg RDA = **1000x RDA**) |
| **Acute Effect** | Transient ↑ Serum Iodine → Transient ↑ TSH (Wolff-Chaikoff) |
| **Thyrotoxicosis Risk** | **Jod-Basedow** in MNG / Latent Graves (Days-Weeks Post) |
| **Hypothyroidism Risk** | Failed Escape (Pre-existing Hashimoto/Autonomous Nodules) |
| **Monitoring** | TSH/fT4 at Baseline, 6-8wks, 3mo Post-Contrast |

## 4. Iodine Deficiency in Pregnancy — Critical

| Trimester | Iodine Requirement | Consequences of Deficiency |
|-----------|-------------------|----------------------------|
| **Pre-conception** | **250 µg/day** | Subfertility, Impaired Ovulation |
| **1st Trimester** | **250 µg/day** | **Cretinism, Neurodevelopmental Defects**, Miscarriage |
| **2nd/3rd Trimester** | **250 µg/day** | **Impaired Neurodevelopment (IQ ↓10-15 pts)**, Preterm Birth |
| **Lactation** | **250 µg/day** | Infant Iodine Supply via Breast Milk |

**WHO Recommendation**: **250 µg/day** throughout Pregnancy & Lactation (vs 150 µg Adult).

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Wolff-Chaikoff vs Jod-Basedow** | W-C = Transient Hypo (Escape); J-B = Hyperthyroidism (Iodine Load + Autonomous Nodules) |
| **Amiodarone Type 1 vs 2** | Type 1 = High RAI (Iodine Load); Type 2 = Low RAI (Destructive) |
| **Iodine Deficiency vs Excess** | Deficiency → Hypothyroidism/Goitre/Cretinism; Excess → Wolff-Chaikoff (Transient) or Jod-Basedow (Hyper) |
| **Escape Phenomenon** | Normal Thyroid Escapes W-C in 10-14 Days; Failure → Persistent Hypothyroidism |
| **Iodine in Pregnancy** | **250 µg/day** (vs 150 µg Adult); Critical for Fetal Brain (1st Tri) |
| **UIC vs TSH** | UIC = Population Status; Neonatal TSH = Programme Effectiveness |

**Mnemonic: IODINE DEFICIENCY DISORDERS**
- **I**odine **D**eficiency = **C**retinism (Mental Retardation + Deaf-Mutism + Spasticity)
- **G**oitre (Endemic)
- **H**ypothyroidism (Adult/Child)
- **I**Q Loss (10-15 Points Even Mild)
- **P**regnancy Loss (Miscarriage, Stillbirth)

**Mnemonic: WOLFF-CHAIKOFF ESCAPE**
- **W**olff-Chaikoff: **H**ypothyroidism **T**ransient (10-14 Days)
- **E**scape: **N**IS Downregulation → Normal Function
- **F**ailed Escape → **P**ermanent **H**ypothyroidism (Autoimmune)

**Mnemonic: AMIODARONE THYROID**
- **A**MIODARONE: 75mg Iodine/Day (500x RDA)
- **T**YPE 1 = **I**ODINE LOAD (Jod-Basedow) = High RAI Uptake
- **T**YPE 2 = **D**estructive = Low RAI (Steroids)
- **H**ypothyroidism = Failed Escape (Wolff-Chaikoff Failure)

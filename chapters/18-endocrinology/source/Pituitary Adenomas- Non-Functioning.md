# Pituitary Adenomas: Non-Functioning

> [!info]
> **Non-functioning pituitary adenomas (NFPAs) = No clinical hormone excess.** Most are **null cell** or **gonadotroph** (α-subunit/FSH/LH secretion without clinical effect). Account for **25-30%** of pituitary adenomas. Present with mass effect, not hormone excess.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Define non-functioning pituitary adenoma (NFPA)
- [ ] Recognise clinical presentation (mass effect, hypopituitarism, stalk effect)
- [ ] Apply appropriate investigations (hormones, MRI, visual fields)
- [ ] Apply management algorithms (observation vs surgery vs radiotherapy)
- [ ] Understand surveillance protocols post-surgery

---

## 1. Definition & Epidemiology

| Feature | Details |
|---------|---------|
| **Definition** | Pituitary adenoma **without clinical hormone excess syndrome** |
| **Cell Types** | **Null cell** (70-80%) → No hormone; **Gonadotroph** (15-25%) → α-subunit/FSH/LH (subclinical) |
| **Prevalence** | **25-30%** of all pituitary adenomas (2nd most common after prolactinoma) |
| **Demographics** | Peak 50-60 years; Slight male predominance |
| **Size at Presentation** | **Macroadenoma (>10mm) in 80-90%** (Present late due to lack of hormone excess) |

---

## 2. Pathology & Immunohistochemistry

| Immunohistochemistry | Staining Pattern | Significance |
|-------------------|------------------|--------------|
| **All Hormones (ACTH, TSH, GH, PRL, FSH, LH, TSH, α-subunit)** | **Negative** | Null cell adenoma |
| **FSH/LH/α-subunit** | **Positive (Focal/Diffuse)** | **Gonadotroph Adenoma** (Often clinically silent) |
| **Single Hormone** | Single Positive | Functioning Adenoma (Exclude) |

**Diagnosis of Exclusion**: NFPA diagnosed **after excluding functioning adenomas** (Normal PRL, GH/IGF-1, ACTH/Cortisol, TSH/fT4, FSH/LH/Testosterone/Oestradiol).

---

## 3. Clinical Presentation

| Presentation | Frequency | Mechanism |
|--------------|-----------|-----------|
| **Visual Field Defects** | 60-80% | Chiasmal compression (Bi-temporal hemianopia) |
| **Headache** | 50-70% | Dural stretch, Cavernous sinus invasion |
| **Hypopituitarism** | 50-60% | Compression of Normal Pituitary Tissue |
| **Incidentaloma** | 10-20% | Imaging for unrelated reason |
| **Cranial Nerve Palsies** | 10-20% | CN III/IV/VI (Cavernous Sinus Invasion) |

### Hypopituitarism Pattern (NFPA)
| Order of Loss (Typical) | Hormone | Frequency |
|-------------------------|---------|-----------|
| **1st** | **GH** | Most common (80-90%) |
| **2nd** | **Gonadotrophs (LH/FSH)** | 70-80% |
| **3rd** | **TSH** | 40-50% |
| **4th** | **ACTH** | 20-30% |
| **Variable** | **Prolactin** | May be ↑ (Stalk Effect) |

**Stalk Effect**: Tumour compresses pituitary stalk → ↓ Dopamine delivery → **Mild Hyperprolactinaemia** (Prolactin usually <2000 mIU/L).

---

## 2. Investigations

### Hormonal Profile (Rule Out Functioning Adenoma)
| Test | Expected in NFPA |
|------|------------------|
| **Prolactin** | Normal or **Mildly Elevated (<2000 mIU/L)** (Stalk Effect) |
| **GH / IGF-1** | Normal |
| **ACTH / 09:00 Cortisol** | Normal |
| **TSH / fT4** | Normal |
| **FSH / LH / Testosterone / Oestradiol** | Normal or Low (Secondary Hypogonadism from Compression) |
| **α-subunit** | May be ↑ (Gonadotroph Adenoma) |

### Imaging
| Modality | Protocol | Purpose |
|----------|----------|---------|
| **MRI Pituitary** | **3mm slices, Gadolinium, Coronal/Sagittal** | Gold Standard; Tumour Size, Extension, Cavernous Sinus Invasion |
| **Visual Fields** | **Goldmann Perimetry / Humphrey** | Baseline; Quaterly Post-op |
| **CT (Rare)** | Thin Slices, Contrast | If MRI Contraindicated |

### Cavernous Sinus Invasion Grading (Knosp Classification)
| Grade | Description | Surgical Resectability |
|-------|-------------|------------------------|
| **Grade 0** | Tumour ≤ Intercarotid Line | Complete Resection Likely |
| **Grade 1** | Tumour > Intercarotid Line, ≤ Tangent to Lateral ICA | Complete Possible |
| **Grade 2** | Tumour > Tangent to Lateral ICA, ≤ Tangent to Medial ICA | Partial Resection |
| **Grade 3** | Tumour > Tangent to Medial ICA, ≤ Tangent to MCC | Subtotal |
| **Grade 4** | Tumour > Tangent to MCC | Gross Total Unlikely |

## 3. Management Algorithm

```
Non-Functioning Pituitary Adenoma (Confirmed)
         │
         ├── MICROADENOMA (<10mm) + ASYMPTOMATIC
         │       → OBSERVATION
         │       MRI 6-12mo → Yearly ×2 → Then 2-5yr
         │       Hormonal Surveillance Annual
         │
         ├── MACROADENOMA (≥10mm) + ASYMPTOMATIC
         │       → SURGERY RECOMMENDED (High risk of future mass effect)
         │       Transsphenoidal Surgery (TSS)
         │
         ├── VISUAL FIELD DEFECTS (Any Size)
         │       → URGENT TRANSSEPHENOIDAL SURGERY (TSS)
         │
         ├── HYPOPITUITARISM (Symptomatic)
         │       → SURGERY (TSS) + Hormone Replacement
         │
         ├── CRANIAL NERVE PALSY (CN III/IV/VI)
         │       → SURGERY (TSS) ± Cavernous Sinus Exploration
         │
         └── RECURRENT / RESIDUAL POST-SURGERY
                 ├── ASYMPTOMATIC RESIDUAL (<1cm) → Observation
                 ├── SYMPTOMATIC / GROWING → Re-operation / Radiotherapy
                 └── NOT SURGICAL CANDIDATE → Stereotactic Radiosurgery (SRS) / Fractionated RT
```

---

## 2. Surgical Management

### Transsphenoidal Surgery (TSS) — Gold Standard
| Aspect | Details |
|--------|---------|
| **Approach** | Endoscopic Endonasal (Preferred) / Microscopic Sub-labial |
| **Goal** | **Gross Total Resection (GTR)** if Possible; Decompression if Cavernous Sinus Invasion |
| **Success Rates** | **GTR**: 70-80% (Grade 0-1); **Decompression**: 90-95% |
| **Complications** | CSF Leak (2-5%), DI (Transient 10-20%, Permanent 1-2%), New Hypopituitarism (5-10%), Epistaxis, Meningitis |
| **Post-op Monitoring** | Visual Fields (Pre/Post), Pituitary Hormones (Day 1, 3mo, 6mo, 1yr), MRI (3mo, 1yr, then yearly×5) |

### Cavernous Sinus Invasion (Knosp Classification)
| Grade | Description | Resection Goal |
|-------|-------------|----------------|
| **Grade 0** | Tumour ≤ Intercarotid Line | **Complete** |
| **Grade 1** | > Intercarotid, ≤ Tangent Lateral ICA | **Complete Possible** |
| **Grade 2** | > Tangent Lateral ICA, ≤ Tangent Medial ICA | **Subtotal** |
| **Grade 3** | > Tangent Medial, ≤ Tangent MCC | Subtotal |
| **Grade 4** | > Tangent MCC | **Decompression Only** |

**Knosp ≥3** → Subtotal resection expected; Adjuvant radiotherapy for residual.

---

## 3. Radiotherapy

| Indication | Type | Timing |
|------------|------|--------|
| **Residual/Recurrent Post-Surgery** | **Stereotactic Radiosurgery (SRS)** / **Fractionated RT** | After Surgical Recovery (3-6mo) |
| **Not Surgical Candidate** | SRS / Fractionated RT | Primary Treatment |
| **Aggressive/Atypical** | Fractionated RT | Adjuvant Post-Surgery |

| RT Type | Dose | Target | Complications |
|---------|------|--------|---------------|
| **SRS (Gamma Knife/CyberKnife)** | 12-15 Gy (Single) | Tumour Margin | Hypopituitarism (10-30% at 10yr), Optic Neuropathy (<1%) |
| **Fractionated RT** | 45-50 Gy / 25-28 Fractions | Tumour Bed | Lower Optic Risk; More Convenient for Large Residual |

**Radiotherapy Effect**: Tumour control 80-90% at 10yr; **Hypopituitarism** develops in 30-50% at 10-15 years.

---

## 4. Surveillance & Follow-up

### Post-Surgery Surveillance Schedule
| Timepoint | Investigations |
|-----------|----------------|
| **Pre-op** | Full Hormones, MRI, Visual Fields |
| **Day 1-2 Post-op** | Cortisol, Electrolytes, Glucose, Visual Fields |
| **6 Weeks** | Full Hormone Panel, Visual Fields |
| **3 Months** | Pituitary Hormones, MRI, Visual Fields |
| **6 Months** | Hormones, MRI (if residual) |
| **1 Year** | Hormones, MRI, Visual Fields |
| **Annually (Years 2-5)** | Hormones, MRI (Alternate Years), Visual Fields |
| **Long-term (>5yr)** | Hormones q2yr, MRI q2-3yr, Visual Fields q2yr |

### Hormonal Replacement Post-Surgery
| Deficiency | Replacement |
|------------|-------------|
| **ACTH/Cortisol** | Hydrocortisone 15-20mg AM + 5-10mg PM |
| **TSH/fT4** | Levothyroxine 1.6 µg/kg/day |
| **Gonadotrophs** | Testosterone / Oestradiol + Progestogen |
| **GH** | Consider if Severe GHD (ITT Peak <3 µg/L) |
| **Prolactin** | Not Replaced (Except Lactation) |

---

## 4. Recurrence & Long-term Outcomes

| Outcome | Rate |
|---------|-------|
| **Recurrence (GTR)** | 10-15% at 10 years |
| **Recurrence (Subtotal)** | 30-50% at 5 years |
| **New Hypopituitarism (Post-SRS)** | 5-10% |
| **New Hypopituitarism (Post-RT)** | 30-50% at 10-15 years |
| **Malignant Transformation** | Extremely Rare (<0.1%) |

## 10. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **NFPA vs Silent Corticotroph** | Silent Corticotroph = ACTH+ on IHC + Silent Cushing; NFPA = All Hormones Negative (except α-subunit/FSH/LH) |
| **NFPA vs Prolactinoma** | Prolactinoma: Prolactin >500 (Micro) / >5000 (Macro); NFPA: Prolactin <2000 (Stalk Effect) |
| **NFPA vs Functioning** | NFPA = No Syndrome; Functioning = Hormone Excess Syndrome |
| **Stalk Effect Prolactin** | <2000 mIU/L; Normalises Post-Surgery; No Galactorrhoea |
| **Gonadotroph vs Null Cell** | Gonadotroph = α-subunit/FSH/LH IHC Positive; Null Cell = All Negative |

**Mnemonic: NFPA MANAGEMENT**
- **N**on-functioning = **N**o Hormone Syndrome
- **F**irst: **F**unctioning Excluded (Normal Hormones)
- **P**resentation: **P**ressure (Visual, Headache, CN Palsy) + **P**ituitary Failure
- **A**lgorithm: Micro → Observe; Macro + Symptoms → **S**urgery (TSS)
- **R**ecurrence: **R**eceed if GTR; **R**ecur if Subtotal → RT

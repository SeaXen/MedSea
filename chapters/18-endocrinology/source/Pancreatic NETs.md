# Pancreatic Neuroendocrine Tumours (pNETs)

> [!info]
> **pNETs = Neuroendocrine Tumours of Pancreatic Origin.** **2nd Most Common Functional NET** (After Carcinoid). **Functional (Hormone-Secreting) vs Non-Functional.** **Insulinoma (Most Common Functional) > Gastrinoma > Glucagonoma > VIPoma > Somatostatinoma.** **Non-Functional = Majority (60-90%).** **MEN1 Association** (Multiple, Higher Malignant Potential).

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Classify pNETs by functionality and grade
- [ ] Recognise clinical syndromes (Insulinoma, Gastrinoma/ZES, Glucagonoma, VIPoma, Somatostatinoma)
- [ ] Apply localisation techniques (EUS, DOTATATE PET, CT/MRI)
- [ ] Apply management algorithms (Surgery vs Medical vs PRRT)
- [ ] Understand MEN1-associated pNETs

## 2. Classification

### By Functionality
| Category | Percentage | Examples |
|----------|-----------|----------|
| **Functional (Hormone-Secreting)** | **10-40%** | Insulinoma, Gastrinoma, Glucagonoma, VIPoma, Somatostatinoma, PPoma, ACTHoma |
| **Non-Functional** | **60-90%** | No Clinical Hormone Syndrome; Often Detected Incidentally or by Mass Effect |

### By Grade (WHO 2019)
| Grade | Ki-67 Index | Mitotic Count (per 2mm²) | Behaviour |
|-------|-------------|--------------------------|-----------|
| **NET G1** | **<3%** | **<2** | Well-Differentiated, Indolent |
| **NET G2** | **3-20%** | **2-20** | Well-Differentiated, Intermediate |
| **NEC G3** | **>20%** | **>20** | Poorly Differentiated, Aggressive |

## 2. Functional pNETs — Clinical Syndromes

### 1. Insulinoma (Most Common Functional pNET)
| Feature | Details |
|---------|---------|
| **Frequency** | **~40-50%** of Functional pNETs |
| **Pathophysiology** | **Autonomous Insulin Secretion** → Hypoglycaemia |
| **Whipple's Triad** | **Symptoms of Hypoglycaemia + Plasma Glucose <2.8 mmol/L + Relief with Glucose** |
| **Clinical** | Fasting Hypoglycaemia (Early Morning), Neuroglycopenic (Confusion, Seizures), Adrenergic (Sweating, Tremor) |
| **Diagnosis** | **72h Supervised Fast**: Glucose <2.8 mmol/L + Insulin ≥6 mU/L + C-Peptide ≥200 pmol/L + Proinsulin ≥5 pmol/L |
| **Localisation** | **EUS** (Sensitivity 90%), **⁶⁸Ga-DOTATATE PET/CT**, **MRI/MRCP**, **Selective Arterial Calcium Stimulation (SACI)** |
| **Treatment** | **Surgical Enucleation** (Curative, 90% Benign); **Diazoxide** (Medical Bridge); **Everolimus** (Unresectable) |

### 2. Gastrinoma (Zollinger-Ellison Syndrome)
| Feature | Details |
|---------|---------|
| **Frequency** | **20-30%** of Functional pNETs |
| **Pathophysiology** | **Gastrin Hypersecretion** → ↑ Gastric Acid → **Refractory PUD** |
| **Clinical** | **Refractory/Recurrent PUD**, Diarrhoea (Acid Inactivation of Lipase), Oesophagitis, Steatorrhoea |
| **Diagnosis** | **Fasting Gastrin >1000 pg/mL + Gastric pH <2** (or Secretin Stimulation: Gastrin Rise >120 pg/mL) |
| **Localisation** | **Duodenal > Pancreatic** (MEN1: Duodenal Predominant); **⁶⁸Ga-DOTATATE PET/CT**, **EUS**, **Secretin Stimulation Test** |
| **MEN1 Association** | **20-25%** of Gastrinomas; **Duodenal Wall** (Multiple, Small) |
| **Treatment** | **High-Dose PPI** (Omeprazole 40-80mg BD) + **Surgical Resection** (Duodenotomy + Enucleation) |

### 3. Glucagonoma
| Feature | Details |
|---------|---------|
| **Frequency** | **5-10%** of Functional pNETs |
| **Pathophysiology** | **Glucagon Hypersecretion** → ↑ Hepatic Gluconeogenesis/Glycogenolysis |
| **Clinical Triad** | **NME (Necrolytic Migratory Erythema)**, **Diabetes**, **Weight Loss** |
| **Other** | Glossitis, Angular Stomatitis, DVT (Hypercoagulable), Anaemia |
| **Diagnosis** | **Fasting Glucagon >500 pg/mL** (Normal <50-100) |
| **Treatment** | **Surgical Resection** (Curative); **Octreotide** (Symptom Control); **Zinc** (Skin); **Insulin** (Diabetes) |

### 4. VIPoma (Verner-Morrison Syndrome)
| Feature | Details |
|---------|---------|
| **Frequency** | **<5%** (Rare) |
| **Pathophysiology** | **VIP Hypersecretion** → ↑ cAMP → ↑ Intestinal Cl⁻/Water Secretion |
| **Clinical** | **WDHA Syndrome**: **Watery Diarrhoea** (Profuse, >1L/day), **Hypokalaemia**, **Achlorhydria** |
| **Diagnosis** | **VIP >75 pg/mL** (Normal <75) |
| **Treatment** | **Surgical Resection**; **Octreotide** (↓ VIP Secretion); **Loperamide**, **Fluid/Electrolyte Replacement** |

### 5. Somatostatinoma
| Feature | Details |
|---------|---------|
| **Frequency** | **<1%** (Rarest) |
| **Clinical** | **Diabetes** (↓ Insulin), **Steatorrhoea** (↓ Pancreatic Enzymes), **Gallstones** (↓ CCK), **Hypochlorhydria**, Weight Loss |
| **Diagnosis** | **Somatostatin >100 pg/mL** (Fasting) |
| **Association** | **NF1** (Associated), **MEN1** |

### 6. PPoma (Pancreatic Polypeptide)
| Feature | Details |
|--------|---------|
| **Frequency** | Rare |
| **Clinical** | Often Asymptomatic; Watery Diarrhoea, Weight Loss |
| **Diagnosis** | **PP >300 pmol/L** (Fasting) |
| **Treatment** | Surgical Resection if Symptomatic |

## 3. Non-Functional pNETs (NF-pNETs)

| Feature | Details |
|---------|---------|
| **Prevalence** | **60-90%** of pNETs |
| **Presentation** | Incidental (Imaging), Mass Effect (Pain, Obstruction, Jaundice), Metastatic (Liver) |
| **Biochemistry** | **Normal Hormones** (May Have ↑ PP, CgA) |
| **Grading** | G1/G2 (Well-Diff) vs G3 (Poorly Diff) |
| **Treatment** | **Surgery if >2cm / Symptomatic / Growing**; **Observation** if <1cm, G1, Asymptomatic |

## 3. Localisation — Imaging Modalities

| Modality | Sensitivity | Specificity | Role |
|----------|-------------|-------------|------|
| **EUS (Endoscopic US)** | **90-95%** | 90%+ | **Best for Small Lesions (<1cm); Guided FNA** |
| **⁶⁸Ga-DOTATATE PET/CT** | **90-95%** | 90%+ | **Gold Standard for Staging** (Whole Body, SSTR2+) |
| **MRI/MRCP (Pancreas Protocol)** | 85-90% | 90%+ | **Anatomic Detail**; Ductal Relationship |
| **CT (Multiphasic)** | 80-90% | 85%+ | **First-Line** (Accessibility) |
| **Selective Arterial Calcium Stimulation (SACI)** | 90%+ | 95%+ | **Insulinoma Localisation** (If Non-Localising) |

## 3. Management by Subtype

### Insulinoma
| Aspect | Details |
|---------|---------|
| **Surgery** | **Enucleation** (Curative; 90% Benign) |
| **Medical** | **Diazoxide** (↓ Insulin Release); **Everolimus** (Unresectable) |
| **Localisation** | EUS, SACI (Selective Arterial Calcium Stimulation) |

### Gastrinoma (ZES)
| Aspect | Details |
|--------|---------|
| **Surgery** | **Enucleation + Duodenotomy** (Duodenal Gastrinomas Often Small/Multiple) |
| **Medical** | **High-Dose PPI** (Omeprazole 40-80mg BD); **Octreotide** (Adjunct) |
| **MEN1** | Duodenal Gastrinomas (Multiple, Small) → **Longitudinal Duodenotomy** |

### Glucagonoma
| Aspect | Details |
|--------|---------|
| **Surgery** | **Resection** (Curative if Localised) |
| **Medical** | **Octreotide** (↓ Glucagon), **Insulin** (Diabetes), **Zinc** (Skin), **Anticoagulation** (DVT Prophylaxis) |

### VIPoma
| Aspect | Details |
|--------|---------|
| **Surgery** | Resection (Curative if Localised) |
| **Medical** | **Octreotide** (↓ VIP → ↓ Diarrhoea), **Loperamide**, **Fluid/Electrolyte Replacement** |

### Non-Functional pNETs
| Size / Grade | Management |
|-------------|------------|
| **<1cm, G1** | **Observation** (EUS/MRI q6-12mo) |
| **1-2cm, G1** | Consider Surgery (Patient Factors) |
| **>2cm or G2/G3** | **Surgical Resection** (Enucleation / Partial Pancreatectomy) |
| **Metastatic** | **SST Analogues (Octreotide/Lanreotide)** → **PRRT (¹⁷⁷Lu-DOTATATE)** → **Chemo (CAPTEM)** → **Everolimus / Sunitinib** |

## 4. MEN1-Associated pNETs

| Feature | MEN1-Associated | Sporadic |
|---------|-----------|----------|
| **Multiplicity** | **Multiple** (Often >3) | Usually Solitary |
| **Gastrinoma** | **Duodenal** (Multiple, Small) > Pancreatic | Pancreatic > Duodenal |
| **Multiplicity** | Common (Multiple Synchronou) | Rare |
| **Genetics** | **MEN1 Mutation** | Sporadic Mutations (MEN1, DAXX/ATRX, MEN1) |
| **Surgery** | **Enucleation + Duodenotomy** (Gastrinoma) | Standard Resection |

## 5. Treatment Algorithms

### Localised Disease
| Scenario | Management |
|-----------|------------|
| **Insulinoma** | **Enucleation** (Laparoscopic) |
| **Gastrinoma** | **Duodenotomy + Enucleation** (Multiple in MEN1) |
| **Glucagonoma / VIPoma / Somatostatinoma** | **Resection** (Distal Pancreatectomy if Body/Tail) |
| **Non-Functional >2cm or G2/G3** | **Resection** (Distal Pancreatectomy / Enucleation / Whipple) |

### Metastatic / Unresectable
| Line | Therapy |
|--------|----------|
| **1st Line** | **Somatostatin Analogues** (Octreotide LAR / Lanreotide) — Symptom + Tumour Control |
| **2nd Line** | **Everolimus** (mTOR Inhibitor) — RADIANT-3: PFS 11 vs 4.6mo |
| **2nd Line** | **Sunitinib** (TKI) — PFS 11.4 vs 5.5mo |
| **3rd Line** | **PRRT (¹⁷⁷Lu-DOTATATE)** — **NETTER-1**: PFS 65% vs 11% (G1/G2) |
| **Chemotherapy** | **CAPTEM** (Capecitabine + Temozolomide) — NEC G3 / High-Grade G2 |

---

## 4. PRRT (Peptide Receptor Radionuclide Therapy)

| Agent | **¹⁷⁷Lu-DOTATATE (Lutathera)** |
|--------|------------------------------|
| **Indication** | **SSTR2+ Progressive NET G1/G2** |
| **Protocol** | **7.4 GBq IV q8wk × 4 Cycles** + Amino Acid Infusion (Renal Protection) |
| **Efficacy (NETTER-1)** | **PFS 65% vs 11% (Octreotide LAR) at 20mo** |
| **Eligibility** | **SSTR2+ (Krenning ≥2), Progressive, G1/G2** |
| **Toxicity** | Nausea, Myelosuppression, Renal (Amino Acid Protection) |

## 9. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Insulinoma vs Factitious** | Insulinoma: C-Peptide ↑, Proinsulin ↑; Factitious: C-Peptide Low, Insulin High |
| **Gastrinoma vs ZES** | ZES = Gastrinoma Syndrome (PUD + Diarrhoea + Hypergastrinaemia) |
| **pNET vs Carcinoid** | pNET = Pancreatic Origin; Carcinoid = GI/Bronchial Origin |
| **Non-Functional pNET** | No Hormone Syndrome; Incidental / Mass Effect |
| **Gastrinoma Location** | **MEN1 = Duodenal (Multiple)**; Sporadic = Pancreatic |
| **Insulinoma Localisation** | **EUS (90%)** > **SACI** (Gold Standard Functional) |
| **Glucagonoma Triad** | **NME + Diabetes + Weight Loss** |
| **VIPoma** | **WDHA Syndrome** (Watery Diarrhoea, Hypokalaemia, Achlorhydria) |
| **Somatostatinoma** | Diabetes + Steatorrhoea + Gallstones + Hypochlorhydria |
| **PRRT** | ¹⁷⁷Lu-DOTATATE; NETTER-1: PFS 65% vs 11% (Octreotide) |
| **Everolimus/Sunitinib** | RADIANT-3: Everolimus PFS 11 vs 4.6mo; Sunitinib 11.4 vs 5.5mo |

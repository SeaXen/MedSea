# Anaplastic Thyroid Cancer (ATC)

> [!info]
> **Anaplastic Thyroid Cancer (ATC) = Undifferentiated Thyroid Carcinoma.** **Rare (1-2% of Thyroid Cancers) but Highly Aggressive.** **Median Survival 3-6 Months.** **Presentation: Rapidly Growing Neck Mass, Airway Compromise, Dysphagia, Hoarseness.** **Multimodal Therapy (Surgery + RT + Chemo) — Palliative in Most.**

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise clinical presentation and differentiate from other thyroid malignancies
- [ ] Apply staging (All Stage IV) and prognostic factors
- [ ] Outline multimodal management algorithm
- [ ] Manage airway emergencies and palliative care

---

## 2. Epidemiology & Pathophysiology

| Feature | Details |
|---------|---------|
| **Incidence** | **1-2%** of Thyroid Cancers |
| **Age** | **>60 Years** (Mean 70) |
| **Sex** | Female > Male (2-3:1) |
| **Risk Factors** | Pre-existing DTC (Dedifferentiation), Radiation Exposure, Goitre, Iodine Deficiency |
| **Genetics** | **TP53** (80%), TERT Promoter, PI3K/AKT, RAS, BRAF (From DTC Background) |

---

## 2. Clinical Presentation

| Feature | Frequency | Details |
|---------|-----------|---------|
| **Rapidly Growing Neck Mass** | **90-100%** | Weeks Duration; Hard, Fixed, Infiltrative |
| **Airway Compromise** | **50-70%** | Stridor, Dyspnoea, Orthopnoea |
| **Dysphagia** | 40-60% | Oesophageal Compression |
| **Hoarseness** | 30-50% | Recurrent Laryngeal Nerve Invasion |
| **Cervical Lymphadenopathy** | 40-60% | Fixed, Matted Nodes |
| **Distant Mets at Diagnosis** | **40-50%** | Lung (90%), Bone, Brain, Liver |
| **Thyroid Storm / Thyrotoxicosis** | Rare | Ectopic TSH/Thyroid Hormone Secretion |

---

## 3. Staging (AJCC 8th) — All Stage IV

| Stage Group | Criteria |
|-------------|----------|
| **IVA** | T4a (Intrathyroidal, Resectable), N0/N1, M0 |
| **IVB** | T4b (Gross Extrathyroidal Extension, Unresectable), Any N, M0 |
| **IVC** | Any T, Any N, **M1** |

---

## 3. Diagnostic Workup

| Investigation | Purpose |
|---------------|---------|
| **FNA Core Biopsy** | **Diagnosis** (Undifferentiated, Pleomorphic, Giant Cells) |
| **IHC** | **Negative**: Thyroglobulin, TTF-1, Calcitonin; **Positive**: Vimentin, p53, Ki-67 (High) |
| **CT Neck/Chest** | Local Extent, Nodal Involvement, Lung Mets |
| **MRI Neck** | Soft Tissue Invasion (Trachea, Oesophagus, Vessels, Nerves) |
| **PET-CT** | Distant Staging (High FDG Avidity) |
| **Laryngoscopy** | Vocal Cord Function (Pre-Op) |

---

## 3. Management Algorithm

```
ATC Diagnosed (FNA + IHC)
         │
         ▼
ASSESS RESPECTABILITY + AIRWAY STATUS
         │
         ├── AIRWAY COMPROMISE
         │       → **TRACHEOSTOMY** (Urgent) / INTUBATION
         │
         ├── POTENTIALLY RESECTABLE (T4a, No Major Vessel Encasement)
         │       → **TOTAL THYROIDECTOMY + CND + MODIFIED RADICAL ND** (If Nodes)
         │       → **ADJUVANT RT** (60-66 Gy) + **CONCURRENT CHEMO** (Doxorubicin/Cisplatin)
         │
         ├── UNRESECTABLE LOCAL (T4b) / NODES / DISTANT METS
         │       → **CONCURRENT CHEMORADIATION** (Definitive)
         │       │       RT: 60-66 Gy / 30-33 Fractions
         │       │       Chemo: **Doxorubicin 60mg/m² q3wk** OR **Cisplatin 40mg/m² Weekly**
         │       │
         │       └── **SYSTEMIC THERAPY** (If Metastatic/Progression)
         │               **Doxorubicin** (Single Agent / Combo)
         │               **Cisplatin** (Single Agent / Combo)
         │               **Paclitaxel / Docetaxel** (Weekly)
         │               **Lenvatinib / Sorafenib** (TKI - Emerging)
         │               **Immunotherapy** (Pembrolizumab - TMB-High/MSI-H)
```

---

## 4. Treatment Modalities

### Surgery
| Indication | Procedure |
|------------|-----------|
| **Resectable (T4a)** | **Total Thyroidectomy** + **Central Neck Dissection** + **Modified Radical Neck Dissection** (If Nodes) |
| **Airway Emergency** | **Tracheostomy** (Preferred Over Intubation for Long-Term) |
| **Palliative** | Debulking for Airway/Oesophageal Compression |

### Radiation Therapy (RT)
| Setting | Dose / Fractionation |
|---------|---------------------|
| **Adjuvant (Post-Op)** | **60-66 Gy / 30-33 Fractions** (Concurrent Chemo) |
| **Definitive (Unresectable)** | **60-66 Gy / 30-33 Fractions** (Concurrent Chemo) |
| **Palliative** | **30-40 Gy / 10-15 Fractions** (Bleeding, Airway, Pain) |

### Chemotherapy
| Regimen | Dose / Schedule | Setting |
|---------|-----------------|---------|
| **Doxorubicin** | 60-75 mg/m² IV q3wk | **Standard Single Agent** |
| **Cisplatin** | 40 mg/m² IV Weekly (Concurrent RT) | **Concurrent Chemoradiation** |
| **Paclitaxel** | 80 mg/m² Weekly | Alternative/Tolerable |
| **Docetaxel** | 75 mg/m² q3wk | Alternative |
| **Cisplatin + Doxorubicin** | Cisplatin 40mg/m² D1 + Doxorubicin 50mg/m² D1 q3wk | Combination (Higher Toxicity) |

### Targeted Therapy / Immunotherapy
| Agent | Indication |
|-------|------------|
| **Lenvatinib** | Progressive, Metastatic (Off-Label/Emerging) |
| **Sorafenib** | Progressive, Metastatic |
| **Pembrolizumab** | **TMB-High / MSI-H / dMMR** (Subset of ATC) |
| **Dabrafenib + Trametinib** | **BRAF V600E Mutant** (Off-Label) |

---

## 4. Airway Management (Critical)

| Scenario | Action |
|----------|--------|
| **Impending Obstruction** | **Awake Fibreoptic Intubation** or **Tracheostomy** (Under Local) |
| **Complete Obstruction** | **Emergency Tracheostomy** / Cricothyroidotomy |
| **Post-Op** | **Tracheostomy Often Required** (Swelling, Nerve Injury) |

---

## 5. Prognosis & Survival

| Parameter | Value |
|-----------|-------|
| **Median Survival** | **3-6 Months** |
| **1-Year Survival** | **20-30%** |
| **5-Year Survival** | **<5%** |
| **Better Prognosis** | <60 Years, Small Tumour (<5cm), Resectable, No Distant Mets |
| **Worse Prognosis** | >70 Years, T4b, Distant Mets, High LDH, Leukocytosis |

---

## 5. Palliative & End-of-Life Care

| Aspect | Management |
|--------|------------|
| **Pain** | Opioid Ladder; Radiotherapy (Bone Mets) |
| **Dyspnoea** | Opioids, Benzodiazepines, Oxygen, Palliative RT |
| **Dysphagia** | NG Tube / PEG; Palliative RT / Stent |
| **Bleeding** | Palliative RT; Tranexamic Acid; Embolisation |
| **Tracheostomy Care** | Humidification, Suctioning, Speech Therapy |

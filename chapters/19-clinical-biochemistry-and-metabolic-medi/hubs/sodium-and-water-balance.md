## Clinical Algorithm: Sodium Disorders

```
SERUM SODIUM DISORDER
         │
         ├── HYPONATRAEMIA (Na⁺ <135 mmol/L)
         │       │
         │       ├── ASSESS VOLUME STATUS
         │       │       ├── HYPOVOLAEMIC → Urine Na⁺ >20 (Renal) vs <20 (Extra-renal)
         │       │       │       → Saline Resuscitation
         │       │       │
         │       ├── EUVOLAEMIC → Urine Osmo >100 + Urine Na⁺ >20
         │       │       ├── TSH/Cortisol Normal → **SIADH**
         │       │       ├── **CSW** → Urine Na⁺ High + Hypovolaemic Signs
         │       │       ├── **Hypothyroidism** / **Adrenal Insufficiency** / **Drugs**
         │       │       │       → Fluid Restriction 1L/d → Demeclocycline → Tolvaptan
         │       │       │
         │       └── HYPERVOLAEMIC → Urine Na⁺ <20 (usually)
         │               → Heart Failure / Cirrhosis / Nephrotic Syndrome
         │               → Fluid Restriction + Diuretics
         │
         └── HYPERNATRAEMIA (Na⁺ >145 mmol/L)
                 │
                 ├── ASSESS VOLUME STATUS
                 │       ├── HYPOVOLAEMIC → Water Loss > Na⁺ Loss
                 │       ├── EUVOLAEMIC → Water Deprivation / Diabetes Insipidus
                 │       └── HYPERVOLAEMIC → Sodium Gain (Hypertonic Saline, Bicarbonate)
                 │
                 ├── WATER DEPRIVATION TEST → Desmopressin Challenge
                 │       ├── Central DI → Responds to Desmopressin (Urine Osmo ↑ >50%)
                 │       └── Nephrogenic DI → No Response
                 │
                 └── MANAGEMENT
                         ├── Free Water Deficit = TBW × (Current Na⁺/140 - 1)
                         ├── Replace over 24-48h (0.9% Na⁺ or 5% Dextrose)
                         ├── Central DI → Desmopressin (Intranasal/PO/IV)
                         └── Nephrogenic DI → Thiazide + Amiloride + Low Salt Diet
```

## Key Formulae & Cut-offs

| Parameter | Formula / Value |
|-----------|-----------------|
| **Free Water Deficit** | TBW × (Current Na⁺/140 - 1) |
| **Corrected Na⁺ (Hyperglycaemia)** | Measured Na⁺ + 1.6 × (Glucose - 5.6)/5.6 |
| **SIADH Criteria** | Euvolaemic + Urine Osmo >100 + Urine Na⁺ >20 + Normal Adrenal/Thyroid |
| **CSW vs SIADH** | CSW = Hypovolaemic; SIADH = Euvolaemic |
| **Water Deficit (DI)** | 4 mL/kg × (Serum Osmo - 285) |
| **Hypernatraemia Correction** | Max 10-12 mmol/L/24h (Avoid Cerebral Oedema) |
| **Hyponatraemia Correction** | Max 8-10 mmol/L/24h (Avoid ODS); 12-15 mmol/L if Severe Symptoms |

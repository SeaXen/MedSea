## Clinical Algorithms

### Obesity & Metabolic Syndrome

```
OBESITY & METABOLIC SYNDROME
         │
         ├── DIAGNOSIS
         │       ├── **BMI**: Overweight 25-29.9; Obese Class I 30-34.9; Class II 35-39.9; Class III ≥40
         │       ├── **Waist Circumference**: Male ≥102cm / Female ≥88cm (Europid); Lower for Asian
         │       ├── **Metabolic Syndrome** (IDF/ATP III): ≥3 of 5
         │       │       ├── Central Obesity (Waist)
         │       │       ├── TG ≥1.7 mmol/L (or on Rx)
         │       │       ├── HDL <1.0 (M) / <1.3 (F) mmol/L (or on Rx)
         │       │       ├── BP ≥130/85 (or on Rx)
         │       │       └── Fasting Glucose ≥5.6 mmol/L (or on Rx)
         │       └── **Exclude Secondary Causes**: Hypothyroidism, Cushing's, PCOS, Drugs (Antipsychotics, Steroids)
         │
         ├── MANAGEMENT
         │       ├── **LIFESTYLE (1st Line)**
         │       │       ├── Diet: 600 kcal/d Deficit (Low Fat/Calorie, Mediterranean, Low GI)
         │       │       ├── Exercise: 150-300 min/wk Moderate + Resistance 2x/wk
         │       │       ├── Behavioural: CBT, Self-Monitoring, Goal Setting
         │       │       └── Target: 5-10% Weight Loss at 6mo (Improves Metabolic Parameters)
         │       │
         │       ├── **PHARMACOTHERAPY** (BMI ≥30 or ≥27 + Comorbidity)
         │       │       ├── **Orlistat** 120mg TDS (Lipase Inhibitor; GI SE)
         │       │       ├── **Liraglutide 3.0mg SC Daily** (GLP-1 RA; Nausea, Pancreatitis Risk)
         │       │       ├── **Semaglutide 2.4mg SC Weekly** (GLP-1 RA; >15% Weight Loss)
         │       │       ├── **Naltrexone/Bupropion** (Craving Reduction; Nausea, Seizure Risk)
         │       │       └── **Phentermine/Topiramate** (Appetite Suppression; Tachycardia, Teratogenic)
         │       │
         │       └── **BARIATRIC SURGERY** (BMI ≥40 or ≥35 + Comorbidity)
         │               ├── **Sleeve Gastrectomy** (Most Common; Restrictive)
         │               ├── **RYGB** (Roux-en-Y; Restrictive + Malabsorptive)
         │               ├── **OAGB** (One-Anastomosis Gastric Bypass)
         │               └── **Follow-Up**: Nutritional Monitoring (Fe, B12, D, Ca), 25-OH D, PTH, LFTs
         │
         └── METABOLIC SYNDROME MANAGEMENT
                 ├── Treat Each Component
                 ├── Intensive Lifestyle (Diet + Exercise)
                 ├── Statin (If CVD Risk >10%), ACEi/ARB (If HTN), Metformin (If Pre-DM/DM)
                 └── Weight Loss → Improves All Components
```

### Lipid Disorders

```
LIPID DISORDERS (DYSLIPIDAEMIA)
         │
         ├── CLASSIFICATION (Fredrickson/WHO)
         │       ├── **Type IIa** (Familial Hypercholesterolaemia): ↑ LDL-C, Normal TG
         │       ├── **Type IIb** (Combined): ↑ LDL-C, ↑ TG
         │       ├── **Type IV** (Familial Combined / Metabolic): Normal LDL, ↑ TG, ↓ HDL
         │       ├── **Type V** (Severe Hypertriglyceridaemia): ↑↑ TG, ↑ Chylomicrons
         │       └── **Familial Hypoalphalipoproteinaemia**: Low HDL, Normal LDL/TG
         │
         ├── TARGETS (ESC/EAS 2019 / NICE)
         │       ├── **Very High Risk** (Established CVD, Diabetes + Target Organ, CKD, FH): **LDL-C <1.4 mmol/L (55 mg/dL) & ≥50% Reduction**
         │       ├── **High Risk** (Diabetes, CKD, >10yr CVD Risk): **LDL-C <1.8 mmol/L (70 mg/dL) & ≥50% Reduction**
         │       ├── **Moderate Risk**: **LDL-C <2.6 mmol/L (100 mg/dL)**
         │       └── **Low Risk**: **LDL-C <3.0 mmol/L (115 mg/dL)**
         │
         ├── MANAGEMENT
         │       ├── **STATIN (1st Line)**
         │       │       ├── High-Intensity (Atorvastatin 40-80mg / Rosuvastatin 20-40mg) → ≥50% LDL Reduction
         │       │       └── Moderate-Intensity (Atorva 10-20mg / Rosuva 5-10mg) → 30-49% Reduction
         │       │
         │       ├── **ADD-ON (If Target Not Met / Intolerant)**
         │       │       ├── **Ezetimibe 10mg OD** (+15-20% LDL Reduction)
         │       │       ├── **PCSK9 Inhibitor** (Evolocumab 140mg SC 2wk / Alirocumab 75-150mg SC 2wk) → -50-60%
         │       │       ├── **Bempedoic Acid** 180mg OD (If Statin Intolerant) → -15-25%
         │       │       └── **Fibrates** (Fenofibrate 145mg) → TG ↓, HDL ↑ (If TG >2.3)
         │       │
         │       └── **SEVERE HYPERTRIGLYCERIDAEMIA** (TG >10 mmol/L)
         │               ├── **Fenofibrate** 145mg OD + Omega-3 (2-4g/d EPA/DHA)
         │               ├── **Insulin Infusion** (Acute Pancreatitis Prevention)
         │               └── **Plasmapheresis** (Refractory, TG >20 mmol/L)
         │
         └── FAMILIAL HYPERCHOLESTEROLAEMIA (FH)
                 ├── **Heterozygous** (1:250): LDL-C >4.9 (Adults) / >4.0 (Children); Tendon Xanthomas, Corneal Arcus
                 ├── **Homozygous** (1:300,000): LDL-C >13; Early CVD, Planar Xanthomas
                 ├── **Genetics**: LDLR (85-90%), APOB, PCSK9
                 └── **Management**: High-Intensity Statin + Ezetimibe → PCSK9 Inhibitor → Lipid Apheresis (HoFH)
```

### Porphyria

```
PORPHYRIA
         │
         ├── CLASSIFICATION
         │       ├── **ACUTE HEPATIC PORPHYRIAS** (Neurovisceral)
         │       │       ├── **AIP** (Acute Intermittent Porphyria) - HMBS Mutation (Most Common)
         │       │       ├── **VP** (Variegate Porphyria) - PPOX Mutation (Acute + Cutaneous)
         │       │       ├── **HCP** (Hereditary Coproporphyria) - CPOX Mutation
         │       │       └── **ADP** (ALAD Deficiency Porphyria) - ALAD Mutation (Ultra-Rare)
         │       │
         │       └── **CUTANEOUS PORPHYRIAS** (Photoactive)
         │               ├── **PCT** (Porphyria Cutanea Tarda) - UROD (Most Common; Hepatic Iron Overload)
         │               ├── **EPP** (Erythropoietic Protoporphyria) - FECH (Ferrochelatase)
         │               ├── **CEP** (Congenital Erythropoietic Porphyria) - UROS (Ultra-Rare)
         │               └── **XLEPP** (X-Linked EPP) - ALAS2
         │
         ├── DIAGNOSIS
         │       ├── **ACUTE ATTACK**
         │       │       ├── **Urine PBG** (↑↑↑) — **Gold Standard Screening**
         │       │       ├── **Urine ALA** (↑)
         │       │       ├── **Urine Porphyrins** (↑ Copro/Uro)
         │       │       ├── **Faecal Porphyrins** (VP/HCP Differentiation)
         │       │       └── **Genetic Testing** (Confirmatory, Family Screening)
         │       │
         │       └── **CUTANEOUS**
         │               ├── Urine/Faecal Porphyrin Profile
         │               └── Plasma Fluorescence Emission (CEP, EPP)
         │
         ├── ACUTE ATTACK MANAGEMENT
         │       ├── **STOP PRECIPITATING DRUGS** (Barbiturates, Sulfonamides, Rifampicin, Phenytoin, Valproate, OCP)
         │       ├── **HAEMIN ARGINATE** 3-4mg/kg IV Daily × 4 Days (Replenishes Haem Pool, Represses ALAS1)
         │       ├── **IV GLUCOSE** 300-400g/d (5-10% Dextrose) — Represses ALAS1
         │       ├── **SUPPORTIVE**: Analgesia (Opioids), Antiemetics, Hyponatraemia Correction, Seizure Prophylaxis (Avoid Phenytoine/Valproate → Levetiracetam)
         │       └── **AVOID**: Barbiturates, Sulfonamides, Ergots, Rifampicin, Phenytoin, Valproate, Oestrogen
         │
         ├── PCT MANAGEMENT
         │       ├── **PHLEBOTOMY** 400-500mL q2wk × Target Ferritin 20-50 µg/L (1st Line)
         │       ├── **LOW-DOSE CHLOROQUINE/HYDROXYCHLOROQUINE** 125mg 2x/week (Mobilises Hepatic Porphyrins)
         │       ├── **AVOID**: Alcohol, Oestrogens, Iron Supplements
         │       └── **TREAT HCV/HIV** (If Co-Infection)
         │
         └── CUTANEOUS MANAGEMENT (EPP/CEP/XLEPP)
                 ├── **PHOTOPROTECTION** (UV-Blocking Clothing, Sunscreens, Window Films)
                 ├── **BETA-CAROTENE** / **LUTEIN** / **N-ACETYLCYSTEINE** (EPP)
                 ├── **AFERESIS / BONE MARROW TRANSPLANT** (Severe CEP)
                 └── **LIVER TRANSPLANT** (End-Stage Hepatic Disease)
```

### Wilson Disease

```
WILSON DISEASE (Hepatolenticular Degeneration)
         │
         ├── GENETICS
         │       ├── **ATP7B** Mutation (Chromosome 13) → Impaired Copper Transport Into Bile / Ceruloplasmin
         │       ├── **Autosomal Recessive** (Carrier 1:100)
         │
         ├── CLINICAL PRESENTATION
         │       ├── **Hepatic** (Childhood/Young Adult): Acute Hepatitis, Chronic Hepatitis, Cirrhosis, Fulminant Hepatic Failure
         │       ├── **Neuropsychiatric** (Young Adult): Tremor (Wing-Beating), Dystonia, Dysarthria, Dementia, Psychiatric
         │       ├── **Ocular**: **Kayser-Fleischer Rings** (Corneal Copper Deposition; Slit-Lamp), Sunflower Cataract
         │       └── **Other**: Renal (Fanconi), Cardiac (Cardiomyopathy), Haematological (Coombs-Negative Haemolytic Anaemia)
         │
         ├── DIAGNOSIS
         │       ├── **Ceruloplasmin** <0.2 g/L (Low; Acute Phase Reactant → Can Be Normal in Acute Inflammation)
         │       ├── **24h Urinary Copper** >100 µg/d (↑↑; >1.6 µmol/24h)
         │       ├── **Kayser-Fleischer Rings** (Slit-Lamp; Present in 95% Neuro, 50% Hepatic)
         │       ├── **Liver Copper** >250 µg/g Dry Weight (Gold Standard; Biopsy)
         │       ├── **Genetic Testing** (ATP7B Mutations; Confirmatory, Family Screening)
         │       └── **Leipzig Score** (≥4 = Diagnosis)
         │
         ├── MANAGEMENT
         │       ├── **LIFELONG CHELATION** (Unless Transplanted)
         │       │       ├── **Penicillamine** 500-1500mg/d (1st Line; Cu Chelation; SE: Proteinuria, Neutropenia, Lupus, Skin)
         │       │       ├── **Trientine** 600-1200mg/d (2nd Line; Fewer SE; No Cross-Reaction with Penicillamine)
         │       │       └── **Zinc Acetate** 50mg TDS (Maintenance/Initial if Mild; Blocks Intestinal Cu Absorption)
         │       │
         │       ├── **MONITORING**
         │       │       ├── 24h Urine Cu (Target 200-500 µg/24h on Chelation)
         │       │       ├── Ceruloplasmin, LFTs, FBC, U&Es, Renal Function q3-6mo
         │       │       └── **Non-Ceruloplasmin Copper** = Total Cu - (3 × Ceruloplasmin) → Target <10 µmol/L
         │       │
         │       ├── **DIET**: Low Copper (Avoid Liver, Shellfish, Nuts, Chocolate, Mushrooms)
         │       └── **LIVER TRANSPLANT** (Fulminant Hepatic Failure / Decompensated Cirrhosis / Refractory Neuro)
```

### Haemochromatosis

```
HAEMOCHROMATOSIS (Iron Overload)
         │
         ├── CLASSIFICATION
         │       ├── **HFE Haemochromatosis** (Type 1) - **C282Y Homozygous** (90%); H63D Compound Het
         │       ├── **Non-HFE** (Types 2-4): Juvenile (HJV/HAMP), TFR2, Ferroportin (SLC40A1)
         │       └── **Secondary**: Transfusional, Ineffective Erythropoiesis (Thalassaemia, Sideroblastic)
         │
         ├── PATHOPHYSIOLOGY
         │       ├── **HFE Mutation** → ↓ Hepcidin Expression → ↑ Ferroportin → ↑ Intestinal Fe Absorption → Iron Overload
         │       └── **Organ Deposition**: Liver (Cirrhosis, HCC), Pancreas (Diabetes), Heart (Cardiomyopathy), Joints (Arthropathy), Skin (Bronzing), Pituitary (Hypogonadism)
         │
         ├── DIAGNOSIS
         │       ├── **Genetic Testing**: **HFE C282Y Homozygous** (Diagnostic in Adults with Biochemical Iron Overload)
         │       ├── **Biochemical**: **Transferrin Saturation (TSAT) >45%** (Early); **Ferritin >300 (M) / >200 (F) µg/L** (Iron Overload)
         │       ├── **MRI Liver** (FerriScan/R2* — Quantifies Hepatic Iron Concentration)
         │       ├── **Liver Biopsy** (If Non-HFE / Ferritin >1000 / Diagnostic Uncertainty): HIC >1.9 (Mild), >3.8 (Moderate), >7.0 (Severe) mg/g DW
         │
         ├── MANAGEMENT
         │       ├── **VENESECTION (PHLEBOTOMY)** — **Gold Standard**
         │       │       ├── **Induction**: 400-500mL Weekly/Biweekly → Target Ferritin 50-100 µg/L, TSAT <50%
         │       │       └── **Maintenance**: 400-500mL q3-4mo (Lifelong)
         │       │
         │       ├── **CHELATION** (If Venesection Contraindicated: Anaemia, Cardiac Failure, Poor Access)
         │       │       └── **Deferasirox** 10-20mg/kg OD / **Deferiprone** 75mg/kg TDS / **Deferoxamine** SC/IV
         │       │
         │       ├── **SCREENING & SURVEILLANCE**
         │       │       ├── 1st-Degree Relatives: HFE Genotyping + TSAT/Ferritin
         │       │       ├── **HCC Surveillance**: US 6-Monthly (If Cirrhosis)
         │       │       ├── Cardiac Echo (If Cardiomyopathy Suspected)
         │       │       └── Endocrine (Diabetes, Hypogonadism, Thyroid)
         │       │
         │       └── **COMPLICATIONS MANAGEMENT**
         │               ├── **Diabetes**: Insulin (Often Required)
         │               ├── **Cardiomyopathy**: Standard HF Rx (ACEi, β-Blocker, MRA, SGLT2i)
         │               ├── **Hypogonadism**: Testosterone/HRT
         │               └── **Arthropathy**: NSAIDs, Joint Replacement (If Severe)
```

### Amyloidosis

```
AMYLOIDOSIS
         │
         ├── CLASSIFICATION
         │       ├── **AL Amyloidosis** (Primary) - **Light Chain** (Plasma Cell Dyscrasia / Myeloma-Associated)
         │       ├── **AA Amyloidosis** (Secondary) - **Serum Amyloid A** (Chronic Inflammation: RA, IBD, TB, FMF)
         │       ├── **ATTR Amyloidosis** (Hereditary) - **Transthyretin Mutations** (Neuropathy/Cardiomyopathy)
         │       ├── **ATTRwt** (Wild-Type) - **Senile Systemic Amyloidosis** (Elderly Males; Cardiac)
         │       └── **Other**: ALECT2, APOAI/II, Fibrinogen, Lysozyme
         │
         ├── CLINICAL MANIFESTATIONS
         │       ├── **Renal**: Nephrotic Syndrome, CKD (AL, AA, ATTR)
         │       ├── **Cardiac**: Restrictive Cardiomyopathy, HFpEF, Arrhythmias, Conduction Disease (AL, ATTR)
         │       ├── **Neurological**: Peripheral Neuropathy (Autonomic/Sensory), Carpal Tunnel (ATTR)
         │       ├── **Hepatic**: Hepatomegaly, Cholestasis (AL, AA)
         │       ├── **GI**: Macroglossia, Motility Disorders, Bleeding (AL)
         │       ├── **Soft Tissue**: Periorbital Purpura, Shoulder Pad Sign (AL)
         │       └── **Haematological**: Coagulopathy (Factor X Deficiency in AL)
         │
         ├── DIAGNOSIS
         │       ├── **Tissue Biopsy + Congo Red Staining** (Apple-Green Birefringence) — **Gold Standard**
         │       ├── **Mass Spectrometry / Immunohistochemistry** (Typing: AL vs AA vs ATTR)
         │       ├── **Serum Free Light Chains (FLC)** κ/λ Ratio (AL: Abnormal Ratio; Normal Total)
         │       ├── **Bone Marrow Biopsy** (AL: Clonal Plasma Cells)
         │       ├── **Genetic Testing** (ATTR: TTR Mutations; >120 Known)
         │       ├── **Cardiac MRI / Echo** (Late Gadolinium Enhancement, Granular Sparkling)
         │       └── **PYP/DPD Scintigraphy** (ATTR Cardiac: Grade 1-3 Uptake; Non-Invasive Diagnosis)
         │
         ├── MANAGEMENT
         │       ├── **AL AMYLOIDOSIS**
         │       │       ├── **Plasma Cell-Directed Therapy** (Bortezomib-Based: CyBorD, Dara-CyBorD)
         │       │       ├── **Autologous SCT** (If Eligible: Age <70, Good PS, ≤2 Organ Involvement)
         │       │       └── **Supportive**: Diuretics, ACEi/ARB (Careful), Compression Stockings
         │       │
         │       ├── **AA AMYLOIDOSIS**
         │       │       ├── **Treat Underlying Inflammation** (TNF Inhibitors, IL-1/6 Blockers, Colchicine for FMF)
         │       │       └── Colchicine (FMF Prophylaxis)
         │       │
         │       ├── **ATTR AMYLOIDOSIS** (Hereditary & Wild-Type)
         │       │       ├── **TTR Stabilisers**: **Tafamidis** 61mg OD (1st Line; Cardiomyopathy), **Diflunisal** (Neuropathy)
         │       │       ├── **Gene Silencers**: **Patisiran** (siRNA IV q3wk), **Inotersen** (ASO SC Weekly), **Vutrisiran** (siRNA SC q3mo)
         │       │       ├── **Liver Transplant** (Hereditary ATTR; Curative for Variant Production)
         │       │       └── **Heart Transplant** (Cardiac ATTR)
         │       │
         │       └── **SUPPORTIVE CARE (All Types)**
         │               ├── Heart Failure Rx (Diuretics, Avoid β-Blockers Initially in ATTR)
         │               ├── Autonomic Neuropathy (Fludrocortisone, Midodrine, Compression)
         │               ├── Renal (Dialysis if ESRD)
         │               └── **Palliative Care** (Advanced Disease)
```

## Key Formulae & Cut-offs

| Parameter | Target / Value |
|-----------|----------------|
| **BMI Classification** | Overweight 25-29.9; Obese I 30-34.9; II 35-39.9; III ≥40 |
| **Metabolic Syndrome (IDF)** | Central Obesity + ≥2 of: TG ≥1.7, HDL <1.0/1.3, BP ≥130/85, Glucose ≥5.6 |
| **LDL-C Targets (ESC)** | Very High Risk <1.4; High Risk <1.8; Mod <2.6; Low <3.0 mmol/L (±50% Reduction) |
| **Statin Intensity** | High: Atorva 40-80/Rosuva 20-40; Mod: Atorva 10-20/Rosuva 5-10 |
| **Ezetimibe** | 10mg OD (+15-20% LDL-C ↓) |
| **PCSK9 Inhibitors** | Evolocumab 140mg SC 2wk; Alirocumab 75-150mg SC 2wk (-50-60% LDL-C) |
| **Porphyria Screening** | **Urine PBG** (↑↑↑) — Gold Standard for Acute Attack |
| **Haemarginate** | 3-4mg/kg IV Daily × 4 Days (Acute Porphyria) |
| **Wilson Ceruloplasmin** | <0.2 g/L (Low); Can Be Normal in Acute Phase |
| **Wilson 24h Urine Cu** | >100 µg/d (↑↑) |
| **Leipzig Score** | ≥4 = Diagnosis (Ceruloplasmin, Urine Cu, KF Rings, Liver Cu, Genetics) |
| **Penicillamine** | 500-1500mg/d (Chelation); Monitor Proteinuria, FBC, LFTs |
| **HFE Haemochromatosis** | C282Y Homozygous (90%); TSAT >45%; Ferritin >300/200 |
| **Venesection Target** | Ferritin 50-100 µg/L; TSAT <50% |
| **Amyloid Typing** | Mass Spec / IHC on Biopsy (AL vs AA vs ATTR) |
| **AL Amyloidosis** | FLC κ/λ Ratio Abnormal; Bortezomib-Based ± Auto-SCT |
| **ATTR Cardiomyopathy** | Tafamidis 61mg OD (1st Line); Patisiran/Inotersen/Vutrisiran (Gene Silencing) |
| **PYP Scan** | ATTR Cardiac: Grade 1-3 Uptake (Non-Invasive Diagnosis) |

## Clinical Algorithms

### Hypocalcaemia & Hypercalcaemia

```
SERUM CALCIUM DISORDER
         │
         ├── HYPOCALCAEMIA (Ionised Ca²⁺ <1.1 mmol/L / Corrected <2.1 mmol/L)
         │       │
         │       ├── ACUTE (Tetany, Seizures, Laryngospasm)
         │       │       └── **IV Calcium Gluconate 10% 10-20mL over 10 min** (Cardiac Monitor)
         │       │
         │       └── CHRONIC MANAGEMENT
         │               ├── Oral Calcium 1-2g/d (Elemental) + **Alfacalcidol/Calcitriol**
         │               ├── Target Ionised Ca²⁺ 1.1-1.3 mmol/L
         │               └── Monitor: Ca²⁺, Phosphate, 24h Urine Ca, Renal Function
         │
         └── HYPERCALCAEMIA (Corrected Ca²⁺ >2.6 mmol/L / Ionised >1.3)
                 │
                 ├── PTH LEVEL
                 │       ├── **PTH SUPPRESSED** → Malignancy / Granulomatous / Vit D Toxicity / Thyrotoxicosis
                 │       │       ├── PTHrP Positive → **Humoral Hypercalcaemia of Malignancy**
                 │       │       ├── 1,25-(OH)₂D High → Sarcoid / TB / Lymphoma
                 │       │       └── Management: Hydration → Bisphosphonate / Denosumab
                 │       │
                 │       ├── **PTH ELEVATED** → Primary / Tertiary Hyperparathyroidism
                 │       │       ├── Localisation (Sestamibi, US, 4D-CT)
                 │       │       └── Surgery Indicated (Age <50, Ca >2.85, Stones, Bone, Renal)
                 │       │
                 │       └── **PTH INAPPROPRIATELY NORMAL** → FHH / Lithium / Thiazides
                 │
                 └── ACUTE MANAGEMENT (Severe >3.5 / Symptomatic)
                         ├── **IV 0.9% NaCl 1-2L** (Hydration, Target UOP >150mL/h)
                         ├── **Furosemide 20-40mg** (After Hydration)
                         ├── **Bisphosphonate**: Zoledronic 4mg IV (Renal Adjust)
                         ├── **Denosumab** 120mg SC (If Bisphosphonate Contraindicated)
                         └── **Calcitonin** 4-8 IU/kg (Rapid Onset, Tachyphylaxis)
```

### Magnesium Disorders

```
SERUM MAGNESIUM DISORDER
         │
         ├── HYPOMAGNESAEMIA (Mg²⁺ <0.7 mmol/L)
         │       ├── CAUSES
         │       │   ├── GI Losses (Diarrhoea, Malabsorption, PPI)
         │       │   ├── Renal Losses (Diuretics, Cisplatin, Aminoglycosides, Aminoglycosides, Gitelman)
         │       │   ├── Redistribution (Hungry Bone, Refeeding, DKA Treatment)
         │       │   └── Alcoholism
         │       │
         │       ├── CLINICAL
         │       │   ├── Hypokalaemia (Refractory), Hypocalcaemia (Refractory PTH)
         │       │   ├── ECG: QT Prolongation, Torsades Risk
         │       │   ├── Neuromuscular: Tremor, Tetany, Seizures, Ataxia
         │       │   └── Arrhythmias (VF, Torsades)
         │       │
         │       └── REPLETION
         │           ├── **IV MgSO₄ 2-4g (8-16 mmol) over 20-30 min** (Acute/Severe)
         │           ├── **IV MgSO₄ 1g/h** (Controlled Infusion)
         │           ├── Oral Mg 10-20 mmol/d (Chronic/Mild)
         │           └── **ALWAYS CORRECT Mg BEFORE K⁺/Ca²⁺** (Refractory if Mg Low)
         │
         └── HYPERMAGNESAEMIA (Mg²⁺ >1.1 mmol/L)
                 ├── CAUSES: Renal Failure, Iatrogenic (Mg Antacids/Laxatives/Enemas), Lithium, Adrenal Insufficiency
                 ├── CLINICAL: Hyporeflexia, Hypotension, Heart Block, Respiratory Depression, Coma
                 └── MANAGEMENT: Stop Source → IV Fluids + Furosemide → Dialysis (Severe)
```

### Phosphate Disorders

```
SERUM PHOSPHATE DISORDER
         │
         ├── HYPOPHOSPHATAEMIA (PO₄ <0.8 mmol/L)
         │       ├── ACUTE (Refeeding Syndrome, DKA Treatment, Respiratory Alkalosis)
         │       │   ├── **IV Phosphate 15-30 mmol over 6-12h** (Severe <0.3)
         │       │   └── Monitor: Ca²⁺, Mg²⁺, K⁺, ECG
         │       │
         │       └── CHRONIC (Malabsorption, Antacids, Oncogenic Osteomalacia, Fanconi)
         │           └── Oral Phosphate 1-2g/d (Neutraphos, Phosphate Sandoz)
         │
         └── HYPERPHOSPHATAEMIA (PO₄ >1.5 mmol/L)
                 ├── ACUTE: Tumour Lysis Syndrome, Rhabdomyolysis, AKI
                 │       ├── Hydration + Allopurinol/Rasburicase
                 │       └── **Dialysis** if Severe / Symptomatic
                 │
                 └── CHRONIC: CKD (CKD-MBD), Hypoparathyroidism, Vit D Excess
                         ├── Dietary Restriction (<800mg/d)
                         ├── Phosphate Binders (CaCO₃, Sevelamer, Lanthanum, Sucroferric)
                         ├── Cinacalcet (If SHPT)
                         └── Dialysis (Renal Replacement)
```

## Key Formulae & Cut-offs

| Parameter | Value / Formula |
|-----------|-----------------|
| **Ionised Ca²⁺ Normal** | 1.15-1.30 mmol/L |
| **Corrected Ca²⁺** | Measured Ca²⁺ + 0.02 × (40 - Albumin g/L) |
| **Hypocalcaemia** | Corrected <2.1 mmol/L / Ionised <1.1 mmol/L |
| **Hypercalcaemia** | Corrected >2.6 mmol/L / Ionised >1.3 mmol/L |
| **PTH in Hypercalcaemia** | Suppressed = Malignancy; Elevated = Primary HPT; Normal = FHH |
| **Hypomagnesaemia** | <0.7 mmol/L (Severe <0.5) |
| **Hypermagnesaemia** | >1.1 mmol/L |
| **IV MgSO₄ 50%** | 1mL = 2 mmol Mg²⁺ (1g = 4 mmol) |
| **Hypophosphataemia** | <0.8 mmol/L (Severe <0.3) |
| **Hyperphosphataemia** | >1.5 mmol/L (Dialysis Target 1.1-1.5) |
| **IV Phosphate** | 15-30 mmol over 6-12h (Monitor Ca²⁺, Avoid if Hypercalcaemic) |
| **Phosphate Binders** | CaCO₃ (if Ca Normal); Sevelamer/Lanthanum (if Ca High/Adynamic) |

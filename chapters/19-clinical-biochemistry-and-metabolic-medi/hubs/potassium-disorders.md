## Clinical Algorithm: Potassium Disorders

```
SERUM POTASSIUM DISORDER
         │
         ├── HYPOKALAEMIA (K⁺ <3.5 mmol/L)
         │       │
         │       ├── ECG CHANGES
         │       │       ├── U Waves, ST Depression, QT Prolongation
         │       │       └── Risk of Arrhythmias (Digoxin Toxicity)
         │       │
         │       ├── ASSESS CAUSE
         │       │       ├── URINE K⁺ >20 mmol/d → **RENAL LOSSES**
         │       │       │       ├── Diuretics (Loop/Thiazide)
         │       │       │       ├── Mineralocorticoid Excess (Conn's, Cushing's)
         │       │       │       ├── Renal Tubular Acidosis
         │       │       │       ├── Magnesium Deficiency
         │       │       │       └── Diabetic Ketoacidosis (Post-Treatment)
         │       │       │
         │       │       └── URINE K⁺ <20 mmol/d → **EXTRarenal LOSSES**
         │       │               ├── GI Losses (Vomiting, Diarrhoea, Laxatives)
         │       │               ├── Insulin/Alkalosis (Transcellular Shift)
         │       │               └── Inadequate Intake
         │       │
         │       └── REPLACEMENT
         │               ├── PO KCl 40-80 mmol/d (Divided Doses)
         │               ├── IV KCl 10-20 mmol/h (Central Line, Cardiac Monitor)
         │               ├── Max Rate: 10 mmol/h (Peripheral), 20-40 mmol/h (Central)
         │               ├── **ALWAYS CHECK Mg²⁺** (Hypomagnesaemia → Refractory Hypokalaemia)
         │               └── **STOP DIGOXIN** if Toxicity Suspected
         │
         └── HYPERAKALAEMIA (K⁺ >5.5 mmol/L)
                 │
                 ├── ECG CHANGES (URGENT if >6.5)
                 │       ├── Peaked T Waves (>6.0)
                 │       ├── PR Prolongation, QRS Widening (>6.5)
                 │       ├── Sine Wave → VF/Asystole (>7.0)
                 │
                 ├── IMMEDIATE STABILISATION (>6.5 or ECG Changes)
                 │       ├── **Calcium Gluconate 10% 10mL IV** (Cardiac Membrane Stabilisation)
                 │       ├── **Insulin 10U Actrapid + 50mL 50% Dextrose** (Shift K⁺ In)
                 │       ├── **Salbutamol 5mg Nebulised** (β₂ Agonist)
                 │       └── **Calcium Resonium 15g PO/PR** (Gut K⁺ Binding)
                 │
                 ├── DEFINITIVE REMOVAL
                 │       ├── **Dialysis** (Refractory, AKI, >7.0)
                 │       └── **Kayexalate** (Slower, Gut Binding)
                 │
                 └── IDENTIFY CAUSE
                         ├── Renal Failure (AKI, CKD)
                         ├── Drugs (ACEi, ARB, K⁺-Sparing, NSAIDs, Heparin)
                         ├── Acidosis (Transcellular Shift)
                         ├── Adrenal Insufficiency (Addison's)
                         ├── Tumour Lysis / Rhabdomyolysis
                         └── Pseudohyperkalaemia (Haemolysis, Fist Clenching)
```

## Key Formulae & Cut-offs

| Parameter | Value / Formula |
|-----------|-----------------|
| **Hypokalaemia Definition** | K⁺ <3.5 mmol/L (Moderate 2.5-3.0; Severe <2.5) |
| **Hyperkalaemia Definition** | K⁺ >5.5 mmol/L (Moderate 6.0-6.5; Severe >6.5) |
| **Urine K⁺ Cut-off** | >20 mmol/d = Renal Loss; <20 = Extrarenal / Transcellular |
| **K⁺ Replacement (IV)** | 10 mmol/h (Peripheral); 20-40 mmol/h (Central, Monitored) |
| **K⁺ Replacement (Oral)** | 40-80 mmol/d (Divided Doses, with Meals) |
| **K⁺ Shift per 10U Insulin** | ↓ K⁺ by 0.5-1.0 mmol/L (Transient, 30-60 min) |
| **Calcium Gluconate 10%** | 10mL IV over 5-10 min (Membrane Stabilisation, NOT K⁺ Lowering) |
| **Dextrose 50% + Insulin** | 50mL 50% Dextrose + 10U Actrapid → Recheck K⁺/Glucose at 30 min |
| **Calcium Resonium** | 15g PO/PR (Sodium Polystyrene Sulfonate) — Delayed 2-4h |
| **Dialysis Indication** | K⁺ >6.5 + ECG Changes / Refractory / AKI / >7.0 |

# Respiratory Acidosis

> [!info]
> **Respiratory Acidosis = Primary ↑ pCO₂ (>6.0 kPa / 45 mmHg) → pH <7.35.** **Failure of Ventilation** → CO₂ Retention. **Acute vs Chronic Differentiation is Critical** (Compensation Differs). **COPD Exacerbation** = Most Common Cause.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Differentiate acute vs chronic respiratory acidosis using compensation rules
- [ ] Identify causes: airway obstruction, hypoventilation, CNS depression, mechanical ventilation issues
- [ ] Apply NIV as first-line for COPD exacerbation
- [ ] Recognise intubation criteria (pH <7.25, GCS <8, exhaustion)
- [ ] Manage acute exacerbation of COPD with respiratory acidosis

## 2. Definitions & Diagnostic Criteria

| Parameter | Normal | Respiratory Acidosis |
|-----------|--------|---------------------|
| **pH** | 7.35-7.45 | **<7.35** |
| **pCO₂** | 4.5-6.0 kPa (35-45 mmHg) | **>6.0 kPa (45 mmHg)** |
| **HCO₃⁻** | 22-26 mmol/L | **Normal (Acute) / Elevated (Chronic)** |
| **pH** | 7.35-7.45 | **<7.35** |

---

## 2. Acute vs Chronic — Critical Distinction

| Feature | **Acute Respiratory Acidosis** | **Chronic Respiratory Acidosis** |
|---------|------------------------------|----------------------------------|
| **Onset** | Minutes-Hours | Days-Weeks (Pre-existing Lung Disease) |
| **Compensation** | **HCO₃⁻ ↑ 1 mmol/L per 10 mmHg pCO₂ ↑** | **HCO₃⁻ ↑ 4 mmol/L per 10 mmHg pCO₂ ↑** |
| **pH Change per 10 mmHg pCO₂** | **↓ 0.08** | **↓ 0.03** |
| **HCO₃⁻ at pCO₂ 80 mmHg** | ~24-26 mmol/L | **~36-40 mmol/L** |
| **Clinical Context** | Acute Opiate Overdose, Asthma Attack, Pulmonary Oedema, Pneumonia | **COPD**, Obesity Hypoventilation, Neuromuscular Disease |
| **Renal Compensation** | **None** (Takes 2-5 Days) | **Complete** (Renal HCO₃⁻ Retention) |

---

## 2. Aetiology

| Category | Causes |
|----------|--------|
| **Airway Obstruction** | **COPD Exacerbation** (Most Common), **Asthma** (Status Asthmaticus), Croup, Foreign Body, Laryngeal Oedema |
| **Hypoventilation (Central)** | **Opiate/Sedative Overdose**, CNS Depression (Stroke, Tumour, Encephalitis), Obesity Hypoventilation Syndrome, Neuromuscular Disease (MND, Guillain-Barré, Myasthenia Gravis) |
| **Mechanical Ventilation** | Inadequate Tidal Volume/Rate, Auto-PEEP, Circuit Disconnection |
| **Chest Wall / Pleural** | Flail Chest, Pneumothorax, Pleural Effusion, Kyphoscoliosis, Ankylosing Spondylitis |

---

## 2. Clinical Presentation

| Feature | Acute | Chronic |
|---------|-------|---------|
| **Mental Status** | Confusion, Drowsiness, **Coma** (Rapid) | **Chronic Confusion**, Morning Headaches, Daytime Somnolence |
| **Respiratory** | Severe Dyspnoea, Tachypnoea, Accessory Muscle Use | Chronic Dyspnoea, Orthopnoea, Cor Pulmonale Signs |
| **Neurological** | Tremor, Asterixis (CO₂ Narcosis), Seizures | Papilloedema (Rare), Polycythemia |
| **Cardiovascular** | Tachycardia, Hypertension, Arrhythmias | **Cor Pulmonale** (RVH, Peripheral Oedema, JVP↑) |

---

## 3. Diagnostic Criteria

| Test | Finding |
|------|---------|
| **ABG** | **pH <7.35**, **pCO₂ >6.0 kPa (45 mmHg)**, HCO₃⁻ Normal (Acute) / ↑ (Chronic) |
| **Compensation Assessment** | **Acute**: HCO₃⁻ ↑ 1/10 mmHg pCO₂; **Chronic**: HCO₃⁻ ↑ 4/10 mmHg pCO₂ |
| **CXR** | COPD Changes, Consolidation, Oedema, Pneumothorax |
| **Spirometry** | Obstructive Pattern (COPD/Asthma), Restrictive (Neuromuscular/Chest Wall) |
| **CT Head** | If CNS Depression Suspected (Stroke, Tumour, Haemorrhage) |
| **Drug Screen** | Opiates, Benzodiazepines, Alcohol |

---

## 3. Compensation Rules

| Type | Expected HCO₃⁻ Change per 10 mmHg pCO₂ ↑ | Expected pH Change per 10 mmHg pCO₂ ↑ |
|---------------------|-------------------------------------------|--------------------------------------|
| **Acute** | **↑ 1 mmol/L** | **↓ 0.08** |
| **Chronic** | **↑ 4 mmol/L** | **↓ 0.03** |

### Quick Assessment
| pCO₂ (mmHg) | Expected HCO₃⁻ (Acute) | Expected HCO₃⁻ (Chronic) |
|-------------|------------------------|--------------------------|
| 60 | 24-26 | 28-30 |
| 70 | 25-27 | 32-34 |
| 80 | 26-28 | 36-38 |
| 90 | 27-29 | 40-42 |

> **If Actual HCO₃⁻ > Expected for Acute** → **Concurrent Metabolic Alkalosis**  
> **If Actual HCO₃⁻ < Expected for Chronic** → **Concurrent Metabolic Acidosis**

## 3. Management Algorithm

```
RESPIRATORY ACIDOSIS (pH <7.35, pCO₂ >6.0 kPa)
         │
         ▼
ASSESS SEVERITY & ACUITY
         │
         ├── ACUTE / SEVERE (pH <7.25, pCO₂ >8.0 kPa, GCS <8, Exhaustion)
         │       → **INTUBATION + INVASIVE MECHANICAL VENTILATION**
         │
         ├── COPD EXACERBATION (pH 7.25-7.35, pCO₂ 6.0-8.0, Conscious)
         │       → **NON-INVASIVE VENTILATION (NIV / BiPAP)** — **1st Line**
         │       │       → IPAP 10-15, EPAP 4-5 cmH₂O; Titrate to pH >7.35
         │       │       → Monitor: pH, pCO₂, SpO₂, GCS q1-2h
         │       │
         │       └── IF NIV FAILS (pH <7.25, Deteriorating GCS, Exhaustion)
         │               → **INTUBATION + IPPV**
         │
         ├── CHRONIC STABLE (Compensated, Asymptomatic)
         │       → **OPTIMISE MAINTENANCE** (Bronchodilators, Steroids, Long-Term O₂)
         │
         └── OTHER CAUSES
                 ├── OPIATE OVERDOSE → **Naloxone 0.4-2mg IV** (Repeat/Infusion)
                 ├── NEUROMUSCULAR → **NIV** (BiPAP) / Tracheostomy + Ventilator
                 ├── OBESITY HYPOVENTILATION → **NIV (BiPAP)** + Weight Loss
                 └── MECHANICAL VENTILATION ISSUE → Adjust Tidal Volume/Rate
```

## 3. NIV (BiPAP) for COPD Exacerbation

| Parameter | Setting |
|-----------|---------|
| **Mode** | **BiPAP (S/T Mode)** or **CPAP + Pressure Support** |
| **IPAP** | **10-15 cmH₂O** (Titrate to Tidal Volume 6-8 mL/kg) |
| **EPAP** | **4-5 cmH₂O** (Counteract Auto-PEEP) |
| **FiO₂** | Target SpO₂ **88-92%** (Avoid Hyperoxia) |
| **Rate** | Backup Rate 12-15/min (S/T Mode) |
| **Monitoring** | ABG q1-2h (First 4h), SpO₂ Continuous, GCS q1h |

### NIV Failure Criteria (→ Intubation)
| Criterion | Threshold |
|-----------|-----------|
| **pH** | **<7.25** (Despite Optimal NIV 1-2h) |
| **GCS** | **<8** / Deteriorating |
| **Respiratory Rate** | >35/min (Worsening) |
| **Haemodynamics** | SBP <90, Unstable Arrhythmia |
| **Inability to Protect Airway** | Excessive Secretions, Impaired Cough |

---

## 3. Intubation Criteria (Absolute)

| Criterion | Threshold |
|-----------|-----------|
| **pH** | **<7.25** (Despite Optimal NIV) |
| **GCS** | **<8** / Unable to Protect Airway |
| **Respiratory Fatigue** | Exhaustion, Paradoxical Breathing, Accessory Muscle Exhaustion |
| **Haemodynamic Instability** | SBP <90 mmHg on Vasopressors |
| **Severe Hypoxaemia** | SpO₂ <85% on FiO₂ 1.0 + NIV |
| **Life-Threatening Arrhythmia** | VF/VT, Severe Bradycardia |

---

## 3. Specific Causes Management

| Cause | Specific Management |
|-------|---------------------|
| **COPD Exacerbation** | **NIV 1st Line** (BiPAP); Steroids 40-50mg Prednisolone 5d; Antibiotics if Purulent Sputum; Bronchodilators (SABA + SAMA Nebulisers) |
| **Opiate Overdose** | **Naloxone 0.4-2mg IV/IM** (Repeat q2-3min or Infusion 0.4-2mg/h); Observe 4-6h Post-Reversal |
| **Asthma (Status Asthmaticus)** | **NIV Caution** (High Intrinsic PEEP Risk); Prefer Intubation + Low Tidal Volume (6-8 mL/kg) + Permissive Hypercapnia |
| **Obesity Hypoventilation** | **BiPAP** (High EPAP 8-10 cmH₂O); Weight Loss; Screen for OSA |
| **Neuromuscular** | **BiPAP** (S/T Mode, Backup Rate); Consider Tracheostomy + Home Ventilator if Chronic |
| **Pneumothorax / Pleural Effusion** | **Chest Drain** (Urgent) |

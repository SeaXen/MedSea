# Metabolic Acidosis

> [!info]
> **Metabolic Acidosis = Primary ↓ HCO₃⁻ (<22 mmol/L) with Compensatory ↓ pCO₂.** **Primary Distinction: High Anion Gap (AGMA) vs Normal Anion Gap (NAGMA).** **MUDPILES Mnemonic for AGMA Causes.**

## 2. Definitions & Diagnostic Criteria

| Parameter | Normal Range | Metabolic Acidosis |
|-----------|--------------|-------------------|
| **pH** | 7.35-7.45 | **<7.35** |
| **HCO₃⁻** | 22-26 mmol/L | **<22 mmol/L** |
| **pCO₂** | 4.5-6.0 kPa (35-45 mmHg) | **Compensatorily Low** |
| **Anion Gap (AG)** | **8-12 mmol/L** | **High (>12) = AGMA; Normal (8-12) = NAGMA** |

> **Anion Gap (AG)** = Na⁺ - (Cl⁻ + HCO₃⁻) [Normal 8-12 mmol/L]
> **Corrected AG (Hypoalbuminaemia)** = AG + 2.5 × (40 - Albumin g/L)

---

## 2. Aetiology — High Anion Gap Metabolic Acidosis (AGMA)

### MUDPILES Mnemonic
| Letter | Cause | Mechanism |
|--------|-------|-----------|
| **M** | **Methanol** | Formic Acid Accumulation |
| **U** | **Uraemia** (CKD Stage 4-5) | Retained Organic Acids |
| **D** | **DKA** (Diabetic Ketoacidosis) | β-Hydroxybutyrate, Acetoacetate |
| **P** | **Paraldehyde** / **Propylene Glycol** | Organic Acid Metabolites |
| **I** | **Iron** / **Isoniazid** | Lactic Acid / Pyruvic Acid Accumulation |
| **L** | **Lactic Acidosis** (Type A: Hypoperfusion; Type B: Metabolic) | Anaerobic Glycolysis / Mitochondrial Dysfunction |
| **E** | **Ethylene Glycol** / **Ethanol** | Glycolic/Oxalic Acid / Lactic Acidosis |
| **S** | **Salicylates** / **Sepsis** | Uncouples Oxidative Phosphorylation / Lactic Acidosis |

### Additional High AG Causes
| Cause | Typical Context |
|-------|----------------|
| **Pyroglutamic Acidosis** (5-Oxoproline) | Chronic Paracetamol Use, Malnutrition, Sepsis, Renal Failure |
| **β-Hydroxybutyrate** (Ketotic) | Starvation, Alcoholism, Low-Carb Diet, Pregnancy |
| **D-Lactic Acidosis** | Short Bowel Syndrome, Jejunoileal Bypass |

## 2. Aetiology — Normal Anion Gap Metabolic Acidosis (NAGMA)

### HARDASS Mnemonic
| Letter | Cause | Mechanism |
|--------|-------|-----------|
| **H** | **Hyperalimentation** (TPN) | Ammonium Chloride Byproduct |
| **A** | **Acetazolamide** | ↓ Proximal HCO₃⁻ Reabsorption |
| **R** | **Renal Tubular Acidosis (RTA)** | Distal (Type 1), Proximal (Type 2), Type 4 (Hypoaldosteronism) |
| **D** | **Diarrhoea** | HCO₃⁻ Loss (Voluminous Watery) |
| **A** | **Adrenal Insufficiency** (Addison's) | Hypoaldosteronism (Type 4 RTA) |
| **S** | **Saline Infusion** (0.9% NaCl) | Hyperchloraemic (Dilutional) |
| **S** | **Spironolactone** / **Ureteric Diversion** | Renal HCO₃⁻ Loss |

## 3. Diagnostic Algorithm

```
METABOLIC ACIDOSIS (HCO₃⁻ <22, pH <7.35)
         │
         ▼
CALCULATE ANION GAP (AG = Na⁺ - Cl⁻ - HCO₃⁻)
         │
         ├── **HIGH AG (>12)** → **AGMA**
         │       ├── **MUDPILES**: Methanol, Uraemia, DKA, Paraldehyde, Iron/Isoniazid, Lactic, Ethylene Glycol/Ethanol, Salicylates
         │       │
         │       └── **CALCULATE DELTA RATIO** (ΔAG / ΔHCO₃⁻)
         │               ├── 1-2 → **Pure AGMA**
         │               ├── <0.4 → **AGMA + NAGMA**
         │               ├── 0.4-0.8 → AGMA + NAGMA (Mild)
         │               └── >2 → **AGMA + Metabolic Alkalosis**
         │
         └── **NORMAL AG (8-12)** → **NAGMA**
                 ├── **HARDASS**: Hyperalimentation, Acetazolamide, RTA, Diarrhoea, Adrenal Insufficiency, Saline, Spironolactone
                 └── **ALWAYS EXCLUDE**: Renal Failure (Uraemia Can Raise AG)
```

## 3. Delta Ratio & Delta-Delta (Mixed Disorders)

| Ratio | Interpretation |
|-------|----------------|
| **ΔAG / ΔHCO₃⁻** | **1-2** → **Pure AGMA** |
| **<0.4** | **AGMA + NAGMA** (Mixed High & Normal AG) |
| **0.4-0.8** | AGMA + NAGMA (Mild NAGMA Component) |
| **>2** | **AGMA + Metabolic Alkalosis** (Concurrent Alkalosis) |

### Delta-Delta Formula
> **Corrected HCO₃⁻** = Measured HCO₃⁻ + ΔAG
> - **Corrected HCO₃⁻ >24** → **Concurrent Metabolic Alkalosis**
> - **Corrected HCO₃⁻ <24** → **Concurrent NAGMA**

## 3. Respiratory Compensation — Winter's Formula

| Disorder | Expected Compensation |
|----------|----------------------|
| **Metabolic Acidosis** | **Expected pCO₂ = 1.5 × HCO₃⁻ + 8 ± 2** |
| **Assessment** | If Actual pCO₂ > Expected → **Concurrent Respiratory Acidosis** |
| | If Actual pCO₂ < Expected → **Concurrent Respiratory Alkalosis** |

## 3. Bicarbonate Therapy Indications

| Indication | Dose & Administration |
|------------|----------------------|
| **pH <7.1** (Severe Acidaemia) | **8.4% NaHCO₃ 50-100 mmol IV** over 30-60 min |
| **Severe AKI** (Urine Output <0.5 mL/kg/h) | Same as Above |
| **Severe Sepsis** (Lactic Acidosis, pH <7.2) | Consider 50-100 mmol |
| **Cardiac Arrest** | 50-100 mmol IV Bolus (Per Protocol) |
| **DKA** | **NOT Routine** (Risk of Paradoxical CNS Acidosis, Hypokalaemia, Volume Overload) |
| **Contraindicated** | Respiratory Acidosis (↑ CO₂ Production), Volume Overload, Alkalosis |

### Bicarbonate Dosing
| Preparation | Dose | Target |
|------------|------|--------|
| **8.4% NaHCO₃** | 50-100 mmol (50-100 mL) over 30-60 min | **pH >7.20** |
| **1.26% NaHCO₃** (1.4%) | 500-1000 mL | IV Infusion (Less Concentrated) |

> **Goal**: Raise pH >7.20, Not Normalise pH. **Avoid Overcorrection** (Post-Hypercapnic Alkalosis).

## 3. Specific Causes — Key Features

### Lactic Acidosis
| Type | Cause | K⁺/Glucose |
|-------|-------|------------|
| **Type A (Hypoperfusion)** | Shock, Sepsis, cardiac Arrest, Mesenteric Ischaemia | Variable |
| **Type B (Metabolic/Toxin)** | Metformin, HAART, Propofol, Thiamine Deficiency, Alcoholic Ketoacidosis | Often Euglycaemic (Metformin) |

### DKA vs HHS
| Feature | DKA | HHS |
|---------|-----|-----|
| **Glucose** | >11.1 mmol/L | **>33.3 mmol/L** |
| **Ketones** | **Positive** (β-HB, Acetoacetate) | Negative/Trace |
| **pH** | **<7.3** (Often <7.1) | >7.3 |
| **Anion Gap** | **High** | Normal/Mildly High |
| **Osmolality** | Elevated | **Very High (>320)** |

### Renal Tubular Acidosis (RTA)
| Type | Defect | Key Features |
|------|--------|--------------|
| **Type 1 (Distal)** | ↓ H⁺ Secretion in Collecting Duct | **Hypokalaemia**, Nephrocalcinosis, Stones |
| **Type 2 (Proximal)** | ↓ HCO₃⁻ Reabsorption | Fanconi Syndrome, Rickets, Glycosuria |
| **Type 4 (Hypoaldosteronism)** | ↓ Aldosterone / Renal Resistance | **Hyperkalaemia**, Mild Acidosis, Diabetes/CKD |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **AGMA vs NAGMA** | AGMA = Anion Gap >12; NAGMA = AG 8-12 |
| **Delta Ratio** | 1-2 = Pure AGMA; <0.4 = AGMA+NAGMA; >2 = AGMA+Met Alkalosis |
| **Winter's Formula** | Expected pCO₂ = 1.5 × HCO₃⁻ + 8 ± 2 (Metabolic Acidosis) |
| **Respiratory Compensation** | If Actual pCO₂ > Expected → Respiratory Acidosis Too |
| **Bicarb in DKA** | **Not Routine** (Paradoxical CNS Acidosis, Hypokalaemia, Volume Overload) |
| **Salicylate** | **Mixed Picture**: Respiratory Alkalosis + Metabolic Acidosis |
| **Ethylene Glycol/Methanol** | Osmolar Gap ↑; Fomepizole/Ethanol + Dialysis |
| **Pyroglutamic Acidosis** | Chronic Paracetamol + Malnutrition/Sepsis/Renal Failure |
| **5-Oxoproline** | Urine Organic Acids; Treated by N-acetylcysteine |
| **DKA vs HHS** | DKA: pH<7.3, Ketones+, AGMA; HHS: pH>7.3, Glucose>33, Normal AG |

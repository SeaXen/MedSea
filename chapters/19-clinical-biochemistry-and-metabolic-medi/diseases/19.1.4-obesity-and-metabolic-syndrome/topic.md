# Obesity & Metabolic Syndrome

> [!info]
> **Obesity = Chronic Relapsing Disease** with Excess Adiposity → Metabolic Dysfunction. **Metabolic Syndrome = Cluster of Cardiometabolic Risk Factors** (Central Obesity + ≥2 of: TG↑, HDL↓, BP↑, Glucose↑). **BMI ≥30 = Obesity**; **BMI ≥25 = Overweight**.

---

## 2. Definitions & Classification

### BMI Classification (WHO)
| Category | BMI (kg/m²) | Risk of Comorbidities |
|----------|-------------|----------------------|
| **Underweight** | <18.5 | Low (but ↑ other risks) |
| **Normal** | 18.5-24.9 | Average |
| **Overweight** | 25.0-29.9 | **Increased** |
| **Obese Class I** | 30.0-34.9 | **Moderate** |
| **Obese Class II** | 35.0-39.9 | **Severe** |
| **Obese Class III** | **≥40.0** | **Very Severe** |

### Waist Circumference (Central Obesity)
| Population | Male | Female |
|------------|------|--------|
| **Europid** | **≥102 cm** | **≥88 cm** |
| **South Asian / Asian** | **≥90 cm** | **≥80 cm** |
| **Japanese** | **≥85 cm** | **≥90 cm** |

## 2. Metabolic Syndrome — Diagnostic Criteria

### IDF Criteria (2005) — **Central Obesity + ≥2 of 4**
| Component | Threshold |
|-----------|-----------|
| **Central Obesity** | Waist ≥94 cm (M) / ≥80 cm (F) (Europid) |
| **Triglycerides** | **≥1.7 mmol/L** (150 mg/dL) or on Rx |
| **HDL-Cholesterol** | **<1.03 mmol/L (M)** / **<1.29 mmol/L (F)** or on Rx |
| **Blood Pressure** | **≥130/85 mmHg** or on Rx |
| **Fasting Glucose** | **≥5.6 mmol/L** (100 mg/dL) or on Rx |

### ATP III / NCEP Criteria (2001) — **≥3 of 5**
| Component | Threshold |
|-----------|-----------|
| **Waist** | >102 cm (M) / >88 cm (F) |
| **Triglycerides** | **≥1.7 mmol/L** |
| **HDL-C** | **<1.03 mmol/L (M)** / **<1.29 mmol/L (F)** |
| **BP** | **≥130/85 mmHg** |
| **Fasting Glucose** | **≥5.6 mmol/L** |

> **Central Obesity is Mandatory** in IDF; **Not Mandatory but Counted** in ATP III.

## 2. Aetiology & Pathophysiology

| Factor | Mechanism |
|--------|-----------|
| **Positive Energy Balance** | Caloric Intake > Expenditure → Adipocyte Hypertrophy/Hyperplasia |
| **Insulin Resistance** | Visceral Adipose → ↑ FFA → Hepatic IR → Hyperinsulinaemia → Lipogenesis |
| **Adipokine Dysregulation** | ↓ Adiponectin, ↑ Leptin (Resistance), ↑ TNF-α, ↑ IL-6 → Inflammation |
| **Sympathetic Overactivity** | Hyperinsulinaemia → SNS Activation → Hypertension |
| **Genetics** | Polygenic (FTO, MC4R, etc.); Monogenic (Leptin/Leptin Receptor Deficiency) |
| **Environment** | Sedentary Lifestyle, Ultra-Processed Foods, Sleep Deprivation, Stress |

## 2. Clinical Complications

| System | Complications |
|--------|---------------|
| **Cardiovascular** | Coronary Artery Disease, Heart Failure (HFpEF), Atrial Fibrillation, Stroke, PAD |
| **Metabolic** | **Type 2 Diabetes**, Dyslipidaemia, NAFLD/NASH, Gout, PCOS |
| **Respiratory** | **Obstructive Sleep Apnoea**, Obesity Hypoventilation Syndrome, Asthma |
| **Musculoskeletal** | Osteoarthritis (Knee, Hip), Low Back Pain, Gout |
| **GI/Hepatic** | **NAFLD/NASH**, Gallstones, GERD, Reflux, Pancreatitis |
| **Reproductive** | **PCOS**, Infertility, Menstrual Irregularity, Erectile Dysfunction |
| **Oncological** | **Endometrial, Breast, Colorectal, Oesophageal, Renal, Pancreatic, HCC** |
| **Psychosocial** | Depression, Anxiety, Stigma, Poor Quality of Life |
| **Other** | Dyslipidaemia, Hypertension, T2DM, CKD, Venous Thromboembolism |

## 2. Management Algorithm — Pyramid Approach

```
OBESITY / METABOLIC SYNDROME DIAGNOSED
         │
         └──► LIFESTYLE INTERVENTION (Foundation — All Patients)
                   │
                   ├── Diet: 600 kcal/d Deficit (Low Fat/Cal, Mediterranean, Low GI)
                   ├── Exercise: 150-300 min/wk Moderate + Resistance 2x/wk
                   ├── Behavioural: CBT, Self-Monitoring, Goal Setting
                   └── Target: **5-10% Weight Loss at 6mo** (Improves Metabolic Parameters)
         │
         ├── PHARMACOTHERAPY (BMI ≥30 or ≥27 + Comorbidity)
         │   ├── **GLP-1 RA** (1st Line Pharmacotherapy)
         │   │       ├── **Semaglutide 2.4mg SC Weekly** (>15% wt loss)
         │   │       ├── **Liraglutide 3.0mg SC Daily** (~8% wt loss)
         │   │       └── **Dulaglutide 1.5-4.5mg Weekly** (Lower Doses Approved)
         │   │
         │   ├── **SGLT2i + GLP-1 RA** (Tirzepatide — Dual GIP/GLP-1)
         │   │       └── **Tirzepatide 15mg SC Weekly** (>20% wt loss)
         │   │
         │   ├── ORAL AGENTS (If GLP-1 Contraindicated/Intolerant)
         │   │       ├── **Orlistat 120mg TDS** (Lipase Inhibitor; GI SE)
         │   │       ├── **Naltrexone/Bupropion** 32/360mg BD (Craving ↓; Nausea, Seizure Risk)
         │   │       └── **Phentermine/Topiramate** (Appetite Suppression; Tachycardia, Teratogenic)
         │   │
         │   └── MONITOR: Weight q4wk; 5% wt loss at 3mo → Continue; <5% → Switch/Stop
         │
         └── BARIATRIC SURGERY (BMI ≥40 or ≥35 + Comorbidity)
                 ├── **Sleeve Gastrectomy** (Most Common; Restrictive)
                 ├── **RYGB** (Roux-en-Y; Restrictive + Malabsorptive)
                 ├── **OAGB** (One-Anastomosis Gastric Bypass)
                 └── POST-OP: Nutritional Monitoring (Fe, B12, D, Ca), 25-OH D, PTH, LFTs
```

## 3. Pharmacotherapy — Detailed

### GLP-1 Receptor Agonists (1st Line)
| Agent | Dose | Weight Loss | Key Adverse Effects |
|-------|------|-------------|---------------------|
| **Semaglutide 2.4mg** | 0.25mg → 0.5 → 1.0 → 1.7 → **2.4mg SC Weekly** (Escalate q4wk) | **15-17%** | Nausea, Vomiting, Diarrhoea, Pancreatitis Risk |
| **Liraglutide 3.0mg** | 0.6mg → 1.2 → 1.8 → 2.4 → **3.0mg SC Daily** | **8-10%** | Nausea, Pancreatitis Risk, Thyroid C-Cell Tumours (Rodent) |
| **Dulaglutide** | 0.75-4.5mg SC Weekly | ~5-10% (Dose-Dependent) | Similar GI Profile |

### Dual GIP/GLP-1 RA
| Agent | Dose | Weight Loss | Status |
|-------|------|-------------|--------|
| **Tirzepatide** | 2.5 → 5 → 7.5 → 10 → 15mg SC Weekly | **>20%** | **Approved (Mounjaro®)** |

### Other Oral Agents (If GLP-1 Contraindicated)
| Agent | Dose | Mechanism | Weight Loss | Key SE |
|-------|------|-----------|-------------|--------|
| **Orlistat** | 120mg TDS with Meals | Lipase Inhibitor | 3-5% | Steatorrhoea, Fat-Soluble Vitamin Deficiency |
| **Naltrexone/Bupropion** | 32/360mg BD (Titrate) | Opioid Antagonist + NDRI | 5-8% | Nausea, Seizure Risk, BP ↑ |
| **Phentermine/Topiramate** | 3.75/23 → 15/92mg OD | Appetite Suppression | 10-12% | Tachycardia, Teratogenic, Depression |
| **Setmelanotide** | MC4R Agonist | Rare Monogenic Obesity (POMC, LEPR, PCSK1) | — | Injection Site, Skin Hyperpigmentation |

## 3. Bariatric Surgery

| Procedure | Mechanism | Weight Loss | Remission T2DM |
|-----------|-----------|-------------|----------------|
| **Sleeve Gastrectomy** | **Restrictive** (80% Stomach Removed) | **60-70% EWL** | **60-80%** |
| **RYGB** (Roux-en-Y) | **Restrictive + Malabsorptive** | **70-80% EWL** | **80-90%** |
| **OAGB** (One-Anastomosis) | Restrictive + Malabsorptive | Similar to RYGB | Similar |
| **Adjustable Gastric Band** | Restrictive (Adjustable) | 40-50% EWL | Lower; Declining Use |

### Surgical Criteria (NICE / IFSO)
| BMI | Comorbidity | Eligibility |
|-----|-------------|-------------|
| **≥40** | None | **Eligible** |
| **35-39.9** | **≥1 Obesity-Related Comorbidity** (T2DM, HTN, OSA, etc.) | **Eligible** |
| **30-34.9** | **T2DM (Recent Onset)** | **Consider** (NICE: BMI 30-35 + T2DM) |

## 3. Post-Operative Monitoring

| Parameter | Frequency |
|-----------|-----------|
| **Weight / BMI** | Monthly ×6mo, Then q3-6mo |
| **Nutritional Labs** | **q3mo ×1yr, Then q6-12mo**: Fe, Ferritin, B12, Folate, Vit D, Ca, PTH, Albumin, LFTs |
| **Bone Density** | DEXA at 1yr, Then q2-3yr |
| **Glucose/HbA1c** | q3mo (T2DM Remission Monitoring) |
| **Medication Review** | De-escalate DM/HTN/Lipid Rx as Weight Loss Occurs |

## 3. Metabolic Syndrome Management

| Component | Target |
|---------|--------|
| **Weight Loss** | 5-10% (Improves All Components) |
| **Blood Pressure** | <130/80 (ACEi/ARB 1st Line) |
| **Lipids** | Statin if CVD Risk >10%; Fenofibrate if TG >2.3 |
| **Glucose** | Metformin if Pre-DM/T2DM; GLP-1/SGLT2i if T2DM |
| **Lifestyle** | Mediterranean Diet, 150min/wk Exercise, Smoking Cessation |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **IDF vs ATP III MetSyn** | **IDF: Central Obesity Mandatory**; ATP III: ≥3 of 5 (No Mandatory) |
| **MetSyn vs Obesity** | MetSyn = Cluster of Risk Factors; Obesity = BMI ≥30 (Can Coexist) |
| **GLP-1 RA vs SGLT2i for Weight** | GLP-1 RA > SGLT2i for Weight Loss; Tirzepatide > Semaglutide |
| **Tirzepatide vs Semaglutide** | Tirzepatide = Dual GIP/GLP-1 → **>20% vs ~15%** |
| **Orlistat vs GLP-1** | Orlistat = Lipase Inhibitor (Less Effective, GI SE); GLP-1 = Hormonal (More Effective) |
| **Bariatric Criteria** | BMI ≥40 or ≥35 + Comorbidity; NICE: BMI 30-35 + T2DM |
| **BMI vs WC** | BMI = General Adiposity; WC = Central/Visceral Adiposity (Better CV Predictor) |
| **Weight Loss Target** | **5-10% at 6mo** (Clinically Meaningful) |
| **T2DM Remission** | RYGB > Sleeve > Band; Remission >80% in RYGB |
| **GLP-1 RA Contraindications** | Personal/Family MTC, MEN2, Pregnancy, Severe GI Disease |

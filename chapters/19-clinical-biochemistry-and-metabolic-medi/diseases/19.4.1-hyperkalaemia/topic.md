# Hyperkalaemia

> [!info]
> **Hyperkalaemia = Serum K⁺ >5.5 mmol/L.** **Life-Threatening Emergency** if >6.5 or ECG Changes. **Immediate Membrane Stabilisation (Calcium Gluconate)** → **Shift K⁺ Into Cells (Insulin/Dextrose)** → **Enhanced Excretion (Dialysis/Resonium)**. **ECG Progression**: Peaked T Waves → PR Prolongation/QRS Widening → Sine Wave → VF/Asystole.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise ECG progression of hyperkalaemia and initiate immediate stabilisation
- [ ] Apply the 4-step emergency management: Calcium → Insulin/Dextrose → Salbutamol → Resonium/Dialysis
- [ ] Differentiate true hyperkalaemia from pseudohyperkalaemia
- [ ] Identify causes: Renal Failure, Drugs (ACEi, K-sparing), Acidosis, Adrenal Insufficiency, Tumour Lysis
- [ ] Apply dialysis indications and monitoring

## 2. Definitions & Severity

| Severity | Serum K⁺ (mmol/L) | ECG Findings | Clinical Urgency |
|----------|-------------------|--------------|------------------|
| **Mild** | 5.5-6.0 | Peaked T Waves (Often Only Finding) | Urgent (Monitor & Treat) |
| **Moderate** | 6.0-6.5 | **Peaked T**, PR Prolongation, **QRS Widening** | **Emergent** (Immediate Stabilisation) |
| **Severe** | **>6.5** | **QRS >120ms**, **Sine Wave**, Loss of P Waves → **VF/Asystole** | **CRITICAL** (ICU, Immediate Dialysis) |

---

## 2. Aetiology & Classification

### Impaired Excretion (Renal)
| Cause | Mechanism |
|-------|-----------|
| **Acute Kidney Injury (AKI)** | ↓ GFR → ↓ K⁺ Excretion |
| **Chronic Kidney Disease (CKD)** | ↓ Nephron Mass → ↓ K⁺ Excretion |
| **Drugs** | **ACEi, ARB, K⁺-Sparing Diuretics (Spironolactone, Amiloride, Triamterene), NSAIDs, Heparin, Trimethoprim, Beta-Blockers, Calcineurin Inhibitors** |
| **Obstructive Uropathy** | Post-renal AKI |
| **Hypoaldosteronism** (Type 4 RTA, Diabetes, HIV, ACEi) | ↓ Aldosterone → ↓ K⁺ Excretion |

### Transcellular Shift (K⁺ Moves Out of Cells)
| Cause | Mechanism |
|---------|-----------|
| **Metabolic Acidosis** | H⁺ Enters Cells → K⁺ Moves Out (0.6 mmol/L K⁺ ↑ per 0.1 pH ↓) |
| **Respiratory Acidosis** | Similar (Less Pronounced) |
| **Insulin Deficiency** (DKA, HHS) | ↓ Insulin → ↓ Na⁺/K⁺-ATPase → K⁺ Shift Out |
| **Tumour Lysis Syndrome** | Massive Cell Lysis → K⁺/Phosphate/Uric Acid Release |
| **Rhabdomyolysis** | Muscle Cell Lysis → K⁺/Phosphate/CK/Myoglobin Release |
| **Haemolysis** | RBC Lysis → K⁺ Release (In Vitro / In Vivo) |
| **Beta-Blockers** | ↓ β₂-Mediated K⁺ Uptake |
| **Succinylcholine** | Massive Depolarisation → K⁺ Release (Burns, Trauma, Denervation) |

### Impaired Excretion + Shift
| Cause | Features |
|--------|----------|
| **Tumour Lysis Syndrome** | AKI + Cell Lysis + Hyperphosphataemia + Hyperuricaemia |
| **Rhabdomyolysis** | AKI + Myoglobinuria + Hyperkalaemia + Hyperphosphataemia |
| **Adrenal Insufficiency (Addison's)** | Hypoaldosteronism + Hyponatraemia + Hyperkalaemia + Hypotension |

## 2. Pseudohyperkalaemia (Spurious)

| Cause | Mechanism |
|-------|-----------|
| **Haemolysis** (During Phlebotomy) | RBC K⁺ Release → Falsely High Serum K⁺ |
| **Thrombocytosis** (>1000×10⁹/L) | Platelet K⁺ Release During Clotting |
| **Leucocytosis** (>100×10⁹/L) | WBC K⁺ Release During Clotting |
| **Fist Clenching / Tourniquet** | Local Muscle Contraction → Local K⁺ Release |
| **Delayed Processing** | K⁺ Leaks from Cells Over Time |

### Detection
| Test | Pseudohyperkalaemia | True Hyperkalaemia |
|------|---------------------|-------------------|
| **Repeat Sample** (No Tourniquet, Free Flow) | **Normalises** | Remains High |
| **Plasma K⁺ (EDTA)** | Normal (No Clotting) | High |
| **Platelet Count** | >1000 ×10⁹/L | Normal |
| **WBC Count** | >100 ×10⁹/L | Normal |
| **Clinical Context** | No ECG Changes | ECG Changes Match K⁺ |

## 3. ECG Progression (Critical for Viva)

| Serum K⁺ | ECG Changes | Clinical Significance |
|----------|-------------|----------------------|
| **5.5-6.0** | **Peaked T Waves** (Tall, Narrow, Tent-Shaped) | Early Sign; Monitor Closely |
| **6.0-6.5** | **PR Prolongation**, **QRS Widening (>100ms)**, Flattened P Waves | **Emergent** — Immediate Stabilisation |
| **6.5-7.0** | **QRS >120ms**, **Sine Wave** (QRS-T Fusion), P Waves Absent | **CRITICAL** — Imminent VF/Asystole |
| **>7.0** | **Sine Wave**, Asystole/VF | **CARDIAC ARREST IMMINENT** |

> **Key**: **ECG Changes Do NOT Always Correlate Perfectly with K⁺ Level** — Treat Patient, Not Number. **Peaked T Waves = Often First Sign**.

## 3. Emergency Management Algorithm (4-Step)

```
HYPERAKALAEMIA (K⁺ >6.5 OR ECG CHANGES)
         │
         ▼
STEP 1: **MEMBRANE STABILISATION** (INSTANT)
         │
         ├── **Calcium Gluconate 10% 10mL IV** (Over 5-10 Min)
         │       **Mechanism**: Antagonises K⁺ Effect on Cardiac Membrane
         │       **Onset**: Immediate; **Duration**: 30-60 Min
         │       **Repeat**: If ECG Changes Persist / Recur
         │
         │       ⚠️ **CAUTION**: In Digoxin Toxicity → Calcium Can Worsen Arrhythmias
         │       ⚠️ CaCl₂ 10% 5-10mL Alternative (Central Line Only; Irritant)
         │
         ▼
STEP 2: **SHIFT K⁺ INTO CELLS** (RAPID, TEMPORARY)
         │
         ├── **Insulin 10U Actrapid + 50mL 50% Dextrose IV**
         │       **Mechanism**: Insulin → ↑ Na⁺/K⁺-ATPase → K⁺ Shift Into Cells
         │       **Onset**: 15-30 Min; **Peak**: 30-60 Min; **Duration**: 2-4 Hours
         │       **Effect**: ↓ K⁺ by 0.5-1.0 mmol/L (Transient)
         │       **Monitor**: CBG q30min (Risk of Hypoglycaemia)
         │
         ├── **Salbutamol 5mg Nebulised** (Adjunct)
         │       **Mechanism**: β₂-Agonist → ↑ cAMP → ↑ Na⁺/K⁺-ATPase
         │       **Onset**: 15 Min; **Effect**: ↓ K⁺ 0.5-1.0 mmol/L
         │
         └── **Sodium Bicarbonate** (If Metabolic Acidosis)
                 8.4% NaHCO₃ 50-100mL IV (Corrects Acidosis → ↓ K⁺ Shift)
         │
         ▼
STEP 3: **ENHANCE EXCRETION** (DEFINITIVE)
         │
         ├── **Calcium Resonium (Sodium Polystyrene Sulfonate) 15g PO/PR**
         │       **Mechanism**: Gut K⁺ Binding (Na⁺/K⁺ Exchange)
         │       **Onset**: 2-4 Hours; **Duration**: 6-8 Hours
         │       **Caution**: Constipation, Colonic Necrosis (Rare)
         │
         ├── **Loop Diuretic** (Furosemide 40-80mg IV)
         │       **Only if Volume Overload / Not Oliguric**
         │       **Effect**: ↑ K⁺ Excretion (If Urine Output Adequate)
         │
         └── **DIALYSIS** (DEFINITIVE TREATMENT)
                 **Indications**: K⁺ >6.5 + ECG / Refractory / AKI / Oliguria / Tumour Lysis
                 **Modalities**: IHD (Preferred), SLED, CVVHDF
                 **Effect**: Rapid K⁺ Removal (3-5 mmol/L per Session)
         │
         ▼
STEP 4: **IDENTIFY & TREAT CAUSE**
         │
         ├── Review Drugs (ACEi, ARB, K-Sparing, NSAIDs, Heparin, Beta-Blockers, Trimethoprim)
         ├── AKI / CKD → Renal Support
         ├── DKA → Insulin + Fluids
         ├── Tumour Lysis → Allopurinol/Rasburicase + Hydration + Dialysis
         ├── Rhabdomyolysis → Hydration + Alkalisation
         ├── Adrenal Insufficiency → Hydrocortisone 100mg IV q6h
         └── Stop/Adjust Offending Drugs (ACEi/ARB/K-Sparing/NSAIDs/Heparin)
```

## 3. Dialysis Indications (Absolute)

| Indication | Details |
|------------|---------|
| **K⁺ >6.5 mmol/L** with ECG Changes | **Absolute** |
| **K⁺ >7.0 mmol/L** (Any ECG) | **Absolute** |
| **Refractory** to Medical Management | >6.5 Despite Calcium + Insulin + Resonium |
| **Oliguric AKI** / **CKD Stage 5** | Unable to Excrete K⁺ |
| **Tumour Lysis Syndrome** | Massive K⁺/Phosphate/Uric Acid Load |
| **Rhabdomyolysis** | Massive K⁺ Release + AKI |

## 3. Monitoring During Treatment

| Parameter | Frequency | Target |
|-----------|-----------|--------|
| **Serum K⁺** | **q30-60 min** (Acute); q4-6h (Stable) | **<5.5 mmol/L** |
| **ECG** | Continuous (Acute); Serial q1-2h | Resolution of Peaked T/QRS Widening |
| **Blood Glucose** | q30-60 min (During Insulin) | 4-10 mmol/L |
| **Serum Ca²⁺** | q1-2h (If Calcium Given) | Normal |
| **Urine Output** | Hourly | >0.5 mL/kg/h |
| **Magnesium** | q6-12h | Normalise if Low |

## 3. Pseudohyperkalaemia — Quick Detection

| Clue | Action |
|------|--------|
| **No ECG Changes** Despite High K⁺ | Suspect Pseudohyperkalaemia |
| **Platelets >1000** or **WBC >100** | Collect EDTA Plasma Sample |
| **Haemolysed Sample** | Repeat with Free Flow, Large Bore Needle |
| **Fist Clenching** | Repeat Without Tourniquet/Clenching |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **True vs Pseudo Hyperkalaemia** | Pseudo: No ECG Changes; EDTA Plasma Normal; Thrombocytosis/Leucocytosis |
| **Calcium Gluconate vs Calcium Chloride** | Gluconate 10% 10mL (Peripheral OK); Chloride 10% 5-10mL (Central Only) |
| **Calcium in Digoxin Toxicity** | **Contraindicated** (Worsens Arrhythmias) |
| **Insulin vs Dextrose Order** | Insulin **AFTER** Dextrose (Or Mixed) — Avoid Hypoglycaemia |
| **Resonium Timing** | Onset 2-4h (Not for Acute Stabilisation) |
| **Dialysis Criteria** | K⁺ >6.5 + ECG / Refractory / AKI / Tumour Lysis |
| **Calcium Effect** | **Membrane Stabilisation ONLY** (Does NOT Lower K⁺) |
| **Acidosis Contribution** | 0.6 mmol/L K⁺ Rise per 0.1 pH Drop |
| **Peaked T Waves** | First ECG Sign (Often Only Finding at K⁺ 5.5-6.0) |
| **Sine Wave** | QRS-T Fusion; Pre-Terminal; Imminent VF/Asystole |

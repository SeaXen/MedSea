# Wilson Disease (Hepatolenticular Degeneration)

> [!info]
> **Wilson Disease = Autosomal Recessive Disorder** of Copper Metabolism due to **ATP7B Mutation** (Chromosome 13). **Impaired Biliary Copper Excretion** → Copper Accumulation in Liver, Brain (Basal Ganglia), Cornea (Kayser-Fleischer Rings), Kidney. **Treatable if Diagnosed Early.**

---

## 2. Genetics & Pathophysiology

| Feature | Details |
|---------|---------|
| **Gene** | **ATP7B** (Chromosome 13q14.3) — Copper-Transporting ATPase |
| **Function** | **Intra-Golgi Copper Transport** → Copper Incorporation into Ceruloplasmin + Biliary Excretion |
| **Mutation Effect** | **Loss-of-Function** → ↓ Ceruloplasmin Incorporation + ↓ Biliary Excretion → **Copper Accumulation** |
| **Inheritance** | **Autosomal Recessive** (Carrier Frequency ~1:100) |
| **Allelic Heterogeneity** | **>800 Mutations** (H1069Q Most Common in Europeans) |
| **Copper Accumulation** | Liver → Brain (Basal Ganglia, Putamen) → Cornea (KF Rings) → Kidney, Heart, Pancreas |

## 2. Clinical Presentation

### Hepatic Presentation (Childhood/Young Adult)
| Presentation | Frequency |
|-------------|-----------|
| **Asymptomatic** | Incidental LFT Abnormality |
| **Acute Hepatitis** | Jaundice, Nausea, Fatigue, Hepatomegaly |
| **Chronic Hepatitis** | Persistent LFT Elevation, Progressive Fibrosis |
| **Cirrhosis** | Portal Hypertension, Ascites, Varices, Splenomegaly |
| **Acute Liver Failure** (Fulminant) | **Coombs-Negative Haemolytic Anaemia**, ALF (High Mortality Without Transplant) |

### Neuropsychiatric Presentation (Young Adult, 20-40yr)
| System | Features |
|--------|----------|
| **Motor** | **Tremor** (Wing-Beating, Rest/Action), **Dystonia**, Rigidity, Bradykinesia, Chorea |
| **Speech** | **Dysarthria** (Scanning, Monotone), Drooling |
| **Psychiatric** | Depression, Bipolar, Psychosis, Personality Change, Cognitive Decline |
| **Ocular** | **Kayser-Fleischer Rings** (Pathognomonic; 95% Neuro, 50% Hepatic), Sunflower Cataract |
| **Other** | Dysphagia, Gait Disturbance, Salivation |

### Other System Involvement
| System | Features |
|---------|----------|
| **Renal** | Fanconi Syndrome (Aminoaciduria, Phosphaturia, Glycosuria, RTA), Nephrolithiasis |
| **Cardiac** | Cardiomyopathy, Arrhythmias |
| **Skeletal** | Joint Pain, Osteoporosis, Premature Osteoarthritis |
| **Endocrine** | Hypogonadism, Menstrual Irregularity, Infertility |
| **Haematological** | **Coombs-Negative Haemolytic Anaemia** (Acute Liver Failure), Thrombocytopenia |

## 2. Diagnostic Criteria — Leipzig Scoring System

| Parameter | Score |
|---------|-------|
| **Kayser-Fleischer Rings** | 2 (Present) / 0 (Absent) |
| **Ceruloplasmin** | 2 (<0.1 g/L), 1 (0.1-0.2), 0 (>0.2) |
| **24h Urinary Copper** | 2 (>1.6 µmol/24h / >100 µg/24h), 1 (0.8-1.6 µmol), 0 (Normal) |
| **Liver Copper (Dry Weight)** | 2 (>4.0 µg/g / >250 µg/g), 1 (1-4 / 50-250), 0 (<1 / <50) |
| **Genetic Testing (ATP7B)** | 4 (2 Mutations), 1 (1 Mutation), 0 (None) |
| **Neurological Symptoms** | 2 (Severe), 1 (Mild), 0 (None) |
| **Coombs-Negative Haemolytic Anaemia** | 1 (Present) |

| Total Score | Interpretation |
|-------------|----------------|
| **≥4** | **Wilson Disease Diagnosed** |
| **3** | **Possible** (Further Testing) |
| **<3** | **Unlikely** |

## 2. Key Diagnostic Investigations

| Test | Wilson Disease Finding | Notes |
|------|-----------------------|-------|
| **Ceruloplasmin** | **<0.2 g/L** (Low) | **Acute Phase Reactant** → Can Be Normal in Acute Inflammation |
| **24h Urinary Copper** | **>100 µg/24h (1.6 µmol/24h)** | ↑↑ (Free Copper) |
| **Kayser-Fleischer Rings** | **Present in 95% Neuro / 50% Hepatic** | **Slit-Lamp Exam Required**; Pathognomonic |
| **Liver Copper (Biopsy)** | **>250 µg/g Dry Weight** (Severe) | Gold Standard; >4.0 µmol/g = Diagnostic |
| **Free Serum Copper** | **>15-20 µg/dL** (Calculated: Total Cu - 3×Ceruloplasmin) | >1.6 µmol/L |
| **Genetic Testing (ATP7B)** | **Biallelic Pathogenic Variants** | Confirmatory; Family Screening |
| **Brain MRI** | **"Face of the Giant Panda" Sign** (Midbrain T2 Hyperintensity), Basal Ganglia T2 Hyperintensity | Supportive |

## 3. Lens Signs

### Kayser-Fleischer Rings
| Feature | Details |
|--------|---------|
| **Location** | **Descemet's Membrane** (Corneal Periphery) |
| **Appearance** | **Golden-Brown / Greenish Ring** at Limbus |
| **Detection** | **Slit-Lamp Examination** (Essential) |
| **Prevalence** | **95% Neurological Wilson**; 50% Hepatic Wilson; <1% Other Liver Disease |

### Sunflower Cataract
| Feature | Details |
|--------|---------|
| **Appearance** | **Radial, Petal-Like Opacities** in Anterior/Posterior Lens Capsule |
| **Significance** | Pathognomonic for Wilson Disease (Rare but Specific) |

## 3. Management

### Chelation Therapy (Lifelong Unless Transplanted)

| Agent | Dose | Indication | Key Adverse Effects |
|-------|------|-----------|---------------------|
| **Penicillamine** | **500-1500mg/day** (Divided Doses) | **1st Line** (If Tolerated) | Proteinuria, Neutropenia, Lupus-like Syndrome, Skin Changes, Taste Disturbance |
| **Trientine** | **600-1200mg/day** (Divided Doses) | **2nd Line** (If Penicillamine Intolerant) | Fewer SE; No Cross-Reaction with Penicillamine |
| **Zinc Acetate** | **50mg TDS** (Elemental Zn 25mg/TDS) | **Maintenance** (After Initial Chelation) / **Initial if Mild/Hepatic** | Gastric Irritation; Cu Deficiency if Overdosed |

### Initiation & Monitoring
| Phase | Monitoring |
|-------|------------|
| **Initial (Months 1-3)** | **24h Urine Cu q2-4wk** (Target 200-500 µg/24h on Chelation), **Ceruloplasmin, LFTs, FBC, U&Es q2-4wk**, **Free Cu <1.6 µmol/L** |
| **Maintenance** | **24h Urine Cu q3-6mo**, **Ceruloplasmin, LFTs, FBC, U&Es q3-6mo**, **Free Cu <1.6 µmol/L** |
| **Zinc Therapy** | Serum Zinc, Urine Cu, FBC, LFTs q3-6mo |

### Pregnancy
| Aspect | Management |
|--------|-------------|
| **Penicillamine** | **Continue** (Reduce Dose 250-500mg/day); Pyridoxine 25mg OD |
| **Trientine** | **Continue** (Safe in Pregnancy) |
| **Zinc** | **Continue** (Safe) |
| **Monitoring** | Monthly Urine Cu, LFTs, FBC; Target Free Cu <1.6 µmol/L |

## 3. Liver Transplantation

### Indications
| Indication | Criteria |
|---------|----------|
| **Acute Liver Failure** | Wilsonian ALF (High Mortality Without Transplant) |
| **Decompensated Cirrhosis** | Refractory Ascites, Variceal Bleeding, Hepatic Encephalopathy |
| **Refractory Neuropsychiatric** | Progressive Despite Optimal Chelation |
| **Fulminant Wilson Disease** | Coombs-Negative Haemolytic Anaemia + ALF |

### Outcomes
| Outcome | Post-Transplant |
|---------|-----------------|
| **Survival** | **1yr >85%, 5yr >80%** |
| **Neurological Recovery** | Often **Incomplete** (Residual Dystonia/Tremor) |
| **Copper Metabolism** | **Normalised** (Donor Liver Provides Functional ATP7B) |

## 3. Family Screening & Genetic Counselling

| Aspect | Recommendation |
|--------|----------------|
| **1st-Degree Relatives** | **Genetic Testing (ATP7B)** + Biochemical (Ceruloplasmin, Urine Cu) |
| **Siblings** | **25% Risk** (Both Parents Carriers); Screen Immediately |
| **Offspring** | **50% Carrier Risk**; Screen if Partner Carrier |
| **Prenatal Diagnosis** | **CVS/Amniocentesis** (If Familial Mutations Known) |
| **Genetic Counselling** | **Autosomal Recessive** → 25% Affected, 50% Carrier, 25% Normal |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Ceruloplasmin Low ≠ Wilson** | Can Be Low in Malnutrition, Nephrotic Syndrome, Protein-Losing Enteropathy; **Confirm with Urine Cu / KF Rings / Genetics** |
| **Ceruloplasmin Normal in Wilson** | **Possible** (Acute Phase Reactant in Inflammation/Infection) |
| **KF Rings Absent** | **Ocular Wilson** (Hepatic Only) — KF Rings Absent in ~50% Hepatic Presentation |
| **Penicillamine vs Trientine** | Penicillamine = 1st Line (More SE); Trientine = 2nd Line (Better Tolerated, No Cross-Reactivity) |
| **Zinc vs Penicillamine** | Zinc = Blocks Absorption (Maintenance); Penicillamine = Chelation (Initial) |
| **Free Copper Formula** | **Total Cu - (3 × Ceruloplasmin)** = Free Copper; Target <1.6 µmol/L on Therapy |
| **Liver Cu Quantitation** | **Gold Standard** (Biopsy); >250 µg/g DW = Diagnostic |
| **Leipzig Score** | ≥4 = Diagnosis; 3 = Possible; <3 = Unlikely |
| **Neurological Recovery Post-Tx** | Incomplete (Residual Dystonia/Tremor Common) |
| **Acute Liver Failure Wilson** | Coombs-Negative Haemolytic Anaemia + ALF → **Urgent Transplant** |
| **Penicillamine in Pregnancy** | Continue (Dose Reduce 250-500mg/d); Pyridoxine 25mg OD |
| **Trientine** | Better Tolerated; No Cross-Reactivity with Penicillamine Allergy |
| **Free Copper Target** | <1.6 µmol/L on Chelation |

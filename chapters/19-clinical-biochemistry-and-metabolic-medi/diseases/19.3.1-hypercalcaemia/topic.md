# Hypercalcaemia

> [!info]
> **Hypercalcaemia = Serum Corrected Calcium >2.6 mmol/L (>10.4 mg/dL) or Ionised Calcium >1.3 mmol/L.** Common Medical Emergency. **PTH-Based Differentiation is Critical**: Suppressed PTH → Malignancy/Granulomatous; Elevated PTH → Primary HPT. **Severe (>3.5) = Medical Emergency**.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Classify hypercalcaemia by PTH status (Suppressed vs Elevated)
- [ ] Differentiate Malignancy, Primary HPT, Granulomatous, and Drug-Induced causes
- [ ] Execute acute management (Hydration → Bisphosphonates → Denosumab)
- [ ] Apply malignancy-specific pathway (PTHrP vs Osteolytic vs Vit D-Mediated)
- [ ] Recognise and manage hypercalcaemic crisis

## 2. Classification by PTH Status (Critical Diagnostic Step)

| PTH Status | Differential Diagnosis | Next Step |
|------------|------------------------|-----------|
| **PTH SUPPRESSED** (<1-2 pmol/L) | **Malignancy** (PTHrP, Osteolytic, Vit D-Mediated), Granulomatous, Vit D Toxicity, Thyrotoxicosis, Immobilisation | PTHrP, 1,25-(OH)₂D, CXR/CT, Tumour Markers |
| **PTH ELEVATED** | **Primary Hyperparathyroidism** (Adenoma 85%, Hyperplasia 15%, Carcinoma <1%) | Localisation (Sestamibi/US/4D-CT) → Surgery |
| **PTH INAPPROPRIATELY NORMAL** | **FHH** (CaSR Mutation), Mild Primary HPT, Lithium/Thiazides | Ca/Cr Ratio, Genetic Testing (CaSR) |

## 2. Aetiology — PTH-Based Approach

### A. PTH SUPPRESSED — Malignancy (Most Common Cause in Hospital)

| Mechanism | Frequency | Mediator | Typical Tumours |
|---------|-----------|----------|-----------------|
| **1. PTHrP (Humoral)** | **80%** | **PTHrP** (PTH-related Protein) → Binds PTH1R | **SCLC**, **Squamous Cell** (Lung, H&N, Cervix), **Renal Cell**, **Breast**, **Ovarian**, **Bladder**, **Pancreatic** |
| **2. Osteolytic (Local Osteolysis)** | **20%** | Local Cytokines (IL-1, IL-6, TNF-α, RANKL) | **Breast**, **Myeloma**, **Prostate**, **Lung (Adenocarcinoma)**, **Thyroid** |
| **3. 1,25-(OH)₂D-Mediated (Vitamin D)** | **<5%** | Ectopic 1α-Hydroxylase → ↑ 1,25-(OH)₂D | **Lymphoma** (Hodgkin/NHL), **Granulomatous** (Sarcoid, TB) |

### B. PTH ELEVATED — Primary Hyperparathyroidism

| Subtype | Frequency | Pathology | Management |
|---------|-----------|-----------|------------|
| **Single Adenoma** | **80-85%** | Solitary Adenoma | **Surgery (Focused Parathyroidectomy)** |
| **Hyperplasia** | **10-15%** | 4-Gland Hyperplasia | Subtotal (3.5-Gland) Parathyroidectomy |
| **Double Adenoma** | 5-10% | 2 Adenomas | Surgery |
| **Carcinoma** | <1% | Parathyroid Cancer | En Bloc Resection |

### C. Granulomatous / Vitamin D-Mediated

| Condition | Mechanism | Key Test |
|---------|-----------|----------|
| **Sarcoidosis** | Ectopic 1α-Hydroxylase → ↑ 1,25-(OH)₂D | ↑ 1,25-(OH)₂D, ACE, Chest X-Ray |
| **Tuberculosis** | Macrophage 1α-Hydroxylase | ↑ 1,25-(OH)₂D, Mantoux/IGRA |
| **Lymphoma** | Ectopic 1α-Hydroxylase | ↑ 1,25-(OH)₂D, Lymph Node Biopsy |
| **Vitamin D Toxicity** | Exogenous Supplementation | ↑ 25-OH D (>150 ng/mL) |

## 2. PTHrP — Key Mediator (Humoral Hypercalcaemia of Malignancy - HHM)

| Feature | Details |
|---------|---------|
| **Structure** | **PTH-related Protein** (139-173 AA); N-Terminal Homology with PTH (1-13 AA) |
| **Receptor** | **PTH1R** (Same as PTH Receptor) — Binds N-Terminus |
| **Actions** | **Identical to PTH**: ↑ Bone Resorption, ↑ Renal Ca²⁺ Reabsorption, ↑ Phosphate Excretion |
| **Assay** | **PTHrP Assay (Immunoradiometric/ELISA)** — Mid-Region / C-Terminal |
| **Specificity** | **Mid-Region PTHrP** (Cross-Reacts Less); C-Terminal Less Specific |

### Differential: HHM vs Primary HPT

| Parameter | **HHM (PTHrP-Mediated)** | **Primary HPT** |
|---------|--------------------------|----------------|
| **PTH** | **Suppressed** (<1-2 pmol/L) | **Elevated / Inappropriately Normal** |
| **PTHrP** | **Elevated** | Negative |
| **Phosphate** | Low | Low/Normal |
| **1,25-(OH)₂D** | Low/Normal | Normal/High |
| **Alkaline Phosphatase** | Often High (Bone Turnover) | Normal/Mildly Elevated |

## 3. Acute Hypercalcaemia — Emergency Management

### Step 1: IV Hydration (First-Line)
| Intervention | Details |
|--------------|---------|
| **Fluid** | **0.9% NaCl 1-2L over 1-2h** (Then 200-300 mL/hr) |
| **Goal** | **Urine Output >150 mL/hr** (Flush Calcium + Improve GFR) |
| **Target** | **Euvolaemia → CVP 8-12 cm H₂O** |
| **Caution** | **Cardiac Failure** → Caution (Monitor JVP, CVP, Lungs) |

### Step 2: Loop Diuretic (After Hydration)
| Drug | Dose | Caution |
|------|------|---------|
| **Furosemide** | 20-40mg IV (After Hydration) | **Only After Adequate Hydration**; Monitor K⁺, Mg²⁺ |

### Step 3: Bisphosphonates (First-Line Anti-Resorptive)
| Drug | Dose | Administration | Onset | Duration |
|------|------|----------------|-------|----------|
| **Zoledronic Acid** | **4mg IV** (15min Infusion) | **First-Line** (Potent, Long-Lasting) | 24-48h | **3-4 Weeks** |
| **Pamidronate** | **60-90mg IV** (2-4h Infusion) | Alternative | 24-48h | 2-3 Weeks |
| **Ibandronate** | 4-6mg IV | Alternative | 24-48h | 2-4 Weeks |

### Renal Impairment Adjustments
| eGFR | Zoledronic Acid | Pamidronate |
|------|-----------------|-------------|
| **≥60** | 4mg (Standard) | 90mg |
| **30-59** | 4mg (Caution) | 60mg |
| **<30** | **Avoid** / 3mg (Caution) | 30mg (Caution) |

### Step 4: Denosumab (Alternative/Refractory)
| Feature | Details |
|---------|---------|
| **Mechanism** | **Anti-RANKL Monoclonal Antibody** → Inhibits Osteoclast Formation/Activity |
| **Dose** | **120mg SC** (Monthly) |
| **Indication** | **Refractory to Bisphosphonates** / **Renal Impairment (eGFR <30)** |
| **Advantage** | **No Renal Adjustment Needed**; Faster Onset; Longer Duration |
| **Monitoring** | **Hypocalcaemia Risk** (Supplement Ca²⁺/Vit D) |
| **Duration** | ~4 Weeks (Monthly Dosing) |

### Step 5: Other Therapies

| Agent | Indication | Dose |
|-------|------------|------|
| **Calcitonin** | **Rapid Onset** (2-4h); Short Duration (2-3 Days) | 4-8 IU/kg IM/SC q6-12h |
| **Gallium Nitrate** | Refractory | 100-200mg/m²/day IV (5d) |
| **Plicamycin (Mithramycin)** | Historical/Refractory | 25µg/kg/day IV (4-7 Days) |
| **Denosumab** | Refractory / Renal Impairment | 120mg SC Monthly |
| **Steroids** | Haematological Malignancy (Lymphoma, Myeloma) | Prednisolone 40-60mg/day |
| **Dialysis** | Severe Renal Failure + Life-Threatening Hypercalcaemia | Emergency HD |

## 3. Acute Hypercalcaemia — Emergency Protocol

```
Hypercalcaemia (Ca²⁺ >3.0 mmol/L or Symptomatic)
         │
         ▼
1. IV 0.9% NaCl 1-2L STAT (Over 1-2h) → Target UOP >150 mL/hr
         │
         ▼
2. Loop Diuretic (AFTER Hydration) → Furosemide 20-40mg IV
         │
         ▼
3. BISPHOSPHONATE: Zoledronic Acid 4mg IV (15min Infusion)
         │
         ▼
4. IF REFRACTORY / RENAL IMPAIRMENT → Denosumab 120mg SC
         │
         ▼
5. SUPPORTIVE
   - Hydration Maintenance (2-3L/day)
   - Monitor Ca²⁺, Mg²⁺, K⁺, Phosphate, Renal Function q6-12h
   - Cardiac Monitoring (Shortened QT, Arrhythmias)
   - Treat Underlying Malignancy
```

## 4. Follow-up & Long-Term Management

| Parameter | Frequency | Target |
|-----------|-----------|--------|
| **Serum Calcium (Corrected)** | q6-12h (Acute); Daily (Subacute) | **<2.6 mmol/L** |
| **Ionised Calcium** | q6-12h (Acute) | **<1.3 mmol/L** |
| **Phosphate** | Daily (Hypophosphataemia Risk) | **>0.8 mmol/L** |
| **Magnesium** | Daily | **>0.7 mmol/L** |
| **Renal Function** | q6-12h | Monitor AKI |
| **PTHrP** | Baseline; Post-Treatment | Monitor Response |
| **Urine Output** | Hourly (Acute) | >100-150 mL/hr |

## 4. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **Commonest Cause Hospital Hypercalcaemia** | **Malignancy** (20-30% of Hospitalised Cancer Patients) |
| **Mechanism 80%** | **PTHrP (Humoral)** — Binds PTH1R → Mimics PTH |
| **PTH in HCM** | **Suppressed** (<1 pmol/L) — Differentiates from Primary HPT |
| **PTHrP** | **PTH-related Protein**; N-Terminal Homology with PTH; Binds PTH1R |
| **Osteolytic (20%)** | Local Cytokines (IL-6, TNF, RANKL) → Local Bone Destruction (Myeloma, Breast) |
| **Vitamin D Mediated** | Lymphoma, Sarcoid, TB → Ectopic 1α-Hydroxylase → ↑ 1,25-(OH)₂D |
| **Emergency Rx** | **Hydration → Zoledronic Acid 4mg IV** (Denosumab if Refractory/Renal Fail) |
| **Zoledronic Acid Dose** | 4mg IV (15min) — Adjust for Renal Impairment |
| **Denosumab** | 120mg SC Monthly; **Preferred if eGFR <30**; No Renal Adjustment |
| **Calcitonin** | Rapid Onset (2-4h); Short Duration (2-3d); Adjunct Only |
| **Differentiation PHPT vs HCM** | PHPT: PTH High; HCM: **PTH Suppressed + PTHrP Positive** |
| **Bisphosphonate Renal Dose** | Zoledronic: eGFR≥60: 4mg; 30-59: 4mg; <30: Avoid/3mg |
| **Hydration First** | **Always Hydrate First** Before Bisphosphonate/Diuretic |
| **Malignancy Types** | SCLC, SQCC, Breast, Renal, Myeloma, Lymphoma (Vit D) |
| **PTHrP Assay** | Mid-Region Preferred; Confirms HHM |
| **Bisphosphonate Onset** | Zoledronic 24-48h; Pamidronate 24-48h |
| **Hypercalcaemia Emergency** | Hydration → Bisphosphonate → Denosumab (Refractory/Renal) |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **HCM vs Primary HPT** | HCM: **PTH Suppressed**, PTHrP+; Primary HPT: PTH High |
| **Osteolytic vs Humoral** | Humoral = PTHrP (Systemic); Osteolytic = Local Cytokines (Local Bone Destruction) |
| **Vitamin D Mediated** | Lymphoma, Sarcoid, TB → 1α-Hydroxylase → ↑ 1,25-(OH)₂D |
| **Zoledronic vs Denosumab** | Denosumab: No Renal Adjustment; Faster; Preferred in Renal Failure |
| **PTHrP vs PTH** | PTHrP Shares N-Terminal Homology with PTH (1-13 AA); Binds Same Receptor (PTH1R) |
| **Calcitonin** | Rapid but Weak/Transient; Adjunct Only |
| **Osteolytic HCM** | Myeloma, Breast, Prostate → Local Bone Destruction (RANKL/IL-6) |
| **1,25-(OH)₂D Mediated** | Lymphoma, Sarcoidosis, TB → Ectopic 1α-Hydroxylase |

**Mnemonic: HYPERCALCAEMIA OF MALIGNANCY**
- **H**ypercalcaemia **M**alignancy → **P**TH **S**uppressed
- **E**ctopic **P**THrP (80%) — **H**umoral
- **O**steolytic (20%) — **L**ocal **C**ytokines
- **V**itamin **D** Mediated (Lymphoma) — **E**ctopic 1α-OHase

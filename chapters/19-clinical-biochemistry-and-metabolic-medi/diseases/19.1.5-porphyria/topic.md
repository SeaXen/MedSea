# Porphyria

> [!info]
> **Porphyrias = Group of Rare Metabolic Disorders** due to Enzyme Defects in Haem Biosynthesis. **Acute Hepatic Porphyrias** (Neurovisceral Attacks) vs **Cutaneous Porphyrias** (Photoactive). **PBG/ALA = Diagnostic Hallmark** in Acute Attack.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Classify porphyrias into acute hepatic vs cutaneous
- [ ] Recognise acute porphyria attack: abdominal pain, neuropathy, psychiatric, hyponatraemia
- [ ] Apply diagnostic criteria: PBG/ALA elevation (Gold Standard)
- [ ] Manage acute attack: Haemarginate, IV Glucose, Avoid Precipitants
- [ ] Recognise cutaneous porphyrias: PCT, EPP, CEP, XLEPP
- [ ] Apply drug precautions (Safe vs Unsafe drugs)

## 2. Classification

### Acute Hepatic Porphyrias (Neurovisceral)
| Type | Enzyme Defect | Gene | Inheritance | Key Features |
|------|---------------|------|-------------|--------------|
| **AIP** (Acute Intermittent Porphyria) | **HMBS** (Hydroxymethylbilane Synthase) | **HMBS** | AD | **Most Common Acute**; Neurovisceral (No Skin) |
| **VP** (Variegate Porphyria) | **PPOX** (Protoporphyrinogen Oxidase) | **PPOX** | AD | **Acute + Cutaneous** (Photosensitivity) |
| **HCP** (Hereditary Coproporphyria) | **CPOX** (Coproporphyrinogen Oxidase) | **CPOX** | AD | Acute + Cutaneous (Milder) |
| **ADP** (ALAD Deficiency Porphyria) | **ALAD** (ALA Dehydratase) | **ALAD** | AR | **Ultra-Rare**; Severe Infantile Onset |

### Cutaneous Porphyrias (Photoactive)
| Type | Enzyme Defect | Gene | Inheritance | Key Features |
|------|---------------|------|-------------|--------------|
| **PCT** (Porphyria Cutanea Tarda) | **UROD** (Uroporphyrinogen Decarboxylase) | **UROD** | AD (80%) / Sporadic (20%) | **Most Common Cutaneous**; Blisters, Scarring, Hypertrichosis |
| **EPP** (Erythropoietic Protoporphyria) | **FECH** (Ferrochelatase) | **FECH** | AR | **Painful Photosensitivity** (Immediate Burning); Liver Risk |
| **CEP** (Congenital Erythropoietic Porphyria) | **UROS** (Uroporphyrinogen III Synthase) | **UROS** | AR | **Severe Photosensitivity** (Infancy); Red Urine, Deformities |
| **XLEPP** (X-Linked EPP) | **ALAS2** | **ALAS2** | XL | Similar to EPP; Higher Liver Risk |

## 2. Haem Biosynthesis Pathway & Enzyme Defects

```
Glycine + Succinyl-CoA
      │
      ▼
ALA Synthase (ALAS) ← **Rate-Limiting** (Repressed by Haem)
      │
      ▼
δ-Aminolaevulinic Acid (ALA)
      │
      ▼
**ALA Dehydratase (ALAD)** → **ADP** (AR)
      │
      ▼
Porphobilinogen (PBG)
      │
      ▼
**PBG Deaminase (HMBS)** → **AIP** (AD)
      │
      ▼
Hydroxymethylbilane
      │
      ▼
Uroporphyrinogen III Synthase (UROS) → **CEP** (AR)
      │
      ▼
Uroporphyrinogen III
      │
      ▼
**Uroporphyrinogen Decarboxylase (UROD)** → **PCT** (AD/Sporadic)
      │
      ▼
Coproporphyrinogen III
      │
      ▼
**Coproporphyrinogen Oxidase (CPOX)** → **HCP** (AD)
      │
      ▼
Protoporphyrinogen IX
      │
      ▼
**Protoporphyrinogen Oxidase (PPOX)** → **VP** (AD)
      │
      ▼
Protoporphyrin IX
      │
      ▼
**Ferrochelatase (FECH)** → **EPP/XLEPP** (AR/XL)
      │
      ▼
**HAEM** (Inserted into Globin / Cytochromes)
```

## 2. Acute Porphyria Attack — Clinical Features

| System | Features |
|--------|----------|
| **Abdominal** | **Severe Colicky Pain** (Most Common), Nausea, Vomiting, Constipation, Ileus |
| **Neurological** | **Peripheral Neuropathy** (Motor > Sensory), Quadriparesis, Areflexia, Cranial Nerve Palsies |
| **Psychiatric** | Anxiety, Agitation, Confusion, Hallucinations, Depression, Seizures |
| **Autonomic** | Tachycardia, Hypertension, Flushing, Sweating, Postural Hypotension |
| **Electrolytes** | **Hyponatraemia (SIADH)**, Hypokalaemia, Hypomagnesaemia |
| **Skin** | **Absent in AIP/ADP**; **Present in VP/HCP** (Photosensitivity, Blisters) |

### Acute Attack Precipitants (Avoid in All Porphyrias!)
| Drug Class | Examples |
|------------|----------|
| **Barbiturates** (Thiopental, Phenobarbital) | **Strong Inducers** (↑ ALAS1) |
| **Sulfonamides** | Co-trimoxazole, Sulfasalazine |
| **Rifampicin** | Potent Inducer |
| **Antiepileptics** | Phenytoin, Carbamazepine, Valproate, Phenobarbital, Ethosuximide |
| **Hormonal** | Oestrogen (OCP), Progesterone, Androgens, Corticosteroids |
| **Anaesthetics** | Thiopental, Enflurane (Some); **Safe**: Propofol, Etomidate, Ketamine, Sevoflurane |
| **Other** | Alcohol (Acute), Fasting/Starvation, Infection, Surgery, Stress |

## 2. Diagnosis — Biochemical (Gold Standard)

### Acute Attack
| Test | Finding | Significance |
|------|---------|--------------|
| **Urine PBG (Quantitative)** | **↑↑↑** (Acute Attack) | **Gold Standard Screening** |
| **Urine ALA** | ↑↑ | Supports Diagnosis |
| **Urine Total Porphyrins** | ↑ (Coproporphyrin, Uroporphyrin) | Supports Diagnosis |
| **Faecal Porphyrins** | ↑ Copro / Uro (VP/HCP Differentiation) | Type Differentiation |
| **Plasma Fluorescence** | Peak ~625nm (VP/HCP) | Cutaneous Porphyrias |
| **Enzyme Assay** | HMBS (AIP), PPOX (VP), CPOX (HCP), ALAD (ADP) | Confirmatory |
| **Genetic Testing** | Mutation ID | **Confirmatory; Family Screening** |

### Biochemical Profile by Type
| Type | Urine PBG | Urine ALA | Urine Porphyrins | Faecal Porphyrins | Skin |
|------|-----------|-----------|------------------|-------------------|------|
| **AIP** | ↑↑↑ | ↑↑ | ↑ Uro/Copro | Normal | **None** |
| **VP** | ↑↑↑ | ↑ | ↑ Uro/Copro | **↑ Proto + Copro** | **Photosensitivity** |
| **HCP** | ↑↑↑ | ↑ | ↑ Copro > Uro | **↑ Copro III** | **Photosensitivity (Milder)** |
| **ADP** | Normal/↑ | ↑↑↑ | ↑ ALA | Normal | None |
| **PCT** | Normal | Normal | ↑ Uro (Carboxylic) | ↑ Iso-Copro | **Blisters, Fragility** |
| **EPP** | Normal | Normal | Normal | ↑ Proto | **Immediate Burning Pain** |

## 2. Acute Attack Management

### 1. IMMEDIATE: STOP PRECIPITATING DRUGS
| Drug Classes to STOP | Examples |
|---------------------|----------|
| **Barbiturates** | Thiopental, Phenobarbital |
| **Sulfonamides** | Co-trimoxazole, Sulfasalazine |
| **Rifampicin** | |
| **Antiepileptics** | Phenytoin, Carbamazepine, Valproate, Phenobarbital |
| **Hormones** | OCP, Progesterone, Androgens |
| **Certain Anaesthetics** | Thiopental, Enflurane |

### 2. SPECIFIC THERAPY
| Agent | Dose | Mechanism |
|-------|------|-----------|
| **Haemarginate** | **3-4 mg/kg IV Daily × 4 Days** | **Replenishes Haem Pool** → Represses ALAS1 (Rate-Limiting Enzyme) |
| **IV Glucose** | **300-400g/day** (5-10% Dextrose) | Represses ALAS1 (Insulin-Mediated) |
| **IV Haemin** | Alternative to Haemarginate (3-4 mg/kg) | Same as Haemarginate |

### 3. SUPPORTIVE CARE
| Issue | Management |
|-------|------------|
| **Pain** | **Opioids** (Morphine, Fentanyl — Safe); **Avoid** NSAIDs (Renal), Pethidine (Seizure Risk) |
| **Nausea/Vomiting** | Ondansetron, Metoclopramide (Safe) |
| **Seizures** | **Levetiracetam** (Safe); **Avoid** Phenytoin, Valproate, Carbamazepine |
| **Hyponatraemia** | Fluid Restriction (SIADH); Hypertonic Saline if Severe |
| **Hypokalaemia/Hypomagnesaemia** | Replace Aggressively |
| **Autonomic Instability** | β-Blocker (Propranolol) for Tachycardia/Hypertension |

### 4. DRUGS TO AVOID (Red List) vs SAFE (Green List)
| **AVOID (Red)** | **SAFE (Green)** |
|------------------|------------------|
| Barbiturates, Sulfonamides, Rifampicin | Paracetamol, NSAIDs (Short-Term) |
| Phenytoin, Carbamazepine, Valproate, Phenobarbital | Levetiracetam, Gabapentin, Lamotrigine |
| Oestrogen/Progesterone (OCP) | Progestogen-Only (Mini-Pill / Implant) |
| Rifampicin, Griseofulvin | Most Antibiotics (Penicillins, Cephalosporins, Macrolides) |
| Alcohol (Acute), Fasting | Normal Diet, Glucose Infusion |
| Thiopental, Enflurane | Propofol, Etomidate, Ketamine, Sevoflurane |

## 3. Cutaneous Porphyrias — Management

### Porphyria Cutanea Tarda (PCT) — Most Common
| Feature | Details |
|--------|---------|
| **Aetiology** | UROD Deficiency (Sporadic 80% — Hepatic Iron Overload; Familial 20% — UROD Mutation) |
| **Triggers** | Alcohol, Oestrogen, Hepatitis C, HIV, Iron Overload, Smoking |
| **Clinical** | **Photosensitivity** (Hands/Face), **Blisters/Fragility**, **Hypertrichosis**, Scarring, Milia |
| **Diagnosis** | **Urine Uroporphyrin ↑** (Carboxylic), **Plasma Fluorescence Peak 625nm**, ↑ Faecal Iso-Coproporphyrin |
| **Management** | **1. Phlebotomy 400-500mL q2wk** → Target Ferritin 20-50 µg/L (1st Line)  
**2. Low-Dose Chloroquine/Hydroxychloroquine** 125mg 2x/week (Mobilises Hepatic Porphyrins)  
**3. Avoid Triggers**: Alcohol, Oestrogen, Iron Supplements  
**4. Treat HCV/HIV** (If Co-Infection) |

### Erythropoietic Protoporphyria (EPP) / XLEPP
| Feature | Details |
|---------|---------|
| **Symptoms** | **Acute Burning Pain** on Sun Exposure (Immediate, No Blisters), Oedema, Erythema |
| **Diagnosis** | **Erythrocyte Protoporphyrin ↑↑** (Total + Free); Plasma Fluorescence; FECH Gene |
| **Management** | **Photoprotection** (UV-Blocking Clothing, Sunscreens, Window Films)  
**Beta-Carotene** 120-180mg/day (Mild Benefit)  
**Afamelanotide** (Scenesse®) — SC Implant q2mo (Approved)  
**Liver Transplant** (End-Stage Liver Disease) |

### Congenital Erythropoietic Porphyria (CEP) — Ultra-Rare
| Feature | Management |
|---------|------------|
| Severe Photosensitivity (Infancy), Red Urine, Deformities, Haemolytic Anaemia | **Splenectomy** (Reduces Haemolysis), **Bone Marrow Transplant** (Curative), Photoprotection |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **AIP vs VP vs HCP** | AIP = No Skin; VP = Proto > Copro in Faeces; HCP = Copro > Proto |
| **Acute vs Cutaneous** | Acute = Neurovisceral (AIP/ADP); Cutaneous = Photosensitivity (PCT/EPP/CEP); VP/HCP = Both |
| **PCT vs EPP** | PCT = Blisters/Fragility (Delayed); EPP = Immediate Burning Pain (No Blisters) |
| **PCT vs VP** | PCT = Uro Porphyrin ↑ (Urine); VP = Proto > Copro in Faeces |
| **PCT Management** | **Phlebotomy 1st Line** (Target Ferritin 20-50); Low-Dose Chloroquine 2nd Line |
| **EPP vs CEP** | EPP = Immediate Burning, No Blisters; CEP = Severe Infantile + Deformities |
| **Safe Antiepileptic** | **Levetiracetam** (Only Safe One); Avoid Phenytoin/Valproate/Carbamazepine |
| **Safe Anaesthetic** | Propofol, Etomidate, Ketamine, Sevoflurane |
| **PCT Precipitants** | Alcohol, Oestrogen, HCV, Iron, Smoking |
| **Glucose in Acute Attack** | 300-400g/d 5-10% Dextrose (Represses ALAS1) |
| **Haemarginate Dose** | 3-4mg/kg IV Daily × 4 Days |
| **VP vs HCP Differentiation** | **VP: Faecal Proto > Copro**; **HCP: Faecal Copro > Proto** |
| **EPP vs XLEPP** | EPP = FECH (AR); XLEPP = ALAS2 (XL); XLEPP Higher Liver Risk |
| **PCT Phlebotomy Target** | Ferritin 20-50 µg/L |
| **CEP** | Splenectomy / Bone Marrow Transplant (Curative) |
| **Genetics** | AIP/VP/HCP = AD; ADP = AR; PCT = AD/Sporadic; EPP = AR; CEP = AR; XLEPP = XL |

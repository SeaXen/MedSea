# Hyponatraemia

> [!info]
> **Hyponatraemia = Serum Na⁺ <135 mmol/L.** Most Common Electrolyte Disorder in Hospitalised Patients (15-30%). **Classification by Volume Status** is Critical. **Correction Rate Limit**: 8-10 mmol/L/24h (12-15 if Severe Symptoms) to Avoid Osmotic Demyelination Syndrome.

## 2. Aetiology & Classification by Volume Status

### Hypovolaemic Hyponatraemia (Na⁺ & Water Loss, Water > Na⁺)
| Cause | Urine Na⁺ | Mechanism |
|-------|-----------|-----------|
| **GI Losses** (Vomiting, Diarrhoea, NG Suction) | **<20 mmol/L** | Renal Na⁺ Conservation |
| **Renal Losses** (Diuretics, Mineralocorticoid Deficiency, Salt-Wasting Nephropathy, Cerebral Salt Wasting) | **>20 mmol/L** | Impaired Renal Na⁺ Reabsorption |
| **Cutaneous Losses** (Burns, Sweating) | Variable | |
| **Third Space Losses** (Pancreatitis, Peritonitis, Obstruction) | <20 | |

### Euvolaemic Hyponatraemia (Water Excess, Normal Na⁺)
| Cause | Key Features |
|-------|-------------|
| **SIADH** (Most Common Cause) | Euvolaemic, Urine Osmo >100, Urine Na⁺ >20, Normal TSH/Cortisol, No Diuretics |
| **Hypothyroidism** | ↑ TSH, Low fT4 |
| **Adrenal Insufficiency** | Hyperkalaemia, Hypotension, Hyperpigmentation |
| **Glucocorticoid Deficiency** | Isolated Cortisol Deficiency |
| **Drugs** (SSRIs, Carbamazepine, Cyclophosphamide, MDMA) | |
| **Beer Potomania / Tea & Toast Diet** | Low Solute Intake → Impaired Diluting Ability |
| **Psychogenic Polydipsia** | Compulsive Water Drinking → Urine Osmo <100 |

### Hypervolaemic Hyponatraemia (Na⁺ & Water Excess, Water > Na⁺)
| Cause | Mechanism |
|-------|-----------|
| **Heart Failure** | ↓ Cardiac Output → ↑ ADH / RAAS → Water Retention |
| **Cirrhosis** | Portal Hypertension → Splanchnic Vasodilatation → Effective Arterial Volume ↓ → ADH ↑ |
| **Nephrotic Syndrome** | Hypoalbuminaemia → ↓ Oncotic Pressure → Effective Volume ↓ |
| **Acute/Chronic Kidney Disease** | Impaired Water Excretion |

## 2. SIADH (Syndrome of Inappropriate ADH Secretion)

### Schwartz-Bartter Diagnostic Criteria (All Must Be Met)
| # | Criterion |
|---|-----------|
| 1 | **Hyponatraemia** (Serum Na⁺ <135 mmol/L) |
| 2 | **Hypo-osmolality** (Serum Osmolality <275 mOsm/kg) |
| 3 | **Inappropriately Concentrated Urine** (Urine Osmolality >100 mOsm/kg) |
| 4 | **Euvolaemia** (No Oedema, Normal JVP, Normal BP) |
| 5 | **Elevated Urine Na⁺** (>20 mmol/L) |
| 6 | **Normal Thyroid, Adrenal, Renal Function** (TSH, Cortisol, Creatinine Normal) |
| 7 | **No Recent Diuretic Use** (Within 1-2 Weeks) |

### Common Causes of SIADH
| Category | Examples |
|----------|----------|
| **Malignancy** | **Small Cell Lung Cancer** (Most Common), Pancreatic, Thymoma, Lymphoma |
| **CNS Disorders** | Stroke, Haemorrhage, Meningitis, Encephalitis, Guillain-Barré, Tumours, Trauma, Neurosurgery |
| **Pulmonary** | Pneumonia, TB, Lung Abscess, Positive Pressure Ventilation |
| **Drugs** | **SSRIs, SNRIs, TCAs**, Carbamazepine, Oxcarbazepine, Chlorpropamide, Cyclophosphamide, Vincristine, MDMA |
| **Post-Operative** | Post-Neurosurgery, Post-Abdominal Surgery |

## 2. Cerebral Salt Wasting (CSW) vs SIADH

| Feature | **SIADH** | **CSW** |
|---------|-----------|---------|
| **Volume Status** | **Euvolaemic** | **Hypovolaemic** |
| **Urine Na⁺** | >20 mmol/L | >20 mmol/L (Often Higher) |
| **Urine Osmolality** | >100 mOsm/kg | >100 mOsm/kg |
| **JVP / Skin Turgor** | Normal | Low / Dry |
| **Postural Blood Pressure** | Normal | **Postural Drop >20 mmHg** |
| **Urine Output** | Low (ADH Effect) | **High (Polyuria)** |
| **Serum Uric Acid** | Low/Normal | High |
| **Pathophysiology** | Inappropriate ADH Secretion | **Renal Salt Wasting** → Volume Depletion → ADH Release |
| **Management** | **Fluid Restriction** | **Saline Resuscitation** + Fludrocortisone |

## 3. Clinical Presentation

| Severity | Serum Na⁺ | Symptoms |
|----------|-----------|----------|
| **Mild** | 130-134 | Asymptomatic, Subtle Cognitive Impairment, Falls Risk |
| **Moderate** | 125-129 | Nausea, Headache, Confusion, Lethargy, Muscle Cramps |
| **Severe** | **<125** | **Seizures, Coma, Respiratory Arrest, Brainstem Herniation** |

## 3. Investigation Algorithm

```
HYPONATRAEMIA (Na⁺ <135)
         │
         ▼
STEP 1: ASSESS VOLUME STATUS (JVP, Skin Turgor, BP, Postural Drop, Urine Output)
         │
         ├── HYPOVOLAEMIC
         │       ├── Urine Na⁺ <20 → Extra-renal Loss (GI, Skin, Third Space)
         │       │       → **0.9% NaCl Resuscitation**
         │       │
         │       └── Urine Na⁺ >20 → Renal Loss (Diuretics, Addison's, CSW, Salt-Wasting)
         │               → **0.9% NaCl Resuscitation** + Treat Cause
         │
         ├── EUVOLAEMIC
         │       ├── **SIADH** (Most Common): Urine Osmo >100, Urine Na⁺ >20, Normal TSH/Cortisol
         │       │       → **Fluid Restriction 1L/d** → Demeclocycline → Tolvaptan
         │       │
         │       ├── **Cerebral Salt Wasting** → Hypovolaemic Signs + High Urine Na⁺
         │       │       → **Saline Resuscitation + Fludrocortisone**
         │       │
         │       ├── **Hypothyroidism** / **Adrenal Insufficiency** → Replace Hormone
         │       │
         │       └── **Drug-Induced** → Stop Offending Drug
         │
         └── HYPERVOLAEMIC
                 → Heart Failure / Cirrhosis / Nephrotic Syndrome / CKD
                 → **Fluid Restriction + Diuretics** (Careful Na⁺ Monitoring)
```

## 3. Acute vs Chronic Management

### Severe Symptomatic Hyponatraemia (Seizures, Coma, GCS <8) — **MEDICAL EMERGENCY**

| Step | Action |
|------|--------|
| **1. Secure Airway** | Intubate if GCS <8 / Seizuring |
| **2. 3% Hypertonic Saline** | **100-150 mL IV over 15-20 min** (Bolus) |
| **3. Repeat Bolus** | If Seizures Persist / Na⁺ Rise <5 mmol/L: Repeat 100-150 mL |
| **4. Target Rise** | **4-6 mmol/L in First 6h** (Max 8-10 mmol/L in 24h) |
| **5. Monitor** | Na⁺ q1-2h (First 6h), Then q4-6h; Neurological Status q1h |
| **6. Stop 3% Saline** | When Na⁺ Rise ≥5 mmol/L OR Symptoms Resolve |

### Chronic / Asymptomatic Hyponatraemia

| Step | Action |
|------|--------|
| **1. Identify & Treat Cause** | Stop Diuretics, Treat Hypothyroidism/Adrenal Insufficiency, Fluid Restriction |
| **2. Fluid Restriction** | **1L/day** (All Fluids) — 1st Line for SIADH |
| **3. Demeclocycline** | 600-1200 mg/day (If Fluid Restriction Fails; Induces Nephrogenic DI) |
| **4. Tolvaptan** | **V2 Receptor Antagonist** 15mg OD → Titrate to 30/45/60mg; Max 30 Days (US) |
| **5. Urea** | 30-60g/day (If Tolvaptan Unavailable; Renal Excretion) |
| **6. Target Rise** | **6-8 mmol/L/24h** (Max 10-12 mmol/L/24h) |

## 3. Correction Rate Limits (Critical)

| Scenario | Max Rate |
|----------|----------|
| **Acute Symptomatic** | **4-6 mmol/L in 6h** (Max 8-10 mmol/L/24h) |
| **Chronic / Asymptomatic** | **6-8 mmol/L/24h** (Max 10-12 mmol/L/24h) |
| **High Risk ODS** (Malnutrition, Liver Disease, Alcoholism, Severe Hyponatraemia <120) | **Slower: 4-6 mmol/L/24h** |

> **Osmotic Demyelination Syndrome (ODS)**: Risk if Correction >10-12 mmol/L/24h (or >18 mmol/L/48h). Presents 2-6 days post-correction with Dysarthria, Dysphagia, Quadriparesis, Pseudobulbar Palsy, Coma. **Irreversible in Many Cases**.

---

## 3. SIADH Specific Management

| Step | Treatment | Dose / Details |
|------|-----------|----------------|
| **1st Line** | **Fluid Restriction** | 800-1000 mL/day (All Fluids) |
| **2nd Line** | **Demeclocycline** | 600-1200 mg/day (PO Divided); Induces Nephrogenic DI; Monitor U&Es, Photosensitivity |
| **3rd Line** | **Tolvaptan** (V2 Antagonist) | 15mg OD → Titrate 30, 45, 60mg; Max 30 Days (US); Monitor LFTs, Na⁺ q4-6h Initially |
| **Alternative** | **Urea** | 30-60g/day PO (Induces Osmotic Diuresis) |

## 3. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **Most Common Cause Hospital Hyponatraemia** | **SIADH** |
| **SIADH Criteria** | **All 7 Schwartz-Bartter Criteria** Must Be Met |
| **SIADH vs CSW** | **Volume Status**: SIADH = Euvolaemic; CSW = Hypovolaemic |
| **CSW Management** | **Saline Resuscitation** (Not Fluid Restriction!) + Fludrocortisone |
| **Correction Rate Limit** | **Max 8-10 mmol/L/24h** (Chronic); **4-6 mmol/L in 6h** (Acute Severe) |
| **ODS Risk** | Correction >10-12 mmol/L/24h → Central Pontine Myelinolysis |
| **3% Saline Dose** | 100-150 mL Bolus over 15-20 min (Severe Symptomatic); Target ↑5 mmol/L in 6h |
| **Fluid Restriction** | **1L/day** (1st Line SIADH); Include ALL Fluids |
| **Demeclocycline** | 600-1200 mg/d; Induces Nephrogenic DI; Photosensitivity/Nephrotoxicity Risk |
| **Tolvaptan** | V2 Antagonist; 15mg OD → 60mg Max; **Max 30 Days** (US); Monitor LFTs |
| **Hyperglycaemia Correction** | Corrected Na⁺ = Measured Na⁺ + 1.6 × (Glucose - 5.6)/5.6 |
| **CSW vs SIADH - Urine Output** | CSW = Polyuria; SIADH = Low Urine Output |
| **CSW Uric Acid** | High (vs Low/Normal in SIADH) |
| **Adrenal Insufficiency** | Always Exclude in Euvolaemic Hyponatraemia (Cortisol + ACTH) |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **SIADH vs CSW** | **Volume Status is Key**: SIADH = Euvolaemic; CSW = Hypovolaemic |
| **CSW Management** | **Saline + Fludrocortisone** (Opposite of SIADH) |
| **Rapid Correction** | >10-12 mmol/L/24h → **Central Pontine Myelinolysis** (ODS) |
| **Demeclocycline** | Induces **Nephrogenic DI**; Photosensitivity / Nephrotoxicity |
| **Tolvaptan** | **V2 Antagonist**; **Max 30 Days** (US); Hepatotoxicity Risk |
| **Fluid Restriction** | **1L/day ALL Fluids** (1st Line SIADH) |
| **3% Saline in Severe** | **100-150 mL Bolus** (Target ↑5 mmol/L in 6h) |
| **Correction Rate** | **Chronic: 6-8/24h Max**; **Acute Severe: 4-6 in 6h** |
| **ODS Presentation** | 2-6 Days Post-Correction: Dysarthria, Dysphagia, Quadriparesis, Coma |
| **Urine Na⁺ in SIADH** | **>20 mmol/L** (Inappropriately Concentrated Urine) |
| **Corrected Na⁺ for Glucose** | Measured Na⁺ + 1.6 × (Glucose - 5.6)/5.6 |

   B. Serum osmolality <275 mOsm/kg
   C. Urine osmolality >100 mOsm/kg

3. **Cerebral salt wasting (CSW) is differentiated from SIADH primarily by:**
   A. Urine sodium concentration
   B. Volume status
   C. Urine osmolality

   A. 4-6 mmol/L/24h
   B. 6-8 mmol/L/24h
   C. 10-12 mmol/L/24h

   A. 3% Hypertonic saline
   B. Fluid restriction <1 L/day

   A. >6 mmol/L/24h

   A. Fluid restriction

## Core Pathways Table (16)

| Topic | Diagnostic Pathway | Treatment Pathway |
|-------|-------------------|-------------------|
| **Hyponatraemia** | Volume → Urine Osmo/Na⁺ → SIADH vs CSW vs Hypovolaemic | 3% NaCl (Severe), Fluid Restrict, Demeclocycline, Tolvaptan |
| **Hypernatraemia** | Volume → Urine Osmo → Central/Nephrogenic DI | Free Water Deficit = TBW × (Current Na⁺/140 - 1) |
| **SIADH** | Euvolaemic + Urine Osmo >100 + Urine Na⁺ >20 + Normal TSH/Cortisol | Fluid Restricted 1L/d → Demeclocycline → Tolvaptan |
| **Diabetes Insipidus** | Water Deprivation → Desmopressin → Central vs Nephrogenic | Central: Desmopressin; Nephrogenic: Thiazide + Amiloride + Low Salt |
| **Hypokalaemia** | ECG → Urine K⁺ >20 (Renal) vs <20 (GI) → Cause | PO KCl 40-80 mmol/d; IV if <2.5 or ECG Changes |
| **Hyperkalaemia** | ECG → Calcium Gluconate → Insulin/Dextrose → Kayexalate/Dialysis | Calcium Gluconate 10% 10mL → Insulin 10U + 50mL 50% Dextrose |
| **Hypocalcaemia** | Ionised Ca²⁺ → Acute (Tetany) vs Chronic | IV Calcium Gluconate 10-20mL 10% → Chronic: Ca²⁺ + Alfacalcidol |
| **Hypercalcaemia** | PTH → Primary HPT vs Malignancy vs Granulomatous | Hydration → Bisphosphonate (Zoledronic) → Denosumab if Refractory |
| **Metabolic Acidosis** | Anion Gap → MUDPILES → Non-Gap (Renal/GI) | Bicarbonate if pH <7.1 or Severe AKI/Sepsis |
| **Metabolic Alkalosis** | Urine Cl⁻ <20 (Chloride-Responsive) vs >20 (Resistant) | 0.9% NaCl + KCl (Responsive); Acetazolamide (Resistant) |
| **Respiratory Acidosis** | Acute: pH↓, pCO₂↑ → Chronic: pH↓, pCO₂↑, HCO₃⁻↑ | Treat Cause (COPD, Opiates); NIV/Intubation if Severe |
| **Respiratory Alkalosis** | pH↑, pCO₂↓ → Hyperventilation Causes | Rebreathing, Treat Anxiety, Salicylate Toxicity |
| **Mixed Disorders** | Delta Ratio (ΔAG/ΔHCO₃⁻) + Delta-Delta | Treat Primary Disorder; Identify Components |
| **Osteoporosis** | DEXA T-Score → FRAX → Indication | Bisphosphonate 1st Line → Denosumab/Romo if Contraindicated |
| **Acute Porphyria** | Urine PBG (↑↑) → ALA → Genetic Confirmation | Haemin Arginate 3-4mg/kg IV daily × 4 days; Avoid Precipitants |

## Quick Reference: Formulae & Cut-offs

| Parameter | Formula / Cut-off |
|-----------|-------------------|
| **Free Water Deficit** | TBW × (Current Na⁺/140 - 1) |
| **Corrected Na⁺ (Hyperglycaemia)** | Measured Na⁺ + 1.6 × (Glucose - 5.6)/5.6 |
| **Anion Gap** | Na⁺ - (Cl⁻ + HCO₃⁻) [Normal 8-12] |
| **Corrected AG (Hypoalbuminaemia)** | AG + 2.5 × (40 - Albumin) |
| **Delta Ratio** | ΔAG / ΔHCO₃⁻ (1-2 = Pure AGMA; <0.4 = NAGMA + AGMA; >2 = AGMA + Met Alk) |
| **Winter's Formula** | Expected pCO₂ = 1.5 × HCO₃⁻ + 8 ± 2 |
| **K⁺ Replacement** | 20-40 mmol KCl per 1 mmol/L deficit (IV); PO 40-80 mmol/d |
| **Water Deficit (DI)** | 4 mL/kg × (Serum Osmo - 285) |
| **Insulin for Hyperkalaemia** | 10U Actrapid + 50mL 50% Dextrose → Recheck K⁺/Glucose 30min |
| **Calcium Gluconate (Acute Hypocalcaemia)** | 10-20mL 10% IV over 10min (Cardiac Monitor) |
| **Zoledronic Acid (Hypercalcaemia)** | 4mg IV over 15min (Adjust for eGFR) |
| **Denosumab (Hypercalcaemia)** | 120mg SC (Monthly) - No Renal Adjustment |
| **Dexamethasone (Porphyria)** | 3-4mg/kg IV daily × 4 days |
| **FRAX Threshold (UK)** | Major Osteoporotic ≥7.5-15% (Age-Dep); Hip ≥3-4.5% |

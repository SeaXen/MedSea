# Long-term Oxygen Therapy (LTOT) in COPD

Related: [[COPD]], [[Respiratory Failure]], [[Airway Diseases/COPD spectrum|COPD spectrum]], [[Acute Medicine and Critical Illness/Oxygen Therapy and NIV|Oxygen Therapy and NIV]]

> [!important]
> **LTOT** = oxygen ≥15 hours/day for **chronic hypoxaemia** in COPD. **Evidence-based**: MRC/NOTT trials show **survival benefit** if PaO₂ ≤7.3 kPa (55 mmHg) or ≤8 kPa (60 mmHg) with cor pulmonale/polycythaemia. **Target**: SpO₂ 90-92% (PaO₂ ≥8 kPa/60 mmHg). **Key FCPS/MRCP**: MRC/NOTT criteria, target SpO₂, assessment for LTOT, portable oxygen, safety, re-assessment, withdrawal criteria.

## Learning Objectives
- Apply LTOT criteria (MRC/NOTT) for COPD
- Distinguish LTOT from ambulatory/short-burst/palliative oxygen
- Apply oxygen prescription (flow rate, duration, target SpO₂)
- Recognise safety concerns (fire, CO₂ retention, trips)
- Apply reassessment and withdrawal criteria

## Definition
**Long-term Oxygen Therapy (LTOT)** = continuous oxygen therapy **≥15 hours/day** (usually continuous/overnight) for **chronic stable hypoxaemia** in advanced COPD (and other chronic lung diseases) — **proven survival benefit**.

## Evidence Base (Landmark Trials)
| Trial | Population | Intervention | Key Finding |
|-------|------------|--------------|-------------|
| **MRC (1981)** | COPD, PaO₂ 40-60 mmHg | O₂ 15h/day vs none | **↓ Mortality** (continuous > nocturnal) |
| **NOTT (1980)** | COPD, PaO₂ ≤55 mmHg | Continuous (24h) vs nocturnal (12h) | **Continuous 24h** → ↓ mortality vs 12h |
| **Göteborg** | COPD, PaO₂ 56-65 mmHg | LTOT vs none | No mortality benefit if PaO₂ >55 mmHg |

> **Consensus**: **Continuous ≥15h/day** (ideally 24h) for **PaO₂ ≤7.3 kPa (55 mmHg)**.

## LTOT Indications (COPD) — BTS/NICE/ERS Consensus

### **Absolute Criteria (Strong Evidence)**
| Criterion | Value |
|-----------|-------|
| **PaO₂** | **≤7.3 kPa (55 mmHg)** at rest, stable, on optimal therapy |
| **SaO₂** | **≤88%** |

### **Relative Criteria (Evidence for Cor Pulmonale/Polycythaemia)**
| Criterion | Value |
|-----------|-------|
| **PaO₂** | **7.3–8.0 kPa (55–60 mmHg)** **PLUS** any of: |
| | **Cor pulmonale** (peripheral oedema, raised JVP, RVH on ECG/echo) |
| | **Polycythaemia** (Hb >18 g/dL, Hct >55%) |
| | **Nocturnal desaturation** (SpO₂ <90% >30% sleep time) |
| | **Exercise desaturation** (if ambulatory O₂ considered) |

> **Not indicated**: PaO₂ >8 kPa (60 mmHg) without complications.

## Exclusion / Caution
| Situation | Action |
|-----------|--------|
| **Active smoker** | **Strongly advise cessation** first; LTOT + smoking = fire risk + negates benefit |
| **Unstable/acute exacerbation** | Treat exacerbation first; reassess after 4-8 weeks stability |
| **Severe untreated OSA** | Treat OSA first (CPAP may correct hypoxaemia) |
| **Severe cognitive impairment** | Assess ability to use safely; consider carer support |
| **Uncontrolled psychiatric illness** | Safety risk |

## Oxygen Prescription (The "Drug" Prescription)

| Component | Details |
|-----------|---------|
| **Flow rate** | **1–2 L/min** via nasal cannula (titrate to target) |
| **Target SpO₂** | **90–92%** (PaO₂ ≥8 kPa / 60 mmHg) — **avoid >94%** (hyperoxia risk) |
| **Duration** | **≥15 hours/day** (ideally continuous 24h) |
| **Device** | **Nasal cannula** standard; **mask** if cannula intolerant |
| **Source** | **Concentrator** (static, home); **Cylinders** (portable); **Liquid O₂** (high volume) |
| **Humidification** | Not routine for ≤4 L/min; add if >4 L/min or dry mucosa |

> **Target PaO₂ ≥8 kPa (60 mmHg)** but **<10.6 kPa (80 mmHg)** — avoid hyperoxia.

## Equipment & Delivery
| System | Use Case |
|--------|----------|
| **Concentrator** | Static home use (continuous flow up to 5-10 L/min) |
| **Portable cylinders** | Ambulatory use (CD, ED, PD types; limited duration) |
| **Conserving devices** | Demand/pulse dose (extends cylinder life) |
| **Liquid oxygen** | High volume users (>21h/day); portable dewars |
| **Transtracheal catheter** | Selected patients (cosmesis, compliance); specialist only |

## Assessment for LTOT (Stepwise)
1. **Clinical stability**: ≥4-8 weeks post-exacerbation, optimal therapy (bronchodilators, rehab, steroids if indicated)
2. **ABG on air** (2 readings, ≥3 weeks apart, stable): PaO₂, PaCO₂, pH
3. **Spirometry**: FEV₁, FVC (document severity)
4. **Echo/ECG**: screen for cor pulmonale (RVH, dilatation)
5. **FBC**: polycythaemia (Hb >18 g/dL)
6. **Overnight oximetry**: nocturnal desaturation screen
6. **Exercise test** (6MWT + oximetry): if ambulatory O₂ considered
7. **Safety assessment**: smoking status, fire risk, cognitive ability, home environment

## Targets & Monitoring
| Parameter | Target | Frequency |
|-----------|--------|-----------|
| **SpO₂** | **90–92%** (at rest, on prescribed flow) | At initiation, then 3-monthly |
| **PaO₂** | **≥8 kPa (60 mmHg)** | ABG at 4-6 weeks, then annually |
| **PaCO₂** | Monitor for rise (>0.5 kPa rise = review) | ABG at 4-6 weeks, then annually |
| **Flow rate** | Adjust to achieve SpO₂ 90-92% | Each visit |
| **Usage** | **≥15h/day** (ideally 24h) | Concentrator hour meter / patient diary |

## Ambulatory / Short-Burst / Palliative Oxygen (Distinct from LTOT)
| Type | Indication | Evidence |
|------|------------|----------|
| **Ambulatory oxygen** | Exercise desaturation (SpO₂ <90% on 6MWT) + improved exercise capacity/dyspnoea | Modest benefit; only if LTOT criteria NOT met |
| **Short-burst oxygen** | Breathlessness relief during ADLs (no desaturation) | **No evidence** of benefit; not recommended |
| **Palliative oxygen** | Terminal breathlessness (hypoxic or not) | Symptom relief; do not confuse with LTOT |

> **Only LTOT has survival benefit** in COPD hypoxaemia.

## Safety & Complications
| Risk | Mitigation |
|------|------------|
| **Fire** | **NO SMOKING** (patient/carers); no open flames, e-cigarettes; "Oxygen in use" signs |
| **CO₂ retention** | Start low (1-2 L/min); target SpO₂ 88-92%; monitor PaCO₂ at 4-6 weeks |
| **Nasal dryness/epistaxis** | Humidifier if >4 L/min; saline spray |
| **Skin irritation** | Padding, alternate cannula/mask |
| **Trip hazards** | Tubing management, cordless phones |
| **Concentrator noise/heat** | Ventilated room, away from bedroom if noise-sensitive |

## Reassessment & Withdrawal
| Timeline | Action |
|----------|--------|
| **4-6 weeks** | ABG, SpO₂, review flow rate; adjust to target |
| **3 months** | ABG, SpO₂, usage (hour meter), FBC, echo |
| **Annually** | ABG, SpO₂, echo, FBC, spirometry, equipment check |
| **Withdrawal criteria** | PaO₂ >8 kPa (60 mmHg) on 2 ABGs, 3 months apart, stable, no cor pulmonale/polycythaemia |

> **Do NOT withdraw** if PaO₂ ≤7.3 kPa, or if PaO₂ 7.3-8 kPa with cor pulmonale/polycythaemia.

## FCPS/MRCP High-Yield Points
1. **LTOT criteria**: PaO₂ ≤7.3 kPa (55 mmHg) OR 7.3-8 kPa + cor pulmonale/polycythaemia
2. **Survival benefit**: MRC/NOTT trials — **continuous ≥15h/day** (ideally 24h)
3. **Target SpO₂**: **90-92%** (PaO₂ ≥8 kPa/60 mmHg); **avoid >94%**
4. **NOTT trial**: continuous 24h > nocturnal 12h for survival
5. **Assessment**: stable COPD, ABG on air (2 readings 3wk apart), echo, FBC, overnight oximetry
6. **Ambulatory O₂**: only if exercise desaturation + improved endurance; **NOT** for breathlessness alone
7. **Short-burst O₂**: **no evidence** of benefit
8. **Safety**: NO SMOKING (fire risk), CO₂ retention (target SpO₂ 88-92%), fire risk
8. **Reassessment**: 4-6 weeks, 3 months, annually; withdraw if PaO₂ >8 kPa stable
9. **Cor pulmonale + polycythaemia** = indications at PaO₂ 7.3-8 kPa
9. **NOTT**: continuous > nocturnal; MRC: 15h/day > none

## Common Confusions / Exam Traps
- **Target SpO₂ = 90-92%**, NOT 94-98% (hyperoxia harms COPD)
- **PaO₂ 7.3-8 kPa needs complication** (cor pulmonale/polycythaemia) for LTOT
- **Smoking + LTOT** = fire risk + negates survival benefit
- **Continuous 24h > nocturnal 12h** (NOTT trial)
- **Ambulatory O₂ ≠ LTOT** — only if exercise desaturation + improved walk distance
- **Short-burst O₂** = no evidence
- **CO₂ retention** = target SpO₂ 88-92%, NOT 94-98%
- **Withdrawal** only if PaO₂ >8 kPa on 2 ABGs 3mo apart, stable

## Summary
**LTOT** = oxygen ≥15h/day for chronic hypoxaemic COPD. **Criteria**: PaO₂ ≤7.3 kPa **OR** 7.3-8 kPa **PLUS** cor pulmonale/polycythaemia. **Evidence**: MRC (15h > none), NOTT (continuous 24h > nocturnal 12h) — **survival benefit**. **Target SpO₂ 90-92%** (PaO₂ ≥8 kPa); **NOT 94-98%**. **Assessment**: stable COPD, 2 ABGs 3 weeks apart, echo, FBC, overnight oximetry. **Prescription**: 1-2 L/min nasal cannula, **≥15h/day (ideally 24h)**, target SpO₂ 90-92%. **Safety**: NO smoking (fire), CO₂ retention (target SpO₂ 88-92%). **Ambulatory O₂**: exercise desaturation + improved walk; **short-burst = no evidence**. **Reassess**: 6 weeks, 3 months, annually; **withdraw if PaO₂ >8 kPa stable**.

   D. 15 L/min
   E. No flow
   **Answer: B** — 2–3 L/min NC or 24–28% Venturi.

10. Which device is preferred for **accurate O₂ delivery in COPD**?
    A. Nasal cannula only
    B. **Venturi mask (fixed performance)**
    C. Simple face mask
    D. Non-rebreather
    E. HFNC
    **Answer: B** — Venturi delivers precise FiO₂, ideal for Type 2 risk.

## Additional SBAs (6–10)

6. A stable COPD patient on home LTOT for 2 years. SpO₂ 96% on 3 L/min. Action:
   A. Reduce O₂
   B. **Wean if PaO₂ >8.0 kPa off O₂ (reassess)**
   C. Stop immediately
   D. Increase O₂
   E. No change
   **Answer: B** — Reassess; if no longer meeting criteria, wean.

7. Ambulatory O₂ in COPD:
   A. Standard for all
   B. **Indicated if exercise desaturation <88% and improves with O₂**
   C. Never used
   D. Only at night
   E. Only when severe
   **Answer: B** — Ambulatory O₂ for exercise-induced desaturation.

8. LTOT in COPD improves:
   A. Lung function
   B. **Survival (when used ≥15 h/day)**
   C. CXR appearance
   D. FEV₁
   E. Sputum
   **Answer: B** — Survival benefit only (NOTT, MRC trials).

9. Smoking with LTOT:
   A. **Absolute contraindication (fire risk)**
   B. Relative contraindication
   C. Allowed
   D. Only outside
   E. Reduce O₂ flow
   **Answer: A** — Fire risk; absolute contraindication.

10. A COPD patient with PaO₂ 7.5 kPa, no cor pulmonale, no polycythaemia. LTOT:
    A. Indicated
    B. **Not indicated (PaO₂ >7.3 and <8.0 with no complications)**
    C. Ambulatory only
    D. Nocturnal only
    E. PRN
    **Answer: B** — PaO₂ 7.5 (between 7.3 and 8.0) without complications does not meet criteria for LTOT; re-assess.

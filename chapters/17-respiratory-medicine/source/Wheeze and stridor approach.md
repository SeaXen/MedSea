# Wheeze and stridor approach

> [!important]
> **Wheeze** = continuous musical sound from **lower airway obstruction** (asthma, COPD). **Stridor** = inspiratory monophonic sound from **upper airway obstruction** (croup, foreign body, anaphylaxis, tumour). Stridor is a **medical emergency** — never ignore it.

Related: [[Asthma]], [[COPD]], [[Foreign body aspiration in adults]], [[Vocal cord dysfunction and inducible laryngeal obstruction|VCD]], [[Upper airway obstruction and stridor]]

> [!tip] **FCPS/MRCP pearl**: **Inspiratory stridor = extrathoracic** (laryngeal, subglottic). **Expiratory wheeze = intrathoracic** (bronchi). **Biphasic = fixed obstruction** (subglottic stenosis, tracheal tumour). **Stridor + anaphylaxis = IM adrenaline**. **Sudden stridor in child = foreign body**.

## Definition

- **Wheeze**: continuous, high-pitched musical sound from **lower airway** obstruction (airway narrowing, secretions)
- **Stridor**: monophonic harsh sound from **upper airway** obstruction (larynx, trachea)
  - **Inspiratory**: extrathoracic (above thoracic inlet)
  - **Expiratory**: intrathoracic (below thoracic inlet)
  - **Biphasic**: fixed or subglottic

## Aetiology

### Wheeze (lower airway)
| Cause | Notes |
|-------|-------|
| **Asthma** | Expiratory, polyphonic, variable |
| **COPD** | Expiratory, chronic, smoking |
| **Bronchiolitis** | Children, viral |
| **Bronchiectasis** | Chronic, often localised |
| **Tumour** (endobronchial) | Fixed monophonic wheeze (focal) |
| **Heart failure** (cardiac asthma) | Bilateral |
| **Anaphylaxis** | Acute, with stridor + hypotension |
| **Vocal cord dysfunction (VCD)** | Inspiratory, often mimics asthma |

### Stridor (upper airway)
| Cause | Notes |
|-------|-------|
| **Croup** (laryngotracheobronchitis) | Children, barking cough, viral |
| **Epiglottitis** | Children, toxic, drooling, sitting up |
| **Foreign body** | Sudden, often children, unilateral wheeze |
| **Anaphylaxis** | Acute, with urticaria, hypotension |
| **Laryngeal oedema** | Post-extubation, ACE-i, hereditary angioedema |
| **Laryngeal tumour** | Smoker, progressive hoarseness |
| **Bilateral vocal cord paralysis** | Post-thyroidectomy, neurological |
| **Subglottic stenosis** | Post-intubation |
| **Tracheal stenosis** | Post-tracheostomy |
| **Laryngomalacia** | Infant, noisy breathing |
| **Retropharyngeal abscess** | Children, fever, drooling |
| **Diphtheria** | Unvaccinated, membrane |

## Pathophysiology

### Airway narrowing
- **Air flow through narrowed airway** → turbulence → sound
- **Stridor = ≥50% airway narrowing** (often critical)
- **Inspiratory stridor** = extrathoracic (larynx, subglottic) — narrows on inspiration (negative pressure)
- **Expiratory stridor** = intrathoracic (trachea, bronchi) — narrows on expiration (positive pressure)
- **Biphasic** = fixed (cartilage) or subglottic

### Acute severe obstruction
- **Complete obstruction** → stridor → aphonia → respiratory arrest
- **Time-critical** (minutes to hours)

## Clinical Features

### Symptoms
- **Noisy breathing** (wheeze, stridor)
- **Dyspnoea**
- **Stridor** (inspiratory, expiratory, biphasic)
- **Hoarseness** (laryngeal)
- **Cough** (croupy, barking, brassy)
- **Drooling, dysphagia** (epiglottitis, abscess)
- **Anaphylaxis symptoms** (urticaria, angioedema)

### Signs
- **Stridor** (auscultate over neck/chest)
- **Wheeze** (auscultate chest)
- **Use of accessory muscles**
- **Tracheal tug, intercostal recession**
- **Cyanosis** (severe)
- **Tachycardia, tachypnoea**
- **Toxic appearance** (epiglottitis, anaphylaxis)
- **Drooling, tripod position** (epiglottitis)
- **Stridor louder when supine** (VCD, laryngomalacia)

## Diagnosis

### Investigations
| Test | Purpose |
|------|---------|
| **ABG** | Hypoxaemia, hypercapnia (severe) |
| **CXR** (inspiration + expiration) | Foreign body, mass, mediastinal shift, hyperinflation |
| **Neck X-ray (AP)** | Subglottic narrowing (steeple sign) in croup |
| **CT neck/chest** | Mass, abscess, foreign body |
| **Bronchoscopy** | Foreign body, tumour, biopsy |
| **Laryngoscopy** | Epiglottitis, vocal cords |
| **Spirometry (flow-volume loop)** | Extrathoracic vs intrathoracic obstruction |
| **Allergy tests** | Anaphylaxis triggers |
| **FEES** (fibreoptic endoscopic evaluation of swallow) | Laryngeal function |

### Flow-volume loop
- **Extrathoracic obstruction**: flattening of **inspiratory** limb (variable)
- **Intrathoracic (variable)**: flattening of **expiratory** limb
- **Fixed**: flattening of both limbs

## Management

### General
- **A–B–C** assessment
- **Position**: sitting up (avoid supine if epiglottitis)
- **O₂** (humidified)
- **Avoid distressing** the patient (esp. epiglottitis)
- **Senior/anaesthesia/ENT** early
- **Keep child with parent** (reduce distress)

### Specific
| Cause | Rx |
|-------|-----|
| **Anaphylaxis** | **IM adrenaline 0.5 mg (1:1000)** + IV fluids + antihistamine + steroids + intubate if needed |
| **Croup** | Dexamethasone/nebulised budesonide + nebulised adrenaline if severe |
| **Epiglottitis** | **Anaesthesia + intubation** (in OR); ceftriaxone + dexamethasone |
| **Foreign body** | **Bronchoscopic removal** (rigid preferred in children) |
| **Asthma/COPD** | Bronchodilators, steroids, O₂ |
| **Laryngeal oedema** | Adrenaline (nebulised), steroids, intubate |
| **Vocal cord dysfunction** | Speech therapy, reassurance |
| **Laryngeal tumour** | ENT evaluation, biopsy, oncology |
| **Bilateral VC paralysis** | Tracheostomy |
| **Subglottic stenosis** | Dilatation, stenting |

### Don'ts
- **Don't examine throat** (epiglottitis) — may cause complete obstruction
- **Don't lie child down** (epiglottitis)
- **Don't delay intubation** in impending obstruction

## FCPS/MRCP High-Yield Summary

| Domain | Key points |
|--------|------------|
| **Stridor** | Upper airway; emergency |
| **Wheeze** | Lower airway (asthma, COPD) |

| **Expiratory** | Intrathoracic (trachea) |
| **Biphasic** | Fixed/subglottic |

## MCQs (10)

1. Inspiratory stridor is caused by:
   A. Tracheal tumour
   B. **Laryngeal obstruction (extrathoracic)**
   C. COPD
   D. Asthma
   E. None
   **Answer: B** — Extrathoracic.

2. Expiratory wheeze is most characteristic of:
   A. Laryngeal oedema
   B. **Lower airway obstruction (asthma, COPD)**
   C. Foreign body
   D. None
   E. Croup
   **Answer: B** — Lower airway.

3. Biphasic stridor suggests:
   A. Asthma
   B. **Fixed (subglottic stenosis, tracheal tumour)**
   C. COPD
   D. None
   E. Pneumonia
   **Answer: B** — Fixed.

4. First-line treatment of anaphylaxis with stridor:
   A. Steroids
   B. **IM adrenaline 0.5 mg (1:1000)**
   C. Antihistamine
   D. Wait
   E. None
   **Answer: B** — IM adrenaline.

5. The "steeple sign" on neck X-ray suggests:
   A. Epiglottitis
   B. **Croup (subglottic narrowing)**
   C. Foreign body
   D. None
   E. Asthma
   **Answer: B** — Croup.

6. The "thumb sign" on lateral neck X-ray suggests:
   A. Croup
   B. **Epiglottitis (swollen epiglottis)**
   C. Foreign body
   D. None
   E. Asthma
   **Answer: B** — Epiglottitis.

7. A child with sudden cough, stridor, unilateral wheeze. First:
   A. Steroids
   B. **Bronchoscopy (foreign body)**
   C. Antibiotics
   D. Wait
   E. None
   **Answer: B** — Foreign body removal.

8. The most common cause of croup:
   A. Bacterial
   B. **Viral (parainfluenza)**
   C. Allergic
   D. None
   E. Fungal
   **Answer: B** — Parainfluenza.

9. VCD (vocal cord dysfunction) is treated by:
   A. **Speech therapy + reassurance (not asthma Rx)**
   B. Steroids
   C. Adrenaline
   D. Antibiotics
   E. None
   **Answer: A** — Speech therapy.

10. Stridor in anaphylaxis indicates:
    A. Mild reaction
    B. **Severe/life-threatening (laryngeal oedema)**
    C. Asthma
    D. None
    E. Anxiety
    **Answer: B** — Severe.

## SBA Questions (10)

1. A 30-year-old with sudden onset stridor and urticaria after eating peanuts. Rx:
   A. Antihistamine
   B. **IM adrenaline 0.5 mg immediately**
   C. Steroids
   D. Wait
   E. None
   **Answer: B** — Adrenaline.

2. A 3-year-old with barking cough and stridor. Most likely:
   A. **Croup (viral)**
   B. Epiglottitis
   C. Asthma
   D. Foreign body
   E. None
   **Answer: A** — Croup.

3. The most appropriate investigation for chronic stridor is:
   A. ABG
   B. **Laryngoscopy +/− bronchoscopy + CT**
   C. CXR
   D. Spirogram
   E. None
   **Answer: B** — Direct visualisation + imaging.

4. A child with sudden unilateral wheeze. Most likely:
   A. Asthma
   B. **Foreign body**
   C. Pneumonia
   D. None
   E. Croup
   **Answer: B** — Foreign body.

5. The first-line treatment of severe croup is:
   A. Antibiotics
   B. **Dexamethasone (oral) ± nebulised adrenaline**
   C. Theophylline
   D. Wait
   E. None
   **Answer: B** — Dexamethasone.

6. A 50-year-old with hoarseness and progressive stridor, smoker. Best next:
   A. Antibiotics
   B. **Laryngoscopy + CT (laryngeal cancer workup)**
   C. Wait
   D. Steroids
   E. None
   **Answer: B** — Laryngeal cancer.

7. The most common cause of stridor in an adult:
   A. Croup
   B. **Laryngeal tumour (smoker) or laryngeal oedema**
   C. Epiglottitis
   D. Foreign body
   E. None
   **Answer: B** — Laryngeal pathology.

8. The flow-volume loop in VCD shows:
   A. **Variable extrathoracic obstruction (flattened inspiratory limb)**
   B. Normal
   C. Fixed obstruction
   D. Restrictive
   E. None
   **Answer: A** — Variable extrathoracic.

9. Bilateral vocal cord paralysis causes:
   A. Hoarseness
   B. **Stridor (often biphasic) +/− respiratory failure**
   C. None
   D. Wheeze
   E. Asthma
   **Answer: B** — Stridor + RF.

10. The best initial treatment of croup at home:
    A. Wait
    B. **Cool/steam air + reassure + oral dexamethasone if severe**
    C. Antibiotics
    D. Adrenaline
    E. None
    **Answer: B** — Cool air + dexamethasone.

## Summary

Stridor = upper airway emergency. Inspiratory = extrathoracic (laryngeal); expiratory = intrathoracic; biphasic = fixed. Anaphylaxis: IM adrenaline. Croup: dexamethasone. Epiglottitis: anaesthesia + intubation. Foreign body: bronchoscopic removal. VCD: speech therapy. Don't examine throat in epiglottitis.

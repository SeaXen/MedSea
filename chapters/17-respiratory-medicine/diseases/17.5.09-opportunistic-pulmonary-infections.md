# Opportunistic pulmonary infections

> [!important]
> **Opportunistic pulmonary infections** occur in **immunocompromised patients** (HIV, transplant, chemotherapy, steroids, immunosuppressants). They include **PJP, fungal infections (Aspergillus, Cryptococcus), CMV, NTM, and reactivation TB**. Diagnosis requires **aggressive workup** (CT, BAL, biopsy) and **empirical broad-spectrum therapy** in critically ill patients.

Related: [[Pneumonia]], [[Pneumocystis jirovecii pneumonia|PJP]], [[Pulmonary aspergillosis syndromes]], [[HIV Infection and AIDS|HIV Infection and AIDS]]

> [!tip] **FCPS/MRCP pearl**: In an HIV patient with **CD4 <200** and pneumonia → think **PJP**. In a transplant/chemo patient with cavitating lesion → think **Aspergillus** or **Nocardia**. In severe neutropenia → think **invasive aspergillosis**. Always consider **TB reactivation** in immunocompromised.

## 1. Definition

**Opportunistic pulmonary infections** = infections caused by organisms that typically do not cause disease in immunocompetent hosts, but exploit **immunocompromised** states.

## 2. Risk Factors (immunocompromise)

| Category | Examples |
|----------|----------|
| **HIV/AIDS** | CD4 <200, <100, <50 |
| **Transplant** (SOT, HSCT) | Immunosuppression, GVHD |
| **Chemotherapy** | Neutropenia, lymphopenia |
| **Steroids** | High-dose, prolonged |
| **Immunosuppressants** | Biologics, anti-TNF |
| **Haematological malignancy** | Leukaemia, lymphoma |
| **Solid tumours** | Advanced disease |
| **Diabetes** | Hyperglycaemia impairs immunity |
| **Chronic disease** | CKD, cirrhosis, malnutrition |
| **Age** | Elderly |

## 3. Common Organisms by Immune Defect

| Defect | Common organisms |
|--------|------------------|
| **CD4 <200 (HIV)** | PJP, bacterial (S. pneumo, H. flu) |
| **CD4 <100 (HIV)** | Toxoplasma, Cryptococcus, MAC |
| **CD4 <50 (HIV)** | CMV, MAC, disseminated fungal |
| **Neutropenia** | Bacteria (Pseudomonas, Staph), invasive aspergillosis, Candida |
| **Cellular immunity (transplant)** | CMV, PJP, fungi, mycobacteria |
| **Humoral immunity (CVID, etc.)** | Encapsulated bacteria |

## 4. Specific Infections

### Pneumocystis jirovecii pneumonia (PJP)
- **CD4 <200** (HIV); transplant; high-dose steroids
- **Subacute** dyspnoea, dry cough, fever
- **CXR**: bilateral perihilar/interstitial infiltrates
- **HRCT**: ground-glass opacities (often upper lobe in non-HIV)
- **Diagnosis**: induced sputum or BAL → silver/Pertussis stain, PCR
- **Treatment**: **TMP-SMX high dose** + **steroids** (if PaO₂ <70 or A-a >35)
- **Prophylaxis**: TMP-SMX when CD4 <200

### Invasive aspergillosis
- **Neutropenia** (especially prolonged)
- **Fever + new infiltrate** despite antibiotics
- **HRCT**: nodule with **halo sign** (GGO around nodule), air crescent sign (late)
- **Diagnosis**: BAL (Aspergillus PCR, galactomannan), biopsy
- **Treatment**: **voriconazole** (1st line), isavuconazole, liposomal amphotericin B
- **Surgery** for localised disease

### Cytomegalovirus (CMV) pneumonia
- **Transplant** (especially HSCT, lung)
- **Fever, dyspnoea, dry cough, hypoxaemia**
- **CXR**: bilateral interstitial infiltrates
- **Diagnosis**: BAL CMV PCR, viral load, intranuclear inclusions ("owl's eye")
- **Treatment**: **ganciclovir** IV (or valganciclovir oral)
- **Prophylaxis**: valganciclovir in high-risk

### Cryptococcus
- **HIV** (CD4 <100), transplant
- **Pneumonia** + meningitis (often)
- **Diagnosis**: serum/CSF cryptococcal antigen (CrAg)
- **Treatment**: amphotericin B + flucytosine → fluconazole

### NTM (non-tuberculous mycobacteria)
- **MAC** (esp. CD4 <50 in HIV), immunocompromised
- **Disseminated disease** in HIV
- **Treatment**: macrolide + ethambutol + rifabutin (×12+ months)

### Nocardia
- **Cell-mediated immunity defect**
- **Cavitating pneumonia** (often upper lobe)
- **CNS dissemination** (brain abscess)
- **Diagnosis**: modified acid-fast stain, culture
- **Treatment**: **TMP-SMX** (or imipenem + amikacin for severe)

### Other
- **Aspergillus fumigatus** → ABPA (in asthma/CF), aspergilloma
- **Candida** (rare pulmonary pathogen)
- **Toxoplasma** (HIV, CD4 <100)
- **Pneumocystis** (covered above)

## 5. Clinical Features

### Symptoms
- **Fever** (often only sign)
- **Dyspnoea** (subacute or acute)
- **Cough** (dry or productive)
- **Hypoxaemia** (may be profound)
- **Constitutional**: weight loss, night sweats, fatigue

### Signs
- **Fever, tachycardia, tachypnoea**
- **Crackles** (diffuse or focal)
- **Hypoxaemia** (silent chest if severe)
- **Signs of underlying disease**

## 6. Diagnosis

### Investigations
| Test | Purpose |
|------|---------|
| **CT chest** | Pattern recognition, biopsy guidance |
| **Sputum** | AFB, fungal, culture, cytology |
| **Blood cultures** | Bacteria, fungi |
| **Galactomannan** | Aspergillus |
| **CMV PCR** | CMV viraemia |
| **Cryptococcal Ag** | CrAg |
| **PJP PCR / silver stain** | PJP on BAL/sputum |
| **BAL** | Bronchoscopy with lavage |
| **Transbronchial biopsy** | For definitive diagnosis |
| **HIV test** | Always consider |
| **CD4 count** | Severity of HIV |
| **Galactomannan** | Aspergillus |

### Empirical treatment
- **Critically ill immunocompromised** with pneumonia:
  - **Empiric broad-spectrum** (anti-bacterial + anti-PJP + anti-fungal + anti-CMV)
  - E.g. meropenem + TMP-SMX + voriconazole + ganciclovir
  - **Adjust** to culture/biopsy

## 7. Management

### General
- **Treat underlying cause** (HIV, malignancy)
- **Antimicrobial therapy** (specific to organism)
- **Supportive**: O₂, ventilation
- **Reduce immunosuppression** (if possible)
- **PJP prophylaxis** when CD4 <200 (TMP-SMX)
- **Vaccinations** (inactivated, avoid live)

### Specific
| Infection | Treatment |
|-----------|-----------|
| **PJP** | TMP-SMX 15–20 mg/kg/day TMP + steroids if PaO₂ <70 |
| **Invasive Aspergillus** | Voriconazole (6–12 weeks) |
| **CMV pneumonia** | Ganciclovir IV |
| **Cryptococcus** | Amphotericin B + flucytosine → fluconazole |
| **MAC** | Macrolide + ethambutol + rifabutin |
| **Nocardia** | TMP-SMX (or imipenem + amikacin) |

## 13. Summary

Opportunistic pulmonary infections occur in immunocompromised. PJP (HIV CD4 <200): TMP-SMX + steroids if severe. Invasive aspergillosis (neutropenia): voriconazole. CMV (transplant): ganciclovir. MAC: macrolide + ethambutol + rifabutin. Always consider TB reactivation. Empiric broad-spectrum in critically ill.

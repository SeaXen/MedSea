# Fever with cough and consolidation approach

> [!important]
> **Fever + cough + consolidation** = **pneumonia** until proven otherwise. Severity assessment (CURB-65, PSI) guides admission decisions. Common pathogens: *S. pneumoniae, H. influenzae, atypicals (Mycoplasma, Legionella, Chlamydia)*. **Always consider TB in chronic/progressive presentations**.

Related: [[Pneumonia]], [[Tuberculosis]], [[ABG Interpretation]], [[Oxygen Therapy and NIV]], [[Respiratory Failure]]

> [!tip] **FCPS/MRCP pearl**: **CURB-65**: **C**onfusion, **U**rea >7, **R**R ≥30, **B**P <90/60, **A**ge ≥65. **Score 0–1**: outpatient; **2**: admit; **3+**: severe (consider ITU). Empirical antibiotics: amoxicillin (mild), amoxicillin + macrolide (moderate), co-amoxiclav + macrolide (severe). Always think TB in non-resolving.

## 1. Definition

**Pneumonia** = infection of the lung parenchyma with consolidation on imaging.

## 2. Aetiology

### By setting
| Setting | Common pathogens |
|---------|------------------|
| **Community-acquired (CAP)** | *S. pneumoniae* (~50%), *H. influenzae*, atypicals (*Mycoplasma, Chlamydia, Legionella*) |
| **Aspiration** | Anaerobes, mixed flora |
| **Hospital-acquired (HAP)** | *Pseudomonas, Klebsiella, Staph aureus* (esp. MRSA) |
| **Ventilator-associated (VAP)** | Multi-drug resistant organisms |
| **Immunocompromised** | PJP, fungi, CMV, atypicals |

### Specific clues
- **Travellers**: Legionella (hotels, AC), *Mycoplasma* (schools)
- **Animal exposure**: *Chlamydia psittaci* (birds), *Coxiella* (livestock)
- **Alcoholic**: *Klebsiella, S. pneumoniae*, anaerobes
- **Aspiration**: anaerobes
- **Post-influenza**: *Staph aureus* (severe)
- **Cystic fibrosis**: *Pseudomonas, Staph aureus*
- **HIV (CD4 <200)**: PJP
- **Neutropenia**: bacteria, fungi (Aspergillus)
- **Travel/sick contacts**: TB

## 3. Clinical Features

### Symptoms
- **Fever** (often rigors)
- **Cough** (productive; rust-coloured sputum in pneumococcus)
- **Dyspnoea**
- **Pleuritic chest pain** (if pleural involvement)
- **Fatigue, myalgia**
- **Confusion** (elderly)

### Signs
- **Fever, tachycardia, tachypnoea**
- **Consolidation signs**: dullness, ↓vocal resonance, bronchial breathing, crackles
- **Pleural rub** (if pleurisy)
- **Cyanosis** (severe)
- **Confusion** (hypoxaemia or sepsis)

## 4. Diagnosis

### Investigations
| Test | Purpose |
|------|---------|
| **CXR** (PA + lateral) | **Gold standard** — consolidation, complications |
| **Sputum** (Gram stain, culture) | Pathogen ID (if severe) |
| **Blood cultures** | If severe |
| **Urinary antigens** | *S. pneumoniae, Legionella* |
| **PCR** (respiratory viral panel) | Influenza, RSV, SARS-CoV-2 |
| **ABG** | Severity, hypoxaemia |
| **FBC, U&E, CRP, LFT** | Severity, sepsis |
| **Serology** | *Mycoplasma, Chlamydia* (paired) |
| **HIV test** | If recurrent/atypical |
| **CT chest** | Complication (abscess, effusion) or non-resolving |
| **Bronchoscopy + BAL** | Immunocompromised, atypical |

### CXR patterns
| Pattern | Common cause |
|---------|--------------|
| **Lobar consolidation** | *S. pneumoniae, Klebsiella* |
| **Bronchopneumonia** (patchy) | *S. pneumoniae, H. influenzae, atypicals* |
| **Interstitial/bilateral** | *Mycoplasma, viral, PJP, miliary TB* |
| **Cavitation** | *Staph aureus, Klebsiella, TB, fungi* |
| **Pleural effusion** | Complication, empyema |

## 5. Severity Assessment

### CURB-65 (BTS)
- **C**onfusion (new)
- **U**rea >7 mmol/L
- **R**espiratory rate ≥30/min
- **B**lood pressure (SBP <90 or DBP ≤60)
- **A**ge ≥65 years
- **Score**:
  - **0–1**: low risk → outpatient
  - **2**: moderate → admit
  - **3–5**: high risk → admit, consider ITU (4–5)

### PSI (Pneumonia Severity Index)
- More complex; similar discrimination
- Use for risk stratification

### Other severity markers
- **Hypoxaemia** (PaO₂ <60)
- **Acidosis** (pH <7.35)
- **Sepsis** (qSOFA, SIRS)
- **Multilobar infiltrates**
- **Effusion/empyema**
- **Bacteraemia**

## 6. Management

### Empirical antibiotics
| Setting | First-line |
|---------|-----------|
| **CAP, low severity (CURB 0–1)** | **Amoxicillin** 500 mg TDS × 5 days |
| **CAP, moderate (CURB 2)** | **Amoxicillin** + **macrolide** (clarithromycin or azithromycin) |
| **CAP, severe (CURB ≥3)** | **Co-amoxiclav** + **macrolide** (or doxycycline); or **levofloxacin** |
| **HAP/VAP** | **Piperacillin-tazobactam** +/− aminoglycoside/macrolide; consider MRSA cover |
| **Aspiration** | **Amoxicillin-clavulanate** or **clindamycin** |
| **Atypical predominant** | **Macrolide** (azithromycin) ± β-lactam |
| **Legionella** | **Levofloxacin** or **azithromycin** (≥7–10 days) |

### Supportive
- **O₂** (target SpO₂ 94–98%)
- **Hydration** (oral if able, IV if not)
- **Analgesia** (pleuritic pain: NSAIDs)
- **Antipyretic** (paracetamol)
- **Physiotherapy** (clearance)
- **Ventilation** if respiratory failure (NIV/CPAP/intubation)

### Complications
| Complication | Management |
|---------------|------------|
| **Pleural effusion** | Tap (diagnostic +/− therapeutic) |
| **Empyema** | Chest drain + antibiotics + fibrinolytics/VATS |
| **Lung abscess** | Prolonged antibiotics (4–6 weeks) |
| **ARDS** | Lung-protective ventilation, prone, treat cause |
| **Septic shock** | Fluids, vasopressors, antibiotics |
| **Death** | Severe CAP, especially elderly, comorbidity |

### Non-resolving pneumonia
- Consider: **TB**, lung cancer, PE, foreign body, empyema, fungal, atypical pathogens
- CT chest + bronchoscopy
- Reassess pathogen coverage

## 7. FCPS/MRCP High-Yield Summary

| Domain | Key points |
|--------|------------|
| **Most common** | *S. pneumoniae* |
| **CURB-65** | Confusion, Urea, RR, BP, Age |
| **CURB 0–1** | Outpatient, amoxicillin |
| **CURB 2** | Admit, amoxicillin + macrolide |
| **CURB 3+** | Severe, co-amoxiclav + macrolide |
| **Red flag** | Confusion, RR ≥30, BP low, hypoxaemia |
| **Atypicals** | Mycoplasma, Chlamydia, Legionella |

   A. Confusion
   B. Urea

   D. **Albumin**

   A. Outpatient
   B. Admit, oral

   A. **Co-amoxiclav + macrolide**

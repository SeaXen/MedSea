# Asbestosis

Related: [[Pneumoconiosis]], [[Silicosis]], [[Mesothelioma]], [[Lung cancer]], [[Pleural plaques]], [[Pleural effusion]], [[Occupational and environmental lung disease]], [[ILD framework]]

> [!important]
> **Asbestosis** = **diffuse pulmonary fibrosis** caused by **inhalation of asbestos fibres**. **Dose-dependent** (cumulative exposure). **Latency 20–40 years**. **Pathognomonic HRCT**: **subpleural curvilinear lines, parenchymal bands, basal fibrosis, honeycombing (late), pleural plaques (circumscribed, calcified, parietal)**. **Key FCPS/MRCP**: **Asbestos bodies (ferruginous bodies)** in BAL/lung tissue, **pleural plaques = exposure marker (not asbestosis)**, **synergistic with smoking for lung cancer** (×50–90 risk), **compensable** (UK IIDB prescribed disease D3/D4/D9).

## Learning Objectives
- Define **asbestosis** and distinguish from **pleural plaques** (exposure marker) and **mesothelioma** (malignancy)
- Recognise **high-risk occupations** and **asbestos types** (crocidolite > amosite > chrysotile)
- Interpret **HRCT findings** (subpleural lines, parenchymal bands, basal fibrosis, honeycombing, pleural plaques)
- Apply **ILO classification** for pneumoconiosis (profusion, size, shape of opacities)
- Diagnose using **asbestos bodies** (ferruginous bodies) in BAL/lung tissue
- Apply **HELPP criteria** (History, Exposure, Latency, Pulmonary function, Pathology/imaging)
- Recognise **synergistic carcinogenicity** with smoking (lung cancer risk ×50–90)
- Manage **complications** (progressive fibrosis, respiratory failure, lung cancer, mesothelioma, cor pulmonale)
- Guide **compensation claims** (UK IIDB prescribed diseases D3, D4, D9)

## Definition
**Asbestosis** = **bilateral diffuse interstitial pulmonary fibrosis** caused by **prolonged, heavy inhalation of asbestos fibres**. **Dose-related** (cumulative exposure). **Latency period 20–40 years** (sometimes 10–15 years with heavy exposure). **Irreversible**, progressive.

> **FCPS/MRCP tip**: **Asbestosis = parenchymal fibrosis** from asbestos. **Pleural plaques = exposure marker only (parietal pleura, benign, no fibrosis)**. **Do not confuse the two**.

## Core Anatomy / Fibre Types
### Asbestos Fibre Types (Amphiboles > Serpentine)
| Type | Category | Potency | Uses |
|------|----------|---------|------|
| **Crocidolite (Blue)** | Amphibole | **Highest** (thin, straight, durable) | Insulation, cement, spray coatings |
| **Amosite (Brown)** | Amphibole | **High** | Insulation board, ceiling tiles |
| **Chrysotile (White)** | Serpentine | Lower (curly, less durable) | Cement, brakes, textiles, roofing |
| **Tremolite/Actinolite/Anthophyllite** | Amphibole | High | Contaminants in chrysotile/vermiculite |

> **FCPS/MRCP tip**: **Amphiboles (crocidolite, amosite) far more fibrogenic and carcinogenic** than chrysotile. **All types cause disease**.

### Asbestos Bodies (Ferruginous Bodies)
- **Asbestos fibre** coated with **iron-protein (ferritin/haemosiderin)** coating in lung
- **Golden-brown, beaded, dumbbell-shaped** rods (10–100 µm)
- **Found in**: BAL fluid (cytospin), lung tissue (histology), sputum
- **Diagnostic criteria**: **>1 asbestos body per mL BAL** (or >1 per gram dry lung) + clinical context

## Pathophysiology
1. **Inhalation** → fibres deposit at **alveolar duct bifurcations** (lower lobes, bases)
2. **Alveolar macrophage phagocytosis** → **frustrated phagocytosis** (long fibres >15–20 µm)
3. **Macrophage activation** → **ROS, TNF-α, IL-1, TGF-β, PDGF** → inflammation
4. **Fibroblast activation** → **collagen deposition** → **interstitial fibrosis**
5. **Asbestos bodies** form (macrophage coats fibre with iron-protein)
6. **Fibrosis starts at respiratory bronchioles** → extends to interstitium → **subpleural fibrosis**
7. **Pleural involvement** → **parietal pleural plaques** (circumscribed, calcified), diffuse pleural thickening

## Clinical Features
### History
- **Occupational exposure** (shipbuilding, insulation, construction, demolition, brake repair, textile, cement, power plants, naval)
- **Latency 20–40 years** (peak incidence 60–80 years)
- **Smoking history** (synergy for lung cancer)

### Symptoms
- **Progressive dyspnoea** (exertional → rest)
- **Dry cough**
- **Chest tightness**
- **Fatigue**, weight loss (late)
- **Clubbing** (late, ~30%)

### Examination
- **Late inspiratory crackles** (basal, "Velcro" — fine, high-pitched)
- **Clubbing** (late)
- **Reduced chest expansion** (basal)
- **Signs of cor pulmonale** (late): JVP ↑, peripheral oedema, RV heave
- **Pleural plaques** (usually asymptomatic, found incidentally)

## Investigations
### 1. HRCT (Gold Standard Imaging)
| Finding | Description |
|---------|-------------|
| **Subpleural curvilinear lines** | **Parallel to pleural surface, 5–10 cm long** — **EARLY SIGN** |
| **Parenchymal bands** | Linear opacities extending from pleura into lung |
| **Basal predominant fibrosis** | Lower lobes > upper lobes |
| **Honeycombing** | **Late**, clustered cystic spaces, basal/subpleural |
| **Pleural plaques** | **Circumscribed, calcified, parietal pleura** (posterolateral, diaphragmatic) — **EXPOSURE MARKER** |
| **Diffuse pleural thickening** | Visceral + parietal, blunts costophrenic angles |
| **Rounded atelectasis** | Folded lung trapped in thickened pleura (comet tail sign) |
| **Mediastinal nodes** | Rarely enlarged |

> **FCPS/MRCP tip**: **Subpleural curvilinear lines + basal fibrosis + pleural plaques = classic asbestosis HRCT**. **Pleural plaques alone ≠ asbestosis**.

### 2. Chest X-Ray (ILO Classification)
| ILO Category | Profusion | Description |
|--------------|-----------|-------------|
| **0** | None | Normal |
| **1/0 – 1/2** | Very few/small | Early |
| **2/1 – 2/2** | Moderate | Definite |
| **3/2 – 3/3** | Profuse | Advanced |

| Opacity Shape | Size | Significance |
|---------------|------|--------------|
| **s** (small rounded) | <1.5 mm | Pinniform |
| **t** | 1.5–3 mm | |
| **u** | 3–10 mm | |
| **Irregular (linear)** | | **Asbestosis typical** |

> **FCPS/MRCP tip**: **ILO classification** standardises pneumoconiosis reporting. **Asbestosis = irregular linear opacities, basal, profusion ≥1/0**.

### 3. Pulmonary Function Tests
| Pattern | Typical Findings |
|---------|------------------|
| **Restrictive** | ↓ TLC, ↓ FVC, ↓ FEV1, **normal FEV1/FVC ratio** |
| **↓ DLCO** | **Early, disproportionate** (early sign) |
| **Airflow obstruction** | May coexist (smoking, small airways disease) |

### 4. Asbestos Bodies (Diagnostic Confirmation)
| Sample | Threshold | Method |
|--------|-----------|--------|
| **BAL (cytospin)** | **>1 body/mL** | Cytospin, light microscopy |
| **Lung tissue** | **>1 body/g dry weight** | Histology (Perls' stain for iron) |
| **Sputum** | Presence | Cytology |

> **FCPS/MRCP tip**: **Asbestos bodies = ferruginous bodies** (iron-coated fibres). **>1/mL BAL = diagnostic** in right clinical context.

### 4. Autoantibodies / Inflammatory Markers
- **ANA** (low titre, speckled) — non-specific, up to 30%
- **RF** (low titre) — non-specific
- **No specific autoantibody** for asbestosis

### 5. Lung Cancer / Mesothelioma Screening
- **Annual CXR** (or LDCT if high risk) for exposed workers
- **Sputum cytology** (if symptoms)
- **CT chest** (if nodule/suspicion)

## Classification / Diagnostic Criteria
### HELPP Criteria (Diagnostic Framework)
| Component | Requirement |
|-----------|-------------|
| **H**istory | Compatible occupational exposure (duration, intensity, fibre type) |
| **E**xposure | Documented cumulative exposure (fibre-years) |
| **L**atency | **≥20 years** (usually 20–40 years) |
| **P**ulmonary function | **Restrictive + ↓ DLCO** |
| **P**athology/Imaging | **HRCT: subpleural lines, basal fibrosis, pleural plaques** + **asbestos bodies** (if available) |

> **Diagnostic certainty**: **Definite** = all 5 criteria; **Probable** = 4/5; **Possible** = 3/5.

### ILO Radiographic Classification (Pneumoconiosis)
- **Profusion** (0–3 scale)
- **Shape/Size** of opacities (s, t, u for rounded; irregular for asbestosis)
- **Zone distribution** (basal for asbestosis)

## Clinical Features Summary
| Feature | Asbestosis | Pleural Plaques | Mesothelioma |
|---------|------------|-----------------|--------------|
| **Pathology** | Parenchymal fibrosis | Parietal pleural thickening | Malignant neoplasm |
| **Symptoms** | Dyspnoea, cough | Usually asymptomatic | Chest pain, dyspnoea, weight loss |
| **CXR/HRCT** | **Irregular opacities, basal fibrosis** | **Circumscribed calcified plaques** (parietal) | **Unilateral pleural thickening, nodular, effusion** |
| **Pulmonary function** | Restrictive, ↓ DLCO | Normal | Restrictive (effusion/tumour) |
| **Latency** | 20–40 years | 15–30 years | 20–50 years |
| **Prognosis** | Progressive fibrosis | Benign (exposure marker) | Poor (median 12 mo) |
| **Compensation** | IIDB D3 | IIDB D9 (if symptomatic) | IIDB D1 |

## Etiology / Risk Factors
### Occupations (High Risk)
- **Shipbuilding/repair** (insulation, lagging)
- **Insulation workers** (laggers, pipe fitters)
- **Construction/demolition** (asbestos removal)
- **Asbestos mining/milling**
- **Textile manufacturing** (asbestos cloth)
- **Cement products** (pipes, sheets)
- **Brake/clutch repair** (mechanics)
- **Power plants / refineries** (insulation)
- **Naval / shipyard** personnel

### Fibre Factors
- **Amphiboles** (crocidolite, amosite) >> **Chrysotile** (fibrogenic & carcinogenic)
- **Fibre dimensions**: Length >15–20 µm, diameter <0.5 µm → most pathogenic
- **Cumulative exposure** (fibre-years) = dose

### Co-factors
- **Smoking** → **synergistic for lung cancer** (×50–90 risk)
- **Age** (older = longer latency)
- **Genetic susceptibility** (GSTM1 null, NAT2 slow acetylator)

## Clinical Features Detail
### Early
- **Exertional dyspnoea** (insidious)
- **Dry cough**
- **Basal end-inspiratory crackles** ("Velcro")
- **Reduced exercise tolerance**

### Advanced
- **Dyspnoea at rest**
- **Clubbing** (~30–50%)
- **Cor pulmonale** (JVP ↑, peripheral oedema, RV heave)
- **Right heart failure** (late)

### Asbestos-Related Pleural Disease (Separate from Asbestosis)
| Entity | Features |
|--------|----------|
| **Pleural plaques** | **Circumscribed, calcified, parietal pleura** (posterolateral, diaphragmatic), **asymptomatic**, exposure marker |
| **Diffuse pleural thickening** | Visceral + parietal, blunts costophrenic angles, may cause restriction |
| **Benign asbestos pleural effusion (BAPE)** | Exudate, eosinophilic, self-limiting, diagnosis of exclusion |
| **Rounded atelectasis** | Folded lung trapped in thickened pleura, comet tail sign |

## Management
### 1. Exposure Cessation
- **Remove from exposure** (redeployment, retirement)
- **No further asbestos exposure**

### 2. Symptomatic / Supportive
- **Oxygen** (if hypoxaemic, LTOT if PaO2 <7.3 kPa)
- **Pulmonary rehabilitation**
- **Vaccinations** (flu, pneumococcal, COVID)
- **Nutritional support** (weight loss in advanced)

### 3. Complication Management
| Complication | Management |
|--------------|------------|
| **Progressive fibrosis** | LTOT, pulmonary rehab, consider antifibrotics (nintedanib/pirfenidone — extrapolated from IPF) |
| **Cor pulmonale** | Diuretics, oxygen, LTOT, avoid fluid overload |
| **Lung cancer** (smoking + asbestos) | **Annual LDCT screening** (high risk), multidisciplinary |
| **Mesothelioma** | Palliative (chemo: cisplatin+pemetrexed, immunotherapy, radiotherapy) |
| **Respiratory failure** | NIV (if hypercapnic), palliative care |

### 4. Smoking Cessation (CRITICAL)
- **Synergistic carcinogenicity**: Asbestos + smoking → **lung cancer risk ×50–90**
- **Smoking cessation reduces risk** (but remains elevated vs never-smoker)

## Drug Interactions / Contraindications / Cautions
### Antifibrotics (if used off-label for progressive fibrotic asbestosis)
- **Nintedanib**: Diarrhoea, LFT, bleeding risk (avoid anticoagulants)
- **Pirfenidone**: Photosensitivity, GI, LFT, rash
- **Monitor**: LFT monthly ×3, then q3mo

### Oxygen
- **Target SpO2 88–92%** if chronic hypercapnia risk (COPD overlap)

### Vaccinations
- **Flu, pneumococcal, COVID** — recommended

## Procedures
### BAL for Asbestos Bodies
- **Indication**: Diagnostic uncertainty (exposure history + fibrosis but no plaques)
- **Technique**: Standard bronchoscopy, 100–200 mL saline, cytospin
- **Interpretation**: **>1 asbestos body/mL** = diagnostic in context

### VATS Biopsy
- **Indication**: Diagnostic uncertainty (IPF vs asbestosis vs CTD-ILD)
- **Samples**: Multiple lobes, include pleura (look for plaques)

### Compensation Assessment (UK)
- **Industrial Injuries Disablement Benefit (IIDB)** — Prescribed Diseases:
  - **D3**: Asbestosis (pulmonary fibrosis)
  - **D4**: Mesothelioma
  - **D9**: Bilateral diffuse pleural thickening / pleural plaques (if symptomatic)
- **Civil litigation** (negligence, exposure history)
- **Diffuse Mesothelioma Payment Scheme (DMPS)** — if employer uninsured/traceable

## Complications
| Complication | Details |
|--------------|---------|
| **Progressive fibrosis** | Restrictive lung disease, respiratory failure |
| **Lung cancer** | **×50–90 risk with smoking** (adenocarcinoma, squamous); **synergy** |
| **Mesothelioma** | Pleural (90%), peritoneal; **latency 20–50 years**; poor prognosis |
| **Cor pulmonale** | Right heart failure from pulmonary hypertension |
| **Respiratory failure** | Hypoxaemic ± hypercapnic |
| **Rounded atelectasis** | Infolded lung in thickened pleura (comet tail sign) |

## Red Flags / Emergencies
- **New pulmonary nodule/mass** → CT, PET, biopsy (lung cancer/mesothelioma)
- **Massive haemoptysis** → BAE
- **Acute dyspnoea + pleural effusion** → mesothelioma, infection, heart failure
- **Sudden deterioration** → pneumothorax, PE, acute exacerbation

## Special Situations
### Smoking + Asbestos = Lung Cancer Synergy
| Exposure | Lung Cancer Relative Risk |
|----------|---------------------------|
| Neither | 1 (reference) |
| Asbestos only | **×5** |
| Smoking only | **×10** |
| **Both** | **×50–90** |

> **Mechanism**: Asbestos fibres → chronic inflammation, ROS, DNA damage; smoking → carcinogens, impaired clearance → **synergistic carcinogenesis**.

### Pleural Plaques vs Asbestosis (Key Distinction)
| Feature | **Pleural Plaques** | **Asbestosis** |
|---------|---------------------|----------------|
| **Location** | Parietal pleura (posterolateral, diaphragmatic) | Lung parenchyma (interstitium) |
| **Pathology** | Collagenous thickening, calcification | Interstitial fibrosis |
| **Symptoms** | Usually **asymptomatic** | **Dyspnoea, cough, crackles** |
| **PFTs** | **Normal** (unless massive) | **Restrictive, ↓ DLCO** |
| **HRCT** | Circumscribed calcified plaques (parietal) | Subpleural lines, basal fibrosis, honeycombing |
| **Prognosis** | **Benign** (exposure marker) | **Progressive fibrosis** |
| **Compensation** | IIDB D9 (if symptomatic) | IIDB D3 |

## Prognosis
- **Variable**: Slowly progressive over decades
- **Median survival** from diagnosis: **10–15 years** (highly variable)
- **Mortality**: Respiratory failure, lung cancer, mesothelioma, cor pulmonale
- **Factors worsening prognosis**: Smoking, high cumulative exposure, early onset, rapid PFT decline, honeycombing

## Common Confusions / Exam Traps
- **Pleural plaques = asbestosis** — WRONG (plaques = benign exposure marker; asbestosis = parenchymal fibrosis)
- **Chrysotile (white) = safe** — WRONG (all types cause disease; amphiboles more potent)
- **Asbestosis = immediate** — WRONG (latency 20–40 years)
- **Asbestos bodies = diagnostic alone** — need clinical/radiological context
- **Pleural plaques cause dyspnoea** — usually asymptomatic; dyspnoea suggests asbestosis or other cause
- **All asbestos = mesothelioma** — mesothelioma is rare; asbestosis more common
- **Compensation for plaques** — only if symptomatic (IIDB D9)
- **Asbestosis = reversible** — NO (irreversible fibrosis)

## Summary
**Asbestosis** = **diffuse pulmonary fibrosis** from **inhaled asbestos fibres** (dose-dependent, latency 20–40 years). **Key HRCT**: **subpleural curvilinear lines + basal fibrosis + pleural plaques (calcified, parietal)**. **Diagnostic confirmation**: **asbestos bodies (ferruginous bodies) >1/mL BAL** or lung tissue. **Fibre potency**: **crocidolite > amosite > chrysotile**. **Pleural plaques** = **benign exposure marker** (parietal, calcified, asymptomatic) — **NOT asbestosis**. **Smoking + asbestos** = **synergistic lung cancer risk ×50–90**. **Mesothelioma** = malignant, latency 20–50 years, poor prognosis. **Complications**: progressive fibrosis, lung cancer (smoking synergy), mesothelioma, cor pulmonale. **Compensation (UK)**: IIDB Prescribed Disease **D3 (asbestosis)**, **D4 (mesothelioma)**, **D9 (pleural plaques/thickening)**. **Management**: exposure cessation, **smoking cessation critical**, LTOT, pulmonary rehab, cancer screening (annual LDCT), compensation claims.

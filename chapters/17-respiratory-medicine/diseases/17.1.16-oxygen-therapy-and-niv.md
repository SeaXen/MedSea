# Oxygen Therapy and Non-Invasive Ventilation (NIV)

Related: [[Respiratory Failure]], [[ABG Interpretation]], [[Acute Medicine and Critical Illness/Oxygen Therapy and NIV|Oxygen Therapy and NIV]], [[Respiratory Failure and Ventilatory Support/Non-Invasive Ventilation (NIV)|NIV]]

> [!important]
> **Oxygen is a drug** — prescribe with target saturation, flow rate, and delivery device. **NIV** = first-line for hypercapnic respiratory failure (AECOPD, cardiogenic pulmonary oedema, immunocompromised). **Key FCPS/MRCP**: target SpO₂ (88-92% at risk CO₂ retention, 94-98% otherwise), NIV indications/contraindications, settings (EPAP/IPAP), escalation to intubation.

## 1. Learning Objectives
- Prescribe oxygen with target saturation, device, and flow rate
- Identify NIV indications, contraindications, and initial settings
- Apply NIV monitoring and escalation criteria
- Differentiate oxygen delivery devices and their FiO₂ ranges
- Recognise NIV failure and escalate to intubation

## 2. Oxygen Therapy

### Target Saturations (BTS Guidelines)
| Patient Group | Target SpO₂ | Rationale |
|---------------|-------------|-----------|
| **At risk of hypercapnia** (COPD, obesity, neuromuscular, CF, severe kyphoscoliosis) | **88–92%** | Avoid hyperoxia-induced hypoventilation / V/Q mismatch |
| **All other patients** | **94–98%** | Standard oxygenation target |
| **Critical illness / cardiac arrest / carbon monoxide / anaphylaxis** | **94–98%** (or 100% initially) | Maximise oxygen delivery |

> **Never** aim for 100% SpO₂ routinely — risk of absorption atelectasis, hyperoxia toxicity.

### Oxygen Delivery Devices

| Device | Flow Rate | Approx. FiO₂ | Indications |
|--------|-----------|--------------|-------------|
| **Nasal cannula** | 1–6 L/min | 0.24–0.44 | Low-flow, stable patients |
| **Simple face mask** | 5–10 L/min | 0.40–0.60 | Short-term, higher flow needed |
| **Non-rebreather mask (NRB)** | 10–15 L/min | **0.85–0.95** | High O₂ requirement, pre-intubation |
| **High-flow nasal oxygen (HFNO)** | Up to 60 L/min | 0.21–1.00 (titratable) | Acute hypoxaemic failure, post-extubation, pre-oxygenation |
| **Venturi mask** | Fixed (device-specific) | **0.24, 0.28, 0.35, 0.40, 0.50, 0.60** | **Precise FiO₂** (CO₂ retention risk, target 88-92%) |
| **Tracheostomy mask / T-piece** | Variable | Variable | Tracheostomy patients |

> **Venturi mask** = **only device delivering precise FiO₂** — essential for CO₂ retention risk (target 88-92%).

### Oxygen Prescription (The "Drug" Prescription)
| Component | Example |
|-----------|---------|
| Target SpO₂ | 88–92% (COPD) / 94–98% (other) |
| Device | Venturi mask 28% / Nasal cannula 2 L/min / NRB 15 L/min |
| Flow rate | As per device |
| Weaning plan | Reduce FiO₂/flow as SpO₂ permits |
| Monitoring | Continuous SpO₂; ABG if deteriorating / at risk CO₂ retention |

> **O₂ on drug chart**: Target SpO₂, device, flow, weaning plan.

### Oxygen Toxicity & Safety
| Risk | Prevention |
|------|------------|
| **Absorption atelectasis** (100% O₂) | Use lowest FiO₂ to achieve target |
| **Hyperoxia-induced vasoconstriction** | Target SpO₂ 88-92% in CO₂ retainers |
| **CO₂ retention (Haldane effect)** | Target SpO₂ 88-92% in COPD; avoid overshoot |
| **Oxygen toxicity (lungs)** | FiO₂ >0.6 prolonged → ARDS risk; minimise duration |
| **Fire hazard** | No smoking/open flames near O₂ |

### Humidification
- **Low flow (<4 L/min)**: not required (nasal cannula)
- **High flow (>4 L/min) or prolonged**: heated humidifier required
- **HFNO**: built-in active humidification (37°C, 100% RH)

## 3. Non-Invasive Ventilation (NIV)

### Definition
**NIV** = ventilatory support via **mask/interface** without endotracheal intubation. **Modes**: CPAP (continuous positive airway pressure) or BiPAP (bilevel positive airway pressure = IPAP + EPAP).

### Indications (Strong Evidence)
| Condition | NIV Mode | Key Target |
|-----------|----------|------------|
| **AECOPD** (pH 7.25–7.35, PaCO₂ >6.5 kPa) | **BiPAP** | pH >7.30, RR ↓, PaCO₂ ↓ |
| **Cardiogenic pulmonary oedema** | **CPAP** (or BiPAP) | SpO₂ >94%, ↓ work of breathing |
| **Immunocompromised** (avoid intubation) | BiPAP | Avoid intubation complications |
| **Post-extubation prevention** (high risk) | CPAP/BiPAP | Prevent re-intubation |
| **Obesity hypoventilation syndrome** | BiPAP (nocturnal) | PaCO₂ normalisation |
| **Neuromuscular / chest wall** | BiPAP (nocturnal) | Symptom control, survival |
| **Post-operative** (abdominal/thoracic) | CPAP/BiPAP | Prevent atelectasis, re-intubation |

### Contraindications
| Absolute | Relative |
|----------|----------|
| **Cardiac / respiratory arrest** | Confused/agitated (unable to cooperate) |
| **Facial trauma / burns** | Copious secretions (aspiration risk) |
| **Upper airway obstruction** | Recent upper GI surgery (aspiration) |
| **Unable to protect airway** (GCS <8) | Severe claustrophobia |
| **Life-threatening hypoxaemia** (requiring intubation) | Recent facial/ocular surgery |
| **Facial deformity** (prevents seal) | Recent pneumothorax (relative) |

### NIV Settings (BiPAP) — Typical Starting Values
| Parameter | Starting Value | Titration |
|-----------|----------------|-----------|
| **EPAP** | **4–5 cmH₂O** | ↑ by 2 cmH₂O if hypoxaemia / OSA |
| **IPAP** | **10–12 cmH₂O** (ΔP = 6–8 cmH₂O) | ↑ by 2–4 cmH₂O if hypercapnia persists |
| **Backup Rate** | 12–15/min | As needed |
| **Inspiratory Time** | 0.8–1.2 s | Adjust for comfort |
| **Rise Time** | 200–300 ms | Faster if high drive |
| **FiO₂** | Titrate to target SpO₂ | 88-92% (CO₂ retainers) / 94-98% |

> **Initial ΔP (IPAP-EPAP) = 6–8 cmH₂O** for AECOPD; titrate to pH/PaCO₂.

### NIV Monitoring (First 1-2 Hours Critical)
| Parameter | Target | Frequency |
|-----------|--------|-----------|
| **pH** | >7.30 (AECOPD) | 30-60 min |
| **PaCO₂** | ↓ >20% from baseline / <6.5 kPa | 30-60 min |
| **Respiratory Rate** | ↓ by ≥25% | 15-30 min |
| **SpO₂** | Target per group | Continuous |
| **Heart Rate / BP** | Stable | 15-30 min |
| **Comfort / Leak** | Minimal leak (<30 L/min) | Continuous |

### NIV Failure Criteria (Escalate to Intubation)
| Criterion | Threshold |
|-----------|-----------|
| **pH** | <7.25 after 1-2h NIV |
| **PaCO₂** | Not falling / rising after 1-2h |
| **Respiratory Rate** | Not decreasing / >35/min |
| **GCS** | <12 / decreasing |
| **Haemodynamic instability** | SBP <90, arrhythmia |
| **Inability to tolerate mask** | Claustrophobia, secretions |
| **Persistent hypoxaemia** | SpO₂ <88% despite FiO₂ 1.0 |

> **Do NOT delay intubation** if failing — NIV delay increases mortality.

### CPAP vs BiPAP
| Feature | CPAP | BiPAP |
|---------|------|-------|
| **Pressure** | Single continuous pressure | IPAP + EPAP (two levels) |
| **Primary Effect** | Splint airways open, ↑ FRC, recruit alveoli | Assist inspiration (↓ work of breathing) + splint airways |
| **Indication** | Cardiogenic pulmonary oedema, OSA, post-extubation | AECOPD, hypercapnic failure, neuromuscular, obesity hypoventilation |
| **Work of Breathing** | No inspiratory support | Reduces inspiratory work |

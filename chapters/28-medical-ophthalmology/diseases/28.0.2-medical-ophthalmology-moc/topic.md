## Key Algorithms & Scoring Systems (Memorise)

| System | Use Case | Key Thresholds |
|--------|----------|----------------|
| **Snellen / LogMAR** | Visual acuity | 6/6 = 1.0; 6/60 = legal blindness (most countries) |
| **IOP (mmHg)** | Glaucoma | Normal 10–21, ocular HTN 22–29, glaucoma ≥21 + damage |
| **Cup:Disc ratio** | Glaucoma suspect | >0.5 suspect, >0.7 advanced |
| **ISNT rule** | Neuroretinal rim | Inferior ≥ Superior ≥ Nasal ≥ Temporal |
| **Humphrey visual field** | Glaucoma | MD, PSD; nasal step, arcuate, paracentral, altitudinal defects |
| **DR grading** | Diabetic retinopathy | None, mild/moderate/severe NPDR, PDR |
| **AMD classification** | Macular degeneration | Dry (drusen, GA) vs Wet (CNV) |
| **Westergren ESR / CRP** | Giant cell arteritis | ESR >50 (or >age/2 men, >age+5/2 women) |
| **Hutchinson sign** | HZO | Nasociliary nerve involvement → eye involvement |
| **RAPD / Marcus-Gunn** | Optic nerve dysfunction | Grade 1–4, swinging flashlight test |
| **König & Schubert** | Uveitis | Anterior, intermediate, posterior, panuveitis |
| **HLA-B27** | Acute anterior uveitis | 50% of cases |
| **BCVA** | Cataract surgery indication | <6/9 (driving) or visual impairment |
| **AAU (Acute anterior uveitis)** | Recurrent | SUN criteria for activity |

## High-Yield Viva Topics (Definitions, Doses, Durations)

| Topic | Expected Answer Points |
|-------|------------------------|
| **CRAO management** | Ocular massage, anterior chamber paracentesis, acetazolamide, IV acetazolamide; GCA workup if >50y |
| **Acute angle closure** | Topical pilocarpine 1–2%, IV acetazolamide 500mg, mannitol, laser peripheral iridotomy |
| **GCA & visual loss** | ESR/CRP, IV methylpred 1g × 3d then oral pred 1mg/kg, temporal artery biopsy within 2w |
| **Diabetic retinopathy screening** | Annual from diagnosis (T2DM); 5y after diagnosis (T1DM); Type 1 prepubertal not screened |
| **Anti-VEGF agents** | Ranibizumab, aflibercept, bevacizumab; 0.5mg intravitreal monthly initially |
| **Open angle glaucoma drops** | Prostaglandin analogues (latanoprost 1st line) → beta-blocker (timolol) → alpha-2 agonist (brimonidine) → carbonic anhydrase inhibitor (dorzolamide) |
| **Herpes zoster ophthalmicus** | Aciclovir 800mg 5×/d × 7–10d, oral valaciclovir alternative |
| **Optic neuritis & MS** | IV methylpred 1g/d × 3d hastens recovery but no effect on long-term outcome; MRI brain+orbit with contrast |
| **Retinal detachment** | Vitreous detachment → tear → detachment; urgent vitreoretinal referral |
| **Cataract surgery** | Phacoemulsification + IOL implant; most common surgery worldwide |
| **Ocular MG diagnosis** | Tensilon (edrophonium) test, anti-AChR Ab, ice pack test, Ocular MG composite score |
| **Differentiating red eye** | Pain + ↓VA = serious; pupil shape, discharge, IOP, corneal involvement |

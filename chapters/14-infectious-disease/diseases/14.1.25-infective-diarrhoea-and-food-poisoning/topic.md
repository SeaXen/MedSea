> [!important]
> **Diarrhoea** = ≥3 loose stools/24h. **Acute** <14d, **Persistent** 14–30d, **Chronic** >30d. **Infective** = majority acute. **Assess hydration, red flags, need for stool culture/PCR, empirical therapy.**

## 1. Learning Objectives
- Classify diarrhoea by duration and mechanism
- Identify red flags for invasive/bloody diarrhoea and systemic illness
- Select appropriate investigations (stool culture, PCR, toxins, endoscopy)
- Apply empirical and pathogen-directed therapy
- Manage dehydration, electrolytes, and complications

## 2. Classification by Duration
| Category | Duration | Common Causes |
|----------|----------|---------------|
| **Acute** | <14 days | Viral (norovirus, rotavirus), bacterial (Campylobacter, Salmonella, Shigella, ETEC), protozoal (Giardia, Cryptosporidium) |
| **Persistent** | 14–30 days | Giardia, Cryptosporidium, Cyclospora, C. difficile, post-infectious IBS |
| **Chronic** | >30 days | IBD, IBS, malabsorption, chronic infections (HIV, TB), medications, neoplasia |

## 3. Classification by Mechanism
| Type | Mechanism | Examples |
|------|-----------|----------|
| **Secretory** | Active ion secretion (CFTR) → water follows | Cholera (CTX), ETEC (LT/ST), VIPoma, laxatives |
| **Osmotic** | Non-absorbable solutes retain water | Lactose intolerance, magnesium, sorbitol, malabsorption |
| **Inflammatory/Invasive** | Mucosal damage → blood, pus, mucus | Shigella, Salmonella, Campylobacter, EIEC, C. difficile, IBD |
| **Fatty (Steatorrhoea)** | Fat malabsorption | Pancreatic insufficiency, coeliac, bile acid diarrhoea |
| **Motility-related** | Altered transit | Hyperthyroidism, diabetic autonomic neuropathy, carcinoid |

## 4. Clinical Assessment

### History
| Element | Key Questions |
|---------|---------------|
| **Duration** | Acute (<14d) vs persistent vs chronic |
| **Frequency/Volume** | Large volume (secretory/osmotic) vs small frequent (inflammatory) |
| **Blood/Mucus** | **Yes = inflammatory/invasive** (dysentery); No = secretory/osmotic |
| **Severity** | ≥6/day, nocturnal, weight loss |
| **Associated** | Fever, abdominal pain, vomiting, tenesmus |
| **Epidemiology** | Travel, antibiotics, hospitalisation, food source, contacts, immunocompromise |

### Red Flags (Admit / Urgent Investigation)
| Red Flag | Significance |
|----------|--------------|
| **Bloody diarrhoea** | Invasive (Shigella, Campylobacter, EHEC, Salmonella, EIEC, C. difficile, IBD) |
| **High fever (>38.5°C)** | Systemic invasion (bacteraemia, sepsis) |
| **Signs of dehydration** | Hypovolaemia, shock risk |
| **Severe abdominal pain** | Surgical abdomen, toxic megacolon, ischaemia |
| **Immunocompromise** | HIV, chemo, steroids — broader differential, severe disease |
| **Elderly / Comorbidities** | Higher mortality, atypical presentation |
| **Recent antibiotics / hospitalisation** | C. difficile risk |
| **Recent travel** | ETEC, Giardia, Cyclospora, Cholera, parasitic |
| **Neurological symptoms** | Botulism, listeria, ciguatera, shellfish poisoning |

## 5. Investigations

### When to Investigate (NICE / IDSA)
| Indication | Tests |
|------------|-------|
| **Bloody/mucoid diarrhoea** | Stool culture + PCR, C. difficile toxin |
| **Fever ≥38.5°C** | Blood cultures, stool culture/PCR |
| **Severe / dehydrated** | U&E, CRP, lactate, VBG/ABG, FBC |
| **Immunocompromised** | Extended PCR panel, O&P, viral PCR, fungal |
| **Persistent >7d** | Stool culture/PCR, O&P, Giardia/Crypto antigen |
| **Recent antibiotics / hospital** | C. difficile toxin/PCR |
| **Recent travel** | O&P, Cyclospora, microsporidia, viral PCR |
| **Foodborne outbreak** | Culture, WGS, public health notification |

### Stool Culture vs PCR
| Test | Advantages | Limitations |
|------|------------|-------------|
| **Culture + AST** | Essential for resistance, susceptibility, outbreak | Slow (2–5d); lower sensitivity |
| **Multiplex PCR panel** | Rapid (1–2h), high sensitivity, multiple targets | No AST; detects DNA not viability; cost |
| **Ova & Parasites (O&P)** | Required for Giardia, Crypto, Cyclospora, helminths | Low sensitivity (3 samples); operator-dependent |
| **C. difficile toxin/PCR** | Gold standard for CDI | PCR more sensitive; toxin more specific |

### Pathogen-Specific Tests
| Pathogen | Best Test |
|----------|-----------|
| **C. difficile** | PCR (toxin B gene) +/− toxin EIA |
| **Giardia / Crypto** | Stool antigen (EIA) or PCR |
| **Cyclospora / Microsporidia** | Modified acid-fast stain / PCR |
| **Vibrio cholerae** | Culture (TCBS) + PCR |
| **EHEC (O157:H7)** | Culture (sorbitol-MacConkey) + PCR (stx1/stx2) |
| **VDRL / RPR** | Not for diarrhoea! (syphilis) |

## 6. Empirical Management Principles

### Rehydration (Cornerstone)
| Severity | ORS / IV Fluid |
|----------|----------------|
| **No dehydration** | ORS ad lib (50–100mL/kg/24h); continue feeding |
| **Some dehydration** | ORS 75mL/kg over 4h; reassess |
| **Severe dehydration** | **IV Ringer's Lactate / Normal Saline** 100mL/kg over 3h (adults: 1L bolus, reassess) |

> **ORS** = WHO reduced osmolarity (245 mOsm/L); **Zinc 20mg/day × 10–14d** in children <5y

### Antimotility Agents
| Agent | Use | Contraindications |
|-------|-----|-------------------|
| **Loperamide** | Acute watery (no blood/fever); adjunct to antibiotics | **Bloody diarrhoea, fever, suspected C. difficile, children <12y** |
| **Racecadotril** | Secretory diarrhoea (adjunct) | Similar to loperamide |

### Antibiotic Therapy (Indicated for Specific Pathogens / Severity)

| Pathogen / Scenario | 1st-line | Duration |
|---------------------|----------|----------|
| **Shigella** | Ciprofloxacin 500mg BD / Azithromycin 500mg OD | 3–5d |
| **Campylobacter** | Azithromycin 500mg OD | 3d |
| **Salmonella (non-typhoidal, invasive)** | Ciprofloxacin 500mg BD / Ceftriaxone 2g IV | 7–10d (IV→oral) |
| **Salmonella Typhi/Paratyphi** | Ceftriaxone 2g IV / Azithromycin 1g OD | 7–14d |
| **ETEC (traveller's diarrhoea)** | Azithromycin 500mg OD × 1–3d / Rifaximin 200mg TDS × 3d | 1–3d |
| **Vibrio cholerae** | Doxycycline 100mg BD × 3d / Azithromycin 1g single | 3d / 1d |
| **C. difficile (mild-mod)** | Vancomycin 125mg QDS PO / Fidaxomicin 200mg BD | 10d |
| **C. difficile (severe)** | Vancomycin 500mg QDS PO/IV + Metronidazole 500mg IV QDS | 10–14d |
| **Giardia** | Metronidazole 400mg TDS / Tinidazole 2g single | 5–7d / 1d |
| **Cryptosporidium** | Nitazoxanide 500mg BD (if immunocompetent) | 3d |
| **EHEC O157:H7** | **No antibiotics** (↑ HUS risk) | Supportive only |
| **Food poisoning (S. aureus, B. cereus)** | Supportive (toxin-mediated) | — |

> **Avoid antibiotics** in: EHEC (↑ HUS), mild viral, self-limiting

## 7. Food Poisoning (Toxin-Mediated)
| Syndrome | Organism / Toxin | Incubation | Key Features |
|----------|------------------|------------|--------------|
| **Staphylococcal** | S. aureus enterotoxin | 1–6h | **Vomiting prominent**, abdominal cramps, diarrhoea; self-limiting 24–48h |
| **Bacillus cereus (emetic)** | Cereulide | 1–6h | Vomiting > diarrhoea; fried rice |
| **Bacillus cereus (diarrhoeal)** | Enterotoxin | 8–16h | Diarrhoea > vomiting; meat, vegetables |
| **Clostridium perfringens** | Enterotoxin | 8–24h | Diarrhoea, cramps; no fever/vomiting; meat/gravy |
| **Botulism** | C. botulinum neurotoxin | 12–72h | **Descending paralysis**, diplopia, dysphagia, respiratory failure; **ANTITOXIN urgent** |
| **Ciguatera** | Ciguatoxin | 2–24h | GI + **neurological** (paradoxical temp reversal, paraesthesia) |
| **Scombroid** | Histamine | 10–60min | Flushing, headache, urticaria, palpitations; **antihistamines** |

## 8. Complications
| Complication | High-Risk Pathogens |
|-------------|---------------------|
| **HUS** (Haemolytic Uraemic Syndrome) | **EHEC O157:H7** (Shiga toxin); Shigella dysenteriae |
| **TTP** | Similar to HUS; ADAMTS13 deficiency |
| **Reactive Arthritis** | Salmonella, Shigella, Campylobacter, Yersinia, Chlamydia (HLA-B27) |
| **Guillain-Barré Syndrome** | Campylobacter jejuni (molecular mimicry) |
| **Septicaemia** | Salmonella, Campylobacter, invasive E. coli |
| **Toxic Megacolon** | C. difficile, Shigella, IBD |
| **Post-infectious IBS** | Campylobacter, Salmonella, Giardia |

## 9. Traveller's Diarrhoea
| Aspect | Recommendation |
|--------|----------------|
| **Definition** | ≥3 loose stools/24h + cramps/nausea/fever during/after travel |
| **Aetiology** | ETEC (~50%), Campylobacter, Salmonella, Shigella, norovirus, Giardia |
| **Prophylaxis** | **Not routine**; Rifaximin 200mg BD (high-risk short-term) |
| **Self-treatment** | Azithromycin 500mg OD × 1–3d + Loperamide + ORS |
| **Standby antibiotics** | Azithromycin 500mg OD × 3d (SE Asia); Rifaximin 200mg TDS (low-risk) |

## 10. FCPS/MRCP High-Yield Points
- **Bloody diarrhoea = invasive** → culture/PCR + empirical antibiotics
- **EHEC O157:H7** = NO antibiotics (↑ HUS risk)
- **C. difficile** = recent antibiotics/hospital → PCR + vancomycin/fidaxomicin
- **Traveller's diarrhoea** = ETEC most common; azithromycin standby
- **Cholera** = rice-water stool, massive secretory → doxycycline/azithro + massive ORS
- **Botulism** = descending paralysis → antitoxin + ventilatory support
- **Reactive arthritis** = post-Salmonella/Shigella/Campylobacter (HLA-B27)
- **HUS** = EHEC O157:H7 (no antibiotics!) + Shigella
- **Food poisoning** = short incubation (S. aureus 1–6h) = vomiting predominant

## 12. Common Confusions / Exam Traps
- **Antibiotics in EHEC** = ↑ HUS (NEVER give)
- **Antibiotics in mild viral** = unnecessary, ↑ resistance
- **Loperamide in bloody diarrhoea** = contraindicated
- **Antibiotics for all traveller's diarrhoea** = only moderate-severe
- **C. difficile** = stop causative antibiotic first; metronidazole no longer 1st-line
- **Cholera** = rice-water stool, massive fluid loss → doxycycline + massive ORS
- **Botulism** = antitoxin + ventilatory support (not antibiotics)
- **HUS triad** = microangiopathic haemolytic anaemia + thrombocytopenia + AKI

## 13. Mnemonics
- **Bloody diarrhoea pathogens**: **S**higella, **S**almonella, **C**ampylobacter, **E**HEC, **C**. difficile, **E**IEC = **SSCEE**
- **No antibiotics**: **E**HEC, **V**iral, **M**ild self-limiting = **EVM**
- **HUS triad**: **M**icroangiopathic **H**aemolytic **A**naemia + **T**hrombocytopenia + **A**KI = **MHA-T**
- **Traveller's diarrhoea**: **E**TEC > **C**ampylobacter > **S**almonella > **S**higella = **ECSS**
- **Food poisoning incubation**: **S**. aureus/**B**. cereus (emetic) = **1–6h** = **SB-6**

## 14. One-Page Revision Summary
**Diarrhoea = Acute (<14d) / Persistent (14–30d) / Chronic (>30d)**
- **Bloody = invasive** → culture/PCR + antibiotics
- **Watery = secretory/osmotic** → rehydration ± antiemetics
- **EHEC O157:H7** = NO antibiotics (HUS risk)
- **C. difficile** = recent antibiotics → vancomycin/fidaxomicin
- **Traveller's** = ETEC most common; azithromycin standby
- **Cholera** = rice-water, massive loss → doxy/azithro + massive ORS
- **Food poisoning** = short incubation = vomiting (S. aureus/B. cereus)
- **Botulism** = descending paralysis → antitoxin + ventilator
- **HUS** = EHEC (NO antibiotics!)

## 15. Must Know / Should Know / Nice to Know

- Bloody diarrhoea = invasive → culture + antibiotics

- Traveller's diarrhoea = azithromycin standby

### Should Know

- Parasitic (Giardia, Crypto, Cyclospora)
- Reactive arthritis (post-infectious)

- Probiotic evidence

### Nice to Know

- Microbiome therapeutics

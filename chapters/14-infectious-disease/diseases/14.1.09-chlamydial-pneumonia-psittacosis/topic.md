# Chlamydial Pneumonia & Psittacosis

> [!info] **Davidson Ch 11 Alignment**: Infectious Disease → Specific Organism Groups → Chlamydiae
> **FCPS/MRCP Focus**: C. pneumoniae (Atypical CAP), C. psittaci (Psittacosis — Bird Exposure), C. trachomatis (LGV/STI — Ch 15), Serology/PCR, Doxycycline/Azithromycin

---

## 🎯 Learning Objectives

- [ ] Distinguish **Chlamydia Species**: *C. pneumoniae* (Respiratory), *C. psittaci* (Psittacosis — Birds), *C. trachomatis* (LGV/STI — Ch 15)
- [ ] Recognise **Clinical Syndromes**: *C. pneumoniae* = Atypical CAP (Young Adults, Persistent Cough), *C. psittaci* = Psittacosis (Bird Exposure, Severe Pneumonia, Horder Nodes)
- [ ] Diagnose: **Serology** (MIF — Gold Standard), **PCR** (Respiratory Specimens), **Culture** (Specialised, Slow)
- [ ] Manage: **Doxycycline 100mg BD** / **Azithromycin 500mg Day 1 then 250mg OD** / **Levofloxacin/Moxifloxacin** (Alternatives)
- [ ] Recognise **Complications**: Endocarditis, Encephalitis, Hepatitis, Reactive Arthritis

---

## 📖 Chlamydia Species Overview

| Species | Primary Disease | Transmission | Key Features |
|---------|----------------|--------------|--------------|
| **C. pneumoniae** (TWAR) | **Atypical CAP**, Bronchitis, Pharyngitis, Sinusitis | Respiratory Droplets (Person-to-Person) | **Young Adults**, **Persistent Cough** (Weeks), **Wheeze**, **Hoarseness**, **Mild** |
| **C. psittaci** | **Psittacosis (Ornithosis)** — Zoonotic Pneumonia | **Bird Droppings/Secretions** (Aerosol), Parrot, Pigeon, Turkey, Duck | **Bird Exposure**, **Severe Pneumonia**, **Horder Nodes** (Splenic), **Hepatitis**, **Endocarditis**, **Meningitis** |
| **C. trachomatis** (L1-L3) | **LGV** (L1-L3) — Genital Ulcer, Buboes, Proctitis; **Ocular** (Trachoma) | Sexual Contact (L1-L3) / Flies/Fomites (Trachoma) | **LGV: Painful Inguinal Buboes**, **Proctocolitis**, **Genital Ulcer**; **Trachoma: Blindness** |

> [!tip] **C. pneumoniae = Atypical CAP in Young Adults** (Persistent Dry Cough, Hoarseness, Low-Grade Fever). **C. psittaci = Bird Exposure + Severe Pneumonia + Horder Nodes (Splenic Microabscesses)**.

---

## 📖 C. pneumoniae — "Walking Pneumonia" Agent

### Epidemiology

| Feature | Details |
|---------|---------|
| **Age** | **Children (5-15y) & Young Adults**; **Reinfection Common** |
| **Transmission** | **Respiratory Droplets** (Person-to-Person); **Incubation 3-4 Weeks** |
| **Seasonality** | **Year-Round** (Peaks Winter/Spring) |
| **Seroprevalence** | **50-70% Adults** (Past Infection Common) |

### Clinical Features

| Feature | Typical Presentation |
|---------|----------------------|
| **Onset** | **Insidious** (1-3 Weeks Prodrome: Sore Throat, Hoarseness, Dry Cough) |
| **Cough** | **Persistent, Dry → Productive**, **Weeks** (Hallmark: **Prolonged Cough >3 Weeks**) |
| **Fever** | Low-Grade / Absent (Often Afebrile) |
| **Extra-Pulmonary** | **Pharyngitis**, **Laryngitis**, **Sinusitis**, **Otitis Media**, **Reactive Arthritis**, **Guillain-Barré** (Rare) |
| **CXR** | **Patchy/Interstitial Infiltrates** (Often Lower Lobe), **Minimal** for Symptom Severity |

> [!tip] **C. pneumoniae = "Walking Pneumonia"**: Young Adult, **Persistent Cough >3 Weeks**, **Hoarseness/Laryngitis**, **Mild Constitutional**, **Patchy CXR** — **Doxycycline/Azithromycin**.

---

## 📖 Psittacosis (Ornithosis) — C. psittaci

### Epidemiology

| Feature | Details |
|---------|---------|
| **Reservoir** | **Birds**: **Psittacines** (Parrots, Macaws, Cockatiels), **Pigeons**, **Poultry** (Turkeys, Ducks), **Wild Birds** |
| **Transmission** | **Inhalation** of **Dried Droppings/Secretions/Feathers** (Aerosol); **Bird Handlers, Pet Owners, Poultry Workers, Vets** |
| **Incubation** | **5-19 Days** (Median 10 Days) |
| **Outbreaks** | **Pet Shops, Aviaries, Poultry Farms, Abattoirs**, **Bird Fairs** |

### Clinical Features

| System | Features |
|---------|----------|
| **Constitutional** | **High Fever**, **Severe Headache**, **Myalgia**, **Photophobia**, **Prostration** |
| **Respiratory** | **Dry Cough → Productive**, **Dyspnoea**, **Pleuritic Pain**, **Rales/Consolidation** |
| **Hepatic** | **Hepatomegaly**, **Mild Transaminitis**, **Jaundice (Rare)** |
| **Splenic** | **Horder Nodes** (Splenic Microabscesses) — **Pathognomonic** (20-30%) |
| **Neurological** | **Meningitis, Encephalitis, Cranial Nerve Palsies** (Rare) |
| **Cardiac** | **Myocarditis, Pericarditis, Endocarditis** (Rare) |
| **Dermatological** | **Maculopapular Rash** (10-20%), **Erythema Nodosum**, **Erythema Multiforme** |

> [!tip] **Psittacosis = Bird Exposure + Severe Pneumonia + Horder Nodes (Splenic Microabscesses) + Relative Bradycardia**. **Horder Nodes = Pathognomonic** (Splenic Microabscesses on Imaging).

## 🔬 Diagnostic Workup

```mermaid
flowchart TD
    A[Atypical Pneumonia / Bird Exposure + Fever] --> B[**Clinical Suspicion**]
    B --> C{**Suspect C. pneumoniae / C. psittaci**}
    C --> D[**Serology: MIF (Gold Standard)**]
    D --> E1[**MIF IgM/IgG** — *4-fold Rise Paired Sera (2-4w)*]
    D --> E2[**CF Test** — *Historical, Less Used*]
    C --> F[**PCR (Sputum/BAL/Throat Swab)**]
    F --> G[**Rapid, High Sensitivity**]
    C --> H[**Culture** (Specialised, Egg Yolk Sac / Cell Culture, BSL-3)]
    H --> I[**Slow (Weeks), Low Sensitivity**]
    C --> J[**Antigen Detection** (DFA/IHC — Sputum/Tissue)]
```

### Diagnostic Tests Comparison

| Test | C. pneumoniae | C. psittaci | Timing | Utility |
|------|---------------|-------------|--------|---------|
| **MIF (Microimmunofluorescence)** | **Gold Standard** | **Gold Standard** | **Acute + Convalescent (2-4w Apart)** | **Confirmatory** (4-fold Rise) |
| **CF (Complement Fixation)** | Older Standard | Older Standard | Convalescent | Historical |
| **PCR (Sputum/BAL/Throat/NP)** | **High Sensitivity** | High Sensitivity | **Acute Phase (Day 1-14)** | **Early Diagnosis** |
| **Culture** | **Specialised (HEL/HL Cells/Egg Yolk)** | Specialised (Yolk Sac/Cell Culture) | Weeks | **Reference Only** (BSL-3) |
| **Serum IgM/IgG (EIA/ELISA)** | Commercial Available | Commercial Available | Acute/Convalescent | Screening |

> [!tip] **MIF = Gold Standard** for both. **PCR = Early Diagnosis**. **IgM/IgG Rise (4-fold) in Paired Sera = Confirmatory**.

## 💊 Management

### Antibiotic Therapy

| Infection | 1st Line | Alternative | Duration |
|-----------|----------|-------------|----------|
| **C. pneumoniae (CAP)** | **Doxycycline 100mg PO BD** | **Azithromycin 500mg Day 1 → 250mg OD** / **Levofloxacin 500mg OD** / **Moxifloxacin 400mg OD** | **10-14 Days** (Persistent Cough → Extend) |
| **C. psittaci (Psittacosis)** | **Doxycycline 100mg PO BD** (or IV if Severe) | **Azithromycin 500mg Day 1 → 250mg OD** / **Erythromycin 500mg QID** (Pregnancy) | **14-21 Days** (Prevent Relapse) |
| **Severe Psittacosis (ICU)** | **Doxycycline 100mg IV BD** + **Hydrocortisone** (If Shock) | **IV Azithromycin** / **Levofloxacin IV** | **14-21 Days** |
| **C. trachomatis (LGV/STI)** | **Doxycycline 100mg BD ×21d** | **Azithromycin 1g Single Dose** (Genital) / **Erythromycin** (Pregnancy) | **21 Days (LGV)** |

> [!warning] **Tetracyclines Contraindicated in Pregnancy/Children <8y** → **Azithromycin / Erythromycin**.

> [!tip] **Psittacosis = Doxycycline 100mg BD × 14-21 Days** (Prevent Relapse). **Severe: IV Doxycycline + ICU Support**.

## 🔬 Laboratory Findings

| Test | C. pneumoniae | C. psittaci |
|------|---------------|-------------|
| **CBC** | Normal / Mild Leukocytosis | **Leukopenia** (Early), **Thrombocytopenia** (Possible) |
| **LFT** | Normal / Mild ↑ ALT/AST | **↑ ALT/AST** (Hepatitis Common), **↑ ALP** |
| **CXR** | **Patchy/Interstitial Infiltrates** (Lower Lobe) | **Consolidation / Multifocal Infiltrates**, **Pleural Effusion** |
| **Serology** | **MIF IgM/IgG** (4-fold Rise) | **MIF IgM/IgG** (4-fold Rise) |
| **PCR** | Sputum/BAL/Throat Swab | Sputum/BAL/Throat Swab |

> [!tip] **Horder Nodes (Splenic Microabscesses) = Pathognomonic for Psittacosis** (Seen on CT/US in 20-30%).

## 🔄 Differential Diagnosis

| Condition | C. pneumoniae | C. psittaci |
|-----------|---------------|-------------|
| **Mycoplasma pneumoniae** | **Younger Age**, **Cold Agglutinins**, **Cold Haemolysis**, **Rash**, **Stevens-Johnson** | **Bird Exposure**, **Horder Nodes**, **Hepatitis** |
| **Legionella** | **Diarrhoea, Confusion, Hyponatraemia**, **Urinary Ag+** | **Bird Exposure**, **Horder Nodes**, **Relative Bradycardia** |
| **Viral Pneumonia** | **Myalgia, GI Symptoms**, **Viral PCR+** | **Bird Exposure**, **Hepatitis**, **Horder Nodes** |
| **Typical Bacterial CAP** | **Abrupt Onset**, **High Fever**, **Lobar Consolidation**, **Leucocytosis** | **Bird Exposure**, **Relative Bradycardia**, **Hepatitis** |
| **Viral Pneumonia (Influenza/COVID)** | **Rapid Onset**, **Myalgia, GI**, **Viral PCR+** | **Bird Exposure**, **Horder Nodes**, **Relative Bradycardia** |

## ❓ Viva Questions

1. **What is the typical clinical presentation of C. pneumoniae pneumonia?**
   - **Young Adult**, **Insidious Onset**, **Persistent Dry Cough >3 Weeks**, **Hoarseness/Laryngitis**, **Low-Grade Fever**, **Patchy CXR**.

2. **What are the classic features of Psittacosis?**
   - **Bird Exposure**, **High Fever, Severe Headache, Myalgia**, **Pneumonia**, **Horder Nodes (Splenic Microabscesses)**, **Relative Bradycardia**, **Hepatitis**.

3. **What is the diagnostic test of choice for Chlamydia species?**
   - **Microimmunofluorescence (MIF)** — **Gold Standard**; **4-fold Rise in Paired Sera (IgG/IgM)**; **PCR** for Early Diagnosis.

4. **How do you treat C. pneumoniae pneumonia?**
   - **Doxycycline 100mg PO BD × 10-14 Days** (or **Azithromycin 500mg Day 1 → 250mg OD ×5 Days**).

5. **How does Psittacosis differ from typical bacterial pneumonia?**
   - **Bird Exposure**, **Relative Bradycardia**, **Horder Nodes (Splenic Microabscesses)**, **Hepatitis**, **Relative Bradycardia**, **Photophobia, Severe Headache**.

6. **What are Horder Nodes and their significance?**
   - **Splenic Microabscesses** seen on Imaging (US/CT) in **20-30% of Psittacosis**; **Pathognomonic**.

6. **What is the treatment of choice for Psittacosis?**
   - **Doxycycline 100mg BD PO/IV × 14-21 Days**; **Azithromycin/Erythromycin if Pregnant/Child**.

7. **How do you differentiate C. pneumoniae from Mycoplasma pneumoniae?**
   - **C. pneumoniae: Younger Adults, Prolonged Cough, Hoarseness, No Cold Agglutinins**; **M. pneumoniae: Younger, Cold Agglutinins, Haemolysis, Rash, Stevens-Johnson Risk**.

7. **What is the significance of Relative Bradycardia in Psittacosis?**
   - **Relative Bradycardia = Fever without Expected Tachycardia**; **Seen in Psittacosis, Typhoid, Legionella, Brucellosis, Drug Effect (Beta-Blockers)**.

8. **How do you diagnose C. pneumoniae in a patient with persistent cough?**
   - **Serology (MIF) — 4-fold Rise Paired Sera**; **PCR (Sputum/BAL)** — Rapid; **CXR: Patchy Interstitial Infiltrates**.

9. **What is the treatment duration for Psittacosis to prevent relapse?**
   - **14-21 Days Doxycycline** (Shorter Courses → Relapse Risk).

10. **Differentiate C. pneumoniae from C. psittaci clinically.**
    - **C. pneumoniae**: Community-Acquired, Young Adult, Persistent Cough, Hoarseness, Mild; **C. psittaci**: Bird Exposure, Severe, Horder Nodes, Hepatitis, Relative Bradycardia.

## 🧠 Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **C. pneumoniae vs M. pneumoniae** | **C. pneumoniae: Hoarseness, No Cold Agglutinins, Older Age**; **M. pneumoniae: Cold Agglutinins+, Rash, Younger, Haemolysis** |
| **Psittacosis vs Ornithosis** | **Same Disease**; **Psittacosis = Parrot-Borne**, **Ornithosis = Bird-Borne (General)** |
| **C. pneumoniae vs Legionella** | **C. pneumoniae: No Diarrhoea/Confusion/Hyponatraemia, No Urinary Ag**; **Legionella: GI/Neuro, Urinary Ag+, Hyponatraemia** |
| **C. psittaci vs Typhoid** | **Psittacosis: Bird Exposure, Horder Nodes, Relative Bradycardia, Hepatitis**; **Typhoid: Relative Bradycardia, Rose Spots, Blood Culture+** |
| **C. pneumoniae vs Viral Pneumonia** | **C. pneumoniae: Persistent Cough >3 Weeks, Hoarseness, Patchy CXR**; **Viral: Rapid Onset, Myalgia, GI, Viral PCR+** |

| Mnemonic | Meaning |
|----------|---------|
| **"Psittacosis = Bird + Horder Nodes + Bradycardia"** | Psittacosis Triad |
| **"C. pneumoniae = Persistent Cough >3 Weeks + Hoarseness"** | C. pneumoniae Hallmarks |
| **"Doxycycline = All Chlamydia"** | Universal Treatment |
| **"Horder Nodes = Splenic Microabscesses = Psittacosis"** | Pathognomonic |
| **"MIF = Gold Standard = 4-fold Rise Paired Sera"** | Serology Gold Standard |
| **"Bird Exposure = Think Psittacosis"** | Exposure History |

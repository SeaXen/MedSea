# Sexually Transmitted Infections (STIs)

Related: [[HIV Infection and AIDS]], [[Pelvic Inflammatory Disease]], [[Epididymo-orchitis]], [[Reactive Arthritis]], [[Dermatology (Genital Ulcers)]]

> [!important]
> **STI = infection transmitted predominantly through sexual contact.** **Syndromic management: urethral discharge, vaginal discharge, genital ulcer, inguinal bubo, scrotal swelling, neonatal conjunctivitis.** **Partner notification and treatment essential.** **Screening: chlamydia/gonorrhoea (NAAT), syphilis (serology), HIV, HBV, HCV.** **Expedited partner therapy (EPT) for chlamydia/gonorrhoea.**

## Learning Objectives
- Diagnose common STIs using clinical syndromes and laboratory tests
- Apply syndromic management algorithms
- Select appropriate antimicrobial therapy for each STI
- Manage complications (PID, epididymo-orchitis, neonatal infection)
- Conduct partner notification and expedited partner therapy
- Screen appropriately (chlamydia, gonorrhoea, syphilis, HIV, HBV, HCV)
- Recognise antimicrobial resistance (gonorrhoea, mycoplasma genitalium)

## Syndromic Classification & Common Aetiology

| Syndrome | Common Pathogens |
|----------|------------------|
| **Urethral discharge (male)** | *Chlamydia trachomatis* (40–50%), *Neisseria gonorrhoeae* (30–40%), *M. genitalium* (15–25%), *Ureaplasma*, *Trichomonas* |
| **Vaginal discharge (female)** | *C. trachomatis*, *N. gonorrhoeae*, *T. vaginalis*, *Candida*, BV, *M. genitalium* |
| **Genital ulcer** | **HSV-1/2** (most common), *Treponema pallidum* (syphilis), *Haemophilus ducreyi* (chancroid), *Klebsiella granulomatis* (donovanosis), LGV |
| **Inguinal bubo** | LGV (*C. trachomatis* L1–L3), chancroid, syphilis |
| **Scrotal swelling** | *C. trachomatis*, *N. gonorrhoeae* (epididymo-orchitis) |
| **Neonatal conjunctivitis** | *C. trachomatis* (5–14 days), *N. gonorrhoeae* (2–5 days), HSV |

## Key STIs — Clinical Features, Diagnosis, Treatment

### 1. Chlamydia trachomatis
| Aspect | Details |
|--------|---------|
| **Clinical** | Often asymptomatic (70% women, 50% men); urethritis, cervicitis, PID, epididymitis, LGV (L1–L3), neonatal conjunctivitis/pneumonia |
| **Diagnosis** | **NAAT (urine, swab)** — gold standard; culture rarely used |
| **Treatment (Uncomplicated)** | **Doxycycline 100mg PO 12h ×7 days** (1st line) |
| **Alternative** | Azithromycin 1g PO single dose (if adherence concern, pregnancy) |
| **LGV** | Doxycycline 100mg PO 12h ×21 days |
| **Pregnancy** | Azithromycin 1g PO single dose (or Amoxicillin 500mg PO 8h ×7d) |
| **Partner notification** | All partners last 6 months (or last partner if >6m) |

### 2. Neisseria gonorrhoeae (Gonorrhoea)
| Aspect | Details |
|--------|---------|
| **Clinical** | Urethritis, cervicitis, pharyngitis, proctitis, PID, DGI (arthritis-dermatitis), neonatal ophthalmia |
| **Diagnosis** | **NAAT (urine, swab)**; **Culture + AST essential** (resistance monitoring) |
| **Treatment (Uncomplicated)** | **Ceftriaxone 500mg IM single dose** (if <150kg) / **1g IM if ≥150kg** |
| **PLUS** | **Doxycycline 100mg PO 12h ×7d** (if chlamydia not excluded) OR Azithromycin 1g PO (if doxy contraindicated) |
| **Pharyngeal** | Ceftriaxone 1g IM single dose (higher failure at pharynx) |
| **DGI** | Ceftriaxone 1g IV/IM 24h ×7d (switch oral after 24–48h improvement) |
| **Resistance** | Rising MICs to ceftriaxone, azithromycin; ceftriaxone 1g IM if decreased susceptibility suspected |
| **Pregnancy** | Ceftriaxone 500mg/1g IM + Azithromycin 1g PO |

### 3. Syphilis (Treponema pallidum)
| Stage | Clinical Features | Treatment |
|-------|-------------------|-----------|
| **Primary** | **Chancre**: painless, clean-based, indurated ulcer at inoculation site (3–90d); regional lymphadenopathy | **Benzathine penicillin G 2.4MU IM single dose** |
| **Secondary** | **Rash** (palms/soles, trunk), **condyloma lata** (moist, infectious), mucous patches, **generalised lymphadenopathy**, fever, malaise (4–10w) | **Benzathine penicillin G 2.4MU IM single dose** |
| **Early Latent** | <1 year post-infection; seropositive, no symptoms | **Benzathine penicillin G 2.4MU IM single dose** |
| **Late Latent** | >1 year or unknown duration; seropositive | **Benzathine penicillin G 2.4MU IM weekly ×3 doses** |
| **Tertiary** | Gummatous (skin, bone, liver), Cardiovascular (aortitis, aneurysm), Neurosyphilis | See Neurosyphilis |
| **Neurosyphilis** | Meningitis, stroke, tabes dorsalis, general paresis, ocular/otic syphilis | **Penicillin G 18–24MU/day IV ÷4h ×14d** (or Ceftriaxone 2g IV 12h ×14d) |
| **Ocular/Otic Syphilis** | Uveitis, optic neuritis, hearing loss | Treat as neurosyphilis (IV penicillin ×14d) |

| Special Situations | Treatment |
|-------------------|-----------|
| **Penicillin allergy (non-pregnant)** | Doxycycline 100mg PO 12h ×14d (early) / ×28d (late); Ceftriaxone 1g IM/IV daily ×10–14d |
| **Pregnancy** | **Penicillin ONLY** (desensitise if allergic); Benzathine penicillin G per stage |
| **HIV coinfection** | Same regimens; closer follow-up (RPR titres); consider neurosyphilis if CD4<350 |
| **Jarisch-Herxheimer** | Fever, rigors, hypotension, worsening rash 2–12h post-treatment; self-limiting; prednisolone not routine |

| Serology | Interpretation |
|----------|----------------|
| **Non-treponemal (RPR, VDRL)** | Quantitative; correlates with disease activity; ↓4-fold = treatment response; can be negative in early primary/late tertiary |
| **Treponemal (TPHA, TPPA, EIA, FTA-ABS)** | Qualitative; remains positive for life; confirms exposure |
| **Algorithm** | **Traditional:** Screen treponemal → confirm non-treponemal; **Reverse:** Screen non-treponemal → confirm treponemal |

---

### 4. Genital Herpes (HSV-1/2)
| Aspect | Details |
|--------|---------|
| **Clinical** | **Primary**: painful vesicles/ulcers, inguinal lymphadenopathy, systemic symptoms (fever, malaise); **Recurrent**: milder, shorter, prodrome (tingling, pain) |
| **Diagnosis** | **NAAT (swab from vesicle/ulcer)** — gold standard; Tzanck smear (multinucleated giant cells) — low sensitivity; Type-specific serology (HSV-1/2 IgG) — for counselling, asymptomatic |
| **Treatment (Episodic)** | **Acyclovir 400mg PO 12h ×5d** (or 800mg 12h ×5d); Valacyclovir 500mg 12h ×5d; Famciclovir 250mg 12h ×5d |
| **Treatment (Suppressive)** | **Acyclovir 400mg PO 12h**; Valacyclovir 500mg PO OD; Famciclovir 250mg PO 12h — **if ≥6 recurrences/year or severe psychosocial impact** |
| **Severe/Immunocompromised** | Acyclovir 5mg/kg IV 8h ×5–7d → oral step-down |
| **Pregnancy** | **Suppressive from 36 weeks** (Acyclovir 400mg 12h or Valacyclovir 500mg OD) to reduce neonatal HSV; C-section if active lesions at delivery |
| **Neonatal HSV** | **Acyclovir 20mg/kg IV 8h ×14d** (SEM) / ×21d (CNS/Disseminated) |

---

### 5. Mycoplasma genitalium
| Aspect | Details |
|--------|---------|
| **Clinical** | Urethritis (often non-gonococcal, non-chlamydial), cervicitis, PID; **increasing macrolide resistance** |
| **Diagnosis** | **NAAT (with macrolide resistance detection if available)** |
| **Treatment (Resistance unknown / sensitive)** | **Doxycycline 100mg PO 12h ×7d → Azithromycin 1g PO day 1, then 500mg daily ×3d** (total 2.5g) |
| **Treatment (Macrolide-resistant)** | **Moxifloxacin 400mg PO OD ×7–14d** (after doxycycline lead-in) |
| **Resistance testing** | Recommended before treatment if available; guides macrolide vs fluoroquinolone choice |

---

### 6. Trichomonas vaginalis
| Aspect | Details |
|--------|---------|
| **Clinical** | Vaginitis (frothy yellow-green discharge, odour, pruritus, strawberry cervix); urethritis (often asymptomatic in men) |
| **Diagnosis** | **NAAT** (gold standard); wet mount (motile trichomonads, sensitivity ~60%); culture |
| **Treatment** | **Metronidazole 2g PO single dose** (or 400mg 12h ×7d); Tinidazole 2g single dose |
| **Partner treatment** | Simultaneous (single dose metronidazole) |
| **Pregnancy** | Metronidazole 2g single dose safe after 1st trimester |

---

### 7. Lymphogranuloma Venereum (LGV)
| Aspect | Details |
|--------|---------|
| **Aetiology** | *C. trachomatis* serovars L1, L2, L3 |
| **Clinical** | **Proctocolitis** (MSM): rectal pain, discharge, bleeding, tenesmus, ulceration; **Inguinal syndrome**: bubo, fistulae; **Genital ulcer** (rare) |
| **Diagnosis** | **NAAT + genotyping (L1–L3)** or **rectal swab for LGV PCR** |
| **Treatment** | **Doxycycline 100mg PO 12h ×21 days** |
| **Alternative** | Erythromycin 500mg PO 6h ×21d (pregnancy) |

---

### 8. Chancroid (Haemophilus ducreyi)
| Aspect | Details |
|--------|---------|
| **Clinical** | Painful, soft, purulent ulcer with ragged edges; **painful inguinal lymphadenopathy (bubo)** |
| **Diagnosis** | Clinical + PCR (specialised); culture difficult |
| **Treatment** | **Azithromycin 1g PO single dose** OR Ceftriaxone 250mg IM single dose OR Ciprofloxacin 500mg PO 12h ×3d OR Erythromycin 500mg PO 6h ×7d |

---

### 9. Donovanosis (Klebsiella granulomatis)
| Aspect | Details |
|--------|---------|
| **Clinical** | Progressive, painless, beefy-red, granulomatous ulcers (pseudobuboes); genital, perineal, perianal |
| **Diagnosis** | Tissue smear (Donovan bodies — intracytoplasmic rods in macrophages); PCR |
| **Treatment** | **Azithromycin 1g PO weekly ×3 weeks** OR Doxycycline 100mg 12h ×3w OR TMP-SMX DS 12h ×3w |

---

### 10. Pelvic Inflammatory Disease (PID)
| Aspect | Details |
|--------|---------|
| **Clinical** | Lower abdominal pain, cervical motion tenderness, adnexal tenderness, fever, abnormal discharge, irregular bleeding |
| **Diagnosis** | Clinical (high index of suspicion); **empirical treatment if criteria met**; laparoscopy gold standard |
| **Treatment (Outpatient, mild-moderate)** | **Ceftriaxone 500mg IM single dose + Doxycycline 100mg PO 12h ×14d + Metronidazole 400mg PO 12h ×14d** |
| **Treatment (Inpatient, severe)** | **Ceftriaxone 2g IV 24h + Doxycycline 100mg PO/IV 12h + Metronidazole 500mg IV 8h** → step-down oral ×14d total |
| **Alternative (if cephalosporin allergy)** | Ofloxacin 400mg PO 12h + Metronidazole 400mg PO 12h ×14d (NOT if gonorrhoea suspected) |

---

### 11. Epididymo-orchitis
| Aspect | Details |
|--------|---------|
| **<35 years / STI risk** | **Ceftriaxone 500mg IM single dose + Doxycycline 100mg PO 12h ×10–14d** (covers *C. trachomatis*, *N. gonorrhoeae*) |
| **>35 years / low STI risk / urinary tract** | **Ciprofloxacin 500mg PO 12h ×10–14d** OR Ofloxacin 200mg PO 12h ×10–14d (covers Enterobacteriaceae) |
| **Mumps orchitis** | Supportive (scrotal elevation, NSAIDs); no antibiotics |

## Partner Notification & Expedited Partner Therapy (EPT)
| Principle | Details |
|-----------|---------|
| **Partner notification** | All sexual partners in relevant lookback period |
| **Lookback periods** | Chlamydia: 6 months; Gonorrhoea: 2 months (or last partner); Syphilis: primary/secondary/early latent = 3m/6m/12m; HIV: all partners since last negative test |
| **EPT** | **Patient-delivered partner therapy** for chlamydia (azithromycin 1g) and gonorrhoea (ceftriaxone 500mg IM + azithromycin 1g) where legal/approved |
| **Abstinence** | Until both treated + 7 days after completion |

## STI Screening Recommendations
| Population | Screen For |
|------------|------------|
| **All sexually active <25** | Annual **Chlamydia** (NAAT urine/swab); **Gonorrhoea** if risk factors |
| **MSM** | **3-site testing**: pharyngeal, rectal, urethral for CT/NG; Syphilis (serology); HIV; HBV/HCV |
| **Pregnancy** | **Syphilis** (booking, 28w, delivery if risk); **HIV** (booking); **HBV** (booking); **Chlamydia/Gonorrhoea** if <25 or risk; **HCV** if risk |
| **New partner / symptomatic** | Full screen (CT, NG, Syphilis, HIV, HBV, HCV) |
| **PrEP users** | 3-monthly: HIV, Syphilis, CT/NG (3-site MSM), Renal function |

## Antimicrobial Resistance Concerns
| Pathogen | Resistance Issue |
|----------|------------------|
| **N. gonorrhoeae** | **Rising ceftriaxone MICs** (ceftriaxone 1g IM if decreased susceptibility); azithromycin resistance; MDR/XDR strains reported |
| **M. genitalium** | **High macrolide resistance** (mutations in 23S rRNA); moxifloxacin resistance emerging |
| **C. trachomatis** | Rare resistance to doxycycline/azithromycin |
| **T. pallidum** | No documented penicillin resistance |

## FCPS/MRCP High-Yield Points
- **Urethral discharge: CT (doxy 7d) → NG (ceftriaxone IM) → Mgen (doxy + azithro/moxi)**
- **Vaginal discharge: CT/NG/T trichomonas/Candida/BV**
- **Genital ulcer: HSV (NAAT, acyclovir) → Syphilis (RPR/TPPA, penicillin) → Chancroid/LGV/Donovanosis**
- **Syphilis stages: Primary/Secondary/Early Latent = 2.4MU BPG ×1; Late Latent/Tertiary = 2.4MU BPG ×3 weekly; Neurosyphilis = Pen G IV 18-24MU/day ×14d**
- **Pregnancy syphilis: Penicillin ONLY (desensitise if allergic)**
- **Gonorrhoea: Ceftriaxone 500mg/1g IM + Doxy 7d (or azithro 1g if doxy contraindicated)**
- **Chlamydia: Doxycycline 100mg 12h ×7d (1st line); Azithro 1g single if pregnancy/adherence**
- **LGV: Doxycycline ×21d**
- **M. genitalium: Doxy lead-in + Azithro (if sensitive) OR Moxifloxacin (if resistant)**
- **PID: Ceftriaxone + Doxy + Metro ×14d**
- **Epididymo-orchitis <35y: Ceftriaxone + Doxy; >35y: Cipro/Ofloxacin**
- **Partner notification: Chlamydia 6m, Gonorrhoea 2m, Syphilis by stage**
- **Screening: <25y annual CT; MSM 3-site CT/NG + Syphilis + HIV; Pregnancy full screen**

## Common Viva Questions
1. **First-line treatment for uncomplicated chlamydia?** Doxycycline 100mg PO 12h ×7 days.
2. **Treatment for uncomplicated gonorrhoea?** Ceftriaxone 500mg IM single dose (1g if ≥150kg) + Doxycycline 100mg 12h ×7d.
3. **Syphilis treatment by stage?** Primary/Secondary/Early Latent: BPG 2.4MU IM ×1; Late Latent/Tertiary: BPG 2.4MU IM weekly ×3; Neurosyphilis: Pen G 18–24MU/day IV ×14d.
4. **Jarisch-Herxheimer reaction?** Fever, rigors, hypotension, worsening rash 2–12h post-penicillin; self-limiting.
5. **LGV treatment?** Doxycycline 100mg 12h ×21 days.
6. **M. genitalium macrolide-resistant treatment?** Doxycycline 100mg 12h ×7d → Moxifloxacin 400mg OD ×7–14d.
7. **PID empirical regimen?** Ceftriaxone 500mg IM + Doxycycline 100mg 12h ×14d + Metronidazole 400mg 12h ×14d.
8. **Epididymo-orchitis in 30-year-old?** Ceftriaxone 500mg IM + Doxycycline 100mg 12h ×10–14d.
9. **Partner notification lookback for chlamydia?** 6 months (or last partner if >6m).
10. **Neonatal HSV treatment?** Acyclovir 20mg/kg IV 8h ×14d (SEM) / ×21d (CNS/Disseminated).

## Common Confusions / Exam Traps
| Confusion | Clarification |
|-----------|---------------|
| Azithromycin 1st line for chlamydia | **Doxycycline 7d = 1st line** (higher cure, less resistance); Azithro = alternative |
| Ceftriaxone IV for uncomplicated gonorrhoea | **Ceftriaxone IM single dose** (IV only for DGI, pharyngeal fallback) |
| Single dose BPG for late latent syphilis | **Late latent = BPG 2.4MU weekly ×3 doses** |
| Doxycycline in pregnancy | **CONTRAINDICATED** (fetal bone/teeth); use Azithro (chlamydia), Amoxicillin, Erythromycin |
| Fluconazole for trichomonas | **Metronidazole/Tinidazole ONLY** |
| Moxifloxacin 1st line for Mgen | **Doxy + Azithro 1st line; Moxifloxacin ONLY if macrolide-resistant** |
| EPT for syphilis | **EPT ONLY for chlamydia/gonorrhoea** (not syphilis — needs staging, penicillin) |
| Asymptomatic partner = no treatment | **Asymptomatic partners MUST be treated** (especially chlamydia/gonorrhoea/syphilis) |
| HIV PrEP = STI protection | **PrEP protects against HIV ONLY, not other STIs** |

## Summary
**Sexually Transmitted Infections (STIs)** are infections transmitted predominantly through sexual contact. **Syndromic presentation:** urethral discharge, vaginal discharge, genital ulcer, inguinal bubo, scrotal swelling, neonatal conjunctivitis. **Chlamydia trachomatis:** commonest bacterial STI; often asymptomatic; **NAAT diagnosis; 1st line: Doxycycline 100mg PO 12h ×7d**; alternative azithromycin 1g single dose; LGV (L1–L3) → doxycycline ×21d. **Neisseria gonorrhoeae:** **NAAT + culture for AST; Ceftriaxone 500mg IM (1g if ≥150kg) + Doxycycline 12h ×7d**; pharyngeal → ceftriaxone 1g IM; DGI → ceftriaxone 1g IV ×7d; **rising ceftriaxone MICs.** **Syphilis (T. pallidum):** stages — primary (chancre), secondary (rash, condyloma lata), early latent (<1y), late latent (>1y), tertiary (gumma, CV, neuro), neurosyphilis, ocular/otic. **Serology:** non-treponemal (RPR/VDRL, quantitative) + treponemal (TPPA/EIA, qualitative). **Treatment:** Primary/Secondary/Early Latent → Benzathine penicillin G 2.4MU IM single dose; Late Latent/Tertiary → BPG 2.4MU IM weekly ×3; Neurosyphilis/Ocular/Otic → Penicillin G 18–24MU/day IV ÷4h ×14d (or Ceftriaxone 2g IV 12h ×14d). **Pregnancy: penicillin ONLY (desensitise if allergic).** **Jarisch-Herxheimer:** fever, rigors, hypotension 2–12h post-treatment. **Herpes (HSV-1/2):** NAAT diagnosis; episodic acyclovir 400mg 12h ×5d; suppressive if ≥6/year acyclovir 400mg 12h. **Pregnancy:** suppressive from 36w; C-section if active lesions. **Neonatal HSV:** acyclovir 20mg/kg IV 8h ×14d (SEM) / ×21d (CNS/Disseminated). **M. genitalium:** NAAT with resistance testing; sensitive → doxycycline 7d → azithromycin 2.5g total; resistant → doxycycline 7d → moxifloxacin 400mg OD ×7–14d. **Trichomonas:** metronidazole 2g single dose. **LGV:** doxycycline ×21d. **Chancroid:** azithromycin 1g single dose. **Donovanosis:** azithromycin 1g weekly ×3. **PID:** ceftriaxone 500mg IM + doxycycline 100mg 12h ×14d + metronidazole 400mg 12h ×14d. **Epididymo-orchitis:** <35y → ceftriaxone + doxycycline 10–14d; >35y → ciprofloxacin/ofloxacin 10–14d. **Partner notification:** chlamydia 6 months, gonorrhoea 2 months, syphilis by stage; EPT for chlamydia/gonorrhoea. **Screening:** <25y annual chlamydia; MSM 3-site CT/NG + syphilis + HIV; pregnancy full screen. **Resistance:** gonorrhoea (ceftriaxone MICs, azithromycin), M. genitalium (macrolide resistance).

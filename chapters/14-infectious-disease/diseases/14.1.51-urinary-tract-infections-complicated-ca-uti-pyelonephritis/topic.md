# Urinary Tract Infections (Complicated, Catheter-Associated, Pyelonephritis)

Related: [[Sepsis and Septic Shock]], [[Antimicrobial Stewardship]], [[Prosthetic Joint Infection]], [[Renal Transplant Infections]]

> [!important]
> **UTI = significant bacteriuria + urinary symptoms.** **Uncomplicated cystitis: healthy non-pregnant women.** **Complicated UTI: structural/functional abnormality, male, pregnant, immunocompromised, catheter, recent instrumentation.** **CA-UTI: catheter ≥2 days + symptoms/signs.** **Pyelonephritis: flank pain, fever, CVA tenderness + bacteriuria.** **Empirical antibiotics by local resistance, severity, risk factors.** **Duration: cystitis 3–5d, pyelonephritis 7–14d, CA-UTI 7d (remove catheter if possible).** **Asymptomatic bacteriuria: treat ONLY in pregnancy, pre-urologic procedure, renal transplant (early).**

## Learning Objectives
- Classify UTI: uncomplicated vs complicated, cystitis vs pyelonephritis, CA-UTI
- Diagnose using symptoms, urinalysis, urine culture thresholds
- Select empirical antibiotics by syndrome, local resistance, MDR risk
- Determine appropriate treatment duration
- Manage asymptomatic bacteriuria (when to treat vs not)
- Recognise and manage complications: emphysematous pyelonephritis, renal abscess, sepsis
- Apply CA-UTI prevention strategies

## Classification
| Category | Definition |
|----------|------------|
| **Uncomplicated cystitis** | Healthy, non-pregnant, premenopausal women; no structural/functional abnormality |
| **Complicated UTI** | **ANY of:** male, pregnant, postmenopausal, immunocompromised, diabetes, structural abnormality (stones, obstruction, reflux), indwelling catheter, recent instrumentation, renal transplant, neurogenic bladder |
| **Pyelonephritis** | UTI + **fever/flank pain/CVA tenderness** (upper tract involvement) |
| **CA-UTI** | **Catheter ≥2 days** + symptoms/signs (fever, suprapubic pain, costovertebral angle tenderness, delirium) + **≥10³ CFU/mL** from catheter specimen |
| **Asymptomatic bacteriuria (ASB)** | **≥10⁵ CFU/mL** (clean-catch) or **≥10² CFU/mL** (catheter) in **ASYMPTOMATIC** patient |

## Aetiology
| Syndrome | Common Pathogens |
|----------|------------------|
| **Uncomplicated cystitis** | *E. coli* (75–90%), *S. saprophyticus* (5–15%), *Klebsiella*, *Proteus* |
| **Complicated UTI / Pyelonephritis** | *E. coli* (50–60%), *Klebsiella*, *Proteus*, *Pseudomonas*, *Enterococcus*, *S. aureus* |
| **CA-UTI** | *E. coli*, *Klebsiella*, *Pseudomonas*, *Enterococcus*, *Candida*, *Proteus*, *Morganella*, *Providencia* (often polymicrobial) |
| **Healthcare-associated / MDR risk** | ESBL *Enterobacteriaceae*, *Pseudomonas*, *Acinetobacter*, *Enterococcus* (VRE), *Serratia* |

> [!tip]
> ***Proteus* → urease → struvite stones, alkaline urine.** ***Pseudomonas* → biofilm on catheters.** ***Candida* in catheterised/ICU patients — treat only if symptomatic + pyuria.**

## Diagnosis
### Urinalysis
| Finding | Significance |
|---------|--------------|
| **Nitrite +** | Specific for Enterobacteriaceae (reduces nitrate) |
| **Leukocyte esterase +** | Pyuria (WBC in urine) |
| **WBC >10/HPF** | Pyuria |
| **Bacteria on microscopy** | ~10⁵ CFU/mL if ≥1 bacterium/HPF |

> [!warning]
> **Pyuria + symptoms = UTI.** **Pyuria alone (no symptoms) = ASB (do not treat except specific indications).** **Elderly: delirium may be ONLY symptom.**

### Urine Culture Thresholds (Significant Bacteriuria)
| Specimen | Threshold | Notes |
|----------|-----------|-------|
| **Clean-catch midstream (female)** | **≥10⁵ CFU/mL** | ≥10² if symptomatic (high pre-test probability) |
| **Clean-catch midstream (male)** | **≥10⁴ CFU/mL** | |
| **Straight catheter** | **≥10² CFU/mL** | |
| **Suprapubic aspiration** | **≥10¹ CFU/mL** | Gold standard |
| **Indwelling catheter** | **≥10³ CFU/mL** | For CA-UTI diagnosis |

## Empirical Antibiotic Therapy
### Uncomplicated Cystitis (Women)
| Agent | Dose | Duration | Notes |
|-------|------|----------|-------|
| **Nitrofurantoin** | 100mg PO 12h (MR) or 50mg 6h | **3 days** (MR: 5 days) | **1st line**; avoid if CrCl <30, G6PD, pregnancy term |
| **Trimethoprim-sulfamethoxazole (TMP-SMX)** | 160/800mg PO 12h | **3 days** | 1st line if local *E. coli* resistance <20% |
| **Fosfomycin** | 3g PO single dose | **Single dose** | Alternative; lower cure rate |
| **Pivmecillinam** | 400mg PO 8h | **3–5 days** | Alternative (not in all countries) |
| **Fluoroquinolones (ciprofloxacin, levofloxacin)** | Cipro 250mg 12h / Levo 250mg OD | 3 days | **Avoid 1st line** (resistance, tendinopathy, C. diff, aortic dissection risk) |

### Complicated UTI / Pyelonephritis (Outpatient / Mild Inpatient)
| Agent | Dose | Duration |
|-------|------|----------|
| **Ciprofloxacin** | 500mg PO 12h (or 750mg XR OD) | **7 days** (pyelo) / 5–7 days (complicated) |
| **Levofloxacin** | 750mg PO OD | **5–7 days** |
| **TMP-SMX** | 160/800mg PO 12h | **7–14 days** (if susceptible) |
| **Cefpodoxime / Cefdinir** | 200mg PO 12h / 300mg PO 12h | 7–14 days |
| **IV → Oral switch** | Ceftriaxone 1–2g IV 24h → oral fluoroquinolone/TMP-SMX | When clinically stable |

### Complicated UTI / Pyelonephritis (Inpatient / Severe / MDR Risk)
| Scenario | Empirical Regimen | Duration |
|----------|-------------------|----------|
| **No MDR risk, stable** | **Ceftriaxone 1–2g IV 24h** | 7–14 days (IV→oral at stability) |
| **MDR risk (prior abx, healthcare, travel, known colonisation)** | **Piperacillin-tazobactam 4.5g IV 6h** OR **Meropenem 1g IV 8h** OR **Cefepime 2g IV 8h** | 7–14 days |
| **+ Enterococcus risk (catheter, instrumentation, abscess)** | Add **Amoxicillin 2g IV 4h** (if E. faecalis) OR **Vancomycin** (if VRE risk) | |
| **+ Pseudomonas risk** | Anti-pseudomonal β-lactam above | |
| **Sepsis / Shock** | **Meropenem 1g IV 8h** (or Pip-tazo) ± Vancomycin ± Aminoglycoside | 7–14 days |

### CA-UTI
| Step | Action |
|------|--------|
| **1. Remove/replace catheter** if ≥2 days (send tip for culture if removed) |
| **2. Empirical antibiotics** (if symptomatic): Same as complicated UTI by severity |
| **3. Duration** | **7 days** (if catheter removed); 10–14 days if catheter MUST remain |
| **4. Catheter change** | **Replace catheter after 48–72h of antibiotics** if remaining |

> [!key]
> **CA-UTI: REMOVE catheter if possible.** **If catheter must stay, change after 48–72h antibiotics.** **Duration 7d if removed, 10–14d if retained.**

## Asymptomatic Bacteriuria (ASB) — When to TREAT
| Population | Treat? |
|------------|--------|
| **Pregnant women** | **YES** (screen at 12–16 weeks; treat to prevent pyelonephritis) |
| **Pre-urologic procedure with mucosal bleeding** | **YES** (pre-op) |
| **Renal transplant (first 1–3 months)** | **YES** (some guidelines) |
| **Neutropenic patients** | Controversial (some treat) |
| **Healthy non-pregnant women** | **NO** |
| **Elderly / LTCF residents** | **NO** |
| **Diabetics** | **NO** |
| **Catheterised patients (chronic)** | **NO** |
| **Spinal cord injury / neurogenic bladder** | **NO** |
| **Pre-non-urologic surgery** | **NO** |

> [!critical]
> **ASB treatment in non-indicated populations → ↑ antibiotic resistance, ↑ C. diff, NO clinical benefit.** **Pyuria in ASB does NOT indicate treatment.**

## Treatment Duration Summary
| Syndrome | Duration |
|----------|----------|
| **Uncomplicated cystitis (women)** | **3 days** (Nitrofurantoin, TMP-SMX); Single dose (Fosfomycin) |
| **Complicated cystitis** | **5–7 days** |
| **Pyelonephritis (outpatient/mild)** | **7 days** (fluoroquinolone) / **10–14 days** (β-lactam/TMP-SMX) |
| **Pyelonephritis (severe/inpatient)** | **10–14 days** (IV→oral at stability) |
| **CA-UTI (catheter removed)** | **7 days** |
| **CA-UTI (catheter retained)** | **10–14 days** (change catheter at 48–72h) |
| **Prostatitis (acute bacterial)** | **4–6 weeks** |
| **Prostatitis (chronic bacterial)** | **6–12 weeks** |

## Complications
| Complication | Features | Management |
|--------------|----------|------------|
| **Emphysematous pyelonephritis** | Gas in renal parenchyme/collecting system (diabetics); high mortality | **Emergency nephrectomy** + broad antibiotics (Pip-tazo/Meropenem) |
| **Renal / Perinephric abscess** | Flank mass, persistent fever, failure to respond | **CT-guided drainage + antibiotics** |
| **Papillary necrosis** | Diabetics, sickle cell, analgesic abuse; flank pain, haematuria | Supportive, antibiotics, urology |
| **Xanthogranulomatous pyelonephritis** | Chronic, staghorn stones, *Proteus*; mass mimics tumour | Nephrectomy |
| **Sepsis / Septic shock** | See Sepsis note | Sepsis bundle + source control |
| **Prostatitis (acute)** | Fever, perineal pain, dysuria, tender prostate | **Fluoroquinolone 4–6 weeks** (prostate penetration) |

## MDR Pathogens in UTI
| Pathogen | Preferred Treatment | Alternatives |
|----------|---------------------|--------------|
| **ESBL Enterobacteriaceae** | **Meropenem 1g IV 8h** (or Ertapenem 1g IV 24h if no Pseudomonas risk) | Fosfomycin IV, Ceftazidime-avibactam, Plazomicin |
| **CRE** | **Ceftazidime-avibactam, Meropenem-vaborbactam, Imipenem-relebactam, Plazomicin, Polymyxin** | Fosfomycin, Tigecycline (poor urine levels) |
| **MDR Pseudomonas** | **Ceftolozane-tazobactam, Ceftazidime-avibactam, Imipenem-relebactam, Polymyxin + high-dose meropenem** | Aminoglycoside + β-lactam |
| **VRE (E. faecium)** | **Linezolid 600mg IV/PO 12h** OR **Daptomycin 8–10mg/kg IV OD** (NOT for urine — inactivated by surfactant) | Tigecycline (poor urine levels) |
| **Enterococcus faecalis (ampicillin-R)** | **Vancomycin** OR **Linezolid** | Daptomycin |
| **Candida (symptomatic + pyuria)** | **Fluconazole 200–400mg PO/IV OD** (if susceptible) | Echinocandin (caspofungin/micafungin) — poor urine levels except micafungin |

> [!tip]
> **Fosfomycin IV excellent for MDR UTI (broad Gram-neg + Enterococcus).** **Daptomycin inactivated in urine — avoid for UTI.** **Tigecycline poor urine levels — avoid for UTI.** **Echinocandins poor urine levels (except micafungin) — fluconazole preferred for Candida UTI.**

## CA-UTI Prevention Bundle
| Component | Action |
|-----------|--------|
| **Avoid unnecessary catheterisation** | Use alternatives (condom catheter, intermittent catheterisation, nappies) |
| **Aseptic insertion** | Sterile technique, smallest appropriate catheter |
| **Closed drainage system** | Maintain unobstructed flow; no disconnections |
| **Catheter securement** | Prevent traction/urethral trauma |
| **Maintain drainage bag below bladder** | Prevent reflux |
| **Daily review of necessity** | **Remove as soon as possible** (nurse-driven protocols) |
| **Periurethral cleaning** | Routine soap/water (no antiseptic benefit) |
| **Antibiotic prophylaxis at insertion** | **NOT routinely recommended** |
| **Catheter material** | Silver-alloy or hydrogel-coated if prolonged (>7d) anticipated |

## Special Situations
| Situation | Management |
|-----------|------------|
| **Pregnancy** | **Screen for ASB at 12–16w; treat positive.** Cystitis: Nitrofurantoin (avoid term), Cephalexin, Fosfomycin. Pyelonephritis: IV Ceftriaxone → oral cephalexin. Avoid fluoroquinolones, TMP-SMX (1st tri), Nitrofurantoin (term). |
| **Renal transplant** | ASB treat early (1–3m); UTI: adjust for immunosuppression interactions (fluconazole ↑ tacrolimus, TMP-SMX ↑ creatinine). |
| **Neurogenic bladder / Spinal cord** | Asymptomatic bacteriuria universal — do NOT treat. Treat only if symptomatic (fever, autonomic dysreflexia, new incontinence). |
| **Prostatitis** | Acute: fluoroquinolone 4–6w (prostate penetration). Chronic: fluoroquinolone/TMP-SMX 6–12w. |
| **Urosepsis with obstruction** | **URGENT decompression (nephrostomy/stent) + antibiotics** — source control critical. |

## FCPS/MRCP High-Yield Points
- **UTI classification: uncomplicated cystitis (healthy women) vs complicated (ALL others)**
- **Diagnosis: symptoms + pyuria + culture (≥10⁵ clean-catch, ≥10² catheter, ≥10³ indwelling catheter)**
- **Uncomplicated cystitis: Nitrofurantoin 3d (1st line), TMP-SMX 3d (if resistance <20%), Fosfomycin single dose**
- **Complicated UTI/Pyelonephritis: Fluoroquinolone 7d (outpatient) or IV Ceftriaxone → oral**
- **MDR risk: prior abx, healthcare, travel, colonisation → Pip-tazo/Meropenem/Cefepime**
- **CA-UTI: catheter ≥2d + symptoms + ≥10³ CFU/mL → REMOVE catheter; 7d antibiotics if removed**
- **Asymptomatic bacteriuria: TREAT only in pregnancy, pre-urologic procedure with bleeding, early renal transplant**
- **Duration: cystitis 3d, pyelo 7–14d, CA-UTI 7d (removed) / 10–14d (retained)**
- **ESBL: carbapenem (meropenem/ertapenem); CRE: newer β-lactamase inhibitors; VRE: linezolid (daptomycin fails in urine)**
- **Emphysematous pyelonephritis: diabetics, gas on imaging → emergency nephrectomy + antibiotics**
- **Prostatitis: fluoroquinolone 4–6w (penetration)**

## Common Viva Questions
1. **What is the urine culture threshold for CA-UTI?** ≥10³ CFU/mL from indwelling catheter specimen.
2. **When do you treat asymptomatic bacteriuria?** Pregnancy, pre-urologic procedure with mucosal bleeding, renal transplant (early).
3. **First-line for uncomplicated cystitis?** Nitrofurantoin 100mg PO 12h ×3 days (or TMP-SMX if resistance <20%).
4. **Duration for pyelonephritis?** 7 days (fluoroquinolone) or 10–14 days (β-lactam/TMP-SMX).
5. **CA-UTI management?** Remove catheter if possible; antibiotics 7d if removed, 10–14d if retained; change catheter at 48–72h.
6. **ESBL UTI treatment?** Carbapenem (meropenem/ertapenem).
7. **Why avoid daptomycin for VRE UTI?** Inactivated by pulmonary surfactant in urine (actually — inactivated by calcium/surfactant; poor urinary penetration).
8. **Emphysematous pyelonephritis management?** Emergency nephrectomy + broad antibiotics.
9. **Asymptomatic bacteriuria in elderly LTCF resident — treat?** NO.
10. **Acute prostatitis treatment and duration?** Fluoroquinolone 4–6 weeks (prostate penetration).

## Common Confusions / Exam Traps
| Confusion | Clarification |
|-----------|---------------|
| Treat all asymptomatic bacteriuria | **Treat ONLY: pregnancy, pre-urologic bleeding, early renal transplant** |
| Pyuria = treat | **Pyuria without symptoms = ASB (do not treat)** |
| Fosfomycin 1st line for cystitis | **Nitrofurantoin/TMP-SMX 1st line; Fosfomycin alternative (lower cure)** |
| Fluoroquinolone 1st line for cystitis | **Avoid 1st line (resistance, tendinopathy, C. diff, aortic risk)** |
| CA-UTI: treat without removing catheter | **REMOVE catheter if possible; change at 48–72h if retained** |
| Daptomycin for VRE UTI | **Daptomycin inactivated in urine — use Linezolid** |
| Tigecycline for MDR UTI | **Poor urine levels — avoid** |
| Echinocandin for Candida UTI | **Poor urine levels (except micafungin) — Fluconazole 1st line** |
| Nitrofurantoin in renal impairment | **Avoid if CrCl <30 (ineffective, toxicity)** |
| ASB in catheterised = treat | **Chronic catheter = ASB universal; do NOT treat** |

## Summary
**Urinary tract infections (UTIs)** classified by host factors and anatomy. **Uncomplicated cystitis** = healthy, non-pregnant, premenopausal women. **Complicated UTI** = male, pregnant, postmenopausal, immunocompromised, diabetes, structural abnormality, indwelling catheter, recent instrumentation, renal transplant, neurogenic bladder. **Pyelonephritis** = UTI + fever/flank pain/CVA tenderness. **CA-UTI** = catheter ≥2 days + symptoms/signs + ≥10³ CFU/mL from catheter specimen. **Asymptomatic bacteriuria (ASB)** = significant bacteriuria without symptoms. **Diagnosis:** symptoms + pyuria (nitrite, leukocyte esterase, WBC>10/HPF) + urine culture. **Thresholds:** clean-catch female ≥10⁵, male ≥10⁴; straight catheter ≥10²; indwelling catheter ≥10³; suprapubic aspiration ≥10¹. **Treatment:** **Uncomplicated cystitis:** Nitrofurantoin 100mg PO 12h ×3d (1st line), TMP-SMX 160/800mg 12h ×3d (if resistance <20%), Fosfomycin 3g single dose. **Complicated cystitis/Pyelonephritis outpatient:** Fluoroquinolone 7d (ciprofloxacin 500mg 12h / levofloxacin 750mg OD) or Ceftriaxone 1–2g IV 24h → oral step-down 7–14d. **Inpatient/severe/MDR risk:** Pip-tazo 4.5g 6h / Meropenem 1g 8h / Cefepime 2g 8h ± vancomycin. **CA-UTI:** **Remove catheter if possible**; antibiotics 7d if removed, 10–14d if retained; change catheter at 48–72h. **ASB: TREAT ONLY** in pregnancy (screen 12–16w), pre-urologic procedure with mucosal bleeding, renal transplant (early). **ESBL:** carbapenem (meropenem/ertapenem). **CRE:** ceftazidime-avibactam, meropenem-vaborbactam, imipenem-relebactam, plazomicin, polymyxin. **VRE:** linezolid (daptomycin inactivated in urine). **Candida UTI:** fluconazole (echinocandins poor urine levels). **Emphysematous pyelonephritis** (diabetics, gas on imaging): emergency nephrectomy + broad antibiotics. **Acute prostatitis:** fluoroquinolone 4–6 weeks (prostate penetration).

## Answer Key with Explanations
### MCQs
1. **C** — Clean-catch midstream female: ≥10⁵ CFU/mL. Male: ≥10⁴. Straight catheter: ≥10². Indwelling catheter: ≥10³.
2. **B** — Nitrofurantoin 100mg PO 12h ×3d is 1st line (IDSA/ESCMID). TMP-SMX if resistance <20%. Fluoroquinolones avoided 1st line. Fosfomycin alternative.
3. **B** — ASB treatment ONLY: pregnancy, pre-urologic procedure with mucosal bleeding, renal transplant early. Elderly LTCF = do NOT treat.
4. **B** — CA-UTI definition: indwelling catheter ≥2 days (≥48h) + symptoms/signs + ≥10³ CFU/mL.
5. **C** — ESBL Enterobacteriaceae: carbapenem (meropenem/ertapenem) is treatment of choice.
6. **B** — Daptomycin is inactivated by calcium and surfactant in urine; poor urinary activity. Linezolid preferred for VRE UTI.
7. **C** — CA-UTI with catheter removed: 7 days antibiotics. If catheter retained: 10–14 days + change catheter at 48–72h.
8. **B** — Emphysematous pyelonephritis: >90% in diabetics; gas in renal parenchyma/collecting system; emergency nephrectomy + antibiotics.
9. **B** — Acute bacterial prostatitis: fluoroquinolone 4–6 weeks (achieves prostatic penetration). β-lactams/TMP-SMX less reliable.
10. **B** — Echinocandins have poor urinary penetration (except micafungin); fluconazole achieves high urinary concentrations.

### SBAs
1. **B** — Uncomplicated cystitis in healthy woman: Nitrofurantoin 3d 1st line.
2. **C** — Pyelonephritis (fever + flank pain) in diabetic (complicated): Fluoroquinolone 7d preferred outpatient. Ceftriaxone→oral also acceptable.
3. **C** — CA-UTI with catheter removed: 7 days antibiotics.
4. **B** — Early renal transplant (≤3 months): ASB treated (some guidelines). Controversial but commonly tested.
5. **A** — *Pseudomonas* UTI: oral fluoroquinolone (ciprofloxacin/levofloxacin) only oral with reliable anti-pseudomonal activity.
6. **C** — Prior recent antibiotics + ESBL risk → empirical carbapenem (meropenem). Ceftriaxone/pip-tazo unreliable for ESBL.
7. **B** — VRE UTI: Linezolid. Daptomycin inactivated in urine. Tigecycline poor urine levels.
8. **B** — Pregnancy ASB: treat. Nitrofurantoin avoid term (1st tri OK but many avoid); TMP-SMX avoid 1st tri (folate antagonist); Fluoroquinolones contraindicated. Cephalexin/amoxicillin safe.
9. **B** — Diabetic + gas in renal parenchyma/collecting system = emphysematous pyelonephritis → emergency nephrectomy + broad antibiotics.
10. **C** — Antibiotic prophylaxis at catheter insertion NOT recommended (promotes resistance). Daily review and prompt removal IS recommended.

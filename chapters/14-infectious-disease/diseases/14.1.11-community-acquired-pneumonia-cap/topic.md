# Community-Acquired Pneumonia (CAP)

Related: [[Hospital-Acquired and Ventilator-Associated Pneumonia]], [[Aspiration Pneumonia]], [[Sepsis and Septic Shock]], [[Tuberculosis (Pulmonary and Extrapulmonary)]], [[Fungal Pneumonias]], [[Viral Pneumonias]]

> [!important]
> **CAP = pneumonia acquired outside hospital (≥48h after admission or <48h if no healthcare contact).** **Severity assessment: CURB-65 (outpatient) / PSI (inpatient) / IDSA/ATS minor/major criteria (ICU).** **Empirical antibiotics by severity and risk factors.** **Legionella urinary antigen + sputum culture + blood cultures before antibiotics.** **Corticosteroids in severe CAP (if no contraindication).** **Switch to oral at clinical stability (usually day 3–5).** **Total 5–7 days treatment.**

## Learning Objectives
- Define CAP and distinguish from HAP/VAP/HCAP
- Apply severity scores: CURB-65, PSI, IDSA/ATS minor/major criteria
- Select empirical antibiotics by severity, risk factors, and local epidemiology
- Order appropriate investigations (Legionella UA, sputum Cx, blood Cx, CXR, lactate)
- Determine site of care (outpatient, ward, ICU)
- Manage complications: empyema, parapneumonic effusion, lung abscess, septic shock, ARDS
- Apply switch criteria (IV to oral) and treatment duration
- Recognise atypical pathogens and when to test/treat

## Definitions
| Term | Definition |
|------|------------|
| **CAP** | Acute lower respiratory tract infection acquired in community (≥48h after admission if no healthcare contact) |
| **HAP** | Pneumonia ≥48h after hospital admission, not incubating at admission |
| **VAP** | Pneumonia ≥48h after endotracheal intubation |
| **HCAP** (retired concept) | Healthcare-associated pneumonia (nursing home, dialysis, recent hospitalisation) — now assessed by risk factors within CAP |

## Aetiology by Setting
| Setting | Typical | Atypical | Viral | Other |
|---------|---------|----------|-------|-------|
| **Outpatient CAP** | *S. pneumoniae* (most common), *H. influenzae*, *M. catarrhalis* | *M. pneumoniae*, *C. pneumoniae*, *Legionella* | Influenza, RSV, SARS-CoV-2 | |
| **Inpatient Ward CAP** | *S. pneumoniae*, *H. influenzae*, *S. aureus*, Enterobacteriaceae | *Legionella*, *M. pneumoniae*, *C. pneumoniae* | Influenza, RSV, SARS-CoV-2 | |
| **ICU CAP** | *S. pneumoniae*, *S. aureus* (MSSA/MRSA), *Pseudomonas*, Enterobacteriaceae | *Legionella* | Influenza, SARS-CoV-2 | *Acinetobacter* (tropical) |

> [!tip]
> ***S. pneumoniae* = #1 cause overall.** ***Legionella* → severe CAP, summer/autumn, GI symptoms, hyponatraemia.** ***M. pneumoniae* → young adults, gradual onset, extrapulmonary.** ***C. pneumoniae* → mild, persistent cough.** **Viral co-infection common (influenza + bacterial).**

## Clinical Features
| Feature | Typical (Pneumococcal) | Atypical (Mycoplasma/Chlamydia/Legionella) |
|---------|------------------------|--------------------------------------------|
| **Onset** | Acute (hours) | Subacute (days) |
| **Fever** | High, rigors | Low-grade to high (Legionella high) |
| **Cough** | Productive, rusty sputum | Dry, non-productive (Legionella may become productive) |
| **Chest pain** | Pleuritic | Less common |
| **Dyspnoea** | Common if extensive | Variable |
| **Extrapulmonary** | Rare | **Headache, myalgia, GI (Legionella), rash (Mycoplasma), haemolysis (Mycoplasma)** |
| **Exam** | Consolidation signs | Often minimal signs vs CXR |

## Severity Assessment — Site of Care Decisions
### CURB-65 (Outpatient / Ward Triage)
| Component | Points |
|-----------|--------|
| **C**onfusion (AMT ≤8 / new disorientation) | 1 |
| **U**rea >7 mmol/L (BUN >19 mg/dL) | 1 |
| **R**espiratory rate ≥30/min | 1 |
| **B**P: SBP <90 or DBP ≤60 mmHg | 1 |
| Age **≥65** years | 1 |

| Score | Mortality | Site of Care |
|-------|-----------|--------------|
| 0–1 | <3% | **Outpatient** |
| 2 | ~9% | **Hospital admission (ward)** |
| 3–5 | 15–40% | **ICU consideration** (assess IDSA/ATS criteria) |

### IDSA/ATS Minor Criteria (for ICU Admission — ≥3 = Severe CAP)
| Criterion | Threshold |
|-----------|-----------|
| Respiratory rate | ≥30/min |
| PaO₂/FiO₂ | ≤250 |
| Multilobar infiltrates | Yes |
| Confusion / disorientation | Yes |
| Uremia (BUN) | ≥20 mg/dL (7 mmol/L) |
| Leukopenia | WCC <4×10⁹/L |
| Thrombocytopenia | Platelets <100×10⁹/L |
| Hypothermia | Core temp <36°C |
| Hypotension requiring fluid resuscitation | Yes |

### IDSA/ATS Major Criteria (Either = Severe CAP / ICU)
- **Invasive mechanical ventilation**
- **Septic shock requiring vasopressors**

> [!critical]
> **ICU admission if: 1 Major OR ≥3 Minor criteria.** **CURB-65 3–5 = consider ICU but use IDSA/ATS criteria for final decision.**

## Investigations
| Test | Indication |
|------|------------|
| **CXR** | **All suspected CAP** (confirmation, multilobar, effusion, cavitation) |
| **Blood cultures** | **Inpatient (ward + ICU)** — before antibiotics |
| **Sputum Gram stain + culture** | **Inpatient (ward + ICU)** — before antibiotics; expectorated > induced |
| **Legionella urinary antigen** | **All inpatients** (serogroup 1 only; rapid, specific) |
| **Pneumococcal urinary antigen** | Inpatients (optional; less specific if colonised) |
| **Respiratory viral PCR panel** | Influenza season, ICU, immunocompromised |
| **ABG** | If SpO₂ <94%, RR ≥30, confusion, lactate ↑ |
| **Lactate** | If sepsis suspected / CURB-65 ≥2 |
| **PCT / CRP** | Support diagnosis, guide antibiotic duration (PCT-guided de-escalation) |
| **HIV test** | All adults with CAP (opt-out) |
| **CT chest** | If CXR unclear, suspected abscess/empyema, immunocompromised |

> [!tip]
> **Legionella UA detects serogroup 1 only (80–90% of cases); negative doesn't exclude.** **Sputum culture yield higher if good sample (<10 squamous epithelial cells/LPF).**

## Empirical Antibiotic Therapy
### Outpatient (CURB-65 0–1, No Comorbidities)
| Regimen | Dose | Duration |
|---------|------|----------|
| **Amoxicillin** | 1g PO 8h | 5 days |
| **Doxycycline** (if penicillin allergy) | 200mg PO stat then 100mg 12h | 5 days |
| **Clarithromycin** (if atypical suspected) | 500mg PO 12h | 5 days |

### Outpatient WITH Comorbidities (COPD, Heart Failure, Diabetes, Liver Disease, Alcohol, Steroids, Asplenia) OR Recent Antibiotics
| Regimen | Dose | Duration |
|---------|------|----------|
| **Amoxicillin + Clarithromycin** | 1g 8h + 500mg 12h | 5–7 days |
| **OR Doxycycline monotherapy** | 200mg stat then 100mg 12h | 5–7 days |
| **OR Levofloxacin** | 500mg PO OD | 5–7 days |

### Inpatient Ward (Non-ICU, CURB-65 ≥2 or Severity Indicators)
| Regimen | Dose | Duration |
|---------|------|----------|
| **Ceftriaxone 2g IV 12h + Clarithromycin 500mg IV/PO 12h** | Standard | 5–7 days (switch oral day 3–5) |
| **OR Levofloxacin 500mg IV/PO OD** | Monotherapy (covers typical + atypical) | 5–7 days |
| **Penicillin allergy (anaphylaxis)** | Levofloxacin 500mg IV/PO OD | 5–7 days |

### ICU (Severe CAP — IDSA/ATS Criteria)
| Regimen | Dose | Duration |
|---------|------|----------|
| **Ceftriaxone 2g IV 12h + Azithromycin 500mg IV/PO OD** | Preferred (macrolide immunomodulation) | 5–7 days |
| **OR Ceftriaxone 2g 12h + Levofloxacin 750mg IV OD** | If macrolide contraindicated | 5–7 days |
| **PLUS MRSA risk (prior MRSA, necrotising, influenza)** | **+ Vancomycin 15–20mg/kg IV 6h** | |
| **PLUS Pseudomonas risk (bronchiectasis, steroids >10mg, broad abx recent)** | **Replace Ceftriaxone → Pip-tazo 4.5g 6h OR Meropenem 1g 8h OR Ceftazidime 2g 8h** | |

> [!key]
> **ICU: β-lactam + macrolide (ceftriaxone + azithromycin) = BEST OUTCOME.** **Add vancomycin/linezolid if MRSA risk. Use antipseudomonal β-lactam if Pseudomonas risk.**

### Special Additions
| Risk Factor | Add |
|-------------|-----|
| **Legionella suspected** (severe, GI symptoms, hyponatraemia, summer/autumn, travel) | **Azithromycin 500mg IV/PO OD** (or Levofloxacin) — macrolide/fluoroquinolone essential |
| **Influenza co-infection** | **Oseltamivir 75mg PO 12h ×5d** (start ASAP, even >48h if severe) |
| **Aspiration** | **Add Metronidazole 500mg IV 8h** (anaerobes) |
| **Cavitation / Lung abscess** | **Prolonged course 2–4 weeks + Metronidazole/Clindamycin** |

## IV to Oral Switch Criteria (Clinical Stability)
| Criteria | Threshold |
|----------|-----------|
| **Temperature** | ≤37.8°C for 24h |
| **Heart rate** | ≤100/min |
| **Respiratory rate** | ≤24/min |
| **SBP** | ≥90 mmHg |
| **SpO₂** | ≥90% on room air |
| **Mental status** | Oriented, no new confusion |
| **Ability to tolerate oral** | Yes |

> [!tip]
> **Usually achieved by day 3–5.** **Switch to same oral counterparts (ceftriaxone → amoxicillin/co-amoxiclav OR levofloxacin PO).** **Discharge when stable + oral tolerated.**

## Treatment Duration
| Scenario | Duration |
|----------|----------|
| **Uncomplicated CAP** | **5 days** (if clinically stable at day 3–5) |
| **Severe CAP / ICU** | **5–7 days** (extend if slow response) |
| **MRSA / Pseudomonas** | **7 days** |
| **Lung abscess / Cavitation** | **2–4 weeks** |
| **Empyema** | **2–4 weeks post-drainage** |

> [!important]
> **Shorter courses (5 days) non-inferior to 7–10 days in mild-moderate CAP.** **PCT-guided: stop if PCT <0.25 or >80% drop.**

## Corticosteroids in Severe CAP
| Indication | **Severe CAP (ICU) without contraindication** |
|------------|-----------------------------------------------|
| **Regimen** | **Dexamethasone 5mg IV 6h ×5 days** (or Hydrocortisone 50mg 6h) |
| **Evidence** | Reduces mortality, ARDS, time to clinical stability (CAPE COD, DEXA-CAP trials) |
| **Contraindications** | Active GI bleed, uncontrolled diabetes, immunosuppression, fungal/viral pneumonia (except influenza) |

## Complications
| Complication | Features | Management |
|--------------|----------|------------|
| **Parapneumonic effusion** | Exudative, neutrophilic, pH >7.2, glucose >3.4, LDH >1000 | Antibiotics alone if uncomplicated |
| **Empyema** | Pus in pleural space, pH <7.2, glucose <3.4, LDH >1000 | **Chest drain + antibiotics ± fibrinolytics (tPA/DNase) ± surgery (VATS/decortication)** |
| **Lung abscess** | Cavitation with air-fluid level, foul sputum | **Prolonged antibiotics 2–4w (clindamycin or metronidazole + β-lactam) ± drainage** |
| **Septic shock** | See Sepsis note | Sepsis bundle |
| **ARDS** | PaO₂/FiO₂ <300, bilateral infiltrates | Low Vt, PEEP, prone, dexamethasone |
| **Necrotising pneumonia** | Multiple cavities, rapid progression | Broad antibiotics ± surgery |

## Pathogen-Specific Pearls
| Pathogen | Clues | Specific Rx |
|----------|-------|-------------|
| **S. pneumoniae** | Rusty sputum, lobar consolidation, high CURB-65 | Penicillin/Amoxicillin if susceptible; Ceftriaxone if resistant |
| **Legionella** | Severe CAP, GI symptoms (diarrhoea), **hyponatraemia**, elevated LFTs, summer/autumn, travel, **Legionella UA +ve** | **Azithromycin 500mg OD OR Levofloxacin 750mg OD** (macrolide/fluoroquinolone MANDATORY) |
| **M. pneumoniae** | Young adults, gradual, **dry cough**, headache, rash (erythema multiforme), cold agglutinins | Doxycycline OR Macrolide (resistance ↑) |
| **C. pneumoniae** | Mild, persistent cough, wheeze | Doxycycline OR Macrolide |
| **S. aureus** (post-influenza) | Rapid progression, cavitation, empyema, haemoptysis | **Flucloxacillin/Cefazolin (MSSA) + Clindamycin (toxin)**; **Vancomycin/Linezolid (MRSA)** |
| **Pseudomonas** | Bronchiectasis, steroids, recent broad abx, structural lung disease | **Pip-tazo/Meropenem/Ceftazidime + Fluoroquinolone** |
| **Klebsiella** | Alcoholics, diabetics, "currant jelly" sputum, cavitation | Ceftriaxone/Cefepime/Meropenem |
| **Influenza** | Seasonal, myalgia, headache, dry cough, bilateral infiltrates | **Oseltamivir 75mg 12h ×5d** (start ASAP) + bacterial cover |

## FCPS/MRCP High-Yield Points
- **CAP = community-acquired; HAP ≥48h admission; VAP ≥48h intubation**
- **CURB-65: 0–1 outpatient, 2 ward, 3–5 consider ICU**
- **ICU: 1 Major (ventilation/vasopressors) OR ≥3 Minor criteria (IDSA/ATS)**
- **Investigations: CXR all; blood cx + sputum cx + Legionella UA + PCT inpatients; viral PCR seasonal**
- **Empirical antibiotics by severity:**
  - Outpatient: Amoxicillin (add macrolide if comorbidities)
  - Ward: Ceftriaxone + Clarithromycin OR Levofloxacin
  - ICU: **Ceftriaxone + Azithromycin** (add Vancomycin if MRSA risk; antipseudomonal β-lactam if Pseudomonas risk)
- **Legionella: severe CAP, GI symptoms, hyponatraemia → Azithromycin/Levofloxacin mandatory**
- **IV→Oral switch at clinical stability (usually day 3–5)**
- **Duration: 5 days uncomplicated; 5–7 days severe; 2–4w cavitation/empyema**
- **Corticosteroids: Dexamethasone 5mg 6h ×5d in severe CAP (no contraindications)**
- **Influenza: Oseltamivir even >48h if severe + bacterial cover**
- **Empyema: pH<7.2, glucose<3.4 → chest drain + fibrinolytics ± VATS**

## Common Viva Questions
1. **What is CURB-65 and how does it guide site of care?** Confusion, Urea>7, RR≥30, BP<90/60, Age≥65. 0–1=outpatient, 2=ward, 3–5=consider ICU.
2. **What are IDSA/ATS major and minor criteria for severe CAP?** Major: invasive ventilation, septic shock. Minor: RR≥30, PaO₂/FiO₂≤250, multilobar, confusion, BUN≥20, WCC<4, platelets<100, temp<36, hypotension needing fluids. ≥3 minor = severe.
3. **Empirical regimen for ICU CAP?** Ceftriaxone + Azithromycin (+ Vancomycin if MRSA risk, antipseudomonal β-lactam if Pseudomonas risk).
4. **When do you add azithromycin/levofloxacin for Legionella?** Severe CAP, GI symptoms, hyponatraemia, summer/autumn, travel.
5. **IV to oral switch criteria?** Afebrile 24h, HR≤100, RR≤24, SBP≥90, SpO₂≥90% RA, oriented, tolerating oral.
6. **Treatment duration for uncomplicated CAP?** 5 days (if stable at day 3–5).
7. **Dexamethasone in severe CAP: dose and duration?** 5mg IV 6h ×5 days.
8. **Empyema biochemical criteria?** pH<7.2, glucose<3.4 mmol/L, LDH>1000 U/L.
9. **Post-influenza S. aureus pneumonia: specific treatment?** Flucloxacillin/Cefazolin + Clindamycin (toxin suppression).
10. **Legionella urinary antigen limitation?** Detects serogroup 1 only (80–90%); negative does not exclude.

## Common Confusions / Exam Traps
| Confusion | Clarification |
|-----------|---------------|
| HCAP still a separate category | **HCAP retired — assess risk factors within CAP** |
| CURB-65 3–5 = automatic ICU | **Use IDSA/ATS criteria for ICU decision; CURB-65 3–5 = consider ICU** |
| Macrolide monotherapy for inpatient | **Inpatient: combination (β-lactam + macrolide) OR respiratory fluoroquinolone monotherapy** |
| Legionella UA rules out Legionella if negative | **Only detects serogroup 1 (80–90%); false negatives occur** |
| Sputum culture before antibiotics not needed | **Inpatient: blood cx + sputum cx BEFORE antibiotics** |
| 7–10 days always needed | **5 days non-inferior for uncomplicated CAP** |
| Dexamethasone for all CAP | **Only SEVERE CAP (ICU) without contraindications** |
| Fluoroquinolone monotherapy for ICU | **ICU: β-lactam + macrolide PREFERRED (better outcomes); fluoroquinolone alternative** |
| MRSA cover for all severe CAP | **Only if risk factors: prior MRSA, necrotising, influenza, cavitation** |
| Pseudomonas cover for all ICU CAP | **Only if risk: bronchiectasis, steroids >10mg, recent broad abx, structural lung disease** |

## Mnemonics
- **CURB-65**: **C**onfusion, **U**rea>7, **R**R≥30, **B**P<90/60, **65** years
- **IDSA MINOR**: **R**R, **P**aO₂/FiO₂, **M**ultilobar, **C**onfusion, **U**remia, **L**eukopenia, **T**hrombocytopenia, **H**ypothermia, **H**ypotension (≥3 = severe)
- **CAP EMPIRICAL**: **O**utpatient: Amox (±Clari); **W**ard: Ceftriaxone + Clari (or Levo); **ICU**: Ceftriaxone + Azithro (+Vanco if MRSA, anti-Pseudo if risk)
- **LEGIONELLA**: **S**evere, **G**I symptoms, **H**yponatraemia, **S**ummer, **T**ravel → **A**zithro/Levo **MANDATORY**
- **SWITCH ORAL**: **A**febrile, **H**R≤100, **R**R≤24, **S**BP≥90, **S**pO₂≥90%, **A**lert, **O**ral able
- **EMPYEMA**: **p**H<7.2, **G**lucose<3.4, **L**DH>1000 → **D**rain + **F**ibrinolytics ± **S**urgery

## Summary
**Community-acquired pneumonia (CAP)** = acute LRTI acquired outside hospital. **Aetiology:** *S. pneumoniae* (most common), *H. influenzae*, *M. catarrhalis*; atypicals: *Legionella* (severe, GI symptoms, hyponatraemia), *M. pneumoniae* (young adults), *C. pneumoniae*; viruses: influenza, RSV, SARS-CoV-2. **Severity:** **CURB-65** (Confusion, Urea>7, RR≥30, BP<90/60, Age≥65): 0–1 outpatient, 2 ward, 3–5 consider ICU. **IDSA/ATS criteria for ICU:** **Major** = invasive ventilation or septic shock with vasopressors; **Minor** = RR≥30, PaO₂/FiO₂≤250, multilobar, confusion, BUN≥20, WCC<4, platelets<100, temp<36, hypotension needing fluids — **≥3 minor = severe CAP**. **Investigations:** CXR all; inpatients: blood cultures, sputum culture, **Legionella urinary antigen** (serogroup 1 only), PCT, viral PCR. **Empirical antibiotics:** **Outpatient** = Amoxicillin (+ Clarithromycin if comorbidities); Penicillin allergy = Doxycycline. **Ward** = Ceftriaxone 2g 12h + Clarithromycin 500mg 12h OR Levofloxacin 500mg OD. **ICU** = **Ceftriaxone 2g 12h + Azithromycin 500mg OD** (+ Vancomycin if MRSA risk; **antipseudomonal β-lactam** — Pip-tazo/Meropenem/Ceftazidime — if Pseudomonas risk). **Legionella suspected** → macrolide/fluoroquinolone **mandatory**. **IV→Oral switch** at clinical stability (afebrile 24h, HR≤100, RR≤24, SBP≥90, SpO₂≥90% RA, oriented, oral able) — usually day 3–5. **Duration:** 5 days uncomplicated; 5–7 days severe; 2–4 weeks cavitation/empyema. **Dexamethasone 5mg IV 6h ×5 days** in severe CAP without contraindications. **Empyema:** pH<7.2, glucose<3.4 mmol/L, LDH>1000 → chest drain + fibrinolytics (tPA/DNase) ± VATS.

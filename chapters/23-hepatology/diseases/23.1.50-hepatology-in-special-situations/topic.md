## 1. Liver Disease in Pregnancy

### Acute Fatty Liver of Pregnancy (AFLP)
| Feature | Detail |
|---------|--------|
| **Incidence** | **1:10,000-15,000** pregnancies |
| **Timing** | **Third trimester** (typically 34-38 weeks) |
| **Pathophysiology** | **Mitochondrial fatty acid oxidation defect** (fetal LCHAD deficiency → maternal accumulation of toxic metabolites) |
| **Swansea Criteria** (≥6 = AFLP) | Nausea/vomiting, abdominal pain, polydipsia/polyuria, encephalopathy, bilirubin >14 µmol/L, hypoglycaemia <4 mmol/L, uric acid >340 µmol/L, leukocytosis >11×10⁹/L, ascites/bright liver on US, PT >14s, AST >120 U/L, ALT >120 U/L, ammonia >100 µmol/L, renal impairment (Cr >150 µmol/L) |
| **Key Labs** | **Low glucose** (hypoglycaemia), **elevated uric acid**, **leukocytosis**, **prolonged PT**, **elevated AST/ALT (mild-moderate)**, **ammonia ↑**, **renal impairment** |
| **Differential** | **HELLP**, **Severe pre-eclampsia**, **Acute viral hepatitis**, **Drug-induced liver injury** |
| **Management** | **Emergency delivery** (regardless of gestation) — **Caesarean section** if maternal/fetal compromise; **ICU support** (glucose, coagulopathy correction, renal support); **Multidisciplinary** (obstetrics, hepatology, intensive care, neonatology) |
| **Prognosis** | **Maternal mortality 10-20%** (improved with early recognition); **Recurrence risk ~25%** in subsequent pregnancies |

### HELLP Syndrome
| Component | Criteria |
|-----------|----------|
| **H**aemolysis | **LDH >600 U/L**, **Schistocytes**, **Hb ↓**, **Low haptoglobin** |
| **E**levated **L**iver enzymes | **AST/ALT >120 U/L** (typically 2-10× ULN) |
| **L**ow **P**latelets | **<100 ×10⁹/L** |

| Classification (Mississippi) | Platelets | AST/ALT | LDH |
|-----------------------------|-----------|---------|-----|
| **Class I** | <50 ×10⁹/L | >120 | >600 |
| **Class II** | 50-100 ×10⁹/L | >120 | >600 |
| **Class III** | 100-150 ×10⁹/L | >120 | >600 |

| Feature | AFLP vs HELLP |
|---------|---------------|
| **Liver Enzymes** | AFLP: Mild-moderate (2-10×); **HELLP: Higher (often >20×)** |
| **Coagulopathy** | **AFLP: Severe (DIC common)**; HELLP: Platelets low, PT mild |
| **Hypoglycaemia** | **AFLP: Common (↓ gluconeogenesis)**; HELLP: Rare |
| **Renal Impairment** | **AFLP: Common**; HELLP: Less common |
| **Ammonia** | **AFLP: ↑↑** (encephalopathy); HELLP: Normal |
| **Uric Acid** | **AFLP: Markedly ↑** (>340 µmol/L); HELLP: Normal/mild ↑ |
| **Timing** | AFLP: **Late 3rd trimester (34-38 wk)**; HELLP: **24-36 wk** |

### Intrahepatic Cholestasis of Pregnancy (ICP)
| Feature | Detail |
|---------|--------|
| **Incidence** | **0.5-1%** pregnancies (higher in South Asian, South American, Scandinavian) |
| **Timing** | **Late 2nd / 3rd trimester** (pruritus → LFTs) |
| **Pathophysiology** | **Hormonal (oestrogen/progesterone) + genetic (ABCB4, ABCB11)** → impaired bile acid transport |
| **Clinical** | **Pruritus (palms/soles, worse at night)**, **No rash** (excoriations only); **Dark urine** |
| **LFTs** | **Bile acids >40 µmol/L** (diagnostic), **ALT/AST mildly ↑** (2-10×), **Bilirubin normal/mild ↑** |
| **Risks to Fetus** | **Stillbirth** (risk ↑ with bile acids >100 µmol/L), **Preterm delivery**, **Meconium staining**, **RDS** |
| **Management** | **Ursodeoxycholic Acid (UDCA) 15-20mg/kg/day** (impr. pruritus, ↓ bile acids); **Vitamin K** (if PT prolonged); **Delivery at 36-37 weeks** (if bile acids >40); **>100 µmol/L → deliver 35-36 wks** |
| **Postpartum** | **Resolution within 2-4 weeks**; **Recurrence risk 60-90%** in subsequent pregnancies |

### Viral Hepatitis in Pregnancy

| Virus | Vertical Transmission | Management |
|-------|----------------------|------------|
| **HBV** | **10-90%** (HBeAg+ high risk) | **TDF 300mg OD from 28 wks** if HBV DNA >200,000 IU/mL; **HBIG + HBV vaccine at birth** (within 12h) |
| **HCV** | **5-6%** (higher if HIV co-infection) | **No DAA in pregnancy** (safety not established); **Postpartum DAA**; Breastfeeding safe |
| **HEV** | **High (50%+**, severe in 3rd trimester) | **Supportive**; **Ribavirin CONTRAINDICATED**; High maternal mortality (15-25%) |

### Vaccination in Pregnancy
| Vaccine | Timing | Safety |
|---------|--------|--------|
| **Hepatitis B** | Any trimester | **Safe** (recombinant) |
| **Hepatitis A** | If high risk | **Safe** (inactivated) |

## 2. Liver Disease in HIV

### HIV-HBV Co-infection
| Aspect | Detail |
|--------|--------|
| **Prevalence** | **5-10%** of HIV+ globally |
| **Natural History** | **Faster fibrosis progression**, higher cirrhosis/HCC risk, lower spontaneous clearance |
| **ART & HBV** | **TDF/FTC** or **TAF/FTC** backbone covers both HIV & HBV |
| **HBV Reactivation** | Risk if **stopping ART** (especially if on TDF/FTC/TAF) |
| **Lamivudine Monotherapy** | **Avoid** (high resistance) |

### HIV-HCV Co-infection
| Aspect | Detail |
|--------|--------|
| **Prevalence** | **20-30%** of HIV+ (higher in PWID) |
| **Fibrosis Progression** | **2-3x faster** than HCV mono-infection |
| **DAA Treatment** | **Same as HCV mono** (SOF/VEL, GLE/PIB, SOF/VEL/VOX); **Check DDI with ART** (e.g., SOF/VEL with boosted PIs — use GLE/PIB) |
| **SVR Rates** | **>95%** (similar to mono-infection) |

### ART Hepatotoxicity
| ART Class | Key Drugs | Toxicity Pattern | Monitoring |
|-----------|-----------|------------------|------------|
| **NRTIs** | Stavudine (d4T), Didanosine (ddI), Zidovudine | **Microvesicular steatosis, lactic acidosis** (mitochondrial toxicity) | LFTs, Lactate, Symptoms q3-6mo |
| **NNRTIs** | **Nevirapine** (hypersensitivity, MHC II), Efavirenz (mild) | **Immune-mediated hepatitis** (Nevirapine: CD4>250 women/>400 men = avoid) | LFTs q2wks ×2mo, then q3mo |
| **PIs** | Ritonavir-boosted (ATAZ/r, DRV/r, LPV/r) | **Cholestatic**, hyperlipidaemia, hyperglycaemia | LFTs, Lipids, Glucose q3-6mo |
| **Integrase Inhibitors** | Raltegravir, Dolutegravir, Bictegravir | **Generally safe**; rare hepatotoxicity | Routine LFTs |

### Immune Reconstitution Inflammatory Syndrome (IRIS)
| Feature | Detail |
|---------|--------|
| **Timing** | **2-12 weeks** after ART initiation |
| **Mechanism** | Immune recovery → exaggerated response to HBV/HCV/other antigens |
| **Presentation** | **Flare of hepatitis** (ALT ↑↑), worsening LFTs, jaundice |
| **Management** | **Continue ART**; **Prednisolone 0.5-1mg/kg** if severe; **Do not stop ART** |

## 3. Liver Disease in the Elderly

### Physiological Changes
| Change | Clinical Impact |
|--------|-----------------|
| **Reduced Liver Volume** | ↓ 20-30% by age 80 |
| **Reduced Hepatic Blood Flow** | ↓ 30-40% (↓ first-pass metabolism) |
| **Reduced Phase I Metabolism** (CYP450) | **↑ Drug levels** (CYP3A4, CYP2D6, CYP2C9) |
| **Phase II Metabolism** (Conjugation) | **Relatively preserved** |
| **Reduced Albumin** | ↓ Binding → ↑ free drug fraction |
| **Reduced Regenerative Capacity** | Slower recovery from injury |

### Drug Dosing Principles in Elderly
| Principle | Application |
|-----------|-------------|
| **Start Low, Go Slow** | Reduce initial dose by 25-50% |
| **Avoid CYP450 Substrates** if possible | Prefer renal excretion / Phase II drugs |
| **Monitor LFTs** | Baseline + q3-6mo if chronic therapy |
| **Beers Criteria** | Avoid high-risk drugs (e.g., amiodarone, methotrexate high-dose) |

### Common Drug Adjustments
| Drug | Adjustment |
|------|------------|
| **Warfarin** | ↓ Dose (↑ sensitivity) |
| **Benzodiazepines** | ↓ Dose (lorazepam/oxazepam preferred — no active metabolites) |
| **Opioids** | ↓ Dose (morphine → oxycodone/alfentanil) |
| **Statins** | Start low (atorvastatin 10mg, rosuvastatin 5mg) |

# Autoimmune Liver Disease

> [!tip] **FCPS/MRCP Priority: HIGH**
> **Autoimmune Liver Disease = AIH, PBC, PSC + Overlap Syndromes** — AIH simplified criteria (ANA/SMA/LKM1, IgG, histology), PBC (AMA, ALP, UDCA 13-15mg/kg, OCA, response criteria), PSC (MRCP beading, IBD link, cholangiocarcinoma surveillance), Overlap syndromes (Paris criteria), IgG4-related cholangitis.

---

## Learning Objectives
By the end of this note you should be able to:
- [ ] Apply **IAIHG simplified criteria** for AIH diagnosis (autoantibodies, IgG, histology, exclusion)
- [ ] Distinguish **AIH Type 1 vs Type 2** and manage with steroids ± azathioprine
- [ ] Diagnose **PBC** (AMA, ALP, histology) and apply **treatment response criteria** (Paris, Toronto, Barcelona)
- [ ] Diagnose **PSC** (MRCP beading, IBD association) and manage surveillance (cholangiocarcinoma)
- [ ] Identify **Overlap Syndromes** (AIH-PBC, AIH-PSC, PBC-PSC) using Paris criteria
- [ ] Recognise **IgG4-related sclerosing cholangitis** (response to steroids)

## 1. Autoimmune Hepatitis (AIH)

### Simplified IAIHG Diagnostic Criteria (Score ≥6 = Probable, ≥7 = Definite)

| Parameter | Cut-off | Points |
|-----------|---------|--------|
| **ANA or SMA** | ≥1:40 | 1 |
| | ≥1:80 | 2 |
| **LKM-1** | ≥1:40 | 2 |
| **IgG** | >ULN | 1 |
| | >1.1×ULN | 2 |
| **Histology (Interface hepatitis)** | Compatible | 1 |
| | Typical | 2 |
| **Exclusion of Viral Hepatitis** | Yes | 2 |

| Score | Diagnosis |
|-------|-----------|
| **≥7** | **Definite AIH** |
| **6** | **Probable AIH** |
| <6 | Not AIH |

### AIH Type 1 vs Type 2

| Feature | Type 1 (Classic) | Type 2 |
|---------|------------------|--------|
| **Autoantibodies** | **ANA and/or SMA** (≥1:40) | **LKM-1** (anti-LKM1) |
| **Age/Peak** | Any age (bimodal: young women, elderly) | Children/young adults |
| **Associated Autoimmunity** | Thyroid, coeliac, RA, Sjogren's | Coeliac, T1DM, vitiligo |
| **HLA Association** | DR3, DR4 | DR7, DR3 |
| **Course** | Chronic, relapsing | More severe, rapid cirrhosis |
| **Treatment Response** | Excellent (steroids) | Good but higher relapse |

### Treatment

| Drug | Dose | Duration | Monitoring |
|------|------|----------|------------|
| **Prednisolone** | **30-40mg/day** (or **Budesonide 3mg TDS** if mild/non-cirrhotic) | **Induction**: until ALT/AST normal + IgG normal | FBC, glucose, BP, bone density, eyes |
| **Azathioprine** | **1-2 mg/kg/day** (1.5 mg/kg typical) | **Maintenance** (add at induction or after 2-4 weeks) | TPMT genotype, FBC, LFT q1-3mo |
| **Alternative** | **Mycophenolate (1g BD)**, **Tacrolimus (0.1mg/kg/day)**, **Ciclosporin** | 2nd-line | FBC, LFT, renal, levels |

> **Remission**: Normal ALT/AST, IgG, no autoantibodies → **maintain azathioprine ≥2 years** before taper.
> **Relapse**: Re-induce with steroids + restart azathioprine → lifelong maintenance if >2 relapses.

### AIH in Pregnancy

| Drug | Safety |
|------|--------|
| **Prednisolone** | Safe (low placental transfer) |
| **Azathioprine** | **Safe** (low placental transfer); continue throughout |
| **MMF/Tacrolimus/Ciclosporin** | **Contraindicated** |
| **Ursodeoxycholic acid** | Safe (if cholestatic component) |

## 2. Primary Biliary Cholangitis (PBC)

### Diagnostic Criteria (All 2 of 3 Required)

| Criterion | Finding |
|-----------|---------|
| **1. Biochemistry** | **ALP >1.5× ULN** (persistent >6 months) |
| **2. Autoantibody** | **AMA positive** (M2 subtype, ≥1:40) |
| **3. Histology** (if needed) | **Florid duct lesion** (non-suppurative destructive cholangitis) |

> **AMA-negative PBC** = AMA-negative but ALP>1.5×ULN + compatible histology + ANA (sp100/gp210) + IgM ↑

### Treatment

| Drug | Dose | Indication | Duration |
|------|------|------------|----------|
| **Ursodeoxycholic Acid (UDCA)** | **13-15 mg/kg/day** (divided doses) | **All PBC** (first-line) | **Lifelong** |
| **Obeticholic Acid (OCA)** | **5mg OD → 10mg OD** (max 10mg) | **UDCA inadequate response** (after 12 months) OR intolerance | Lifelong |

### Response Criteria (Assess at 12 Months)

| Criteria | ALP Target | Bilirubin Target | Transaminases |
|----------|------------|------------------|---------------|
| **Paris I** | <3×ULN | ≤1×ULN | <2×ULN |
| **Paris II** | <1.5×ULN | ≤1×ULN | Normal |
| **Toronto** | <1.67×ULN | ≤1×ULN | — |
| **Barcelona** | Normal | Normal | Normal |

> **Inadequate Response** → Add **Obeticholic Acid 5mg OD** (titrate to 10mg if ALP still >1.67×ULN at 3 months, tolerability permitting).

### PBC Complications & Surveillance

| Complication | Surveillance |
|--------------|--------------|
| **Osteoporosis** | **DEXA scan at diagnosis**, then q2-3yrs; Calcium 1g + Vit D 800-1000IU daily |
| **Fat-soluble vitamin deficiency** | Check A, D, E, K annually; supplement if deficient |
| **HCC Surveillance** | **If cirrhosis**: 6-monthly US ± AFP |
| **Portal Hypertension** | Screen for varices if cirrhosis; SBP risk if ascites |

## 3. Primary Sclerosing Cholangitis (PSC)

### Diagnostic Criteria

| Modality | Finding |
|----------|---------|
| **MRCP (Diagnostic)** | **Multifocal strictures + dilatations** ("beading") of intra/extrahepatic bile ducts |
| **ERCP** | **Gold standard** (if MRCP equivocal / therapeutic intervention needed) |
| **Histology** (if needed) | **Periductal "onion-skin" fibrosis** |
| **Autoantibodies** | **p-ANCA (60-80%)**, ANA, SMA; **AMA negative** |
| **IBD Association** | **70-80% UC**; 5-10% Crohn's; **PSC may precede IBD diagnosis** |

> **Small-duct PSC** = Normal MRCP/ERCP + histology only; better prognosis.

### PSC & IBD

| Feature | Detail |
|---------|--------|
| **IBD Type** | **Ulcerative Colitis (80-90%)**, Crohn's (5-10%) |
| **PSC ↑ Risk of** | **Colorectal Cancer** (if UC extent >left colon) |
| **Surveillance** | **Colonoscopy q1-2yrs** (chromoendoscopy + targeted biopsies) |
| **Liver Disease** | PSC may **precede IBD diagnosis** by years |

### PSC Management & Surveillance

| Aspect | Recommendation |
|--------|----------------|
| **UDCA** | **High-dose (28-30mg/kg)** — **No mortality benefit** (may worsen outcomes in some studies); **not routinely recommended** |
| **ERCP** | **Dominant stricture dilation/stenting** (symptomatic, cholangitis, abnormal LFTs) |
| **Cholangiocarcinoma (CCA) Surveillance** | **Annual MRI/MRCP + CA19-9** (sensitivity limited); **ERCP + brush cytology** if dominant stricture |
| **Liver Transplant** | **Only curative option**; 5-yr survival ~85%; **Recurrence ~20-30%** |
| **IBD Surveillance** | **Colonoscopy q1-2yr chromoendoscopy** if UC >left colon |

## 4. Overlap Syndromes

### Paris Criteria (AIH-PBC)

| Syndrome | Criteria (Need 2/3 from each) |
|----------|------------------------------|
| **AIH-PBC** | **AIH**: ALT >5×ULN, IgG >2×ULN or ANA/SMA ≥1:80, Interface hepatitis<br>**PBC**: ALP >2×ULN or AMA+, Florid duct lesion |
| **AIH-PSC** | **AIH**: ALT >5×ULN, IgG >2×ULN or ANA/SMA ≥1:80, Interface hepatitis<br>**PSC**: MRCP beading, IBD |
| **PBC-PSC** | **PBC**: AMA+, ALP↑, Florid duct lesion<br>**PSC**: MRCP beading |

### Treatment of Overlap Syndromes

| Overlap | Treatment |
|---------|-----------|
| **AIH-PBC** | **UDCA 13-15mg/kg + Prednisolone 30-40mg ± Azathioprine** (treat AIH component first) |
| **AIH-PSC** | **Prednisolone + Azathioprine** (UDCA not proven) |
| **PBC-PSC** | **UDCA 13-15mg/kg** + **ERCP for dominant strictures** |

## 4. IgG4-Related Sclerosing Cholangitis (IgG4-SC)

### Diagnostic Criteria (International Consensus)

| Criterion | Finding |
|-----------|---------|
| **1. IgG4** | **Serum IgG4 >135 mg/dL** (≥2×ULN) |
| **2. Imaging** | **MRCP/ERCP**: Diffuse/segmental biliary strictures + pancreatic involvement |
| **3. Histology** (if needed) | **IgG4+ plasma cells >10/HPF** + storiform fibrosis + obliterative phlebitis |
| **4. Response to Steroids** | **Rapid improvement** (LFTs, imaging within 2-4 weeks) |

### Treatment

| Drug | Dose | Duration | Relapse |
|------|------|----------|---------|
| **Prednisolone** | **40mg/day** (0.6mg/kg) | **2-4 weeks induction** then taper over 3-6 months | **Common** (30-50%) → add azathioprine/rituximab |
| **Rituximab** | 375mg/m² weekly ×4 | Induction remission in relapsing/refractory | — |

## 5. FCPS/MRCP High-Yield Summary

| Topic | Key Points |
|-------|------------|
| **AIH Simplified Criteria** | **Autoantibodies (ANA/SMA/LKM1 ≥1:40/80) + IgG >ULN + Interface Hepatitis + Exclude Viral** → ≥6 probable, ≥7 definite |
| **AIH Type 1 vs 2** | **Type 1**: ANA/SMA, young women/elderly; **Type 2**: LKM-1, children, more severe |
| **AIH Treatment** | **Prednisolone 30-40mg/day → taper to azathioprine 1.5mg/kg maintenance ≥2 years** |
| **AIH in Pregnancy** | **Prednisolone + Azathioprine SAFE**; MMF/Tacrolimus CONTRAINDICATED |
| **PBC Diagnosis** | **ALP >1.5×ULN + AMA+ + Florid Duct Lesion** (2 of 3) |
| **PBC Treatment** | **UDCA 13-15mg/kg lifelong**; **OCA 5-10mg** if inadequate response at 12mo |
| **PBC Response Criteria** | **Paris I**: ALP<3×ULN, Bil<1×ULN, AST<2×ULN at 12mo |
| **PSC Diagnosis** | **MRCP beading + IBD (70-80% UC)**; AMA-negative, p-ANCA+ |
| **PSC Surveillance** | **Annual MRI/MRCP + CA19-9** (CCA); **Colonoscopy q1-2yr** (if UC) |
| **Overlap Syndromes** | **Paris Criteria**: AIH-PBC (2/3 AIH + 2/3 PBC), AIH-PSC, PBC-PSC |
| **IgG4-SC** | **IgG4 >135 + Biliary strictures + Steroid-responsive**; Prednisolone 40mg → taper |

## 6. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **AIH Type 1 vs Type 2** | **Type 1: ANA/SMA, young women/elderly**; **Type 2: LKM-1, children, more severe** |
| **PBC vs PSC — AMA, ALP, IBD** | **PBC: AMA+, ALP↑, F duct lesion, female; PSC: AMA-, MRCP beading, IBD (UC 80%), male** |
| **UDCA in PSC** | **High-dose UDCA (28-30mg/kg) — NO mortality benefit, possible harm**; NOT routinely recommended |
| **AIH-PBC vs AIH-PSC Overlap** | **AIH-PBC**: Paris criteria (2 AIH + 2 PBC); **AIH-PSC**: AIH features + PSC MRCP/IBD |
| **IgG4-SC vs PSC** | **IgG4-SC**: IgG4 >135, steroid-responsive, pancreatic involvement; **PSC**: No IgG4, IBD, no steroid response |
| **UDCA in PBC vs PSC** | **PBC**: UDCA 13-15mg/kg FIRST-LINE; **PSC**: UDCA 28-30mg/kg NOT RECOMMENDED routinely |
| **AIH vs DILI** | AIH: Autoantibodies, IgG↑, Interface hepatitis, chronic; DILI: Drug temporal relationship, RUCAM, resolves on withdrawal |
| **PBC vs Autoimmune Cholangitis** | **AMA-negative PBC (autoimmune cholangitis)**: ANA sp100/gp210+, IgM↑, same treatment/response |

**Mnemonic: AUTOIMMUNE-LIVER**
- **A**IH: **Simplified Criteria (Ab, IgG, Histology, Exclude Viral) ≥6/7**
- **U**rsodeoxycholic: **PBC 1st Line (13-15mg/kg), PSC NOT Recommended**
- **T**ype 1 vs 2 AIH: **ANA/SMA (Type 1) vs LKM-1 (Type 2)**
- **O**beticholic Acid: **PBC 2nd Line (Inadequate UDCA Response)**
- **I**gG4-SC: **IgG4 >135 + Steroid Responsive + Pancreas**
- **M**RCP Beading: **PSC (UC 80%, CCA Surveillance)**
- **M**ulti-organ: **IBD (PSC 70-80% UC, Colorectal Ca Surveillance)**
- **U**DCA in PBC: **13-15mg/kg Lifelong; PBC Response Paris I (ALP<3x, Bil<1x, AST<2x)**
- **N**egative AMA PBC: **Autoimmune Cholangitis = ANA sp100/gp210, Same Rx**
- **E**mployee PSC: **No UDCA Benefit; ERCP Dominant Stricture; CCA Surveillance**
- **L**iver Transplant: **All Indications (AIH/PSC Failure, PBC UDCA Fail, CCA Early)**
- **I**gG4: **>135 + Steroid Responsive + Pancreas**
- **V**aricella: **AIH Relapse → Restart Steroids + Azathioprine**
- **R**elapse AIH: **Restart Pred + Azathioprine; >2 Relapses = Lifelong Maintenance**

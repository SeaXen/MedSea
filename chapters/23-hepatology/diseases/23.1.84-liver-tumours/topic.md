## 1. Hepatocellular Carcinoma (HCC)

### Surveillance (Who & How Often)

| Population | Interval | Modality |
|------------|----------|----------|
| **Cirrhosis (all aetiologies)** | **6-monthly** | **US ± AFP** |
| **Chronic HBV (Asian M>40, F>50; African >20; Family history)** | **6-monthly** | US ± AFP |
| **NASH F3 Fibrosis** | **12-monthly** | US ± AFP |
| **Post-SVR HCV with cirrhosis** | **6-monthly** | US ± AFP |

> **AFP cut-off**: **>20 ng/mL** raises suspicion; **>200 ng/mL + mass = HCC diagnosis** (with imaging)

---

## 2. LI-RADS (Liver Imaging Reporting and Data System)

| Category | Definition | HCC Probability | Management |
|----------|------------|-----------------|------------|
| **LR-1** | Definitely benign | <1% | Routine surveillance |
| **LR-2** | Probably benign | <10% | Routine surveillance |
| **LR-3** | Intermediate | 10-50% | Short-interval follow-up (3-6mo) |
| **LR-4** | Probably HCC | >50% | Diagnostic; consider biopsy if uncertainty |
| **LR-5** | **Definitely HCC** | >95% | **Non-invasive diagnosis** (treat) |
| **LR-M** | Probably malignant, not HCC specific | — | Biopsy / MDT |
| **LR-TIV** | Tumour in vein | — | Stage IV (BCLC D) |

### Major LI-RADS Features (for LR-4/LR-5)

| Feature | LR-5 (Definite HCC) | LR-4 (Probable HCC) |
|---------|---------------------|---------------------|
| **Arterial Phase Hyperenhancement (APHE)** | Present | Present |
| **Washout (Portal Venous/Delayed)** | Present | Present or absent |
| **Capsule** | Present | Present or absent |
| **Threshold Growth** | ≥50% size increase in ≤6mo | May be present |
| **Size** | **≥10mm** | **10-19mm** (if APHE only = LR-4) |

---

## 3. BCLC Staging & Treatment Algorithm

| BCLC Stage | Criteria | Treatment Options | Median Survival |
|------------|----------|-------------------|-----------------|
| **0 (Very Early)** | Single <2cm, Child-Pugh A, PS 0 | **Ablation** (RFA/MWA) > Resection > Transplant | >5 years (>70%) |
| **A (Early)** | Single ≤5cm or ≤3 nodules ≤3cm, Child-Pugh A-B, PS 0 | **Resection** (if preserved liver function) / **Transplant** (Milan criteria) / **Ablation** | >5 years (50-70%) |
| **B (Intermediate)** | Multinodular, Child-Pugh A-B, PS 0 | **TACE** (cTACE/DEB-TACE) | 20-30 months |
| **C (Advanced)** | Vascular invasion / Extrahepatic spread / PS 1-2 | **Systemic Therapy** (Atezolizumab+Bevacizumab 1L; Sorafenib/Lenvatinib 2L) | 10-15 months |
| **D (Terminal)** | Child-Pugh C, PS 3-4 | **Best Supportive Care** | <3 months |

### Milan Criteria (Transplant Eligibility)
- **Single tumour ≤5cm** OR **≤3 nodules each ≤3cm**
- **No vascular invasion / extrahepatic spread**
- **Child-Pugh A/B, PS 0**

### Up-to-Seven Criteria (Extended Transplant Criteria)
- **Number of tumours + largest tumour diameter (cm) ≤7**

## 4. Treatment Modalities

| Modality | Indication | Key Details |
|----------|------------|-------------|
| **Resection** | Single tumour, Child-Pugh A, normal bilirubin, no portal hypertension (HVPG <10) | 5-yr OS 50-70%; recurrence 70% at 5yr |
| **Ablation (RFA/MWA)** | **BCLC 0/A** (≤3cm, ≤3 nodules), not resectable | RFA <3cm; MWA larger; 5-yr OS 40-60% |
| **Liver Transplant** | **Milan Criteria** (or Up-to-Seven) | 5-yr OS 70-80%; recurrence <15% |
| **TACE (cTACE/DEB-TACE)** | **BCLC B** (multinodular, preserved liver) | Repeat q4-12wks; median OS 20-30mo |
| **TARE (SIRT/Y-90)** | BCLC B/C, bridge to transplant, portal vein thrombosis | Alternative to TACE |
| **Systemic Therapy (BCLC C)** | **1L: Atezolizumab + Bevacizumab** (IMbrave150); **2L: Sorafenib / Lenvatinib** | Atezolizumab/Bevacizumab: median OS 19.2mo |
| **Sorafenib** | 400mg BD | First TKI; median OS 10.7mo |
| **Lenvatinib** | 12mg (BW≥60kg) / 8mg (BW<60kg) | Non-inferior to sorafenib; median OS 13.6mo |

## 5. Metastatic Liver Disease

### Workup for Unknown Primary

| Step | Investigation |
|------|---------------|
| **1. Imaging** | CT Chest/Abdomen/Pelvis + MRI Liver |
| **2. Endoscopy** | Gastroscopy + Colonoscopy |
| **3. Tumour Markers** | CEA, CA19-9, CA125, AFP, PSA, CA15-3 |
| **4. Biopsy + IHC Panel** | **CK7, CK20, CDX2, TTF-1, Napsin A, GATA3, ER, PR, HER2, PAX8, CDX2, SATB2** |
| **5. Genetic** | NGS panel if CUP (Cancer of Unknown Primary) |

### Common Primary → Liver Mets Patterns

| Primary | Pattern | Key IHC |
|---------|---------|---------|
| **Colorectal** | Multiple, peripheral | CK20+, CDX2+, SATB2+, CK7- |
| **Pancreatic/Biliary** | Multiple, central/hilar | CK7+, CK20+/-, CA19-9+ |
| **Breast** | Multiple, subcapsular | ER+, PR+, GATA3+, HER2 variable |
| **Lung (Adenocarcinoma)** | Multiple, random | TTF-1+, Napsin A+, CK7+ |
| **Neuroendocrine** | Multiple, hypervascular | Chromogranin A, Synaptophysin, Ki-67 |
| **Melanoma** | Variable, often pigmented | S100, HMB45, Melan-A, SOX10 |

## 4. Benign Liver Tumours

| Tumour | Features | Diagnosis | Management |
|--------|----------|-----------|------------|
| **Haemangioma** | Most common; **T2 hyperintense, flash fill** on MRI; peripheral nodular enhancement | MRI (T2 bright, centripetal fill) | **Observe** (asymptomatic); resect if giant >10cm + symptomatic |
| **FNH (Focal Nodular Hyperplasia)** | **Central scar, spoke-wheel arterial enhancement**, isointense on T1/T2; **Gadoxetate uptake** (hepatobiliary phase retention) | MRI (central scar, hepatocyte-specific uptake) | **Observe** (benign); resect if diagnostic uncertainty |
| **Hepatic Adenoma** | **Oral contraceptive association**; **Risk: haemorrhage, malignant transformation** (β-catenin mut); HNF1α, inflammatory, β-catenin subtypes | MRI (fat content, arterial enhancement); biopsy if uncertain | **Stop OCP**; **resect if >5cm, male, β-catenin mut, symptomatic**; surveillance if <5cm female |
| **Regenerative Nodules** | Cirrhosis background; low T1, high T2 | MRI (low T1, high T2) | Part of cirrhosis; not true tumour |

## 6. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **LI-RADS LR-4 vs LR-5** | **LR-5**: APHE + Washout + Capsule (≥10mm); **LR-4**: APHE + (Washout or Capsule) 10-19mm; **LR-3**: APHE only or indeterminate |
| **BCLC A vs B** | A: Single ≤5cm or ≤3 nodules ≤3cm (curative intent); B: Multinodular beyond Milan (TACE) |
| **TACE vs TARE** | **TACE**: Embolisation + chemo (cTACE/DEB); **TARE**: Y-90 radioembolisation (SIRT); TARE for PVT, bridge to transplant |
| **Ablation vs Resection** | **Ablation**: <3cm, not surgical candidate; **Resection**: Child A, no portal hypertension, preserved liver function |
| **Adenoma Subtypes** | **HNF1α**: steatotic, low HCC risk; **Inflammatory**: CRP↑, serum amyloid A; **β-catenin**: **High HCC risk** (resect); **Unclassified** |
| **LI-RADS LR-M** | **Malignant but not HCC** (cholangiocarcinoma, metastases); **LR-TIV**: Tumour in vein (portal/hepatic vein) |
| **AFP in HCC** | >20 = suspicious; >200 + mass = diagnostic (with imaging); not sensitive enough alone for screening |
| **Up-to-Seven vs Milan** | Milan: single ≤5cm or ≤3 nodules ≤3cm; Up-to-7: size + number ≤7 (expanded transplant criteria) |

**Mnemonic: HCC-LI-RADS-BCLC**
- **H**CC Surveillance: **Cirrhosis q6mo US±AFP**
- **C**CR: **Ablation/Resection/Transplant**
- **L**I-RADS: **LR-5 Definite HCC, LR-4 Probable, LR-3 Follow-up, LR-M Non-HCC Malignant**
- **I**maging: **APHE + Washout + Capsule = LR-5**
- **B**CLC: **0/A Curative, B TACE, C Systemic (Atezo/Bev), D BSC**
- **C**urative: **Ablation <3cm, Resection Child A, Transplant Milan**
- **L**iver Mets: **CT CAP + Endoscopy + IHC (CK7/20, CDX2, TTF-1)**
- **B**enign: **Haemangioma (T2 Bright), FNH (Central Scar + Retention), Adenoma (OCP, β-catenin = HCC Risk)**
- **A**FP: **>200 + Mass = HCC Diagnosis**
- **T**ACE: **BCLC B Multinodular**
- **S**ystemic: **Atezo+Bev 1L, Sora/Lenv 2L**
- **R**esection: **Child A, No PH, HVPG<10**
- **A**blation: **RFA/MWA <3cm**

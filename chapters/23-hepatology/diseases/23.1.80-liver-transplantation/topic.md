## 2. Indications for Liver Transplant

### 1. Chronic Liver Disease (MELD-Based)
| Indication | Criteria |
|------------|----------|
| **Decompensated Cirrhosis** | **MELD ≥15** (or ≥12 in Some Regions) |
| **Refractory Ascites** | Despite LVP + Albumin + Diuretics |
| **Recurrent Variceal Bleed** | Despite NSBB + EVL + TIPS |
| **Recurrent HE** | Despite Lactulose + Rifaximin |
| **HRS** | Type 1 or Refractory Type 2 |
| **HPS / PoPH** | Severe (PaO₂ <60 / mPAP >35) |

### 2. Acute Liver Failure (Super-Urgent)
| Indication | Criteria |
|------------|----------|
| **King's College Criteria** | pH <7.3 **OR** (PT >100s + Cr >300 + Grade III/IV HE) |
| **Non-Paracetamol** | Any 3 of: Age <10 or >40, Aetiology Unknown/Drug, Jaundice >7d Before HE, PT >50s, PT >100s |

### 3. HCC (Within Transplant Criteria)
| Criteria | Detail |
|----------|--------|
| **Milan Criteria** | Single ≤5cm **OR** ≤3 Nodules ≤3cm, No Vascular Invasion |
| **UCSF Criteria** (Expanded) | Single ≤6.5cm **OR** ≤3 Nodules ≤4.5cm, Total Diameter ≤8cm |
| **Downstaging** | TACE/TARE → Meet Milan → Transplant |
| **Exception Points** | HCC Within Milan = **MELD Exception Points** (e.g., 28 in US) |

### 4. Metabolic / Genetic
| Indication | Detail |
|------------|--------|
| **Wilson Disease (ALF or Neuro)** | Corrects Metabolic Defect |
| **Alpha-1 AT (Cirrhosis)** | |
| **Familial Amyloid Polyneuropathy** | Liver Produces Mutant TTR |

## 3. MELD Score

### MELD (Original)
```
MELD = 3.78 × ln(Bilirubin mg/dL) + 11.2 × ln(INR) + 9.57 × ln(Creatinine mg/dL) + 6.43
```
- **Bilirubin, INR, Creatinine**: All Capped at 4.0 (Minimum 1.0)
- **Dialysis**: Creatinine = 4.0
- **Range**: 6-40 (Higher = Sicker, Higher Priority)

### MELD-Na (Current Standard)
```
MELD-Na = MELD + 1.32 × (137 - Na) - [0.033 × MELD × (137 - Na)]
```
- **Na Capped**: 125-137 mmol/L
- **Range**: 6-40

> **FCPS/MRCP**: Don't Memorize Formula — Know **Components, Range, and that Higher = Sicker**

### MELD Exceptions (Common)
| Condition | Exception Points |
|-----------|------------------|
| **HCC Within Milan** | **28** (US) / Variable (UK) |
| **HPS (PaO₂ <60)** | **22** |
| **PoPH (mPAP >35, PVR >240)** | **22** |
| **Familial Amyloid Polyneuropathy** | **24** |
| **Primary Oxaluria** | **28** |
| **Recurrent Cholangitis (PSC)** | Case-by-Case |

## 4. Contraindications

### Absolute
| Contraindication | Reason |
|------------------|--------|
| **Active Sepsis** (Uncontrolled) | Immunosuppression Will Worsen |
| **Extrahepatic Malignancy** (<5y Disease-Free) | Recurrence Risk |
| **Advanced Cardiopulmonary Disease** | Perioperative Mortality |
| **AIDS** (CD4 <200, Uncontrolled) | |
| **Active Substance Abuse** (Alcohol <6mo Abstinence) | Recurrence Risk |
| **Severe Psychiatric Illness** (Non-Adherence) | |
| **Portal Vein Thrombosis (Complete, Non-Reconstructible)** | Technical Failure (Partial OK) |

### Relative
| Contraindication | Consideration |
|------------------|---------------|
| **Age >70** | Biological Age > Chronological |
| **Obesity (BMI >35)** | Technical Difficulty, Complications |
| **Severe Malnutrition / Sarcopenia** | Poor Outcomes |
| **Non-Adherence History** | Multidisciplinary Assessment |
| **Limited Social Support** | Requires Caregiver |

## 5. Pre-Transplant Assessment

| Domain | Tests |
|--------|-------|
| **Cardiac** | Echo, Stress Test (If >50 or Risk Factors), Coronary Angio if Indicated |
| **Pulmonary** | CXR, PFTs, ABG, VQ Scan (If HPS Suspected), RHC (If PoPH Suspected) |
| **Renal** | U&E, eGFR, Proteinuria |
| **Infection** | HIV, HBV, HCV, CMV, EBV, VZV, TB (Quantiferon), Syphilis |
| **Malignancy** | Age-Appropriate Screening (Colonoscopy, Mammography, PSA, CT Chest/Abdo/Pelvis) |
| **Psychosocial** | Psychiatry, Social Work, Addiction, Financial, Support System |
| **Nutrition** | BMI, DEXA, Handgrip, Mid-Arm Circumference, Serum Albumin |

## 6. Immunosuppression Regimens

### Induction (Perioperative)
| Agent | Use |
|-------|-----|
| **Basiliximab** (IL-2 Receptor Antagonist) | Standard Induction (2 Doses: Day 0, Day 4) |
| **Thymoglobulin** (ATG) | High Immunologic Risk (Retransplant, High PRA) |
| **Steroids** | IV Methylprednisolone Intra-op + Taper |

### Maintenance (Triple Therapy Common)
| Regimen | Components |
|---------|------------|
| **Standard** | **Tacrolimus + MMF + Steroids** (Taper to Off by 3-6mo) |
| **CNI-Sparing** | Tacrolimus + mTOR (Sirolimus/Everolimus) — If Renal Impairment |
| **Steroid-Free** | Tacrolimus + MMF (No Steroids from Day 1) |
| **HCC/Malignancy** | **mTOR-Based** (Sirolimus/Everolimus) — Anti-Proliferative |

### Drug Details

| Drug | Target Level (Maintenance) | Key Side Effects |
|------|----------------------------|------------------|
| **Tacrolimus** | 5-10 ng/mL (Early), 3-7 ng/mL (Late) | Nephrotoxicity, Neurotoxicity (Tremor, HA), Diabetes, HTN, HyperK, Hyperlipidemia |
| **Cyclosporine** | 100-300 ng/mL | Hirsutism, Gingival Hyperplasia, HTN, Nephrotoxicity, Hyperlipidemia |
| **MMF / MPA** | AUC 30-60 mg·h/L | GI (Diarrhea, Nausea), Myelosuppression (Leukopenia), Teratogenic |
| **Sirolimus / Everolimus** | 5-15 ng/mL / 3-8 ng/mL | Impaired Wound Healing, Hyperlipidemia, Proteinuria, Pneumonitis, Thrombocytopenia |
| **Prednisolone** | Taper to 5mg by 3mo, Off by 6-12mo | Diabetes, Osteoporosis, Infection, Weight Gain, Cataracts |

## 1. Gallstone Disease

### Spectrum
| Condition | Features | Management |
|-----------|----------|------------|
| **Asymptomatic Gallstones** | Incidental on imaging | **Observe** (no prophylactic cholecystectomy) |
| **Biliary Colic** | **Episodic RUQ pain**, post-prandial, 30min-6h, no fever/LFT abnormality | **Elective cholecystectomy** |
| **Acute Cholecystitis** | **RUQ pain + fever + Murphy's sign + leukocytosis**; US: wall thickening >3mm, pericholecystic fluid | **Tokyo Guidelines**; **Early cholecystectomy (≤72h)** |
| **Chronic Cholecystitis** | Recurrent colic, thickened wall on US | **Elective cholecystectomy** |

### Tokyo Guidelines 2018 (TG18) — Acute Cholecystitis

| Severity | Criteria | Management |
|----------|----------|------------|
| **Grade I (Mild)** | No organ dysfunction, no extensive inflammation | **Early cholecystectomy (≤72h)** |
| **Grade II (Moderate)** | **Any**: WBC >18k, palpable GB, disease >72h, Marked local inflammation | **Early cholecystectomy (≤72h)**; consider PTGBD if high surgical risk |
| **Grade III (Severe)** | **Organ dysfunction** (cardiovascular, neurological, respiratory, renal, hepatic, haematological) | **Urgent biliary drainage (PTGBD/ERCP)** → delayed cholecystectomy |

### Mirizzi Syndrome
| Type | Mechanism | Management |
|------|-----------|------------|
| **Type I** | Stone impacted in Hartmann's pouch → extrinsic compression of CHD | **Cholecystectomy + CBD exploration** (lap/open) |
| **Type II** | Cholecystocholedochal fistula (erosion into CBD) | **Cholecystectomy + fistula repair + T-tube** |
| **Type III** | Fistula involving CHD proper | **Biliary reconstruction (hepaticojejunostomy)** |
| **Type IV** | Complete destruction of CHD wall | **Hepaticojejunostomy** |

## 2. Choledocholithiasis

### Presentation & Diagnosis
| Feature | Finding |
|---------|---------|
| **Symptoms** | RUQ pain, jaundice, cholangitis, pancreatitis |
| **LFTs** | **Cholestatic pattern** (ALP ↑↑, GGT ↑↑, bilirubin ↑) |
| **Imaging** | **MRCP** (non-invasive, 95% sensitivity) / **ERCP** (therapeutic) / **EUS** (if MRCP equivocal) |
| **Predictors** (ASGE) | **High probability**: CBD stone on imaging, cholangitis, bilirubin >4.3 mg/dL (73 µmol/L), dilated CBD >6mm + stones |

### ERCP Indications (ASGE)
| Priority | Indication |
|----------|------------|
| **Emergent (<24h)** | **Ascending cholangitis**, **Gallstone pancreatitis with cholangitis** |
| **Urgent (24-72h)** | **Gallstone pancreatitis (no cholangitis)**, **Choledocholithiasis with symptoms** |
| **Elective** | **Asymptomatic CBD stone** (incidental), **Biliary strictures** |

## 3. Ascending Cholangitis

### Charcot's Triad vs Reynolds' Pentad
| Component | Charcot's Triad | Reynolds' Pentad (Severe) |
|-----------|-----------------|---------------------------|
| **Fever** | ✓ | ✓ |
| **RUQ Pain** | ✓ | ✓ |
| **Jaundice** | ✓ | ✓ |
| **Altered Mental Status** | | ✓ |
| **Hypotension/Shock** | | ✓ |

> **Charcot's Triad** = **~50-70% sensitive**; **Reynolds' Pentad** = **<10% sensitive** but specific for severe cholangitis

### Tokyo Guidelines 2018 (TG18) — Acute Cholangitis

| Severity | Criteria | Management |
|----------|----------|------------|
| **Grade I (Mild)** | No organ dysfunction | **Antibiotics + Elective ERCP (24-48h)** |
| **Grade II (Moderate)** | **Any**: WBC >12k/<4k, Temp >39°C, Age >75, Hyperbilirubinemia >5mg/dL, Hypoalbuminemia <25g/L | **Antibiotics + Early ERCP (<24h)** |
| **Grade III (Severe)** | **Organ dysfunction**: CV (pressor), Neurological (GCS<14), Respiratory (PaO2/FiO2<300), Renal (Cr>2), Hepatic (INR>1.5), Haematological (Plt<100k) | **Resuscitation → Antibiotics → Emergency ERCP/PTBD (<12h)** |

### Acute Cholangitis Management Algorithm

```mermaid
flowchart TD
    A[Suspected Cholangitis] --> B[Resuscitation: Fluids, Bloods, Blood cultures]
    B --> C[**IV Antibiotics IMMEDIATELY**]
    C --> D{Severity (TG18)}
    D -->|Grade I| E[Elective ERCP 24-48h]
    D -->|Grade II| F[**Early ERCP <24h**]
    D -->|Grade III| G[**ICU + Emergency ERCP/PTBD <12h**]
    C --> H[**Antibiotic Choice**: Piperacillin-tazobactam 4.5g IV q6h OR Ceftriaxone 2g q12h + Metronidazole 500mg q8h]
    F --> I[ERCP: Sphincterotomy + Stone extraction ± Stent]
    G --> I
    E --> I
```

### Antibiotic Regimens

| Severity | Regimen |
|----------|---------|
| **Mild/Moderate** | **Piperacillin-tazobactam 4.5g IV q6h** OR **Ceftriaxone 2g IV q12h + Metronidazole 500mg IV q8h** |
| **Severe/Septic Shock** | **Meropenem 1g IV q8h** OR **Piperacillin-tazobactam 4.5g q6h + Vancomycin** (if MRSA risk) |
| **Duration** | **5-7 days** (step-down to oral if improving) |

## 4. Gallstone Pancreatitis

### Diagnosis
| Criterion | Finding |
|-----------|---------|
| **Clinical** | Epigastric pain radiating to back, nausea/vomiting |
| **Lipase/Amylase** | **>3× ULN** |
| **Imaging** | **CT abdomen** (revised Atlanta classification); **US** for gallstones; **MRCP** for CBD stones |

### Management

| Severity (Atlanta) | Management |
|--------------------|------------|
| **Mild** | Conservative (fluids, analgesia, early feeding) + **Cholecystectomy same admission** |
| **Moderate** | ICU if needed, fluids, monitor for complications |
| **Severe** | ICU, organ support, consider necrosectomy if infected necrosis |

### ERCP Timing in Gallstone Pancreatitis

| Scenario | ERCP Timing |
|----------|-------------|
| **Concurrent Cholangitis** | **Emergent (<24h)** |
| **Severe Pancreatitis + CBD Stone** | **Early (<24-48h)** |
| **Mild/Moderate, No Cholangitis** | **No routine ERCP**; **Cholecystectomy same admission** |
| **CBD Stone on Imaging, No Pancreatitis** | **Elective ERCP** before cholecystectomy |

## 4. Biliary Strictures

### Classification
| Type | Causes | Features |
|------|--------|----------|
| **Benign** | **Post-cholecystectomy (iatrogenic)**, **Post-ERCP**, **Primary sclerosing cholangitis**, **Chronic pancreatitis**, **Ischemic (hepatic artery thrombosis)**, **Trauma**, **Radiation** | Usually single, proximal to injury site |
| **Malignant** | **Cholangiocarcinoma**, **Pancreatic cancer**, **Gallbladder cancer**, **Ampullary carcinoma**, **Metastatic nodes** | Progressive, weight loss, CA19-9 ↑, irregular stricture on imaging |

### Diagnostic Workup
| Step | Investigation |
|------|---------------|
| **1. LFTs** | Cholestatic pattern (ALP ↑↑, GGT ↑↑) |
| **2. Imaging** | **MRCP** (morphology, location, length); **CT CAP** (malignancy staging) |

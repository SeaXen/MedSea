# Drug-Induced Liver Injury (DILI)

> [!tip] **FCPS/MRCP Priority: HIGH**
> **DILI = Major cause of acute liver injury** — Classification: **Intrinsic (dose-dependent)** vs **Idiosyncratic (unpredictable)**; Patterns: **Hepatocellular, Cholestatic, Mixed**; Common culprits: **Paracetamol, Anti-TB, Antibiotics (amoxicillin-clavulanate, flucloxacillin), NSAIDs, ART, Immune checkpoint inhibitors**; **RUCAM scoring**; Management: **Withdraw drug, NAC for paracetamol, supportive, rechallenge principles**.

---

## Learning Objectives
By the end of this note you should be able to:
- [ ] Classify **DILI**: **Intrinsic (dose-dependent, predictable)** vs **Idiosyncratic (unpredictable, immune-mediated)**
- [ ] Identify **injury patterns**: **Hepatocellular, Cholestatic, Mixed** (R-value)
- [ ] Recognize **common culprit drugs** by pattern
- [ ] Apply **RUCAM scoring** for causality assessment
- [ ] Manage **paracetamol toxicity** (NAC regimen)
- [ ] Apply **rechallenge principles** (when safe vs contraindicated)

## 1. Classification

| Type | Mechanism | Features | Examples |
|------|-----------|----------|----------|
| **Intrinsic (Type A)** | **Dose-dependent, predictable**, direct toxin/metabolite | **Reproducible**, short latency, dose-related | **Paracetamol**, Carbon tetrachloride, Amanita phalloides, Niacin (high dose) |
| **Idiosyncratic (Type B)** | **Unpredictable, immune-mediated** (hapten, P-i concept) | **Unpredictable**, variable latency (days-months), **not dose-related**, host factors (HLA, genetics) | **Anti-TB (isoniazid, rifampicin, pyrazinamide), antibiotics (co-amoxiclav, flucloxacillin), NSAIDs, ART, ICI**, herbals |

---

## 2. Injury Patterns (R-Value = ALT/ULN ÷ ALP/ULN)

| Pattern | R-Value | ALT/ALP | Typical Drugs |
|---------|---------|---------|---------------|
| **Hepatocellular** | **R ≥5** | ALT ↑↑ > ALP | **Paracetamol**, Isoniazid, Valproate, Methotrexate, NIACIN, Halothane |
| **Cholestatic** | **R ≤2** | ALP ↑↑ > ALT | **Co-amoxiclav**, Flucloxacillin, **Erythromycin**, Chlorpromazine, **Azathioprine**, **Oral contraceptives**, Anabolic steroids |
| **Mixed** | **2 < R < 5** | Both ↑ | **Anti-TB (rifampicin/pyrazinamide)**, **Amoxicillin-clavulanate**, **Carbamazepine**, **Phenytoin**, **Trimethoprim-sulfamethoxazole**, **Terfenadine** |

---

## 3. Common Culprit Drugs by Pattern

### Hepatocellular (R ≥5)
| Drug | Key Features |
|------|--------------|
| **Paracetamol** | **Dose-dependent**; CYP2E1 → NAPQI → glutathione depletion → centrilobular necrosis; **NAC antidote** |
| **Isoniazid (INH)** | **Dose-independent idiosyncratic** but dose-related risk; latency 1-6 months; slow acetylator risk |
| **Valproate** | **Dose-related** (children <3y polytherapy); mitochondrial toxicity; **hyperammonaemic encephalopathy** |
| **Methotrexate** | **Dose-dependent** (cumulative); fibrosis/cirrhosis; monitor LFTs, FBC, albumin |
| **Niacin (High dose)** | **Sustained-release** > time-release; hepatotoxicity |

### Cholestatic (R ≤2)
| Drug | Key Features |
|------|--------------|
| **Co-amoxiclav** | **Most common cause** of antibiotic DILI; **latency 1-3 weeks**; typically cholestatic; **re-exposure = rapid recurrence** |
| **Flucloxacillin** | **Latency 2-4 weeks**; cholestatic; **elderly >50y** at higher risk; **re-exposure = rapid** |
| **Erythromycin (estolate)** | **Children/young adults**; cholestatic; **hypersensitivity** |
| **Chlorpromazine** | **Phenothiazine**; typically 2-4 weeks; **cholestatic + eosinophilia**; cross-reactivity with other phenothiazines |
| **Azathioprine/6-MP** | **Cholestatic**; rare; **NUDT15/TPMT** variants |
| **Oral Contraceptives** | **Oestrogen component**; benign cholestasis; **disappear after stopping** |

### Mixed (2 < R < 5)
| Drug | Key Features |
|------|--------------|
| **Anti-TB** (Rifampicin, Pyrazinamide, INH) | **Pyrazinamide** = most hepatotoxic; **Rifampicin** = cholestatic + enzyme inducer; **INH** = hepatocellular; monitor LFTs q2wks ×2mo then monthly |
| **Co-amoxiclav** | Often mixed; latency 1-3 weeks |
| **Carbamazepine** | **Hypersensitivity (DRESS)**; latency 2-8 weeks; **HLA-B*15:02** (Asians) |
| **Phenytoin** | **Hypersensitivity**; latency 2-8 weeks; **HLA-B*15:02** |
| **Trimethoprim-Sulfamethoxazole** | **Idiosyncratic**, latency 1-3 weeks; mixed/cholestatic |
| **Allopurinol** | **DRESS/Stevens-Johnson**; **HLA-B*58:01** (Asians); allopurinol hypersensitivity syndrome |

### Special Categories

| Category | Key Drugs | Features |
|----------|-----------|----------|
| **Anti-TB** | Isoniazid, Rifampicin, Pyrazinamide, Ethambutol | Pyrazinamide most hepatotoxic; monitor LFTs baseline, q2wks×2mo, then monthly |
| **ART (HIV)** | NRTIs (steatosis, lactic acidosis), NNRTIs (nevirapine hypersensitivity), PIs (cholestatic) | **Nevirapine**: CD4>250 (women)/>400 (men) → avoid; **Efavirenz** preferred |
| **Immune Checkpoint Inhibitors (ICI)** | Anti-CTLA-4 (ipilimumab), Anti-PD-1 (pembrolizumab/nivolumab) | **Immune-mediated hepatitis**; **Grade 2 (ALT 3-5x): hold ICI + pred 0.5-1mg/kg**; **Grade 3-4 (ALT>5x): high-dose pred 1-2mg/kg + hold ICI** |
| **Herbal/Traditional** | **Germander, Chaparral, Kava, Skullcap, Pyrrolizidine alkaloids** | Often hepatocellular; under-reported |

## 2. RUCAM (Roussel Uclaf Causality Assessment Method)

| Domain | Score |
|--------|-------|
| **Time to onset** | +2 (5-90d), +1 (<5d or >90d hepatocellular; <5d or >90d cholestatic), -2 (incompatible) |
| **Course after cessation** | +3 (decrease ≥50% in 8d hepatocellular; 30d cholestatic), +1 (decrease ≥50% in 30d hepatocellular; 60d cholestatic), 0 (no info), -2 (increase) |
| **Risk factors** | +1 (Age ≥55), +1 (Alcohol), +1 (Pregnancy) |
| **Concomitant drugs** | -1 (none), 0 (incompatible timing), -2 (compatible timing, plausible) |
| **Non-drug causes excluded** | +2 (all excluded), +1 (6 main excluded), 0 (incomplete), -2 (not done) |
| **Previous info on drug** | +3 (well-known), +2 (limited), 0 (unknown), -1 (no hepatotoxicity) |
| **Rechallenge** | +3 (ALT ↑↑ on re-exposure), +2 (ALT ↑ only), 0 (no info), -2 (negative) |

| Total Score | Causality |
|-------------|-----------|
| **≥8** | **Highly probable** |
| **6-7** | **Probable** |
| **3-5** | **Possible** |
| **1-2** | **Unlikely** |
| **≤0** | **Excluded** |

## 3. Paracetamol Toxicity (Specific Management)

### Toxic Dose
| Population | Toxic Dose (single) |
|------------|---------------------|
| **Adult** | **>150 mg/kg** or **>7.5g** |
| **Child** | **>150 mg/kg** |
| **High-risk** (alcohol, fasting, CYP inducers) | **>75 mg/kg** |

### NAC Regimen (IV)
| Phase | Dose | Volume | Duration |
|-------|------|--------|----------|
| **Loading** | **150 mg/kg** | 200ml 5% glucose | 1 hour |
| **Phase 2** | **50 mg/kg** | 500ml 5% glucose | 4 hours |
| **Phase 3** | **100 mg/kg** | 1000ml 5% glucose | 16 hours |
| **Total** | **300 mg/kg** | | 21 hours |

> **Continue NAC** if: INR >1.3, ALT rising, encephalopathy, renal failure, or <8h post-ingestion with nomogram above line.

### Paracetamol Nomogram (UK)
- **Treatment line**: 100 mg/L at 4h → 15 mg/L at 15h (log-linear)
- **High-risk line** (alcohol, fasting, inducers): **half treatment line**

## 4. Management Principles

### General Approach
| Step | Action |
|------|--------|
| **1. Suspect DILI** | New LFT abnormality + drug exposure (prescribed/OTC/herbal) |
| **2. Exclude other causes** | Viral (A,B,C,E), autoimmune (AIH), biliary, ischaemic, alcohol, Wilson, malignancy |
| **3. Calculate R-value** | **R = (ALT/ULN) ÷ (ALP/ULN)** → Hepatocellular / Cholestatic / Mixed |
| **4. Apply RUCAM** | Score causality |
| **5. Stop suspected drug** | **Immediate withdrawal** (most important step) |
| **6. Supportive care** | Monitor LFTs, INR, bilirubin, glucose, signs of ALF |
| **7. Specific therapy** | **Paracetamol → NAC**; **ICI hepatitis** → steroids; **DRESS** → stop drug + prednisolone 1mg/kg |

### Rechallenge Principles

| Scenario | Recommendation |
|----------|----------------|
| **Definite/Probable DILI (RUCAM ≥6)** | **NEVER** rechallenge (risk of fatal recurrence) |
| **Possible DILI (RUCAM 3-5)** | **Avoid if possible**; if essential, different drug class; monitor LFTs weekly |
| **Different drug, same class** | **Cross-reactivity possible** (e.g., co-amoxiclav → other penicillins; sulfonamides → other sulfonamides); **avoid** |
| **Essential drug, no alternative** | **Risk-benefit discussion**, specialist hepatology input, close LFT monitoring (weekly ×4wks, then monthly) |

## 6. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Intrinsic vs Idiosyncratic** | **Intrinsic**: Dose-dependent, predictable (Paracetamol); **Idiosyncratic**: Unpredictable, host-dependent, immune (Co-amoxiclav, INH) |
| **R-Value Calculation** | **R = (ALT/ULN) ÷ (ALP/ULN)**; use **ULN of YOUR LAB** |
| **Hepatocellular vs Cholestatic** | **Hepatocellular**: ALT dominant (>5x ALP); **Cholestatic**: ALP dominant (<2x ALT); **Mixed**: 2-5 |
| **Paracetamol Nomogram** | **UK treatment line**: 100mg/L at 4h → 15mg/L at 15h; **High-risk (alcohol/fasting) = half line** |
| **Co-amoxiclav vs Flucloxacillin** | **Co-amoxiclav**: 1-3 wks, mixed/cholestatic; **Flucloxacillin**: 2-4 wks, cholestatic; **Both: re-exposure = rapid recurrence** |
| **RUCAM Rechallenge** | **Positive rechallenge = +3**; **Definite/Probable DILI = NEVER rechallenge** |
| **Anti-TB Hepatotoxicity** | **Pyrazinamide > Rifampicin > INH**; Monitor monthly ×2mo; **Stop if ALT>3xULN (symptomatic) or >5xULN (asymptomatic)** |
| **ICI Hepatitis Grading** | **Grade 1 (<3x): Continue ICI + monitor**; **Grade 2 (3-5x): Hold + Pred 0.5-1mg/kg**; **Grade 3-4 (>5x): Pred 1-2mg/kg + Hold ICI** |
| **VALPROATE Toxicity** | **Children <3y + polytherapy = High risk**; **Hyperammonaemic encephalopathy** (treat with carnitine) |
| **Allopurinol DRESS** | **HLA-B*58:01 (Asians)**; **Allopurinol hypersensitivity syndrome** → stop immediately |

**Mnemonic: DILI-CHAMPION**
- **D**ILI Types: **Intrinsic (Paracetamol) vs Idiosyncratic (Co-amoxiclav, INH, ICI)**
- **I**ntrinsic: **Dose-dependent, Predictable, Paracetamol**
- **L**iver Patterns: **Hepatocellular (R≥5: Paracetamol, INH), Cholestatic (R≤2: Co-amoxiclav), Mixed (2-5: Anti-TB)**
- **I**diosyncratic: **Unpredictable, Immune, Co-amoxiclav, Flucloxacillin, INH, ICI**
- **C**o-amoxiclav: **#1 Antibiotic DILI, Latency 1-3wk, Mixed/Cholestatic**
- **H**ALO: **Flucloxacillin (Cholestatic, 2-4wks, Re-exposure Rapid)**
- **A**nti-TB: **Pyrazinamide > Rifampicin > INH; Monitor Monthly**
- **M**onitoring: **Baseline → q2wks×2mo → Monthly; Stop ALT>3xULN(Sx) / >5xULN(Asx)**
- **P**aracetamol: **>150mg/kg → NAC 150→50→100 mg/kg/1h/4h/16h**
- **I**CI Hepatitis: **G2 Hold+Pred0.5-1; G3/4 Pred1-2+Hold**
- **O**ther: **Valproate (<3y polytherapy → Hyperammonaemia), CBZ/PHT/Allopurinol DRESS (HLA)**
- **N**ever Rechallenge: **RUCAM ≥6 = NEVER; Possible = Avoid/Monitor**

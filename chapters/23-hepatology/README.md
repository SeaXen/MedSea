## Key Algorithms & Scoring Systems (Memorize for Viva)

| Algorithm/Score | Use Case | Key Cut-offs |
|----------------|----------|--------------|
| **King's College Criteria** | ALF transplant referral | pH <7.3 OR (PT >100s + Cr >300 + Grade III/IV HE) |
| **CLIF-C ACLF** | ACLF grading | Organ failures (liver, kidney, brain, coagulation, circulation, respiration) |
| **Child-Pugh** | Cirrhosis severity | A: 5-6, B: 7-9, C: 10-15 |
| **MELD / MELD-Na** | Transplant priority | 6-40 (higher = sicker) |
| **Maddrey DF** | Alcoholic hepatitis severity | >32 = severe (consider steroids) |
| **Glasgow/ABIC** | Alcoholic hepatitis prognosis | ABIC >9 = 90-day mortality >50% |
| **Lille Score** | Steroid response Day 7 | >0.45 = non-response (stop steroids) |
| **FIB-4 / NFS / APRI** | NAFLD fibrosis risk | FIB-4 <1.3 low, >2.67 high risk |
| **Child-Pugh / MELD** | Variceal bleed prognosis | Child C or MELD >18 = high mortality |
| **HVPG** | Portal hypertension | ≥10 mmHg = CSPH; ≥12 mmHg = variceal risk |
| **West Haven** | HE grading | Grade I-IV |
| **HRS-AKI/HRS-CKD** | Hepatorenal syndrome | Scr rise, no shock, no nephrotoxins, no parenchymal disease |
| **LI-RADS** | HCC diagnosis on imaging | LR-1 to LR-5 / LR-M / LR-TIV |
| **BCLC** | HCC staging & treatment | 0-A: curative; B: TACE; C: systemic; D: BSC |

## High-Yield Viva Topics (FCPS/MRCP)

### Must-Know Definitions
- [ ] ALF vs ACLF definition (interval, encephalopathy, pre-existing liver disease)
- [ ] CSPH vs HVPG ≥10 mmHg
- [ ] ACLF-ACLF vs ACLF definitions (organ failures)
- [ ] Compensated vs decompensated cirrhosis
- [ ] AIH simplified criteria (ANA/SMA/LKM1, IgG, histology, exclusion)
- [ ] PBC criteria (ALP >1.5x ULN + AMA + histology)
- [ ] PSC diagnosis (MRCP beading, IBD association)

### Must-Know Doses/Durations
- [ ] NAC regimen for paracetamol ALF (IV 150mg/kg/1h, 50mg/kg/4h, 100mg/kg/16h)
- [ ] Prednisolone for alcoholic hepatitis (40mg/day ×28 days taper)
- [ ] Lactulose dose (15-30ml BD to 2-3 soft stools/day)
- [ ] Rifaximin 550mg BD add-on for recurrent HE
- [ ] Terlipressin 2mg IV q4-6h + albumin 1g/kg for HRS-AKI
- [ ] UDCA 13-15mg/kg/day for PBC
- [ ] Obeticholic acid 5-10mg OD if UDCA inadequate response
- [ ] Nucleos(t)ide analogues: entecavir 0.5mg, tenofovir 245mg
- [ ] DAA regimens: SOF/VEL, GLE/PIB, SOF/VEL/VOX

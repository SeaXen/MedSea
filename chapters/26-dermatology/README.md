- Erythema multiforme, figurate erythemas
- Inherited bullous disorders (EB), dermatitis herpetiformis
- Targeted therapy/immunotherapy cutaneous toxicity
- Fungal infections (dermatophytes, candidiasis, deep mycoses)
- Benign melanocytic lesions, dysplastic naevus syndrome
- Pigmentation disorders (melasma, PIH, albinism)
- Nail disorders (psoriasis, LP, onychomycosis, systemic signs)
- Connective tissue disease skin signs
- Genodermatoses (ichthyoses, ectodermal dysplasia, phakomatoses)
- Post-transplant skin cancer surveillance

## Tier 3 (5% of marks — differentiators)

- Rare papulosquamous (PRP, parapsoriasis, CTCL early)
- Rare bullous (pemphigus variants, bullous lupus, EBA)
- Rare drug reactions (SDRIFE, baboon syndrome, fixed drug eruption variants)
- Rare infections (mycobacteria, leishmaniasis, onchocerciasis)
- Rare skin tumours (Merkel cell, DFSP, AFX, adnexal tumours)
- Rare genodermatoses (Darier, Hailey-Hailey, porokeratosis)
- Complex immunocompromise scenarios

## Key Algorithms & Scoring Systems (Memorise)

| System | Use Case | Key Thresholds |
|--------|----------|----------------|
| **PASI** | Psoriasis severity | <10 mild, 10-20 moderate, >20 severe |
| **SCORTEN** | SJS/TEN mortality | ≥3 = high mortality (>35%) |
| **RegiSCAR** | DRESS diagnosis | Score ≥5 = definite |
| **EUROSCAR** | AGEP diagnosis | Score ≥8 = confirmed |
| **ABCDE** | Melanoma screening | Asymmetry, Border, Colour, Diameter, Evolution |
| **Breslow thickness** | Melanoma prognosis | ≤1mm, 1-2mm, 2-4mm, >4mm |
| **Clark level** | Melanoma microstaging | I-V (dermis invasion depth) |
| **UAS7** | Urticaria activity | Daily scoring ×7 days |
| **EASI** | Atopic dermatitis severity | 0-72 scale |
| **SCORAD** | Atopic dermatitis severity | Objective + subjective |
| **CASPAR** | Psoriatic arthritis | ≥3 points = PsA |
| **DAPSA** | PsA disease activity | Remission <4, LDA 4-14, MDA 15-28, HDA >28 |
| **DLQI** | Quality of life impact | 0-30 (higher = worse) |
| **HLA associations** | Drug reactions | HLA-B*58:01 (allopurinol), B*15:02 (carbamazepine), B*57:01 (abacavir) |
| **LRINEC** | Necrotising fasciitis | ≥6 = high suspicion |

## High-Yield Viva Topics

| Topic | Expected Answer Points |
|-------|------------------------|
| **SJS vs TEN vs overlap** | BSA detachment: SJS <10%, overlap 10-30%, TEN >30%; mucosal sites ≥2; SCORTEN |
| **DRESS vs AGEP** | DRESS: eosinophilia, organ involvement, latency 2-6w; AGEP: neutrophilic pustules, latency 1-5d, EUROSCAR |
| **Pemphigus vs BP DIF** | Pemphigus: IgG intercellular (fishnet); BP: IgG/C3 linear basement membrane |
| **Melanoma staging** | TNM 8th: Breslow, ulceration, mitotic rate, SLNB, satellite/in-transit |
| **Psoriasis biologics** | TNFα → IL-17/23 → JAKi; failure criteria, screening (TB, hepatitis) |
| **Urticaria stepwise** | H1 → ↑dose → add H2/LTRA → omalizumab → cyclosporine |
| **Vitiligo management** | Topical steroids/TCI → NB-UVB → JAKi (ruxolitinib) → surgery (stable) |
| **Skin disease in pregnancy** | Category D/X: retinoids, MTX, MMF, hydroxyurea; Safe: UVB, topical steroids, azathioprine |
| **Kaposi sarcoma** | HHV-8, HIV/CD4<200, classic/endemic/iatrogenic; staging (TIS) |
| **Necrotising fasciitis** | LRINEC ≥6, surgical emergency, broad-spectrum ABx + IVIG consideration |
| **Erythroderma causes** | Psoriasis, drugs, CTCL, eczema, PRP; "red man syndrome" workup |
| **Cutaneous T-cell lymphoma** | MF vs SS: blood involvement (B1/B2), Sézary count >1000/μL |

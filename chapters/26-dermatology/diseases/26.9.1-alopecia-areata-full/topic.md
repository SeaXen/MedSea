## Definition

Alopecia areata (AA) is a chronic autoimmune non-scarring alopecia affecting ~2% of population, all ages, both sexes equally. Pathogenesis: T-cell mediated attack on the hair follicle immune privilege, with CD8+ NKG2D+ T cells targeting anagen hair bulb autoantigens. Strong association with other autoimmune diseases (vitiligo, thyroid disease, type 1 diabetes, pernicious anaemia, coeliac, SLE). Genetic predisposition: HLA-DRB1*04, *11, *16. Triggered by stress, infection, vaccines, hormonal changes.

## Clinical Features and Presentation

Clinical: sudden onset, well-circumscribed, smooth, completely bald patches with normal underlying scalp (no scaling, no scarring, no erythema). 'Exclamation mark hairs' (short, broken hairs tapered at proximal end) at active border. Positive 'hair pull test' at expanding margins. Patterns: patchy (single/few patches, most common), totalis (100% scalp), universalis (100% body hair), ophiasis (band-like along occipital/temporal margins), sisaipho (reverse ophiasis, central scalp), incognita (diffuse). Nail changes: pitting (geometric, regular), trachyonychia (sandpaper nails), red lunulae, onychorrhexis — in 10-50%. Trichoscopy: yellow dots (sebum in empty follicles), black dots (cadaver hairs), exclamation marks, tapered hairs, short vellus regrowth. Severity scoring: SALT (Severity of Alopecia Tool) score.

## 1. DEFINITION & EPIDEMIOLOGY

**Alopecia Areata (AA)** is a **T-cell mediated autoimmune non-scarring alopecia** characterized by **patchy, non-scarring hair loss** with preservation of follicular ostia.

| Parameter | Details |
|-----------|---------|
| **Prevalence** | 0.1-0.2% (lifetime risk 1.7-2.1%) |
| **Age** | Any; **peak 15-30 yrs**; 20% <16 yrs |
| **Sex** | Equal |
| **Genetics** | **Polygenic**; **HLA-DRB1*04, *11, *13**; **CTLA4, IL2RA, IL2/21, ULBP3/6**; 10-20% family history |
| **Autoimmune comorbidities** | **Thyroid (Hashimoto's/Graves')**, Vitiligo, Atopy, T1DM, Pernicious anaemia, SLE, Psoriasis, Down syndrome |
| **Course** | **Unpredictable**: spontaneous remission (30-50% at 1 yr), relapse common |

## 2. CLASSIFICATION & CLINICAL PATTERNS

| Pattern | Description | Prognosis |
|---------|-------------|-----------|
| **Patchy AA** | 1+ well-circumscribed oval/round patches | Best; 50% spontaneous regrowth at 1 yr |
| **Alopecia Totalis (AT)** | **Complete scalp hair loss** | Guarded; 10-30% regrowth |
| **Alopecia Universalis (AU)** | **Complete body hair loss** (scalp, face, body) | Poor; <10% regrowth |
| **Ophiasis** | Band-like occipital/temporal loss | Poor; resistant |
| **Ophiasis Inversus** | Fringe of hair loss sparing occiput | Poor |
| **Sisaipho** | Diffuse thinning + patchy | Variable |
| **Diffuse AA** | Diffuse shedding without patches | Mimics telogen effluvium |
| **Nail AA** | Pitting, trachyonychia, भू्र regression | 10-20%; correlates with severity |

---

## 3. PATHOPHYSIOLOGY

| Mechanism | Details |
|-----------|---------|
| **Autoimmune** | **CD8+ NKG2D+ T-cells** → attack **anagen hair follicle** (bulb) — **loss of immune privilege** |
| **Autoantigens** | **Melanocyte proteins** (tyrosinase, TRP-1, TRP-2, gp100); **Keratin 16/17**; **ULBP3** (NKG2D ligand) |
| **Genetic** | **HLA-DRB1*04, *11, *13**; **CTLA4, IL2RA, IL2/21, ULBP3/6, IKZF4** |
| **Cytokines** | **IFN-γ, IL-2, IL-15, IL-21** → JAK/STAT activation → MHC-I upregulation on follicle → CD8+ recruitment |
| **Collapse of Immune Privilege** | Normal anagen follicle: low MHC-I, TGF-β, IGF-1, FGF; AA: ↑ MHC-I, ↓ TGF-β, ↑ IFN-γ → CD8+ attack |

## 3. CLINICAL FEATURES

### 3.1 Scalp Hair Loss
| Feature | Description |
|---------|-------------|
| **Patches** | Well-circumscribed, smooth, non-scarring, **no scaling** |
| **Exclamation Mark Hairs** | **Short, broken hairs** (3-4 mm) tapering at base — **pathognomonic** at active border |
| **Black Dots** | Destructed hairs at follicular orifice (dermoscopy) |

## 4. DIAGNOSTIC WORKUP

### 4.1 Clinical Diagnosis
- **History:** Acute onset patchy hair loss, no scarring, no scaling
- **Examination:** Patches, exclamation mark hairs, nail exam
- **Pull Test:** Positive at active border

### 4.2 Trichoscopy (Dermoscopy of Scalp) — **Key Diagnostic Tool**
| Feature | Significance |
|---------|--------------|
| **Yellow Dots** | Sebum/keratin in empty follicles — **most sensitive** |
| **Black Dots** | Destructed hairs at orifice |
| **Exclamation Mark Hairs** | Active disease border |
| **Broken Hairs** | Active |
| **Short Vellus Hairs** | Regrowth |
| **Absent Yellow Dots** | Scarring alopecia (LPP, FFA) |

### 4.3 Laboratory Investigations
| Test | Indication | Frequency |
|------|------------|-----------|
| **TSH, Anti-TPO, Anti-Tg** | **Thyroid autoimmunity** (15-25%) | Baseline + annual |
| **FBC, Ferritin, B12, Zinc** | Nutritional | If diffuse/thinning |
| **ANA, ESR** | If SLE/scleroderma suspected | If clinical |
| **HbA1c** | Type 1 DM | If indicated |
| **IgE, Eosinophils** | Atopy | If history |

### 4.3 Scalp Biopsy (If Diagnostic Uncertainty)
- **H&E:** **Peribulbar lymphocytic infiltrate** ("swarm of bees") around anagen bulb
- **Terminal:Vellus ratio** <2:1 (normal >4:1)
- **Pigment incontinence** in dermis
- **DIF:** Negative (vs lupus, lichen planus)

| SALT Score | Severity |
|------------|----------|
| **<25%** | Mild |
| **25-49%** | Moderate |
| **50-74%** | Severe |
| **75-99%** | Very severe |

## 5. MANAGEMENT

### 6.1 Treatment Algorithm
```
New Diagnosis AA
    │
    ▼ Assess Severity (SALT) + Pattern + Age
    │
    ▼ REASSURE (spontaneous regrowth 30-50% at 1 yr)
    │
    ▼ Patchy, <50% scalp (Mild-Moderate)
    │       │
    │       ▼ **Intralesional Triamcinolone** 2.5-10 mg/mL (max 20 mg/session) q4-6w
    │       ▼ Potent **Topical Corticosteroid** (Clobetasol 0.05% foam/solution) daily
    │       ▼ **TCI** (Tacrolimus 0.1% / Pimecrolimus 1%) if face/flexures
    │       ▼ **Topical Minoxidil 5%** BID (adjunct)
    │
    ▼ Extensive / Totalis / Universalis / Ophiasis / Rapidly Progressive
    │       │
    │       ▼ **Systemic Corticosteroids** (Pred 0.5-1 mg/kg/day × 4-8w → taper)
    │       ▼ **Oral Mini-pulse** (Dexamethasone 5 mg 2 consecutive days/week)
    │       ▼ **Contact Immunotherapy** (DPCP / SADBE / DNCB) — **Gold standard for extensive**
    │       ▼ **JAK Inhibitors** (Baricitinib/Upadacitinib/Ritlecitinib) — **New paradigm**
    │       ▼ **Systemic Immunosuppressants** (MTX, AZA, MMF, Cyclosporine) — 2nd line
    │
    ▼ Pediatric (<12 yrs)
    │       ▼ IL Triamcinolone (lower dose) / Topical TCS / TCI
    │       ▼ Avoid systemic unless severe/rapid progression
    │
    ▼ Refractory / Relapsing
    │       ▼ JAK Inhibitors (Baricitinib 4mg, Upadacitinib 15-30mg, Ritlecitinib 30-50mg)
    │       ▼ DPCP re-sensitisation / maintenance
    │       ▼ MTX / AZA / MMF / Cyclosporine
```

---

## 5.1 Intralesional Corticosteroids (First-Line for Patchy)
| Agent | Concentration | Volume | Max/Session | Interval |
|-------|---------------|--------|-------------|----------|
| **Triamcinolone Acetonide** | **2.5-10 mg/mL** | 0.1 mL/cm² | **20 mg** (2 mL of 10 mg/mL) | **q4-6 weeks** |

> **Technique:** Multiple superficial injections 1 cm apart at active border + centre; avoid deep injection (atrophy)
> **Contraindications:** AT/AT/AU (too many injections); children <10 yrs (pain)

### 5.2 Topical Therapy
| Agent | Indication | Regimen |
|-------|------------|---------|
| **Clobetasol 0.05% foam/solution** | Patchy AA | Daily × 3-6 months |
| **Minoxidil 5% solution/foam** | Adjunct / regrowth maintenance | BID |
| **TCI (Tacrolimus 0.1%/Pimecrolimus 1%)** | Face, children | BID |

### 5.3 Contact Immunotherapy (DPCP) — **Gold Standard for Extensive AA**
| Step | Protocol |
|------|----------|
| **Sensitisation** | **2% DPCP** in acetone on 2×2 cm upper arm/back |
| **Challenge (2 weeks)** | **0.001% DPCP** on scalp patch → assess reaction |
| **Weekly Treatment** | Escalating concentration (0.001% → 0.01% → 0.1% → 1%) to maintain mild eczema |
| **Maintenance** | Once allergic contact dermatitis established → **weekly/biweekly** |
| **Response** | **Regrowth in 50-70%** at 6-12 months; relapse common on stopping |

> **Alternative sensitisers:** SADBE (squaric acid dibutyl ester), DNCB (dinitrochlorobenzene — mutagenic)

### 5.4 Systemic Corticosteroids
| Regimen | Dose | Duration |
|---------|------|----------|
| **Prednisolone** | 0.5-1 mg/kg/day (max 60 mg) | 4-8 weeks → taper |
| **Oral Mini-pulse (OMP)** | **Dexamethasone 5 mg** (or Betamethasone 5 mg) on **2 consecutive days/week** | 3-6 months |
| **IV Methylprednisolone** | 500 mg-1g daily × 3 days | Pulse for rapid control |

> **Relapse rate post-steroids:** 50-80% at 6-12 months

### 5.4 JAK Inhibitors — **Game Changer (FDA Approved 2022)**
| Drug | Dose | Indication | Key Trial |
|------|------|------------|-----------|
| **Baricitinib** | **4 mg OD** (2 mg if eGFR<60) | **AA ≥18 yrs, SALT ≥50%** | BRAVE-AA1/2 (40% SALT75 at 36 wks) |
| **Upadacitinib** | **15-30 mg OD** | AA ≥12 yrs | Phase 3 (up to 50% SALT75) |
| **Ritlecitinib** | **30-50 mg OD** | AA ≥12 yrs | Phase 2/3 |

| Monitoring | Frequency |
|------------|-----------|
| **CBC, LFT, Lipids** | Baseline, 4w, 8w, then q3m |
| **CPK** | Upadacitinib (baseline, q3m) |
| **VZV IgG** | Baseline (vaccinate if -ve) |
| **TB Screen** | CXR/IGRA |
| **Pregnancy Test** | Baseline, during |

> **Contraindications:** Active infection, TB, HBV/HCV, uncontrolled HTN, pregnancy, thrombosis risk

## 6. KEY VIVA POINTS

| Question | Answer |
|----------|--------|
| **AA pathophysiology?** | **CD8+ NKG2D+ T-cells** attack anagen follicle (loss of immune privilege); **IFN-γ, IL-15, JAK/STAT** pathway |
| **Exclamation mark hairs?** | Short, tapered broken hairs at active border — **pathognomonic** |
| **Trichoscopy yellow dots?** | Sebum/keratin in empty follicles — **most sensitive** for AA |
| **SALT score?** | Severity: <25% mild, 25-49% moderate, 50-74% severe, 75-99% very severe, 100% totalis |
| **First-line patchy AA?** | **Intralesional triamcinolone** 2.5-10 mg/mL q4-6w (max 20 mg/session) |
| **Extensive AA treatment?** | **Contact immunotherapy (DPCP)** — gold standard; **JAK inhibitors** (new paradigm) |
| **DPCP protocol?** | Sensitise 2% → challenge 0.001% → weekly escalation (0.001%→0.01%→0.1%→1%) |
| **JAK inhibitors approved for AA?** | **Baricitinib 4mg OD**, Upadacitinib 15-30mg, Ritlecitinib 30-50mg (FDA 2022+) |
| **JAKi monitoring?** | CBC, LFT, lipids, **VZV IgG**, TB screen, pregnancy avoid, thrombosis risk |
| **AA vs Telogen Effluvium?** | AA: patches, exclamation hairs, yellow dots; TE: diffuse shedding, hx trigger 3m ago |
| **AA vs Trichotillomania?** | TT: irregular patches, broken hairs different lengths, no exclamation marks |
| **AA vs Scarring Alopecia?** | AA: follicular ostia preserved, no scarring; LPP/FFA/CCA: scarring, loss of ostia |
| **Ophiasis prognosis?** | Poor; band-like occipital loss; resistant |
| **Nail pitting in AA?** | 10-20%; correlates with severity |
| **Baricitinib dose?** | 4 mg OD (2 mg if eGFR<60); monitor CBC/LFT/lipids/VZV/TB |
| **Spontaneous regrowth?** | 30-50% at 1 year (patchy); <10% AT/AU |

## 7. SALT SCORE QUICK CALCULATION

```
Example: 50% loss on vertex (40%) + 20% on right parietal (18%)
SALT = (0.50 × 40) + (0.20 × 18) = 20 + 3.6 = 23.6% → MILD
```

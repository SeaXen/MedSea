## Definition

Systemic lupus erythematosus (SLE) is a multisystem autoimmune disease with diverse cutaneous manifestations. 70-80% of SLE patients have skin involvement at some point; skin disease is the first manifestation in 20-25%. Female predominance (9:1), peak onset 15-45 years, more common and severe in non-white populations. Pathogenesis: autoantibody formation, immune complex deposition, complement activation, photosensitivity (UV-induced keratinocyte apoptosis exposes nuclear antigens).

## Clinical Features and Presentation

Specific cutaneous LE (interface dermatitis on biopsy): (1) Acute CLE — malar/butterfly rash (spares nasolabial folds), photosensitive, transient; (2) Subacute CLE — annular polycyclic or papulosquamous, photosensitive, anti-Ro/SSA positive in 70%, mild systemic disease; (3) Chronic CLE (DLE) — discoid lesions, scarring, atrophy, follicular plugging, hypopigmentation, alopecia (scarring), 5-10% progress to SLE. Non-specific: photosensitivity, alopecia (diffuse, non-scarring), livedo reticularis, Raynaud's, vasculitis, urticaria, bullous LE. Other: oral ulcers (painless, hard palate), nasopharyngeal ulcers, periungual telangiectasia. Investigations: ANA, anti-dsDNA (specific, correlates with disease activity), anti-Smith (specific), anti-Ro/La, complement C3/C4 (low in active disease), CBC (cytopenias), urinalysis, skin biopsy + DIF (lupus band — granular IgG/C3 at BMZ).

## 1. DEFINITION & EPIDEMIOLOGY

**Systemic Lupus Erythematosus (SLE)** is a chronic, multisystem autoimmune disease characterized by **autoantibody production**, **immune complex deposition**, and **multiorgan inflammation**. Cutaneous involvement occurs in **70-85%** of patients.

| Parameter | Details |
|-----------|---------|
| **Prevalence** | 20-150/100,000; **F:M = 9:1** (reproductive years) |
| **Age of onset** | 15-45 years (peak 20-30) |
| **Ethnicity** | Higher in African, Asian, Hispanic populations |
| **Genetics** | **HLA-DR2, DR3**; **IRF5, STAT4, BLK, TNFAIP3, PTPN22**; monozygotic twins 25-50% concordance |
| **Triggers** | UV light (photosensitivity), drugs, hormones, infections, stress |

## 2. CLASSIFICATION OF CUTANEOUS LUPUS (GILLAM & SONTHEIMER)

| Category | Subtypes | Features |
|----------|----------|----------|
| **Acute Cutaneous LE (ACLE)** | **Malar rash (butterfly)**, **photosensitive** maculopapular eruption | **Correlates with systemic flare**; spares nasolabial folds |
| **Subacute Cutaneous LE (SCLE)** | **Annular** (polycyclic), **psoriasiform** (papulosquamous) | **Anti-Ro/SSA + (70-90%)**; photo-distributed; **non-scarring** |
| **Chronic Cutaneous LE (CCLE)** | **Discoid LE (DLE)** (most common), lupus profundus, chilblain LE, lupus tumidus | **Scarring, dyspigmentation**, alopecia; DLE: **localized vs generalized** |
| **Intermittent** | Tumid LE, lupus erythematosus tumidus | Urival, oedematous plaques; no interface dermatitis |

## 3. CLINICAL FEATURES BY SUBTYPE

### 3.1 Acute Cutaneous LE (ACLE)
| Feature | Description |
|---------|-------------|
| **Malar rash** | **Erythematous butterfly rash** over cheeks/bridge of nose; **spares nasolabial folds** |
| **Photosensitive eruption** | Maculopapular on face, V-neck, dorsa hands; **worsens with UV** |
| **Bullous LE** | Rare; vesicles/bullae on sun-exposed skin; **anti-type VII collagen** (mimics EBA) |
| **Toxic epidermal necrolysis-like** | Rare; full-thickness necrosis; high mortality |

### 3.2 Subacute Cutaneous LE (SCLE)
| Feature | Description |
|---------|-------------|
| **Annular polycyclic** | **Expanding rings** with central clearing; trailing scale inside border |
| **Psoriasiform** | **Papulosquamous plaques**; mimics psoriasis; **photo-distributed** |
| **Anti-Ro/SSA** | **70-90% positive**; also **Anti-La/SSB** (30-50%) |
| **Drug-induced** | Hydrochlorothiazide, PPIs, TNF-α inhibitors, terbinafine, calcium channel blockers |

### 3.3 Chronic Cutaneous LE (CCLE) - DLE
| Feature | Description |
|---------|-------------|
| **Discoid plaques** | **Well-demarcated, erythematous, scaly** plugs; **follicular hyperkeratosis** (carpet tack sign) |
| **Healing** | **Scarring, atrophy, dyspigmentation** (hypo/hyper); **permanent alopecia** if scalp |
| **Distribution** | Face, scalp, ears, neck; **conchal bowl involvement** (specific) |
| **Mucosal** | Oral discoid lesions (erosions, white plaques) |
| **Lupus Profundus (Lupus Panniculitis)** | Deep subcutaneous nodules/plaques → **atrophic depressed scars** |
| **Chilblain LE** | Acral violaceous plaques/nodules (cold-induced); **TREX1 mutations** (familial) |

## 4. PATHOPHYSIOLOGY

| Mechanism | Details |
|-----------|---------|
| **Loss of Tolerance** | Defective apoptosis → nuclear antigen exposure → **autoantibodies** (anti-dsDNA, Sm, Ro, La) |
| **Immune Complex Deposition** | **ICs in dermo-epidermal junction** → complement activation → inflammation |
| **Type I IFN Signature** | **Plasmacytoid DC → IFN-α** → DC maturation → T/B cell activation |
| **Photosensitivity** | UV → keratinocyte apoptosis → autoantigen exposure (Ro/La) → **IFN-κ** → immune activation |
| **NETosis** | Impaired neutrophil extracellular trap clearance → autoantigen exposure |

> **Total ≥10** = Classified as SLE (with positive ANA ≥1:80 as entry criterion)

### 5.2 Serology
| Antibody | Sensitivity | Specificity | Clinical Correlation |
|----------|-------------|-------------|---------------------|
| **ANA** | 95-98% | Low | Entry criterion; **IF positive on HEp-2** |
| **Anti-dsDNA** | 60-70% | **High (95%)** | **Correlates with renal activity** |
| **Anti-Sm** | 20-30% | **Very High (99%)** | **Pathognomonic** for SLE |
| **Anti-Ro/SSA** | 30-50% | Moderate | **SCLE, neonatal lupus, Sjögren overlap** |
| **Anti-La/SSB** | 10-15% | Moderate | With Anti-Ro |
| **Anti-RNP** | 20-30% | Moderate | **MCTD (high titre)** |
| **Antiphospholipid (aCL, LAC, anti-β2GPI)** | 30-40% | High | **APS (thrombosis, pregnancy morbidity)** |
| **Low C3/C4** | — | — | **Complement consumption = active disease** |

### 5.3 Skin Biopsy
| Subtype | Histopathology |
|---------|----------------|
| **ACLE/SCLE** | **Interface dermatitis**, vacuolar basal degeneration, **mucin deposition**, perivascular/periappendageal lymphocytic infiltrate |
| **DLE** | **Interface dermatitis + follicular plugging**, basal degeneration, **follicular hyperkeratosis**, **scarring**, **mucin**, **Civatte bodies** |
| **Lupus Profundus** | **Lobular panniculitis**, lymphocytic infiltrate, **mucin**, **adipocyte necrosis** |

### 5.4 Direct Immunofluorescence (DIF) — **Gold Standard for DLE/SCLE**
| Finding | Pattern |
|---------|---------|
| **IgG, IgM, IgA, C3** | **Linear granular/continuous at BMZ** ("lupus band test") |
| **Site** | **Lesional AND clinically normal sun-exposed skin** (vs pemphigus/BP) |
| **Significance** | **Positive in 90% DLE lesions**; 60-70% non-lesional sun-exposed skin |

## 6. DIFFERENTIAL DIAGNOSIS

| Condition | Key Differentiators |
|-----------|---------------------|
| **Rosacea** | Centrofacial, papules/pustules, no scarring, no autoantibodies |
| **Seborrheic Dermatitis** | Greasy scales, nasolabial folds, scalp; no scarring |
| **Dermatomyositis** | Heliotrope, Gottron, mechanic's hands; anti-Mi-2/TIF1γ |
| **Polymorphous Light Eruption (PMLE)** | Papules/vesicles on sun-exposed; resolves without scarring; no autoantibodies |
| **Lichen Planus** | Purple polygonal papules, Wickham striae, oral reticular; anti-CD4 T-cells |
| **Sarcoidosis** | Lupus pernio (violaceous plaques nose/cheeks); granulomas on biopsy |
| **Rosai-Dorfman** | Massive cervical lymphadenopathy, S100+ histiocytes |

## 7. MANAGEMENT

### 6.1 General Measures
| Measure | Details |
|---------|---------|
| **Strict Photoprotection** | **SPF 50+ broad-spectrum**, protective clothing, avoid midday sun |
| **Smoking Cessation** | Reduces cutaneous activity; improves hydroxychloroquine efficacy |
| **Vitamin D** | Supplement if deficient (avoidance of sun) |
| **Camouflage** | Dermacolor, makeup for dyspigmentation |

### 6.2 Topical Therapy (First-Line for Cutaneous)
| Agent | Indication | Regimen |
|-------|------------|---------|
| **Potent TCS** (Clobetasol, Betamethasone dipropionate) | Localized DLE, ACLE, SCLE | Daily × 2-4 wks → pulse |
| **TCI** (Tacrolimus 0.1%, Pimecrolimus 1%) | **Face, flexures, mucosal** | BID; no atrophy |
| **Intralesional Triamcinolone** | Resistant DLE plaques | 2.5-5 mg/mL q4-6w |
| **Topical Calcineurin Inhibitors** | First-line for **face/mucosal** | BID; no atrophy risk |

### 6.3 Systemic Therapy

| Drug | Dose | Indication | Key Trial |
|------|------|------------|-----------|
| **Hydroxychloroquine (HCQ)** | **200-400 mg/day** (≤5 mg/kg) | **All SLE patients**; **first-line cutaneous** | **Reduces flares, damage, thrombosis** |
| **Chloroquine** | 250 mg/day | If HCQ intolerant | Less retinopathy |
| **Quinacrine** | 100 mg/day | Refractory + HCQ | Adjunct; corneal deposits |
| **Prednisolone** | 0.5-1 mg/kg/day | Severe ACLE/SCLE, flare | Bridge to steroid-sparing |
| **Methotrexate** | 7.5-25 mg/week + Folate | Refractory SCLE/DLE | Steroid-sparing |
| **Mycophenolate Mofetil (MMF)** | 2-3 g/day | Refractory cutaneous | Steroid-sparing |
| **Azathioprine** | 1-2 mg/kg/day | Refractory; pregnancy | TPMT test |
| **Methotrexate** | 10-25 mg/week | Refractory DLE/SCLE | FBC, LFT q3m |
| **Belimumab** (anti-BAFF) | 10 mg/kg IV q4w | Active SLE despite standard | BLISS-52/76 |
| **Anifrolumab** (anti-IFNAR) | 300 mg IV q4w | Moderate-severe | TULIP-1/2 |
| **Rituximab** | 1000 mg × 2 / 375 mg/m²×4 | Refractory, severe | LUNAR, RITUXILUP |
| **Voclosporin** | 23.7 mg BD | Lupus nephritis | AURORA |

### 6.4 Subtype-Specific Management
| Subtype | First-Line | Refractory |
|---------|------------|------------|
| **ACLE** | HCQ + TCS + photoprotection | Prednisolone + MMF/MTX |
| **SCLE** | **HCQ + TCS/TCI** + photoprotection | MTX, MMF, **HCQ failure → Belimumab** |
| **DLE** | **HCQ + Potent TCS/TCI** + intralesional | MTX, MMF, **Dapsone**, Thalidomide, Isotretinoin |
| **Lupus Profundus** | HCQ + systemic steroids | MTX, MMF, Cyclophosphamide |
| **Chilblain LE** | Nifedipine, HCQ, TCS | TREX1 mutation → JAK inhibitors |

## 7. NEONATAL LUPUS ERYTHEMATOSUS (NLE)

| Feature | Details |
|---------|---------|
| **Cause** | Transplacental **maternal Anti-Ro/SSA, Anti-La/SSB** |
| **Skin** | **Annular SCLE-like lesions**; face/scalp; **resolve 6-8 months** (maternal IgG clearance) |
| **Cardiac** | **Congenital Complete Heart Block (CHB)** — **2% risk**; **permanent** — pacing if <50 bpm |
| **Screening** | **Fetal echo 18-24 wks** if mother Anti-Ro/La +ve |
| **Prevention** | **HCQ throughout pregnancy** (reduces CHB risk 50%) |

## 8. KEY VIVA POINTS

| Question | Answer |
|----------|--------|
| **ACLE vs SCLE vs DLE clinical difference?** | ACLE: malar rash + photosensitivity; SCLE: annular/psoriasiform photo-distributed; DLE: scarring discoid plaques |
| **SCLE autoantibody?** | **Anti-Ro/SSA 70-90%**; Anti-La/SSB 30-50% |
| **DLE histopathology?** | Interface dermatitis + **follicular plugging** + basement membrane thickening + scarring |
| **DIF "lupus band test"?** | **IgG/IgM/IgA/C3 linear granular at BMZ** in lesional + non-lesional sun-exposed skin |
| **SCLE vs Psoriasis?** | SCLE: photo-distributed, Anti-Ro+, non-scarring, annular/papulosquamous; Psoriasis: extensor, silvery scale, Auspitz+, Koebner+ |
| **Drug-induced SCLE culprits?** | HCTZ, PPIs, TNF-α inhibitors, terbinafine, calcium channel blockers |
| **HCQ dose & monitoring?** | 200-400 mg/day (≤5 mg/kg); **annual ophthalmology** (retinopathy), renal dose adjust |
| **Neonatal lupus(CHB) prevention?** | HCQ in pregnancy reduces CHB risk 50%; fetal echo 18-24 wks if Anti-Ro/La+ |
| **SCLE vs PMLE?** | SCLE: Anti-Ro+, annular/psoriasiform, persistent; PMLE: no autoantibodies, papules/vesicles, resolves without scarring |
| **DLE scarring management?** | Intralesional triamcinolone, surgical excision, laser (CO2/fractional) for atrophy |
| **Lupus band test positive in?** | **DLE (90% lesional, 60-70% non-lesional sun-exposed)**; also ACLE, SCLE |
| **Anti-Sm significance?** | **Pathognomonic for SLE** (99% specific) |
| **HCQ retinopathy screening?** | Baseline + annual after 5 years (OCT, visual field, fundus autofluorescence) |
| **Malar rash vs Rosacea?** | Malar: spares nasolabial folds, photosensitive, systemic SLE features; Rosacea: centrofacial, papules/pustules, flushing |

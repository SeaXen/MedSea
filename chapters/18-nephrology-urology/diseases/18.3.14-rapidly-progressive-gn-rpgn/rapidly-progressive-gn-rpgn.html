NEPHROLOGY & UROLOGY
Rapidly Progressive Glomerulonephritis (RPGN)
1 Learning Objectives
  • Define RPGN (rapid AKI + active sediment + crescents)
  • Classify by aetiology (Anti-GBM, Immune-complex, Pauci-immune)
  • Recognise clinical presentations and serological patterns
  • Apply urgent diagnostic workup (serology, biopsy)
  • Manage emergently based on category (pulse steroids + CyC/rituximab ± PLEX)
  • Differentiate from other causes of rapid AKI
  • Apply to FCPS/MRCP emergency nephrology vignettes
2 Definition & Clinical Presentation
FeatureDetail
-----------------
**Definition****Rapid loss of GFR** (50% decline in days–weeks) + **active urinary sediment** (haematuria, RBC casts, proteinuria)
**Pathology****Crescentic GN** — cellular crescents in **>50% glomeruli**
**Timecourse**Days to weeks (vs CKD = months–years)
**Presentation**AKI, haematuria (microscopic/macroscopic), proteinuria (subnephrotic), hypertension, oedema, ± pulmonary haemorrhage
3 Aetiological Classification (Crucial for Management)
CategoryMechanismSerologyExamples
-----------------------------------------
**Type I: Anti-GBM**Anti-α3(IV)NC1 collagen IgG**Anti-GBM+**, ANCA−**Anti-GBM disease** (Goodpasture's if pulmonary)
**Type II: Immune-Complex**Immune complex deposition**Low C3/C4**, ANA/dsDNA+, Cryo+, ANCA−**Lupus nephritis (III/IV), Post-strep, Cryoglobulinaemia, IE, IgA (rare), MPGN**
**Type III: Pauci-immune****ANCA-mediated** (neutrophil activation)**ANCA+ (p-ANCA/MPO or c-ANCA/PR3)**, C3/C4 normal**GPA, MPA, EGPA, Renal-limited vasculitis**
Double-positive (Anti-GBM + ANCA) = 20–30% of Type I — treat as both.

6 Emergency Management (Start IMMEDIATELY)
General Principles
  • Do NOT wait for biopsy if clinical + serology clear
  • Pulse IV Methylprednisolone 500–1000mg × 3 days → Oral Prednisolone 1mg/kg/day
  • Cyclophosphamide (IV 500mg/m² q2wk × 3mo OR oral 2mg/kg/day × 3mo)
  • Plasma Exchange (PLEX) based on category
Category-Specific Regimens

Anti-GBM: PLEX = life-saving (removes antibody). Dialysis-dependent at presentation → <10% renal recovery.

#### Type II: Immune-Complex (SLE, Cryo, Post-strep, IE) <table class="ctable"><tr><th>Aetiology</th><th>Regimen</th></tr><tr><td>-----------</td><td>---------</td></tr><tr><td>SLE (Class III/IV)</td><td>Pulse steroids → MMF 2–3g/day OR IV CyC 500mg/m² q2wk + steroids</td></tr><tr><td>Cryoglobulinaemia</td><td>PLEX + Rituximab + Steroids (if severe/RPGN)</td></tr><tr><td>Post-strep</td><td>Usually self-limited; supportive; steroids if crescentic >50%</td></tr><tr><td>Infective Endocarditis</td><td>Antibiotics (surgery if needed); immunosuppression contraindicated</td></tr><tr><td>IgA Nephropathy</td><td>Rarely RPGN; if so: steroids ± CyC</td></tr></table>

Identify and treat infection first (IE, post-strep) — immunosuppression can worsen.

#### Type III: Pauci-immune (ANCA Vasculitis — GPA, MPA, EGPA) <table class="ctable"><tr><th>Component</th><th>Regimen</th></tr><tr><td>-----------</td><td>---------</td></tr><tr><td>PLEX</td><td>If Cr >500 μmol/L or pulmonary haemorrhage (PEXIVAS trial: PLEX not routine for all)</td></tr><tr><td>Induction</td><td>Rituximab 375mg/m² weekly × 4 (preferred) OR IV CyC 500mg/m² q2wk × 3mo</td></tr><tr><td>Steroids</td><td>Pulse → Oral taper (RAVE/PEXIVAS: lower cumulative steroid better)</td></tr><tr><td>Maintenance</td><td>Rituximab q6mo OR Azathioprine/MMf (MAINRITSAN: rituximab superior)</td></tr></table>

ANCA RPGN: Rituximab non-inferior to CyC (RAVE); preferred for relapsing/young women.

9 High-Yield FCPS/MRCP Points
[!important] - RPGN = rapid AKI + active sediment + crescents >50%. - Three categories: Anti-GBM (Type I), Immune-complex (Type II), Pauci-immune/ANCA (Type III). - Emergency workup: Anti-GBM, ANCA (MPO/PR3), ANA/dsDNA, C3/C4, Cryoglobulins, Hepatitis. - Anti-GBM: Linear IgG, pulmonary haemorrhage common, PLEX mandatory × 14d. - ANCA vasculitis: Rituximab preferred induction (non-inferior to CyC, fertility-sparing). - Double-positive (Anti-GBM + ANCA): 20–30% — treat as both. - PLEX: Anti-GBM = always; ANCA = if Cr >500 or pulmonary bleed. - SLE RPGN: MMF or CyC + steroids (not PLEX routine). - Prognosis: Dialysis-dependent Anti-GBM → <10% recovery; ANCA high relapse → maintenance.

TrapCorrection
------------------
**RPGN = always ANCA**Only **Type III** = ANCA; Type I = Anti-GBM; Type II = Immune-complex

**Rituximab for all**Anti-GBM: CyC standard (rituximab alternative); ANCA: rituximab preferred; SLE: MMF/CyC
**Crescents = RPGN****>50% crescents = RPGN**; <50% = crescentic GN but not RPGN clinically
**Anti-GBM = no ANCA****20–30% double-positive** — worse prognosis, ↑ relapse
**Wait for biopsy before treatment****NO — treat emergently on serology**; biopsy confirms
**SLE RPGN = PLEX**PLEX not routine for SLE; MMF/CyC + steroids first
**All pulmonary haemorrhage = Goodpasture's**Also ANCA vasculitis, SLE, cryoglobulinaemia, catastrophic APS
**ANCA types****p-ANCA/MPO** = MPA/EGPA/renal-limited; **c-ANCA/PR3** = GPA
**Post-strep = RPGN rare**Post-strep usually self-limited; RPGN rare, steroids if crescents >50%
11 Mnemonics
  • RPGN Types: I = Immune Anti-GBM; II = Immune Complex; III = Pauci-immune ANCA = ICAP
  • Anti-GBM: Linear IgG + Pulmonary = LIP
  • ANCA: PR3 = PR3 GPA; MPO = MPA/EGPA
  • PLEX: Anti-GBM = Always; ANCA = Always if Cr>500/Pulmonary
  • Double-positive: Anti-GBM + ANCA = Awful (worse prognosis)

20 Summary

Rapidly Progressive Glomerulonephritis (RPGN) is a Must Know 🔴 emergency nephrology topic. Definition: Rapid AKI (days–weeks) + active sediment + crescents >50%. Three Categories: Type I = Anti-GBM (linear IgG); Type II = Immune-complex (SLE, Cryo, Post-strep, IE); Type III = Pauci-immune/ANCA (GPA, MPA, EGPA). Emergency Workup: Anti-GBM, ANCA (MPO/PR3), ANA/dsDNA, C3/C4, Cryoglobulins, Hepatitis. Management: Anti-GBM: PLEX daily × 14d + CyC + Steroids (mandatory). ANCA: Rituximab preferred + Steroids; PLEX if Cr >500 or pulmonary bleed (PEXIVAS). SLE: MMF/CyC + steroids (PLEX not routine). Cryo: PLEX + Rituximab + Steroids. Double-positive (20–30%): Treat as both. Prognosis: Dialysis-dependent Anti-GBM <10% recovery; ANCA high relapse → maintenance rituximab. Exam focus: 3 types, serology, PLEX indications, rituximab vs CyC, double-positive, SLE vs ANCA management, crescent threshold.

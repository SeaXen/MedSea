> [!important]
> **Lupus Nephritis = renal manifestation of SLE. Affects 40–60% of SLE patients. ISN/RPS 2003 Class I–VI guides treatment. Class III/IV = proliferative (induction: MMF or CYC + steroids); Class V = membranous (add MMF/CNI); Class VI = advanced sclerosis (no immunosuppression).**

---

## 1. Learning Objectives
- Classify lupus nephritis using ISN/RPS 2003 classification (Class I–VI)
- Correlate histology with clinical presentation and treatment
- Apply induction and maintenance regimens per class
- Monitor response and manage flares

## 2. Epidemiology & Risk Factors

| Feature | Detail |
|---------|--------|
| **Prevalence** | 40–60% of SLE patients develop LN |
| **Demographics** | Young women (9:1 F:M); higher severity in African, Asian, Hispanic ancestry |
| **Risk Factors** | Male sex, young age at SLE onset, high anti-dsDNA, low complement, hypertension, delay in treatment |
| **Genetics** | HLA-DR2/DR3, complement deficiencies (C1q, C4), FCGR polymorphisms, APOL1 (African ancestry) |

---

## 3. ISN/RPS 2003 Classification (Light Microscopy)

| Class | Light Microscopy | Immunofluorescence | Electron Microscopy | Clinical Features | Treatment |
|-------|------------------|-------------------|---------------------|-------------------|-----------|
| **Class I** | Minimal mesangial | Mesangial IgG, IgM, C3, C1q | Mesangial deposits | Asymptomatic, normal renal function | None (monitor) |
| **Class II** | Mesangial proliferative | Mesangial deposits (full-house) | Mesangial deposits | Mild proteinuria, haematuria | Supportive; steroids if proteinuria >1g |
| **Class III** | **Focal (<50% glomeruli) endocapillary/proliferative** | Granular: IgG, IgM, IgA, C3, C1q (capillary + mesangial) | Subendothelial + mesangial | Active sediment, proteinuria, haematuria, renal impairment | **Induction: MMF or CYC + steroids** |
| **Class IV** | **Diffuse (≥50% glomeruli) endocapillary/proliferative** | **Granular "full-house"** (IgG, IgM, IgA, C3, C1q) | **Subendothelial + mesangial** | **Severe**: nephritic/nephrotic, active sediment, renal impairment | **Induction: MMF or CYC + steroids** |
| **Class IV-S / IV-G** | IV-S: Segmental (>50% tuft); IV-G: Global | — | Subendothelial + mesangial | IV-G worse prognosis | Same as Class IV |
| **Class V** | **Membranous** (diffuse GBM thickening) | **Granular capillary wall** (IgG, C3) | **Subepithelial** deposits | Nephrotic syndrome; may overlap III/IV (III+V, IV+V) | **Class V alone: steroids + MMF/CNI**; **Mixed: treat as III/IV** |
| **Class VI** | **Advanced sclerosis** (>90% globally sclerosed) | Variable | Global sclerosis | Advanced CKD, poor response | **No immunosuppression** (renal replacement) |

## 4. Activity & Chronicity Indices

| Activity Index (AI) | Chronicity Index (CI) |
|---------------------|----------------------|
| Endocapillary hypercellularity | Glomerulosclerosis |
| Neutrophils/karyorrhexis | Fibrous crescents |
| Wire loops | Tubular atrophy |
| Cellular crescents | Interstitial fibrosis |
| Hyaline deposits | |
| **Max AI = 24** | **Max CI = 12** |

> [!key]
> **High AI = active inflammation = responds to immunosuppression. High CI = irreversible damage = poor response.**

---

## 5. Clinical Presentation by Class

| Class | Presentation |
|-------|--------------|
| **I/II** | Asymptomatic, microscopic haematuria, subnephrotic proteinuria (<1g) |
| **III** | Proteinuria 1–3g, active sediment (RBC casts), mild renal impairment |
| **IV** | **Nephritic/nephrotic**, heavy proteinuria (>3g), active sediment, HTN, renal impairment |
| **V** | **Nephrotic syndrome** (heavy proteinuria, oedema, hypoalbuminaemia); may have active sediment if mixed |
| **VI** | Advanced CKD, small kidneys, minimal proteinuria |

---

## 6. Serology & Monitoring

| Marker | Significance |
|--------|-------------|
| **Anti-dsDNA** | ↑ titre correlates with disease activity (esp. Class III/IV) |
| **Complement (C3, C4)** | **Low C3/C4 = active disease**; normalisation = response |
| **Anti-C1q** | Highly specific for LN; correlates with activity |
| **Urinalysis** | Proteinuria (UPCR), active sediment (RBC casts, WBC casts) |
| **Renal biopsy** | Gold standard for classification; repeat for flare assessment |

---

## 7. Treatment — Induction (Class III/IV ± V)

| Regimen | Dose/Duration | Notes |
|---------|---------------|-------|
| **MMF (Preferred)** | **MMF 2–3 g/day** (target 2–3g) + **Prednisolone 0.5–1 mg/kg/day** (max 60mg) × 6mo | **ALMS trial**: MMF non-inferior to CYC; better safety; **1st line per KDIGO/EULAR** |
| **Cyclophosphamide (Euro-Lupus)** | **IV CYC 500 mg × 6 fortnightly** (low-dose) + steroids | **Euro-Lupus**: equally effective, less gonadal toxicity; preferred in Caucasians |
| **Cyclophosphamide (NIH)** | **IV CYC 0.5–1g/m² monthly × 6mo** + steroids | Higher dose; more toxicity; used in severe/refractory |
| **Add-on** | **Hydroxychloroquine** (all SLE), ACEi/ARB, BP control, statin | HCQ reduces flares, improves survival |

> [!key]
> **Induction 6 months**. Assess response at 3mo (proteinuria ↓50%) and 6mo (remission).

---

## 8. Treatment — Maintenance (Class III/IV)

| Agent | Dose | Duration |
|-------|------|----------|
| **MMF** | 1–2 g/day | **Minimum 3 years** (longer if high-risk) |
| **Azathioprine** | 1.5–2 mg/kg/day | Alternative if MMF intolerant |
| **CNI (Tac/CsA)** | Tac 0.05–0.1 mg/kg/day | Option for membranous overlap |
| **Prednisolone** | Taper to ≤5–7.5 mg/day | Lowest effective dose |
| **Hydroxychloroquine** | 200–400 mg/day | **Lifelong** (reduces flares, thrombosis, damage) |

---

## 9. Class V (Membranous) LN Treatment

| Scenario | Treatment |
|----------|-----------|
| **Pure Class V** (nephrotic) | **Prednisolone + MMF** (or CNI) × 6mo induction → MMF maintenance |
| **Mixed III+V / IV+V** | **Treat as Class III/IV** (MMF/CYC + steroids) |
| **Subnephrotic proteinuria** | ACEi/ARB + HCQ + monitor |

---

## 10. Flare Management

| Flare Type | Definition | Management |
|------------|------------|------------|
| **Renal Flare** | ↑Proteinuria >50% from baseline, new active sediment, ↑Cr | **Re-biopsy** (class may change); restart induction regimen |
| **Extra-renal Flare** | Arthritis, rash, serositis, cytopenias | Treat per SLE guidelines (steroids, HCQ, immunosuppressants) |

---

## 11. High-Yield FCPS/MRCP Points

> [!important]
> - **LN affects 40–60% SLE**; ISN/RPS Class I–VI guides treatment
> - **Class III/IV = proliferative** → **MMF or CYC + steroids** (6mo induction)
> - **MMF 1st line** (ALMS trial); **Euro-Lupus CYC** (low-dose, less toxicity)
> - **Class V = membranous**: Pure = steroids + MMF/CNI; Mixed = treat as III/IV
> - **Class VI = advanced sclerosis** → **no immunosuppression**
> - **Full-house IF** = IgG, IgM, IgA, C3, C1q (pathognomonic for SLE)
> - **Activity vs Chronicity Index**: AI = treatable; CI = irreversible
> - **Monitor**: anti-dsDNA, C3/C4, UPCR, active sediment
> - **Maintenance**: MMF 1–2g/day × ≥3 years + HCQ lifelong
> - **Flare**: re-biopsy (class may change); restart induction

## 12. Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **All LN classes need immunosuppression** | Class I/II = supportive; Class VI = no immunosuppression |
| **Class V = always pure membranous** | Often mixed (III+V, IV+V) → treat as proliferative |
| **CYC = always high-dose NIH** | **Euro-Lupus low-dose (500mg × 6)** equally effective, less toxicity |
| **MMF = only maintenance** | **MMF = 1st line INDUCTION** (ALMS trial) |
| **Anti-dsDNA = always correlates** | Not 100%; some flares without rise; complement more reliable |
| **Flare = same class** | **Re-biopsy essential** — class often changes (e.g., IV → III, III → V) |
| **HCQ = optional** | **HCQ mandatory all SLE** (reduces flares, thrombosis, organ damage) |
| **Maintenance = 1 year** | **Minimum 3 years** (high relapse if stopped early) |

## 22. Summary

**Lupus Nephritis** is a **Must Know 🔴** topic.
**Key takeaway:** ISN/RPS Class I–VI. **Full-house IF** (IgG, IgM, IgA, C3, C1q) pathognomonic. **Class III/IV = proliferative** → **MMF 2–3g/day + steroids × 6mo** (1st line, ALMS); **Euro-Lupus CYC** 500mg × 6 alternative. **Class V = membranous**: pure = steroids + MMF/CNI; mixed = treat as III/IV. **Class VI = no immunosuppression**. Maintenance **≥3 years** (MMF 1–2g/day) + **HCQ lifelong**. **Flare = re-biopsy + restart induction**. Monitor anti-dsDNA, C3/C4, UPCR, sediment.
**Exam focus:** Classification, full-house IF, induction regimens (MMF vs CYC), Class V pure vs mixed, Class VI, maintenance duration, flare = re-biopsy, HCQ mandate.
**Clinical relevance:** Early biopsy-guided treatment prevents ESRD; MMF improved safety profile; HCQ reduces long-term damage.

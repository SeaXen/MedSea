2 Urinalysis — The "Renal Biopsy of the Poor"
Dipstick Analysis
ParameterNormalAbnormal Significance
-----------------------------------------
**pH**4.5–8.0Low: acidosis, UTI (urea-splitters); High: alkalosis, urea-splitters
**Specific Gravity**1.005–1.030Low: DI, CKD; High: dehydration, glycosuria, proteinuria
**Protein**Negative**Proteinuria** (glomerular > tubular > overflow); false +ve: alkaline, bloody
**Glucose**Negative**Glycosuria** (DM, renal glycosuria, pregnancy, SGLT2i)
**Ketones**Negative**Ketonuria** (DKA, starvation, low carb diet)
**Blood**Negative**Haematuria** (RBC >3/HPF); false +ve: myoglobin, Hb
**Leucocytes**Negative**Leucocyturia** (UTI, sterile pyuria = TB, stones, interstitial nephritis)
**Nitrites**Negative**Nitrituria** (Gram-negative UTI, sensitivity ~50%)
**Bilirubin**NegativeLiver disease, haemolysis
**Urobilinogen**NormalIncreased: haemolysis, haemolytic anaemia
[!key] Proteinuria: Dipstick detects albumin; ACR = gold standard for quantification. Haematuria: Dipstick +ve + microscopy RBC >3/HPF = true haematuria.
3 Urine Microscopy
ElementNormalAbnormal Significance
---------------------------------------
**RBC**0–2/HPF**>3/HPF = haematuria**; **dysmorphic = glomerular**; isomorphic = non-glomerular
**WBC**0–5/HPF**>5/HPF = pyuria**; sterile pyuria = TB, interstitial nephritis, stones, carcinoma
**Epithelial cells**FewSquamous = contamination; Transitional = bladder/TCC; Renal tubular = ATN, interstitial nephritis
**Casts**Hyaline few**RBC cast = GN**; **WBC cast = pyelo/interstitial nephritis**; Granular/muddy brown = ATN; Waxy/broad = advanced CKD; Fatty = nephrotic syndrome
**Crystals**Few urates**Cystine** (hexagonal = cystinuria), **Uric acid** (needles/rosettes), **Calcium oxalate** (envelopes), **Struvite** (coffin-lid = struvite/Proteus)
**Microorganisms**NoneBacteria = UTI; Yeast = Candida; Parasites (Schistosoma)

6 Urinary Casts
Cast TypeAppearanceSignificance
-------------------------------------
**Hyaline**Clear, low refractiveNormal (dehydration, fever, exercise)
**RBC**Red-brown, retain RBC shape**Glomerulonephritis** (pathognomonic)
**WBC / Pus**White, granular**Pyelonephritis**, acute interstitial nephritis
**Granular**Brown, granular**ATN**, severe glomerular disease
**Muddy brown**Dark, granular**ATN** (classic)
**Waxy / Broad**High refractive, broad**Advanced CKD**, chronic tubular damage
**Fatty / Oval fat bodies**Maltese crosses**Nephrotic syndrome** (lipiduria)
7 Urinary Crystals
CrystalShapeClinical Association
-------------------------------------
**Cystine****Hexagonal** (flat, hexagonal plates)**Cystinuria** (autosomal recessive)
**Uric acid**Needles, rosettes, rhombic**Gout, tumour lysis, acidic urine**
**Calcium oxalate****Envelope-shaped** (bipyramidal), dumbbells**Hyperoxaluria, ethylene glycol**, stones
**Calcium phosphate**Amorphous, stellateAlkaline urine, hyperparathyroidism
**Struvite (Triple phosphate)****Coffin-lid / coffin-lid shaped** (prismatic)**Struvite stones** (Proteus, Klebsiella - urea-splitting organisms)
**Ammonium urate**Thorny apples, cactus-likeLiver disease, laxative abuse
**Leucine / Tyrosine**Yellow-brown, fine/silkySevere liver disease, tyrosinaemia
**Cholesterol**Rectangular, notchedNephrotic syndrome (lipiduria)
[!key] Cystine = Hexagonal; Uric acid = Needles/rosettes; CaOx = Envelopes; Struvite = Coffin-lid; Cholesterol = Rectangular/notched (nephrotic); Tyrosine/Leucine = liver failure.
8 Proteinuria Quantification
TestMethodNormalBest For
--------------------------------
**ACR**Albumin:creatinine ratio (spot)<3 mg/mmol**Gold standard**
**PCR**Protein:creatinine ratio<15 mg/mmolTotal protein (incl. non-albumin)
**24h urine protein**24h collection<150 mg/dayGold standard (cumbersome)
**Dipstick**Semi-quantitativeNegativeScreening only
ACR CategoryRange (mg/mmol)TermSignificance
---------------------------------------------------
**A1**<3Normal—
**A2**3–30MicroalbuminuriaEarly diabetic nephropathy, hypertensive nephropathy
**A3**>30MacroalbuminuriaOvert proteinuria; >220 = nephrotic range
PCR vs ACR: PCR measures total protein; ACR specific for albumin. PCR > ACR if tubular proteinuria (Fanconi, myeloma).
9 Haematuria Investigation
StepAction
--------------
1**Confirm**: Repeat dipstick + microscopy (>3 RBC/HPF)
2**Morphology**: Dysmorphic (>50%) = glomerular; Isomorphic = non-glomerular
3**Proteinuria**: ACR >30 + haematuria = glomerular disease
4**Renal function**: eGFR, BP
5**Imaging**: US KUB (stone, tumour, cyst); CT urogram (haematuria >40yr)
6**Cystoscopy**: If >40yr or risk factors (smoker, analgesic abuse)
7**Serology**: ASOT (post-strep), ANA, ANCA, anti-GBM, C3/C4, HIV, Hep B/C

12 High-Yield FCPS/MRCP Points
[!important] - Urinalysis = renal biopsy of the poor; first test in renal disease - Proteinuria: ACR >30 = proteinuria; ACR >300 = nephrotic range - Haematuria: >3 RBC/HPF + dysmorphic = glomerular; isomorphic = non-glomerular - Casts: RBC = GN; WBC = pyelo/interstitial; granular = ATN; waxy = advanced CKD; fatty = nephrotic - Crystals: Cystine (hex), urate (needles), Ca oxalate (envelopes), struvite (coffin-lid) - ACR: <3 (A1), 3–30 (A2), >30 (A3) — nephrotic if >220 mg/mmol - Casts: RBC = GN; WBC = pyelo/interstitial; granular = ATN; waxy = CKD - Crystals: Cystine (hex), urate (needles), Ca oxalate (envelopes), struvite (coffin-lid) - ACR categories: A1 <3, A2 3–30, A3 >30 mg/mmol - Glomerular haematuria: dysmorphic RBCs + RBC casts - Non-glomerular: isomorphic RBCs, no casts
13 Common Confusions / Exam Traps
TrapCorrection
------------------
**Dipstick protein = all protein**Dipstick = albumin only; misses tubular proteins (Bence Jones)
**Microalbuminuria = diabetic only**Also hypertensive, early CKD, heart failure
**Red urine = haematuria**Myoglobin, haemoglobin, beetroot, rifampicin, porphyria
**Proteinuria = always glomerular**Tubular (Fanconi, drugs), overflow (myeloma), postural
**Casts = only in GN**WBC = pyelo/interstitial; granular = ATN; waxy = CKD
**All crystals = stones**Crystalluria ≠ stones (can be normal in concentrated urine)
**ACR = PCR**ACR = albumin only; PCR = total protein (incl. tubular)
**Microalbuminuria = only diabetes**Also hypertension, early CKD, heart failure

# Urological Malignancies (RCC, Bladder Cancer, Urothelial Cancer)

**Related:** [[Male Reproductive Tract Disorders (BPH, Prostate Cancer, Testicular Cancer)]], [[Urinary Tract Obstruction]], [[Urinary Tract Infection (UTI, Pyelonephritis, Prostatitis, Catheter-Associated)]], [[Nephrology and Urology MOC]]

> [!important]
> **Urological malignancies: Renal Cell Carcinoma (RCC) = commonest renal malignancy (3% adult cancers); Bladder Cancer = commonest urothelial malignancy; Upper Tract Urothelial Carcinoma (UTUC) = rare. Key: RCC = VHL mutation, clear cell histology; Bladder Ca = smoking, occupational; UTUC = Balkan nephropathy link. Staging = TNM; Treatment: surgery, systemic therapy (IO, TKI), intravesical (BCG).**

## Renal Cell Carcinoma (RCC)

### Epidemiology & Risk Factors
| Factor | Association |
|--------|-------------|
| **Incidence** | ~3% of adult cancers; 60–70yo; M:F 1.5:1 |
| **Smoking** | **2x risk** (dose-dependent) |
| **Obesity** | ↑ risk (adipokines, insulin resistance) |
| **Hypertension** | Independent risk factor |
| **Acquired Cystic Kidney Disease** | Dialysis patients (30–50x risk) |
| **Genetic** | **VHL syndrome** (3p25 deletion/mutation) → clear cell RCC; HLRCC (fumarate hydratase) → papillary type 2; BHD (FLCN) → chromophobe/oncocytoma |
| **Occupational** | Trichloroethylene, cadmium, asbestos |

### Histological Subtypes
| Subtype | % | Genetics | Prognosis |
|---------|---|----------|-----------|
| **Clear Cell** | **70–80%** | **VHL loss (3p)**, PBRM1, SETD2, BAP1 | Intermediate |
| **Papillary Type 1** | 10–15% | MET activation | Better |
| **Papillary Type 2** | 5–10% | Fumarate hydratase (HLRCC), NRF2 | Worse |
| **Chromophobe** | 5% | TP53, PTEN, BHD (FLCN) | Better |
| **Collecting Duct** | <1% | SMARCB1, SETD2 | Aggressive |
| **Unclassified** | 3–5% | Variable | Variable |

### Clinical Presentation
| Feature | Detail |
|---------|--------|
| **Classic Triad** | Flank pain, haematuria, palpable mass (now <10%) |
| **Incidental** | **>50%** (CT/US for other reasons) |
| **Paraneoplastic** | **Erythrocytosis (EPO)**, hypercalcaemia (PTHrP), hypertension (renin), amyloidosis, Stauffer syndrome (hepatic dysfunction) |
| **Metastatic** | Lung (50%), bone, liver, brain, adrenal |

### Staging (TNM AJCC 8th)
| T Stage | Definition |
|---------|------------|
| **T1a** | ≤4 cm, renal only |
| **T1b** | 4–7 cm, renal only |
| **T2a** | 7–10 cm, renal only |
| **T2b** | >10 cm, renal only |
| **T3a** | Perinephric fat / renal vein / IVC (infradiaphragmatic) |
| **T3b** | IVC supradiaphragmatic / IVC wall |
| **T3c** | IVC wall invasion |
| **T4** | Beyond Gerota's fascia (adrenal, adjacent organs) |
| **N1** | Regional nodes |
| **M1** | Distant mets |

### Management by Stage
| Stage | Treatment |
|-------|-----------|
| **T1a (≤4 cm)** | **Active surveillance** (elderly/comorbid), **Ablation** (RFA/cryo), **Partial nephrectomy** (gold standard if feasible) |
| **T1b (4–7 cm)** | **Partial nephrectomy** (preferred), Radical nephrectomy |
| **T2 (>7 cm)** | **Radical nephrectomy** |
| **T3** | Radical nephrectomy + IVC thrombectomy if needed |
| **T4** | Radical nephrectomy + multivisceral resection (if resectable) |
| **Metastatic (M1)** | **Cytoreductive nephrectomy** (selected) + **Systemic therapy** |

> [!key]
> **Nephron-sparing (partial) = preferred for T1 if technically feasible** — equivalent cancer control, better renal function.

### Systemic Therapy (Metastatic RCC)
| Line | Regimen |
|------|---------|
| **1st Line (Clear Cell)** | **IO + TKI**: Pembrolizumab + axitinib, Nivolumab + cabozantinib, Avelumab + axitinib |
| | **IO + IO**: Nivolumab + ipilimumab (intermediate/poor risk IMDC) |
| **2nd Line** | TKI (cabozantinib, axitinib, lenvatinib+everolimus) if IO-naïve; IO if TKI-naïve |
| **Non-Clear Cell** | TKI (sunitinib, cabozantinib); clinical trials |

### IMDC Prognostic Model (Metastatic RCC)
| Factor | Points |
|--------|--------|
| **Hb < LLN** | 1 |
| **Corrected Ca > ULN** | 1 |
| **KPS <80%** | 1 |
| **Time from diagnosis to treatment <1yr** | 1 |
| **Neutrophils > ULN** | 1 |
| **Platelets > ULN** | 1 |
| **Risk Groups** | Favourable (0), Intermediate (1–2), Poor (≥3) |

## Bladder Cancer

### Epidemiology & Risk Factors
| Factor | Detail |
|--------|--------|
| **Incidence** | 4th commonest cancer in men; 70–80yo; M:F 3:1 |
| **Smoking** | **#1 risk (50% cases); 2–4x risk** |
| **Occupational** | **Aromatic amines** (benzidine, β-naphthylamine) — dye, rubber, leather, textile, paint |
| **Chemotherapy** | Cyclophosphamide (acrolein), ifosfamide |
| **Radiation** | Pelvic RT |
| **Chronic Irritation** | Stones, indwelling catheter, schistosomiasis (S. haematobium → squamous cell) |
| **Genetic** | Lynch syndrome, RB1 mutation |

### Classification by Invasion
| Category | Definition | % |
|----------|------------|---|
| **NMIBC** (Non-Muscle Invasive) | Ta, Tis (CIS), T1 | **75–80%** |
| **MIBC** (Muscle Invasive) | T2–T4 | 20–25% |

### TNM Staging (Bladder)
| Stage | Definition |
|-------|------------|
| **Ta** | Non-invasive papillary |
| **Tis (CIS)** | Carcinoma in situ (flat, high-grade) |
| **T1** | Lamina propria invasion (subepithelial connective tissue) |
| **T2** | Muscularis propria invasion (T2a superficial, T2b deep) |
| **T3** | Perivesical fat (T3a micro, T3b macro) |
| **T4** | Adjacent organs (prostate, uterus, vagina, pelvic/abdominal wall) |

### NMIBC Risk Stratification (EAU/AUA)
| Risk Group | Criteria | Recurrence/Progression Risk |
|------------|----------|----------------------------|
| **Low** | Single, primary, Ta, LG, <3cm, no CIS | Recur 15%, Prog <1% |
| **Intermediate** | Multiple/recurrent Ta LG, or Ta HG <3cm | Recur 30–50%, Prog 5–10% |
| **High** | Any T1, any HG, any CIS, multiple/large Ta HG | Recur 50–70%, Prog 15–30% |

### Management

#### NMIBC
| Step | Treatment |
|------|-----------|
| **TURBT** | **Complete resection** (including muscle for T1); **Re-TURBT** mandatory for T1, incomplete, no muscle in specimen |
| **Low Risk** | **Single post-op instillation** (mitomycin C/epirubicin/gemcitabine within 24h) |
| **Intermediate** | **Induction BCG** (6 weekly) ± **Maintenance BCG** (3 weekly at 3,6,12,18,24,30,36mo) |
| **High Risk** | **BCG induction + maintenance (1–3yr)**; Radical cystectomy if BCG-unresponsive |
| **BCG-Unresponsive** | **Radical cystectomy** (gold standard); Intravesical alternatives (gemcitabine/docetaxel, pembrolizumab, nadofaragene firadenovec) |

#### MIBC
| Step | Treatment |
|------|-----------|
| **Localised (T2–T4a, N0)** | **Radical cystectomy + pelvic lymphadenectomy + urinary diversion** (ileal conduit, neobladder, continent diversion) |
| **Neoadjuvant** | **Cisplatin-based chemo (GC/MVAC) → cystectomy** (survival benefit) |
| **Adjuvant** | If not given neoadjuvant + high risk (pT3/T4, N+) |
| **Trimodality (Bladder Preservation)** | **Maximal TURBT + concurrent chemoradiation** (selected patients) |
| **Metastatic** | **Cisplatin-based chemo (GC)** → **IO maintenance (avelumab)**; IO if cisplatin-ineligible |

## Upper Tract Urothelial Carcinoma (UTUC)

| Feature | Detail |
|---------|--------|
| **Incidence** | 5–10% of urothelial cancers; 70–80yo |
| **Risk Factors** | Smoking, **aromatic amines**, **Balkan nephropathy (aristolochic acid)**, Lynch syndrome, analgesic nephropathy |
| **Presentation** | Haematuria (90%), flank pain, UTI, incidental |
| **Diagnosis** | CT urogram (gold standard), ureteroscopy + biopsy, urine cytology (low sensitivity) |
| **Staging** | Same TNM as bladder (Ta–T4, N, M) |
| **Risk Strat** | **Low**: Ta LG, solitary, <2cm; **High**: T1, HG, CIS, multifocal, >2cm, hydronephrosis |
| **Treatment** | **Radical nephroureterectomy (RNU) + bladder cuff** (gold standard high-risk); **Kidney-sparing** (ureteroscopic ablation/laser, segmental resection) for low-risk |
| **Adjuvant** | Cisplatin-based chemo if pT3/T4 or N+ (controversial) |
| **Surveillance** | **High risk of bladder recurrence** — cystoscopy q3mo × 2yr |

## High-Yield FCPS/MRCP Points

> [!important]
> - **RCC = commonest renal cancer; Clear cell 70–80%; VHL mutation**
> - **RCC classic triad (flank pain, haematuria, mass) <10%; incidental >50%**
> - **RCC paraneoplastic: erythrocytosis (EPO), hypercalcaemia (PTHrP)**
> - **RCC staging: T1a ≤4cm → partial nephrectomy/ablation/surveillance; T1b 4–7 → partial; T2+ → radical**
> - **Metastatic RCC: IO+TKI 1st line (pembro+axiti, nivo+cabo); IMDC risk stratification**
> - **Bladder Ca: smoking #1; occupational aromatic amines; M>F 3:1**
> - **NMIBC (75%): Ta, Tis, T1; MIBC (25%): T2+**
> - **NMIBC risk: Low (single Ta LG) → single post-op chemo; High (T1/HG/CIS) → BCG induction+maintenance**
> - **BCG-unresponsive → radical cystectomy**
> - **MIBC: neoadjuvant cisplatin chemo → radical cystectomy + diversion**
> - **UTUC: Balkan nephropathy (aristolochic acid); RNU + bladder cuff; high bladder recurrence**
> - **Schistosomiasis → squamous cell bladder cancer**

## Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **All RCC = radical nephrectomy** | **T1a/T1b → partial/ablation/surveillance** (nephron-sparing) |
| **Clear cell = only RCC** | Papillary, chromophobe, collecting duct, unclassified |
| **VHL = only clear cell** | VHL = clear cell; but also sporadic clear cell have VHL loss |
| **Bladder Ca = only TCC** | Squamous (schisto), adenocarcinoma, small cell, sarcomatoid |
| **NMIBC = always BCG** | Low risk = single post-op chemo only; BCG for intermediate/high |
| **BCG = 6 weeks only** | **Maintenance BCG (1–3yr) essential for high risk** (SWOG/Sylvester) |
| **MIBC = cystectomy only** | **Neoadjuvant cisplatin chemo → cystectomy** = standard |
| **UTUC = same as bladder** | RNU + cuff; kidney-sparing for low risk; high bladder recurrence risk |
| **Schistosomiasis = TCC** | **Squamous cell carcinoma** (chronic inflammation) |
| **RCC mets = only lung** | Bone, liver, brain, adrenal also common |

## Mnemonics

- **RCC Subtypes**: **C**lear (70%), **P**apillary (15%), **C**hromophobe (5%), **C**ollecting **D**uct (<1%) = **CPCCD**
- **RCC Paraneoplastic**: **E**rythrocytosis (EPO), **H**ypercalcaemia (PTHrP), **H**ypertension (Renin) = **EHH**
- **RCC Staging**: **T1**a ≤4, **T1**b 4–7, **T2** 7–10, **T3** fat/vein/IVC, **T4** Gerota's = **T1T2T3T4**
- **RCC Systemic**: **I**O+**T**KI 1st line; **I**MDC risk = **ITC**
- **Bladder Risk**: **S**moking **#1**; **A**romatic **A**mines = **S#1 AA**
- **NMIBC**: **T**a, **T**is, **T**1 = **TTT**
- **NMIBC Risk**: **L**ow (Ta LG single), **I**ntermediate (multi/recur Ta LG), **H**igh (T1/HG/CIS) = **LIH**
- **BCG**: **I**nduction 6wk + **M**aintenance 1–3yr = **IM**
- **MIBC**: **N**eo**A**djuvant **C**isplatin → **C**ystectomy = **NAC**
- **UTUC**: **R**adical **N**ephro**U**reterectomy (**R**NU) + **B**ladder **C**uff = **RNB**

## Summary

**Urological Malignancies** is a **Must Know 🔴** topic.
**Key takeaway:** **RCC**: clear cell 70% (VHL loss); paraneoplastic: erythrocytosis, hypercalcaemia. **Staging**: T1a ≤4cm → partial/ablation/surveillance; T2+ → radical. **Metastatic**: IO+TKI 1st line; IMDC risk stratification. **Bladder Ca**: smoking #1, aromatic amines. **NMIBC** (75%): Ta/Tis/T1. Risk: Low (Ta LG single) → post-op chemo; High (T1/HG/CIS) → **BCG induction + maintenance**. **BCG-unresponsive** → radical cystectomy. **MIBC**: neoadjuvant cisplatin → radical cystectomy + diversion. **UTUC**: Balkan = aristolochic acid; **RNU + cuff**; bladder surveillance q3mo. **Schistosomiasis** → squamous cell.
**Exam focus:** RCC subtypes/staging/management, metastatic therapy, bladder risk factors, NMIBC risk stratification, BCG protocol, MIBC neoadjuvant, UTUC management, schistosomiasis histology.
**Clinical relevance:** Nephron-sparing surgery preserves renal function; IO revolutionised metastatic RCC; BCG maintenance critical for NMIBC; cystectomy morbidity high but curative for MIBC.

# Functional Anatomy and Physiology of the Kidney and Urinary Tract

**Related:** [[Glomerular Filtration Rate]], [[Investigation of Renal and Urinary Tract Disease]], [[Urine Investigations]], [[Blood Tests in Renal Disease]], [[Acute Kidney Injury (AKI)]], [[Chronic Kidney Disease (CKD)]], [[Nephrology and Urology MOC]]

> [!important]
> **Kidney = bean-shaped retroperitoneal organs (T12–L3); Right lower than left (liver). Functional unit = Nephron (~1 million/kidney). Key functions: Filtration (GFR ~125 mL/min), Reabsorption (99%), Secretion, Endocrine (EPO, Renin, 1,25-OH Vit D), Acid-base, BP regulation. Understanding tubular handling along nephron segments is essential for interpreting diuretic action, acid-base, electrolyte disorders.**

## Gross Anatomy

### Kidney Position & Relations
| Feature | Right Kidney | Left Kidney |
|---------|--------------|-------------|
| **Vertebral Level** | T12–L3 | T12–L3 (slightly higher) |
| **Superior Relation** | Right adrenal | Left adrenal |
| **Anterior Relations** | Liver (hepatorenal recess), duodenum, hepatic flexure | Spleen, stomach, pancreas, splenic flexure |
| **Posterior Relations** | Diaphragm, 12th rib, quadratus lumborum, psoas | Diaphragm, 11th/12th rib, quadratus lumborum, psoas |
| **Hilum Structures** | Renal vein (anterior), Renal artery, Ureter (posterior) — **VAU** | Same |

### Renal Capsules & Fasciae
| Layer | Description |
|-------|-------------|
| **Fibrous Capsule** | True capsule, adherent to parenchyma |
| **Perirenal Fat** | Adipose tissue surrounding kidney |
| **Renal Fascia (Gerota's)** | Condensation of fascia: **Anterior (Zuckerkandl)**, **Posterior (Lateroconal)** — encloses kidney + perirenal fat |
| **Pararenal Fat** | External to renal fascia |

### Internal Anatomy
| Region | Features |
|--------|----------|
| **Cortex** | Outer 1/3; contains glomeruli, proximal/distal tubules, cortical collecting ducts |
| **Medulla** | Inner 2/3; **8–18 renal pyramids**; base at cortex, apex = **papilla** draining into minor calyx |
| **Renal Columns (Bertin)** | Cortical tissue between pyramids |
| **Calyces** | **Minor (2–3 per pyramid)** → **Major (2–3)** → **Renal Pelvis** → **Ureter** |
| **Renal Sinus** | Fat-filled space containing pelvis, calyces, vessels, lymphatics |

### Vasculature
| Vessel | Description |
|--------|-------------|
| **Renal Artery** | Branch of abdominal aorta (L1–L2); **Segmental branches**: Superior, Anterosuperior, Anteroinferior, Inferior, Posterior |
| **Interlobar Arteries** | Through renal columns to corticomedullary junction |
| **Arcuate Arteries** | Arch over base of pyramids at corticomedullary junction |
| **Interlobular Arteries** | Into cortex → **Afferent Arterioles** → **Glomerulus** → **Efferent Arterioles** |
| **Peritubular Capillaries** | From efferent (cortical nephrons) → surround tubules |
| **Vasa Recta** | From efferent (juxtamedullary nephrons) → descend into medulla → countercurrent exchange |
| **Renal Vein** | Anterior to artery at hilum; Left longer, passes anterior to aorta; **Left gonadal/adrenal/suprarenal veins drain into left renal vein** |

### Lymphatics & Nerves
| System | Details |
|--------|---------|
| **Lymphatics** | Drain to lateral aortic / hilar nodes |
| **Sympathetic (T10–L1)** | Vasoconstriction (α1), Renin release (β1), Na+ reabsorption |
| **Parasympathetic (Vagus)** | Minor role |
| **Afferent (Sensory)** | Renal colic (T10–L1), visceral pain |

## Nephron — Functional Unit

### Types of Nephrons
| Type | % | Features |
|------|---|----------|
| **Cortical** | **85%** | Short loops (outer medulla); glomeruli in outer cortex; peritubular capillaries |
| **Juxtamedullary** | **15%** | Long loops (inner medulla/papilla); glomeruli near corticomedullary junction; **Vasa recta** essential for concentration |

### Nephron Segments & Transport

#### 1. Glomerulus (Filtration)
| Parameter | Value/Mechanism |
|-----------|-----------------|
| **GFR** | ~125 mL/min (180 L/day) |
| **Filtration Barrier** | **Fenestrated Endothelium** → **GBM** (type IV collagen, heparan sulphate, negative charge) → **Podocyte Slit Diaphragm** (nephrin) |
| **Forces** | **P_GC** (hydrostatic, ~45 mmHg) — **P_BS** (hydrostatic, ~15 mmHg) — **π_GC** (oncotic, ~30 mmHg) = **NFP ~10 mmHg** |
| **Autoregulation** | **Myogenic** (afferent stretch) + **Tubuloglomerular Feedback (TGF)** (macula densa NaCl sensing → afferent arteriole tone) |
| **Regulation** | Afferent (myogenic, TGF, sympathetic, Ang II, PGE2, NO); Efferent (Ang II > Afferent) |

#### 2. Proximal Convoluted Tubule (PCT) — **Reabsorbs ~65% of filtered load**
| Transport | Mechanism | Clinical Relevance |
|-----------|-----------|-------------------|
| **Na+** | **Na+/H+ Exchanger (NHE3)** (luminal), **Na+/K+ ATPase** (basolateral) | **Carbonic Anhydrase Inhibitors** (acetazolamide) → ↓ Na+/H+ exchange → ↓ Na+, HCO3- reabsorption |
| **HCO3-** | **Na+/H+** → H+ + HCO3- → H2CO3 → CO2 + H2O (CA) → reabsorbed | **Proximal RTA (Type 2)** = HCO3- wasting |
| **Glucose** | **SGLT2** (early PCT), **SGLT1** (late PCT) → secondary active | **SGLT2 Inhibitors** (dapagliflozin) → glycosuria; **SGLT1/2** = Fanconi (cystinosis) |
| **Amino Acids** | Na+-dependent cotransporters | **Hartnup** (neutral AA), **Cystinuria** (dibasic AA) |
| **Phosphate** | **Na+/Pi Cotransporter (NaPi-IIa)** (PTH inhibits) | **PTH** → phosphaturia; **FGF23** → phosphaturia |
| **Uric Acid** | **URAT1** (reabsorption), **ABCG2** (secretion) | **Uricosurics** (probenecid) → inhibit URAT1; **Allopurinol** → xanthine oxidase inhibitor |
| **Water** | Passive (aquaporin-1) | **Osmotic Diuresis** (mannitol, glucose) → ↓ water reabsorption |
| **Secretion** | Organic acids/bases, drugs (PAH, penicillin, creatinine) | **Cimetidine/Trimethoprim** inhibit creatinine secretion → ↑ serum Cr |

#### 3. Loop of Henle — **Creates Medullary Gradient**
| Segment | Permeability | Transport |
|---------|-------------|-----------|
| **Thin Descending** | **Water HIGH** (AQP1); Solute LOW | Passive water efflux → tubular fluid concentrated |
| **Thin Ascending** | Water LOW; Solute PASSIVE (NaCl) | Passive NaCl efflux → interstitium |
| **Thick Ascending (TAL)** | Water IMPERMEABLE; Solute ACTIVE | **NKCC2** (Na+/K+/2Cl- cotransporter) → **generates gradient**; **Ca2+/Mg2+** paracellular (paracellin-1/claudin-16); **ROMK** (K+ recycling) |

> [!key]
> **TAL = "Diluting Segment"**: Generates medullary hypertonicity; produces hypotonic tubular fluid.
> **NKCC2** = Target of **Loop Diuretics** (furosemide, bumetanide).
> **ROMK** = K+ channel; mutations = **Bartter Syndrome**.
> **Ca2+/Mg2+** reabsorption = Paracellular, driven by lumen-positive voltage.

#### 4. Distal Convoluted Tubule (DCT)
| Transport | Mechanism | Clinical Relevance |
|-----------|-----------|-------------------|
| **Na+** | **NCC** (Na+/Cl- cotransporter) | **Thiazides** inhibit NCC → natriuresis, ↓ Ca2+ excretion |
| **Ca2+** | **TRPV5** (apical), **NCX1/PMCA** (basolateral) | **PTH** ↑ Ca2+ reabsorption (via TRPV5); **Thiazides** ↑ Ca2+ reabsorption |
| **K+** | **ROMK** (secretion), **H+/K+ ATPase** (intercalated cells) | **K+ secretion** ↑ with flow, aldosterone |
| **Acid-Base** | **α-Intercalated**: H+ ATPase (acid secretion); **β-Intercalated**: HCO3- secretion | **Distal RTA (Type 1)** = H+ ATPase defect → metabolic acidosis, nephrocalcinosis |

#### 5. Collecting Duct (CD) — **Final Urine Modification**
| Cell Type | Function | Regulation |
|-----------|----------|------------|
| **Principal Cells** | **ENaC** (Na+ reabsorption) → lumen-negative voltage → **K+ secretion (ROMK)**, **H+ secretion (H+ ATPase)** | **Aldosterone** ↑ ENaC, ROMK, H+ ATPase; **ADH (Vasopressin)** ↑ AQP2 water channels |
| **α-Intercalated** | **H+ ATPase** (acid secretion), **HCO3- reabsorption** | **Acidosis** ↑ H+ secretion |
| **β-Intercalated** | **Pendrin** (HCO3- secretion, Cl- reabsorption) | **Alkalosis** ↑ HCO3- secretion |

> [!key]
> **ENaC** = Target of **Amiloride / Triamterene** (K+-sparing diuretics).
> **AQP2** = Target of **ADH**; mutations = **Nephrogenic DI**.
> **Pendrin** mutations = **Pendred Syndrome** (deafness + goitre).

## Glomerular Filtration Rate (GFR) — Determinants & Regulation

| Factor | Effect on GFR |
|--------|---------------|
| **Renal Plasma Flow (RPF)** | ↑ RPF → ↑ GFR (flow-limited) |
| **Glomerular Hydrostatic Pressure (P_GC)** | ↑ P_GC → ↑ GFR (afferent dilation / efferent constriction) |
| **Bowman's Space Pressure (P_BS)** | ↑ P_BS → ↓ GFR (obstruction) |
| **Glomerular Oncotic Pressure (π_GC)** | ↑ π_GC → ↓ GFR (high protein, efferent constriction) |
| **Filtration Coefficient (Kf)** | ↑ Kf → ↑ GFR (surface area, permeability) |

### Autoregulation (Maintains GFR constant over MAP 80–180 mmHg)
| Mechanism | Mechanism |
|-----------|-----------|
| **Myogenic** | Afferent arteriole stretch → vasoconstriction (stretch-activated Ca2+ channels) |
| **Tubuloglomerular Feedback (TGF)** | **Macula Densa** senses ↑ NaCl delivery → adenosine/ATP → afferent vasoconstriction |

### Hormonal/Neural Regulation
| Agent | Afferent | Efferent | Net GFR |
|-------|----------|----------|---------|
| **Angiotensin II** | Mild constriction | **Strong constriction** | Maintained/↑ (↑ P_GC) |
| **Prostaglandins (PGE2/PGI2)** | **Dilation** | — | **Maintained** (protective in volume depletion) |
| **NO** | **Dilation** | — | **↑ GFR** |
| **Sympathetic (α1)** | **Constriction** | Constriction | **↓ GFR** (severe volume depletion, shock) |
| **Dopamine (Low dose)** | **Dilation** | — | **↑ GFR** (renal dose) |
| **ANP/BNP** | **Dilation** | Constriction | **↑ GFR** |

> [!key]
> **NSAIDs** inhibit prostaglandins → **loss of afferent dilation** → ↓ GFR (especially in volume depletion, CKD, heart failure).
> **ACEi/ARB** ↓ Ang II → **efferent dilation** → ↓ P_GC → ↓ GFR (therapeutic in proteinuria; caution in bilateral RAS/solitary kidney).

## Countercurrent Multiplication & Urine Concentration

### 1. Countercurrent Multiplication (Loop of Henle)
| Step | Mechanism |
|------|-----------|
| **TAL** actively reabsorbs NaCl (NKCC2) → interstitium becomes hypertonic |
| **Descending limb** permeable to water → water leaves → tubular fluid concentrated |
| **Ascending limb** impermeable to water → NaCl leaves passively (thin) + actively (thick) → interstitium more hypertonic |
| **Recycling** → progressive increase in medullary osmolarity (cortex ~300 → papilla ~1200–1400 mOsm/kg) |

### 2. Countercurrent Exchange (Vasa Recta)
| Feature | Function |
|---------|----------|
| **Descending vasa recta** | Gains solute, loses water → equilibrates with interstitium |
| **Ascending vasa recta** | Loses solute, gains water → returns to cortex |
| **U-shaped** | Minimizes washout of medullary gradient |

### 3. Urine Concentration (Collecting Duct)
| Step | Mechanism |
|------|-----------|
| **ADH (Vasopressin)** binds V2 receptor → cAMP → **AQP2 vesicles** insert into apical membrane |
| **Water** moves passively down osmotic gradient → concentrated urine |
| **Max concentrating ability** | ~1200–1400 mOsm/kg (depends on medullary gradient + ADH) |
| **Dilute urine** | ADH absent → CD impermeable to water → hypotonic urine (<100 mOsm/kg) |

## Acid-Base Handling by Kidney

### Net Acid Excretion (NAE)
**NAE = (Titratable Acid + NH4+) — HCO3- excreted**
**Normal NAE = 40–100 mEq/day** (matches metabolic acid production)

### Mechanisms by Segment
| Segment | Mechanism |
|---------|-----------|
| **PCT** | **HCO3- reabsorption** (~80–90%): Na+/H+ exchanger (NHE3) + Carbonic Anhydrase |
| **TAL** | **HCO3- reabsorption** (~10–15%): Na+/H+ exchanger |
| **CD (α-Intercalated)** | **H+ secretion** (H+ ATPase); **NH4+ trapping**: NH3 diffuses → NH4+ trapped in acidic urine; **HCO3- regeneration** (new HCO3- added to blood) |
| **β-Intercalated** | **HCO3- secretion** (Pendrin) in alkalosis |

### Ammoniagenesis
| Factor | Effect |
|--------|--------|
| **Acidosis** | ↑ Glutamine → ↑ NH4+ + HCO3- (new HCO3- to blood) |
| **Hypokalaemia** | ↑ Ammoniagenesis (paradoxical) |
| **Cortisol** | ↑ Glutaminase → ↑ NH4+ |

> [!key]
> **NH4+ excretion** = major adaptive response to chronic acidosis.
> **Urine pH** can be <5.0 in acidosis (maximal acidification).
> **Distal RTA (Type 1)** = inability to acidify urine <5.5 despite acidosis.

## Endocrine Functions of Kidney

| Hormone | Source | Function | Clinical Relevance |
|---------|--------|----------|-------------------|
| **Erythropoietin (EPO)** | Peritubular fibroblasts (cortex) | Stimulates erythropoiesis | **Anaemia of CKD** = EPO deficiency → recombinant EPO/darbepoetin |
| **Renin** | Juxtaglomerular cells (afferent arteriole) | Cleaves angiotensinogen → Ang I → Ang II (ACE) | **RAAS activation** = HTN, volume retention; **ACEi/ARB** block |
| **1,25-(OH)2 Vitamin D (Calcitriol)** | Proximal tubule (1α-hydroxylase) | Active Vit D → Ca2+ absorption, bone | **CKD-MBD** = ↓ calcitriol → secondary hyperparathyroidism; **Alfacalcidol/Calcitriol** replacement |
| **Prostaglandins (PGE2, PGI2)** | Medullary interstitial cells | Vasodilation (afferent), renin release, medullary blood flow | **NSAIDs** inhibit → ↓ GFR, ↑ K+, ↓ renin |
| **Kallikrein-Kinin** | Distal tubule | Vasodilation, natriuresis | Role in BP regulation |

## Clinical Correlations — Diuretic Sites of Action

| Diuretic Class | Site | Target | Effects |
|----------------|------|--------|---------|
| **Carbonic Anhydrase Inhibitors** (Acetazolamide) | PCT | Carbonic Anhydrase | ↓ Na+/HCO3- reabsorption; metabolic acidosis; weak diuretic |
| **Loop Diuretics** (Furosemide, Bumetanide, Torsemide) | **TAL (NKCC2)** | NKCC2 | **Potent natriuresis**; ↓ medullary gradient; ↑ Ca2+/Mg2+ excretion; ototoxicity |
| **Thiazides** (Hydrochlorothiazide, Chlorthalidone, Indapamide) | **DCT (NCC)** | NCC | Natriuresis; **↓ Ca2+ excretion** (↑ Ca2+ reabsorption); hyponatraemia, hypokalaemia, hyperuricaemia |
| **K+-Sparing (ENaC Blockers)** (Amiloride, Triamterene) | **CD (ENaC)** | ENaC | Weak natriuresis; **↓ K+ secretion** (hyperkalaemia risk); no effect on Ca2+ |
| **K+-Sparing (Aldosterone Antagonists)** (Spironolactone, Eplerenone) | **CD (MR)** | Mineralocorticoid Receptor | ↓ ENaC expression; **↓ K+ secretion**; anti-androgen (spironolactone); **hyperkalaemia** |
| **SGLT2 Inhibitors** (Dapagliflozin, Empagliflozin) | **PCT (SGLT2)** | SGLT2 | **Glycosuria + natriuresis**; ↓ intraglomerular pressure; CV/renal protection |
| **Osmotic Diuretics** (Mannitol) | PCT / Descending Limb | Osmotic pressure | ↓ water reabsorption; used for cerebral oedema, forced diuresis |

> [!key]
> **Loop diuretics** = most potent; **Thiazides** = first-line HTN (with Ca2+ benefit); **K+-sparing** = hyperkalaemia risk (especially with ACEi/ARB, CKD); **SGLT2i** = game-changer in DKD/CKD/HF.

## Electrolyte Handling Summary

| Electrolyte | Filtered Load | Reabsorbed (%) | Primary Site | Key Regulators |
|-------------|---------------|----------------|--------------|----------------|
| **Na+** | ~25,000 mmol/day | 99.5% | PCT (65%), TAL (25%), DCT (5%), CD (3%) | Ang II, Aldosterone, ANP, Sympathetic |
| **K+** | ~700 mmol/day | 90% (secreted) | PCT/TAL (reabsorption), CD (secretion) | Aldosterone, Flow, pH, Na+ delivery |
| **Cl-** | ~25,000 mmol/day | 99% | Passive (with Na+), NKCC2, NCC, Pendrin | Na+ coupling |
| **HCO3-** | ~4,300 mmol/day | ~100% (reclaimed) | PCT (80–90%), TAL (10–15%), CD (new HCO3- via NH4+) | Acidosis, Ang II, CA inhibitors |
| **Ca2+** | ~10,000 mg/day | 98% | PCT (60%), TAL (20% paracellular), DCT (10% active TRPV5), CD | PTH, Vit D, Thiazides, Loop diuretics |
| **Mg2+** | ~2,400 mg/day | 95% | PCT (15%), TAL (70% paracellular), DCT (10% TRPM6) | PTH, EGF, Loop diuretics |
| **PO4** | ~4,000 mg/day | 85–90% | PCT (NaPi-IIa) | PTH, FGF23, Vit D |
| **Uric Acid** | ~600 mg/day | 90% (reabsorbed) + secreted | PCT (URAT1 reabs, ABCG2 sec) | URAT1 inhibitors (probenecid), volume status |

> [!key]
> **Ca2+**: Thiazides ↑ reabsorption (DCT); Loops ↓ reabsorption (TAL paracellular voltage).
> **Mg2+**: Loop diuretics cause Mg2+ wasting (TAL paracellular).
> **K+**: Secretion in CD driven by Na+ reabsorption (ENaC) → lumen-negative voltage.
> **HCO3-**: PCT reabsorbs bulk; CD generates new HCO3- via NH4+ excretion.

## High-Yield FCPS/MRCP Points

> [!important]
> - **Nephron = functional unit**: Cortical (85%) vs Juxtamedullary (15%, long loops, vasa recta)
> - **GFR ~125 mL/min**; Autoregulation (myogenic + TGF) maintains constant over MAP 80–180
> - **Filtration barrier**: Endothelium → GBM (negative charge) → Podocyte slit diaphragm (nephrin)
> - **PCT**: 65% reabsorption; NHE3 (Na+/H+), SGLT2 (glucose), NaPi-IIa (phosphate), URAT1 (urate)
> - **Loop of Henle**: Countercurrent multiplication; TAL = NKCC2 (loop diuretics); Ca2+/Mg2+ paracellular
> - **DCT**: NCC (thiazides); TRPV5 (Ca2+); ROMK (K+); α/β-intercalated (acid-base)
> - **CD**: ENaC (amiloride); AQP2 (ADH); ROMK (K+); H+ ATPase (acid); Pendrin (β-intercalated)
> - **GFR Regulation**: Afferent (myogenic, TGF, prostaglandins, NO); Efferent (Ang II); ACEi/ARB → efferent dilation
> - **Countercurrent**: Multiplication (TAL) + Exchange (Vasa Recta) → Medullary gradient → Concentration (ADH → AQP2)
> - **Acid-Base**: PCT (HCO3- reclamation), CD (H+ secretion, NH4+ trapping, new HCO3- generation)
> - **Endocrine**: EPO (anaemia), Renin (RAAS), Calcitriol (Ca2+), Prostaglandins (NSAIDs inhibit → ↓ GFR)
> - **Diuretics**: CAi (PCT), Loop (TAL/NKCC2), Thiazide (DCT/NCC), K+-sparing (ENaC/MR), SGLT2i (PCT/SGLT2), Osmotic (PCT/Descending)
> - **Electrolytes**: Na+ (RAAS), K+ (aldosterone/flow), Ca2+ (PTH/Vit D/thiazides), Mg2+ (loop diuretics waste), PO4 (PTH/FGF23), UA (URAT1/ABCG2)

## Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **GFR = RPF** | GFR = ~20% of RPF (Filtration Fraction ~0.2) |
| **All Na+ reabsorbed in PCT** | PCT = 65%; TAL = 25%; DCT = 5%; CD = 3–5% |
| **Loop diuretics spare K+** | **Loop diuretics cause K+ WASTING** (↑ flow to CD → ↑ K+ secretion) |
| **Thiazides cause hypercalciuria** | **Thiazides ↓ Ca2+ excretion** (↑ Ca2+ reabsorption in DCT) |
| **K+-sparing = only amiloride** | **Also spironolactone/eplerenone (MR antagonists)** |
| **ADH acts on PCT** | **ADH acts on CD (AQP2)** |
| **Aldosterone acts on PCT** | **Aldosterone acts on CD (principal cells → ENaC)** |
| **Bicarbonate reabsorbed in PCT only** | **New HCO3- generated in CD via NH4+ excretion** |
| **EPO produced by liver in adults** | **EPO produced by renal peritubular fibroblasts (cortex)** |
| **Renin from macula densa** | **Renin from JG cells (afferent arteriole)**; Macula densa = TGF sensor |

## Mnemonics

- **Nephron Segments**: **P**CT → **L**oop → **D**CT → **C**D = **PLDC**
- **Loop of Henle**: **D**escending = **W**ater out; **A**scending = **S**olute out = **DW-AS**
- **TAL Transport**: **N**a/**K**/2**C**l = **NKCC2** (Furosemide target)
- **DCT Transport**: **N**a/**C**l = **NCC** (Thiazide target)
- **CD Transport**: **E**Na**C** (Amiloride target); **A**Q**P**2 (ADH target)
- **Diuretic Sites**: **C**A (PCT), **L**oop (TAL), **T**hiazide (DCT), **K**-sparing (CD) = **CLTK**
- **Ca2+ Handling**: **T**hiazides **↑** Ca2+ reabsorption; **L**oop **↓** Ca2+ = **T↑ L↓**
- **K+ Secretion**: **A**ldosterone + **F**low + **N**a+ delivery = **AFN**
- **Acid-Base**: **P**CT **R**eclaims **H**CO3; **C**D **G**enerates **N**ew **H**CO3 = **PR-CHGN**
- **NH4+ Trapping**: **N**H3 diffuses → **H**+ adds → **N**H4+ trapped = **NHN**
- **Endocrine**: **E**PO, **R**enin, **C**alcitriol, **P**rostaglandins = **ERCP**
- **Autoregulation**: **M**yogenic + **T**GF = **MT**

## Summary

**Functional Anatomy and Physiology of the Kidney and Urinary Tract** is a **Must Know 🔴** topic.
**Key takeaway:** **Kidney**: T12–L3, hilum VAU. **Nephron**: 1M; Cortical (85%) vs Juxtamedullary (15%). **Glomerulus**: Endothelium → GBM (neg charge) → Podocyte (nephrin). **GFR**: P_GC - P_BS - π_GC; Autoregulation (Myogenic + TGF). **Segments**: PCT (65%, NHE3/SGLT2/NaPi/URAT1) → Loop (Descending H2O, Ascending solute; TAL NKCC2) → DCT (NCC, TRPV5, ROMK, intercalated) → CD (ENaC, AQP2, ROMK, H+ ATPase, Pendrin). **Autoregulation**: Afferent (myogenic, TGF, PG, NO); Efferent (Ang II). **ACEi/ARB** = efferent dilation. **Countercurrent**: Multiplication (TAL) + Exchange (Vasa recta) → Gradient → ADH (V2/AQP2) → Concentration. **Acid-Base**: PCT HCO3- reclamation; CD H+ secretion + NH4+ trapping = new HCO3-. **Endocrine**: EPO (fibroblasts), Renin (JG), Calcitriol (PCT), PG (NSAIDs inhibit). **Diuretics**: CAi(PCT), Loop(TAL/NKCC2), Thiazide(DCT/NCC), K+-sparing(ENaC/MR), SGLT2i(PCT/SGLT2), Osmotic. **Electrolytes**: K+ secretion (aldosterone, flow); Ca2+ thiazide↑/loop↓; Mg2+ loop wasting; PO4 PTH/FGF23. **Bartter** (TAL), **Gitelman** (DCT), **Liddle** (ENaC), **RTA** (Type 1 H+ ATPase, Type 2 NHE3/CA).
**Exam focus:** Nephron segments & transporters, GFR regulation, countercurrent mechanism, acid-base, endocrine, diuretic sites/mechanisms, electrolyte handling, tubular disorders.
**Clinical relevance:** Understanding tubular transport enables rational diuretic use, acid-base management, electrolyte correction, and interpretation of renal drug effects.

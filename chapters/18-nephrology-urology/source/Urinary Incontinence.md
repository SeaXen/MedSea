# Urinary Incontinence

**Related:** [[Clinical Examination of the Kidney and Urinary Tract]], [[Urinary Tract Infection (UTI, Pyelonephritis, Prostatitis, Catheter-Associated)]], [[Male Reproductive Tract Disorders (BPH, Prostate Cancer, Testicular Cancer)]], [[Nephrology and Urology MOC]]

> [!important]
> **Urinary incontinence (UI) = involuntary leakage of urine. Types: Stress (SUI), Urge (UUI), Mixed (MUI), Overflow, Functional. Prevalence: Women > Men; increases with age. Impact: QoL, skin breakdown, falls, social isolation. Management: Conservative → Pharmacological → Surgical.**

---

## Learning Objectives
- Classify types of urinary incontinence
- Assess and diagnose UI (history, examination, investigations)
- Apply conservative, pharmacological, and surgical management
- Address special populations (elderly, post-prostatectomy, neurogenic)
- Apply to FCPS/MRCP clinical scenarios

## Classification of Urinary Incontinence

| Type | Mechanism | Key Features |
|------|-----------|--------------|
| **Stress UI (SUI)** | ↑ Intra-abdominal pressure > urethral closure pressure | Leakage with cough, sneeze, exercise, lifting; **no urgency** |
| **Urge UI (UUI)** | Detrusor overactivity (involuntary contractions) | **Urgency** + frequency + nocturia + leakage; "key-in-lock" syndrome |
| **Mixed UI (MUI)** | Combination of SUI + UUI | Features of both; one may predominate |
| **Overflow UI** | Chronic urinary retention → bladder overdistension | **Constant dribbling**, poor stream, hesitancy, incomplete emptying; **PV residual high** |
| **Functional UI** | Normal lower urinary tract but physical/cognitive impairment | Mobility issues, dementia, environmental barriers |
| **Continuous UI** | Fistula, ectopic ureter, sphincter destruction | Constant leakage (no storage) |

## Prevalence & Risk Factors

| Population | Prevalence |
|------------|------------|
| **Women** | 25–45% (any UI); SUI commonest in young/middle; UUI increases with age |
| **Men** | 10–30% (post-prostatectomy SUI common; UUI with BPH) |
| **Elderly (>65)** | 30–50% (nursing home higher) |

| Risk Factor | SUI | UUI |
|-------------|-----|-----|
| **Age** | ↑ | ↑↑ |
| **Parity/Vaginal Delivery** | ↑↑ | ↑ |
| **Menopause** | ↑ (oestrogen loss) | ↑ |
| **Obesity** | ↑↑ | ↑ |
| **Chronic Cough/Constipation** | ↑ | |
| **Neurological Disease** | | ↑ (MS, stroke, Parkinson's) |
| **BPH/Prostate Surgery** | ↑ (post-TURP) | ↑ (BOO) |
| **Medications** | Diuretics, α-blockers | Anticholinergics, diuretics, caffeine |

## Assessment & Diagnosis

### History
| Domain | Questions |
|--------|-----------|
| **Type** | Leakage with activity (SUI) vs urgency (UUI) vs constant (overflow) |
| **Severity** | Pads/day, volume, interference with activities |
| **Impact** | QoL scores (ICIQ-UI SF, King's Health Questionnaire) |
| **Voiding Diary** | **3-day diary**: frequency, volumes, leakage episodes, fluid intake |
| **Red Flags** | Haematuria, recurrent UTI, pain, neurological symptoms, pelvic mass |

### Examination
| Component | SUI Focus | UUI/Overflow Focus |
|-----------|-----------|-------------------|
| **Abdominal** | Mass, scars, distension | |
| **Pelvic (Women)** | **Cough stress test** (leakage with cough), pelvic organ prolapse (POP-Q), vaginal atrophy | |
| **Genital (Men)** | Penile/scrotal, meatus | Prostate (DRE: size, consistency, nodules) |
| **Neurological** | | Perineal sensation, anal tone, bulbocavernosus reflex, lower limbs |
| **Post-Void Residual (PVR)** | Usually low | **High (>100–150 mL)** = retention/overflow |

### Investigations
| Test | Indication |
|------|------------|
| **Urinalysis + Culture** | Exclude UTI, haematuria, glycosuria |
| **Urodynamics** | **Complex/unclear diagnosis**, pre-surgery, neurogenic, failed conservative tx |
| **Cystoscopy** | Haematuria, recurrent UTI, failed surgery, suspicion of malignancy/fistula |
| **Imaging** | US KUB (hydronephrosis, PVR), MRI (if fistula/sphincter) |
| **Pad Test** | 1-hour (standardised) or 24-hour (quantify leakage) |

### Urodynamics Key Parameters
| Parameter | SUI | UUI | Overflow |
|-----------|-----|-----|----------|
| **Valsalva Leak Point Pressure (VLPP)** | **Low (<60 cmH2O)** = intrinsic sphincter deficiency | Normal/high | N/A |
| **Detrusor Pressure at Max Flow (PdetQmax)** | Normal | **High** (overactivity) | **Low** (underactivity) |
| **Maximum Urethral Closure Pressure (MUCP)** | **Low** | Normal | Normal |
| **Bladder Compliance** | Normal | Normal/low | Normal |
| **Detrusor Overactivity** | Absent | **Present** (phasic/terminal) | Absent |

## Management

### Conservative (1st Line for All Types)
| Intervention | Details |
|--------------|---------|
| **Lifestyle** | Weight loss (BMI <25), smoking cessation, fluid management (1.5–2L/day, reduce caffeine/alcohol), treat constipation |
| **Pelvic Floor Muscle Training (PFMT)** | **SUI: 1st line**; 8–12 weeks supervised; ⱗ "Knack" (pre-contraction before stress) |
| **Bladder Training** | **UUI: 1st line**; scheduled voiding, urge suppression, progressive interval increase |
| **Continence Products** | Pads, sheaths (men), catheters (last resort) |
| **Vaginal Oestrogen (Post-menopausal)** | Topical (creme/pessary/ring) — improves urethral mucosa, PFMT response |

### Pharmacological

#### Stress UI
| Drug | Mechanism | Notes |
|------|-----------|-------|
| **Duloxetine** | SNRI → ↑ urethral sphincter tone (Onuf's nucleus) | **Only drug approved for SUI**; 20mg BD → 40mg BD; nausea common; avoid if hepatic/severe renal |

#### Urge UI / OAB
| Drug | Mechanism | Dose | Notes |
|------|-----------|------|-------|
| **Anticholinergics** | M3 blockade → ↓ detrusor contractions | Oxybutynin 5mg TDS/IR, 10mg OD/ER; Tolterodine 2mg BD/4mg OD; Solifenacin 5mg OD; Darifenacin 7.5mg OD; Fesoterodine 4–8mg OD | **1st line**; side effects: dry mouth, constipation, blurred vision, cognitive (avoid elderly) |
| **β3-Agonist** | **Mirabegron** → β3 relaxation → ↑ bladder capacity | 25mg OD → 50mg OD | **Alternative if anticholinergic intolerant**; less cognitive; hypertension monitoring |
| **Combination** | Anticholinergic + β3-agonist | Solifenacin + mirabegron | Refractory OAB |

#### Overflow / Neurogenic
| Approach | Detail |
|----------|--------|
| **Clean Intermittent Catheterisation (CIC)** | Gold standard for retention/neurogenic |
| **Alpha-blockers** | Tamsulosin (if BOO component) |
| **Bethanechol** | Cholinergic agonist (if detrusor underactivity) — limited evidence |

---

### Surgical

#### Stress UI (Women)
| Procedure | Indication | Notes |
|-----------|------------|-------|
| **Retropubic Mid-Urethral Sling (TVT)** | **Gold standard SUI** | **>85% cure**; mesh complications (erosion, pain, obstruction) |
| **Transobturator Sling (TOT)** | Alternative | Less bladder/bowel injury; similar efficacy |
| **Single-Incision Mini-Sling** | Less invasive | Lower long-term data |
| **Colposuspension (Burch)** | Open/laparoscopic | Historical; durable; more invasive |
| **Bulking Agents** | Intrinsic sphincter deficiency (low VLPP) | Periurethral injection; lower durability; repeat needed |
| **Artificial Urinary Sphincter (AUS)** | **Men (post-prostatectomy SUI)**; failed slings | **Gold standard men**; 3-component; mechanical failure ~10–15%/yr |

#### Urge UI / OAB (Refractory)
| Procedure | Indication |
|-----------|------------|
| **Botulinum Toxin A (Intradetrusor)** | Refractory OAB; 100U/200U; repeat q6–12mo; risk retention (CIC needed) |
| **Sacral Neuromodulation (SNM)** | Refractory OAB/UUI; test phase → permanent; reversible |
| **Percutaneous Tibial Nerve Stimulation (PTNS)** | Refractory OAB; weekly × 12 then maintenance |
| **Augmentation Cystoplasty** | End-stage small capacity bladder; enterocystoplasty |

#### Overflow (BOO/Neurogenic)
| Procedure | Indication |
|-----------|------------|
| **TURP / HoLEP** | BPH with overflow |
| **Urethroplasty** | Stricture |
| **CIC** | Neurogenic detrusor underactivity |

## Special Populations

### Post-Prostatectomy Incontinence (Men)
| Timeline | Management |
|----------|------------|
| **0–3 months** | PFMT, lifestyle, pads; expect recovery |
| **3–12 months** | PFMT + biofeedback; duloxetine; consider early sling if persistent |
| **>12 months** | **Artificial Urinary Sphincter (AUS) = gold standard**; Adjuvant sling (male sling) if mild-moderate |

### Elderly / Nursing Home
| Approach | Detail |
|----------|--------|
| **Prompted Voiding** | Scheduled toileting + positive reinforcement |
| **Habit Training** | Identify pattern → scheduled voiding |
| **Indwelling Catheter** | **Last resort** (comfort care only); suprapubic preferred if long-term |

### Neurogenic Bladder
| Condition | Management |
|-----------|------------|
| **Suprasacral (UMN)** | Detrusor overactivity + DSD → anticholinergic + CIC; botulinum toxin |
| **Sacral/Infrasacral (LMN)** | Detrusor underactivity + areflexia → CIC; bethanechol (limited) |
| **Spinal Cord Injury** | Autonomic dysreflexia risk; CIC + anticholinergic + botulinum |

## High-Yield FCPS/MRCP Points

> [!important]
> - **SUI: leakage with cough/sneeze/exercise; no urgency; women > men**
> - **UUI: urgency + frequency + nocturia + leakage; "key-in-lock"**
> - **MUI: both SUI + UUI features**
> - **Overflow: constant dribbling + high PVR (>100–150mL)**
> - **Functional: normal tract, impaired mobility/cognition**
> - **Assessment: 3-day voiding diary, cough stress test, DRE, PVR**
> - **SUI 1st line: PFMT (8–12wk); drug: duloxetine; surgery: TVT (women), AUS (men)**
> - **UUI 1st line: bladder training; drug: anticholinergics (oxybutynin, solifenacin) or mirabegron; refractory: botox, SNM, PTNS**
> - **Overflow: high PVR; CIC gold standard; treat cause (TURP, stricture)**
> - **Post-prostatectomy: PFMT 0–3mo; AUS >12mo if persistent**
> - **Vaginal oestrogen: post-menopausal adjunct**
> - **Pad test: 1-hr or 24-hr quantify leakage**

## Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **SUI = urgency** | **SUI = NO urgency**; UUI = urgency |
| **PFMT only for women** | **Men post-prostatectomy: PFMT 1st line** |
| **Anticholinergics for SUI** | **Anticholinergics for UUI/OAB**; SUI = duloxetine |
| **Mirabegron = anticholinergic** | **β3-agonist** (different mechanism, less cognitive) |
| **TVT = for men** | **TVT = women**; **AUS = men** (post-prostatectomy) |
| **All incontinence = surgery** | **Conservative 1st line for ALL types** |
| **Overflow = just dribbling** | **Must confirm high PVR (>100–150mL)** |
| **Elderly = just pads** | **Prompted voiding, habit training, treat reversible causes** |
| **Post-TURP SUI = immediate AUS** | **Wait 12 months; PFMT first; most recover** |
| **Urodynamics always needed** | **Only if complex/unclear, pre-surgery, neurogenic, failed conservative** |

## Mnemonics

- **UI Types**: **S**tress, **U**rge, **M**ixed, **O**verflow, **F**unctional = **SUMOF**
- **SUI**: **C**ough, **S**neeze, **E**xercise = **CSE** (no urgency)
- **UUI**: **U**rgency, **F**requency, **N**octuria = **UFN**
- **Overflow**: **H**igh **P**VR, **D**ribbling = **HPD**
- **Assessment**: **D**iary (3-day), **C**ough **S**tress, **P**VR, **U**rodynamics = **DCS-PVU**
- **SUI Tx**: **P**FMT → **D**uloxetine → **T**VT/**A**US = **PDTA**
- **UUI Tx**: **B**ladder training → **A**nticholinergic/**M**irabegron → **B**otox/**S**NM/**P**TNS = **BAM-BSP**
- **Overflow**: **P**VR >100 → **C**IC → **T**URP = **PCT**
- **Post-Prostatectomy**: **0–3mo PFMT**, **>12mo AUS** = **03-12**
- **Red Flags**: **H**aematuria, **R**ecurrent UTI, **P**ain, **N**euro, **M**ass = **HRPNM**

## Summary

**Urinary Incontinence** is a **Must Know 🔴** topic.
**Key takeaway:** **SUI**: leakage with stress (cough/sneeze), no urgency; **PFMT 1st line**, duloxetine drug, **TVT/TOT (women)**, **AUS (men post-prostatectomy)**. **UUI**: urgency + frequency + nocturia; **bladder training 1st line**, **anticholinergics/mirabegron**, refractory = **botox/SNM/PTNS**. **Overflow**: constant dribbling + **high PVR >100mL**; **CIC gold standard**; treat cause. **Assessment**: 3-day diary, cough stress test, PVR, urodynamics if complex. **Post-prostatectomy**: PFMT 0–3mo, **AUS >12mo**. **Vaginal oestrogen**: post-menopausal adjunct. **Elderly**: prompted voiding, habit training.
**Exam focus:** Types & features, assessment algorithm, PFMT vs bladder training, duloxetine vs anticholinergics/mirabegron, TVT vs AUS, botox/SNM for refractory UUI, CIC for overflow, post-prostatectomy timeline, urodynamics interpretation.
**Clinical relevance:** UI massively impacts QoL; stepwise conservative → pharmacological → surgical approach; most patients improve with PFMT/bladder training; surgery reserved for refractory.

# Urine Investigations

**Related:** [[Investigation of Renal and Urinary Tract Disease]], [[Functional Anatomy and Physiology of the Kidney and Urinary Tract]], [[Glomerular Filtration Rate]], [[Blood Tests in Renal Disease]], [[Acute Kidney Injury (AKI)]], [[Chronic Kidney Disease (CKD)]], [[Nephrology and Urology MOC]]

> [!important]
> **Urine is the "renal biopsy of the poor". Systematic: Dipstick → Microscopy → Quantification → Culture. Key: Proteinuria (ACR/PCR), Haematuria (dysmorphic = glomerular), Casts (RBC=GN, WBC=pyelo, granular=ATN, waxy=CKD), Crystals (cystine=hex, urate=needles, CaOx=envelope, struvite=coffin-lid).**

## Urinalysis — The "Renal Biopsy of the Poor"

### Dipstick Analysis
| Parameter | Normal | Abnormal Significance |
|-----------|--------|----------------------|
| **pH** | 4.5–8.0 | Low: acidosis, UTI (urea-splitters); High: alkalosis, urea-splitters |
| **Specific Gravity** | 1.005–1.030 | Low: DI, CKD; High: dehydration, glycosuria, proteinuria |
| **Protein** | Negative | **Proteinuria** (glomerular > tubular > overflow); false +ve: alkaline, bloody |
| **Glucose** | Negative | **Glycosuria** (DM, renal glycosuria, pregnancy, SGLT2i) |
| **Ketones** | Negative | **Ketonuria** (DKA, starvation, low carb diet) |
| **Blood** | Negative | **Haematuria** (RBC >3/HPF); false +ve: myoglobin, Hb |
| **Leucocytes** | Negative | **Leucocyturia** (UTI, sterile pyuria = TB, stones, interstitial nephritis) |
| **Nitrites** | Negative | **Nitrituria** (Gram-negative UTI, sensitivity ~50%) |
| **Bilirubin** | Negative | Liver disease, haemolysis |
| **Urobilinogen** | Normal | Increased: haemolysis, haemolytic anaemia |

> [!key]
> **Proteinuria**: Dipstick detects albumin; ACR = gold standard for quantification.
> **Haematuria**: Dipstick +ve + microscopy RBC >3/HPF = true haematuria.

## Urine Microscopy

| Element | Normal | Abnormal Significance |
|---------|--------|----------------------|
| **RBC** | 0–2/HPF | **>3/HPF = haematuria**; **dysmorphic = glomerular**; isomorphic = non-glomerular |
| **WBC** | 0–5/HPF | **>5/HPF = pyuria**; sterile pyuria = TB, interstitial nephritis, stones, carcinoma |
| **Epithelial cells** | Few | Squamous = contamination; Transitional = bladder/TCC; Renal tubular = ATN, interstitial nephritis |
| **Casts** | Hyaline few | **RBC cast = GN**; **WBC cast = pyelo/interstitial nephritis**; Granular/muddy brown = ATN; Waxy/broad = advanced CKD; Fatty = nephrotic syndrome |
| **Crystals** | Few urates | **Cystine** (hexagonal = cystinuria), **Uric acid** (needles/rosettes), **Calcium oxalate** (envelopes), **Struvite** (coffin-lid = struvite/Proteus) |
| **Microorganisms** | None | Bacteria = UTI; Yeast = Candida; Parasites (Schistosoma) |

## Proteinuria Quantification

| Test | Method | Normal | Use |
|------|--------|--------|-----|
| **ACR** | Albumin:creatinine ratio (spot) | <3 mg/mmol | **Gold standard**; corrects for concentration |
| **PCR** | Protein:creatinine ratio (spot) | <15 mg/mmol | Includes non-albumin; overestimates if tubular protein |
| **24h urine protein** | 24h collection | <150 mg/day | Gold standard but cumbersome |
| **Dipstick** | Semi-quantitative | Negative | Screening only |

| ACR Category | Range (mg/mmol) | Term |
|--------------|-----------------|------|
| **A1** | <3 | Normal |
| **A2** | 3–30 | Microalbuminuria |
| **A3** | >30 | Macroalbuminuria |

## Haematuria

| Type | Definition | Cause |
|------|------------|-------|
| **Microscopic** | >3 RBC/HPF on microscopy | Dipstick +ve → confirm microscopy |
| **Macroscopic** | Visible blood in urine | Stone, tumour, trauma, severe GN |

| Type | RBC Morphology | Cause |
|------|----------------|-------|
| **Glomerular** | **Dysmorphic** (>50% dysmorphic), RBC casts | GN, vasculitis, IgA nephropathy |
| **Non-glomerular** | **Isomorphic** (uniform), no casts | Stones, tumours, UTI, trauma, exercise |

> [!key]
> **Dysmorphic RBCs + RBC casts = Glomerular haematuria**
> **Isomorphic RBCs + no casts = Non-glomerular (urological)**

---

## Urinary Casts

| Cast Type | Appearance | Significance |
|-----------|------------|--------------|
| **Hyaline** | Clear, low refractive | Normal (dehydration, fever, exercise) |
| **RBC** | Red-brown, retain RBC shape | **Glomerulonephritis** (pathognomonic) |
| **WBC / Pus** | White, granular | **Pyelonephritis**, acute interstitial nephritis |
| **Granular** | Brown, granular | **ATN**, severe glomerular disease |
| **Muddy brown** | Dark, granular | **ATN** (classic) |
| **Waxy / Broad** | High refractive, broad | **Advanced CKD**, chronic tubular damage |
| **Fatty / Oval fat bodies** | Maltese crosses | **Nephrotic syndrome** (lipiduria) |

---

## Urinary Crystals

| Crystal | Shape | Clinical Association |
|---------|-------|---------------------|
| **Cystine** | **Hexagonal** (flat, hexagonal plates) | **Cystinuria** (autosomal recessive) |
| **Uric acid** | Needles, rosettes, rhombic | **Gout, tumour lysis, acidic urine** |
| **Calcium oxalate** | **Envelope-shaped** (bipyramidal), dumbbells | **Hyperoxaluria, ethylene glycol**, stones |
| **Calcium phosphate** | Amorphous, stellate | Alkaline urine, hyperparathyroidism |
| **Struvite (Triple phosphate)** | **Coffin-lid / coffin-lid shaped** (prismatic) | **Struvite stones** (Proteus, Klebsiella - urea-splitting organisms) |
| **Ammonium urate** | Thorny apples, cactus-like | Liver disease, laxative abuse |
| **Leucine / Tyrosine** | Yellow-brown, fine/silky | Severe liver disease, tyrosinaemia |
| **Cholesterol** | Rectangular, notched | Nephrotic syndrome (lipiduria) |

> [!key]
> **Cystine = Hexagonal**; **Uric acid = Needles/rosettes**; **CaOx = Envelopes**; **Struvite = Coffin-lid**;
> **Cholesterol = Rectangular/notched** (nephrotic); **Tyrosine/Leucine** = liver failure.

## Proteinuria Quantification

| Test | Method | Normal | Best For |
|------|--------|--------|----------|
| **ACR** | Albumin:creatinine ratio (spot) | <3 mg/mmol | **Gold standard** |
| **PCR** | Protein:creatinine ratio | <15 mg/mmol | Total protein (incl. non-albumin) |
| **24h urine protein** | 24h collection | <150 mg/day | Gold standard (cumbersome) |
| **Dipstick** | Semi-quantitative | Negative | Screening only |

| ACR Category | Range (mg/mmol) | Term | Significance |
|--------------|-----------------|------|--------------|
| **A1** | <3 | Normal | — |
| **A2** | 3–30 | Microalbuminuria | Early diabetic nephropathy, hypertensive nephropathy |
| **A3** | >30 | Macroalbuminuria | Overt proteinuria; >220 = nephrotic range |

> **PCR vs ACR**: PCR measures total protein; ACR specific for albumin. PCR > ACR if tubular proteinuria (Fanconi, myeloma).

## Haematuria Investigation

| Step | Action |
|------|--------|
| 1 | **Confirm**: Repeat dipstick + microscopy (>3 RBC/HPF) |
| 2 | **Morphology**: Dysmorphic (>50%) = glomerular; Isomorphic = non-glomerular |
| 3 | **Proteinuria**: ACR >30 + haematuria = glomerular disease |
| 4 | **Renal function**: eGFR, BP |
| 5 | **Imaging**: US KUB (stone, tumour, cyst); CT urogram (haematuria >40yr) |
| 6 | **Cystoscopy**: If >40yr or risk factors (smoker, analgesic abuse) |
| 7 | **Serology**: ASOT (post-strep), ANA, ANCA, anti-GBM, C3/C4, HIV, Hep B/C |

## Urine Culture & Sensitivity

| Indication | Method |
|-----------|----------|
| **Symptomatic UTI** | Midstream urine (MSU) |
| **Catheter specimen** | Catheter specimen (CSU) |
| **Suprapubic aspirate** | Infants, contaminated samples |
| **Quantitative culture** | >10⁵ CFU/mL = significant bacteriuria |

| Organism | Common Context |
|----------|----------------|
| **E. coli** | 70–80% community UTI |
| **Klebsiella** | Hospital, catheter, stones |
| **Proteus** | Stones (urease → struvite), catheter |
| **Enterococcus** | Hospital, catheter, instrumentation |
| **Pseudomonas** | Hospital, catheter, immunocompromised |
| **Staph saprophyticus** | Young women, community UTI |

## Special Urine Tests

| Test | Indication | Method |
|---------|------------|--------|
| **Bence Jones Protein** | Myeloma, amyloidosis | Urine immunofixation, heating (precipitates at 50–60°C) |
| **Myoglobin** | Rhabdomyolysis | Dipstick +ve for blood, no RBCs on microscopy |
| **Haemoglobin** | Intravascular haemolysis | Dipstick +ve blood, no RBCs on microscopy |
| **Porphyrins** | Porphyria | Fluorescence, chromatography |
| **Catecholamines** | Pheochromocytoma | 24h urine VMA, metanephrines |
| **Cortisol** | Cushing's | 24h urinary free cortisol |
| **VMA/HVA** | Neuroblastoma | 24h urine VMA, HVA |
| **5-HIAA** | Carcinoid syndrome | 24h urine 5-HIAA |

## High-Yield FCPS/MRCP Points

> [!important]
> - **Urinalysis** = renal biopsy of the poor; first test in renal disease
> - **Proteinuria**: ACR >30 = proteinuria; ACR >300 = nephrotic range
> - **Haematuria**: >3 RBC/HPF + dysmorphic = glomerular; isomorphic = non-glomerular
> - **Casts**: RBC = GN; WBC = pyelo/interstitial; granular = ATN; waxy = advanced CKD; fatty = nephrotic
> - **Crystals**: Cystine (hex), urate (needles), Ca oxalate (envelopes), struvite (coffin-lid)
> - **ACR**: <3 (A1), 3–30 (A2), >30 (A3) — nephrotic if >220 mg/mmol
> - **Casts**: RBC = GN; WBC = pyelo/interstitial; granular = ATN; waxy = CKD
> - **Crystals**: Cystine (hex), urate (needles), Ca oxalate (envelopes), struvite (coffin-lid)
> - **ACR categories**: A1 <3, A2 3–30, A3 >30 mg/mmol
> - **Glomerular haematuria**: dysmorphic RBCs + RBC casts
> - **Non-glomerular**: isomorphic RBCs, no casts

## Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **Dipstick protein = all protein** | Dipstick = albumin only; misses tubular proteins (Bence Jones) |
| **Microalbuminuria = diabetic only** | Also hypertensive, early CKD, heart failure |
| **Red urine = haematuria** | Myoglobin, haemoglobin, beetroot, rifampicin, porphyria |
| **Proteinuria = always glomerular** | Tubular (Fanconi, drugs), overflow (myeloma), postural |
| **Casts = only in GN** | WBC = pyelo/interstitial; granular = ATN; waxy = CKD |
| **All crystals = stones** | Crystalluria ≠ stones (can be normal in concentrated urine) |
| **ACR = PCR** | ACR = albumin only; PCR = total protein (incl. tubular) |
| **Microalbuminuria = only diabetes** | Also hypertension, early CKD, heart failure |

# Bartter & Gitelman Syndromes (+ Liddle Syndrome)

**Related:** [[Functional Anatomy and Physiology of the Kidney and Urinary Tract]], [[Renal Tubular Acidosis (RTA Types 1, 2, 4)]], [[Electrolyte, Water, and Acid-Base Disorders]], [[Chronic Kidney Disease (CKD)]], [[Nephrology and Urology MOC]]

> [!important]
> **Bartter & Gitelman = inherited salt-wasting tubulopathies with hypokalaemic metabolic alkalosis. Bartter = TAL defect (NKCC2/ROMK/CLCNKB/BSND/MAGED2) → polyhydramnios, premature, nephrocalcinosis. Gitelman = DCT defect (NCC/SLC12A3) → hypomagnesaemia, hypocalciuria, later onset. Liddle = ENaC gain-of-function → hypertension, hypokalaemia, alkalosis, low renin/aldo.**

---

## Learning Objectives
- Differentiate Bartter vs Gitelman vs Liddle by site, labs, and clinical features
- Apply genetic classification (Bartter Types 1–5, Gitelman, Liddle)
- Diagnose using urine electrolytes, renin/aldosterone, genetic testing
- Manage each syndrome (NSAIDs/K+ for Bartter, Mg/K+ for Gitelman, amiloride for Liddle)
- Apply to FCPS/MRCP paediatric and adult vignettes

## Bartter Syndrome

### Genetics & Classification
| Type | Gene | Protein | Location | Features |
|------|------|---------|----------|----------|
| **Type 1** | **SLC12A1** | **NKCC2** | TAL | Antenatal, severe, polyhydramnios |
| **Type 2** | **KCNJ1** | **ROMK (Kir1.1)** | TAL | Antenatal, severe, transient neonatal hyperK |
| **Type 3** | **CLCNKB** | **ClC-Kb** | TAL | Classic/classic, later onset, variable |
| **Type 4** | **BSND** | **Barttin** (β-subunit ClC-Ka/ClC-Kb) | TAL + inner ear | **Sensorineural deafness** |
| **Type 5** | **MAGED2** | **MAGE-D2** | TAL (X-linked) | Transient antenatal, resolves postnatally |

> **All Bartter types** = TAL salt wasting → ↑ prostaglandin E2 → polyuria, polyhydramnios.

---

### Pathophysiology (TAL Defect)
- **TAL** = Na⁺-K⁺-2Cl⁻ cotransporter (NKCC2) + ROMK + ClC-Kb/Barttin
- **Defect** → ↓ NaCl reabsorption → **salt wasting** → volume depletion → ↑ renin/aldo
- **Consequences**: Polyuria (↓ medullary gradient), hypokalaemia, metabolic alkalosis, hypercalciuria → **nephrocalcinosis**

---

### Clinical Presentation
| Feature | Antenatal (Types 1,2,4) | Classic (Type 3) |
|---------|-------------------------|------------------|
| **Presentation** | **Prenatal**: polyhydramnios, prematurity | Childhood/adult: failure to thrive, polyuria |
| **Polyuria/Polydipsia** | Severe (neonatal) | Moderate-severe |
| **Growth** | **Failure to thrive** | Short stature |
| **Nephrocalcinosis** | **Common** (hypercalciuria) | Common |
| **Deafness** | **Type 4 (BSND) only** | — |
| **Hyperkalaemia** | **Type 2 (KCNJ1) transient neonatal** | — |

---

### Biochemical Profile
| Parameter | Bartter |
|-----------|---------|
| **K+** | **Low** (hypokalaemia) |
| **Cl-** | Low |
| **HCO3-** | **High** (metabolic alkalosis) |
| **Renin** | **High** |
| **Aldosterone** | **High** |
| **BP** | **Normal/Low** (volume depletion) |
| **Urine Ca2+** | **High** (hypercalciuria) → nephrocalcinosis |
| **Urine Mg2+** | Normal/mildly ↑ |
| **Urine PGE2** | **High** |

---

### Management
| Intervention | Details |
|--------------|---------|
| **NSAIDs (Indomethacin)** | **First-line** — inhibits PGE2 → ↑ NaCl reabsorption in TAL, ↓ renin/aldo, ↑ K+ |
| **K+ supplements** | KCl (high dose, often 50–100 mmol/day) |
| **K+-sparing diuretics** | Spironolactone/Amiloride (add-on) |
| **ACEi/ARB** | Sometimes used (↓ Ang II) |
| **Growth hormone** | If severe growth failure |
| **Avoid** | Loop diuretics (worsen wasting) |

## Gitelman Syndrome

### Genetics
- **SLC12A3** (chr16q13) → **NCC (Na-Cl cotransporter)** in **DCT**
- **Autosomal recessive**
- Prevalence ~1:40,000 (higher in East Asia)

---

### Pathophysiology (DCT Defect)
- **NCC loss-of-function** → ↓ NaCl reabsorption in DCT → mild volume depletion → ↑ renin/aldo
- **Paradoxical Mg2+ wasting** (DCT = major Mg reabsorption site via TRPM6)
- **Hypocalciuria** (↑ proximal Ca2+ reabsorption due to volume depletion)

---

### Clinical Presentation
| Feature | Typical |
|---------|---------|
| **Age at onset** | **Adolescence/adulthood** (often asymptomatic until stress) |
| **Symptoms** | Fatigue, muscle cramps, tetany (hypomagnesaemia), salt craving |
| **BP** | **Normal/Low** |
| **Growth** | Normal (vs Bartter failure to thrive) |
| **Nephrocalcinosis** | **Absent** (hypocalciuria) |
| **Chondrocalcinosis** | Adult: CPPD deposition (pseudogout) |

---

### Biochemical Profile
| Parameter | Gitelman |
|-----------|----------|
| **K+** | **Low** (hypokalaemia) |
| **Cl-** | Low |
| **HCO3-** | **High** (metabolic alkalosis) |
| **Renin** | **High** |
| **Aldosterone** | **High** |
| **Mg2+** | **Low** (hypomagnesaemia — hallmark) |
| **Urine Ca2+** | **Low** (hypocalciuria — distinguishes from Bartter) |
| **Urine Mg2+** | **High** (renal Mg wasting) |

---

### Management
| Intervention | Details |
|--------------|---------|
| **Mg2+ supplementation** | **Oral Mg (glycinate/chloride) — mainstay**; IV if severe |
| **K+ supplements** | KCl (moderate doses) |
| **K+-sparing diuretics** | **Amiloride** (blocks ENaC → ↓ K+ wasting); Spironolactone |
| **NSAIDs** | Less effective than Bartter (PGE2 not as central) |
| **High salt diet** | Liberal NaCl intake |
| **Avoid** | Thiazides (target is NCC — already defective), loop diuretics |

---

## Liddle Syndrome (Pseudohyperaldosteronism)

### Genetics
- **SCNN1B** or **SCNN1G** (chr16p12) → **ENaC β/γ subunits** gain-of-function
- **Autosomal dominant**
- Constitutive ENaC activation → Na+ reabsorption independent of aldosterone

---

### Pathophysiology
- **ENaC always open** → ↑ Na+ reabsorption in CD → volume expansion → hypertension
- ↓ Renin, ↓ Aldosterone (suppressed)
- ↑ K+ & H+ secretion → **hypokalaemia, metabolic alkalosis**

---

### Clinical Presentation
| Feature | Liddle |
|---------|--------|
| **Hypertension** | **Early-onset, severe, resistant** |
| **Hypokalaemia** | Marked |
| **Metabolic alkalosis** | Yes |
| **Renin** | **Low (suppressed)** |
| **Aldosterone** | **Low (suppressed)** |
| **Family history** | Autosomal dominant |

---

### Differential: Liddle vs Hyperaldosteronism
| Feature | Liddle | Primary Hyperaldosteronism |
|---------|--------|---------------------------|
| **Aldosterone** | **Low** | **High** |
| **Renin** | Low | Low |
| **ENaC mutation** | Yes | No |
| **Response to spironolactone** | **Poor** (ENaC mutation) | Good |
| **Response to amiloride/triamterene** | **Excellent** | Good |

---

### Management
| Drug | Dose/Mechanism |
|------|----------------|
| **Amiloride** | **First-line** 5–20mg/day — blocks mutated ENaC directly |
| **Triamterene** | Alternative ENaC blocker |
| **Avoid** | Spironolactone (ineffective — acts via MR, not ENaC) |
| **Low Na+ diet** | Adjunct |

## High-Yield FCPS/MRCP Points

> [!important]
> - **Bartter** = TAL defect, antenatal polyhydramnios, nephrocalcinosis, **indomethacin**.
> - **Gitelman** = DCT defect, **hypomagnesaemia + hypocalciuria**, later onset, **Mg²⁺ + amiloride**.
> - **Liddle** = ENaC gain-of-function, **hypertension + hypokalaemia + LOW renin/aldo**, **amiloride** (not spiro).
> - **Bartter vs Gitelman**: Nephrocalcinosis (Bartter YES, Gitelman NO); Hypomagnesaemia (Bartter NO, Gitelman YES); Hypocalciuria (Bartter NO, Gitelman YES).
> - **Genetics**: Bartter Types 1–5 (NKCC2, ROMK, ClC-Kb, Barttin, MAGE-D2); Gitelman = NCC (SLC12A3); Liddle = ENaC (SCNN1B/G).
> - **Urinary Ca2+** is the key discriminator: high in Bartter, low in Gitelman.

## Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **Bartter = hypertension** | Bartter = **normo/hypotension**; **Liddle = hypertension** |
| **Gitelman = childhood** | Gitelman = **adolescent/adult** (often diagnosed 20–40yr) |
| **All hypokalaemic alkalosis = Bartter/Gitelman** | **Liddle** also causes this but with **hypertension + low renin/aldo** |
| **Spironolactone works in Liddle** | **No — amiloride/triamterene only** (blocks ENaC directly) |
| **Indomethacin works in Gitelman** | Less effective; **Mg²⁺ replacement is mainstay** |
| **Bartter = hypocalciuria** | **Bartter = hypercalciuria → nephrocalcinosis**; Gitelman = hypocalciuria |
| **Gitelman = hypercalciuria** | **Gitelman = hypocalciuria** (volume depletion ↑ prox Ca2+ reabs) |
| **Liddle = high aldosterone** | **Liddle = LOW aldosterone** (suppressed by volume expansion) |
| **All Bartter types = deafness** | **Only Type 4 (BSND/barttin)** has sensorineural deafness |
| **MAGED2 = permanent** | **Type 5 (MAGED2)** = transient antenatal, resolves after birth (X-linked) |

- **Bartter B's**: **B**artter = **B**aby (antenatal), **B**ig Ca²⁺ (nephrocalcinosis), **B**ad growth, **B**lock NKCC2/ROMK
- **Gitelman G's**: **G**itelman = **G**rown-up (adult), **G**ranny (hypomagnesaemia), **G**enerous Ca²⁺ reabs (hypocalciuria)
- **Liddle L's**: **L**iddle = **L**ow renin/aldo, **L**efty (hypertension), **L**ocks ENaC open

- **Rx**: **B**artter = **I**ndomethacin; **G**itelman = **M**g + **A**miloride; **L**iddle = **A**miloride

## Summary
**Bartter, Gitelman & Liddle Syndromes** are **Must Know 🔴** tubulopathy topics.  
**Bartter**: TAL defect (NKCC2/ROMK/ClC-Kb/Barttin/MAGED2), AR (Types 1–4), X-linked (Type 5). Antenatal polyhydramnios, FTT, **nephrocalcinosis** (hypercalciuria), high renin/aldo. **Rx: Indomethacin + K+**.  
**Gitelman**: DCT defect (NCC/SLC12A3), AR. Adolescent/adult, **hypomagnesaemia + hypocalciuria** (hallmark), NO nephrocalcinosis, high renin/aldo. **Rx: Mg²⁺ + Amiloride + K+**.  
**Liddle**: CD ENaC gain-of-function (SCNN1B/G), AD. **Hypertension**, hypokalaemia, alkalosis, **LOW renin/aldo**. **Rx: Amiloride (NOT spiro)**.  
**Key discriminators**: Urine Ca²⁺ (Bartter HIGH, Gitelman LOW); BP (Liddle HIGH, others LOW); Mg²⁺ (Gitelman LOW); Renin/aldo (Liddle LOW, others HIGH).  
**Genetics**: Bartter 1–5 (NKCC2, ROMK, ClC-Kb, Barttin, MAGED2); Gitelman (NCC); Liddle (ENaC).  
**Exam focus**: Site, age, nephrocalcinosis, Mg/Ca urine, BP, renin/aldo, specific Rx.

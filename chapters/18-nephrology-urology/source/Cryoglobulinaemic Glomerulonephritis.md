# Cryoglobulinaemic Glomerulonephritis

**Related:** [[Glomerular Diseases — Overview and Classification]], [[MPGN and C3 Glomerulopathy (DDD, C3GN)]], [[Secondary Glomerulonephritides — Lupus Nephritis]], [[Anti-GBM Disease (Goodpasture Syndrome)]], [[Hepatitis C]], [[Nephrology and Urology MOC]]

> [!important]
> **Cryoglobulinaemic GN = immune-complex MPGN from cryoglobulin precipitation in glomeruli. Types: Type I (monoclonal, malignancy), Type II (mixed monoclonal + polyclonal, HCV), Type III (mixed polyclonal, SLE/RA). HCV = 80–90% of mixed cryoglobulinaemia. Low C4 is hallmark. Renal: MPGN pattern, nephritic/nephrotic. Treat HCV (DAAs) ± rituximab; severe: plasma exchange + rituximab + steroids.**

---

## Learning Objectives
- Classify cryoglobulinaemia (Types I, II, III)
- Recognise clinical triad (purpura, arthralgia, weakness) + renal involvement
- Interpret complement profile (Low C4, Low/Normal C3, Cryoglobulins+, RF+)
- Manage based on type and severity (DAAs for HCV, rituximab, plasma exchange)
- Differentiate from other MPGN causes
- Apply to FCPS/MRCP immune-complex GN vignettes

## Classification (Brouet)

| Type | Composition | Association | Prevalence |
|------|-------------|-------------|------------|
| **Type I** | **Monoclonal** (IgM/IgG/IgA κ/λ) | **Lymphoproliferative** (Waldenström, myeloma, CLL, lymphoma) | 10–15% |
| **Type II** | **Mixed**: Monoclonal IgM (κ) + Polyclonal IgG | **HCV (80–90%)**, also lymphoproliferative | **50–60%** |
| **Type III** | **Mixed**: Polyclonal IgM + Polyclonal IgG | **SLE, RA, Sjögren's**, other autoimmune | 25–30% |

> **Type II (HCV) = most common mixed cryoglobulinaemia.** Type I = malignancy. Type III = autoimmune.

---

## Pathogenesis
- **Cryoglobulins** = immunoglobulins that **precipitate at <37°C** and dissolve on rewarming
- **Type II/III**: IgM (with rheumatoid factor activity) binds polyclonal IgG → immune complexes
- **Cold exposure** → precipitation in capillaries (skin, kidneys, nerves) → vasculitis
- **Renal**: Subendothelial deposits → **MPGN pattern** (mesangial expansion, GBM duplication)

---

## Clinical Presentation

### Classic Triad (Meltzer's Triad)
| Feature | Frequency |
|---------|-----------|
| **Palpable purpura** (lower limbs) | 80–90% |
| **Arthralgia** (symmetrical, non-erosive) | 60–80% |
| **Weakness / Fatigue** | 70–90% |

### Renal Involvement (20–60%)
| Presentation | Frequency |
|--------------|-----------|
| **Nephritic syndrome** (haematuria, RBC casts, hypertension) | Common |
| **Nephrotic syndrome** | Less common |
| **Mixed nephritic-nephrotic** | Common |
| **AKI / RPGN** | Severe cases |
| **Asymptomatic proteinuria/haematuria** | Early |

### Extrarenal
| System | Manifestation |
|--------|---------------|
| **Skin** | Purpura, ulcers, livedo reticularis, Raynaud's |
| **Joints** | Symmetrical arthralgia (knees, ankles, wrists) |
| **Nerves** | **Peripheral neuropathy** (sensory > motor, mononeuritis multiplex) |
| **Liver** | HCV chronic hepatitis, cirrhosis |
| **Lungs** | Interstitial pneumonitis (rare) |

## Laboratory Findings

| Test | Typical Finding | Significance |
|------|-----------------|--------------|
| **Cryoglobulins** | **Positive** (send warm tube!) | Diagnostic |
| **C4** | **Low (hallmark)** | Classical pathway consumption |
| **C3** | Low or Normal | Variable consumption |
| **Rheumatoid Factor (RF)** | **High** (IgM RF) | Monoclonal IgM has RF activity |
| **Serum Immunofixation** | **Monoclonal IgM κ (Type II)** | Confirms type |
| **HCV RNA** | **Positive (80–90% Type II)** | Aetiology |
| **ANA / ENA** | Positive (Type III/SLE) | Autoimmune |
| **ANCA** | Usually negative | Differentiate from ANCA vasculitis |
| **Urine** | Haematuria, Proteinuria, RBC casts | Renal involvement |

> **Key**: **Low C4 + High RF + Cryoglobulins+** = cryoglobulinaemic vasculitis.

---

## Renal Histology
| Finding | Description |
|---------|-------------|
| **LM** | **MPGN pattern** (mesangial expansion, GBM double contours "tram-track") |
| **IF** | **Granular IgM, IgG, C3** (capillary wall + mesangium) |
| **EM** | **Subendothelial deposits + microtubular/fibrillary structures** (cryoglobulin precipitates) |
| **Microtubules** | **30–50nm hollow microtubules** (pathognomonic for cryoglobulins) |

---

## Management

### Type I (Monoclonal) — Treat Underlying Malignancy
| Approach | Detail |
|----------|--------|
| **Primary** | Treat lymphoproliferative disorder (rituximab-based chemo, BTKi, etc.) |
| **Renal** | Plasmapheresis if severe RPGN/acute |
| **Rituximab** | Often effective (targets B-cell clone) |

### Type II (HCV-Associated) — **Treat HCV First**
| Severity | Regimen |
|----------|---------|
| **Mild (proteinuria <1g, preserved eGFR)** | **DAAs alone** (sofosbuvir/velpatasvir × 12wk) — cryoglobulins clear with HCV cure |
| **Moderate (proteinuria 1–3.5g, eGFR >30)** | **DAAs + Rituximab** (375mg/m² weekly × 4) |
| **Severe (RPGN, nephrotic, AKI, eGFR <30)** | **Plasma Exchange + Rituximab + Steroids → then DAAs** |

> **DAAs cure HCV → cryoglobulins disappear in 90%**. Rituximab for persistent/severe vasculitis.

### Type III (Autoimmune) — Treat Underlying Disease
| Disease | Management |
|---------|------------|
| **SLE** | Standard lupus nephritis protocol (MMF/CyC + steroids) |
| **RA / Sjögren's** | Rituximab + steroids ± immunosuppressants |
| **Severe** | Plasma exchange + rituximab + steroids |

---

## Plasma Exchange (PLEX) Indications
| Indication | Protocol |
|------------|----------|
| **RPGN** (rapidly rising Cr) | 1–1.5 plasma vol daily × 7–14 days |
| **Severe neuropathy** (mononeuritis multiplex) | Same |
| **Severe skin ulcers/gangrene** | Same |
| **Hyperviscosity** (Type I, high monoclonal) | Same |

> **PLEX removes cryoglobulins acutely**; bridge to immunosuppression/DAAs.

## High-Yield FCPS/MRCP Points

> [!important]
> - **Cryoglobulinaemic GN** = MPGN from cryoglobulin precipitation.
> - **Types**: I (monoclonal, malignancy), **II (mixed monoclonal+polyclonal, HCV 80–90%)**, III (polyclonal, SLE/RA).
> - **Hallmark labs**: **Low C4, High RF, Cryoglobulins+**, HCV RNA+ (Type II).
> - **Renal**: MPGN pattern, subendothelial deposits + **microtubules (30–50nm)** on EM.
> - **Clinical**: Purpura, arthralgia, weakness + neuropathy.
> - **Type II (HCV)**: **DAAs first-line** (cure HCV → cryoglobulins clear); add rituximab if moderate/severe.
> - **Type I**: Treat malignancy (rituximab-chemo).
> - **Type III**: Treat autoimmune (SLE/RA protocol).
> - **Severe**: **Plasma exchange + rituximab + steroids**.
> - **Differential**: MPGN (C3G, IC-MPGN), SLE, ANCA, anti-GBM, IE.

## Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **All cryoglobulinaemia = HCV** | **Type I = malignancy**; Type III = SLE/RA |
| **Low C3 + Low C4 = always SLE** | **Cryoglobulinaemia also Low C3 + Low C4** (classical pathway) |
| **RF = only RA** | **High RF in cryoglobulinaemia** (monoclonal IgM has RF activity) |
| **DAAs alone for all HCV cryo** | **Severe (RPGN/nephrotic) = PLEX + rituximab + steroids first** |
| **Cryoglobulins = send cold tube** | **Send WARM tube (37°C)** — cold precipitates in tube, false negative |
| **Type I = MPGN** | Type I = usually **no renal** or minimal; Type II/III = MPGN |
| **Cryoglobulins = only renal** | **Skin, joints, nerves, liver** all involved |
| **ANCA positive in cryo** | **Usually ANCA negative** — differentiates from ANCA vasculitis |

## Mnemonics
- **Types**: **I** = **I**mmune malignancy (monoclonal); **II** = **I**I = **H**CV (mixed); **III** = **I**II = **A**utoimmune (polyclonal)
- **Type II**: **H**CV + **M**ixed = **HM** (80–90%)
- **Lab triad**: **L**ow C4, **H**igh RF, **C**ryoglobulins+ = **LHC**
- **EM**: **M**icrotubules **3**0–50nm = **M3**
- **Treatment**: **H**CV = **D**AAs; **M**alignancy = **R**ituximab-chemo; **A**utoimmune = **R**ituximab/immunosuppression

## Summary
**Cryoglobulinaemic Glomerulonephritis** is a **Must Know 🔴** immune-complex GN topic.  
**Classification**: **Type I** (monoclonal, malignancy), **Type II** (mixed monoclonal+polyclonal, **HCV 80–90%**), **Type III** (polyclonal, SLE/RA).  
**Labs**: **Low C4** (hallmark), **High RF**, **Cryoglobulins+** (send **warm tube**), HCV RNA+ (Type II).  
**Renal**: MPGN pattern, **subendothelial microtubules (30–50nm)** on EM.  
**Clinical**: **Meltzer's triad** (purpura, arthralgia, weakness) + neuropathy, skin ulcers.  
**Management**: **Type II (HCV)** — DAAs first; add rituximab if moderate/severe; **PLEX + Rituximab + Steroids** for RPGN/severe. **Type I** — treat malignancy. **Type III** — treat autoimmune.  
**PLEX indications**: RPGN, severe neuropathy, hyperviscosity, alveolar haemorrhage.  
**Exam focus**: Three types, Low C4/High RF, microtubules EM, warm tube, DAAs for HCV, PLEX+Rituximab for severe, Meltzer's triad.

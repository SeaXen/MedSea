# Kidney Transplantation (Indications, Workup, Immunosuppression, Complications)

**Related:** [[Chronic Kidney Disease (CKD)]], [[Renal Replacement Therapy — Dialysis (HD, PD), Indications, Complications]], [[Nephrology and Urology MOC]]

> [!important]
> **Kidney transplant = treatment of choice for ESRD. Indications: ESRD (G5), irreversible CKD G4–5. Live donor > deceased donor. Immunosuppression: Induction (basiliximab/ATG) + Maintenance (Tacrolimus + MMF + Pred). Complications: rejection (acute/chronic), infection, malignancy, CVD, recurrence.**

---

## Learning Objectives
- Identify indications and contraindications for kidney transplantation
- Perform recipient and donor workup

- Manage post-transplant complications (rejection, infection, malignancy, CVD)

## Indications & Contraindications

### Indications
| Indication | Criteria |
|------------|----------|
| **ESRD** | eGFR <15 (G5) on dialysis or imminent |
| **Advanced CKD** | G4 (eGFR 15–29) with rapid decline or preparing for pre-emptive transplant |
| **Pre-emptive Transplant** | Before dialysis (ideal) — better graft/patient survival |

### Absolute Contraindications
| Condition | Reason |
|-----------|--------|
| **Active malignancy** | Except treated skin cancers (BCC/SCC), some CNS tumours with long remission |
| **Active systemic infection** | TB, HIV (if uncontrolled), hepatitis B/C (if active replication) |
| **Severe irreversible cardiopulmonary disease** | Unable to survive surgery/immunosuppression |
| **Severe psychiatric illness** | Non-adherence risk |
| **Active substance abuse** | Alcohol, IV drugs |

### Relative Contraindications
| Condition | Management |
|-----------|------------|
| **Age >70** | Individualised; biological age > chronological |
| **Obesity (BMI >35)** | Weight loss recommended before listing |
| **Peripheral vascular disease** | Vascular assessment; may need revascularisation |
| **Prior transplant** | Sensitisation (PRA/DSA) — desensitisation protocols |
| **Chronic viral infections** | HIV (if controlled, CD4>200, undetectable VL), HBV (on antiviral), HCV (DAA treatable) |

## Recipient Workup

| Domain | Tests |
|--------|-------|
| **Cardiovascular** | ECG, Echo, Stress test (if >50 or DM/CVD risk), Coronary angiography (if high risk) |
| **Pulmonary** | CXR, Spirometry (if COPD/asthma), ABG if severe |
| **Infectious** | HIV, HBV (HBsAg, anti-HBc, anti-HBs), HCV (Ab, PCR), CMV IgG, EBV IgG, VZV IgG, TB (IGRA/skin test), Syphilis, Strongyloides (if endemic) |
| **Malignancy Screen** | Age-appropriate: colonoscopy, mammography, PSA, cervical, skin |
| **Immunological** | **ABO typing, HLA typing, PRA (panel reactive antibody), DSA (donor-specific antibody), Crossmatch** |
| **Vascular Access** | Doppler US iliac vessels (if prior surgery/vascular disease) |
| **Urological** | US KUB, VCUG if reflux/neurogenic bladder history; native nephrectomy if large polycystic/recurrent infection |
| **Bone/Mineral** | DEXA (if >50 or steroids), Ca, PO4, PTH, Vit D |
| **Nutritional** | BMI, albumin, pre-albumin |
| **Psychosocial** | Support system, adherence history, financial, substance use |

> [!key]
> **HLA mismatch: 0–6 (A, B, DR). Lower mismatch = better survival. PRA >80% = highly sensitised = hard to match. DSA = contraindication to that donor.**

## Donor Types & Workup

### Living Donor
| Category | Details |
|----------|---------|
| **Related** | 1st degree (parent, sibling, child) — best HLA match |
| **Unrelated** | Spouse, friend, altruistic (non-directed) |
| **Paired Exchange** | Incompatible pairs swap donors |
| **ABO-incompatible** | Desensitisation (rituximab, plasmapheresis, IVIG) |

### Living Donor Evaluation
| Domain | Criteria |
|--------|----------|
| **Age** | 18–70 (older if excellent health) |
| **GFR** | **>90 mL/min/1.73m²** (measured, not estimated) |
| **Proteinuria** | **<300 mg/day** (ACR <3 mg/mmol) |
| **BP** | **<140/90** off meds; no end-organ damage |
| **Anatomy** | CT angiography — single artery preferred; venous anatomy |
| **Medical** | No diabetes, no hypertension (or well-controlled single drug), no CKD, no stones, no malignancy, no significant cardiac/pulmonary disease |
| **Psychosocial** | Voluntary, informed, no coercion; adequate support |

### Deceased Donor
| Type | Criteria |
|------|----------|
| **DBD** (Donation after Brain Death) | Standard; better graft survival |
| **DCD** (Donation after Circulatory Death) | Maastricht categories I–IV; higher DGF risk |
| **ECD** (Extended Criteria Donor) | >60yo, or 50–59 with 2 of: HTN, CVA death, Cr >1.5; KDPI >85% |
| **Standard Criteria** | <50yo, no comorbidities |

> [!key]
> **KDPI (Kidney Donor Profile Index): 0–100%; lower = better. KDPI >85% = ECD. DCD = higher DGF but similar long-term survival.**

## Allocation & Matching

| System | Details |
|--------|---------|
| **UK (NHSBT)** | HLA match + waiting time + sensitisation + age matching + geography |
| **US (UNOS)** | KDPI + EPTS (Estimated Post-Transplant Survival) + HLA + waiting time + DSA |
| **Virtual Crossmatch** | Recipient DSA vs donor HLA — predicts positive crossmatch |
| **Actual Crossmatch** | **Complement-dependent cytotoxicity (CDC)** or **Flow cytometry** — final go/no-go |

## Immunosuppression

### Induction Therapy (Peri-transplant)
| Agent | Dose | Indication |
|-------|------|------------|
| **Basiliximab** (IL-2R antagonist) | 20mg IV Day 0 + Day 4 | **Low immunologic risk** (1st transplant, low PRA, no DSA) |
| **Anti-thymocyte Globulin (ATG)** | 1.5mg/kg/day × 3–5 days (total 4.5–7.5mg/kg) | **High immunologic risk** (retransplant, high PRA, DSA, DCD, African ancestry) |
| **Alemtuzumab** | 30mg IV Day 0 (or split) | Some centres for low-risk (off-label) |
| **No Induction** | — | Rare (e.g., identical twin) |

### Maintenance (Triple Therapy Standard)
| Drug | Typical Dose | Target/Monitoring |
|------|--------------|-------------------|
| **Tacrolimus** (CNI) | **0.05–0.1 mg/kg/day** (BD) | **Trough 5–10 ng/mL** (early 8–12; late 5–8); adjust for toxicity |
| **Mycophenolate Mofetil (MMF)** | **1000–1500 mg BD** (max 3g/day) | MPA AUC if available; FBC, LFT |
| **Prednisolone** | **20mg/day → taper to 5mg by 3–6mo** | Lowest effective dose; minimize |

> [!key]
> **Tacrolimus + MMF + Pred = standard triple therapy.** Ciclosporin (older CNI) rarely used now. Belatacept (CTLA4-Ig) alternative CNI-sparing (monthly IV) — better GFR, less CNI toxicity, but higher acute rejection, EBV risk.

### CNI-Sparing / Alternative Regimens
| Regimen | Use Case |
|---------|----------|
| **Tacrolimus + MMF (steroid-free/withdrawal)** | Selected low-risk; avoid steroid side effects |
| **Belatacept + MMF + Pred** | CNI toxicity (nephrotoxicity, diabetes, neurotoxicity); EBV seropositive only |
| **Ciclosporin + MMF + Pred** | Tacrolimus intolerance (neurotoxicity, diabetes) |
| **Sirolimus/Everolimus (mTOR) + CNI reduction** | Malignancy risk reduction; wound healing issues (avoid early post-Tx) |

## Rejection

### Acute Cellular Rejection (ACR) — Banff Classification
| Grade | Histology | Treatment |
|-------|-----------|-----------|
| **Borderline** | Mild tubulitis, minimal inflammation | Optimise CNI; consider steroid pulse |
| **IA** | Moderate tubulitis (t2), interstitial infiltrate (i2) | **Methylprednisolone 500mg IV × 3 days** |
| **IB** | Severe tubulitis (t3), interstitial infiltrate (i3) | **Methylprednisolone 500mg IV × 3–5 days** ± **ATG** if steroid-resistant |
| **IIA/IIB** | **Arterial inflammation (v1/v2)** — vascular rejection | **ATG** (steroids alone inadequate) |
| **III** | **Transmural arteritis (v3)** — severe vascular | **ATG + plasma exchange ± IVIG** |

### Antibody-Mediated Rejection (AMR)
| Criteria (Banff) | Treatment |
|------------------|-----------|
| **DSA + C4d+ (or ptc >0) + microvascular inflammation (g+ptc)** | **Plasma exchange + IVIG + rituximab** ± **bortezomib** (refractory); **eculizumab** (severe) |
| **Chronic AMR** | Transplant glomerulopathy (cg >0) + DSA; difficult to treat; optimise CNI, consider belatacept |

### Monitoring for Rejection
| Parameter | Frequency |
|-----------|-----------|
| **Serum Creatinine** | 2–3x/week early; weekly × 1mo; monthly × 6mo; then q3mo |
| **Donor-Specific Antibodies (DSA)** | q3–6mo (or if Cr rise); Luminex single antigen bead |
| **Protocol Biopsy** | 3–6mo, 1yr (some centres) — detects subclinical rejection |
| **Urinalysis** | Proteinuria, haematuria |

## Post-Transplant Complications

### Infection (Commonest Cause of Death Year 1)
| Timeframe | Typical Infections |
|-----------|-------------------|
| **0–1 month** | Surgical (wound, UTI, pneumonia), donor-derived, HSV, CMV (if D+/R–) |
| **1–6 months** | **CMV (peak)**, EBV, BK virus, PJP, fungal (Candida, Aspergillus), community-acquired |
| **>6 months** | Community-acquired (UTI, pneumonia, skin), chronic viral (BK, CMV), TB |

**Prophylaxis (Standard):**
| Infection | Prophylaxis |
|-----------|-------------|
| **PJP** | TMP-SMX 1 year (or dapsone/atovaquone if allergic) |
| **CMV** | valganciclovir 3–6mo (D+/R– highest risk); pre-emptive monitoring alternative |
| **HSV/VZV** | Aciclovir/valaciclovir 3mo |
| **Fungal** | Fluconazole (if high risk) |
| **BK Virus** | No prophylaxis; **screen plasma/urine PCR q3mo × 2yr** |

### Malignancy (Leading Cause of Death After Year 1)
| Cancer | Risk | Screening |
|--------|------|-----------|
| **Skin (SCC, BCC, Melanoma)** | **50–100x** (sun protection, dermatology q6–12mo) |
| **PTLD** (Post-Transplant Lymphoproliferative Disorder) | EBV-driven; early (<1yr) = polymorphic; late = monomorphic | EBV PCR monitoring; reduce immunosuppression 1st step |
| **Kaposi Sarcoma** | HHV-8; reduce immunosuppression ± sirolimus |
| **Renal Cell Carcinoma** | In native kidneys (especially ADPKD, ESRD) | US native kidneys q1–2yr |
| **Cervical/Anal** | HPV; enhanced screening |

### Cardiovascular Disease
| Risk Factor | Management |
|-------------|------------|
| **Hypertension** | **<130/80**; CCB + ACEi/ARB (if allograft function allows); avoid CNI-induced |
| **Dyslipidaemia** | Statin (atorvastatin 20–40mg) — **interacts with CNI (dose adjust tacrolimus)** |
| **Diabetes** | **New-onset diabetes after transplant (NODAT)** — tacrolimus > ciclosporin; steroids; lifestyle + metformin/SGLT2i/GLP-1 RA |
| **Obesity** | Weight management |

### Recurrent Disease (in Allograft)
| Disease | Recurrence Rate | Management |
|---------|----------------|------------|
| **FSGS** | 30–50% (early, proteinuria) | **Plasmapheresis + rituximab + CNI optimisation** |
| **IgA Nephropathy** | 20–30% (late) | Optimise BP, ACEi/ARB, SGLT2i, fish oil |
| **Membranous Nephropathy** | <10% (if anti-PLA2R+) | Monitor anti-PLA2R; treat if recurrence |
| **C3G / aHUS** | High (complement-mediated) | Eculizumab/ravulizumab prophylaxis/pre-emptive |
| **Oxalosis** | Very high | Combined liver-kidney transplant |
| **ATTR Amyloidosis** | Nil (donor liver clears) | Liver-kidney if variant |

---

## BK Virus Nephropathy (BKVN)

| Feature | Detail |
|---------|--------|
| **Screening** | Plasma/urine BK PCR q3mo × 2yr (then annually) |
| **Diagnosis** | **Plasma BK >10⁴ copies/mL** + allograft dysfunction + **biopsy (SV40 T-antigen +)** |
| **Decoy Cells** | Urine cytology — suspicious but not diagnostic |
| **Treatment** | **Reduce immunosuppression** (↓ MMF/stop → ↓ Tac); **lefuoflovir/quinolones** (weak evidence); **IVIG**; **cidofovir** (nephrotoxic) |

---

## Pregnancy Post-Transplant

| Recommendation | Detail |
|----------------|--------|
| **Timing** | **Wait ≥1 year** post-transplant (stable graft, low immunosuppression) |
| **Immunosuppression** | **Stop MMF (teratogenic)** → switch to AZA/CNI/Pred pre-conception; **Tacrolimus/CsA/Pred/AZA safe**; Stop mTOR (sirolimus/everolimus) |
| **Monitoring** | Monthly Cr, BP, tacrolimus levels, DSA; obstetric + transplant joint care |
| **Outcomes** | Good if graft function stable; ↑ preeclampsia, preterm, SGA |

## High-Yield FCPS/MRCP Points

> [!important]
> - **Transplant = treatment of choice for ESRD; live donor > deceased**
> - **Recipient workup: CV, infection (HIV/HBV/HCV/CMV/TB), malignancy, HLA/PRA/DSA, psychosocial**
> - **Living donor: GFR >90, proteinuria <300mg, BP <140/90, single artery preferred**
> - **Induction: basiliximab (low risk) vs ATG (high risk/sensitised)**
> - **Maintenance: Tacrolimus + MMF + Pred (standard); target Tac trough 5–10 ng/mL**
> - **Rejection: ACR = steroid pulse ± ATG; AMR = PLEX + IVIG + rituximab + DSA monitoring**
> - **Infection: 0–1mo (surgical), 1–6mo (CMV, BK, PJP), >6mo (community); prophylaxis: TMP-SMX, valganciclovir, aciclovir**
> - **Malignancy: skin cancer (SCC/BCC) 50–100x; PTLD (EBV); Kaposi (HHV-8); screen skin q6–12mo**
> - **CVD: BP <130/80, statin (interacts with tacrolimus), NODAT (tacrolimus risk)**
> - **BK virus: screen PCR q3mo × 2yr; plasma >10⁴ + biopsy = BKVN; ↓ immunosuppression**
> - **Recurrent disease: FSGS (30–50%, PLEX+RTX), IgA (20–30%), aHUS (eculizumab)**
> - **Pregnancy: wait ≥1yr; stop MMF/mTOR; Tac/CsA/Pred/AZA safe**

## Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| **All patients get basiliximab** | **High risk (PRA>80%, DSA, retransplant, DCD) → ATG** |
| **Tacrolimus trough same always** | **Early (1–3mo): 8–12; Late: 5–8 ng/mL** |
| **MMF dose fixed 1g BD** | **1000–1500mg BD; reduce if leucopenia/GI side effects** |
| **Steroids stopped at 3mo** | **Taper to 5mg by 3–6mo; many continue low-dose lifelong** |
| **CNI = only nephrotoxicity** | Also neurotoxicity (tremor, headache, seizures), diabetes, hypertension, hyperkalaemia, hyperuricaemia |
| **Belatacept = for everyone** | **EBV seropositive only; higher acute rejection; monthly IV** |
| **DSA = always rejection** | Can be de novo non-complement fixing; monitor trend |
| **Protocol biopsy = routine everywhere** | Not universal; centre-dependent |
| **BKVN = treat with antivirals** | **1st step = reduce immunosuppression (↓ MMF, ↓ Tac)** |
| **Pregnancy = stop all immunosuppression** | **Stop MMF/mTOR only; continue Tac/CsA/Pred/AZA** |
| **Graft survival = patient survival** | Death with functioning graft common (CVD, infection, malignancy) |

## Summary

**Kidney Transplantation** is a **Must Know 🔴** topic.
**Key takeaway:** **Treatment of choice for ESRD.** Live donor > deceased. **Recipient workup:** CV, infection (HIV/HBV/HCV/CMV/TB), malignancy, HLA/PRA/DSA, psychosocial. **Living donor:** GFR >90, proteinuria <300mg, BP <140/90, single artery. **Induction:** basiliximab (low risk) vs **ATG (high risk/sensitised)**. **Maintenance:** **Tac + MMF + Pred**; Tac trough early 8–12, late 5–8. **Rejection:** ACR = steroid pulse ± ATG; AMR = PLEX + IVIG + rituximab. **Infection:** 0–1mo surgical, 1–6mo CMV/BK/PJP, >6mo community; **prophylaxis:** TMP-SMX, valganciclovir, aciclovir. **Malignancy:** skin 50–100x, PTLD (EBV), Kaposi (HHV-8). **CVD:** BP <130/80, statin (CNI interaction), NODAT. **BK virus:** PCR q3mo × 2yr; >10⁴ = BKVN → reduce IS. **Recurrent FSGS:** 30–50% → PLEX + rituximab. **Pregnancy:** wait ≥1yr; **STOP MMF/mTOR**; Tac/CsA/Pred/AZA safe.
**Exam focus:** Indications/workup, induction choice, tacrolimus targets, ACR vs AMR treatment, infection prophylaxis, malignancy screening, BK virus, recurrent FSGS, pregnancy management.
**Clinical relevance:** Transplant offers best survival/QoL; careful immunosuppression balance prevents rejection vs infection/malignancy; long-term follow-up essential.

# Pyridoxine (B6), Biotin & Pantothenic Acid

**Related:** [[Nutritional Factors in Disease MOC]], [[Davidson Chapter 22 - Nutritional Factors in Disease Hierarchy]], [[../00_Index/Medicine MOC|Medicine MOC]]

> [!important]
> **Pyridoxine (B6) = PLP cofactor for 140+ enzymes (transamination, decarboxylation, glycogen phosphorylase, haem synthesis); deficiency = sideroblastic anaemia, peripheral neuropathy, seizures, cheilosis, glossitis; Biotin = carboxylase cofactor; deficiency = dermatitis, alopecia, conjunctivitis; Pantothenic acid = CoA/ACP; deficiency RARE (burning feet syndrome).**

## 2. Definitions / Key Concepts

| Term | Definition |
|------|------------|
| **Pyridoxine (PN)** | Alcohol form; dietary; converted to PLP |
| **Pyridoxal (PL)** | Aldehyde form; phosphorylated to PLP |
| **Pyridoxamine (PM)** | Amine form; transamination intermediate |
| **PLP (Pyridoxal 5'-Phosphate)** | Active cofactor; Schiff base with lysine ε-amino group; >140 enzymes |
| **Transamination** | PLP-dependent; amino acid + α-ketoacid ↔ new amino acid + new α-ketoacid; ALT, AST |
| **Decarboxylation** | PLP-dependent: DOPA → dopamine (aromatic L-amino acid decarboxylase), 5-HTP → serotonin, glutamate → GABA (GAD), histidine → histamine |
| **Glycogen Phosphorylase** | PLP-dependent; glycogen → glucose-1-P; muscle/liver; PLP = structural (not catalytic) |
| **ALAS2 (δ-ALA Synthase 2)** | PLP-dependent; mitochondrial; glycine + succinyl-CoA → δ-ALA (haem synthesis); deficiency → sideroblastic anaemia |
| **Sideroblastic Anaemia** | Ring sideroblasts (Fe-loaded mitochondria around nucleus); microcytic/hypochromic/dimorphic; ↑ferritin, ↑Fe, ↓TIBC |
| **PLP Deficiency → ↓GABA** | Glutamate decarboxylase (GAD) requires PLP; ↓GABA → seizures (esp. infants), hyperirritability |
| **Pyridoxine Toxicity** | Sensory neuronopathy (dorsal column: loss vibration/position sense, ataxia); dose >100 mg/day chronic; reversible on cessation |
| **INH/Penicillamine/Hydralazine/OCP** | Form hydrazones with PLP → ↑renal excretion; INH also inhibits pyridoxal kinase |
| **Biotin (B7)** | Carboxylase cofactor: ATP + HCO₃⁻ + substrate → carboxylated substrate; holocarboxylase synthetase attaches to lysine |
| **Acetyl-CoA Carboxylase (ACC)** | Acetyl-CoA → malonyl-CoA (rate-limiting fatty acid synthesis) |
| **Pyruvate Carboxylase (PC)** | Pyruvate → oxaloacetate (anaplerosis, gluconeogenesis) |
| **Propionyl-CoA Carboxylase (PCC)** | Propionyl-CoA → methylmalonyl-CoA (odd-chain FA, BCAA, cholesterol) |
| **Methylcrotonyl-CoA Carboxylase (MCC)** | Leucine catabolism |
| **Avidin** | Raw egg white glycoprotein; binds biotin 10¹⁵ affinity → biotin deficiency; heat denatures avidin |
| **Biotinidase Deficiency** | BTD mutation (AR) → cannot recycle biotin; late-onset (weeks–months); seizures, hypotonia, dermatitis, alopecia, organic aciduria |
| **Holocarboxylase Synthetase Deficiency** | HLCS mutation (AR) → cannot attach biotin to carboxylases; early-onset (neonatal); severe |
| **Pantothenic Acid (B5)** | Component of CoA (4'-phosphopantetheine) + ACP (fatty acid synthase); ubiquitous in foods |

## 3. Core Content

### Section 1: Pyridoxine (B6) Metabolism
**Forms:** Pyridoxine (PN, alcohol), pyridoxal (PL, aldehyde), pyridoxamine (PM, amine); interconvertible.
**Absorption:** Jejunum; passive diffusion; phosphorylated by pyridoxal kinase → PLP (liver, RBC, brain).
**PLP-dependent enzymes (6 classes):**
1. **Transaminases:** ALT, AST, BCAT (BCAA) — amino group transfer
2. **Decarboxylases:** DOPA decarboxylase, 5-HTP decarboxylase, GAD (glutamate → GABA), histidine decarboxylase
3. **Glycogen phosphorylase:** Structural PLP; glycogenolysis
4. **Haem synthesis:** ALAS2 (δ-ALA synthase 2) — erythroid-specific
5. **Cystathionine β-synthase (CBS):** Homocysteine + serine → cystathionine (transsulfuration); PLP + B12 (methylcobalamin) for homocysteine remethylation
6. **Kynureninase:** Tryptophan → niacin pathway; INH/B6 deficiency → inhibition
**Catabolism:** PLP → pyridoxic acid (excreted in urine); 4-pyridoxic acid = major metabolite.

### Section 2: Pyridoxine Deficiency
**Causes:** INH, penicillamine, hydralazine, isoniazid, OCP, alcohol, malabsorption, dialysis, pregnancy, lactation, 5-FU, cycloserine, theophylline.
**Clinical Features:**
| System | Features |
|--------|----------|
| **Haematological** | **Sideroblastic anaemia** (↓ALAS2 → ↓haem → ring sideroblasts); microcytic/hypochromic/dimorphic; ↑serum Fe, ↑ferritin, ↓TIBC |
| **Neurological** | Peripheral neuropathy (sensory, axonal); ↓GABA → seizures (infants), hyperirritability, EEG changes |
| **Dermatological** | Cheilosis, angular stomatitis, glossitis, seborrhoeic dermatitis (nasolabial, scrotal) |
| **Metabolic** | ↓Cystathionine β-synthase → ↑homocysteine; ↓kynureninase → ↑xanthurenic acid (tryptophan load test) |

**Diagnosis:** Plasma PLP <20 nmol/L (ref 20–150); erythrocyte aspartate aminotransferase activity coefficient (EAST-AC) >1.8; urinary 4-pyridoxic acid <2 µmol/day; tryptophan load test ↑xanthurenic acid.

### Section 3: Pyridoxine Toxicity
**Sensory neuronopathy:** Dose >100 mg/day chronic (typically >500 mg/day); large sensory fibre degeneration (dorsal columns); loss of vibration/position sense, sensory ataxia; **reversible on stopping** (months); motor spared.
**Mechanism:** PLP competes with GABA?; direct toxicity to dorsal root ganglia.
**Treatment:** Stop pyridoxine; symptomatic; recovery over 6–24 months.

### Section 4: Biotin (B7) Metabolism
**Sources:** Liver, egg yolk, nuts, seeds, salmon, avocado, sweet potato; gut flora synthesis (significant contribution).
**Absorption:** Small intestine; SMVT (sodium-dependent multivitamin transporter, SLC5A6); also colonic absorption from flora.
**Holocarboxylase synthetase (HLCS):** Attaches biotin to lysine ε-amino group of apocarboxylases (ATP-dependent).
**Biotinidase (BTD):** Recycles biotin from biocytin (biotin-lysine) and dietary protein-bound biotin; serum biotinidase activity = diagnostic.
**Carboxylases (4 mitochondrial):**
1. **Acetyl-CoA Carboxylase (ACC):** Acetyl-CoA + HCO₃⁻ → malonyl-CoA (fatty acid synthesis rate-limiting)
2. **Pyruvate Carboxylase (PC):** Pyruvate + HCO₃⁻ → oxaloacetate (anaplerosis, gluconeogenesis)
3. **Propionyl-CoA Carboxylase (PCC):** Propionyl-CoA → methylmalonyl-CoA (odd-chain FA, BCAA: Val, Ile, Met, Thr; cholesterol side-chain)
4. **Methylcrotonyl-CoA Carboxylase (MCC):** Leucine → β-methylcrotonyl-CoA → β-methylglutaconyl-CoA

### Section 5: Biotin Deficiency
**Causes:**
- **Raw egg whites:** Avidin binds biotin (Kd 10⁻¹⁵ M); heat denatures avidin
- **TPN without biotin** (historical; now added)
- **Prolonged broad-spectrum antibiotics** → ↓gut flora synthesis
- **Biotinidase deficiency (BTD):** AR; profound (<10%) or partial (10–30%); neonatal screening (many countries)
- **Holocarboxylase synthetase deficiency (HLCS):** AR; neonatal onset; severe
- **Anticonvulsants (valproate, carbamazepine, phenytoin, phenobarbital)** → ↑biotin catabolism
- **Pregnancy/lactation** → ↑demand, marginal status common

**Clinical Features:**
- **Dermatitis:** Scaly, erythematous, periorificial (eyes, nose, mouth, perineum); "biotin-deficient facies"
- **Alopecia:** Total (universalis) or patchy
- **Conjunctivitis:** Blepharoconjunctivitis
- **Neurological:** Hypotonia, lethargy, seizures, ataxia, developmental delay (untreated children)
- **Metabolic:** Lactic acidosis, organic aciduria (3-hydroxyisovaleric acid, methylcitrate), hyperammonaemia (↓PCC → propionyl-CoA accumulation)
- **Immunological:** ↓T-cell function, candidiasis

**Diagnosis:** Serum biotinidase activity (profound <10%, partial 10–30%); urinary organic acids (↑3-hydroxyisovalerate, methylcitrate); plasma biotin (low); genetic testing BTD/HLCS.

### Section 6: Pantothenic Acid (B5)
**CoA synthesis:** Pantothenate → 4'-phosphopantetheine → CoA (universal acyl carrier); ACP (fatty acid synthase).
**Ubiquitous:** "Pantos" = everywhere; deficiency virtually unknown in humans.
**Experimental deficiency (burning feet syndrome):** Paresthesia, burning feet, fatigue, headache, insomnia, GI distress; prisoner studies WWII.
**Pantetheine / Pantethine:** Used for dyslipidaemia (↓TG, ↑HDL); cystinosis (cysteamine depletion).

## 5. High-Yield FCPS/MRCP Points

> [!important]
> - **Must know:** PLP cofactor for transamination, decarboxylation, glycogen phosphorylase, ALAS2 (haem), CBS (homocysteine); deficiency = sideroblastic anaemia, neuropathy, seizures, cheilosis; INH/penicillamine/hydralazine/OCP cause; toxicity = sensory neuronopathy >100 mg/day; Biotin → carboxylases (ACC, PC, PCC, MCC); deficiency = raw egg whites (avidin), antibiotics, biotinidase deficiency → dermatitis, alopecia, conjunctivitis, organic aciduria; Pantothenic acid = CoA/ACP; deficiency RARE (burning feet)
> - **Common viva:** Sideroblastic anaemia mechanism (ALAS2); pyridoxine drugs (INH, penicillamine, hydralazine); pyridoxine toxicity vs deficiency; biotin carboxylases; avidin-raw egg white; biotinidase deficiency screening; burning feet syndrome = pantothenic acid
> - **Exam trap:** Confusing pyridoxine toxicity (sensory neuronopathy) with deficiency (neuropathy); sideroblastic anaemia = microcytic but ↑Fe/ferritin; biotin deficiency vs zinc deficiency (both dermatitis/alopecia); pantothenic acid deficiency never seen clinically

## 6. Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| Pyridoxine toxicity = deficiency neuropathy | **Toxicity = sensory neuronopathy (dorsal column, >100 mg/day)**; Deficiency = sensorimotor axonal neuropathy |
| Sideroblastic = iron deficiency | **Sideroblastic: microcytic but ↑Fe, ↑ferritin, ring sideroblasts**; Iron deficiency: ↓Fe, ↓ferritin |
| Biotin deficiency = zinc deficiency | **Both: dermatitis + alopecia**; Biotin: conjunctivitis, organic aciduria, seizures; Zinc: diarrhoea, immune, taste loss |
| Pantothenic acid deficiency = common | **EXTREMELY RARE**; only experimental "burning feet syndrome" |
| Avidin in cooked eggs | **Heat denatures avidin**; only RAW egg whites bind biotin |
| Biotinidase deficiency = HLCS deficiency | **BTD: biotin recycling defect (later onset); HLCS: biotin attachment defect (neonatal, severe)** |
| Pyridoxine for all neuropathies | **Only B6-responsive** (sideroblastic, INH, deficiency); toxic at high dose |

## 14. Exam Answer Modes

### Long Answer / Essay (20 min)
**Topic:** "Pyridoxine, Biotin, and Pantothenic Acid: metabolism, deficiency, and toxicity"
- Pyridoxine: PN/PL/PM → PLP; >140 enzymes (transaminases, decarboxylases, glycogen phosphorylase, ALAS2, CBS, kynureninase); deficiency: sideroblastic anaemia, neuropathy, seizures, cheilosis; drugs: INH, penicillamine, hydralazine, OCP; toxicity: sensory neuronopathy >100 mg/day
- Biotin: carboxylase cofactor (ACC, PC, PCC, MCC); avidin (raw egg), antibiotics, BTD/HLCS deficiency; features: periorificial dermatitis, alopecia, conjunctivitis, organic aciduria, seizures
- Pantothenic acid: CoA + ACP component; ubiquitous; deficiency only experimental (burning feet syndrome)

### Short Note (7 min)
**Topic:** "Sideroblastic Anaemia in Pyridoxine Deficiency"
- **Mechanism:** ALAS2 (δ-ALA synthase 2) is PLP-dependent; mitochondrial enzyme in erythroblasts; glycine + succinyl-CoA → δ-ALA (first step haem synthesis)
- **Deficiency:** ↓PLP → ↓ALAS2 activity → ↓haem synthesis → iron accumulates in mitochondria → **ring sideroblasts** (Prussian blue +)
- **Blood picture:** Microcytic hypochromic dimorphic (mixed microcytic + normocytic); ↑serum iron, ↑ferritin, ↑transferrin saturation, ↓TIBC
- **Causes:** Pyridoxine deficiency (INH, alcohol), lead poisoning, myelodysplasia, alcohol, drugs (chloramphenicol)
- **Treatment:** Pyridoxine 50–100 mg/day (pyridoxine-responsive); if unresponsive → transfusion support

### Viva Answer (3 min)
**Q:** "Differentiate pyridoxine deficiency neuropathy from pyridoxine toxicity neuropathy."
"A: **Deficiency:** sensorimotor axonal neuropathy (distal, symmetrical, glove-stocking); **Toxicity:** pure **sensory neuronopathy** (dorsal column: loss vibration/position sense, sensory ataxia, pseudoathetosis); **dose >100 mg/day chronic**; toxicity reversible on stopping (6–24 months); motor spared in toxicity."

### Ward Case Discussion (5 min)
**Case:** 35M on INH/RIF/PZA/EMB for TB, 3 months, presents with fatigue, pallor. Hb 8 g/dL, MCV 72, dimorphic blood film, ring sideroblasts on marrow, serum Fe 180 µg/dL, ferritin 800.

## 18. Answer Key with Explanations

### MCQs
1. **C** — PLP is the catalytically active coenzyme form.
2. **B** — ALAS2 (erythroid δ-ALA synthase) requires PLP; first step of haem synthesis.
3. **B** — Sideroblastic anaemia: iron overload (↑Fe, ↑ferritin, ↓TIBC) due to impaired haem synthesis → mitochondrial iron accumulation.
4. **B** — INH forms hydrazone with PLP → increased renal excretion; also inhibits pyridoxal kinase.
5. **B** — Chronic high-dose pyridoxine (>100 mg/day) → sensory neuronopathy (dorsal column degeneration); motor spared; reversible on cessation.
6. **A** — APMM: Acetyl-CoA carboxylase (FA synthesis), Pyruvate carboxylase (gluconeogenesis/anaplerosis), Propionyl-CoA carboxylase (odd-chain FA/BCAA), Methylcrotonyl-CoA carboxylase (leucine catabolism).
7. **B** — Avidin (raw egg white) binds biotin with extremely high affinity (Kd 10⁻¹⁵ M); cooking denatures avidin.
8. **B** — BTD (biotinidase) recycles biotin from biocytin/dietary protein (later onset); HLCS attaches biotin to apocarboxylases (neonatal, severe).
9. **B** — DAC: Dermatitis (periorificial scaly), Alopecia (universalis), Conjunctivitis.
10. **B** — Pantothenic acid experimental deficiency = burning feet syndrome (paresthesia, dysesthesia); virtually never seen in normal populations.

### SBAs
1. **B** — INH-induced sideroblastic anaemia: pyridoxine 100 mg/day reverses; prophylaxis 10–25 mg/day should be routine; continue INH.
2. **B** — Biotin deficiency: scaly periorificial dermatitis, alopecia, conjunctivitis, metabolic acidosis, ↑3-hydroxyisovalerate (MCC substrate); TPN without biotin or raw egg whites.
3. **B** — Pyridoxine toxicity: sensory neuronopathy (large fibre dorsal column) at doses >100 mg/day chronic; absent sensory NCS, normal motor; reversible on stopping.
4. **B** — Avidin in raw egg whites consumed by mother → secreted in breast milk → infant biotin deficiency; maternal biotin supplementation also helps.
5. **B** — Pantothenic acid deficiency = "burning feet syndrome" (experimental, WWII prisoners); virtually never seen in clinical practice; ubiquitous in foods.

## 19. Summary

**Pyridoxine (B6), Biotin (B7) & Pantothenic Acid (B5)** are **Must Know 🔴** topics for FCPS/MRCP.
**Key takeaway:** Pyridoxine → PLP (transamination, decarboxylation, glycogen phosphorylase, ALAS2, CBS, kynureninase); deficiency = sideroblastic anaemia, neuropathy, seizures, cheilosis; drugs (INH, penicillamine, hydralazine, OCP) cause deficiency; toxicity = sensory neuronopathy >100 mg/day. Biotin → carboxylases (ACC, PC, PCC, MCC); deficiency: raw egg whites (avidin), antibiotics, BTD/HLCS → periorificial dermatitis, alopecia, conjunctivitis, organic aciduria. Pantothenic acid = CoA + ACP; ubiquitous; deficiency virtually non-existent (burning feet syndrome).
**Exam focus:** Sideroblastic anaemia mechanism, B6 drug interactions, pyridoxine toxicity vs deficiency, biotin carboxylases (APMM), avidin, biotinidase vs HLCS, DAC features.
**Clinical relevance:** INH prophylaxis mandatory; TPN biotin supplementation; anticonvulsant-biotin monitoring; pyridoxine dosing safety.

## 2. Definitions / Key Concepts

| Term | Definition |
|------|------------|
| **Riboflavin (B2)** | Flavin mononucleotide (FMN) + flavin adenine dinucleotide (FAD); yellow fluorescent |
| **FMN** | Riboflavin-5'-phosphate; 1-electron transfers; Complex I (NADH dehydrogenase) |
| **FAD** | FMN + ATP; 2-electron transfers; Complex II (succinate dehydrogenase), acyl-CoA dehydrogenase, MTHFR |
| **Glutathione Reductase** | FAD-dependent; regenerates GSH from GSSG; riboflavin deficiency → ↓GSH → oxidative stress |
| **MTHFR** | FAD-dependent; 5,10-methylene-THF → 5-methyl-THF (homocysteine remethylation) |
| **Cheilosis** | Fissures at corners of mouth (angular stomatitis); riboflavin deficiency |
| **Magenta Glossitis** | Smooth, beefy red, swollen tongue; atrophy of papillae; B2/B3/B12 deficiency |
| **Corneal Vascularisation** | Superficial neovascularisation of cornea; pathognomonic riboflavin deficiency |
| **Niacin (B3)** | Nicotinic acid + nicotinamide; precursor of NAD⁺/NADP⁺; diet + tryptophan (60:1) |
| **NAD⁺/NADH** | Oxidised/reduced; catabolic dehydrogenases (glycolysis, Krebs, β-oxidation) |
| **NADP⁺/NADPH** | Anabolic (fatty acid synthesis, glutathione reductase, PPP) |
| **Pellagra** | 3 D's: **D**ermatitis, **D**iarrhoea, **D**ementia (+ **D**eath); photosensitivity rash |
| **Casal's Necklace** | Symmetrical hyperpigmented rash on sun-exposed neck/Casal area; pellagra hallmark |
| **Tryptophan → Niacin** | 60 mg tryptophan → 1 mg niacin; requires B2 (FAD), B6 (PLP), Fe; kynurenine pathway |
| **Hartnup Disease** | SLC6A19 mutation → neutral AA malabsorption (tryptophan) → pellagra-like rash, cerebellar ataxia |
| **Carcinoid Syndrome** | Serotonin-secreting NET → tryptophan → serotonin (not niacin) → pellagra |
| **Isoniazid (INH)** | Anti-TB; hydrazine → inhibits pyridoxal kinase → B6 deficiency AND kynureninase → niacin deficiency |
| **Niacin Flush** | Nicotinic acid → prostaglandin D2 → cutaneous vasodilation, pruritus; absent with nicotinamide |
| **NAD⁺ Precursors** | Nicotinamide riboside (NR), nicotinamide mononucleotide (NMN) — anti-ageing research |

## 3. Core Content

### Section 1: Riboflavin (B2) Metabolism
**Sources:** Milk, eggs, liver, green veg, fortified cereals; light-sensitive (opaque containers).
**Absorption:** Small intestine; carrier-mediated (RFVT1/SLC52A1, RFVT2/SLC52A2, RFVT3/SLC52A3); phosphorylation to FMN → FAD.
**Cofactors:** FMN (Complex I), FAD (Complex II, acyl-CoA dehydrogenase, MTHFR, glutathione reductase, xanthine oxidase, monoamine oxidase).
**Deficiency (ariboflavinosis):** Rare isolated; usually with other B-vitamin deficiencies.
**Features:** Cheilosis, angular stomatitis, magenta glossitis, corneal vascularisation (superficial neovascularisation), seborrhoeic dermatitis (nasolabial, scrotal/vulval), normocytic normochromic anaemia (↓erythropoietin), peripheral neuropathy.
**At risk:** Alcohol, elderly, anorexia, phototherapy (bilirubin degradation), vegans (no dairy).
**Diagnosis:** Erythrocyte glutathione reductase activity coefficient (EGRAC) >1.2–1.4; urinary riboflavin <27 µg/g creatinine.
**Treatment:** Riboflavin 10–30 mg/day oral; multivitamin if multiple deficiencies.

### Section 2: Niacin (B3) Metabolism
**Forms:** Nicotinic acid (niacin), nicotinamide (niacinamide); both → NAD/NAD.
**Dietary sources:** Meat, fish, poultry, peanuts, legumes, fortified grains; coffee (high).
**Tryptophan pathway:** 60 mg tryptophan → 1 mg niacin (via kynurenine pathway); requires B2 (FAD for kynurenine 3-monooxygenase), B6 (PLP for kynureninase), Fe.
**NAD⁺ synthesis pathways:**
1. Preiss-Handler: Nicotinic acid → NaMN → NaAD → NAD⁺
2. Salvage: Nicotinamide → NMN → NAD⁺
3. De novo: Tryptophan → kynurenine → 3-hydroxykynurenine → 3-hydroxyanthranilate → quinolinate → NAD⁺
**Functions:** NAD⁺ (catabolism: glycolysis, PDH, Krebs, β-oxidation, ETC); NADPH (anabolism: fatty acid/cholesterol synthesis, PPP, glutathione reductase, CYP450).

### Section 3: Pellagra — Clinical Features
| D | Feature | Details |
|---|---------|---------|
| **Dermatitis** | Photosensitive rash | Symmetrical, sun-exposed (face, neck-Casal's necklace, hands-glove, feet-boot); hyperpigmented, scaling, vesiculobullous → hyperkeratotic |
| **Diarrhoea** | GI involvement | Watery, sometimes bloody; glossitis, stomatitis, gastritis, malabsorption |
| **Dementia** | Neuropsychiatric | Confusion, memory loss, disorientation, depression, hallucinations, psychosis; reversible if early |
| **Death** | Untreated | Multi-organ failure; cardiovascular collapse |

**Pellagrous dermatosis:** "Casal's necklace" (broad collar rash on neck), "gauntlet" rash on forearms, "boot" rash on legs; sharply demarcated at衣领/ sleeve lines.

### Section 4: Secondary Niacin Deficiency
| Cause | Mechanism |
|-------|-----------|
| **Hartnup Disease** | SLC6A19 (B⁰AT1) mutation → neutral AA malabsorption (tryptophan) → ↓niacin synthesis; cerebellar ataxia, nystagmus, pellagra rash; autosomal recessive |
| **Carcinoid Syndrome** | Serotonin-secreting NET → tryptophan → 5-HTP → serotonin (via TPH, AADC) → ↑5-HIAA; tryptophan diverted from niacin pathway |
| **Isoniazid (INH)** | Hydrazine metabolite → inhibits pyridoxal kinase (B6) → ↓PLP → kynureninase inhibition → ↑xanthurenic acid, ↓niacin; give pyridoxine 10–25 mg/day prophylaxis |
| **Carcinoid + INH** | Dual hit → severe pellagra |
| **Malnutrition / Alcohol** | ↓Intake + ↓tryptophan → niacin |
| **Carcinoid + Hartnup** | Rare combination |

### Section 5: Diagnosis & Treatment
**Diagnosis:** Urinary **N¹-methylnicotinamide (NMN)** or **N¹-methyl-2-pyridone-5-carboxamide (2-PYR)** excretion <0.8 mg/day; plasma NAD⁺; clinical response to niacin.
**Treatment:**
- **Nicotinamide 300 mg/day PO** (divided doses) — preferred (no flush)
- **Nicotinic acid 100–300 mg/day** — causes flush (prostaglandin D2); aspirin 30 min prior reduces
- **Hartnup:** High-protein diet + nicotinamide 50–100 mg/day
- **Carcinoid:** Nicotinamide 300 mg/day + treat NET; tryptophan supplementation controversial
- **INH prophylaxis:** Pyridoxine 10–25 mg/day (prevents B6 + B3 deficiency)

### Section 6: Niacin Toxicity & Therapeutic Use
**Therapeutic (lipid-lowering):** Nicotinic acid 1.5–3 g/day → ↓LDL 10–20%, ↓TG 20–50%, ↑HDL 15–35%; **flushing** (PGD2 via GPER109A on Langerhans cells); laropiprant (PGD2 receptor antagonist) withdrawn.
**Adverse:** Hepatotoxicity (time-release > immediate-release), hyperglycaemia, hyperuricaemia (↓renal urate excretion), GI upset, hypotension.
**Nicotinamide:** No flush, no lipid effect, safe in high dose; used for pellagra, actinic keratosis prevention, osteoarthritis (cartilage).
**NAD⁺ boosters:** Nicotinamide riboside (NR), NMN — anti-ageing, sirtuin activation; research phase.

## 6. Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| Niacin = nicotinamide for lipids | **Only nicotinic acid** lowers lipids; nicotinamide = no flush, no lipid effect |
| Pellagra rash = photosensitive eczema | **Sharply demarcated at clothing lines** (Casal's necklace, glove, boot) |
| Tryptophan → niacin always works | **Needs B2 (FAD), B6 (PLP), Fe**; Hartnup = no tryptophan absorption |
| Corneal vascularisation = B12 deficiency | **Riboflavin deficiency** = superficial corneal vascularisation; B12 = optic neuropathy |
| INH only causes B6 deficiency | **Also causes niacin deficiency** (kynureninase inhibition via B6 depletion) |
| Carcinoid diarrhoea = pellagra diarrhoea | **Carcinoid: secretory (5-HT); Pellagra: inflammatory (glossitis, gastritis)** |
| Hartnup = cystinuria | **Hartnup: neutral AA transporter (SLC6A19); Cystinuria: dibasic AA (SLC3A1/SLC7A9)** |
| Niacin flush = anaphylaxis | **Prostaglandin D2-mediated**; aspirin 325 mg 30 min prior prevents |

- Niacin: NAD/NADP; tryptophan 60:1 (requires B2, B6, Fe); pellagra: 3 D's (dermatitis Casal's necklace, diarrhoea, dementia); Casal's necklace hallmark

### Short Note (7 min)
**Topic:** "Pellagra"
- **3 D's:** Dermatitis (photosensitive, Casal's necklace, glove/boot distribution), Diarrhoea (glossitis, gastritis), Dementia (confusion, psychosis), Death if untreated
- **Pathophysiology:** Niacin deficiency → ↓NAD/NADP → impaired cellular respiration
- **Secondary causes:** Hartnup (tryptophan malabsorption), carcinoid (tryptophan → serotonin), INH (B6 depletion → kynureninase inhibition), malnutrition
- **Treatment:** Nicotinamide 300 mg/day PO divided (no flush); dietary niacin/tryptophan; treat underlying cause

### Viva Answer (3 min)

Hartnup: autosomal recessive, high-protein diet + nicotinamide; Pellagra: treat cause + nicotinamide."

2. **B** — Carcinoid: serotonin-secreting NET diverts tryptophan from niacin pathway → secondary pellagra.
3. **B** — INH prophylaxis: pyridoxine 10–25 mg/day prevents B6 deficiency (neuropathy) and secondary B3 deficiency (pellagra).
4. **B** — Cheilosis, angular stomatitis, magenta glossitis, corneal vascularisation = classic riboflavin (B2) deficiency (C-4 mnemonic).
5. **B** — Aspirin 325 mg 30 min before nicotinic acid inhibits PGD2 synthesis → reduces flush; laropiprant withdrawn.

## 19. Summary

**Riboflavin (B2) & Niacin (B3)** are **Must Know 🔴** topics for FCPS/MRCP.
**Key takeaway:** Riboflavin → FMN (Complex I) + FAD (Complex II, MTHFR, GSH reductase); deficiency = cheilosis, angular stomatitis, magenta glossitis, **corneal vascularisation**, normocytic anaemia. Niacin → NAD/NADP; tryptophan 60:1 (needs B2, B6, Fe); **pellagra = 3 D's (Dermatitis Casal's necklace, Diarrhoea, Dementia, Death)**. Secondary: Hartnup (SLC6A19 neutral AA → tryptophan malabsorption + cerebellar ataxia), carcinoid (tryptophan → serotonin), INH (B6 depletion → kynureninase inhibition). **Treatment: Nicotinamide 300 mg/day (no flush); INH prophylaxis: pyridoxine 10–25 mg/day.**
**Exam focus:** Riboflavin C-4 signs, pellagra 3 D's + Casal's necklace, Hartnup + cerebellar signs, carcinoid tryptophan diversion, INH dual B6/B3 deficiency, nicotinamide vs nicotinic acid.
**Clinical relevance:** Alcohol malnutrition = multiple B-vitamin deficiencies; TB treatment monitoring; carcinoid syndrome workup; genetic transporter disorders.

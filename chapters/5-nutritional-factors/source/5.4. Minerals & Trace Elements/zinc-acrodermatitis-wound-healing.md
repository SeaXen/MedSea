## 2. Definitions / Key Concepts

| Term | Definition |
|------|------------|
| **Zinc (Zn²⁺)** | Essential trace element; 2–3 g body (60% muscle, 30% bone, 0.1% plasma); structural/catalytic/regulatory |
| **ZIP4 (SLC39A4)** | Apical enterocyte Zn importer; gene mutation = **acrodermatitis enteropathica** (AR) |
| **ZnT1 (SLC30A1)** | Basolateral Zn exporter (enterocyte) |
| **ZIP10, ZIP11, ZnT2, ZnT3** | Tissue-specific Zn transporters; ZIP = import, ZnT = transport out of cytosol |
| **ZnT2 (SLC30A2)** | Mammary gland Zn transport; mutation = transient neonatal Zn deficiency (low breast milk Zn) |
| **Metallothionein (MT)** | Cysteine-rich Zn/Cu-binding protein; Zn induces MT in enterocyte → binds Cu → ↓Cu absorption; antioxidant |
| **Zn Finger Motif** | Cys2His2 (most common) or Cys4 structure; Zn tetrahedrally coordinates; DNA/RNA/protein binding |
| **Zn Superoxide Dismutase (SOD1)** | Zn/Cu-dependent; antioxidant; Cu-Zn SOD |
| **Zn Metalloenzymes** | >300; alcohol dehydrogenase, carbonic anhydrase, alkaline phosphatase, carboxypeptidase, DNA/RNA polymerase |
| **MTF1** | Metal-responsive transcription factor; Zn sensing → MT gene expression |
| **Acrodermatitis Enteropathica (AE)** | AR; SLC39A4 (ZIP4) mutation; dermatitis (perioral/perianal), alopecia, diarrhoea, irritability |
| **Periorificial Dermatitis** | Erythematous, scaly, vesiculobullous; perioral, perianal, perineal, acral; Zn deficiency |
| **Dysseborrhoeic Dermatitis** | Zn deficiency rash in seborrheic distribution; differentiates from vitamin B3/B6 deficiency |
| **Beau's Lines** | Transverse nail ridges; Zn, severe illness, chemo |
| **Hypogonadism (Zn)** | ↓Testosterone, ↓spermatogenesis, delayed puberty; Zn-dependent testosterone synthesis |
| **Wilson's Disease** | ATP7B mutation → Cu overload; **Zn therapy** (induces MT) |
| **Acute Zn Deficiency** | NUTRI-IV (Nutritional Intervention), TPN without Zn, penicillamine |
| **Wilson + Zn Trial** | Zn vs penicillamine (CHELATE trial; Zn non-inferior for neurological WD) |
| **Sideroblastic Anaemia (Cu Def)** | Cu deficiency (Zn excess) → ↓hephaestin → ↓Fe utilisation → ring sideroblasts |

## 3. Core Content

### Section 1: Zinc Metabolism
**Body content:** 2–3 g in adult; 60% muscle, 30% bone, 0.1% plasma; liver, kidney, brain, retina also rich.
**Absorption:** Duodenum, proximal jejunum; **ZIP4 (SLC39A4)** apical import; passive diffusion at high dose; **ZnT1** basolateral export; **MT** sequesters in enterocyte.
**Regulation:** ZIP4 expression ↑in Zn deficiency (adaptive); MT binds Zn (prevents overshoot); **paracellular** (tight junctions) at high dose.
**Enhancers:** Animal protein (muscle, fish, eggs), histidine, cysteine.
**Inhibitors:** Phytates (whole grains, legumes), oxalates, polyphenols, **Fe** (competes), **Cu** (competes), **Ca** (dairy), oral contraceptives.
**Plasma:** Bound to albumin 60%, α2-macroglobulin 30%, transferrin 10%; **0.1% free**; plasma Zn poorly reflects tissue stores.
**Excretion:** Pancreatic (Zn in pancreatic juice) 30%; intestinal (desquamation); sweat (minor); renal 10% (filtered + reabsorbed).
**Tissue turnover:** Slow; body pool replaced every 1–2 weeks.

### Section 2: Functions
1. **Structural:** >300 **zinc finger proteins** (DNA/RNA binding, transcription factors); steroid receptors (oestrogen, androgen, glucocorticoid); receptor tyrosine kinases; GPCRs; cytoskeletal (tubulin).
2. **Catalytic:** >300 **Zn metalloenzymes**:
   - **Alcohol dehydrogenase** (Zn + NAD⁺)
   - **Carbonic anhydrase** (CO₂ + H₂O ↔ H₂CO₃; Zn + His)
   - **Alkaline phosphatase** (Zn + Mg²⁺)
   - **Carboxypeptidases** (pancreatic)
   - **RNA polymerase** (DNA transcription)
   - **Reverse transcriptase** (HIV)
   - **Matrix metalloproteinases** (collagen degradation)
   - **Cu/Zn-SOD** (SOD1, antioxidant)
3. **Regulatory:** Transcription factors, immune (T-cell development, cytokine production), signal transduction, wound healing (collagen synthesis, epithelial cell migration), growth/development, fertility/testosterone, antioxidant, apoptosis.

### Section 3: Requirements & Sources
**RDA (IOM 2001):**
- Adult male: **11 mg/day** (was 15 mg)
- Adult female: 8 mg/day
- Pregnancy: 11 mg/day (UK 7; some 12–15)
- Lactation: 12 mg/day
- Children: 2–8 mg by age
- Smokers/alcohol: +20–30%

**UL:** 40 mg/day (acute toxicity); chronic high → Cu deficiency
**Sources (mg/100 g):**
- Oysters 78, Beef 12, Pumpkin seeds 10, Cashews 5, Chickpeas 2, Lentils 1.5
- Animal protein > plant (phytate binding reduces bioavailability)

### Section 4: Acrodermatitis Enteropathica (AE)
**Genetics:** AR; **SLC39A4 (ZIP4)** mutation on chromosome 8q24; loss of ZIP4 function → ↓Zn absorption; presents after weaning (breast milk Zn ligand compensates partially).
**Classic Triad:**
1. **Dermatitis:** Perioral, perianal, perineal, acral, hands, feet; erythematous, scaly, vesiculobullous, pustular, eczematoid; **alopecia, nail dystrophy, paronychia**
2. **Diarrhoea:** Chronic, watery; malabsorption
3. **Alopecia:** Total/partial
4. **Mood changes:** Apathy, irritability, depression
**Other:** Hypogonadism, growth retardation, immune dysfunction (candidiasis, recurrent infections), ophthalmic (blepharitis, conjunctivitis, photophobia), taste/smell loss.
**Biochemistry:** ↓Plasma Zn <70 µg/dL; ↓Alkaline phosphatase (Zn-dependent); low sweat Zn.
**Treatment:** **Zn supplementation 3 mg/kg/day** elemental (lifelong); clinical response in 5–10 days; monitor plasma Zn 3-monthly.

### Section 5: Acquired Zinc Deficiency
| Category | Examples |
|----------|----------|
| **Malabsorption** | Coeliac, Crohn's, short bowel, bariatric (RYGB), CF, biliary atresia |
| **Chronic disease** | Liver cirrhosis (↑renal Zn loss), CKD/dialysis, DM (hyperzincuria) |
| **Drugs** | **Penicillamine** (chelation), diuretics, ACE inhibitors, antimetabolites, OCP |
| **↑ Demand** | Pregnancy, lactation, premature infants, growth, burns, surgery, sepsis |
| **Diet** | Vegan, alcoholism, elderly ("tea-and-toast"), anorexia, formula without Zn |
| **Genetic** | AE (ZIP4), acrodermatitis enteropathica, MTF1 mutations (rare) |
| **Hepatic** | Acrodermatitis dysseborrhoica in liver disease (empirical Zn) |

**Clinical features (acquired):**
- **Skin:** Perioral/perianal dermatitis, eczema, hyperkeratosis, seborrhoeic rash; similar to AE
- **Hair:** Alopecia, brittle hair
- **Nails:** Beau's lines, paronychia, onychodystrophy
- **GI:** Diarrhoea, anorexia, taste abnormalities
- **Neuro:** Depression, irritability, cognitive decline, taste (dysgeusia) and smell (anosmia) loss
- **Immune:** ↓T-cell function, lymphopenia, recurrent infections (skin, respiratory, GI)
- **Reproductive:** Hypogonadism, ↓spermatogenesis, ↓testosterone, infertility
- **Wound healing:** Delayed; collagen synthesis impairment
- **Eye:** Night blindness (Zn in retinol dehydrogenase), photophobia

### Section 6: Zinc in Wilson's Disease
**Mechanism:** Zn induces **metallothionein (MT)** in enterocyte; MT has high affinity for Cu; Cu bound to MT is shed into faeces with enterocyte desquamation; **↓Cu absorption** (1 mEq/day reduction); mild chelation effect on body Cu.
**Indications:** Maintenance (chelators first line for acute neurological WD), pregnancy, presymptomatic, children; CHELATE trial (2024): Zn non-inferior to penicillamine for neurological WD.
**Dose:** Adults 50 mg PO ×3/day; children (5–15y) 25 mg ×3/day; **empty stomach** (avoid food/iron/calcium reducing absorption).
**Time to effect:** 6–8 weeks; titrate to 24h urinary Cu <100 µg/day, non-ceruloplasmin Cu <25 µg/dL.
**Side effects:** GI (nausea, abdominal pain); monitor LFT, Zn levels; rare Cu deficiency with chronic use.
**Vs Penicillamine/Trientine:** Zn only weakly chelates body Cu; not for active neurological disease; better for maintenance/pregnancy/paediatrics.

### Section 7: Zinc & Other Therapeutic Uses
| Use | Dose | Notes |
|-----|------|-------|
| **Wilson disease** | 50 mg ×3 adult; 25 mg ×3 child | Induces MT; empty stomach |
| **Acrodermatitis enteropathica** | 3 mg/kg/day elemental | Lifelong; titrate to plasma Zn 70–120 µg/dL |
| **Acute diarrhoea (children)** | 10–20 mg/day ×10–14 days | WHO/UNICEF ORS + Zn; ↓duration 25% |
| **Common cold (zinc lozenges)** | ≥75 mg/day (acetate/gluconate) | ↓duration if started <24h; nasal symptoms; ionic form |
| **Age-related macular degeneration (AMD)** | 25–80 mg/day (AREDS2) | Slows progression; Zn + Cu + antioxidants |
| **Sickle cell disease** | 25 mg/day | ↓pain crises, infections |
| **Recurrent infections in elderly** | 15–30 mg/day | ↑Immune function (T-cell) |
| **TPN** | 3 mg/day (acute); 12 mg/day (stable) | Add to TPN; deficiency 1–2 months |
| **Pica (in pregnancy)** | 30–60 mg/day | Reduces pica in Zn-deficient |

### Section 8: Zinc Excess (Toxicity)
**Acute:** Nausea, vomiting, GI distress, headache, lethargy (single dose 1–2 g); Zn from contaminated food/water (galvanised containers, high-zinc denture cream).
**Chronic (industrial, ZnO fumes):** Metal fume fever (acute, self-limiting, 12–48h), copper deficiency, ↓immune function, ↓HDL, sideroblastic anaemia (Cu depletes).
**Drug interactions:** **Penicillamine, EDTA** (chelate Zn); quinolones/tetracyclines ↓absorption (separate by 2h); PPIs ↓Zn absorption (long-term).
**High-dose supplement risks:** >100 mg/day → GI upset, Cu deficiency, immune impairment.

### Section 9: Special Populations & Clinical Scenarios
**Pregnancy/lactation:** ↑Demand; Zn deficiency → pre-eclampsia, IUGR, preterm birth, neurodevelopmental delay; supplementation 15–30 mg/day; IZiNCG recommends 15 mg in pregnancy.
**Elderly:** Poor diet, polypharmacy (PPIs, diuretics), chronic disease → marginal status; supplementation 15–30 mg/day may improve immune function.
**HIV:** Zn deficiency common; supplementation may slow disease progression but excess → viral replication (theoretical).
**Malaria:** Zn supplementation reduces severity in children (P. falciparum); WHO recommends in endemic areas.
**Pneumonia:** Zn deficiency → ↑susceptibility, severity; supplementation reduces incidence in children.

## 6. Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| Acrodermatitis enteropathica = iron deficiency | **AE = ZIP4 Zn transporter mutation**; presents after weaning (breast milk Zn compensates) |
| Zn deficiency = pellagra | **Niacin: photosensitive Casal's necklace; Zn: periorificial, not sun-exposed** |
| Wilson's disease: Zn chelates body Cu | **Zn induces MT → blocks Cu absorption (enterocyte)**; mild chelation only; not acute WD |
| All dermatitis in alcohol = pellagra | **Zn deficiency also presents in alcohol** (↓intake, ↑urinary loss, ↓absorption) |
| Oral Zn supplement with food | **Empty stomach** (avoid food/Ca/Fe/quinolones) — chelation/competition |
| Zinc supplement for everyone with low appetite | **Taste loss (dysgeusia) = classic Zn deficiency**; supplement if confirmed |
| Acute diarrhoea Rx = ORS alone | **WHO: ORS + Zn 10–20 mg/day ×10–14 days** (all children) |
| AMD prevention: vitamins only | **AREDS2: Zn 25 + Cu 2 + vit C/E + lutein/zeaxanthin**; Zn essential |
| Zn supplementation is always safe | **Excess → Cu deficiency, ↓immune, sideroblastic anaemia; UL 40 mg/day** |

## 18. Answer Key with Explanations

### MCQs
1. **C** — RDA zinc 11 mg/day for adult males; 8 mg for females.
2. **B** — Acrodermatitis enteropathica: SLC39A4 (ZIP4) mutation on chromosome 8q24; loss of apical enterocyte Zn importer.
3. **B** — Classic AE triad: dermatitis (perioral, perianal, acral), diarrhoea, alopecia.
4. **C** — AE treatment: Zn 3 mg/kg/day elemental (lifelong); clinical response 5–10 days.
5. **B** — Zn induces metallothionein in enterocyte; MT has high affinity for Cu; Cu-MT shed in faeces with enterocyte desquamation; ↓Cu absorption.
6. **C** — Wilson's disease: Zn 50 mg ×3/day adult; 25 mg ×3 child; empty stomach.
7. **B** — WHO: ORS + Zn 10–20 mg/day ×10–14 days for acute childhood diarrhoea; ↓duration 25%.
8. **D** — Zn deficiency impairs wound healing, T-cell immunity, spermatogenesis, taste/smell; NOT erythrocyte Hb (Fe).
9. **A** — AREDS2: Zn 25 + Cu 2 + Vit C + Vit E + lutein/zeaxanthin; NOT B12.
10. **B** — Chronic Zn excess: Cu deficiency, sideroblastic anaemia, immune dysfunction, GI upset; UL 40 mg.

### SBAs
1. **B** — Classic AE: post-weaning infant, perioral/perianal dermatitis + alopecia + diarrhoea + ↓plasma Zn = ZIP4 mutation; Rx 3 mg/kg/day.
2. **B** — Wilson's + pregnancy: switch to Zn 50 mg ×3 (safe in pregnancy, MT induction); chelators teratogenic; consider continuing Zn until pregnancy end + breastfeeding.
3. **B** — PPIs long-term ↓Zn absorption (↑gastric pH); dysgeusia + nail changes + infections = Zn deficiency; supplement Zn.
4. **B** — WHO: ORS + Zn 10–20 mg/day ×10–14 days for acute childhood diarrhoea; ↓duration 25%, ↓subsequent episodes.
5. **B** — Penicillamine-induced Zn/Cu deficiency with sideroblastic anaemia; switch to Zn + copper repletion.

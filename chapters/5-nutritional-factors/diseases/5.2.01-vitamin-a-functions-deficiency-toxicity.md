## 2. 2. Definitions / Key Concepts

| Term | Definition |
|------|------------|
| **Retinol** | Preformed vitamin A (alcohol); transport form in blood (bound to RBP + transthyretin) |
| **Retinal** | Aldehyde; visual chromophore (11-cis-retinal + opsin = rhodopsin) |
| **Retinoic Acid** | Acid; nuclear receptor ligand (RAR/RXR) → gene transcription (growth, differentiation) |
| **β-Carotene** | Provitamin A carotenoid (2 retinal equivalents); plant source; cleavage by BCO1 |
| **RAE (Retinol Activity Equivalent)** | 1 µg retinol = 1 µg RAE = 12 µg β-carotene = 24 µg α-carotene/β-cryptoxanthin |
| **RBP** | Retinol-binding protein; transports retinol in 1:1 complex with transthyretin |
| **BCO1** | β-carotene 15,15'-oxygenase; cleaves β-carotene → 2 retinal; polymorphic (↓conversion) |
| **Rhodopsin** | Visual pigment (rods); 11-cis-retinal + opsin; light → all-trans-retinal → neural signal |
| **Night Blindness (X1A)** | Inability to see in dim light; earliest sign; impaired rod function |
| **Bitot's Spots (X1B)** | Triangular foamy deposits on bulbar conjunctiva (temporal); keratinised epithelium |
| **Conjunctival Xerosis (X1B)** | Dry, lusterless conjunctiva; loss of goblet cells; keratinisation |
| **Corneal Xerosis (X2)** | Dry, hazy cornea; loss of lustre; precedes ulceration |
| **Keratomalacia (X3A/X3B)** | Corneal softening/liquefaction; X3A <1/3, X3B >1/3; ophthalmic emergency |
| **Follicular Hyperkeratosis** | Phrynoderma; horny plugs in hair follicles (extensor surfaces); vitamin A deficiency |
| **Hypervitaminosis A (Acute)** | Single massive dose (>100x RDA): nausea, vomiting, headache, bulging fontanelle (infants) |
| **Hypervitaminosis A (Chronic)** | Prolonged excess (>10x RDA months): dry skin, alopecia, hepatomegaly, hyperostosis, teratogenic |
| **Pseudotumor Cerebri (PIVS)** | ↑ICP without mass lesion; papilloedema, headache, VI nerve palsy; vit A / tetracycline / withdrawal |
| **Carotenoderma** | Yellow-orange skin (palms/soles, nasolabial folds); NO scleral icterus; β-carotene excess |
| **Isotretinoin (13-cis-retinoic acid)** | Severe acne; teratogenic Category X; requires pregnancy prevention programme (iPledge); side effects: cheilitis, dry skin, ↑TG, ↑LFT, mood changes |
| **WHO X Classification** | X1A night blindness, X1B Bitot's/conjunctival xerosis, X2 corneal xerosis, X3A keratomalacia <1/3, X3B >1/3, XS corneal scar |

## 3. 3. Core Content

### 1. Section 1: Vitamin A Forms, Absorption & Transport
**Forms:** Preformed (retinyl esters: animal liver, fish oil, dairy, eggs) + Provitamin A carotenoids (β-carotene, α-carotene, β-cryptoxanthin: carrots, sweet potato, spinach, mango, papaya).
**Absorption:** Retinyl esters → pancreatic lipase/intestinal esterases → retinol → mixed micelles → enterocyte → re-esterified → chylomicron → lymph. Carotenoids → passive diffusion → enterocyte → BCO1 cleavage → retinal → retinol (less efficient, variable).
**Transport:** Chylomicron remnant uptake by liver → stored as retinyl esters (stellate cells) → mobilised → retinol + RBP + transthyretin → systemic delivery → STRA6 receptor uptake.
**Storage:** Liver (80–90% total body); normal liver reserve = 100–300 µg/g; deficiency when <20 µg/g (≈0.7 µmol/L plasma retinol).
**Excretion:** Bile (retinol glucuronide) → faeces; urine (polar metabolites).

### 2. Section 2: Visual Cycle & Epithelial Function
**Rhodopsin cycle (rods):** Dark: 11-cis-retinal + opsin → rhodopsin (λmax 498 nm). Photon → 11-cis → all-trans retinal → metarhodopsin II → activates transducin (Gt) → activates cGMP phosphodiesterase → cGMP hydrolysed → cGMP-gated Na⁺ channels close → hyperpolarisation → ↓glutamate release → bipolar cell signal.
**Regeneration:** all-trans-retinal → all-trans-retinol (RDH) → transported to RPE → isomerised to 11-cis-retinol (RPE65) → oxidised to 11-cis-retinal (RDH) → back to photoreceptor.
**Epithelial differentiation:** Retinoic acid → binds RAR/RXR nuclear receptors → regulates HOX genes, growth factors → maintains columnar epithelium, goblet cells, mucus secretion. Deficiency → squamous metaplasia (respiratory, GI, GU tracts) → ↑infection risk.

### 3. Section 3: Immune Function & Measles
**Mechanism:** Retinoic acid → ↑T-cell proliferation, Th2 differentiation, IgA secretion, neutrophil function, macrophage phagocytosis, NK activity.
**Measles mortality:** WHO: vitamin A supplementation (200,000 IU day 1 + day 2) ↓measles mortality 50% in children >12m; ↓diarrhoea mortality 33%.
**Infection cycle:** Infection → ↓retinol (acute phase: ↓RBP synthesis, ↓retinol mobilisation) → ↓immune function → worse infection.

### 4. Section 4: Deficiency (VAD) — Clinical Features
| Stage | WHO Code | Features |
|-------|----------|----------|
| Night blindness | X1A | Impaired dark adaptation; history "cannot see at dusk" |
| Conjunctival xerosis | X1B | Dry, dull conjunctiva; loss of goblet cells |
| Bitot's spots | X1B | Foamy, triangular, temporal conjunctiva; keratin debris |
| Corneal xerosis | X2 | Dull, hazy cornea; punctate epithelial erosions |
| Keratomalacia | X3A (<1/3), X3B (>1/3) | Corneal softening, ulceration, perforation risk; EMERGENCY |
| Corneal scar | XS | Bilateral, central; permanent blindness |
| Systemic | — | Follicular hyperkeratosis (phrynoderma: shoulders, thighs, buttocks); growth retardation; ↑infection severity; anaemia; impaired spermatogenesis |

**At risk:** Pregnant/lactating women, preschool children (6–59m), malabsorption (coeliac, CF, biliary atresia), strict vegan, zinc/protein deficiency (↓RBP).

### 5. Section 5: WHO Prevention & Treatment
**Prophylaxis (universal in high-prevalence areas):**
- Infants 6–12m: 100,000 IU (30 mg RE) oral q4–6m
- Children 12–59m: 200,000 IU (60 mg RE) oral q4–6m
- Postpartum women: 200,000 IU single dose within 8 weeks delivery
- Pregnant women: **NOT** high-dose (teratogenic); max 10,000 IU/day or 25,000 IU/week

**Treatment (active xerophthalmia):**
- **Day 1:** 200,000 IU (6–12m: 100,000 IU) oral
- **Day 2:** Repeat same dose
- **Day 14:** Repeat same dose
- Keratomalacia: **Topical antibiotics** (prevent secondary infection) + **eye padding**; corneal transplant if perforated

### 6. Section 6: Hypervitaminosis A (Toxicity)
| Type | Dose | Features | Management |
|------|------|----------|------------|
| **Acute** | >100x RDA single dose (>300,000 IU adult) | Nausea, vomiting, severe headache, drowsiness, bulging fontanelle (infants), diplopia (VI palsy) | Supportive; stop vitamin A; mannitol for ↑ICP |
| **Chronic** | >10x RDA months (>25,000 IU/day) | Dry scaly skin, cheilitis, alopecia, hepatomegaly, splenomegaly, hyperostosis (long bones), arthralgia, ↑ICP (PIVS), teratogenicity | Stop vitamin A; monitor LFT, lipids, bone density; PIVS: acetazolamide, LP |

**Teratogenicity:** Retinoic acid → HOX gene dysregulation → neural crest defects (craniofacial, cardiac, thymic, CNS); **Critical window: weeks 3–8 gestation**. Isotretinoin (13-cis-RA): Category X; strict contraception (iPledge); avoid pregnancy 1 month before/during/1 month after.

### 7. Section 7: β-Carotene & Carotenoids
- **Provitamin A:** β-carotene (2 retinal), α-carotene, β-cryptoxanthin (1 retinal each); lycopene/lutein/zeaxanthin = no vitamin A activity.
- **Absorption:** Fat-dependent (3–5 g fat/meal); food matrix (cooking ↑bioavailability); BCO1 genetic variation (↓conversion 50% in some).
- **Carotenoderma:** Yellow-orange palms/soles, nasolabial folds; **sclerae WHITE**; harmless; reversible on stopping.
- **CARET/ATBC trials:** High-dose β-carotene (20–30 mg/day) + smokers → ↑lung cancer mortality; **avoid in smokers/asbestos workers**.

## 7. 7. Mnemonics

- **Vit A forms:** **R-R-R** = **R**etinol (transport), **R**etinal (vision), **R**etinoic acid (genes)

- **Ocular progression:** **X1A (Night) → X1B (Bitot's/Xerosis) → X2 (Corneal xerosis) → X3 (Keratomalacia) → XS (Scar)**

- **Teratogen:** **HOX GENES** → neural crest defects; Critical window **weeks 3–8**
- **β-carotene:** **CAROTENODERMA** = **C**arotenoderma, **A**sx, **R**eversible, **O**ranges palms, **T** = no toxicity, **E**yes white, **N**o teratogen, **O**K in smokers? **NO** (↑lung cancer CARET/ATBC)
- **RAE:** **RAE** = **R**etinol **A**ctivity **E**quivalent; 1 RAE = 1 µg retinol = 12 µg β-carotene = 24 µg other

### 1. MCQs
1. **B** — 11-cis-retinal is the chromophore; retinol is transport/storage; retinoic acid is nuclear signalling; β-carotene is provitamin.
2. **B** — X1A = night blindness; X1B = Bitot's/conjunctival xerosis; X2 = corneal xerosis; X3 = keratomalacia.
3. **B** — Bitot's spots: foamy, triangular, temporal conjunctival deposits of keratinised epithelium.
4. **C** — WHO: 200k IU day 1, 2, 14 for children >12m; 100k IU for 6–12m.
5. **B** — Retinoic acid → RAR/RXR → HOX gene transcription → craniofacial/cardiac/CNS defects.
6. **B** — Weeks 3–8 = organogenesis, neural crest migration.
7. **B** — Chronic: hepatomegaly, hyperostosis (long bones), PIVS (↑ICP, papilloedema), alopecia, dry skin.
8. **B** — Carotenoderma: yellow palms/soles, sclerae white; harmless, reversible.
9. **A** — 1 µg β-carotene = 1/12 µg RAE (since 12 µg β-carotene = 1 µg RAE).

### 2. SBAs
1. **B** — Category X, absolute contraindication; stop immediately; fetal anomaly scan; counsel risk.
2. **B** — X1A (night blindness) + X1B (Bitot's) = xerophthalmia; WHO treatment: 200k IU day 1, 2, 14.
3. **B** — Chronic toxicity: bone pain (hyperostosis), headache/papilloedema (PIVS), dry skin; 50k IU/day ×2y = chronic.
4. **C** — Pregnancy: max 10k IU/day preformed vit A; high-dose teratogenic; β-carotene preferred source.
5. **C** — CF: fat malabsorption → all fat-soluble vitamins (A, D, E, K) deficient; need water-miscible preparations.

## 19. 19. Summary

**Vitamin A** is a **Must Know 🔴** topic for FCPS/MRCP.
**Key takeaway:** Vitamin A (retinol/retinal/retinoic acid/β-carotene) essential for vision (rhodopsin cycle), epithelial integrity (retinoic acid → RAR/RXR → HOX), immunity. Deficiency = xerophthalmia (X1A night blindness → X3B keratomalacia → XS blindness) + phrynoderma + growth failure + ↑infection mortality. WHO: prophylaxis 100k/200k IU q4–6m; treatment 200k day 1,2,14. Toxicity: chronic (hepatomegaly, hyperostosis, PIVS), teratogenic (isotretinoin Cat X, HOX genes, weeks 3–8). β-carotene: carotenoderma only; smokers ↑lung cancer.
**Exam focus:** Visual cycle, WHO classification/treatment, teratogenicity window, toxicity triad, RAE conversions.
**Clinical relevance:** Measles mortality reduction; VAD prevention in preschool children; isotretinoin counselling; CF fat-soluble vitamin supplementation.

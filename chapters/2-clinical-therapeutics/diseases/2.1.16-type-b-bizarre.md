## 2. 2. 📖 Definition & Core Features

| Feature | Type B (Bizarre) |
|---------|------------------|
| **Synonym** | Idiosyncratic |
| **Dose-dependence** | **No** (may need threshold but not linear) |
| **Predictability** | **Unpredictable** from pharmacology |
| **Incidence** | **Rare** (1:1,000 – 1:100,000) |
| **Mortality** | **High** (can be fatal) |
| **Mechanism** | Immunological (hypersensitivity) or non-immunological idiosyncrasy |
| **Rechallenge** | **Absolutely contraindicated** |
| **Genetic predisposition** | Often (HLA alleles, enzyme polymorphisms) |

## 3. 3. 🧬 Classification by Mechanism

### 1. A. Immunological (Hypersensitivity Reactions — Gell & Coombs)

| Type | Mechanism | Mediators | Onset | Classic Drug Examples | Clinical Syndrome |
|------|-----------|-----------|-------|----------------------|-------------------|
| **Type I** (Immediate) | IgE-mediated mast cell/basophil degranulation | Histamine, tryptase, leukotrienes, PGD₂ | **Minutes–hours** | Penicillins, cephalosporins, NSAIDs, muscle relaxants, insulin, latex | **Anaphylaxis, urticaria, angioedema, bronchospasm, anaphylactoid** |
| **Type II** (Cytotoxic) | IgG/IgM + complement → cell lysis | Complement, ADCC | **Days–weeks** | Penicillins, cephalosporins (haemolytic anaemia); Methyldopa (AIHA); Quinidine (thrombocytopenia); Heparin (HIT) | **Immune haemolytic anaemia, thrombocytopenia, neutropenia** |
| **Type III** (Immune Complex) | IgG/IgM-antigen complexes → vasculitis | Complement, neutrophils | **1–3 weeks** | Penicillins, sulfonamides, allopurinol, biologics (infliximab), vaccines | **Serum sickness, vasculitis, glomerulonephritis, drug fever** |
| **Type IV** (Delayed T-cell) | T-cell mediated (CD4+/CD8+) | Cytokines (IFN-γ, TNF-α, IL-5) | **Days–weeks** | **Allopurinol, sulfonamides, antiepileptics, abacavir, dapsone, NSAIDs** | **Morbilliform, SJS/TEN, DRESS, AGEP, AHS, maculopapular exanthema (MPE)** |

> **Subtypes of Type IV (PHIL):**
> - **IVa** (CD4+ Th1): Maculopapular, DRESS
> - **IVb** (CD4+ Th2): Eosinophilic (DRESS, AHS)
> - **IVc** (CD8+ cytotoxic): **SJS/TEN, AGEP, MPE** — **granulysin, perforin, Fas-FasL**
> - **IVd** (T-cell → neutrophil): **AGEP** — IL-8, CXCL8

### 2. B. Non-Immunological Idiosyncrasy
| Mechanism | Example | Syndrome |
|-----------|---------|----------|
| **Enzyme deficiency (pharmacogenetic)** | G6PD deficiency + oxidative drugs (primaquine, dapsone, sulfonamides) | **Haemolytic anaemia** |
| | CYP2D6 poor metaboliser + codeine | **No analgesia** |
| | CYP2D6 ultra-rapid + codeine | **Toxic morphine levels → apnoea** |
| | NAT2 slow acetylator + isoniazid | **Peripheral neuropathy, hepatotoxicity** |
| | TPMT deficiency + azathioprine/6-MP | **Myelosuppression** |
| **Mitochondrial toxicity** | Valproate, linezolid, NRTIs (zidovudine) | **Hepatic steatosis, lactic acidosis, neuropathy** |
| **Channelopathy** | QT-prolonging drugs in congenital LQTS | **Torsades de Pointes** |
| **Idiosyncratic hepatotoxicity** | Isoniazid, valproate, diclofenac, amoxicillin-clavulanate | **DILI (Hy's Law)** |

## 4. 4. ⚠️ High-Mortality Type B Syndromes (Must Know for Exams)

### 1. 1. Stevens-Johnson Syndrome / Toxic Epidermal Necrolysis (SJS/TEN)
| Feature | SJS | TEN | Overlap (SJS/TEN) |
|---------|-----|-----|-------------------|
| **BSA detachment** | **<10%** | **>30%** | **10–30%** |
| **Mucosal sites** | ≥2 | ≥2 | ≥2 |
| **Typical drugs** | Allopurinol, sulfonamides, AEDs (CZP, PHT, LMG, OXC), NSAIDs, nevirapine | Same | Same |
| **HLA associations** | **B*15:02** (CZP/PHT/OXC), **B*58:01** (Allopurinol), **A*31:01** (CZP DRESS) | | |
| **Mortality** | ~5–10% | ~30–50% | ~15–25% |
| **SCORTEN** | 7 factors: age>40, HR>120, cancer, BSA>10%, BUN>28, Glc>14, HCO₃<20 | | |

**SCORTEN Mortality Prediction:**
| Score | Mortality |
|-------|-----------|
| 0–1 | 3% |
| 2 | 12% |
| 3 | 35% |
| 4 | 58% |
| 5 | 90%+ |

**Management:** Stop culprit **immediately** → ICU/burn unit → supportive (fluids, electrolytes, wound care, infection prevention) → **IVIG (controversial)**, **cyclosporine**, **etanercept** (emerging) → **NO systemic steroids** (controversial, ↑ sepsis risk)

### 2. 2. DRESS (Drug Reaction with Eosinophilia and Systemic Symptoms)
| Feature | Details |
|---------|---------|
| **Triad** | Fever + Rash + Organ involvement |
| **Typical onset** | **2–8 weeks** after drug start (delayed) |
| **Key drugs** | Allopurinol, AEDs (CZP, PHT, PB, LMG, VPA), sulfonamides, dapsone, abacavir, minocycline, allopurinol |
| **HLA** | **B*58:01** (Allopurinol), **B*15:02** (CZP), **A*31:01** (CZP), **B*13:01** (Dapsone) |
| **RegiSCAR criteria** | Fever ≥38°C, lymphadenopathy, eosinophilia >1.5×10⁹/L or atypical lymphs, organ involvement (liver, kidney, lung, heart), rash, reaction >2wks |
| **Mortality** | ~10% (liver failure, myocarditis) |
| **Management** | Stop drug → systemic steroids (pred 1mg/kg) → slow taper 6–12wks → monitor LFTs, renal, cardiac |

---

### 3. 3. AGEP (Acute Generalised Exanthematous Pustulosis)
| Feature | Details |
|---------|---------|
| **Presentation** | **Acute febrile** + **non-follicular pustules** on oedematous erythema |
| **Onset** | **1–5 days** (rapid) |
| **Key drugs** | **Beta-lactams (esp. aminopenicillins)**, quinolones, hydroxychloroquine, diltiazem, sulfonamides |
| **Histology** | Subcorneal pustules, neutrophilic infiltration |
| **Biomarker** | **Neutrophilia** (not eosinophilia) |
| **Course** | Self-limiting **<2 weeks** after withdrawal |
| **Mortality** | Low (<5%) |

---

### 4. 4. AHS (Anticonvulsant Hypersensitivity Syndrome) / DRESS by AEDs
| Feature | Details |
|---------|---------|
| **Drugs** | Phenytoin, carbamazepine, phenobarbital, lamotrigine, valproate |
| **Triad** | Fever, rash, systemic involvement (hepatitis, nephritis, pneumonitis) |
| **Onset** | 1–8 weeks |
| **HLA** | **HLA-B*15:02** (CZP), **HLA-A*31:01** (CZP DRESS), **HLA-B*38:01** (dapsone/CZP) |
| **Cross-reactivity** | **High among aromatic AEDs** (CZP, PHT, PB, OXC) — avoid all if reaction to one |
| **Management** | Stop all aromatic AEDs → switch to non-aromatic (levetiracetam, topiramate, gabapentin, lacosamide, zonisamide) |

---

### 5. 5. Torsades de Pointes (Drug-Induced)
| Feature | Details |
|---------|---------|
| **Mechanism** | **hERG (IKr) blockade** → prolonged repolarisation → afterdepolarisations → R-on-T |
| **Risk factors** | Female, elderly, hypokalaemia, hypomagnesaemia, bradycardia, congenital LQTS, polypharmacy (multiple QT drugs) |
| **High-risk drugs** | **Antiarrhythmics (Class Ia, III)**, **antipsychotics (haloperidol, thioridazine, ziprasidone)**, **antidepressants (citalopram, escitalopram, TCAs)**, **antibiotics (macrolides, fluoroquinolones)**, **antiemetics (ondansetron, domperidone)**, **antifungals (azole)**, **methadone** |
| **Monitoring** | Baseline ECG, repeat if risk factors; **QTc >500ms or Δ>60ms = high risk** |
| **Acute management** | **IV Magnesium sulfate 2g bolus** (even if normal Mg²⁺) → correct K⁺/Mg²⁺ → pacing/isoprenaline if paused → stop offending drug |

---

### 6. 6. Anaphylaxis (Type I)
| Feature | Details |
|---------|---------|
| **Criteria** | Acute onset + skin/mucosal + respiratory compromise **OR** hypotension **OR** GI symptoms |
| **Common drugs** | Penicillins, cephalosporins, NSAIDs, neuromuscular blockers, radiocontrast, insulin, biologics |
| **Management** | **IM Adrenaline 1:1000 (0.5mg = 0.5mL) anterolateral thigh** → supine + legs raised → O₂ → IV fluids → chlorphenamine + hydrocortisone (adjuncts) → observe 6–12h |
| **Tryptase** | Peak 1–2h, half-life 2h — sample at 1–2h and 24h (baseline) |

---

### 7. 7. DILI (Drug-Induced Liver Injury) — Idiosyncratic (Type B)
| Pattern | Drugs | Features |
|---------|-------|----------|
| **Hepatocellular** (ALT>ALP) | Isoniazid, valproate, paracetamol (overdose = Type A), nevirapine, flucloxacillin | **Hy's Law**: ALT>3×ULN + Bilirubin>2×ULN → **10–50% mortality** |
| **Cholestatic** (ALP>ALT) | **Flucloxacillin**, amoxicillin-clavulanate, azathioprine, chlorpromazine, anabolic steroids | Pruritus, jaundice, prolonged |
| **Mixed** | Diclofenac, sulfonamides, allopurinol | Intermediate |
| **HLA** | **DRB1*07:01/DQA1*02:01** (Flucloxacillin), **B*57:01** (Flucloxacillin in some) | |

## 5. 5. 🎯 FCPS/MRCP High-Yield Summary

| Syndrome | Key Drug(s) | Onset | Key Feature | Mortality | Management |
|----------|-------------|-------|-------------|-----------|------------|
| **Anaphylaxis** | Penicillins, NSAIDs, NM blockers | Minutes–hrs | Hypotension, angioedema, wheeze | High if delayed | **IM Adrenaline 0.5mg** |
| **SJS/TEN** | Allopurinol, Sulfa, AEDs, NSAIDs | 1–4 weeks | BSA detachment, mucosal erosions | 5–50% | **Stop drug, ICU, supportive** |
| **DRESS** | Allopurinol, AEDs, Dapsone, Abacavir | 2–8 weeks | Fever, eosinophilia, organ involvement | ~10% | Stop drug, **steroids 1mg/kg, slow taper** |
| **AGEP** | Beta-lactams, quinolones | 1–5 days | Non-follicular pustules, neutrophilia | Low | Stop drug, supportive |
| **Torsades** | QT-prolonging drugs + risk factors | Variable | Polymorphic VT, R-on-T | High | **IV MgSO₄ 2g**, correct K/Mg, pacing |
| **DILI (Hy's Law)** | INH, Valproate, Flucloxacillin, Augmentin | Variable | ALT>3×ULN + Bil>2×ULN | 10–50% | Stop drug, monitor, transplant eval |

## 7. 7. 🤯 Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Type B vs Type C/D/E/F** | Type B = **idiosyncratic** (unpredictable); Type C = **chronic** (time-dependent); D = **delayed** (carcinogenesis); E = **end of use** (withdrawal); F = **failure** (inefficacy) |
| **SJS/TEN vs DRESS vs AGEP** | SJS/TEN: **necrosis/bullae**, mucosal, high mortality; DRESS: **eosinophilia**, organ involvement, delayed onset; AGEP: **pustules**, neutrophilia, rapid onset, benign |
| **AHS vs DRESS** | AHS = **DRESS caused specifically by aromatic AEDs** — same spectrum; cross-reactivity among aromatic AEDs |
| **Torsades = Type A or B?** | **Type B** — idiosyncratic (not all get it at same dose); requires susceptibility (congenital LQTS, electrolytes, polypharmacy) |
| **Anaphylaxis vs Anaphylactoid** | Anaphylaxis = **IgE-mediated (Type I)**; Anaphylactoid = **non-IgE**, direct mast cell activation (e.g., radiocontrast, opioids, vancomycin "red man") — **treatment identical** |

**Mnemonics:**
- **"B = Bizarre = Bad (high mortality) = idiosyncratic"**
- **"SCORTEN"** = **S**kin >10%, **C**ancer, **O**ld >40, **R**enal (BUN>28), **T**achy >120, **E**lectrolytes (HCO₃<20, Glc>14), **N**eoplasia
- **"REGISCAR for DRESS"** = **R**ash, **E**osinophilia, **G**lobular (atypical lymphs), **I**ntermittent fever, **S**ystemic organ, **C**ytology, **A**llopurinol/AEDs, **R**eaction >2wks
- **"Hy's Law"** = **H**epatocellular (ALT>3×) + **Y**ellow (Bil>2×) = **High mortality**
- **"IV Mg for Torsades"** = **M**agnesium **Mg** for **M**echanism (hERG block)

## 11. 11. 📝 Exam Answer Modes

### 1. Long Question (10 marks): *"Discuss the classification, clinical features, and management of severe cutaneous adverse reactions (SCARs)."*
1. **Classification** (2): Type B (idiosyncratic), Type IV T-cell mediated; SCARs = SJS/TEN, DRESS, AGEP, AHS
2. **SJS/TEN** (3): Definition (BSA), SCORTEN, drugs (allopurinol, sulfa, AEDs), HLA (B*15:02, B*58:01), management (stop drug, ICU, supportive, IVIG/cyclosporine/etanercept, NO steroids)
3. **DRESS** (2): RegiSCAR criteria, 2–8wk onset, eosinophilia/organ involvement, drugs, HLA, steroids 1mg/kg taper 6–12wks
4. **AGEP** (2): Pustules, neutrophilia, 1–5d onset, beta-lactams, self-limiting
5. **Cross-reactivity** (1): Aromatic AEDs — avoid all if one causes SCAR

### 2. Short Question (5 marks): *"SJS vs TEN vs DRESS — distinguish."*
| Feature | SJS | TEN | DRESS |
|---------|-----|-----|-------|
| BSA detachment | <10% | >30% | Variable (usually <10%) |
| Mucosal | ≥2 sites | ≥2 sites | Often |
| Onset | 1–4 wks | 1–4 wks | 2–8 wks |
| Key labs | — | — | **Eosinophilia, atypical lymphs** |
| Mortality | 5–10% | 30–50% | ~10% |
| HLA | B*15:02 | B*15:02 | B*58:01, A*31:01 |

### 3. Viva (2 min): *"Patient on allopurinol 2 weeks, fever, rash, eosinophilia 3×10⁹, ALT 200. Diagnosis? Management?"*
- **DRESS** (RegiSCAR criteria met)
- **Stop allopurinol immediately**
- **Prednisolone 1mg/kg**, slow taper 6–12 weeks
- Monitor LFTs, renal, cardiac
- Screen HLA-B*58:01 (retrospective)
- Alternative: Febuxostat (caution CV risk)

### 4. Ward Round (30 sec): *"Patient on clarithromycin + citalopram develops syncope. ECG shows polymorphic VT, twisting points. Immediate management?"*
- **Torsades de Pointes** — **IV Magnesium sulfate 2g bolus** (even if Mg²⁺ normal)
- Stop clarithromycin + citalopram (both QT prolonging)
- Correct K⁺ >4.5, Mg²⁺ >1.0
- Temporary pacing / isoprenaline if recurrent

### 5. Last-Night Revision (1-liners):
- Type B = Bizarre = Idiosyncratic = Unpredictable = High mortality
- SJS <10%, TEN >30%, Overlap 10–30% BSA
- SCORTEN: 7 factors, score 3 = 35% mortality
- DRESS: 2–8 wks, eosinophilia, organ involvement, steroids taper 6–12wks
- AGEP: pustules, neutrophilia, 1–5d, beta-lactams, benign
- Torsades: hERG block → IV Mg 2g
- Hy's Law: ALT>3× + Bil>2× = 10–50% mortality
- HLA: B*15:02 (CZP), B*58:01 (Allo), B*57:01 (ABC), A*31:01 (CZP DRESS), DRB1*07:01 (Fluclox)
- Aromatic AEDs cross-react: CZP, PHT, PB, OXC, LMG → switch to LEV, TPM, GBP, LCM

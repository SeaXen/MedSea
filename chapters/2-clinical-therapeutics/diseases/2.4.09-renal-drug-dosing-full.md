# Renal Drug Dosing in CKD

> [!tip] **FCPS/MRCP Priority: HIGHEST**

## 1. Renal Function Estimation — Cockcroft-Gault vs eGFR

### Cockcroft-Gault (CrCl) — **USE FOR DRUG DOSING**
> **CrCl (mL/min) = (140 - Age) × Weight (kg) × (0.85 if Female) ÷ (72 × SCr mg/dL)**

| Parameter | Detail |
|-----------|--------|
| **Age** | Years |
| **Weight** | **Actual Body Weight (ABW)** — Unless obese (BMI>30) → Use **Adjusted Body Weight (AdjBW)** = IBW + 0.4×(ABW-IBW) |
| **Sex** | ×0.85 for Female |
| **SCr** | mg/dL (Use **stable** baseline; If SCr <0.8, use 0.8 to avoid overestimation) |

### eGFR (CKD-EPI 2021) — **USE FOR CKD STAGING**
| Stage | eGFR (mL/min/1.73m²) | Description |
|-------|----------------------|-------------|
| **G1** | ≥90 | Normal/High |
| **G2** | 60-89 | Mildly decreased |
| **G3a** | 45-59 | Mild-moderate |
| **G3b** | 30-44 | Moderate-severe |
| **G4** | 15-29 | Severe |
| **G5** | <15 | Kidney failure |

### When to Use Which?
| Situation | Use |
|-----------|-----|
| **Drug Dosing** (BNF, FDA, EMA labels) | **Cockcroft-Gault CrCl** |
| **CKD Diagnosis/Staging** (KDIGO) | **eGFR (CKD-EPI)** |
| **Drug Monographs (BNF)** | Often list **both** — Follow the label |
| **Aminoglycosides, Vancomycin, DOACs** | **CrCl (Cockcroft-Gault)** |

## 2. Dose Adjustment Principles

| Strategy | When | Examples |
|----------|------|----------|
| **Reduce Dose** (Same frequency) | Concentration-dependent drugs; Narrow TI; Linear PK | Digoxin, Lithium, Warfarin (monitor INR), Phenytoin |
| **Extend Interval** (Same dose) | Time-dependent killing; Concentration-dependent with post-antibiotic effect | **Aminoglycosides (Extended-interval)**, **Vancomycin (Intermittent)**, Beta-lactams (Prolonged infusion) |
| **Reduce Dose + Extend Interval** | Severe impairment; Dual clearance pathways | Some antivirals, Chemotherapy |
| **Avoid / Contraindicated** | Metabolites accumulate → Toxicity; No safe dose established | **Metformin (eGFR<30)**, Gadolinium (eGFR<30), Nitrofurantoin (eGFR<30), Fondaparinux (CrCl<20) |

### General Rules by CrCl
| CrCl (mL/min) | General Approach |
|---------------|------------------|
| **≥60** | Normal dose (Check drug-specific) |
| **30-59** | Reduce dose **OR** Extend interval (Drug-specific) |
| **15-29** | Significant dose reduction; Extend interval; **Avoid nephrotoxins** |
| **<15 / Dialysis** | **Post-dialysis dosing** for renally cleared drugs; Avoid drugs dialysed out |

## 3. Common Drug Adjustments — Quick Reference

### High-Yield "Must Know" Drugs for Exams

| Drug | CrCl 30-59 | CrCl 15-29 | CrCl <15 / Dialysis | Key Points |
|------|------------|------------|---------------------|------------|
| **Metformin** | Max **1g BD** (Caution if <45) | **CONTRAINDICATED** (Lactic acidosis) | Contraindicated | Stop if eGFR<30; Restart if eGFR≥30 & stable |
| **ACEi / ARB** | Standard dose; Monitor K, Cr | Reduce dose; Monitor closely | Start low; Monitor; Hold if K>5.5 | **Do not withhold for CrCl>30** unless hyperkalaemia/AKI |
| **DOACs** | See table below | See table below | **Avoid DOACs** → Warfarin | **CrCl-based** (Not eGFR) |
| **Aminoglycosides** | Extended interval; **Monitor levels** | Extended interval; **Monitor levels** | **Post-HD dosing** | Hartford nomogram; Once-daily preferred |
| **Vancomycin** | Standard; **AUC-guided** | Standard; **AUC-guided** | **Post-HD 15-20mg/kg** | AUC 400-600; Trough 10-20 (historical) |
| **Gabapentin / Pregabalin** | **Reduce dose** (Gabapentin 300mg TDS→BD; Pregabalin 150mg BD→OD) | **Significant reduction** (Gabapentin 300mg OD; Pregabalin 75mg OD) | **Dose post-HD** | Gabapentin: CrCl-based; Pregabalin: CrCl-based |
| **Allopurinol** | Reduce (Max 200mg/day if <30) | **100mg/day max** | 100mg **post-HD** | Febuxostat alternative (Hepatic metabolism) |
| **Colchicine** | Reduce (0.5mg OD) | 0.5mg alternate days | Avoid / 0.5mg post-HD | Neuromyotoxicity risk with statins |
| **Digoxin** | Standard; **Monitor level** | Reduce dose; Monitor level | Standard; Monitor level | Narrow TI; Hypokalaemia ↑ toxicity |
| **Lithium** | Monitor level; Reduce if needed | Reduce dose; **Level q1-2wk** | Reduce dose; Level q1wk | **Contraindicated if severe CKD** (Risk toxicity) |
| **NSAIDs** | **Avoid** (AKI risk) | **Contraindicated** | Contraindicated | Use Paracetamol, Topical NSAID |
| **Nitrofurantoin** | Avoid if CrCl<45 (↓ Efficacy + Toxicity) | **Contraindicated** | Contraindicated | Use alternative ABX for UTI |
| **Fondaparinux** | **Reduce** (1.5mg if 20-50) | **Contraindicated** (<20) | Contraindicated | LMWH preferred in CKD |
| **LMWH (Enoxaparin)** | Standard; Anti-Xa if <30 | **Anti-Xa monitoring** (Target 0.5-1.0) | **Anti-Xa monitoring**; Consider UFH | Accumulates in CKD |
| **Piperacillin-Tazobactam** | Standard | Extend interval (4.5g q8h→q12h) | **Post-HD** | Prolonged infusion (4h) in severe infections |
| **Meropenem** | Standard | Extend interval (1g q8h→q12h) | **Post-HD** | Prolonged infusion (3h) |
| **Acyclovir** | Reduce (IV: 5mg/kg q12h→q24h if <25) | Reduce (IV: 5mg/kg q24h) | **Post-HD** | Neurotoxicity if accumulated |

### DOAC Renal Dosing (CrCl-Based — **Critical for Exams**)

| DOAC | Indication | CrCl ≥50 | CrCl 30-49 | CrCl 15-29 | CrCl <15 / Dialysis |
|------|------------|----------|------------|------------|---------------------|
| **Dabigatran** | AF / VTE | 150mg BD / 110mg BD | **110mg BD** (if >80y/Verapamil) | **CONTRAINDICATED** | Contraindicated |
| **Rivaroxaban** | AF | 20mg OD | **15mg OD** | **15mg OD** (Caution) | **Avoid** → Warfarin |
| | VTE Treatment | 15mg BD ×21d → 20mg OD | **15mg OD** | **10mg OD** | Avoid |
| **Apixaban** | AF | 5mg BD | **5mg BD** | **2.5mg BD** (if ≥2: Age≥80, Wt≤60, Cr≥133) | **2.5mg BD** (Off-label) |
| | VTE | 10mg BD ×7d → 5mg BD | 5mg BD | **2.5mg BD** | Avoid |
| **Edoxaban** | AF / VTE | 60mg OD | **30mg OD** | **30mg OD** | Avoid |

**Pearls:**
- **CrCl, NOT eGFR** for DOAC dosing
- **Dabigatran:** Only DOAC with **dialysis clearance** (but still contraindicated <30)
- **Apixaban:** Least renal clearance (27%) → **Most flexible in CKD**
- **Rivaroxaban:** Once daily → **CrCl 15-29 = 15mg OD (AF)** — Controversial, many avoid
- **Anti-Xa monitoring** NOT routine for DOACs

## 4. Dialysis Dosing Principles

### Haemodialysis (HD) — Thrice Weekly
| Drug Property | Dosing Approach |
|---------------|-----------------|
| **Highly dialysable** (Low MW, Low protein binding, Low Vd) | **Post-HD supplemental dose** (e.g., Vancomycin, Aminoglycosides, Gabapentin, Acyclovir, Pyridoxine) |
| **Poorly dialysable** (High MW, High protein binding, High Vd) | **Standard dosing** (e.g., Warfarin, Phenytoin, Valproate, Tacrolimus, Ciclosporin, DOACs) |
| **Intermediate** | **Post-HD dose adjustment** based on PK studies |

### Peritoneal Dialysis (PD) — Continuous
- Continuous clearance → **Less dramatic fluctuations**
- **Dose as for CrCl 10-15 mL/min** (Continuous low clearance)
- IP administration possible for some (Vancomycin, Aminoglycosides, Heparin)

### CRRT (CVVH/CVVHDF) — Continuous
- **High clearance** for small molecules → **Higher doses often needed**
- **Drug-specific CRRT dosing guides** (e.g., Vancomycin 15-20mg/kg q24-48h, Meropenem 1g q8h prolonged infusion)
- **Effluent flow rate** determines clearance (Typically 20-25 mL/kg/h)

### HD Dosing — Common Post-HD Supplemental Doses
| Drug | Post-HD Dose |
|------|--------------|
| **Vancomycin** | 15-20 mg/kg (after HD session) |
| **Gentamicin / Amikacin** | 1-2 mg/kg (after HD) / Re-dose per levels |
| **Gabapentin** | 200-300 mg (after HD) |
| **Pregabalin** | 75-150 mg (after HD) |
| **Acyclovir (IV)** | 5 mg/kg (after HD) |
| **Pyridoxine (B6)** | 50 mg (after HD) — With Isoniazid |
| **Folic Acid** | 5 mg (after HD) |
| **Water-soluble vitamins** | Post-HD replacement |

## 5. Drug-Specific Algorithms (Viva Favourites)

### Metformin in CKD
```mermaid
flowchart TD
    A[Patient on Metformin] --> B{eGFR}
    B -->|≥45| C[Full dose OK (Max 2g/day)]
    B -->|30-44| D[Max 1g BD; **Caution**; Review 3-6 monthly]
    B -->|<30| E[**STOP Metformin** — Lactic acidosis risk]
    E --> F[eGFR improves ≥30?]
    F -->|Yes| G[Restart 500mg OD, titrate]
    F -->|No| H[Alternative: SU, DPP4, GLP1, SGLT2 (if eGFR≥20), Insulin]
```

### ACEi/ARB in CKD — "Start Low, Go Slow, Monitor"
```mermaid
flowchart TD
    A[Start ACEi/ARB in CKD] --> B{Baseline K, Cr}
    B -->|K<5.0, Cr stable| C[Start **Low Dose** (Ramipril 1.25mg, Losartan 25mg)]
    C --> D[**Recheck U&Es at 1-2 weeks**]
    D -->|Cr ↑ <30% & K ≤5.5| E[Continue; Titrate to target dose]
    D -->|Cr ↑ ≥30% or K >5.5| F[**Hold ACEi/ARB**; Investigate (Renal artery stenosis? Diuretic? NSAID?)]
    D -->|K 5.0-5.5| G[Reduce K+ intake; Review K-sparing diuretics; Recheck 1wk]
```

### Aminoglycoside Extended-Interval (Hartford Nomogram)
```mermaid
flowchart TD
    A[Gentamicin/Amikacin Indicated] --> B[Calculate CrCl (Cockcroft-Gault)]
    B --> C[Dose: **Gentamicin 5-7mg/kg** / **Amikacin 15-20mg/kg** IV]
    C --> D[Infuse over 30-60 min]
    D --> E[**Check level 6-14h post-dose** (Pre-2nd dose if q24h)]
    E --> F{Plot on Hartford Nomogram}
    F -->|Interval A| G[q24h]
    F -->|Interval B| H[q36h]
    F -->|Interval C| I[q48h]
    F -->|Interval D| J[**Traditional dosing** (q8h) + Levels]
    G --> K[Monitor: Pre-dose level <1, Renal fn q2-3d]
```

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **eGFR vs CrCl for DOACs** | **DOAC dosing uses CrCl (Cockcroft-Gault)** — eGFR overestimates in elderly/frail |
| **Metformin cutoff** | **eGFR <30 = Contraindicated**; 30-44 = Max 1g BD with caution; **Not CrCl** |
| **ACEi/ARB in CKD** | **Do NOT withhold for CrCl>30** unless K>5.5 or Cr↑>30% — Renoprotective benefit outweighs |
| **Aminoglycoside monitoring** | **Extended-interval:** Pre-dose (trough) only, Hartford nomogram; **Traditional:** Peak + Trough |
| **Vancomycin trough vs AUC** | **AUC-guided (400-600) is standard**; Trough-only deprecated (Poor correlation with AUC) |
| **Gabapentin vs Pregabalin renal adj** | Both need adjustment; **Gabapentin more granular (CrCl-based)**; Pregabalin simpler (CrCl-based bands) |
| **LMWH in CKD** | **Anti-Xa monitoring if CrCl<30**; Enoxaparin accumulates; **UFH alternative** if uncontrolled |
| **Contrast + Metformin** | **Hold Metformin 48h post-contrast** if eGFR<60 (Risk lactic acidosis if AKI develops) |

**Mnemonic: RENAL-DRUGS**
- **R**efrain NSAIDs, Nitrofurantoin, Fondaparinux (<20-30)
- **E**xclude Metformin (<30)
- **N**arrow TI drugs: Digoxin, Lithium, Phenytoin, Warfarin — Levels
- **A**minoglycosides/Vancomycin — Extended interval, Levels, Post-HD
- **L**ow dose ACEi/ARB — Monitor K, Cr 1-2wk
- **D**OACs — CrCl-based table (Apixaban most flexible)
- **R**educe Gabapentin/Pregabalin/Allopurinol/Colchicine
- **U**niversal CrCl formula (Cockcroft-Gault) for dosing
- **G**FR (eGFR) for staging, NOT dosing
- **S**upplement water-soluble vitamins Post-HD

## 12. Exam Answer Modes

### Long Answer Skeleton
1. Calculate CrCl (Cockcroft-Gault) with actual/adjusted weight
2. Identify CKD stage (eGFR)
3. For each drug: State adjustment per CrCl stage
4. Specify monitoring parameters & frequency
5. Address dialysis dosing if applicable

## Summary
Renal drug dosing uses **Cockcroft-Gault CrCl** (not eGFR). **Metformin contraindicated <30**. **DOACs** require CrCl-based dose reduction (Apixaban most flexible). **ACEi/ARB** are renoprotective — continue unless K>5.5 or Cr↑>30%. **Aminoglycosides** use extended-interval Hartford nomogram. **Vancomycin** targets AUC 400-600 (Bayesian). **Gabapentin/Pregabalin/Allopurinol/Colchicine** need CrCl-based reduction. **Dialysis** requires post-HD supplemental dosing for renally cleared drugs. **NSAIDs, Nitrofurantoin, Fondaparinux** contraindicated in advanced CKD.

## Answer Key with Explanations

### MCQs
1. **B** — Cockcroft-Gault CrCl is standard for drug dosing labels (BNF, FDA, EMA)
2. **B** — Metformin contraindicated at eGFR <30 (Lactic acidosis risk); 30-44 = Max 1g BD
3. **B** — Apixaban 2.5mg BD for AF if CrCl 15-29 AND ≥2 of (Age≥80, Wt≤60kg, Cr≥133) — typically met in this range
4. **B** — Extended-interval aminoglycosides: **Pre-dose (trough) level only**, plotted on Hartford nomogram
4. **B** — Vancomycin AUC target for MRSA = **400-600 mg·h/L** (AUC/MIC >400)
5. **C** — Hold ACEi/ARB if **K>5.5** or **Cr rise >30%** from baseline
6. **B** — Allopurinol max **100mg/day** if CrCl <30 (Oxypurinol accumulation)
7. **B** — Vancomycin is highly dialysable → **Post-HD supplemental dose (15-20mg/kg)**
8. **B** — NSAIDs **Contraindicated** in CKD G4/G5 (AKI risk, Na retention, HF exacerbation)
9. **B** — LMWH accumulates in CKD → **Anti-Xa monitoring** (Target 0.5-1.0 IU/mL) if CrCl<30
10. **B** — Rivaroxaban AF: **15mg OD for CrCl 15-49**; 20mg OD for ≥50

### SBAs
1. **A** — CrCl = (140-75)×80×1 / (72×1.8) = 65×80/129.6 = 40.1 → ~40
2. **B** — eGFR 38 = G3b; Metformin max 1g BD is correct; Linagliptin = No renal adjustment (DPP4)
3. **C** — CrCl 28: Dabigatran contraindicated <30; Rivaroxaban 15mg only for CrCl 15-49 (AF) but Apixaban preferred; Apixaban 2.5mg BD for CrCl 15-29
4. **B** — Hartford: Level at 10h ~2 mg/L → Interval B (q36h) typically
5. **B** — AUC >700 → Nephrotoxicity risk (Also trough >20 correlates)
6. **B** — CrCl ~20: Gabapentin 300mg OD (or 200-300mg OD); Pregabalin 75mg OD

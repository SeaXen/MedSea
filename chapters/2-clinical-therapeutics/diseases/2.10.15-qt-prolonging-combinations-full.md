## 💊 Major QT-Prolonging Drug Classes

| Class | High-Risk Drugs (Strongest Evidence) | Mechanism |
|-------|--------------------------------------|-----------|
| **Class Ia Antiarrhythmics** | Quinidine, Procainamide, Disopyramide | hERG (IKr) blockade |
| **Class III Antiarrhythmics** | **Amiodarone, Dronedarone, Sotalol, Ibutilide, Dofetilide** | hERG blockade |
| **Antipsychotics (Typical)** | **Haloperidol, Chlorpromazine, Thioridazine, Pimozide, Droperidol, Fluphenazine** | hERG blockade |
| **Antipsychotics (Atypical)** | **Quetiapine, Ziprasidone, Risperidone (high dose), Olanzapine, Clozapine, Amisulpride, Sertindole** | hERG blockade |
| **Antidepressants** | **Citalopram (>40mg), Escitalopram (>20mg), TCAs (Amitriptyline, Nortriptyline, Imipramine, Clomipramine), Venlafaxine (high dose), Mirtazapine** | hERG blockade |
| **Macrolide Antibiotics** | **Erythromycin, Clarithromycin, Azithromycin (lower risk)** | hERG blockade + CYP3A4 inhibition |
| **Fluoroquinolones** | **Moxifloxacin, Levofloxacin, Ciprofloxacin, Gemifloxacin** | hERG blockade |
| **Antifungals (Azoles)** | **Ketoconazole, Itraconazole, Voriconazole, Posaconazole, Fluconazole (high dose)** | hERG blockade + CYP3A4 inhibition |
| **Antiemetics** | **Ondansetron, Granisetron, Dolasetron, Domperidone, Droperidol** | hERG blockade |
| **Opioids** | **Methadone, Levomethadyl, Tramadol (high dose)** | hERG blockade |
| **Others** | **Chloroquine/Hydroxychloroquine, Arsenic trioxide, Tizanidine, Fingolimod, Vandetanib, Lapatinib** | hERG blockade |

## ⚡ Risk Factors for Torsades (TDR Score)

| Risk Factor | Points |
|-------------|--------|
| **Female sex** | 1 |
| **Age ≥65** | 1 |
| **Hypokalaemia (K⁺ <3.5)** | 1 |
| **Hypomagnesaemia (Mg²⁺ <0.7)** | 1 |
| **Bradycardia (HR <60)** | 1 |
| **Congenital LQTS / FHx SCD** | 2 |
| **Heart failure / LVH** | 1 |
| **Liver disease (Child-Pugh B/C)** | 1 |
| **≥2 QT-prolonging drugs** | 1 per additional drug |
| **Baseline QTc >450ms (M) / >470ms (F)** | 1 |

**Score ≥3 = High Risk** (intensive monitoring)

## 🚨 High-Risk Combinations (Additive hERG Blockade)

| Combination | Clinical Context | Monitoring |
|-------------|------------------|------------|
| **Amiodarone + Beta-blocker/CCB** | AF rate control | Baseline ECG, q1–3mo |
| **Antiarrhythmic + Antipsychotic** | ICD patient with psychosis | Baseline ECG, avoid if possible |
| **Macrolide + Antipsychotic** | Pneumonia in schizophrenia | Baseline ECG, switch macrolide |
| **Fluoroquinolone + Antidepressant (Citalopram)** | UTI in depression | Baseline ECG, avoid moxifloxacin |
| **Azole + Opioid (Methadone)** | Fungal infection in OAT | Baseline ECG, avoid if possible |
| **TCAs + Antipsychotic** | Refractory depression/psychosis | Baseline ECG, limit doses |
| **Domperidone + PPI + Macrolide** | Gastroparesis | Avoid domperidone >30mg/day |
| **Sotalol + Diuretic** | HF with AF | K⁺/Mg²⁺ replacement, ECG q1–3mo |

## 📊 ECG Monitoring Protocol

| Situation | Baseline ECG | Follow-up |
|-----------|--------------|-----------|
| **Starting 1st QT drug** (low risk pt) | Optional | If symptoms |
| **Starting 1st QT drug** (high risk pt: female, elderly, K⁺↓, Mg²⁺↓, HF, LQTS) | **Mandatory** | **48–96h, then q1–3mo** |
| **Adding 2nd QT drug** | **Mandatory** | **48–96h, then q1–3mo** |
| **Dose increase of QT drug** | **Mandatory** | **48–96h** |
| **Electrolyte abnormality (K⁺<3.5, Mg²⁺<0.7)** | **Mandatory** | **Daily until corrected** |
| **New bradycardia / AV block** | **Mandatory** | **48–96h** |

## ⚡ Thresholds for Action

| QTc (Bazett) | ΔQTc from Baseline | Action |
|--------------|-------------------|--------|
| **<450ms (M) / <470ms (F)** | Any | Continue, routine monitoring |
| **450–500ms (M) / 470–500ms (F)** | **>60ms** | ↓ dose, correct electrolytes, consider alternative |
| **>500ms** | Any | **STOP/QTc drug if possible**, correct K⁺/Mg²⁺, cardiology review |
| **>500ms + symptoms** (syncope, palpitations) | N/A | **EMERGENCY** — IV MgSO₄ 2g, pacing, ICU |

## 🎯 FCPS/MRCP High-Yield Scenarios

| Scenario | Risk Assessment | Action |
|----------|----------------|--------|
| 70F on citalopram 40mg, started clarithromycin for pneumonia | **High** (female, elderly, 2 QT drugs, CYP3A4i ↑ citalopram) | **Switch clarithro → doxycycline**; ECG baseline + 48h; correct K⁺/Mg²⁺ |
| 55M on amiodarone, started haloperidol for delirium | **High** (amiodarone + haloperidol = additive hERG) | **Avoid haloperidol**; use quetiapine low dose (caution) or risperidone; ECG baseline |
| 40F on methadone 80mg, needs fluconazole for candidiasis | **High** (methadone + azole = CYP3A4i + hERG) | **Avoid fluconazole**; use echinocandin; if fluconazole essential → ECG, ↓ methadone, TDM |
| 80M on furosemide 80mg, started sotalol for AF | **High** (hypokalaemia risk + sotalol) | **K⁺/Mg²⁺ repletion**; target K⁺ >4.5; ECG baseline + 48h; monitor renal |
| 30M on citalopram 20mg, presents with syncope, ECG QTc 520ms | **EMERGENCY** | **IV MgSO₄ 2g**, stop citalopram, correct electrolytes, cardiology, ICU |

## 🤯 Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **QT vs QTc** | QT = raw interval; QTc = heart-rate corrected (Bazett: QTc = QT/√RR). **Use QTc.** |
| **Bazett overcorrects at high HR** | At HR >100, Bazett overestimates QTc; Fridericia (QT/RR¹ᐟ³) preferred but Bazett standard |
| **All antipsychotics equal QT risk?** | **Typical (haloperidol, thioridazine, pimozide) > Atypical (ziprasidone, quetiapine > risperidone, olanzapine)** |
| **Azithromycin = safe QT?** | **Lower risk** than clarithro/erythro, but not zero — still caution in high-risk pts |
| **MgSO₄ for Torsades — even if Mg²⁺ normal?** | **YES** — MgSO₄ stabilises membrane, suppresses EADs regardless of serum level |

**Mnemonics:**
- **"QT DRUGS"** = **Q**uinidine, **T**ricyclics, **D**ronedarone, **R**isperidone, **U**ndesirable **G**roups: **A**ntiarrhythmics (Ia/III), **A**ntipsychotics, **M**acrolides, **F**luoroquinolones, **A**zoles, **A**ntiemetics, **M**ethadone
- **"RISK FACTORS"** = **R**hythm (brady), **I**ons (K⁺↓, Mg²⁺↓), **S**ex (female), **K**idney/Liver, **F**Hx LQTS, **A**ge>65, **C**ombination drugs, **T**orsades hx, **O**rders (QTc>500), **R**ate (brady)
- **"MAGNESIUM FIRST"** = **MgSO₄ 2g IV** for **ALL** Torsades (even normal Mg²⁺)
- **"MAX DOSES"** = Citalopram 40mg (20mg if risk), Domperidone 30mg/day, Escitalopram 20mg

## 📝 Exam Answer Modes

### Short Question (5 marks): *"Drug-induced QT prolongation — mechanism, risk factors, monitoring"*
- Mechanism: hERG (IKr) blockade → delayed repolarisation → EADs → R-on-T → Torsades
- Risk factors: Female, elderly, hypoK/hypoMg, bradycardia, LQTS, HF, multiple QT drugs
- Monitoring: Baseline ECG if high risk/≥2 drugs; 48–96h follow-up; QTc>500 or Δ>60 = stop/alternative

### Viva (2 min): *"70F on citalopram 40mg, clarithromycin added for pneumonia. ECG QTc 480ms. Management?"*
- **High risk**: female, elderly, 2 QT drugs, clarithro inhibits CYP3A4 → ↑ citalopram
- **Stop clarithromycin** → switch to doxycycline/amoxicillin
- **Reduce citalopram to 20mg** (max in >65)
- **Correct K⁺ >4.5, Mg²⁺ >1.0**
- **Repeat ECG 48h**
- If QTc >500ms → IV MgSO₄, cardiology

### Ward Round (30 sec): *"Patient on sotalol for AF, started furosemide 40mg for ankle oedema. 3 days later: K⁺ 3.0. Plan?"*
- **High Torsades risk**: sotalol + hypokalaemia
- **IV KCl replacement**, target K⁺ >4.5
- **MgSO₄ if Mg²⁺ low**
- **ECG now** — if QTc >500ms → hold sotalol, IV MgSO₄
- Consider switching sotalol → amiodarone/ablation

### Last-Night Revision (1-liners):
- hERG (IKr) blockade = QT prolongation mechanism
- QTc >500ms or Δ>60ms = high risk
- MgSO₄ 2g IV for ALL Torsades (even normal Mg²⁺)
- Classes: Antiarrhythmics, Antipsychotics, TCAs/SSRIs, Macrolides, Fluoroquinolones, Azoles, Antiemetics, Methadone
- Moxifloxacin highest FQ risk; Azithromycin lowest macrolide risk
- Citalopram max 40mg (20mg if risk); Domperidone max 30mg/day
- Risk: Female, Elderly, HypoK/Mg, Brady, LQTS, HF, Multiple drugs

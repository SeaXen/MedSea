## 2. 2. 📊 Target Ranges & Pharmacokinetics

| Drug | **Target Trough (ng/mL)** | **AUC Target** | **Key PK** |
|------|---------------------------|----------------|------------|
| **Tacrolimus (Tac)** | **Kidney: 5–15** (early 10–15, late 5–10)
**Liver: 5–10**
**Heart/Lung: 10–15** | **AUC 200–300 ng·h/mL** (limited use) | **CYP3A4/5 substrate**
**P-gp substrate**
**High Vd, low bioavailability (20–25%)**
**t½ ~12h** |
| **Ciclosporin (CsA)** | **Kidney: 100–300** (early 200–300, late 100–200)
**Liver: 100–200**
**C2 monitoring (2h post): 800–1500** | **AUC 3000–5000 ng·h/mL** | **CYP3A4/5 substrate**
**P-gp substrate**
**Low bioavailability (20–50%), variable**
**t½ ~6–12h** |
| **Sirolimus (SRL)** | **4–12** (de novo kidney: 8–12; with CsA: 4–8) | **AUC 100–200 ng·h/mL** | **CYP3A4/5 substrate**
**P-gp substrate**
**High Vd, low bioavailability (14%)**
**t½ ~60h** |
| **Everolimus (EVR)** | **3–8** (with CsA: 3–5; with Tac: 6–10) | **AUC 30–80 ng·h/mL** | **CYP3A4/5 substrate**
**P-gp substrate**
**t½ ~30h** |

**No CYP metabolism**

## 3. 3. 🔄 Major CYP3A4/P-gp Interactions — **CRITICAL**

### 1. Strong Inhibitors (↑ Levels 2–10x) — **REDUCE DOSE**
| Inhibitor | **Tac / CsA / SRL / EVR Dose Adjustment** | Notes |
|-----------|------------------------------------------|-------|
| **Ketoconazole** | **↓ 66–80%** (Tac: divide by 3–5) | **Potent CYP3A4 + P-gp inhibitor** |
| **Itraconazole** | **↓ 50–66%** | |
| **Posaconazole** | **↓ 50–66%** | |
| **Voriconazole** | **↓ 66–90%** (Tac: **reduce to 1/10–1/3 dose**) | **Most potent** — monitor levels **daily** |
| **Fluconazole** | **↓ 50%** | |
| **Clarithromycin / Erythromycin** | **↓ 50–66%** | **Azithromycin safer** (no CYP3A4 inhibition) |
| **Verapamil / Diltiazem** | **↓ 30–50%** | |
| **Dronedarone** | **↓ 50%** | |
| **Amiodarone** | **↓ 30–50%** | |
| **Grapefruit Juice** | **↓ 30–50%** | Intestinal CYP3A4 + P-gp inhibition |
| **Protease Inhibitors** (Ritonavir, etc.) | **↓ 80–90%** | Avoid if possible |

### 2. Strong Inducers (↓ Levels 50–90%) — **INCREASE DOSE / AVOID**
| Inducer | **Effect** | Management |
|---------|------------|------------|
| **Rifampicin** | **↓ 80–90%** (near complete loss) | **AVOID**; if essential: ↑ dose 3–5x, monitor levels **daily** |
| **Rifabutin** | **↓ 50–80%** | ↑ Dose, monitor |
| **Carbamazepine / Phenytoin / Phenobarbital** | **↓ 50–80%** | ↑ Dose, monitor |
| **St John's Wort** | **↓ 50–80%** | **AVOID** |
| **Dexamethasone (high dose)** | **↓ 30–50%** | Monitor |

## 4. 4. ☠️ Toxicity Profiles

| Drug | **Nephrotoxicity** | **Neurotoxicity** | **Metabolic / Other** | **Infection / Malignancy** |
|------|-------------------|-------------------|----------------------|----------------------------|
| **Tacrolimus** | **Afferent arteriolar vasoconstriction** → ↑ Cr, ↓ GFR (dose-dep, reversible) | **Tremor, Headache, Insomnia, Seizures, PRES** (high levels) | **New-onset diabetes (PTDM) — 15–20%**, Hyperkalaemia, Hypomagnesaemia, Hyperlipidaemia | ↑ CMV, BK virus, **PTLD (EBV)**, Skin cancer |
| **Ciclosporin** | **Afferent vasoconstriction** → ↑ Cr (dose-dep); **Thrombotic microangiopathy** (TMA) | **Tremor, Hirsutism, Gingival hyperplasia, Seizures, PRES** | **Hyperlipidaemia**, Hyperuricaemia/gout, Hypertension, Hyperkalaemia | ↑ CMV, BK, PTLD, Skin cancer |
| **Sirolimus / Everolimus** | **Proteinuria** (podocyte injury), ↓ Wound healing, Lymphocele | **Oral ulcers**, Dysgeusia | **Hyperlipidaemia (↑ TG/Chol)**, **Hyperglycaemia**, Thrombocytopenia, Anaemia, Impaired wound healing, Pneumonitis (rare) | ↑ Infections (atypical), PTLD risk lower than CNI |
| **Mycophenolate** | **Rare** | Rare (dizziness) | **GI (diarrhoea, N/V) — 30–40%**, **Bone marrow suppression (leukopenia, anaemia)** | ↑ CMV, HSV, VZV; **Teratogenic** |

## 7. 7. 🤯 Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Tacrolimus trough 5–15 — what does it mean?** | Early post-Tx (0–3mo) = **10–15**; Maintenance (>3mo) = **5–10** |
| **C2 monitoring** | **2h post-dose CsA level** — correlates better with AUC; used in some centres |
| **Voriconazole interaction** | **Most potent** — Tac ↓ to **1/10–1/3 dose (90% reduction)** |
| **Azithromycin vs Clarithromycin** | **Azithromycin = NO CYP3A4 inhibition** — safe with CNI |
| **Tacrolimus vs CsA diabetes** | **Tacrolimus >> CsA** for new-onset diabetes (PTDM) |
| **Sirolimus proteinuria** | **Podocyte injury** — if >1g/day, consider switch to CNI |
| **Mycophenolate + Rifampicin** | **Rifampicin ↓ MPA AUC 30–50%** (disrupts enterohepatic recycling) |

**Mnemonics:**
- **"TAC RANGE"** = **Kidney: 10-15 early, 5-10 late**; **Liver: 5-10**; **Heart/Lung: 10-15**
- **"CSA RANGE"** = **Kidney: 200-300 early, 100-200 late**; **C2: 800-1500**
- **"SRL RANGE"** = **4-12** (higher if CNI-free)
- **"VORI = 1/10 TAC"** = Voriconazole → **Tacrolimus dose 1/10–1/3**
- **"FLUCONAZOLE = 1/2 TAC"** = Fluconazole → **Tacrolimus 50% dose**
- **"CLARI = 1/2 CSA"** = Clarithromycin → **Ciclosporin 50-66%**
- **"RIFA = AVOID CNI"** = Rifampicin → **↓ CNI 80-90%** — AVOID
- **"AZITHRO = SAFE"** = Azithromycin = no CYP3A4 inhibition
- **"TAC = DIABETES, TREMOR, PRES"**
- **"CSA = HIRSUTISM, GINGIVAL, HYPERLIPID, GOUT, HTN, TMA"**
- **"SRL = PROTEINURIA, WOUND HEALING, HYPERLIPID, PNEUMONITIS"**
- **"MMF = GI, BONE MARROW, TERATOGENIC"**

## 11. 11. 📝 Exam Answer Modes

### 1. Short Question (5 marks): *"Tacrolimus target trough ranges and key drug interactions."*
- **Kidney**: Early (0–3mo) **10–15**; Late (>3mo) **5–10** ng/mL
- **Liver**: **5–10**; **Heart/Lung**: **10–15**
- **Strong inhibitors (↓ dose)**: Voriconazole (**1/10–1/3**), Itraconazole/Posaconazole (**1/3**), Fluconazole (**1/2**), Clarithromycin/Erythromycin (**1/2**), Verapamil/Diltiazem (**1/2**)
- **Strong inducers (AVOID/↑ dose)**: Rifampicin (**AVOID**, ↓ 90%), Carbamazepine/Phenytoin/Phenobarbital, St John's Wort
- **Azithromycin safe** (no CYP3A4 inhibition)

### 2. Viva (1 min): *"Kidney transplant patient on tacrolimus 4mg BD, started on voriconazole for aspergillosis. Tacrolimus level next day 45 ng/mL (target 5–10). Action?"*
- **Voriconazole = potent CYP3A4/P-gp inhibitor → ↑ tacrolimus 5–10x**
- **IMMEDIATELY reduce tacrolimus to 1/10–1/3 dose** (e.g., 4mg BD → 0.5mg daily or 1mg daily)
- **Monitor tacrolimus level DAILY** until stable
- Re-titrate after voriconazole stopped (return to baseline over 3–5 days)

### 3. Ward Round (30 sec): *"Transplant patient on ciclosporin, prescribed clarithromycin for chest infection. Alternative?"*
- **Clarithromycin inhibits CYP3A4 → ↑ ciclosporin 2–3x**
- **SWITCH to azithromycin** (no CYP3A4 inhibition)
- If clarithromycin essential: ↓ ciclosporin 50–66%, monitor levels daily

### 4. Last-Night Revision (1-liners):
- Tac: Kidney 10-15→5-10; Liver 5-10; Heart/Lung 10-15
- CsA: Kidney 200-300→100-200; C2 800-1500
- SRL: 4-12; EVR: 3-8
- Vori + Tac = 1/10–1/3 dose (DAILY levels); Fluconazole = 1/2; Clari = 1/2
- Rifampicin = AVOID (↓ 90%); Azithromycin = SAFE
- Tac toxicity = PTDM, Tremor, PRES, Nephro (afferent vasoconstriction)
- CsA toxicity = Hirsutism, Gingival, TMA, Gout, HTN, Hyperlipid
- SRL = Proteinuria, Wound healing, Hyperlipid, Pneumonitis
- MMF = GI, Bone marrow, Teratogenic

## 2. 📊 Classification of Renal ADRs

| Lesion Type | Mechanism | Typical Drugs | Key Features |
|-------------|-----------|---------------|--------------|
| **Acute Interstitial Nephritis (AIN)** | **Type IV hypersensitivity** (T-cell mediated) | **NSAIDs, Antibiotics (β-lactams, sulfonamides, rifampicin), PPIs, Diuretics (thiazide, loop), Allopurinol, ICD drugs** | Subacute (days–weeks), fever, rash, eosinophilia (blood/urine), sterile pyuria, ↑ Cr |
| **Acute Tubular Necrosis (ATN)** | Direct tubular toxicity / ischaemia | **Aminoglycosides, Amphotericin B, Contrast, Cisplatin, Vancomycin (high trough), Tenofovir, Lithium** | ↑ Cr, muddy brown casts, FENa >2%, non-oliguric often |
| **Crystal Nephropathy** | Intraluminal crystal precipitation | **Acyclovir, Indinavir, Atazanavir, Sulfonamides, Methotrexate (high-dose), Triamterene, Ciprofloxacin** | Flank pain, haematuria, crystals on US/microscopy, ↑ Cr |
| **Glomerular Disease** | Immune complex / podocyte injury | **Gold, Penicillamine, NSAIDs (MCD), Interferon-α, Heparin (HIT), Cancer immunotherapy (ICI)** | Proteinuria (nephrotic/nephritic), haematuria, ↑ Cr |
| **Thrombotic Microangiopathy (TMA)** | Endothelial injury | **Ciclosporin, Tacrolimus, Sirolimus, Clopidogrel, Ticlopidine, Quinine, Gemcitabine, VEGF inhibitors** | MAHA (schistocytes), thrombocytopenia, ↑ Cr, neurological |
| **Electrolyte / Tubular** | Specific tubular dysfunction | **Lithium (NDI), Amphotericin B (RTA type 1), Ifosfamide (Fanconi), Tenofovir (Fanconi), Cisplatin (Mg²⁺/K⁺ wasting)** | Polyuria, electrolyte abnormalities, ↑ Cr |

## 3. 🔬 Acute Interstitial Nephritis (AIN) — *Most Common Drug-Induced Renal Lesion*

### Clinical Presentation
| Feature | Typical |
|---------|---------|
| **Onset** | Days to weeks (NSAIDs: weeks–months) |
| **Symptoms** | Fever (30–50%), rash (25–50%), arthralgia |
| **Labs** | ↑ Cr, **eosinophilia** (blood 30%, urine 80%), **sterile pyuria**, **WBC casts** |
| **Urine** | Low-grade proteinuria (<1g/d), haematuria |
| **Biopsy** | Interstitial oedema, **inflammatory infiltrate (eosinophils, lymphocytes, plasma cells)**, tubulitis |

### High-Risk Drugs for AIN
| Drug Class | Specific Agents | Notes |
|------------|-----------------|-------|
| **NSAIDs** | All (esp. fenoprofen, ibuprofen) | **Most common cause**; delayed onset (weeks–months); often no fever/rash/eosinophilia |
| **Antibiotics** | β-lactams (penicillins, cephalosporins), **Sulfonamides**, Rifampicin, Quinolones | Classic triad (fever, rash, eosinophilia) more common |
| **PPIs** | Omeprazole, Esomeprazole, Lansoprazole | Increasing recognition; often insidious |
| **Diuretics** | Thiazides, Loop, Triamterene | Often with other features |
| **Allopurinol** | Allopurinol | Part of DRESS/hypersensitivity |
| **ICI (Immunotherapy)** | Pembrolizumab, Nivolumab, Ipilimumab | **Immune checkpoint inhibitor nephritis** — distinct entity, high-dose steroids |

### Management
1. **STOP offending drug immediately**
2. **Supportive**: hydration, avoid nephrotoxins, correct electrolytes
3. **Corticosteroids**: Prednisolone 1mg/kg/day (0.5–1mg/kg) × 2–4 weeks → taper 4–8 weeks
   - Best if started **early (<7–10 days from Cr rise)**
   - NSAID-AIN: less responsive to steroids
4. **Renal replacement** if indicated (KDIGO criteria)
5. **Follow-up**: Monitor Cr for 6–12 months (risk of CKD)

## 4. 💊 Aminoglycoside Nephrotoxicity (ATN)

| Feature | Details |
|---------|---------|
| **Mechanism** | Proximal tubular uptake → lysosomal phospholipidosis → mitochondrial dysfunction |
| **Risk Factors** | **Extended-interval dosing (↓ risk)**, age>60, pre-existing CKD, dehydration, loop diuretics, amphotericin, vancomycin, contrast |
| **Presentation** | Non-oliguric ↑ Cr (day 5–10), **mdrd casts**, FENa >2%, tubular proteinuria (β2-microglobulin, NAG) |
| **Monitoring** | **Extended-interval (Hartford)**: single daily dose; **trough <1 mg/L** at 18h post-dose |
| **Management** | Stop/hold aminoglycoside; hydration; avoid other nephrotoxins; usually reversible |
| **TDM** | **Target**: Peak 14–20 mg/L (traditional); **Trough <1 mg/L** (extended-interval) |

## 5. 💎 Crystal Nephropathy

| Drug | Crystals | Presentation | Prevention |
|------|----------|--------------|------------|
| **Acyclovir** | Needle-like (urine pH dependent) | AKI, flank pain, crystalluria | **Hydration** (IV 1.5–2L/day), adjust for renal, avoid rapid infusion |
| **Indinavir / Atazanavir** | Rectangular / star-shaped | Flank pain, haematuria, obstructive uropathy | **Hydration** (2–3L/day); atazanavir less nephrotoxic |
| **Sulfonamides** | Sheaf-of-wheat | Crystalluria, obstruction | Hydration, alkalinisation (historical) |
| **Methotrexate (HD)** | MTX + 7-OH-MTX | AKI, tubular precipitation | **Leucovorin rescue**, hydration, urine alkalinisation (pH>7), monitor levels |
| **Ciprofloxacin** | Needle-like | Rare crystalluria | Hydration |

## 6. 🩸 Thrombotic Microangiopathy (TMA)

| Drug | Context | Features |
|------|---------|----------|
| **Ciclosporin / Tacrolimus / Sirolimus** | Transplant | MAHA, thrombocytopenia, ↑ Cr, neurological; **dose-dependent**; ↓ dose/stop |
| **Clopidogrel / Ticlopidine** | Antiplatelet | Rare; onset days–weeks; **stop drug → plasma exchange** |
| **Quinine / Quinine** | Malaria/leg cramps | Immune-mediated; **stop immediately** |
| **Gemcitabine / VEGF inhibitors** | Chemotherapy | Dose-dependent; may need drug cessation |
| **HIT (Heparin)** | Anticoagulation | **4Ts score**; ↓ platelets >50%; **stop heparin, argatroban/bivalirudin** |

## 7. 📋 KDIGO AKI Staging (for Drug-Induced AKI)

| Stage | Serum Creatinine | Urine Output |
|-------|------------------|--------------|
| **1** | **1.5–1.9× baseline** OR **≥26.5 μmol/L increase** | **<0.5 mL/kg/h for 6–12h** |
| **2** | **2.0–2.9× baseline** | **<0.5 mL/kg/h for ≥12h** |
| **3** | **≥3.0× baseline** OR **≥354 μmol/L** OR **RRT** OR **<0.3 mL/kg/h for ≥24h** / **anuria ≥12h** |

## 10. 🤯 Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **AIN vs ATN** | AIN = **immune** (eosinophilia, WBC casts, sterile pyuria, fever/rash); ATN = **toxic/ischaemic** (muddy brown casts, FENa>2%, no eosinophilia) |
| **NSAID-AIN vs Antibiotic-AIN** | NSAID = insidious, no triad, older; Antibiotic = classic triad, younger |
| **Crystal nephropathy vs AIN** | Crystal = **flank pain, crystalluria, obstructive**; AIN = **allergic features, eosinophilia** |
| **TMA vs HUS/TTP** | Drug-induced TMA = **stop drug first**; TTP = **ADAMTS13 deficiency → plasma exchange**; HUS = **STEC/shiga toxin** |
| **Lithium NDI vs DI** | **NDI** = renal resistance to ADH (lithium, amphotericin); **DI** = central ADH deficiency |

**Mnemonics:**
- **"AIN DRUGS"** = **A**ntibiotics (β-lactams, sulfa, rifampicin), **I**buprofen/NSAIDs, **N**exium/PPIs, **D**iuretics, **R**ifampicin, **U**ric acid (allopurinol), **G**old, **S**ulfonamides
- **"ATN TOXINS"** = **A**minoglycosides, **T**enofovir, **N**ephrotoxins (Ampho B, Cisplatin, Contrast, Vancomycin)
- **"CRYSTALS"** = **C**iprofloxacin, **R**ifampicin (rare), **Y** (Acyclovir), **S**ulfonamides, **T**riamterene, **A**tazanavir/Indinavir, **L**ithium (not crystal), **S** (Methotrexate HD)
- **"TMA DRUGS"** = **T**acrolimus/Ciclosporin/Sirolimus, **M**itomycin/Gemcitabine/VEGF, **A**ntiplatelet (Clopidogrel/Ticlopidine), **D**rug (Quinine), **R**are, **U**ncommon, **G**emcitabine, **S**top drug
- **"LITHIUM"** = **L**ithium → **N**ephrogenic **D**I (NDI) → **Polyuria, Hypernatremia** → **Amiloride**

## 14. 📝 Exam Answer Modes

### Long Question (10 marks): *"Classify drug-induced kidney injury. Discuss AIN in detail."*
1. **Classification** (2): 6 categories (AIN, ATN, Crystal, Glomerular, TMA, Tubular) with mechanisms
2. **AIN** (4): Pathophysiology (Type IV), drugs (NSAIDs #1, antibiotics, PPIs), clinical features (classic triad vs NSAID-AIN), biopsy, management (stop drug, steroids 1mg/kg)
3. **Other key lesions** (3): ATN (aminoglycosides, TDM), Crystal (acylovir hydration), TMA (CNIs, clopidogrel)
4. **AKI Staging** (1): KDIGO 3 stages

### Short Question (5 marks): *"NSAID-induced AIN vs Antibiotic-induced AIN"*
| Feature | NSAID-AIN | Antibiotic-AIN |
|---------|-----------|----------------|
| Onset | Weeks–months | Days–weeks |
| Classic triad | **Often absent** | **Present** (fever, rash, eosinophilia) |
| Age | Older | Any |
| Eosinophilia | Often absent | Common |
| Biopsy | Similar | Similar |
| Steroid response | Less responsive | More responsive |

### Viva (2 min): *"70yo on ibuprofen 3 months for OA, Cr 180 (baseline 90), no fever/rash, urine eosinophils+. Diagnosis? Management?"*
- **NSAID-induced AIN** (insidious, no classic triad)
- **STOP ibuprofen immediately**
- Hydration, avoid nephrotoxins
- **Prednisolone 1mg/kg/day × 2–4 weeks** → taper 4–8 weeks
- Monitor Cr weekly ×4, then monthly ×6–12mo
- Switch to paracetamol/topical NSAID for OA

### Ward Round (30 sec): *"Patient on IV acyclovir for HSV encephalitis, day 3: flank pain, crystalluria, Cr rising. Action?"*
- **Acyclovir crystal nephropathy**
- **Aggressive IV hydration** (2–3L/day), **urine alkalinisation** (target pH>7), slow infusion
- Adjust dose for renal function
- Stop if severe AKI

### Last-Night Revision (1-liners):
- AIN = Type IV hypersensitivity; NSAIDs #1 cause (insidious); antibiotics = classic triad
- AIN management: Stop drug → Steroids 1mg/kg × 2–4wk → taper
- ATN: Aminoglycosides, Ampho B, Contrast, Cisplatin, Vancomycin (trough>15), Tenofovir
- Crystal nephropathy: Acyclovir (hydration), Indinavir/Atazanavir (hydration), MTX HD (alkalinisation + leucovorin)
- TMA: CNIs (dose-dependent), Clopidogrel (immune → PLEX)
- Lithium = NDI (polyuria, hypernatremia) + chronic interstitial → amiloride
- KDIGO: Stage 1 (1.5× Cr), Stage 2 (2×), Stage 3 (3×/RRT)

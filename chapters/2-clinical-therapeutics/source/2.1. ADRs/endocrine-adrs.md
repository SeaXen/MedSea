## 2. 📊 Classification of Endocrine ADRs

| Endocrine Axis | ADR | Key Drugs | Clinical Features |
|----------------|-----|-----------|-------------------|
| **ADH / Water** | **SIADH** (Hyponatraemia) | **SSRIs, SNRIs, TCAs, Carbamazepine, Oxcarbazepine, Cyclophosphamide, Vincristine, Clofibrate, Chlorpropamide, NSAIDs, Opioids, Trimethoprim** | Euvolaemic hyponatraemia, ↓ serum osmolality, ↑ urine osmolality, ↑ urine Na⁺ |
| | **Diabetes Insipidus (NDI)** | **Lithium, Demeclocycline, Amphotericin B, Ofloxacin, Foscarnet** | Polyuria, polydipsia, hypernatremia, low urine osmolality |
| **Glucose / Diabetes** | **Drug-Induced Hyperglycaemia / Diabetes** | **Corticosteroids, Thiazides (high-dose), Beta-blockers, Atypical antipsychotics (olanzapine, clozapine), Tacrolimus, Ciclosporin, Sirolimus, Protease inhibitors, L-asparaginase, Nicotinic acid, Pentamidine** | New-onset DM, worsening glycaemic control, DKA/HHS risk |
| | **Hypoglycaemia** | **Insulin, Sulfonylureas (glibenclamide), Beta-blockers (mask symptoms), ACEi/ARB (↑ insulin sensitivity), SGLT2i + insulin/sulfonylurea, Alcohol, Quinolones, Pentamidine, Sulfonamides** | Neuroglycopenic + adrenergic symptoms; BB masks adrenergic |
| **Adrenal** | **Adrenal Insufficiency (Primary)** | **Ketoconazole, Metronidazole, Rifampicin (induces cortisol metabolism), Phenytoin, Phenobarbital, Mitotane, Etomidate (acute), Checkpoint inhibitors (autoimmune hypophysitis)** | Fatigue, hypotension, hyponatraemia, hyperkalaemia, ↑ ACTH |
| | **Adrenal Insufficiency (Secondary)** | **Long-term glucocorticoids (HPA axis suppression), Opioids, Checkpoint inhibitors (hypophysitis)** | Fatigue, hypotension, hyponatraemia, **normal K⁺**, low/normal ACTH |
| | **Critical Illness-Related Corticosteroid Insufficiency (CIRCI)** | **Sepsis, trauma, major surgery** | Vasopressor-refractory hypotension |
| **Thyroid** | **Hypothyroidism** | **Lithium, Amiodarone, Tyrosine kinase inhibitors (sunitinib), Checkpoint inhibitors, Interferon-α, Thalidomide, Bevacizumab, Iodine excess** | Fatigue, weight gain, cold intolerance, ↑ TSH, ↓ fT4 |
| | **Thyrotoxicosis (Amiodarone-Induced Thyrotoxicosis - AIT)** | **Amiodarone (Type 1: iodine-induced; Type 2: destructive thyroiditis)** | Weight loss, tachycardia, AF, tremor; ↑ fT3/fT4, ↓ TSH |
| | **Thyroiditis** | **Checkpoint inhibitors, Interferon-α, Alemtuzumab, Lithium (rare)** | Transient thyrotoxicosis → hypothyroidism |
| **Bone / Calcium** | **Osteoporosis / Osteonecrosis** | **Glucocorticoids (major), Aromatase inhibitors, Androgen deprivation therapy, Heparin (long-term), Anticonvulsants (enzyme inducers), PPIs (long-term), Loop diuretics, Chemotherapy** | Fractures, bone pain, ↓ BMD |
| | **Osteonecrosis of Jaw (ONJ)** | **Bisphosphonates (IV > oral), Denosumab, Anti-angiogenics (bevacizumab)** | Exposed bone in maxilla/mandible >8 weeks |
| **Reproductive** | **Gynaecomastia** | **Spironolactone, Finasteride/Dutasteride, Cimetidine, Ketoconazole, Calcium channel blockers (verapamil), ACEi (rare), Antipsychotics (risperidone → ↑ prolactin), GnRH agonists** | Bilateral breast tissue proliferation in males |
| | **Sexual Dysfunction** | **SSRIs (most common), Antipsychotics (↑ prolactin), Beta-blockers, Thiazides, Spironolactone, Finasteride, Opioids, GnRH agonists** | Libido loss, erectile dysfunction, anorgasmia |
| | **Hyperprolactinaemia** | **Antipsychotics (typical > atypical; risperidone > others), Metoclopramide, Domperidone, Verapamil, Methyldopa, SSRIs (rare)** | Galactorrhoea, amenorrhoea, infertility, gynaecomastia, sexual dysfunction |

## 3. 💧 SIADH — **Common, High-Yield**

### Diagnostic Criteria (Bartter & Schwartz)
| Criterion | Required |
|-----------|----------|
| **Hyponatraemia** (Na⁺ <135 mmol/L) | ✓ |
| **Low serum osmolality** (<275 mOsm/kg) | ✓ |
| **Inappropriately concentrated urine** (Uosm >100, usually >Serum osmolality) | ✓ |
| **High urine sodium** (UNa⁺ >40 mmol/L, usually >100) | ✓ |
| **Euvolaemia** (no oedema, no dehydration) | ✓ |
| **Normal thyroid, adrenal, renal function** | ✓ |
| **No diuretic use** (recent) | ✓ |

### Top Culprit Drugs
| Drug Class | Specific Agents | Risk |
|------------|-----------------|------|
| **Antidepressants** | **SSRIs (citalopram, sertraline, fluoxetine), SNRIs (venlafaxine), TCAs (amitriptyline)** | **High** (esp. elderly, ↓ body weight) |
| **Antiepileptics** | **Carbamazepine, Oxcarbazepine, Valproate, Levetiracetam** | **High** |
| **Chemotherapy** | **Cyclophosphamide, Vincristine, Ifosfamide, Cisplatin** | High |
| **Antipsychotics** | **Haloperidol, Risperidone, Olanzapine** | Moderate |
| **Opioids** | **Morphine, Fentanyl, Tramadol** | Moderate |
| **Others** | **Trimethoprim, NSAIDs, Chlorpropamide, Clofibrate** | Moderate |

### Management Algorithm
```mermaid
flowchart TD
    A[Hyponatraemia + Suspected SIADH] --> B[**STOP CULPRIT DRUG IF POSSIBLE**]
    B --> C{Severity / Symptoms}
    C -->|**Severe (Na⁺ <120 or symptoms: seizures, confusion, coma)**| D[**CRITICAL CARE**<br/>**Hypertonic saline 3% (100-150mL over 10-20min)**<br/>Target: ↑ Na⁺ by 4-6 mmol/L in first 2h<br/>**Max ↑ 8-10 mmol/L in 24h** (avoid ODS)<br/>Furosemide 20-40mg IV to promote free water excretion<br/>Monitor Na⁺ q1-2h]
    C -->|**Moderate (Na⁺ 120-130, mild symptoms)**| E[**Fluid Restriction 800-1000mL/day**<br/>**Oral salt tablets / IV 0.9% NaCl** if hypovolaemic<br/>**Demeclocycline 600-1200mg/day** (if chronic, refractory)<br/>**Tolvaptan 15-30mg/day** (vaptan, hospital only)<br/>Monitor Na⁺ q6-12h]
    C -->|**Mild (Na⁺ 130-135, asymptomatic)**| F[**Fluid Restriction 1-1.5L/day**<br/>Stop culprit drug<br/>Monitor Na⁺ daily<br/>Review medications]
    D & E & F --> G[**Identify & stop causative drug**<br/>Switch antidepressant (mirtazapine, bupropion, agomelatine lower SIADH risk)<br/>Switch AED (levetiracetam, topiramate, lamotrigine, lacosamide low SIADH risk)]
```

## 4. 🍬 Drug-Induced Hyperglycaemia / Diabetes

| Drug | Mechanism | Monitoring |
|------|-----------|------------|
| **Corticosteroids** | ↑ Gluconeogenesis, ↓ peripheral glucose uptake, ↑ insulin resistance | **CBG q6h** if high-dose; HbA1c at 3mo; **sliding scale insulin** |
| **Thiazides (High-dose >25mg)** | ↓ Insulin secretion (K⁺ depletion), ↑ insulin resistance | Fasting glucose at 4–12 weeks |
| **Beta-blockers** | ↓ Insulin secretion (β2 blockade), mask hypoglycaemia | HbA1c if long-term |
| **Atypical Antipsychotics** | **Olanzapine, Clozapine > Risperidone > Quetiapine/Aripiprazole**; weight gain + insulin resistance | **Fasting glucose, lipids, weight at baseline, 4wks, 12wks, then annually** |
| **Calcineurin Inhibitors** | **Tacrolimus > Ciclosporin**; β-cell toxicity, ↓ insulin secretion | **Fasting glucose weekly ×4 then monthly** |
| **Protease Inhibitors** | Insulin resistance, lipodystrophy | Fasting glucose at 3–6 months |
| **L-Asparaginase** | ↓ Insulin synthesis (asparagine depletion) | CBG monitoring |

### Steroid-Induced Hyperglycaemia — Specifics
| Dose | Pattern | Management |
|------|---------|------------|
| **Low (<10mg prednisolone)** | Minimal | Monitor |
| **Moderate (10–30mg)** | Post-prandial predominance | **Metformin + Gliclazide**; add basal insulin if needed |
| **High (>30mg / IV)** | Fasting + post-prandial | **Basal-bolus insulin**; **sliding scale inadequate** |

## 5. 🏥 Adrenal Insufficiency (AI)

### Types & Drug Causes
| AI Type | Mechanism | Key Drugs |
|---------|-----------|-----------|
| **Primary (Adrenal)** | Direct adrenal toxicity / enzyme inhibition / autoimmune | **Ketoconazole (CYP11B1 inhibition), Metronidazole, Rifampicin (induces cortisol metabolism), Phenytoin, Phenobarbital, Mitotane (adrenolytic), Etomidate (11β-hydroxylase inhibition), Checkpoint inhibitors (autoimmune adrenalitis)** |
| **Secondary (Pituitary)** | HPA axis suppression / hypophysitis | **Long-term glucocorticoids (>3 weeks), Opioids (↓ ACTH pulse), Checkpoint inhibitors (hypophysitis)** |
| **CIRCI** | Relative insufficiency in critical illness | Sepsis, trauma, major surgery (not drug-induced per se) |

### Diagnosis
| Test | Primary AI | Secondary AI |
|------|------------|--------------|
| **Cortisol (08:00h)** | Low | Low/Normal |
| **ACTH** | **High** | Low/Normal |
| **Electrolytes** | **Hyponatraemia, Hyperkalaemia** | Hyponatraemia, **Normal K⁺** |
| **Renin** | High | Normal |
| **Short Synacthen Test** | **No response** | Delayed/blunted response |

### Management
| Situation | Treatment |
|-----------|-----------|
| **Acute Adrenal Crisis** | **Hydrocortisone 100mg IV STAT**, then 200mg/24h continuous infusion or 50mg q6h; **IV 0.9% NaCl 1L bolus**, then 2–3L/24h; **Fludrocortisone 50–100mcg/day** once stable (primary only); **Identify/treat trigger** |
| **Chronic Primary AI** | **Hydrocortisone 15–25mg/day** (⅔ AM, ⅓ PM) + **Fludrocortisone 50–100mcg/day** |
| **Chronic Secondary AI** | **Hydrocortisone 15–25mg/day** (⅔ AM, ⅓ PM); **NO fludrocortisone** |
| **Perioperative (on steroids >3wks)** | **Hydrocortisone 50mg IV induction**, then 25mg q8h ×24–72h (major surgery); 25mg IV induction then 25mg q8h ×24h (minor) |
| **Drug-Induced (Ketoconazole, Rifampicin, etc.)** | **Stop offending drug**; stress-dose steroids if needed |

## 6. 🦋 Thyroid Dysfunction

### Amiodarone-Induced Thyroid Dysfunction
| Type | Mechanism | Features | Management |
|------|-----------|----------|------------|
| **AIT Type 1** (Iodine-induced) | Excess iodine (75mg/200mg tab) → Jod-Basedow in nodular goitre | ↑ fT3/fT4, ↓ TSH; **thyroid uptake normal/high** | **Thionamides (carbimazole 20–40mg) + Potassium perchlorate 200–400mg/day**; consider amiodarone cessation |
| **AIT Type 2** (Destructive thyroiditis) | Direct cytotoxic thyroiditis → hormone leak | ↑ fT3/fT4, ↓ TSH; **thyroid uptake LOW**, ↑ IL-6 | **Prednisolone 30–40mg/day** taper 2–3mo; Thionamides **ineffective**; consider amiodarone cessation |
| **Amiodarone Hypothyroidism** | Wolff-Chaikoff failure (impaired escape) in susceptible | ↑ TSH, ↓ fT4; **amiodarone usually continued** | **Levothyroxine replacement**; monitor TSH 6–8 weekly |

### Lithium-Induced Thyroid Dysfunction
| Dysfunction | Mechanism | Management |
|-------------|-----------|------------|
| **Hypothyroidism** (most common) | ↓ Thyroid hormone synthesis/release; impairs coupling | **Levothyroxine**; **continue lithium** if psychiatric indication strong |
| **Goitre** | Compensatory TSH-driven | Monitor; levothyroxine if hypothyroid |
| **Thyrotoxicosis** (rare) | Destructive thyroiditis | Usually transient; beta-blockers |

### Checkpoint Inhibitor Thyroiditis
| Pattern | Features | Management |
|---------|----------|------------|
| **Thyrotoxicosis → Hypothyroidism** | Transient destructive thyroiditis (weeks); then permanent hypothyroidism | **Beta-blockers for thyrotoxic phase**; **Levothyroxine for hypothyroid phase**; **Continue ICI** unless severe |

## 7. 🦴 Bone & Calcium ADRs

### Glucocorticoid-Induced Osteoporosis (GIOP)
| Feature | Details |
|---------|---------|
| **Mechanism** | ↓ Osteoblast activity, ↑ osteoclast lifespan, ↓ calcium absorption, ↓ sex hormones, ↑ RANKL/OPG ratio |
| **Risk** | **Dose ≥7.5mg prednisolone/day for ≥3 months** |
| **Fracture Risk** | ↑ Vertebral (early), ↑ Non-vertebral (late) |
| **Prevention** | **Calcium 1000mg + Vitamin D 800–1000 IU/day**; **Bisphosphonate (alendronate 70mg/week) if >3mo ≥7.5mg**; **Denosumab**; **Teriparatide** (high risk) |
| **Monitoring** | **DEXA at baseline, then q1–2y** |

### ONJ (Osteonecrosis of Jaw)
| Risk Factors | **IV bisphosphonates (zoledronate) > oral; Denosumab; Anti-angiogenics (bevacizumab); Dental procedures; Poor oral hygiene; Corticosteroids; Diabetes** |
|--------------|----------------------------------------------------------------------------------------|
| **Presentation** | Exposed bone in maxilla/mandible **>8 weeks**, pain, infection |
| **Management** | **Stop antiresorptive**; conservative (chlorhexidine, antibiotics, analgesics); **surgery if refractory**; dental referral |

## 10. 🤯 Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **SIADH vs Cerebral Salt Wasting (CSW)** | SIADH = **euvolaemic**, urine Na⁺ high; CSW = **hypovolaemic**, urine Na⁺ high, **high urine output** |
| **Primary vs Secondary AI — K⁺** | Primary = **hyperkalaemia** (aldosterone deficiency); Secondary = **normal K⁺** (aldosterone intact via RAAS) |
| **AIT Type 1 vs Type 2** | Type 1 = **iodine excess in nodular goitre** (uptake normal/high) → thionamides; Type 2 = **destructive thyroiditis** (uptake LOW) → steroids |
| **Steroid hyperglycaemia pattern** | **Post-prandial > fasting** (esp. moderate dose); high dose = both |
| **Sliding scale insulin** | **Inadequate for steroid hyperglycaemia** — need basal-bolus |
| **Lithium thyroid** | **Hypothyroidism > goitre > thyrotoxicosis**; continue lithium if needed |
| **GIOP threshold** | **≥7.5mg prednisolone/day for ≥3 months** |

**Mnemonics:**
- **"SIADH DRUGS"** = **S**SRIs, **I**SNIs/**I**mipramine, **A**ntiepileptics (Carbamazepine, Oxcarbazepine), **D**emeclocycline (treatment, not cause), **H**Cyclophosphamide/Vincristine
- **"ADRENAL CRISIS"** = **A**drenal crisis = **H**ydrocortisone 100mg IV + **S**aline 1L + **F**ludrocortisone (primary)
- **"PRIMARY vs SECONDARY AI"** = **Primary = HyperK + High ACTH**; **Secondary = Normal K + Low ACTH**
- **"AIT TYPE 1 vs 2"** = **Type 1 = Iodine-induced (uptake HIGH) → Thionamides**; **Type 2 = Destructive (uptake LOW) → Steroids**
- **"GIOP THRESHOLD"** = **7.5mg prednisolone × 3 months** = START bisphosphonate
- **"HYPERPROLACTINAEMIA"** = **Risperidone > Haloperidol > Olanzapine > Quetiapine/Aripiprazole/Clozapine** → Switch to aripiprazole/quetiapine

## 14. 📝 Exam Answer Modes

### Long Question (10 marks): *"Discuss drug-induced SIADH — causes, diagnosis, and management."*
1. **Definition & Criteria** (2): Bartter & Schwartz criteria (6 points)
2. **Causes** (2): Drug table (SSRIs, SNRIs, TCAs, Carbamazepine, Oxcarbazepine, Chemo, Antipsychotics, Opioids, Trimethoprim)
3. **Differential** (1): CSW (hypovolaemic vs euvolaemic)
4. **Management** (4): Severity-based (Severe: Hypertonic saline 3%, target ↑4–6/2h, max 8–10/24h; Moderate: Fluid restriction, demeclocycline/tolvaptan; Mild: Fluid restriction); **STOP culprit drug**; switch to low-risk alternatives (mirtazapine, bupropion, levetiracetam)
4. **Monitoring** (1): Na⁺ frequency by severity

### Short Question (5 marks): *"Primary vs Secondary Adrenal Insufficiency — differentiation"*
| Feature | Primary AI | Secondary AI |
|---------|------------|--------------|
| **Cortisol** | Low | Low |
| **ACTH** | **High** | **Low/Normal** |
| **Hyponatraemia** | Yes | Yes |
| **Potassium** | **High** | **Normal** |
| **Renin** | High | Normal |
| **Synacthen Test** | No response | Blunted/delayed response |
| **Causes** | Adrenal destruction (autoimmune, ketoconazole, mitotane) | Pituitary/HPA (steroids, opioids, ICI hypophysitis) |
| **Fludrocortisone** | **Required** | **Not required** |

### Viva (2 min): *"Patient on amiodarone 200mg daily for 1 year. TSH 0.01, fT4 35, fT3 12. Radioiodine uptake LOW. IL-6 elevated. Diagnosis? Management?"*
- **AIT Type 2 (Destructive thyroiditis)** — uptake LOW, IL-6 ↑
- **Prednisolone 30–40mg/day**, taper over 2–3 months
- **Thionamides INEFFECTIVE** (destructive, not hyperfunction)
- Consider amiodarone cessation if feasible
- Beta-blockers for symptomatic control
- Monitor thyroid function monthly

### Ward Round (30 sec): *"Elderly on sertraline 100mg, Na⁺ 122, drowsy. Management?"*
- **SSRI-induced SIADH (Moderate-Severe)**
- **STOP sertraline**
- **Fluid restrict 800–1000mL/day**
- **Consider hypertonic saline if symptomatic** (drowsy = symptomatic)
- **Switch to mirtazapine or bupropion** (lower SIADH risk)
- Monitor Na⁺ q6–12h

### Last-Night Revision (1-liners):
- SIADH = Euvolaemic hyponatraemia + High urine osmo/Na⁺; SSRIs, Carbamazepine, Chemo
- Severe SIADH = Hypertonic saline 3%, ↑ Na⁺ 4–6/2h, max 8–10/24h
- Steroid hyperglycaemia = Post-prandial pattern → Basal-bolus insulin (sliding scale inadequate)
- Primary AI = Hyponatraemia + Hyperkalaemia + High ACTH → Fludrocortisone needed
- Secondary AI = Hyponatraemia + Normal K⁺ + Low ACTH → No fludrocortisone
- Adrenal crisis = Hydrocortisone 100mg IV + Saline 1L
- AIT Type 1 = Iodine-induced, uptake HIGH → Thionamides + Perchlorate
- AIT Type 2 = Destructive thyroiditis, uptake LOW → Prednisolone (Thionamides NO)
- Amiodarone hypo = Levothyroxine, continue amiodarone
- GIOP = Pred ≥7.5mg ×3mo → Ca/VitD + Bisphosphonate
- ONJ = IV Bisphosphonates/Denosumab > oral; exposed bone >8wks
- Hyperprolactinaemia = Risperidone > Haloperidol → Switch to Aripiprazole/Quetiapine

> 1. **SIADH (Severe)** → **Hypertonic saline 3%, ↑ Na⁺ 4–6/2h, STOP culprit**
> 2. **ADRENAL CRISIS** → **Hydrocortisone 100mg IV + Saline 1L + Fludrocortisone (primary)**
> 3. **AIT TYPE 2** → **Uptake LOW → Prednisolone (NOT thionamides)**
>
> **STEROID HYPERGLYCAEMIA** = Basal-bolus insulin (sliding scale NO)
> **PRIMARY AI** = HyperK + High ACTH + Fludrocortisone
> **SECONDARY AI** = Normal K + Low ACTH + No Fludrocortisone
> **GIOP** = Pred ≥7.5mg ×3mo → Bisphosphonate
> **HYPERPROLACTINAEMIA** = Risperidone → Switch to Aripiprazole/Quetiapine

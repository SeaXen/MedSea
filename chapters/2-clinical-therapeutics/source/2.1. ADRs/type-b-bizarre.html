CHAPTER 2 CLINICAL THERAPEUTICS AND GOOD PRESCRIBING / 2.1. ADRS
Type B Bizarre
1 🎯 Learning Objectives
  • [ ] Define Type B ADR: idiosyncratic, non-dose-dependent, unpredictable
  • [ ] Classify Type B by mechanism: immunological (hypersensitivity I–IV), non-immunological idiosyncrasy
  • [ ] Recognise high-mortality syndromes: SJS/TEN, DRESS, AGEP, AHS, Torsades, Anaphylaxis
  • [ ] Apply causality assessment (Naranjo, WHO-UMC, RUCAM) to Type B reactions
  • [ ] Know absolute contraindication to rechallenge
2 📖 Definition & Core Features
FeatureType B (Bizarre)
---------------------------
**Synonym**Idiosyncratic
**Dose-dependence****No** (may need threshold but not linear)
**Predictability****Unpredictable** from pharmacology
**Incidence****Rare** (1:1,000 – 1:100,000)
**Mortality****High** (can be fatal)
**Mechanism**Immunological (hypersensitivity) or non-immunological idiosyncrasy
**Rechallenge****Absolutely contraindicated**
**Genetic predisposition**Often (HLA alleles, enzyme polymorphisms)

2. DRESS (Drug Reaction with Eosinophilia and Systemic Symptoms)
FeatureDetails
------------------
**Triad**Fever + Rash + Organ involvement
**Typical onset****2–8 weeks** after drug start (delayed)
**Key drugs**Allopurinol, AEDs (CZP, PHT, PB, LMG, VPA), sulfonamides, dapsone, abacavir, minocycline, allopurinol
**HLA****B*58:01** (Allopurinol), **B*15:02** (CZP), **A*31:01** (CZP), **B*13:01** (Dapsone)
**RegiSCAR criteria**Fever ≥38°C, lymphadenopathy, eosinophilia >1.5×10⁹/L or atypical lymphs, organ involvement (liver, kidney, lung, heart), rash, reaction >2wks
**Mortality**~10% (liver failure, myocarditis)
**Management**Stop drug → systemic steroids (pred 1mg/kg) → slow taper 6–12wks → monitor LFTs, renal, cardiac
3. AGEP (Acute Generalised Exanthematous Pustulosis)
FeatureDetails
------------------
**Presentation****Acute febrile** + **non-follicular pustules** on oedematous erythema
**Onset****1–5 days** (rapid)
**Key drugs****Beta-lactams (esp. aminopenicillins)**, quinolones, hydroxychloroquine, diltiazem, sulfonamides
**Histology**Subcorneal pustules, neutrophilic infiltration
**Biomarker****Neutrophilia** (not eosinophilia)
**Course**Self-limiting **<2 weeks** after withdrawal
**Mortality**Low (<5%)
4. AHS (Anticonvulsant Hypersensitivity Syndrome) / DRESS by AEDs
FeatureDetails
------------------
**Drugs**Phenytoin, carbamazepine, phenobarbital, lamotrigine, valproate
**Triad**Fever, rash, systemic involvement (hepatitis, nephritis, pneumonitis)
**Onset**1–8 weeks
**HLA****HLA-B*15:02** (CZP), **HLA-A*31:01** (CZP DRESS), **HLA-B*38:01** (dapsone/CZP)
**Cross-reactivity****High among aromatic AEDs** (CZP, PHT, PB, OXC) — avoid all if reaction to one
**Management**Stop all aromatic AEDs → switch to non-aromatic (levetiracetam, topiramate, gabapentin, lacosamide, zonisamide)
5. Torsades de Pointes (Drug-Induced)
FeatureDetails
------------------
**Mechanism****hERG (IKr) blockade** → prolonged repolarisation → afterdepolarisations → R-on-T
**Risk factors**Female, elderly, hypokalaemia, hypomagnesaemia, bradycardia, congenital LQTS, polypharmacy (multiple QT drugs)
**High-risk drugs****Antiarrhythmics (Class Ia, III)**, **antipsychotics (haloperidol, thioridazine, ziprasidone)**, **antidepressants (citalopram, escitalopram, TCAs)**, **antibiotics (macrolides, fluoroquinolones)**, **antiemetics (ondansetron, domperidone)**, **antifungals (azole)**, **methadone**
**Monitoring**Baseline ECG, repeat if risk factors; **QTc >500ms or Δ>60ms = high risk**
**Acute management****IV Magnesium sulfate 2g bolus** (even if normal Mg²⁺) → correct K⁺/Mg²⁺ → pacing/isoprenaline if paused → stop offending drug
6. Anaphylaxis (Type I)
FeatureDetails
------------------
**Criteria**Acute onset + skin/mucosal + respiratory compromise **OR** hypotension **OR** GI symptoms
**Common drugs**Penicillins, cephalosporins, NSAIDs, neuromuscular blockers, radiocontrast, insulin, biologics
**Management****IM Adrenaline 1:1000 (0.5mg = 0.5mL) anterolateral thigh** → supine + legs raised → O₂ → IV fluids → chlorphenamine + hydrocortisone (adjuncts) → observe 6–12h
**Tryptase**Peak 1–2h, half-life 2h — sample at 1–2h and 24h (baseline)
7. DILI (Drug-Induced Liver Injury) — Idiosyncratic (Type B)
PatternDrugsFeatures

**Hepatocellular** (ALT>ALP)Isoniazid, valproate, paracetamol (overdose = Type A), nevirapine, flucloxacillin**Hy's Law**: ALT>3×ULN + Bilirubin>2×ULN → **10–50% mortality**

7 🤯 Confusions & Mnemonics
ConfusionClarification
--------------------------
**Type B vs Type C/D/E/F**Type B = **idiosyncratic** (unpredictable); Type C = **chronic** (time-dependent); D = **delayed** (carcinogenesis); E = **end of use** (withdrawal); F = **failure** (inefficacy)
**SJS/TEN vs DRESS vs AGEP**SJS/TEN: **necrosis/bullae**, mucosal, high mortality; DRESS: **eosinophilia**, organ involvement, delayed onset; AGEP: **pustules**, neutrophilia, rapid onset, benign
**AHS vs DRESS**AHS = **DRESS caused specifically by aromatic AEDs** — same spectrum; cross-reactivity among aromatic AEDs
**Torsades = Type A or B?****Type B** — idiosyncratic (not all get it at same dose); requires susceptibility (congenital LQTS, electrolytes, polypharmacy)
**Anaphylaxis vs Anaphylactoid**Anaphylaxis = **IgE-mediated (Type I)**; Anaphylactoid = **non-IgE**, direct mast cell activation (e.g., radiocontrast, opioids, vancomycin "red man") — **treatment identical**

Mnemonics:

  • "B = Bizarre = Bad (high mortality) = idiosyncratic"
  • "SCORTEN" = Skin >10%, Cancer, Old >40, Renal (BUN>28), Tachy >120, Electrolytes (HCO₃<20, Glc>14), Neoplasia
  • "REGISCAR for DRESS" = Rash, Eosinophilia, Globular (atypical lymphs), Intermittent fever, Systemic organ, Cytology, Allopurinol/AEDs, Reaction >2wks
  • "Hy's Law" = Hepatocellular (ALT>3×) + Yellow (Bil>2×) = High mortality
  • "IV Mg for Torsades" = Magnesium Mg for Mechanism (hERG block)

Viva (2 min): *"Patient on allopurinol 2 weeks, fever, rash, eosinophilia 3×10⁹, ALT 200. Diagnosis? Management?"*
  • DRESS (RegiSCAR criteria met)
  • Stop allopurinol immediately
  • Prednisolone 1mg/kg, slow taper 6–12 weeks

Ward Round (30 sec): *"Patient on clarithromycin + citalopram develops syncope. ECG shows polymorphic VT, twisting points. Immediate management?"*
  • Torsades de Pointes — IV Magnesium sulfate 2g bolus (even if Mg²⁺ normal)
  • Stop clarithromycin + citalopram (both QT prolonging)
  • Correct K⁺ >4.5, Mg²⁺ >1.0
  • Temporary pacing / isoprenaline if recurrent

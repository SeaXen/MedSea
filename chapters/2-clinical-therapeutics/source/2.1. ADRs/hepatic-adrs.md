## 2. 📊 DILI Classification by Biochemical Pattern

| Pattern | Definition (R-value) | Typical Drugs | Prognosis |
|---------|---------------------|---------------|-----------|
| **Hepatocellular** | **R = ALT/ULN ÷ ALP/ULN ≥ 5** | **Paracetamol (OD), Isoniazid, Valproate, Nevirapine, Flucloxacillin (some), Methotrexate, Halothane** | **Worse** — higher mortality, transplant |
| **Cholestatic** | **R ≤ 2** | **Flucloxacillin, Amoxicillin-Clavulanate, Azathioprine, Chlorpromazine, Anabolic steroids, OCP, Sulfonamides, Allopurinol** | Better — prolonged but lower mortality |
| **Mixed** | **2 < R < 5** | **Diclofenac, Sulfonamides, Allopurinol, Phenytoin, Carbamazepine, Tuberculosis drugs** | Intermediate |

> **R-value** = (ALT ÷ ULN) ÷ (ALP ÷ ULN)

---

## 3. ⚠️ Hy's Law — **Prognostic Rule (Exam Essential)**

| Criteria | Significance |
|----------|--------------|
| **1. Hepatocellular injury** (ALT >3× ULN) | |
| **2. Jaundice** (Total Bilirubin >2× ULN) | |
| **3. No obstruction** (imaging negative) | |
| **4. No other cause** (viral, autoimmune, etc.) | |

**If ALL 4 met → 10–50% mortality or transplant**  
**Hy's Law = "The law of mortality"** — stop drug immediately, transplant evaluation

## 4. 🧬 Intrinsic vs Idiosyncratic DILI

| Feature | **Intrinsic (Type A)** | **Idiosyncratic (Type B)** |
|---------|------------------------|----------------------------|
| **Dose-dependence** | **Yes** — predictable | **No** — unpredictable |
| **Incidence** | Common | Rare |
| **Latency** | Short (hours–days) | Variable (weeks–months) |
| **Mechanism** | Direct toxicity / metabolite | Immune / metabolic idiosyncrasy |
| **Examples** | **Paracetamol overdose**, Methotrexate (high-dose), Niacin, Valproate (microvesicular steatosis) | **Isoniazid, Flucloxacillin, Augmentin, Diclofenac, Allopurinol, Carbamazepine, Nevirapine** |
| **Rechallenge** | Toxic at same dose | **Absolutely contraindicated** |

## 5. 💊 High-Risk Drugs by Pattern

### Hepatocellular (R ≥ 5)
| Drug | Key Features |
|------|--------------|
| **Paracetamol** (overdose) | **NAPQI toxicity** → centrilobular necrosis; **NAC antidote**; Rumack-Matthew nomogram |
| **Isoniazid** | **Hepatocellular**, 1–3 months; ↑ risk age>35, alcohol, rifampicin; monitor LFTs monthly |
| **Valproate** | **Microvesicular steatosis** → hyperammonaemia; risk <3y, polytherapy, metabolic disorders |
| **Nevirapine** | **Hypersensitivity syndrome** (CD4>250 ♀, >400 ♂); rash + hepatitis; HLA-DRB1*01:01 |
| **Methotrexate** (high-dose) | **Folic acid rescue**; monitor levels at 24h/48h; leucovorin rescue |
| **Halothane** | **Immune-mediated** (trifluoroacetyl hapten); ↓ use now |

### Cholestatic (R ≤ 2)
| Drug | Key Features |
|------|--------------|
| **Flucloxacillin** | **Cholestatic**, 2–4 weeks; **HLA-DRB1*07:01/DQA1*02:01**; prolonged jaundice; elderly >50 |
| **Amoxicillin-Clavulanate** | **Cholestatic/mixed**, 1–6 weeks; ↑ risk >60y, male; **avoid re-exposure** |
| **Azathioprine** | **Cholestatic**, nodular regenerative hyperplasia; TPMT testing |
| **Chlorpromazine** | **Cholestatic**, 2–4 weeks; fever, eosinophilia |
| **OCP (estrogen)** | **Cholestatic**, benign; benign hepatic adenoma risk long-term |
| **Sulfonamides** | **Mixed/cholestatic**, hypersensitivity context |

### Mixed (2 < R < 5)
| Drug | Key Features |
|------|--------------|
| **Diclofenac** | **Mixed**, 1–3 months; ↑ risk elderly, alcohol; **avoid in chronic liver disease** |
| **Allopurinol** | **DRESS context**, mixed pattern; HLA-B*58:01 |
| **Phenytoin/Carbamazepine** | **AHS/DRESS**, mixed; HLA associations |
| **Anti-TB drugs (INH/RIF/PZA)** | **Combined hepatotoxicity**; RIF induces INH metabolism → toxic metabolite |

## 6. 🔬 RUCAM (Roussel Uclaf Causality Assessment Method) — *DILI Gold Standard*

| Domain | Max Score | Key Points |
|--------|-----------|------------|
| **1. Time to onset** (5–90d = +2) | +2 | <5d or >90d = -1 |
| **2. Course after cessation** (↓50% in 8d = +3) | +3 | ≥50% in 30d = +2; no info = 0 |
| **3. Risk factors** (Age≥55, Alcohol, Pregnancy = +1 each) | +2 | Max +2 |
| **4. Concomitant drugs** (Compatible hepatotoxin = -2) | 0 to -3 | None = 0 |
| **5. Exclusion of alternatives** (All ruled out = +2) | +2 to -3 | <3 ruled out = -2; non-hepatic probable = -3 |
| **6. Previous hepatotoxicity** (In label = +2) | +2 | Published = +1; none = 0 |
| **7. Rechallenge** (Positive = +3) | +3 to -2 | Negative = -2; not done = 0 |

**Total: -8 to +14**

| Score | Category |
|-------|----------|
| >8 | **Highly probable** |
| 6–8 | **Probable** |
| 3–5 | **Possible** |
| 1–2 | **Unlikely** |
| ≤0 | **Excluded** |

## 7. 📋 Monitoring & Management Algorithm

```mermaid
flowchart TD
    A[Suspected DILI] --> B[**STOP SUSPECT DRUG IMMEDIATELY**]
    B --> C{Pattern?}
    C -->|Hepatocellular| D[**Hy's Law Check**: ALT>3xULN + Bil>2xULN?]
    D -->|YES| E[**URGENT TRANSPLANT REFERRAL**<br/>ICU if encephalopathy/coagulopathy<br/>NAC if paracetamol/early ALF<br/>Monitor: LFTs daily, INR, glucose, renal, lactate]
    D -->|NO| F[Monitor LFTs q48–72h<br/>Resolution: ALT halves in 1–2wk<br/>Refer if: ALF criteria, no improvement 2wk]
    C -->|Cholestatic/Mixed| G[Monitor LFTs weekly<br/>Cholestatic: may take months to resolve<br/>Ursodeoxycholic acid 15mg/kg for pruritus<br/>Exclude obstruction (US/MRCP)]
    E & F & G --> H[**RUCAM Assessment** for causality<br/>Document: drug, timeline, pattern, exclusions<br/>Report: Yellow Card / FDA MedWatch<br/>**NEVER RECHALLENGE**]
```

## 9. ❓ Viva Questions (12)

| Q | Answer |
|---|--------|
| 1. Define Hy's Law. What is its prognostic significance? | **ALT>3×ULN + Bil>2×ULN (no obstruction, no other cause) → 10–50% mortality/transplant**; immediate drug cessation |
| 2. Classify DILI patterns by R-value. | Hepatocellular R≥5; Cholestatic R≤2; Mixed 2<R<5 |
| 3. Paracetamol overdose — mechanism, antidote, monitoring? | **NAPQI** (CYP2E1) depletes glutathione → necrosis; **NAC** (glutathione precursor); **Rumack-Matthew nomogram** (4–24h); monitor INR, creatinine, glucose, lactate |
| 4. Flucloxacillin DILI — pattern, HLA, time course? | **Cholestatic**; **HLA-DRB1*07:01/DQA1*02:01**; 2–4 weeks; prolonged jaundice; elderly |
| 5. Isoniazid hepatotoxicity — risk factors, monitoring? | Age>35, alcohol, rifampicin co-administration; **monthly LFTs**; symptomatic hepatitis = stop |
| 6. Valproate hepatotoxicity — mechanism, high-risk group, monitoring? | **Microvesicular steatosis** (mitochondrial β-oxidation inhibition); **<3y, polytherapy, metabolic disorders**; monitor LFTs, ammonia |
| 7. RUCAM domains and scoring? | 7 domains (Time, Course, Risk factors, Concomitant drugs, Exclusion, Previous toxicity, Rechallenge); -8 to +14; >8 = highly probable |
| 8. Amoxicillin-clavulanate DILI — pattern, risk factors? | Cholestatic/mixed; **>60y, male**; avoid re-exposure |
| 9. Microvesicular vs macrovesicular steatosis? | **Microvesicular**: small droplets, mitochondrial toxicity (valproate, tetracycline, Reye's), **hyperammonaemia**; **Macrovesicular**: large droplets, metabolic (NAFLD, alcohol) |
| 10. Anti-TB drug hepatotoxicity — which drug is main culprit? Mechanism? | **Isoniazid** (hydrazine metabolite); **Rifampicin induces CYP2E1** → ↑ toxic INH metabolite; PZA adds risk |
| 11. Methotrexate high-dose — rescue, monitoring? | **Leucovorin (folinic acid) rescue** based on 24h/48h MTX levels; monitor LFTs, renal, CBC |
| 12. DILI mimics — what must be excluded (RUCAM Domain 5)? | Hepatitis A/B/C/E, CMV, EBV, HSV, **autoimmune hepatitis** (ANA, ASMA, IgG), **Wilson's** (ceruloplasmin), **α1-antitrypsin**, **biliary obstruction** (US/MRCP), **ischaemic hepatitis**, **alcohol**, **NAFLD** |

## 10. 🤯 Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Hepatocellular vs Cholestatic** | R-value = (ALT/ULN) ÷ (ALP/ULN); Hepatocellular = ALT predominant; Cholestatic = ALP predominant |
| **Hy's Law = diagnosis?** | No — **prognostic rule**; indicates high mortality, not causality |
| **Intrinsic vs Idiosyncratic** | Intrinsic = dose-dependent, predictable (paracetamol OD); Idiosyncratic = unpredictable, immune/metabolic (flucloxacillin) |
| **Valproate = steatosis type?** | **Microvesicular** (mitochondrial β-oxidation inhibition) → hyperammonaemia |
| **RUCAM vs Naranjo** | RUCAM = **DILI-specific**, liver-focused domains; Naranjo = general ADR |

**Mnemonics:**
- **"HY'S LAW"** = **H**epatocellular (ALT>3×) + **Y**ellow (Bil>2×) = **High mortality**
- **"R-VALUE"** = **R**atio = **ALT/ULN ÷ ALP/ULN**; **≥5 = Hepatocellular**, **≤2 = Cholestatic**, **2–5 = Mixed**
- **"INH RISK"** = **I**NH hepatotoxicity: **A**ge>35, **L**cohol, **R**ifampicin co-Rx
- **"VALPROATE"** = **V**alproate → **Microvesicular steatosis** → **Hyperammonaemia** (not ALT-driven)
- **"FLUCLOX"** = **F**lucloxacillin → **Cholestatic** → **HLA-DRB1*07:01** → Elderly >50
- **"NAC"** = **N**-acetylcysteine for **P**aracetamol **O**verdose (and early ALF)

## 14. 📝 Exam Answer Modes

### Long Question (10 marks): *"Discuss drug-induced liver injury (DILI) — classification, Hy's Law, and management."*
1. **Classification** (2): R-value patterns (Hepatocellular R≥5, Cholestatic R≤2, Mixed 2<R<5) with examples
2. **Hy's Law** (2): ALT>3×ULN + Bil>2×ULN, no obstruction/other cause → 10–50% mortality; prognostic not diagnostic
3. **Intrinsic vs Idiosyncratic** (2): Dose-dependent vs unpredictable; examples
4. **Key drugs** (2): Paracetamol (NAPQI, NAC), INH (monthly LFTs), Flucloxacillin (cholestatic, HLA), Valproate (microvesicular), Anti-TB
5. **Management** (2): Stop drug, Hy's Law = transplant referral, RUCAM causality, never rechallenge

### Short Question (5 marks): *"Hy's Law and RUCAM"*
- **Hy's Law**: ALT>3×ULN + Bil>2×ULN (no obstruction, no other cause) = 10–50% mortality; stop drug immediately
- **RUCAM**: 7 domains (-8 to +14); Time, Course, Risk factors, Concomitant drugs, Exclusion of alternatives, Previous hepatotoxicity, Rechallenge; >8 = highly probable

### Viva (2 min): *"60M on flucloxacillin 3 weeks, jaundice, ALP 400, ALT 100, Bil 80. Pattern? Causality? Management?"*
- **Pattern**: Cholestatic (R = 2.5/3 = 0.8 < 2)
- **Causality**: Flucloxacillin classic; **RUCAM**: Time +2 (3wk), Course +2 (↓), Risk +1 (age), Concomitant 0, Exclusion +2 (viral/autoimmune/biliary ruled out), Previous +2 (in label), Rechallenge 0 = **9 = Highly probable**
- **Management**: Stop flucloxacillin; monitor LFTs weekly; may persist months; **avoid re-exposure**; UDCA for pruritus

### Ward Round (30 sec): *"Patient on isoniazid 2 months, ALT 500 (ULN 40), Bil 50 (ULN 20). Action?"*
- **Hy's Law criteria MET** (ALT>3×, Bil>2×) → **STOP INH immediately**, transplant referral, NAC considered, monitor INR/encephalopathy

### Last-Night Revision (1-liners):
- R = ALT/ULN ÷ ALP/ULN; ≥5 Hepatocellular, ≤2 Cholestatic, 2–5 Mixed
- Hy's Law = ALT>3× + Bil>2× = 10–50% mortality
- Paracetamol = NAPQI → NAC (Rumack-Matthew)
- INH = Hepatocellular, monthly LFTs, Hy's Law risk
- Flucloxacillin = Cholestatic, HLA-DRB1*07:01, elderly
- Valproate = Microvesicular steatosis → hyperammonaemia
- Augmentin = Cholestatic, >60y male
- RUCAM = DILI gold standard; 7 domains, >8 highly probable
- NEVER rechallenge idiosyncratic DILI

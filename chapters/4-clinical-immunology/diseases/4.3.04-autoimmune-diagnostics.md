## 🧠 Core Concept: Autoimmune Diagnostics Workflow

```mermaid
flowchart TD
    A[Clinical Suspicion of Autoimmunity] --> B[ANA Screening (HEp-2 IIF)]
    B --> C{ANA Positive?}
    C -->|Yes| D[ENA Profile (Line Blot/ELISA)]
    D --> E[Specific Autoantibody Profile]
    E --> F[Clinical Correlation + Other Tests]
    F --> G[Diagnosis]
    C -->|No| H[Low Probability<br/>Consider: Seronegative, Pre-test probability]
    H --> I[Alternative: ANCA, RF, aPL, TSHR Ab, etc.]
    I --> J{Positive?}
    J -->|Yes| F
    J -->|No| K[Low Probability<br/>Re-assess Clinical Picture]
```

## 1️⃣ Antinuclear Antibody (ANA) — HEp-2 IIF

### ANA Patterns (ICAP Nomenclature)

| Pattern (ICAP) | AC Code | Antigens | Associated Diseases |
|----------------|---------|----------|---------------------|
| **Homogeneous** | AC-1 | dsDNA, Histones, Nucleosomes | **SLE**, Drug-induced lupus, Juvenile SLE |
| **Speckled** | AC-4, AC-5 | **ENA**: Sm, RNP, Ro/SSA, La/SSB, Jo-1, Ku, PCNA | **SLE, Sjögren's, MCTD, SSc, PM/DM, SLE** |
| **Centromere** | AC-3 | CENP-A, CENP-B | **Limited SSc (CREST)**, Primary Biliary Cholangitis |
| **Nucleolar** | AC-8, AC-9, AC-10 | Fibrillarin, RNA Pol I/III, PM-Scl, U3 RNP | **Systemic Sclerosis**, PM-Scl (Overlap) |
| **Discrete Nuclear Dots** | AC-6 | Sp100, PML | **PBC** (Sp100), **Acute Promyelocytic Leukaemia** (PML) |
| **Nuclear Envelope** | AC-11 | Nuclear pore complex (Gp210, Tpr) | **PBC** (Gp210) |
| **Cytoplasmic** | AC-13 to AC-23 | Jo-1 (Cytoplasmic), **Ribosomal P** | **Anti-synthetase (Jo-1)**, **PBC** (Ribosomal P) |
| **Mitotic Spindle Apparatus** | AC-24 | Centrosome proteins | Rare, Some cancers |
| **Rods & Rings** | AC-25 | PML-RARα | **APL (t(15;17))** — Specific! |

### ANA Titre Interpretation
| Titre | Significance |
|-------|--------------|
| **<1:80** | Negative (Low pre-test probability) |
| **1:80 - 1:160** | Low positive — Common in healthy elderly, Non-specific |
| **1:320 - 1:640** | Moderate positive — Significant if clinical suspicion |
| **≥1:640** | High positive — Strong association with SLE/Sjogren/SSc/MCTD |

> **Key**: ANA is **Screening Test** — High sensitivity (~98% for SLE), Low specificity (5-20% healthy positives)

### ANA Algorithm
```mermaid
flowchart TD
    A[ANA Positive] --> B{Titre ≥1:160?}
    B -->|No| C[Low Clinical Significance<br/>If High Suspicion → Repeat/ENA]
    B -->|Yes| D[ENA Profile (Line Blot/ELISA/Array)]
    D --> E{Nuclear Pattern?}
    E -->|Homogeneous| F[Anti-dsDNA, Anti-Sm, Histones<br/>→ SLE, Drug-induced Lupus]
    E -->|Speckled| G[ENA Subtypes<br/>Sm/RNP/Ro/La/Jo-1/Ku/PM-Scl<br/>→ SLE, MCTD, Sjögren, SSc, PM/DM]
    E -->|Centromere| H[Anti-Centromere<br/>→ Limited SSc (CREST)]
    E -->|Nucleolar| I[Anti-Scl-70, Fibrillarin, RNA Pol III<br/>→ SSc Subtypes]
    E -->|Cytoplasmic| J[Anti-Jo-1, Ribosomal P<br/>→ Anti-synthetase, PBC]
```

| **Anti-PL-7** | Threonyl-tRNA synthetase | Anti-synthetase | Anti-synthetase, ILD-predominant |

### ENA Testing Methods
| Method | Principle | Advantages |
|--------|-----------|------------|
| **Line Blot (Immunoblot)** | Membrane strips with immobilised antigens | Visual, Semi-quantitative, Multiple antigens |
| **Addressable Laser Bead Immunoassay (ALBIA)** | Luminex xMAP technology | Quantitative, High-throughput, Multiplex |
| **ELISA** | Solid-phase immunoassay | Quantitative, Automated |
| **Immunofluorescence (IIF)** | HEp-2 cells + specific antigen | Pattern recognition |

## 3️⃣ ANCA (Anti-Neutrophil Cytoplasmic Antibodies)

### ANCA Patterns (IIF on Ethanol-Fixed Neutrophils)

| Pattern | IIF Appearance | Target Antigen | ELISA Target | Clinical Association |
|---------|----------------|----------------|--------------|---------------------|
| **c-ANCA** (Cytoplasmic) | **Granular cytoplasmic** | **PR3** (Proteinase 3) | **PR3-ANCA** | **GPA (90%)**, c-ANCA = GPA (Specificity >90%) |
| **p-ANCA** (Perinuclear) | **Nuclear rim** | **MPO** (Myeloperoxidase) | **MPO-ANCA** | **MPA (70%), EGPA (40%)**, Drug-induced |
| **Atypical p-ANCA** | Perinuclear + Cytoplasmic | **Multiple** (BPI, Lactoferrin, Cathepsin G, Elastase) | **Atypical p-ANCA** | **UC, PSC, AIH, CF, RA** |
| **Atypical c-ANCA** | Cytoplasmic atypical | Various | Various | Drug-induced, RA, CF |
| **Negative** | No pattern | — | — | Does not exclude vasculitis (5-10% ANCA-negative AAV) |

### ANCA Interpretation Algorithm
```mermaid
flowchart TD
    A[ANCA Positive by IIF] --> B{Pattern?}
    B -->|c-ANCA| C[PR3-ANCA ELISA<br/>Positive → GPA (90% specificity)]
    B -->|p-ANCA| D[MPO-ANCA ELISA<br/>Positive → MPA/EGPA]
    B -->|Atypical p/c-ANCA| E[Consider Drug-induced, UC, PSC, AIH, CF, RA]
    C & D --> F[Confirm with Clinical Correlation<br/>Biopsy if Indicated]
```

### ANCA Testing Algorithm (BSR/BSV Guidelines)
| Scenario | Recommended Testing |
|----------|--------------------|
| **Suspected AAV** (Pulmonary-renal, Glomerulonephritis, Pulmonary haemorrhage) | **IIF + PR3/MPO ELISA** (Parallel) |
| **Suspected EGPA** | **IIF + MPO-ANCA ELISA** (PR3 rarely positive) |
| **Suspected Drug-Induced Vasculitis** | **IIF + ELISA** — Check drug history (PTU, Hydralazine, Minocycline) |
| **IBD / PSC / AIH** | **Atypical p-ANCA** — Supports diagnosis, Not diagnostic alone |
| **ANCA Negative but Clinical Suspicion High** | Repeat in 2-4 weeks, Consider biopsy, Consider alternative diagnoses |

### ANCA Interpretation Table

| ANCA Result | Most Likely Diagnosis | Next Step |
|-------------|----------------------|-----------|
| **c-ANCA + PR3+** | **GPA (90% specificity)** | Biopsy (ENT/Lung/Kidney) for granulomas |
| **p-ANCA + MPO+** | **MPA (70%)** or **EGPA (40%)** | Differentiate by clinical features (Asthma/Eosinophilia = EGPA) |
| **p-ANCA + MPO-** | **Atypical p-ANCA** | Consider UC, PSC, AIH, RA, Drug-induced |
| **c-ANCA + PR3-** | Atypical c-ANCA | Drug-induced, CF, RA |
| **Negative ANCA** | **5-10% of AAV are ANCA-negative** | Consider biopsy, Alternative diagnoses |

## 4️⃣ Rheumatoid Factor (RF) & Anti-CCP

### Rheumatoid Factor (RF)
| Feature | Detail |
|---------|--------|
| **Target** | Fc portion of IgG (Usually IgM-RF, also IgG/IgA-RF) |
| **Sensitivity (RA)** | 60-80% |
| **Specificity** | 85% (Positive in Sjögren's, SLE, Chronic infections, Cryoglobulinaemia, Healthy elderly) |
| **Prognostic Value** | **High titre = More erosive disease, Extra-articular manifestations** |
| **Isotypes** | IgM-RF (Standard), IgA-RF (Mucosal), IgG-RF (Severe disease) |

### Anti-CCP (Anti-Citrullinated Protein Antibody)
| Feature | Detail |
|---------|--------|
| **Target** | Cyclic Citrullinated Peptide (CCP) — Citrullinated fibrinogen, Fibrinogen, Vimentin, α-Enolase |
| **Sensitivity (RA)** | 60-70% |
| **Specificity** | **>95%** (Gold standard for RA diagnosis) |
| **Prognostic** | **High titre = Erosive disease, Severe RA, Extra-articular manifestations** |
| **Generations** | **CCP2 (CCP3)** — Current standard (CCP1 obsolete) |
| **Pre-clinical** | **Present years before clinical RA** — Predictive value |

### RF vs Anti-CCP in RA
| Feature | RF | Anti-CCP |
|---------|-----|----------|
| **Sensitivity** | 60-80% | 60-70% |
| **Specificity** | 85% | **>95%** |
| **Prognostic Value** | Moderate | **High (Erosive disease)** |
| **Early RA** | Lower sensitivity | **Higher sensitivity in early RA** |

## 5️⃣ Complement Testing

### Complement Components & Pathways
| Pathway | Trigger | Key Components | Key Regulators |
|---------|---------|----------------|----------------|
| **Classical** | Immune complexes (IgG/IgM) | C1q, C1r, C1s, C4, C2 | C1-INH, C4BP |
| **Lectin** | MBL/MASP binding to pathogens | MBL, MASP-1/2, C4, C2 | C1-INH |
| **Alternative** | Spontaneous C3 hydrolysis | C3, Factor B, Factor D, Properdin | Factor H, Factor I, Properdin |
| **Terminal (MAC)** | C5 convertase | C5, C6, C7, C8, C9 (MAC) | CD59, CD46, CD55 |

### Complement Levels Interpretation
| Pattern | Interpretation | Conditions |
|-----------|----------------|------------|
| **Low C3, Low C4, Low CH50** | **Classical Pathway Consumption** | **SLE (Active), Cryoglobulinaemia, Post-streptococcal GN, Infective Endocarditis** |
| **Low C3, Normal C4** | **Alternative Pathway Consumption** | **C3 Nephritic Factor (C3NeF), C3 Glomerulopathy (DDD/C3GN), Partial Lipodystrophy** |
| **Low C4, Normal C3** | **Classical Pathway Deficiency** | **C1q/C1r/C1s/C4/C2 Deficiency (SLE-like), HAE (C1-INH Deficiency)** |
| **Low C4, Low C3, Normal CH50** | **Early/Selective Classical Consumption** | Early SLE, Local consumption |
| **Normal C3, C4, Low CH50** | **Terminal Pathway Deficiency** | **C5-C9 Deficiency, Terminal Complement Deficiency** |

### Hereditary Complement Deficiencies
| Deficiency | Inheritance | Clinical Features |
|------------|-------------|-------------------|
| **C1q** | AR | **SLE-like (Early onset, Photosensitivity, Nephritis)**, Recurrent infections |
| **C1r/C1s** | AR | Similar to C1q |
| **C4** | AR | **SLE-like, Recurrent infections** |
| **C2** | AR | **Most common classical deficiency**, SLE-like, Recurrent infections |
| **C3** | AR | **Severe recurrent pyogenic infections, Autoimmunity (SLE-like)** |
| **C5-C9** | AR | **Recurrent Neisseria (Meningitis, Gonorrhoea)** |
| **C1-INH (HAE)** | AD | **Angioedema (Bradykinin), Normal C1q, Low C4, Normal C3** |

### Complement in Disease Monitoring
| Disease | Complement Pattern | Monitoring Utility |
|---------|-------------------|-------------------|
| **SLE** | **Low C3, Low C4, Low CH50** = **Active disease** (Renal flare) | **q3-6m** (C3, C4, CH50, dsDNA) |
| **Post-Streptococcal GN** | Low C3, Normal C4 | Resolves in 6-8 weeks |
| **Cryoglobulinaemia** | Low C4, Low/Normal C3 | Cryoglobulins, Rheumatoid factor |
| **aHUS** | Low C3, Normal C4 (Alternative pathway) | ADAMTS13, CFH, CFI, MCP, CFB mutations |
| **C3 Glomerulopathy** | Low C3, Normal C4, C3NeF+ | C3NeF, Genetic testing (CFH, CFI, CFB) |

## 6️⃣ Cryoglobulins

### Classification (Brouet)
| Type | Composition | Associated Conditions |
|------|-------------|----------------------|
| **Type I** | **Monoclonal Ig** (Usually IgM, IgG) | **Waldenström Macroglobulinaemia**, Multiple Myeloma, CLL |
| **Type II** | **Monoclonal IgM (κ) + Polyclonal IgG** | **HCV (90%)**, Lymphoproliferative disorders |
| **Type III** | **Polyclonal IgM + Polyclonal IgG** | **HCV, SLE, RA, Sjögren's, Infections** |

### Cryoglobulin Testing
| Step | Method |
|------|--------|
| **Collection** | **Pre-warmed syringes/tubes (37°C)**, Transport at 37°C, Clot at 37°C |
| **Processing** | Centrifuge at 37°C → Cryoprecipitate at 4°C (72h) → Wash → Resuspend |
| **Typing** | Immunofixation (Type I) / Immunofixation + Rheumatoid Factor (Type II/III) |
| **Quantification** | Cryocrit (% Volume of cryoprecipitate) |

### Clinical Features by Type
| Type | Clinical Features |
|------|-------------------|
| **Type I** | **Hyperviscosity Syndrome** (Visual, Neurological), Raynaud's, Glomerulonephritis (Rare) |
| **Type II** | **MPGN (Type I)**, Palpable Purpura, Arthralgia, Peripheral Neuropathy, **HCV Association (90%)** |
| **Type III** | Similar to Type II but less severe, **HCV, SLE, RA, Infections** |

## 6️⃣ Antiphospholipid Antibodies (aPL)

### Testing Panel
| Test | Method | Target |
|------|--------|--------|
| **Lupus Anticoagulant (LA)** | **dRVVT** (Dilute Russell's Viper Venom Time) + **aPTT** (Silica/kaolin) | Phospholipid-dependent coagulation inhibition |
| **Anti-Cardiolipin (aCL)** | ELISA | IgG/IgM against Cardiolipin/β2-GPI complex |
| **Anti-β2-Glycoprotein I (β2-GPI)** | ELISA | IgG/IgM against β2-GPI (Co-factor for aCL) |

### Interpretation Criteria (Sydney 2006)
| Criterion | Threshold |
|-----------|-----------|
| **LA** | **Positive dRVVT** (Screen + Confirm) **OR** Prolonged aPTT not corrected by normal plasma |
| **aCL IgG/IgM** | **>40 GPL/MPL** (Medium/High titre) on 2 occasions ≥12w apart |
| **Anti-β2-GPI IgG/IgM** | **>40 SGU/SSM** on 2 occasions ≥12w apart |

### APS Clinical Criteria
| Clinical | Laboratory |
|-----------|------------|
| **Vascular Thrombosis** (Arterial, Venous, Small vessel) | **LA+** OR **aCL IgG/IgM >40 GPL/MPL** OR **Anti-β2-GPI IgG/IgM >40 SGU/SSM** (2 occasions ≥12w) |
| **Pregnancy Morbidity** | Same laboratory criteria |

## 🧩 Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **"ANA Positive = SLE"** | **NO.** ANA positive in 5-20% healthy, Sjögren's, SSc, MCTD, RA, Drug-induced, Infections, Malignancy. |
| **"ANA Negative = No SLE"** | **NO.** 2-5% SLE are ANA-negative (Anti-dsDNA rogue, Technical failure, Early disease). |
| **"c-ANCA = GPA 100%"** | **NO.** 90% specificity; 10% false + (Drugs, CF, Other); 5-10% GPA are ANCA-negative. |
| **"p-ANCA = MPA Only"** | **NO.** p-ANCA = MPO-ANCA = MPA (70%), **EGPA (40%)**, Drug-induced, UC, PSC, AIH. |
| **"RF = RA"** | **NO.** RF in Sjögren's, SLE, Cryoglobulinaemia, Chronic Infections, Healthy Elderly (5-10%). |
| **"Anti-CCP = RA Only"** | **NO.** **High specificity (>95%)**, but also in **Pre-RA**, **PsA, SSc, CVID**; High titre = Erosive RA. |
| **"Low C3 + Low C4 = Only SLE"** | **NO.** Also **Cryoglobulinaemia, Post-Streptococcal GN, Infective Endocarditis, Subacute Bacterial Endocarditis**. |
| **"p-ANCA = MPA Only"** | **NO.** **EGPA (40%)**, Drug-induced, UC, PSC, AIH, Atypical p-ANCA in IBD. |
| **"ANA Negative = No Autoimmune Disease"** | **NO.** Seronegative SLE (2-5%), **Seronegative RA**, **ANCA Vasculitis**, **Adult Stills**, **IBD**. |
| **"ENA Panel = All Autoantibodies"** | **NO.** Standard ENA = 10-15 antigens; Missing: **dsDNA (Separate), ANCA (Separate), aPL (Separate), RF (Separate), HLA (Separate)**. |

> **Mnemonic: AUTOIMMUNE DIAGNOSTICS**  
> **A**NA Patterns: **Homogeneous (dsDNA/SLE), Speckled (ENA), Centromere (Ltd SSc), Nucleolar (SSc), Cytoplasmic (Jo-1)**  
> **U**niversal Screening: **ANA** — High Sensitivity, Low Specificity → **If +ve → ENA Panel**  
> **T**argeted ENA: **Sm (SLE), RNP (MCTD), Ro/La (Sjögren), Jo-1 (Antisynthetase), Scl-70 (SSc), Centromere (Ltd SSc), RNA Pol III (SSc+SRC)**  
> **O**NCogene vs TSG: **Not relevant here** (Just mnemonic filler)  
> **I**mmunoglobulins: **RF (IgM Anti-IgG), Anti-CCP (Citrullinated Peptide)**  
> **M**imicry & **E**pitope Spreading: **Diagnosis Clues**  
> **M**HC: **HLA-B27 (AS), DR4 (RA), DQ2/8 (Celiac), DR3/4 (T1D), B51 (Behcet)**  
> **U**nusual ANCA: **c-ANCA/PR3=GPA, p-ANCA/MPO=MPA/EGPA, Atypical=UC/PSC/AIH**

1️⃣ HPV Natural History & Carcinogenesis
Infection Outcomes
OutcomeProbabilityTimeline

Carcinogenesis Pathway (High-Risk HPV)
StepMolecular Mechanism
---------------------------
**1. Viral Entry**HPV Infects Basal Keratinocytes (Microtrauma)
**2. Viral Replication****Episomal** (Productive) in Differentiating Epithelium
**3. Immune Evasion****E6/E7 Oncoproteins** Downregulate HLA, Interferon Response
**4. Cell Cycle Dysregulation****E7 → pRb Degradation** → E2F Release → Uncontrolled S-Phase Entry
**5. Apoptosis Block****E6 → p53 Degradation** → Failed DNA Repair/Apoptosis
**6. Genomic Instability**Chromosomal Aberrations, Integration into Host Genome
**7. Clonal Expansion**Integrated HPV (E6/E7 Overexpression) → CIN → Cancer
Key: E6 → p53 Degradation (Apoptosis Block); E7 → pRb Degradation (Cell Cycle Dysregulation)

AspectImpact

-----------------

  • Screening: HPV Primary → Negative = Routine Recall; HPV 16/18 Positive = Colposcopy; Other HR HPV + Cytology → Abnormal = Colposcopy, Normal = Repeat 12 Months
  • CIN Management: CIN1 = Observe (12-Month HPV Test); CIN2 = Excision (LLETZ) Preferred; CIN3 = Excision Mandatory; Test of Cure at 6 Months (HPV Test)
  • Anogenital Warts: HPV 6, 11; Patient-Applied (Podophyllotoxin, Imiquimod); Provider (Cryotherapy, TCA, Surgical); Pregnancy: Cryo/TCA/Surgical Only
  • Non-Cervical Cancers: Anal (90% HPV, Annual Screen MSM/HIV), Oropharyngeal (70% HPV+, 16 Dominant, No Screen), Vulvar/Vaginal/Penile (40-70%)
  • Special Populations: HIV/Immunocompromised = 3-Dose Vaccine, Annual Screen, More Extensive Disease; Pregnancy = Defer Vaccine, Cryo/TCA/Surgical for Warts Only

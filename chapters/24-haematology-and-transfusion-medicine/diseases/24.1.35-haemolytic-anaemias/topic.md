# Haemolytic Anaemias: Classification, Diagnosis & Management

## Learning Objectives
- [ ] Classify haemolytic anaemias (Immune vs Non-immune, Intra/Extravascular)
- [ ] Apply diagnostic criteria (Reticulocytosis, LDH, Bilirubin, Haptoglobin)
- [ ] Differentiate immune vs non-immune causes
- [ ] Apply disease-specific management (AIHA, Thalassaemia, Sickle Cell, G6PD, PNH)
- [ ] Identify FCPS/MRCP high-yield diagnostic and management pearls

---

## Definition & Pathophysiology

| Feature | Detail |
|---------|--------|
| **Definition** | Anaemia due to **reduced RBC survival** (<120 days) |
| **Mechanism** | **Extravascular** (Macrophage phagocytosis in spleen/liver) vs **Intravascular** (RBC lysis in circulation) |
| **Key Principle** | **Haemolysis = ↓ RBC Survival** → ↑ Erythropoiesis (Reticulocytosis) IF Marrow Compensates |

---

## Classification

```mermaid
flowchart TD
    A[Haemolytic Anaemia] --> B{Mechanism}
    B --> C[Intravascular]
    B --> D[Extravascular]
    C --> C1[DIC, TTP/HUS, HUS]
    C --> C2[Marchiafava-Micheli (PNH)]
    C --> C3[Mechanical (Mechanical Heart Valves, DIC)]
    C --> C4[Severe Burns, Clostridium]
    D --> D1[Immune: AIHA (Warm, Cold, Mixed)]
    D --> D2[Non-immune: Membrane (HS, Elliptocytosis), Enzyme (G6PD, PK), Haemoglobinopathies (Sickle, Thalassaemia), Enzyme (PK)]
```

| Feature | **Intravascular** | **Extravascular** |
|---------|-------------------|-------------------|
| **Site** | Bloodstream | Spleen / Liver / Bone Marrow |
| **Mechanism** | Direct RBC Lysis | Macrophage Phagocytosis |
| **Haemoglobinuria** | **YES** (Haemoglobinuria) | No |
| **Haptoglobin** | **Markedly Low/Absent** | Low/Normal |
| **Haemopexin** | Low | Normal/Low |
| **LDH** | **Markedly Elevated** | Elevated |
| **Bilirubin** | Unconjugated ↑ | Unconjugated ↑ |
| **Reticulocytes** | ↑↑ | ↑ |
| **Urine Haemosiderin** | **Present (Chronic)** | Absent |
| **Urobilinogen** | ↑↑ | ↑ |

> **FCPS/MRCP**: **Intravascular = Haemoglobinuria + Haemopexin Low**; **Extravascular = Splenomegaly, No Haemoglobinuria**.

## Investigations: The "Haemolysis Panel"

| Test | Intravascular | Extravascular | Normal |
|------|---------------|---------------|--------|
| **Reticulocytes** | ↑↑ | ↑ | 0.5-2.5% |
| **LDH** | **↑↑↑** | ↑ | Normal |
| **Bilirubin** | Unconjugated ↑↑ | Unconjugated ↑ | Normal |
| **Haptoglobin** | **Absent/Markedly Low** | Low | Normal |
| **Haemopexin** | **Low** | Normal | Normal |
| **Urine Haemoglobin** | **Positive** | Negative | Negative |
| **Urine Haemosiderin** | **Positive (Chronic)** | Negative | Negative |
| **Urobilinogen** | ↑↑ | ↑ | Normal |

> **FCPS/MRCP**: **Reticulocytosis + High LDH + Low Haptoglobin = Haemolysis** (Triad).

## Key Differential: Intravascular vs Extravascular

| Feature | **Intravascular** | **Extravascular** |
|--------|-------------------|-------------------|
| **Haemoglobinuria** | **YES** | NO |
| **Haemopexin** | **Low** | Normal |
| **Haptoglobin** | **Absent** | Low |
| **Urine Haemosiderin** | **Present** | Absent |
| **LDH** | **Very High** | High |
| **Common Causes** | PNH, TTP/HUS, DIC, Mechanical, Burns | AIHA, HS, G6PD, Sickle, Thalassaemia |

## Immune Haemolytic Anaemia (AIHA)

### Classification

| Type | Antibody | Thermal Amplitude | Key Features |
|------|----------|-------------------|--------------|
| **Warm AIHA** | **IgG** | 37°C | **Most Common (70%)**; Rh Ab; DAT IgG+ ± C3 |
| **Cold AIHA** | **IgM** | 0-4°C | Acrocyanosis, Raynaud; DAT C3+ Only |
| **Mixed AIHA** | IgG + IgM | Both | Both Warm & Cold Features |
| **Paroxysmal Cold Haemoglobinuria (PCH)** | **Donath-Landsteiner (IgG)** | Biphasic | Post-infectious (Syphilis, Viral); Children |

### Diagnosis (DAT = Direct Antiglobulin Test / Coombs Test)

| Type | DAT Pattern |
|------|-------------|
| **Warm AIHA** | **IgG +/- C3** |
| **Cold AIHA** | **C3 Only** |
| **CAD** | **C3 Only** (High Titre Cold Agglutinin) |
| **PCH** | **C3 Only** (Donath-Landsteiner IgG) |

### Management
| Type | First-Line | Second-Line |
|------|-----------|-------------|
| **Warm AIHA** | **Prednisolone 1-1.5 mg/kg** | Rituximab, Splenectomy, Azathioprine, MMF |
| **Cold AIHA (CAD)** | **Avoid Cold**; Rituximab ± Bendamustine | Plasma Exchange (Acute) |
| **PCH** | Supportive, Warmth | Usually Self-Limiting (Post-Infectious) |

## Non-Immune Haemolytic Anaemias

### 1. Hereditary Spherocytosis (HS)
| Feature | Detail |
|---------|--------|
| **Inheritance** | **Autosomal Dominant** (75%); Recessive (25%) |
| **Defect** | **Membrane Skeleton** (Spectrin, Ankyrin, Band 3, Protein 4.2) |
| **Smear** | **Spherocytes** (Microcytic, Hyperchromatic, No Central Pallor) |
| **Labs** | ↑ MCHC, ↑ Retic, ↑ Bilirubin, ↑ LDH, Low Haptoglobin |
| **Tests** | **Eosin-5-Maleimide (EMA) Binding ↓**; Osmotic Fragility ↑ |
| **Complications** | **Cholelithiasis (Pigment Stones)**, Aplastic Crisis (Parvovirus B19) |
| **Management** | Folic Acid; **Splenectomy** (If Severe/Transfusion Dependent) → ↑ Infection Risk (Encapsulated) |

### 2. G6PD Deficiency
| Feature | Detail |
|---------|--------|
| **Inheritance** | **X-Linked Recessive** (Males Affected, Females Carriers) |
| **Enzyme** | **Glucose-6-Phosphate Dehydrogenase** (PPP NADPH Production) |
| **Trigger** | **Oxidative Stress**: Fava Beans, Sulfa Drugs, Antimalarials (Primaquine), Naphthalene, Dapsone |
| **Presentation** | **Acute Haemolytic Crisis** (Back Pain, Dark Urine, Jaundice) |
| **Smear** | **Heinz Bodies** (Supravital Staining), Bite Cells, Blister Cells |
| **Diagnosis** | **G6PD Enzyme Assay** (Normal during Acute Crisis — Test After Recovery) |
| **Management** | **Avoid Triggers**; Folic Acid; Supportive (Transfusion if Severe) |

### 3. Paroxysmal Nocturnal Haemoglobinuria (PNH)
| Feature | Detail |
|---------|--------|
| **Defect** | **PIGA Mutation** → Deficient GPI Anchor → ↓ CD55 (DAF), CD59 (MIRL) |
| **Mechanism** | Uncontrolled Complement Activation → **Intravascular Haemolysis** |
| **Triad** | **Haemolytic Anaemia + Thrombosis (Budd-Chiari) + Cytopenias** |
| **Nocturnal Haemoglobinuria** | **Morning Haemoglobinuria** (Complement Activity ↑ at Night) |
| **Diagnosis** | **Flow Cytometry (FLAER/CD55/CD59 Deficiency)** |
| **Thrombosis Risk** | **Venous > Arterial**; Budd-Chiari (Hepatic Vein), Cerebral Vein |
| **Management** | **Eculizumab** (Anti-C5) → ↓ Haemolysis, ↓ Thrombosis; **Ravulizumab** (Long-Acting C5 Inhibitor); **Allogeneic SCT** (Curative) |

### 4. Microangiopathic Haemolytic Anaemia (MAHA)
| Condition | Key Features | Management |
|-----------|--------------|------------|
| **TTP** | **ADAMTS13 <10%**, Pentad (Fever, Thrombocytopenia, MAHA, Neuro, Renal) | **Plasma Exchange (Daily) + Rituximab** |
| **HUS** | **Shiga Toxin (E. coli O157:H7)**, Renal Failure Dominant | Supportive, Eculizumab (Atypical) |
| **DIC** | Consumptive Coagulopathy, ↑ D-Dimer, ↓ Fibrinogen | Treat Underlying Cause |
| **HELLP** | Pregnancy, Haemolysis, Elevated Liver Enzymes, Low Platelets | Delivery |
| **Mechanical** | Prosthetic Valves, LVAD, Burns | Anticoagulation, Valve Replacement |

## Haemoglobinopathies

### Sickle Cell Disease (SCD)
| Feature | Detail |
|---------|--------|
| **Genetics** | **Autosomal Recessive** (β-globin Glu6Val) |
| **HbS Polymerisation** | Deoxygenation → Polymerisation → Sickling |
| **Crises** | **Vaso-occlusive (Pain)**, Acute Chest Syndrome, Stroke, Priapism, Splenic Sequestration |
| **Diagnosis** | **Hb Electrophoresis (HbS)**, HPLC, Sickle Solubility Test |
| **Management** | **Hydroxyurea** (↑ HbF), Voxelotor (↑ Hb Affinity), Crizanlizumab (P-Selectin Inhibitor), Penicillin Prophylaxis (Children), Folic Acid, **Transfusion** (Stroke Prevention, Acute Chest), **L-Glutamine** |
| **Monitoring** | TCD (Stroke Screening), Annual Organ Screening |

### Thalassaemia
| Type | Genetics | Hb Pattern | Management |
|------|----------|------------|------------|
| **Beta Thal Major** | β⁰/β⁰ | HbF >90%, No HbA | Lifelong Transfusion + Iron Chelation |
| **Beta Thal Intermedia** | β⁺/β⁰ or β⁺/β⁺ | Variable | Transfusions Intermittent, Chelation |
| **Beta Thal Minor** | β/β⁰ or β/β⁺ | HbA2 >3.5%, Microcytic | Genetic Counselling |
| **Alpha Thal** | α-Globin Deletions | HbH (3α), Bart's (4α) | HbH: Transfusion/Supplement; Bart's: Lethal |

## Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| Intravascular vs Extravascular | **IV: Haemoglobinuria, Haemopexin Low, Bite Cells**; **EV: Splenomegaly, Spherocytes** |
| AIHA Warm vs Cold | **Warm = IgG (DAT IgG+)**; **Cold = IgM (DAT C3+)** |
| G6PD vs Pyruvate Kinase | G6PD: Oxidative Stress → Heinz Bodies; PK: ATP Deficit → Echinocytes |
| PNH vs TTP | PNH: CD55/59 Deficient, Chronic Intravascular; TTP: ADAMTS13<10%, Acute MAHA + Thrombocytopenia + Neuro |
| TTP vs HUS | TTP: ADAMTS13<10%, Neuro Prominent; HUS: Shiga Toxin, Renal Dominant |
| Sickle Cell Crisis Types | VOC (Pain), ACS (Chest Syndrome), Stroke, Priapism, Splenic Sequestration |
| Thalassaemia Major vs Minor | Major: Transfusion Dependent; Minor: Carrier, Microcytic, HbA2 >3.5% |
| AIHA vs DIC | AIHA: Isolated Haemolysis; DIC: Coagulopathy + Fibrinolysis |

# Disseminated Intravascular Coagulation (DIC)

> [!info] **Davidson Ch 25 Alignment**: Bleeding and Thrombotic Disorders → DIC
> **FCPS/MRCP Focus**: ISTH overt DIC score, sepsis/trauma/obstetric triggers, replacement therapy (FFP/cryo/platelets), NO therapeutic heparin, treat underlying cause

---

## 🎯 Learning Objectives

- [ ] Define DIC: **Acquired syndrome** of **systemic intravascular coagulation activation** → thrombin generation → fibrin formation → **consumption of platelets/coagulation factors** + **fibrinolysis** → bleeding ± thrombosis
- [ ] Apply **ISTH Overt DIC Score**: Platelets, Fibrinogen, PT, D-dimer/FDP → **≥5 = Overt DIC**
- [ ] Identify **triggers**: Sepsis (most common), Trauma, Obstetric (abruption, AFE, retained products), Malignancy, Transplant, Vascular anomalies
- [ ] Differentiate **Overt DIC** (bleeding) from **Non-overt/Sepsis-induced coagulopathy (SIC)** (organ dysfunction, score <5)
- [ ] Manage: **TREAT UNDERLYING CAUSE** (antibiotics, delivery, surgery); **Replacement** if bleeding/invasive procedure; **NO routine heparin prophylaxis**

---

## 📖 Definition & Pathophysiology

```mermaid
flowchart TD
    A[Trigger: Sepsis, Trauma, Obstetric, Cancer] --> B[Tissue Factor (TF) Exposure]
    B --> C[Extrinsic Pathway Activation]
    C --> D[Thrombin Generation]
    D --> E1[Fibrin Formation → Microthrombi]
    D --> E2[Platelet Activation & Consumption]
    D --> E3[Coagulation Factor Consumption]
    E1 & E2 & E3 --> F[**Consumptive Coagulopathy**]
    F --> G[**Bleeding** (Factor/Platelet Deficiency)]
    F --> H[**Microvascular Thrombosis** → Organ Dysfunction]
    H --> I[Fibrinolysis Activation (tPA Release)]
    I --> J[**D-dimer / FDP Elevation**]
```

| Type | Mechanism | Clinical |
|------|-----------|----------|
| **Acute DIC** | Sudden massive TF exposure (sepsis, trauma, obstetric) | **Bleeding dominant**, rapid onset |
| **Chronic DIC** | Low-grade continuous TF (solid tumours, vascular anomalies) | **Thrombosis dominant**, compensated |

## 🔬 Diagnostic Workup – ISTH Overt DIC Score

| Parameter | Score |
|-----------|-------|
| **Platelet Count** | >100 = 0; 50-100 = 1; <50 = 2 |
| **Fibrinogen (g/L)** | >1.0 = 0; <1.0 = 1 |
| **PT (sec prolonged)** | <3 = 0; 3-6 = 1; >6 = 2 |
| **D-dimer / FDP** | No increase = 0; Moderate = 2; Strong = 3 |

**Interpretation**: **≥5 = Overt DIC**; **<5 = Non-overt (SIC) – monitor**

> [!tip] **FCPS/MRCP**: **ISTH Score ≥5 = Overt DIC**. **Order: Platelets, Fibrinogen, PT, D-dimer**. **DIC = Consumption (↓Plt, ↓Fib, ↑PT, ↑D-dimer)**. **Treat underlying cause FIRST**.

## 🩺 Common Triggers

| Category | Examples |
|----------|----------|
| **Sepsis** | **Gram-negative (E. coli, Pseudomonas), Gram-positive, Meningococcal, Fungal** – most common |
| **Obstetric** | **Abruptio placentae, Amniotic fluid embolism, Retained dead fetus, HELLP, Postpartum haemorrhage** |
| **Trauma/Burns** | Major trauma, crush injury, severe burns, neurotrauma |
| **Malignancy** | **Acute promyelocytic leukaemia (APML)**, Adenocarcinomas (pancreas, lung, prostate), Metastatic disease |
| **Vascular** | Giant haemangioma (Kasabach-Merritt), Aortic aneurysm |
| **Transplant** | Solid organ, HSCT (engraftment syndrome, VOD/SOS) |
| **Other** | Heat stroke, Snake bite, Severe liver disease, Haemolysis |

## 💊 Management

### 1. Treat the Underlying Cause – **MOST IMPORTANT**

| Trigger | Specific Treatment |
|---------|-------------------|
| **Sepsis** | **Broad-spectrum antibiotics + Source control** |
| **Obstetric** | **Delivery** (abruption), **Uterine evacuation** (retained products), **Supportive for AFE** |
| **APML** | **ATRA + ATO** (reverses DIC rapidly) |
| **Malignancy** | **Chemotherapy / Tumour resection** |
| **Trauma** | **Surgical control, Massive transfusion protocol** |

### 2. Replacement Therapy – **Only if Bleeding or Invasive Procedure**

| Component | Threshold for Replacement | Dose |
|-----------|---------------------------|------|
| **Platelets** | **<50** (prophylactic); **<100** (CNS bleed/surgery) | 1 ATD / 10 kg |
| **Fibrinogen** | **<1.5 g/L** (bleeding); **<1.0** (ISTH score) | **Cryoprecipitate 1 pool/10 kg** or **Fibrinogen concentrate 25-50 mg/kg** |
| **Coagulation Factors** | **PT >1.5x normal** (bleeding/surgery) | **FFP 15-20 mL/kg** |
| **RBCs** | Hb <7-8 (symptomatic) | Standard |

> [!warning] **NO ROUTINE PROPHYLACTIC REPLACEMENT** in non-bleeding DIC. **Replacement only for active bleeding or before invasive procedures**.

### 3. Anticoagulation – **CONTROVERSIAL / GENERALLY AVOIDED**

| Scenario | Recommendation |
|----------|----------------|
| **Overt DIC with bleeding** | **NO Heparin** |
| **Overt DIC with thrombosis (purpura fulminans, VTE)** | **Therapeutic anticoagulation (LMWH/UFH) + Replacement** |
| **Chronic DIC / Thrombosis-predominant** | **Consider LMWH** (if no bleeding) |
| **Purpura Fulminans (Meningococcal)** | **Protein C concentrate + Heparin + FFP** |

> [!warning] **DIC ≠ Heparin prophylaxis**. **Therapeutic heparin ONLY if thrombosis dominates (purpura fulminans, VTE) AND no active bleeding**.

### 4. Specific Scenarios

| Scenario | Key Points |
|----------|------------|
| **APML** | **ATRA + ATO reverses DIC**; Monitor fibrinogen, platelets; **Avoid heparin** |
| **Obstetric DIC** | **Delivery is definitive**; Replace fibrinogen early (<2 g/L); **Cell salvage** if massive |
| **Purpura Fulminans** | **Protein C concentrate** (if available) + **Heparin + FFP**; Limb ischaemia → fasciotomy |

## 🔄 Differential Diagnosis

| Condition | Distinguishing Features |
|-----------|------------------------|
| **TTP** | **ADAMTS13 <10%, Schistocytes, Normal Fibrinogen/PT**, No underlying sepsis/trauma |
| **HIT** | **Heparin 5-14d, PF4/heparin Ab+, Thrombosis**, Normal PT/Fibrinogen |
| **Liver Disease** | **Prolonged PT, Low Fibrinogen**, **Factor V LOW** (DIC = FV normal), No D-dimer spike |
| **Vitamin K Deficiency** | **PT ↑↑, PTT ↑, Normal Fibrinogen/Platelets/D-dimer**, Responds to Vit K |
| **Massive Transfusion** | **Dilutional coagulopathy**; Improves with replacement; No D-dimer spike initially |
| **Anticoagulant Overdose** | **History**, Specific factor pattern (Warfarin = II,VII,IX,X; DOAC = specific assays) |

## ❓ Viva Questions

1. **What is the ISTH Overt DIC scoring system?**
   - Platelets (>100=0, 50-100=1, <50=2), Fibrinogen (>1=0, <1=1), PT (<3s=0, 3-6=1, >6=2), D-dimer (none=0, moderate=2, strong=3). **≥5 = Overt DIC**

2. **What is the most common cause of DIC?**
   - **Sepsis** (especially Gram-negative)

3. **How do you manage DIC?**
   - **1. Treat underlying cause** (antibiotics, delivery, ATRA for APML); **2. Replace factors/platelets ONLY if active bleeding or invasive procedure**; **3. Avoid routine heparin**

4. **What are the replacement thresholds in DIC?**
   - **Platelets <50** (prophylactic), **<100** (CNS/surgery); **Fibrinogen <1.5 g/L** (bleeding); **PT >1.5x normal** → FFP

5. **When is heparin indicated in DIC?**
   - **Thrombosis-dominant DIC** (purpura fulminans, VTE) **WITHOUT active bleeding**; **Protein C concentrate + Heparin for purpura fulminans**

6. **How does DIC differ from TTP?**
   - **DIC: PT↑, Fibrinogen↓, D-dimer↑↑, underlying trigger**; **TTP: ADAMTS13<10%, normal PT/Fibrinogen, schistocytes**

7. **How does DIC differ from liver disease coagulopathy?**
   - **Liver: Factor V LOW**; **DIC: Factor V NORMAL** (consumed equally but liver can't synthesise)

8. **What is the obstetric management of DIC?**
   - **Delivery is definitive** (abruption); **Replace fibrinogen early (<2 g/L)**; **Massive transfusion protocol**; **Cell salvage**

9. **How does APML-associated DIC differ?**
   - **Rapidly reversible with ATRA + ATO**; **Monitor fibrinogen closely**; **Avoid heparin**

10. **What is purpura fulminans and its treatment?**
    - **Severe thrombotic DIC** (meningococcal); **Protein C concentrate + Heparin + FFP**; Limb ischaemia → **fasciotomy**

## 🧠 Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **DIC vs TTP** | **DIC = PT↑, Fib↓, D-dimer↑↑, trigger**; **TTP = ADAMTS13<10%, normal PT/Fib** |
| **DIC vs Liver Disease** | **Liver = Factor V LOW**; **DIC = Factor V NORMAL** |
| **DIC vs HIT** | **HIT = Heparin + PF4 Ab + thrombosis**; **DIC = Trigger + consumption** |
| **Heparin in DIC** | **NO routine prophylaxis**; **Only therapeutic if thrombosis dominant + no bleeding** |
| **Replacement** | **Only if BLEEDING or PROCEDURE**; NOT prophylactic for lab numbers |

| Mnemonic | Meaning |
|----------|---------|
| **"DIC = PT↑, Plt↓, Fib↓, D-dimer↑↑"** | Lab pattern |
| **"ISTH ≥5 = Overt DIC"** | Score threshold |
| **"Treat Cause First"** | Management priority |
| **"Bleed = Replace; No Bleed = Watch"** | Replacement rule |
| **"APML = ATRA Fixes DIC"** | APML-specific |
| **"Liver = Low Factor V"** | Differentiation |
| **"Purpura Fulminans = Protein C + Heparin"** | Specific treatment |

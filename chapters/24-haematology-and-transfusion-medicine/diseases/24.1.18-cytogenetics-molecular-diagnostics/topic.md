## 📖 Cytogenetics

### Karyotyping

| Aspect | Details |
|--------|---------|
| **Sample** | **Bone Marrow (Heparin)** or **Peripheral Blood** (if high blast count) |
| **Culture** | **24-48h** (Unstimulated) + **PHA Stimulation** (72h) for Lymphocytes |
| **Banding** | **G-banding (Trypsin/Giemsa)** - Standard; **Q/R-banding** (Alternative) |
| **Resolution** | **400-550 Bands** (Standard); **High-resolution (850+)** for Subtle Abnormalities |
| **Analysis** | **20 Metaphases** (Standard); **50** (If Mosaicism Suspected) |
| **Reporting** | **ISCN 2020** Nomenclature |

### Chromosome Nomenclature (ISCN 2020)

| Notation | Meaning | Example |
|----------|---------|---------|
| **p / q** | Short arm / Long arm | 7q- |
| **t(A;B)(p1;q2)** | **Translocation** between Chr A & B | t(9;22)(q34;q11) |
| **inv(A)(p1q2)** | **Inversion** within Chromosome A | inv(16)(p13.1q22) |
| **del(A)(q1q2)** | **Deletion** | del(5)(q13q33) / del(5q) |
| **dup(A)(q1q2)** | **Duplication** | dup(1q) |
| **ins(A;B)(p1;q2q3)** | **Insertion** | ins(3;21)(q21;q22q22) |
| **+A / -A** | **Gain / Loss of Whole Chromosome** | +8, -7 |
| **i(A)(q1)** / **dic(A;B)** | **Isochromosome / Dicentric** | i(17q), dic(9;20) |
| **/ ** | **Mosaicism** / **Cell Line** | 46,XX/47,XX,+8 |

### Key Recurrent Abnormalities

| Abnormality | Disease | Significance |
|-------------|---------|--------------|
| **t(9;22)(q34;q11)** | **CML, ALL** | **BCR::ABL1**, TKI target |
| **t(15;17)(q24;q21)** | **APML (AML-M3)** | **PML::RARA**, ATRA/ATO |
| **t(8;21)(q22;q22)** | **AML** | **RUNX1::RUNX1T1**, Favourable |
| **inv(16)(p13.1q22)** | **AML** | **CBFB::MYH11**, Favourable |
| **t(11;14)(q13;q32)** | **MCL** | **CCND1::IGH**, Cyclin D1 |
| **t(14;18)(q32;q21)** | **FL** | **IGH::BCL2**, BCL2 overexpression |
| **t(11;18)** | **MALT** | **API2::MALT1** |
| **-5 / del(5q)** | **MDS, AML** | Poor Prognosis |
| **-7 / del(7q)** | **MDS, AML** | Poor Prognosis |
| **Complex Karyotype (≥3)** | **MDS, AML** | **Adverse (ELN 2022)** |
| **Monosomal Karyotype** | **MDS, AML** | **Adverse** |

## 🔬 FISH (Fluorescence In Situ Hybridisation)

### Probe Types

| Probe Type | Design | Application |
|------------|--------|-------------|
| **Dual-Fusion** | **Two Fusion Signals** (Red+Green) | **BCR::ABL1 (t(9;22))**, **PML::RARA (t(15;17))**, **RUNX1::RUNX1T1 (t(8;21))** |
| **Break-Apart** | **Split Signals** (Red+Green Separate) | **KMT2A (11q23)**, **ALK**, **ETV6**, **MYC** |
| **Deletion** | **Loss of Signal** | **17p/TP53**, **13q14 (RB1)**, **20q**, **5q**, **7q** |
| **Enumeration / Copy Number** | **Count Signals** | **+8**, **HER2 Amplification**, **MYC Amplification** |

### FISH vs Karyotype

| Feature | **FISH** | **Karyotype** |
|---------|----------|---------------|
| **Resolution** | **100 kb - 1 Mb** | **5-10 Mb** |
| **Throughput** | **200-500 cells** | **20-50 Metaphases** |
| **Targeted** | **Yes (Specific Probe)** | **Genome-wide** |
| **Detection** | **Interphase + Metaphase** | **Metaphase Only** |
| **Sensitivity** | **1-5%** | **5-10%** |
| **Turnaround** | **24-48h** | **7-14 Days** |

> [!tip] **FISH = Rapid, Targeted, Higher Sensitivity**; **Karyotype = Genome-wide, Lower Resolution, Detects Novel/Complex Changes**.

## 🧬 Molecular Diagnostics

### PCR-based Methods

| Method | Application | Sensitivity |
|--------|-------------|-------------|
| **Qualitative RT-PCR** | **Fusion Gene Detection** (BCR::ABL1, PML::RARA, CBFB::MYH11) | **10⁻⁴ - 10⁻⁵** |
| **Quantitative RT-PCR (qRT-PCR)** | **MRD** (BCR::ABL1 IS, NPM1, CBFB::MYH11) | **10⁻⁴ - 10⁻⁶** |
| **Digital PCR (dPCR)** | **Absolute Quantification** (No Standard Curve) | **10⁻⁵ - 10⁻⁶** |
| **Allele-Specific PCR** | **Hotspot Mutations** (FLT3-ITD, NPM1, JAK2 V617F) | **10⁻³ - 10⁻⁴** |
| **ASO-PCR (Allele-Specific Oligonucleotide)** | **Patient-specific MRD** (IGH/TCR Rearrangements) | **10⁻⁴ - 10⁻⁶** |

### Next-Generation Sequencing (NGS)

| Approach | Application | Advantages |
|----------|-------------|------------|
| **Targeted Panel (50-100 genes)** | **Diagnosis, Prognosis, Therapy Selection** (AML, MDS, MPN, Lymphoma) | **Cost-effective**, **Deep Coverage (500-1000x)**, **Fast (3-5 days)** |
| **Whole Exome (WES)** | **Research, Novel Drivers** | **All Coding Regions**, **Novel Variants** |
| **Whole Genome (WGS)** | **Research, Structural Variants** | **Complete Genome**, **SV Detection**, **Non-coding** |

### NGS Variant Interpretation (ACMG/AMP 2015)

| Category | Criteria |
|----------|----------|
| **Pathogenic** | **PVS1 + PS1-4 + PM1-6 + PP1-5** (≥1 Very Strong + ≥1 Strong OR ≥2 Strong) |
| **Likely Pathogenic** | Strong + Moderate OR ≥3 Moderate |
| **VUS (Uncertain Significance)** | Insufficient Evidence |
| **Likely Benign** | Benign Supporting ≥1 |
| **Benign** | BA1 (Population Freq >5%) OR BS1-4 |

### Key Mutations by Disease

| Disease | Key Mutations | Clinical Utility |
|---------|---------------|------------------|
| **AML** | **FLT3-ITD/TKD, NPM1, CEBPA, IDH1/2, DNMT3A, TP53, ASXL1, RUNX1, SRSF2, U2AF1, KIT** | **Prognosis (ELN 2022)**, **Targeted Therapy (Midostaurin, Gilteritinib, Ivosidenib, Enasidenib, Venetoclax)** |
| **MDS** | **SF3B1, SRSF2, U2AF1, ZRSR2, ASXL1, RUNX1, TP53, DNMT3A, TET2, EZH2** | **Prognosis (IPSS-M)**, **Luspatercept (SF3B1)** |
| **MPN** | **JAK2 V617F, CALR Type 1/2, MPL, TET2, ASXL1, SRSF2, IDH1/2** | **Diagnosis, Prognosis (MIPSS70), Ruxolitinib (JAK2)** |
| **ALL** | **ETV6::RUNX1, BCR::ABL1, KMT2A-r, TCF3::PBX1, Ph-like (CRLF2, JAK2, EPOR)** | **Risk Stratification, TKI (Ph+)**, **Targeted (Dasatinib, Ruxolitinib)** |
| **CLL** | **TP53, ATM, NOTCH1, SF3B1, BIRC3, MYD88, IGHV Status** | **Prognosis, Treatment Choice (BTKi vs Venetoclax)** |
| **DLBCL** | **MYC/BCL2/BCL6 (Double/Triple Hit), EZH2, TP53, BCL2, CARD11, MYD88** | **DH/TH Identification**, **EZH2mut → Tazemetostat** |

## 📊 Integration: WHO/ICC Classification + Prognostic Systems

### AML (ELN 2022 Risk Stratification)

| Risk | Genetics |
|------|----------|
| **Favourable** | **t(8;21), inv(16), NPM1mut (no FLT3-ITD or low AR), CEBPAbiallelic** |
| **Intermediate** | **NPM1mut + FLT3-ITDhigh**, **Normal Karyotype**, **Other** |
| **Adverse** | **TP53mut, Complex Karyotype, Monosomal Karyotype, -5/del5q, -7/del7q, inv(3)/t(3;3), KMT2A-r (non-t(9;11)), FLT3-ITDhigh + Adverse Cytogenetics** |

### MDS (IPSS-R / IPSS-M)

| IPSS-R Variable | Scoring |
|-----------------|---------|
| **Cytogenetics** | Very Good (0) → Very Poor (4) |
| **Blasts %** | ≤2% (0), 2-<5% (0.5), 5-<10% (1), 10-<20% (2), >20% (3) |
| **Hb** | ≥10 (0), 8-<10 (1), <8 (1.5) |
| **Plt** | ≥100 (0), 50-<100 (0.5), <50 (1) |
| **ANC** | ≥0.8 (0), <0.8 (0.5) |

### Multiple Myeloma (R-ISS)

| Stage | Criteria |
|-------|----------|
| **I** | ISS I + **No High-risk FISH** (17p-, t(4;14), t(14;16)) + Normal LDH |
| **II** | Not I or III |
| **III** | **ISS III (β2M >5.5) + High-risk FISH OR High LDH** |

## 🎯 Minimal Residual Disease (MRD) by Molecular Methods

| Method | Target | Sensitivity | Standard |
|-------|--------|-------------|----------|
| **qRT-PCR (Fusion Genes)** | BCR::ABL1 (IS), PML::RARA, CBFB::MYH11, NPM1 | **10⁻⁴ - 10⁻⁶** | **International Scale (IS)** |
| **Digital PCR** | Fusion Genes, Mutations | **10⁻⁵ - 10⁻⁶** | Absolute Quantification |
| **NGS (Clonotypic)** | IGH/TCR Rearrangement, Mutations | **10⁻⁴ - 10⁻⁶** | **Clonotype Tracking** |
| **Multiparameter Flow** | LAIP | **10⁻⁴ - 10⁻⁵** | **≥1M Events** |

### MRD Clinical Utility

| Disease | MRD Application |
|---------|----------------|
| **CML** | **BCR::ABL1 IS (qRT-PCR)** → **TFR Criteria (MR4.5 sustained ≥2y)** |
| **AML** | **NPM1, CBFB::MYH11, FLT3-ITD** → **Transplant Decision** |
| **ALL** | **IGH/TCR Rearrangement, ETV6::RUNX1, BCR::ABL1** → **Intensity Adjustment** |
| **CLL** | **IGH Clonotype (NGS/dPCR)** → **Stop Therapy (uMRD)** |
| **Myeloma** | **Clonotypic NGS (10⁻⁵-10⁻⁶)** → **Sustained MRD-neg → Treatment Stop** |

## ❓ Viva Questions

1. **What is the difference between Karyotype and FISH?**
   - **Karyotype = Genome-wide, 5-10 Mb resolution, Metaphase only**; **FISH = Targeted, 100kb-1Mb, Interphase+Metaphase, Higher sensitivity**

2. **What are the common FISH probe types and their uses?**
   - **Dual-Fusion**: BCR::ABL1, PML::RARA; **Break-Apart**: KMT2A, ALK; **Deletion**: 17p, 5q, 7q; **Enumeration**: +8

3. **What is the ELN 2022 favourable risk group in AML?**
   - **t(8;21), inv(16), NPM1mut without FLT3-ITD (or low AR), CEBPAbiallelic**

4. **What is the significance of TP53 mutation in MDS/AML?**
   - **Adverse Prognosis**, **Resistant to Chemotherapy**, **IPSS-M Very High Risk**, **Venetoclax/Azacitidine Preferred**

5. **What is the International Scale (IS) for BCR::ABL1 monitoring?**
   - **Standardised qRT-PCR**; **Major Molecular Response (MMR) = BCR::ABL1 ≤0.1% IS**; **MR4.5 = ≤0.0032% (TFR threshold)**

6. **How does IPSS-M differ from IPSS-R?**
   - **IPSS-M adds 17 Molecular Mutations** (TP53, FLT3, RAS, Splicing Factors, etc.) → **Better Risk Stratification**

7. **What is the significance of TP53 mutation in MDS?**
   - **Worst Prognosis**, **IPSS-M Very High Risk**, **Multi-hit TP53 = Adverse**, **Responds poorly to HMAs, Better with Azacitidine + Eprenetapopt (Trials)**

7. **What is the role of Molecular testing in Multiple Myeloma?**
   - **R-ISS Staging**: ISS + **High-risk FISH (17p, t(4;14), t(14;16))** + **LDH**; **MRD by NGS (Clonotypic)**

8. **What is the difference between qRT-PCR and dPCR for MRD?**
   - **qRT-PCR = Relative Quant (Standard Curve)**; **dPCR = Absolute Quant (Partitioning), No Standard Curve, Higher Precision**

9. **What are the key mutations in AML with prognostic/therapeutic significance?**
   - **FLT3-ITD/TKD (Midostaurin/Gilteritinib)**, **NPM1 (Prognosis)**, **IDH1/2 (Ivosidenib/Enasidenib)**, **TP53 (Adverse)**, **CEBPA (Favourable if Biallelic)**

10. **How does NGS integrate into WHO/ICC classification?**
    - **Defining Genetic Abnormalities** (SF3B1, TP53, NPM1, CEBPA, KMT2A-r) → **Diagnostic Criteria**; **Prognostic Systems (ELN, IPSS-M, R-ISS) = Molecular + Cytogenetic + Clinical** |

## 🧠 Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Karyotype vs FISH** | **Karyotype = Genome-wide**; **FISH = Targeted, Faster, More Sensitive** |
| **FISH Probe Types** | **Dual-Fusion = Translocation (BCR::ABL1)**; **Break-Apart = Gene Split (KMT2A)**; **Deletion = Loss (17p)** |
| **ELN 2022 vs 2017** | **2022: TP53 = Adverse regardless; NPM1/FLT3-ITDlow = Favourable** |
| **IPSS-R vs IPSS-M** | **IPSS-M = IPSS-R + 17 Mutations** |
| **qRT-PCR vs dPCR** | **qRT-PCR = Relative (Std Curve)**; **dPCR = Absolute (Partitioning)** |
| **BCR::ABL1 IS** | **Standardised % (not copies)**; **MMR = ≤0.1%, MR4.5 = ≤0.0032%** |

| Mnemonic | Meaning |
|----------|---------|
| **"Karyotype = Genome Map"** | Full chromosome view |
| **"FISH = Flashlight = Targeted"** | Specific probe |
| **"ELN 2022: Fav = t(8;21), inv16, NPM1/FLT3low, CEBPAbi"** | Favourable risk |
| **"ELN 2022: Adv = TP53, Complex, Mono, -5/-7"** | Adverse risk |
| **"IPSS-M = IPSS-R + 17 Genes"** | Molecular IPSS |
| **"IS = International Scale = Standardised BCR::ABL1"** | CML monitoring |
| **"MR4.5 = 0.0032% = TFR"** | Treatment-free remission |

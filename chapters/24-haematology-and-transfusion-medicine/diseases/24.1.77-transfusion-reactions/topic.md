# Transfusion Reactions

> [!info]
> **Transfusion Reactions = Adverse Events** occurring during or after blood product transfusion. **Classification by Timing** (Acute ≤24h vs Delayed >24h) and **Mechanism** (Immune vs Non-immune). **Immediate Recognition + Stop Transfusion** = Core Management.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Classify transfusion reactions by timing and mechanism
- [ ] Recognise clinical features of each reaction type
- [ ] Apply immediate management algorithm (Stop, Support, Investigate)
- [ ] Apply reporting requirements (SHOT, local transfusion lab)

---

## 2. Classification

| Category | Reaction Types |
|----------|----------------|
| **Acute (≤24h) - Immune** | Acute Haemolytic, Anaphylactic, Febrile Non-Haemolytic (FNHTR), TRALI |
| **Acute (≤24h) - Non-Immune** | TACO, Bacterial Sepsis, Hypothermia, Citrate Toxicity, Hyperkalaemia |
| **Delayed (>24h) - Immune** | Delayed Haemolytic (DHTR), Delayed Serologic (DSTR), TBvHD |
| **Delayed (>24h) - Non-Immune** | Iron Overload, Post-Transfusion Purpura |

---

## 3. Acute Haemolytic Transfusion Reaction (AHTR)

| Feature | Details |
|---------|---------|
| **Mechanism** | **ABO Incompatibility** → Anti-A/B IgM → Complement Activation → Intravascular Haemolysis |
| **Severity** | Most Severe Reaction; Mortality ~10-20% |
| **Onset** | Minutes to Hours (During/Soon After Transfusion) |
| **Clinical** | Fever, Chills, Flank Pain, Dark Urine (Hb), Hypotension, DIC, Oliguria, Renal Failure |
| **Labs** | **↑ Plasma Hb, ↓ Haptoglobin, ↑ LDH, ↑ Bilirubin, ↑ Urobilinogen, Positive DAT**, Schistocytes |
| **Immediate Management** | **STOP TRANSFUSION IMMEDIATELY**; Maintain IV Access; Send Unit + New Sample for **Group/ABO Check**; **Hydration (3L 0.9% NaCl)**; Alkalinise Urine (If Hburia); Monitor Urine Output, U&Es, Coagulation |

---

## 3. Anaphylactic Reaction

| Feature | Details |
|---------|---------|
| **Mechanism** | **IgA Deficiency (1:5000)** + Anti-IgA IgE → Anaphylaxis on Plasma/Platelet Exposure |
| **Onset** | Minutes (Rapid) |
| **Clinical** | **Airway Obstruction** (Stridor, Angioedema), Hypotension, Bronchospasm, Urticaria, Nausea/Vomiting |
| **Immediate Management** | **STOP TRANSFUSION**; **Adrenaline 0.5mg IM (1:1000)**; Airway Support (O₂, Intubation if Needed); IV Fluids (0.9% NaCl); Chlorphenamine 10mg IV; Hydrocortisone 200mg IV |
| **Future Transfusions** | **Washed RBCs/Platelets** or **IgA-Deficient Components** |

---

## 3. Febrile Non-Haemolytic Transfusion Reaction (FNHTR)

| Feature | Details |
|---------|---------|
| **Mechanism** | **Cytokine Release** (IL-1, IL-6, TNF-α) from Leucocytes in Stored Blood; **Recipient Anti-Leucocyte Antibodies** |
| **Onset** | **1-4 Hours** (During/After Transfusion) |
| **Clinical** | **Fever ≥38°C (or 1.5°C Rise)**, Rigors, Tachycardia, No Haemolysis/Allergy |
| **Diagnosis** | **Exclusion** — Rule Out AHTR, Sepsis, TRALI |
| **Management** | **Stop Transfusion** (If Severe); Paracetamol 1g PO/IV; **Slow Transfusion Rate**; **Leucodepleted Components** (Prevention); Antipyretics Pre-Transfusion (If Recurrent) |

---

## 3. Transfusion-Associated Circulatory Overload (TACO)

| Feature | Details |
|---------|---------|
| **Mechanism** | Volume Overload → Pulmonary Oedema (Left Heart Failure) |
| **Risk Factors** | **Elderly, Cardiac Failure, Renal Failure, Positive Fluid Balance, Rapid Transfusion** |
| **Clinical** | **Acute Dyspnoea, Orthopnoea, Hypertension, Tachycardia, Bilateral Crepitations, JVP↑**, Raised BNP/NT-proBNP |
| **CXR** | Pulmonary Oedema, Cardiomegaly |
| **Management** | **STOP/Slow Transfusion**; Sit Upright; **Oxygen**; **IV Furosemide 20-40mg**; **NIV/Intubation** if Severe; **Diuretics Prophylaxis** (If High Risk) |

---

## 3. Transfusion-Related Acute Lung Injury (TRALI)

| Feature | Details |
|---------|---------|
| **Mechanism** | **Donor Anti-HLA/HNA Antibodies** → Recipient Neutrophil Activation → Pulmonary Capillary Leak |
| **Onset** | **≤6 Hours** Post-Transfusion |
| **Clinical** | **Acute Hypoxaemia**, Bilateral Pulmonary Infiltrates, **No Cardiac Failure**, Fever, Hypotension |
| **Diagnosis** | **Exclusion** — Bilateral Infiltrates + NO Left Atrial Hypertension (BNP Normal/Low); **Donor Antibody Testing** |
| **Management** | **Supportive** — O₂, NIV/Intubation; **Stop Transfusion**; **NO Diuretics** (Unlike TACO); Report to Blood Service (Donor Deferral) |
| **Mortality** | **5-10%** (Respiratory Failure) |

---

## 4. Delayed Haemolytic Transfusion Reaction (DHTR)

| Feature | Details |
|---------|---------|
| **Mechanism** | **Anamnestic Antibody Response** (IgG) to Previously Sensitised Antigen (Kidd, Duffy, Kell, Rh) |
| **Timing** | **3-14 Days** Post-Transfusion |
| **Clinical** | Fever, Falling Hb (Unexplained), Mild Jaundice, **Rarely Severe** |
| **Labs** | **↓ Hb, ↑ Bilirubin, ↑ LDH, ↓ Haptoglobin, Positive DAT**, New Alloantibody Identified |
| **Management** | Supportive; **Future: Antigen-Negative Blood**; Monitor Hb |

---

## 4. Transfusion-Associated Graft vs Host Disease (TA-GvHD)

| Feature | Details |
|---------|---------|
| **Mechanism** | **Donor T Lymphocytes** Engraft in **Immunocompromised Recipient** → Attack Host Tissues |
| **Timing** | **7-21 Days** Post-Transfusion |
| **Clinical** | **Fever, Rash (Maculopapular → Desquamation), Diarrhoea, Hepatitis, Pancytopenia (Marrow Aplasia)** |
| **Risk Groups** | **Immunocompromised** (Haematology, Chemo, HSCT, HIV, Intrauterine), **HLA-Matched Relatives** |
| **Mortality** | **>90%** (Marrow Aplasia → Infection, Haemorrhage) |
| **Prevention** | **IRRADIATED COMPONENTS** (25 Gy) — **Mandatory** for At-Risk Groups |
| **Treatment** | **Supportive Only** (No Effective Treatment) |

---

## 4. Bacterial Sepsis (Transfusion-Transmitted)

| Feature | Details |
|---------|---------|
| **Source** | **Platelets** (Stored at 22°C — Bacterial Growth); Rarely RBCs |
| **Organisms** | **Staph epidermidis, Staph aureus, Strep, Gram-negatives (E. coli, Klebsiella)** |
| **Clinical** | **High Fever, Rigors, Hypotension, Shock** (Rapid Onset During/Soon After) |
| **Management** | **STOP TRANSFUSION**; Blood Cultures (Patient + Unit); **Broad-Spectrum IV Antibiotics**; Fluid Resuscitation; Vasopressors |

---

## 4. Other Acute Reactions

| Reaction | Key Features | Management |
|----------|--------------|------------|
| **Hypothermia** | Core Temp <35°C (Massive/Rapid Cold Blood Transfusion) | Blood Warmers; Thermal Blankets |
| **Citrate Toxicity** | Hypocalcaemia (Citrate Chelates Ca²⁺) — **Massive Transfusion** | **IV Calcium Gluconate 10% 10-20mL**; Monitor Ionised Ca²⁺ |
| **Hyperkalaemia** | Stored RBCs K⁺ Leakage — **Massive/Neonatal Transfusion** | Monitor K⁺; Calcium Gluconate; Insulin/Dextrose; Dialysis |
| **Air Embolism** | Air in Giving Set → Right Heart/Arterial Embolism | **Trendelenburg + Left Lateral Decubitus**; O₂; Aspiration via CVC |

---

## 5. Delayed Reactions

| Reaction | Timing | Key Features |
|----------|--------|--------------|
| **Delayed Serologic TR (DSTR)** | Days-Weeks | New Alloantibody Detected, **No Haemolysis** |
| **Post-Transfusion Purpura (PTP)** | 5-12 Days | **Severe Thrombocytopenia** (Anti-HPA-1a); IVIG 1g/kg |
| **Iron Overload** | Months-Years | **Ferritin >1000, Transferrin Sat >50%**; Chelation (Deferasirox/Deferiprone/Deferoxamine) |
| **Transfusion-Transmitted Infection** | Months-Years | HBV, HCV, HIV, CMV, Malaria, Syphilis — **Screening + NAT** |

## 5. Immediate Management Algorithm

```
TRANSFUSION REACTION SUSPECTED
         │
         ▼
1. STOP TRANSFUSION IMMEDIATELY
         │
         ▼
2. MAINTAIN IV ACCESS (Do Not Flush Line)
         │
         ▼
3. ASSESS ABCDE
         │
         ├── ANAPHYLAXIS → Adrenaline 0.5mg IM, Airway, Fluids, Chlorphenamine, Hydrocortisone
         ├── TACO → Sit Up, O₂, Furosemide, NIV
         ├── TRALI → O₂, NIV/Intubation, NO Diuretics
         ├── SEPSIS → Blood Cultures (Patient + Unit), IV Antibiotics, Fluids
         ├── AHTR → **Hydration (3L 0.9% NaCl)**, Alkalinise Urine, Monitor Renal
         └── FNHTR → Paracetamol, Slow Rate, Leucodepleted
         │
         ▼
4. SEND SAMPLES: Group/ABO Check, FBC, U&Es, LFTs, Coag, LDH, Haptoglobin, Urine Hb, Blood Cultures
         │
         ▼
5. RETURN UNIT + GIVING SET TO TRANSFUSION LAB
         │
         ▼
6. REPORT: Local Transfusion Lab + SHOT (Serious Hazards of Transfusion)
```

## 5. Reporting Requirements (SHOT)

| Reportable Events | To Whom |
|-------------------|---------|
| **All Serious Reactions** (Death, Life-Threatening, Major Morbidity) | **SHOT** (UK National Haemovigilance) |
| **AHTR, Anaphylaxis, TRALI, TA-GvHD, Bacterial Sepsis, TACO (Severe), DHTR** | **SHOT + Hospital Transfusion Team** |
| **Near Misses** (Wrong Blood in Tube, Near Miss Transfusion) | **SHOT + Local Governance** |

---

## 5. Prevention Strategies

| Reaction | Prevention |
|----------|------------|
| **AHTR** | **Positive Patient ID**, Group/ABO Check, Electronic Crossmatch, Bedside Check |
| **Anaphylaxis** | **IgA-Deficient Components** (If Known IgA Deficient); Washed Components |
| **FNHTR** | **Leucodepletion** (Universal in UK); Antipyretics Pre-Transfusion |
| **TACO** | **Slow Rate** (1 Unit/2-3h), Diuretics Pre-Transfusion (High Risk), Avoid Overtransfusion |
| **TRALI** | **Male-Only Plasma/Platelets** (Lower Anti-HLA/HNA); Leucodepletion |
| **TA-GvHD** | **Irradiated Components (25 Gy)** — Mandatory for At-Risk |
| **Bacterial Sepsis** | **Platelet Bacterial Screening**; Diversion Pouch; Cold Storage (Research) |

## 5. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **AHTR** | **Stop Transfusion**; **Hydration 3L NaCl**; ABO Incompatibility; Mortality 10-20% |
| **Anaphylaxis** | Anti-IgA in IgA Deficient; **Adrenaline 0.5mg IM**; Washed/IgA-Deficient Blood |
| **FNHTR** | **Fever + Rigors**; Cytokines; **Leucodepletion Prevents**; Exclusion Diagnosis |
| **TACO** | **Volume Overload**; Dyspnoea, Hypertension, BNP↑; **Furosemide + O₂** |
| **TRALI** | **Donor Anti-HLA/HNA**; Hypoxaemia + Bilateral Infiltrates + **No Cardiac Failure**; **No Diuretics** |
| **TA-GvHD** | **Immunocompromised + HLA-Matched Relative**; **Irradiated Blood Prevents**; Mortality >90% |
| **DHTR** | **Anamnestic IgG (Kidd, Duffy, Kell)**; 3-14 Days; New Alloantibody |
| **Bacterial Sepsis** | **Platelets (22°C)**; Staph epidermidis; **Blood Cultures + IV Antibiotics** |
| **Citrate Toxicity** | **Hypocalcaemia**; Massive Transfusion; **IV Calcium Gluconate** |
| **SHOT Reporting** | All Serious Reactions; Near Misses; Annual Report |
| **Irradiated Blood** | **TA-GvHD Prevention**; 25 Gy; Immunocompromised, Intrauterine, Relatives |
| **Leucodepletion** | **FNHTR, CMV, Platelet Alloimmunisation Prevention**; Universal in UK |
| **TRALI vs TACO** | TRALI = No Cardiac Failure, BNP Normal; TACO = BNP High, Fluid Overload |
| **TRALI Prevention** | **Male-Only Plasma/Platelets**; Lower Anti-HLA/HNA |

# Acromegaly

> [!info]
> **Acromegaly = Chronic GH Excess** (Usually Pituitary Macroadenoma). **Insidious Onset** → Coarse Features, Acral Enlargement, Organomegaly. **IGF-1 = Screening Test**; **OGTT GH Suppression** = Gold Standard Diagnosis. **Transsphenoidal Surgery** = First-Line Treatment.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise clinical features (acral enlargement, coarse features, organomegaly)
- [ ] Apply diagnostic criteria (IGF-1, OGTT GH suppression)
- [ ] Interpret pituitary MRI and plan surgical approach
- [ ] Outline medical management (somatostatin analogues, pegvisomant, dopamine agonists)
- [ ] Manage complications (cardiomyopathy, sleep apnoea, glucose intolerance, colonic polyps)

---

## 2. Epidemiology & Aetiology

| Feature | Details |
|---------|---------|
| **Incidence** | **3-4 per million/year** |
| **Prevalence** | **40-70 per million** |
| **Peak Age** | **40-50 Years** |
| **Sex** | **Equal** (M:F = 1:1) |
| **Aetiology** | **Pituitary GH-Secreting Adenoma** (>95%) — **Macroadenoma >10mm (80-90%)** |
| **Rare Causes** | Ectopic GHRH (Pancreatic NET, Bronchial Carcinoid), Ectopic GH (Rare) |
| **Genetics** | **AIP** (Young-Onset, Familial), **GPR101**, **GNAS** (McCune-Albright), **MEN1** |

## 3. Clinical Features

### Acral & Facial Changes
| Feature | Description |
|---------|-------------|
| **Acral Enlargement** | **Hands/Feet Enlargement** → Ring/Shoe Size Increase, Spade-Like Hands |
| **Facial** | **Coarse Features**, Prognathism, Macroglossia, Wide Spacing Teeth, Prominent Supraorbital Ridges |
| **Skin** | Thickened, Oily, Hyperhidrosis, Skin Tags, Acanthosis Nigricans |
| **Soft Tissue** | Soft Tissue Swelling, Carpal Tunnel Syndrome, Arthralgias |

### Systemic Manifestations
| System | Complications |
|--------|---------------|
| **Cardiovascular** | **Cardiomyopathy** (LVH, Diastolic Dysfunction), Hypertension, Sleep Apnoea (Obstructive), Arrhythmias |
| **Respiratory** | **Obstructive Sleep Apnoea** (50-80%), Upper Airway Obstruction |
| **Metabolic** | **Impaired Glucose Tolerance / Diabetes** (30-50%), Insulin Resistance, Dyslipidaemia |
| **Gastrointestinal** | **Colonic Polyps** (↑ Risk 2-3x, Adenomatous), Diverticulosis |
| **Musculoskeletal** | Arthralgias, Joint Swelling, Osteoarthritis, Carpal Tunnel Syndrome |
| **Reproductive** | Hypogonadism (↓ LH/FSH), Menstrual Irregularities, Infertility, Galactorrhoea (If Co-Secreting PRL) |

## 4. Diagnosis — Biochemical & Imaging

### Screening & Confirmation

| Test | Normal | Acromegaly |
|------|--------|-------------|
| **IGF-1 (Age-Adjusted)** | Normal | **↑↑ (2-3x ULN)** — **Best Screening Test** |
| **OGTT (75g Glucose)** | **GH Suppressed <1 µg/L** at 60-120min | **GH NOT Suppressed ≥1 µg/L** (Gold Standard) |
| **Random GH** | <0.4 µg/L (Nadir) | **>1 µg/L** (If IGF-1 Elevated) |
| **MRI Pituitary** | Normal | **Macroadenoma >10mm (80-90%)** |

### Diagnostic Criteria
| Criterion | Threshold |
|---------|-----------|
| **IGF-1** | **Age-Adjusted >ULN** (Usually 2-3x ULN) |
| **OGTT GH** | **Not Suppressed <1 µg/L** at 60-120min (Gold Standard) |
| **Random GH** | >1 µg/L (If IGF-1 Elevated) |
| **MRI Pituitary** | Macroadenoma >10mm (80-90%) |

### Differential Diagnosis of Elevated IGF-1
| Condition | IGF-1 | OGTT GH Suppression | Other Features |
|---------|-------|---------------------|----------------|
| **Acromegaly** | ↑↑ | **Not Suppressed (≥1 µg/L)** | Pituitary Macroadenoma |
| **Pubertal Gigantism** | ↑↑ | Not Suppressed | Open Epiphyses, Age <18 |
| **Pregnancy** | ↑ (Placental GH) | N/A | Physiological |
| **Hepatic Failure** | ↓ (Impaired Synthesis) | N/A | Liver Disease |
| **Renal Failure** | ↑ (↓ Clearance) | Variable | ↑ Creatinine |
| **Malnutrition/Anorexia** | ↓↓ | Paradoxical ↑ GH | Low BMI, Starvation |

## 5. Pituitary MRI — Imaging Protocol

| Parameter | Specification |
|-----------|---------------|
| **Slice Thickness** | **3mm** (No Gap) |
| **Sequences** | T1 Pre/Post-Gadolinium (Coronal, Sagittal), T2 |
| **Dynamic Contrast** | T1 Post-Gad at 30s, 60s, 90s (Microadenoma Delayed Enhancement) |
| **Field of View** | Sella + Suprasellar Region + Optic Chiasm |

### Adenoma Classification
| Type | Size | Frequency |
|------|------|-----------|
| **Microadenoma** | <10mm | 10-20% |
| **Macroadenoma** | **≥10mm** | **80-90%** |
| **Giant** | **>40mm** | Rare |

## 6. Complications — High Yield for Exams

| System | Complication | Screening |
|--------|--------------|-----------|
| **Cardiovascular** | **Cardiomyopathy** (LVH, Diastolic Dysfunction), **Hypertension** (30-50%), **Sleep Apnoea** (50-80%) | Echo, Sleep Study, BP Monitoring |
| **Respiratory** | **Obstructive Sleep Apnoea** (50-80%) | Sleep Study (Polysomnography) |
| **Metabolic** | **Impaired Glucose Tolerance / Diabetes** (30-50%), Dyslipidaemia | HbA1c, Fasting Glucose, Lipids |
| **GI** | **Colonic Polyps** (↑ 2-3x Risk), Diverticulosis | Colonoscopy at Diagnosis, q3-5yr |
| **Joints** | Arthralgias, Osteoarthritis, Carpal Tunnel Syndrome | Clinical Exam |
| **Reproductive** | Hypogonadism (↓ LH/FSH), Menstrual Irregularities, Infertility | LH, FSH, Testosterone/Oestradiol, Prolactin |
| **Neoplastic Risk** | **Thyroid Nodules** (↑), **Colonic Polyps** (↑ 2-3x) | Thyroid US, Colonoscopy |

## 7. Management Algorithm

```
Acromegaly Diagnosed (↑ IGF-1 + GH Not Suppressed on OGTT)
         │
         ▼
MRI PITUITARY (3mm Slices) → Macroadenoma >10mm (90%)
         │
         ▼
**TRANSSEPHENOIDAL SURGERY (TSS) — 1st LINE**
         │
         ├── CURE (Post-Op IGF-1 Normal + OGTT GH <1 µg/L)
         │       → SURVEILLANCE (Annual IGF-1 + OGTT)
         │
         └── RESIDUAL DISEASE (IGF-1 Elevated / GH Not Suppressed)
                 │
                 ├── RE-OPERATION (If Accessible Residual)
                 └── MEDICAL THERAPY
                         ├── SOMATOSTATIN ANALOGUES (1st Line Medical)
                         │       Octreotide LAR 20-40mg IM q4wk
                         │       Lanreotide Autogel 60-120mg SC q4wk
                         │       Pasireotide LAR 40-60mg IM q4wk (2nd Line)
                         │
                         ├── PEGVISOMANT (GH Receptor Antagonist)
                         │       10-30mg SC Daily (If SST Refractory)
                         │
                         ├── DOPAMINE AGONISTS (Adjunct)
                         │       Cabergoline 0.5-3mg/week
                         │
                         └── RADIOTHERAPY (Stereotactic / Fractionated)
                                 → If Surgery + Medical Fail
```

## 8. Medical Therapy — Agents & Dosing

### Somatostatin Analogues (1st Line Medical)
| Agent | Dose | Route | Frequency | Key Features |
|-------|------|-------|-----------|--------------|
| **Octreotide LAR** | 20-40mg | IM | q4 Weeks | Standard 1st Line |
| **Lanreotide Autogel** | 60-120mg | SC | q4 Weeks | Alternative; Self-Administration |
| **Pasireotide LAR** | 40-60mg | IM | q4 Weeks | **2nd Line**; ↑ Hyperglycaemia Risk |

### Pegvisomant (GH Receptor Antagonist) — 2nd/3rd Line
| Feature | Details |
|---------|---------|
| **Mechanism** | **GH Receptor Antagonist** — Blocks GH Action (↓ IGF-1) |
| **Dose** | **10-30mg SC Daily** (Titrate to IGF-1 Normalisation) |
| **Indication** | **SST Refractory** / **Intolerant** / **Post-RT** |
| **Advantage** | Normalises IGF-1 Without ↓ GH (GH ↑↑ on Pegvisomant) |
| **Monitoring** | **IGF-1 Normalisation** (GH Levels Rise — Do Not Use for Monitoring) |

### Dopamine Agonists (Adjunct)
| Agent | Dose | Role |
|-------|------|------|
| **Cabergoline** | 0.5-3mg/week | Adjunct; Mild GH Suppression; Low Cost |

|-----------|--------|

## 12. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **IGF-1 vs GH for Diagnosis** | **IGF-1 = Screening** (Stable); **OGTT GH = Confirmation** (Dynamic) |
| **GH Random** | Useless (Pulsatile); Use IGF-1 or OGTT |
| **Gigantism vs Acromegaly** | Gigantism = Pre-Epiphyseal Fusion; Acromegaly = Post-Fusion |
| **Pegvisomant Effect** | **GH ↑↑ on Treatment** (Displaces GH from Receptor); **IGF-1 Normalises** |
| **Pasireotide vs Octreotide** | Pasireotide = SSTR1/2/3/5 (Broader); ↑ Hyperglycaemia Risk |
| **Surgery Cure Rate** | Micro 80-90%; Macro 50-60% |
| **Remission Definition** | IGF-1 Normal + OGTT GH <1 µg/L |
| **Sleep Apnoea** | 50-80% in Acromegaly; Screen with Sleep Study |
| **Colonic Polyps** | 2-3x Increased Risk; Colonoscopy at Diagnosis |
| **Pegvisomant Monitoring** | Monitor **IGF-1** (NOT GH — GH Rises on Treatment) |

# Prolactin Axis

> [!info]
> **Prolactin = Lactotroph product.** Unique anterior pituitary hormone under **tonic dopaminergic inhibition**. Most common functioning pituitary adenoma (prolactinoma). Key concepts: dopamine inhibition, macroprolactinaemia, cabergoline treatment.

## 2. Prolactin Physiology

### Synthesis & Secretion
| Feature | Details |
|---------|---------|
| **Source** | Anterior Pituitary **Lactotrophs** (15-20% of cells) |
| **Structure** | 199 amino acid polypeptide; **Macroprolactin** = PRL-IgG complex (large, inactive) |
| **Regulation** | **Dopamine (PIH)** = **Tonic inhibition** (Main regulator); **TRH** (stimulatory); **Oestrogen** (stimulates lactotroph hyperplasia); Suckling, Stress, Sleep → ↑ |
| **Circadian Rhythm** | Peak during **sleep**; Nadir mid-morning |
| **Pregnancy** | **↑ 10-20x** (Oestrogen stimulates lactotroph hyperplasia); Increases from 6 weeks, plateaus at term |
| **Postpartum** | Suckling → Prolactin surges → Lactation; Lactational amenorrhoea (↓ GnRH) |

### Receptor & Actions
| Receptor | PRL-R (Cytokine receptor family, JAK2/STAT5 pathway) |
|----------|------------------------------------------------------|
| **Main Actions** | **Mammogenesis** (Breast development), **Lactogenesis** (Milk synthesis), **Galactopoiesis** (Milk maintenance) |
| **Gonadal Effects** | High PRL → ↓ GnRH pulsatility → ↓ LH/FSH → Hypogonadism, Infertility, Galactorrhoea |

## 3. Hyperprolactinaemia — Diagnostic Algorithm

```
Prolactin >500 mIU/L (or > upper reference limit)
         │
         ▼
EXCLUDE PHYSIOLOGICAL / PHARMACOLOGICAL CAUSES:
         │
         ├── DRUGS: Antipsychotics (Risperidone > Olanzapine > Quetiapine), Metoclopramide, Antidepressants (SSRIs, TCAs), OCP, Verapamil, Opioids
         ├── HYPOTHYROIDISM (TRH-driven) → Check TSH
         ├── RENAL FAILURE (Reduced clearance)
         ├── PREGNANCY
         ├── NIPPLE STIMULATION / STRESS (Recent)
         └── MACROPROLACTINAEMIA
         │
         ▼
REPEAT PROLACTIN (Fasting, Non-stressed, Avoid Nipple Stimulation)
         │
         ├── NORMAL → Transient/Physiological
         │
         └── PERSISTENTLY ELEVATED
                 │
                 ▼
PEG PRECIPITATION TEST → MACROPROLACTINAEMIC?
                 │
                 ├── YES → MACROPROLACTINAEMIA (Benign, No treatment)
                 │
                 └── NO → TRUE HYPERPROLACTINAEMIA
                         │
                         ▼
                 MRI PITUITARY (3mm slices, Gadolinium)
                         │
                         ├── NO TUMOUR + Prolactin <1000 → IDIOPATHIC / STALK EFFECT
                         ├── MICROADENOMA (<10mm) → CABERGOLINE
                         ├── MACROADENOMA (≥10mm) → CABERGOLINE / SURGERY
                         └── NO TUMOUR + Prolactin >1000 → STALK EFFECT / ECTOPIC (Rare)
```

## 4. Macroprolactinaemia

| Feature | Details |
|---------|---------|
| **Definition** | Prolactin complexed with **IgG autoantibody** (PRL-IgG complex) |
| **Characteristics** | **High molecular weight** (>150 kDa); **Biologically inactive** (cannot cross capillary endothelium) |
| **Prevalence** | 10-25% of hyperprolactinaemia |
| **Diagnosis** | **PEG (Polyethylene Glycol) Precipitation** → Precipitates macroprolactin; **Recovery <60%** = Macroprolactinaemia |
| **Clinical Significance** | **Benign**; No pituitary tumour; No galactorrhoea/hypogonademia; **No treatment needed** |
| **Pitfall** | Can coexist with true hyperprolactinaemia → Measure monomeric PRL after PEG |

## 5. Prolactinoma — Classification & Management

### Classification
| Type | Size | Prolactin Level | Prevalence |
|------|------|----------------|------------|
| **Microprolactinoma** | **<10 mm** | Usually **500-5000 mIU/L** | ~60% |
| **Macroprolactinoma** | **≥10 mm** | **>5000 mIU/L** (often >10,000) | ~40% |
| **Giant** | **>40 mm** | Very high | Rare |

### Clinical Presentation
| Feature | Microprolactinoma | Macroprolactinoma |
|---------|--------------------|-------------------|
| **Galactorrhoea** | Common | Common |
| **Hypogonadism** | Amenorrhoea, Infertility, Low libido | Severe (Mass effect + PRL suppression of GnRH) |
| **Visual Fields** | Normal | **Bi-temporal Hemianopia** (Chiasmal compression) |
| **Headache** | Rare | Common |
| **Other Pituitary Deficits** | Rare | Common (Hypopituitarism) |

---

## 6. Prolactinoma — Management

### Microprolactinoma (<10mm)
| Aspect | Protocol |
|--------|----------|
| **1st Line** | **Cabergoline** 0.25mg BD → Titrate weekly by 0.25mg BD |
| **Target Dose** | 0.5-1mg BD (Max 2mg BD) |
| **Goal** | Prolactin normalisation + Tumour shrinkage (>50% volume reduction) |
| **Monitoring** | Prolactin q4-8w during titration; MRI at 6-12m if macro features |
| **Withdrawal** | After **2-3 years** normal PRL + **No tumour on MRI** → Trial off cabergoline |

### Macroprolactinoma (≥10mm)
| Aspect | Protocol |
|--------|----------|
| **1st Line** | **Cabergoline** 0.5mg BD → Titrate to 1-2mg BD (Max 3mg BD) |
| **Target** | Prolactin normalisation + **>50% volume reduction** |
| **Surgery Indications** | Cabergoline resistant/intolerant; Apoplexy; Chiasmal compression with visual loss despite medical Rx |
| **Visual Fields** | **Mandatory** (Bi-temporal hemianopia = Urgent decompression) |
| **Visual Improvement** | Usually within days-weeks of cabergoline initiation |

### Cabergoline Dosing Schedule
| Week | Microprolactinoma | Macroprolactinoma |
|------|-------------------|-------------------|
| **1-2** | 0.25mg BD | 0.5mg BD |
| **3-4** | 0.5mg BD | 1mg BD |
| **5-8** | 1mg BD | 1.5-2mg BD |
| **Max** | 2mg BD | 3mg BD (rarely >2mg) |

---

## 7. Cabergoline — Key Points

| Aspect | Details |
|--------|---------|
| **Mechanism** | **D2 Dopamine Receptor Agonist** → Inhibits Prolactin secretion + Shrinks tumour |
| **Dose Range** | 0.25-3mg/week (divided BD); Titrate slowly |
| **Half-life** | **63-69 hours** (long; allows BD/Weekly dosing) |
| **Onset** | Prolactin falls within hours; Tumour shrinkage over weeks-months |
| **Side Effects** | Nausea (40%), Postural hypotension, Nasal congestion, Impulse control disorders (Rare), Valvulopathy (High dose, chronic) |
| **Monitoring** | Echocardiogram if dose >2mg/day or duration >5 years; Prolactin q3-6m |

---

## 8. Withdrawal / Discontinuation Criteria

| Criteria | Required |
|----------|----------|
| **Duration** | **≥2-3 years** on stable dose |
| **Biochemical** | Prolactin **normalised** on stable dose for ≥2 years |
| **Radiological** | **No tumour visible on MRI** (or microadenoma resolved) |
| **Process** | Taper cabergoline over 3-6 months; Recheck Prolactin q3mo; Restart if recurrence |

---

## 9. Pregnancy & Prolactinoma

| Scenario | Microprolactinoma | Macroprolactinoma |
|----------|-------------------|-------------------|
| **Pre-conception** | Stop cabergoline if Prolactin normal + MRI stable | Continue cabergoline (risk of growth) |
| **During Pregnancy** | Stop cabergoline; Monitor PRL q trimester; Visual fields if macro | **Continue cabergoline** (Risk of growth >10%) |
| **Delivery** | Vaginal preferred; Monitor BP | C-section if chiasmal compression risk |
| **Postpartum** | Breastfeeding → Prolactin ↑; Cabergoline suppresses lactation | Breastfeeding → Cabergoline suppresses lactation |
| **Monitoring** | Visual fields q trimester if macro | MRI if new neuro symptoms |

**Key Point**: **Microprolactinoma** → Stop cabergoline pre-conception; **Macroprolactinoma** → Continue cabergoline throughout pregnancy.

## 10. Stalk Effect / Disconnection Hyperprolactinaemia

| Feature | Details |
|---------|---------|
| **Mechanism** | Pituitary stalk compression → ↓ Dopamine delivery to lactotrophs → **Disinhibition** |
| **Prolactin Level** | Usually **<2000 mIU/L** (rarely >3000) |
| **MRI** | **Non-functioning macroadenoma** compressing stalk / Suprasellar mass |
| **Management** | Treat underlying mass (Surgery if macroadenoma); Prolactin normalises post-decompression |
| **Key Differentiator** | **Visual field defects** common; Other pituitary deficits often coexist |

---

## 11. Drug-Induced Hyperprolactinaemia

| Drug Class | Mechanism | Typical Prolactin Level |
|------------|-----------|-------------------------|
| **Antipsychotics** (Typical > Atypical) | D2 Antagonism → Disinhibition | Risperidone > Olanzapine > Quetiapine > Clozapine > Aripiprazole (Low) |
| **Metoclopramide / Domperidone** | D2 Antagonism (GI motility) | Moderate-High |
| **Antidepressants** (SSRIs, TCAs) | Serotonergic modulation | Mild-Moderate |
| **Opioids** | μ-receptor → ↓ Dopamine | Variable |
| **Verapamil / Alpha-methyldopa** | Central D2 effect | Mild |
| **OCP** | Oestrogen → Lactotroph hyperplasia | Mild |

**Management**: Switch to lower-risk agent if possible; If essential → Monitor; Cabergoline if symptomatic and drug cannot be stopped.

## 13. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Prolactinoma vs Stalk Effect** | Stalk effect: Non-functioning macroadenoma + prolactin <2000 + Visual fields + Other deficits |
| **Macroprolactinaemia vs True HyperPRL** | PEG precipitation → Recovery <60% = Macro; True hyper = Monomeric PRL > monomeric upper limit |
| **Cabergoline vs Bromocriptine** | Cabergoline preferred: Longer half-life, Better tolerability, Higher efficacy, Once/twice weekly |
| **Prolactin in Pregnancy** | Physiological ↑ 10-20x; Not pathological; No treatment |
| **Micro vs Macro Prolactin** | Micro: 500-5000; Macro: >5000 (often >10,000) |
| **Cabergoline Stop** | After 2-3yr normal PRL + MRI resolution |
| **Pregnancy + Macro** | **Continue cabergoline** (Growth risk) |
| **Pregnancy + Micro** | Stop cabergoline (Low risk) |

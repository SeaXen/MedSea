# Osteoporosis & Metabolic Bone Disease

> [!info]
> **Osteoporosis = Systemic Skeletal Disease** Characterised by Low Bone Mass & Microarchitectural Deterioration → **Increased Fragility Fracture Risk.** **Commonest Metabolic Bone Disease.** **DEXA T-Score ≤-2.5 = Osteoporosis.** **FRAX Tool** for 10-Year Fracture Probability.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Diagnose osteoporosis using DEXA T-scores and clinical risk factors
- [ ] Apply FRAX tool for 10-year major osteoporotic/hip fracture probability
- [ ] Apply treatment thresholds (NOF, NICE, ESCEO guidelines)
- [ ] Prescribe first-line (bisphosphonates) and second-line (denosumab, teriparatide, romosozumab) therapies
- [ ] Manage glucocorticoid-induced osteoporosis (GIOP)
- [ ] Monitor treatment efficacy and duration (drug holidays)

## 2. Definition & Diagnosis

### WHO Criteria (DEXA T-Scores)
| Category | T-Score (SD from Young Adult Mean) |
|----------|-----------------------------------|
| **Normal** | **≥-1.0 SD** |
| **Osteopenia** | **-1.0 to -2.5 SD** |
| **Osteoporosis** | **≤-2.5 SD** |
| **Severe/Established Osteoporosis** | **≤-2.5 SD + Fragility Fracture** |

**Measurement Sites**: **Lumbar Spine (L1-L4)** + **Total Hip / Femoral Neck** (Lowest T-Score Used for Diagnosis).

---

## 3. Fracture Risk Assessment — FRAX® Tool

### FRAX® (WHO) — 10-Year Major Osteoporotic Fracture / Hip Fracture Probability
| Input Variables | Required |
|-----------------|----------|
| **Age, Sex, Weight, Height** | Mandatory |
| **Previous Fracture** | Yes/No |
| **Parental Hip Fracture** | Yes/No |
| **Current Smoking** | Yes/No |
| **Glucocorticoids** | Current/Past (>5mg Pred >3mo) |
| **Rheumatoid Arthritis** | Yes/No |
| **Secondary Osteoporosis** | T1DM, Untreated Hypothyroidism, etc. |
| **Alcohol ≥3 units/day** | Yes/No |
| **BMD (Femoral Neck)** | Optional (Improves Accuracy) |

### Treatment Thresholds (UK / NOGG)
| Scenario | Intervention Threshold (10-Year Probability) |
|----------|---------------------------------------------|
| **Major Osteoporotic Fracture** | **≥7.5% (Age 50-59) → ≥15% (Age 70+)** |
| **Hip Fracture** | **≥3% (Age 50-59) → ≥4.5% (Age 70+)** |

> **Note**: Thresholds Vary by Country/Guideline (NOGG vs NICE vs ESCEO).

## 4. Clinical Risk Factors for Fracture

| Category | Risk Factors |
|----------|--------------|
| **Non-Modifiable** | Age >65, Female, Caucasian/Asian, Family History (Parental Hip Fracture), Previous Fragility Fracture, Early Menopause (<45), Low BMI (<19) |
| **Modifiable** | Low BMI, Smoking, Alcohol >3 Units/Day, Sedentary, Low Calcium/Vit D Intake, Falls Risk |
| **Diseases** | RA, T1DM, Hyperthyroidism, Coeliac, IBD, Chronic Liver/Kidney Disease, Anorexia, Premature Menopause, Premature Andropause |

---

## 5. Secondary Causes of Osteoporosis (Always Exclude)

| Category | Conditions |
|----------|------------|
| **Endocrine** | Hyperthyroidism, Hyperparathyroidism, Cushing's, Hypogonadism (Male/Female), T1DM, Acromegaly |
| **GI** | Coeliac, IBD, Gastrectomy, Bariatric Surgery, Chronic Pancreatitis |
| **Haematological** | Multiple Myeloma, Lymphoma, Leukaemia |
| **Rheumatological** | RA, Ankylosing Spondylitis, SLE |
| **Medications** | **Glucocorticoids (>5mg Pred >3mo)**, Aromatase Inhibitors, AEDs (Enzyme Inducers), PPIs (Long-Term), Heparin, Methotrexate, Depo-Provera, GnRH Analogues |
| **Other** | Immobilisation, Organ Transplant, HIV, HIV Meds (TDF), Loop Diuretics, Loop Diuretics |

## 6. Diagnosis & Workup

### Essential Investigations
| Test | Purpose |
|------|---------|
| **DEXA (Lumbar Spine + Hip)** | **Gold Standard** — T-Score Diagnosis |
| **Lateral Spine X-Ray / VFA** | Vertebral Fracture Assessment (VFA) — Silent Fractures |
| **Bloods** | Ca, Phos, ALP, 25-OH Vit D, TSH, TFTs, FBC, U&E, LFT, FSH/LH/Oestradiol (Women), Testosterone (Men), 25-OH Vit D |
| **Fracture Risk** | **FRAX®** (With/Without BMD) |
| **Lateral Spine X-Ray** | If Height Loss >3cm, Kyphosis, Back Pain |

### Laboratory Exclusion of Secondary Causes
| Test | Target |
|------|--------|
| **Calcium, Phosphate, ALP** | Normal (Exclude Hyperparathyroidism/Osteomalacia) |
| **TSH, fT4** | Normal (Exclude Hyperthyroidism) |
| **25-OH Vit D** | **>50 nmol/L (20 ng/mL)** |
| **TSH, Testosterone (Men)** | Exclude Hypogonadism |
| **FSH/LH/Oestradiol** | Exclude Premature Menopause |
| **Coeliac Screen (TTG-IgA)** | If GI Symptoms |

## 7. Treatment — Pharmacological Therapy

### Treatment Indications (NOGG / NICE / ESCEO)
| Scenario | Treatment Indicated |
|----------|---------------------|
| **Osteoporosis (T ≤-2.5)** | **Yes** |
| **Osteopenia (T -1 to -2.5) + FRAX >Threshold** | **Yes** |
| **Fragility Fracture (Any T-Score)** | **Yes** |
| **Glucocorticoid-Induced** | **Yes** (If ≥5mg Pred >3mo + T ≤-1.5) |

---

## 8. Pharmacological Therapy — First-Line

### Oral Bisphosphonates (First-Line)
| Drug | Dose | Administration | Monitoring |
|------|------|----------------|------------|
| **Alendronate** | **70mg Weekly** PO | **First-Line**; Fast 30min, Upright 30min, Water 200mL | DEXA 2-3yr; eGFR if <35 |
| **Risedronate** | 35mg Weekly / 150mg Monthly | Alternative (Better GI Tolerance) | |
| **Ibandronate** | 150mg Monthly PO / 3mg IV q3mo | GI Intolerance | |
| **Zoledronic Acid** | **5mg IV Annual** | **Alternative if Oral Contraindicated**; Renal Adjust | eGFR ≥35 (Avoid <35) |

### Bisphosphonate Monitoring
| Parameter | Frequency |
|-----------|-----------|
| **DEXA** | **2-3 Years** After Initiation; Then 2-3 Yearly |
| **Renal Function** | Annual (eGFR) — Avoid if eGFR <35 (Oral); <35 (IV Zoledronic) |
| **Adherence** | Assess at Each Visit |

### Bisphosphonate Duration & Drug Holiday
| Duration | Action |
|----------|--------|
| **3-5 Years (Oral)** / **3 Years (IV)** | **Reassess** → **Drug Holiday** if Low Risk (T-Score >-2.5, No Fracture) |
| **High Risk** (Prior Fracture / T ≤-3.0 / Steroid) | **Continue** Beyond 5-6 Years |
| **Drug Holiday Monitoring** | DEXA q1-2yr; Restart if T-Score ≤-2.5 or Fracture |

---

## 9. Second-Line / Alternative Therapies

| Drug | Dose | Indication | Advantage |
|------|------|------------|-----------|
| **Denosumab** | **60mg SC q6mo** | **First-Line if Bisphosphonate Contraindicated/Intolerant**; **GIOP** | No Renal Adjustment; Potent ↓ Fracture |
| **Teriparatide** | **20µg SC Daily** (Max 24 Months) | **Severe Osteoporosis** (T ≤-3.5, Prior Fracture, Failed Bisphosphonate) | **Anabolic** (↑ Bone Formation) |
| **Romosozumab** | **210mg SC Monthly ×12mo** | **Severe** (Very High Fracture Risk) | **Anabolic + Antiresorptive** |
| **Teriparatide** | 20µg SC Daily (Max 24mo) | Severe; Failed/Intolerant Bisphosphonates | Anabolic |
| **Raloxifene** | 60mg OD | Postmenopausal Women; **Breast Cancer Risk Reduction** | SERM |
| **HRT** | Oestrogen ± Progestogen | Early Postmenopause (<10yr) | Symptom Relief + Bone |

## 10. Glucocorticoid-Induced Osteoporosis (GIOP)

### Risk Stratification
| Risk | Criteria | Action |
|------|----------|--------|
| **High Risk** | **≥5mg Pred >3mo** + **T-Score ≤-1.5** OR **Prior Fracture** OR **Age >70** | **Start Bone Protection Immediately** |
| **Moderate** | **≥5mg Pred >3mo** + **T-Score >-1.5** | **Consider** (If High FRAX) |
| **Low** | **<5mg Pred** OR **<3mo** | Lifestyle + Calcium/Vit D |

### First-Line for GIOP
| Drug | Dose |
|------|------|
| **Oral Bisphosphonate** (Alendronate/Risedronate) | **First-Line** |
| **Zoledronic Acid 5mg IV Annual** | If Oral Contraindicated |
| **Denosumab 60mg SC q6mo** | **Preferred if High Fracture Risk / eGFR <35** |
| **Teriparatide** | **Preferred if Very High Risk / Prior Fracture** (Anabolic Superior in GIOP) |

**Calcium + Vitamin D Mandatory** with All Therapies.

---

## 11. Treatment Duration & Monitoring

### Bisphosphonate Holiday
| Scenario | Action |
|----------|--------|
| **Low Risk** (No Fracture, T >-2.5 after 3-5yr) | **Drug Holiday** (Stop 1-3 Years) → Reassess DEXA q1-2yr |
| **High Risk** (Prior Fracture, T ≤-2.5, Steroid) | **Continue** (May Extend to 10yr) |
| **Denosumab** | **NO HOLIDAY** — Rapid Bone Loss on Stopping → Transition to Bisphosphonate |

### Monitoring Schedule
| Parameter | Frequency |
|-----------|-----------|
| **DEXA** | **2-3 Years** After Initiation; Then Q2-3yr |
| **Height** | Annual (Vertebral Fracture Screen) |
| **Biochemistry** | Annual (Ca, Phos, ALP, 25-OH Vit D, Renal) |
| **Adherence** | Each Visit |

| **Workup** | **Testosterone, LH, FSH, Prolactin, TSH, 25-OH Vit D** |
| **Treatment** | **Testosterone Replacement** (If Hypogonadal) + **Osteoporosis Therapy** |

## 14. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Osteopenia vs Osteoporosis** | Osteopenia: T -1 to -2.5; Osteoporosis: T ≤-2.5 |
| **T-Score vs Z-Score** | T-Score = vs Young Adult Mean; Z-Score = vs Age/Sex Matched (Use Z in <50yr) |
| **Bisphosphonate Holiday** | Oral: 3-5yr → Holiday; IV Zoledronic: 3yr → Holiday; **Denosumab = NO HOLIDAY** |
| **Denosumab Holiday** | **Contraindicated** — Rapid Bone Loss → Transition to Bisphosphonate |
| **Teriparatide Duration** | **Max 24 Months** (Rat Osteosarcoma Risk); Anabolic |
| **Romosozumab Duration** | **12 Months Only** → Follow with Antiresorptive |
| **FRAX Without BMD** | Valid (But Less Accurate); Add BMD for Accuracy |
| **Zoledronic Acid Renal** | eGFR ≥35; <35 → Avoid/Reduce dose |
| **Teriparatide Max Duration** | **24 Months Lifetime Max** (Rat Osteosarcoma Data) |
| **Romosozumab Duration** | **12 Months Only** → Follow with Antiresorptive |

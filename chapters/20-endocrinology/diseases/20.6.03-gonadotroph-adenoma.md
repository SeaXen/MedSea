# Gonadotroph Adenoma

> [!info]
> **Gonadotroph Adenoma = Pituitary Adenoma Secreting FSH and/or LH (and α-Subunit).** **Usually Clinically Silent / Non-Functioning.** **Most Present as Macroadenomas with Mass Effect.** **No Classical Hormone Excess Syndrome** (Unlike Prolactinoma/Acromegaly/Cushing).

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise the clinical presentation (mass effect rather than hormone excess)
- [ ] Interpret hormonal profile (FSH/LH/α-subunit elevation without clinical syndrome)
- [ ] Apply diagnostic workup (MRI, hormonal panel, visual fields)
- [ ] Apply management algorithm (observation vs surgery)
- [ ] Differentiate from other pituitary adenomas

---

## 2. Pathophysiology & Classification

| Feature | Details |
|---------|---------|
| **Cell Origin** | **Gonadotrophs** (FSH/LH Producing Cells) — ~10-15% of Anterior Pituitary |
| **Hormones Secreted** | **FSH**, **LH**, **α-Subunit** (Common), **β-Subunits** (FSH-β, LH-β) |
| **Clinical Syndrome** | **Usually NONE** ("Silent" / Non-Functioning) |
| **Size at Presentation** | **Macroadenoma (>10mm) in >80%** — Presents Late Due to Lack of Hormone Excess |

### Hormone Profile
| Hormone | Typical Pattern |
|---------|---------------|
| **FSH** | ↑↑ (Often Disproportionately High vs LH) |
| **LH** | ↑ or Normal |
| **α-Subunit** | **Consistently Elevated** (Most Sensitive Marker) |
| **Free α-Subunit** | **Markedly Elevated** (Specific Marker) |
| **Testosterone/Oestradiol** | Low/Normal (Due to ↓ Pulsatile LH) — **Not Elevated** |

---

## 3. Clinical Presentation — Mass Effect Dominates

| Presentation | Frequency | Details |
|------------|-----------|---------|
| **Incidentaloma** | 20-30% | Found on MRI for Headache/Trauma |
| **Visual Field Defects** | 50-70% | **Bi-temporal Hemianopia** (Chiasmal Compression) |
| **Headache** | 40-60% | Frontal/Retro-orbital |
| **Hypopituitarism** | 40-50% | GH → Gonadotrophs → TSH → ACTH Order |
| **Cranial Nerve Palsies** | 10-20% | CN III/IV/VI (Cavernous Sinus Invasion) |
| **Hormone Excess Symptoms** | **Absent** | No Classical Syndrome (No Hyperprolactinaemia, Acromegaly, Cushing) |

---

## 4. Biochemical Profile

### Hormonal Pattern
| Hormone | Typical Finding | Significance |
|---------|-----------------|--------------|
| **FSH** | **↑↑** (Often Disproportionately > LH) | Gonadotroph Hyperplasia/Neoplasia |
| **LH** | ↑ or Normal | Often Less Than FSH |
| **α-Subunit** | **↑↑ (Markedly Elevated)** | **Most Sensitive Marker** |
| **Free α-Subunit** | **Markedly Elevated** | Specific for Gonadotroph Adenoma |
| **LH/FSH Ratio** | **<1** (FSH Predominates) | Opposite to PCOS / Menopause |
| **Testosterone / Oestradiol** | **Low / Normal** | Dysregulation; Not Elevated |
| **Prolactin** | Normal / Mildly ↑ (Stalk Effect) | Stalk Compression (Disinhibition) |

---

## 5. Diagnosis — Workup Algorithm

```
Pituitary Incidentaloma / Mass Effect Symptoms
         │
         ▼
HORMONAL PANEL: FSH, LH, α-Subunit, Free α-Subunit, Testosterone/Oestradiol
         │
         ├── FSH ↑↑, α-Subunit ↑↑, LH Normal/↑, Testosterone/Oestradiol Low/Normal
         │       → **GONADOTROPH ADENOMA** (Silent)
         │
         ├── Prolactin ↑↑ → PROLACTINOMA
         ├── GH/IGF-1 ↑ → ACROMEGALY
         ├── ACTH/Cortisol ↑ → CUSHING DISEASE
         ├── TSH ↑ + fT4/fT3 ↑ → TSH-oma
         └── All Normal / Mild α-Subunit ↑ → NON-FUNCTIONING / NULL CELL ADENOMA
```

### Key Diagnostic Features
| Feature | Gonadotroph Adenoma | Non-Functioning (Null Cell) |
|---------|--------------------|-----------------------------|
| **FSH/LH** | **↑↑ / ↑ (FSH > LH)** | Normal / Slightly ↑ |
| **α-Subunit** | **Markedly Elevated** | Normal / Mildly ↑ |
| **Free α-Subunit** | **Markedly ↑** | Normal |
| **Sex Steroids** | Low / Normal (Not Elevated) | Low (Due to Compression) |
| **Clinical Syndrome** | **None (Silent)** | None (Silent) |

## 6. Imaging & Localisation

### MRI Pituitary (3mm, Gadolinium)
| Feature | Findings |
|---------|----------|
| **Size** | **Macroadenoma >10mm** (>80%) |
| **Enhancement** | Homogeneous / Heterogeneous |
| **Extension** | Suprasellar (Chiasm), Cavernous Sinus (Knosp Grade) |
| **Signal** | T1 Hypointense, T2 Variable |

### Cavernous Sinus Invasion (Knosp Classification)
| Grade | Description | Surgical Resectability |
|-------|-------------|------------------------|
| **Grade 0** | Tumour ≤ Intercarotid Line | Complete Resection Likely |
| **Grade 1** | > Intercarotid, ≤ Tangent to Lateral ICA | Complete Possible |
| **Grade 2** | > Lateral ICA, ≤ Tangent to Medial ICA | Partial Resection |
| **Grade 3** | > Medial ICA, ≤ Tangent to MCC | Subtotal |
| **Grade 4** | > Tangent to MCC | Gross Total Unlikely |

---

## 7. Management Algorithm

```
Gonadotroph Adenoma Confirmed (Hormones + MRI)
         │
         ├── MICROADENOMA (<10mm) + ASYMPTOMATIC
         │       → OBSERVATION
         │       MRI 6-12mo → Yearly ×2 → Then 2-5yrly
         │       Hormonal Surveillance Annual
         │
         ├── MACROADENOMA (≥10mm) + ASYMPTOMATIC
         │       → SURGERY RECOMMENDED (High Risk Future Mass Effect)
         │       Transsphenoidal Surgery (TSS)
         │
         ├── VISUAL FIELD DEFECTS (Any Size)
         │       → URGENT TRANSSEPHENOIDAL SURGERY (TSS)
         │
         ├── HYPOPITUITARISM (Symptomatic)
         │       → SURGERY (TSS) + Hormone Replacement
         │
         └── RECURRENT / RESIDUAL POST-SURGERY
                 ├── ASYMPTOMATIC RESIDUAL (<1cm) → Observation
                 ├── SYMPTOMATIC / GROWING → Re-operation / Radiotherapy
                 └── NOT SURGICAL CANDIDATE → Stereotactic Radiosurgery (SRS)
```

---

## 8. Surgical Management

### Transsphenoidal Surgery (TSS)
| Aspect | Details |
|--------|---------|
| **Approach** | **Endoscopic Endonasal** (Preferred) / Microscopic Sub-labial |
| **Goal** | **Gross Total Resection (GTR)** if Possible; Decompression if Cavernous Sinus Invasion |
| **Success Rates** | **GTR 70-80%** (Knosp 0-1); **Decompression 90-95%** |
| **Complications** | CSF Leak (2-5%), DI (10-20% Transient), New Hypopituitarism (5-10%), Epistaxis |

### Cavernous Sinus Invasion (Knosp)
| Grade | Description | Resection |
|-------|-------------|----------|
| **Grade 0** | ≤ Intercarotid Line | Complete Likely |
| **Grade 1** | > Intercarotid, ≤ Tangent Lateral ICA | Complete Possible |
| **Grade 2** | > Lateral ICA, ≤ Medial ICA | **Subtotal Expected** |
| **Grade 3** | > Medial ICA, ≤ MCC | Subtotal |
| **Grade 4** | > MCC | Decompression Only |

---

## 9. Post-Operative Management

| Timepoint | Assessments |
|-----------|-------------|
| **Day 1-2** | Hormones (FSH, LH, α-Subunit, Testosterone/Oestradiol), Cortisol, Electrolytes, Visual Fields |
| **6 Weeks** | Full Hormone Panel, Visual Fields |
| **3 Months** | Full Hormones, MRI, Visual Fields |
| **6 Months** | Hormones, MRI (If Residual), Visual Fields |
| **Annual ×5yr** | Hormones, MRI, Visual Fields |
| **Long-term** | 2-3 Yearly MRI + Hormones + Visual Fields |

### Post-Op Hormone Replacement
| Deficiency | Replacement |
|------------|-------------|
| **ACTH/Cortisol** | Hydrocortisone 15-20mg AM + 5-10mg PM |
| **TSH/fT4** | Levothyroxine 1.6µg/kg/day |
| **Gonadotrophs** | Testosterone / HRT (If Hypogonadal) |
| **GH** | RH-GH if Confirmed GHD (ITT Peak <3 µg/L) |
| **Prolactin** | Not Replaced (Except Lactation) |

## 12. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Gonadotroph vs Non-Functioning** | Gonadotroph = **FSH/LH/α-Subunit ↑** (Biochemically Active); Non-Func = All Hormones Normal |
| **Gonadotroph vs Prolactinoma** | Prolactinoma = PRL ↑↑ + Galactorrhoea/Amenorrhoea; Gonadotroph = FSH/LH/α-Subunit ↑, PRL Normal |
| **Gonadotroph vs TSH-oma** | TSH-oma: TSH ↑ + fT4/fT3 ↑; Gonadotroph: TSH Normal |
| **Gonadotroph vs Null Cell** | Gonadotroph = α-Subunit/FSH/LH ↑; Null Cell = All Hormones Normal |
| **FSH > LH** | Characteristic of Gonadotroph Adenoma (vs PCOS LH>FSH) |
| **α-Subunit** | **Best Marker** for Gonadotroph Adenoma (More Sensitive than FSH/LH) |
| **Free α-Subunit** | More Specific than Total α-Subunit |
| **Surgery** | **Recommended for All Macroadenomas** (Even Asymptomatic) |
| **Knosp ≥3** | Subtotal Resection Expected; Adjuvant RT Considered |

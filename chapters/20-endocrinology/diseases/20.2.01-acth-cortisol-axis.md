# ACTH/Cortisol Axis (HPA Axis)

> [!info]
> **HPA axis = CRH → ACTH → Cortisol.** Central to stress response, glucose homeostasis, immune regulation, and circadian rhythm. Dysregulation → Cushing syndrome, adrenal insufficiency, steroid-induced suppression.

---

## 1. Learning Objectives
By the end of this note you should be able to:

- [ ] Apply feedback loops to distinguish primary, secondary, and tertiary adrenal insufficiency

- [ ] Distinguish Cushing disease, ectopic ACTH, and adrenal Cushing

## 2. HPA Axis — Anatomy & Physiology

### Hormone Synthesis & Secretion
| Step | Hormone | Source | Regulation |
|------|---------|--------|------------|
| **1. CRH** | Corticotropin-Releasing Hormone | Hypothalamus (PVN) | Stress ↑; Circadian (peak 06-08h); Negative feedback by Cortisol |
| **2. ACTH** | Adrenocorticotropic Hormone | Anterior Pituitary (Corticotrophs) | **CRH** (stimulatory); **Cortisol** (negative feedback); **AVP** (synergistic) |
| **3. Cortisol** | Glucocorticoid | Adrenal Cortex (Zona fasciculata/reticularis) | **ACTH** (stimulatory); **Negative feedback** at pituitary + hypothalamus |

### Precursor Pathway (Cortisol Synthesis)
```
Cholesterol → (STAR) → Pregnenolone → (CYP17A1) → 17-OH Pregnenolone
         ↓
         (CYP21A2) → 17-OH Progesterone → (CYP11B1) → 11-Deoxycortisol → **Cortisol** (CYP11B1)
```

| Enzyme | Gene | Product | Deficiency Syndrome |
|--------|------|---------|---------------------|
| **STAR** | STAR | Cholesterol transfer | Lipoid CAH |
| **CYP11A1** | CYP11A1 | Pregnenolone | Lipoid CAH |
| **CYP17A1** (17α-hydroxylase/17,20-lyase) | CYP17A1 | 17-OH steroids, DHEA | 17α-OH deficiency |
| **CYP21A2** (21-hydroxylase) | CYP21A2 | 11-Deoxycortisol, DOC | **21-OH deficiency (95% CAH)** |
| **CYP11B1** (11β-hydroxylase) | CYP11B1 | Cortisol | 11β-OH deficiency |
| **CYP11B2** (Aldosterone synthase) | CYP11B2 | Aldosterone | — |

## 3. Regulation & Secretion Patterns

| Aspect | Details |
|--------|---------|
| **CRH** | Hypothalamus (PVN); **Stimulated by**: Stress, Hypoglycaemia, Exercise; **Inhibited by**: Cortisol (negative feedback) |
| **ACTH** | Anterior pituitary (Corticotrophs); **Stimulated by**: CRH, AVP; **Inhibited by**: Cortisol (negative feedback) |
| **Cortisol** | Adrenal cortex (Zona fasciculata); **Stimulated by**: ACTH; **Negative feedback** on CRH + ACTH |

### Secretion Patterns
| Pattern | Details |
|---------|---------|
| **Circadian Rhythm** | **Peak 06:00-08:00** (cortisol ~400-700 nmol/L); **Nadir 23:00-24:00** (<50 nmol/L) |
| **Pulsatile** | ACTH/Cortisol pulses ~hourly; ~15-20 pulses/24h |
| **Stress Response** | Acute stress → ↑ CRH + AVP → ↑ ACTH → ↑ Cortisol (within minutes) |
| **Feedback** | **Long-loop**: Cortisol → ↓ CRH + ACTH; **Short-loop**: ACTH → ↓ CRH |

## 4. HPA Axis Feedback Patterns — Diagnostic Key

| Disorder | Cortisol | ACTH | CRH | Interpretation |
|----------|----------|------|-----|----------------|
| **Normal** | Normal | Normal | Normal | — |
| **Primary AI (Addison)** | ↓↓ | **↑↑** | ↑↑ | Adrenal failure |
| **Secondary AI (Pituitary)** | ↓ | **↓/N** | ↑ | Pituitary failure |
| **Tertiary AI (Hypothalamic/Steroid)** | ↓ | ↓/N | **↓** | Steroid withdrawal, hypothalamic lesion |
| **Cushing Disease** | ↑↑ | **N/↑** | ↓ | Pituitary ACTH adenoma |
| **Adrenal Cushing** | ↑↑ | **↓↓** | ↓↓ | Adrenal autonomy |
| **Ectopic ACTH** | ↑↑ | **↑↑** | ↓ | Ectopic ACTH source |

**Key Diagnostic Rule**: **Primary = Target bad + Tropic opposite**; **Secondary = Both low**; **Tertiary = Same as secondary but CRH low**

## 5. Cushing Syndrome — Differential Diagnosis

### Diagnostic Approach
```
Cushing Suspected → Confirm (24h UFC×2 + Late-night Salivary Cortisol×2 + 1mg DST)
         │
         ├── NEGATIVE → Excluded
         │
         └── POSITIVE → Measure ACTH (09:00, EDTA, Ice)
                 │
                 ├── ACTH LOW (<10 pg/mL) → ACTH-INDEPENDENT
                 │       Unilateral adrenal mass → Adrenal Adenoma/Carcinoma
                 │       Bilateral hyperplasia → PPNAD / AIMAH
                 │
                 └── ACTH NORMAL/HIGH (>10-20) → ACTH-DEPENDENT
                         │
                         ├── HDDST (2mg q6h × 48h)
                         │       Suppresses >50% → **Cushing Disease** (Pituitary)
                         │       No suppression → Ectopic ACTH / Adrenal Cushing
                         │
                         └── If Equivocal → CRH Test / IPSS / ⁶⁸Ga-DOTATATE PET
```

### Cushing Subtypes — Comparison

| Feature | **Cushing Disease** | **Adrenal Cushing** | **Ectopic ACTH** |
|---------|---------------------|---------------------|------------------|
| **ACTH** | Normal / High | **Low / Suppressed** | **High** |
| **HDDST** | **Suppresses >50%** | No suppression | **No suppression** |
| **Imaging** | Pituitary MRI (microadenoma) | Adrenal CT (unilateral/bilateral) | CT Chest/Abdomen; DOTATATE PET |
| **IPSS** | Central:Peripheral ACTH gradient >2 | N/A | N/A |
| **Treatment** | Transsphenoidal surgery | Adrenalectomy | Resection of ectopic source |

## 6. Adrenal Insufficiency — Classification & Diagnosis

### Classification
| Type | ACTH | Cortisol | Key Features |
|------|------|----------|--------------|
| **Primary (Addison)** | **↑↑** | ↓↓ | Hyperpigmentation, ↑K⁺, ↓Na⁺, Hypotension |
| **Secondary (Pituitary)** | ↓/N | ↓ | No hyperpigmentation; Other pituitary deficits; Aldosterone preserved |
| **Tertiary (Hypothalamic/Steroid)** | ↓/N | ↓ | Steroid withdrawal; HPA suppression; Aldosterone preserved |

### Diagnostic Thresholds
| Test | Normal | Adrenal Insufficiency |
|------|--------|----------------------|
| **09:00 Cortisol** | 200-700 nmol/L | <100 nmol/L (diagnostic); 100-500 = equivocal |
| **Synacthen 250µg (30min)** | **≥500-550 nmol/L** | **<400-500 nmol/L** |
| **Synacthen 1µg (30min)** | ≥500-550 nmol/L | <400-500 nmol/L |
| **ITT (Peak Cortisol)** | >500-550 nmol/L | <500-550 nmol/L |

### Synacthen Test Protocol
| Parameter | Standard (250µg) | Low-Dose (1µg) |
|-----------|------------------|----------------|
| **Dose** | 250µg IM/IV | 1µg IV |
| **Sampling** | 0, 30, 60min | 0, 20, 30, 60min |
| **Normal Response** | Cortisol ≥500-550 at 30/60min | Same |
| **AI Diagnosis** | Cortisol <400-500 at 30/60min | Same |
| **Indication** | General screening | Secondary AI (more physiological) |

## 8. Dynamic Testing — HPA Axis

### Synacthen Test (Standard 250µg)
| Time | Cortisol (nmol/L) | Interpretation |
|------|-------------------|----------------|
| **0 min** | 200-700 | Baseline |
| **30 min** | **≥500-550** | **Normal** |
| **60 min** | **≥500-550** | **Normal** |
| **<400-500 at 30/60min** | | **Adrenal Insufficiency** |

### Dexamethasone Suppression Tests

#### Low-Dose DST (LDDST) — Screening
| Protocol | Overnight 1mg at 23:00 → Cortisol 09:00 next day |
|----------|--------------------------------------------------|
| **Cut-off** | **Cortisol <50 nmol/L** = Normal suppression |
| **Non-suppressed** | Cortisol ≥50 nmol/L → Proceed to HDDST |

#### High-Dose DST (HDDST) — Differential
| Protocol | 2mg q6h × 8 doses (48h) OR Overnight 8mg |
|----------|------------------------------------------|
| **Cushing Disease** | **Cortisol suppresses >50%** (or <50% of baseline) |
| **Ectopic ACTH / Adrenal** | **No suppression** (<50% suppression) |

**DST Pitfalls**: Enzyme inducers (Phenytoin, Rifampicin) → False non-suppression; Obesity/Depression/Alcohol → False positive; Pregnancy/OCP → Use salivary cortisol/24h UFC.

### OGTT for Acromegaly (GH Suppression)
| Protocol | 75g Glucose PO → GH at 0, 30, 60, 90, 120min |
|----------|---------------------------------------------|
| **Normal** | **GH suppressed <1 µg/L** at 60-120min |
| **Acromegaly** | **GH not suppressed ≥1 µg/L** |

### Insulin Tolerance Test (ITT) — Gold Standard
| Protocol | Insulin 0.1 U/kg IV → Glucose <2.2 mmol/L |
|----------|------------------------------------------|
| **Cortisol Peak** | **>500-550 nmol/L** = Normal HPA axis |
| **GH Peak** | **>3 µg/L** (adults) = Normal GH reserve |
| **Contraindications** | IHD, CVD, Epilepsy, Severe hypopituitarism, Age >60, Pregnancy |

## 9. Stress-Dose Steroid Protocol

| Stress Level | Hydrocortisone IV | Duration |
|--------------|-------------------|----------|
| **Minor** (Dental, Minor surgery) | 25mg pre-op | Single dose |
| **Moderate** (Major surgery, Sepsis, Trauma) | 50-100mg pre-op → 50mg q6-8h | 24-48h |
| **Major** (Cardiac surgery, Septic shock, Major trauma) | **100mg IV bolus** → 50-100mg q6h | 48-72h |
| **Labour/Delivery** | 100mg IV at onset → 50mg q6h | Until 24h post-delivery |

**Oral Stress Dosing**: Triple usual oral dose for duration of illness.

## 10. Steroid Withdrawal & HPA Recovery

| Steroid Duration | HPA Suppression | Recovery Time |
|------------------|----------------|---------------|
| **<3 weeks** | Minimal | Days |
| **>3 weeks** | **Significant** (adrenal atrophy) | **Weeks-months** |
| **High dose (>40mg pred equivalent)** | Severe | Months |
| **Inhaled/Topical (high dose)** | Moderate | Weeks |

### Stress Dose Protocol (Oral → IV)
| Stress Level | Hydrocortisone IV | Oral Equivalent |
|--------------|-------------------|-----------------|
| **Minor** (Dental, Minor surgery) | 25mg stat | 2-3x usual dose |
| **Moderate** | 50-100mg → 50mg q6-8h × 24-48h | 3x usual dose |
| **Major** | 100mg IV bolus → 50-100mg q6h × 48-72h | N/A (IV required) |

## 11. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **Circadian Cortisol** | Peak 06-08h (400-700 nmol/L); Nadir 23-24h; Loss of rhythm in Cushing |
| **Synacthen 250µg** | Cortisol ≥500-550 at 30/60min = Normal; <400-500 = AI |
| **Synacthen 1µg** | More sensitive for secondary AI; Technically demanding |
| **LDDST** | 1mg overnight; Cortisol <50 = Normal suppression; ≥50 = Cushing possible |
| **HDDST** | 2mg q6h × 48h; >50% suppression = Cushing Disease; No suppression = Ectopic/Adrenal |
| **ACTH in Cushing** | Disease: N/↑; Adrenal: ↓↓; Ectopic: ↑↑ |
| **ITT** | Gold Standard GH+Cortisol; Insulin 0.1U/kg → Glucose <2.2; Contra: IHD, CVD, Epilepsy, Age>60 |
| **Cortisol Cut-offs** | DST: <50; Synacthen: ≥500-550; ITT: >500-550; Morning: 200-700 nmol/L |
| **ITT Contraindications** | IHD, CVD, Epilepsy, Severe hypopit, Age>60, Pregnancy |
| **Steroid Withdrawal** | >3 weeks = HPA suppression; Taper over weeks-months |
| **Stress Dosing** | Minor 25mg; Moderate 50-100mg; Major 100mg + 50mg q6h |
| **Adrenal Crisis** | 100mg Hydrocortisone IV STAT + 1L 0.9% NaCl; Never delay for tests |
| **Primary vs Secondary AI** | Primary: Hyperkalaemia + Hyperpigmentation; Secondary: Neither |

## 12. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **LDDST vs HDDST** | LDDST = **Screening** (is it Cushing?); HDDST = **Differential** (Pituitary vs Ectopic/Adrenal) |
| **Synacthen 250µg vs 1µg** | 250µg = Standard screening; 1µg = More sensitive for secondary/early AI |
| **GH Stimulation** | ITT = Gold standard but dangerous; Glucagon = Safer alternative; GHRH+Arg = Most sensitive for adults |
| **DST Failure** | Most common cause = **Drugs (enzyme inducers)**; Obesity/Depression/Alcohol also |
| **Cortisol Cut-offs** | DST: <50; Synacthen: ≥500-550; ITT: >500-550; Morning: 200-700 nmol/L |
| **ITT vs Glucagon** | ITT tests cortisol + GH simultaneously; Glucagon safer but only GH well-validated |
| **OGTT for Acromegaly** | **Only validated suppression test for acromegaly**; GH <1 µg/L = Normal |

**Mnemonic: HPA AXIS FEEDBACK PATTERNS**
- **Primary AI**: Cortisol↓ + ACTH↑↑ (Opposite)
- **Secondary AI**: Cortisol↓ + ACTH↓ (Same)
- **Cushing Disease**: ACTH N/↑ (Pituitary makes ACTH)
- **Adrenal Cushing**: ACTH↓↓ (Adrenal autonomous)
- **Ectopic ACTH**: ACTH↑↑ (Tumour makes ACTH)

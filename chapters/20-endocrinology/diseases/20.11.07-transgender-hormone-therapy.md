# Transgender Hormone Therapy (Gender-Affirming Hormone Therapy)

> [!info]
> **Gender-Affirming Hormone Therapy (GAHT) = Feminising (Transfeminine) or Masculinising (Transmasculine) Hormone Regimens.** **Goal: Align Secondary Sex Characteristics with Gender Identity.** **Multidisciplinary Care (Endocrinology, Mental Health, Surgery, Primary Care).** **Evidence-Based, Patient-Centred, Harm-Reduction Approach.**

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Prescribe evidence-based feminising and masculinising hormone regimens
- [ ] Apply monitoring protocols (hormones, metabolic, cardiovascular, bone, cancer screening)
- [ ] Counsel on fertility preservation options
- [ ] Manage peri-operative hormone considerations for gender-affirming surgery
- [ ] Apply long-term health surveillance

---

## 2. General Principles

| Principle | Details |
|-----------|---------|
| **Informed Consent Model** | **Capacity-Based** (Not Mandatory Mental Health Assessment in Many Guidelines) |
| **Assessment** | **WPATH SOC8 / Endocrine Society Guidelines** — Persistent Gender Incongruence, Capacity, Medical Optimisation |
| **Mental Health** | **Supportive, Not Gatekeeping** — Co-Morbid Mental Health Assessment/Treatment |
| **Harm Reduction** | **Lower Doses Acceptable** If High Doses Contraindicated; Goal = Affirmation, Not Perfection |
| **Multidisciplinary Team** | Endocrinology, Mental Health, Primary Care, Surgery, Fertility, Social Work |

## 3. Feminising Hormone Therapy (Transfeminine / MTF)

### Goals
| Goal | Target |
|------|--------|
| **Breast Development** | Oestrogen-Driven (2-3 Years for Maximal) |
| **Fat Redistribution** | Gynoid Pattern (Hips, Thighs) |
| **Skin Changes** | Softer, Thinner |
| **Testicular Atrophy** | ↓ Testosterone Production |
| **↓ Body Hair** | Mild Reduction (Laser/Electrolysis Often Needed) |
| **Voice** | **No Pitch Change** (Voice Therapy/Surgery Needed) |

### Hormone Regimens

| Agent | Dose | Route | Notes |
|------|------|-------|-------|
| **Oestradiol (17β-Oestradiol)** | **2-6mg/day** (Start 2mg, Titrate) | **Transdermal Preferred** (Patch/Gel/Spray) | **1st Line**; ↓ VTE Risk |
| **Oestradiol Valerate** | 5-20mg IM q2wk | IM | Alternative |
| **Anti-Androgen (GnRH Agonist)** | **Goserelin 3.6mg SC q4wk** / **Leuprorelin 3.75mg IM q4wk** | SC/IM | **Best** (Suppressed T to Castrate) |
| **Cyproterone Acetate** | 10-50mg/day | PO | **Alternative** (↓ T + Progestogenic); ↓ Bone Density Risk |
| **Spironolactone** | 50-200mg/day | PO | **Common (USA)**; K⁺-Sparing Diuretic; ↑ VTE Risk with Oral Oestrogen |
| **Finasteride / Dutasteride** | 1-5mg/day | PO | ↓ DHT (Hair/Prostate); No T Suppression |

### Preferred Regimens
| Regimen | Components |
|---------|------------|
| **Gold Standard** | **Transdermal Oestradiol + GnRH Agonist** |
| **Alternative (Cost)** | Transdermal Oestradiol + Cyproterone 10-25mg |
| **USA Common** | Oral/Transdermal Oestradiol + Spironolactone 100-200mg |

## 4. Masculinising Hormone Therapy (Transmasculine / FTM)

### Goals
| Goal | Target |
|------|--------|
| **Voice Deepening** | Testosterone-Driven (Irreversible) |
| **Facial/Body Hair** | ↑ Growth (Terminal Hair) |
| **Fat Redistribution** | Android Pattern (Abdomen) |
| **Muscle Mass/Strength** | ↑ |
| **Cessation of Menses** | ↑ Testosterone → Endometrial Atrophy |
| **Clitoral Enlargement** | Irreversible |

### Hormone Regimens

| Agent | Dose | Route | Notes |
|------|------|-------|-------|
| **Testosterone Undecanoate** | **1000mg (4mL) IM q10-14 weeks** | **IM (Nebido)** | **1st Line** (Steady Levels, Long Duration) |
| **Testosterone Enantate/Cypionate** | **250mg IM q2-3 weeks** / **100mg IM weekly** | IM | Alternative; Peaks/Troughs |
| **Testosterone Gel** | **50-100mg/day** (Pump/ Sachet) | **Transdermal** | Physiological; Daily Application |
| **Testosterone Pellets** | 600-1200mg SC q4-6mo | SC Implant | Long-Lasting; Surgical |
| **Dutasteride (Adjunct)** | 0.5mg OD | PO | If Prostate Concerns / Hairline |

## 6. Target Hormone Ranges

| Therapy | Target Range | Notes |
|---------|-------------|-------|
| **Feminising — Oestradiol** | **200-400 pg/mL (730-1500 pmol/L)** | Physiological Female Range |
| **Feminising — Testosterone** | **<50 ng/dL (<1.7 nmol/L)** | Castrate Level |
| **Masculinising — Testosterone** | **400-700 ng/dL (14-24 nmol/L)** | Physiological Male Range |
| **Masculinising — Oestradiol** | **<50 pg/mL (<180 pmol/L)** | Suppressed |

## 7. Contraindications / Cautions

### Absolute Contraindications
| Condition | Feminising | Masculinising |
|---------|------------|---------------|
| **Active VTE / Thrombophilia** | **Absolute** (Oestrogen) | Relative (Testosterone ↑ Hct) |
| **Active Breast Cancer** | **Absolute** | — |
| **Active Prostate Cancer** | — | **Absolute** |
| **Severe Liver Disease** | **Absolute** | **Absolute** |
| **Uncontrolled CVD** | **Absolute** | **Relative** |

### Relative Contraindications (Individualise)
| Condition | Feminising | Masculinising |
|---------|------------|---------------|
| **Age >50 / CVD Risk** | Transdermal Oestrogen | Monitor Hct, Lipids |
| **Migraine with Aura** | Transdermal Oestrogen | Monitor Hct |
| **Obesity (BMI >30)** | Transdermal + GnRH Agonist | Monitor Hct, Sleep Apnoea |
| **Smoking** | **Strongly Counsel Cessation** | **Strongly Counsel Cessation** |
| **Prolactinoma** | Caution (Oestrogen ↑) | — |
| **Porphyria** | Caution | Caution |

## 8. Fertility Preservation (Pre-GAHT)

| Population | Options | Timing |
|-----------|---------|--------|
| **Transfeminine (AMAB)** | **Sperm Cryopreservation** (Before/Soon After GAHT) | **Ideally Before GAHT** |
| **Transmasculine (AFAB)** | **Oocyte/Embryo Cryopreservation** (Before Testosterone) | **Before Testosterone** |
| **Post-GAHT** | May Require Temporary Hormone Cessation (3-6mo) | Variable Recovery |

> **Key**: **Counsel BEFORE Starting GAHT** — Testosterone Suppresses Ovulation/Spermatogenesis Rapidly.

---

## 9. Surgical Considerations

| Surgery | Peri-Operative Hormone Management |
|---------|-----------------------------------|
| **Vaginoplasty** | **Stop Oestrogen 2-4 Weeks Pre-Op** (VTE Risk); Restart Post-Op When Mobile |
| **Phalloplasty / Metoidioplasty** | **Continue Testosterone** (Wound Healing); Monitor Hct |
| **Mastectomy (Top Surgery)** | **Continue Testosterone**; No VTE Risk Change |
| **Breast Augmentation** | **Stop Oestrogen 2-4 Weeks Pre-Op**; Restart Post-Op |
| **Facial Feminisation** | Continue / Adjust Per Surgeon |
| **Tracheal Shave** | Continue / Adjust |

## 12. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Oral vs Transdermal Oestrogen** | Transdermal = ↓ VTE, ↓ Liver First-Pass, ↓ Triglycerides |
| **Cyproterone vs Spironolactone** | Cyproterone = Progestogenic + Anti-Androgen; Spironolactone = K⁺-Sparing Diuretic + Anti-Androgen |
| **GnRH Agonist vs Antagonist** | **Agonist** (Downregulation) = Standard; **Antagonist** = For Fertility/Precocious Puberty |
| **Testosterone Undecanoate vs Enantate** | Undecanoate = q10-14wk (Steady); Enantate = q2-3wk (Peaks/Troughs) |
| **VTE Risk** | Oestrogen ↑ Risk; **Transdermal ↓ VTE vs Oral** |
| **HCT Rise on Testosterone** | **Castrate → Replace** → Hct ↑↑ (Polycythaemia Risk); Monitor q3-6mo |
| **Fertility** | **Preserve BEFORE GAHT** — Testosterone/Oestrogen Rapidly Suppress Gonadal Function |
| **Breast Cancer Screening** | **If >5yr Oestrogen + >50** → Mammogram per Guidelines |
| **Prostate Cancer Screening** | **Not Reduced by Testosterone** (If >50 / Symptoms) |
| **Cervical Screening** | **If Cervix Present (FTM)** — Per Guidelines |
| **Voice** | **Testosterone Deepens** (Irreversible); Oestrogen Does Not Raise Pitch |
| **Menses Cessation** | **Testosterone → Endometrial Atrophy** (Usually 2-6mo) |
| **Bone Density** | DEXA if >50 / Risk Factors (Both Directions) |

# Endocrine Investigation Algorithms

> [!info]
> **A systematic algorithmic approach is the standard of care in endocrinology.** This note provides the step-by-step flowcharts used in clinical practice to move from clinical suspicion → biochemical confirmation → dynamic testing → imaging → genetic testing → MDT discussion.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Apply the universal endocrine investigation algorithm (Paired Hormone → Feedback Logic → Dynamic Test → Imaging → Genetics)
- [ ] Execute specific algorithms for the 12 most common endocrine presentations
- [ ] Avoid common investigation pitfalls (wrong test, wrong timing, wrong interpretation)
- [ ] Structure a viva-ready investigation plan for any endocrine scenario

## 2. The Universal Endocrine Investigation Algorithm

```
CLINICAL SUSPICION OF ENDOCRINE DISORDER
                    │
                    ▼
STEP 1: PAIRED HORMONE MEASUREMENT
   • Measure TARGET HORMONE + TROPIC HORMONE simultaneously
   • Examples: TSH + fT4, ACTH + Cortisol, LH/FSH + Testosterone/Oestradiol, PTH + Calcium
                    │
                    ▼
STEP 2: FEEDBACK LOGIC ANALYSIS
   │
   ├── PRIMARY DISORDER: Target Hormone ABNORMAL + Tropic Hormone OPPOSITE direction
   │     (e.g., High T4 + Low TSH = Primary Hyperthyroidism)
   │
   ├── SECONDARY DISORDER: Target Hormone LOW + Tropic Hormone LOW/INAPPROPRIATELY NORMAL
   │     (e.g., Low T4 + Low/Normal TSH = Central Hypothyroidism)
   │
   └── TERTIARY DISORDER: Same as secondary but RELEASING HORMONE LOW (steroid withdrawal, hypothalamic lesion)
                    │
                    ▼
STEP 3: DYNAMIC TESTING (if baseline equivocal or need confirmation)
   ├── DEFICIENCY SUSPECTED → STIMULATION TEST
   │     (Synacthen for AI, ITT/Glucagon for GHD, GnRH for Hypogonadism, TRH for Central Hypo)
   │
   ├── EXCESS/AUTONOMY SUSPECTED → SUPPRESSION TEST
   │     (LDDST/HDDST for Cushing, OGTT for Acromegaly, Saline/Fludro for Conn)
   │
   └── DI/WEAKNESS → PROVOCATION TEST
         (Water Deprivation + DDAVP, ITT for GH/Cortisol)
                    │
                    ▼
STEP 4: LOCALISATION IMAGING
   • Pituitary: MRI Pituitary (with gadolinium, 3mm slices)
   • Adrenal: CT Abdomen (adrenal protocol, 1mm slices, washout)
   • Thyroid: US Thyroid (TI-RADS) → FNA (Bethesda)
   • Adrenal (functioning): Adrenal Venous Sampling (AVS) for Conn
   • NET: ⁶⁸Ga-DOTATATE PET/CT (gold standard)
                    │
                    ▼
STEP 5: GENETIC TESTING (if syndromic, familial, young onset)
   • MEN1, RET, MEN1, CDKN1B, CaSR, GNAS, GNAS1, KCNJ11, ABCC8, etc.
                    │
                    ▼
STEP 6: MDT DISCUSSION
   • Endocrinologist, Surgeon, Radiologist, Pathologist, Geneticist, Oncologist (if malignant)
                    │
                    ▼
TREATMENT PLAN + SURVEILLANCE SCHEDULE
```

## 3. Presentation-Specific Algorithms

### Algorithm 1: HYPERTHYROIDISM

```
Clinical Thyrotoxicosis (Symptoms + Signs)
         │
         ▼
TSH + fT4 + fT3
         │
         ├── TSH SUPPRESSED + fT4/fT3 ↑ → PRIMARY HYPERTHYROIDISM
         │       │
         │       ├── TRAb POSITIVE → GRAVES DISEASE
         │       │       Management: Carbimazole/PTU → RAI or Surgery
         │       │
         │       ├── TRAb NEGATIVE + Thyroid US + RAI Uptake
         │       │       ├── Diffuse ↑ uptake → GRAVES (Seronegative)
         │       │       ├── Focal ↑ uptake → TOXIC ADENOMA
         │       │       └── Multifocal ↑ uptake → TOXIC MULTINODULAR GOITRE
         │       │
         │       ├── LOW RAI Uptake + Painful Thyroid → SUBACUTE THYROIDITIS
         │       ├── LOW RAI Uptake + Painless + Postpartum → POSTPARTUM THYROIDITIS
         │       ├── LOW RAI Uptake + Amiodarone/Lithium → DRUG-INDUCED
         │       └── LOW RAI Uptake + No Pain → SILENT THYROIDITIS / FACTITIOUS
         │
         └── TSH NORMAL/HIGH + fT4/fT3 ↑ → SECONDARY HYPERTHYROIDISM (TSH-OMA)
                 TSH NORMAL/HIGH + fT4/fT3 ↑ + Pituitary MRI → TSH-SECRETING ADENOMA
```

### Algorithm 2: HYPOTHYROIDISM

```
Symptoms + Signs of Hypothyroidism
         │
         ▼
TSH + fT4
         │
         ├── TSH HIGH + fT4 LOW → PRIMARY HYPOTHYROIDISM
         │       ├── Anti-TPO/TgAb POSITIVE → HASHIMOTO THYROIDITIS
         │       ├── Post-thyroidectomy / Post-RAI → IATROGENIC
         │       ├── Iodine Deficiency / Excess → NUTRITIONAL
         │       └── Drug-Induced (Lithium, Amiodarone, Tyrosine Kinase Inhibitors)
         │
         ├── TSH LOW/NORMAL + fT4 LOW → CENTRAL HYPOTHYROIDISM
         │       ├── Pituitary MRI → Macroadenoma / Empty Sella / Infiltration
         │       ├── Post-Surgery / Radiotherapy → IATROGENIC
         │       └── Genetic (PROP1, POU1F1, LHX3) → CONGENITAL
         │
         └── TSH HIGH + fT4 NORMAL → SUBCLINICAL HYPOTHYROIDISM
                 TSH >10 → Treat; TSH 5-10 + Ab+/Pregnant/Lipids/Symptoms → Treat; Else Monitor
```

### Algorithm 3: THYROID NODULE

```
Thyroid Nodule Detected (Clinical or Incidental)
         │
         ▼
TSH
         │
         ├── TSH SUPPRESSED → HYPERFUNCTIONING ("HOT") NODULE
         │       → RAI Scan → Focal Uptake → TOXIC ADENOMA → Surgery/RAI
         │
         └── TSH NORMAL/HIGH → "COLD" NODULE
                 │
                 ▼
             THYROID ULTRASOUND (TI-RADS)
                 │
                 ├── TR1/TR2 → BENIGN → Surveillance (1-2yr)
                 ├── TR3 → FNA if ≥2.5cm
                 ├── TR4 → FNA if ≥1.5cm
                 └── TR5 → FNA if ≥1cm
                 │
                 ▼
             FNA CYTOLOGY (BETHESDA)
                 │
                 ├── I (Non-diagnostic) → Repeat FNA
                 ├── II (Benign) → Surveillance
                 ├── III (AUS/FLUS) → Repeat FNA / Molecular Testing
                 ├── IV (FN/SFN) → Diagnostic Lobectomy / Molecular
                 ├── V (Suspicious) → Near-Total Thyroidectomy
                 └── VI (Malignant) → Total Thyroidectomy ± RAI
```

### Algorithm 4: CUSHING SYNDROME

```
Clinical Suspicion (Features + Screening)
         │
         ▼
CONFIRM: 24h UFC ×2 + Late-night Salivary Cortisol ×2 + 1mg DST
         │
         ├── NEGATIVE → Cushing Excluded
         │
         └── POSITIVE → CONFIRMED CUSHING
                 │
                 ▼
         MEASURE PLASMA ACTH (09:00, EDTA, Ice)
                 │
                 ├── ACTH LOW (<10 pg/mL) → ACTH-INDEPENDENT
                 │       ├── Adrenal CT → Unilateral Mass → ADRENAL ADENOMA/CARCINOMA → Surgery
                 │       ├── Bilateral Hyperplasia → PPNAD / AIMAH → Genetic Testing
                 │       └── No Mass → Ectopic CRH (Rare)
                 │
                 └── ACTH NORMAL/HIGH (>10-20) → ACTH-DEPENDENT
                         │
                         ├── HDDST (2mg q6h × 48h)
                         │       ├── SUPPRESSES >50% → CUSHING DISEASE
                         │       │       MRI Pituitary (3mm slices)
                         │       │       ├── Microadenoma (<10mm) → Transsphenoidal Surgery
                         │       │       └── Macroadenoma → Surgery ± IPSS if equivocal
                         │       │
                         │       └── NO SUPPRESSION → ECTOPIC ACTH / ADRENAL
                         │               │
                         │               ├── ACTH HIGH → ECTOPIC ACTH
                         │               │       CT Chest/Abd/Pelvis → ⁶⁸Ga-DOTATATE PET → Bronchoscopy
                         │               │
                         │               └── ACTH LOW → ADRENAL CUSHING (Re-check Imaging)
                         │
                         └── IF HDDST EQUIVOCAL → CRH TEST / IPSS / ⁶⁸Ga-DOTATATE PET
```

### Algorithm 5: ADRENAL INSUFFICIENCY

```
Clinical Suspicion (Fatigue, Hypotension, Hyponatraemia, Hyperkalaemia, Hyperpigmentation)
         │
         ▼
0900 CORTISOL + ACTH (Paired)
         │
         ├── CORTISOL <100 + ACTH HIGH → PRIMARY AI (ADDISON)
         │       Anti-21-OH Ab / Adrenal CT → AUTOIMMUNE / TB / HIV / HAEMORRHAGE
         │
         ├── CORTISOL <100 + ACTH LOW/NORMAL → SECONDARY/TERTIARY AI
         │       ├── MRI Pituitary → Macroadenoma / Empty Sella / Infiltration
         │       ├── Post-Surgery/Radiotherapy → IATROGENIC
         │       └── Steroid Withdrawal → TERTIARY (HPA Suppression)
         │
         ├── CORTISOL 100-500 + ACTH HIGH → EARLY PRIMARY AI
         │
         ├── CORTISOL 100-500 + ACTH LOW/NORMAL → EQUIVOCAL
         │       → SYNACTHEN TEST (250µg or 1µg)
         │
         └── CORTISOL >500 + ACTH LOW → NORMAL
                 (If clinical suspicion persists → Synacthen Test)
```

### Algorithm 5b: ADRENAL CRISIS — EMERGENCY ALGORITHM

```
ACUTE HYPOTENSION + HYPONATRAEMIA + HYPERKALAEMIA + HYPOGLYCAEMIA
         IN PATIENT WITH KNOWN/UNKNOWN ADRENAL INSUFFICIENCY
         │
         ▼
IMMEDIATE: HYDROCORTISONE 100mg IV STAT + 1L 0.9% NaCl OVER 1 HOUR
         │
         ├── HYDROCORTISONE 50-100mg IV Q6H
         ├── FLUIDS: 2-3L 0.9% NaCl OVER 24H
         ├── GLUCOSE 5-10% IF HYPOGLYCAEMIC
         ├── MONITOR: U&Es, Glucose Q1-2H, CARDIAC MONITOR
         ├── TREAT PRECIPITANT (ANTIBIOTICS, FLUIDS, STRESS DOSES)
         └── DO NOT DELAY HYDROCORTISONE FOR TESTS
```

### Algorithm 6: HYPERCALCAEMIA

```
Corrected Calcium >2.6 mmol/L (or Ionised >1.3)
         │
         ▼
PTH + Phosphate + Alk Phos + 25-OH Vit D + U&Es + Protein
         │
         ├── PTH HIGH / INAPPROPRIATELY NORMAL → PTH-MEDIATED
         │       ├── 24h Urine Calcium/Creatinine >0.02 → PRIMARY HYPERPARATHYROIDISM
         │       │       Sestamibi/US → Surgery (if criteria met)
         │       ├── 24h Urine Ca/Cr <0.02 → FHH (CaSR MUTATION)
         │       │       Genetic Testing → Family Screening
         │       └── PTH HIGH + CKD/Hx SECONDARY → TERTIARY HPT → Surgery
         │
         └── PTH SUPPRESSED (<10 pg/mL) → NON-PTH MEDIATED
                 ├── PTHrP HIGH → HUMORAL HYPERCALCAEMIA OF MALIGNANCY
                 │       (Squamous Ca, Breast, RCC, Bladder, Ovarian)
                 ├── 1,25-OH Vit D HIGH → GRANULOMATOUS (Sarcoid, TB, Lymphoma)
                 ├── 25-OH Vit D HIGH → VIT D TOXICITY
                 └── MEDICATIONS (Thiazides, Lithium, Vit A, Theophylline)
```

### Algorithm 7: HYPOCALCAEMIA

```
Corrected Calcium <2.15 mmol/L (or Ionised <1.1)
         │
         ▼
PTH + Phosphate + Alk Phos + 25-OH Vit D + Mg + U&Es
         │
         ├── PTH LOW → HYPOPARATHYROIDISM
         │       ├── Post-Thyroidectomy → TRANSIENT (days-weeks) vs PERMANENT
         │       ├── AUTOIMMUNE (APS-1, CaSR Ab) → Chronic
         │       ├── MAGNESIUM DEFICIENCY → Functional Hypoparathyroidism
         │       └── AUTOIMMUNE POLYGLANDULAR SYNDROME
         │
         ├── PTH HIGH → SECONDARY HPT / PSEUDOHYPOPARATHYROIDISM
         │       ├── CKD → Phosphate Binders, Vit D Analogues, Cinacalcet
         │       ├── VIT D DEFICIENCY (25-OH D <30) → Cholecalciferol/Calcifediol
         │       ├── PSEUDOHYPOPARATHYROIDISM (PHP1a) → PTH ↑, Ca ↓, Phos ↑, GNAS
         │       └── ACUTE PANCREATITIS → Ca Saponification
         │
         └── PTH INAPPROPRIATELY NORMAL → Mg Deficiency / Early Secondary
                 Check Mg → Replace Mg first
```

### Algorithm 8: HYPERNATRAEMIA / DIABETES INSIPIDUS

```
Serum Na+ >145 mmol/L + Polyuria (>3L/day) + Thirst
         │
         ▼
Serum Osmolality + Urine Osmolality + Urine Na+
         │
         ├── Serum Osm HIGH (>295) + Urine Osm LOW (<300) → DIABETES INSIPIDUS
         │       │
         │       ├── WATER DEPRIVATION TEST + DESMOPRESSIN
         │       │       ├── URINE OSM ↑ >50% → CENTRAL DI → Desmopressin
         │       │       └── NO RESPONSE → NEPHROGENIC DI → Thiazide + Low Salt + Amiloride
         │       │
         │       └── CAUSES: Pituitary Surgery/Trauma/Infiltration (Central); Li/Demeclocycline/HyperCa (Nephrogenic)
         │
         └── Serum Osm HIGH + Urine Osm HIGH → OSMOTIC DIURESIS (Hyperglycaemia, Mannitol, Urea)
```

---

### Algorithm 8b: HYPONATRAEMIA / SIADH

```
Serum Na+ <135 mmol/L
         │
         ▼
ASSESS VOLUME STATUS
         │
         ├── HYPOVOLAEMIC (Low JVP, Dry Mucosa, Low BP)
         │       Urine Na+ <20 → EXTRA-RENAL LOSSES (GI, Skin, Third Space)
         │       Urine Na+ >20 → RENAL LOSSES (Diuretics, AKI, Cerebral Salt Wasting, Adrenal Insufficiency)
         │
         ├── HYPERVOLAEMIC (Oedema, High JVP)
         │       Heart Failure / Cirrhosis / Nephrotic Syndrome
         │
         └── EUVOLAEMIC (Normal JVP, No Oedema)
                 │
                 ├── Serum Osm <275 + Urine Osm >100 + Urine Na+ >20 → SIADH
                 │       Exclude: Hypothyroidism, Adrenal Insufficiency, Drugs (SSRI, Carbamazepine)
                 │       Management: Fluid Restriction <1L → Demeclocycline → Tolvaptan (Severe)
                 │
                 ├── Hypothyroidism / Adrenal Insufficiency → Treat Primary
                 ├── RENAL FAILURE → Dialysis
                 ├── BEER POTOMANIA / TEA-TOAST DIET → Low Solute Intake
                 └── RESET OSMOSTAT → Asymptomatic, Stable Na+ 125-130
```

### Algorithm 9: HYPOKALAEMIA

```
Serum K+ <3.5 mmol/L
         │
         ▼
ASSESS: BP, Acid-Base, Urine K+, Creatinine, Medications
         │
         ├── HYPERSECRETION OF ALDOSTERONE (Primary Aldosteronism)
         │       BP HIGH + Metabolic Alkalosis + Low Renin
         │       → ARR (Aldosterone/Renin Ratio) >70 → Confirmatory Tests → CT Adrenal → AVS
         │
         ├── RENAL LOSSES Without Aldosterone Excess
         │       Diuretics (Loop/Thiazide) → Most Common
         │       RTA (Type 1/2), Tubulointerstitial Disease
         │       Magnesium Deficiency (Refractory Hypokalaemia) → Replace Mg First
         │
         ├── GI LOSSES
         │       Diarrhoea, Vomiting, Laxative Abuse → Urine K+ <20 (Appropriate Conservation)
         │
         ├── INTRACELLULAR SHIFT
         │       Insulin, Beta-agonists, Alkalosis, Refeeding Syndrome
         │
         └── HYPOKALAEMIC PERIODIC PARALYSIS (Genetic, Thyrotoxicosis)
```

### Algorithm 10: INCIDENTAL ADRENAL MASS

```
Adrenal Incidentaloma on Imaging
         │
         ▼
HORMONAL WORKUP (All Three)
         │
         ├── 1mg DST (Cushing) → Cortisol <50 nmol/L = Excluded
         ├── Metanephrines (Phaeo) → Plasma Free Metanephrines Normal = Excluded
         ├── Aldosterone/Renin (Conn) → ARR Normal = Excluded
         │
         ├── ALL NORMAL → NON-FUNCTIONING
         │       │
         │       ├── SIZE <4cm + BENIGN FEATURES (<10 HU, Washout >60%) → SURVEILLANCE
         │       │       CT 6-12mo → Yearly ×2 → Hormonal Recheck 1yr
         │       │
         │       └── SIZE ≥4cm OR SUSPICIOUS (>10 HU, Washout <40%, Heterogeneous) → SURGERY
         │
         └── ANY ABNORMAL → FUNCTIONING → SPECIFIC MANAGEMENT
                 ├── Cushing → Surgery
                 ├── Phaeo → Alpha-blockade → Surgery
                 ├── Conn → AVS → Surgery (Unilateral) / Medical (Bilateral)
```

### Algorithm 10b: PHAEOCHROMOCYTOMA — PRE-OP PREPARATION

```
Confirmed Phaeochromocytoma (Biochemistry + Imaging)
         │
         ▼
ALPHA-BLOCKADE (Phenoxybenzamine) 10mg BD → TITRATE 20-40mg BD OVER 2+ WEEKS
         │
         ├── TARGET: SBP <130/80 SUPINE, <120/70 STANDING
         ├── HEART RATE 60-80 BPM
         │
         └── AFTER ALPHA-BLOCKADE ESTABLISHED → BETA-BLOCKADE (Propranolol/Metoprolol)
                 (NEVER BETA-BLOCKADE FIRST — UNOPPOSED α → HYPERTENSIVE CRISIS)
         │
         │
         ▼
SURGERY (Laparoscopic Adrenalectomy)
         │
         ├── INTRAOP: PROPHYLACTIC OCTREOTIDE 200-500µg IV (PREVENT CARCINOID-LIKE CRISIS)
         ├── POST-OP: MONITOR GLUCOSE, BP, ELECTROLYTES
         └── GENETIC TESTING (RET, VHL, SDHx, NF1, MAX, TMEM127) → FAMILY SCREENING
```

### Algorithm 11: PROLACTINOMA

```
Hyperprolactinaemia (Prolactin >500 mIU/L)
         │
         ▼
EXCLUDE CAUSES:
         • Drugs (Antipsychotics, Metoclopramide, Antidepressants, OCP)
         • Hypothyroidism (TRH-driven)
         • Renal Failure
         • Macroprolactinaemia (PEG Precipitation)
         • Pregnancy
         • Nipple Stimulation/Stress
         │
         ▼
MRI PITUITARY (3mm slices, Gadolinium)
         │
         ├── MICROADENOMA (<10mm) → CABERGOLINE 0.25mg BD → TITRATE
         │       Target: Prolactin Normal + Tumour Shrinkage
         │       WITHDRAWAL: After 2-3yr Normal Prolactin + No Tumour on MRI
         │
         ├── MACROADENOMA (≥10mm) → CABERGOLINE (Higher Doses) → Surgery if Resistant/Intolerant/Apoplexy
         │       VISUAL FIELDS MANDATORY (Bi-temporal Hemianopia)
         │
         └── NO TUMOUR + Prolactin >1000 → MACROPROLACTINAEMIA OR STALK EFFECT (Large Non-Functioning)
```

### Algorithm 12: MALE HYPOGONADISM

```
Symptoms (Low Libido, ED, Fatigue, Infertility, Reduced Hair, Osteoporosis)
         │
         ▼
TOTAL TESTOSTERONE (09:00, Fasting) ×2
         │
         ├── Testosterone <12 nmol/L (or <8 on 2 occasions) + Symptoms → HYPOGONADISM
         │       │
         │       ├── LH/FSH HIGH → PRIMARY (TESTICULAR)
         │       │       Karyotype (Klinefelter), Mumps, Chemo/Radiotherapy, Autoimmune, Genetic
         │       │
         │       └── LH/FSH LOW/NORMAL → SECONDARY (PITUITARY/HYPOTHALAMIC)
         │               MRI Pituitary → Macroadenoma / Hyperprolactinaemia / Kallmann / Functional
         │
         └── Testosterone 12-15 + Symptoms → BORDERLINE
                 Repeat + Free Testosterone (SHBG) + LH/FSH
```

## 6. Flashcards
- **Q: Endocrine axis logic**
  **A: High target + High tropic = Primary hyperfunction; High target + Low tropic = Autonomous; Low target + High tropic = Primary failure; Low target + Low tropic = Central failure**

- **Q: Stimulation vs Suppression**
  **A: Deficiency → Stimulation test; Excess → Suppression test**

- **Q: Paired hormones always**

- **Q: Cushing algorithm**
  **A: 1mg DST → Late-night salivary cortisol → ACTH → Imaging/IPSS**

- **Q: AI algorithm**
  **A: 0900 cortisol → Synacthen → ACTH → Primary vs Secondary/Tertiary**

- **Q: Thyroid algorithm**
  **A: TSH → fT4/fT3 → TRAb/RAI → Graves vs Toxic nodule vs Thyroiditis**

- **Q: Pituitary algorithm**
  **A: Hormone panel → MRI ± Visual fields → Micro vs Macro, Functioning vs Non-functioning**

- **Q: Thyroid nodule**
  **A: TI-RADS US → FNA Bethesda → Risk stratification → Management**

# Anterior Pituitary Hormones Overview

> [!info]
> **Anterior pituitary (adenohypophysis) produces 6 key hormones** from 5 distinct cell types. Understanding each hormone's regulation, target, and clinical effects is fundamental to all pituitary pathology.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Identify the 5 anterior pituitary cell types and their hormones
- [ ] Describe the regulation of each hormone (releasing/inhibiting factors)
- [ ] State the physiological actions of each hormone
- [ ] Apply feedback loop logic to anterior pituitary axes
- [ ] Recognise clinical features of excess and deficiency for each hormone

## 2. Anterior Pituitary Cell Types & Hormones

| Cell Type | % of Cells | Hormone(s) | Abbreviation | Target Organ |
|-----------|------------|------------|--------------|--------------|
| **Corticotroph** | 15-20% | Adrenocorticotropic Hormone | **ACTH** | Adrenal Cortex (Cortisol) |
| **Thyrotroph** | 5-10% | Thyroid-Stimulating Hormone | **TSH** | Thyroid (T4/T3) |
| **Somatotroph** | 40-50% | Growth Hormone | **GH** | Liver (IGF-1), Tissues |
| **Gonadotroph** | 10-15% | Follicle-Stimulating Hormone | **FSH** | Gonads (Folliculogenesis/Spermatogenesis) |
| | | Luteinizing Hormone | **LH** | Gonads (Ovulation/Leydig) |
| **Lactotroph** | 15-20% | Prolactin | **PRL** | Breast (Lactation) |

**Note**: Gonadotrophs produce both FSH and LH. Total = 6 distinct hormones.

## 3. Hypothalamic Regulation of Anterior Pituitary

| Axis | Releasing Hormone | Inhibiting Hormone | Pituitary Hormone | Target Hormone |
|------|-------------------|-------------------|-------------------|----------------|
| **HPA** | **CRH** (Corticotropin-Releasing Hormone) | — | ACTH | Cortisol |
| **HPT** | **TRH** (Thyrotropin-Releasing Hormone) | Somatostatin, Dopamine | TSH | T4/T3 |
| **GH/IGF-1** | **GHRH** (Growth Hormone-Releasing Hormone) | **Somatostatin** (GHIH) | GH | IGF-1 (Liver) |
| **HPG** | **GnRH** (Gonadotropin-Releasing Hormone) | — | FSH + LH | Oestradiol/Testosterone |
| **Prolactin** | **TRH** (stimulatory) | **Dopamine** (PIH, Prolactin-Inhibiting Hormone) | Prolactin | — (Breast) |

**Portal Circulation**: Hypothalamic hormones travel via hypophyseal portal vessels to anterior pituitary.

## 4. Individual Hormone Profiles

### ACTH (Corticotropin)
| Feature | Details |
|---------|---------|
| **Precursor** | **POMC** (Pro-opiomelanocortin) → ACTH + β-LPH → β-Endorphin + γ-LPH |
| **Regulation** | **CRH** (stimulatory); **Cortisol** (negative feedback at pituitary + hypothalamus) |
| **Secretion** | Pulsatile, **Diurnal peak 06:00-08:00**, nadir 23:00-24:00; Stress → ↑ |
| **Target** | Adrenal Cortex (Zona fasciculata/reticularis) → **Cortisol** |
| **Clinical Excess** | **Cushing Syndrome** (ACTH-dependent: Cushing disease, ectopic ACTH) |
| **Clinical Deficiency** | **Secondary Adrenal Insufficiency** (pituitary failure) |

---

### TSH (Thyroid-Stimulating Hormone)
| Feature | Details |
|---------|---------|
| **Structure** | **α-subunit** (shared with LH, FSH, hCG) + **β-subunit** (TSH-specific) |
| **Regulation** | **TRH** (stimulatory); **T3/T4** (negative feedback, mainly T3 at pituitary); Somatostatin, Dopamine (inhibitory) |
| **Secretion** | Pulsatile; **Diurnal peak 02:00-04:00**; Cold exposure → ↑ |
| **Target** | Thyroid (TSH-R) → **T4/T3** synthesis & release |
| **Clinical Excess** | **Secondary Hyperthyroidism** (TSH-oma); **TSH-secreting adenoma** |
| **Clinical Deficiency** | **Central Hypothyroidism** (pituitary/hypothalamic failure) |

---

### GH (Growth Hormone / Somatotropin)
| Feature | Details |
|---------|---------|
| **Structure** | 191 amino acid polypeptide; 22kDa (major) + 20kDa variant |
| **Regulation** | **GHRH** (stimulatory); **Somatostatin (GHIH)** (inhibitory); **Ghrelin** (stimulatory); **IGF-1** (negative feedback); Glucose/Insulin (inhibitory) |
| **Secretion** | **Pulsatile**; **Major peak during slow-wave sleep**; Stress, Exercise, Hypoglycaemia, Fasting → ↑; Glucose, Obesity, Ageing → ↓ |
| **Target** | **Liver** (IGF-1 production); Direct on tissues (lipolysis, anti-insulin) |
| **Clinical Excess** | **Acromegaly** (adult); **Gigantism** (child, pre-epiphyseal fusion) |
| **Clinical Deficiency** | **GHD**: Child = Growth failure; Adult = Metabolic syndrome, ↓ QoL |

---

### FSH (Follicle-Stimulating Hormone)
| Feature | Details |
|---------|---------|
| **Structure** | α-subunit (shared) + **β-subunit** (FSH-specific) |
| **Regulation** | **GnRH** (stimulatory); **Inhibin B** (ovary/testis) → **Negative feedback** on FSH selectively; Oestradiol/Testosterone (general) |
| **Secretion** | Pulsatile; **Follicular phase**: FSH > LH; **Mid-cycle**: LH > FSH |
| **Target** | **Ovary**: Granulosa cells → Folliculogenesis, Oestradiol; **Testis**: Sertoli cells → Spermatogenesis, Inhibin B |
| **Clinical Excess** | Rare; Ovarian hyperstimulation syndrome |
| **Clinical Deficiency** | **Hypogonadotropic hypogonadism**; Infertility, Absent spermatogenesis |

---

### LH (Luteinizing Hormone)
| Feature | Details |
|---------|---------|
| **Structure** | α-subunit (shared) + **β-subunit** (LH-specific) |
| **Regulation** | **GnRH** (stimulatory); Oestradiol/Testosterone (negative feedback); **Positive feedback** (Oestradiol mid-cycle → LH surge) |
| **Secretion** | Pulsatile; **Mid-cycle surge** (Ovulation); **LH > FSH** in PCOS (LH:FSH >2:1) |
| **Target** | **Ovary**: Theca cells → Androgens → Oestradiol; **Ovulation**; Corpus luteum → Progesterone; **Testis**: Leydig cells → Testosterone |
| **Clinical Excess** | **PCOS** (LH:FSH >2:1); LH-secreting adenoma (rare) |
| **Clinical Deficiency** | Hypogonadotropic hypogonadism; Anovulation, Low testosterone |

---

### Prolactin (PRL)
| Feature | Details |
|---------|---------|
| **Structure** | 199 amino acid polypeptide; **Macroprolactin** (PRL-IgG complex) |
| **Regulation** | **Dopamine (PIH)** = **Tonically inhibitory** (main regulator); **TRH** (stimulatory); **Oestrogen** (stimulates lactotroph hyperplasia); Suckling, Stress, Sleep → ↑ |
| **Secretion** | **Diurnal**: Peak during sleep; **Pregnancy**: ↑ 10-20x (oestrogen-driven); **Postpartum**: Suckling → surges |
| **Target** | **Breast**: Mammogenesis, Lactogenesis, Galactopoiesis; **Gonads**: High PRL → ↓ GnRH → Hypogonadism |
| **Clinical Excess** | **Prolactinoma** (most common functioning pituitary adenoma); **Drug-induced** (antipsychotics, metoclopramide); Hypothyroidism (TRH-driven); Renal failure |
| **Clinical Deficiency** | Rare; Postpartum failure of lactation (Sheehan syndrome) |

## 5. Anterior Pituitary Axes — Quick Reference

| Axis | Releasing Hormone | Pituitary Hormone | Target Hormone | Key Feedback |
|------|-------------------|-------------------|----------------|--------------|
| **HPA** | CRH | ACTH | Cortisol | Cortisol → ↓ CRH + ACTH |
| **HPT** | TRH | TSH | T4/T3 | T3/T4 → ↓ TRH + TSH |
| **GH/IGF-1** | GHRH / Somatostatin | GH | IGF-1 (Liver) | IGF-1 → ↓ GH + GHRH |
| **HPG** | GnRH | FSH + LH | Oestradiol/Testosterone | Steroids + Inhibin → ↓ GnRH + LH/FSH |
| **Prolactin** | TRH (stim) / Dopamine (inhib) | Prolactin | — | Dopamine → ↓ PRL |

## 6. Clinical Patterns of Anterior Pituitary Dysfunction

| Pattern | Hormone Profile | Common Causes |
|---------|----------------|---------------|
| **Pan-hypopituitarism** | All axes deficient | Pituitary surgery, Radiotherapy, Apoplexy, Infiltration (sarcoid, haemochromatosis) |
| **Selective deficiency** | Single axis | Microadenoma, Autoimmune (lymphocytic hypophysitis), Genetic (PROP1, POU1F1) |
| **Functioning adenoma** | Single hormone excess | Prolactinoma, GH-oma (acromegaly), ACTH-oma (Cushing), TSH-oma |
| **Non-functioning adenoma** | Mass effect + hypopituitarism | Gonadotroph/Null cell adenoma (large, compress normal pituitary) |
| **Stalk effect** | ↑ Prolactin (disinhibition) + other axes low | Large non-functioning adenoma compressing stalk → ↓ Dopamine → ↑ PRL |

## 7. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **Anterior pituitary hormones** | ACTH, TSH, GH, FSH, LH, Prolactin (6 hormones) |
| **POMC derivatives** | ACTH, β-LPH, β-Endorphin, γ-LPH |
| **TSH structure** | α-subunit (shared) + β-subunit (unique) |
| **Ghrelin** | Stimulates GH (via GHS-R1a receptor on somatotrophs) |
| **Somatostatin** | Inhibits GH, TSH, Insulin, Glucagon, Gastrin |
| **Dopamine** | **PIH** = Prolactin-Inhibiting Hormone (tonic inhibition) |
| **Prolactin regulation** | **Dopamine (PIH)** = Main inhibitor; TRH = stimulatory; Oestrogen = stimulatory |
| **Prolactinoma** | Most common functioning pituitary adenoma; PRL >1000 → macroprolactinoma |
| **Macroprolactinaemia** | PRL-IgG complex; PEG precipitation to confirm; clinically insignificant |
| **Sheehan syndrome** | Postpartum pituitary necrosis → Panhypopituitarism + failure of lactation |
| **Growth hormone pulsatility** | Major peak in slow-wave sleep; suppressed by glucose, insulin, obesity |
| **IGF-1** | Liver-derived; mediates most GH effects; better screening than random GH |
| **Gonadotrophs** | Produce both FSH and LH; α-subunit shared, β-subunits distinct |
| **Inhibin B** | Selective FSH negative feedback (from Sertoli/granulosa cells) |

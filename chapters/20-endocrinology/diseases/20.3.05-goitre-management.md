# Goitre Management

> [!info]
> **Goitre = Thyroid Enlargement** (WHO: >18mL Women, >25mL Men by US; or Visible/Palpable Neck Swelling). **Aetiological Classification Essential** (Diffuse vs Nodular, Toxic vs Non-Toxic). **TSH First** → Guides Further Workup.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Classify goitre (Diffuse/Nodular, Toxic/Non-Toxic, Endemic/Sporadic)
- [ ] Apply investigation algorithm starting with TSH
- [ ] Manage based on aetiology (Iodine, Autoimmune, Nodular, Malignancy)
- [ ] Apply indications for surgery, radioiodine, and observation

---

## 2. Classification

| Classification | Types |
|----------------|-------|
| **By Morphology** | **Diffuse** (Whole Gland), **Nodular** (Single/Multinodular), **Mixed** |
| **By Function** | **Toxic** (Hyperthyroid), **Non-Toxic (Euthyroid)**, **Hypothyroid** |
| **By Epidemiology** | **Endemic** (Iodine Deficiency, >10% Population), **Sporadic** |

### WHO Goitre Grading (Palpation)
| Grade | Definition |
|-------|------------|
| **Grade 0** | Not Palpable, Not Visible |
| **Grade 1** | Palpable but Not Visible (Neck Extended) |
| **Grade 2** | Visible (Neck Normal) + Palpable |

---

## 3. Aetiological Classification

| Category | Causes |
|----------|--------|
| **Iodine Deficiency** | **Commonest Worldwide (Endemic Goitre)** |
| **Autoimmune** | **Hashimoto (Hypothyroid)**, **Graves (Toxic)** |
| **Nodular** | **Multinodular Goitre (MNG)**, **Toxic Nodule**, **Toxic MNG** |
| **Physiological** | Puberty, Pregnancy (hCG, ↑ TBG) |
| **Drug-Induced** | **Lithium, Amiodarone, Iodine Excess** |
| **Congenital** | Dyshormonogenesis (Dyshormonogenic CH) |
| **Malignancy** | Thyroid Cancer (Papillary, Follicular, Medullary, Anaplastic) |
| **Infiltrative** | Sarcoidosis, Amyloid, Riedel Thyroiditis, Lymphoma |
| **Genetic** | Pendred Syndrome, TSHR Mutations, NIS Defects |

---

## 4. Investigation Algorithm

```
GOITRE DETECTED (Clinical / US / Incidental)
         │
         ▼
**TSH + fT4 + fT3** (First-Line)
         │
         ├── **TSH LOW** → Hyperthyroid Goitre
         │       ├── TRAb Positive → **GRAVES DISEASE**
         │       ├── TRAb Negative + Nodule → **TOXIC NODULE / TOXIC MNG**
         │       │       (Technetium Scan: Hot vs Cold)
         │       └── **DRUG-INDUCED** (Amiodarone, Lithium)
         │
         ├── **TSH HIGH** → Hypothyroid Goitre
         │       ├── TPOAb Positive → **HASHIMOTO THYROIDITIS**
         │       ├── TPOAb Negative → **IODINE DEFICIENCY / DYSHORMONOGENESIS**
         │       └── **DRUG-INDUCED** (Lithium, Amiodarone)
         │
         └── **TSH NORMAL** → Euthyroid Goitre
                 │
                 ├── **MULTINODULAR GOITRE (MNG)** — Most Common
                 ├── **SOLITARY NODULE** — TI-RADS Assessment
                 ├── **IODINE DEFICIENCY** (Endemic Areas)
                 ├── **INFILTRATIVE** (Sarcoid, Amyloid, Riedel, Lymphoma)
                 └── **MALIGNANCY** (If Suspicious Features)
```

## 5. Further Investigations (After TSH)

| Test | Indication |
|------|------------|
| **Thyroid Ultrasound (TI-RADS)** | **All Nodular Goitres**; Size, Echogenicity, Vascularity, Lymph Nodes |
| **Thyroid Antibodies (TPOAb, TgAb, TRAb)** | Autoimmune Aetiology Suspected |
| **Radioisotope Scan (Tc-99m)** | **Toxic Nodular Goitre** (Hot vs Cold Nodules) |
| **FNA (Bethesda)** | Suspicious Nodules (TI-RADS 3-5) |
| **CT/MRI** | Retrosternal Extension, Tracheal Compression, Invasive Features |
| **Urinary Iodine** | Population Assessment (Not Routine Individual) |
| **Calcitonin** | If Medullary Cancer Suspected (Family Hx, Nodule) |

---

## 6. Euthyroid Multinodular Goitre (MNG) — Most Common

### Management Options
| Option | Indication |
|--------|------------|
| **Observation** | **Asymptomatic, Benign US, No Compression**; US q6-12mo, then Annually |
| **Levothyroxine Suppression** | **Not Recommended Routinely** (No Proven Size Reduction; Risk of Iatrogenic Thyrotoxicosis, Bone Loss) |
| **Radioiodine (RAI)** | **Large MNG, Cosmetic Concern, Elderly/Unfit for Surgery**; Dose 150-300 MBq; ↓ Volume 30-50% |
| **Surgery (Thyroidectomy)** | **Compression** (Trachea/Oesophagus), **Retrosternal Extension**, **Cosmetic**, **Suspicion of Malignancy**, **Failure of RAI** |

### Indications for Surgery in MNG
| Indication | Details |
|------------|---------|
| **Compressive Symptoms** | Dyspnoea, Dysphagia, Stridor, Orthopnoea |
| **Retrosternal Extension** | >50% Gland Below Sternal Notch; CT for Assessment |
| **Cosmetic Concern** | Visible Neck Swelling (Patient Choice) |
| **Suspicion of Malignancy** | Bethesda V/VI, Rapid Growth, Fixation, Vocal Cord Palsy |
| **Failure of RAI** | Persistent/Regrowing After RAI |

---

## 7. Endemic Goitre (Iodine Deficiency)

| Aspect | Management |
|--------|------------|
| **Population** | **Universal Salt Iodisation (USI)** — Gold Standard Prevention |
| **Individual** | **Iodine Supplementation** (KI 150-200 µg/day); **Pregnancy/Lactation: 250 µg/day** |
| **Severe Iodine Deficiency** | **Iodised Oil (Lipiodol)** 1-2mL IM/Year (If Salt Iodisation Not Available) |
| **Monitoring** | Median Urinary Iodine Concentration (Target 100-199 µg/L) |

---

## 8. Toxic Goitre Management

| Aetiology | Key Management |
|-----------|----------------|
| **Graves Disease** | ATD (Carbimazole/PTU) → RAI / Surgery (See Graves Note) |
| **Toxic Adenoma** | **RAI (1st Line)**; Surgery if Large/Compression/Refractory |
| **Toxic MNG** | **RAI (1st Line)**; Surgery if Large/Compression/Refractory |
| **Amiodarone-Induced** | Type 1 (Iodine Excess) → ATD + KPerchlorate; Type 2 (Destructive) → Steroids |
| **Lithium-Induced** | Stop Lithium if Possible; ATD if Persistent |

---

## 9. Retrosternal / Substernal Goitre

| Feature | Management |
|---------|------------|
| **Definition** | >50% Thyroid Volume Below Sternal Notch |
| **Imaging** | **CT Chest** (Extent, Tracheal Deviation, Vascular Encroachment) |
| **Symptoms** | Dyspnoea, Dysphagia, Stridor, Superior Vena Cava Syndrome |
| **Surgery** | **Total/Subtotal Thyroidectomy** via Cervical Approach (Usually); Sternotomy Rare |
| **Pre-Op** | **Vocal Cord Assessment** (Laryngoscopy); **CT for Vascular/Tracheal Anatomy** |

---

## 10. Goitre & Pregnancy

| Aspect | Management |
|--------|------------|
| **Physiological Goitre** | **Common (30-40%)**; hCG + ↑ TBG; Resolves Postpartum |
| **Iodine Requirement** | **250 µg/day** (vs 150 µg Non-Pregnant) |
| **New Goitre in Pregnancy** | Assess TSH; FNA if Nodule >1cm Suspicious |
| **Surgery** | **2nd Trimester** (If Urgent); Avoid 1st/3rd if Possible |
| **RAI** | **Contraindicated** |

---

## 11. Riedel Thyroiditis (Fibrosing Variant)

| Feature | Management |
|---------|------------|
| **Presentation** | **Hard, Fixed, "Woody" Thyroid**; Invades Strap Muscles, Trachea, RLN |
| **Diagnosis** | **Biopsy** (Dense Fibrosis, Lymphoplasmacytic Infiltrate); **IgG4-Related Disease** |
| **Complications** | Tracheal Compression, RLN Palsy, Hypoparathyroidism |
| **Medical** | **Prednisolone** (1mg/kg → Taper); **Tamoxifen**; **Rituximab** (IgG4-RD) |
| **Surgical** | **Isthmusectomy / Tracheostomy** for Airway; **Avoid Total Thyroidectomy** (High RLN/Hypocalcaemia Risk) |

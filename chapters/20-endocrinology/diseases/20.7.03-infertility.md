# Infertility

> [!info]
> **Infertility = Failure to Conceive After 12 Months** of Regular Unprotected Intercourse (6 Months if Female >35). **Prevalence: 1 in 7 Couples.** **Male Factor (30-40%), Female Factor (30-40%), Combined (20-30%), Unexplained (10-20%).**

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Apply the infertility investigation algorithm (male + female)
- [ ] Select appropriate ovulation induction agent (letrozole vs clomifene vs gonadotrophins)
- [ ] Indicate IVF/ICSI and interpret success rates

## 2. Definitions & Epidemiology

| Definition | Criteria |
|-----------|----------|
| **Infertility** | Failure to Conceive After **12 Months** Regular Unprotected Intercourse (6 Months if Female ≥35) |
| **Primary Infertility** | Never Conceived |
| **Secondary Infertility** | Previous Conception (Regardless of Outcome) |
| **Prevalence** | **1 in 7 Couples** (15%) |
| **Distribution** | Male 30-40%, Female 30-40%, Combined 20-30%, Unexplained 10-20% |

### Age-Related Decline
| Female Age | Monthly Fecundability | Cumulative 12-Month Pregnancy Rate |
|------------|----------------------|-----------------------------------|
| **<30** | 20-25% | 85-90% |
| **30-34** | 15-20% | 75-80% |
| **35-37** | 10-15% | 60-70% |
| **38-40** | 5-10% | 40-50% |
| **41-42** | <5% | 20-30% |
| **>42** | <5% | <10% |

## 3. Investigation Algorithm

### Initial Assessment (Both Partners)
| Partner | Tests |
|---------|-------|
| **Female** | **TSH, Prolactin, FSH/LH (Day 2-4), Oestradiol (Day 2-4), AMH, Pelvic US (AFC, Uterus), HSG/HyCoSy (Tubal Patency), Cervical Smear/HPV, Rubella/Varicella Immunity, Chlamydia/Gonorrhoea Screen** |
| **Male** | **Semen Analysis** (WHO 2021 Criteria: Volume ≥1.5mL, Conc ≥15M/mL, Progressive Motility ≥32%, Normal Morphology ≥4%), **Hormones** (Testosterone, LH, FSH, Prolactin) if Abnormal Semen |

### Timing of Investigation
| Scenario | Timing |
|---------|--------|
| **Female <35** | After **12 Months** Unprotected Intercourse |
| **Female 35-39** | After **6 Months** |
| **Female ≥40** | **Immediate** |
| **Known Cause** (e.g., Amenorrhoea, Azospermia) | **Immediate** |

## 4. Female Factor Investigation

### Ovulation Assessment
| Method | Details |
|------|---------|
| **Progesterone (Day 21)** | **>30 nmol/L** = Ovulation Confirmed |
| **Urinary LH Kits** | LH Surge Detection (Home) |
| **Basal Body Temperature** | Biphasic Shift (Post-Ovulatory Rise) |
| **Ultrasound (Follicular Tracking)** | Follicle Growth → Rupture (Gold Standard) |

### Tubal Patency
| Test | Indication | Pros/Cons |
|------|-----------|-----------|
| **HSG (Hysterosalpingography)** | **First-Line** | X-Ray; Radiation; Therapeutic (Oil Contrast) |
| **HyCoSy** | No Radiation; Office-Based | Operator-Dependent; False Positives |
| **Laparoscopy + Dye** | Gold Standard | Invasive; Diagnostic + Therapeutic |

### Ovarian Reserve
| Test | Interpretation |
|------|----------------|
| **AMH** | **Best Marker**; **<1.0 ng/mL = Low**; >3.5 = High (PCOS) |
| **AFC (Antral Follicle Count)** | **<5-7 = Low**; **>15 = High (PCOS)** |
| **FSH (Day 2-4)** | **>10 IU/L = Diminished Reserve**; >20 = Severely Diminished |

## 5. Male Factor Investigation

### Semen Analysis (WHO 2021 Criteria)
| Parameter | Lower Reference Limit (5th Percentile) |
|-----------------------------|----------------------------------------|
| **Volume** | **≥1.5 mL** |
| **Concentration** | **≥15 million/mL** |
| **Total Motility** | **≥40%** (Progressive **≥32%**) |
| **Progressive Motility** | **≥32%** |
| **Normal Morphology** | **≥4%** (Strict Kruger Criteria) |
| **Vitality** | **≥58% Live** |
| **pH** | **≥7.2** |

### Additional Male Tests
| Test | Indication |
|------|----------|
| **Hormones** (T, LH, FSH, Prolactin) | Abnormal Semen Analysis |
| **Scrotal US** | Varicocele, Testicular Mass, Obstruction |
| **Genetic** | Karyotype (Klinefelter), Y-Microdeletion (If Severe Oligo/Azoospermia), CFTR (CAVD) |
| **DNA Fragmentation** | Recurrent Miscarriage, Failed IVF, Unexplained Infertility |

## 6. Female Factor Management

### Anovulation (PCOS / Hypothalamic Amenorrhoea)
| Agent | Dose | Protocol | Pregnancy Rate |
|------|------|----------|----------------|
| **Letrozole** | 2.5-5mg Days 2-6 | **1st Line** (PCOS) | **22-25%/Cycle** |
| **Clomifene** | 50-150mg Days 2-6 | 2nd Line | 15-20%/Cycle |
| **Gonadotrophins (rFSH/hMG)** | 75-225 IU/day | 3rd Line / PCOS Resistant | 20-25%/Cycle |
| **Ovulation Trigger** | **hCG 5000-10000 IU** (Lead Follicle ≥18mm) | | |

### Tubal Factor
| Pathology | Management |
|-----------|------------|
| **Proximal Blockage** | Tubal Cannulation (HSG-Guided) |
| **Distal Blockage / Hydrosalpinx** | **Salpingectomy** → IVF (Hydrosalpinx ↓ Implantation 50%) |
| **Periadnexal Adhesions** | Laparoscopic Adhesiolysis → IVF if Severe |

### Endometriosis
| Stage | Management |
|--------|------------|
| **Minimal/Mild (I-II)** | Expectant / Laparoscopic Ablation → Natural / IUI |
| **Moderate/Severe (III/IV)** | Surgery (Excision) → IVF (If Not Conceiving 6-12mo Post-Op) |

### Uterine Factors
| Pathology | Management |
|----------|------------|
| **Fibroids (Submucosal/Intramural >4cm)** | Hysteroscopic/Myomectomy → IVF |
| **Polyps** | Hysteroscopic Polypectomy |
| **Adenomyosis** | Medical (GnRH Agonist) → IVF |
| **Septate Uterus** | Hysteroscopic Septoplasty |
| **Asherman Syndrome** | Hysteroscopic Adhesiolysis + Oestrogen + IUCD (Prevent Re-Adhesion) |

## 7. Male Factor Management

### Lifestyle & Medical
| Intervention | Evidence |
|------------|----------|
| **Weight Loss** (BMI >25) | ↑ Sperm Count/Motility |
| **Antioxidants** (CoQ10, Vit C/E, Zinc, Se, L-Carnitine) | Meta-Analysis: ↑ Sperm Parameters |
| **Smoking Cessation** | ↑ Count, Motility, Morphology |
| **Alcohol Reduction** | <14 Units/Week |
| **Heat Avoidance** | Hot Tubs, Tight Underwear, Laptop on Lap |

### Surgical
| Condition | Procedure |
|---------|----------|
| **Varicocele** (Grade 2-3) | **Microsurgical Varicocelectomy** → ↑ Pregnancy Rate (If Clinical + Abnormal SA) |
| **Obstructive Azoospermia** | Vasovasostomy / Vasoepididymostomy |
| **Ejaculatory Duct Obstruction** | TURED (Transurethral Resection) |

### Assisted Reproduction (Male Factor)
| Scenario | Technique |
|----------|-----------|
| **Mild Oligo/Asthenozoospermia** | **IUI** (If TMSC >5 Million) |
| **Severe Oligo/Astheno/Teratozoospermia** | **ICSI** |
| **Non-Obstructive Azoospermia (NOA)** | **Micro-TESE** → ICSI |
| **Obstructive Azoospermia (OA)** | **TESE/PESA** + **ICSI** |
| **Genetic Risk** | **Karyotype + Y-Microdeletion** Before ICSI |

## 8. Assisted Reproductive Technology (ART)

### IVF vs ICSI Indications
| Technique | Indication |
|----------|------------|
| **IVF** | Tubal Factor, Unexplained, Mild Male Factor, Endometriosis (Mild), Ovulation Induction Failure |
| **ICSI** | **Severe Male Factor** (Oligo/Astheno/Teratozoospermia), Failed Fertilisation IVF, Frozen Sperm, Surgical Sperm Retrieval, PGD/PGS |

### Ovarian Stimulation Protocols (IVF)
| Protocol | Indication | Key Features |
|----------|-----------|--------------|
| **Long Agonist (GnRH-a)** | Standard (Good Responders) | Down-Regulation → Stimulation |
| **Antagonist (GnRH-ant)** | **Preferred** (OHSS Risk ↓, Shorter) | Flexible Start (Day 6 Stim) |
| **Short Agonist** | Poor Responders | Flare Effect |
| **Mild/Minimal Stimulation** | Poor Responders / OHSS Risk | Low Dose Gonadotrophins + Antagonist |

### Trigger & Timing
| Trigger | Dose | Timing |
|---------|------|--------|
| **hCG** | 5000-10,000 IU | 36h Post-Trigger = Oocyte Retrieval |
| **GnRH Agonist Trigger** | 0.2mg Triptorelin / 0.2mg Leuprorelin | **OHSS Prevention** (No LH Surge) |
| **Dual Trigger** | hCG 1000-2000 IU + GnRH-a 0.2mg | Poor Responders / OHSS Risk |

## 9. IVF/ICSI Outcomes & Complications

### Success Rates (HFEA/ESHRE Data)
| Female Age | Live Birth Rate per Embryo Transfer |
|------------|-------------------------------------|
| **<35** | **30-35%** |
| **35-37** | 20-25% |
| **38-39** | 15-20% |
| **40-42** | 10-15% |
| **>42** | **<5%** |

### OHSS (Ovarian Hyperstimulation Syndrome)
| Grade | Criteria | Management |
|---------|-----------|------------|
| **Mild** | Bloating, Mild Abdominal Pain | Outpatient, Oral Fluids |
| **Moderate** | Ascites, Nausea/Vomiting | Admission, IV Fluids, Monitor |
| **Severe** | Tense Ascites, Haemoconcentration, Oliguria, DVT Risk | **ICU**, Paracentesis, Albumin, Anticoagulation, Cabergoline (Dopamine Agonist) |

**Prevention**: GnRH Agonist Trigger, Coasting, Cabergoline, Albumin, Freeze-All Strategy.

## 10. Special Scenarios

### Poor Ovarian Response (POR) — Bologna Criteria
| Criteria (≥2 of 3) | |
|-------------------|---|
| **Age ≥40** / **AMH <1.1 ng/mL** / **AFC <5** | |
| **Previous Poor Response** | ≤3 Oocytes with Standard Stimulation |

| Strategy | Protocol |
|----------------|---------------|
| **Mild Stimulation** | Low-Dose Gonadotrophins + Antagonist |
| **Agonist Flare** | GnRH-a Flare Protocol |
| **DuoStim** | Dual Stimulation (Follicular + Luteal Phase) |
| **Accumulation** | Freeze All Embryos Across Cycles |

### Poor Responder Management
| Strategy | Outcome |
|----------|---------|
| **Mild Stimulation** | Lower Dose Gonadotrophins + Antagonist |
| **GnRH Agonist Flare** | Short Protocol |
| **DuoStim** | Dual Stimulation (Follicular + Luteal) |
| **Embryo Accumulation** | Freeze All Across Cycles |

### Fertility Preservation
| Indication | Method |
|------------|---------|
| **Cancer (Chemo/Radiotherapy)** | Oocyte/Embryo Cryopreservation (Oocyte Vitrification) |
| **Gender Affirmation** | Oocyte/Sperm Cryopreservation Pre-Hormones |
| **Elective (Social)** | Oocyte Cryopreservation (Age <35 Optimal) |

# Thyroid Nodule Evaluation (TI-RADS & Bethesda)

> [!info]
> **Thyroid Nodules are Common (50-60% on Ultrasound).** **TI-RADS Risk Stratification** guides FNA decisions. **Bethesda Classification** standardises cytology reporting. **Malignancy Risk: 5-10% Overall**, Higher in Solitary Nodules, Males, Extremes of Age.

---

## 2. Thyroid Nodule — Clinical Assessment

| Feature | Benign Features | Suspicious Features |
|---------|-----------------|---------------------|
| **Age** | >50 years | **<20 or >70 years** |
| **Sex** | Female | **Male** |
| **Radiation History** | None | **Childhood/Neck Irradiation** |
| **Symptoms** | None | Hoarseness, Dysphagia, Dyspnoea, Rapid Growth |
| **Family History** | None | MTC, MEN2, Familial NMTC |

## 3. Physical Examination

| Feature | Benign | Suspicious |
|---------|--------|------------|
| **Consistency** | Soft, Rubbery | **Hard, Firm** |
| **Mobility** | Mobile | **Fixed to Surrounding Tissue** |
| **Cervical Lymph Nodes** | Not Palpable | **Palpable/Lymphadenopathy** |

## 4. Ultrasound — TI-RADS (ACR) Risk Stratification

### TI-RADS Features & Points

| Feature | Points | Examples |
|---------|--------|----------|
| **Composition** | | |
| Cystic / Almost Cystic | 0 | Anechoic / <25% Solid |
| Spongiform | 0 | Microcystic (>50% Microcysts) |
| Mixed Cystic/Solid | 1 | 25-50% Solid |
| **Solid / Almost Solid** | **2** | **>50% Solid** |
| **Echogenicity** | | |
| Anechoic | 0 | Cystic |
| Hyperechoic / Isoechoic | 1 | Brighter/Same as Strap Muscle |
| **Hypoechoic** | **2** | **Darker than Strap Muscle** |
| **Markedly Hypoechoic** | **3** | **Darker than Adjacent Muscle** |
| **Shape** | | |
| Wider-than-Tall | 0 | Transverse > AP |
| **Taller-than-Wide** | **3** | **AP > Transverse** |
| **Margin** | | |
| Smooth | 0 | Well-Defined |
| Ill-Defined | 0 | Blurred |
| **Lobulated / Irregular** | **2** | Bumpy, Infiltrative |
| **Extrathyroidal Extension** | **3** | Muscle Invasion |
| **Echogenic Foci** | | |
| None / Large Comet-tail | 0 | Colloid Crystals / Ring-down Artefact |
| **Macrocalcifications** | **1** | Coarse Calcifications |
| **Peripheral (Rim) Calcifications** | **2** | Eggshell Calcification |
| **Punctate (Micro) Calcifications** | **3** | **Psammoma Bodies / Microcalcifications** |

## 5. TI-RADS Categories & FNA Thresholds

| TR Level | Points | Risk of Malignancy | FNA Threshold |
|----------|--------|-------------------|---------------|
| **TR1** | 0 | **Benign** (<1%) | No FNA |
| **TR2** | 2 | **Not Suspicious** (<2%) | No FNA |
| **TR3** | 3 | **Mildly Suspicious** (2-5%) | **≥2.5 cm** |
| **TR4** | 4-6 | **Moderately Suspicious** (5-20%) | **≥1.5 cm** |
| **TR5** | **≥7** | **Highly Suspicious** (>20%) | **≥1.0 cm** |

### Simplified Decision
| TR Level | Nodule Size | Action |
|----------|------------|--------|
| **TR1** | Any | No FNA, No Follow-up |
| **TR2** | Any | No FNA, No Follow-up |
| **TR3** | **≥2.5 cm** | FNA |
| **TR4** | **≥1.5 cm** | FNA |
| **TR5** | **≥1.0 cm** | FNA |

### Multiple Nodules
- **Biopsy 2 Highest-Risk Nodules** (Highest TR Level, Then Largest)
- Do Not Biopsy All Nodules (Unnecessary)

## 6. Bethesda System for Reporting Thyroid Cytopathology

| Category | Diagnosis | Risk of Malignancy | Management |
|----------|-----------|-------------------|------------|
| **I** | **Non-Diagnostic / Unsatisfactory** | — | **Repeat FNA** (6-12 weeks) |
| **II** | **Benign** | **0-3%** | **Clinical Follow-up** (US 1-2yr) |
| **III** | **AUS/FLUS** (Atypia of Undetermined Significance / Follicular Lesion of Undetermined Significance) | **10-30%** | **Repeat FNA** (3-6mo) **OR** **Molecular Testing** (Afirma, ThyroSeq) |
| **IV** | **FN/SFN** (Follicular Neoplasm / Suspicious for Follicular Neoplasm) | **15-30%** | **Diagnostic Lobectomy** (Histology Required) OR Molecular Testing |
| **V** | **Suspicious for Malignancy** | **60-75%** | **Near-Total Thyroidectomy** (Surgical Consult) |
| **VI** | **Malignant** | **97-99%** | **Total Thyroidectomy ± Central Neck Dissection** |

### Molecular Testing (Bethesda III/IV)
| Test | Indication | Action if Positive |
|------|------------|-------------------|
| **Afirma GSC** | AUS/FLUS (III) | If "Suspicious" → Surgery |
| **ThyroSeq v3** | AUS/FLUS (III) or FN/SFN (IV) | If "High Risk" → Surgery |
| **ThyGeNEXT/ThyraMIR** | AUS/FLUS, FN/SFN | Guides Surgery vs Observation |

---

## 7. Management Algorithm by Bethesda Category

```
Thyroid Nodule → TSH → TI-RADS US → FNA (if indicated) → Bethesda Category
         │
         ├── I (Non-Diagnostic) → Repeat FNA (6-12 weeks)
         │
         ├── II (Benign) → Surveillance US (12-24 months)
         │
         ├── III (AUS/FLUS)
         │       ├── Repeat FNA (3-6 months) → If Still III → Molecular Testing
         │       └── Molecular Testing → Positive → Surgery; Negative → Observe
         │
         ├── IV (FN/SFN)
         │       ├── Molecular Testing (Preferred) OR Diagnostic Lobectomy
         │       └── If Molecular Positive → Surgery
         │
         ├── V (Suspicious for Malignancy)
         │       → Near-Total Thyroidectomy ± Central Neck Dissection
         │
         └── VI (Malignant)
                 → Total Thyroidectomy ± Central Neck Dissection
                 → RAI Ablation (if Indicated)
```

## 8. Special Scenarios

### Thyrotoxic Nodule (TSH Suppressed)
| Finding | Management |
|---------|------------|
| **"Hot" Nodule (RAI Uptake Focal)** | **Toxic Adenoma** → Surgery (Hemithyroidectomy) / RAI |
| **Multifocal "Hot" Nodules** | Toxic MNG → RAI / Surgery |

### Non-Diagnostic FNA (Bethesda I)
| Situation | Action |
|-----------|--------|
| **1st Non-Diagnostic** | Repeat FNA (6-12 weeks, US-Guided) |
| **2nd Non-Diagnostic** | Consider Core Biopsy / Surgery (if High Suspicion) |

### Thyroglobulin (Tg) Monitoring
| Scenario | Utility |
|----------|---------|
| **Post-Thyroidectomy (DTC)** | **Surveillance Marker** (Stimulated Tg >1-2 ng/mL → Recurrence) |
| **Post-RAI** | Stimulated Tg <1 ng/mL = Excellent Response |
| **Anti-Tg Antibodies** | Interfere with Assay → Measure TgAb Simultaneously |

---

## 9. Surveillance & Follow-up

| Bethesda Category | Follow-up Protocol |
|-------------------|-------------------|
| **II (Benign)** | US **12-24 months** ×2 → Then Discharge if Stable |
| **III (AUS/FLUS)** | Repeat FNA 3-6mo OR Molecular → If Negative → US 12-24mo |
| **IV (FN/SFN)** | Post-Surgery: Pathology → If Benign → US 6-12mo |
| **V/VI (Malignant)** | Post-Thyroidectomy: Tg, US, TSH q6mo → Yearly |

## 11. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **TI-RADS vs Bethesda** | TI-RADS = US Risk Stratification (Pre-FNA); Bethesda = Cytology Reporting (Post-FNA) |
| **TR3 vs TR4 Threshold** | TR3: FNA if ≥2.5cm; TR4: FNA if ≥1.5cm |
| **Bethesda III vs IV** | III = AUS/FLUS (Repeat FNA/Molecular); IV = FN/SFN (Diagnostic Lobectomy) |
| **Molecular Testing** | For Bethesda III/IV; Not for V/VI (Already Surgical) |
| **Malignancy Risk** | Benign <3%; AUS/FLUS 10-30%; FN/SFN 15-30%; Suspicious 60-75%; Malignant 97-99% |
| **Non-Diagnostic FNA** | Repeat 6-12wk; If 2nd Non-Diagnostic → Core Biopsy/Surgery |

**Mnemonic: TI-RADS FNA THRESHOLDS**
- **T**R1/2: **None**
- **T**R3: **2.5cm**
- **T**R4: **1.5cm**
- **T**R5: **1.0cm**

**Mnemonic: BETHESDA CATEGORIES**
- **I** = **I**nadequate (Non-Diagnostic)
- **II** = **II**deal (Benign)
- **III** = **III**ntermediate (AUS/FLUS)
- **IV** = **IV** (FN/SFN - Follicular Neoplasm)
- **V** = **V**ery Suspicious
- **VI** = **VI**catory (Malignant)

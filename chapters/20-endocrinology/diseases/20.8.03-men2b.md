# MEN2B (Wagenmann-Froehlich Syndrome)

> [!info]
> **MEN2B = Wagenmann-Froehlich Syndrome.** **RET M918T Mutation (95%).** **Most Aggressive MEN2 Subtype.** **Triad: MTC + Phaeochromocytoma + Marfanoid Habitus + Mucosal Neuromas.** **NO Primary Hyperparathyroidism.** **Prophylactic Thyroidectomy <1 Year.**

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise the distinct clinical phenotype of MEN2B (Marfanoid habitus, mucosal neuromas)
- [ ] Apply RET M918T-based prophylactic thyroidectomy timing (<1 year)
- [ ] Differentiate MEN2B from MEN2A (No HPT, Marfanoid habitus, mucosal neuromas)
- [ ] Manage MTC, phaeochromocytoma in MEN2B context
- [ ] Apply genetic counselling and family screening

## 2. Genetics & Pathophysiology

| Feature | Details |
|---------|---------|
| **Gene** | **RET Proto-Oncogene** (Chromosome 10q11.2) |
| **Mutation** | **M918T (Exon 16) — 95%**; Rarely A883F (Exon 15) |
| **Mechanism** | **Gain-of-Function** → Constitutive RET Kinase Activation → MAPK/PI3K Pathways |
| **Inheritance** | **Autosomal Dominant** (Penetrance ~100% for MTC by Adulthood) |
| **Penetrance** | **MTC ~100%**; Phaeochromocytoma ~50%; **No HPT** |

## 3. Clinical Features — MEN2B Phenotype

| Feature | Details |
|---------|---------|
| **Medullary Thyroid Cancer (MTC)** | **100% Penetrance**; Earliest Manifestation; Aggressive |

| **Ocular** | Medullated Corneal Nerves (Slit-Lamp); Thickened Corneal Nerves |

## 4. RET Mutation — MEN2B Specific

| Feature | Details |
|---------|---------|
| **Mutation** | **M918T (Exon 16)** — **95%** of MEN2B |
| **Rare** | **A883F (Exon 15)** — ~5% |
| **Mechanism** | **Constitutive RET Kinase Activation** → MAPK/PI3K Pathways |
| **Phenotype Severity** | **Most Aggressive** MEN2 Subtype (Earlier Onset, More Aggressive MTC) |

### RET Codon Risk Stratification (ATA Guidelines)
| Risk Level | RET Codon | MTC Risk | Prophylactic Thyroidectomy |
|----------|------------|----------|----------------------------|
| **Highest (Level A)** | **M918T (Exon 16)** | **Highest** (Aggressive, Early Metastasis) | **<1 Year** (6-12 Months) |
| **High (Level B)** | **Codon 634** (Cysteine) | High | **5 Years** |
| **Moderate (Level C)** | 609, 611, 618, 620, 630, 768, 804, 891 | Moderate | 5-10 Years |

**MEN2B = Level A (M918T) → Prophylactic Thyroidectomy <1 Year (Ideally 6-12 Months).**

## 5. Clinical Features — MEN2B Distinctive Phenotype

### Mucosal Neuromas (Pathognomonic)
| Feature | Details |
|---------|---------|
| **Location** | **Lips, Tongue, Buccal Mucosa, Conjunctiva, Eyelids** |
| **Appearance** | **Multiple, Polypoid, Pedunculated, 1-5mm** |
| **Onset** | **Early Childhood** (Often <10 Years) |
| **Significance** | **Pathognomonic for MEN2B** (vs MEN2A) |

### Marfanoid Habitus
| Feature | Details |
|---------|---------|
| **Skeletal** | Tall, Slender, Dolichostenomelia, Arachnodactyly, Pectus Excavatum, High-Arched Palate |
| **Joints** | Joint Laxity, Pes Planus, Scoliosis |
| **Face** | High Forehead, Malar Flattening, Prognathism |

### Gastrointestinal
| Feature | Details |
|---------|---------|
| **Ganglioneuromatosis** | Diffuse Intestinal Ganglioneuromatosis → Constipation, Megacolon, Megaloesophagus |
| **Hirschsprung-Like** | Aganglionic Segments → Functional Obstruction |
| **Megalocornea** | Enlarged Cornea (>13mm) |

### Ocular
| Feature | Details |
|---------|---------|
| **Medullated Corneal Nerves** | **Slit-Lamp Finding** — Thickened, Visible Corneal Nerves |
| **Significance** | Highly Specific for MEN2B |

## 6. MTC in MEN2B — Most Aggressive

| Feature | Details |
|---------|---------|
| **Penetrance** | **100%** (Earliest Onset — Often Childhood) |
| **Aggressiveness** | **Highest** of All MEN2 Subtypes; Early Metastasis (Nodes, Liver, Lung, Bone) |
| **Calcitonin** | **Very High** at Diagnosis; Exponential Rise |
| **Prophylactic Thyroidectomy** | **<1 Year (6-12 Months)** — Earliest of All RET Mutations |

---

## 7. Phaeochromocytoma in MEN2B

| Feature | Details |
|---------|---------|
| **Frequency** | **~50%** (Similar to MEN2A) |
| **Bilaterality** | **Common (50-70%)** |
| **Biochemistry** | Plasma Free Metanephrines / Normetanephrines |
| **Imaging** | **⁶⁸Ga-DOTATATE PET/CT** (Best) |
| **Pre-Op** | **Alpha-Blockade First** (Phenoxybenzamine) → Beta-Blockade After |
| **Surgery** | Laparoscopic Adrenalectomy; **Cortical-Sparing if Bilateral** |

---

## 8. Prophylactic Thyroidectomy — MEN2B Specific

| Aspect | Details |
|--------|---------|
| **RET Mutation** | **M918T (Exon 16) — 95%** |
| **Risk Level** | **Level A (Highest)** — ATA Guidelines |
| **Thyroidectomy Timing** | **<1 Year** (Ideally **6-12 Months**) |
| **Procedure** | **Total Thyroidectomy + Central Neck Dissection (Level VI)** |
| **Post-Op** | **Calcitonin Undetectable** = Cure; Monitor Basal + Stimulated Calcitonin |
| **Levothyroxine** | Lifelong Replacement (Start Post-Op) |

---

## 9. MEN2B vs MEN2A — Key Differences

| Feature | **MEN2B** | **MEN2A (Sipple)** |
|---------|---------|------------------|
| **MTC** | **100% (Earliest, Most Aggressive)** | 95% (Later Onset) |
| **Phaeochromocytoma** | **50% (Bilateral Common)** | 50% (Bilateral Common) |
| **Primary HPT** | **ABSENT** (Key Difference) | **Present (20-30%)** |
| **Marfanoid Habitus** | **Present** (Tall, Arachnodactyly) | **Absent** |
| **Mucosal Neuromas** | **Present (Pathognomonic)** | **Absent** |
| **Gastrointestinal** | **Ganglioneuromatosis (Megacolon)** | **Absent** |
| **RET Mutation** | **M918T (95%)** | **Codon 634 (85%)** |
| **Prophylactic Thyroidectomy** | **<1 Year (6-12 Months)** | **Age 5 (Codon 634)** |
| **Genetics** | **RET M918T (95%)** | **Codon 634 (85%)** |

## 10. Management — MEN2B Specific

### Medullary Thyroid Cancer (MTC)
| Aspect | Details |
|--------|---------|
| **Prophylactic Thyroidectomy** | **<1 Year (6-12 Months)** — **Earliest of All RET Mutations** |
| **Surgery** | **Total Thyroidectomy + Central Neck Dissection (Level VI)** |
| **Post-Op** | **Levothyroxine Replacement**; **Calcitonin Monitoring** (Basal + Stimulated) |

### Phaeochromocytoma
| Feature | Details |
|---------|---------|
| **Biochemistry** | Free Metanephrines / Normetanephrines |
| **Imaging** | **⁶⁸Ga-DOTATATE PET/CT** (Gold Standard) |
| **Pre-Op** | **Alpha-Blockade First** (Phenoxybenzamine) → Beta-Blocker After |
| **Surgery** | Laparoscopic Adrenalectomy; **Cortical-Sparing if Bilateral** |

### Mucosal Neuromas / GI
| Feature | Management |
|---------|---------|
| **Mucosal Neuromas** | Symptomatic (Bleeding, Irritation) → Laser Excision |
| **Ganglioneuromatosis** | Constipation/Megacolon → Laxatives, Prokinetics; Surgery if Obstruction |

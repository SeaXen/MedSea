# Hypoparathyroidism

> [!info]
> **Hypoparathyroidism = Deficient PTH Secretion** → Hypocalcaemia + Hyperphosphataemia. **Post-Surgical (Post-Thyroidectomy) = Commonest Cause (75%).** Autoimmune, Genetic, Infiltrative also. **No PTH → ↓ Bone Resorption, ↓ Renal Ca²⁺ Reabsorption, ↓ 1,25-(OH)₂D Synthesis → Hypocalcaemia + Hyperphosphataemia.**

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Classify hypoparathyroidism by aetiology (post-surgical, autoimmune, genetic)
- [ ] Recognise clinical features of hypocalcaemia (tetany, seizures, QT prolongation)
- [ ] Execute acute management of hypocalcaemic tetany (IV calcium gluconate)
- [ ] Prescribe chronic replacement (calcium, alfacalcidol/calcitriol)
- [ ] Recognise pseudohypoparathyroidism (PTH resistance)

## 2. Aetiology & Classification

| Category | Cause | Frequency |
|----------|-------|-----------|
| **Post-Surgical** | **Post-Thyroidectomy** (Total/Subtotal); **Parathyroidectomy** | **75%** (Commonest) |
| **Autoimmune** | Isolated / **APS-1** (Autoimmune Polyendocrine Syndrome Type 1) | 10-15% |
| **Genetic** | **DiGeorge Syndrome (22q11.2 Deletion)**; **CASR** (Activating); **GCM2**; **PTH Gene** | 5-10% |
| **Infiltrative/Infiltrative** | Haemochromatosis, Wilson's, Metastases, Sarcoidosis | Rare |
| **Radiation/Radiotherapy** | Neck Irradiation; Radioactive Iodine | Rare |
| **Magnesium Deficiency** | **Functional Hypoparathyroidism** (Mg Required for PTH Secretion) | Reversible |

---

## 3. Pathophysiology

| Deficiency | Consequence |
|------------|-------------|
| **↓ PTH** | ↓ Bone Resorption (↓ Ca²⁺ Release); ↓ Renal Ca²⁺ Reabsorption (Distal Tubule); ↓ Renal 1α-Hydroxylase → **↓ 1,25-(OH)₂D** → ↓ Intestinal Ca²⁺ Absorption |
| **Net Effect** | **Hypocalcaemia** + **Hyperphosphataemia** (↓ Renal Phosphate Excretion) |
| **Mg²⁺ Deficiency** | Impairs PTH Secretion **AND** PTH Action (Functional Hypoparathyroidism) |

---

## 4. Aetiology & Clinical Features

### Post-Surgical (Commonest, 75%)
| Type | Onset | Features |
|------|---------|---------|
| **Transient** | <6 Months Post-Thyroidectomy | Usually Recovers; Parathyroid Stunned |
| **Permanent** | **>6-12 Months** | Permanent PTH Deficiency; Requires Lifelong Replacement |
| **Risk Factors** | Total Thyroidectomy, Central Neck Dissection, Re-operation, Thyroid Cancer |

### Autoimmune (Isolated / APS-1)
| Syndrome | Components |
|--------|-----------|
| **Isolated** | Autoimmune Hypoparathyroidism Alone; **CAII, NACHT-LRR** Antibodies |
| **APS-1 (APECED)** | **AIRE Gene** (AR); **Triad: Hypoparathyroidism + Adrenal Insufficiency + Chronic Mucocutaneous Candidiasis**; Also: Enamel Hypoplasia, Hepatitis, Keratitis |

### Genetic
| Syndrome | Gene | Features |
|----------|------|----------|
| **DiGeorge Syndrome (22q11.2 Del)** | **TBX1** | Hypoparathyroidism, Thymic Aplasia (T-cell Deficiency), Conotruncal Heart Defects, Cleft Palate, Facial Dysmorphism |
| **CASR Activating Mutation** | **CASR** (Gain-of-Function) | Autosomal Dominant; Hypocalcaemia, Hypercalciuria, Bartter-Like Syndrome |
| **GCM2** | **GCM2** | Familial Isolated Hypoparathyroidism (AR/AD) |
| **PTH Gene** | **PTH** | Familial Isolated Hypoparathyroidism (AR) |

### Magnesium Deficiency
| Feature | Details |
|---------|---------|
| **Mechanism** | Mg²⁺ Required for PTH Secretion (Adenylate Cyclase) + PTH Action (G-Protein Coupling) |
| **Presentation** | Hypocalcaemia (Resistant to Ca²⁺), Hypokalaemia, Hypomagnesaemia, Seizures, Tetany |
| **Treatment** | **IV Magnesium Sulphate** (Correct Mg²⁺ First → PTH Recovers) |

---

## 5. Clinical Features

### Acute Hypocalcaemia (Tetany)
| Feature | Details |
|---------|---------|
| **Perioral Paraesthesia** | Numbness/Tingling Lips, Fingers, Toes |
| **Carpopedal Spasm** | **Trousseau's Sign** (Carpal Spasm on BP Cuff Inflation >20mmHg Above SBP); **Chvostek's Sign** (Facial Twitch on Facial Nerve Tap) |
| **Laryngospasm** | Stridor, Airway Obstruction (Emergency) |
| **Generalised Seizures** | Generalised Tonic-Clonic |
| **Cardiac** | **Prolonged QT** → Torsades de Pointes, VF |
| **Neuropsychiatric** | Anxiety, Irritability, Confusion, Seizures, Depression |
| **Ectodermal** | Dry Skin, Brittle Nails, Hair Loss, Cataracts (Chronic) |

### Chronic Hypocalcaemia
| Feature | Details |
|---------|---------|
| **Basal Ganglia Calcification** | CT Brain: Bilateral Basal Ganglia Calcification |
| **Cataracts** | Posterior Subcapsular |
| **Dental** | Enamel Hypoplasia, Caries |
| **Neuropsychiatric** | Depression, Anxiety, Dementia, Parkinsonism |
| **Skin/Hair/Nails** | Dry Skin, Coarse Hair, Brittle Nails |

## 6. Diagnosis & Investigations

### Biochemical Confirmation
| Test | Expected in Hypoparathyroidism |
|------|--------------------------------|
| **Serum Calcium (Corrected)** | **Low (<2.15 mmol/L / 8.6 mg/dL)** |
| **Ionised Calcium** | **Low (<1.1 mmol/L)** |
| **Phosphate** | **High (>1.5 mmol/L)** |
| **PTH (Intact)** | **Low / Undetectable** (<1.6 pmol/L) |
| **Magnesium** | Normal (Unless Mg²⁺ Deficiency) |
| **25-OH Vitamin D** | Usually Normal/Low (Secondary to Low PTH) |
| **Creatinine** | Normal (Exclude Renal Failure) |
| **24h Urine Calcium** | Low (Tubular Reabsorption ↑) |

### Differential Diagnosis
| Condition | Calcium | Phosphate | PTH | Key Differentiator |
|-----------|---------|-----------|-----|-------------------|
| **Hypoparathyroidism** | Low | High | **Low/Undetectable** | Low PTH |
| **Pseudohypoparathyroidism** | Low | High | **High** | PTH Resistance |
| **Vitamin D Deficiency** | Low | Low/Normal | **High** (Secondary) | Low 25-OH D |
| **Renal Failure** | Low | High | High (Secondary) | Elevated Creatinine |
| **Pseudohypoparathyroidism Type 1a** | Low | High | High | Albright Hereditary Osteodystrophy (Short Stature, Obesity, Short Metacarpals) |

## 7. Acute Management — Hypocalcaemic Tetany (Emergency)

### Immediate IV Calcium
| Drug | Dose | Administration |
|------|--------|--------------|
| **10% Calcium Gluconate** | **10-20 mL (1-2g) IV** | **Slow IV Over 10 Min** (Cardiac Monitor) |
| **Repeat** | If Persistent Tetany | Every 10-20 Min Until Controlled |

### Follow-Up Infusion
| Infusion | Rate |
|---------|-------|
| **10% Calcium Gluconate** | **1-2 g/hr (10-20 mL/hr)** in 5% Dextrose / 0.9% Saline |
| **Target** | Ionised Ca²⁺ >1.1 mmol/L; Symptom Relief |

### Adjuncts
| Drug | Dose | Indication |
|------|------|------------|
| **Magnesium Sulphate** | 2-4g IV (If Hypomagnesaemia) | Mg <0.7 mmol/L |
| **Vitamin D (Alfacalcidol/Calcitriol)** | 1-2 µg IV/PO | Accelerate Ca²⁺ Absorption |
| **Oral Calcium** | Calcium Carbonate 1-2g TDS | After IV Stabilisation |

---

## 8. Chronic Management

### Calcium Supplementation
| Preparation | Elemental Ca²⁺ | Dose |
|--------------|----------------|------|
| **Calcium Carbonate** | 40% (400mg/tablet) | 1-2g TDS with Meals |
| **Calcium Citrate** | 21% | 1-2g TDS (Better Absorption if Achlorhydria) |
| **Target** | **Total Elemental Ca²⁺ 1.5-2g/day** | Divided Doses |

### Vitamin D Analogues (Essential for Absorption)
| Agent | Dose | Advantage |
|---------|------|----------|
| **Alfacalcidol (1α-OH D3)** | **0.25-1 µg/day** | Rapid Onset; No Renal Activation Needed |
| **Calcitriol (1,25-(OH)₂D₃)** | **0.25-0.5 µg/day** | Active Form; Short Half-Life |
| **Dose Titration** | **Target: Ionised Ca²⁺ 1.1-1.3 mmol/L**; Adjust q2-4wk |

### Magnesium
| Indication | Dose |
|------------|------|
| **Hypomagnesaemia** | **MgSO₄ 2-4g IV** (Then Oral Mg 300-600mg/day) |

### Thiazide Diuretic (Adjunct)
| Role | Dose |
|------|------|
| **Reduce Urinary Ca²⁺ Excretion** | **Hydrochlorothiazide 25mg OD** (If Hypercalciuria) |

## 9. Monitoring & Targets

| Parameter | Target | Frequency |
|-----------|--------|-----------|
| **Serum Calcium (Corrected)** | **2.2-2.5 mmol/L** (Slightly Low Normal) | Monthly (Stable); Weekly (Titration) |
| **Ionised Calcium** | **1.1-1.3 mmol/L** | q1-2wk (Titration) |
| **Phosphate** | **<1.5 mmol/L** (Avoid Hyperphosphataemia) | Monthly |
| **24h Urine Calcium** | **<7.5 mmol/day (300mg)** | 3-6 Monthly (Monitor Nephrolithiasis) |
| **Renal Function** | eGFR, Creatinine | 6-Monthly |
| **25-OH Vitamin D** | >50 nmol/L (20 ng/mL) | Annually |
| **Bone Density** | DEXA (Spine/Hip) | q2-3 Years |

## 10. Special Situations

### Pregnancy
| Aspect | Management |
|--------|-----------|
| **Calcium Demand** | ↑ Fetal Demand → ↑ Maternal Requirement |
| **Vitamin D** | **Alfacalcidol/Calcitriol Safe**; Increase Dose 50-100% |
| **Target** | **Ionised Ca²⁺ 1.1-1.3 mmol/L** (Maternal); Fetal Ca²⁺ Dependent on Maternal |
| **Breastfeeding** | Continue Replacement; Monitor Infant Ca²⁺ |

### Post-Thyroidectomy (Transient vs Permanent)
| Type | Duration | Management |
|---------|----------|------------|
| **Transient** | <6 Months | Calcium + Vit D; Wean as PTH Recovers |
| **Permanent** | **>6-12 Months** | Lifelong Replacement; PTH Undetectable |

## 11. Pseudohypoparathyroidism (PHP) — PTH Resistance

| Type | Gene | Features | Biochemistry |
|--------|------|----------|-------------|
| **PHP Type 1a** | **GNAS** (Maternal Imprinting) | **Albright Hereditary Osteodystrophy (AHO)**: Short Stature, Obesity, Round Face, Short Metacarpals (4th/5th), Subcutaneous Ossifications | **High PTH**, Hypocalcaemia, Hyperphosphataemia |
| **PHP Type 1b** | **GNAS** (Methylation Defect) | Renal Resistance Only (No AHO) | High PTH, Hypocalcaemia |
| **Pseudopseudohypoparathyroidism (PPHP)** | **GNAS** (Paternal) | **AHO Features Only** | Normal Ca/Phos/PTH |

## 13. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Hypoparathyroidism vs Pseudohypoparathyroidism** | Hypopara: **Low PTH**; PHP: **High PTH** (Resistance) |
| **Phosphate** | Hypopara: **High**; Vit D Def: **Low** |
| **PHP Type 1a vs 1b** | 1a = AHO Phenotype (Short Stature, Round Face, Short Metacarpals); 1b = Renal Resistance Only |
| **PPHP** | **Pseudopseudohypoparathyroidism** = AHO Features + Normal Biochemistry |
| **Mg Deficiency** | Causes **Functional Hypoparathyroidism**; Correct Mg First Before Ca |
| **Vitamin D in Hypopara** | **Alfacalcidol/Calcitriol** (Active Analogues) — Not Cholecalciferol |

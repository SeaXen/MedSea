# MEN1 (Multiple Endocrine Neoplasia Type 1)

> [!info]
> **MEN1 = Wermer Syndrome = Autosomal Dominant** (Penetrance >95% by Age 50). **"3 Ps": Parathyroid Hyperplasia, Pituitary Adenoma, Pancreatic NETs.** **MEN1 Gene (Menin, Tumour Suppressor)** on Chromosome 11q13. **Screening: Annual Biochemical + 5-Yearly Imaging.**

---

## 2. Diagnostic Criteria

| Clinical Criterion | Definition |
|---------------------|------------|
| **Classic** | **≥2 of 3 Ps** (Parathyroid, Pituitary, Pancreas) |
| **Familial** | **1 of 3 Ps** + **1st-Degree Relative with MEN1** |

**Genetic Confirmation**: **MEN1 Mutation** (Menin, Tumour Suppressor, Chromosome 11q13)

---

## 3. The "3 Ps" — Core Manifestations

### 1. Primary Hyperparathyroidism (PHPT) — Most Common (90-95%)
| Feature | Details |
|--------|---------|
| **Frequency** | **90-95%** (Earliest, Often First Manifestation) |
| **Pathology** | **Multigland Hyperplasia** (90%) / Single Adenoma (10%) |
| **Biochemistry** | Hypercalcaemia, **Inappropriately Normal/High PTH**, Low Phosphate |
| **Age of Onset** | 20-40 Years (Earlier Than Sporadic PHPT) |
| **Management** | **Subtotal (3.5-Gland) Parathyroidectomy** (Standard) |

### 2. Pituitary Adenoma
| Feature | Details |
|----------|---------|
| **Frequency** | **30-40%** |
| **Types** | **Prolactinoma (60%)** > GH (Somatotroph) > Non-Functioning > ACTH (Cushing) > TSH-oma |
| **Prolactinoma** | Often Macroadenoma; Cabergoline 1st Line |
| **GH-Secreting (Acromegaly)** | 10-20% of MEN1 Pituitary; Earlier Than Sporadic |
| **Cushing Disease** | Rare in MEN1 |
| **Surgery** | TSS (Transsphenoidal) for Macroadenomas / Functioning Microadenomas |

### 3. Pancreatic NETs (pNETs)
| Feature | Details |
|---------|---------|
| **Frequency** | **30-50%** (Often Multiple) |
| **Functioning** | **Gastrinoma (ZES)** (40%) → **Duodenal** (Multiple, Small); **Insulinoma** (10-20%); **Non-Functioning** (30-40%) |
| **Gastrinoma (ZES)** | **Duodenal > Pancreatic**; Multiple Small Tumours; **Secretin Stimulation Test** (+ve) |
| **Insulinoma** | **Benign** (90%); **Whipple's Triad**; Surgical Enucleation |
| **Non-Functioning** | **Malignant Potential** High if >2cm; Surgical Resection if >2cm |
| **Malignant Potential** | **pNETs >2cm** → Higher Malignant Risk; Lymph Node Metastases Common |

---

## 4. Associated Tumours (Beyond the 3 Ps)

| Tumour | Frequency | Clinical Significance |
|--------|-----------|----------------------|
| **Adrenal** | **20-40%** | Usually **Non-Functioning** Incidentalomas; Rare Cortisol/Androgen Excess |
| **Thymic Carcinoid** | 5-10% | **MEN1-Associated**; Often Malignant; **Male Predominance** |
| **Bronchial Carcinoid** | 5-10% | **Female Predominance**; Less Aggressive than Thymic |
| **Lipomas** | 30% | Subcutaneous, Visceral |
| **Angiofibromas / Collagenomas** | Skin | Facial, Trunk |
| **Meningioma** | Rare | Suprasellar / Convexity |

---

## 5. Screening & Surveillance Protocol

| Component | Frequency | Modality |
|-----------|-----------|----------|
| **Calcium + PTH** | **Annual** (From Age 5-10) | Blood |
| **Pituitary Hormones** (PRL, IGF-1, ACTH, TSH, fT4, LH/FSH) | **Annual** | Blood |
| **Fasting Gut Hormones** (Gastrin, Insulin, Glucagon, PP, Chromogranin A) | **Annual** (From Age 20) | Blood |
| **MRI Pituitary** | **Every 3 Years** (Or Sooner if Functioning) | MRI (3mm, Gadolinium) |
| **CT/MRI Pancreas** | **Every 1-3 Years** (From Age 20) | Multiphasic CT / MRI |
| **Chest CT** | **Every 3-5 Years** (Thymic/Bronchial Carcinoid) | CT Chest |
| **Adrenal Imaging** | **Every 2-3 Years** | CT Abdomen |
| **Skin Exam** | Annual | Clinical |

## 6. Genetic Testing & Counselling

| Aspect | Details |
|----------|---------|
| **Gene** | **MEN1 (Menin)** — Chromosome 11q13, Tumour Suppressor |
| **Inheritance** | **Autosomal Dominant**; **Penetrance >95% by Age 50** |
| **Testing Indication** | Clinical Diagnosis + 1st-Degree Relatives |
| **Family Screening** | **Predictive Testing** (If Mutation Known); Offer to 1st-Degree Relatives |
| **Genetic Counselling** | Mandatory Pre/Post-Test; Reproductive Options (PGD) |

## 7. Management by Organ System

### Parathyroid (PHPT in MEN1)
| Aspect | Details |
|---------|---------|
| **Pathology** | **4-Gland Hyperplasia (90%)** (vs 80-85% Single Adenoma in Sporadic) |
| **Surgery** | **Subtotal (3.5-Gland) Parathyroidectomy** + Autotransplant (Forearm) |
| **Recurrence** | **15-30%** (Due to Hyperplasia / Remnant Tissue) |
| **IOPTH** | >50% Drop at 10min Post-Excision (Miami Criterion) |

### Pituitary Adenoma
| Type | Management |
|-------|------------|
| **Prolactinoma** | **Cabergoline 1st Line** (Dose Titration); Surgery if Macro + Visual Defects |
| **Acromegaly (GH)** | TSS (1st Line) → SST Analogues / Pegvisomant |
| **Non-Functioning** | Surgery if Macro + Mass Effect / Apoplexy Risk |

### Pancreatic NETs
| Tumour Type | Management |
|-----------|------------|
| **Gastrinoma (ZES)** | **High-Dose PPI** + **Surgical Resection** (Duodenotomy + Enucleation) |
| **Insulinoma** | **Enucleation** (Curative; 90% Benign) |
| **Non-Functioning >2cm** | **Surgical Resection** (Malignant Potential) |
| **<2cm** | Surveillance (EUS/MRI q6-12mo) |

---

## 8. Surveillance — Lifetime Protocol

| Age to Start | Surveillance | Frequency |
|--------------|--------------|-----------|
| **5-10 Years** | Serum Calcium, PTH | Annual |
| **10-15 Years** | Pituitary Hormones, MRI Pituitary | Annual / 3-Yearly |
| **20 Years** | Fasting Gut Hormones (Gastrin, Insulin, Glucagon, PP, CgA) | Annual |
| **20 Years** | CT/MRI Pancreas, Chest CT | 1-3 Years |
| **All Ages** | Adrenal Imaging | Every 2-3 Years |
| **Post-Transplant / Post-PTX** | Lifelong Surveillance | |

## 9. Genetic Counselling

| Aspect | Recommendation |
|--------|----------------|
| **Testing** | **MEN1 Sequencing** (If Clinical Diagnosis / Family History) |
| **Predictive Testing** | **Offer to 1st-Degree Relatives** (If Mutation Known) |
| **Prenatal** | CVS/Amniocentesis (If Parental Mutation Known) |
| **Reproductive** | **PGD (Preimplantation Genetic Diagnosis)** Available |

## 11. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **MEN1 vs MEN2A** | MEN1 = 3Ps (Parathyroid, Pituitary, Pancreas); MEN2A = MTC + Phaeo + HPT |
| **MEN1 PHPT vs Sporadic PHPT** | MEN1 = Multigland Hyperplasia (90%); Sporadic = Single Adenoma (80%) |
| **Gastrinoma Location** | **Duodenal > Pancreatic** in MEN1 (Opposite Sporadic) |
| **Non-Functioning pNET** | >2cm → Surgery (Malignant Potential); <2cm → Surveillance |
| **Adrenal in MEN1** | Usually Non-Functioning Incidentaloma; Rarely Functional |
| **MEN1 vs MEN2** | MEN1 = Parathyroid + Pituitary + Pancreas; MEN2 = MTC + Phaeo ± HPT |

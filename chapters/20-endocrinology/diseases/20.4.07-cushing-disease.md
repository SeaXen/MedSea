# Cushing Disease

> [!info]
> **Cushing Disease = ACTH-Secreting Pituitary Microadenoma causing Cushing Syndrome.** **Commonest cause of endogenous Cushing Syndrome (70% of ACTH-dependent cases).** **Microadenoma in 80-90%.** Gold standard treatment: **Transsphenoidal Surgery (TSS).**

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Diagnose Cushing disease using ACTH, HDDST, CRH test, IPSS
- [ ] Interpret MRI pituitary findings
- [ ] Outline surgical approach (TSS) and remission criteria
- [ ] Manage residual/recurrent disease and medical therapy options
- [ ] Differentiate from ectopic ACTH and adrenal Cushing

## 2. Pathophysiology & Genetics

| Aspect | Details |
|--------|---------|
| **Cell Origin** | **Corticotroph Adenoma** (Pituitary Anterior Lobe) |
| **Size** | **Microadenoma <10mm** (80-90%); Macroadenoma 10-20% |
| **Pathogenesis** | **USP8 Mutations** (40%); **USP48** (Rare); **ATP1A1, BRAF** (Rare) → ↑ EGFR Signalling → ↑ POMC Transcription |
| **Genetics** | **Sporadic** (Most); **MEN1** (Rare); **Carney Complex** (PRKAR1A); **FIP** (Familial Isolated Pituitary Adenomas) |
| **Epidemiology** | **Women 3-5:1**; Peak 20-50 Years; **70% of ACTH-Dependent Cushing** |

## 3. Clinical Presentation

| Feature | Frequency | Details |
|---------|-----------|---------|
| **Weight Gain / Central Obesity** | >90% | Moon Face, Buffalo Hump, Centripetal |
| **Skin** | 70-80% | Striae (>1cm, Violaceous), Thin Skin, Bruising, Acne |
| **Musculoskeletal** | 60% | Proximal Myopathy, Osteoporosis, Vertebral Fractures |
| **Cardiovascular** | 80% | Hypertension, Metabolic Syndrome, Atherosclerosis |
| **Glucose** | 20-50% | Diabetes / IGT |
| **Psychiatric** | 50-70% | Depression, Anxiety, Cognitive Impairment |
| **Reproductive** | 50-70% | Oligomenorrhoea, Amenorrhoea, Hirsutism, Low Libido |
| **Ophthalmological** | Rare | Visual Field Defects (If Macroadenoma >10mm) |

## 4. Diagnosis — Confirmation & Differentiation

### Diagnostic Algorithm
```
Cushing Syndrome Confirmed (2/3 Tests Positive)
         │
         ▼
PLASMA ACTH (09:00, EDTA, Ice)
         │
         ├── ACTH <10 pg/mL → ACTH-INDEPENDENT → Adrenal CT
         │
         └── ACTH ≥10-20 pg/mL → ACTH-DEPENDENT
                 │
                 ├── HDDST (2mg q6h × 48h)
                 │       ├── >50% Suppression → **Cushing Disease**
                 │       │       MRI Pituitary (3mm, Gadolinium)
                 │       │       IPSS if Equivocal
                 │       │
                 │       └── No Suppression → Ectopic ACTH / Adrenal Cushing
                 │
                 └── IF HDDST EQUIVOCAL
                         ├── CRH Test (ACTH ↑ >50% = Cushing Disease)
                         ├── IPSS (Central:Peripheral ACTH Gradient >2 Basal / >3 Post-CRH)
                         └── ⁶⁸Ga-DOTATATE PET (Ectopic Localisation)
```

### Key Differentiators

| Feature | **Cushing Disease** | **Ectopic ACTH** | **Adrenal Cushing** |
|---------|---------------------|------------------|---------------------|
| **ACTH** | Normal / High | **High (>200 pg/mL)** | **Suppressed (<10 pg/mL)** |
| **HDDST (>50% Suppression)** | **Yes** | No | No |
| **CRH Test** | ACTH ↑ >50% | No Rise | N/A |
| **IPSS** | Central:Peripheral >2 (Basal) >3 (Post-CRH) | No Gradient | N/A |
| **MRI Pituitary** | Microadenoma (80-90%) | Normal / Ectopic Mass | Normal |
| **Lung/Chest CT** | Normal | **Mass (SCLC, Carcinoid)** | Normal |

## 5. MRI Pituitary — Imaging Protocol

| Parameter | Specification |
|-----------|---------------|
| **Sequence** | T1 (Pre/Post-Gadolinium), T2, Coronal/Sagittal |
| **Slice Thickness** | **3 mm** (No Gap) |
| **Dynamic Contrast** | T1 Post-Gadolinium at 30s, 60s, 90s (Microadenoma Enhancement Delay) |
| **Sensitivity** | 60-70% for Microadenoma; >95% for Macroadenoma |

### MRI Findings
| Finding | Significance |
|---------|--------------|
| **Hypointense on T1 Post-Gad** | Microadenoma (Delayed Enhancement vs Normal Pituitary) |
| **Marked Enhancement** | Normal Pituitary / Macroadenoma |
| **Deviation of Stalk** | Contralateral to Microadenoma |

## 6. Inferior Petrosal Sinus Sampling (IPSS) — Gold Standard

### Indications
| Indication | Details |
|-----------|---------|
| **MRI Negative / Equivocal** | Microadenoma Not Visualised / Incidentaloma |
| **HDDST Equivocal** | Discordant Results |
| **Discordant Clinical/Biochemical** | Suspicion of Ectopic Despite Positive HDDST |

### Procedure
| Step | Details |
|------|---------|
| **Catheters** | Bilateral Inferior Petrosal Sinuses (IPS) + Peripheral Vein |
| **Sampling** | Basal ACTH ×2 → CRH 100µg IV → ACTH at 1, 2, 3, 5, 10, 15min |
| **Gradients** | **Basal Central:Peripheral >2** OR **Post-CRH >3** = Pituitary Source |

### Interpretation
| Result | Interpretation |
|--------|----------------|
| **Central:Peripheral >2 (Basal) or >3 (Post-CRH)** | **Cushing Disease** (Pituitary ACTH) |
| **No Gradient** | **Ectopic ACTH** |

## 7. Management — Cushing Disease

### First-Line: Transsphenoidal Surgery (TSS)

| Aspect | Details |
|--------|---------|
| **Approach** | **Endoscopic Endonasal** (Preferred) / Microscopic Sub-labial |
| **Cure Rate** | **Microadenoma: 80-90%**; **Macroadenoma: 50-60%** |
| **Remission Criteria (Consensus)** | |
| **Post-Op Day 1** | **Cortisol <50 nmol/L (2 µg/dL)** |
| **6 Weeks** | **UFC <ULN** + **Late-Night Salivary Cortisol <Upper Limit** |
| **6 Months** | Normal UFC + LNSC + Normal ACTH/Cortisol Axis |
| **Recurrence Rate** | **10-20% at 10 Years** |

### Post-Op Management
| Timepoint | Action |
|-----------|---------|
| **Day 1** | **Cortisol <50 nmol/L** = Biochemical Remission |
| **Week 1-2** | **Hydrocortisone Replacement** (15-20mg AM + 5-10mg PM) — **Essential** (Adrenal Suppression) |
| **6 Weeks** | **Wean Hydrocortisone** if UFC Normal; If Not → Assess Recurrence |
| **6 Months** | Confirm Sustained Remission (UFC, LNSC, ACTH, Cortisol) |
| **Annual** | UFC, LNSC, MRI (If Residual/Recurrent) |

---

## 8. Residual / Recurrent Disease — Management

| Scenario | Management |
|----------|------------|
| **Residual Post-TSS** | **Repeat TSS** (If Accessible) OR **Medical Therapy** |
| **Recurrent (After Remission)** | **Repeat TSS** (Preferred) / Medical / Radiotherapy |
| **Medical Therapy** | **Ketoconazole** 200-400mg q6h; **Metyrapone**; **Osilodrostat**; **Pasireotide** (SST Analog) |
| **Bilateral Adrenalectomy** | Last Resort (If Medical Fails / Life-Threatening) |
| **Radiotherapy** | **Stereotactic / Fractionated** (If Surgery/Medical Fail) |

### Medical Therapy Options
| Drug | Dose | Mechanism | Monitoring |
|------|------|-----------|------------|
| **Ketoconazole** | 200-400mg q6h | Inhibits CYP11B1 (11β-hydroxylase) | LFTs q2wk; Adrenal Insufficiency Risk |
| **Metyrapone** | 250-750mg q8h | Inhibits 11β-hydroxylase | Hypotension, Oedema, Hirsutism |
| **Osilodrostat** | 2-30mg BD | Inhibits 11β-hydroxylase | ECG (QT), Hypokalaemia |
| **Pasireotide** | 600-900µg q12h SC | SSTR5 Agonist (↓ ACTH) | **Hyperglycaemia** (Monitor Glucose) |

## 9. Pituitary Radiotherapy
| Type | Indication | Dose |
|------|------------|------|
| **Stereotactic (SRS/GK)** | Small Residual (<2cm) | 15-25 Gy (Single Fraction) |
| **Fractionated RT** | Larger Residual / Not SRS Candidate | **45-50 Gy / 25-28 Fractions** |
| **Onset of Effect** | **5-10 Years** (Delayed) | |
| **Complication** | **Hypopituitarism (30-50% at 10-15yr)** | |

## 12. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Cushing Disease vs Syndrome** | Disease = Pituitary ACTH Adenoma; Syndrome = All Causes of Hypercortisolism |
| **HDDST >50% Suppression** | Cushing Disease; Ectopic/Adrenal = No Suppression |
| **IPSS vs CRH Test** | IPSS = Gold Standard Localisation; CRH Test = Physiological (ACTH Rise >50% = Pituitary) |
| **Micro vs Macroadenoma** | Micro <10mm (80-90%); Macro ≥10mm (Cavernous Sinus Invasion Risk) |
| **Post-Op Hydrocortisone** | **Mandatory** — Adrenal Suppression from Chronic ACTH Excess |
| **Recurrence** | 10-20% at 10yr; Monitor UFC/LNSC Annually |
| **IPSS Gradient** | >2 Basal / >3 Post-CRH = Pituitary Source |
| **Pasireotide** | SST5 Agonist; ↓ ACTH; **Hyperglycaemia** (Monitor Glucose) |

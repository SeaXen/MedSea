# Carcinoid Syndrome

> [!info]
> **Carcinoid Syndrome = Clinical Syndrome from Bioactive Secretion by Metastatic Neuroendocrine Tumour (NET).** **Serotonin (5-HT) = Main Mediator**. **Classic Triad: Flushing, Diarrhoea, Valvular Heart Disease.** **Midgut NETs (Ileum, Appendix, Right Colon) + Liver Mets → Systemic Release**. **Diagnosis: 24h Urine 5-HIAA**.

---

## 2. Pathophysiology

| Mechanism | Mediator | Effect |
|---------|----------|--------|
| **Serotonin (5-HT)** | Tumour Cell Secretion → Systemic Circulation | **Flushing**, **Diarrhoea**, **Valvular Fibrosis** (Right-Sided), Bronchospasm, Fibrosis (Retroperitoneal, Mesenteric) |
| **Kallikrein → Bradykinin** | ↑ Vascular Permeability | **Flushing** (Intense, Prolonged), Angioedema, Hypotension |
| **Prostaglandins** (PGE2, PGF2α) | Tumour Secretion | Diarrhoea, Flushing, Bronchospasm |
| **Other** | Substance P, Neuropeptide K, Motilin, ACTH (Ectopic Cushing) | Variable |

### Why Liver Metastases Required
- **Portal Venous Drainage** → Tumour Products → **Liver** → **First-Pass Metabolism** (Monoamine Oxidase, Lung) → **Systemic Circulation Only If Liver Metastases** (or Ovarian Primary → Systemic Venous Drainage)
- **Foregut NETs** (Bronchus, Stomach) → Systemic Drainage → **Carcinoid Syndrome Without Liver Mets**
- **Hindgut NETs** (Rectum, Distal Colon) → **No Carcinoid Syndrome** (No 5-HT Secretion)

## 2. Clinical Features — Classic Triad

| Feature | Frequency | Details |
|---------|-----------|---------|
| **Flushing** | **90%** | **Dry** (Serotonin) vs **Wet** (Bradykinin); Triggers: Alcohol, Stress, Tyramine, Cheese, Exercise |
| **Diarrhoea** | **80%** | Watery, Explosive, Nocturnal; **Secretory** (5-HT → ↑ cAMP → Cl⁻ Secretion) |
| **Valvular Heart Disease** | **50-60%** | **Right-Sided** (Tricuspid Regurgitation, Pulmonary Stenosis); **Left-Sided Rare** (PFO/Shunt) |
| **Bronchospasm** | 10-20% | Wheeze, Dyspnoea (5-HT, Bradykinin) |
| **Fibrosis** | 10-20% | **Retroperitoneal**, Mesenteric, Pleural (Trapped Lung), Cardiac (Right-Sided) |
| **Pellagra** | Rare | Niacin Deficiency (Tryptophan → Serotonin) |
| **Carcinoid Crisis** | Rare | **Life-Threatening**: Severe Flushing, Hypotension, Bronchospasm, DIC; Trigger: Anaesthesia, Embolisation, Chemotherapy |

---

## 2. Diagnosis — Biochemical & Imaging

### Biochemical Markers
| Marker | Sensitivity | Specificity | Notes |
|--------|-------------|-------------|-------|
| **24h Urine 5-HIAA** | **75-85%** | **High** | **Gold Standard**; Collect in Acid (HCl); Avoid High-Serotonin Foods (Banana, Walnut, Tomato, Pineapple, Kiwi, Avocado) 48h Prior |
| **Plasma Chromogranin A (CgA)** | **90-95%** | 80% | **General NET Marker**; ↑ in Renal Failure, PPI Use, Atrophic Gastritis |
| **Plasma Serotonin** | Variable | Variable | Less Standardised |
| **Plasma Kynurenine** | Emerging | — | Tryptophan Metabolism Ratio (Kynurenine/Tryptophan) |

### 5-HIAA Collection Protocol
| Step | Instruction |
|---------|-------------|
| **Diet** | **Low-Serotonin Diet 48h** (Avoid: Banana, Walnut, Tomato, Pineapple, Kiwi, Avocado, Plum, Eggplant, Chocolate, Coffee, Tea) |
| **Collection** | **24h Urine + HCl Preservative** (10-20mL Conc. HCl) |
| **Normal** | **<15 mg/24h (8 mg/24h Some Labs)** |
| **Positive** | **>25-30 mg/24h** (Often >100 in Syndrome) |

## 3. Carcinoid Heart Disease (Right-Sided Valvulopathy)

| Feature | Details |
|---------|---------|
| **Pathology** | **Fibrotic Plaques** on Valvular Endocardium (Serotonin → Fibroblast Activation → Collagen Deposition) |
| **Valves Affected** | **Tricuspid Regurgitation (TR)** > **Pulmonary Stenosis (PS)**; **Left-Sided Rare** (PFO/ASD → Systemic Circulation) |
| **Echo Findings** | **Thickened, Retracted, Immobile** Leaflets; **Fixed, Restricted Opening** (PS); **Wide-Open** (TR) |
| **Clinical** | Right Heart Failure (Peripheral Oedema, Ascites, JVD, Hepatomegaly), Exertional Dyspnoea |
| **Prognosis** | **Poor** (Median Survival 1-2 Years Untreated); Valve Replacement Improves Survival |

## 4. Management — Acute & Chronic

### Acute Carcinoid Crisis (Medical Emergency)
| Trigger | Surgery, Embolisation, Chemotherapy, Biopsy, Stress |
|---------|-----------------------------------------------------|
| **Features** | Severe Flushing, Hypotension, Bronchospasm, DIC, Cardiovascular Collapse |
| **Immediate Rx** | **Octreotide 500-1000µg IV Bolus** → **50-100µg/hr Infusion** |
| **Adjuncts** | Fluids, Vasopressors (Noradrenaline), Bronchodilators, Antihistamines, Corticosteroids |
| **Prophylaxis** | **Octreotide 200-500µg IV Pre-Procedure** (Anaesthesia, Embolisation, Biopsy) |

---

## 4. Chronic Management

### Somatostatin Analogues (1st Line)
| Agent | Dose | Route | Frequency |
|-------|------|-------|-----------|
| **Octreotide LAR** | 20-30mg IM | q4 Weeks |
| **Lanreotide Autogel** | 60-120mg SC | q4 Weeks |
| **Pasireotide LAR** | 40-60mg IM | q4 Weeks (2nd Line; ↑ Hyperglycaemia) |

| Effect | Details |
|--------|---------|
| **Symptom Control** | **Flushing ↓ 70-90%**, **Diarrhoea ↓ 50-70%** |
| **Tumour Control** | **Stabilisation** (↓ Growth Rate); **PFS Prolongation** (CLARINET) |
| **Side Effects** | GI (Nausea, Steatorrhoea), Gallstones, Hyperglycaemia, Injection Site Pain |

### Telotristat Ethyl (Tryptophan Hydroxylase Inhibitor)
| Indication | Refractory Diarrhoea on SST Analogues |
|----------|---------------------------------------|
| **Dose** | **250mg TDS** (With Meals) |
| **Mechanism** | **Tryptophan Hydroxylase Inhibitor** → ↓ Serotonin Synthesis |
| **Effect** | ↓ Diarrhoea Frequency, ↓ 5-HIAA |
| **Adverse** | Nausea, Abdominal Pain, Peripheral Oedema |

### Interferon-α (Historical/Refractory)
| Dose | 3-9 MU SC 3x/week |
|--------|-------------------|
| **Efficacy** | Symptom Control, Tumour Stabilisation |
| **Toxicity** | Flu-Like, Depression, Myelosuppression, Thyroiditis |

## 4. Carcinoid Heart Disease — Management

### Medical
| Agent | Role |
|-------|--------|
| **Diuretics** | Loop (Furosemide) + Aldosterone Antagonist (Spironolactone) |
| **Anticoagulation** | **Warfarin** (Atrial Fibrillation / Thrombus Risk) |
| **Octreotide** | Symptom Control; May Slow Valve Progression |

### Surgical
| Indication | Procedure |
|-----------|---------|
| **Severe TR/PS + Symptoms** | **Valve Replacement** (Bioprosthetic Preferred — Avoid Anticoagulation) |
| **Timing** | **Before Severe RV Dysfunction** (TAPSE <16mm, RVSP >60mmHg) |
| **Percutaneous** | **Balloon Valvuloplasty (PS)** / **Transcatheter Valve Replacement** (TR) — Emerging |

## 5. Tumour Control & Surveillance

### Tumour Grading (WHO 2019) → Determines Follow-Up
| Grade | Ki-67 | Mitotic Count (2mm²) | Surveillance |
|-------|-------|----------------------|--------------|
| **NET G1** | <3% | <2 | **Annual** CT/MRI + 5-HIAA + CgA |
| **NET G2** | 3-20% | 2-20 | **6-Monthly** CT/MRI + 5-HIAA + CgA |
| **NEC G3** | >20% | >20 | **3-Monthly** CT + 5-HIAA + CgA |

### Somatostatin Analogues (Tumour Control)
| Agent | Dose | Indication |
|-------|------|-----------|
| **Octreotide LAR** | 20-30mg IM q4wk | 1st Line |
| **Lanreotide Autogel** | 60-120mg SC q4wk | 1st Line Alternative |
| **Pasireotide LAR** | 40-60mg IM q4wk | 2nd Line (Resistant; ↑ Hyperglycaemia) |

### Peptide Receptor Radionuclide Therapy (PRRT)
| Agent | **¹⁷⁷Lu-DOTATATE (Lutathera)** |
|--------|------------------------------|
| **Indication** | **SSTR2+ Progressive NET G1/G2** (NETTER-1 Trial) |
| **Protocol** | **7.4 GBq IV q8wk × 4 Cycles** + Amino Acid Infusion (Renal Protection) |
| **Efficacy** | **PFS 65% vs 11% (Octreotide LAR)** at 20mo (NETTER-1) |
| **Toxicity** | Nausea, Myelosuppression, Renal (Amino Acid Protection) |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Carcinoid Syndrome vs Flushing** | Syndrome = Flushing + Diarrhoea + Valvular Disease (+/- Bronchospasm) |
| **Right vs Left Heart** | **Right-Sided** (Portal Drainage → Liver → Systemic); Left Only if PFO/Shunt |
| **5-HIAA Diet** | Avoid Banana, Walnut, Tomato, Pineapple, Kiwi, Avocado, Chocolate, Coffee 48h |
| **Carcinoid Crisis** | Peri-Procedural; **Octreotide 500-1000µg IV Bolus → Infusion** |
| **Valve Replacement** | **Bioprosthetic Preferred** (Avoid Anticoagulation in Cachectic Patients) |
| **PRRT** | ¹⁷⁷Lu-DOTATATE; NETTER-1: PFS 65% vs 11% (Octreotide) |
| **Telotristat** | Tryptophan Hydroxylase Inhibitor → ↓ Serotonin → ↓ Diarrhoea (Refractory) |
| **Carcinoid vs Flushing** | Carcinoid = Triad (Flush, Diarrhoea, Valve) + 5-HIAA↑ |
| **Octreotide vs Lanreotide** | Similar Efficacy; Lanreotide SC (Self-Admin); Octreotide IM (Pro) |
| **Telotristat Ethyl** | Tryptophan Hydroxylase Inhibitor; Refractory Diarrhoea Only |

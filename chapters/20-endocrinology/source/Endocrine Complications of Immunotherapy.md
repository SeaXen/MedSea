# Endocrine Complications of Immunotherapy

> [!info]
> **Immune Checkpoint Inhibitor (ICI) Endocrine Toxicities (irAEs) = Autoimmune Endocrinopathies Triggered by ICIs.** **Anti-CTLA-4 (Ipilimumab) > Anti-PD-1/PD-L1** for Hypophysitis; **Anti-PD-1/PD-L1 > Anti-CTLA-4** for Thyroiditis, Diabetes, Adrenal Insufficiency. **Hormone Replacement = Mainstay (Not Immunosuppression for Hormone Deficiency).**

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise ICI-related endocrine irAEs (Hypophysitis, Thyroiditis, Adrenal Insufficiency, Type 1 Diabetes)
- [ ] Apply grading (CTCAE) and management algorithms
- [ ] Differentiate primary vs secondary adrenal insufficiency in ICI context
- [ ] Apply monitoring schedules for patients on ICIs
- [ ] Manage ICI continuation vs hold vs discontinuation

## 1. ICI Endocrine Toxicities — Overview

| Agent Class | Commonest Endocrine irAE | Incidence | Onset |
|-------------|--------------------------|-----------|-------|
| **Anti-CTLA-4** (Ipilimumab) | **Hypophysitis** | **5-15%** (High) | 6-12 Weeks |
| **Anti-PD-1/PD-L1** (Nivolumab, Pembrolizumab, Atezolizumab) | **Thyroiditis** (Hyper → Hypo) | **5-10%** | 6-18 Weeks |
| **Anti-PD-1/PD-L1** | **Primary Adrenal Insufficiency** | **1-5%** | 8-20 Weeks |
| **Anti-PD-1/PD-L1** | **Type 1 Diabetes (Autoimmune)** | **<1%** | Variable |
| **Combination (CTLA-4 + PD-1)** | **Higher Rates** (All Endocrine) | **20-30%** (Any) | Earlier |

## 2. Hypophysitis (Anti-CTLA-4 Predominant)

### Clinical Features
| Feature | Details |
|--------|---------|
| **Onset** | **6-12 Weeks** After ICI Initiation |
| **Symptoms** | **Headache** (Bifrontal), Fatigue, Visual Disturbance, Nausea, Polydipsia/Polyuria (DI) |
| **Signs** | Visual Field Defects (Bitemporal), Cranial Nerve Palsies |

### Biochemical Pattern
| Hormone | Pattern |
|---------|---------|
| **ACTH/Cortisol** | **First Affected** (Central Adrenal Insufficiency) → **Low/Inappropriately Normal ACTH, Low Cortisol** |
| **TSH/fT4** | **Central Hypothyroidism** → Low/Normal TSH, Low fT4 |
| **LH/FSH** | **Central Hypogonadism** → Low/Normal LH/FSH, Low Sex Steroids |
| **Prolactin** | **Variable** (May Be High/Normal/Low) |

### MRI Findings
| Finding | Description |
|---------|---------|
| **Pituitary Enlargement** | **Homogeneous Enhancement** (Inflammatory) |
| **Stalk Thickening** | >2-3mm |
| **Mass Effect** | Optic Chiasm Compression (Visual Fields) |

### Grading & Management (CTCAE v5.0)

| Grade | Clinical | Management |
|-------|----------|------------|
| **G1** | Asymptomatic; Biochemical Only | **Continue ICI**; Hormone Replacement; Monitor q2-4wk |
| **G2** | Symptomatic (Mild); No End-Organ Dysfunction | **Hold ICI**; Hormone Replacement; **Prednisolone 0.5-1mg/kg** if Symptomatic |
| **G3** | Severe; End-Organ Dysfunction (Adrenal Crisis Risk) | **Discontinue ICI**; **IV Hydrocortisone 100mg q6h** → Oral Taper; **High-Dose Steroids** (Pred 1-2mg/kg) |
| **G4** | Life-Threatening (Adrenal Crisis, Coma) | **Discontinue ICI**; **ICU**; IV Hydrocortisone 100mg q6h; High-Dose Steroids |

### Hormone Replacement Protocol
| Axis | Replacement | Notes |
|------|------------|------|
| **Adrenal (AI)** | **Hydrocortisone 15-20mg AM + 5-10mg PM** | **STRESS DOSES** for Illness/Surgery |
| **Thyroid** | **Levothyroxine 1.6µg/kg/day** | Start **after** Cortisol Replacement (Else Crisis) |
| **Gonadal** | Testosterone / Oestrogen + Progestogen | If Symptomatic / Fertility Desired |

### ICI Management
| Severity | ICI Action | Steroid |
|----------|------------|---------|
| **G1** | **Continue** | None |
| **G2** | **Hold** | Prednisolone 0.5-1mg/kg (If Symptomatic) |
| **G3/G4** | **Discontinue** | **IV Hydrocortisone 100mg q6h** → Oral Taper (4-12 Weeks) |

## 2. Thyroiditis (Anti-PD-1/PD-L1 Predominant)

### Natural History
| Phase | Timing | TSH | fT4/fT3 | TRAb |
|-------|--------|-----|---------|------|
| **Thyrotoxic** | **6-12 Weeks** | **Suppressed** (<0.01) | **High** | **Negative** |
| **Hypothyroid** | **12-24 Weeks** | **High** | **Low** | **Negative** |
| **Permanent Hypo** | 20-40% | High | Low | Negative |

> **Key**: **TRAb Negative** (vs Graves); **Painless** (vs Subacute Thyroiditis).

### Clinical Features
| Phase | Symptoms |
|-------|----------|
| **Thyrotoxic** | Palpitations, Tremor, Weight Loss, Anxiety, Heat Intolerance |
| **Hypothyroid** | Fatigue, Weight Gain, Cold Intolerance, Constipation, Depression |

### Management

| Phase | Treatment |
|-------|-----------|
| **Thyrotoxic** | **Beta-Blocker (Propranolol 20-40mg TDS)** for Symptom Control; **No Antithyroid Drugs** (Destructive) |
| **Hypothyroid** | **Levothyroxine 1.6µg/kg/day** (Start Low 25-50µg in Elderly) |
| **Subclinical** | Monitor q6-8wk; Treat if TSH >10 or Symptomatic |

### Grading & ICI Management (CTCAE)

| Grade | Thyroid Function | ICI Action |
|-------|------------------|------------|
| **G1** | Asymptomatic; fT4 Normal | **Continue ICI** |
| **G2** | Symptomatic Thyrotoxic / Hypo | **Continue ICI** (May Hold if Severe) |
| **G3** | Severe Thyrotoxic (AF, HF) / Severe Hypo (Myxoedema) | **Hold ICI** → Consider Steroids (Pred 0.5-1mg/kg) |
| **G4** | Life-Threatening (Thyroid Storm / Myxoedema Coma) | **Discontinue ICI** → ICU + Steroids |

## 2. Primary Adrenal Insufficiency (Anti-PD-1/PD-L1)

| Feature | Details |
|---------|---------|
| **Incidence** | **1-5%** (Higher with Combo) |
| **Onset** | **8-20 Weeks** |
| **Mechanism** | **Autoimmune Adrenalitis** (Anti-Adrenal Antibodies) |
| **Biochemistry** | **Low Cortisol, High ACTH, Hyperkalaemia, Hyponatraemia** |
| **Imaging** | Adrenal Enlargement (Early) → Atrophy (Late) |

### Management
| Severity | Management |
|----------|------------|
| **G1** | Monitor; Continue ICI |
| **G2** | **Hydrocortisone 15-20mg AM + 5-10mg PM**; Hold ICI |
| **G3** | **IV Hydrocortisone 100mg q6h**; Discontinue ICI |
| **G4** | **Adrenal Crisis Protocol** (IV Hydrocortisone 100mg q6h + Fluids); Discontinue ICI |

---

## 2. Type 1 Diabetes (ICI-Induced)

| Feature | Details |
|---------|---------|
| **Incidence** | **<1%** (Rare but Critical) |
| **Mechanism** | **Autoimmune β-Cell Destruction** (Anti-GAD, IA-2, ZnT8+) |
| **Presentation** | **Acute Diabetic Ketoacidosis (DKA)** Often First Presentation |
| **Onset** | Variable (Weeks-Months) |
| **Biochemistry** | **Autoantibodies Positive** (GAD65, IA-2, ZnT8); C-Peptide Low |

### Management
| Scenario | Management |
|---------|-----------|
| **DKA at Presentation** | **Standard DKA Protocol** (Fluids, Insulin, K⁺); **ICI Hold/Discontinue** |
| **New-Onset Diabetes** | **Insulin Initiation** (Basal-Bolus); **Discontinue ICI** (Usually) |
| **Autoantibodies** | **GAD65, IA-2, ZnT8** Positive (Confirm Autoimmune) |

## 2. ICI Monitoring Schedule

| Timepoint | Tests |
|-----------|-------|
| **Baseline (Pre-ICI)** | TSH, fT4, fT3, Cortisol (9am), ACTH, Glucose, HbA1c, Electrolytes |
| **Every 3-4 Weeks (During ICI)** | TSH, fT4, Cortisol (9am), Glucose, Electrolytes |
| **At Symptom Onset** | Full Endocrine Panel (TSH, fT4, fT3, Cortisol, ACTH, Prolactin, LH/FSH, Testosterone/Oestradiol, Glucose, HbA1c, Electrolytes) |
| **Post-ICI Completion** | q3mo × 1yr, then q6mo |

## 3. Management Algorithm — ICI Endocrine irAEs

```
ICI Patient + New Endocrine Symptoms / Abnormal Labs
         │
         ▼
IDENTIFY AXIS: HPA / HPT / HPG / ADRENAL / PANCREAS
         │
         ▼
GRADE (CTCAE v5.0)
         │
         ├── G1: Asymptomatic / Biochemical Only
         │     → CONTINUE ICI; Hormone Replacement; Monitor q2-4wk
         │
         ├── G2: Symptomatic / Mild End-Organ Dysfunction
         │     → HOLD ICI; Hormone Replacement; Prednisolone 0.5-1mg/kg (If Symptomatic)
         │
         ├── G3: Severe / End-Organ Dysfunction
         │     → DISCONTINUE ICI; IV Hydrocortisone 100mg q6h; High-Dose Steroids
         │
         └── G4: Life-Threatening
               → DISCONTINUE ICI; ICU; IV Hydrocortisone 100mg q6h; High-Dose Steroids
```

## 4. Steroid Tapering Protocol (Post-irAE)

| Phase | Dose | Duration |
|-------|------|----------|
| **Induction** | **Prednisolone 1-2mg/kg/day** | Until Symptoms Resolve + Hormones Stabilised |
| **Taper** | **Reduce 10mg q1-2wk** to 20mg | 4-8 Weeks |
| **Slow Taper** | **2.5-5mg q2-4wk** to 5mg | 8-12 Weeks |
| **Physiological** | **Hydrocortisone 15-20mg AM + 5-10mg PM** | Lifelong if Permanent AI |

> **Key**: **Taper Slowly to Avoid Adrenal Crisis**; Faster Taper if ICI Rechallenge Planned.

---

## 4. ICI Rechallenge After irAE

| Scenario | Decision |
|---------|----------|
| **Endocrine G1/G2** | **Consider Rechallenge** (If Hormones Stable on Replacement) |
| **Endocrine G3/G4** | **Generally Avoid Rechallenge** (High Recurrence Risk) |
| **Non-Endocrine irAE G3/G4** | **Generally Avoid Rechallenge** |
| **Pre-Rechallenge** | Hormones Stable on Replacement; No Acute Symptoms; MDT Discussion |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Hypophysitis vs Primary Adrenal AI** | Hypophysitis = **Central** (Low/Normal ACTH); Primary AI = **High ACTH** |
| **ICI Thyroiditis vs Graves** | ICI Thyroiditis = **TRAb Negative**, Painless, Thyrotoxic→Hypo Sequence |
| **ICI Thyroiditis vs Subacute** | ICI = Painless, TRAb(-); Subacute = Painful, TRAb(-) |
| **Hypophysitis ACTH Pattern** | **Central AI**: Low/Inappropriately Normal ACTH, Low Cortisol |
| **Primary AI ACTH Pattern** | **Primary**: High ACTH, Low Cortisol, Hyperkalaemia, Hyponatraemia |
| **Steroid for Hypophysitis** | **High-Dose for Acute (Pred 1-2mg/kg)**; Then Taper |
| **ICI Management** | G1/G2: Hold/Continue; G3/G4: Discontinue |
| **Thyroiditis Sequence** | Thyrotoxic (Weeks 6-12) → Hypothyroid (Weeks 12-24) → Permanent (20-40%) |
| **Thyroiditis G3** | Severe Thyrotoxic (AF, HF) / Severe Hypo (Myxoedema) |
| **Type 1 DM ICI** | **DKA Often First Presentation**; Autoantibodies +; C-Peptide Low |
| **Steroid Taper** | Slow (4-12+ Weeks); Physiological Replacement if Permanent |
| **Rechallenge** | G1/G2 Endocrine: Consider; G3/G4: Generally Avoid |

## Flashcards
- **Q: ICI hypophysitis (anti-CTLA-4)**
  **A: Ipilimumab: 5-10%; diffuse pituitary enlargement → ACTH/TSH/LH/FSH deficiency; 6-12 wks**

- **Q: ICI thyroiditis (anti-PD-1)**
  **A: Nivolumab/pembrolizumab: 5-10%; destructive thyroiditis → thyrotoxicosis → hypothyroidism; 6-12 wks**

- **Q: ICI adrenal insufficiency**
  **A: Primary adrenalitis (cortisol↓, ACTH↑) or secondary (hypophysitis); check 8am cortisol, ACTH**

- **Q: ICI type 1 DM**
  **A: Acute onset, DKA common, low C-peptide, positive GAD65/IA-2/ZnT8; anti-PD-1 > anti-CTLA-4**

- **Q: ICI hypophysitis Rx**
  **A: Hold ICI; hydrocortisone 100mg IV q6h → pred 1-2mg/kg taper; thyroxine AFTER cortisol**

- **Q: ICI thyroiditis Rx**
  **A: Thyrotoxic phase: beta-blocker ± steroids; hypothyroid phase: levothyroxine; often continue ICI**

- **Q: ICI adrenal Rx**
  **A: Hydrocortisone 15-25mg/day (10/5/5 or 10/5); fludrocortisone if primary; hold ICI if Grade ≥3**

- **Q: ICI DM Rx**
  **A: Insulin mandatory; hold ICI if DKA; consider restarting after stabilisation with specialist input**

- **Q: Monitoring**
  **A: Baseline: TSH, fT4, 8am cortisol, ACTH, glucose, HbA1c, electrolytes; q4-6wk on treatment**

- **Q: Grade toxicity**
  **A: G1: asymptomatic; G2: symptomatic, hold ICI; G3: severe, hold + high-dose steroids; G4: life-threatening, permanent stop**

- **Q: Anti-CTLA-4 vs PD-1**
  **A: CTLA-4: higher hypophysitis, colitis; PD-1: higher thyroiditis, pneumonitis, diabetes**

- **Q: Combination ICI**
  **A: Ipi+Nivo: highest endocrine toxicity (hypophysitis 10-15%, thyroiditis 15-20%, adrenal 5%)**

- **Q: Rechallenge**
  **A: After Grade 2: consider restart if controlled on hormone replacement + steroids tapered**

- **Q: Primary vs secondary AI**
  **A: Primary: cortisol↓ ACTH↑ hyperkalaemia hyponatraemia; Secondary: cortisol↓ ACTH↓/normal euK**

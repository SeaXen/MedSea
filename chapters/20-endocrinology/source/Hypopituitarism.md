# Hypopituitarism

> [!info]
> **Hypopituitarism = Deficiency of one or more anterior pituitary hormones.** Caused by pituitary/hypothalamic damage. **Order of loss: GH → Gonadotrophs → TSH → ACTH** (most to least vulnerable). Panhypopituitarism = All axes deficient.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Define hypopituitarism and classify by severity (partial vs panhypopituitarism)
- [ ] Identify aetiologies (pituitary, hypothalamic, systemic)
- [ ] Apply the hierarchical pattern of hormone loss to clinical presentations
- [ ] Prescribe appropriate hormone replacement regimens
- [ ] Manage special situations (pregnancy, surgery, adrenal crisis, Sheehan syndrome)

---

## 1. Definition & Classification

| Category | Definition |
|----------|------------|
| **Partial Hypopituitarism** | Deficiency of 1-2 anterior pituitary axes |
| **Panhypopituitarism** | Deficiency of **all** anterior pituitary axes |
| **Selective Deficiency** | Isolated single hormone deficiency (e.g., isolated GHD, isolated ACTH deficiency) |

---

## 2. Aetiology

| Category | Specific Causes |
|----------|-----------------|
| **Pituitary Adenoma** | Macroadenoma (compression), Apoplexy, Surgery, Radiotherapy |
| **Hypothalamic** | Craniopharyngioma, Meningioma, Glioma, Sarcoidosis, LCH, Trauma |
| **Vascular** | Pituitary Apoplexy, Sheehan Syndrome (Postpartum Necrosis), Carotid Aneurysm |
| **Inflammatory/Infiltrative** | Sarcoidosis, LCH, Haemochromatosis, TB, Metastases, Lymphoma |
| **Genetic** | PROP1, POU1F1, LHX3, HESX1, SOX3, OTX2 |
| **Autoimmune** | Lymphocytic Hypophysitis (Postpartum), Autoimmune Polyendocrine Syndromes (APS-1, APS-2) |
| **Iatrogenic** | Surgery (Transsphenoidal), Radiotherapy (Delayed), Drugs (Checkpoint Inhibitors) |
| **Trauma** | Basal Skull Fracture, Diffuse Axonal Injury |

---

## 2. Pattern of Hormone Loss — Hierarchical Vulnerability

| Order of Loss | Hormone | Mechanism | Clinical Presentation |
|---------------|---------|-----------|----------------------|
| **1st (Most Vulnerable)** | **GH** | Somatotrophs most sensitive to compression/ischaemia | Children: Growth Failure; Adults: Metabolic Syndrome, ↓ QoL, ↑ Visceral Fat |
| **2nd** | **Gonadotrophs (LH/FSH)** | Gonadotrophs sensitive to compression | Infertility, Amenorrhoea, Low Libido, Low Testosterone/Oestradiol |
| **3rd** | **TSH** | Thyrotrophs less sensitive | Central Hypothyroidism (Fatigue, Cold Intolerance, Weight Gain) |
| **4th (Least Vulnerable)** | **ACTH** | Corticotrophs most resistant | **Adrenal Insufficiency** (Fatigue, Hypotension, Hyponatraemia, Hyperkalaemia) |

**Mnemonic**: **G**o **G**o **T**o **A**drenal (GH → Gonadotrophs → TSH → ACTH)

## 3. Aetiology — Detailed

### Pituitary Adenoma (Most Common Cause)
| Type | Mechanism |
|------|-----------|
| **Macroadenoma** | Direct Compression of Normal Pituitary Tissue |
| **Pituitary Apoplexy** | Haemorrhagic Infarction → Acute Hypopituitarism |
| **Post-Surgical** | Transsphenoidal Resection (Damage to Normal Pituitary) |
| **Post-Radiotherapy** | Delayed (5-10yr) Progressive Hypopituitarism |

### Hypothalamic / Suprasellar
| Condition | Mechanism |
|---------|-----------|
| **Craniopharyngioma** | Compression of Pituitary Stalk + Hypothalamus |
| **Meningioma** | Suprasellar Compression |
| **Infiltrative** | Sarcoidosis, LCH, Haemochromatosis, TB, Metastases |
| **Vascular** | Carotid Aneurysm (Compress Pituitary Stalk) |

### Vascular
| Condition | Details |
|---------|---------|
| **Pituitary Apoplexy** | Acute Haemorrhage/Infarction → Acute Panhypopituitarism |
| **Sheehan Syndrome** | Postpartum Pituitary Necrosis (Vasospasm + Enlarged Gland) |
| **Carotid Aneurysm** | Compression of Pituitary Stalk |

### Infiltrative / Inflammatory
| Condition | Features |
|---------|----------|
| **Sarcoidosis** | Neurosarcoid; CSF ACE ↑; Pituitary Stalk Thickening |
| **LCH** | Diabetes Insipidus + Anterior Hypopituitarism; Bone Lesions |
| **Haemochromatosis** | Iron Deposition in Pituitary + Liver + Heart + Pancreas |
| **Lymphocytic Hypophysitis** | Postpartum; Autoimmune; MRI: Diffuse Pituitary Enlargement |
| **Autoimmune Polyendocrine Syndromes** | APS-1 (AIRE), APS-2 (Addison + Thyroid + DM1) |

### Genetic
| Gene | Inheritance | Phenotype |
|------|-------------|-----------|
| **PROP1** | AR | Multiple Pituitary Hormone Deficiency (CPHD) |
| **POU1F1** | AR | GH + TSH + PRL Deficiency |
| **LHX3** | AR | CPHD + Rigid Cervical Spine |
| **HESX1** | AR/AD | Septo-optic Dysplasia + CPHD |
| **SOX3** | X-linked | CPHD + Mental Retardation |
| **OTX2** | AD | CPHD + Eye Anomalies |

## 3. Clinical Presentation by Axis

### GH Deficiency
| Population | Features |
|------------|----------|
| **Children** | **Growth Failure** (Height < -2 SDS), ↓ Growth Velocity, ↓ Bone Age, Hypoglycaemia |
| **Adults** | ↑ Visceral Fat, ↓ Lean Mass, ↓ BMD, ↓ QoL, Dyslipidaemia, ↓ Exercise Capacity, ↑ Cardiovascular Risk |

### Gonadotropin Deficiency
| Sex | Features |
|-----|----------|
| **Male** | Low Libido, ED, Infertility, ↓ Beard/Body Hair, ↓ Muscle Mass, Hot Flushes, Osteoporosis |
| **Female** | **Secondary Amenorrhoea** (OLIGO/AMEN), Infertility, Hot Flushes, Vaginal Atrophy, Osteoporosis |

### TSH Deficiency (Central Hypothyroidism)
| Features | Details |
|--------|---------|
| **Biochemistry** | **Low fT4 + Inappropriately Low/Normal TSH** |
| **Clinical** | Fatigue, Cold Intolerance, Weight Gain, Constipation, Bradycardia, Hyperlipidaemia |
| **Differentiation** | Primary Hypo: TSH ↑↑; Central: TSH Low/Normal + fT4 Low |

### ACTH Deficiency (Secondary Adrenal Insufficiency)
| Feature | Details |
|---------|---------|
| **Biochemistry** | Low Cortisol + **Low/Inappropriately Normal ACTH** |
| **Clinical** | Fatigue, Weakness, Anorexia, Weight Loss, Hypotension, Hypoglycaemia, Hyponatraemia, **NO Hyperpigmentation** (ACTH Low) |
| **Differentiation** | Primary: Cortisol ↓ + ACTH ↑↑ + Hyperpigmentation; Secondary: Cortisol ↓ + ACTH ↓/N + No Pigmentation |

## 4. Common Aetiology-Specific Presentations

### Sheehan Syndrome (Postpartum Pituitary Necrosis)
| Feature | Details |
|---------|---------|
| **Pathophysiology** | Pituitary Enlarges 2x in Pregnancy → Vasospasm/Thrombosis Postpartum → Infarction |
| **Risk Factors** | Postpartum Haemorrhage, Hypotension, DIC, Multiple Gestation |
| **Onset** | **Acute** (Days) or **Subacute/Insidious** (Months-Years) |
| **Classic Triad** | **Agallactorrhoea + Amenorrhoea + Loss of Pubic/Axillary Hair** |
| **Hormones Lost** | **GH > Gonadotrophs > TSH > ACTH** (ACTH often preserved initially) |
| **Diagnosis** | MRI: Pituitary Infarction → Empty Sella; Hormonal Panels |
| **Management** | Lifelong Hormone Replacement; Glucocorticoid Priority; GH if Indicated |

### Lymphocytic Hypophysitis
| Feature | Details |
|--------|---------|
| **Timing** | **Postpartum** (3-6 months) or Peripartum |
| **Pathology** | Autoimmune Lymphocytic Infiltration of Pituitary |
| **MRI** | **Diffuse Pituitary Enlargement**, Stalk Thickening, Homogeneous Enhancement |
| **Hormones** | ACTH Deficiency (Most Common), TSH, Gonadotrophs; Prolactin ↑ (Stalk Effect) |
| **Management** | High-Dose Steroids (Pred 1mg/kg) + Hormone Replacement; Taper Over Months |

### Pituitary Apoplexy
| Feature | Details |
|--------|---------|
| **Definition** | Acute Haemorrhage/Infarction in Pituitary Adenoma |
| **Triggers** | Coagulopathy, Hypertension, Pregnancy, Trauma, Anticoagulation, Dynamic Testing |
| **Clinical** | **Sudden Severe Headache**, Visual Loss, Ophthalmoplegia, Hypotension, Altered Consciousness |
| **Emergency Mgmt** | **Hydrocortisone 100mg IV STAT** + Fluids + **Dexamethasone 4-8mg IV q6h** + Urgent MRI → TSS if Deteriorating |

## 4. Diagnosis — Investigations

### Hormonal Panel
| Axis | Tests | Interpretation |
|-------|-------|----------------|
| **GH/IGF-1** | IGF-1 (Age-Adjusted), GH (Random/Stim) | ↓ IGF-1 + Low GH Peak on Stim = GHD |
| **Gonadal** | Testosterone/Oestradiol + LH/FSH | Low Steroid + Low/Normal LH/FSH = Secondary Hypogonadism |
| **Thyroid** | fT4 + TSH | Low fT4 + Low/Normal TSH = Central Hypothyroidism |
| **Adrenal** | 09:00 Cortisol + ACTH | Low Cortisol + Low/Normal ACTH = Secondary AI |
| **Prolactin** | Prolactin | Normal/Low (unless Stalk Effect) |
| **GH Stim** | ITT / Glucagon / GHRH+Arg | Peak GH <3 µg/L (Adults) = GHD |

### Imaging
| Modality | Indication |
|-----------|------------|
| **MRI Pituitary (3mm, Gadolinium)** | **Gold Standard**; Pituitary Stalk, Adenoma, Infiltration |
| **CT** | If MRI Contraindicated; Bony Detail (Craniopharyngioma) |
| **Visual Fields (Goldmann/Humphrey)** | Bitemporal Hemianopia (Chiasmal Compression) |

### Genetic Testing
| Gene | Indication |
|------|----------|
| **PROP1** | Multiple Pituitary Hormone Deficiency (CPHD) |
| **POU1F1** | GH + TSH + PRL Deficiency |
| **LHX3** | CPHD + Rigid Cervical Spine |
| **HESX1** | Septo-optic Dysplasia + CPHD |
| **SOX3** | CPHD + Mental Retardation (X-linked) |

## 5. Hormone Replacement Therapy

### Glucocorticoid (ACTH Deficiency)
| Regimen | Dose |
|---------|-------|
| **Hydrocortisone** | **15-20mg AM + 5-10mg PM** (or 10mg TDS) |
| **Prednisolone** | 5mg OD (Longer half-life) |
| **Stress Dosing** | **Minor**: 25mg IV; **Moderate**: 50-100mg IV; **Major**: 100mg IV + 50mg q6h |
| **Monitoring** | Weight, BP, Glucose, BP, Skin, Bone (DEXA q2-3yr) |

### Thyroxine (TSH Deficiency)
| Regimen | Dose |
|---------|-------|
| **Levothyroxine** | **1.6 µg/kg/day** (e.g., 100-150µg) |
| **Initiation** | Start Low (25-50µg) → Titrate q4-6wk by 25µg |
| **Target** | fT4 Upper Half of Normal; TSH N/A (Pituitary) |
| **Special** | **Initiate BEFORE Glucocorticoid** (Avoid Adrenal Crisis) |

### Sex Steroid Replacement
| Population | Regimen |
|-----------|---------|
| **Male (Secondary Hypogonadism)** | **Testosterone Gel** 5g/day (50mg) / IM Nebido 1000mg q10-14wk / Enantate 250mg q3-4wk |
| **Female (POI/Secondary Hypogonadism)** | **HRT** (Oestradiol + Progestogen) until Age 50; **Fertility**: hCG/FSH if Desired |
| **Fertility (Male)** | **hCG** 1500-5000 IU 2-3x/week OR **rFSH** + **hCG** |

### GH Replacement (Confirmed GHD)
| Population | Dose | Monitoring |
|------------|-------|----------|
| **Adult** | 0.1-0.3 mg/day SC → Titrate to IGF-1 Mid-Normal | IGF-1 q1-3mo ×1yr, then 6-monthly |
| **Child** | 0.025-0.035 mg/kg/day SC | Height Velocity, IGF-1, Bone Age q6-12mo |

## 5. Special Situations

### Pregnancy in Hypopituitarism
| Axis | Management |
|-------|----------|
| **Glucocorticoid** | Increase **20-40%** in 2nd/3rd Trimester; **Stress Doses for Delivery** |
| **Thyroxine** | ↑ Dose **30%** at +ve Test; Monitor TSH/fT4 q4-6wk |
| **Sex Steroids** | Continue HRT; **Stop Gonadotrophins**; Discuss Fertility Plans |
| **GH** | **Stop** (Contraindicated in Pregnancy) |
| **Delivery** | Stress Dose Hydrocortisone (100mg IV q6h) During Labour |

### Surgery in Hypopituitarism
| Phase | Management |
|--------|-----------|
| **Pre-op** | Stress Dose Hydrocortisone 100mg IV; Thyroxine continued |
| **Peri-op** | 50-100mg Hydrocortisone IV q6-8h × 24-48h; Taper |
| **Post-op** | Resume Usual Doses; Monitor U&Es, Glucose, BP |

### Radiotherapy-Induced Hypopituitarism
| Timeline | Features |
|----------|----------|
| **6-12 months** | GH Deficiency (First) |
| **2-5 years** | Gonadotroph Deficiency |
| **5-10 years** | TSH Deficiency |
| **>10 years** | ACTH Deficiency (Rare) |
| **Surveillance** | Annual Hormones for 10+ Years Post-RT |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Primary vs Secondary AI** | Primary: Cortisol↓ + ACTH↑↑ + Hyperpigmentation; Secondary: Cortisol↓ + ACTH↓ + NO Hyperpigmentation |
| **Central vs Primary Hypothyroidism** | Central: fT4↓ + TSH Low/Normal; Primary: TSH↑↑ + fT4↓ |
| **Hormone Loss Order** | GH → Gonad → TSH → ACTH (Mnemonic: **G**o **G**o **T**o **A**drenal) |
| **Replacement Priority** | **Thyroxine FIRST, then Glucocorticoid** (Avoid Adrenal Crisis) |
| **ITT in Hypopituitarism** | Contraindicated (Cortisol <100); Use Glucagon/GHRH+Arg |
| **GHD in Adults** | ↑ Visceral Fat, ↓ BMD, ↓ QoL, ↑ CV Risk |
| **Sheehan Syndrome** | Postpartum Pituitary Necrosis → Agalactorrhoea + Amenorrhoea + Hair Loss; MRI: Empty Sella |
| **Replacement Priority** | **Thyroxine BEFORE Glucocorticoid** (Prevents Adrenal Crisis) |
| **Stress Doses** | Minor 25mg; Moderate 50-100mg; Major 100mg + 50mg q6h |

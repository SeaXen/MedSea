# Prolactinoma

> [!info]
> **Prolactinoma = Prolactin-Secreting Pituitary Adenoma.** **Commonest Functioning Pituitary Adenoma (40-50%).** **Microprolactinoma (<10mm) 60%; Macroprolactinoma (≥10mm) 40%.** **Cabergoline = 1st Line Treatment.** **Macroprolactinoma in Pregnancy = Continue Cabergoline.**

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Diagnose prolactinoma using prolactin levels and MRI
- [ ] Differentiate true prolactinoma from macroprolactinaemia and stalk effect
- [ ] Apply cabergoline dosing and titration protocols
- [ ] Manage prolactinoma in pregnancy (micro vs macro)
- [ ] Apply withdrawal criteria and monitor for recurrence

---

## 2. Classification & Epidemiology

| Type | Size | Prolactin Level | Prevalence |
|------|------|-----------------|------------|
| **Microprolactinoma** | **<10mm** | 500-5000 mIU/L | **~60%** |
| **Macroprolactinoma** | **≥10mm** | **>5000 mIU/L** (Often >10,000) | **~40%** |
| **Giant** | **>40mm** | Very High | Rare |

| Feature | Details |
|---------|---------|
| **Incidence** | **Most Common Functioning Pituitary Adenoma (40-50%)** |
| **Sex Ratio** | **Female > Male (5:1)** |
| **Peak Age** | **20-40 Years** |
| **Genetics** | **MEN1** (Rare), **AIP** (Young-Onset Familial), **GPR101** |

## 2. Clinical Presentation

| Feature | Microprolactinoma | Macroprolactinoma |
|---------|-------------------|-------------------|
| **Prolactin Level** | 500-5000 mIU/L | **>5000 mIU/L** (Often >10,000) |
| **Presentation** | Galactorrhoea, Amenorrhoea, Infertility, Low Libido | Visual Field Defects (Bi-temporal Hemianopia), Headache, Hypopituitarism |
| **Visual Fields** | Normal | **Bi-temporal Hemianopia** (Chiasmal Compression) |
| **Headache** | Rare | Common |

### Other Clinical Features
| System | Features |
|---------|----------|
| **Reproductive** | Galactorrhoea, Amenorrhoea, Oligomenorrhoea, Infertility, Low Libido, Erectile Dysfunction |
| **Bone** | Osteoporosis (Oestrogen/Testosterone Deficiency) |
| **Metabolic** | Weight Gain, Insulin Resistance |

## 3. Differential Diagnosis of Hyperprolactinaemia

```
Prolactin >500 mIU/L
         │
         ▼
EXCLUDE PHYSIOLOGICAL / PHARMACOLOGICAL CAUSES:
         │
         ├── DRUGS: Antipsychotics (Risperidone > Olanzapine > Quetiapine), Metoclopramide, Antidepressants, OCP, Verapamil, Opioids
         ├── HYPOTHYROIDISM (TRH-driven) → Check TSH
         ├── RENAL FAILURE (Reduced Clearance)
         ├── PREGNANCY
         ├── NIPPLE STIMULATION / STRESS (Recent)
         └── MACROPROLACTINAEMIA
         │
         ▼
REPEAT PROLACTIN (Fasting, Non-stressed, Avoid Nipple Stimulation)
         │
         ├── NORMAL → Transient/Physiological
         │
         └── PERSISTENTLY ELEVATED
                 │
                 ▼
         PEG PRECIPITATION TEST → MACROPROLACTINAEMIA?
                 │
                 ├── YES → MACROPROLACTINAEMIA (Benign, No Treatment)
                 │
                 └── NO → TRUE HYPERPROLACTINAEMIA
                         │
                         ▼
                 MRI PITUITARY (3mm Slices, Gadolinium)
                         │
                         ├── NO TUMOUR + Prolactin <1000 → IDIOPATHIC / STALK EFFECT
                         ├── MICROPROLACTINOMA (<10mm) → CABERGOLINE
                         ├── MACROPROLACTINOMA (≥10mm) → CABERGOLINE / SURGERY
                         └── NO TUMOUR + Prolactin >1000 → STALK EFFECT / ECTOPIC (Rare)
```

## 3. Macroprolactinaemia

| Feature | Details |
|---------|---------|
| **Definition** | Prolactin Complexed with **IgG Autoantibody** (PRL-IgG Complex) |
| **Characteristics** | **High Molecular Weight** (>150 kDa); **Biologically Inactive** |
| **Prevalence** | 10-25% of Hyperprolactinaemia |
| **Diagnosis** | **PEG (Polyethylene Glycol) Precipitation** → **Recovery <60%** = Macroprolactinaemia |
| **Clinical Significance** | **Benign**; No Pituitary Tumour; **No Treatment Needed** |
| **Pitfall** | Can Coexist with True Hyperprolactinaemia → Measure Monomeric PRL Post-PEG |

## 3. Prolactinoma — Management

### Microprolactinoma (<10mm)
| Aspect | Protocol |
|--------|----------|
| **1st Line** | **Cabergoline** 0.25mg BD → Titrate Weekly by 0.25mg BD |
| **Target Dose** | 0.5-1mg BD (Max 2mg BD) |
| **Goal** | Prolactin Normalisation + Tumour Shrinkage (>50% Volume Reduction) |
| **Monitoring** | Prolactin q4-8w During Titration; MRI at 6-12mo if Macro Features |
| **Withdrawal** | After **2-3 Years** Normal Prolactin + **No Tumour on MRI** → Taper Over 3-6 Months |

### Macroprolactinoma (≥10mm)
| Aspect | Protocol |
|--------|----------|
| **1st Line** | **Cabergoline** 0.5mg BD → Titrate to 1-2mg BD (Max 3mg BD) |
| **Target** | Prolactin Normalisation + **>50% Volume Reduction** |
| **Surgery Indications** | Cabergoline Resistant/Intolerant; Apoplexy; Chiasmal Compression with Visual Loss Despite Medical Rx |
| **Visual Fields** | **Mandatory** (Bi-temporal Hemianopia = Urgent Decompression) |

### Cabergoline Dosing Schedule
| Week | Microprolactinoma | Macroprolactinoma |
|------|-------------------|-------------------|
| **1-2** | 0.25mg BD | 0.5mg BD |
| **3-4** | 0.5mg BD | 1mg BD |
| **5-8** | 1mg BD | 1.5-2mg BD |
| **Max** | 2mg BD | 3mg BD (Rarely >2mg) |

---

## 3. Cabergoline — Key Pharmacology

| Property | Details |
|--------|---------|
| **Mechanism** | **D2 Dopamine Receptor Agonist** → Inhibits Prolactin Secretion + Shrinks Tumour |
| **Half-Life** | **63-69 Hours** (Long; Allows BD/Weekly Dosing) |
| **Onset** | Prolactin Falls Within Hours; Tumour Shrinkage Over Weeks-Months |
| **Side Effects** | Nausea (40%), Postural Hypotension, Nasal Congestion, Impulse Control Disorders (Rare), Valvulopathy (High Dose, Chronic) |
| **Monitoring** | Echocardiogram If Dose >2mg/day or Duration >5 Years; Prolactin q3-6mo |

---

## 3. Withdrawal / Discontinuation Criteria

| Criteria | Required |
|----------|----------|
| **Duration** | **≥2-3 Years** on Stable Dose |
| **Biochemical** | Prolactin **Normalised** on Stable Dose for **≥2 Years** |
| **Radiological** | **No Tumour Visible on MRI** (or Microadenoma Resolved) |
| **Process** | Taper Cabergoline Over 3-6 Months; Recheck Prolactin q3mo; Restart if Recurrence |

## 4. Pregnancy & Prolactinoma

| Scenario | Microprolactinoma | Macroprolactinoma |
|----------|-------------------|-------------------|
| **Pre-Conception** | Stop Cabergoline if Prolactin Normal + MRI Stable | **Continue Cabergoline** (Growth Risk >10%) |
| **During Pregnancy** | Stop Cabergoline (Low Growth Risk) | **Continue Cabergoline** (Growth Risk >10%) |
| **Monitoring** | Prolactin q Trimester; Visual Fields q Trimester (If Macro) | Prolactin q Trimester; Visual Fields q Trimester; MRI if Symptoms |
| **Delivery** | Vaginal Preferred | C-Section if Chiasmal Compression Risk |
| **Postpartum** | Breastfeeding → Prolactin ↑; Cabergoline Suppresses Lactation | Breastfeeding → Cabergoline Suppresses Lactation |

**Key Point**: **Microprolactinoma** → Stop Cabergoline; **Macroprolactinoma** → **Continue Cabergoline** (Growth Risk >10%).

## 3. Stalk Effect / Disconnection Hyperprolactinaemia

| Feature | Details |
|---------|---------|
| **Mechanism** | Pituitary Stalk Compression → ↓ Dopamine Delivery to Lactotrophs → **Disinhibition** |
| **Prolactin Level** | Usually **<2000 mIU/L** (Rarely >3000) |
| **MRI** | **Non-Functioning Macroadenoma** Compressing Stalk / Suprasellar Mass |
| **Management** | Treat Underlying Mass (Surgery if Macroadenoma); Prolactin Normalises Post-Decompression |
| **Key Differentiator** | **Visual Field Defects** Common; Other Pituitary Deficits Often Coexist |

---

## 4. Drug-Induced Hyperprolactinaemia

| Drug Class | Typical Prolactin Level | Management |
|------------|------------------------|------------|
| **Antipsychotics** (Typical > Atypical) | Risperidone > Olanzapine > Quetiapine > Clozapine > **Aripiprazole (Low)** | Switch to Lower-Risk Agent (Aripiprazole); If Essential → Monitor |
| **Metoclopramide / Domperidone** | D2 Antagonism (GI Motility) | Switch if Possible; Reduce Dose |
| **Antidepressants** (SSRIs, TCAs) | Serotonergic Modulation | Switch if Possible |
| **Opioids** | μ-Receptor → ↓ Dopamine | Reduce Dose / Switch |
| **Verapamil / Alpha-Methyldopa** | Central D2 Effect | Switch if Possible |
| **OCP** | Oestrogen → Lactotroph Hyperplasia | Switch to POP / Non-Hormonal |

| **Cabergoline Stop** | After 2-3yr Normal PRL + MRI Resolution |
| **Pregnancy + Macro** | **Continue Cabergoline** (Growth Risk >10%) |

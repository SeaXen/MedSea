# Menopause & Hormone Replacement Therapy (HRT)

> [!info]
> **Menopause = Permanent Cessation of Menses for ≥12 Months** (Mean Age 51 Years). **Perimenopause = Transition Period** (Irregular Cycles, Vasomotor Symptoms). **HRT = Most Effective Treatment for Vasomotor Symptoms & Bone Protection.** **Individualised Risk-Benefit Assessment** is Key.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Diagnose menopause and perimenopause (clinical vs biochemical)
- [ ] Apply individualised HRT prescribing (indications, contraindications, route, duration)
- [ ] Differentiate oestrogen-only vs combined HRT regimens
- [ ] Counsel on risks/benefits (WHI, MWS data) and non-hormonal alternatives
- [ ] Manage special populations (premature menopause, breast cancer survivors, cardiovascular disease)

## 2. Definitions & Staging

| Stage | Definition | Typical Age |
|-------|------------|-------------|
| **Premenopause** | Regular Cycles, Normal Ovarian Function | <45 Years |
| **Perimenopause** | **Irregular Cycles** (Cycle Length Variability >7 Days) + Vasomotor Symptoms | **45-55 Years** (Mean 47) |
| **Menopause** | **12 Consecutive Months Amenorrhoea** (Retrospective Diagnosis) | **Mean 51 Years** (Range 45-55) |
| **Postmenopause** | >12 Months Post Final Menstrual Period | >51 Years |
| **Premature Menopause** | **<40 Years** | <40 Years (See POI Note) |
| **Early Menopause** | **40-45 Years** | 40-45 Years |
| **Surgical Menopause** | Bilateral Oophorectomy (Abrupt Hormone Loss) | Any Age |

### STRAW+10 Staging
| Stage | Criteria | Hormonal Profile |
|-------|-----------|------------------|
| **-3 to -2** | Reproductive | Regular Cycles, Normal FSH |
| **-1** | Early Perimenopause | Variable Cycle Length (>7 Days Variability), ↑ FSH |
| **0** | Late Perimenopause | ≥60 Days Amenorrhoea, ↑↑ FSH, ↓ Inhibin B |
| **+1a/b** | Early Postmenopause (1-3yr / 3-8yr) | ↑↑ FSH, ↓↓ Oestradiol |
| **+2** | Late Postmenopause | Stable ↑ FSH, Low Oestradiol |

## 2. Clinical Features

### Vasomotor Symptoms (VMS) — Hallmark
| Symptom | Frequency | Description |
|--------|-----------|-------------|
| **Hot Flushes** | **75-85%** | Sudden Heat, Flushing, Sweating, Palpitations |
| **Night Sweats** | 60-80% | Sleep Disruption, Fatigue |
| **Urogenital (GSM)** | **50-70%** | Vaginal Dryness, Dyspareunia, Urinary Frequency/UTI |
| **Psychological** | 40-60% | Mood Swings, Anxiety, Depression, Irritability, Brain Fog |
| **Sleep** | 40-60% | Insomnia, Night Sweats, Fragmented Sleep |
| **Musculoskeletal** | 50-60% | Joint Pain, Muscle Stiffness, Frozen Shoulder |
| **Sexual** | 30-50% | Low Libido, Dyspareunia (GSM) |

---

## 2. Diagnosis

| Method | Criteria |
|--------|----------|
| **Clinical** | **≥12 Months Amenorrhoea** (Age >45) = Clinical Diagnosis |
| **Biochemical (If <45 or Uncertain)** | **FSH >30 IU/L** on 2 Occasions (4-6 Weeks Apart) + Low Oestradiol |
| **Perimenopause** | Clinical (Irregular Cycles + VMS) — Biochemical Not Required |

> **Routine FSH Not Needed >45 Years** with Typical Presentation.

## 3. Hormone Replacement Therapy (HRT) — Indications & Contraindications

### Indications for HRT
| Indication | Strength |
|-----------|----------|
| **Vasomotor Symptoms** | **Primary Indication** (Gold Standard) |
| **Genitourinary Syndrome (GSM)** | Local Oestrogen if Systemic Contraindicated |
| **Bone Protection** | **If High Fracture Risk** (T-Score ≤-2.5, Prior Fracture) |
| **Premature Menopause (<45)** | **Strong Indication** (Until Age 50-51) |
| **Quality of Life** | Significant Symptom Burden Affecting Function |

### Absolute Contraindications
| Condition | Details |
|---------|---------|
| **Breast Cancer** (Current/Recent) | Absolute |
| **Endometrial Cancer** (Active) | Absolute |
| **VTE** (Active/History Unprovoked) | Absolute |
| **Stroke / TIA / MI** (Active/Recent) | Absolute |
| **Unexplained Vaginal Bleeding** | Investigate First |
| **Active Liver Disease** | Severe Impairment |
| **Porphyria Cutanea Tarda** | Rare |

### Relative Contraindications (Individualise)
| Condition | Consideration |
|-----------|---------------|
| **Age >60 / >10 Years Post-Menopause** | ↑ VTE/Stroke Risk (WHI Data) |
| **Controlled Hypertension** | Monitor BP; Transdermal Preferred |
| **Migraine with Aura** | Transdermal Preferred (↓ Stroke Risk) |
| **Gallbladder Disease** | Oral ↑ Risk; Transdermal Preferred |
| **Fibroids / Endometriosis** | May Worsen; Monitor |
| **Family History Breast Cancer** | Individualised; Transdermal Preferred |

## 3. HRT Regimens — Types & Routes

### Oestrogen-Only (Hysterectomised Women)
| Route | Dose | Advantages |
|--------|------|------------|
| **Oral** | Oestradiol 1-2mg Daily | Simple, Cheap |
| **Transdermal Patch** | 25-100µg/24h (Weekly/Biweekly) | **↓ VTE Risk**, Steady Levels |
| **Gel/Spray** | 0.5-1.5mg/day | Flexible Dosing, Low VTE Risk |
| **Implant** | 50-100mg (6-Monthly) | Long-Lasting, No Daily Adherence |
| **Vaginal (Local Only)** | Pessary/Ring/Cream | **GSM Only** (No Systemic Effect) |

### Combined HRT (Oestrogen + Progestogen) — Intact Uterus
| Regimen | Oestrogen | Progestogen | Schedule |
|---------|-----------|-------------|----------|
| **Sequential (Cyclical)** | Continuous | **10-14 Days/Month** | Withdrawal Bleed |
| **Continuous Combined** | Continuous | Continuous | **No Bleed** (After 1yr Post-Menopause) |
| **Long-Cycle** | Continuous | Quarterly (3-Monthly) | Reduced Bleeding Frequency |

### Progestogen Options (Intact Uterus)
| Agent | Dose (Sequential) | Dose (Continuous) | Notes |
|------|------------------|-------------------|-------|
| **Medroxyprogesterone Acetate (MPA)** | 10mg Days 1-12 | 2.5-5mg Daily | Cheap; Androgenic |
| **Dydrogesterone** | 10mg Days 1-12 | 5-10mg Daily | Neutral Metabolic Profile |
| **Micronised Progesterone** | 200mg Days 15-26 | 100mg Daily | **Best Safety Profile** (↓ Breast Cancer Risk) |
| **Levonorgestrel IUS (52mg)** | — | **20µg/day (Intrauterine)** | **Best Endometrial Protection**; ↓ Systemic SE |

## 3. Tibolone (Alternative)

| Feature | Details |
|---------|---------|
| **Mechanism** | **SERM + Weak Oestrogenic/Androgenic/Progestogenic** Activity |
| **Dose** | **2.5mg OD** |
| **Indications** | Vasomotor Symptoms, Osteoporosis Prevention, Libido |
| **Contraindications** | Breast Cancer, VTE, Stroke, Undiagnosed Vaginal Bleeding |
| **Advantage** | No Withdrawal Bleed; ↑ Libido |
| **Caution** | ↑ Stroke Risk (>60y), ↑ Breast Cancer Risk (Long-Term) |

## 4. Non-Hormonal Alternatives

| Agent | Indication | Dose | Evidence |
|-------|----------|------|----------|
| **CBT** | VMS, Mood, Sleep | 6-8 Sessions | Strong (NICE 1st Line) |
| **SSRIs/SNRIs** (Paroxetine, Venlafaxine, Escitalopram) | VMS | Low Dose | Moderate |
| **Gabapentin** | 300-900mg/day | VMS (Breast Cancer Survivors) | Moderate |
| **Clonidine** | 50-75µg BD | VMS | Weak |
| **Oxybutynin** | 5mg BD | VMS (Off-Label) | Weak |
| **Fezolinetant (NK3 Antagonist)** | 45mg OD | New (2023); VMS | Strong (New) |
| **Phytoestrogens** (Soy, Red Clover) | Food/Supplements | Variable | Weak/Moderate |
| **Black Cohosh** | Herbal | Variable | Weak/Inconsistent |

## 4. HRT — Risks vs Benefits (WHI / MWS Evidence)

| Outcome | Combined HRT (oestrogen + progestogen) | Oestrogen-Only (Hysterectomy) |
|-----------|----------------------------------------|-------------------------------|
| **Cardiovascular** | ↑ CHD (Small ↑ if >10yr Post-Menopause); **No ↑ if <10yr** | Neutral / ↓ CHD if <10yr Post-Menopause |
| **Stroke** | **↑ Risk** (All Ages) | **↑ Risk** (All Ages) |
| **VTE** | **↑↑ Risk** (2-3x) | **↑ Risk** (1.5x) |
| **Breast Cancer** | **↑ Risk** (Combined > Oestrogen-Only) | **↑ Risk** (Long-Term >10yr) |
| **Endometrial Cancer** | **Protected** (Progestogen) | **↑↑ Risk** (If No Progestogen) |
| **Ovarian Cancer** | Slight ↑ | Slight ↑ |
| **Colorectal Cancer** | **↓ Risk** | **↓ Risk** |
| **Hip Fracture** | **↓↓ Risk** | **↓↓ Risk** |
| **Mortality (All-Cause)** | Neutral / Slight ↓ (If Started <60) | Neutral / Slight ↓ |

> **Key**: **"Window of Opportunity"** — **Start HRT <10yr Post-Menopause / Age <60** → Net Benefit (↓ CHD, ↓ Mortality). **>10yr Post-Menopause / Age >60** → Net Harm (↑ CHD, Stroke, VTE).

## 4. Duration & Monitoring

| Aspect | Recommendation |
|--------|----------------|
| **Duration** | **Shortest Effective Duration**; Individualised; **Annual Review** |
| **Typical Duration** | **2-5 Years** for VMS; Longer if Osteoporosis Prevention |
| **Review** | **Annual** (Symptoms, Bleeding, Breast Exam, BP, BP, Weight, Breast Exam) |
| **Stopping** | **Gradual Taper** (3-6 Months) to Avoid Rebound Symptoms |
| **Vaginal Oestrogen** | **Long-Term Safe** (Local Only) — Continue Indefinitely for GSM |

### Monitoring on HRT
| Parameter | Frequency |
|-----------|-----------|
| **Blood Pressure** | Every Visit |
| **Breast Exam** | Annual (+ Mammogram per National Screening) |
| **Blood Pressure** | Each Visit |
| **Weight / BMI** | Annual |
| **Lipids / Glucose** | Annual |
| **LFTs** | If Oral HRT / Baseline |
| **Mammogram** | Per National Screening Programme |
| **Bone Density (DEXA)** | If Indicated (Baseline, then q2-3yr) |

## 5. Special Populations

### Premature Menopause (<40) / POI
| Aspect | Recommendation |
|--------|----------------|
| **HRT Indication** | **Strong** — Until Age 50-51 (Natural Menopause Age) |
| **Dose** | Higher Oestrogen Dose (Estradiol 100-200µg/day Patch) |
| **Bone Protection** | Critical (Peak Bone Mass Not Achieved) |
| **Fertility** | Donor Oocyte IVF (If Desired); HRT Not Contraceptive |

### Breast Cancer Survivors
| Aspect | Recommendation |
|--------|----------------|
| **Systemic HRT** | **Contraindicated** (ER+ Tumours) |
| **Vaginal Oestrogen** | **Low-Dose** (Pessary/Ring/Cream) — **Safe** (Minimal Absorption) |
| **Non-Hormonal** | CBT, SSRIs, Gabapentin, Fezolinetant, Vaginal Moisturisers/Lubricants |

### Cardiovascular Disease
| Scenario | Recommendation |
|--------|----------------|
| **Established CVD** | **Transdermal Oestrogen** Preferred; Avoid Oral |
| **Stroke/TIA** | Avoid Systemic HRT; **Local Vaginal Only** |
| **Hypertension** | Control BP First; Transdermal Preferred |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **HRT vs ERT** | HRT = Oestrogen + Progestogen (Intact Uterus); ERT = Oestrogen Only (Hysterectomy) |
| **Sequential vs Continuous** | Sequential = Cyclical Progestogen (Withdrawal Bleed); Continuous = No Bleed (Post-Menopause >1yr) |
| **Transdermal vs Oral** | Transdermal = ↓ VTE, ↓ Stroke, ↓ Gallbladder, ↓ Triglycerides |
| **Tibolone vs HRT** | Tibolone = SERM + Weak Oestrogen/Androgen/Progestogen; No Bleed; ↑ Stroke >60 |
| **Window of Opportunity** | **<10yr Post-Menopause / Age <60** → Net CVD Benefit; **>10yr / >60** → Net Harm |
| **Vaginal Oestrogen in Breast Cancer** | **Safe** (Low Systemic Absorption); Systemic HRT Contraindicated |
| **Progestogen Types** | Micronised Progesterone = Best Breast Safety; LNG-IUS = Best Endometrial Protection |
| **Duration** | Shortest Effective; Annual Review; Taper Gradually (3-6mo) |
| **Local Oestrogen** | Safe Long-Term; No Progestogen Needed; First-Line for GSM |

# Toxic Nodular Goitre & Toxic Adenoma

> [!info]
> **Toxic Multinodular Goitre (TMNG) = Multiple Autonomous Nodules causing Hyperthyroidism. 2nd Commonest Cause (10-15%). Toxic Adenoma = Single Autonomous "Hot" Nodule. 3rd Commonest (3-5%). Both: Autonomous Function → TSH Suppressed, fT4/fT3 ↑, TSH Suppressed, RAI Uptake Focal/Multifocal.**

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Differentiate TMNG, Toxic Adenoma, and Graves on clinical, biochemical, and imaging grounds
- [ ] Apply investigation algorithm (TSH, fT4, RAI Uptake, US)
- [ ] Compare management strategies (RAI vs Surgery vs ATDs)
- [ ] Recognise specific complications (Atrial Fibrillation in Elderly, Malignancy Risk)
- [ ] Manage in Special Populations (Elderly, Pregnancy, Cardiac Disease)

## 2. Toxic Multinodular Goitre (TMNG)

### Epidemiology & Risk Factors
| Feature | Details |
|--------|---------|
| **Prevalence** | **2nd Commonest** Hyperthyroidism (10-15%) |
| **Age** | **>50 Years** (Peak 60-70y) |
| **Sex** | **Female > Male** (2:1) |
| **Risk Factors** | Long-standing Multinodular Goitre, Iodine Deficiency → Sufficiency, Ageing, Smoking |
| **Pathogenesis** | **Autonomous Nodules** (TSH-R Mutations, Gsα Mutations) → TSH-Independent Growth + Hormone Secretion |

### Clinical Presentation
| Feature | TMNG vs Graves |
|---------|----------------|
| **Age** | **>50 Years** (vs 30-50 in Graves) |
| **Goitre** | **Long-standing, Multinodular, Large** | Diffuse, Smaller |
| **Symptoms** | Often Subclinical/Mild; **Atrial Fibrillation Common** (15-20%) | Classic Thyrotoxicosis |
| **Eye Signs** | **Absent** (No Ophthalmopathy) | **Present (25-50%)** |
| **Dermopathy/Acropachy** | **Absent** | **Present (Graves Only)** |
| **Tracheal Compression** | Common (Large Goitre) | Rare |

## 2. Investigation — TMNG vs Graves

| Investigation | **TMNG** | **Graves** | **Toxic Adenoma** |
|--------------|----------|------------|-------------------|
| **TSH** | **Suppressed** | Suppressed | Suppressed |
| **fT4 / fT3** | ↑ (Often Mild) | ↑↑ | ↑ (Often Mild) |
| **TRAb** | **Negative** | **Positive (>95%)** | Negative |
| **RAI Uptake** | **Multifocal / Patchy** ("Salt & Pepper") | **Diffuse Homogeneous ↑** | **Focal "Hot" Nodule** |
| **Thyroid US** | **Multiple Nodules** (Variable Echogenicity) | Diffuse Enlargement, Hypervascular | **Single "Hot" Nodule** |
| **TRAb** | **Negative** | **Positive (>95%)** | Negative |
| **TgAb / TPOAb** | Usually Negative | Positive (80-90%) | Negative |
| **RAI Scan Appearance** | **"Salt & Pepper" / Multifocal** | **Diffuse Homogeneous** | **Single "Hot" Spot** |

## 2. Toxic Adenoma (Plummer's Disease)

### Key Features
| Feature | Details |
|---------|---------|
| **Definition** | **Single Autonomous "Hot" Nodule** (TSH-Independent) |
| **Prevalence** | **3-5%** of Hyperthyroidism |
| **Age** | **Younger** (30-50y) vs TMNG |
| **Sex** | Female > Male (3:1) |
| **Pathogenesis** | **Somatic TSH-R Mutation** (Constitutive Activation) / **Gsα Mutation** |
| **Presentation** | Solitary Nodule; Often Asymptomatic/Mild Thyrotoxicosis |
| **Malignancy Risk** | **Low (<1%)** (vs 5-10% in Cold Nodules) |

---

## 3. Investigation — Differential Diagnosis

### RAI Uptake Scan — Key Differentiator
| Pattern | Diagnosis |
|---------|-----------|
| **Diffuse Homogeneous ↑** | **Graves Disease** |
| **Multifocal / "Salt & Pepper"** | **Toxic Multinodular Goitre** |
| **Focal "Hot" Nodule** (Rest of Thyroid Suppressed) | **Toxic Adenoma** |
| **Low / Absent** | Thyroiditis / Factitious / Iatrogenic |

### Thyroid Ultrasound
| Feature | TMNG | Toxic Adenoma | Graves |
|---------|------|---------------|--------|
| **Nodules** | **Multiple** (Variable Echo) | **Single Dominant "Hot" Nodule** | Diffuse Enlargement |
| **Vascularity** | Variable (Nodule-Dependent) | **Marked Intranodular** | Diffuse Hypervascular |
| **Background** | Multinodular | Normal/Suppressed Parenchyma | Diffuse Heterogeneous |

---

## 3. Management — TMNG

### Principles
| Principle | Detail |
|---------|--------|
| **ATDs Not Curative** | Nodules Autonomous → Relapse Universal on Stopping |
| **Definitive Therapy Required** | **RAI (1st Line) OR Surgery** |
| **ATDs Role** | Bridge to Definitive Rx / Elderly Frail / Pre-op Preparation |

### Management Algorithm
```
TMNG Diagnosed (TSH Suppressed, fT4/fT3 ↑, TRAb Negative, Multifocal RAI Uptake)
         │
         ├── ELDERLY / FRAIL / COMORBIDITIES
         │       → **RAI (¹³¹I)** — **1st Line Definitive**
         │       Dose: 400-800 MBq (Higher than Graves due to Lower Uptake)
         │
         ├── LARGE GOITRE (>80g) / COMPRESSIVE SYMPTOMS
         │       → **SURGERY (Total Thyroidectomy)** — Preferred
         │       Pre-op: ATDs → Euthyroid → Lugol's Iodine 5-7 Days
         │
         ├── YOUNGER PATIENT / PATIENT PREFERENCE
         │       → **RAI** (Definitive, Non-Invasive) OR **Surgery**
         │
         └── ATRIAL FIBRILLATION / CARDIAC DISEASE
                 → **RAI Preferred** (Lower Perioperative Risk)
                 Rate Control (β-Blockers / Digoxin) + Anticoagulation (CHA₂DS₂-VASc)
```

### RAI in TMNG
| Parameter | Details |
|---------|---------|
| **Dose** | **400-800 MBq** (Higher than Graves due to Lower Uptake Fraction) |
| **Efficacy** | **Cure Rate 80-90%** (May Need Repeat Dose) |
| **Hypothyroidism** | **>90% at 2-3 Years** (Higher than Graves) |
| **Pre-treatment** | Stop ATDs 3-5 Days; Low Iodine Diet 2 Weeks |
| **Post-RAI** | Monitor TSH/fT4 q4-6wk; Levothyroxine when Hypothyroid |

---

## 3. Management — Toxic Adenoma

### Management Algorithm
```
Toxic Adenoma (Single Hot Nodule, TSH Suppressed, fT4/fT3 ↑, TRAb Negative)
         │
         ├── **SURGERY (HEMITHYROIDECTOMY)** — **CURATIVE, 1st Line**
         │       Pre-op: ATDs → Euthyroid → Lugol's Iodine 5-7 Days
         │       Complications: RLN Injury (<1%), Hypocalcaemia, Hypothyroidism (Rare)
         │
         ├── **RAI (¹³¹I)** — Alternative if Surgery Contraindicated
         │       Dose: 400-600 MBq (Targeted to Nodule)
         │       Efficacy: 80-90% Cure; Hypothyroidism >90% at 2-3 Years
         │
         └── ATDS (Carbimazole) — Bridge Only / Frail Elderly
                 Not Curative; Relapse Universal on Stopping
```

### Comparison: TMNG vs Toxic Adenoma Management
| Aspect | **TMNG** | **Toxic Adenoma** |
|------|--------|-------------------|
| **1st Line Definitive** | **RAI** (Elderly) / Surgery (Large/Compressive) | **Surgery (Hemithyroidectomy)** |
| **ATDs Role** | Bridge / Pre-op | Bridge Only |
| **Surgery Type** | **Total Thyroidectomy** | **Hemithyroidectomy** |
| **RAI Dose** | Higher (400-800 MBq) | Standard (400-600 MBq) |
| **Malignancy Risk** | Low (Dominant Nodule <1%) | **Very Low (<0.5%)** |

---

## 4. Special Considerations in TMNG

### Atrial Fibrillation (Common in Elderly TMNG)
| Aspect | Management |
|---------|----------|
| **Prevalence** | **15-25%** of TMNG (vs 10-15% Graves) |
| **Mechanism** | Thyroid Hormone → ↑ β-Receptor Sensitivity + Electrical Remodelling |
| **Management** | **Rate Control** (β-Blocker 1st Line) + **Anticoagulation** (CHA₂DS₂-VASc) |
| **Thyroid Control** | **RAI Definitive** → Restores Sinus Rhythm in 60-70% |

### Malignancy Risk in TMNG
| Aspect | Details |
|--------|---------|
| **Dominant Nodule** | **<1% Malignancy** (If Functional/Hot) |
| **Cold Nodules in MNG** | **5-10% Malignancy** — Require FNA (Bethesda) |
| **RAI Uptake Cold Nodule** | FNA Mandatory |

### Pregnancy in TMNG
| Aspect | Management |
|--------|------------|
| **Rare** (Age >50) | If Occurs: **PTU 1st Trimester** → Switch Carbimazole 2nd/3rd |
| **RAI Contraindicated** | Surgery if Definitive Rx Needed |

## 5. Follow-up & Surveillance (Post-RAI / Surgery)

| Timeline | Investigations |
|----------|---------------|
| **4-6 Weeks** | TSH, fT4, fT3 |
| **3 Months** | TSH, fT4, fT3 |
| **6 Months** | TSH, fT4, fT3 |
| **12 Months** | TSH, fT4, fT3 |
| **Annually** | TSH (Lifelong if Post-RAI/Total Thyroidectomy) |

| Complication | Post-RAI | Post-Surgery |
|--------------|----------|--------------|
| **Hypothyroidism** | **>90% at 2-3 Years** (TMNG) | **10-20%** (Hemithyroidectomy) |
| **Recurrence** | <5% (May Need Repeat RAI) | <2% (Hemithyroidectomy) |
| **Malignancy** | No Increase | No Increase |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **TMNG vs Graves** | TMNG: Multifocal RAI, No TRAb, No Eye Signs, Elderly, AF Common |
| **Toxic Adenoma vs Graves** | Toxic Adenoma = Single Hot Nodule; Graves = Diffuse Uptake |
| **TMNG vs Simple MNG** | TMNG = Biochemical Hyperthyroidism; Simple MNG = Euthyroid |
| **RAI in TMNG** | Higher Dose (400-800 MBq) vs Graves (400-600 MBq) |
| **Toxic Adenoma Surgery** | Hemithyroidectomy (Curative) vs Total Thyroidectomy (TMNG) |
| **Malignancy** | Hot Nodule <1%; Cold Nodule in MNG = 5-10% Risk |
| **RAI Dose** | TMNG: 400-800 MBq; Toxic Adenoma: 400-600 MBq |

**Mnemonic: TMNG vs TOXIC ADENOMA**
- **T**MN = **T**oo **M**any **N**odules (Multifocal)
- **T**oxic **A**denoma = **A**lone **N**odule (Single)
- **R**AI: **T**MN = Multifocal "Salt & Pepper"; **A**denoma = **F**ocal Hot Spot
- **S**urgery: **T**MN = Total Thyroidectomy; **A**denoma = Hemi = **H**alf
- **R**ecurrence: **T**MN after RAI <5%; **A**denoma after Hemi <2%

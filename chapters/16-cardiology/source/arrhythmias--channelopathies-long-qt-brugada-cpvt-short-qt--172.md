# Channelopathies

## 1. Long QT Syndrome (LQTS)
- **LQT1** (KCNQ1, swimming triggers, exercise — especially swimming)
- **LQT2** (KCNH2, K+, auditory triggers, emotion, postpartum)
- **LQT3** (SCN5A, Na+, sleep/rest)
- **Diagnosis**: QTc ≥480 ms (definite), or Schwartz score
- **Rx**: beta-blocker (propranolol, nadolol), avoid QT-prolonging drugs, ICD if high risk, LCSD for refractory
- LQT1: avoid competitive swimming

## 2. Brugada Syndrome
- **SCN5A** (Na+), autosomal dominant
- Type 1 Brugada pattern: coved ST ↑ ≥2 mm in V1-V2 with negative T
- Triggers: fever, sodium channel blockers, vagal tone, sleep
- **Risk stratification**: spontaneous type 1, syncope, VF, family history
- **Rx**: ICD if symptomatic/high risk; quinidine for VT storm; **fever control critical**
- Avoid Class IC antiarrhythmics, cocaine, alcohol binge

## 3. CPVT (Catecholaminergic Polymorphic VT)
- **RyR2** (autosomal dominant) or **CASQ2** (recessive)
- Exercise/emotion-induced bidirectional VT
- Normal resting ECG
- **Rx**: nadolol (max dose) + flecainide; ICD if breakthrough; LCSD

## 4. Short QT Syndrome
- Rare, very short QTc (<330 ms)
- AF, VF risk
- **Rx**: ICD, quinidine

> **Pearls** — All channelopathies: **avoid triggers, treat with beta-blocker (nadolol often), consider ICD** based on risk. **Brugada + fever = code!** Cool aggressively. CPVT can present as SCD in young athletes.

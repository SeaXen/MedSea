## 1. Introduction
Cardiac involvement in systemic disease is often **subclinical but lethal**. Three broad groups: **(1) Connective tissue disease (CTD)**, **(2) Infiltrative / metabolic**, **(3) Endocrine / nutritional**. Pattern recognition of cardiac features (pericarditis, LVH, restrictive CM, conduction block) narrows the differential.

### Cardinal Cardiac Syndromes
- **Pericarditis** → SLE, RA, scleroderma, uraemia
- **Libman-Sacks endocarditis** → SLE, antiphospholipid syndrome
- **Myocardial fibrosis / restrictive CM** → amyloidosis, sarcoidosis, scleroderma
- **LVH with low-voltage ECG** → cardiac amyloidosis (the "voltage–mass mismatch")
- **Biventricular iron-overload CM** → haemochromatosis
- **High-output cardiac failure** → thiamine deficiency (beriberi), severe anaemia, thyrotoxicosis
- **Conduction block in young patient** → sarcoidosis, Lyme, congenital

## 2. Pathophysiology

### CTD Mechanisms
- **Immune complex deposition** (SLE, RA) → pericarditis, myocarditis
- **Microvascular spasm & fibrosis** (scleroderma) → myocardial ischaemia, PAH
- **Valvular verrucae** (Libman-Sacks) → both surfaces, sterile, embolic risk

### Infiltrative
- **Amyloid** = misfolded protein (ATTR transthyretin, AL light-chain) → extracellular deposition → restrictive CM, low voltage
- **Sarcoid** = non-caseating granulomas → basal septum → AV block, VT
- **Iron overload** → hydroxyl radical damage → systolic + diastolic dysfunction
- **Fabry** = α-galactosidase A deficiency → globotriaosylceramide accumulation → LVH, stroke, CKD

### Endocrine / Nutritional
- **Thyrotoxicosis** → ↑β-receptor density, ↑HR, AF, high-output failure
- **Acromegaly** → GH/IGF-1 → concentric LVH, valvular thickening
- **Thiamine (B1)** deficiency → impaired ATP generation → high-output failure with **vasodilation**

## 3. Classification

### Connective Tissue Disease
| Disease | Cardiac Hallmark |
|---|---|
| **SLE** | Pericarditis (most common), Libman-Sacks, accelerated CAD, myocarditis |
| **Antiphospholipid syndrome** | Valvular thickening, thrombosis, recurrent miscarriage |
| **Systemic sclerosis** | Myocardial fibrosis, PAH (leading COD), conduction disease |
| **Rheumatoid arthritis** | Pericarditis, nodules, vasculitis, accelerated CAD |
| **MCTD** | Puffy hands, PAH, overlap features |
| **Sjögren** | Pericarditis, vasculitis, ↑lymphoma risk |

### Infiltrative / Metabolic
| Disease | Hallmark |
|---|---|
| **Cardiac amyloidosis (ATTR / AL)** | Restrictive CM, low-voltage ECG, LGE global subendocardial |
| **Cardiac sarcoidosis** | Basal septal LGE, AV block, VT, FDG-PET +ve |
| **Haemochromatosis** | Biventricular CM, ferritin >1000, TSAT >45% |
| **Fabry disease** | LVH, stroke, CKD, α-Gal A deficiency |

### Endocrine / Nutritional
| Condition | Cardiac Effect |
|---|---|
| **DM** | Accelerated CAD, diastolic dysfunction, autonomic neuropathy, silent ischaemia |
| **Hyperthyroidism** | AF, high-output HF, thyrotoxic CM |
| **Hypothyroidism** | Pericardial effusion, bradycardia, diastolic dysfunction |
| **Cushing** | HTN, metabolic syndrome, LVH |
| **Acromegaly** | Biventricular hypertrophy, valvular thickening, HTN |
| **Thiamine (B1) deficiency** | High-output failure (wet beriberi) |

## 4. Clinical Features — Pattern Clues

### SLE / APS
- Young female, malar rash, arthritis, serositis
- **Pericardial rub** (SLE), **sterile verrucous vegetations** (Libman-Sacks → both valve surfaces)
- Recurrent arterial / venous thrombosis, miscarriages, livedo reticularis (APS)

### Scleroderma
- Skin thickening, Raynaud, sclerodactyly
- **PAH** (dyspnoea, loud P2, RV heave) — leading COD
- Renal crisis, interstitial lung disease

### Amyloidosis
- HFpEF + **low-voltage ECG + thick LV walls** (the classic mismatch)
- Bilateral carpal tunnel, macroglossia, periorbital purpura (AL)
- ATTRwt: elderly, aortic stenosis + amyloidosis overlap

### Sarcoidosis
- Young / middle-aged, Black ethnicity, lung + skin
- **Syncope, AV block in <50 y**, ventricular arrhythmia
- Bilateral hilar lymphadenopathy, ↑ACE, hypercalcaemia

### Haemochromatosis
- "Bronze diabetes" — skin pigmentation, DM, liver disease
- Restrictive → dilated CM, conduction disease

## 5. Investigations

### First-line
- **ECG** (low voltage → amyloid; AV block → sarcoid; LVH → acromegaly / Fabry)
- **Echocardiogram** (LV wall thickness, diastolic pattern, EF, pericardial effusion)
- **Troponin + NT-proBNP** (AL amyloid — prognostic)
- **CXR** (cardiomegaly, hilar nodes in sarcoid)

### Second-line / Disease-specific
| Disease | Key Test |
|---|---|
| **Cardiac amyloid** | Serum / urine **immunofixation + free light chains (FLC)** (AL); **99mTc-PYP scan** (ATTR) + FLC negative; **CMR** LGE global subendocardial; fat / endomyocardial biopsy |
| **Cardiac sarcoid** | **CMR** (basal septum LGE) + **FDG-PET** (active inflammation); ↑ACE; biopsy (non-caseating granuloma) |
| **Haemochromatosis** | **Ferritin** (>200 µg/L women, >300 men), **TSAT >45%**, **HFE C282Y**; MRI **T2\* <20 ms** for cardiac iron |
| **Fabry** | α-Gal A activity, **GLA** gene, lyso-Gb3 |
| **SLE / CTD** | ANA, **anti-dsDNA**, **anti-Sm**, **anti-SSA/Ro, SSB/La**, **anti-Scl-70**, **anti-centromere**, **anti-U1-RNP**; **C3 / C4** ↓ |
| **APS** | **Lupus anticoagulant, anti-cardiolipin, anti-β2-GPI** (persist ≥12 wk) |
| **Hyperthyroid** | **↓TSH, ↑FT4 / T3**, TPO antibodies |
| **Acromegaly** | **↑IGF-1**, GH not suppressed on OGTT, pituitary MRI |
| **Cushing** | **24-h urinary free cortisol**, **1 mg dexamethasone suppression**, midnight salivary cortisol |
| **Thiamine** | Whole-blood thiamine, lactate ↑ |
| **CKD** | Urea, creatinine, eGFR, urinalysis |

## 6. Differential Diagnosis (Restrictive / Infiltrative CM)

| Condition | Echo / ECG | Key Diagnostic Clue |
|---|---|---|
| **Cardiac amyloidosis (AL)** | Thick walls, low voltage, speckled | ↑FLC ratio, κ/λ, PYP negative |
| **Cardiac amyloidosis (ATTRwt)** | Thick walls, low voltage, AS overlap | PYP grade 2–3 + FLC negative |
| **Cardiac sarcoidosis** | Basal septal thinning, aneurysm | CMR LGE + FDG-PET +ve, ↑ACE |
| **Haemochromatosis** | Dilated / restrictive, mid-cavity obliteration | Ferritin >1000, HFE C282Y, T2\* <20 ms |
| **Fabry disease** | Concentric LVH, mild AV thickening | α-Gal A ↓, GLA gene + |
| **Endomyocardial fibrosis** | Apical obliteration, bright endocardium | Tropical, eosinophilia |
| **Hypertrophic CM** | Asymmetric septal hypertrophy | Sarcomere mutation, family history |
| **Constrictive pericarditis** | Septal bounce, resp variation | Pericardial thickening on CT/CMR |

## 7. Complications
- **Heart failure** (HFrEF or HFpEF) — most common cardiac complication
- **Conduction disease** — complete AV block (sarcoid, amyloidosis)
- **Ventricular arrhythmia / SCD** — sarcoid, amyloidosis, ICM
- **Pulmonary hypertension** — scleroderma (leading COD)
- **Valvular disease** — Libman-Sacks, RA nodules, endocarditis (SLE/APS)
- **Accelerated atherosclerosis** — SLE, RA, DM, CKD
- **Pericardial effusion → tamponade** — SLE, RA, uraemia, myxoedema

## 8. Management

### Disease-Specific Therapy
| Disease | Specific Therapy |
|---|---|
| **AL amyloidosis** | **Chemotherapy + autologous stem cell transplant**; supportive diuretics; **avoid digoxin & CCB** |
| **ATTR amyloidosis** | **Tafamidis** (TTR stabiliser); patisiran / vutrisiran (RNAi); supportive |
| **Cardiac sarcoidosis** | **Prednisolone 0.5–1 mg/kg** ± MMF / methotrexate; **ICD** for VT / EF <35% |
| **Haemochromatosis** | **Venesection** (target ferritin 50–100); **deferasirox** if anaemic |
| **Fabry disease** | **Enzyme replacement** (agalsidase α/β) or oral chaperone **migalastat** |
| **SLE pericarditis** | NSAIDs → **colchicine** → **hydroxychloroquine** → steroids; pericardiocentesis if tamponade |
| **Scleroderma PAH** | ERA (bosentan, ambrisentan), **PDE5i** (sildenafil / tadalafil), **riociguat**, **selexipag**; screen with echo + NT-proBNP yearly |
| **APS with thrombosis** | **Lifelong warfarin INR 2.5–3.5**; LMWH in pregnancy |
| **Hyperthyroid** | **Carbimazole** / PTU, β-blocker; definitive: **radio-iodine** or thyroidectomy |
| **Hypothyroid** | **Levothyroxine** replacement (start low, go slow if IHD) |
| **Cushing** | Transsphenoidal surgery; **ketoconazole / metyrapone** if not surgical |
| **Acromegaly** | Transsphenoidal surgery; **somatostatin analogues** (octreotide, lanreotide); **pegvisomant** |
| **Thiamine deficiency (wet beriberi)** | **IV thiamine 100 mg** → then oral; supportive HF therapy |
| **Diabetic CM** | **SGLT2 inhibitor** + **GLP-1 RA** (CV mortality benefit), ACEi, statin |

### Supportive Cardiac Care
- HF: **diuretics, ACEi/ARB, β-blocker, MRA, SGLT2i** (GDMT — but avoid β-blocker / digoxin / CCB in amyloid)
- Anticoagulation as indicated (AF, APS, mechanical valve)
- **Vaccinations** (influenza, pneumococcal)

## 9. Secondary Prevention & Long-term
- **Treat underlying disease** aggressively — most cardiac features regress
- **Lifelong immunosuppression** in many CTDs (azathioprine, MMF, biologics)
- **Surveillance** echo + ECG every 6–12 months in amyloid, sarcoid, haemochromatosis
- **Family screening** for hereditary syndromes (ATTR, Fabry, HFE, Marfan)
- **Lifestyle:** Mediterranean diet, weight loss, BP / glycaemic / lipid control
- **Avoid** cardiotoxic drugs where possible; review drug interactions
- **Genetic counselling** in inherited forms
- **Palliative care** referral in advanced cardiac amyloid

---

## 10. Exam Pearls & High-Yield

- **Libman-Sacks endocarditis** = **sterile verrucae on both valve surfaces**, in SLE / APS
- **Antiphospholipid syndrome triad:** thrombosis + pregnancy morbidity + persistent aPL antibodies
- **Scleroderma → PAH** is the **leading cause of death**; screen annually with echo + NT-proBNP
- **MCTD** = anti-U1-RNP **high titre** + puffy hands + Raynaud + overlap features
- **Cardiac amyloid = low voltage + thick walls** on echo (voltage–mass mismatch)
- **ATTR** → **PYP scan positive, FLC negative**; **AL** → **PYP negative, FLC positive** (κ/λ abnormal ratio)
- **Cardiac sarcoid:** basal septal LGE on CMR + FDG-PET uptake; **prednisolone + ICD**
- **Haemochromatosis:** ferritin >1000, TSAT >45%, **HFE C282Y**; **venesection** is treatment of choice
- **Fabry disease:** α-galactosidase A deficiency, **GLA gene**; **agalsidase ERT** or **migalastat**
- **DM cardiac:** SGLT2i + GLP-1 RA reduce CV mortality (EMPA-REG, LEADER, DAPA-HF)
- **Hyperthyroid + AF:** control rate with β-blocker; treat hyperthyroidism first — many revert to sinus rhythm
- **Wet beriberi** = thiamine deficiency + high-output failure + warm extremities — give **thiamine BEFORE glucose**

<callout icon="🎯" color="rose_bg">
**CLINICAL SIGNIFICANCE —** Suspect infiltrative disease in any patient with **HFpEF + thick LV walls + low ECG voltage** (amyloid), **AV block in a young adult** (sarcoid), or **bronze skin + DM + liver disease + CM** (haemochromatosis). Tissue diagnosis + targeted therapy can transform prognosis; **avoid conventional HF drugs in amyloid (digoxin, CCB, β-blocker)**.

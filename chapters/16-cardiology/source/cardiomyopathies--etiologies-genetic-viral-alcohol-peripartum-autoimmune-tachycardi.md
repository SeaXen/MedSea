# Dilated Cardiomyopathy (DCM) — Etiologies

## 1. Definition
DCM is a **non-ischaemic cardiomyopathy** with **LV dilatation and systolic dysfunction** (LVEF <40–45%) unexplained by coronary disease, valve disease, or hypertension alone. It is the **most common referral for transplant** in adults <50.

## 2. Categorisation (Classic "Ischaemic vs Non-Ischaemic")
- **Ischaemic cardiomyopathy** — most common cause of LV dysfunction (NOT strictly DCM)
- **Non-ischaemic DCM** — many aetiologies, listed below

## 3. Etiologic Breakdown
| Category | Examples |
|---|---|
| **Genetic** | TTN truncating variants (~25%), LMNA, FLNC, PLN, RBM20, SCN5a |
| **Viral / post-infectious** | Coxsackie B, parvovirus B19, HHV-6, SARS-CoV-2 |
| **Toxic — alcohol** | >80 g/day for >5 yrs; 5–10% of DCM |
| **Peripartum** | Last month pregnancy → 5 months postpartum |
| **Autoimmune / inflammatory** | Giant cell myocarditis, sarcoid, eosinophilic |
| **Tachycardia-induced** | AF/flutter, SVT, PVC-induced (frequent >10%) |
| **Chemotherapy** | Anthracyclines, trastuzumab, cyclophosphamide |
| **Endocrine** | Hypothyroid, hyperthyroid, Cushing, acromegaly |
| **Infiltrative** | Amyloidosis, haemochromatosis, sarcoid |
| **Nutritional** | Thiamine (beriberi), selenium, carnitine deficiency |
| **Neuromuscular** | Duchenne/Becker MD, myotonic dystrophy, Friedreich ataxia |
| **Stress** | Takotsubo (apical ballooning) |

## 4. Clinical Clues to Specific Etiology
| Clue | Suspect |
|---|---|
| **Skeletal myopathy, family** | Neuromuscular / genetic |
| **AV block, AF, CK↑** | LMNA, desmin, dystrophinopathy |
| **Palpitations, syncope, AΦ** | Plakophilin-2 / ARVC overlap |
| **Pregnancy / postpartum** | PPCM |
| **Heavy alcohol** | Alcoholic CM (reversible on abstinence) |
| **Iron overload, LFTs, arthropathy** | Haemochromatosis |
| **Thick walls + low voltage** | Amyloid |
| **Conduction disease + VT** | Cardiac sarcoid |
| **Frequent PVCs (>10%)** | PVC-induced |

## 5. Workup
- **History:** alcohol, chemotherapy, family, pregnancy, neuromuscular symptoms
- **Exam:** neuropathy, skin (carpal tunnel → amyloid, hyperpigmentation → haemochromatosis)
- **ECG:** low voltage (amyloid), AV block (sarcoid, LMNA), persistent ST ↑ (aneurysm)
- **Echo:** LV size, LVEF, RV involvement, valves
- **CMR with LGE** — pattern narrows aetiology (mid-wall, RV insertion, subepicardial)
- **Genetic panel** (TTN, LMNA, FLNC, PLN, RBM20) for familial
- **Specific tests:** SPEP/UPEP, ferritin/TSAT, viral serology, troponin

## 6. Family Screening
- **First-degree relatives:** ECG + echo at diagnosis; repeat every 3–5 yrs (or sooner if symptomatic)
- Cascade **genetic testing** if pathogenic variant in proband

> **Pearls**
> - **TTN truncating variants** are the most common genetic cause of DCM (~25%).
> - Always ask about **alcohol, chemotherapy, pregnancy, family** — common reversible causes.
> - **CMR with LGE pattern** narrows aetiology: mid-wall = DCM, RV insertion = sarcoid/HTN, subepicardial = myocarditis.
> - **Tachycardia-induced** is reversible — control rate/rhythm.
> - Up to **30–40% of "idiopathic" DCM** has a genetic basis — counsel relatives.

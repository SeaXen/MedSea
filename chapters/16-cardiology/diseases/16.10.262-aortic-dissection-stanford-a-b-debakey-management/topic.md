# Aortic Dissection (Stanford, DeBakey, Management)

## 1. Definition
**Intimal tear → blood enters media → splits into true + false lumen**. Propagates antegrade or retrograde. Mortality **1–2% per hour** in untreated Type A.

### Risk Factors
- **Hypertension** (most important, 70%)
- Connective tissue: **Marfan, Loeys-Dietz, Ehlers-Danlos type IV**
- **Bicuspid aortic valve (BAV)**
- Atherosclerosis, vasculitis (Takayasu, GCA)
- Pregnancy (3rd trimester), cocaine, trauma, iatrogenic (catheter, IABP)
- Cocaine, amphetamines, weightlifting

---

## 2. Classification

### Stanford (clinical, drives treatment)
| Type | Involvement | Treatment |
|---|---|---|
| **A** | Ascending aorta ± arch ± descending | **Emergency surgery** |
| **B** | Descending only (distal to LSCA) | Medical ± TEVAR if complicated |

### DeBakey (anatomical)
- **I** — Ascending + arch + descending
- **II** — Ascending only
- **III** — Descending only

---

## 3. Clinical Features
- **Sudden "tearing / ripping"** chest or back pain
- Type A: anterior chest; Type B: interscapular
- **Pulse deficit** (30% Type A), **BP differential** > 20 mmHg
- **Acute AR murmur** (40% Type A)
- Neurological: stroke, syncope, paraplegia
- Hypotension / tamponade (rupture into pericardium)

---

## 4. Management
### Type A — Emergency Open Repair
- **Bentall procedure** (root + valve + re-implant coronaries)
- **Valve-sparing David procedure** if valve OK
- Immediate surgery — mortality 1–2%/h untreated

### Type B — Medical First
- **HR < 60** with **IV beta-blocker** (esmolol, labetalol)
- **SBP 100–120** mmHg (add nitroprusside after HR controlled)
- Analgesia (morphine / fentanyl)
- **TEVAR** if: malperfusion, rupture, refractory pain / HTN, expansion

---

## 5. Key Pearls

> **Pearls**
- **Type A = surgery** (1–2% mortality per hour)
- **Type B = medical ± TEVAR**
- **HR < 60** with IV BB first, **then SBP 100–120**
- Never thrombolyse a Type A (worsens rupture)
- Long-term: BB + ACEi/ARB + statin, lifelong BP control

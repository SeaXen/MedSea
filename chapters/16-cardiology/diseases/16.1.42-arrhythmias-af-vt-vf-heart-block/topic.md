# Arrhythmias in ACS

## 1. Definition
**Arrhythmias** occur in >75% of MI patients, especially in the first 48 h. Mechanisms: ischemia, reperfusion, autonomic surge, structural change. Can be **supraventricular** (AF, SVT) or **ventricular** (VT/VF, bradyarrhythmia/AV block).

## 2. Key Features
- **VF / sustained VT**: most common in first 24–48 h (ischaemia/reperfusion); primary VF (no HF/shock) ≠ long-term ICD indication
- **Accelerated idioventricular rhythm (AIVR)**: benign, common post-reperfusion
- **AF**: 5–20% of MI; worse prognosis; from atrial ischaemia, ↑atrial pressure, pericarditis
- **Sinus bradycardia/AV block**:
  - Inferior MI → AV node ischaemia (RCA territory) → 1st/2nd type I/3rd block; usually resolves
  - Anterior MI → septal necrosis → infranisian block (RBBB + LAFB, Mobitz II, 3rd) → **permanent pacing** often needed
- **Long QT / Torsades**: from antiarrhythmics, electrolyte disturbance

## 3. Management
- **VF/pulseless VT**: ACLS — defibrillation, amiodarone, lidocaine
- **Sustained VT (stable)**: amiodarone, lidocaine; cardioversion if unstable
- **AF**: rate control (β-blocker, CCB); anticoagulate per CHA2DS2-VASc; cardioversion if unstable
- **Bradyarrhythmias**: atropine, temporary pacing; permanent pacemaker for persistent high-degree block (especially anterior MI)
- **Correct**: K⁺ >4.0, Mg²⁺ >2.0
- **ICD**: only for **secondary** prevention or late (>48 h) ventricular arrhythmia + LVEF ≤35%

> **Pearls**
> - **Primary VF in first 48 h** ≠ ICD indication (low recurrence)
> - **Inferior MI + AV block**: usually transient (RCA)
> - **Anterior MI + AV block**: high mortality, needs permanent pacing
> - Keep **K⁺ >4.0, Mg²⁺ >2.0** in MI

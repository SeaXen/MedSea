# Hypertension — Thresholds & Targets

## 1. Definition
Treatment thresholds and BP targets vary by **guideline** and **patient profile** (age, comorbidity, CV risk, frailty). Modern trials (SPRINT, STEP) favour more aggressive targets in high-risk patients.

## 2. Key Features
- **ACC/AHA 2017**: treat if ≥130/80 (high risk) or ≥140/90 (low risk); target **<130/80** for most
- **ESC/ESH 2018**: treat if ≥140/90 (most); consider at 130–139 if very high risk; target **<130/80** if tolerated, 130–139 in elderly
- **KDIGO 2021 (CKD)**: target **<120 systolic** (SPRINT-like), if tolerated
- **Diabetes**: target <130/80
- **Elderly (≥65 y)**: target 130–139 (avoid <120 in frail)
- **Pregnancy**: treat if ≥140/90 (chronic), ≥160/110 (severe)
- **SPRINT trial**: target <120 (intensive) vs <140 (standard) — intensive ↓ CV events 25%

## 3. Pearls
- **Lower is better**, but **avoid orthostatic hypotension** in elderly
- Out-of-office BP (ABPM/home) should be used to titrate
- Frail elderly: relax targets; aim 140–150
- Initial therapy: ACEi/ARB or CCB or thiazide; combination usually needed
- Single-pill combinations improve adherence

> **Pearls**
> - **SPRINT** trial supports intensive target <120 in high-risk
> - **KDIGO** now recommends <120 systolic in CKD
> - **Elderly**: avoid <120 SBP, watch for orthostasis
> - Out-of-office BP measurement is the standard for titration

# Antihypertensive Drug Classes

## 1. First-Line Classes
- **ACEi** (e.g., lisinopril, ramipril): ↓RAAS, vasodilation, renal protection; dry cough, hyperK+, angioedema
- **ARB** (losartan, valsartan): if ACEi-intolerant; same benefits, less cough
- **CCB — DHP** (amlodipine): vasodilation; peripheral edema, headache
- **CCB — non-DHP** (diltiazem, verapamil): rate + BP; avoid with HFrEF, β-blocker
- **Thiazide** (HCTZ, chlorthalidone, indapamide): natriuresis; hypoK+, hyponatremia, glucose

## 2. Second-Line / Add-On
- **β-blocker** (metoprolol, bisoprolol): compelling for CAD, HFrEF; not first-line for HTN alone
- **MRA** (spironolactone, eplerenone): resistant HTN, HFrEF; hyperK+
- **α-blocker** (doxazosin): BPH; orthostasis
- **Central** (clonidine, moxonidine): rebound HTN
- **Direct vasodilators** (hydralazine): pregnancy, refractory

## 3. Compelling Indications
- **HFrEF**: ACEi/ARB + β-blocker + MRA + SGLT2i (SGLT2i also HTN benefit)
- **DM/CKD**: ACEi/ARB
- **Pregnancy**: labetalol, nifedipine, methyldopa

## 4. Pearls
> **Pearls**: Start 2 drugs if BP **>20/10** above target. **A+C** (ACEi+CCB) or **A+D** (ACEi+thiazide) are preferred combos per ACCOMPLISH.

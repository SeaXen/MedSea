# Acromegaly and the Heart

## 1. Definition
- Excess **GH/IGF-1** — usually pituitary adenoma
- Develops over years; cardiac disease = leading cause of death

## 2. Cardiac Features
- **Biventricular hypertrophy** (early concentric, later dilated) — **acromegalic cardiomyopathy**
- **HF**: HFrEF or HFpEF
- **Valvular disease**: AR, MR (esp. mitral prolapse) from fibrotic changes
- **HTN**: 30% (Na+ retention, vascular remodeling)
- **Arrhythmias**: AF, SVT, ventricular ectopy
- **CAD**: accelerated atherosclerosis

## 3. Diagnosis
- **IGF-1** (screening) — age- and sex-adjusted
- **OGTT with GH** (failure to suppress <1 μg/L = confirm)
- **MRI pituitary** for adenoma
- Echo: LVH, diastolic dysfunction, valvular

## 4. Management
- **Transsphenoidal surgery** (first-line)
- **Medical**: somatostatin analogs (octreotide, lanreotide), GH receptor antagonist (**pegvisomant**), cabergoline
- **Radiotherapy** for residual
- Standard HF Rx; **careful** rehydration post-surgery (glucose/Na+)
- Screen for colonoscopy, sleep apnea

> **Pearls**: Coarse features + HTN + DM + cardiomegaly = **acromegaly**. Normalize **IGF-1** improves CV outcomes. Even treated patients need lifetime cardiac surveillance.

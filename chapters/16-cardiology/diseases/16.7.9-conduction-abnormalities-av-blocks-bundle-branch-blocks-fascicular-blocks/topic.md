# Conduction Abnormalities (AV Blocks, Bundle Branch Blocks, Fascicular Blocks)

**Audience:** FCPS / MRCP exam prep

## 1. Introduction
Conduction system disease can occur at the AV node, His bundle, or bundle branches. **Localisation matters for prognosis and pacing decisions.**

## 2. AV Blocks

### First-Degree AV Block
- PR interval >200 ms (1 large square)
- Every P followed by QRS
- Usually benign, no pacing needed
- Causes: high vagal tone (athletes), ischaemia, drugs (β-blockers, CCB, digoxin), inferior MI, AV nodal disease

### Second-Degree AV Block

| Type | Pattern | Site | Prognosis |
|---|---|---|---|
| **Mobitz I (Wenckebach)** | Progressive PR lengthening then dropped QRS | AV node | Usually benign |
| **Mobitz II** | Constant PR with intermittent dropped QRS | His-Purkinje | Risk of progression to CHB — pacing |
| **2:1 / 3:1** | Fixed conduction ratio | Either | Investigate; often needs pacing |
| **High-grade** | ≥2 consecutive P waves non-conducted | Infra-Hisian | Pacemaker |

### Third-Degree (Complete) Heart Block
- **Complete AV dissociation** — P waves and QRS independent
- Atrial rate > ventricular rate
- QRS width localises escape: **narrow** = junctional (40–60 bpm, AV nodal block); **wide** = ventricular (20–40 bpm, infranodal)
- **Cannon a waves**, soft S1, changing intensity
- Symptoms: Stokes-Adams attacks, syncope, fatigue
- **Treatment: atropine (temporary), isoprenaline, transcutaneous/transvenous pacing, permanent pacemaker**

## 3. Bundle Branch Blocks (QRS ≥120 ms)

### Right Bundle Branch Block (RBBB)
- **rSR' ("M-shaped") in V1**
- **Wide, slurred S in I, V6**
- QRS ≥120 ms
- Causes: **PE, RVH, RBBB at baseline, anterior MI, atrial septal defect (rsR' V1), Brugada**

### Left Bundle Branch Block (LBBB)
- **Broad monomorphic R in I, V6** (no Q)
- **QS or rS in V1**
- QRS ≥120 ms
- Discordant ST/T (ST opposite direction to QRS)
- Causes: **severe CAD, severe LV dysfunction, aortic stenosis, dilated cardiomyopathy**
- **New LBBB + chest pain = STEMI equivalent** (Sgarbossa criteria to diagnose STEMI in LBBB)

## 4. Fascicular Blocks (Hemiblocks)

### Left Anterior Fascicular Block (LAHB)
- LAD (–30° to –90°)
- qR in aVL, rS in II, III, aVF
- QRS <120 ms
- Causes: anterior MI, LVH, Chagas, Lenègre disease

### Left Posterior Fascicular Block (LPHB)
- RAD
- rS in I, aVL, qR in II, III, aVF
- QRS <120 ms
- **Diagnosis of exclusion** (need to rule out RVH, PE, lateral MI, lead reversal)
- Causes: inferior MI, aortic dissection, Lenègre

## 5. Bifascicular Block
- RBBB + LAHB (or LPHB)
- Risk of progression to CHB if acute (e.g. anterior MI)
- **Pacemaker indicated** if syncope, intermittent CHB, prolonged HV interval

## 6. Trifascicular Block
- Bifascicular block + 1° AV block (or alternating bundle branch block)
- Indicates diffuse conduction system disease
- **Strong indication for pacing**

## 7. Pacing Indications Summary
- **Symptomatic bradycardia** (any cause)
- **CHB** (any aetiology, any site)
- **Mobitz II**, high-grade, 2:1 with bifascicular block
- **Sinus node dysfunction** with symptoms
- **Bifascicular block with syncope** (after negative workup)
- **Post-MI:** persistent 2°/3° block, new bifascicular block

> **Pearls**
> - **Mobitz I** is usually nodal (benign), **Mobitz II** is infranodal (malignant).
> - **New LBBB in chest pain = reperfusion** (don't wait for Sgarbossa score to act).
> - **Bifascicular block + syncope = EP study ± pacemaker**, even if no documented block.
> - **Alternating LBBB / RBBB** = "bilateral bundle branch block" = high risk of asystole → permanent pacing.

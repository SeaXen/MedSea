# ICD Programming

## 1. Goal
- **Avoid inappropriate shocks** (50% of all shocks historically)
- Allow detection of true VT/VF
- Use **ATP (anti-tachycardia pacing)** to terminate VT painlessly

## 2. Detection Zones (Modern Programming)
- **VT zone** (~180–220 bpm) — ATP first, then shock if not terminated
- **VF zone** (>220 bpm) — direct shock
- **SVT discriminators** — avoid shocking AF, SVT

## 3. Key Trials
- **MADIT-RIT** (2012): high-rate cutoff (≥200 bpm) + delayed detection → ↓ inappropriate shocks by 76%
- **ADVANCE III** (2013): longer detection intervals
- Modern: avoid therapy <200 bpm in primary prevention

## 4. ATP (Anti-tachycardia Pacing)
- **Burst pacing** (8 pulses, ~85% of VT cycle length) terminates 75% of VT
- Painless vs shock
- Programming: 1–2 ATP attempts before shock in VT zone

## 5. SVT Discriminators
- **Interval stability** (AF vs VT)
- **Onset** (paroxysmal vs gradual)
- **Morphology** (WCT discrimination)
- **AV relationship** (1:1 SVT, AV dissociation in VT)

## 6. Special Cases
- **Primary prevention**: high-rate cutoff, longer detection
- **Secondary prevention**: lower zone, more aggressive
- **ICD in children/congenital**: higher rates, more ATP

> **Pearls** — Modern ICD programming: **high-rate cutoff, delayed detection, ATP first**. **MADIT-RIT** paradigm ↓ inappropriate shocks by 76%. Always enable SVT discriminators.

## Key Algorithms & Scoring Systems (Memorise)

| System | Domain | Key Thresholds |
|--------|--------|----------------|
| **FAST / ABCD2** | TIA/Stroke | ABCD2 ≥4 = high risk |
| **NIHSS / mRS** | Stroke severity/outcome | NIHSS ≥16 severe |
| **GCS / FOUR Score** | Coma | GCS ≤8 = intubation |
| **McDonald 2017** | MS diagnosis | DIS + DIT |
| **El Escorial** | ALS diagnosis | UMN + LMN in ≥3 regions |
| **Movement Disorder Society** | PD diagnosis | Bradykinesia + rigidity/tremor |
| **NIA-AA / IWG** | AD diagnosis | Amyloid + tau + neurodegeneration |
| **SCORTEN** | SJS/TEN | ≥3 = 35% mortality |
| **APACHE II / SOFA** | ICU/Encephalopathy | Organ failure scoring |
| **MMSE / MoCA** | Cognitive screen | MoCA <26 = impairment |
| **IASP / Neuropathic Pain** | Neuropathic pain | DN4, LANSS |
| **Epworth Sleepiness Scale** | Narcolepsy/OSA | >10 = excessive sleepiness |

## 6. Chapter-Level SBAs (Single Best Answer)

> These SBAs test application of **Neurology and Stroke Medicine**-specific scoring systems and clinical decision algorithms.

**SBA1.** A patient with tia/stroke requires risk stratification using **FAST / ABCD2**. What is the clinical significance of the threshold value?
A. FAST / ABCD2 < threshold = no clinical action needed
B. **FAST / ABCD2 ≥ threshold = ABCD2 ≥4 = high risk**
C. FAST / ABCD2 score is purely prognostic without therapeutic implications
D. FAST / ABCD2 should only be calculated by specialists
E. FAST / ABCD2 replaces clinical judgement entirely

> **Answer:** B
> **Rationale:** FAST / ABCD2 thresholds guide clinical decisions: ABCD2 ≥4 = high risk.

**SBA2.** A patient with stroke severity/outcome requires risk stratification using **NIHSS / mRS**. What is the clinical significance of the threshold value?
A. NIHSS / mRS < threshold = no clinical action needed
B. **NIHSS / mRS ≥ threshold = NIHSS ≥16 severe**
C. NIHSS / mRS score is purely prognostic without therapeutic implications
D. NIHSS / mRS should only be calculated by specialists
E. NIHSS / mRS replaces clinical judgement entirely

> **Answer:** B
> **Rationale:** NIHSS / mRS thresholds guide clinical decisions: NIHSS ≥16 severe.

**SBA3.** A patient with coma requires risk stratification using **GCS / FOUR Score**. What is the clinical significance of the threshold value?
A. GCS / FOUR Score < threshold = no clinical action needed
B. **GCS / FOUR Score ≥ threshold = GCS ≤8 = intubation**
C. GCS / FOUR Score score is purely prognostic without therapeutic implications
D. GCS / FOUR Score should only be calculated by specialists
E. GCS / FOUR Score replaces clinical judgement entirely

> **Answer:** B
> **Rationale:** GCS / FOUR Score thresholds guide clinical decisions: GCS ≤8 = intubation.

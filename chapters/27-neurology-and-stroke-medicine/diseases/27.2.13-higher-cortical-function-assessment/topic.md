# Higher Cortical Function Assessment

> [!tip] **MMSE** = screening; **MoCA** = more sensitive for MCI/executive; **ACE-III** = most comprehensive brief battery.
> [!tip] **Localisation:** Dominant (usually L) = language, calculation, Gerstmann. Non-dominant (R) = neglect, visuospatial, dressing apraxia. Frontal = executive, personality.

## 1. Definition / Epidemiology / Classification
**Definition:** Integrated cognitive functions mediated by association cortices: attention, memory, language, praxis, gnosis, executive, visuospatial.

| Metric | Value |
|--------|-------|
| Dementia | 5-7% >65 yr; 20% >85 yr |
| MCI | 15-20% >65 yr; 10-15%/yr convert |
| Aphasia post-stroke | 30-40% acute, 10-15% chronic |
| Neglect post-R MCA | 40-80% acute, 30% persistent |

| Domain | Function | Test | Lobe |
|--------|----------|------|------|
| Attention | Concentration | Digit span, months backwards | Frontal-parietal |
| Memory | Registration, recall | 3-word recall | Temporal/hippocampal |
| Language | Comprehension, expression | Naming, repetition, fluency | Dominant hemisphere |
| Praxis | Learned motor | Pantomime, imitation | Dominant parietal |
| Executive | Planning, flexibility | Fluency, Trails B | Prefrontal |
| Visuospatial | Spatial | Clock drawing, copying | Non-dominant parietal |

## 2. Aetiology / Pathophysiology
- **Memory:** AD, hippocampal stroke, limbic encephalitis, Wernicke-Korsakoff
- **Language:** Dominant MCA stroke, FTD, tumour
- **Neglect:** Non-dominant (R) MCA/parietal lesion
- **Executive:** FTD, frontal stroke, NPH, MS, subcortical dementias
- **Diffuse:** Delirium, encephalopathy, CJD, Hashimoto's

## 3. Clinical Features

**Bedside tests:**
| Test | Normal | Abnormal → |
|------|--------|------------|
| Orientation | All correct | Disorientation to time (delirium/dementia) |
| Attention | ≥5/7 serial 7s; months backward | Frontal/diffuse |
| Registration (3 words ×3) | 3/3 | Encoding failure |
| Recall (5 min) | 3/3 | Hippocampal |
| Naming | Watch, pen, larynx | Anterior temporal |
| Repetition | "No ifs, ands or buts" | Perisylvian |
| Comprehension | 3-step command | Wernicke |
| Praxis | "Show how to wave" | Dominant parietal |
| Clock drawing (10 past 11) | Spaced, intact | Non-dominant parietal |
| Fluency (FAS 1 min) | ≥15 words | Frontal |

**Aphasia syndromes:**
| Type | Fluency | Comprehension | Repetition | Lesion |
|------|---------|--------------|------------|--------|
| Broca | Non-fluent | Good | Poor | L inferior frontal |
| Wernicke | Fluent (paraphasic) | Poor | Poor | L superior temporal |
| Conduction | Fluent | Good | **Poor** | Arcuate fasciculus |
| Global | Non-fluent | Poor | Poor | Whole L MCA |
| Transcortical motor | Non-fluent | Good | **Preserved** | ACA/anterior watershed |
| Transcortical sensory | Fluent | Poor | **Preserved** | Posterior watershed |
| Anomic | Fluent | Good | Good | Anterior temporal |

**Other syndromes:**
- **Gerstmann:** acalculia, agraphia, finger agnosia, L-R disorientation → L angular gyrus
- **Hemispatial neglect:** R parietal; line bisection, clock, cancellation, extinction
- **Anton:** Cortical blindness with denial; bilateral occipital
- **Capgras:** Familiar person = imposter; bilateral frontal-temporal
- **Prosopagnosia:** Cannot recognise faces; bilateral fusiform
- **Dressing apraxia:** R parietal
- **Apraxia (ideomotor):** Cannot perform on command but uses spontaneously

## 4. Diagnostic Approach
1. Screen (MMSE/MoCA) → 2. Full ACE-III + bedside tests → 3. Identify domain pattern → 4. Aetiology workup
| Test | Range | MCI | Mild | Moderate | Severe |
|------|-------|-----|------|----------|--------|
| MMSE | 30 | - | 21-26 | 10-20 | <10 |
| MoCA | 30 | 19-25 | 14-18 | 5-13 | <5 |
| ACE-III | 100 | <88 | 88-82 | 81-60 | <60 |

## 5. Investigations
- **Bloods:** FBC, U&E, Ca, TSH, B12, folate, glucose, LFT, syphilis, HIV
- **CT head:** All new cognitive decline (mass, hydrocephalus, stroke, SDH)
- **MRI brain:** Hippocampal atrophy (AD), vascular, FTD pattern, PCA
- **CSF:** NPH (opening pressure), autoimmune (NMDAR, LGI1), CJD (14-3-3, RT-QuIC)
- **FDG-PET:** Hypometabolism pattern (AD = temporoparietal, FTD = frontal, DLB = occipital sparing)
- **Amyloid/tau PET:** Amyloid positivity in AD
- **DAT-SPECT:** DLB/PDD (nigrostriatal loss)
- **Neuropsychology:** Full WAIS/WMS/Trails/Boston naming; for MCI vs dementia, capacity, baseline

## 6. Differential Diagnosis
| Differential | Distinguishing | Key Test |
|--------------|----------------|----------|
| Dementia vs Delirium | Acute, fluctuating, attention | Bedside, history |
| Pseudodementia (depression) | Effortful "don't know", mood | Trial of antidepressant |
| MCI vs Dementia | Single domain, function preserved | Neuropsychology |
| FTD vs AD | Early behaviour/language, sparing of memory | MRI (frontal atrophy) |
| LBD vs AD | Visual hallucinations, parkinsonism, RBD | DAT-SPECT |
| Vascular dementia | Stepwise, focal signs | MRI lacunes, WMH |
| CJD | Rapid (weeks-months), myoclonus | EEG (PSWCs), CSF RT-QuIC |
| PCA (AD variant) | Visuospatial, Balint, sparing memory | MRI posterior |
| Autoimmune encephalitis | Rapid, seizures, psychiatric | CSF antibodies |
| Wernicke | Triad confusion/ataxia/ophthalmoplegia | Thiamine response |

## 7. Management
**Reversible:** Hypothyroid → levothyroxine; B12 IM; Wernicke → IV thiamine 500mg TDS before glucose; NPH → VP shunt (tap test positive); autoimmune → methylpred 1g ×3-5d ± IVIG/PLEX; tumour → surgery/oncology; infection → antimicrobials.

**Symptomatic (AD):**
| Drug | Indication | Dose | Notes |
|------|-----------|------|-------|
| Donepezil | Mild-mod AD | 5mg nocte → 10mg 4 wk | Nausea, bradycardia, vivid dreams |
| Rivastigmine | AD, PDD | Patch 4.6 → 9.5mg/24h | GI tolerability, only licensed in PDD |
| Galantamine | AD | 8mg MR → 16 → 24mg | MR |
| Memantine | Mod-severe AD | 5mg → 10mg BD | NMDA antagonist |

**Behavioural:** Non-pharm first (reorientation, routine, environment). Severe: atypical antipsychotic (quetiapine 12.5-25mg) cautiously - CVAE/mortality risk. **Avoid** haloperidol, benzodiazepines. Mood: SSRI (sertraline, citalopram). Sleep: melatonin, mirtazapine.

**Supportive:** Capacity assessment, lasting power of attorney, advance care planning early. SALT, OT, physiotherapy, social support, day centres, respite.

## 8. Drug Interactions
| Drug | Issue | Action |
|------|-------|--------|
| Anticholinesterases | Bradycardia, syncope, urinary retention | ECG; caution β-blockers, asthma, SSS |
| Donepezil + NSAIDs | ↑GI bleeding | PPI cover |
| Memantine | Confusion, hallucinations | Reduce in eGFR <30 |
| Antipsychotics in dementia | ↑Stroke, mortality | Lowest dose, shortest, review |
| Benzodiazepines | Falls, paradoxical agitation | Avoid in elderly cognitive impairment |

## 9. Procedures
- Bedside cognitive testing (core skill)
- LP: NPH, suspected infection, autoimmune, CJD
- Neuropsychology referral (full assessment, capacity)
- FDG-PET/Amyloid PET for atypical dementia

## 10. Complications
- Falls/hip fracture (30-50% in dementia): vitamin D, environment
- Delirium superimposed (25% in hospital): avoid sedatives, screen
- Behavioural disturbances (50-70% late): non-pharm first
- Aspiration: SALT, feeding decisions
- Pressure sores (late): reposition, mattress
- Mortality: AD median survival 8-10 yr

## 11. Red Flags
- Rapid cognitive decline (weeks-months): MRI + CSF autoimmune
- New focal deficit: urgent imaging
- Seizure + cognitive change: EEG, autoimmune panel
- Young onset (<65): genetic counselling
- Wernicke triad: IV thiamine immediately
- Suicidal ideation: psych, remove means

## 12. Prognosis
| Condition | Prognosis |
|-----------|-----------|
| AD | 8-10 yr median |
| Vascular | 4-7 yr; stepwise |
| LBD | 6-8 yr; falls, neuroleptic sensitivity |
| FTD | 6-11 yr; behaviour, swallowing |
| Autoimmune | Variable; reversible with treatment |
| CJD | 6-12 months (median) |
| Korsakoff | 50% chronic; Wernicke 10-20% mortality |

## 13. Topic Correlation
Aphasia ↔ [[Stroke Syndromes]], [[Cranial Nerve Examination]]; Memory loss ↔ [[Dementia]], [[Transient Amnesia]]; Neglect ↔ [[Stroke Syndromes]]; Frontal ↔ [[FTD]], [[NPH]]; Posterior ↔ [[PCA]], [[Visual Hallucinations]].

## 14. Special Situations
- **Pregnancy:** Avoid gadolinium; radiation risk for PET
- **Elderly:** Pre-test probability high; include hearing/visual loss, drugs
- **Renal:** Memantine dose reduce; caution contrast
- **DVLA:** Dementia + MCI = must cease driving, notify DVLA. Group 1: licence on individual assessment. Group 2: permanent bar
- **Capacity:** Mental Capacity Act; presume capacity; functional test (Understand, Retain, Weigh, Communicate)

## Viva Questions
1. **Q:** 5-min bedside cognitive assessment? **A:** Orientation → attention (months backwards) → registration (3 words) → language (naming, repetition, comprehension) → memory (recall) → praxis → visuospatial (clock) → executive (verbal fluency).
2. **Q:** MMSE vs MoCA? **A:** MMSE quick, less executive sensitive. MoCA better for MCI - tests executive, abstraction, visuospatial. NICE recommends MoCA for MCI.
3. **Q:** Broca vs Wernicke vs conduction? **A:** Broca = non-fluent, good comp, poor repet. Wernicke = fluent, poor comp, poor repet. Conduction = fluent, good comp, **poor repet** (arcuate).
4. **Q:** Gerstmann? **A:** Acalculia, agraphia, finger agnosia, L-R disorientation; L angular gyrus.
5. **Q:** Neglect - test, lesion? **A:** R MCA/inferior parietal. Tests: line bisection, clock, cancellation, extinction.
6. **Q:** Reversible dementias? **A:** Hypothyroid, B12/folate, syphilis, HIV, NPH, tumour, SDH, depression, autoimmune, drugs, Wernicke.
7. **Q:** DVLA rules dementia? **A:** Cease driving, notify DVLA. Group 2 permanent bar. MCI: must stop + notify.
8. **Q:** AD treatment? **A:** Donepezil 5-10mg (or rivastigmine/galantamine) for mild-mod. Add memantine for mod-severe. Non-pharm support.
9. **Q:** Wernicke vs Korsakoff? **A:** Wernicke = acute confusion/ataxia/ophthalmoplegia (IV thiamine). Korsakoff = chronic anterograde amnesia, confabulation (irreversible).
10. **Q:** Capacity principles? **A:** Presume; Understand + Retain + Weigh + Communicate. Functional, not outcome.
11. **Q:** Executive tests? **A:** Verbal fluency (FAS), Trails B, Luria 3-step, clock, Wisconsin card sort, Stroop.
12. **Q:** MMSE 28/30 with obvious deficit? **A:** MMSE insensitive to executive/visuospatial; use MoCA/ACE-III. Consider bvFTD, LBD, PCA.

## Common Confusions
| Confusion | Clarification |
|-----------|---------------|
| MMSE normal ≠ no cognitive impairment | Misses executive/visuospatial - use MoCA |
| Delirium ≠ dementia | Delirium = acute, fluctuating, attention, often reversible |
| Broca = aware, Wernicke = anosognosic | Wernicke unaware of language deficit |
| Conduction = poor repet, fluent, good comp | Arcuate fasciculus |
| Transcortical = preserved repetition | Distinguishes from Broca/Wernicke |
| Neglect ≠ hemianopia | Hemianopia = visual field; neglect = attentional |
| Anton = cortical blindness + denial | Bilateral occipital; pupillary reflex intact |
| PCA = AD variant | Early visuospatial, memory spared early |
| Apraxia test | Use object first; apraxia = cannot on command |

## Mnemonics
1. **GLASGOW** (MMSE items) — orientation, registration, attention, recall, language, visuospatial
2. **NUNPUF** (aphasia screen) — naming, understanding, naming, praxis, utterance, fluency
3. **DRESS** (Gerstmann) — dominant, R-L, eye, speech, subtract (or 4 features: Acalculia, Agraphia, finger agnosia, L-R disorientation)
4. **FAST** (language PACES) — fluency, assess comprehension, sentence repetition, test naming
5. **HEAD** (praxis) — hold, execute, analyse, demonstrate
6. **WATER** (aphasia) — **W**ernicke, **A**nomia, **T**ranscortical, **E**xpressive (Broca), **R**eceptive (Wernicke/Conduction), **G**lobal

## MCQs (10)
1. **Q:** Best bedside executive test? **A:** B (Verbal fluency) — Recall=temporal; Clock=visuospatial; Naming=language.
2. **Q:** Fluent + poor comp + poor repet aphasia? **A:** B (Wernicke) — superior temporal gyrus.
3. **Q:** Acalculia, agraphia, finger agnosia, L-R disorientation? **A:** B (Dominant parietal angular gyrus - Gerstmann).
4. **Q:** Most specific neglect test? **A:** C (Line bisection) — visual fields = hemianopia.
5. **Q:** Visual hallucinations + parkinsonism + fluctuating cognition? **A:** C (LBD).
6. **Q:** UK DVLA rule for new AD? **A:** C (Cease driving + notify DVLA). Group 2 permanent bar.
7. **Q:** Rapid dementia + myoclonus + PSWCs? **A:** B (CJD - 6-12 months).
8. **Q:** First-line symptomatic AD? **A:** B (Donepezil). Memantine added mod-severe.
9. **Q:** Capacity - 4 abilities? **A:** B (Understand, Retain, Weigh, Communicate).
10. **Q:** Clock numbers crowded to one side? **A:** C (Non-dominant parietal - neglect).

**MCQ Answers:** 1B, 2B, 3B, 4C, 5C, 6C, 7B, 8B, 9B, 10C

## SBA Questions (10)
1. **Q:** MMSE 24, MoCA 18, CT hippocampal atrophy? **A:** B (AD). FTD = behaviour/language; LBD = hallucinations/parkinsonism; Vascular = stepwise.
2. **Q:** Disinhibition + frontal atrophy, age 65? **A:** B (bvFTD).
3. **Q:** Visual hallucinations + parkinsonism + neuroleptic sensitivity? **A:** B (DLB). PDD = motor first >1 yr; Charles Bonnet = isolated, normal cognition.
4. **Q:** Asymmetric apraxia, alien limb, parietal atrophy? **A:** B (CBS).
5. **Q:** Visuospatial decline, sparing memory, posterior atrophy? **A:** B (PCA - AD variant).
6. **Q:** Anterograde + retrograde amnesia + confabulation + mammillary atrophy? **A:** B (Korsakoff).
7. **Q:** Donepezil 5mg, next step at 4 weeks? **A:** B (Increase to 10mg if tolerated).
8. **Q:** Wernicke - first treatment? **A:** B (IV thiamine 500mg BEFORE glucose).
9. **Q:** Gait apraxia, incontinence, dementia, ventriculomegaly, tap test +? **A:** B (NPH - VP shunt).
10. **Q:** Severe agitation in dementia - safest drug? **A:** B (Quetiapine 12.5-25mg). Avoid haloperidol/BZD.

**SBA Answers:** 1B, 2B, 3B, 4B, 5B, 6B, 7B, 8B, 9B, 10B

## Flashcards
- **MMSE:** 30 points; Orientation 10, Registration 3, Attention 5, Recall 3, Language 8, Visuospatial 1; <24 abnormal
- **MoCA:** 30 points; sensitive for MCI; <26 abnormal
- **ACE-III:** 100 points; <88 abnormal
- **Aphasia areas:** Broca (inferior frontal); Wernicke (superior temporal); Arcuate = conduction
- **Conduction aphasia:** Fluent, good comp, **poor repetition**
- **Transcortical aphasias:** Repetition **preserved**
- **Gerstmann:** Acalculia, Agraphia, finger Agnosia, L-R disorientation → L angular gyrus
- **Neglect:** R MCA/parietal; line bisection, clock, cancellation
- **Anton:** Cortical blindness + denial; bilateral occipital
- **Capgras:** Familiar = imposter; bilateral frontal-temporal
- **DLB triad:** Visual hallucinations, parkinsonism, fluctuating cognition + RBD + neuroleptic sensitivity
- **NPH:** Wet, wobbly, wacky; tap test → VP shunt
- **Reversible dementia screen:** TSH, B12, folate, syphilis, HIV, CT (NPH, tumour, SDH)
- **DVLA dementia:** Cease driving + notify; Group 2 permanent bar
- **Cholinesterase inhibitors:** Donepezil 5-10mg, Rivastigmine patch, Galantamine; Rivastigmine in PDD
- **Memantine:** Moderate-severe AD; NMDA antagonist; 5mg→10mg BD

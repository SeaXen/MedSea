# Neuroinvestigations Overview

> [!tip] **Investigate to ANSWER a question** — never order tests without a clinical hypothesis. Localise first → then choose the investigation.

## Learning Objectives
- [ ] Match modality to clinical question (structure vs function, urgent vs elective)
- [ ] Order appropriate first-line tests for common neurological presentations
- [ ] Recognise indications/contraindications for LP, MRI, CT, EEG, NCS
- [ ] Interpret basic CSF, imaging, neurophysiology results
- [ ] Apply ALARA/safety principles (radiation, contrast, pregnancy)

---

## 1. Definition / Epidemiology / Classification

### Definition
A framework for selecting appropriate investigations in neurological disease based on **anatomical localisation** and **pathophysiological hypothesis**. Investigations are categorised as:
- **Structural** (CT, MRI, angiography)
- **Functional** (EEG, NCS/EMG, evoked potentials, PET/SPECT)
- **Biochemical/Cellular** (CSF, blood, urine, biopsy)
- **Molecular** (genetic, antibody panels)

### Classification by Modality
| Modality | What it Shows | Best For |
|----------|---------------|----------|
| **CT head** | Acute blood, bone, calcification, mass effect | Acute stroke, trauma, haemorrhage |
| **MRI brain** | Parenchymal detail, demyelination, tumour, infection | MS, tumours, encephalitis, stroke (DWI) |
| **CT/MR angiography** | Vascular lumen, aneurysms, stenosis | SAH, stroke, dissection |
| **EEG** | Cortical electrical activity | Epilepsy, encephalopathy, brain death |
| **NCS/EMG** | Peripheral nerve/muscle function | Neuropathy, myopathy, NMJ disorders |
| **Evoked potentials** | Central pathway conduction | MS (demyelination), optic neuritis |
| **LP + CSF** | Intrathecal milieu | Infection, inflammation, haemorrhage |
| **PET/SPECT** | Metabolic/perfusion patterns | Dementia, tumour, refractory epilepsy |

## 2. When to Order — Clinical Triggers

| Presentation | First-Line Investigation |
|--------------|--------------------------|
| **Thunderclap headache** | **CT head (urgent)** → LP if CT -ve (SAH) |
| **Focal deficit, sudden** | **CT head (urgent)** → MRI/CTP if stroke |
| **Headache + red flags** | CT/MRI ± LP |
| **First seizure** | MRI brain, EEG, bloods (NOT routine CT) |
| **Suspected MS** | MRI brain + cord, CSF (OCB), VEP |
| **Confusion/coma** | CT, metabolic screen, EEG, LP if febrile |
| **Acute flaccid weakness** | MRI spine, NCS, LP, CK |
| **Peripheral neuropathy** | NCS/EMG, bloods (glucose, B12, autoimmune) |
| **Visual loss** | Visual acuity, fundoscopy, VEP, MRI |
| **Movement disorder** | MRI brain (Haller-Volhard), copper/ceruloplasmin, DAT-SPECT |
| **Dementia (<65 yrs)** | MRI, CSF, EEG, HIV, syphilis, autoimmune |
| **Suspected myopathy** | CK, NCS/EMG, MRI muscle, genetic/biopsy |

## 3. CT Head — When & Why

### Indications
- **Trauma** (NICE head injury criteria)
- **Suspected stroke** (rule out haemorrhage before thrombolysis)
- **Thunderclap headache** (SAH)
- **Altered consciousness / first seizure** in ED
- **Known malignancy** with new symptoms
- **Acute hydrocephalus** assessment
- **Bone windows** for skull base, fractures

### What CT Shows Well
- Acute blood (hyperdense), mass effect, midline shift
- Hydrocephalus, large infarcts (loss of grey-white differentiation)
- Calcification (tumours, infections, Fahr's)
- Bone fractures

### What CT Misses
- **Posterior fossa** lesions (beam-hardening artefact)
- **Acute ischaemia** <6h (early infarcts subtle)
- **Demyelination** (small plaques)
- **Meningitis/encephalitis** (early)
- **Small tumours** (<1 cm)

### CT Angiography (CTA)
- **Stroke workup** (large vessel occlusion — candidate for thrombectomy)
- **Aneurysm** (CTA replaces DSA for most SAH)
- **Dissection** (carotid/vertebral)
- **Vasculitis** (segmental narrowing, beading)

### CT Perfusion (CTP)
- **Stroke**: penumbra vs core (mismatch = salvageable tissue)
- **Vasospasm** post-SAH

## 4. MRI Brain — When & Why

### Standard Sequences
| Sequence | Shows |
|----------|-------|
| **T1** | Anatomy; grey/white matter contrast; post-contrast enhancement |
| **T2/FLAIR** | Oedema, demyelination, gliosis; FLAIR suppresses CSF |
| **DWI** | **Acute ischaemia (cytotoxic oedema)**; abscess vs tumour |
| **ADC** | Confirms restricted diffusion |
| **SWI/T2*** | Microbleeds, cavernoma, haemosiderin |
| **MRA** | Vascular anatomy (without contrast) |
| **Post-gadolinium T1** | Blood-brain barrier breakdown (tumour, MS, infection) |

### Indications
- **All non-acute** structural brain pathology
- **Suspected MS** (periventricular, juxtacortical, infratentorial, cord)
- **Tumour** (better tissue characterisation)
- **Encephalitis** (HSV — temporal; limbic — autoimmune)
- **Posterior fossa** lesions (CT misses)
- **Cavernoma, AV malformation**
- **Cord compression/myelopathy** (MRI = gold standard)
- **Vasculitis** (vessel wall imaging)

### Contraindications
- **Absolute:** Non-MRI-compatible pacemaker, cochlear implant, ferromagnetic aneurysm clips, metallic intraocular FB
- **Relative:** Claustrophobia, severe obesity, pregnancy (1st trimester — gadolinium contraindicated)
- **Caution:** Renal impairment with gadolinium (NSF risk with older agents)

## 5. Lumbar Puncture & CSF

### Indications
| Indication | CSF Test |
|------------|----------|
| **Suspected meningitis/encephalitis** | Cells, glucose, protein, culture, PCR (HSV, VZV, EBV, enterovirus, TB) |
| **SAH (CT -ve)** | Xanthochromia (≥12h after onset), bilirubin |
| **Suspected MS** | OCB (paired with serum), IgG index |
| **Guillain-Barré** | Albuminocytological dissociation (↑protein, normal cells) |
| **Autoimmune encephalitis** | Antibody panel (NMDA-R, LGI1, CASPR2, GABA-B, etc.) |
| **Malignancy** | Cytology, flow cytometry |
| **IIH** | Opening pressure >25 cmCSF |
| **NPH** | Large-volume LP (tap test) |

### Contraindications
- **Raised ICP** (papilloedema + focal deficit; risk of herniation)
- **Mass lesion** on imaging
- **Coagulopathy** (INR >1.4, platelets <50, on DOACs <48h)
- **Skin infection** at puncture site
- **Suspected spinal epidural abscess** (defer until imaging)

### Normal CSF Values
- **Opening pressure:** 6–25 cmCSF (lateral decubitus)
- **Appearance:** Crystal clear ("crystal clear, like water from a tap")
- **WCC:** <5/µL (mononuclear); no neutrophils
- **Protein:** 0.15–0.45 g/L
- **Glucose:** 2/3 of serum (>2.5 mmol/L if serum normal)
- **No OCB** in serum-matched sample

### CSF Patterns
| Condition | Cells | Protein | Glucose |
|-----------|-------|---------|---------|
| **Bacterial meningitis** | ↑↑↑ neutrophils | ↑↑ | ↓↓ |
| **Viral meningitis** | ↑ lymphocytes | ↑ | Normal |
| **TB/fungal** | ↑ lymphocytes | ↑↑ | ↓ |
| **SAH** | RBCs / xanthochromia | ↑ | Normal |
| **GBS** | Normal cells | ↑↑ (>1 g/L) | Normal |
| **MS** | Normal / mild ↑ | Normal/mild ↑ | Normal (OCB+) |
| **Autoimmune encephalitis** | Mild ↑ lymph | Mild ↑ | Normal |

## 6. Neurophysiology — Quick Guide

### EEG (See EEG topic)
- **First seizure:** Routine EEG (yield ~50%)
- **Suspected NCSE:** Continuous EEG (cEEG)
- **Encephalopathy:** Triphasic waves (metabolic), PEDs (CJD)
- **Brain death:** Confirmatory test

### NCS/EMG (See NCS-EMG topic)
- **Symmetric distal sensory neuropathy** → chronic axonal (diabetes, B12, idiopathic)
- **Asymmetric motor > sensory with ↑protein** → demyelinating (CIDP)
- **Acute flaccid areflexia** → GBS vs polymyositis vs spinal cord
- **Myopathy:** ↑CK, myopathic EMG, MRI muscle

### Evoked Potentials (See EP topic)
- **VEP:** Delayed P100 in MS (optic neuritis)
- **BAEP:** Acoustic neuroma, MS brainstem lesions
- **SSEP:** Cord lesions, intraoperative monitoring

## 7. Vascular Imaging

| Modality | Indication | Pros | Cons |
|----------|-----------|------|------|
| **CTA** | Acute stroke, SAH, dissection | Fast, widely available | Iodinated contrast, radiation |
| **MRA** | Outpatient workup, screening | No radiation | Slower, less sensitive to calcium |
| **DSA** | Aneurysm treatment, vasculitis detail | Gold standard, therapeutic | Invasive, stroke risk ~0.5% |
| **Carotid Doppler** | Asymptomatic/TIA, carotid stenosis | Bedside, cheap | Operator-dependent, limited intracranial view |
| **TCD** | Vasospasm in SAH, PFO detection | Bedside, no contrast | Limited windows |

## 8. Laboratory / Serology

### First-line (every neurological patient)
- **FBC** (anaemia, infection, haematological malignancy)
- **U&E, glucose, Ca2+, Mg2+** (metabolic encephalopathy, seizures)
- **LFT, albumin** (hepatic encephalopathy, drug dosing)
- **TFT** (Hashimoto's encephalopathy, myopathy)
- **ESR, CRP** (vasculitis, infection, giant cell arteritis)
- **Vitamin B12, folate** (myeloneuropathy, dementia)
- **HIV, syphilis serology** (many presentations)

### Targeted
- **Autoimmune panel:** ANA, ENA, ANCA (vasculitis, lupus)
- **Anti-neuronal antibodies** (paraneoplastic, autoimmune encephalitis)
- **Anti-GM1, GQ1b** (GBS, MFS)
- **Anti-AChR, MuSK** (myasthenia)
- **Anti-NMO/MOG** (neuromyelitis optica)
- **CK, aldolase** (myopathy)
- **Ammonia, lactate, pyruvate** (metabolic encephalopathy)
- **Heavy metals, porphyrins** (suspected intoxication)

## 9. Genetic Testing

- **Huntington's:** CAG repeat expansion
- **Muscular dystrophy:** Dystrophin gene, FSHD, myotonic dystrophy
- **Channelopathies:** SCN1A (Dravet), KCNQ2 (BFNS)
- **Mitochondrial:** mtDNA mutations (MELAS, MERRF, LHON)
- **CADASIL:** NOTCH3
- **Genetic panels:** epilepsy, ataxia, spastic paraplegia, leukodystrophy
- **WES/WGS:** Diagnostic odyssey (after counselling)

## 11. Choice of Investigation — Summary Table

| Question | Best Test |
|----------|-----------|
| Acute stroke vs bleed? | CT head |
| Stroke — LVO candidate? | CTA |
| Acute infarct? | MRI DWI |
| Demyelination (MS)? | MRI brain + cord, CSF OCB |
| SAH? | CT (acute) → LP (≥12h) → CTA |
| Encephalitis? | MRI, LP (HSV PCR), EEG |
| Epilepsy focus? | MRI, video-EEG, PET, SPECT, MEG |
| Peripheral neuropathy? | NCS/EMG, bloods |
| Myopathy? | CK, NCS/EMG, MRI, biopsy, genetic |
| Dementia cause? | MRI, CSF, PET (amyloid/tau), DAT-SPECT |
| Spinal cord lesion? | MRI whole spine |
| Cavernoma? | MRI SWI |

## 12. Red Flags & Pre-Test Considerations

- **Pregnancy:** Avoid CT (radiation), gadolinium; prefer MRI (no gad in T1) and ultrasound
- **Renal impairment:** Caution with gadolinium (NSF) and iodinated contrast
- **Claustrophobia / pacemakers:** May preclude MRI
- **Anticoagulation:** Stop before LP (warfarin INR<1.4, DOAC 48h, LMWH 24h)
- **Coagulopathy:** Correct before LP or epidural
- **Allergy:** Iodinated contrast premedication; shellfish allergy NOT a contraindication

## 13. FCPS/MRCP High-Yield Summary

| Category | Key Points |
|----------|------------|
| **CT vs MRI** | CT = acute blood, bone, calcification; MRI = parenchyma, demyelination, posterior fossa |
| **DWI** | Most sensitive for acute ischaemia (minutes); also distinguishes abscess (restricted) from tumour |
| **LP timing in SAH** | Wait ≥12h after headache onset for xanthochromia to develop |
| **GBS CSF** | Albuminocytological dissociation: ↑protein, normal cells (after 1st week) |
| **MS CSF** | OCB in CSF, not serum (intrathecal synthesis); ↑IgG index |
| **NMO vs MS** | Anti-AQP4 (NMO); anti-MOG (MOGAD) — longitudinally extensive cord lesions |
| **Pregnancy & imaging** | No radiation (X-ray/CT); MRI without gad in T1; ultrasound preferred |
| **LP contraindications** | Raised ICP, mass lesion, coagulopathy, infection at site |
| **EEG in first seizure** | Routine EEG within 24-48h; if normal, does NOT exclude epilepsy |
| **PET in dementia** | FDG-PET: temporoparietal hypometabolism in AD; DAT-SPECT for DLB vs AD |

---

## 14. Special Situations

| Situation | Consideration |
|-----------|---------------|
| **Pregnancy** | MRI without gad preferred; defer CT; avoid gad (1st trimester) |
| **Renal impairment** | Avoid gad (NSF risk); hydrate before iodinated contrast |
| **Pacemaker** | MR-conditional devices only; older devices contraindicated |
| **Child** | MRI preferred (avoid radiation); sedation may be needed |
| **Anticoagulated** | LP requires INR<1.4, platelets >50, hold DOACs 48h, LMWH 24h |
| **Diabetes** | HbA1c for neuropathy; consider GBS, CIDP mimic |
| **HIV** | Consider opportunistic CNS infections, PML, primary CNS lymphoma |
| **Cancer patient** | Brain MRI with contrast (metastases, PML, paraneoplastic) |

## Viva Questions

1. **Q:** What does DWI show in acute stroke vs chronic infarct?
   **A:** Acute = bright (restricted diffusion); chronic = normal or bright on T2/FLAIR but no restricted diffusion.
2. **Q:** When is LP contraindicated?
   **A:** Raised ICP, mass lesion, coagulopathy, infection at site, suspected epidural abscess.
3. **Q:** What CSF pattern suggests GBS?
   **A:** Albuminocytological dissociation = ↑protein with normal cell count.
4. **Q:** How long after SAH onset should LP be performed for xanthochromia?
   **A:** ≥12 hours (bilirubin from RBC breakdown).
5. **Q:** MRI contraindication?
   **A:** Non-MRI-conditional pacemaker, cochlear implant, ferromagnetic aneurysm clips, metallic intraocular FB.

## Common Confusions / Exam Traps

| Confusion | Clarification |
|-----------|---------------|
| CT can miss early infarct | Use MRI DWI if <6h or posterior fossa |
| LP in SAH too early → negative | Wait ≥12h for xanthochromia |
| Gadolinium in pregnancy | Avoid (1st trimester) — risk to fetus |
| OCBs in CSF not serum | Intrathecal synthesis = MS pattern |
| Routine EEG "rules out" epilepsy | 50% yield; sleep-deprived/ambulatory improves |

---

## Mnemonics
1. **LOCALISE FIRST** — clinical localisation drives investigation choice
2. **STRUCTURE vs FUNCTION** — CT/MRI = anatomy; EEG/NCS = physiology
3. **CT MISSES** — **P**osterior fossa, **I**schaemia <6h, **S**mall demyelination, **M**eningitis early
4. **DO LP WHEN** — **D**iagnosis suspected, **O**pening pressure needed, **P**atients stable (no ICP)
5. **GAD** — **G**adolinium caution in **A**llergy, **D**ialysis (NSF risk)

## MCQs (10)

1. **Q:** Which MRI sequence is most sensitive for acute ischaemic stroke?
   **A:** FLAIR **B:** DWI **C:** T1 **D:** T2
   **Ans:** B — DWI detects cytotoxic oedema within minutes; FLAIR takes hours.

2. **Q:** CSF opening pressure >25 cmCSF in lateral decubitus position suggests?
   **A:** Normal **B:** IIH or hydrocephalus **C:** Hypovolaemia **D:** Spinal stenosis
   **Ans:** B — ICP >25 cmCSF = raised; think IIH (obese woman, papilloedema) or communicating hydrocephalus.

3. **Q:** Albuminocytological dissociation in CSF is characteristic of:
   **A:** MS **B:** GBS **C:** Bacterial meningitis **D:** SAH
   **Ans:** B — GBS: ↑protein, normal cells (after 1st week).

4. **Q:** CT head is preferred over MRI in which scenario?
   **A:** MS diagnosis **B:** Posterior fossa lesion **C:** Acute trauma **D:** Demyelination
   **Ans:** C — CT = fast, sensitive for acute blood/fracture; MRI better for parenchymal detail.

5. **Q:** LP in suspected SAH should be performed at what minimum time after onset?
   **A:** Immediately **B:** ≥6h **C:** ≥12h **D:** ≥48h
   **Ans:** C — Xanthochromia develops 12h after RBC breakdown.

6. **Q:** Which is a contraindication to MRI?
   **A:** Cochlear implant (non-MRI conditional) **B:** Pregnancy (2nd trimester) **C:** Diabetes **D:** Epilepsy
   **Ans:** A — Pacemakers, cochlear implants, ferromagnetic clips = absolute contraindications (unless conditional).

7. **Q:** CSF shows WCC 2000 (90% neutrophils), protein 4 g/L, glucose 1.0/serum 5.0. Diagnosis?
   **A:** Viral meningitis **B:** Bacterial meningitis **C:** GBS **D:** MS
   **Ans:** B — Neutrophilic pleocytosis + very low glucose + very high protein = bacterial.

8. **Q:** MRA is the best initial test for which condition?
   **A:** Acute SAH **B:** Carotid stenosis screening **C:** Cavernoma **D:** MS plaques
   **Ans:** B — MRA (or CTA) is non-invasive for carotid stenosis; cavernoma = MRI SWI.

9. **Q:** Anti-NMO (AQP4) antibody is associated with:
   **A:** MS **B:** Neuromyelitis optica **C:** Myasthenia gravis **D:** GBS
   **Ans:** B — AQP4 = NMO (longitudinally extensive cord, optic neuritis); MOG = MOGAD.

10. **Q:** DAT-SPECT is most useful to differentiate:
    **A:** AD vs FTD **B:** DLB vs AD **C:** VaD vs AD **D:** CJD vs AD
    **Ans:** B — Dopamine transporter SPECT shows reduced striatal uptake in DLB.

## SBA Questions (10)

1. **Scenario:** A 35-year-old woman presents with sudden onset right hemiparesis and aphasia, 2 hours ago. CT head is normal. Next step?
   **A:** MRI brain DWI **B:** Repeat CT in 24h **C:** LP **D:** EEG
   **Ans:** A — Acute stroke, CT may be normal early; MRI DWI detects ischaemia in minutes. CTA also needed to assess for LVO (thrombectomy candidate).

2. **Scenario:** A 70-year-old man with severe headache (worst ever), 6 hours. CT head normal. Next step?
   **A:** Discharge with analgesia **B:** MRI brain **C:** LP for xanthochromia **D:** EEG
   **Ans:** C — Thunderclap headache + normal CT (early) → LP. Wait ≥12h for optimal xanthochromia yield; at 6h CT is sensitive, but if clinical suspicion remains high, LP.

3. **Scenario:** A 25-year-old woman with bilateral lower limb weakness, areflexia, 5 days post gastroenteritis. CSF: protein 2.5 g/L, WCC 2. Diagnosis?
   **A:** MS **B:** GBS **C:** Transverse myelitis **D:** Spinal cord compression
   **Ans:** B — Post-infectious, ascending flaccid areflexic paralysis + albuminocytological dissociation = GBS. Treatment: IVIG.

4. **Scenario:** MRI brain in a 30-year-old woman with optic neuritis shows periventricular, juxtacortical, and spinal cord lesions. Best next CSF test?
   **A:** HSV PCR **B:** OCB + IgG index **C:** Anti-NMO **D:** Anti-GQ1b
   **Ans:** B — Lesions meet dissemination in space (McDonald); CSF OCB supports MS diagnosis (intrathecal IgG).

5. **Scenario:** A 60-year-old man with cognitive decline. MRI shows global atrophy, no vascular lesions. CSF: Aβ42 ↓, tau ↑. Diagnosis?
   **A:** DLB **B:** Alzheimer's disease **C:** FTD **D:** VaD
   **Ans:** B — CSF biomarkers (↓Aβ42, ↑total tau, ↑phospho-tau) support AD; MRI atrophy pattern (hippocampal) supports.

6. **Scenario:** A 45-year-old man with progressive proximal weakness, CK 5000, EMG myopathic. Next investigation?
   **A:** NCS only **B:** MRI muscle + consider biopsy **C:** LP **D:** DAT-SPECT
   **Ans:** B — Inflammatory myopathy suspected (polymyositis/IBM); MRI guides biopsy site, biopsy confirms.

7. **Scenario:** A 65-year-old with new-onset seizures. MRI brain shows multiple ring-enhancing lesions at grey-white junction. HIV+. Diagnosis?
   **A:** Primary CNS lymphoma **B:** Toxoplasmosis **C:** PML **D:** TB
   **Ans:** B — Multiple ring-enhancing + HIV = toxoplasmosis (empiric treatment; if no response, biopsy for PCNSL).

8. **Scenario:** LP for suspected meningitis. Opening pressure 35 cmCSF. CT shows mild cerebral oedema. What is the next step?
   **A:** Perform LP immediately **B:** Defer LP, treat empirically, get MRI **C:** Repeat CT in 24h **D:** Lumbar drain
   **Ans:** B — Raised ICP + mass effect = LP contraindicated (herniation risk). Treat empirically, defer LP or do after imaging clarifies.

9. **Scenario:** A 25-year-old woman with double vision, fatigable ptosis. Anti-AChR antibody negative. Next antibody test?
   **A:** Anti-MuSK **B:** Anti-NMO **C:** Anti-GM1 **D:** Anti-NMDA
   **Ans:** A — Seronegative MG (anti-AChR -ve) → test anti-MuSK (and LRP4). Treat with immunosuppression ± thymectomy.

10. **Scenario:** Young woman with stroke + livedo reticularis + recurrent miscarriages. Best antibody test?
    **A:** Anti-dsDNA **B:** Anti-cardiolipin / lupus anticoagulant **C:** ANCA **D:** Anti-Jo1
    **Ans:** B — Antiphospholipid syndrome: recurrent thromboses, miscarriages, livedo, stroke. Triple-positive antibodies confirm.

## Flashcards

- **Q:** Best MRI for acute stroke?
  **A:** DWI (bright within minutes)
- **Q:** LP contraindication triad?
  **A:** Raised ICP, Mass lesion, Coagulopathy
- **Q:** GBS CSF?
  **A:** Albuminocytological dissociation (↑protein, normal cells)
- **Q:** LP for SAH timing?
  **A:** ≥12h after headache onset (xanthochromia)
- **Q:** MRA vs CTA for stroke?
  **A:** Both useful; CTA faster in acute; MRA no iodinated contrast
- **Q:** What is DAT-SPECT for?
  **A:** Differentiate DLB (↓striatal uptake) from AD
- **Q:** MRI contraindications?
  **A:** Non-MRI-conditional pacemaker, cochlear implant, ferromagnetic clips
- **Q:** CSF OCBs in which disease?
  **A:** MS (intrathecal IgG synthesis)
- **Q:** Gadolinium risk in renal failure?
  **A:** Nephrogenic systemic fibrosis (NSF)
- **Q:** When is contrast-enhanced MRI preferred?
  **A:** Tumours, abscesses, MS activity, inflammation

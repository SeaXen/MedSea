# Neurological Emergencies Approach

> [!tip] **Time-critical neurological emergencies include:** stroke, status epilepticus, coma, raised ICP, meningitis/encephalitis, GBS, spinal cord compression, myasthenic crisis. Use **ABCDE + GCS + glucose + targeted emergency investigations**.

## Learning Objectives
- [ ] Apply ABCDE approach to neurological emergencies
- [ ] Perform and document GCS accurately
- [ ] Differentiate structural vs metabolic coma
- [ ] Identify time-critical presentations
- [ ] Initiate emergency management (stroke thrombolysis, SE protocol, meningitis treatment)
- [ ] Recognise when to escalate (ICU, neurosurgery)

---

## 1. Definition / Classification

### Definition
**Neurological emergency:** A presentation where delay in diagnosis or treatment leads to **irreversible neurological injury or death**. Examples: stroke (1.9M neurons/minute), status epilepticus, meningitis, raised ICP, cord compression, GBS with respiratory failure.

### Classification
| Category | Examples |
|----------|----------|
| **Vascular** | Ischaemic stroke, ICH, SAH, venous sinus thrombosis, dissection |
| **Seizure** | Status epilepticus, NCSE, post-ictal state |
| **Infection** | Bacterial meningitis, viral encephalitis, brain abscess, subdural empyema |
| **Inflammatory** | ADEM, NMOSD, transverse myelitis, autoimmune encephalitis |
| **Trauma** | TBI, spinal cord injury |
| **Neoplastic** | Raised ICP from tumour, spinal cord compression, leptomeningeal disease |
| **Toxic / Metabolic** | Coma (drugs, CO, hypoglycaemia), hyponatraemia, hepatic |
| **Neuromuscular** | GBS, myasthenic crisis, severe periodic paralysis |
| **Hydrocephalus** | Acute obstructive hydrocephalus |

## 2. Initial Approach — ABCDE

### Airway + Breathing
- Patent airway? Consider intubation if GCS ≤8, threatened airway
- O2 15L via non-rebreather mask
- Ventilate if apnoeic / inadequate

### Circulation
- IV access (×2 large bore), fluids
- BP, HR, ECG monitoring
- Correct hypotension (noradrenaline if needed)

### Disability
- **GCS** (Eye, Verbal, Motor — see below)
- **Pupils** (size, reactivity, asymmetry)
- **Glucose** (capillary + venous) — **NEVER MISS HYPOGLYCAEMIA**
- Brief focal neurological exam

### Exposure
- Temperature
- Skin (rash — meningococcal, needle marks — IVDU)
- Neck stiffness (meningism)

### Always
- **Capillary glucose** at the bedside
- If <4 mmol/L or unknown: **50 mL 50% dextrose IV + Thiamine 100 mg IV** (if alcohol/malnourished — give thiamine FIRST)

## 3. GCS (Glasgow Coma Scale)

### Components
| Component | Score | Best Response |
|-----------|-------|---------------|
| **Eye (E)** | 1-4 | Spontaneous (4) → no response (1) |
| **Verbal (V)** | 1-5 | Oriented (5) → no response (1) |
| **Motor (M)** | 1-6 | Obeys commands (6) → no response (1) |
| **Total** | 3-15 | |

### Eye (E)
- 4: Spontaneous
- 3: To voice
- 2: To pain
- 1: None

### Verbal (V)
- 5: Oriented
- 4: Confused
- 3: Inappropriate words
- 2: Incomprehensible sounds
- 1: None

### Motor (M)
- 6: Obeys commands
- 5: Localises pain
- 4: Withdraws from pain
- 3: Flexion to pain (decorticate)
- 2: Extension to pain (decerebrate)
- 1: None

### Key Points
- **Intubated patient:** V = 1T (tube); document as such
- **GCS ≤8:** Consider intubation
- **GCS ≤13:** Trauma → CT head
- **Track trends** — single value less useful than trajectory

## 4. Differential Diagnosis — Structural vs Metabolic

### Metabolic / Diffuse
- Symmetric pupils (small or normal, reactive)
- No focal deficit (early)
- Tremor, asterixis, multifocal myoclonus
- Examples: hypoglycaemia, drugs, hepatic/renal, CO2 narcosis

### Structural / Focal
- Asymmetric pupils, anisocoria, abnormal reactivity
- Focal deficit (hemiparesis, cranial nerve)
- Cushing's triad (HTN, bradycardia, irregular breathing) — raised ICP
- Examples: stroke, tumour, abscess, haemorrhage

### Coma Mimics
| Condition | Distinguishing Features |
|-----------|--------------------------|
| **Locked-in syndrome** | Quadriplegia, anarthria, vertical eye movements/blinking preserved |
| **Psychogenic (catatonia, conversion)** | Eye closure resistance, "blowing" eyelids, normal reflexes |
| **Akinetic mutism** | Severe frontal/bilateral lesion; awake, no speech/movement |
| **Vegetative state** | Sleep-wake cycles, no awareness |
| **Brain death** | Absent brainstem reflexes, apnoea |

---

## 5. Pupillary Patterns

| Pattern | Cause |
|---------|-------|
| **Small, reactive (miosis)** | Metabolic, pontine, opiates |
| **Bilateral fixed mid-size** | Midbrain lesion, severe anoxia |
| **Bilateral fixed dilated** | Severe anoxia, brain death, drugs (atropine) |
| **Unilateral fixed dilated** | CN III compression (uncal herniation), trauma |
| **Small reactive + Horner's** | Lateral medullary (Wallenberg), sympathetic lesion |
| **Nystagmus + brainstem signs** | Vestibular / brainstem |

## 6. Specific Emergencies

### Acute Stroke
- **Time of onset = last known well**
- **NIHSS** documentation
- **CT head** (rule out bleed)
- **IV thrombolysis (Alteplase / Tenecteplase)** within **4.5h** (3h if >80 years, anticoagulated, prior stroke + DM)
- **Mechanical thrombectomy** within **6h** (extended to 24h with CT perfusion selection for LVO)
- **BP <185/110** for thrombolysis; <180/105 for 24h after
- **Swallow screen** before oral intake
- **Stroke unit** admission

### Subarachnoid Haemorrhage (SAH)
- **Thunderclap headache** ± neck stiffness, photophobia, ↓consciousness
- **CT head** (sensitivity drops after 6h)
- **LP** for xanthochromia if CT negative (≥12h after onset)
- **CTA** for aneurysm
- **Treatment:** Endovascular coiling or surgical clipping; nimodipine (vasospasm prevention)
- Complications: re-bleed, hydrocephalus, vasospasm (day 4-14)

### Intracerebral Haemorrhage (ICH)
- **BP control** (<140 mmHg, but avoid rapid drop)
- **Reversal of anticoagulation:** Vitamin K + PCC (warfarin); Andexanet alfa or 4-factor PCC (DOACs); Idarucizumab (dabigatran)
- **Neurosurgery** for cerebellar ICH >3 cm, lobar with mass effect
- **Hematoma expansion** risk

### Bacterial Meningitis
- **Empirical antibiotics IMMEDIATELY** (within 1h) — don't delay for LP
- **Dexamethasone** (before or with antibiotics) — ↓ mortality (S. pneumoniae)
- **Empirical:** Ceftriaxone 2g IV 12H + Ampicillin 2g IV 4H (>50 yrs, immunocompromised) ± Vancomycin
- **LP** after stabilisation, CT first if signs of raised ICP

### Viral Encephalitis (HSV)
- **IV Acyclovir 10 mg/kg 8H** empirically for all suspected encephalitis
- **LP** for HSV PCR (and other viruses)
- **MRI** (temporal lobe changes in HSV)
- Delay >24h increases mortality

### Status Epilepticus
- See Status Epilepticus note
- **ABCDE → Glucose → Lorazepam 4mg IV → Levetiracetam 60 mg/kg → ICU + cEEG**

### Raised ICP
- **Signs:** Headache, vomiting, papilloedema, Cushing's triad, ↓GCS
- **Herniation signs:** CN III palsy (uncal), decerebrate posturing, Cushing's
- **Treatment:**
  - Head elevation 30°
  - IV Mannitol 0.5-1 g/kg OR Hypertonic saline (3% NaCl)
  - Hyperventilation (brief, bridge to surgery)
  - Definitive: Surgical decompression (haematoma, tumour)

### Spinal Cord Compression
- **Acute back pain + myelopathy** (weakness, sensory level, bladder/bowel)
- **MRI whole spine** (URGENT)
- **IV dexamethasone** (tumour-related compression)
- **Neurosurgical decompression** within 24h if neurological deficit

### Guillain-Barré Syndrome (GBS)
- **FVC, NIF** (respiratory monitoring)
- **20/30/40 rule:** FVC <20 mL/kg, NIF <30 cmH2O, FVC <40% predicted → ICU/intubation
- **IVIG 0.4 g/kg/day ×5** OR **Plasmapheresis 5 exchanges**
- **DVT prophylaxis**, autonomic monitoring

### Myasthenic Crisis
- **Acute neuromuscular respiratory failure** in MG patient
- **FVC, NIF, ABG**, swallow assessment
- **Intubation** if FVC <20 mL/kg or bulbar failure
- **Plasma exchange** (preferred) OR **IVIG**
- Avoid: aminoglycosides, fluoroquinolones, beta-blockers, magnesium
- Search for trigger (infection, drug change)

## 7. Investigations — Immediate

| Test | Indication |
|------|-----------|
| **Capillary glucose** | **All** |
| **ABG** (lactate, pCO2, O2) | All serious presentations |
| **FBC, U&E, LFT, glucose, Ca2+, Mg2+, PO4-** | All |
| **Coagulation, INR, APTTr** | Stroke, ICH, before LP |
| **Troponin** | Stroke mimics (MI) |
| **Ammonia** | Hepatic encephalopathy |
| **Toxicology** | Drug overdose, alcohol |
| **CK, urine myoglobin** | Rhabdo (prolonged SE, NMS) |
| **CT head** | Focal, trauma, anticoagulated, ↓GCS |
| **LP** | Meningitis/encephalitis, SAH |
| **MRI brain / spine** | Time-permitting |

## 8. Coma Algorithm

```
ABCs + IV + O2 + Monitor
  ↓
GLUCOSE — if low: D50 + Thiamine
  ↓
ABG, electrolytes, tox screen
  ↓
CT head (if focal, trauma, ↓GCS, anticoagulated)
  ↓
Look for signs of meningitis — LP if CT normal
  ↓
Empirical antibiotics/antivirals if suspected
  ↓
MRI / EEG if no diagnosis
  ↓
ICU monitoring, supportive care
```

## 9. When to Escalate

| Trigger | Action |
|---------|--------|
| **GCS ≤8 / threatened airway** | ICU, intubation |
| **Status epilepticus** | ICU, cEEG |
| **FVC <20 mL/kg / bulbar failure** | ICU, intubation (GBS, MG) |
| **Cushing's triad / herniation** | ICU, urgent neurosurgery |
| **Massive stroke with ↓consciousness** | Stroke unit / ICU |
| **Meningitis with shock** | ICU |
| **Cardiac arrest / ROSC** | Targeted temperature management, ICU |

## 10. FCPS/MRCP High-Yield Summary

| Category | Key Points |
|----------|------------|
| **ABCDE first** | Always, before neurological specifics |
| **Glucose first** | Hypoglycaemia mimics everything |
| **GCS ≤8** | Intubate |
| **Thiamine before glucose** | In alcohol/malnutrition |
| **Stroke onset = last known well** | Critical for thrombolysis |
| **IV TPA window** | 4.5h (Alteplase) |
| **Thrombectomy** | LVO up to 6h; up to 24h with CTP |
| **SAH** | CT + LP (≥12h) + CTA |
| **Bacterial meningitis** | Antibiotics <1h + dexamethasone |
| **HSV encephalitis** | IV acyclovir 10 mg/kg 8H |
| **SE** | T=5min BZD, T=20min 2nd line, T=40min anaesthetic |
| **GBS respiratory** | FVC <20 → ICU |
| **MG crisis** | PLEX (preferred) or IVIG |
| **Raised ICP** | Head up, mannitol, hypertonic saline |

## 11. Special Situations

| Situation | Consideration |
|-----------|---------------|
| **Pregnancy** | Consider eclampsia, cerebral venous thrombosis; lower threshold for MRI |
| **Children** | Febrile SE, meningitis, non-accidental injury |
| **Elderly** | Polypharmacy, falls, anticoagulants |
| **Anticoagulated** | Reversal for ICH; avoid LP |
| **Immunocompromised** | Toxoplasmosis, PML, fungal meningitis |
| **HIV** | Opportunistic CNS infections |
| **Spinal cord injury** | Cervical — respiratory failure; autonomic dysreflexia |

## Viva Questions

1. **Q:** Define GCS ≤8 and management implication.
   **A:** Coma (no eye opening, no comprehensible speech, no motor response to commands); intubate to protect airway.
2. **Q:** First investigations in a comatose patient?
   **A:** Glucose, ABG, electrolytes, tox screen, CT head, LP if febrile.
3. **Q:** When to give thrombolysis for stroke?
   **A:** Within 4.5h of last known well (Alteplase); exclusions include ICH, recent surgery, INR >1.7, BP >185/110.
4. **Q:** Empirical meningitis treatment in adults?
   **A:** Ceftriaxone 2g IV 12H + Dexamethasone (before/with); add Ampicillin if >50 yrs / Listeria risk.
5. **Q:** GBS ICU criteria?
   **A:** FVC <20 mL/kg, NIF <30 cmH2O, bulbar failure, autonomic instability.

## Common Confusions

| Confusion | Clarification |
|-----------|---------------|
| GCS 3 = brain death | GCS 3 ≠ brain death; need formal brainstem tests |
| Pupils reactive = no herniation | Pupillary signs may be late in central herniation |
| Thrombolysis = aspirin too | Thrombolysis within 24h excludes antiplatelets |
| Bacterial meningitis = wait for LP | Don't delay antibiotics for LP |
| GBS = progressive weakness only | Autonomic instability, respiratory failure are critical |

## Mnemonics

1. **ABCDE** — Airway, Breathing, Circulation, Disability, Exposure
2. **GCS ≤8 = INTUBATE** — protective airway
3. **GLUCOSE FIRST** — hypoglycaemia mimics stroke, SE, coma
4. **20/30/40** — FVC <20 / NIF <30 / <40% predicted in GBS = ICU
5. **4.5h** — thrombolysis window
6. **DOOR TO NEEDLE** — <60 min target for thrombolysis
7. **STATUS EPILEPTICUS T=5/20/40/60** — BZD / 2nd line / anaesthetic / SRSE

## SBA Questions (10)

1. **Scenario:** A 65-year-old with sudden right hemiparesis and aphasia, last well 2h ago. BP 180/95. CT normal. Best action?
   **A:** Aspirin 300 mg **B:** IV Alteplase (if no contraindications, BP <185/110) **C:** Aspirin 75 mg **D:** Wait
   **Ans:** B — Within window, normal CT; ensure BP <185/110 first.

2. **Scenario:** A 30-year-old with fever, headache, neck stiffness. CT normal. Best next step?
   **A:** LP for CSF analysis (after starting empirical antibiotics if any delay)
   **A:** **LP** for CSF (cell count, glucose, protein, culture, PCR). Empirical antibiotics if >30 min delay expected.
   **Ans:** B

3. **Scenario:** A 25-year-old with sudden severe headache, neck stiffness, normal CT head at 4h. Next step?
   **A:** Discharge **B:** LP for xanthochromia (≥12h after onset) **C:** Aspirin **D:** Triptan
   **Ans:** B — LP after 12h for xanthochromia; or CTA if clinical suspicion remains.

4. **Scenario:** A 60-year-old comatose patient with pinpoint pupils, RR 8. Likely cause?
   **A:** Pontine lesion / opiates **B:** SAH **C:** Seizure **D:** Hypoglycaemia
   **Ans:** A — Pinpoint reactive pupils = pontine or opiates; trial naloxone.

5. **Scenario:** GBS patient, FVC 18 mL/kg, NIF 25 cmH2O. Next action?
   **A:** Discharge **B:** ICU for respiratory monitoring / intubation; start IVIG **C:** Aspirin **D:** MRI brain
   **Ans:** B — FVC <20 / NIF <30 = ICU; consider intubation.

6. **Scenario:** Patient with sudden unilateral fixed dilated pupil + contralateral hemiparesis. Diagnosis?
   **A:** Diabetic CN III palsy (pupil spared) **B:** Uncal herniation (CN III compression) **C:** Migraine **D:** TIA
   **Ans:** B — Neurosurgical emergency.

7. **Scenario:** Myasthenic crisis with FVC 12 mL/kg, bulbar failure. Best treatment?
   **A:** IVIG only **B:** Plasma exchange (preferred for crisis) **C:** Pyridostigmine only **D:** Steroids only
   **Ans:** B — PLEX preferred for myasthenic crisis (faster onset).

8. **Scenario:** A patient with acute ICH on warfarin, INR 3.5. Reversal:
   **A:** Vitamin K + PCC (4-factor) **B:** Aspirin **C:** Heparin **D:** Tranexamic acid
   **Ans:** A — Vitamin K + 4-factor PCC (e.g., Beriplex).

9. **Scenario:** A 40-year-old with new-onset seizures, LP shows lymphocytic pleocytosis, normal glucose, negative HSV PCR, MRI FLAIR hyperintensity in medial temporal lobes. Best treatment?
   **A:** Antibiotics **B:** Acyclovir + immunotherapy (autoimmune encephalitis) **C:** Anticonvulsants only **D:** Aspirin
   **Ans:** B — Suspect autoimmune (LGI1, GABA-B, etc.); immunotherapy while antibody workup pending.

10. **Scenario:** A 50-year-old with sudden headache, ↓consciousness, BP 200/110, HR 50, irregular breathing. Diagnosis?
    **A:** SAH **B:** Raised ICP with Cushing's triad **C:** Stroke **D:** Status epilepticus
    **Ans:** B — Cushing's triad suggests raised ICP; treat with mannitol, head elevation, neurosurgery consult.

## High-Yield Exam Points (FCPS/MRCP)

| Topic | Key Facts |
|-------|-----------|
| **Type 3c diabetes** | Pancreatic exocrine insufficiency + diabetes · Low faecal elastase · No autoantibodies · Often insulin requiring |
| **Chronic pancreatitis** | Alcohol, smoking, genetic (PRSS1, SPINK1, CFTR) · Pain, malabsorption, diabetes late |
| **Pancreatic cancer** | New-onset diabetes (>50y, weight loss) as presentation · ↑CA19-9 · CT pancreas |
| **Cushing syndrome** | Cortisol excess → insulin resistance · Dexamethasone suppression test · Midnight cortisol |
| **Acromegaly** | GH/IGF-1 excess → insulin resistance · OGTT for GH suppression · Pituitary MRI |
| **Thyrotoxicosis** | ↑HGP, ↓insulin clearance → hyperglycaemia · Resolves with euthyroidism |
| **Phaeochromocytoma** | Catecholamines → ↑glycogenolysis, ↓insulin secretion · Plasma metanephrines · MIBG |
| **Glucagonoma** | Glucagon excess → hyperglycaemia, NME rash, weight loss, DVT · Plasma glucagon |
| **Steroid diabetes** | Post-prandial hyperglycaemia prominent · Morning dose → afternoon hyperglycaemia · NPH insulin or basal-bolus |
| **Antipsychotics** | Olanzapine, clozapine > risperidone > aripiprazole → weight gain, insulin resistance |
| **Calcineurin inhibitors** | Tacrolimus > ciclosporin → β-cell toxicity, insulin resistance · Monitor post-transplant |
| **ICI diabetes** | Anti-PD-1/PD-L1 → fulminant type 1, DKA at presentation · Check autoantibodies |
| **Thiazides/β-blockers** | Thiazide: hypokalaemia → ↓insulin secretion · β-blocker: ↓insulin secretion, mask hypoglycaemia |

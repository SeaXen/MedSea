# Thiazolidinediones (Pioglitazone)

> [!info]
> **Pioglitazone = PPAR-γ agonist**; ↑ insulin sensitivity in adipose, muscle, liver; **↓ HbA1c 0.5–1.0%**; **weight gain 2–4kg**, **fluid retention**, **HF contraindicated**, **fracture risk** (women), **bladder cancer** signal; **renal safe** (no dose adjustment); add-on if metformin/SU contraindicated; **avoid HF, active bladder cancer, high fracture risk**.

## 2. Mechanism of Action

| Step | Action |
|-------|--------|
| **1. Target** | **PPAR-γ** (Peroxisome Proliferator-Activated Receptor gamma) — nuclear receptor |
| **2. Activation** | Forms heterodimer with RXR → binds **PPRE** (PPAR Response Elements) on DNA |
| **3. Gene transcription** | ↑ Adiponectin, ↓ resistin, ↓ TNF-α, ↓ leptin; ↑ fatty acid oxidation, ↓ lipolysis |
| **4. Tissue effects** | **Adipose**: ↑ differentiation, ↑ lipid storage (subcutaneous > visceral); **Muscle**: ↑ GLUT4, ↑ insulin signalling; **Liver**: ↓ gluconeogenesis, ↓ lipid accumulation |
| **5. Net effect** | **↑ Insulin sensitivity** (peripheral > hepatic); ↓ glucose, ↓ TG, ↑ HDL |

---

## 3. Dosing & Pharmacokinetics

| Property | Pioglitazone |
|----------|--------------|
| **Starting dose** | **15mg OD** |
| **Titration** | ↑ to **30mg OD** after 4–8 weeks; **max 45mg OD** |
| **Onset** | **4–8 weeks** (gene transcription) |
| **Peak effect** | 12–16 weeks |
| **Half-life** | 16–24h (active metabolites) |
| **Renal impairment** | **No dose adjustment** (hepatic metabolism) |
| **Hepatic impairment** | **Contraindicated** (Child-Pugh B/C) |
| **Administration** | With or without food |

---

## 4. Efficacy

| Outcome | Effect |
|---------|--------|
| **HbA1c reduction** | **−0.5 to −1.0%** (monotherapy/add-on) |
| **Fasting glucose** | ↓ 2–3 mmol/L |
| **Post-prandial glucose** | ↓ 2–3 mmol/L |
| **Triglycerides** | ↓ 10–20% |
| **HDL** | ↑ 5–10% |
| **LDL** | ↑ 5–10% (large buoyant particles) |
| **Weight change** | **+2 to +4kg** (adipose expansion + fluid) |
| **Durability** | **Longest** of oral agents (ADOPT: durable to 4–5y) |

---

## 5. Adverse Effects & Major Risks

| Risk | Incidence | Mechanism | Management |
|------|-----------|-----------|------------|
| **Weight gain** | **2–4kg** (dose-related) | Adipocyte differentiation + fluid retention | Counsel; monitor; consider GLP-1/SGLT2i if problematic |
| **Fluid retention / Oedema** | **3–5%** (ankle, peripheral) | ↑ ENaC in collecting duct; ↑ vascular permeability | **Contraindicated in HF**; reduce dose; loop diuretic if mild |
| **Heart failure** | **↑ Hospitalisation 2–3x** | Fluid retention + direct cardiac effects | **ABSOLUTE CONTRAINDICATION** (NYHA I–IV) |
| **Fracture risk** | **↑ 2x** (women > men; distal limbs) | ↓ Bone formation, ↑ bone resorption; PPAR-γ in osteoblasts | Avoid in osteoporosis; DEXA monitor; calcium/vit D |
| **Bladder cancer** | **Signal in PROactive/Epic** (HR ~1.2–1.5) | Unclear; PPAR-γ in urothelium | **Avoid if active/prior**; counsel; urine cytology if haematuria |
| **Macular oedema** | Rare (<0.1%) | Fluid retention | Discontinue if visual changes |
| **Anaemia** | Mild (Hb ↓ 0.5–1g/dL) | Haemodilution (fluid retention) | Monitor; usually not clinically significant |

---

## 6. Contraindications

| Absolute | Relative |
|----------|----------|
| **NYHA Class I–IV Heart Failure** | **Osteoporosis / high fracture risk** |
| **Active bladder cancer** | **Prior bladder cancer** (caution) |
| **Hepatic impairment** (Child-Pugh B/C) | **Macular oedema** |
| **Pregnancy / Breastfeeding** | **Elderly women** (fracture risk) |
| **Type 1 DM / DKA** | **Macular oedema history** |

**Rosiglitazone**: **Withdrawn in EU/UK** (CV risk); not available.

## 7. Monitoring Requirements

| Parameter | Frequency |
|-----------|-----------|
| **HbA1c** | 3–6 monthly |
| **Weight & oedema** | Each visit |
| **BP** | Each visit |
| **LFTs** | Baseline, 3m, 6m, then annually |
| **Haemoglobin** | Baseline, annually (mild anaemia) |
| **Bone health** | DEXA if risk factors; calcium/vit D |
| **Bladder symptoms** | Ask re haematuria annually; urine dipstick if symptoms |

## 8. Position in Treatment Algorithm (NICE NG28 / ADA)

| Line | Role |
|------|------|
| **Add-on to metformin** | If **SU contraindicated** (hypo risk, renal impairment) |
| **Add-on to SU** | If **metformin contraindicated** (eGFR <30) |
| **Triple therapy** | Metformin + SU + **Pioglitazone** (if GLP-1/SGLT2i not suitable) |
| **Avoid in** | **HF, bladder cancer, fracture risk, hepatic impairment** |
| **Preferred over** | Sulfonylurea if **hypo risk high** (eGFR <30, elderly) |
| **Avoid if** | GLP-1 RA / SGLT2i available (superior CV/renal benefit) |

## 9. Key Trials

| Trial | Finding |
|-------|---------|
| **PROactive** (2005) | Pioglitazone ↓ macrovascular composite (MI, stroke, CV death) in T2DM + CVD; **↑ HF hospitalisation** |
| **ADOPT** (2006) | Pioglitazone **most durable** glycaemic control vs metformin/SU (4–5y); **↑ weight, oedema, fractures** |
| **IRIS** (2016) | Pioglitazone **↓ stroke/MI 24%** in insulin resistance + prior stroke/TIA (non-diabetic) |
| **Meta-analyses** | **Bladder cancer signal** (HR 1.2–1.5); **"Black box" warning** in US |

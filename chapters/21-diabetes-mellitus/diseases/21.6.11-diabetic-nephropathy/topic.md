# Diabetic Nephropathy (Diabetic Kidney Disease - DKD)

> [!info]
> **DKD = leading cause of ESRD** (40–50% of incident dialysis); KDIGO G+A staging; natural history: hyperfiltration → normoalbuminuria → microalbuminuria (A2) → macroalbuminuria (A3) → declining GFR → ESRD; **SGLT2i, GLP-1 RA, finerenone** = disease-modifying beyond ACEi/ARB.

## 2. Natural History (Mogensen Model - T1DM; Similar T2DM)

| Stage | GFR | Albuminuria | Clinical | Duration |
|-------|-----|-------------|----------|----------|
| **1. Hyperfiltration** | **↑↑** (>120–150) | Normal (A1) | Asymptomatic | Diagnosis – 5y |
| **2. Silent / Normoalbuminuric** | Normal/High | **A1 (<30 mg/g)** | Asymptomatic | 5–15y |
| **3. Incipient / Microalbuminuria** | Normal | **A2 (30–300 mg/g)** | **Earliest detectable** | 10–20y |
| **4. Overt / Macroalbuminuria** | **↓ Declining** | **A3 (>300 mg/g)** | Hypertension, oedema | 15–25y |
| **5. ESRD** | **<15 (G5)** | Variable | Uraemia, need RRT | 20–30y |

**T2DM**: More heterogeneous; **normoalbuminuric DKD** common (30–50% of DKD); may have retinopathy absent.

---

## 3. Pathophysiology

| Mechanism | Key Mediators |
|-----------|---------------|
| **Haemodynamic** | Hyperfiltration → glomerular hypertension → afferent/efferent arteriolar dysregulation |
| **Metabolic** | AGE formation, PKC activation, polyol pathway, hexosamine pathway → oxidative stress |
| **Inflammatory** | NF-κB, TGF-β, MCP-1, TNF-α → macrophage infiltration, fibrosis |
| **RAAS** | ↑ Ang II → efferent constriction → glomerular hypertension; direct profibrotic |
| **SGLT2-mediated** | Proximal tubular glucose reabsorption → ↑ distal NaCl delivery → macula densa → afferent dilatation |

## 4. KDIGO Staging (G + A) — Risk Matrix

### GFR Categories (G)
| G | eGFR (mL/min/1.73m²) |
|---|----------------------|
| **G1** | ≥90 |
| **G2** | 60–89 |
| **G3a** | 45–59 |
| **G3b** | 30–44 |
| **G4** | 15–29 |
| **G5** | <15 |

### Albuminuria Categories (A)
| A | ACR (mg/mmol) | ACR (mg/g) | Term |
|---|---------------|------------|------|
| **A1** | <3 | <30 | Normal/mildly increased |
| **A2** | 3–30 | 30–300 | Moderately increased (micro) |
| **A3** | >30 | >300 | Severely increased (macro) |

### Risk Matrix (Heat Map)
| | **A1** (Green) | **A2** (Yellow) | **A3** (Orange/Red) |
|---|---|---|---|
| **G1, G2** | Low | **Moderate** | **High** |
| **G3a** | Moderate | **High** | **Very High** |
| **G3b** | High | **Very High** | **Very High** |
| **G4, G5** | Very High | **Very High** | **Very High** |

## 5. Screening & Monitoring

| Test | Frequency | Threshold for Action |
|------|-----------|---------------------|
| **UACR** (first morning urine) | **Annual** (T1DM from 5y post-dx; T2DM from dx) | **A2: ACEi/ARB max, SGLT2i**; **A3: + finerenone, nephrology** |
| **eGFR** (CKD-EPI) | Annual (6m if <60) | **G3b: adjust meds**; **G4: nephrology**; **G5: RRT planning** |
| **BP** | Every visit | **<130/80** (KDIGO <120/80) |
| **HbA1c** | 3–6 monthly | Individualised target |

**UACR > eGFR for early detection** — A2 precedes GFR decline.

---

## 6. Disease-Modifying Therapy — The "Big Four"

| Drug Class | Agents | Key Trials | Indication in DKD |
|------------|--------|------------|-------------------|
| **ACEi / ARB** | Ramipril, losartan, etc. | RENAAL, IDNT, MICRO-HOPE | **First-line**; max tolerated dose; ↓ intraglomerular pressure |
| **SGLT2i** | Dapagliflozin, empagliflozin, canagliflozin | **CREDENCE, DAPA-CKD, EMPA-KIDNEY** | **eGFR ≥20** (dapa/empa); ↓ progression 30–40%; non-diabetic CKD too |
| **GLP-1 RA** | Semaglutide, dulaglutide, liraglutide | **FLOW, AWARD-7, LEADER** | **eGFR ≥15**; ↓ albuminuria, ↓ eGFR decline, CV benefit |
| **Finerenone** | Finerenone 10–20mg | **FIDELIO-DKD, FIGARO-DKD** | **eGFR ≥25, A2/A3**; non-steroidal MRA; monitor K+ |

**Additive benefit** — each class works via distinct pathways; **combination is standard**.

---

## 7. Management by KDIGO Stage

| Stage | Management |
|-------|------------|
| **G1–2, A1** | Optimise glucose/BP/lipids; annual screen; lifestyle |
| **G1–2, A2** | **ACEi/ARB max dose** + **SGLT2i** (if eGFR ≥20); BP <130/80; GLP-1 RA if T2DM |
| **G1–2, A3** | Above + **finerenone** (if eGFR ≥25); nephrology referral |
| **G3a, A2/A3** | Above + **adjust metformin** (max 1g BD); avoid NSAIDs; monitor K+/eGFR 3–6m |
| **G3b, A2/A3** | Above + **reduce/stop metformin** if <45; **SGLT2i still OK** (dapa/empa ≥20); nephrology co-care |
| **G4, any A** | Nephrology lead; **stop metformin**; adjust all renally-cleared drugs; RRT planning; avoid contrast |
| **G5** | RRT (HD/PD/transplant); palliative option |

## 8. Non-Diabetic Kidney Disease in Diabetes — When to Suspect

| Clue | Suggests |
|------|----------|
| **No retinopathy** (especially T1DM) | Non-diabetic (e.g., hypertensive nephrosclerosis, GN) |
| **Rapid GFR decline** (>5 mL/min/yr) | Non-diabetic / superimposed |
| **Active urine sediment** (RBC casts, dysmorphic RBC) | Glomerulonephritis |
| **Heavy proteinuria** (>3g/day) early in course | Nephrotic syndrome (MN, FSGS) |
| **Short diabetes duration** (<5y T1DM, <10y T2DM) with A3 | Unlikely DKD |
| **Systemic symptoms** (rash, arthralgia, haemoptysis) | Vasculitis, SLE, amyloidosis |

**Renal biopsy** if diagnosis uncertain — especially if immunosuppression considered.

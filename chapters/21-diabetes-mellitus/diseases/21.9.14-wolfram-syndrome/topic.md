# Wolfram syndrome

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise Wolfram syndrome (DIDMOAD)
- [ ] Identify WFS1 gene and inheritance
- [ ] Apply multidisciplinary management
- [ ] Counsel on prognosis and genetic counselling

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Acronym** | DIDMOAD: Diabetes Insipidus, Diabetes Mellitus, Optic Atrophy, Deafness |
| **Gene** | WFS1 (wolframin) - 4p16.1; also WFS2 (CISD2) |
| **Inheritance** | Autosomal recessive |
| **Prevalence** | 1:500,000 - 1:1,000,000 |

## 3. Clinical Features / Presentation
| Feature | Detail |
|--------|--------|
| **Diabetes Mellitus** | Onset 4-10y; insulin-requiring; non-autoimmune |
| **Diabetes Insipidus** | Central (pituitary); polyuria/polydipsia; onset ~14y |
| **Optic Atrophy** | Progressive; onset ~11y; colour vision loss; legal blindness by adulthood |
| **Deafness** | Sensorineural; high-frequency; onset ~13y |
| **Neurological** | Ataxia, peripheral neuropathy, autonomic dysfunction, psychiatric |
| **Urinary** | Neurogenic bladder; recurrent UTIs |
| **Endocrine** | Hypogonadism, growth hormone deficiency, hypothyroidism |

## 4. Classification / Staging / Grading

### WFS1 vs WFS2
| Gene | Features |
|-------|----------|
| **WFS1** (Wolframin) | Classic DIDMOAD; ER calcium handling |
| **WFS2** (CISD2) | Similar; also peptic ulcer disease, bleeding diathesis |

## 4. Classification / Staging / Grading

### Progression Timeline
| Age | Onset |
|-----|-------|
| **4-10y** | Diabetes Mellitus |
| **~11y** | Optic Atrophy |
| **~13y** | Deafness |
| **~14y** | Diabetes Insipidus |
| **Adolescence** | Neurological decline |

## 5. Diagnosis & Investigations
| Test | Finding |
|------|---------|
| **Genetic testing** | WFS1 mutation (biallelic); WFS2 (CISD2) |
| **Ophthalmology** | Optic atrophy (temporal pallor) |
| **Audiometry** | Sensorineural deafness |
| **Water deprivation test** | Central DI |
| **MRI brain** | Cerebellar/brainstem atrophy (late) |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **MIDD (m.3243A>G)** | Maternal inheritance; diabetes + deafness; no DI/OA |
| **MODY** | Autosomal dominant; no DI/OA/deafness |
| **Isolated DI** | No DM/OA/deafness |
| **Alström syndrome** | ALMS1; obesity, cardiomyopathy, hepatic steatosis |

## 7. Management
| System | Management |
|--------|------------|
| **Diabetes Mellitus** | Insulin (basal-bolus); tight control slows complications |
| **Diabetes Insipidus** | Desmopressin (DDAVP) oral/nasal; monitor electrolytes |
| **Optic Atrophy** | Low vision aids; regular ophthalmology |
| **Deafness** | Hearing aids; cochlear implant if severe |
| **Neurological** | Physiotherapy; orthotics for ataxia |
| **Hypogonadism** | Hormone replacement |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **Wolfram = MIDD?** | NO - Wolfram = AR, DIDMOAD; MIDD = maternal, no DI/OA |
| **All DI + DM = Wolfram?** | NO - need optic atrophy + deafness |
| **WFS1 = WFS2?** | WFS1 = classic; WFS2 = also peptic ulcer, bleeding |

# Bariatric Surgery Indications

> [!info]
> **NICE NG191**: BMI ≥35 + T2DM **OR BMI ≥30 + recent T2DM (<10y)**; **ADA**: BMI ≥30 + T2DM; **RYGB/Sleeve/OAGB** = main procedures; **75–85% remission**; **multidisciplinary assessment mandatory**; lifelong follow-up.

## 2. Indications for Surgery

### NICE NG191 (2023) — UK Standard
| Criterion | Threshold |
|-----------|-----------|
| **BMI + T2DM** | **BMI ≥35 kg/m²** + T2DM **OR** **BMI ≥30 kg/m²** + **recent T2DM (<10 years)** |
| **Ethnicity adjustment** | **South Asian/Chinese/Other Asian**: BMI thresholds **−2.5 kg/m²** (i.e., ≥32.5 / ≥27.5) |
| **Failed non-surgical** | Structured weight management (Tier 3) **≥12 months** |
| **Age** | **18–65** (consider >65 individually) |
| **Commitment** | Lifelong follow-up, nutritional supplementation |

### ADA 2024 / ASMBS — International
| Criterion | Threshold |
|-----------|-----------|
| **BMI + T2DM** | **BMI ≥30 kg/m²** + T2DM (no duration limit) |
| **BMI + Obesity complications** | BMI ≥35 (any); BMI 30–34.9 + comorbidity |
| **Asian adjustment** | BMI ≥27.5 + T2DM; BMI ≥25 + obesity complications |

## 3. Surgical Procedures — Comparison

| Procedure | Type | Remission (1y) | Remission (5y) | Mechanism | Key Risks |
|-----------|------|----------------|----------------|-----------|-----------|
| **RYGB** (Roux-en-Y) | Mixed (restrictive + malabsorptive) | **80–85%** | **50–60%** | Small pouch + biliopancreatic limb bypass → **GLP-1 ↑**, nutrient sensing, restriction | Marginal ulcer, internal hernia, dumping, micronutrient deficiency |
| **Sleeve Gastrectomy** (SG) | Restrictive + hormonal | **70–80%** | **40–50%** | **Ghrelin ↓** (fundus removed), GLP-1 ↑, restriction | Leak, stricture, reflux, weight regain |
| **OAGB** (One-Anastomosis / Mini-Gastric Bypass) | Mixed | **80–85%** | **50–60%** | Single anastomosis; simpler than RYGB | Bile reflux, marginal ulcer, micronutrient deficiency |
| **BPD/DS** (Biliopancreatic Diversion / Duodenal Switch) | Predominantly malabsorptive | **85–95%** | **60–70%** | Extensive bypass; greatest weight loss | Severe malabsorption, protein deficiency, frequent diarrhoea |
| **Adjustable Gastric Band** | Restrictive only | **40–50%** | **20–30%** | Adjustable restriction | Band erosion, slippage, port infection, **low remission** — **not recommended** for T2DM |

## 4. Preoperative Assessment (Multidisciplinary)

| Domain | Assessment |
|-------|------------|
| **Medical** | HbA1c, lipids, renal, liver, thyroid, iron, B12, folate, vitamin D, coagulation |
| **Cardiac** | ECG, echo if indicated (HF, valve disease) |
| **Respiratory** | Sleep study (OSA screening); CPAP if severe |
| **GI** | Endoscopy (H. pylori, Barrett's, hiatus hernia) |
| **Psychological** | Eating disorder screen (BED, night eating); depression/anxiety; capacity for consent |
| **Nutritional** | Diet history, protein intake, micronutrient baseline |
| **Anaesthetic** | Airway, Mallampati, VTE risk (Caprini score) |
| **Consent** | Procedure-specific risks, lifelong supplementation, follow-up commitment |

**Team**: Surgeon, physician/endocrinologist, dietitian, psychologist, anaesthetist, specialist nurse.

---

## 5. Preoperative Optimisation

| Target | Intervention |
|--------|--------------|
| **HbA1c** | **<64 mmol/mol (8.0%)** if possible (reduces infection risk) |
| **Weight loss** | **5–10% pre-op** (liver shrinkage diet: 800kcal 2 weeks) |
| **Smoking** | **Stop ≥6 weeks** (wound healing, anastomotic leak) |
| **Medications** | Stop SGLT2i (euglycaemic DKA risk); adjust insulin; stop anticoagulants per protocol |
| **Nutritional** | High protein (1.5g/kg); multivitamin; vitamin D if deficient |
| **VTE prophylaxis** | LMWH preoperative + extended post-discharge (28 days) |

---

## 6. Postoperative Diabetes Management — Immediate

| Timeframe | Glucose Management |
|-----------|-------------------|
| **Day 0–1** | **IV insulin infusion** (sliding scale) OR **SC basal + correction**; NPO → clear fluids |
| **Day 1–2** | Transition to **SC insulin**; reduce total dose **50–70%** (incretin effect, caloric restriction) |
| **Week 1–2** | **Stop most glucose meds** (except metformin if eGFR ok); monitor capillary glucose 4–6x/day |
| **Month 1** | **Off all glucose meds** if remission (HbA1c <48); continue monitoring |
| **Month 3** | Assess remission (HbA1c <48 off meds); nutritional review |

**Key**: **Rapid remission** often within days (incretin effect, caloric restriction, weight loss).

---

## 7. Long-Term Outcomes

| Outcome | RYGB | Sleeve | OAGB |
|---------|------|--------|------|
| **T2DM Remission (1y)** | 80–85% | 70–80% | 80–85% |
| **Remission (5y)** | 50–60% | 40–50% | 50–60% |
| **Weight loss (1y)** | 30–35% TBWL | 25–30% TBWL | 30–35% TBWL |
| **Weight regain (5y)** | 15–20% | 20–25% | 15–20% |
| **Microvascular benefit** | ↓ Retinopathy, nephropathy, neuropathy | Similar | Similar |
| **Macrovascular benefit** | ↓ MACE, mortality | Similar | Similar |
| **Mortality (30-day)** | 0.1–0.3% | 0.1–0.2% | 0.2–0.3% |

---

## 8. Lifelong Follow-Up Requirements

| Timeline | Assessments |
|----------|-------------|
| **6 weeks** | Wound, diet progression, glucose, nutritional (B12, iron, folate, D, Ca) |
| **3 months** | Full bloods, HbA1c, weight, diet adherence, supplementation |
| **6 months** | Full nutritional panel, DEXA (baseline), medication review |
| **Annual (lifelong)** | **Full bloods (FBC, U&E, LFT, iron, B12, folate, D, Ca, PTH, Cu, Zn, Se)**, HbA1c, weight, DEXA q2y, endoscopy if symptoms |
| **Supplementation (lifelong)** | **Multivitamin + B12 (1mg IM 3-monthly or oral 1mg daily) + Iron + Calcium/Vit D + Folate + Thiamine** |

| Topic | Key Point |
|-------|-----------|
| **NICE BMI threshold** | **≥35 + T2DM** OR **≥30 + recent T2DM (<10y)** |
| **Ethnicity adjustment** | **South Asian: −2.5 BMI** (threshold ≥32.5 / ≥27.5) |
| **ADA threshold** | **BMI ≥30 + T2DM** |
| **Best remission** | **RYGB/OAGB** 80–85%; **Sleeve** 70–80% |
| **Remission durability** | RYGB 50–60% at 5y; Sleeve 40–50% |
| **Mechanism** | **RYGB/OAGB**: incretin (GLP-1) + restriction + malabsorption; **Sleeve**: ghrelin ↓ + GLP-1 ↑ |
| **Pre-op liver shrink** | **800kcal 2 weeks** pre-op |
| **Smoking cessation** | **≥6 weeks** pre-op |
| **SGLT2i pre-op** | **STOP** (euglycaemic DKA risk) |
| **Lifelong supplements** | **B12, Iron, Ca/VitD, Folate, Thiamine, Multivitamin** |
| **Follow-up** | Annual lifelong nutritional monitoring mandatory |

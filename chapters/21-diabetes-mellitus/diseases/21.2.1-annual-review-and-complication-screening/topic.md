# Annual Review and Complication Screening

> [!info]
> **NICE 9 Care Processes** = core annual review; achieves QOF indicators; optimal diabetes care = all 9 completed; care plan agreed; structured education offered; complications screened.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] List all 9 NICE care processes
- [ ] Perform systematic annual review
- [ ] Interpret screening results and escalate
- [ ] Apply QOF diabetes indicators
- [ ] Create shared care plan

---

## 2. NICE 9 Care Processes (Essential Annual Checks)

| # | Care Process | Target / Action |
|---|--------------|-----------------|
| **1** | **HbA1c** | Individualised target (see Glycaemic targets); measure 3–6 monthly until stable, then 6-monthly |
| **2** | **Blood Pressure** | Per guideline targets (NICE <140/90, <130/80 if CKD/albuminuria) |
| **3** | **Lipids** (Total, HDL, LDL, non-HDL, TG) | Non-HDL <2.6 (primary) / <2.2 (secondary); statin per risk |
| **4** | **Urine ACR** (albumin:creatinine ratio) | Screen for DKD; ACR <3 (A1); if A2/A3 → ACEi/ARB max, SGLT2i, finerenone |
| **5** | **eGFR** (CKD-EPI) | Monitor CKD stage; adjust meds (metformin, SGLT2i, insulin) |
| **6** | **BMI** | Weight management; if ≥25 → lifestyle referral; if ≥30 → consider GLP-1/SGLT2i |
| **7** | **Smoking Status** | Document; offer cessation (NRT, varenicline, behavioural) |
| **8** | **Foot Examination** | **Neurovascular assessment** (10g monofilament, vibration, pulses); classify risk (IWGDF 1–3) |
| **9** | **Retinal Screening** | Digital photography; grading (R0–R3, M0–M1); annual (or 2-yearly if R0 x2) |

**All 9 completed** = high-quality care; NICE QS6 / QOF DM002–DM012.

## 3. Expanded Annual Review Checklist

| Domain | Assessments | Frequency |
|--------|-------------|-----------|
| **Glycaemic Control** | HbA1c, CGM metrics (TIR, TBR, GMI), hypoglycaemia history | HbA1c 6m; CGM each visit |
| **Cardiovascular** | BP, lipids, ECG (if symptoms), QRISK3/ASCVD, aspirin/statin review | Annual |
| **Renal** | UACR, eGFR, KP 2023 risk (if DKD) | Annual (6m if A2/A3 or eGFR<60) |
| **Feet** | Inspection, monofilament, vibration (128Hz), pulses (DP/PT), capillary refill, deformity, footwear | Annual (3–6m if high risk) |
| **Eyes** | Retinal photography (mydriatic); OCT if DMO suspected | Annual (2y if R0 x2) |
| **Neuropathy** | Symptom questionnaire (MNSI), 10g monofilament, vibration | Annual |
| **Psychosocial** | PHQ-9/GAD-7 (depression/anxiety), PAID/DDS (diabetes distress), eating disorder screen | Annual |
| **Sexual Health** | Erectile dysfunction (men), sexual dysfunction (women) | Ask annually |
| **Vaccinations** | Influenza (annual), Pneumococcal (PPV23), COVID, Hep B (if not immune) | Per schedule |
| **Medication Review** | Adherence, side effects, interactions, deprescribing (elderly), sick day rules | Each visit |
| **Lifestyle** | Diet, activity, alcohol, sleep, smoking | Each visit |
| **Pre-conception** | Folic acid 5mg, HbA1c <48, medication review, complication screen | If planning pregnancy |

## 4. Complication Screening Details

### Foot Risk Classification (IWGDF / Scottish)
| Risk Level | Findings | Action |
|------------|----------|--------|
| **1 (Low)** | Normal sensation, palpable pulses, no deformity | Annual screen; self-care education |
| **2 (Moderate)** | Neuropathy **OR** absent pulses **OR** deformity **OR** skin changes | 3–6 monthly podiatry; footwear assessment; self-care + professional care |
| **3 (High)** | Previous ulcer/amputation **OR** active Charcot **OR** critical ischaemia **OR** on dialysis | 1–3 monthly specialist MDT; urgent referral if active problem |

### Retinopathy Screening (ETDRS / English National Programme)
| Grade | Description | Referral |
|-------|-------------|----------|
| **R0** | No retinopathy | Routine annual |
| **R1** | Background NPDR (microaneurysms, dot/blot haemorrhages) | Annual |
| **R2** | Pre-proliferative (venous beading, IRMA, multiple haemorrhages) | Ophthalmology (6–12 weeks) |
| **R3** | Proliferative (new vessels, vitreous haemorrhage, preretinal haemorrhage) | **Urgent ophthalmology (2–4 weeks)** |
| **M0** | No maculopathy | Routine |
| **M1** | Maculopathy (exudates within 1DD of fovea, retinal thickening) | Ophthalmology (4–8 weeks) |

### DKD Screening (KDIGO)
| Test | Frequency | Action Threshold |
|------|-----------|------------------|
| **UACR** | Annual (6m if A2/A3) | A2 (3–30): ACEi/ARB max, SGLT2i; A3 (>30): + finerenone, nephrology |
| **eGFR** | Annual (6m if <60) | G3a (45–59): monitor; G3b (30–44): adjust meds; G4 (15–29): nephrology; G5 (<15): RRT planning |

## 5. Care Planning — Shared Decision Making

**NICE NG28**: Agree individualised care plan at each review covering:
- **Glycaemic target** (individualised)
- **Cardiovascular risk** (QRISK3, statin, aspirin, BP target)
- **Complication screening results** (traffic light system)
- **Medication changes** agreed
- **Lifestyle goals** (SMART)
- **Education referral** (if not completed/need refresher)
- **Next review date**
- **Patient copy** provided

## 6. QOF Diabetes Indicators (2024/25)

| Indicator | Description | Points |
|-----------|-------------|--------|
| **DM002** | Register of patients ≥17y with diabetes | 5 |
| **DM003** | HbA1c ≤58 mmol/mol (7.5%) | 17 |
| **DM004** | HbA1c ≤48 mmol/mol (6.5%) | 10 |
| **DM005** | BP ≤140/90 | 11 |
| **DM006** | Non-HDL <2.6 (or >40% reduction) | 6 |
| **DM007** | Statin prescribed (if CVD/QRISK3≥10%) | 5 |
| **DM008** | UACR checked | 3 |
| **DM009** | eGFR checked | 3 |
| **DM010** | Foot exam (neurovascular) | 3 |
| **DM011** | Retinal screen | 3 |
| **DM012** | Smoking status + cessation offer | 3 |

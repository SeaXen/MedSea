## High-Yield Exam Points (FCPS/MRCP)

| Topic | Key Facts |
|-------|-----------|
| **Screening** | ADA: age ≥35 or BMI ≥25 + risk factor · NICE: risk assessment + HbA1c/FPG · USPSTF: 35-70 BMI ≥25 |
| **Metformin** | 1st line · Start 500mg OD → 2g/day · Contraindicated eGFR <30 · B12 monitoring · Lactic acidosis rare |
| **Sulfonylureas** | ↑Insulin secretion · Hypoglycaemia (esp. elderly, CKD) · Weight gain · 2nd/3rd line |
| **DPP-4i** | Weight neutral · Low hypoglycaemia · Linagliptin no renal dose adjust · Neutral CV |
| **SGLT2i** | ↓Glucose reabsorption · CV/renal benefit · Genital mycotic infection · Euglycaemic DKA risk · eGFR thresholds |
| **TZDs** | Pioglitazone · ↑Insulin sensitivity · Fluid retention, HF, fracture, bladder cancer · 2nd/3rd line |
| **GLP-1 RA** | Semaglutide, liraglutide, dulaglutide · CV benefit · Weight loss · Nausea, pancreatitis risk |
| **Tirzepatide** | Dual GIP/GLP-1 · Superior HbA1c/weight vs semaglutide · SURPASS/SURMOUNT trials |
| **ADA/EASD 2023** | ASCVD/HF/CKD → SGLT2i/GLP-1 RA 1st line · Obesity → GLP-1 RA/tirzepatide · Cost/access considered |
| **NICE NG28** | Metformin → dual → triple → insulin · SGLT2i if ASCVD/HF/CKD · GLP-1 RA if BMI ≥35 |
| **Bariatric surgery** | BMI ≥35 (or ≥30 with comorbidities) · Remission rates 60-80% (RYGB/SG) |

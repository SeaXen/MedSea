# Alpha-Glucosidase Inhibitors

> [!info]
> **Acarbose (Glucobay) = reversible alpha-glucosidase inhibitor**; delays complex carbohydrate digestion in small intestinal brush border; ↓ post-prandial glucose excursion; **no hypoglycaemia as monotherapy**; **GI side effects** (flatulence, diarrhoea, abdominal pain) limit use; **rarely used in UK**; contraindicated IBD, obstruction.

## 2. Mechanism of Action

| Step | Action |
|------|--------|
| **1. Target** | **Alpha-glucosidase enzymes** (sucrase, maltase, isomaltase) on small intestinal brush border |
| **2. Inhibition** | **Reversible competitive inhibition** → delays oligosaccharide/disaccharide breakdown |
| **3. Result** | **Delayed glucose absorption** → ↓ post-prandial glucose peak; ↑ distal carbohydrate delivery → colonic fermentation |
| **4. Secondary** | ↑ GLP-1, GIP (distal nutrient delivery); ↓ triglycerides (delayed fat absorption) |

**No effect on insulin secretion or sensitivity** — works purely on carbohydrate digestion.

---

## 3. Pharmacokinetics

| Property | Acarbose |
|----------|----------|
| **Bioavailability** | **<2%** (minimal systemic absorption) |
| **Onset** | With first meal |
| **Peak effect** | 1–2 hours post-dose |
| **Duration** | Covers meal + 2–3h |
| **Elimination** | Faecal (>90%), renal (<2% as metabolites) |
| **Renal impairment** | **No dose adjustment** (minimal absorption) — but avoid if eGFR <25 (accumulation) |

---

## 4. Dosing & Titration

| Phase | Dose |
|-------|------|
| **Starting** | **25mg TDS** (with first bite of each main meal) |
| **Titration** | ↑ 25mg TDS every **4–8 weeks** based on HbA1c/PP glucose |
| **Target** | **50mg TDS** (standard); **max 100mg TDS** (200mg TDS in some countries) |
| **Administration** | **Swallow whole with first bite** of meal (or chew) |

**Important**: Must be taken **with first bite** — not before or after meal.

---

## 5. Efficacy

| Outcome | Effect Size |
|--------|-------------|
| **HbA1c reduction** | **−0.5 to −0.8%** (monotherapy) |
| **Post-prandial glucose** | ↓ **2–3 mmol/L** (major effect) |
| **Fasting glucose** | Minimal change |
| **Weight** | **Neutral or slight loss** (−0.5 to −1kg) |
| **Triglycerides** | ↓ 10–15% |
| **CVD outcomes** | No dedicated CVOT; ACE trial (acarbose in IGT) ↓ progression to T2DM, ↓ CV events |

---

## 6. Adverse Effects

| Adverse Effect | Incidence | Management |
|----------------|-----------|------------|
| **Flatulence** | **40–80%** (H2, CO2, CH4 from colonic fermentation) | Slow titration; low fibre initially; simethicone; usually improves 4–8w |
| **Diarrhoea** | 15–30% | Hydration; loperamide if needed; reduce dose |
| **Abdominal pain / distension** | 10–20% | Slow titration; smaller meals |
| **Hepatotoxicity** | Rare (<1%) — elevated transaminases | Monitor LFTs baseline, 3m, 6m; stop if >3x ULN |

**Side effects dose-related** — **slow titration essential**.

---

## 7. Contraindications & Cautions

| Contraindication | Reason |
|------------------|--------|
| **Inflammatory bowel disease** | Exacerbates diarrhoea, ulceration |
| **Intestinal obstruction / hernia** | Risk of obstruction (gas, distension) |
| **Significant chronic intestinal disease** | Malabsorption, short bowel |
| **Renal impairment eGFR <25** | Accumulation (minimal but theoretical) |
| **Pregnancy / Breastfeeding** | Safety not established |
| **Children** | Not studied |

---

## 8. Drug Interactions

| Interaction | Effect |
|-------------|--------|
| **Digoxin** | ↓ Digoxin absorption → monitor levels |
| **Other oral drugs** | May delay absorption → separate by 1–2h if critical |
| **Insulin / Sulfonylurea** | **No hypoglycaemia alone** — but **additive** if combined (reduce insulin/SU) |
| **Cholestyramine** | May bind acarbose → separate administration |

---

## 9. Special Use: Post-Bariatric Late Dumping (Reactive Hypoglycaemia)

| Indication | **Late dumping syndrome** (reactive hypoglycaemia post-RYGB/Sleeve) |
|------------|---------------------------------------------------------------------|
| **Dose** | **50–100mg TDS** with meals |
| **Mechanism** | Delays carbohydrate absorption → blunts insulin surge |
| **Alternative** | Octreotide (if acarbose fails) |

---

## 10. Position in Treatment Algorithm (NICE NG28 / ADA)

| Line | Role |
|------|------|
| **Monotherapy** | If metformin contraindicated/intolerant + **post-prandial hyperglycaemia dominant** |
| **Add-on** | Rarely used; **not in NICE pathway** (limited evidence, GI side effects) |
| **NICE** | Not in main pathway; specialist use only |
| **ADA** | Low-cost option if post-prandial focus; not preferred |

# Retinopathy Treatment (Laser, Anti-VEGF, Vitrectomy)

> [!info]
> **Treatment triad**: **PRP** (PDR gold standard), **Anti-VEGF** (CI-DMO first-line; PDR alternative), **Vitrectomy** (non-clearing VH, TRD, NVG); laser types: PRP (pan-retinal) vs focal/grid (macular); systemic optimisation essential adjunct.

## 2. Pan-Retinal Photocoagulation (PRP) — PDR Gold Standard

### Indications (ETDRS)
- **High-Risk PDR** (HRCs) — **urgent**
- **Early PDR** (no HRC) — within 4–6 months
- **Severe NPDR** (very severe, 4-2-1 rule) — consider if poor compliance/follow-up

### Parameters (ETDRS Protocol)
| Parameter | Value |
|-----------|-------|
| **Burn size** | **500 µm** |
| **Duration** | **0.1 second** |
| **Power** | **100–200 mW** (barely visible grey-white burn) |
| **Number of burns** | **1200–1600** (2–3 sessions) |
| **Pattern** | **Mid-periphery to equator**; spare macula (2 DD temporal), disc, major vessels |
| **Spacing** | 1 burn width apart |

### Efficacy (ETDRS)
- **50% reduction** in severe vision loss (<5/200) at 5 years
- **Regression of NVD/NVE** in 60–80%
- **Recurrence** if incomplete — repeat as needed

### Complications
| Complication | Incidence | Management |
|--------------|-----------|------------|
| **Peripheral visual field loss** | ~20–30% | May affect driving (DVLA standards) |
| **Night vision loss** | Common | Counsel; avoid night driving if severe |
| **DMO worsening** | **15–30%** | Anti-VEGF ± focal laser; usually transient |
| **Choroidal detachment** | Rare (<1%) | Conservative; cycloplegia |
| **Vitreous haemorrhage** | Rare | Observe; vitrectomy if non-clearing |
| **Pain** | During procedure | Topical anaesthetic; sub-Tenon's if extensive |

## 3. Focal/Grid Laser — Macular (Non-Centre DMO)

### Indications
- **Non-centre-involving DMO** (CSME outside 500µm from fovea)
- **Adjunct to anti-VEGF** (Protocol I: modest injection reduction)

### Parameters (ETDRS)
| Type | Parameters |
|------|------------|
| **Focal** (microaneurysms) | **50–100 µm**, 0.05–0.1s, 50–200mW; direct treatment of leaking MAs |
| **Grid** (diffuse leakage) | **50–100 µm**, 0.05–0.1s, 50–200mW; **2 burn widths apart**; avoid fovea (500µm) |

### Efficacy
- **50% reduction** in moderate visual loss (ETDRS)
- **Less used now** for CI-DMO (anti-VEGF superior)
- **Still first-line for non-centre-involving DMO**

## 4. Anti-VEGF Intravitreal Therapy

### Agents & Dosing

| Agent | Dose | Volume | Regimens |
|-------|------|--------|----------|
| **Aflibercept (Eylea)** | **2mg** | 0.05mL | **DMO**: 5 monthly → 8w → 12w (T&E); **PDR**: monthly ×3 → PRN/T&E |
| **Ranibizumab (Lucentis)** | 0.5mg | 0.05mL | **DMO**: 3 monthly → PRN/T&E; **PDR**: monthly ×3 → PRN |
| **Bevacizumab (Avastin)** | 1.25mg | 0.05mL | Off-label; same regimens; **cost-effective** |
| **Faricimab (Vabysmo)** | 6mg | 0.05mL | Dual Ang-2/VEGF; 16-week interval potential |

### Indications by Disease

| Disease | Anti-VEGF Role |
|---------|----------------|
| **CI-DMO** | **First-line** (aflibercept preferred if VA ≤6/12) |
| **Non-centre DMO** | Alternative if laser contraindicated |
| **PDR (HRC)** | **Alternative to PRP** (Protocol S, CLARITY); less field loss |
| **PDR (Early)** | Alternative; monitor for regression |
| **Pre-proliferative + DMO** | Treat both simultaneously |
| **Media opacity** (cataract/VH) | Only option (laser not possible) |

### DMO Protocols Comparison

| Protocol | Regimen | Key Feature |
|----------|---------|-------------|
| **PRN (Pro Re Nata)** | Monthly ×3 → treat if CST >300µm or VA drop ≥5 letters | Fewer injections; variable control |
| **T&E (Treat & Extend)** | Monthly ×3 → extend 2–4w if stable | **Preferred** — more consistent control; fewer visits |
| **Fixed interval** | Monthly / 8-weekly | High burden; guaranteed control |

## 5. Dexamethasone Implant (Ozurdex)

| Feature | Detail |
|---------|--------|
| **Dose** | **0.7mg** biodegradable implant |
| **Duration** | **4–6 months** sustained release |
| **Indications** | Anti-VEGF inadequate; **pseudophakic**; vitrectomised; non-compliant |
| **Advantages** | Long interval; no monthly visits; treats inflammation |
| **Complications** | **Cataract** (phakic), **IOP rise** (30% >25mmHg), endophthalmitis |

---

## 6. Vitrectomy — Indications & Timing

| Indication | Timing | Procedure |
|------------|--------|-----------|
| **Non-clearing vitreous haemorrhage** | **>3 months** (or <3m if dense + no view for PRP) | 23/25G PPV; core vitrectomy ± PRP (endolaser) ± gas |
| **Tractional RD (macula-on)** | **Urgent (within days)** | Delamination ± endolaser ± gas/silicone oil |
| **Tractional RD (macula-off)** | **Within 1 week** | As above; lower visual prognosis |
| **Combined TRD + rhegmatogenous** | **Urgent** | Buckle + vitrectomy ± tamponade |
| **Neovascular glaucoma (NVG)** | **Emergency** | PRP + IOP control → vitrectomy if angle closure |
| **Ghost cell glaucoma** | **Urgent** | Vitrectomy + washout ± tamponade |
| **Endophthalmitis** | **Emergency** | Tap + inject ± vitrectomy |

### Tamponade Agents
| Agent | Duration | Use |
|-------|----------|-----|
| **SF6 gas** | 2–3 weeks | Standard TRD |
| **C3F8 gas** | 6–8 weeks | Complex/recurrent TRD |
| **Silicone oil** | Months (removal 3–6m) | Complex TRD, proliferative vitreoretinopathy, patient non-compliant posturing |

---

## 7. Combined / Sequential Therapy

| Scenario | Approach |
|----------|----------|
| **PDR + CI-DMO** | **Anti-VEGF first** (treats both); add PRP if NVD persistent |
| **PDR + Non-centre DMO** | PRP + focal/grid laser |
| **Post-vitrectomy PDR** | Endolaser at surgery; anti-VEGF for residual DMO |
| **Anti-VEGF tachyphylaxis** | Switch agent (beva → aflibercept); add dexamethasone (pseudophakic); add laser |

---

## 8. Complications Summary

| Treatment | Key Complications |
|-----------|-------------------|
| **PRP** | Field loss, night blindness, DMO worsening, choroidal detachment |
| **Focal/Grid** | Paracentral scotoma (if too close to fovea), CNV (rare) |
| **Anti-VEGF** | Endophthalmitis (0.05%), IOP spike, cataract, thromboembolic (theoretical), tachyphylaxis |
| **Dexamethasone** | Cataract, IOP rise, endophthalmitis |
| **Vitrectomy** | Cataract progression (90% at 1y), retinal breaks, IOP issues, endophthalmitis, silicone oil emulsification |

| **Vitrectomy: VH** | Non-clearing >3 months |
| **Vitrectomy: TRD** | Macula-on urgent; macula-off within 1 week |
| **NVG** | Emergency: PRP + IOP control + anti-VEGF |

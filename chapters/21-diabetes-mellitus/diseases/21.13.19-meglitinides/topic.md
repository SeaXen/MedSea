# Meglitinides

> [!info]
> **Meglitinides (Repaglinide, Nateglinide) = rapid-acting insulin secretagogues**; **pre-meal dosing** (15min before); **short duration** (4–6h); ↓ post-prandial glucose; **hypoglycaemia risk**; **weight gain**; **renal adjustment**; alternative to sulfonylurea for **irregular meal patterns**; **rarely used UK** (cost, dosing frequency).

## 2. Agents

| Property | **Repaglinide** (Prandin) | **Nateglinide** (Starlix) |
|----------|---------------------------|---------------------------|
| **Receptor** | **SUR1** (distinct from SU site on K-ATP channel) | Same |
| **Onset** | **15–30 min** | **10–20 min** |
| **Peak** | 1 hour | 30–60 min |
| **Duration** | **4–6 hours** | **3–4 hours** |
| **Potency** | Higher | Lower |
| **Meal flexibility** | **Take 15 min before** each meal; omit if meal skipped | Same |

---

## 3. Mechanism of Action

| Step | Action |
|-------|--------|
| **1. Binding** | Binds **SUR1 subunit** of K-ATP channel (distinct site from sulfonylureas) |
| **2. Channel closure** | ↓ K+ efflux → membrane depolarisation |
| **3. Ca2+ influx** | Voltage-gated Ca2+ channels open |
| **4. Insulin release** | **Rapid, short-lived** insulin secretory burst |
| **Key difference from SU** | **Shorter duration**; **glucose-dependent** (less release at low glucose) |

---

## 4. Dosing & Titration

### Repaglinide
| Phase | Dose |
|-------|------|
| **Starting** | **0.5mg TDS** (pre-meal) |
| **Titration** | Double dose every **1–2 weeks** based on PP glucose |
| **Max** | **4mg TDS** (16mg/day) |
| **Renal adjustment** | **Child-Pugh A/B**: 0.5mg TDS start, max 4mg TDS; **Child-Pugh C**: avoid |

### Nateglinide
| Phase | Dose |
|-------|------|
| **Starting** | **60mg TDS** (pre-meal) |
| **Titration** | ↑ to **120mg TDS** if PP glucose not controlled |
| **Max** | **120mg TDS** |
| **Renal adjustment** | **No adjustment** (hepatic metabolism) — caution eGFR <30 |

**Administration**: **1–15 minutes before each main meal**; **omit if meal skipped**.

---

## 5. Efficacy

| Outcome | Repaglinide | Nateglinide |
|---------|-------------|-------------|
| **HbA1c reduction** | **−1.0 to −1.5%** | **−0.5 to −1.0%** |
| **PP glucose reduction** | **−3 to −4 mmol/L** | **−2 to −3 mmol/L** |
| **Fasting glucose** | Modest ↓ | Minimal |
| **Weight gain** | **1–3kg** | **1–2kg** |
| **Hypoglycaemia risk** | **Low–moderate** (glucose-dependent) | **Low** (lowest among secretagogues) |

---

## 6. Adverse Effects

| Adverse Effect | Incidence | Management |
|----------------|-----------|------------|
| **Hypoglycaemia** | **Low–moderate** (glucose-dependent; less than SU) | ✓ Omit dose if meal skipped; ✓ Educate on hypo symptoms |
| **Weight gain** | 1–3kg | Dietary counselling; consider GLP-1/SGLT2i if significant |
| **GI (nausea, abdominal pain)** | <5% | Take with food |
| **Liver enzymes** | Rare ↑ ALT/AST | Monitor baseline, 3m; stop if >3x ULN |
| **Upper respiratory infection** | <5% (repaglinide) | Symptomatic |

---

## 7. Contraindications & Cautions

| Condition | Repaglinide | Nateglinide |
|-----------|-------------|-------------|
| **Type 1 DM** | Contraindicated | Contraindicated |
| **DKA** | Contraindicated | Contraindicated |
| **Severe hepatic impairment** | **Contraindicated** | Caution (hepatic metabolism) |
| **Severe renal impairment** | Caution (no dose adjustment) | **No adjustment** |
| **Pregnancy** | Category C (avoid) | Category C (avoid) |
| **Gemfibrozil** | **Contraindicated** (↑ repaglinide 8x) | Caution |
| **CYP3A4 inhibitors** (azole, clarithromycin) | Dose reduce / avoid | Caution |
| **CYP2C8 inhibitors** (trimethoprim, montelukast) | Caution | **Contraindicated with gemfibrozil** |

---

## 8. Key Difference: Meglitinide vs Sulfonylurea

| Feature | **Meglitinide** | **Sulfonylurea** |
|---------|-----------------|------------------|
| **Dosing** | **Pre-meal (TDS)** | **Once/twice daily** |
| **Meal skipping** | **Omit dose** (no hypo) | **Hypo risk** if dose taken without meal |
| **Duration** | **Short (3–6h)** | **Long (12–24h)** |
| **Hypoglycaemia** | **Lower** (glucose-dependent) | **Higher** (prolonged) |
| **Weight gain** | Similar | Similar |
| **Renal adjustment** | Repaglinide: hepatic; Nateglinide: none needed | Most need adjustment (except gliclazide) |
| **Cost** | Higher | **Low (generic)** |
| **Use case** | **Irregular meals**, shift work, renal impairment | Regular meals, cost-sensitive |

## 9. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **Meglitinides** | Repaglinide, Nateglinide — rapid-acting secretagogues |
| **Dosing** | **Pre-meal (15min before)**; **omit if meal skipped** |
| **Repaglinide max** | **4mg TDS** (16mg/day) |
| **Nateglinide max** | **120mg TDS** |
| **Hypoglycaemia risk** | **Lower than SU** (glucose-dependent, short duration) |
| **Repaglinide hepatic** | Contraindicated severe hepatic impairment |
| **Repaglinide + gemfibrozil** | **Contraindicated** (↑ 8x AUC via CYP2C8/OATP1B1) |
| **Nateglinide renal** | **No dose adjustment** needed |
| **Indication** | **Irregular meals**, shift work, renal impairment (alternative to SU) |
| **UK use** | **Rarely used** (cost, TDS dosing, generic SU preferred) |

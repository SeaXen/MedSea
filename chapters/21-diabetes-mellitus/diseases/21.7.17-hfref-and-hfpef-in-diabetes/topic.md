# HFrEF and HFpEF in Diabetes

> [!info]
> **Diabetes = 2–5× HF risk**; **HFpEF more common in T2DM** (obesity, HTN, metabolic); **HFrEF also increased**; **diabetic cardiomyopathy** = metabolic + structural remodelling; **NT-proBNP lower in obesity/diabetes** — adjust thresholds; **SGLT2i/GLP-1 RA** = disease-modifying.

## 2. Epidemiology

| Parameter | Data |
|-----------|------|
| **HF prevalence in T2DM** | **15–25%** (vs 1–2% non-diabetes) |
| **Relative risk** | **2–5×** higher |
| **HFpEF proportion** | **50–60%** of HF in T2DM (vs 40–50% non-diabetes) |
| **HFrEF proportion** | 40–50% |
| **Mortality** | **2–3× higher** vs HF without diabetes |
| **Hospitalisation** | **3–4× higher** HF admission rate |

---

## 3. Diabetic Cardiomyopathy — Pathophysiology

| Mechanism | Features |
|-----------|----------|
| **Metabolic** | Impaired glucose oxidation → ↑ fatty acid oxidation → ↓ cardiac efficiency; insulin resistance; lipotoxicity |
| **Structural** | LV hypertrophy (concentric > eccentric), fibrosis (interstitial > replacement), diastolic dysfunction |
| **Microvascular** | Coronary microvascular dysfunction (↓ flow reserve) without epicardial CAD |
| **Autonomic** | CV autonomic neuropathy → resting tachycardia, exercise intolerance, silent ischaemia |
| **Inflammatory** | Systemic inflammation (IL-6, TNF-α, CRP) → myocardial inflammation |

**Diagnosis**: HF symptoms + diabetes + **no other cause** (CAD, HTN, valvular) + evidence of LV dysfunction/structure.

---

## 4. Diagnosis in Diabetes — NT-proBNP Nuances

### Standard NT-proBNP Cutoffs (ESC 2021)
| Setting | HFrEF | HFpEF |
|---------|-------|-------|
| **Acute** | >300 pg/mL | >300 pg/mL |
| **Chronic** | >125 pg/mL | >125 pg/mL |

### Diabetes/Obesity Adjustment
- **Obesity (BMI >35)**: NT-proBNP **30–50% lower** for same HF severity
- **Diabetes**: Independent NT-proBNP lowering effect
- **eGFR <60**: NT-proBNP ↑ (renal clearance)

**Practical approach**: Use **lower threshold** in diabetes/obesity (e.g., **>50–75 pg/mL chronic**); combine with **clinical assessment + echo**.

### Echo Parameters for HFpEF (HFA-PEFF / H₂FPEF Score)
| Parameter | Threshold |
|-----------|-----------|
| **E/e'** | **>14** (average) or **>15** (septal) |
| **LA volume index** | **>34 mL/m²** |
| **TR velocity** | **>2.8 m/s** |
| **RV dysfunction** | TAPSE <17mm, S' <10 cm/s |
| **Global longitudinal strain** | **>-16%** (impaired) |

## 5. Guideline-Directed Medical Therapy (GDMT) in Diabetes

### HFrEF (LVEF ≤40%) — **Four Pillars + SGLT2i**
| Class | Agents | Diabetes Consideration |
|-------|--------|------------------------|
| **ACEi/ARB/ARNI** | Sacubitril/valsartan (ARNI) preferred | **ARNI > ACEi** (PARADIGM-HF: ↓ HF hosp 21%, ↓ new diabetes) |
| **Beta-blocker** | Bisoprolol, carvedilol, nebivolol | **Carvedilol** may improve insulin sensitivity |
| **MRA** | Spironolactone, eplerenone | Monitor K+ (diabetes + CKD + MRA = hyperkalaemia risk) |
| **SGLT2i** | **Dapagliflozin, Empagliflozin** | **eGFR ≥20**; **Class I, Level A**; ↓ CV death/HF hosp 25–30% |
| **Vericiguat** | If symptomatic on GDMT | VICTORIA trial — ↓ CV death/HF hosp |

### HFpEF (LVEF >40%) — **Limited Evidence, SGLT2i + GLP-1 RA**
| Class | Agents | Evidence |
|-------|--------|----------|
| **SGLT2i** | **Dapagliflozin, Empagliflozin** | **DELIVER, EMPEROR-Preserved** — ↓ CV death/HF hosp 18–21% **regardless of EF** |
| **GLP-1 RA** | **Semaglutide** | **STEP-HFpEF, FIGHT** — ↓ symptoms, weight, NT-proBNP (symptomatic HFpEF, BMI ≥30) |
| **MRA** | Spironolactone | TOPCAT: neutral overall; ↓ hosp in Americas subgroup |
| **ARNI** | Sacubitril/valsartan | PARAGON-HF: neutral (trend in EF <57%) |
| **Diuretics** | Loop + thiazide | Symptom relief only |

## 6. Specific Diabetes Management in HF

| Issue | Recommendation |
|-------|----------------|
| **Metformin** | **Continue if eGFR ≥30** (historical lactic acidosis concern disproven) |
| **SGLT2i** | **Preferred** — HF + renal + glycaemic benefit; eGFR ≥20 |
| **GLP-1 RA** | **Consider** for HFpEF + obesity (semaglutide); no HF harm (FIGHT: neutral CV) |
| **DPP-4i** | **Avoid saxagliptin/alogliptin** (SAVOR/EXAMINE: ↑ HF hosp) |
| **TZDs** | **Contraindicated** (fluid retention → HF exacerbation) |
| **Sulfonylureas** | Neutral HF; avoid if hypoglycaemia risk |
| **Insulin** | Neutral HF; weight gain concern |

---

## 7. HFpEF in Diabetes — Specific Management

| Priority | Action |
|----------|--------|
| **1. SGLT2i** | Dapagliflozin/empagliflozin — **first disease-modifying therapy** for HFpEF |
| **2. Weight loss** | **Semaglutide** (STEP-HFpEF: ↓ symptoms, 6-min walk, weight, NT-proBNP) |
| **3. BP control** | <130/80; avoid excessive diuresis (preload dependence) |
| **4. Rate control** | Beta-blocker if AF; avoid excessive bradycardia |
| **5. Comorbidities** | Treat anaemia, sleep apnoea, thyroid, iron deficiency |
| **6. Exercise** | Cardiac rehab → improves VO₂ max, QoL |

---

## 8. Monitoring in HF + Diabetes

| Parameter | Frequency |
|-----------|-----------|
| **NT-proBNP** | 3–6 monthly (trend more than absolute) |
| **eGFR, K+** | 1–2w post SGLT2i/MRA/ARNI change; then 3–6m |
| **HbA1c** | 6 monthly (target individualised; avoid hypoglycaemia) |
| **Weight** | Each visit (fluid status + semaglutide effect) |
| **Echo** | 6–12 monthly if stable; sooner if decompensation |

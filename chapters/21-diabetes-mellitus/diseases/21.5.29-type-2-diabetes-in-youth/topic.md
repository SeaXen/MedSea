# Type 2 diabetes in youth

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise rising T2DM in youth (obesity-driven)
- [ ] Apply diagnostic criteria and screening (ADA: BMI >= 85th percentile + 2 RF; age >= 10 or puberty)
- [ ] Apply management: metformin 1st line; GLP-1 RA/SGLT2i if HbA1c > 75
- [ ] Manage comorbidities (PCOS, NAFLD, sleep apnoea, hypertension)

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Trend** | Rising with childhood obesity; 10-19y peak; T2DM > T1DM in some ethnic groups |
| **Risk Factors** | Obesity (BMI >= 95th percentile), FH T2DM, ethnicity (South Asian, African-Caribbean), acanthosis nigricans, PCOS, maternal GDM |
| **Screening** | ADA: age >= 10 or puberty + BMI >= 85th percentile + 2 risk factors |

## 3. Clinical Features / Presentation
| Feature | T2DM Youth | T1DM Youth |
|---------|------------|------------|
| **Obesity** | Usually present (BMI >= 95th percentile) | Absent/normal |
| **Acanthosis nigricans** | Common (insulin resistance marker) | Rare |
| **FH T2DM** | Strong (multiple relatives) | Variable |
| **Autoantibodies** | Negative | Positive (2+) |
| **C-peptide** | Preserved/high | Low/absent |
| **DKA at dx** | Rare | 25-30% |

## 4. Classification / Staging / Grading

### Screening Criteria (ADA)
| Criteria | Threshold |
|----------|-----------|
| **Age** | >= 10 years or onset of puberty |
| **BMI** | >= 85th percentile |
| **Risk Factors** | >= 2 of: FH T2DM, high-risk ethnicity, acanthosis nigricans, PCOS, maternal GDM, hypertension, dyslipidaemia |

### HbA1c Targets
| Age | Target |
|-----|--------|
| **All** | <53 mmol/mol (7.0%) if achievable without hypo |
| **If high risk** | <58 mmol/mol (7.5%) |

## 5. Diagnosis & Investigations
| Test | Role |
|------|------|
| **HbA1c / FPG / OGTT** | Diagnosis (ADA criteria) |
| **Autoantibodies** | Exclude T1DM (negative in T2DM) |
| **C-peptide** | Preserved in T2DM |
| **Lipids, LFT, UACR** | Baseline comorbidities |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **T1DM** | Autoantibodies +, low C-peptide, DKA common |
| **MODY** | <25y, non-obese, strong FH, negative autoantibodies, preserved C-peptide |
| **Secondary DM** | Steroid, Cushing, acromegaly, etc. |

## 7. Management

### Stepwise Algorithm
```mermaid
flowchart TD
    A[T2DM Youth] --> B[Lifestyle + Metformin]
    B --> C{HbA1c > 75mmol/mol (9%)?}
    C -->|Yes| D[Add GLP-1 RA (liraglutide 1.8mg) or SGLT2i (dapagliflozin 10mg)]
    C -->|No| E[Optimise metformin; lifestyle]
    D --> F{HbA1c target met?}
    F -->|No| G[Add insulin; basal-bolus]
    F -->|Yes| H[Continue + monitor]
    E --> H
```

### Lifestyle Intervention
| Component | Recommendation |
|----------|----------------|
| **Diet** | Reduced calorie; low GI; dietitian-led |
| **Exercise** | 60min/day moderate-vigorous |
| **Screen time** | <2 hours/day |
| **Family involvement** | Essential; whole-family approach |

### Comorbidity Screening (Annual)
| Condition | Screen |
|-----------|--------|
| **Dyslipidaemia** | Fasting lipid profile |
| **Hypertension** | BP at each visit |
| **NAFLD** | ALT, USS liver |
| **PCOS** | Irregular menses, hyperandrogenism |
| **Sleep apnoea** | Snoring, daytime sleepiness |
| **Depression** | PHQ-9 |

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **Screening** | Age >= 10 or puberty + BMI >= 85th percentile + 2 RF (ADA) |
| **Presentation** | Obesity, acanthosis nigricans, FH T2DM, negative Abs |
| **1st line** | Lifestyle + Metformin |
| **2nd line** | GLP-1 RA (liraglutide) or SGLT2i (dapagliflozin) if HbA1c > 75 |
| **Comorbidities** | PCOS, NAFLD, HTN, dyslipidaemia, sleep apnoea |
| **HbA1c target** | <53 (7.0%) if achievable without hypo |

## 9. Viva Questions
| Question | Expected Answer |
|----------|-----------------|
| **What are the screening criteria for T2DM in youth?** | Age >= 10 or puberty + BMI >= 85th percentile + 2 risk factors (FH, ethnicity, acanthosis, PCOS, GDM, HTN, dyslipidaemia) |
| **How does T2DM youth differ from T1DM?** | Obesity, acanthosis nigricans, negative Abs, preserved C-peptide, strong FH, no DKA |
| **What is 1st line therapy for T2DM in youth?** | Lifestyle + Metformin |
| **When do you add GLP-1 RA or SGLT2i in youth?** | HbA1c > 75mmol/mol (9%) on metformin |
| **What comorbidities to screen for?** | PCOS, NAFLD, HTN, dyslipidaemia, sleep apnoea, depression |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **T2DM youth = T1DM?** | NO - obesity, acanthosis, negative Abs, preserved C-peptide |
| **Metformin in children?** | Safe >10y; extend-release preferred for GI tolerance |
| **Insulin in T2DM youth?** | If HbA1c > 75 on metformin + GLP-1/SGLT2i or symptomatic |
| **Bariatric surgery in youth?** | Consider if BMI > 35 + comorbidity or BMI > 40; specialist centre |

**Mnemonic: T2DM-YOUTH-RISE**
- **T**2DM youth: rising with obesity
- **2** screening: age >= 10 or puberty + BMI > 85th percentile + 2 RF
- **D**iagnosis: obesity, acanthosis, FH, negative Abs
- **M**etformin 1st line
- **Y**outh obesity: BMI > 95th percentile
- **O**besity: acanthosis nigricans marker
- **U**S screening: ADA age >= 10 or puberty + BMI > 85th percentile + 2RF
- **T**ype 2 vs 1: obesity, acanthosis, neg Abs, preserved C-peptide
- **H**bA1c target: <53 (7.0%)
- **R**ising trend: obesity epidemic
- **I**nsulin if HbA1c > 75 on metformin + GLP-1/SGLT2i
- **S**creen comorbidities: PCOS, NAFLD, HTN, dyslipidaemia, apnoea
- **E**thnicity: South Asian, African-Caribbean higher risk

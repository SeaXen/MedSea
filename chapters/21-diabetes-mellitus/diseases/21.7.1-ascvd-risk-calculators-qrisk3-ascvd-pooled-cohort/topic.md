# ASCVD Risk Calculators (QRISK3, ASCVD Pooled Cohort)

> [!info]
> **QRISK3** = UK primary care (includes deprivation, ethnicity, CKD, RA, SLE, medications); **ASCVD Pooled Cohort** = US/ACC/AHA (race, lipid-focused); both ↑ risk with diabetes but neither fully captures diabetes duration/complications.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Compare QRISK3 vs ASCVD Pooled Cohort equations
- [ ] Identify variables unique to each calculator

- [ ] Communicate risk to patients effectively

## 2. QRISK3 (UK / NICE)

### Variables (20+)
| Traditional | QRISK3-Specific |
|-------------|-----------------|
| Age, sex, smoking, SBP, TC, HDL | **Townsend deprivation score** |
| Treated hypertension | **Ethnicity** (detailed: Indian, Pakistani, Bangladeshi, etc.) |
| Diabetes (T1/T2 + duration) | **CKD stage** (eGFR + ACR) |
| BMI | **Rheumatoid arthritis** |
| Family history premature CVD | **Systemic lupus erythematosus** |
|  | **Atrial fibrillation** |
|  | **Migraine** |
|  | **Corticosteroid use** |
|  | **Antipsychotic use** |
|  | **Severe mental illness** |
|  | **ED (erectile dysfunction)** |

### Diabetes in QRISK3
- **T1DM and T2DM separate variables**
- **Duration** of diabetes included
- **HbA1c** not directly in equation (but via diabetes variable)

### Output
- **10-year CVD risk %** (CHD + stroke + TIA)
- NICE CG181: **≥10% 10-year risk** → offer statin (atorvastatin 20mg)

## 3. ASCVD Pooled Cohort Equations (ACC/AHA 2013)

### Variables
| Variable | Notes |
|----------|-------|
| Age | 40–79 years |
| Sex | Male / Female |
| Race | **White / Black / Other** (coefficients differ) |
| Total cholesterol | mg/dL |
| HDL cholesterol | mg/dL |
| Systolic BP | mmHg |
| BP treatment | Yes/No |
| Diabetes | Yes/No (binary, no duration) |
| Smoking | Current smoker |

### Race Controversy
- Separate equations for Black vs White/Other
- **Black individuals**: Lower coefficients → lower calculated risk for same risk factors
- **2023 ACC/AHA**: Moving toward **race-free** equations (PREVENT)

### Output
- **10-year hard ASCVD risk** (fatal/nonfatal MI, fatal/nonfatal stroke)
- ACC/AHA 2018 Cholesterol Guideline: **≥7.5% 10-year risk** → moderate/high-intensity statin

## 4. Comparison Table

| Feature | QRISK3 | ASCVD Pooled Cohort |
|---------|--------|---------------------|
| **Population** | UK primary care (THIN/QResearch) | US cohorts (ARIC, CARDIA, CHS, FHS) |
| **Age range** | 25–84 | 40–79 |
| **Ethnicity** | Detailed (16 categories) | Binary (Black/White/Other) |
| **Deprivation** | **Townsend score** | No |
| **CKD** | eGFR + ACR | No |
| **Autoimmune** | RA, SLE | No |
| **Medications** | Steroids, antipsychotics | No |
| **Mental health** | SMI, migraine | No |
| **Diabetes detail** | T1/T2 + duration | Binary only |
| **Outcome** | CVD (CHD+stroke+TIA) | Hard ASCVD (MI+stroke) |
| **Statin threshold** | ≥10% (NICE) | ≥7.5% (ACC/AHA) |
| **Calibration** | UK population | US population |

## 5. Diabetes as Risk Enhancer — Beyond Calculators

**Neither calculator fully captures**:
- Diabetes duration
- HbA1c trajectory
- Microvascular complications (retinopathy, nephropathy, neuropathy)
- Macrovascular complications (prior CVD)
- CKD (in ASCVD eq)
- Insulin use vs oral agents
- Hypoglycaemia episodes

### ACC/AHA 2018 "Risk Enhancers" (Diabetes-Specific)
If 10-year risk **5–7.5%** (borderline) or **7.5–20%** (intermediate), these favour statin:
- **Diabetes duration ≥10 years** (T2DM) or **≥20 years** (T1DM)
- **Albuminuria** (ACR ≥30)
- **eGFR <60**
- **Retinopathy**
- **Neuropathy**
- **ABI <0.9**
- **Family history premature CVD**
- **hsCRP ≥2 mg/L**
- **Lp(a) ≥50 mg/dL**
- **APOB ≥130 mg/dL**

## 6. Practical Application in Diabetes

| Scenario | Calculator | Action |
|----------|------------|--------|
| **UK primary care** | **QRISK3** (mandated) | If ≥10% → atorvastatin 20mg |
| **US / international** | **ASCVD Pooled Cohort** | If ≥7.5% → moderate/high-intensity statin |
| **T1DM >40y or >10y duration** | **Don't rely on calculator** | **Automatic high-intensity statin** (NICE/ACC) |
| **CKD G3+** | QRISK3 includes; ASCVD doesn't | QRISK3 preferred |
| **Ethnic minority UK** | QRISK3 detailed ethnicity | QRISK3 required |
| **Patient communication** | Both give % | Use **"heart age"**, visual aids, shared decision-making |

---

## 7. Limitations & Future

| Limitation | Solution |
|------------|----------|
| **No diabetes duration in ASCVD** | Use risk enhancers; consider PREVENT (2023) |
| **Race in ASCVD** | Race-free equations in development |
| **HbA1c not in either** | Emerging models include HbA1c |
| **Dynamic risk** (treatment changes) | Lifetime risk models; serial calculation |
| **Young adults <40** | Not validated; use clinical judgement |
| **Frailty/elderly** | Not validated >79 (QRISK3) / >79 (ASCVD) |

## 8. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **QRISK3 unique variables** | Deprivation (Townsend), detailed ethnicity, CKD (eGFR+ACR), RA, SLE, steroids, antipsychotics, SMI, migraine, ED |
| **QRISK3 diabetes** | T1 vs T2 separate; duration included |
| **ASCVD Pooled Cohort** | Race-specific equations (Black/White/Other); no CKD, no deprivation |
| **Statin threshold QRISK3** | **≥10%** 10-year CVD risk (NICE CG181) |
| **Statin threshold ASCVD** | **≥7.5%** 10-year hard ASCVD risk (ACC/AHA) |
| **Diabetes as risk equivalent** | T1DM >40y or >10y duration; T2DM >40y → automatic high-intensity statin |
| **Risk enhancers** | Albuminuria, eGFR<60, retinopathy, neuropathy, ABI<0.9, family history, hsCRP≥2, Lp(a)≥50 |
| **HbA1c in calculators** | Not included in either |
| **Communication** | Use heart age, visual aids, shared decision-making |
| **Limitations** | <40y not validated; >79/84 not validated; frailty not captured |

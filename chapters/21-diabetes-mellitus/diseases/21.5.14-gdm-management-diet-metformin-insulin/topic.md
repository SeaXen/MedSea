# GDM Management (Diet, Metformin, Insulin)

> [!info]
> **Stepwise management**: Medical nutrition therapy (MNT) → Metformin (if fasting/pre-prandial not at target) → Insulin (if post-prandial not at target or metformin contraindicated/declined).

## 2. Glycaemic Targets in GDM (NICE NG3 / ADA)

| Target | NICE | ADA | Notes |
|--------|------|-----|-------|
| **Fasting** | **<5.3 mmol/L** | <5.3 mmol/L (95 mg/dL) | Key driver of macrosomia |
| **1-hour post-prandial** | **<7.8 mmol/L** | <7.8 mmol/L (140 mg/dL) | Preferred by ADA |
| **2-hour post-prandial** | **<6.4 mmol/L** | <6.7 mmol/L (120 mg/dL) | NICE stricter |
| **HbA1c** | <48 mmol/mol (6.5%) | <48 mmol/mol (6.5%) | If measured (2nd/3rd trimester) |
| **Pre-prandial** | — | <5.3 mmol/L | ADA |

**Continuous glucose monitoring**: TIR 3.5–7.8 mmol/L >70% (ATTD pregnancy targets).

---

## 3. Step 1: Medical Nutrition Therapy (MNT) — First Line

| Component | Recommendation |
|-----------|----------------|
| **Calories** | 30 kcal/kg ideal body weight (not actual); ~1800–2200 kcal/day |
| **Carbohydrate** | **175g/day minimum** (for fetal brain); 40–45% total calories |
| **Distribution** | 3 meals + 2–3 snacks (prevent nocturnal ketosis) |
| **Type** | Low GI, high fibre; avoid SSBs, refined carbs |
| **Monitoring** | SMBG 4–7x/day (fasting + 1h or 2h post each meal) |
| **Exercise** | 30 min moderate daily (walking post-meal ↓ glucose 1–2 mmol/L) |

**Failure criteria** (NICE): >2 fasting ≥5.3 OR >2 post-prandial ≥7.8 (1h) or ≥6.4 (2h) in 1 week → escalate.

---

## 4. Step 2: Metformin — First-Line Pharmacotherapy

| Aspect | Details |
|--------|---------|
| **Indication** | Fasting ≥5.3 mmol/L OR pre-prandial ≥ target despite MNT (NICE) |
| **Contraindications** | eGFR <30, acute illness (hypoxia risk), patient decline |
| **Starting dose** | 500 mg OD with evening meal |
| **Titration** | 500 mg weekly to 500 mg BD → 1g BD (max **2g/day** in pregnancy; 3g non-pregnant) |
| **Renal monitoring** | eGFR q4–6 weeks; stop if <30 |
| **Folic acid** | Continue 5mg daily (metformin ↓ folate absorption) |
| **B12** | Monitor annually if long-term |

**Evidence** (MiG trial, 2008; follow-up):
- Metformin vs insulin: Similar neonatal outcomes (LGA, hypoglycaemia, NICU)
- **Less maternal weight gain** (−2kg), **less hypoglycaemia**, **higher treatment satisfaction**
- **20–45% need supplemental insulin** (metformin alone insufficient)

**NICE preference**: Metformin before insulin for fasting hyperglycaemia.

---

## 5. Step 3: Insulin — When Metformin Fails/Contraindicated

### Indications for Insulin
- Post-prandial glucose not at target on metformin + MNT
- Fasting not at target on max metformin
- Metformin contraindicated/declined
- GDM diagnosed >34 weeks (insulin faster onset)
- Multiple gestation, macrosomia on scan, polyhydramnios

### Regimens

| Regimen | Indication | Dosing |
|---------|------------|--------|
| **Basal only** (NPH or detemir/glargine) | **Isolated fasting hyperglycaemia** | 0.2 u/kg at bedtime; titrate to fasting <5.3 |
| **Bolus only** (rapid analogue) | Isolated post-prandial hyperglycaemia | Per ICR (start 1u/10g carb); pre-bolus 10–15 min |
| **Basal-bolus** (MDI) | Both fasting + post-prandial high | Basal 0.2–0.3 u/kg + bolus per ICR |
| **CSII (pump)** | Intensive needs, pre-existing pump | Standard pump settings; tighter targets |

**Insulins safe in pregnancy**:
- **Rapid**: Aspart, Lispro, **Faster aspart (Fiasp)**, **Lispro-aabc (Lyumjev)** — Category B
- **Basal**: **Detemir**, **Glargine U100** — Category B (extensive data); Degludec limited data but used
- **Avoid**: NPH (less predictable), Glargine U300 (limited pregnancy data)

### Titration Protocol (Basal)
1. Start 0.2 u/kg (or 4–6u) at bedtime
2. Adjust by 10–20% every 2–3 days
3. Target fasting <5.3 mmol/L
4. If nocturnal hypo suspected → CGM or 3 AM check

### Titration Protocol (Bolus)
1. ICR start: 1 unit per 10–15g carb
2. Pre-bolus 10–15 min before meal
3. Target 1h <7.8, 2h <6.4
4. Adjust by 1 unit per 5g carb error

## 6. Delivery Timing (NICE/RCOG)

| Scenario | Timing | Mode |
|----------|--------|------|
| **GDM diet/metformin controlled, no complications** | **38–40 weeks** | Induction if not spontaneous |
| **GDM on insulin** | **37–38+6 weeks** | Induction |
| **Poor control / macrosomia / polyhydramnios** | **36–37 weeks** | Consider C-section if EFW >4.5kg |
| **Previous stillbirth / severe complications** | Individualised | Earlier delivery |

**Intrapartum management**:
- Stop SC insulin; **IV insulin sliding scale** (or continue basal insulin pump at 80% rate)
- Target 4–7 mmol/L (NICE) / 70–110 mg/dL (ADA)
- Dextrose 5% + insulin infusion per protocol

## 7. Postpartum Management

| Timing | Action |
|--------|--------|
| **Immediate** | Stop all glucose-lowering therapy (metformin/insulin) |
| **Before discharge** | Check fasting glucose (exclude persistent diabetes) |
| **6–12 weeks** | **75g OGTT** (not HbA1c — unreliable <3 months postpartum) |
| **Annual** | HbA1c or FPG (lifelong T2DM risk 50% at 10y, 70% at 20y) |
| **Breastfeeding** | Encouraged — improves maternal glucose, infant outcomes |
| **Contraception** | Discuss pre-pregnancy; avoid oestrogen if VTE risk |

---

## 8. Metformin in Breastfeeding

- **Safe** — minimal transfer to milk (<0.5% maternal dose)
- Continue if needed for postpartum glucose / PCOS
- Monitor infant for GI symptoms (rare)

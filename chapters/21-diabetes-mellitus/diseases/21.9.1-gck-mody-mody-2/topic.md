# GCK-MODY (MODY 2)

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise GCK-MODY clinical features
- [ ] Understand why no treatment is needed
- [ ] Avoid unnecessary treatment
- [ ] Recognise in pregnancy (GDM overlap)

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Gene** | GCK (glucokinase) |
| **Chromosome** | 7p15.3 |
| **Prevalence** | 2nd most common MODY (30-35% of MODY) |
| **Mechanism** | Glucokinase = pancreatic glucose sensor; heterozygous loss -> ↑glucose set-point |

## 3. Clinical Features / Presentation
| Feature | Detail |
|--------|--------|
| **Onset** | Neonatal/childhood; often asymptomatic |
| **Hyperglycaemia** | Mild fasting (5.5-8.0 mmol/L); stable lifelong |
| **HbA1c** | 39-58 mmol/mol (5.7-7.5%); stable |
| **Autoantibodies** | Negative |
| **C-peptide** | Preserved |
| **OGT** | Mild impairment (2h ~8-10 mmol/L) |

## 4. Classification / Staging / Grading
| Feature | GCK-MODY | HNF1A-MODY | T1DM | T2DM |
|---------|----------|------------|------|------|
| **Treatment** | **None** | Sulfonylurea | Insulin | Metformin ± others |
| **Hyperglycaemia** | Mild fasting (5.5-8.0) | Progressive | Severe | Variable |
| **Complications** | **Virtually none** | Microvascular | High | High |
| **Pregnancy** | **Treat only if maternal** | Treat with SU | Insulin | Metformin/insulin |

## 5. Diagnosis & Investigations
| Test | Finding |
|------|---------|
| **Fasting glucose** | 5.5-8.0 mmol/L (stable) |
| **HbA1c** | 39-58 mmol/mol (5.7-7.5%) |
| **2h OGTT** | Mild elevation |
| **Autoantibodies** | Negative |
| **C-peptide** | Preserved |
| **Genetic testing** | GCK heterozygous mutation |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **T1DM** | Autoantibodies +, severe hyperglycaemia |
| **T2DM** | Obese, progressive, insulin resistance |
| **HNF1A-MODY** | Progressive, low CRP, SU responsive |
| **Gestational DM** | Resolves postpartum; GCK may persist |

## 7. Management

### No Treatment Needed
| Principle | Detail |
|-----------|--------|
| **No glucose-lowering therapy** | Stable mild hyperglycaemia; no microvascular complications |
| **Annual monitoring** | HbA1c, fasting glucose |
| **Pregnancy** | Treat **only if maternal GCK mutation** (fetal risk of macrosomia) |
| **Insulin/SU** | **Contraindicated** - causes hypoglycaemia |

### Pregnancy Management
| Fetal Genotype | Management |
|----------------|------------|
| **Maternal GCK mutation** | Insulin if fetal GCK mutation NOT inherited (fetus can't compensate) |
| **Fetus inherits GCK mutation** | **No treatment** (fetus has same set-point) |

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **No treatment** | Stable mild hyperglycaemia; no complications |
| **Fasting glucose** | 5.5-8.0 mmol/L (stable lifelong) |
| **HbA1c** | 39-58 mmol/mol (5.7-7.5%) |
| **Complications** | **Virtually none** (no micro/macrovascular) |
| **Pregnancy** | Treat ONLY if mother has GCK mutation & fetus does NOT |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **All MODY need treatment?** | NO - GCK-MODY needs **no treatment** |
| **GCK-MODY = GDM?** | NO - GCK is genetic, lifelong; GDM resolves postpartum |
| **All MODY need insulin?** | NO - GCK needs none; HNF1A needs SU; HNF1B needs insulin |

# Monogenic diabetes in children

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Differentiate MODY from T1DM/T2DM in children
- [ ] Recognise neonatal diabetes (<6 months) and management
- [ ] Apply genetic testing criteria (probability > 6% cost-effective)
- [ ] Manage GCK-MODY (no treatment) vs HNF1A-MODY (sulfonylurea responsive)

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **MODY** | 1-2% of diabetes; autosomal dominant; onset < 25y |
| **Neonatal DM** | < 6 months; 1:100,000-500,000; KCNJ11/ABCC8 most common |
| **Prevalence** | ~100,000 MODY in UK; underdiagnosed |

## 3. Clinical Features / Presentation

### MODY Red Flags in Children
| Feature | MODY | T1DM | T2DM |
|---------|------|------|------|
| **Age** | < 25y (often < 15) | < 25y | > 10y |
| **BMI** | Non-obese | Normal/low | Obese |
| **FH** | **Autosomal dominant (3 generations)** | Variable | Variable |
| **Autoantibodies** | **Negative** | Positive (2+) | Negative |
| **C-peptide** | Preserved | Low/absent | Preserved/high |
| **Insulin requirement** | Often not needed (GCK) or SU responsive (HNF1A) | Immediate | Variable |

### Neonatal Diabetes
| Feature | Transient (TNDM) | Permanent (PNDM) |
|---------|------------------|-----------------|
| **Remission** | By 18 months | Never |
| **Relapse** | Adolescence/adulthood | Never |
| **Genetics** | 6q24 imprinting (70%) | KCNJ11, ABCC8, INS |
| **Sulfonylurea** | Variable | **KCNJ11/ABCC8: 90% responsive** |

## 4. Classification / Staging / Grading

### Common MODY Types
| Type | Gene | Features | Treatment |
|------|------|----------|-----------|
| **MODY 2 (GCK)** | GCK | Mild fasting hyperglycaemia (5.5-8.0); stable | **No treatment needed** |
| **MODY 3 (HNF1A)** | HNF1A | Progressive; low renal threshold; low CRP | **Sulfonylurea (excellent response)** |
| **MODY 1 (HNF4A)** | HNF4A | Macrosomia, neonatal hypoglycaemia | Sulfonylurea responsive |
| **MODY 5 (HNF1B)** | HNF1B | Renal cysts, GU malformations | Often insulin needed |

## 5. Diagnosis & Investigations
| Test | Indication |
|------|------------|
| **Autoantibodies** | Negative in MODY/neonatal |
| **C-peptide** | Preserved |
| **Genetic panel** | If probability > 6% |
| **Targeted testing** | If FH suggests specific gene |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **T1DM** | Autoantibodies +, low C-peptide, DKA common |
| **T2DM** | Obese, older, no FH of autosomal dominant |
| **Transient hyperglycaemia** | Stress, steroids; resolves |

## 7. Management

### MODY-Specific
| Type | Management |
|------|------------|
| **GCK-MODY (MODY2)** | **No treatment**; monitor HbA1c annually |
| **HNF1A-MODY (MODY3)** | **Sulfonylurea (gliclazide MR)**; excellent response |
| **HNF4A-MODY (MODY1)** | Sulfonylurea responsive |
| **HNF1B-MODY (MODY5)** | Often insulin; monitor renal/GU |

### Neonatal Diabetes
| Gene | Management |
|------|------------|
| **KCNJ11/ABCC8** | **Switch to sulfonylurea** (tolbutamide/gliclazide 0.5-1mg/kg/day) |
| **6q24 (TNDM)** | Insulin until remission; monitor for relapse |
| **INS** | Insulin |
| **ECF2AK3 (Wolcott-Rallison)** | Insulin; multi-system |

### Genetic Counselling
| Aspect | Detail |
|--------|--------|
| **Inheritance** | Autosomal dominant (MODY); mostly de novo (neonatal) |
| **Prenatal** | Available if familial mutation known |
| **Family screening** | Offer to relatives |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **All MODY = insulin?** | NO - GCK = no treatment; HNF1A/4A = SU responsive; HNF1B = often insulin |
| **Neonatal DM = T1DM?** | NO - < 6 months; KCNJ11/ABCC8 -> sulfonylurea responsive |
| **All neonatal DM permanent?** | NO - transient (TNDM) remits by 18mo, relapses later |

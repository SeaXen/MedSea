# Diabetic nephropathy vs other proteinuric diseases

## 1. Learning Objectives
- [ ] Differentiate DKD from other causes of proteinuria in diabetes
- [ ] Recognise red flags for non-diabetic kidney disease
- [ ] Select appropriate investigations (immunology, complement, biopsy)
- [ ] Apply biopsy indications in diabetes

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Clinical dilemma** | ~30% of proteinuric diabetes have non-DKD component |
| **Biopsy yield** | Non-diabetic pathology in 20–30% of biopsied diabetics with proteinuria |
| **Key principle** | **Not all proteinuria in diabetes = DKD** |

## 3. Clinical Features / Presentation
| Red Flag for Non-DKD | Significance |
|----------------------|--------------|
| **Rapid eGFR decline** | >5 mL/min/1.73m²/year |
| **Active urinary sediment** | RBC >10/µL, RBC casts, WBC casts |
| **No diabetic retinopathy** | Especially if DM >10y or A3 proteinuria |
| **Systemic symptoms** | Fever, rash, arthralgia, haemoptysis, neuropathy |
| **Low BMI at diagnosis** | Type 2 DM usually overweight |
| **Family history** | Hereditary kidney disease (Alport, ADPKD, Fabry) |
| **Short DM duration** | <5y with heavy proteinuria |

## 4. Classification / Staging / Grading

### Common Non-DKD Proteinuric Diseases in Diabetes

| Disease | Typical ACR | Key Features | Investigations |
|---------|-------------|--------------|----------------|
| **Hypertensive nephrosclerosis** | A1–A2 | Long-standing HTN; no retinopathy; bland sediment | Clinical; exclude DKD |
| **Chronic GN (IgA, membranous, FSGS)** | A2–A3 | Haematuria, RBC casts; may have low complement | Autoantibodies, complement, **biopsy** |
| **Amyloidosis** | A3 (nephrotic) | Nephrotic syndrome; systemic (cardiac, neuropathy, macroglossia) | **Congo red +ve**; FLC, SAP scan |
| **Multiple myeloma** | A3 (Bence Jones) | CRAB features; Bence Jones proteinuria | **Urine protein electrophoresis**; FLC, bone marrow |
| **Diabetic AIN** | Variable | Drug-induced (NSAIDs, PPI, antibiotics); eosinophilia | Drug history; **biopsy** |
| **Obstructive uropathy** | Variable | Hydronephrosis on US; post-void residual | US bladder/kidneys |
| **Alport syndrome** | A2→A3 | FH haematuria/renal failure; hearing loss, ocular | Genetic (COL4A5); biopsy |
| **Fabry disease** | A2→A3 | Angiokeratomas, acroparaesthesia, cornea verticillata | α-galactosidase A; genetic (GLA) |

## 5. Diagnosis & Investigations

### Stepwise Workup for Atypical Proteinuria
| Step | Investigation | Purpose |
|------|---------------|---------|
| **1** | **Urine microscopy** | RBC casts = glomerulonephritis; WBC casts = AIN/pyelonephritis |
| **2** | **Autoantibodies** | ANA, dsDNA, ANCA, anti-GBM, C3, C4 |
| **3** | **Serum FLC + electrophoresis** | Myeloma, amyloidosis (AL) |
| **4** | **Complement C3/C4** | Low C3 = post-strep, MPGN, lupus; Low C4 = cryoglobulinaemia |
| **5** | **Viral serology** | HBV, HCV, HIV (membranous, MPGN) |
| **6** | **Renal ultrasound** | Size, obstruction, cysts (ADPKD) |
| **7** | **Renal biopsy** | If diagnosis unclear or immunosuppression considered |

### Biopsy Indications in Diabetes
| Indication | Criteria |
|------------|----------|
| **Rapid decline** | eGFR drop >5 mL/min/yr |
| **Active sediment** | RBC >10/µL, RBC/WBC casts |
| **No retinopathy** | DM >10y with A3 proteinuria |
| **Systemic features** | Fever, rash, arthralgia, haemoptysis |
| **Short duration** | DM <5y with heavy proteinuria |
| **Immunosuppression considered** | Before starting steroids/rituximab/etc. |

## 6. Differential Diagnosis Table

| Feature | DKD | Non-DKD Proteinuria |
|---------|-----|---------------------|
| **Retinopathy** | Usually present (if DM>10y) | Often absent |
| **Proteinuria onset** | Gradual (years) | May be acute/subacute |
| **Haematuria** | Rare | Common (RBC casts = GN) |
| **eGFR decline** | Slow (2–4 mL/min/yr) | Rapid (>5 mL/min/yr) |
| **Systemic symptoms** | Absent | May be present |
| **Biopsy (if done)** | Kimmelstiel-Wilson, GBM thickening | Immune deposits, amyloid, myeloma casts |

## 7. Management
| Scenario | Action |
|---------|--------|
| **Strong DKD features** | Continue DKD management (ACEi/ARB, SGLT2i, etc.) |
| **Red flags present** | Complete workup (above); consider biopsy |
| **Biopsy-proven non-DKD** | Treat specific disease (immunosuppression for GN, chemo for myeloma, etc.) |
| **Mixed DKD + non-DKD** | Treat both; DKD management + disease-specific |

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **Red flags** | Rapid eGFR decline, active sediment, no retinopathy, systemic symptoms, FH |
| **Urine microscopy** | RBC casts = GN; WBC casts = AIN/pyelonephritis |
| **Biopsy indications** | Rapid decline, active sediment, no retinopathy, systemic features, short DM duration |
| **Common non-DKD** | Hypertensive nephrosclerosis, GN (IgA, membranous), amyloidosis, myeloma |
| **Not all proteinuria in DM = DKD** | ~20–30% have non-diabetic component |

## 9. Viva Questions
| Question | Expected Answer |
|----------|-----------------|
| **What are the red flags suggesting non-diabetic kidney disease in a diabetic with proteinuria?** | Rapid eGFR decline (>5 mL/min/yr), active urinary sediment (RBC casts), absent retinopathy (esp if DM>10y), systemic symptoms, family history, short DM duration with heavy proteinuria |
| **When do you biopsy a diabetic with proteinuria?** | Rapid decline >5mL/min/yr, active sediment, absent retinopathy with heavy proteinuria, systemic features suggesting systemic disease, DM duration <5y with A3 proteinuria |
| **What does RBC casts in urine indicate?** | **Glomerulonephritis** — not DKD (DKD typically has bland sediment) |
| **How does hypertensive nephrosclerosis differ from DKD?** | Long-standing HTN, bland sediment, no retinopathy, usually A1–A2 proteinuria |
| **What investigation confirms amyloidosis?** | **Congo red staining (apple-green birefringence)** on biopsy; serum free light chains, SAP scan |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **All proteinuria in DM = DKD?** | NO — 20–30% have non-diabetic component; always check red flags |
| **No retinopathy = no DKD?** | If DM >10y and no retinopathy with heavy proteinuria → think non-DKD |
| **Biopsy always needed?** | NO — only if red flags or immunosuppression considered |

**Mnemonic: DDx-PROTEINURIA-DM**
- **D**KD red flags: rapid eGFR decline, active sediment, no retinopathy
- **D**M duration: <5y + heavy proteinuria = biopsy
- **x** (exclude): non-DKD causes
- **P**roteinuria: not all in DM = DKD
- **R**BC casts = GN (not DKD)
- **O**phthalmoscopy: no retinopathy + DM>10y = red flag
- **T**ubular: WBC casts = AIN/pyelonephritis
- **E**lectrophoresis: myeloma, amyloidosis
- **I**mmunology: ANA, ANCA, anti-GBM, C3, C4
- **N**ephrotic syndrome: consider amyloid, myeloma, membranous
- **U**ltrasound: size, cysts, obstruction
- **R**enal biopsy: if red flags or immunosuppression considered
- **I**gA nephropathy: common GN in DM
- **A**myloidosis: Congo red, FLC
- **M**yeloma: Bence Jones, CRAB
- **D**iabetic AIN: drugs (NSAIDs, PPI, abx)
- **N**o retinopathy + proteinuria = think non-DKD

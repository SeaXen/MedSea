# Mononeuropathies and Radiculoplexus Neuropathy

> [!info]
> **Mononeuropathy = single nerve injury** (compression/ischaemia); **Cranial III palsy** (pupil-sparing) classic; **Diabetic amyotrophy (radiculoplexus)** = severe pain → weakness/wasting; **Mononeuropathy multiplex** = vasculitis until proven otherwise; **Carpal tunnel** common (median at wrist).

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Identify common diabetic mononeuropathies (cranial, limb)
- [ ] Diagnose diabetic amyotrophy (radiculoplexus neuropathy)
- [ ] Differentiate mononeuropathy multiplex from vasculitis
- [ ] Manage compression neuropathies (carpal tunnel, ulnar)
- [ ] Apply prognosis and recovery timelines

## 2. Cranial Nerve Palsies

| Nerve | Frequency | Presentation | Key Feature |
|-------|-----------|--------------|-------------|
| **III (Oculomotor)** | **Most common** | Acute **diplopia**, **ptosis**, **eye down/out** | **Pupil-sparing** (parasympathetic fibres peripheral → spared in ischaemia) |
| **VI (Abducens)** | Common | **Horizontal diplopia** (worse on lateral gaze), esotropia | No ptosis, pupil normal |
| **IV (Trochlear)** | Rare | **Vertical diplopia** (worse on downgaze/reading), head tilt | Superior oblique palsy |
| **VII (Facial)** | Uncommon | **Bell's palsy** (LMN face weakness); **Rule out stroke** (forehead spared in UMN) | Hyperacusis, taste loss (ant 2/3) |

**Diabetic III palsy**: **Pupil-sparing** = **ischaemic microvascular**; **Pupil-involving** = compressive (aneurysm, tumour) → **urgent imaging**.

**Recovery**: 8–12 weeks (most); if not → re-image.

---

## 3. Limb Mononeuropathies (Compression / Ischaemia)

| Nerve | Site | Presentation | Provocative Test |
|-------|------|--------------|------------------|
| **Median (Carpal Tunnel)** | **Wrist** (carpal tunnel) | **Nocturnal paraesthesia** (thumb/index/middle), thenar wasting, weak abduction | **Phalen's** (wrist flexion 60s), **Tinel's** (wrist) |
| **Ulnar** | **Elbow** (cubital tunnel) | Medial hand paraesthesia, **Froment's sign** (adductor pollicis weak), thenar wasting (late) | Tinel's at elbow; elbow flexion test |
| **Radial** | **Spiral groove** (humerus) | **Wrist drop**; sensory loss dorsal hand (snuffbox) | Tinel's spiral groove |
| **Common Peroneal** | **Fibular neck** | **Foot drop**; sensory loss lateral leg/dorsum foot | Tinel's fibular neck |
| **Femoral** | **Inguinal ligament / pelvis** | **Quadriceps wasting**, knee buckling, sensory loss anteromedial thigh | Hip extension weakness; **diabetic amyotrophy overlap** |
| **Lateral Cutaneous Nerve of Thigh** | **Inguinal ligament (meralgia paresthetica)** | **Burning/numbness** anterolateral thigh | Tinel's at ASIS |

**Diabetes ↑ risk** for all compression neuropathies (nerve susceptibility + oedema).

## 4. Diabetic Amyotrophy (Radiculoplexus Neuropathy / Bruns-Garland)

| Feature | Details |
|---------|---------|
| **Other names** | Proximal diabetic neuropathy, lumbosacral radiculoplexus neuropathy |
| **Demographics** | **Older T2DM** (mean 60y); often **recent diagnosis** or **poor control**; weight loss |
| **Onset** | **Severe, asymmetric pain** (hip, thigh, buttock) → **weakness + wasting** (quadriceps, glutei) |
| **Progression** | Pain peaks at 4–12 weeks → weakness plateau → slow recovery (6–24 months) |
| **Signs** | Quadriceps wasting, knee jerk reduced/absent, **hip flexion weak**, sensory loss anteromedial thigh |
| **CSF** | **Raised protein** (often >1g/L), normal cells (albuminocytological dissociation) |
| **MRI** | **STIR/T2 hyperintensity** in lumbosacral plexus; nerve root enhancement |
| **NCS/EMG** | **Active denervation** (fibrillations) in L2–L4 myotomes; reduced recruitment |

**Pathophysiology**: **Microvasculitis / immune-mediated** (not pure ischaemia) — inflammatory infiltrate in nerve biopsy.

---

## 5. Diabetic Amyotrophy Management

| Step | Intervention |
|------|--------------|
| **1. Pain control** | **Pregabalin/gabapentin/duloxetine**; opioids short-term if severe |
| **2. Immunotherapy** (if early/severe) | **IVIG** 2g/kg over 5 days; **IV methylprednisolone** 1g × 3–5 days → oral taper; **PLEX** (refractory) |
| **3. Physiotherapy** | **Quadriceps strengthening**, gait training, falls prevention |
| **4. Glycaemic control** | Optimise HbA1c; no specific evidence for immunotherapy benefit, but standard |
| **5. Prognosis** | **Incomplete recovery** common (residual weakness/wasting); 50% walk unaided at 2y |

---

## 6. Mononeuropathy Multiplex

| Feature | Significance |
|---------|--------------|
| **Definition** | **≥2 non-contiguous nerves** affected sequentially or simultaneously |
| **In diabetes** | **Think systemic vasculitis** (PAN, ANCA, cryoglobulinaemia, rheumatoid) |
| **Workup** | **ESR, CRP, ANCA, ANA, cryoglobulins, complement, hepatitis B/C, HIV, SPEP, urine Bence Jones** |
| **Nerve biopsy** | **Vasculitic changes** (necrotising inflammation, vessel wall destruction) |
| **Treatment** | **Immunosuppression** (steroids + cyclophosphamide/rituximab/azathioprine) per vasculitis protocol |

**Key**: Isolated mononeuropathies in diabetes = common; **multiplex = vasculitis until proven otherwise**.

---

## 7. Recovery Timelines

| Neuropathy Type | Typical Recovery |
|-----------------|------------------|
| **CN III / VI / IV palsy** | **8–12 weeks** (pupil-sparing) |
| **Facial nerve (Bell's)** | 6–12 weeks (85% full recovery) |
| **Carpal tunnel (mild)** | Weeks-months (splint/injection); surgery if persistent |
| **Ulnar at elbow** | Months (conservative); surgery if wasting |
| **Peroneal / Femoral** | 3–12 months (often incomplete) |
| **Diabetic amyotrophy** | **6–24 months** (often residual deficit) |

## 8. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **CN III palsy in diabetes** | **Pupil-sparing** = microvascular ischaemia; pupil-involving = compressive (aneurysm) |
| **CN VI palsy** | Horizontal diplopia, esotropia, no ptosis |
| **Carpal tunnel** | Nocturnal paraesthesia thumb/index/middle; Phalen's +ve, Tinel's wrist |
| **Diabetic amyotrophy** | Severe hip/thigh pain → quadriceps wasting/weakness; older T2DM; CSF protein ↑ |
| **Amyotrophy CSF** | Albuminocytological dissociation (protein >1g/L) |
| **Amyotrophy treatment** | Pain control + **IVIG/steroids/PLEX** (early); physio |
| **Mononeuropathy multiplex** | **Vasculitis** until proven otherwise — full vasculitis screen |
| **Recovery CN III** | 8–12 weeks; if persistent → re-image |
| **Femoral neuropathy** | Quadriceps wasting, knee buckling; overlaps with amyotrophy |
| **Meralgia paresthetica** | Lateral cutaneous nerve thigh; burning anterolateral thigh |

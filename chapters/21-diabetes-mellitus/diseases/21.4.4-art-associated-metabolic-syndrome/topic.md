## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Definition** | ART-induced metabolic disturbances: dyslipidaemia, insulin resistance, lipodystrophy, diabetes |
| **Prevalence** | 20-50% on ART; PI > INSTI > NNRTI |
| **Risk** | PI (lopinavir, atazanavir) > INSTI (dolutegravir, bictegravir) > NNRTI |
| **Mechanism** | PI: ↓GLUT4, ↑lipogenesis; INSTI: weight gain, ↑adiposity |

## 3. Clinical Features / Presentation
| Syndrome Component | ART Association | Features |
|--------------------|----------------|----------|
| **Dyslipidaemia** | PI (lopinavir > atazanavir) | ↑TG, ↑LDL, ↓HDL; ↑apoB |
| **Insulin resistance** | PI, INSTI (dolutegravir, bictegravir) | ↑Fasting glucose, HOMA-IR |
| **Lipodystrophy** | PI (older), NRTI (stavudine, zidovudine) | Lipoatrophy (face, limbs); lipohypertrophy (abdomen, buffalo hump) |
| **Weight gain** | INSTI (dolutegravir, bictegravir) | ↑BMI, central adiposity |
| **Diabetes** | PI, INSTI, cumulative ART exposure | New-onset T2DM; ↑HbA1c |

| ART Class | Agents | Dyslipidaemia | Insulin Resistance | Weight Gain | Lipodystrophy |

## 7. Management

### ART Optimisation
```mermaid
flowchart TD
    A[ART Metabolic Syndrome] --> B{Identify culprit drug}
    B --> C{PI-based?}
    C -->|Yes| D[Switch to INSTI (DTG/BIC) if viral suppression]
    C -->|No| E{INSTI with weight gain?}
    E -->|Yes| F[Consider switch (RPV/DOR) if suppressed]
    E -->|No| G[Add metabolic Rx]
    G --> H[Statin + fibrate if TG high]
    G --> I[Metformin for IR/diabetes]
    G --> J[GLP-1 RA/SGLT2i if T2DM]
```

### Metabolic Management
| Issue | 1st Line | 2nd Line |
|-------|----------|----------|
| **Dyslipidaemia** | Statin (atorvastatin 20-40mg) | Ezetimibe; fibrate (if TG >5) |
| **Insulin resistance** | Metformin (1st line) | GLP-1 RA / SGLT2i (if T2DM) |
| **Weight gain** | Lifestyle (diet/exercise) | Switch ART (PI -> INSTI; INSTI -> RPV/DOR) |
| **Lipodystrophy** | Switch from PI/NRTI (stavudine/ZDV) | Tesamorelin (if HIV + lipodystrophy) |

### Drug Interactions (Statin + ART)
| ART | Statin Interaction | Recommendation |
|-----|-------------------|----------------|
| **PI (boosted)** | ↑ simvastatin/atorvastatin (CYP3A4) | Atorvastatin ≤20mg; rosuvastatin 10mg; pitavastatin |
| **INSTI** | Minimal | Standard doses |
| **NNRTI (EFV)** | ↓ atorvastatin (CYP2B6) | Rosuvastatin/atorvastatin (monitor) |

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **PI** | Lopinavir/r > atazanavir/r: dyslipidaemia, IR, lipodystrophy |
| **INSTI** | Dolutegravir/bictegravir: weight gain, IR; raltegravir/doravirine neutral |
| **NRTI** | TAF > TDF weight gain; stavudine/ZDV -> severe lipodystrophy |
| **Screening** | Annual: glucose, lipids, weight, BMI, waist circumference |
| **Management** | Switch PI -> INSTI (DTG/BIC); metabolic Rx (metformin, statin, GLP-1/SGLT2i) |
| **Statin + PI** | Atorvastatin ≤20mg; rosuvastatin 10mg; avoid simvastatin |

## 9. Viva Questions
| Question | Expected Answer |
|----------|-----------------|
| **Which ART classes cause metabolic syndrome?** | PI (lopinavir/r > atazanavir/r) > INSTI (dolutegravir/bictegravir) > NNRTI/NRTI |
| **Which PI causes most dyslipidaemia?** | Lopinavir/ritonavir > atazanavir/ritonavir |
| **Which INSTI causes most weight gain?** | Dolutegravir, bictegravir > raltegravir, doravirine |
| **How do you manage ART-induced dyslipidaemia?** | Switch PI -> INSTI if virologically suppressed; statin (atorvastatin ≤20mg with PI) |
| **What is the metabolic syndrome with TAF vs TDF?** | TAF > TDF: weight gain, ↑triglycerides; TDF: renal/bone toxicity |
| **How do you manage lipodystrophy?** | Switch from PI/NRTI (stavudine/ZDV); tesamorelin if HIV + lipodystrophy |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **All INSTI cause weight gain?** | NO - dolutegravir/bictegravir > raltegravir/doravirine |
| **PI = all metabolic syndrome?** | PI: dyslipidaemia + IR + lipodystrophy; INSTI: weight gain + IR |
| **TAF = TDF?** | TAF: ↑weight, ↑TG; TDF: renal/bone toxicity; TDF cheaper |

**Mnemonic: ART-METABOLIC**
- **A**RT classes: PI > INSTI > NNRTI > NRTI (metabolic risk)
- **R**itonavir-boosted PI: lopinavir > atazanavir (dyslipidaemia, lipodystrophy)
- **T**AF: weight gain, ↑TG > TDF
- **M**etabolic syndrome: IR, dyslipidaemia, lipodystrophy, weight gain
- **E**favirenz: dyslipidaemia (NNRTI)
- **T**esamorelin: lipodystrophy treatment
- **A**TAZANAVIR/r: less dyslipidaemia than lopinavir/r
- **B**ictegravir/DTG: weight gain in INSTI
- **O**lder NRTI: stavudine/ZDV -> severe lipoatrophy
- **L**ipodystrophy: lipoatrophy (face/limbs) + lipohypertrophy (abdomen/buffalo)
- **I**NSTI: DTG/BIC weight gain > RAL/DOR neutral
- **C**ombination: PI + statin -> atorva ≤20mg; rosa 10mg; avoid simva
- **S**witch: PI -> INSTI if suppressed; metabolic improvement</think>

# DPP-4 Inhibitors

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Classify DPP-4 inhibitors and their dosing
- [ ] Apply renal dose adjustments (linagliptin unique = no adjustment)
- [ ] Recognise saxagliptin HF hospitalisation risk
- [ ] Position in ADA/EASD algorithm (2nd/3rd line, no CV benefit)
- [ ] Monitor for bullous pemphigoid, pancreatitis, arthralgia

---

## 2. Definition & Epidemiology

| Feature | Detail |
|--------|--------|
| **Drug Class** | Dipeptidyl peptidase-4 inhibitors |
| **Mechanism** | Inhibit DPP-4 enzyme → ↓degradation of endogenous GLP-1 & GIP → ↑glucose-dependent insulin secretion, ↓glucagon secretion |
| **Agents & Doses** | **Linagliptin** 5mg OD (no renal adjust); **Sitagliptin** 100mg OD (50mg if eGFR 30-45, 25mg if <30); **Vildagliptin** 50mg BD (50mg OD if eGFR<30); **Alogliptin** 25mg OD (12.5mg if eGFR 30-45, 6.25mg if <30); **Saxagliptin** 5mg OD (2.5mg if eGFR≤50) |
| **HbA1c Reduction** | 0.5-0.8% (5-9 mmol/mol) |
| **Weight Change** | Neutral (0 to –0.5kg) |
| **Hypoglycaemia Risk** | Very low (glucose-dependent action) |

## 4. Classification / Staging / Grading

### Agent Comparison

| Agent | Standard Dose | Renal Adjustment | Key Features |
|-------|---------------|------------------|--------------|
| **Linagliptin** | 5mg OD | **NO adjustment** (biliary excretion) | **CKD safe**; CARMELINA (renal neutral); CAROLINA (vs glimepiride) |
| **Sitagliptin** | 100mg OD | eGFR 30-45: 50mg; <30: 25mg | TECOS (CV neutral); widely used |
| **Vildagliptin** | 50mg BD | eGFR<30: 50mg OD | Requires LFT monitoring (hepatotoxicity rare) |
| **Alogliptin** | 25mg OD | eGFR 30-45: 12.5mg; <30: 6.25mg | EXAMINE (post-ACS); CV neutral |
| **Saxagliptin** | 5mg OD | eGFR≤50: 2.5mg | **SAVOR-TIMI: ↑HF hospitalisation** (HR 1.27); avoid in HF |

### CVOT Evidence

| Trial | Agent | Population | MACE | HF Hospitalisation |
|-------|-------|------------|------|-------------------|
| **TECOS** | Sitagliptin | T2DM + ASCVD | Neutral | Neutral |
| **SAVOR-TIMI** | Saxagliptin | T2DM + ASCVD/risk | Neutral | **↑27% (HR 1.27)** |
| **EXAMINE** | Alogliptin | Post-ACS (15-90d) | Neutral | Neutral |
| **CARMELINA** | Linagliptin | T2DM + ASCVD/CKD | Neutral | Neutral |
| **CAROLINA** | Linagliptin | T2DM + CV risk | Non-inferior to glimepiride | Neutral |

## 5. Diagnosis & Investigations
| Investigation | Role | Monitoring |
|---------------|------|------------|
| **HbA1c** | Efficacy | 3-monthly till target, then 6-monthly |
| **Renal function (eGFR)** | Dose adjustment (except linagliptin) | Before initiation, then 6-12 monthly |
| **LFT (vildagliptin)** | Hepatotoxicity monitoring | Pre-treatment, then per local protocol |
| **Heart failure symptoms** | Saxagliptin risk | Screen if on saxagliptin |

## 6. Differential Diagnosis

| Condition | Distinguishing Features |
|-----------|-------------------------|
| **Bullous pemphigoid** | Tense blisters on erythematous base; ↑with DPP-4i (sitagliptin, linagliptin); **STOP drug**, refer dermatology |
| **Pancreatitis** | Severe epigastric pain radiating to back, ↑lipase >3×ULN; rare but reported; **STOP if suspected** |
| **Arthralgia** | Joint pain; onset weeks-months after start; **STOP drug**, resolves in weeks |
| **HF decompensation (saxagliptin)** | New/worsening dyspnoea, oedema, weight gain; **avoid saxagliptin in HF** |

## 7. Management

### Initiation & Positioning

| Step | Action |
|------|--------|
| **1. Select agent** | **Linagliptin 5mg OD** (no renal adjust, CKD safe); Sitagliptin if eGFR≥30; **Avoid saxagliptin if HF** |
| **2. Check renal function** | Adjust dose per table (except linagliptin) |
| **3. Add-on therapy** | After metformin; **no CV/renal benefit** — use if SGLT2i/GLP-1 RA not tolerated/contraindicated |
| **4. Combine** | With metformin, SGLT2i, insulin, TZD; **avoid with GLP-1 RA** (same pathway) |

### Adverse Effect Management

| Adverse Effect | Management |
|----------------|------------|
| **Bullous pemphigoid** | **STOP DPP-4i**; topical/systemic steroids; dermatology referral |
| **Arthralgia** | **STOP DPP-4i**; switch agent class; resolves in weeks |
| **Pancreatitis** | **STOP DPP-4i**; supportive care; avoid rechallenge |
| **HF symptoms (saxagliptin)** | **STOP saxagliptin**; switch to linagliptin/sitagliptin/alogliptin/vildagliptin; treat HF per guidelines |

### Special Situations

| Situation | Management |
|-----------|------------|
| **CKD** | **Linagliptin** preferred (no adjust); others dose reduce |
| **HF** | **Avoid saxagliptin**; linagliptin/sitagliptin/alogliptin/vildagliptin OK |
| **With GLP-1 RA** | **Do not combine** — same pathway (↑GLP-1) |
| **With insulin** | Safe; ↓insulin dose if hypoglycaemia |
| **Pregnancy** | Not recommended (Category B but limited data); insulin preferred |

| **Saxagliptin = other DPP-4i?** | NO — **saxagliptin ↑HF hosp**; avoid in HF |

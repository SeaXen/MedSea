# Glycaemic Targets

> [!info]
> **No single target** — individualise by diabetes type, duration, age, frailty, complications, hypoglycaemia risk, pregnancy; **NICE/ADA/ESC broadly aligned**: T1DM <48 mmol/mol (6.5%); T2DM <48–58; pregnancy <48; elderly <53–75 by frailty.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] State HbA1c targets by guideline and population
- [ ] Apply individualisation principles (hypo risk, life expectancy, complications)

## 2. HbA1c Targets by Guideline

### NICE NG28 / NG17
| Population | Target | Notes |
|------------|--------|-------|
| **T1DM (adults)** | **<48 mmol/mol (6.5%)** | If achievable without problematic hypoglycaemia |
| **T2DM (adults)** | **<48 mmol/mol (6.5%)** | If achievable without hypoglycaemia; relax to **<58 (7.5%)** if high hypo risk, frailty, limited life expectancy |
| **T2DM (new diagnosis)** | **<48 mmol/mol** | Intensive early control → legacy effect (UKPDS) |
| **Pregnancy (pre-existing)** | **<48 mmol/mol** | Pre-conception and throughout |
| **Children T1DM** | **<48 mmol/mol** | If achievable without severe hypo |

### ADA 2024 Standards of Care
| Population | Target | Notes |
|------------|--------|-------|
| **Most non-pregnant adults** | **<53 mmol/mol (7.0%)** | Individualise |
| **If achievable safely** | **<48 mmol/mol (6.5%)** | Especially T1DM, new T2DM |
| **High hypo risk / limited life expectancy / advanced complications** | **<58–64 mmol/mol (7.5–8.0%)** | Relaxed |
| **Pregnancy (pre-existing)** | **<48 mmol/mol (6.5%)** | <42 (6.0%) if achievable without hypo |
| **GDM** | Not applicable (use glucose targets) | |

### ESC/EASD 2023
| Population | Target |
|------------|--------|
| **T1DM** | **<48 mmol/mol (6.5%)** |
| **T2DM** | **<53 mmol/mol (7.0%)**; <48 if safe |
| **Elderly/frail** | Individualised by frailty (see below) |
| **Pregnancy** | <48 (6.5%) |

## 3. Individualisation Framework (ADA/ESC/NICE Consensus)

| Factor | Tighten Target (<48 / <6.5%) | Relax Target (>58 / >7.5%) |
|--------|------------------------------|----------------------------|
| **Diabetes duration** | Short (<5y T2DM; new T1DM) | Long (>10–20y) |
| **Age** | Young (<40–50) | Elderly (>65–70) |
| **Frailty** | Fit (CFS 1–3) | Frail (CFS 5–7) |
| **Hypoglycaemia risk** | No severe hypo, good awareness | Recurrent Level 2/3, unawareness |
| **Complications** | None / early | Advanced CVD, CKD G4–5, proliferative retinopathy |
| **Life expectancy** | >10–15 years | <5–10 years |
| **Patient preference** | Motivated, engaged | Overwhelmed, competing priorities |
| **Comorbidities** | Few | Many |

**HbA1c <42 (6.0%)**: Not routinely recommended — ↑ mortality signal in ACCORD, ↑ hypoglycaemia.

## 4. Frailty-Based Targets (Elderly) — See "Frailty and Individualised HbA1c Targets"

| Frailty Category (CFS) | HbA1c Target |
|------------------------|--------------|
| **Fit (1–3)** | **<53 mmol/mol (7.0%)** |
| **Vulnerable (4)** | **<58 mmol/mol (7.5%)** |
| **Frail (5–7)** | **<64 mmol/mol (8.0%)** |
| **Severely frail / End-of-life (8–9)** | **<75 mmol/mol (9.0%)** — symptom control only |

## 5. CGM-Based Targets (ATTD / ADA / EASD Consensus) — Primary for Insulin Users

| Metric | Target (General) | Target (Pregnancy) | Target (High Risk/Frail) |
|--------|------------------|-------------------|--------------------------|
| **TIR** (3.9–10.0 mmol/L / 70–180 mg/dL) | **>70%** | **>70%** (3.5–7.8) | **>50%** |
| **TBR Level 1** (3.0–3.9 / 54–69) | **<4%** | **<4%** | **<1%** |
| **TBR Level 2** (<3.0 / <54) | **<1%** | **<1%** | **<0.5%** |
| **TAR Level 1** (10.1–13.9 / 181–250) | **<25%** | **<25%** | **<50%** |
| **TAR Level 2** (>13.9 / >250) | **<5%** | **<5%** | **<10%** |
| **CV** (Coefficient of Variation) | **<36%** | **<36%** | **<40%** |
| **GMI** (Glucose Management Indicator) | **<53 (7.0%)** | **<48 (6.5%)** | Individualised |

**TIR >70% ≈ HbA1c <53 (7.0%)**; each 5% TIR ≈ 0.5% HbA1c.

## 7. Special Situations

| Situation | Target |
|-----------|--------|
| **Pre-conception (T1/T2)** | **HbA1c <48 (6.5%)**; <42 (6.0%) if safe |
| **Pregnancy (T1/T2)** | Fasting <5.3; 1h <7.8; 2h <6.4; HbA1c <48 (NICE) / <42 if safe (ADA) |
| **GDM** | Same as pregnancy targets |
| **Inpatient (non-critical)** | **Pre-meal 5–10**; random <12; avoid <4 |
| **Inpatient (critical/ICU)** | **6–10 mmol/L** (NICE-SUGAR: intensive <6 ↑ mortality) |
| **Post-transplant** | Individualised; steroid hyperglycaemia common |
| **Cystic fibrosis-related diabetes** | <53 (7.0%) if stable |

## 8. The "Legacy Effect" — Early Intensive Control

| Trial | Population | Intervention | Long-Term Benefit |
|-------|------------|--------------|-------------------|
| **UKPDS** (1998/2008) | New T2DM | HbA1c ~7.0% vs ~7.9% | **10y post-trial**: ↓ MI 15%, ↓ death 13%, ↓ microvascular 24% |
| **DCCT/EDIC** (1993/2005) | T1DM | HbA1c ~7.0% vs ~9.0% | **30y follow-up**: ↓ CVD 30%, ↓ retinopathy/nephropathy progression |

**Implication**: **Early intensive control** in new T2DM and T1DM has lifelong benefits — "metabolic memory."

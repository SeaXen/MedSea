# Time in range (TIR) targets

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Apply ATTD consensus CGM targets (TIR, TBR, TAR, GMI, CV)
- [ ] Interpret AGP report for clinical decision-making
- [ ] Apply CGM metrics in clinical practice (insulin adjustment, lifestyle)
- [ ] Recognise limitations of CGM metrics vs HbA1c

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **TIR** | Time in Range (70-180 mg/dL / 3.9-10.0 mmol/L) |
| **TBR** | Time Below Range (<70 mg/dL / <3.9 mmol/L) |
| **TAR** | Time Above Range (>180 mg/dL / >10.0 mmol/L) |
| **GMI** | Glucose Management Indicator (estimates HbA1c from CGM mean) |
| **CV** | Coefficient of Variation (glucose variability) |

## 4. Classification / Staging / Grading

### ATTD Consensus Targets (2019/2023)
| Metric | T1DM Target | T2DM Target | Pregnancy |
|--------|-------------|-------------|-----------|
| **TIR (70-180 mg/dL)** | **>70%** | **>70%** | >70% (63-140 mg/dL) |
| **TBR <70 mg/dL** | **<4%** | **<4%** | <4% |
| **TBR <54 mg/dL** | **<1%** | **<1%** | <1% |
| **TAR >180 mg/dL** | <25% | <25% | <25% |
| **TAR >250 mg/dL** | <5% | <5% | — |
| **CV** | **<36%** | **<36%** | <36% |
| **GMI** | ≈ HbA1c | ≈ HbA1c | ≈ HbA1c |

### TIR Categories (Clinical)
| TIR | Category | Action |
|-----|----------|--------|
| **<50%** | Very poor | Immediate intensification |
| **50-70%** | Suboptimal | Intensify therapy |
| **>70%** | Target | Maintain |
| **>85%** | Excellent | Consider deintensify if hypo risk |

## 5. Diagnosis & Investigations
| Test | Role |
|------|------|
| **CGM (rtCGM/isCGM)** | Primary TIR measurement (14-day wear minimum) |
| **AGP Report** | Standardised visualisation (median, IQR, TIR, TBR, TAR, GMI, CV) |
| **HbA1c** | Complementary; GMI ≈ HbA1c |

## 7. Management

### CGM-Guided Therapy
| AGP Pattern | Action |
|-------------|--------|
| **High TAR** | Increase basal/bolus; review carb counting |
| **High TBR** | Reduce basal/bolus; adjust timing; carb before bed |
| **High CV** | Review meal regularity; carb consistency; bolus timing |
| **Wide IQR** | Review carb estimation; bolus timing; snack patterns |

### Pregnancy-Specific

## 9. Viva Questions
| Question | Expected Answer |
|----------|-----------------|
| **What are the ATTD TIR targets for T1DM?** | TIR >70% (70-180 mg/dL); TBR <4% (<70), <1% (<54); TAR <25% (>180); CV <36% |
| **What is GMI?** | Glucose Management Indicator - estimates HbA1c from CGM mean glucose |
| **What is the relationship between TIR and HbA1c?** | TIR 70% ≈ HbA1c 7.0% (53 mmol/mol); each 10% TIR ≈ 0.5% HbA1c |
| **When is CV >36%?** | High glucose variability; associated with hypoglycaemia risk and complications |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **TIR replaces HbA1c?** | NO - complementary; HbA1c = 3-month average; TIR = daily variability |
| **TIR same in pregnancy?** | NO - pregnancy range 63-140 mg/dL; tighter |
| **GMI = HbA1c?** | Approximate; GMI from CGM mean; HbA1c from RBC glycation |

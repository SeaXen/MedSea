## 4. Classification / Staging / Grading

### Mechanisms of Insulin Resistance in HIV
| Mechanism | Details |
|---------|---------|
| **Chronic inflammation** | ↑IL-6, TNF-α, CRP → ↓IRS-1 signalling |
| **Mitochondrial toxicity** | NRTI (stavudine, zidovudine) → mtDNA depletion |
| **ART effects** | PI: ↓GLUT4, ↑lipogenesis; INSTI (DTG/BIC): weight gain, ↑adiposity |
| **Lipodystrophy** | Lipoatrophy → ↓adiponectin; lipohypertrophy → visceral fat |
| **HCV co-infection** | HCV core protein → IRS-1 serine phosphorylation |

## 5. Diagnosis & Investigations
| Test | Role |
|------|------|
| **HOMA-IR** | (Fasting insulin × fasting glucose) / 22.5; >2.5 = IR |
| **Fasting glucose/HbA1c** | Screen for diabetes (HbA1c falsely low in CKD) |
| **Lipid profile** | TG, HDL, LDL, apoB |
| **Adiponectin/leptin** | Low adiponectin = IR marker |
| **Visceral fat** | CT/MRI (gold standard); waist circumference surrogate |
| **HCV RNA** | Screen all PLWH; treat if positive |

## 7. Management

### HIV-Specific IR Management
```mermaid
flowchart TD
    A[HIV + Insulin Resistance] --> B[Screen: HOMA-IR, lipids, HCV, visceral fat]
    B --> C{On ART?}
    C -->|Yes| D[ART review: PI -> INSTI if suppressed]
    C -->|No| E[Start ART (DTG/BIC preferred; minimise weight gain)]
    D --> F[Metabolic Rx]
    E --> F
    F --> G[Metformin 1st line IR]
    F --> H[GLP-1 RA/SGLT2i if T2DM]
    F --> I[Statin + fibrate if dyslipidaemia]
    F --> J[Weight loss if obese]
    F --> K[HCV treatment if co-infected]
```

### Management Strategies
| Domain | Intervention |
|--------|--------------|
| **ART optimisation** | Switch PI -> INSTI (DTG/BIC) if suppressed; avoid older NRTI |
| **Metformin** | 1st line for IR (if eGFR >=30); weight neutral |
| **GLP-1 RA / SGLT2i** | If T2DM + obesity/CV/CKD; weight loss benefit |
| **Statin** | Atorvastatin 20-40mg (CV risk reduction) |
| **Weight loss** | 5-7% → ↓IR 30-50%; lifestyle + GLP-1 RA/SGLT2i |
| **HCV treatment** | DAA cure → ↓IR (SVR12) |

### HCV Co-infection
| Aspect | Management |
|---------|------------|
| **Screen** | All PLWH: HCV Ab then RNA |
| **Treat** | DAA (SOF/VEL, GLE/PIB); 8-12 weeks |
| **Benefit** | SVR12 → ↓IR, ↓diabetes risk, ↓liver complications |
| **Timing** | Treat early; before cirrhosis |

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **Mechanisms** | Chronic inflammation, ART (PI/INSTI), mitochondrial toxicity, lipodystrophy, HCV |
| **Prevalence** | IR 40-60%; DM 10-15% (vs 8-10% general) |
| **ART** | PI (lopinavir) > INSTI (DTG/BIC) > NNRTI for IR |
| **Lipodystrophy** | Lipoatrophy (face/limbs) + lipohypertrophy (abdomen) → IR |
| **HCV** | Co-infection common; DAA cure → ↓IR |
| **Screening** | Annual fasting glucose/HbA1c; HOMA-IR if available |
| **Management** | ART switch (PI->INSTI); metformin; GLP-1/SGLT2i if T2DM; HCV cure |

## 9. Viva Questions
| Question | Expected Answer |
|----------|-----------------|
| **What are the mechanisms of insulin resistance in HIV?** | Chronic inflammation (IL-6, TNF-α), ART (PI/INSTI), mitochondrial toxicity (NRTI), lipodystrophy, HCV |
| **How does chronic inflammation cause IR in HIV?** | ↑IL-6, TNF-α, CRP → serine phosphorylation of IRS-1 → ↓insulin signalling |
| **Which ART drugs cause insulin resistance?** | PI (lopinavir/r > atazanavir/r), INSTI (dolutegravir, bictegravir), older NRTI (stavudine, zidovudine) |
| **How does lipodystrophy contribute to IR?** | Lipoatrophy -> ↓adiponectin; lipohypertrophy -> visceral adiposity -> inflammation |
| **Does HCV co-infection worsen IR?** | YES - HCV core protein → IRS-1 serine phosphorylation; DAA cure improves IR |
| **What is the screening for diabetes in HIV?** | Annual fasting glucose/HbA1c; HOMA-IR if available; CV risk assessment |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **HIV IR = traditional metabolic syndrome?** | NO - HIV IR has unique drivers (ART, chronic inflammation, lipodystrophy, HCV) |
| **All ART = IR?** | PI > INSTI (DTG/BIC) > NNRTI; raltegravir/doravirine neutral |
| **HCV cure = IR cure?** | DAA cure (SVR12) improves IR but may not fully normalise |
| **Lipodystrophy = just cosmetic?** | NO - metabolic consequences: IR, dyslipidaemia, diabetes risk |

# Statin Therapy in Diabetes (Intensity, Targets)

> [!info]
> **Diabetes = high CVD risk** — NICE: atorvastatin 20mg primary prevention (QRISK3 ≥10%); high-intensity 80mg secondary; ADA/ACC: high-intensity for most diabetes; **LDL target <1.8 (secondary) or <2.6 (primary)**; non-HDL <2.6/3.4; ezetimibe/PCSK9i if not at target.

## 2. Statin Intensity Classification

| Intensity | Statins & Doses | LDL Reduction |
|-----------|-----------------|---------------|
| **High** | Atorvastatin **40–80mg**, Rosuvastatin **20–40mg** | **≥50%** |
| **Moderate** | Atorvastatin 10–20mg, Rosuvastatin 10mg, Simvastatin 20–40mg, Pravastatin 40–80mg | **30–50%** |
| **Low** | Simvastatin 10mg, Pravastatin 10–20mg, Fluvastatin 20–40mg | **<30%** |

## 3. Guideline Recommendations

### NICE CG181 / NG28 (UK)

| Population | Statin | Dose | Target |
|------------|--------|------|--------|
| **Primary prevention**: T2DM, QRISK3 **≥10%** | Atorvastatin | **20mg** (moderate) | **Non-HDL <2.6** or **>40% reduction** |
| **Primary prevention**: T1DM **>40y OR >10y duration OR microvascular complications** | Atorvastatin | **20mg** | Non-HDL <2.6 |
| **Secondary prevention** (established CVD) | Atorvastatin | **80mg** (high) | **LDL <1.8** or **Non-HDL <2.2** or **>50% reduction** |
| **CKD (eGFR <60)** | Atorvastatin | 20mg (primary) / 80mg (secondary) | Per above |

**Review**: 3 months → check lipids, LFTs → adjust if <40% non-HDL reduction.

### ADA 2024 / ACC/AHA 2018

| Population | Statin Intensity | Target |
|------------|------------------|--------|
| **Diabetes, age 40–75** | **High-intensity** (atorva 40–80 / rosuva 20–40) | **LDL <1.8** if ASCVD; **<2.6** if primary |
| **Diabetes, age <40, risk factors** | Moderate/high | LDL <2.6 |
| **Diabetes, age >75** | Moderate (clinical judgement) | Individualised |
| **Established ASCVD** | **High-intensity** | **LDL <1.4** (very high risk) / <1.8 |
| **CKD (eGFR 15–59)** | Moderate/high | Per ASCVD status |

**ADA adds**: If **LDL >2.6 on max tolerated statin** → add **ezetimibe** → if still >1.8 (ASCVD) or >2.6 (primary) → **PCSK9i**.

### ESC/EAS 2019

| Risk Category | LDL Target | Non-HDL Target | Statin Intensity |
|---------------|------------|----------------|------------------|
| **Very high** (diabetes + CVD or diabetes + target organ damage or diabetes >20y) | **<1.4** (55 mg/dL) | **<1.8** | High + ezetimibe if needed |
| **High** (diabetes no other risk factors, age >40) | **<1.8** | **<2.2** | High |
| **Moderate** (diabetes age <40, no complications) | **<2.6** | **<2.6** | Moderate |

## 4. Lipid Targets Summary

| Scenario | **LDL-C Target** | **Non-HDL-C Target** | **ApoB Target** |
|----------|------------------|----------------------|-----------------|
| **Secondary prevention / Very high risk** | **<1.4** (ESC) / **<1.8** (NICE/ADA) | **<2.2** (NICE) / **<1.8** (ESC) | **<65 mg/dL** |
| **Primary prevention / High risk** | **<1.8** / **<2.6** | **<2.6** | **<80 mg/dL** |
| **CKD** | Per above | Per above | Per above |

**Non-HDL = Total Cholesterol − HDL**; better predictor than LDL in diabetes (hypertriglyceridaemia); **preferred by NICE**.

---

## 5. Add-On Therapy (If Not at Target on Max Tolerated Statin)

| Agent | Dose | LDL Reduction | Indication | Key Trials |
|-------|------|---------------|------------|------------|
| **Ezetimibe** | 10mg OD | **15–20%** | First add-on; cheap, safe | IMPROVE-IT (↓ events on simva+ezetimibe) |
| **PCSK9i** (alirocumab, evolocumab) | 75–150mg SC 2w / 300–420mg SC monthly | **50–60%** | Very high risk, LDL not at target | FOURIER, ODYSSEY |
| **Bempedoic acid** | 180mg OD | **15–20%** | Statin intolerant; add to ezetimibe | CLEAR Outcomes |
| **Inclisiran** (siRNA) | 284mg SC day 1, 90, then 6-monthly | **50%** | Very high risk; adherence advantage | ORION-10, -11 |

## 6. Statin Intolerance — Management Algorithm

```
Statin intolerance (muscle symptoms, LFT rise)
    │
    ├─► Confirm: CK if myalgia; LFT if hepatotoxicity
    │
    ├─► Try alternate statin (rosuvastatin often better tolerated)
    │
    ├─► Reduce dose / alternate day dosing
    │
    ├─► Switch to non-statin:
    │     ├─► Ezetimibe 10mg (first)
    │     ├─► Bempedoic acid 180mg (if ezetimibe insufficient)
    │     └─► PCSK9i (if very high risk, LDL not at target)
    │
    └─► Refer lipid clinic if complex
```

**CoQ10**: Not recommended routinely (no RCT evidence for statin myalgia).

## 7. Monitoring

| Test | Baseline | 3 Months | Annually |
|------|----------|----------|----------|
| **Lipid profile** (fasting preferred) | ✓ | ✓ (check % reduction) | ✓ |
| **LFTs (ALT)** | ✓ | ✓ | ✓ (if symptomatic) |
| **CK** | Only if myalgia | Only if myalgia | Only if myalgia |
| **HbA1c** | ✓ | — | ✓ (statins ↑ HbA1c ~2–3 mmol/mol; new T2DM risk ~0.1%/yr) |

**New-onset diabetes on statin**: Small absolute risk; CVD benefit outweighs; do not stop statin.

| **PCSK9i** | If LDL not at target on max statin + ezetimibe; very high risk |

| **Statin + diabetes risk** | Small ↑ HbA1c/new T2DM; **do not stop** for this — CVD benefit >> risk |

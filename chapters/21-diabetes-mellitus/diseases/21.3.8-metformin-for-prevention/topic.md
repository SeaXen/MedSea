## 4. Classification / Staging / Grading

### ADA/USPSTF Criteria for Metformin Prevention
| Criterion | Threshold |
|-----------|-----------|
| **BMI** | ≥35 kg/m2 (or ≥23 Asian) |
| **Age** | <60 years |
| **Prediabetes** | IFG, IGT, or HbA1c 39-47 mmol/mol (5.7-6.4%) |
| **History** | Gestational diabetes |
| **Rising HbA1c** | Despite lifestyle intervention |

### Efficacy by Subgroup (DPP)
| Subgroup | Metformin Reduction |
|----------|---------------------|
| **BMI ≥35** | 53% |
| **Age <60** | 44% |
| **Age ≥60** | Minimal (NS) |
| **GDM history** | Effective |
| **Overall** | 31% |

## 7. Management

### Prescribing Protocol
| Step | Action |
|-------|--------|
| **1** | Confirm prediabetes (IFG/IGT/HbA1c 39-47) |
| **2** | Assess criteria (BMI≥35, age<60, GDM history, rising HbA1c) |
| **3** | Start metformin 500mg OD with food |
| **4** | Titrate to 1000mg BD over 4 weeks (max 2000-2500mg) |
| **5** | Monitor: HbA1c 6-monthly; eGFR annually; B12 annually |

### Monitoring
| Parameter | Frequency |
|-----------|-----------|
| **HbA1c** | 6-monthly |
| **eGFR** | Annually (contraindicated if <30) |
| **B12** | Annually (deficiency risk) |
| **Weight** | Quarterly |

# Proliferative Diabetic Retinopathy (PDR)

> [!info]
> **PDR = neovascularisation** (NVD at disc, NVE elsewhere) driven by retinal ischaemia/VEGF; **high-risk characteristics (HRCs)** define urgent PRP need; **PRP = gold standard**; anti-VEGF alternative; vitrectomy for tractional RD/non-clearing vitreous haemorrhage.

## 2. Classification (ETDRS / ENDESP = R3)

| Feature | Description |
|---------|-------------|
| **NVD** (New Vessels at Disc) | On/within 1 DD of optic disc; **most ominous** |
| **NVE** (New Vessels Elsewhere) | Beyond 1 DD from disc; along vascular arcades |
| **Vitreous haemorrhage (VH)** | Blood in vitreous cavity → sudden vision loss |
| **Pre-retinal haemorrhage** | Between retina and posterior hyaloid |
| **Fibrovascular proliferation** | Epiretinal membrane + vessels → traction |
| **Tractional retinal detachment (TRD)** | Fibrovascular contraction → retinal elevation |

---

## 3. High-Risk Characteristics (HRCs) — **ETDRS Definition**

**Any ONE = High-Risk PDR (requires urgent PRP)**:

| HRC | Definition |
|-----|------------|
| **1. Large NVD** | NVD **>1/3 disc area** (≈ >standard photograph disc area) |
| **2. NVD + Haemorrhage** | **Any NVD** (any size) + **vitreous or pre-retinal haemorrhage** |
| **3. Large NVE + Haemorrhage** | **NVE >1/2 disc area** + **vitreous/pre-retinal haemorrhage** |

**No HRC** = **Early PDR** (still needs treatment but less urgent).

---

## 4. Pathophysiology

| Step | Mechanism |
|------|-----------|
| **1. Ischaemia** | Capillary non-perfusion (NPDR progression) → retinal hypoxia |
| **2. HIF-1α stabilisation** | Hypoxia → HIF-1α ↑ → **VEGF transcription** |
| **3. VEGF upregulation** | VEGF diffuses → binds VEGFR2 on endothelial cells |
| **4. Neovascularisation** | New vessels grow from **retinal vessels** (not choroidal) → **fragile, leaky** |
| **5. Complications** | Bleeding → VH; Contraction → TRD; Rubeosis → neovascular glaucoma |

---

## 5. Clinical Presentation

| Scenario | Symptoms |
|----------|----------|
| **Asymptomatic PDR** | No symptoms; detected on screening (R3) |
| **Vitreous haemorrhage** | **Sudden painless vision loss** ("floaters", "cobwebs", "curtain") |
| **Tractional RD** | Progressive visual field loss, metamorphopsia |
| **Neovascular glaucoma** | **Pain**, red eye, high IOP, rubeosis iridis (NVI) — ophthalmic emergency |

---

## 6. Management

### A. Pan-Retinal Photocoagulation (PRP) — **Gold Standard**
| Aspect | Detail |
|--------|--------|
| **Indication** | **HRC PDR** (urgent); Early PDR (within 4–6 months) |
| **Mechanism** | Ablate peripheral retina → ↓ VEGF production → regression of new vessels |
| **Protocol** (ETDRS) | **1200–1600 burns** (500µm, 0.1s, 100–200mW); 2–3 sessions |
| **Effectiveness** | **50% ↓ severe vision loss** (ETDRS); regression of NVD/NVE |
| **Side effects** | **Peripheral field loss** (driving), **night vision loss**, **DMO worsening** (15–30%), **choroidal detachment** (rare) |
| **Follow-up** | 4–6 weekly until regression; then 3–6 monthly |

### B. Anti-VEGF (Alternative / Adjunct)
| Agent | Regimen | Role |
|-------|---------|------|
| **Ranibizumab/Aflibercept** | Monthly ×3 → PRN / T&E | **Non-inferior to PRP** for regression (Protocol S, CLARITY); **less field loss** |
| **Bevacizumab** (off-label) | 1.25mg intravitreal | Cost-effective; similar efficacy |
| **Indications** | **Early PDR**, **DMO + PDR**, **media opacity** (prevents PRP), **patient refusal PRP** |
| **Disadvantage** | **Requires frequent injections**; rebound if stopped; cost; **no peripheral ablation** |

### C. Vitrectomy
| Indication | Timing |
|------------|--------|
| **Non-clearing vitreous haemorrhage** | **>3 months** (or <3m if severe + no view for PRP) |
| **Tractional retinal detachment** | **Macula-threatening** → urgent; macula-off → within 1w |
| **Combined TRD + rhegmatogenous** | Urgent |
| **Ghost cell glaucoma** | Urgent |
| **Procedure** | 23/25G pars plana; delamination ± endolaser ± gas/silicone oil |

---

## 7. Complications

| Complication | Management |
|--------------|------------|
| **Vitreous haemorrhage** | Observe 3m; PRP if view; vitrectomy if non-clearing |
| **Tractional RD** | Vitrectomy + membrane peeling ± endolaser ± tamponade |
| **Neovascular glaucoma (NVG)** | **Emergency**: IOP control (topical + systemic) + **PRP** (or cryo) + anti-VEGF; drainage device if refractory |
| **Rubeosis iridis (NVI)** | New vessels on iris → NVG precursor → urgent PRP |
| **PRP-induced DMO** | Anti-VEGF ± focal laser; usually transient |

---

## 8. Systemic Optimisation (Essential Adjunct)
| Target | Rationale |
|--------|-----------|
| **HbA1c <53 (7.0%)** | DCCT: intensive control ↓ PDR progression 50% |
| **BP <130/80** | UKPDS: tight BP ↓ progression |
| **Lipids** | Fenofibrate + statin ↓ progression (ACCORD-Eye) |
| **SGLT2i/GLP-1 RA** | CVOTs: ↓ DR progression (post-hoc) |
| **Anaemia correction** | Hb <11 → ↑ VEGF; correct if possible |

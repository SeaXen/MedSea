# Thyrotoxicosis diabetes

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise thyrotoxicosis as cause of hyperglycaemia
- [ ] Explain mechanism (↑HGP, ↓insulin clearance)
- [ ] Manage hyperglycaemia in thyrotoxicosis
- [ ] Recognise resolution with euthyroidism

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Definition** | Transient hyperglycaemia due to thyroid hormone excess |
| **Mechanism** | Thyroxine -> ↑HGP, ↓insulin clearance, ↑insulin degradation |
| **Prevalence** | 1-2% of thyrotoxicosis develop hyperglycaemia |
| **Resolution** | Resolves with euthyroidism |

## 3. Clinical Features / Presentation
| Feature | Presentation |
|---------|--------------|
| **Hyperglycaemia** | Often mild-moderate; fasting + post-prandial |
| **Thyrotoxic signs** | Weight loss, tachycardia, tremor, heat intolerance, goitre |
| **Cardiac** | Tachycardia, atrial fibrillation, high output HF |
| **Glucose pattern** | Post-prandial prominent; fasting may be normal |

## 4. Classification / Staging / Grading
| Feature | Detail |
|--------|--------|
| **Mechanism** | ↑HGP, ↓insulin clearance, ↑insulin degradation |
| **Reversibility** | Resolves with euthyroidism |
| **Type** | Usually mild; rarely DKA (if underlying DM) |

## 5. Diagnosis & Investigations
| Test | Finding |
|-------|---------|
| **TSH** | Suppressed (<0.01 mU/L) |
| **Free T4/T3** | Elevated |
| **Glucose/HbA1c** | Mild elevation; HbA1c may be falsely low |
| **TSH receptor antibodies** | Positive (Graves) |
| **Thyroid ultrasound/Doppler** | If nodule |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **T2DM** | No thyrotoxic signs; persistent hyperglycaemia |
| **Stress hyperglycaemia** | Acute illness; resolves |
| **Steroid diabetes** | Known steroid use; post-prandial pattern |
| **Cushing/Acromegaly** | Specific features; cortisol/GH excess |

## 7. Management
| Aspect | Management |
|-------|------------|
| **Thyrotoxicosis** | **Treat underlying** (carbimazole, radioiodine, surgery) |
| **Glucose control** | Often resolves with euthyroidism; monitor |
| **Insulin** | Rarely needed; short-term if severe |
| **Metformin** | If persistent after euthyroidism |
| **DKA risk** | Rare (unless underlying DM); monitor if unwell |

# Pre-Mixed Insulin

> [!info]
> **Pre-mixed insulin = fixed ratio** of rapid-acting + intermediate-acting (NPH or protamine) in one pen; **BD dosing** (pre-breakfast, pre-dinner); **for patients with regular meals, unable to carb count**; **inferior post-prandial control** vs basal-bolus; **higher hypoglycaemia risk**; **not first-line** (NICE: only if basal-bolus not feasible).

## 2. Available Pre-Mixed Insulins

| Product | Ratio (Rapid/Intermediate) | Rapid Component | Intermediate Component | Profile |
|---------|----------------------------|-----------------|------------------------|---------|
| **Novomix 30** | **30/70** | Aspart (30%) | Aspart protamine (70%) | Peak ~2–4h; duration ~12–16h |
| **Humalog Mix 25** | **25/75** | Lispro (25%) | Lispro protamine (75%) | Peak ~1–3h; duration ~12–16h |
| **Humalog Mix 50** | **50/50** | Lispro (50%) | Lispro protamine (50%) | Higher rapid for larger meals |
| **Humulin M3** | **30/70** | Human regular (30%) | NPH (70%) | Human insulin; more peak |
| **Insuman Comb 25/75** | **25/75** | Human regular (25%) | NPH (75%) | Human insulin |

**Most common**: **Novomix 30 (30/70)** and **Humalog Mix 25 (25/75)**.

---

## 3. Indications for Pre-Mixed Insulin (NICE NG28)

| Indication | Criteria |
|-----------|----------|
| **Regular meal pattern** | **Fixed mealtimes** (breakfast, dinner ~12h apart); similar carb content |
| **Unable/Unwilling to carb count** | Cognitive impairment, literacy issues, no structured education access |
| **Basal-bolus not feasible** | Injection burden (4–5/day), needle phobia, poor dexterity |
| **Simplification** | Elderly, care home, complex regimens failing |
| **Cost-conscious** | Single pen vs multiple; but analogues cost similar |

**NICE position**: **Only if basal-bolus not possible** — not first-line.

---

## 4. Contraindications / Avoid

| Scenario | Reason |
|----------|--------|
| **Variable meal times** | Fixed peak → hypoglycaemia if meal delayed |
| **Variable carbohydrate intake** | Fixed ratio → mismatch |
| **Shift work** | Circadian mismatch |
| **High post-prandial excursions** | Fixed rapid % often insufficient |
| **Hypoglycaemia unawareness** | Higher nocturnal hypo risk (intermediate peak) |
| **Renal impairment** | NPH/protamine accumulation → prolonged effect |

---

## 5. Initiation Protocol

### Starting Dose
| Method | Calculation |
|--------|-------------|
| **Weight-based** | **0.3–0.5 units/kg/day** total (split ~50/50 or 60/40 breakfast/dinner) |
| **Fixed** | **10–12 units BD** (pre-breakfast, pre-dinner) |
| **From basal insulin** | Same total daily units split BD (e.g., basal 30u → 15u + 15u) |

**Typical start**: **10–12 units pre-breakfast + 10–12 units pre-dinner**.

### Stop Sulfonylurea
- **Taper over 1–2 weeks** as pre-mixed initiated (hypoglycaemia risk)

### Continue
- **Metformin** (Yes)
- **SGLT2i / GLP-1 RA / DPP-4i** (Yes)
- **Pioglitazone** (Caution: weight gain + oedema)

## 6. Titration Algorithm

| Target | Pre-Breakfast Glucose | Pre-Dinner Glucose | Action |
|--------|----------------------|-------------------|--------|
| **Fasting / Pre-lunch** | **4–7 mmol/L** | — | Adjust **dinner dose** |
| **Pre-dinner** | — | **4–7 mmol/L** | Adjust **breakfast dose** |
| **If <4.0** | Any | Any | **↓ 2–4 units** relevant dose |
| **If 4–7** | Any | Any | **No change** |
| **If 7–10** | Any | Any | **↑ 2–4 units** relevant dose |
| **If >10** | Any | Any | **↑ 4–6 units** relevant dose |

**Frequency**: **Weekly** until stable; then monthly.

**Nocturnal hypoglycaemia** → reduce dinner dose 10–20%; consider bedtime snack.

---

## 7. Pre-Mixed vs Basal-Bolus Comparison

| Feature | **Pre-Mixed (BD)** | **Basal-Bolus (QDS)** |
|---------|---------------------|----------------------|
| **Injections/day** | 2 | 4–5 |
| **Meal flexibility** | **Low** (fixed times, carbs) | **High** (per meal) |
| **Carb counting** | Not needed | **Essential** |
| **Post-prandial control** | Moderate (fixed rapid %) | **Superior** (dose per meal) |
| **Hypoglycaemia risk** | **Higher** (nocturnal peak) | Lower (basal flat, bolus matched) |
| **Weight gain** | Similar | Similar |
| **Education needed** | Low | High (DAFNE) |
| **Cost** | Single pen | Multiple pens |

---

## 8. Transition from Pre-Mixed to Basal-Bolus

**Indications for Switch**:
- HbA1c > target despite titration
- Recurrent hypoglycaemia (esp. nocturnal)
- Variable lifestyle/meals
- Patient motivated + able to carb count
- Access to structured education (DAFNE)

**Method**:
1. Stop pre-mixed
2. Start basal insulin (glargine/detemir/degludec) at **50–70% of previous total daily dose**
3. Add rapid-acting bolus at meals per ICR
4. Monitor closely for 1–2 weeks

| **Best for** | **Regular meals**, unable to carb count, elderly |

| **Hypoglycaemia risk** | **Higher** (nocturnal intermediate peak) |

| **Titration** | Weekly; adjust breakfast dose by pre-dinner glucose, dinner dose by fasting |
| **Switch to basal-bolus** | If HbA1c not controlled, recurrent hypo, variable lifestyle |
| **Key limitation** | Fixed rapid:intermediate ratio → mismatch with meals |

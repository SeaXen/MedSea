# Cushing syndrome diabetes

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise Cushing syndrome as cause of secondary diabetes
- [ ] Apply diagnostic tests (dexamethasone suppression, midnight cortisol)
- [ ] Manage diabetes in Cushing syndrome
- [ ] Apply definitive treatment (surgery, medical therapy)

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Definition** | Hypercortisolism causing insulin resistance and diabetes |
| **Prevalence** | 20-50% of Cushing syndrome develop diabetes |
| **Mechanism** | Cortisol -> ↑HGP, ↓GLUT4, ↑lipolysis -> insulin resistance |
| **Type** | Usually T2DM-like (insulin resistance); can unmask latent DM |

## 3. Clinical Features / Presentation
| Feature | Presentation |
|---------|--------------|
| **Diabetes** | Often mild-moderate; post-prandial hyperglycaemia prominent |
| **Cushing features** | Moon face, central obesity, striae, bruising, hypertension, proximal myopathy |
| **Hypokalaemia** | Common (cortisol mineralocorticoid effect) |
| **Metabolic** | Central obesity, dyslipidaemia, hypertension |

## 4. Classification / Staging / Grading

### Diabetes in Cushing Syndrome
| Feature | Detail |
|---------|--------|
| **Type** | Insulin resistance (T2DM-like) |
| **Severity** | Correlates with cortisol levels |
| **Reversibility** | Often improves/resolves with eucortisolism |

## 5. Diagnosis & Investigations
| Test | Role | Interpretation |
|-------|------|----------------|
| **1mg overnight dexamethasone suppression** | Screening | Cortisol >50 nmol/L = positive |
| **48h low-dose dexamethasone (2mg/day)** | Confirmation | Day 3 9am cortisol >50 nmol/L = positive |
| **Late-night salivary cortisol** | Screening | >2x upper limit |
| **24h urinary free cortisol** | Confirmation | >3x upper limit (x2) |
| **ACTH** | Differentiate | Low = adrenal; High = pituitary/ectopic |
| **Imaging (MRI pituitary / CT adrenal)** | Localisation | Microadenoma vs adenoma vs ectopic |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **Pseudo-Cushing** | Alcoholism, depression, obesity; mild hypercortisolism; dexamethasone suppression usually normal |
| **Metabolic syndrome** | Obesity, hypertension, dyslipidaemia; normal cortisol dynamics |
| **Pseudo-Cushing (alcohol)** | Resolves with abstinence; history |

## 7. Management

### Diabetes Management
| Approach | Detail |
|----------|--------|
| **Insulin** | If severe hyperglycaemia; flexible dosing |
| **Metformin** | First-line oral; improves insulin sensitivity |
| **SGLT2i/GLP-1 RA** | If CVD/CKD; weight loss benefit |
| **Sulfonylureas** | Avoid (weight gain, hypoglycaemia) |
| **Steroids** | Not applicable (cortisol excess primary) |

### Definitive Treatment (Curative)
| Aetiology | Treatment |
|-----------|-----------|
| **Pituitary adenoma** | Transsphenoidal surgery (first-line) |
| **Adrenal adenoma** | Adrenalectomy (laparoscopic) |
| **Ectopic ACTH** | Tumour resection; bilateral adrenalectomy if unresectable |
| **Medical (if surgery contraindicated)** | Ketoconazole, metyrapone, mifepristone, pasireotide |

| Topic | Key Points |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **Cushing diabetes = T1DM?** | NO - insulin resistance (T2DM-like); C-peptide normal/high |
| **All Cushing = diabetes?** | NO - 20-50% develop DM; depends on cortisol severity/duration |
| **Post-prandial glucose** | Often disproportionately elevated (cortisol post-prandial peak) |

**Mnemonic: CUSHING-DIABETES**
- **C**ushing: cortisol excess -> insulin resistance
- **U**ncoupling: GLUT4 down -> glucose uptake down
- **S**teroid: cortisol -> HGP up, lipolysis up
- **H**yperglycaemia: post-prandial prominent
- **I**nsulin resistance: T2DM-like
- **N**egative feedback: lost (ACTH not suppressed)
- **G**lucose test: DST 1mg >50 nmol/L
- **D**examethasone suppression: diagnostic
- **I**mproves with cure: surgery -> eucortisolism
- **A**CTH: pituitary vs adrenal vs ectopic
- **B**ilateral adrenalectomy: last resort
- **E**ctopic ACTH: lung, thymus tumours
- **T**ranssphenoidal: pituitary surgery
- **E**xogenous steroids: iatrogenic Cushing -> similar DM**</think>

# Physical Activity and Exercise

> [!info]
> **ADA/NICE/ESC: 150min/week moderate aerobic + 2–3x/week resistance**; post-meal walk ↓ PP glucose 1–2mmol/L; HIIT time-efficient; **exercise snippets** (sedentary breaks) every 30min; adjust insulin/carbs; CGM-guided; avoid if glucose >14 + ketones.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Prescribe exercise per guideline recommendations (frequency, intensity, type)
- [ ] Differentiate aerobic, resistance, HIIT, flexibility benefits
- [ ] Manage glucose around exercise (insulin, carbs, CGM)
- [ ] Apply special considerations (complications, elderly, pregnancy)
- [ ] Address sedentary behaviour reduction

---

## 2. Guideline Recommendations (ADA 2024 / NICE / ESC)

| Component | Prescription |
|-----------|--------------|
| **Aerobic** | **≥150 min/week** moderate (3–6 METs) OR **75 min/week** vigorous (>6 METs) |
| **Frequency** | **≥3 days/week**; no >2 consecutive days without |
| **Resistance** | **2–3 sessions/week** (non-consecutive); 8–10 exercises, 1–3 sets, 8–15 reps |
| **Flexibility/Balance** | **2–3x/week** (yoga, tai chi) — especially elderly/neuropathy |
| **Sedentary breaks** | **Every 30min** — stand, walk 2–3min (↓ post-prandial glucose) |

**Total**: ~150min aerobic + 60min resistance + flexibility weekly.

---

## 3. Exercise Types & Glycaemic Effects

| Type | Examples | Glycaemic Effect | Key Notes |
|------|----------|------------------|-----------|
| **Moderate aerobic** | Brisk walk, cycling, swimming, elliptical | **↓ glucose during/after** (↑ insulin sensitivity 24–48h) | 40–60% HRmax or RPE 12–13 (talk test) |
| **Vigorous aerobic** | Running, HIIT, fast cycling | **Rapid ↓ glucose** (↑ GLUT4 translocation) | Risk of delayed hypoglycaemia (6–12h post) |
| **HIIT** | 4×4min at 85–95% HRmax + 3min recovery | **Time-efficient**; ↑ VO2max, insulin sensitivity | **Not for unfit/comorbid** without screening |
| **Resistance** | Weights, bands, bodyweight | **↑ muscle mass** → ↑ basal metabolic rate; ↓ insulin resistance | Major muscle groups; progressive overload |
| **Flexibility/Balance** | Yoga, tai chi, stretching | Indirect (↓ falls, ↑ adherence) | Essential for neuropathy/elderly |

## 4. Glucose Management Around Exercise

### Pre-Exercise Assessment
| Glucose | Action |
|---------|--------|
| **<4.0 mmol/L** | **Treat hypo first**; delay exercise |
| **4.0–6.9** | **Carb snack 10–20g** (if on insulin/sulfonylurea) |
| **7.0–10.0** | **Safe to start** (ideal) |
| **10.1–13.9** | Safe; monitor if on insulin |
| **14.0–16.6** | **Check ketones**; if negative → OK with caution |
| **>16.7 + ketones** | **DO NOT EXERCISE** — risk DKA |

### Insulin/Carb Adjustment

| Exercise Type | Insulin Adjustment | Carb Intake |
|---------------|-------------------|-------------|
| **Aerobic 30–60min** | ↓ Basal 20–50% (if pump: temp basal 50–80%); ↓ Bolus 25–50% | 15–30g/hr if >60min or glucose <7 |
| **HIIT / Resistance** | ↓ Basal 10–20%; ↓ Bolus 25% | Often not needed (glucose may rise initially) |
| **Prolonged (>90min)** | ↓ Basal 50–80%; consider carb + protein during | 30–60g/hr + electrolytes |

**CGM-guided**: Trend arrows ↑ → reduce carb; ↓ → add carb; → → standard.

### Post-Exercise
- **Monitor 6–24h** (delayed hypoglycaemia risk, esp. after aerobic)
- **Reduce basal 10–20% overnight** if evening exercise
- **Protein + carb recovery** (e.g., milk, yogurt) if prolonged

## 5. Special Populations

| Population | Modifications |
|------------|---------------|
| **T1DM** | CGM essential; detailed insulin/carb plan; ketone check if >14 |
| **Elderly / Frail** | **Chair-based**, balance, resistance bands; HIIT contraindicated; supervision |
| **Peripheral neuropathy** | **Non-weight-bearing** (cycling, swimming, chair); daily foot check; proper footwear |
| **Autonomic neuropathy** | **Avoid Valsalva**; monitor HR/BP; supine exercises; gradual position changes |
| **CKD / Dialysis** | Exercise **on non-dialysis days**; avoid vascular access arm; fluid balance |
| **Pregnancy (GDM/pre-existing)** | **Moderate only**; avoid supine >16w; HR <140; no contact sports; 150min/wk |
| **Cardiovascular disease** | **Cardiac rehab** referral; stress test if high intensity; avoid ischaemic threshold |

## 6. Sedentary Behaviour — Independent Risk Factor

| Recommendation | Evidence |
|----------------|----------|
| **Break sitting every 30min** | 2–3min light walk → **↓ post-prandial glucose 20–30%** |
| **Standing desk** | ↓ Sedentary time 1–2h/day |
| **TV/computer breaks** | Ad breaks = movement cues |
| **Step target** | **7,000–10,000 steps/day** (dose-response for mortality) |

**Key**: Sedentary time **independent of exercise** predicts mortality and T2DM progression.

## 7. Exercise Prescription Template (FITT-VP)

| Component | Prescription |
|-----------|--------------|
| **Frequency** | Aerobic ≥3d/wk; Resistance 2–3d/wk; Flexibility 2–3d/wk |
| **Intensity** | Moderate (RPE 12–13, "talk test") → Vigorous as tolerated |
| **Time** | Aerobic 30–60min/session; Resistance 20–30min |
| **Type** | Aerobic + Resistance + Flexibility/Balance |
| **Volume** | 150min aerobic + 60min resistance weekly |
| **Progression** | Increase duration → frequency → intensity (gradual) |

## 8. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **Aerobic minimum** | **150min/week moderate** (≥3 days, no >2 days off) |
| **Resistance** | **2–3x/week**, non-consecutive, major muscle groups |
| **Sedentary breaks** | **Every 30min** — 2–3min walk |
| **Post-meal walk** | 10–15min → ↓ PP glucose 1–2mmol/L |
| **Pre-exercise glucose <4** | Treat hypo first |
| **Pre-exercise glucose >14 + ketones** | **Do not exercise** |
| **T1DM on insulin** | ↓ Bolus 25–50%; ↓ Basal 20–50% (temp basal); carb snack if <7 |
| **Delayed hypo** | 6–12h post-aerobic; reduce overnight basal |
| **Neuropathy** | Non-weight-bearing (cycling, swimming) |
| **HIIT** | Time-efficient; screen first; not for unfit/comorbid |

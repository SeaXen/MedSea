# m.3243A>G (MELAS/MIDD)

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise mitochondrial diabetes due to m.3243A>G mutation
- [ ] Identify MELAS and MIDD syndromes
- [ ] Apply maternal inheritance counselling
- [ ] Manage diabetes and extra-pancreatic features

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Mutation** | m.3243A>G in tRNA-Leu (UUR) |
| **Syndromes** | MELAS (mitochondrial encephalomyopathy, lactic acidosis, stroke-like episodes) / MIDD (maternally inherited diabetes and deafness) |
| **Inheritance** | Maternal (mtDNA); heteroplasmy determines severity |
| **Prevalence** | ~1% of diabetes; most common mtDNA diabetes |

## 4. Classification / Staging / Grading

### Heteroplasmy & Severity
| Heteroplasmy Level | Phenotype |
|--------------------|-----------|
| **<10%** | Asymptomatic carrier |
| **10-30%** | MIDD (diabetes + deafness) |
| **30-90%** | MIDD + cardiomyopathy + myopathy |
| **>90%** | MELAS (severe) |

## 5. Diagnosis & Investigations
| Test | Finding |
|------|---------|
| **Genetic testing** | m.3243A>G mutation (blood, urine, hair follicles) |
| **Heteroplasmy level** | Quantify in blood/urine |
| **Audiometry** | High-frequency sensorineural hearing loss |
| **ECG/Echo** | Hypertrophic cardiomyopathy |
| **MRI brain** | Stroke-like lesions (MELAS) |
| **Lactate** | Elevated (esp. CSF in MELAS) |
| **Urine protein** | Proteinuria (FSGS pattern in MIDD) |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **T2DM + deafness** | No maternal inheritance; no heteroplasmy |
| **Wolfram syndrome** | WFS1 gene; DI, DM, OA, D (DIDMOAD) |
| **Alport syndrome** | Deafness + renal; COL4A5 |
| **Diabetic cardiomyopathy** | No deafness; no maternal inheritance |

## 7. Management

### Diabetes Management
| Aspect | Detail |
|--------|--------|
| **Treatment** | Insulin often needed (progressive beta-cell failure) |
| **Avoid** | Metformin (lactic acidosis risk); glitazones (fluid retention) |
| **SGLT2i** | Caution (renal impairment, low BMI) |
| **GLP-1 RA** | Caution (weight loss may worsen myopathy) |
| **Insulin** | Often required; variable doses |

### Extra-pancreatic Management
| System | Management |
|--------|------------|
| **Cardiomyopathy** | Echo surveillance; ACEi/ARB, beta-blocker |
| **Deafness** | Hearing aids; cochlear implant if severe |
| **Renal (FSGS)** | ACEi/ARB; nephrology referral |
| **Lactic acidosis** | Avoid metformin; IV fluids, bicarbonate if acute |
| **Stroke-like episodes (MELAS)** | Arginine IV; avoid triggers |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **All diabetes + deafness = MIDD?** | NO - Alport (COL4A5), Wolfram (WFS1), maternal inheritance key |
| **MELAS = just stroke?** | No - stroke-like episodes + lactic acidosis + myopathy + diabetes |
| **Heteroplasmy = always tested?** | Yes - blood, urine, hair; correlates with severity |

**Mnemonic: MELAS-MIDD-MUTATION**
- **M**ELAS: Mitochondrial Encephalomyopathy, Lactic Acidosis, Stroke-like
- **E**ncephalomyopathy: brain + muscle
- **L**actic acidosis: elevated lactate (blood/CSF)
- **A**stroke-like: not true strokes; cortical
- **S**treptomycin: aminoglycosides worsen (avoid)
- **M**IDD: Maternally Inherited Diabetes & Deafness
- **I**nheritance: Maternal (mtDNA)
- **D**eafness: sensorineural, high-frequency
- **D**iabetes: progressive, insulin-requiring
- **M**utation: m.3243A>G (tRNA-Leu)
- **U**UR: tRNA-Leu(UUR)
- **T**eroplasmy: low=MIDD; high=MELAS
- **A**void metformin: lactic acidosis risk
- **I**nsulin often needed
- **O**phthalmoplegia: chronic progressive (MELAS)
- **N**ephropathy: FSGS (MIDD); rare in MELAS</think>

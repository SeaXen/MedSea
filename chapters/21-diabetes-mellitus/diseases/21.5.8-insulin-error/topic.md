# Hypoglycaemia in Hospital / Insulin Error

> [!info]
> **Inpatient hypoglycaemia = 10–20% of diabetes admissions**; **insulin errors** = leading preventable cause (wrong dose, wrong insulin, wrong timing, sliding scale misuse); **protocolised regimens + capillary glucose monitoring + staff education** reduce events.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Define inpatient hypoglycaemia thresholds
- [ ] Identify common insulin prescribing/administration errors
- [ ] Apply prevention strategies (protocols, prescriptions, monitoring)
- [ ] Manage inpatient hypoglycaemia per severity
- [ ] Report and review incidents (learning culture)

---

## 2. Definitions (ADA / NICE / JBDS)

| Category | Capillary / Venous Glucose | Clinical Context |
|----------|----------------------------|------------------|
| **Level 1 (Alert)** | **3.9–3.0 mmol/L** (70–54 mg/dL) | Asymptomatic or mild symptoms; intervention needed |
| **Level 2 (Clinically Significant)** | **<3.0 mmol/L** (<54 mg/dL) | Neuroglycopenic symptoms; urgent treatment |
| **Level 3 (Severe)** | **<2.0 mmol/L** (<40 mg/dL) OR **requires assistance** | Altered consciousness, seizure, coma; **medical emergency** |

**Inpatient target**: Avoid <4.0 mmol/L; treat at 3.9 mmol/L.

## 3. Common Insulin Errors in Hospital

| Error Type | Examples | Prevention |
|------------|----------|------------|
| **Wrong dose** | 10 units instead of 1; 40 units instead of 4; decimal point | **Unit-specific syringes**; double-check; **no "U" abbreviation** (write "units") |
| **Wrong insulin** | Rapid given instead of basal; basal instead of rapid; concentrated (U200/U300) ≠ U100 | **Colour-coded labels**; **tall-man lettering** (Hum**U**lin vs Hum**A**log); separate storage |
| **Wrong timing** | Rapid insulin given **without meal**; basal given at wrong time; pre-meal given post-meal | **Link insulin to meals** on chart; **pre-meal = 15–20 min before**; meal tray verification |
| **Sliding scale misuse** | **Sliding scale AS ONLY REGIMEN** (no basal); excessive correction doses; no meal bolus | **Basal-bolus mandatory**; sliding scale **supplemental only**; max correction dose cap |
| **Omitted dose** | Basal insulin held for "low glucose" (e.g., 5.5) without review | **Never omit basal** without senior review; reduce by 20–50% instead |
| **Duplicate dose** | Nurse + patient both give; shift handover failure | **Electronic MAR**; double-sign for insulin; patient self-administration policy |
| **Renal adjustment failure** | Full insulin dose in eGFR <30 → prolonged hypoglycaemia | **Dose reduce per eGFR**; pharmacy review |
| **Concentrated insulin error** | U200/U300/U500 given as U100 volume | **Separate prescribing**; dedicated syringes; **volume check** |

## 4. High-Risk Clinical Scenarios

| Scenario | Specific Risks |
|----------|----------------|
| **NPO / Fasting** (pre-op, procedures) | Baseline insulin still needed; reduce basal 20–50%; dextrose infusion if prolonged |
| **Renal impairment** (AKI, CKD G4–5) | Insulin clearance ↓ → reduce dose 30–50%; monitor q4–6h |
| **Corticosteroids** (dexamethasone) | Hyperglycaemia day 1–2 → ↑ insulin; then hypoglycaemia as steroid wears off |
| **Enteral/Parenteral Nutrition** | Feed interruption → hypoglycaemia; insulin matched to feed rate |
| **Elderly / Frail / Dementia** | Erratic intake, polypharmacy, renal impairment, autonomic neuropathy |
| **Post-bariatric surgery** | Altered anatomy, rapid gastric emptying, dumping → unpredictable glucose |
| **Critical illness** (ICU) | Stress hyperglycaemia → insulin infusion; target 6–10 mmol/L (NICE-SUGAR) |

## 5. Prevention Strategies (JBDS / NICE)

### Prescribing
- **Basal-bolus regimen** for all insulin-requiring inpatients (sliding scale **supplemental only**)
- **Pre-printed / electronic order sets** with dose calculators
- **No "U" abbreviation** — write "units"
- **Concentrated insulins** (U200, U300, U500) — separate prescribing pathway
- **Renal dosing guidelines** on chart (eGFR-based)

### Administration
- **Double-check** by two nurses for insulin (or barcode scanning)
- **Meal-coordinated** rapid insulin (pre-meal 15–20 min)
- **Capillary glucose** before **every** insulin dose (unless continuous infusion)
- **Patient self-administration** if competent (assessed, documented)

### Monitoring
- **Capillary glucose** pre-meals, bedtime, 3am (if on insulin)
- **Target 5–10 mmol/L** (individualised; tighter in critical care)
- **Hypo alert at 3.9** — inform doctor, review insulin
- **Continuous glucose monitoring (CGM)** emerging for inpatients

### Systems
- **Hypoglycaemia protocol** on every ward (IV dextrose, glucagon, escalation)
- **Mandatory reporting** of Level 2/3 events (Datix/Safety system)
- **Monthly audit** of hypoglycaemia rates, insulin errors, harm
- **Staff education** (induction + annual) — insulin safety, device use

## 6. Inpatient Hypoglycaemia Treatment

| Severity | Conscious? | Treatment |
|----------|------------|-----------|
| **Level 1** (3.0–3.9) | Yes | **15–20g fast-acting carb** (glucose gel, juice, 4–5 dextrose tabs); recheck 15 min |
| **Level 2** (<3.0) | Yes | **20g fast-acting carb** + complex carb; recheck 15 min; review insulin |
| **Level 3** (<2.0 or needs help) | **No / Confused** | **IV 20% dextrose 100mL** (or 10% 200mL); **IM glucagon 1mg** if no IV access; recheck 10 min |
| **Seizure/Coma** | No | **IV 20% dextrose 100mL** + **glucagon 1mg IM**; protect airway; **senior review**; stop insulin infusion |

**Post-treatment**: Observe 30–60 min; recheck glucose; identify cause; adjust regimen.

---

## 7. Reporting & Learning

| Step | Action |
|------|--------|
| **Immediate** | Treat patient; ensure safety; document on chart |
| **Within 24h** | Report via incident system (Datix); classify harm (none/low/moderate/severe/death) |
| **Within 72h** | **Root cause analysis** for Level 3 or recurrent Level 2 |
| **Monthly** | Ward/unit report: rates, themes, actions |
| **Quarterly** | Hospital-wide review; share learning; update protocols |

**Just culture** — focus on system failures, not individual blame.

## 8. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **Inpatient hypo definition** | Level 1: 3.0–3.9; Level 2: <3.0; Level 3: <2.0 or needs help |
| **Most common insulin error** | Wrong dose (especially 10x), wrong insulin, wrong timing |
| **Sliding scale** | **Supplemental only** — never sole regimen |
| **Never omit basal** | Reduce 20–50% if concerned; never hold without review |
| **NPO patient** | Basal insulin still needed (reduce 20–50%); dextrose if prolonged |
| **Renal impairment** | Reduce insulin 30–50% (eGFR <30); monitor q4–6h |
| **Concentrated insulin** | U200/U300/U500 — separate prescribing, dedicated syringes |
| **Level 3 treatment** | **IV 20% dextrose 100mL** + glucagon 1mg IM |
| **Reporting** | All Level 2/3 mandatory; root cause for Level 3 |
| **Target inpatient glucose** | **5–10 mmol/L** (avoid <4.0) |

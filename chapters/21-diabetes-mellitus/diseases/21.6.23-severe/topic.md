# DKA severity grading (mild/moderate/severe)

## 1. Learning Objectives
- [ ] Apply DKA severity grading (mild/moderate/severe) using pH, bicarbonate, ketones, mental status, potassium
- [ ] Triaging to appropriate care setting (ward vs HDU vs ICU)
- [ ] Adjust monitoring intensity by severity
- [ ] Recognise severe DKA complications requiring ICU

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Grading System** | JBDS (UK) / ADA (US) — similar cut-offs |
| **Mild** | pH 7.25–7.30, bicarb 15–18, alert, K+ >3.5 |
| **Moderate** | pH 7.00–7.24, bicarb 10–15, alert/drowsy, K+ 3.0–3.5 |
| **Severe** | pH <7.00, bicarb <10, drowsy/coma, K+ <3.0 |

## 3. Clinical Features / Presentation
| Feature | Mild | Moderate | Severe |
|---------|------|----------|--------|
| **Consciousness** | Alert | Drowsy | Coma/GCS <12 |
| **Respiratory rate** | Normal | Kussmaul | Kussmaul + fatigue |
| **Hemodynamics** | Stable | Tachycardic | Hypotensive, shock |
| **Potassium** | >3.5 | 3.0–3.5 | <3.0 (critical) |

## 4. Classification / Staging / Grading
| Parameter | Mild | Moderate | Severe |
|-----------|------|----------|--------|
| **Venous pH** | 7.25–7.30 | 7.00–7.24 | <7.00 |
| **Bicarbonate** | 15–18 | 10–15 | <10 |
| **Ketonaemia** | >3 | >3 | >3 |
| **Mental status** | Alert | Drowsy | Coma/GCS ≤12 |
| **Potassium** | >3.5 | 3.0–3.5 | <3.0 |
| **Care setting** | General ward | HDU | ICU |
| **Monitoring** | q2–4h | q1–2h | Continuous + arterial line |

## 5. Diagnosis & Investigations
| Investigation | Mild | Moderate | Severe |
|---------------|------|----------|--------|
| **VBG** | q4h | q2–4h | q1–2h / arterial line |
| **Glucose** | q1–2h | q1h | q1h |
| **Ketones** | q4h | q2–4h | q1–2h |
| **K+** | q2–4h | q1–2h | q1h / continuous |
| **ECG** | Baseline | q4–6h | Continuous |
| **Fluid balance** | q4h | q1–2h | Hourly |
| **Neurology (GCS)** | q4h | q2h | q1h / continuous |

(See DKA diagnosis criteria note — severity grading applies after diagnosis confirmed)

## 7. Management by Severity
| Aspect | Mild | Moderate | Severe |
|--------|------|----------|--------|
| **Location** | Medical ward | HDU | ICU |
| **Insulin** | FRII 0.1U/kg/hr | FRII 0.1U/kg/hr | FRII 0.1U/kg/hr + consider 0.14U/kg/hr if no improvement |
| **Fluids** | 0.9% NaCl per protocol | 0.9% NaCl + closer monitoring | 0.9% NaCl + CVP/arterial line + inotropes if shock |
| **K+ replacement** | 40mmol/L fluids | 40–60mmol/L, q1–2h checks | 60mmol/L, q1h checks, cardiac monitor |
| **Bicarbonate** | No | No | Yes (pH<7.0): 100mmol/2h |
| **Phosphate** | Oral if <0.5 | IV if <0.3 | IV if <0.3 |
| **Cerebral oedema watch** | Children: q1h neuro | Children: q30min neuro | Children: continuous ICU monitoring |

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **JBDS/ADA grading** | Mild: pH 7.25–7.30; Mod: 7.0–7.24; Sev: <7.0 |
| **Care setting** | Mild = ward; Moderate = HDU; Severe = ICU |
| **Severe DKA triggers** | pH <7.0, K+ <3.0, GCS <12, shock, oliguria, elderly/comorbid |
| **Monitoring frequency** | Increases with severity (see table) |
| **Bicarbonate indication** | ONLY severe (pH <7.0) |
| **Cerebral oedema** | Children <20y; rapid osmolar fall; mannitol 0.5–1g/kg; hyperventilation; ICU |

## 9. Viva Questions
| Question | Expected Answer |
|----------|-----------------|
| **How do you grade DKA severity?** | Mild: pH 7.25–7.30, bicarb 15–18, alert, K+ >3.5. Moderate: pH 7.0–7.24, bicarb 10–15, drowsy, K+ 3.0–3.5. Severe: pH <7.0, bicarb <10, coma/GCS<12, K+ <3.0. |
| **Where do you manage each grade?** | Mild: medical ward. Moderate: HDU. Severe: ICU. |
| **What monitoring frequency for severe DKA?** | Glucose q1h, ketones q1–2h, VBG/ABG q1–2h, K+ q1h, continuous ECG, hourly fluid balance, q1h GCS. |
| **When is bicarbonate indicated?** | ONLY in severe DKA with pH <7.0: 100mmol in 400ml water + 20mmol KCl over 2h. |
| **What is the cerebral oedema risk?** | Children <20y; rapid effective osmolar drop >3 mOsm/kg/hr; headache, bradycardia, hypertension, ↓GCS; treat with mannitol 0.5–1g/kg IV. |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **Severity based on glucose?** | NO — severity based on pH/bicarb/mental status/K+, NOT glucose level |
| **Moderate vs severe borderline** | pH 7.0 exactly = severe; GCS 12 = severe; K+ 3.0 = severe |

**Mnemonic: DKA-GRADE**
- **D**ecide care setting by grade
- **K**+ <3.0 = severe
- **A**cidosis: pH <7.0 = severe
- **G**CS <12 = severe
- **R**espiratory fatigue = severe
- **A**rterial line for severe
- **D**uration of monitoring ↑ with severity
- **E**scalate insulin if no improvement in severe

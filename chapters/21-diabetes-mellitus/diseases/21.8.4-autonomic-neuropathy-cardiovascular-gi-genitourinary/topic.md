# Autonomic Neuropathy (Cardiovascular, GI, Genitourinary)

> [!info]
> **Autonomic neuropathy = common, underdiagnosed**; CV: orthostatic hypotension, silent MI, exercise intolerance; GI: gastroparesis (nausea, vomiting, erratic glucose); GU: neurogenic bladder, ED; sudomotor: anhidrosis, gustatory sweating; **mortality ↑ 2–3×** (CV autonomic).

## 2. Cardiovascular Autonomic Neuropathy (CAN)

### Clinical Features
| Manifestation | Features |
|---------------|----------|
| **Orthostatic hypotension** | **SBP drop ≥20 / DBP ≥10** within 3min standing; syncope, falls, "coat-hanger" pain |
| **Resting tachycardia** | **HR >100 bpm** (denervation supersensitivity) |
| **Fixed heart rate** | Loss of respiratory sinus arrhythmia (RSA) |
| **Exercise intolerance** | Inadequate HR response (chronotropic incompetence) |
| **Silent myocardial ischaemia** | **Angina absent** in 25–50% of diabetic MI;-screen with ECG/stress echo |
| **Post-prandial hypotension** | SBP drop >20mmHg after meals (splanchnic pooling) |

### Screening Tests (Ewing Battery)
| Test | Normal | Abnormal (CAN) |
|------|--------|----------------|
| **Heart Rate Variability (HRV)** — deep breathing | **E:I ratio >1.15** | <1.10 (severe), 1.10–1.15 (moderate) |
| **30:15 Ratio** (standing) | **>1.04** | <1.00 |
| **Valsalva Ratio** | **>1.21** | <1.11 |
| **Orthostatic BP** | SBP drop <20, DBP <10 | **≥20/10 drop** |
| **BP Response to Sustained Handgrip** | DBP rise >16mmHg | <10mmHg |

**CAN Staging**: Early (1 abnormal), Definite (≥2), Severe (≥3 + orthostatic hypotension).

## 3. Gastrointestinal Autonomic Neuropathy

### Gastroparesis
| Feature | Details |
|---------|---------|
| **Symptoms** | Nausea, vomiting (undigested food), early satiety, bloating, **erratic glucose**, weight loss |
| **Diagnosis** | **Gastric emptying scintigraphy** (gold standard): 4h retention >10% (solids) / 4h >60% (liquids); **C13 breath test** alternative |
| **Exclude** | Mechanical obstruction (endoscopy/CT), medications (opioids, GLP-1 RA, anticholinergics) |

### Other GI Manifestations
| Condition | Symptoms |
|-----------|----------|
| **Diabetic diarrhoea** | Nocturnal, watery, incontinence; bacterial overgrowth common |
| **Constipation** | Slow transit; overflow diarrhoea possible |
| **Oesophageal dysmotility** | Dysphagia, reflux (impaired LES) |

### Gastroparesis Management
| Step | Intervention |
|------|--------------|
| **1. Dietary** | **Small frequent meals** (6/day); **low fat, low fibre**; liquid/blended; **avoid carbonated** |
| **2. Glycaemic optimisation** | Hyperglycaemia delays emptying; avoid hypoglycaemia (unpredictable absorption) |
| **3. Prokinetics** | **Metoclopramide** 10mg TDS (max 5 days — tardive dyskinesia risk); **Domperidone** 10mg TDS (QT caution); **Erythromycin** 250mg TDS (motilin agonist — tachyphylaxis); **Prucalopride** 1–2mg OD (5-HT4 agonist) |
| **4. Antiemetics** | Ondansetron, cyclizine, prochlorperazine |
| **5. Refractory** | **Gastric electrical stimulation (GES)** — Enterra Therapy; **Jejunostomy feeding**; **TPN** (last resort) |

## 4. Genitourinary Autonomic Neuropathy

### Neurogenic Bladder
| Stage | Features |
|-------|----------|
| **Early** | Impaired sensation → increased capacity, infrequent voiding |
| **Late** | Detrusor underactivity → retention, overflow incontinence, recurrent UTI |

**Diagnosis**: Post-void residual (PVR) >100mL (ultrasound); urodynamics if complex.

**Management**: Timed voiding, double voiding, **intermittent self-catheterisation (ISC)**, bethanechol (limited), sacral neuromodulation.

### Erectile Dysfunction (ED)
- **Prevalence**: 50–75% of diabetic men (3× non-diabetic); **earlier onset, more severe**
- **Aetiology**: Neurogenic + vasculogenic + endothelial (NO deficiency) + hormonal (low T)
- **Screening**: **SHIM/IIEF-5** questionnaire annually
- **Management**:
  1. **PDE5 inhibitors** (sildenafil, tadalafil, vardenafil) — **1st line**; check nitrate contraindication
  2. **Vacuum erection device**
  3. **Intracavernosal injections** (alprostadil)
  4. **Penile prosthesis** (refractory)

### Female Sexual Dysfunction
- Arousal, lubrication, orgasm difficulty; under-recognised
- Management: Lubricants, topical oestrogen, referral psychosexual

## 5. Sudomotor Dysfunction

| Manifestation | Features |
|---------------|----------|
| **Anhidrosis** | Dry skin (distal → proximal); fissures, ulcer risk |
| **Gustatory sweating** | **Facial/neck sweating** with eating (spicy/hot foods) — Frey's syndrome variant |
| **Compensatory hyperhidrosis** | Truncal sweating (proximal to anhidrotic area) |

---

## 6. Systemic Management Principles

| Domain | Action |
|--------|--------|
| **Glycaemic control** | DCCT: intensive → **50% ↓ CAN incidence**; target HbA1c <53 (7.0%) |
| **Risk factor modification** | BP <130/80, statin, smoking cessation, weight management |
| **Fall prevention** | Review meds (antihypertensives, sedatives); orthostatic training; hip protectors |
| **Exercise** | Supervised cardiac rehab; avoid Valsalva; monitor HR/BP |

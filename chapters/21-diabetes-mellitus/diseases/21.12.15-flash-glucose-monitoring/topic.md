## 1. Definition & Exam Relevance
**Flash Glucose Monitoring (isCGM)** = Intermittently scanned Continuous Glucose Monitoring. Sensor worn for 14 days; user scans with reader/phone to get glucose value + 8h trend graph.  
**Device**: FreeStyle Libre (Libre 2, Libre 3).  
**FCPS/MRCP relevance**: NICE NG17 recommends as option for T1DM; exam questions on technology, indications, limitations vs rtCGM.

---

## 2. Sensor Specifications
| Feature | Libre 1 | Libre 2 | Libre 3 |
|---------|---------|---------|---------|
| **Wear duration** | 14 days | 14 days | 14 days |
| **Scan method** | Reader/Phone NFC | Reader/Phone NFC + Bluetooth alerts | Phone only (Bluetooth) |
| **Alarms** | None | Optional hypo/hyper alerts | Real-time alerts |
| **Size** | Standard | Standard | **World's smallest** (2 pennies) |
| **Warm-up** | 1 hour | 1 hour | 1 hour |
| **Calibration** | Factory (none) | Factory (none) | Factory (none) |
| **Accuracy (MARD)** | ~11.4% | ~9.2% | ~7.9% |

---

## 3. Indications (NICE NG17 / NHS England)
- **All T1DM adults** — offer rtCGM first; isCGM if rtCGM not appropriate/preferred
- **T2DM on insulin** (multiple daily injection) — if recurrent/severe hypoglycaemia, impaired awareness, or need for intensive monitoring
- **Pregnancy with T1DM** — rtCGM preferred; isCGM if rtCGM unavailable
- **Children/YP with T1DM** — rtCGM first line

**NHS England criteria**: HbA1c ≥58 mmol/mol (7.5%) OR disabling hypoglycaemia OR impaired awareness OR pregnancy OR need for intensive insulin management.

## 4. Key Metrics & Reports
| Metric | Definition | Target (T1DM) |
|--------|------------|---------------|
| **Time in Range (TIR)** | 3.9–10 mmol/L | >70% (>16.8h/day) |
| **Time Below Range (TBR)** | <3.9 mmol/L | <4% (<1h/day) |
| **Time Below Range L2** | <3.0 mmol/L | <1% (<15 min/day) |
| **Time Above Range (TAR)** | >10 mmol/L | <25% (<6h/day) |
| **Time Above Range L2** | >13.9 mmol/L | <5% (<1.2h/day) |
| **Glucose Variability (CV)** | SD/Mean × 100 | <36% |
| **Glucose Management Indicator (GMI)** | Estimated HbA1c from CGM | Correlates with lab HbA1c |
| **Daily Scans** | User-initiated scans | >8 scans/day (LibreLink) |

**LibreView reports**: Ambulatory Glucose Profile (AGP), Daily Patterns, Time in Range, Sensor Usage, Glucose Variability.

---

## 5. isCGM vs rtCGM — Exam Differentiation
| Feature | isCGM (Flash) | rtCGM (Dexcom G6/G7, Guardian) |
|---------|---------------|--------------------------------|
| **Data transmission** | User-initiated scan | Automatic (Bluetooth) |
| **Real-time alarms** | Libre 2/3 only | Yes (all) |
| **Predictive alerts** | No | Yes |
| **Calibration** | None | G6: none; G7: none; Guardian: 2-4×/day |
| **Sensor life** | 14 days | G6: 10d; G7: 10d; Guardian: 7d |
| **Warm-up** | 1 hour | G6: 2h; G7: 30m; Guardian: 2h |
| **Acetaminophen interference** | No | G6: yes (falsely high); G7: minimal |
| **Cost (NHS)** | Lower | Higher |

---

## 6. Limitations & Artefacts
| Issue | Cause | Management |
|-------|-------|------------|
| **Compression artefact** | Lying on sensor → falsely low | Reposition; confirm with SMBG if symptomatic |
| **First 24h inaccuracy** | Sensor equilibration | Use SMBG for calibration decisions |
| **Last 24h drift** | Sensor degradation | Expect wider variation |
| **Temperature extremes** | Sauna/hot tub/cold exposure | Avoid; sensor range 10–45°C |
| **MRI/CT/X-ray** | Sensor damage | Remove before imaging |
| **Ascorbic acid (high dose)** | Falsely elevates | Avoid megadose vit C |

---

## 7. Clinical Application
- **Pattern review**: Weekly AGP review → basal/bolus adjustments
- **Hypoglycaemia prevention**: Trend arrows (↓, ↓↓) prompt pre-emptive carbs
- **Post-prandial review**: 2h scan after meals → bolus adjustment
- **Exercise**: Pre-exercise scan + trend → carb intake decision
- **Sick days**: Frequent scans + ketone check if >13.9 mmol/L

**Libre 2/3 alarms**: Set hypo alert at 3.9 mmol/L (default); customizable.

---

## 8. FCPS/MRCP Exam Pearls
- **MARD Libre 3**: ~7.9% (best in class for isCGM)
- **Warm-up**: 1 hour (vs 30 min–2h for rtCGM)
- **No calibration needed** — factory calibrated
- **Compression lows**: Classic trap — "patient wakes at 3am with 2.5 mmol/L on Libre but feels fine" → confirm with SMBG
- **NICE NG17**: rtCGM first line for T1DM; isCGM if patient prefers or rtCGM contraindicated
- **Driving**: Scan before driving (no alarm = must scan manually on Libre 1; Libre 2/3 have alarms)
- **Pregnancy**: rtCGM preferred; isCGM acceptable if rtCGM not available

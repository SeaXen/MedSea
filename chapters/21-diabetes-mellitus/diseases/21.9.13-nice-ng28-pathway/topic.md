# NICE NG28 Pathway (Type 2 Diabetes Management)

> [!info]
> **NICE NG28 (2022)** = UK guideline for T2DM management; **metformin first-line**; **comorbidity-driven** second-line (SGLT2i/GLP-1 RA if CVD/HF/CKD/obesity); **SU not first-line**; **basal insulin** if HbA1c >58 on dual therapy; **pre-mixed only** if basal-bolus not feasible; **individualised HbA1c**; **structured education at diagnosis**.

## 2. Stepwise Pharmacological Algorithm

### Step 1: First-Line — Metformin
| Detail | Recommendation |
|--------|----------------|
| **Drug** | **Metformin** (standard release) |
| **Titration** | 500mg OD → 500mg BD → 1g BD over 4–8 weeks |
| **Extended release** | If GI intolerance |
| **Contraindication** | eGFR <30 (stop); 30–45 (max 1g BD) |
| **If contraindicated/intolerant** | Consider **SGLT2i** or **GLP-1 RA** or **DPP-4i** or **Pioglitazone** |

### Step 2: Second-Line — **Comorbidity-Driven** (Add to Metformin)

| Clinical Scenario | **Preferred Add-On** | Alternative |
|-------------------|----------------------|-------------|
| **Established CVD** (MI, stroke, PAD) | **SGLT2i** (dapagliflozin/empagliflozin) **+ GLP-1 RA** | If SGLT2i/GLP-1 RA not suitable: **DPP-4i** (linagliptin) |
| **Heart Failure** (HFrEF or HFpEF) | **SGLT2i** (dapagliflozin/empagliflozin) **first** | GLP-1 RA if obese HFpEF (semaglutide 2.4mg) |
| **CKD** (eGFR <60 or ACR ≥30) | **SGLT2i** (dapa/empa ≥20) **+ GLP-1 RA** | **Finerenone** if ACR ≥30, eGFR ≥25 |
| **Obesity** (BMI ≥30; ≥27.5 South Asian) | **GLP-1 RA** (semaglutide/tirzepatide) | SGLT2i |
| **High hypoglycaemia risk** (elderly, renal, frail) | **SGLT2i** / **GLP-1 RA** / **DPP-4i** (linagliptin) / **Pioglitazone** | Avoid SU, insulin |
| **No specific comorbidity** | **SGLT2i** or **GLP-1 RA** (preferred) | **DPP-4i** (linagliptin) or **Pioglitazone** |
| **Cost primary concern** | **DPP-4i** (generic sitagliptin) / **Pioglitazone** | Sulfonylurea (gliclazide) |

**Key**: **Sulfonylureas NOT first-line add-on** — only if other options unsuitable.

### Step 3: Third-Line — Dual/Triple Therapy
- **Metformin + SGLT2i + GLP-1 RA** (ideal for CVD/HF/CKD/obesity)
- **Metformin + SGLT2i + DPP-4i** (if GLP-1 RA not tolerated)
- **Metformin + GLP-1 RA + DPP-4i** (if SGLT2i contraindicated)
- **Metformin + Pioglitazone + DPP-4i** (if others unavailable)

### Step 4: Insulin Initiation
| Trigger | Criteria |
|---------|----------|
| **Basal insulin** | HbA1c **>58 mmol/mol (7.5%)** on **dual therapy** (metformin + add-on) **AND** symptomatic hyperglycaemia OR HbA1c >75 (9%) |
| **Urgent insulin** | HbA1c **>86 (10%)** OR symptomatic (osmotic) OR ketosis |
| **Basal-bolus** | If basal optimised (fasting 4–7) but HbA1c >58 + post-prandial >8.5 |
| **Pre-mixed** | **Only if basal-bolus not feasible** (regular meals, no carb counting) |

## 3. Individualised HbA1c Targets

| Population | Target | Rationale |
|------------|--------|-----------|
| **Most adults** | **<48 mmol/mol (6.5%)** | If achievable without problematic hypoglycaemia |
| **High hypoglycaemia risk / frailty / limited life expectancy** | **<58 mmol/mol (7.5%)** | Balance benefit vs harm |
| **Long duration / advanced complications** | **<64 mmol/mol (8.0%)** | Avoid intensive control risks (ACCORD) |
| **Pregnancy (pre-existing)** | **<48 mmol/mol (6.5%)** | Pre-conception and throughout |
| **Children/Young people** | **<48 mmol/mol (6.5%)** | If achievable safely |

**Relax targets** if: hypoglycaemia unawareness, recurrent severe hypo, advanced frailty, limited life expectancy.

## 4. Structured Education (NICE Quality Standard)

| Requirement | Detail |
|-------------|--------|
| **Offer at diagnosis** | **T1DM**: DAFNE; **T2DM**: DESMOND/X-PERT |
| **Timing** | Within **12 months** of diagnosis |
| **Annual review** | Assess need for refresher at each annual review |
| **Cultural adaptation** | Available in appropriate languages |
| **Digital options** | Can supplement (DAFNE Online, Oviva, Liva) if accessible |

## 5. Monitoring & Annual Review (9 Care Processes)

| Care Process | Target / Action |
|--------------|-----------------|
| **HbA1c** | 3–6 monthly until stable; then 6-monthly |
| **Blood Pressure** | <140/90 (<130/80 if CKD/albuminuria) |
| **Lipids** | Non-HDL <2.6 (primary) / <2.2 (secondary); statin per QRISK3 |
| **Urine ACR** | Annual (6-monthly if A2/A3) |
| **eGFR** | Annual (6-monthly if <60) |
| **BMI** | Annual; weight management if ≥25 |
| **Smoking** | Annual; cessation support |
| **Foot examination** | Annual (neurovascular); risk stratification (IWGDF 0–3) |
| **Retinal screening** | Annual (2-yearly if R0 ×2); refer per ETDRS grading |

## 6. Drug-Specific NICE Positioning

| Drug Class | NICE Position | Key Criteria |
|------------|---------------|--------------|
| **Metformin** | **First-line** | Universal |
| **SGLT2i** | **Preferred 2nd-line** | If CVD/HF/CKD/obesity |
| **GLP-1 RA** | **Preferred 2nd-line** | If CVD/HF/CKD/obesity |
| **DPP-4i** | **Alternative** | If SGLT2i/GLP-1 RA not suitable; linagliptin preferred (no renal adjust) |
| **Pioglitazone** | **Alternative** | If others unsuitable; **avoid HF/fracture/bladder cancer** |
| **Sulfonylurea** | **Not first/second-line** | Only if others unsuitable; gliclazide preferred |
| **Meglitinides** | **Specialist** | Irregular meals, renal impairment |
| **Alpha-glucosidase** | **Specialist** | Post-prandial dominant, post-bariatric dumping |
| **Basal insulin** | **Step 4** | HbA1c >58 on dual therapy |
| **Basal-bolus** | **Step 5** | Basal optimised + HbA1c >58 |
| **Pre-mixed** | **Only if basal-bolus not feasible** | Regular meals, no carb counting |

## 7. Special Populations

| Population | Key Modifications |
|------------|-------------------|
| **Renal impairment (eGFR <30)** | Stop metformin; **SGLT2i (dapa/empa ≥20), GLP-1 RA, DPP-4i (linagliptin)** |
| **Heart failure** | SGLT2i first; **avoid TZD, saxagliptin/alogliptin** |
| **Elderly/Frail** | Relaxed HbA1c (<58–64); **avoid SU/insulin if possible**; SGLT2i/GLP-1 RA/DPP-4i |
| **Pregnancy** | **Insulin** (detemir/glargine U100, aspart/lispro); stop oral agents (except metformin if ADA) |
| **South Asian ethnicity** | BMI thresholds **−2.5**; earlier intervention |

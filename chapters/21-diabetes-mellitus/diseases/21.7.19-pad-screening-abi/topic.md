# PAD Screening (ABI)

> [!info]
> **ABI = Ankle-Brachial Index** = highest ankle SBP / highest brachial SBP; **screen diabetics ≥50y** (or <50y with risk factors); **diabetes calcifies vessels → ABI >1.3 = non-compressible** (use TBI); **<0.9 = PAD**; **<0.4 (0.5 in DM) = CLI**.

## 2. ABI Technique

### Equipment
- **Doppler probe** (8–10 MHz) + sphygmomanometer
- **Patient**: Supine, rested 10 minutes, room warm

### Procedure
1. **Brachial pressures**: Measure **both arms** (right & left) — use **higher** as reference
2. **Ankle pressures**: Measure **dorsalis pedis (DP)** and **posterior tibial (PT)** on **each leg**
3. **Calculation per leg**: 
   - **ABI = Highest ankle SBP (DP or PT) / Highest brachial SBP**
4. **Record**: Right ABI, Left ABI

### Interpretation (Standard)

| ABI Value | Interpretation |
|-----------|----------------|
| **>1.3** (or >1.4) | **Non-compressible arteries** (medial calcification) — **use TBI** |
| **1.0–1.3** | **Normal** |
| **0.9–1.0** | **Borderline** — consider further testing if symptomatic |
| **0.4–0.9** | **PAD** (mild 0.7–0.9; moderate 0.4–0.7) |
| **<0.4** (or **<0.5 in diabetes**) | **Critical Limb Ischaemia (CLI)** |

---

## 3. Diabetes-Specific Adjustments

| Issue | Impact | Solution |
|-------|--------|----------|
| **Medial arterial calcification** (Mönckeberg's) | Vessels non-compressible → **falsely high ABI** (>1.3) | **Use Toe-Brachial Index (TBI)** |
| **Threshold for CLI** | Standard <0.4; **diabetes <0.50** (more sensitive) | Use lower threshold |
| **Sensitivity in diabetes** | ABI <0.9 misses ~30% PAD (calcification) | TBI + clinical assessment |

### TBI (Toe-Brachial Index)
- **Method**: Photoplethysmography (PPG) cuff on **great toe** + brachial
- **Normal**: **>0.70**
- **PAD**: **<0.70**
- **CLI**: **<0.25** (or <0.30 in diabetes)
- **Advantage**: Toe arteries **do not calcify** (no media)

---

## 4. Screening Recommendations (NICE / ESC / ADA)

| Population | Screening Frequency |
|------------|---------------------|
| **All T2DM ≥50y** | **Baseline + annual** (or 2-yearly if normal) |
| **T1DM ≥50y or ≥10y duration** | Baseline + annual |
| **Diabetes <50y + risk factors** | Baseline + risk-based |
| **Risk factors**: smoking, HTN, dyslipidaemia, CKD, prior CVD, foot ulcer | |
| **Prior PAD / revascularisation** | **Annual** |
| **Prior amputation** | **Annual** (contralateral surveillance) |

**NICE CG147**: Consider ABI in diabetes with leg symptoms or foot ulcer.

---

## 5. Alternative / Adjunctive Tests

| Test | Indication | Advantage |
|------|------------|-----------|
| **TBI** | ABI >1.3 (non-compressible); CLI confirmation | Not affected by calcification |
| **TcPO2** | CLI assessment; wound healing prediction | Microcirculation; <30 mmHg = poor healing |
| **Duplex Ultrasound** | Anatomic localisation; planning revascularisation | Non-invasive, real-time |
| **CT Angiography** | Pre-procedural planning; complex anatomy | High resolution; contrast nephropathy risk |
| **MR Angiography** | Contrast allergy; CKD (gadolinium caution) | No ionising radiation |
| **Pulse Volume Recording (PVR)** | Qualitative waveform; ABI unmeasurable | Simple, bedside |

---

## 6. Management Based on ABI

| ABI Result | Action |
|------------|--------|
| **>1.3** | **Repeat with TBI**; assess clinically; consider CTA if symptomatic |
| **1.0–1.3** | Reassure; annual rescreen; risk factor optimisation |
| **0.9–1.0** | Borderline; repeat in 6–12 months; optimise risk factors |
| **0.7–0.9** | **Mild PAD** — risk factor modification (statin, antiplatelet, BP, smoking, exercise) |
| **0.4–0.7** | **Moderate PAD** — vascular referral; supervised exercise; consider imaging |
| **<0.4 (<0.5 DM)** | **CLI** — **urgent vascular MDT**; WIfI classification; revascularisation |

---

## 7. Diabetic Foot Integration

| Foot Risk (IWGDF) | ABI Role |
|-------------------|----------|
| **Risk 0–1** | Annual screen |
| **Risk 2** (LOPS + PAD) | ABI confirms PAD; 3–6m podiatry MDT; vascular review |
| **Risk 3** (prior ulcer) | ABI/TBI annually; surveillance for CLI |
| **Active ulcer** | ABI/TBI at presentation; urgent vascular if CLI |

---

## 8. Limitations of ABI

| Limitation | Impact |
|------------|--------|
| **Calcification (ABI >1.3)** | Misses PAD in 30% diabetics — **TBI mandatory** |
| **Operator dependency** | Requires training; standardise protocol |
| **Resting only** | May miss functional limitation — **exercise ABI** if equivocal |
| **Inflow vs outflow** | Does not localise — duplex/CTA needed for intervention |

# HHS vs DKA Differentiation

> [!info]
> **HHS vs DKA** = spectrum of hyperglycaemic emergencies; **HHS**: glucose >30, minimal ketones, pH >7.3, older, mortality 10–20%; **DKA**: glucose 14–30, high ketones, pH <7.3, younger, mortality 1–2%; **Mixed (30–40%)** = features of both; treat based on dominant pathology.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Differentiate HHS and DKA using biochemical and clinical criteria
- [ ] Identify mixed HHS/DKA presentations
- [ ] Apply treatment priorities for each (fluids first both; insulin urgency differs)
- [ ] Recognise euDKA as special DKA variant
- [ ] Avoid diagnostic pitfalls

---

## 2. Diagnostic Criteria Comparison

| Parameter | **HHS** | **DKA** | **Mixed** |
|-----------|---------|---------|-----------|
| **Plasma Glucose** | **>30 mmol/L** (often 40–60) | **14–30 mmol/L** (rarely >40) | Variable |
| **β-hydroxybutyrate** | **<3 mmol/L** (often <1) | **≥3 mmol/L** (often >5) | **≥3 mmol/L** |
| **Ketonuria** | Negative / trace | **+++ / ++++** | Positive |
| **Venous pH** | **>7.3** | **<7.3** (often <7.1) | **<7.3** |
| **Bicarbonate** | **>15–18 mmol/L** | **<15 mmol/L** | <15 mmol/L |
| **Anion Gap** | Normal / mildly elevated | **>10** (often >20) | >10 |
| **Effective Osmolality** | **>320 mOsm/kg** | 290–320 | >320 |
| **Serum Na+ (corrected)** | **>145** (often >150) | Normal / low (pseudohyponatraemia) | Often elevated |

---

## 3. Clinical Presentation Comparison

| Feature | **HHS** | **DKA** |
|---------|---------|---------|
| **Age** | **>60y** (median 70) | **<50y** (median 30–40) |
| **Diabetes Type** | **T2DM** (>90%) | **T1DM** (50–60%), T2DM |
| **Onset** | **Days–weeks** (gradual) | **Hours** (acute) |
| **Consciousness** | **Confusion, obtundation, coma** | **Alert** (unless severe/cerebral oedema) |
| **Dehydration** | **Severe** (8–10L deficit) | Moderate (4–6L) |
| **Kussmaul Respiration** | Absent / mild | **Prominent** |
| **Abdominal Pain** | Uncommon | **Common** (often surgical mimic) |
| **Precipitant** | Infection, MI, stroke, drugs, renal failure | Missed insulin, infection, new-onset |

---

## 4. Mixed HHS/DKA (30–40% of Hyperglycaemic Emergencies)

**Definition**: Meets criteria for **BOTH** HHS (glucose >30, osmolality >320) **AND** DKA (ketones ≥3, pH <7.3, bicarb <15).

| Feature | Mixed Presentation |
|---------|-------------------|
| **Glucose** | Very high (>30, often >40) |
| **Ketones** | High (≥3 mmol/L) |
| **pH** | <7.3 |
| **Osmolality** | >320 |
| **Patient** | Often older T2DM with severe precipitant (sepsis, MI, stroke) |
| **Mortality** | **5–10%** (intermediate) |
| **Treatment** | **Fluids + insulin** (both urgent); monitor K+ closely (shift from HHS hyperK to hypoK) |

**Key**: In mixed cases, **insulin is more urgent** than pure HHS (due to ketoacidosis).

---

## 5. Treatment Priorities — Key Differences

| Priority | **HHS** | **DKA** | **Mixed** |
|----------|---------|---------|-----------|
| **1. Fluids** | **URGENT** (1–2L 0.9% NaCl first hour) | **URGENT** (1–2L first hour) | **URGENT** |
| **2. Insulin** | **DELAYED** (start after 1–2L fluids; 0.05u/kg/h) | **EARLY** (start with fluids; 0.1u/kg/h) | **EARLY** (like DKA) |
| **3. K+ Management** | Often **high initially** (shift from acidosis/renal); monitor closely | **High initially** → rapid **fall** with insulin; aggressive replacement | **Both patterns** — monitor 1–2h |
| **4. Bicarbonate** | **Never** (no acidosis) | **Rarely** (pH <6.9 only) | **Rarely** (pH <6.9) |
| **5. Glucose Target** | **Slow fall** (3–5 mmol/L/h; avoid cerebral oedema) | **3–5 mmol/L/h** | **3–5 mmol/L/h** |
| **6. Dextrose** | Start when glucose <14 | Start when glucose <14 | Start when glucose <14 |

**Why insulin differs**: HHS = no significant ketoacidosis → fluids alone ↓ glucose (osmotic diuresis resumes); insulin risks hypokalaemia/rapid osmolality drop. DKA = ketoacidosis requires insulin to stop ketogenesis.

---

## 6. Diagnostic Pitfalls

| Pitfall | Reality |
|---------|---------|
| **"Glucose 25 = DKA"** | Could be HHS with mild ketosis or mixed |
| **"pH 7.25 = DKA"** | Could be HHS with mild lactic acidosis (sepsis) |
| **"Older T2DM = HHS"** | T2DM can have DKA (especially SGLT2i, severe illness) |
| **"No ketonuria = not DKA"** | **Check blood β-hydroxybutyrate** — urine misses β-OHB |
| **"Euglycaemia = not DKA"** | **Euglycaemic DKA** (SGLT2i) — glucose <14 |
| **"Normal Na+ = normal osmolality"** | **Correct Na+** for glucose: Na+corr = Na+ + (Glucose−5.5)×0.4 |

---

## 7. Euglycaemic DKA — Special Variant

| Feature | Typical DKA | **Euglycaemic DKA** |
|---------|-------------|---------------------|
| **Glucose** | >14 | **<14 (8–13)** |
| **SGLT2i** | Uncommon | **Majority** |
| **Diagnosis** | Easy (high glucose) | **Delayed** (glucose "reassuring") |
| **Key Action** | Standard | **Stop SGLT2i; dextrose earlier** |

## 9. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **HHS glucose** | **>30 mmol/L** |
| **DKA glucose** | **14–30 mmol/L** |
| **HHS ketones** | **<3 mmol/L** |
| **DKA ketones** | **≥3 mmol/L** |
| **HHS pH** | **>7.3** |
| **DKA pH** | **<7.3** |
| **HHS mortality** | **10–20%** |
| **DKA mortality** | **1–2%** |
| **Mixed HHS/DKA** | 30–40%; treat with **early insulin + fluids** |
| **Insulin: HHS** | Delayed (after 1–2L fluids) |
| **Insulin: DKA** | Early (with fluids) |

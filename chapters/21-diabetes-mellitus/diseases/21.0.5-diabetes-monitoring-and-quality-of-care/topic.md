## High-Yield Exam Points (FCPS/MRCP)

| Topic | Key Facts |
|-------|-----------|
| **HbA1c targets (individualised)** | Standard: <53mmol/mol (7.0%) · Tighter: <48mmol/mol (6.5%) if achievable without hypo · Looser: <58-64mmol/mol (7.5-8.0%) if elderly/frail/comorbidities/hypo risk |
| **TIR targets** | >70% time 70-180mg/dL (3.9-10.0mmol/L) · <4% <54mg/dL (3.0mmol/L) · <1% <54mg/dL · CV <36% |
| **SMBG frequency** | Type 1/insulin: pre-meal, bedtime, pre-driving, suspected hypo (4-10x/day) · Type 2 non-insulin: individualised |
| **Annual review (9 care processes)** | 1) HbA1c 2) BP 3) Cholesterol 4) Retinopathy screening 5) Foot check 6) ACR 7) eGFR 8) BMI 9) Smoking |
| **Retinopathy screening** | Digital photography · Annual if no DR · Refer if R2/R3 or M1 (maculopathy) |
| **Nephropathy screening** | ACR (spot urine) + eGFR (CKD-EPI) · ACR <3 (A1), 3-30 (A2), >30 (A3) mg/mmol |
| **Neuropathy/foot screening** | 10g monofilament, vibration (128Hz tuning fork), ANNS · IWGDF risk classification |
| **CV risk assessment** | QRISK3 (UK) / ASCVD Pooled Cohort (US) · Statin if 10-year risk >10% or age 40-75 with DM |
| **DAFNE** | Dose Adjustment For Normal Eating · 5-day structured education for type 1 · Carb counting, insulin adjustment |
| **DESMOND/X-PERT** | Structured education for type 2 · Self-management, lifestyle, medication understanding |
| **Diabetes distress** | PAID scale, DDS · Not depression · Address with education, support, shared decision-making |
| **Depression in DM** | 2x risk · PHQ-9 screening · SSRIs 1st line · Collaborative care model |
| **Eating disorders (diabulimia)** | Insulin omission for weight loss · DKA recurrent · High mortality · Multidisciplinary (psych, diabetes, dietitian) |

# HHS Mortality and Prognosis

> [!info]
> **HHS mortality 10–20%** (vs DKA 1–2%); **older, comorbid, delayed presentation**; independent predictors: age >60, coma (GCS <14), hypotension, pH <7.2, Na+ >160, K+ >6, renal failure (Cr >200), Osm >350; **early recognition + aggressive fluids + treat precipitant** key to survival.

## 2. Mortality Data

| Population | In-Hospital Mortality | Key Difference |
|------------|----------------------|----------------|
| **HHS** | **10–20%** (up to 25–30% in some series) | Older, comorbid, delayed Dx |
| **DKA** | **1–2%** (0.5% in specialised centres) | Younger, earlier presentation |
| **Mixed HHS/DKA** | 5–10% | Intermediate |

**Mortality has not significantly improved** in 20 years — reflects population aging and comorbidity burden.

---

## 3. Independent Mortality Predictors (Multivariate Analyses)

| Predictor | Threshold | Mechanism |
|-----------|-----------|-----------|
| **Age** | **>60–65 years** | Comorbidity, reduced reserve |
| **Coma / Altered consciousness** | **GCS <14** (or <12) | Cerebral oedema, osmotic CNS injury |
| **Hypotension** | **SBP <90 mmHg** | Severity of dehydration; cardiovascular collapse |
| **Acidosis** | **pH <7.2** (or HCO3 <15) | Mixed HHS/DKA; cellular dysfunction |
| **Hypernatraemia** | **Na+ >160 mmol/L** (corrected) | Degree of free water deficit; cerebral dehydration |
| **Hyperkalaemia** | **K+ >6.0 mmol/L** | Renal failure; arrhythmia risk |
| **Renal failure** | **Creatinine >200–250 µmol/L** (or eGFR <30) | Multi-organ failure; impaired clearance |
| **Effective osmolality** | **>350 mOsm/kg** | Osmotic CNS injury |
| **Precipitant** | **Sepsis, MI, stroke, PE** | Underlying lethal illness |
| **Delayed presentation** | >24h symptoms | Established organ failure |

**Combined scores**: Some centres use **HHS mortality score** (age + GCS + SBP + osmolality + renal function) but none universally validated.

---

## 4. Causes of Death in HHS

| Immediate Cause | Proportion | Notes |
|-----------------|------------|-------|
| **Cardiovascular collapse** (MI, arrhythmia, HF) | **40–50%** | Dehydration, electrolyte shifts, underlying CAD |
| **Sepsis / Septic shock** | **20–30%** | Precipitant (UTI, pneumonia, abdominal); immunocompromised |
| **Cerebral complications** (oedema, thrombosis, haemorrhage) | **10–15%** | Osmotic shifts, hypercoagulability |
| **Renal failure** | **5–10%** | ATN, cortical necrosis, contrast |
| **Thromboembolism** (PE, DVT) | **5%** | Hypercoagulable state; prophylaxis essential |
| **Respiratory failure** | **<5%** | Aspiration, ARDS |

**Most deaths occur within 24–48h** of admission.

---

## 5. Prognostic Factors — Favourable vs Unfavourable

| Favourable | Unfavourable |
|------------|--------------|
| Age <65 | Age >75 |
| Alert (GCS 15) | Coma (GCS <12) |
| SBP >100 | SBP <90 (despite fluids) |
| pH >7.3 | pH <7.2 |
| Na+ <150 | Na+ >160 |
| K+ 3.5–5.5 | K+ >6.0 or <3.0 |
| Creatinine <150 | Creatinine >250 |
| Osmolality <320 | Osmolality >350 |
| No precipitant / minor | Sepsis, MI, stroke, PE |
| No comorbidities | Multiple (HF, CKD, COPD, cancer) |

---

## 6. Early Interventions to Reduce Mortality

| Intervention | Evidence |
|--------------|----------|
| **Early recognition** (high index of suspicion in elderly with confusion) | Reduces time to treatment |
| **Aggressive IVF resuscitation** (0.9% NaCl 1–2L first hour) | Corrects hypovolaemia, improves renal perfusion |
| **Early insulin** (after 1–2h fluids, 0.05u/kg/h) | Prevents ongoing catabolism; avoid premature |
| **Treat precipitant** (antibiotics, PCI, stroke unit) | Addresses primary driver of mortality |
| **Electrolyte management** (K+, phosphate, Mg2+) | Prevents arrhythmias |
| **Thromboprophylaxis** (LMWH) | All HHS admissions — hypercoagulable |
| **Avoid rapid Na+ correction** | Max 10–12 mmol/L/24h (prevent central pontine myelinolysis) |
| **Monitor neurological status** (hourly GCS) | Early detection of cerebral oedema/deterioration |
| **ICU/HDU admission** if GCS <14, SBP <90, pH <7.2, K+ >6, Cr >200 | Higher nursing ratio, invasive monitoring |

---

## 7. Long-Term Outcomes in Survivors

| Outcome | Data |
|---------|------|
| **1-year mortality** | **20–30%** (cardiovascular, sepsis, cancer) |
| **Recurrent HHS** | **5–10%** (often same precipitant) |
| **Functional decline** | Common in elderly (delirium, deconditioning) |
| **New-onset diabetes diagnosis** | ~30% of HHS = first presentation |
| **Quality of life** | Reduced in elderly survivors |

---

## 8. Comparison: HHS vs DKA Mortality

| Factor | HHS | DKA |
|--------|-----|-----|
| **Mortality** | **10–20%** | **1–2%** |
| **Median age** | **65–75y** | **30–40y** |
| **Comorbidities** | Multiple (CVD, CKD, HF) | Few |
| **Precipitant** | Often occult (silent MI, sepsis) | Obvious (infection, missed insulin) |
| **Presentation delay** | Days (gradual) | Hours (acute) |
| **Cerebral oedema** | Rare | 0.5–1% (children) |
| **Key preventable** | Early fluids + treat precipitant | Early insulin + fluids + K+ |

# Basal Insulin Initiation

> [!info]
> **Basal insulin = first injectable for T2DM** failing oral therapy; **glargine U100 / detemir / degludec** preferred; start **10 units or 0.1–0.2 u/kg**; **titrate 2–4 units weekly** to fasting target (4–7 mmol/L); **once daily** (same time); combine with metformin ± SGLT2i/DPP-4i; stop sulfonylurea.

## 2. When to Initiate Basal Insulin (NICE NG28 / ADA)

| Trigger | Criteria |
|---------|----------|
| **HbA1c above target** | **>58 mmol/mol (7.5%)** on **dual/triple oral therapy** (metformin + SU ± DPP-4i/SGLT2i/pioglitazone) |
| **Symptomatic hyperglycaemia** | Osmotic symptoms despite oral therapy |
| **Contraindication to orals** | CKD G4–5 (eGFR <30), recurrent hypoglycaemia on SU, intolerance |
| **Specific populations** | Pregnancy (pre-existing T2DM), steroid-induced, post-transplant |

**Not first-line**: Newly diagnosed T2DM (lifestyle + metformin first).

---

## 3. Basal Insulin Choice

| Insulin | Duration | Profile | Advantages | Dosing |
|---------|----------|---------|------------|--------|
| **Glargine U100** (Lantus, Biosimilars) | **24h** | Flat, peakless | **Most evidence**; cheap (biosimilars); once daily | Once daily (any time, consistent) |
| **Detemir** (Levemir) | **16–24h** | Flat; slightly lower weight gain | **Unit-for-unit ~same as glargine**; predictable | Once or **twice daily** |
| **Degludec** (Tresiba) | **>42h** | Ultra-long, flat | **Flexible timing** (±8h); lowest hypoglycaemia (esp. nocturnal); U100/U200 | Once daily (any time) |
| **Glargine U300** (Toujeo) | **>24h** | More concentrated, flatter | Lower hypoglycaemia vs U100; U300 pen | Once daily |
| **NPH (Isophane)** | **12–16h** | Pronounced peak (4–8h) | Cheap; **higher nocturnal hypo**; twice daily | Twice daily |

**NICE/ADA preference**: **Glargine U100, detemir, or degludec** (long-acting analogues) over NPH.

---

## 4. Starting Dose

| Method | Calculation |
|--------|-------------|
| **Weight-based** | **0.1–0.2 units/kg** (e.g., 80kg → 8–16 units) |
| **Fixed** | **10 units** (most guidelines) |
| **High HbA1c (>86/10%)** | 0.2–0.3 u/kg or 20 units |

**Practical**: Start **10 units once daily** (bedtime or morning) for most; adjust for weight/renal function.

---

## 5. Titration Algorithm (NICE / ADA Consensus)

### Target
- **Fasting glucose 4–7 mmol/L** (individualise: 5–8 if hypoglycaemia risk)

### Titration Schedule
| Approach | Dose Adjustment | Frequency |
|----------|-----------------|-----------|
| **Weekly (standard)** | **Fasting mean** (3 days): <4 → ↓ 2–4u; 4–7 → no change; 7–10 → ↑ 2–4u; >10 → ↑ 4–6u | Weekly |
| **Self-titration (patient-led)** | **Treat-to-target**: FBS <4 → ↓ 2u; 4–5 → no change; 5.1–7 → ↑ 1u; 7.1–10 → ↑ 2u; >10 → ↑ 4u | Every 3 days |
| **Automated (connected pen/app)** | Algorithm-driven | Per protocol |

**Key**: **Nocturnal hypoglycaemia** → reduce dose 10–20%; **persistent fasting >10** → review adherence, diet, add bolus.

---

## 6. Combination Therapy with Basal Insulin

| Oral Agent | Continue? | Adjustment |
|------------|-----------|------------|
| **Metformin** | **Yes** (first-line) | Continue full dose |
| **SGLT2i** | **Yes** | Continue (synergistic weight/glucose benefit) |
| **GLP-1 RA** | **Yes** (add-on) | Reduce basal 10–20% if added |
| **DPP-4i** | **Yes** | Continue |
| **Pioglitazone** | If needed | Caution: weight gain, oedema, HF |
| **Sulfonylurea** | **Stop** (hypoglycaemia risk) | Taper over 2–4 weeks |
| **Meglitinide** | Stop | — |
| **Acarbose** | Stop | — |

---

## 7. Monitoring & Safety

| Parameter | Frequency |
|-----------|-----------|
| **Fasting glucose** | Daily (self-monitored); review weekly |
| **HbA1c** | 3 months post-initiation; then 6-monthly |
| **Hypoglycaemia** | Review at each visit; ask about nocturnal |
| **Weight** | Monthly (basal insulin ~2–4kg gain) |
| **Injection technique** | Review annually (site rotation, needle length) |

---

## 8. Special Populations

| Population | Adjustments |
|------------|-------------|
| **Elderly / Frail** | Start 5–8u; target fasting 6–8; degludec preferred (flexible timing) |
| **CKD G3–G4** | Start low (5–8u); detemir/glargine U100 OK; **degludec caution** (limited data); monitor |
| **CKD G5 / Dialysis** | Insulin clearance ↓ → **reduce 30–50%**; monitor closely |
| **Pregnancy** | **Detemir/glargine U100** (Category B); tight targets; frequent titration |
| **Ramadan** | Basal at iftar; reduce 20–30%; monitor |

# Critical Limb Ischaemia (CLI)

> [!info]
> **CLI = rest pain >2 weeks + tissue loss (ulcer/gangrene) + objective arterial disease**; Rutherford 4–6 / Fontaine III–IV; **amputation risk 25–40% at 1 year revascularisation**; **urgent vascular MDT**; WIfI classification guides therapy; diabetes accelerates PAD and worsens outcomes.

## 2. Definition & Classification

### Rutherford Classification (Clinical)
| Category | Clinical Picture |
|----------|------------------|
| **0** | Asymptomatic |
| **1** | Mild claudication |
| **2** | Moderate claudication |
| **3** | Severe claudication |
| **4** | **Ischaemic rest pain** (typically nocturnal, relieved by dependency) |
| **5** | **Minor tissue loss** (non-healing ulcer, focal gangrene) |
| **6** | **Major tissue loss** (extensive gangrene, functional foot loss) |

**CLI = Rutherford 4, 5, 6**

### Fontaine Classification
| Stage | Clinical Picture |
|-------|------------------|
| **I** | Asymptomatic |
| **IIa** | Mild claudication (>200m) |
| **IIb** | Severe claudication (<200m) |
| **III** | **Rest pain** |
| **IV** | **Ulceration/gangrene** |

**CLI = Fontaine III–IV**

---

## 3. Diagnostic Criteria (ESC / SVS)

**CLI = ALL THREE**:
1. **Symptoms**: Rest pain **>2 weeks** (typically nocturnal, dorsal foot, relieved by dependency) **OR** tissue loss (ulcer/gangrene)
2. **Haemodynamic evidence** (ONE of):
   - **ABI ≤0.40** (or **<0.50** with diabetes/medial calcification)
   - **Ankle pressure <50 mmHg** (or <70 mmHg with diabetes)
   - **Toe pressure <30 mmHg** (or TBI <0.25)
   - **TcPO2 <30 mmHg** (or <40 mmHg with diabetes)
3. **Arterial disease** on imaging (duplex/CTA/MRA) amenable to revascularisation

**Diabetes adjustment**: Use **lower thresholds** (ABI <0.50, toe pressure <40 mmHg) due to medial calcification.

---

## 4. WIfI Classification (SVS) — **Amputation Risk Stratification**

| Component | Grade 0 | Grade 1 | Grade 2 | Grade 3 |
|-----------|---------|---------|---------|---------|
| **Wound** | None | Small, shallow ulcer; no gangrene | Deep ulcer, exposed bone; gangrene <2 toes | Extensive gangrene; forefoot/heel loss |
| **Ischaemia** | ABI >0.80, TP >60 | ABI 0.6–0.8, TP 40–60 | ABI 0.4–0.6, TP 30–40 | ABI <0.4, TP <30, TcPO2 <30 |
| **Foot Infection** | None | Local (≤2cm erythema) | Deep (≥2cm, abscess, osteomyelitis) | Sepsis, necrotising fasciitis |

**WIfI Clinical Stage** (combines W+I+fI) → **1-year amputation risk & revascularisation benefit**:
| Stage | Amputation Risk | Revascularisation Benefit |
|-------|-----------------|---------------------------|
| **1 (Very Low)** | <5% | Low |
| **2 (Low)** | 5–10% | Moderate |
| **3 (Moderate)** | 10–40% | **High** |
| **4 (High)** | **>40%** | **High** (but risk remains) |

---

## 5. Management — Urgent Vascular MDT

### Principles
- **Revascularisation = primary therapy** (endovascular first for most)
- **Wound care** = offloading, debridement, infection control (MDT)
- **Medical optimisation** = statin, antiplatelet, BP, glucose, smoking cessation

### Revascularisation Strategy (BASIL-2, BEST-CLI)

| Lesion Pattern | Preferred Approach |
|----------------|-------------------|
| **TASC A/B** (short, focal) | **Endovascular first** (angioplasty ± stent) |
| **TASC C/D** (long, diffuse, tibial) | **Endovascular first** (modern data); bypass if endo fails/anatomy unsuitable |
| **Inflow disease** (aortoiliac) | Endovascular (iliac stenting) |
| **Outflow disease** (tibial/pedal) | Endovascular (pedal loop technique) |

**Best outcomes**: **Multidisciplinary vascular MDT** (vascular surgeon, interventional radiologist, diabetologist, podiatrist).

### Medical Therapy
| Drug | Dose | Indication |
|------|------|------------|
| **Aspirin** | 75mg OD | **All CLI** (unless contraindicated) |
| **Clopidogrel** | 75mg OD | If aspirin intolerant; **or DAPT post-endovascular** |
| **Statin** | Atorvastatin 80mg | **High-intensity** — pleiotropic stabilisation |
| **Rivaroxaban 2.5mg BD + Aspirin** | VOYAGER PAD | **Post-revascularisation** (↓ ALI/amputation) |
| **Cilostazol** | 100mg BD | Symptomatic claudication (not CLI) |

---

## 6. Amputation — Indications & Levels

| Indication | Level |
|------------|-------|
| **Non-reconstructable disease** (no target vessel) | Most distal viable level |
| **Life-threatening sepsis** (wet gangrene, necrotising) | **Guillotine** (emergency) → definitive later |
| **Failed revascularisation** (persistent rest pain/tissue loss) | Definitive level per perfusion |
| **Non-ambulatory / non-reconstructable** | AKA (if popliteal disease) / BKA (if tibial disease) |

**BKA preferred over AKA** if popliteal flow adequate — better prosthetic outcome, lower mortality.

---

## 7. Outcomes

| Metric | Value |
|--------|-------|
| **1-year amputation-free survival** | ~60–70% (with revascularisation) |
| **1-year major amputation** | 25–40% (without revascularisation) |
| **1-year mortality** | 20–25% (CV dominant) |
| **5-year survival** | ~40–50% |
| **Diabetes multiplier** | **2–3× amputation risk** vs non-diabetes |

# Lipodystrophy syndromes

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Classify lipodystrophy syndromes (generalised vs partial)
- [ ] Recognise genetic vs acquired forms
- [ ] Apply management: metreleptin, insulin, metabolic control
- [ ] Recognise associated complications

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Definition** | Selective loss of adipose tissue -> metabolic complications |
| **Classification** | Generalised (CGL) vs Partial (FPLD); Genetic vs Acquired |
| **Prevalence** | Rare: CGL 1:10M; FPLD 1:1M |
| **Metabolic consequence** | Severe insulin resistance, diabetes, hypertriglyceridaemia, hepatic steatosis |

## 3. Clinical Features / Presentation

### Congenital Generalised Lipodystrophy (CGL)
| Type | Gene | Features |
|-------|------|----------|
| **CGL1** | AGPAT2 | Severe; early onset; acanthosis nigricans; hypertrophic cardiomyopathy |
| **CGL2** | BSCL2 | Severe; intellectual disability; cardiomyopathy risk |
| **CGL3** | CAV1 | Mild; vitamin D resistance |
| **CGL4** | PTRF (CAVIN1) | Myopathy; pyloric stenosis |

### Familial Partial Lipodystrophy (FPLD)
| Type | Gene | Features |
|-------|------|----------|
| **FPLD1** | Unknown | Women; gluteofemoral fat loss |
| **FPLD2 (Dunnigan)** | LMNA | Puberty onset; limb/trunk fat loss; face/neck fat gain; high CVD risk |
| **FPLD3** | PPARG | Early onset; severe insulin resistance |
| **FPLD4** | PLIN1 | Adult onset; mild |
| **FPLD5** | CIDEC | Childhood onset; severe |

### Acquired Lipodystrophy
| Type | Features |
|--------|----------|
| **Generalised (Lawrence)** | Autoimmune; post-infectious; panniculitis |
| **Partial (Barraquer-Simons)** | Centripetal fat loss; complement C3 nephritic factor; MPGN |
| **Drug-induced** | Protease inhibitors (HIV); insulin injections (local) |

## 4. Classification / Staging / Grading

### Generalised vs Partial
| Feature | Generalised (CGL, Lawrence) | Partial (FPLD, Barraquer-Simons) |
|---------|-----------------------------|----------------------------------|
| **Fat loss** | Near-total | Partial (limbs, trunk) |
| **Metabolic severity** | Extreme | Variable |
| **Onset** | Infancy/childhood | Puberty/adulthood |
| **Genetic** | CGL1-4 (AGPAT2, BSCL2, CAV1, PTRF) | FPLD1-6 (LMNA, PPARG, etc.) |

## 5. Diagnosis & Investigations
| Test | Purpose |
|------|---------|
| **Genetic testing** | AGPAT2, BSCL2, LMNA, PPARG, others |
| **Body composition** | DEXA, MRI (adipose tissue quantification) |
| **Metabolic** | Glucose, HbA1c, lipids (TG >> cholesterol), insulin |
| **Leptin** | Low (hallmark) |
| **Liver** | Ultrasound/MRI (steatosis); ALT/AST |
| **Heart** | Echo (cardiomyopathy in CGL) |
| **Kidney** | Proteinuria (CGL2) |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **T2DM with obesity** | Increased fat mass (vs loss) |
| **Cushing syndrome** | Central obesity + fat redistribution |
| **Anorexia nervosa** | Voluntary weight loss; psychopathology |
| **Ageing** | Gradual fat redistribution |

## 7. Management

### Metabolic Management
| Target | Strategy |
|--------|----------|
| **Diabetes** | High-dose insulin (often >2 U/kg/day); U-500 insulin; CSII |
| **Hypertriglyceridaemia** | Fibrates; omega-3; avoid alcohol; strict glucose control |
| **Leptin replacement** | **Metreleptin** (FDA approved for GL; off-label for PL) |
| **Hepatic steatosis** | Weight loss (if partial); pioglitazone? (caution) |

### Metreleptin Therapy
| Aspect | Detail |
|--------|--------|
| **Indication** | FDA approved for GL; off-label for PL with severe metabolic complications |
| **Dose** | 0.06 mg/kg/day SC (adjust by response) |
| **Effects** | ↓HbA1c 2-3%; ↓TG 50-60%; ↓hepatic fat; ↑satiety |
| **Monitoring** | Leptin levels; HbA1c, TG, LFT; TSH (thyroid autoimmunity risk) |
| **Side effects** | Anti-leptin antibodies (neutralising); hypoglycaemia; T-cell lymphoma (theoretical) |

### Complication Management
| Complication | Management |
|--------------|------------|
| **Diabetes** | High-dose insulin; CSII; metreleptin |
| **Hypertriglyceridaemia** | Fibrates + omega-3; avoid alcohol/estrogen |
| **Cardiomyopathy** | Echo surveillance; standard HF therapy |
| **Autoimmunity** | Screen (CGL2); treat if present |

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **Lipodystrophy = anabolic?** | NO - CATABOLIC state; fat loss -> metabolic chaos |
| **All lipodystrophy = genetic?** | NO - acquired forms exist (Lawrence, Barraquer-Simons) |
| **Leptin = weight loss drug?** | In lipodystrophy = REPLACEMENT therapy for deficiency |

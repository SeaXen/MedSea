## 3. Clinical Features / Presentation
| Feature | Presentation |
|---------|--------------|
| **Diabetes** | Often mild-moderate; post-prandial hyperglycaemia |
| **Acromegaly features** | Coarse features, macroglossia, sweating, prognathism, large hands/feet |
| **Cardiovascular** | Hypertension, cardiomyopathy, premature atherosclerosis |
| **Respiratory** | Sleep apnoea (macroglossia, soft tissue hypertrophy) |
| **Metabolic** | Insulin resistance, dyslipidaemia, visceral adiposity |

## 4. Classification / Staging / Grading
| Feature | Detail |
|--------|--------|
| **Mechanism** | GH/IGF-1 -> ↓insulin receptor signalling, ↑lipolysis, ↓GLUT4 |
| **Reversibility** | Improves with biochemical control (surgery/medical) |

## 5. Diagnosis & Investigations
| Test | Role | Interpretation |
|-------|------|----------------|
| **IGF-1** | Screening | Age-matched >ULN = positive |
| **OGTT with GH** | **Gold standard** | GH not suppressed <1 ug/L (or <0.4 modern assays) |
| **Random GH** | Screening | >1 ug/L suggests acromegaly |
| **MRI pituitary** | Localisation | Microadenoma vs macroadenoma |
| **HbA1c/glucose** | Monitor DM | Monitor response to treatment |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **Gigantism** | Epiphyseal closure not yet fused |
| **Pseudoacromegaly** | Severe insulin resistance + acanthosis; normal GH/IGF-1 |
| **Pituitary macroadenoma (non-functioning)** | No GH excess; mass effect only |
| **Type 2DM + obesity** | No acromegaly features; normal GH/IGF-1 |

## 7. Management

### Diabetes Management (Pre-cure)
| Approach | Detail |
|----------|--------|
| **Lifestyle** | Weight loss; exercise; low GI diet |
| **Metformin** | 1st line; improves insulin sensitivity |
| **SGLT2i/GLP-1 RA** | If CVD/CKD; weight loss benefit |
| **Sulfonylureas** | Avoid (weight gain) |
| **Insulin** | If severe hyperglycaemia |

### Definitive Treatment
| Aetiology | Treatment |
|-----------|-----------|
| **Pituitary adenoma** | Transsphenoidal surgery (first-line) |
| **Medical (if surgery contraindicated/recurrent)** | **Somatostatin analogues** (octreotide, lanreotide); **Pegvisomant** (GH receptor antagonist); **Dopamine agonists** (cabergoline) |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **Acromegaly DM = T1DM?** | NO - insulin resistance, not autoimmune |
| **GH directly causes hyperglycaemia?** | GH -> insulin resistance -> compensatory hyperinsulinaemia -> beta-cell exhaustion |
| **All acromegaly = diabetes?** | NO - 25% DM, 50% IGT; depends on duration/severity |

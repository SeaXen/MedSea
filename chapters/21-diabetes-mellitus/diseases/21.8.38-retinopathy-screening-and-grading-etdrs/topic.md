# Retinopathy Screening and Grading (ETDRS)

> [!info]
> **English National Diabetic Eye Screening Programme (ENDESP)** — **digital mydriatic photography**; **ETDRS-based grading** R0–R3 (retinopathy) + M0–M1 (maculopathy); **annual** if R1–R2, **2-yearly** if R0 ×2; **urgent referral** R3/M1.

---

## 2. Screening Programme (ENDESP / UK)

| Element | Specification |
|---------|---------------|
| **Modality** | **Mydriatic digital retinal photography** (2-field: macula + disc; or 3-field) |
| **Mydriasis** | Tropicamide 1% (phenylephrine 2.5% if needed) |
| **Frequency** | **Annual** (default); **2-yearly** if **R0 at two consecutive screens** |
| **Age range** | **12 years and older** (all diabetes types) |
| **Quality standards** | Sensitivity **>95%** for referable retinopathy; Specificity **>80%** |
| **Fail-safe** | Recall if ungradable (cataract, small pupil) — re-screen 4–6 weeks |

---

## 3. ETDRS Grading — Retinopathy (R)

| Grade | Description | Lesions | Referral |
|-------|-------------|---------|----------|
| **R0** | **No retinopathy** | None | Routine (annual / 2y if R0×2) |
| **R1** | **Background NPDR** (Mild) | **Microaneurysms** ± dot/blot haemorrhages ± hard exudates <1 DD from fovea | Annual |
| **R2** | **Pre-proliferative NPDR** (Moderate/severe) | **Venous beading**, **IRMA** (intraretinal microvascular abnormalities), **multiple haemorrhages** (>4 quadrants), **CWS** (cotton wool spots) | **Ophthalmology 6–12 weeks** |
| **R3** | **Proliferative DR (PDR)** | **New vessels** (NVD at disc, NVE elsewhere), **vitreous/pre-retinal haemorrhage**, **fibrovascular proliferation** | **Urgent ophthalmology 2–4 weeks** |

**Key**: Any R grade + **M1** = referable maculopathy (see below).

---

## 4. ETDRS Grading — Maculopathy (M)

| Grade | Description | OCT / Clinical Findings | Referral |
|-------|-------------|-------------------------|----------|
| **M0** | **No maculopathy** | No exudate/retinal thickening within 1 DD of fovea | Per R grade |
| **M1** | **Referable maculopathy** | **Exudate within 1 DD of fovea** OR **retinal thickening** (circinate exudates, CSME on OCT) | **Ophthalmology 4–8 weeks** |

**CSME (Clinically Significant Macular Oedema)** — ETDRS definition:
- Retinal thickening **≤500µm** from foveal centre
- **Hard exudates ≤500µm** with adjacent thickening
- **Zone of thickening ≥1 DD** involving centre

**OCT**: Central subfield thickness **>300µm (men)** / **>310µm (women)** / **>250µm (central 1mm)** = DMO.

---

## 5. Screening Interval Algorithm

```
Screen Result → Next Screen
R0, M0  → Annual (if first R0) → **2-yearly if R0 x 2**
R1, M0  → Annual
R1, M1  → Refer ophthalmology (4-8w); annual screen continues
R2, M0  → Refer ophthalmology (6-12w); annual screen continues
R2, M1  → Refer ophthalmology (4-8w); annual screen continues
R3, M0  → **Urgent refer** (2-4w); annual screen continues
R3, M1  → **Urgent refer** (2-4w + 4-8w); annual screen continues
Ungradable → Re-screen 4-6 weeks (cataract → refer for surgery)
```

---

## 6. Referral Criteria Summary

| Screen Result | Referral Timeframe | Specialty |
|---------------|-------------------|-----------|
| **R3** (any M) | **2–4 weeks** | **Urgent ophthalmology** |
| **R2** (any M) | **6–12 weeks** | Ophthalmology |
| **Any R + M1** | **4–8 weeks** | Ophthalmology (macula clinic) |
| **Ungradable** (media opacity) | **4–6 weeks** (re-screen) | Cataract surgery if indicated |
| **R0/R1, M0** | No referral | Continue screening |

---

## 7. Special Populations

| Population | Screening Modification |
|------------|------------------------|
| **Pregnancy (pre-existing diabetes)** | **Trimester 1**, **Trimester 2**, **Trimester 3** (3 screens); postpartum if R2/R3/M1 |
| **GDM** | Screen if pre-existing retinopathy or develops T2DM |
| **Children <12y** | Not routinely screened; start at 12y or puberty |
| **End-stage renal disease** | High risk — ensure annual screen compliance |
| **Learning disabilities** | Reasonable adjustments (longer appointments, familiariser) |

## 8. Quality Assurance & Metrics

| Metric | Target |
|--------|--------|
| **Uptake** | **>80%** of eligible population |
| **Sensitivity** (referable DR) | **>95%** |
| **Specificity** | **>80%** |
| **Time to result** | <3 weeks |
| **Time to referral appointment** | Per timeframe above |
| **Ungradable rate** | <5% |

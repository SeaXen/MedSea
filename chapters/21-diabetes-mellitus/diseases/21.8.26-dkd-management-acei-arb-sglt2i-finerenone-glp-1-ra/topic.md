# DKD Management (ACEi/ARB, SGLT2i, Finerenone, GLP-1 RA)

> [!info]
> **DKD management = "Big Four"** — ACEi/ARB (foundation) + SGLT2i + GLP-1 RA + Finerenone; **additive, complementary mechanisms**; start early (A2), continue through G3–4; individualise by eGFR/K+.

## 2. Treatment Algorithm — Sequential Add-On

### Step 1: ACEi OR ARB (Foundation)
| Aspect | Detail |
|--------|--------|
| **Indication** | **All DKD A2/A3** (even normotensive) |
| **Target dose** | **Maximum tolerated** (e.g., ramipril 10mg, losartan 100mg) |
| **Mechanism** | Efferent arteriolar dilatation → ↓ intraglomerular pressure |
| **Monitor** | K+, creatinine at **1–2 weeks**, then 3–6 monthly |
| **Acceptable rise** | Creatinine **<30%** (eGFR drop <30%) = expected; >30% → halve dose |
| **Hyperkalaemia** | K+ >5.5 → reduce/hold; >6.0 → stop; treat K+, restart low dose |
| **Contraindication** | Bilateral renal artery stenosis; K+ >5.5 at baseline |

**ACEi vs ARB**: Equivalent renal benefit; ARB if ACEi cough/angioedema; **do not combine** (↑ hyperkalaemia, AKI without benefit).

---

### Step 2: Add SGLT2 Inhibitor (If eGFR ≥20)

| Agent | eGFR Range | Dose | Key Trials |
|-------|------------|------|------------|
| **Dapagliflozin** | **≥20** | 10mg OD | **DAPA-CKD, CREDENCE** |
| **Empagliflozin** | **≥20** | 10mg OD | **EMPA-KIDNEY, EMPEROR-Reduced** |
| **Canagliflozin** | **≥30** | 100mg OD | CREDENCE |

**Mechanism**: Afferent arteriolar constriction (via tubuloglomerular feedback) → ↓ intraglomerular pressure; ↓ proximal glucose/Na reabsorption → ↓ workload.

**eGFR dip**: **Initial 3–5 mL drop expected** (haemodynamic) — **do not stop**; stabilises at new baseline.

**Contraindications**: T1DM (euglycaemic DKA risk), recurrent GU infections, eGFR <20 (most).

---

### Step 3: Add GLP-1 RA (If T2DM, eGFR ≥15)

| Agent | eGFR Range | Dose | Key Trials |
|-------|------------|------|------------|
| **Semaglutide SC** | **≥15** | 0.5→1.0mg weekly | **FLOW, SUSTAIN-6** |
| **Dulaglutide** | **≥15** | 0.75–1.5mg weekly | **AWARD-7, REWIND** |
| **Liraglutide** | **≥15** | 1.2–1.8mg daily | LEADER |

**Mechanism**: ↓ Inflammation, ↓ oxidative stress, weight loss, BP reduction, direct renal effects.

**No dose adjustment** for renal impairment — **safe down to dialysis**.

---

### Step 4: Add Finerenone (If T2DM, A2/A3, eGFR ≥25)

| Aspect | Detail |
|--------|--------|
| **Dose** | 10mg OD (if eGFR 25–60); **20mg OD** (if eGFR ≥60) |
| **Key Trials** | **FIDELIO-DKD, FIGARO-DKD** |
| **Mechanism** | Non-steroidal MRA → anti-inflammatory, anti-fibrotic (no BP effect at low dose) |
| **Monitor K+** | **At 4 weeks**, then monthly ×3, then quarterly |
| **Hold if** | K+ >5.5; restart at lower dose when K+ <5.0 |
| **Contraindication** | eGFR <25, K+ >5.0 at baseline, Addison's |

---

## 3. Treatment by KDIGO Stage

| Stage | Regimen |
|-------|---------|
| **G1–2, A2** | ACEi/ARB max + **SGLT2i** + GLP-1 RA (if T2DM) |
| **G1–2, A3** | Above + **Finerenone** (eGFR ≥25) |
| **G3a, A2** | Above (metformin max 1g BD) |
| **G3a, A3** | Above + Finerenone |
| **G3b, A2/A3** | Above (metformin contraindicated if eGFR <30 NICE) |
| **G4, any A** | ACEi/ARB max tolerated + SGLT2i (dapa/empa ≥20) + GLP-1 RA + Finerenone (eGFR ≥25); **nephrology co-care** |
| **G5 / Dialysis** | Continue ACEi/ARB + GLP-1 RA; stop SGLT2i/finerenone/metformin |

---

## 4. Blood Pressure Target

| Guideline | Target |
|-----------|--------|
| **KDIGO 2024** | **<120/80** (office) |
| **NICE** | **<130/80** (if A2/A3) |
| **ADA/ESC** | **<130/80** |

**Preferred agents**: ACEi/ARB → CCB (amlodipine) → Thiazide-like (indapamide/chlorthalidone, if eGFR >30) → Spironolactone (if resistant, K+ ≤4.5).

## 6. Common Pitfalls & Solutions

| Problem | Solution |
|---------|----------|
| **Creatinine rise >30% on ACEi/ARB** | **Expected** if <30% (haemodynamic); if >30% → halve dose, recheck 1w |
| **Hyperkalaemia on ACEi/ARB + SGLT2i + Finerenone** | Review K+ binders (patiromer, SZC); reduce finerenone first; loop diuretic if volume overloaded |
| **Volume depletion on SGLT2i** | Reduce diuretic dose; ensure euvolaemia; counsel on sick day rules |
| **Euglycaemic DKA (SGLT2i)** | **Stop SGLT2i** if acute illness/surgery/starvation; check ketones if unwell |
| **Non-adherence** | Simplify regimen; once-daily dosing; fixed-dose combinations |

## 7. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **First-line DKD** | **ACEi/ARB max dose** (even if normotensive) |
| **SGLT2i in DKD** | **eGFR ≥20** (dapa/empa); cana ≥30; add to ACEi/ARB |
| **Finerenone** | **eGFR ≥25 + A2/A3**; monitor K+ at 4w; hold if >5.5 |
| **GLP-1 RA renal** | **No dose adjust** down to eGFR 15; FLOW trial benefit |
| **ACEi + ARB combo** | **Contraindicated** — ↑ hyperkalaemia, AKI, no extra benefit |
| **Creatinine rise ACEi/ARB** | **<30% acceptable** (haemodynamic); >30% → reduce dose |
| **BP target KDIGO** | **<120/80**; NICE/ADA <130/80 |
| **Metformin in DKD** | G3a: max 1g BD; G3b/G4: contraindicated (NICE) |
| **SGLT2i eGFR dip** | **Initial 3–5 mL drop expected** — do not stop |
| **Patiromer/SZC** | K+ binders enable RAASi/SGLT2i/finerenone continuation |

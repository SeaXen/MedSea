## High-Yield Exam Points (FCPS/MRCP)

| Topic | Key Facts |
|-------|-----------|
| **rtCGM vs isCGM** | rtCGM: real-time, alerts (Dexcom G6/G7, Medtronic Guardian) · isCGM: flash, scan (FreeStyle Libre 2/3) |
| **CGM metrics (ATTD)** | TIR 70-180mg/dL >70% · TBR <54mg/dL <4% ( <1% <54) · TAR >180mg/dL <25% · GMI ≈ HbA1c · CV <36% |
| **CGM-guided dosing** | Trend arrows: ↑↑ add 10-20% bolus, ↑ add 5-10%, → no change, ↓ reduce 5-10%, ↓↓ reduce 10-20% + carbs |
| **Pump types** | Tethered (tubing): Medtronic, Tandem · Patch: Omnipod, Accu-Chek Solo · Closed-loop: AID |
| **AID systems** | Hybrid closed-loop: algorithm adjusts basal only (user announces meals) · Full closed-loop: no meal announcement |
| **Commercial HCL** | Medtronic 670G/780G · Tandem Control-IQ · CamAPS FX (Dana-i/FreeStyle) · Loop (DIY) · Beta Bionics iLet |
| **Connected pens** | NovoPen 6/ES, Esysta, InPen, Gocap · Dose tracking, reminders, data sharing |
| **Digital platforms** | Diasend/Glooko, LibreView, Clarity, CareLink · Remote monitoring, pattern analysis, telehealth |

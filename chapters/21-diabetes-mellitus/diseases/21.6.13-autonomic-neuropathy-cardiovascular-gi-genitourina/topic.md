# Autonomic neuropathy (cardiovascular, GI, genitourinary)

## 1. Learning Objectives
- [ ] Classify autonomic neuropathy: cardiovascular, gastrointestinal, genitourinary, sudomotor
- [ ] Diagnose cardiovascular autonomic neuropathy (CAN): orthostatic hypotension, HRV, silent MI
- [ ] Manage gastroparesis: prokinetics, dietary, gastric emptying study
- [ ] Manage erectile dysfunction / neurogenic bladder
- [ ] Apply screening tests: Valsalva, tilt table, HRV, gastric emptying

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Definition** | Autonomic nerve fibre damage from chronic hyperglycaemia |
| **Prevalence** | ~20–40% diabetes; CAN 20%, gastroparesis 5–12%, ED 30–50% men |
| **Risk Factors** | HbA1c, duration, age, DSPN, microvascular complications |

## 3. Clinical Features / Presentation

### Cardiovascular (CAN)
| Feature | Presentation |
|---------|--------------|
| **Orthostatic hypotension** | ↓SBP ≥20 mmHg or ↓DBP ≥10 mmHg within 3min standing; syncope, falls |
| **Resting tachycardia** | >100 bpm (early); fixed heart rate (late) |
| **Silent myocardial ischaemia** | No angina despite ischaemia (↓afferent cardiac nerves); ↑sudden cardiac death |
| **Exercise intolerance** | Blunted HR response; ↓exercise capacity |
| **HRV** | ↓Time/frequency domain measures |

### Gastrointestinal
| Feature | Presentation |
|---------|--------------|
| **Gastroparesis** | Nausea, vomiting (undigested food), early satiety, bloating, erratic glucose, weight loss |
| **Diarrhoea/Constipation** | Autonomic dysregulation; nocturnal diarrhoea common |
| **Oesophageal dysmotility** | Dysphagia, reflux |

### Genitourinary
| Feature | Presentation |
|---------|--------------|
| **Erectile dysfunction** | 30–50% men; autonomic + vascular + psychological |
| **Retrograde ejaculation** | Bladder neck incompetence; infertility |
| **Neurogenic bladder** | Atonic (retention, overflow) or spastic (frequency, urgency, incontinence); recurrent UTIs |

### Sudomotor
| Feature | Presentation |
|---------|--------------|
| **Anhidrosis** | Distal (feet/legs); ↓thermoregulation |
| **Gustatory sweating** | Facial/neck sweating with eating (Frey's syndrome variant) |

## 4. Classification / Staging / Grading

### CAN Staging (Ewing / CAN Subscore)
| Stage | Criteria |
|-------|----------|
| **Early** | Abnormal HRV (1+ test) |
| **Definite** | Orthostatic hypotension + abnormal HRV |
| **Severe** | Orthostatic hypotension + resting tachycardia + silent MI + HRV severely reduced |

### Gastroparesis Grading (GCSI)
| Grade | Symptoms |
|-------|----------|
| **Mild** | Nausea, early satiety |
| **Moderate** | Vomiting 1–3x/week, weight loss <5% |
| **Severe** | Daily vomiting, weight loss >5%, hospitalisation |

## 5. Diagnosis & Investigations

### Cardiovascular Autonomic Tests (Ewing Battery)
| Test | Normal | Abnormal |
|------|--------|----------|
| **HRV (deep breathing)** | E:I ratio >1.5 | ≤1.5 |
| **Valsalva ratio** | >1.5 | ≤1.2 |
| **30:15 ratio** | >1.04 | ≤1.0 |
| **Orthostatic BP** | ↓SBP <10 | ↓SBP ≥20 |
| **Sustained handgrip** | DBP rise >16 | ≤10 |

### Gastroparesis
| Investigation | Details |
|---------------|---------|
| **Gastric emptying scintigraphy** | **Gold standard**: T½ solids >120min (or >40min liquids) |
| **13C-octanoate breath test** | Alternative; non-radioactive |
| **Wireless motility capsule** | pH, pressure, transit time |

### Genitourinary
| Investigation | Details |
|---------------|---------|
| **IIEF-5** | ED severity score |
| **Urodynamics** | Neurogenic bladder type |
| **Post-void residual** | >150ml = retention |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **Autonomic neuropathy (non-diabetic)** | Amyloid, autoimmune (paraneoplastic), HIV, alcohol, drugs |
| **Medication effects** | β-blockers (mask tachycardia), diuretics (orthostasis), anticholinergics |
| **Cardiac disease** | Heart failure, arrhythmias (independent of CAN) |
| **GI obstruction** | Mechanical vs gastroparesis (imaging) |
| **Psychogenic ED** | Morning erections preserved; situational |

## 7. Management

### Cardiovascular Autonomic Neuropathy
| Intervention | Details |
|--------------|---------|
| **Orthostatic hypotension** | ↑Salt/fluid (2–3L/day), compression stockings, head-up tilt sleep, **fludrocortisone** 0.1–0.3mg OD, **midodrine** 2.5–10mg TDS |
| **Silent MI prevention** | Intensive risk factor control; consider stress testing if high risk |
| **Resting tachycardia** | Ivabradine if HR >70 on max β-blocker; avoid non-selective β-blockers |

### Gastroparesis
| Line | Agent | Dose | Notes |
|------|-------|------|-------|
| **Dietary** | Small frequent meals (6–8/day), low fat/fibre, liquid calories | — | Foundation |
| **1st line prokinetic** | **Metoclopramide** | 10mg TDS (max 5 days continuous) | D2 antagonist; **EPS risk** (limit 5d); avoid parkinsonism |
| | **Domperidone** | 10mg TDS | Peripheral D2; less CNS EPS; cardiac QT risk |
| | **Erythromycin** | 250mg TDS (short-term 2–4wk) | Motilin agonist; tachyphylaxis; QT |
| **2nd line** | Prucalopride / Tegaserod | Specialist | 5-HT4 agonists |
| **Refractory** | GES (gastric electrical stimulation), pyloroplasty, jejunostomy | Surgical | Specialist centres |

### Erectile Dysfunction
| Line | Intervention |
|------|--------------|
| **1st** | **PDE5 inhibitors** (sildenafil 50mg, tadalafil 10mg, vardenafil 10mg) — take 30–60min pre-activity |
| **2nd** | Vacuum erection device |
| **3rd** | Intraurethral alprostadil (MUSE) |
| **4th** | Intracavernosal alprostadil injection |
| **5th** | Penile prosthesis |

### Neurogenic Bladder
| Type | Management |
|------|------------|
| **Atonic (retention)** | Clean intermittent catheterisation (CIC); bethanechol 10–25mg TDS |
| **Spastic (urgency)** | Anticholinergics (oxybutynin, tolterodine); botulinum toxin A; sacral neuromodulation |

```mermaid
flowchart TD
    A[Autonomic Neuropathy] --> B{System}
    B -->|CV| C[Orthostatic HTN: fludrocortisone/midodrine]
    C --> D[Silent MI: risk factor control]
    B -->|GI| E[Gastroparesis: diet → metoclopramide/domperidone/erythromycin]
    B -->|GU| F[ED: PDE5i → vacuum → alprostadil → prosthesis]
    F --> G[Bladder: CIC (atonic) / anticholinergics (spastic)]
    B -->|Sudomotor| H[Anhidrosis: avoid heat; gustatory sweating: reassure]
```

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **CAN** | Orthostatic hypotension (↓SBP≥20), resting tachycardia, silent MI, HRV ↓ |
| **Orthostatic hypotension Rx** | Fludrocortisone 0.1–0.3mg OD, midodrine 2.5–10mg TDS, compression stockings |
| **Gastroparesis** | Nausea, vomiting, erratic glucose; gastric emptying T½ >120min; metoclopramide/domperidone/erythromycin |
| **ED** | PDE5i 1st line; vacuum → alprostadil → prosthesis |
| **Neurogenic bladder** | CIC for retention; anticholinergics/botox for urgency |
| **Screening** | Annual: orthostatic BP, HRV (if available), symptom questionnaire |

## 9. Viva Questions
| Question | Expected Answer |
|----------|-----------------|
| **What is cardiovascular autonomic neuropathy (CAN)?** | Orthostatic hypotension (↓SBP≥20), resting tachycardia >100bpm, silent MI, reduced HRV, exercise intolerance |
| **How do you diagnose orthostatic hypotension?** | ↓SBP ≥20 mmHg or ↓DBP ≥10 mmHg within 3 minutes of standing |
| **How do you manage orthostatic hypotension?** | Non-drug: ↑salt/fluid, compression stockings, head-up tilt; Drug: fludrocortisone 0.1–0.3mg OD, midodrine 2.5–10mg TDS |
| **What is gastroparesis and how do you diagnose it?** | Delayed gastric emptying without obstruction; symptoms: nausea, vomiting, early satiety, erratic glucose; **Gastric emptying scintigraphy**: T½ solids >120min |
| **What are the prokinetics for gastroparesis?** | Metoclopramide 10mg TDS (max 5d, EPS risk), Domperidone 10mg TDS (less EPS, QT risk), Erythromycin 250mg TDS (short-term, tachyphylaxis) |
| **How do you manage erectile dysfunction in diabetes?** | 1st: PDE5 inhibitors (sildenafil 50mg, tadalafil 10mg); 2nd: vacuum device; 3rd: intraurethral/intracavernosal alprostadil; 4th: penile prosthesis |
| **What is a silent MI in diabetes?** | Myocardial ischaemia/infarction without typical angina due to cardiac autonomic denervation; ↑sudden cardiac death risk |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **CAN = all autonomic?** | NO — CAN = cardiovascular only; autonomic includes GI, GU, sudomotor |
| **Metoclopramide long-term?** | NO — max 5 days continuous (EPS risk); use domperidone for chronic |
| **PDE5i with nitrates?** | CONTRAINDICATED — severe hypotension |

**Mnemonic: AUTO-NOMIC**
- **A**utonomic: CV, GI, GU, Sudomotor
- **U**pright: orthostatic hypotension (↓SBP≥20) → fludrocortisone/midodrine
- **T**achycardia: resting >100 (early CAN)
- **O**rthostatic: ↓SBP≥20 → midodrine/fludrocortisone
- **N**ausea/vomiting: gastroparesis (T½>120min)
- **O**esophageal: dysmotility
- **M**edication: metoclopramide (5d max), domperidone, erythromycin
- **I**mpotence: PDE5i → vacuum → alprostadil → prosthesis
- **C**atheterisation: CIC for atonic bladder
- **N**eurogenic bladder: anticholinergics for urgency, CIC for retention

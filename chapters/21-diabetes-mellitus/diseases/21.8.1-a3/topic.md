# Albuminuria Categories (A1/A2/A3)

> [!info]
> **KDIGO Albuminuria categories (A1–A3)** based on **UACR (Urine Albumin:Creatinine Ratio)**; A2 = microalbuminuria (earliest DKD); A3 = macroalbuminuria (overt nephropathy); confirm with **2/3 samples in 3–6 months**.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Define A1, A2, A3 categories with thresholds
- [ ] Apply correct sampling technique (first morning, avoid confounders)
- [ ] Interpret confirmation requirements
- [ ] Link albuminuria category to treatment decisions

---

## 2. KDIGO Albuminuria Categories

| Category | UACR (mg/mmol) | UACR (mg/g) | Terminology | Clinical Significance |
|----------|----------------|-------------|-------------|----------------------|
| **A1** | **<3** | **<30** | Normal / Mildly increased | **Normoalbuminuria**; low risk |
| **A2** | **3–30** | **30–300** | Moderately increased | **Microalbuminuria**; **earliest DKD**; ↑ CVD risk |
| **A3** | **>30** | **>300** | Severely increased | **Macroalbuminuria**; overt nephropathy; high ESRD/CVD risk |

---

## 3. Sampling Requirements

### Preferred Specimen
- **First morning void** (after overnight sleep) — most reliable, least variability
- **Random spot urine** acceptable if first morning not possible
- **Avoid 24-hour collection** (burdensome, no superiority for screening)

### Confounders to Avoid (Transient Albuminuria)
| Confounder | Action |
|------------|--------|
| **Exercise** (vigorous, 24h prior) | Avoid; repeat if positive |
| **Fever / Acute illness** | Defer until recovered |
| **UTI / Menstruation** | Defer |
| **Heart failure (acute)** | Defer |
| **Uncontrolled hypertension** (SBP >160) | Defer |
| **Hyperglycaemia** (acute marked) | Defer |
| **Recent contrast** | Defer 48h |

---

## 4. Confirmation of Chronicity

| Initial Result | Confirmation Protocol |
|----------------|------------------------|
| **A2 (3–30)** | **Repeat 2/3 samples within 3–6 months**; if ≥2/3 A2 → confirmed microalbuminuria |
| **A3 (>30)** | **Repeat 1–2 samples**; if confirmed → overt nephropathy |
| **A1 (<3)** | Annual screening (no confirmation needed) |

**Do not diagnose DKD on single sample** — biological variability 30–40%.

---

## 5. Clinical Action by Category

| Category | Action |
|----------|--------|
| **A1** | Annual UACR + eGFR; optimise BP/glucose/lipids |
| **A2** | **ACEi/ARB max dose** (even if normotensive); **add SGLT2i** (eGFR ≥20); BP <130/80; 6-monthly monitoring; consider finerenone if T2DM |
| **A3** | **ACEi/ARB max + SGLT2i + finerenone** (eGFR ≥25); **nephrology referral**; 3–6 monthly monitoring; GLP-1 RA if T2DM |

---

## 6. Albuminuria Reduction as Surrogate Endpoint

| Drug Class | Typical UACR Reduction | Trial Evidence |
|------------|------------------------|----------------|
| **ACEi/ARB** | **30–50%** | RENAAL, IDNT, MICRO-HOPE |
| **SGLT2i** | **30–40%** | CREDENCE, DAPA-CKD, EMPA-KIDNEY |
| **Finerenone** | **30–35%** | FIDELIO-DKD, FIGARO-DKD |
| **GLP-1 RA** | **20–30%** | FLOW, AWARD-7 |
| **Combined** | **Additive** (50–70%+) | Standard of care |

**KDIGO target**: **>30% UACR reduction** at 3–6 months predicts long-term renal protection.

---

## 7. Special Populations

| Population | Consideration |
|------------|---------------|
| **T1DM <5y duration** | Screen **from 5 years post-diagnosis** (not at dx) |
| **T2DM** | Screen **at diagnosis** (duration unknown) |
| **Pregnancy** | Screen each trimester; A2/A3 → obstetric nephrology |
| **Children** | Same thresholds; first morning void essential |
| **Non-diabetic CKD** | Same A1–A3 categories apply; SGLT2i indicated (DAPA-CKD, EMPA-KIDNEY) |

---

## 8. Alternative Markers (If UACR Unreliable)

| Marker | Use |
|--------|-----|
| **24h urinary albumin** | Gold standard but burdensome; >30 mg/24h = A2; >300 = A3 |
| **Albumin:Creatinine ratio (random)** | Acceptable if first morning not possible |
| **Dipstick protein** | **Not for screening** (misses microalbuminuria; false +ve/-ve) |

---

## 9. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **A1 threshold** | **<3 mg/mmol** (<30 mg/g) |
| **A2 (micro)** | **3–30 mg/mmol** (30–300 mg/g) — **earliest DKD** |
| **A3 (macro)** | **>30 mg/mmol** (>300 mg/g) — overt nephropathy |
| **Specimen** | **First morning void** preferred |
| **Confirmation** | **2/3 samples in 3–6 months** for A2; 1–2 for A3 |
| **Transient causes** | Exercise, fever, UTI, menstruation, HF, uncontrolled HTN |
| **Action for A2** | ACEi/ARB max + SGLT2i; BP <130/80 |
| **Action for A3** | + Finerenone + nephrology referral |
| **UACR reduction target** | **>30% at 3–6 months** predicts renal protection |
| **Screening T1DM** | From **5 years post-diagnosis** |
| **Screening T2DM** | **At diagnosis** |

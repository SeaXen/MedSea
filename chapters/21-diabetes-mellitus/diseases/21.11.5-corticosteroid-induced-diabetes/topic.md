# Corticosteroid-induced diabetes

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise steroid-induced hyperglycaemia pattern
- [ ] Apply management: NPH insulin, basal-bolus, sliding scale
- [ ] Adjust for steroid tapering
- [ ] Monitor for complications

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Definition** | Hyperglycaemia induced by glucocorticoid therapy |
| **Incidence** | 20-50% on steroids develop hyperglycaemia |
| **Mechanism** | Cortisol -> ↑HGP, ↓insulin sensitivity, ↓β-cell function |
| **Pattern** | **Post-prandial hyperglycaemia prominent** (steroids given AM) |

## 3. Clinical Features / Presentation
| Feature | Steroid DM | T2DM |
|---------|------------|------|
| **Pattern** | Post-prandial >> fasting | Fasting + post-prandial |
| **Timing** | Peaks 4-8h post-dose (AM steroids) | Throughout day |
| **Fasting glucose** | Often normal | Elevated |
| **Dose relationship** | Direct (higher dose = worse hyperglycaemia) | Chronic |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **T2DM** | Chronic; fasting + post-prandial; no steroid history |
| **Stress hyperglycaemia** | Acute illness; resolves; steroids not given |
| **Cushing syndrome** | Endogenous; features; dexamethasone suppression |

## 7. Management

### Insulin Regimens
| Steroid Dose | Insulin Strategy |
|--------------|------------------|
| **Prednisolone >20mg** | **Basal-bolus** (NPH BD or glargine OD + rapid analogue pre-meal) |
| **Prednisolone 10-20mg** | **NPH insulin BD** (0.2-0.3 U/kg/day) |
| **Prednisolone <10mg** | May not need insulin; monitor |
| **IV methylprednisolone pulse** | IV insulin infusion (sliding scale) |

### NPH Insulin Protocol (Classic)
| Dose | Timing |
|------|--------|
| **2/3 total dose** | Morning (covers AM steroid peak) |
| **1/3 total dose** | Evening |

### Steroid Tapering
| Phase | Insulin Adjustment |
|-------|-------------------|
| **Weaning** | Reduce insulin proportionally to steroid reduction |
| **Off steroids** | Stop insulin (unless pre-existing DM) |

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **Pattern** | Post-prandial >> fasting (peaks 4-8h post-AM steroid) |
| **NPH insulin** | 2/3 AM, 1/3 PM (covers steroid peak) |
| **Sliding scale** | NOT recommended as sole therapy |
| **Taper** | Reduce insulin proportionally to steroid taper |
| **HbA1c** | Falsely low if recent onset |

## 9. Viva Questions
| Question | Expected Answer |
|----------|-----------------|
| **What is the characteristic glucose pattern in steroid diabetes?** | Post-prandial hyperglycaemia prominent; fasting often normal; peaks 4-8h post-AM steroid |
| **Why is NPH insulin preferred?** | Peak action 4-8h matches steroid peak (AM dose); 2/3 AM, 1/3 PM |
| **Is sliding scale insulin sufficient?** | NO - reactive, erratic, increases hypoglycaemia risk |
| **How do you manage tapering steroids?** | Reduce insulin proportionally to steroid dose reduction; stop when steroids stopped |
| **Why is fasting glucose often normal?** | Steroids given AM -> post-prandial peak; overnight clearance normal |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **Steroid DM = permanent?** | Usually resolves when steroids stopped (unless pre-existing DM) |
| **Sliding scale sufficient?** | NO - reactive, increases hypo risk; use basal/basal-bolus |
| **All steroids = same risk?** | NO - IV pulse > oral > inhaled/topical; dose-dependent |

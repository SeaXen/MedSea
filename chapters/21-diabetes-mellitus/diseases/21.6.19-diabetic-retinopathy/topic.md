# Diabetic Retinopathy

> [!info]
> **Diabetic retinopathy: leading cause of blindness in working-age adults** — microvascular complication; **ETDRS grading** drives screening/referral; **NPDR → PDR → DMO**; **Anti-VEGF 1st line for DMO**; **PRP for PDR**; **Annual screening** (2-yearly if no DR).

---

## 1. Learning Objectives
- [ ] Apply ETDRS grading for screening and referral decisions
- [ ] Differentiate NPDR, PDR, and DMO features
- [ ] State screening intervals by DR severity
- [ ] Select treatment: anti-VEGF for DMO, PRP for PDR, vitrectomy for VH/TRD
- [ ] Counsel on glycaemic/BP/lipid control for progression prevention

---

## 2. Definition & Epidemiology

| Feature | Detail |
|---------|--------|
| **Definition** | Microvascular retinal damage from chronic hyperglycaemia → capillary occlusion, leakage, neovascularisation |
| **Prevalence** | Any DR: ~35% T1DM, ~25% T2DM; Vision-threatening DR: ~10% T1DM, ~5% T2DM |
| **Incidence** | T1DM: ~90% at 20y; T2DM: ~60% at 20y |
| **Risk Factors** | Duration of diabetes, HbA1c, hypertension, dyslipidaemia, pregnancy, nephropathy, anaemia |
| **Pathophysiology** | Pericyte loss → capillary basement membrane thickening → occlusion → ischaemia → VEGF → leakage + neovascularisation |

---

## 3. Clinical Features / Presentation

| Presentation | Frequency | Key Features |
|-------------|-----------|--------------|
| **Asymptomatic** | Early NPDR | No visual symptoms; detected on screening |
| **Blurred vision** | DMO | Central vision loss, distortion, reading difficulty |
| **Floaters / sudden vision loss** | Vitreous haemorrhage (PDR) | Painless, "cobwebs", "curtain" |
| **Visual field defects** | Tractional retinal detachment | Peripheral then central loss |

> **Red Flags**: Sudden vision loss (vitreous haemorrhage, TRD); rapid progression in pregnancy; new vessels on disc (NVD) or elsewhere (NVE).

---

## 4. Classification / Staging / Grading

### ETDRS Grading Scale (Simplified)
| ETDRS Level | Retinopathy Stage | Key Features | Referral Timing |
|-------------|-------------------|--------------|-----------------|
| **10–12** | No DR | No abnormalities | Annual |
| **20–35** | Mild–Moderate NPDR | Microaneurysms only (20); Haemorrhages ± exudates ± cotton wool (35) | Annual / 6–12 months |
| **43–53** | **Severe NPDR** | **4:2:1 rule**: 4 quadrants haem, 2 venous beading, 1 IRMA | **3–6 months** |
| **61–65** | **Early PDR** | NVD <1/3 disc area, NVE <1/2 disc area, no VH | **Urgent (2–4 weeks)** |
| **71–75** | **High-risk PDR** | NVD ≥1/3 disc, NVE ≥1/2 disc, VH, pre-retinal haem | **Urgent (1–2 weeks)** |
| **M1** | **Diabetic Macular Oedema (DMO)** | Retinal thickening ≤1DD from fovea (centre-involving) | **Urgent (2–4 weeks)** |

> **4:2:1 Rule (Severe NPDR)**: Haemorrhages/microaneurysms in **4 quadrants** OR Venous beading in **2 quadrants** OR Intraretinal microvascular abnormalities (IRMA) in **1 quadrant**

### DMO Classification
| Type | Definition | Treatment Priority |
|------|------------|-------------------|
| **Centre-involving DMO (CI-DMO)** | Thickening ≤1DD from fovea | **Anti-VEGF 1st line** |
| **Non-centre-involving DMO** | Thickening >1DD from fovea | Laser (if threatening centre) |

---

## 5. Diagnosis & Investigations

| Investigation | Role | Key Details |
|---------------|------|-------------|
| **Digital retinal photography** | Screening (mydriatic 45°/2-field) | Annual (2-yearly if no DR); quality assured |
| **OCT (Optical Coherence Tomography)** | DMO diagnosis/monitoring | Central subfield thickness (CST); CI-DMO if CST >300–320µm (device-dependent) |
| **Fluorescein angiography** | PRP planning, NVE/NVD confirmation | If media opacity; OCT-A emerging alternative |
| **HbA1c, BP, lipids** | Systemic risk factor control | HbA1c <53mmol/mol; BP <130/80; LDL <1.8mmol/L |

---

## 6. Differential Diagnosis

| Condition | Distinguishing Features |
|-----------|-------------------------|
| **Hypertensive retinopathy** | AV nicking, copper/silver wiring, flame haemorrhages; no microaneurysms, no neovascularisation |
| **Retinal vein occlusion** | Sectoral haemorrhages, disc oedema, venous dilatation; unilateral; no microaneurysms elsewhere |
| **Radiation retinopathy** | Delayed (months-years post-RT); microaneurysms, capillary non-perfusion; history RT |
| **Ocular ischaemic syndrome** | Carotid stenosis; mid-peripheral haemorrhages, neovascularisation of iris/angle; low IOP |
| **Age-related macular degeneration** | Drusen, geographic atrophy, CNV; age >55; spares periphery |

---

## 7. Management

### Screening & Referral
| DR Stage | Screening Interval | Referral |
|----------|-------------------|----------|
| **No DR (R0)** | 2 years (UK NSC) | None |
| **Mild NPDR (R1)** | Annual | Routine ophthalmology |
| **Moderate NPDR (R2)** | 6–12 months | Routine ophthalmology |
| **Severe NPDR (R3)** | 3–6 months | **Urgent (2–4 weeks)** |
| **PDR (R4)** | — | **Urgent (1–2 weeks)** |
| **DMO (M1)** | — | **Urgent (2–4 weeks)** |

### Treatment
| Indication | 1st Line | 2nd Line / Adjunct | Details |
|------------|----------|-------------------|---------|
| **Centre-involving DMO** | **Anti-VEGF** (Ranibizumab 0.5mg, Aflibercept 2mg, Bevacizumab 1.25mg off-label) | Dexamethasone implant (Ozurdex) if anti-VEGF inadequate | Monthly loading ×3, then treat-and-extend / PRN |
| **Non-centre DMO** | **Laser** (ETDRS grid) | Anti-VEGF if threatening centre | Focal/grid laser to leaking microaneurysms |
| **PDR (Early/High-risk)** | **PRP (Panretinal Photocoagulation)** | Anti-VEGF (adjunct for VH/NV regression) | 1200–1600 burns over 2–3 sessions; avoids macula |
| **Vitreous haemorrhage** | Observe 3 months (if no PDR tx needed) → **Vitrectomy** if non-clearing | Anti-VEGF pre-op to regress NV | Early vitrectomy if PDR + VH (DRVS) |
| **Tractional RD** | **Vitrectomy** | — | Urgent if macula threatened |

> **Anti-VEGF agents**: Ranibizumab (Lucentis, licensed), Aflibercept (Eylea, licensed), Bevacizumab (Avastin, off-label, cost-effective)

### Systemic Risk Factor Control
| Target | Evidence |
|--------|----------|
| **HbA1c <53mmol/mol (7.0%)** | DCCT/UKPDS: ↓DR onset/progression 30–40% per 1% HbA1c ↓ |
| **BP <130/80** | UKPDS: tight BP ↓DR progression 34% |
| **Lipids: Fenofibrate** | FIELD/ACCORD-Eye: fenofibrate ↓DR progression (esp NPDR); consider if dyslipidaemia |
| **Aspirin** | No effect on DR; safe for CVD |

## 8. FCPS/MRCP High-Yield Summary

| Topic | Key Points |
|-------|------------|
| **Screening** | Digital photography; Annual (2-yearly if no DR); ETDRS grading |
| **ETDRS referral thresholds** | Severe NPDR (43–53): 2–4 weeks; PDR (61+): 1–2 weeks; DMO (M1): 2–4 weeks |
| **4:2:1 Rule** | Severe NPDR: 4 quadrant haem OR 2 quadrant venous beading OR 1 quadrant IRMA |
| **DMO treatment** | **Centre-involving: Anti-VEGF 1st line** (Ranibizumab/Aflibercept/Bevacizumab); Non-centre: Laser |
| **PDR treatment** | **PRP** (1200–1600 burns); Anti-VEGF adjunct for NV/VH |
| **Vitreous haemorrhage** | Observe 3mo → Vitrectomy if non-clearing; Early if PDR (DRVS) |
| **Tractional RD** | Vitrectomy urgent if macula threatened |
| **Systemic control** | HbA1c <53, BP <130/80, Fenofibrate for DR progression |
| **Pregnancy** | Screen each trimester; rapid progression risk |

## 9. Viva Questions

| Question | Expected Answer |
|----------|-----------------|
| **What is the ETDRS grading for severe NPDR?** | **4:2:1 Rule**: Haemorrhages/microaneurysms in 4 quadrants OR Venous beading in 2 quadrants OR IRMA in 1 quadrant (ETDRS levels 43–53) |
| **Screening intervals by DR stage?** | No DR: 2-yearly; Mild NPDR: annual; Moderate NPDR: 6–12 months; Severe NPDR: 3–6 months; PDR/DMO: urgent referral |
| **What is the 1st line treatment for centre-involving DMO?** | **Anti-VEGF** (Ranibizumab 0.5mg, Aflibercept 2mg, Bevacizumab 1.25mg off-label); monthly ×3 then treat-and-extend/PRN |
| **What is the treatment for PDR?** | **PRP (Panretinal Photocoagulation)** — 1200–1600 burns over 2–3 sessions, sparing macula; Anti-VEGF adjunct for NV/VH |
| **When do you refer severe NPDR urgently?** | ETDRS 43–53 (4:2:1 rule) → **Urgent referralwithin 2–4 weeks** |
| **How does fenofibrate affect DR?** | FIELD/ACCORD-Eye: fenofibrate ↓DR progression (30–40%); consider in NPDR with dyslipidaemia |
| **What is the difference between CI-DMO and non-CI DMO?** | CI-DMO: thickening ≤1DD from fovea → Anti-VEGF 1st line; Non-CI DMO: >1DD → Laser (if threatening centre) |
| **Screening in pregnancy?** | Pre-conception, then each trimester; rapid progression risk; postpartum 6 months |

## 10. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **PRP for DMO?** | NO — PRP for **PDR**; **Focal/grid laser** for non-centre DMO; **Anti-VEGF** for centre-involving DMO |
| **Anti-VEGF for PDR?** | Adjunct only (NV regression pre-PRP, VH); **PRP is definitive** |
| **Bevacizumab unlicensed?** | Yes — off-label; cost-effective; NICE allows if ranibizumab/aflibercept not suitable |

**Mnemonic: DR-ETDRS-421**
- **D**R: leading cause blindness working age
- **R**etinopathy: microaneurysms → haemorrhages → cotton wool → venous beading → IRMA → NVE/NVD → VH/TRD
- **E**TDRS grading: 10-12 none, 20-35 mild/mod, 43-53 severe, 61+ PDR, M1 DMO
- **T**wo-yearly screen if no DR; **A**nnual if mild; **6-12mo** if mod; **3-6mo** if severe
- **D**MO: CI-DMO → Anti-VEGF 1st; Non-CI → Laser
- **R**eferral: Severe NPDR 2-4wk; PDR 1-2wk; DMO 2-4wk
- **S**ystemic: HbA1c<53, BP<130/80, Fenofibrate ↓progression
- **4**: 4 quadrant haemorrhages
- **2**: 2 quadrant venous beading
- **1**: 1 quadrant IRMA
- **= Severe NPDR (43-53) → Urgent referral**

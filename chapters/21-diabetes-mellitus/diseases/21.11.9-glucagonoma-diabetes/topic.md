# Glucagonoma diabetes

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise glucagonoma as rare cause of secondary diabetes
- [ ] Apply diagnostic criteria (plasma glucagon, imaging)
- [ ] Recognise NME rash and associated features
- [ ] Apply management (surgery, somatostatin analogues, insulin)

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Definition** | Alpha-cell tumour secreting glucagon causing diabetes |
| **Incidence** | 1:20-50 million; ~1% of pancreatic NETs |
| **Mechanism** | Glucagon -> ↑HGP, ↓insulin effect -> diabetes |
| **Median age** | 50-60 years |

## 4. Classification / Staging / Grading

### Glucagonoma Syndrome Features
| Feature | Prevalence |
|---------|------------|
| **Diabetes** | 90% |
| **NME rash** | 70% |
| **Weight loss** | 90% |
| **DVT** | 10-15% |
| **Anaemia** | 60% |
| **Hypoaminoacidaemia** | Yes |

## 5. Diagnosis & Investigations
| Test | Finding |
|------|---------|
| **Plasma glucagon** | **>500-1000 pg/mL** (normal <50-100) |
| **CT/MRI pancreas** | Mass in tail/body (60%) |
| **Somatostatin receptor imaging** | Octreoscan (SSTR2+) |
| **Amino acids** | Low (hypoaminoacidaemia) |
| **Zn/Cu/Se** | Low (due to rash) |
| **Octreoscan** | SSTR2 positive |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **NME-like rash** | Zn deficiency, biotin deficiency, pellagra |
| **Phaeochromocytoma** | Catecholamine excess; HTN spells |
| **Phaeochromocytoma** | Paroxysmal HTN; metanephrines high |
| **T2DM + rash** | No glucagon excess; rash different |

## 7. Management

### Surgical (Curative if localised)
| Stage | Treatment |
|-------|-----------|
| **Localised** | Distal pancreatectomy (+splenectomy if tail) |
| **Metastatic** | Debulking; somatostatin analogues; systemic therapy |

### Medical Management
| Agent | Role |
|-------|------|
| **Somatostatin analogues** | Octreotide/lanreotide -> ↓glucagon; improves rash, weight loss, DM |
| **Insulin** | If DM uncontrolled |
| **Metformin** | If insulin resistance component |
| **Anticoagulation** | Prophylactic (DVT risk) |
| **Zinc supplementation** | Improves rash (hypoaminoacidaemia) |

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **Triad** | DM + NME rash + weight loss |
| **Glucagon** | >500-1000 pg/mL (normal <50-100) |
| **Rash** | NME: migratory, blistering, perineum/groin/legs |
| **Weight loss** | Catabolic (glucagon -> ↑catabolism) |
| **DVT** | 10-15% (hypercoagulable) |
| **Treatment** | Surgery (curative); somatostatin analogues ↓glucagon |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **Glucagonoma = common?** | NO - extremely rare (1:20-50M) |
| **Rash = diabetic dermopathy?** | NO - NME is different (migratory, blistering, perineum) |
| **All DM + rash = glucagonoma?** | NO - rule out Zn/Pellagra/biotin deficiency |

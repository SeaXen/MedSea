# BP Targets in Diabetes (NICE, ADA, ESC)

> [!info]
> **NICE**: <140/90 (<130/80 if CKD/diabetes with albuminuria); **ADA/ESC**: <130/80 for all diabetes; **ACCORD BP**: intensive <120 systolic ↓ stroke but ↑ adverse events; **SPRINT**: non-diabetes intensive benefit.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] State BP targets per NICE, ADA, ESC guidelines
- [ ] Apply ACCORD BP and SPRINT trial evidence
- [ ] Choose first-line and add-on antihypertensives

- [ ] Monitor for adverse effects (AKI, hyperkalaemia, hypotension)

## 2. Guideline Targets Comparison

| Guideline | General Diabetes Target | CKD / Albuminuria / High Risk | Age/Specific |
|-----------|------------------------|-------------------------------|--------------|
| **NICE NG28 / NG136** | **<140/90** | **<130/80** (if ACR ≥30 or eGFR <60) | <150/90 if >80y frail |
| **ADA 2024** | **<130/80** | **<130/80** (same) | Individualised if >65/frail |
| **ESC 2023** | **<130/80** | **<130/80** | <140/90 if >80/frail |
| **KDIGO 2021** | — | **<120/80** (CKD) | — |
| **ACC/AHA 2017** | — | <130/80 | — |

**Key difference**: NICE allows <140/90 for uncomplicated diabetes; ADA/ESC/KDIGO uniformly target <130/80.

## 3. Key Trial Evidence

### ACCORD BP (2010) — T2DM, High CVD Risk
| Arm | SBP Target | Primary Outcome (MI, stroke, CV death) | Adverse Events |
|-----|------------|----------------------------------------|----------------|
| **Intensive** | **<120 mmHg** | **No significant difference** (1.87%/yr vs 2.09%/yr; HR 0.88, p=0.20) | **↑ Hypotension, syncope, AKI, electrolyte imbalance** |
| **Standard** | **<140 mmHg** | Reference | Lower |

**Subgroup**: **Stroke significantly reduced** (0.32% vs 0.53%/yr; HR 0.59) with intensive.

### SPRINT (2015) — Non-Diabetes, High CVD Risk
| Arm | SBP Target | Primary Outcome | Adverse Events |
|-----|------------|-----------------|----------------|
| **Intensive** | **<120 mmHg** | **↓ 25%** (HR 0.75) | ↑ Hypotension, syncope, AKI, electrolyte |
| **Standard** | **<140 mmHg** | Reference | Lower |

**SPRINT NOT in diabetes** — but informs intensive targets.

### UKPDS (1998) — T2DM, Newly Diagnosed
- **Tight control** (<150/85) vs **less tight** (<180/105)
- **↓ 24%** diabetes-related endpoints; **↓ 32%** stroke; **↓ 37%** microvascular

## 4. First-Line Agents in Diabetes

| Drug Class | Agent Examples | Position | Rationale |
|------------|----------------|----------|-----------|
| **ACE Inhibitor** | Ramipril, lisinopril, perindopril | **1st line** (all guidelines) | Renal protection (↓ albuminuria), CV benefit, ↓ new-onset diabetes |
| **ARB** | Losartan, candesartan, valsartan, olmesartan | **1st line** (ACEi intolerant) | Same as ACEi; less cough |
| **CCB** (Dihydropyridine) | Amlodipine, felodipine, lercanidipine | **1st line add-on** / alternative | Potent BP lowering; neutral metabolic; **avoid verapamil/diltiazem with beta-blockers** |
| **Thiazide/Thiazide-like** | Indapamide, chlorthalidone, hydrochlorothiazide | **2nd/3rd line** | Cheap; ↑ glucose mildly (↓ insulin secretion); chlorthalidone > HCTZ |

**Beta-blockers**: **Not 1st line** in diabetes (mask hypoglycaemia, ↑ insulin resistance, weight gain) — use if CAD, HF, post-MI.

---

## 5. Treatment Algorithm (NICE/ADA Consensus)

```
Step 1: ACEi or ARB (titrate to max tolerated)
    │
    ├─► If Black African/Caribbean: ACEi/ARB + CCB (less ACEi response)
    │
Step 2: Add CCB (amlodipine) OR Thiazide-like (indapamide/chlorthalidone)
    │
    ├─► If CKD/albuminuria: prefer CCB (less metabolic)
    │
Step 3: Triple therapy: ACEi/ARB + CCB + Thiazide
    │
Step 4: Add 4th agent:
    ├─► Spironolactone 25mg (if K+ ≤4.5, eGFR >45) — RESISTANT HTN evidence
    ├─► Beta-blocker (if CAD/HF)
    ├─► Alpha-blocker (doxazosin)
    ├─► Central agent (clonidine, moxonidine)
    │
Step 5: Referral to hypertension specialist
```

## 6. Special Considerations

| Situation | Management |
|-----------|------------|
| **CKD (eGFR <60, ACR ≥30)** | Target <130/80 (NICE) / <120/80 (KDIGO); ACEi/ARB mandatory; monitor K+/creatinine 1–2w after start/change |
| **Albuminuria** | ACEi/ARB max dose; add SGLT2i (↓ albuminuria); finerenone |
| **Postural hypotension** | Check standing BP; reduce/stop alpha-blockers, diuretics; review timing |
| **Resistant HTN** (≥4 drugs including diuretic) | Exclude adherence, white coat, secondary causes (hyperaldosteronism, RAS); spironolactone 25mg |
| **Pregnancy** | Target <135/85; **avoid ACEi/ARB** (teratogenic); use labetalol, nifedipine, methyldopa |
| **Elderly/frail** | Target <140/90 (NICE) / <140/90 (ESC >80); avoid overtreatment; standing BP check |

## 7. Monitoring & Safety

| Parameter | Frequency |
|-----------|-----------|
| **BP (clinic)** | Every visit until controlled; then 3–6 monthly |
| **Home BP (HBPM)** | 7-day averages (discard day 1); target <135/85 |
| **ABPM** | If white coat suspected |
| **Renal function (U&E, eGFR)** | 1–2w post ACEi/ARB/diuretic change; then 6–12 monthly |
| **Potassium** | Same as renal; check if on spironolactone/ACEi/ARB + SGLT2i |
| **Orthostatic BP** | Annual (elderly, autonomic neuropathy) |

## 8. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **NICE target** | <140/90 general; **<130/80 if CKD (ACR≥30 or eGFR<60)** |
| **ADA/ESC target** | **<130/80** for all diabetes |
| **ACCORD BP** | Intensive <120: ↓ stroke but **no CV death/MI benefit**, ↑ AKI/hypotension |
| **SPRINT** | Non-diabetes: intensive <120 ↓ CV events; not directly applicable to diabetes |
| **First-line** | **ACEi or ARB** (renal protection, CV benefit, ↓ new diabetes) |
| **Add-on** | **CCB** (amlodipine) or **thiazide-like** (indapamide/chlorthalidone) |
| **Beta-blockers** | **Not 1st line** in diabetes (mask hypo, weight gain, insulin resistance) |
| **Resistant HTN** | **Spironolactone 25mg** (if K+ ≤4.5, eGFR >45) — best evidence |
| **CKD + diabetes** | ACEi/ARB mandatory; max dose; monitor K+/creat 1–2w |
| **Pregnancy** | Labetalol, nifedipine, methyldopa; **avoid ACEi/ARB** |

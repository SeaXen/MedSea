# GLP-1 Receptor Agonists in Heart Failure (FIGHT, STEP-HFpEF)

> [!info]
> **GLP-1 RA in HF: nuanced** — **HFrEF: neutral** (FIGHT: liraglutide no benefit); **HFpEF + obesity: benefit** (STEP-HFpEF: semaglutide); **weight loss drives improvement**; **not first-line for HF** (SGLT2i preferred); safe in HF (no harm).

## 2. Key Trials

### FIGHT Trial (2017) — Liraglutide in HFrEF
| Aspect | Detail |
|--------|--------|
| **Population** | **300 patients**, HFrEF (EF ≤40%), advanced (recent hospitalisation), **±T2DM** (mostly non-diabetic) |
| **Intervention** | Liraglutide 1.8mg daily vs placebo (180 days) |
| **Primary Outcome** | Hierarchical clinical outcome (death, HF hosp, time to event) |
| **Result** | **Neutral** (HR 1.00, p=0.50); no benefit in any subgroup |
| **Safety** | More serious AEs (GI); **no increase in CV mortality** |

**Conclusion**: **Liraglutide not beneficial in HFrEF** — do not use for HF treatment.

### STEP-HFpEF Trial (2023) — Semaglutide in HFpEF + Obesity
| Aspect | Detail |
|--------|--------|
| **Population** | **529 patients**, HFpEF (EF ≥45%), **BMI ≥30**, NYHA II–III, **±T2DM** (mostly non-diabetic) |
| **Intervention** | **Semaglutide 2.4mg weekly** (max dose) vs placebo (52 weeks) |
| **Primary Outcomes** | 1) Change in **KCCQ-CSS** (symptom score) 2) Change in **6-minute walk distance** |
| **Results** | **KCCQ-CSS: +7.8 points** (p<0.001); **6MWD: +21.5m** (p<0.001); **Weight: −13.3%**; **NT-proBNP: −42%** |
| **Safety** | GI side effects (nausea); no increase in serious AEs |

**Conclusion**: **Semaglutide 2.4mg improves symptoms, exercise capacity, weight, NT-proBNP in HFpEF + obesity**.

### LIVE Trial (2023) — Liraglutide Post-MI
| Aspect | Detail |
|--------|--------|
| **Population** | Post-MI with **EF ≤40%** and/or **diabetes**, high-risk |
| **Result** | **Neutral** for CV death/HF hosp; **weight loss, HbA1c improvement** |
| **Implication** | No HF benefit post-MI |

## 3. Mechanism & Physiology

| Mechanism | Effect |
|-----------|--------|
| **Weight loss** | ↓ Preload, ↓ afterload, ↓ cardiac workload (major in STEP-HFpEF) |
| **Direct cardiac effects** | ↑ Myocardial glucose uptake, ↓ lipotoxicity, ↓ oxidative stress (preclinical) |
| **Haemodynamic** | ↓ BP (2–4 mmHg), ↓ HR (2–3 bpm) — modest |
| **Anti-inflammatory** | ↓ CRP, IL-6, TNF-α |
| **Renal** | ↓ Albuminuria, stable eGFR |

**Key difference from SGLT2i**: GLP-1 RA benefit **largely weight-mediated**; SGLT2i has **direct cardiorenal mechanisms** independent of glucose/weight.

---

## 4. Positioning in HF Guidelines (ESC 2023 / ACC/AHA)

| HF Type | SGLT2i | GLP-1 RA |
|---------|--------|----------|
| **HFrEF** | **Class I, Level A** (dapa/empa) | **Not recommended** (FIGHT neutral) |
| **HFmrEF (41–49%)** | **Class I, Level A** | Consider if obese + HFpEF features |
| **HFpEF (EF ≥50%)** | **Class I, Level A** (dapa/empa) | **Semaglutide 2.4mg** if **BMI ≥30** (STEP-HFpEF) — emerging |

**NICE**: SGLT2i first-line for HF; GLP-1 RA for weight management in T2DM + obesity.

---

## 5. Practical Algorithm — GLP-1 RA in HF

```
Patient with HF + T2DM/Obesity
    │
    ├─► HFrEF (EF ≤40%)
    │     → **SGLT2i first-line** (dapa/empa)
    │     → GLP-1 RA **not for HF** (use for glucose/weight if needed)
    │
    ├─► HFpEF (EF ≥50%)
    │     → **SGLT2i first-line** (dapa/empa)
    │     → If **BMI ≥30** + symptomatic (NYHA II–III):
    │           **Add semaglutide 2.4mg** (STEP-HFpEF)
    │
    └─► T2DM + HF (any EF) — glucose-lowering choice
          1. **SGLT2i** (HF + renal + glucose)
          2. **GLP-1 RA** (add if HbA1c not controlled + obesity)
          3. Avoid DPP-4i (saxa/alo ↑ HF hosp), TZDs (contraindicated)
```

## 6. Adverse Effects in HF Context

| Adverse Effect | Relevance to HF |
|----------------|-----------------|
| **GI (nausea, vomiting)** | May cause **volume depletion** → worsen HF; start low, go slow |
| **Tachycardia** | Semaglutide ↑ HR 2–3 bpm; monitor AF |
| **Hypoglycaemia** | Low risk alone; ↑ with insulin/sulfonylurea |
| **Pancreatitis** | Rare; avoid if history |
| **Medullary thyroid cancer** | Contraindicated if MEN2 / family history |

## 7. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **FIGHT trial** | Liraglutide in HFrEF → **neutral** (no benefit) |
| **STEP-HFpEF** | **Semaglutide 2.4mg in HFpEF + BMI ≥30** → ↑ KCCQ, ↑ 6MWD, −13% weight, −42% NT-proBNP |
| **LIVE trial** | Liraglutide post-MI → neutral |
| **SGLT2i vs GLP-1 RA** | SGLT2i = **first-line for HF** (all EF); GLP-1 RA = obesity-driven benefit in HFpEF |
| **HFrEF + GLP-1 RA** | **Not recommended** (FIGHT neutral) |
| **HFpEF + obesity** | **Semaglutide 2.4mg** (STEP-HFpEF) |
| **Weight loss** | Primary mechanism in STEP-HFpEF (13% loss) |
| **NT-proBNP** | −42% with semaglutide in STEP-HFpEF |
| **Positioning** | SGLT2i first; GLP-1 RA add-on for glucose/weight/obesity-HFpEF |
| **Semaglutide dose** | **2.4mg weekly** (max dose, not 1mg) for HFpEF |

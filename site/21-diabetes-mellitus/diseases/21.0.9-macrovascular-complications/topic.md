## High-Yield Exam Points (FCPS/MRCP)

| Topic | Key Facts |
|-------|-----------|
| **CV risk equivalent** | Diabetes = CHD risk equivalent (but debate: type 1 vs type 2, age, duration) |
| **Risk calculators** | QRISK3 (UK, includes DM) · ASCVD Pooled Cohort (US, includes DM) |
| **Statin intensity** | High-intensity atorvastatin 40-80mg (age 40-75) · Moderate if >75 or intolerance |
| **Aspirin primary prevention** | ADA: 10-year ASCVD risk >10% (age >50 + risk factor) · ESC: age >50 + risk factor · NICE: not routinely |
| **BP targets** | ADA/ESC: <130/80 · NICE: <140/90 (clinic) / <135/85 (ABPM) · <130/80 if CKD/albuminuria |
| **HF in diabetes** | HFrEF (EF ≤40%) · HFpEF (EF ≥50%) · HFmrEF (41-49%) · Diabetic cardiomyopathy |
| **SGLT2i in HF** | DAPA-HF (HFrEF), EMPEROR-Reduced (HFrEF), EMPEROR-Preserved (HFpEF), DELIVER (HFpEF) → all benefit |
| **GLP-1 RA in HF** | FIGHT (liraglutide neutral in HFrEF) · STEP-HFpEF (semaglutide improves symptoms in HFpEF) |
| **PAD screening** | ABI <0.9 diagnostic · TBI if calcified vessels · Screen if >50 or symptoms |
| **Critical limb ischaemia** | Rest pain, tissue loss, ABI <0.4 · Urgent vascular referral |

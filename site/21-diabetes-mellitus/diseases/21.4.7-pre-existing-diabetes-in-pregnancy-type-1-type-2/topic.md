# Pre-existing diabetes in pregnancy (type 1, type 2)

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Apply pre-conception care for pre-existing diabetes (T1DM/T2DM)
- [ ] State glycaemic targets in pregnancy
- [ ] Manage medication changes (stop teratogens, switch to insulin)
- [ ] Apply intrapartum and postpartum management

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Apply pre-conception care for pre-existing diabetes (T1DM/T2DM)
- [ ] State glycaemic targets in pregnancy

- [ ] Apply intrapartum and postpartum management

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Definition** | Diabetes diagnosed before pregnancy (T1DM or T2DM) |
| **Prevalence** | ~1% pregnancies (increasing with T2DM) |
| **Risk** | Miscarriage, congenital anomalies, macrosomia, pre-eclampsia, stillbirth |

## 3. Clinical Features / Presentation
| Phase | Features |
|-------|----------|
| **Pre-conception** | Optimise HbA1c, stop teratogens, folic acid, screen complications |
| **1st Trimester** | Insulin sensitivity up then down; nausea/vomiting; hypoglycaemia risk |
| **2nd/3rd Trimester** | Insulin resistance up 2-3x; insulin requirements increase 2-3x |
| **Delivery** | 38-39 weeks if well-controlled; earlier if complications |
| **Postpartum** | Insulin requirements drop dramatically; breastfeeding adjustments |

## 4. Classification / Staging / Grading

### Glycaemic Targets (ADA/NICE/ADIPS aligned)
| Measure | Target |
|---------|--------|
| **Fasting / Pre-meal** | **<5.3 mmol/L** (95 mg/dL) |
| **1-hour post-prandial** | **<7.8 mmol/L** (140 mg/dL) |
| **2-hour post-prandial** | **<6.4 mmol/L** (115 mg/dL) |
| **HbA1c** | **<48 mmol/mol (6.5%)** pre-conception; **<48** in pregnancy if achievable without hypo |

## 5. Diagnosis & Investigations
| Investigation | Timing |
|---------------|--------|
| **HbA1c** | Pre-conception (target <48); monthly in pregnancy (falsely low 2nd/3rd tri) |
| **SMBG** | 7x/day (pre/post meals, bedtime, 3am) |
| **CGM** | Recommended (NICE: all T1DM in pregnancy) |
| **Retinal screen** | Pre-conception, each trimester, 6mo postpartum |
| **Renal (ACR/eGFR)** | Pre-conception, each trimester |
| **Thyroid (T1DM)** | Pre-conception, each trimester |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **GDM** | Diagnosed in pregnancy; no pre-pregnancy DM |
| **Stress hyperglycaemia** | Acute illness; resolves |
| **T1DM presenting in pregnancy** | Ketosis, weight loss, young; check autoantibodies |

## 7. Management

### Pre-conception Care (Start 3-6 months before)
| Component | Action |
|-----------|--------|
| **HbA1c** | **<48 mmol/mol (6.5%)** if achievable without severe hypo |
| **Folic acid** | **5mg OD** (start >=3mo pre-conception -> 12 weeks) |
| **Medication review** | **STOP**: ACEi/ARB, statins, SGLT2i, GLP-1 RA, sulfonylureas, TZDs |
| **Insulin** | Switch to insulin (basal-bolus or CSII); optimise doses |
| **Screening** | Retinal, renal, cardiac, thyroid (T1DM), contraception until ready |

### Medication Safety in Pregnancy
| Drug | Safety | Notes |
|------|--------|-------|
| **Insulin** | **Safe** (does not cross placenta) | Gold standard; human insulin/analogues |
| **Metformin** | **Safe** (MiG trial) | Crosses placenta but no teratogenicity; 1st line oral if needed |
| **Glibenclamide** | Alternative | Crosses placenta; up neonatal hypoglycaemia; 2nd line |
| **SGLT2i / GLP-1 RA / DPP-4i / TZD / SU (except glibenclamide)** | **Contraindicated** | Teratogenicity / limited safety data |

### Intrapartum Management
| Aspect | Management |
|--------|------------|
| **Timing** | 38-39 weeks if well-controlled; 37-38 if complications |
| **Glucose target** | **4-7 mmol/L** (72-126 mg/dL) |
| **Insulin** | IV insulin infusion (sliding scale or algorithm); stop SC insulin |
| **Monitoring** | Hourly glucose; ketone monitoring |

### Postpartum
| Aspect | Management |
|--------|------------|
| **Insulin** | **Dramatic drop** (reduce to pre-pregnancy doses) |
| **Breastfeeding** | Adjust insulin/calories; monitor for hypoglycaemia |
| **Contraception** | Discuss pre-delivery; LARC preferred |
| **Follow-up** | 6-13 week review; annual HbA1c; retinal/renal screen |

```mermaid
flowchart TD
    A[Pre-existing DM] --> B[Pre-conception: HbA1c<48, Folic 5mg, Stop teratogens]
    B --> C[Pregnancy confirmed]
    C --> D[Switch to insulin (basal-bolus/CSII)]
    D --> E[Targets: FBG<5.3, 1h<7.8, 2h<6.4]
    E --> F[Monthly MDT review]
    F --> G[Delivery 38-39wk: IV insulin target 4-7]
    G --> H[Postpartum: reduce insulin; OGTT if T2DM; annual screen]
```

## 8. FCPS/MRCP High-Yield Summary

| Topic | Key Points |
|-------|------------|
| **Pre-conception HbA1c** | **<48 mmol/mol (6.5%)** |
| **Folic acid** | **5mg OD** (>=3mo pre-conception -> 12 weeks) |
| **Teratogens to STOP** | ACEi/ARB, statins, SGLT2i, GLP-1 RA, sulfonylureas, TZDs |
| **Glycaemic targets** | FBG <5.3, 1h <7.8, 2h <6.4 mmol/L |
| **Insulin** | Basal-bolus or CSII; requirements 2-3x increase in 3rd trimester |
| **Intrapartum** | IV insulin infusion; target 4-7 mmol/L |
| **Postpartum** | Insulin requirements drop; OGTT 6-13wk if T2DM; annual screen |

## 9. Viva Questions

| Question | Expected Answer |
|----------|-----------------|
| **What is the pre-conception HbA1c target for pre-existing diabetes?** | **<48 mmol/mol (6.5%)** |
| **What folic acid dose is recommended?** | **5mg OD** from >=3 months pre-conception to 12 weeks gestation |
| **Which medications must be stopped pre-conception?** | ACEi/ARB, statins, SGLT2i, GLP-1 RA, sulfonylureas, TZDs |
| **What are the glycaemic targets in pregnancy for pre-existing DM?** | FBG <5.3, 1h <7.8, 2h <6.4 mmol/L |
| **How do insulin requirements change in pregnancy?** | 1st tri: may decrease; 2nd/3rd: increase 2-3x (placental insulin resistance) |
| **What is the intrapartum glucose target?** | **4-7 mmol/L** via IV insulin infusion |
| **What happens to insulin requirements postpartum?** | **Dramatic drop** (placental insulin resistance removed); reduce to pre-pregnancy doses |
| **When do you do OGTT postpartum for T2DM?** | **6-13 weeks** (75g OGTT); annual HbA1c thereafter |

## 10. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Metformin in pregnancy?** | MiG trial: safe, crosses placenta but no teratogenicity; but insulin preferred for pre-existing DM |
| **Glibenclamide vs other SU?** | Glibenclamide crosses placenta but less than older SU; 2nd line oral if metformin fails |
| **SGLT2i/GLP-1 RA in pregnancy?** | Contraindeaated; teratogenicity risk / limited data |
| **HbA1c in pregnancy?** | Falsely low in 2nd/3rd tri (increased RBC turnover); use SMBG/CGM |

**Mnemonic: PRE-EXISTING-DM-PREG**
- **P**re-conception: HbA1c<48, **Folic 5mg**, Stop teratogens
- **R**etinopathy: screen pre-conception, each trimester
- **E**xisting DM: T1DM/T2DM known before pregnancy
- **E**xisting: HbA1c<48, Folic 5mg, Stop ACEi/ARB/Statin/SGLT2i/GLP-1/SU
- **X**isting DM targets: FBG<5.3, 1h<7.8, 2h<6.4
- **I**nsulin: basal-bolus/CSII; 2-3x increase 3rd tri
- **S**GLT2i/GLP-1/SU: STOP (teratogenic/contraindicated)
- **T**eratogens: ACEi/ARB/Statin/SGLT2i/GLP-1/SU/TZD
- **I**ntrapartum: IV insulin 4-7 mmol/L
- **N**eonates: macrosomia, hypoglycaemia, RDS risk
- **I**nsulin drop postpartum: dramatic reduction
- **N**eonatal hypo: monitor glucose
- **G**DM vs Pre-existing: HbA1c>=48/FPG>=7.0 at booking = pre-existing
- **D**elivery: 38-39wk if controlled
- **M**etformin: safe but insulin preferred for pre-existing DM

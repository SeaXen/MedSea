# Insulin Pump Types (Tethered, Patch, Closed-Loop)

> [!info]
> **CSII = Continuous Subcutaneous Insulin Infusion** — Tethered (tubed), Patch (tubeless), and Closed-Loop (AID) systems; only rapid-acting insulin used; NICE criteria for funding.

## 2. Pump Classification

| Category | Type | Examples | Key Feature |
|----------|------|----------|-------------|
| **Tethered (Tubed)** | Traditional | Medtronic 780G, Tandem t:slim X2, Ypsomed mylife YpsoPump | Tubing connects pump to infusion set |
| **Patch (Tubeless)** | Wearable/pod | **Insulet Omnipod 5**, Omnipod DASH, CeQur Simplicity, V-Go | Pod adheres to body; no tubing; controlled by PDA/phone |
| **Closed-Loop / AID** | Hybrid (HCL) | **Medtronic 780G, Tandem Control-IQ, Omnipod 5, CamAPS FX, Loop (DIY)** | Automates basal ± some correction boluses |
| | Full (experimental) | Beta Bionics iLet, DIY bicorneal | Automates basal + bolus |

## 3. Tethered Pumps (Traditional)

| Pump | CGM Integration | AID Algorithm | Key Features |
|------|-----------------|---------------|--------------|
| **Medtronic 780G** | Guardian 3/4 | **PID** (SmartGuard) | Meal detection, auto-correction bolus, 70–180 target adjustable |
| **Tandem t:slim X2** | Dexcom G6/G7 | **MPC** (Control-IQ) | Predictive 30 min ahead, sleep/exercise modes, remote monitoring |
| **Ypsomed mylife YpsoPump** | Dexcom G6/G7 / FreeStyle Libre 3 | **CamAPS FX** (on phone) | Small, customisable, CamAPS FX app-based AID |

**Pros**: Large insulin reservoir (300u), proven reliability, multiple AID options, screen on pump
**Cons**: Tubing (snagging, dislodgement), visible device, infusion set changes q2–3d

---

## 4. Patch Pumps (Tubeless)

| Pump | CGM Integration | AID Algorithm | Key Features |
|------|-----------------|---------------|--------------|
| **Insulet Omnipod 5** | Dexcom G6/G7 | **MPC** (SmartAdjust) | Pod holds 200u; controller = phone; waterproof (IP28); no tubing |
| **Omnipod DASH** | Dexcom G6/G7 | Manual (no AID) | Phone controller; no AID |
| **CeQur Simplicity** | None (standalone) | None | 3-day wearable; 200u; mechanical bolus button |
| **V-Go** | None | None | Disposable 24h; basal + on-demand bolus (mechanical) |

**Pros**: No tubing, discreet, waterproof, easy to wear, no infusion set tubing
**Cons**: Smaller reservoir (200u), pod change q3d (vs q2–3d set), adhesive issues, cost higher, limited AID (Omnipod 5 only)

## 5. NICE Criteria for CSII Access (TA151 / NG17/NG18)

### Adults (T1DM)
**All must be met**:
1. **T1DM** (or insulin-deficient T3c/pancreatectomy)
2. **HbA1c ≥69 mmol/mol (8.5%)** on MDI **OR** disabling hypoglycaemia (recurrent severe/unpredictable) **OR** HbA1c target not met despite optimal MDI + education + CGM
3. **Structured education completed** (DAFNE or equivalent)
4. **Competent in carb counting, SMBG/CGM, sick day rules**
5. **Commitment to follow-up** (pump team)

### Children/Young People (T1DM)
- **All children <12y**: CSII first-line if MDI fails (NG18)
- **Age ≥12y**: Same as adults

### Pregnancy
- CSII recommended if MDI fails to achieve targets (NICE NG3)

### T2DM
- Not routinely funded; exceptional case-by-case (severe insulin resistance, high TDD)

## 6. Pump Mechanics — Core Concepts

| Concept | Description | Typical Range |
|---------|-------------|---------------|
| **Basal rate** | Continuous insulin delivery (units/hour) | 0.05–2.0 u/hr; 24+ programmable rates |
| **Basal profile** | 24h pattern (often 3–6 segments) | Dawn phenomenon: ↑ 2–8 AM |
| **Temporary basal** | % of programmed basal (duration 30min–24h) | Exercise: 50–80%; Illness: 120–150% |
| **Standard bolus** | Immediate delivery for meals | Per ICR |
| **Extended / Square bolus** | Delivered over 30min–8h | High fat/protein meals, gastroparesis |
| **Dual / Multi-wave bolus** | Immediate + extended portion | Pizza, curry, restaurant meals |
| **Correction bolus** | Based on ISF + target | Per ISF |
| **Active Insulin / IOB** | Insulin still acting from previous boluses | Tracks 3–5h; prevents stacking |

## 7. Infusion Sets & Sites

| Set Type | Angle | Length | Best For |
|----------|-------|--------|----------|
| **Straight (90°)** | 90° | 6–9mm | Most adults, abdomen/thigh |
| **Angled (30–45°)** | 30–45° | 13–17mm | Lean, muscular, children |
| **Steel needle** | 90° | 6–8mm | Allergy to Teflon; short-term |
| **Soft cannula (Teflon)** | 90° or angled | 6–17mm | Standard |

**Site rotation**: Abdomen (avoid 2" around umbilicus), thighs, buttocks, upper arms. Change **every 2–3 days** (72h max).

## 8. Common Problems & Troubleshooting

| Problem | Cause | Solution |
|---------|-------|----------|
| **Unexplained hyperglycaemia** | Occlusion, dislodgement, insulin degradation, site lipohypertrophy | Check site; change set; rotate sites; new insulin vial |
| **Nocturnal hypoglycaemia** | Basal too high; exercise effect; alcohol | ↓ Basal 10–20% overnight; temp basal; review AGP |
| **Dawn phenomenon** | Growth hormone/cortisol surge 3–8 AM | ↑ Basal 2–3 AM; or AID handles automatically |
| **Pump alarm: Occlusion** | Kinked cannula, lipohypertrophy, insulin precipitation | Change set + site; check for lumps |
| **Adhesive failure** | Sweat, oil, allergy | Skin prep (IV prep, Mastisol); different adhesive; patch pump |

| Topic | Key Point |
|-------|-----------|

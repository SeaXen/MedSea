# Sulfonylurea responsiveness

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Explain mechanism of sulfonylurea responsiveness in KCNJ11/ABCC8 neonatal diabetes
- [ ] Apply switch protocol from insulin to sulfonylurea
- [ ] Monitor for dose adjustments and adverse effects

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Mechanism** | KATP channel mutation prevents closure -> sulfonylurea binds SUR1 -> closes channel -> insulin secretion |
| **Responsiveness** | ~90% of KCNJ11/ABCC8 PNDM respond to sulfonylureas |
| **Dose** | Higher than MODY (0.5-3 mg/kg/day) |

## 4. Classification / Staging / Grading

### Switch Protocol
| Step | Action |
|------|--------|
| **1** | Confirm KCNJ11/ABCC8 mutation |
| **2** | Start gliclazide MR 0.5 mg/kg/day (max 30mg if <10kg) |
| **3** | Reduce insulin by 50% day 1; stop insulin by day 3-5 if glucose controlled |
| **4** | Titrate SU weekly by 0.25-0.5 mg/kg/day to max 3 mg/kg/day |
| **5** | Monitor glucose 4-6x/day for first 2 weeks |

## 7. Management

### Dose Titration
| Week | Dose (mg/kg/day) | Monitoring |
|------|------------------|------------|
| **1** | 0.5 | 4-6x/day glucose; reduce insulin 50% |
| **2** | 1.0 | Stop insulin if glucose <10 mmol/L |
| **4** | 1.5-2.0 | Monitor for hypoglycaemia |
| **8** | Up to 3.0 | Max dose; monitor neuro development |

### Long-term Follow-up
| Parameter | Frequency |
|-----------|-----------|
| **HbA1c** | 3-monthly |
| **Growth** | 6-monthly (height, weight) |
| **Neurological** | Annual (DEND/iDEND) |
| **Dose adjustment** | As needed with growth |

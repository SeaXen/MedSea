# Type 1 diabetes in children

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Apply ISPAD guidelines for T1DM in children
- [ ] Manage DKA in children (cerebral oedema risk)
- [ ] Apply HbA1c targets by age
- [ ] Manage school, exercise, sick days in children

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Incidence** | Rising 3-5%/year; peak 10-14y; bimodal 4-7 and 10-14y |
| **Presentation** | 25-30% in DKA at diagnosis (<5y highest risk) |
| **Glycaemic Targets** | HbA1c <53mmol/mol (7.0%) if achievable without severe hypo |

## 3. Clinical Features / Presentation
| Age Group | Presentation |
|-----------|--------------|
| **<5 years** | Often DKA at diagnosis; non-specific symptoms (lethargy, polyuria) |
| **5-12 years** | Classic osmotic symptoms; weight loss |
| **Adolescents** | Similar to adults; adherence challenges; eating disorder risk |

## 4. Classification / Staging / Grading

### HbA1c Targets by Age (ISPAD)
| Age | Target HbA1c |
|-----|--------------|
| **<6 years** | <58 mmol/mol (7.5%) |
| **6-12 years** | <53 mmol/mol (7.0%) |
| **>12 years** | <53 mmol/mol (7.0%) |
| **All** | Individualised; avoid severe hypo |

### DKA in Children - Cerebral Oedema Risk
| Risk Factor | Significance |
|-------------|--------------|
| **<5 years** | Highest risk |
| **New-onset T1DM** | 0.5-1% incidence |
| **Rapid fluid/insulin** | Avoid rapid osmolar drop |
| **Low pCO2** | Hyperventilation risk |

## 5. Diagnosis & Investigations
| Test | Frequency |
|------|-----------|
| **HbA1c** | 3-monthly |
| **SMBG/CGM** | 4-8x/day; CGM recommended (NICE: all T1DM) |
| **Thyroid (TFT)** | Annual (autoimmune thyroiditis 15-20%) |
| **Coeliac (TTG)** | Annual (screening) |
| **Lipids** | Annual from age 12 |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **MODY** | <25y, non-obese, strong FH, negative autoantibodies |
| **Stress hyperglycaemia** | Resolves; no autoantibodies |
| **Cystic fibrosis-related DM** | Exocrine insufficiency, respiratory symptoms |

## 7. Management

### School & Daily Life
| Aspect | Management |
|--------|------------|
| **Individual Healthcare Plan** | Required (school, parents, diabetes team) |
| **Insulin** | Basal-bolus or CSII; CSII preferred (NICE) |
| **CGM** | All T1DM (NICE); predictive alerts for hypoglycaemia |
| **Exercise** | Pre-exercise carb if glucose <7; reduce basal if CSII |
| **Sick day rules** | Check ketones if ill/glucose>14; supplemental insulin |

### Transition to Adult Care
| Component | Detail |
|---------|--------|
| **Age** | 16-18 years (planned transition) |
| **Process** | Joint paediatric-adult clinics; DAFNE-Youth; psychologist |
| **Key Issues** | Adherence, eating disorders, contraception, alcohol, driving |

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **DKA at diagnosis** | 25-30% children; <5y highest risk |
| **HbA1c targets** | <6y: <58; >6y: <53 mmol/mol (avoid severe hypo) |
| **CGM** | NICE: all T1DM; predictive alerts for hypo |
| **CSII preferred** | NICE: CSII if HbA1c>58 on MDI or severe hypo |
| **Screening** | Annual thyroid, coeliac, lipids (age>12) |
| **Transition** | 16-18y; joint clinics; psychologist; DAFNE-Youth |
| **Cerebral oedema** | Kids DKA: mannitol 0.5-1g/kg; avoid rapid osmolar drop |

## 9. Viva Questions

| Question | Expected Answer |
|----------|-----------------|
| **What is the HbA1c target for a 4-year-old with T1DM?** | <58 mmol/mol (7.5%) |
| **What is the HbA1c target for a 14-year-old with T1DM?** | <53 mmol/mol (7.0%) |
| **What is the DKA cerebral oedema risk in children?** | <5y highest; 0.5-1% incidence; mannitol 0.5-1g/kg; avoid rapid osmolar drop >3 mOsm/kg/hr |
| **When do you transition to adult care?** | 16-18 years; planned joint paediatric-adult clinics |
| **What screening is needed for T1DM children?** | Annual: TFT, coeliac (TTG), lipids (age>12), retinal (age>12) |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **Children = same HbA1c as adults?** | NO - <6y: <58 mmol/mol (allows slightly higher to avoid hypo) |
| **CSII for all children?** | NICE: CSII if HbA1c>58 on MDI or severe hypo; not all |
| **Cerebral oedema only in DKA?** | Mostly DKA; can occur with rapid glucose correction in HHS |

**Mnemonic: T1DM-KIDS-CARE**
- **T**1DM kids: peak 10-14y, bimodal 4-7, 10-14
- **1** DKA at dx: 25-30% (highest <5y)
- **D**KA cerebral oedema: mannitol 0.5-1g/kg, avoid rapid osmolar drop
- **M**erge HbA1c targets: <6y <58, >6y <53
- **K**IDS: COE (cerebral oedema) risk highest <5y
- **I**nsulin: MDI or CSII (NICE prefers CSII if HbA1c>58)
- **D**AILY: CGM recommended (NICE all T1DM)
- **S**creening: thyroid, coeliac, lipids (age>12)
- **C**gm: predictive alerts for hypo
- **A**dherence: teen challenge; psychologist help
- **R**etinal screen: age>12y
- **E**ating disorders: screen teens (diabulimia)
- **T**ransition: 16-18y, joint clinics, DAFNE-Youth
- **S**chool plan: Individual Healthcare Plan mandatory

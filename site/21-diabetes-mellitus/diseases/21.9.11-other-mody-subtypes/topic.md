# Other MODY subtypes

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise less common MODY subtypes (HNF1B, PDX1, NEUROD1, KCNJ11, BLK, APPL1)
- [ ] Identify distinguishing clinical features
- [ ] Apply appropriate management per subtype

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Collective prevalence** | ~5-10% of MODY |
| **Key subtypes** | HNF1B (MODY5), PDX1 (MODY4), NEUROD1 (MODY6), KCNJ11 (MODY13), BLK (MODY11), APPL1 (MODY14) |

## 3. Clinical Features / Presentation

### HNF1B-MODY (MODY5)
| Feature | Detail |
|---------|--------|
| **Gene** | HNF1B (17cen-q21.3) |
| **Key features** | Renal cysts, renal dysplasia, GU malformations, hyperuricaemia |
| **Diabetes** | Variable; often insulin-requiring |
| **Extra-pancreatic** | Liver dysfunction, genital tract malformations, hypomagnesaemia |
| **Autoantibodies** | Negative |
| **C-peptide** | Variable |

### NEUROD1-MODY (MODY6)
| Feature | Detail |
|---------|--------|
| **Gene** | NEUROD1 (2q32) |
| **Features** | Pancreatic/brain development; neurological features possible |
| **Diabetes** | Young onset; insulin often required |
| **Neurological** | Learning difficulties, cerebellar hypoplasia |

### PDX1-MODY (MODY4)
| Feature | Detail |
|---------|--------|
| **Gene** | PDX1 (13q12.1) |
| **Features** | Pancreatic duodenal homeobox 1; pancreatic agenesis if homozygous |
| **Diabetes** | Variable; may need insulin |

### KCNJ11-MODY (MODY13)
| Feature | Detail |
|---------|--------|
| **Gene** | KCNJ11 (11p15.1) - Kir6.2 subunit |
| **Features** | Overlap with neonatal diabetes |
| **Sulfonylurea** | Responsive |

## 5. Diagnosis & Investigations
| Test | Indication |
|------|------------|
| **Genetic panel** | All MODY genes if probability >6% |
| **Renal ultrasound** | HNF1B-MODY (renal cysts) |
| **Magnesium** | HNF1B-MODY (hypomagnesaemia) |
| **Liver function** | HNF1B-MODY (liver dysfunction) |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|--------------------------|
| **HNF1A-MODY** | Low CRP, excellent SU response |
| **GCK-MODY** | Mild fasting hyperglycaemia, no treatment |
| **T1DM** | Autoantibodies +, low C-peptide |
| **T2DM** | Obese, older, insulin resistance |

## 7. Management

### HNF1B-MODY (MODY5)
| Issue | Management |
|-------|------------|
| **Diabetes** | Often insulin-requiring; SU variable |
| **Renal** | Nephrology follow-up; monitor eGFR, magnesium |
| **GU** | Urology review; prenatal counselling |
| **Magnesium** | Supplement if hypomagnesaemia |

### NEUROD1/PDX1
| Aspect | Management |
|--------|------------|
| **Diabetes** | Often insulin-requiring |
| **Neurological** | Paediatric neurology follow-up (NEUROD1) |

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **HNF1B-MODY** | Renal cysts, GU malformations, hypomagnesaemia; insulin often needed |
| **NEUROD1** | Neurological features; insulin often needed |
| **KCNJ11** | Sulfonylurea responsive |
| **Rare MODY** | PDX1, NEUROD1, BLK, APPL1 - insulin often needed |

## 9. Viva Questions
| Question | Expected Answer |
|----------|-----------------|
| **What distinguishes HNF1B-MODY?** | Renal cysts, GU malformations, hypomagnesaemia; insulin often needed |
| **What is the key feature of NEUROD1-MODY?** | Neurological involvement (learning difficulties, cerebellar hypoplasia) |
| **How do you manage HNF1B-MODY diabetes?** | Often insulin-requiring; monitor renal function, magnesium |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **All MODY = sulfonylurea responsive?** | NO - HNF1B/PDX1/NEUROD1 often insulin-requiring |
| **HNF1B = renal only?** | No - GU malformations, liver, magnesium, pancreas |

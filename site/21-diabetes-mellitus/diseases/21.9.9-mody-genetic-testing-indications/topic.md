## 4. Classification / Staging / Grading

### Clinical Indications for Genetic Testing (MODY Probability > 6%)
| Criterion | Points |
|-----------|--------|
| **Age at diagnosis < 25y** | High |
| **BMI < 25 kg/m2** | High |
| **Family history** | 3 generations affected (autosomal dominant) |
| **Negative autoantibodies** | GAD65, IA-2, ZnT8, IAA all negative |
| **C-peptide preserved** | >200 pmol/L at 3+ years post-diagnosis |
| **Non-insulin requiring** | >3 years post-diagnosis without insulin |

### MODY Probability Calculator
| Tool | Variables |
|------|-----------|
| **UK MODY calculator** | Age, BMI, FH, autoantibodies, C-peptide, treatment |
| **Threshold** | >6% = cost-effective for testing |

### Testing Strategy
| Approach | Indication |
|----------|------------|
| **Targeted panel** | Known MODY genes (HNF1A, GCK, HNF4A, HNF1B, etc.) |
| **Single gene** | Strong FH suggesting specific gene |
| **Exome/genome** | Negative panel; atypical features |

## 6. Differential Diagnosis
| Condition | Why Test? |
|-----------|-----------|
| **T1DM** | Exclude if autoantibodies +, low C-peptide |
| **T2DM** | Exclude if obese, older, insulin resistance |
| **Secondary DM** | Exclude pancreatic/endocrine/drug cause |

## 7. Management

### Testing Algorithm
```mermaid
flowchart TD
    A[Possible MODY] --> B[Calculate probability (UK calculator)]
    B --> C{Probability > 6%?}
    C -->|Yes| D[Targeted MODY panel]
    C -->|No| E[No genetic testing; clinical diagnosis]
    D --> F[Result: pathogenic variant?]
    F -->|Yes| G[Confirm diagnosis; family cascade testing]
    F -->|No| H[Consider exome/genome if high suspicion]
    F -->|VUS| I[Segregation analysis; functional studies]
```

### Cascade Testing
| Step | Action |
|------|--------|
| **1** | Identify pathogenic variant in proband |
| **2** | Offer predictive testing to 1st degree relatives |
| **3** | Pre-test counselling (insurance, family planning) |
| **4** | Prenatal diagnosis if requested |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **Test everyone <25?** | NO - use probability calculator; >6% threshold |
| **All diabetes <25 = MODY?** | NO - T1DM, T2DM, secondary more common |
| **Negative Abs = MODY?** | NO - need full clinical picture |

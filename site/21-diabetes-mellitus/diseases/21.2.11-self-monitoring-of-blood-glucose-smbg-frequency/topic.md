# Self-monitoring of blood glucose (SMBG) frequency

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Apply SMBG frequency guidelines per diabetes type and treatment
- [ ] Identify when to increase monitoring (illness, driving, pregnancy)
- [ ] Apply structured SMBG protocols (paired testing, pattern management)
- [ ] Recognise limitations of SMBG vs CGM

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Definition** | Capillary blood glucose testing by patient |
| **Standard** | Fingerstick glucose meter; ISO 15197 accuracy (±15% >100 mg/dL) |
| **CGM alternative** | rtCGM/isCGM; reduces SMBG need |

## 4. Classification / Staging / Grading

### SMBG Frequency by Patient Group
| Group | Frequency | Timing |
|-------|-----------|--------|
| **T1DM / T2DM on insulin** | **4-10x/day** | Pre-meals, bedtime, pre-driving, 3am (basal check), suspected hypo |
| **T2DM on SU/GLP-1/SGLT2i** | 1-2x/day | Pre-breakfast, pre-dinner, alternating |
| **T2DM on metformin/DPP-4i** | 2-3x/week | Pre-breakfast, pre-dinner (rotating) |
| **T2DM diet-only** | Individualised | Pre-breakfast (if monitoring) |
| **Pregnancy (T1/T2/GDM)** | **7x/day** | Fasting, 1h post-meals x3, bedtime, 3am |
| **High hypoglycaemia risk** | **Additional 3am check** | + CGM recommended |

### Structured SMBG Protocols
| Protocol | Schedule | Purpose |
|----------|----------|---------|
| **Paired testing** | Pre-meal + 2h post-meal | Assess carb/insulin match |
| **Pattern management** | 7-point profile (fasting, 3 meals pre/post, bed) | Pattern identification |
| **Basal check** | 3am glucose | Assess basal insulin adequacy |

## 5. Diagnosis & Investigations
| Test | Role |
|------|------|
| **SMBG** | Daily decision-making; dose adjustment |
| **CGM** | Pattern recognition; nocturnal hypo detection |
| **HbA1c** | 3-month average; long-term control |
| **CGM vs SMBG** | CGM reduces SMBG need; SMBG for calibration/confirmation |

## 7. Management

### Structured SMBG Protocol (Example: T1DM)
| Time | Purpose |
|------|---------|
| **Fasting** | Basal insulin adequacy |
| **Pre-breakfast** | Bolus calculation |
| **2h post-breakfast** | Breakfast bolus adequacy |
| **Pre-lunch** | Pre-meal correction |
| **2h post-lunch** | Lunch bolus adequacy |
| **Pre-dinner** | Dinner bolus calculation |
| **2h post-dinner** | Dinner bolus adequacy |
| **Bedtime** | Overnight basal; hypo prevention |
| **3am (if needed)** | Basal check |

### Situational Increases
| Situation | Additional Checks |
|-----------|------------------|
| **Illness** | q2-4h; ketones if glucose >14 |
| **Pre-driving** | Mandatory (DVLA) |
| **Exercise** | Pre, during (if >30min), post |
| **Alcohol** | Bedtime + 3am |
| **Pregnancy** | 7x/day (fasting, 1h post-meals x3, bedtime, 3am) |
| **Ramadan** | Pre-dawn, midday, pre-iftar, 2h post-iftar, bedtime |

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **T1DM/Insulin T2DM** | 4-10x/day (pre-meals, bedtime, 3am, pre-driving) |
| **T2DM non-insulin** | Individualised (2-3x/week rotating pre-meals) |
| **Pregnancy** | 7x/day (fasting, 1h post x3, bedtime, 3am) |
| **Illness** | q2-4h + ketones |
| **Driving** | Pre-driving mandatory |
| **CGM** | Reduces SMBG; predictive alerts; TIR/TBR |

## 9. Viva Questions
| Question | Expected Answer |
|----------|-----------------|
| **How often should a T1DM patient on basal-bolus test?** | 4-10x/day (pre-meals, bedtime, pre-driving, 3am, suspected hypo) |
| **What is the SMBG frequency for T2DM on metformin only?** | Individualised (e.g., 2-3x/week pre-breakfast/pre-dinner rotating) |
| **What is the SMBG frequency in pregnancy?** | 7x/day: fasting, 1h post-meals x3, bedtime, 3am |
| **When is 3am glucose check indicated?** | Suspected nocturnal hypoglycaemia; basal insulin adjustment; pregnancy |
| **What is paired testing?** | Pre-meal + 2h post-meal to assess bolus insulin adequacy |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **SMBG = CGM?** | NO - CGM continuous; SMBG intermittent; CGM reduces SMBG need |
| **All T2DM = daily SMBG?** | NO - individualised; non-insulin often 2-3x/week |
| **HbA1c replaces SMBG?** | NO - HbA1c = 3-month average; SMBG = daily decisions |

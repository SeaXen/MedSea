# Diabetic macular oedema (DMO)

## 1. Learning Objectives
- [ ] Differentiate centre-involving (CI-DMO) vs non-centre DMO
- [ ] Apply OCT criteria for DMO diagnosis (central subfield thickness)
- [ ] Select 1st line treatment: anti-VEGF for CI-DMO, laser for non-centre
- [ ] Manage anti-VEGF treatment protocols (loading, treat-and-extend, PRN)
- [ ] Recognise steroid implant (Ozurdex) as 2nd line

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Definition** | Retinal thickening ≤1DD (disc diameter) from fovea due to VEGF-mediated vascular leakage |
| **Prevalence** | ~7% T1DM, ~5% T2DM; leading cause of vision loss in DR |
| **Risk Factors** | HbA1c, hypertension, dyslipidaemia, duration, pregnancy, renal disease |
| **Pathophysiology** | Hyperglycaemia → VEGF ↑ → vascular permeability → retinal oedema; BRB breakdown |

## 3. Clinical Features / Presentation
| Presentation | Frequency | Key Features |
|-------------|-----------|--------------|
| **Blurred vision** | Most common | Central vision loss, distorted (metamorphopsia) |
| **Reading difficulty** | Common | Need brighter light, magnification |
| **Colour desaturation** | Less common | Washed-out colours |
| **Asymptomatic** | Early | Detected on screening OCT |

## 4. Classification / Staging / Grading

### DMO Classification (ETDRS / OCT)
| Type | Definition | OCT Criteria | Treatment Priority |
|------|------------|--------------|-------------------|
| **Centre-involving DMO (CI-DMO)** | Thickening ≤1DD from fovea | CST >300–320µm (device-dependent) | **Anti-VEGF 1st line** |
| **Non-centre-involving DMO** | Thickening >1DD from fovea | CST normal; peripheral thickening | Laser if threatening centre |

### Severity (OCT-based)
| Severity | Central Subfield Thickness (CST) |
|----------|----------------------------------|
| **Mild** | 300–400µm |
| **Moderate** | 400–600µm |
| **Severe** | >600µm |

### M-grading (Screening)
| Grade | Definition |
|-------|------------|
| **M0** | No DMO |
| **M1** | DMO present (retinal thickening ≤1DD from fovea) → **Urgent referral** |

## 5. Diagnosis & Investigations
| Investigation | Role | Key Details |
|---------------|------|-------------|
| **OCT (Optical Coherence Tomography)** | **Gold standard** | CST measurement; cystoid spaces; subretinal fluid; monitor response |
| **Digital photography** | Screening | M1 if thickening ≤1DD from fovea |
| **Fluorescein angiography** | Pre-laser / atypical | Focal leaks; capillary non-perfusion |
| **HbA1c, BP, lipids** | Systemic control | HbA1c <53; BP <130/80; fenofibrate |

## 6. Differential Diagnosis
| Condition | Distinguishing Features |
|-----------|-------------------------|
| **Cystoid macular oedema (CMO)** | Post-surgery, uveitis, retinal vein occlusion; similar OCT |
| **Age-related macular degeneration (AMD)** | Drusen, CNV, subretinal fluid; age >55; no DR |
| **Central serous chorioretinopathy (CSCR)** | Young male, stress, serous detachment; no DR |
| **Macular telangiectasia** | Bilateral, temporal to fovea, right-angle vessels |

## 7. Management

### Anti-VEGF Therapy (1st Line for CI-DMO)
| Agent | Dose | Licensing | Regimen |
|-------|------|-----------|---------|
| **Ranibizumab** | 0.5mg (0.05ml) | Lucentis (licensed) | Monthly ×3 → treat-and-extend / PRN |
| **Aflibercept** | 2mg (0.05ml) | Eylea (licensed) | Monthly ×3 → 8-weekly → treat-and-extend |
| **Bevacizumab** | 1.25mg (0.05ml) | Avastin (off-label) | Monthly ×3 → treat-and-extend / PRN |

> **Loading phase**: 3 monthly injections mandatory before extend/PRN
> **Treat-and-extend**: Extend interval by 2–4 weeks if dry; shorten if fluid recurs
> **PRN (pro re nata)**: Re-treat if CST ↑>50µm or vision ↓>5 letters

### Steroid Implant (2nd Line)
| Agent | Dose | Indication | Risks |
|-------|------|------------|-------|
| **Dexamethasone implant (Ozurdex)** | 700µg | Anti-VEGF inadequate/refractory | **IOP rise (30%)**, cataract progression |

### Laser (Non-centre DMO / Historical)
| Indication | Details |
|------------|---------|
| **Non-centre DMO** | Focal/grid laser to leaking microaneurysms >1DD from fovea |
| **Failed anti-VEGF** | Consider if persistent after 6+ injections |

### Systemic Control
| Target | Evidence |
|--------|----------|
| **HbA1c <53mmol/mol** | DCCT/UKPDS: ↓DMO progression |
| **BP <130/80** | UKPDS: tight BP ↓DR/DMO |
| **Fenofibrate** | FIELD/ACCORD-Eye: ↓DMO progression |

```mermaid
flowchart TD
    A[DMO Detected] --> B{CI-DMO (CST >300µm, ≤1DD)?}
    B -->|Yes| C[Anti-VEGF 1st line]
    B -->|No| D[Laser if threatening centre]
    C --> E[Loading: 3 monthly]
    E --> F{Treat-and-extend or PRN?}
```

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **CI-DMO definition** | Retinal thickening ≤1DD from fovea; CST >300–320µm |
| **1st line treatment** | **Anti-VEGF** (Ranibizumab 0.5mg, Aflibercept 2mg, Bevacizumab 1.25mg off-label) |
| **Loading phase** | 3 monthly injections before extend/PRN |
| **Treat-and-extend** | Extend interval 2–4wk if dry; most common regimen |
| **PRN** | Re-treat if CST ↑>50µm or vision ↓>5 letters |
| **2nd line** | Dexamethasone implant (Ozurdex) — watch IOP rise |
| **Non-centre DMO** | Laser (focal/grid) |
| **Systemic** | HbA1c <53, BP <130/80, fenofibrate ↓progression |

## 9. Viva Questions
| Question | Expected Answer |
|----------|-----------------|
| **What is centre-involving DMO?** | Retinal thickening ≤1 disc diameter from fovea; CST >300–320µm on OCT |
| **What is the 1st line treatment for CI-DMO?** | **Anti-VEGF** — Ranibizumab 0.5mg, Aflibercept 2mg, Bevacizumab 1.25mg (off-label); monthly ×3 loading then treat-and-extend/PRN |
| **What is the difference between treat-and-extend and PRN?** | T&E: proactive extension by 2–4wk if dry; PRN: reactive re-treatment if CST↑>50µm or vision↓>5 letters |
| **When do you use dexamethasone implant (Ozurdex)?** | Anti-VEGF inadequate after 6+ injections; monitor IOP (30% rise) |
| **How does fenofibrate affect DMO?** | FIELD/ACCORD-Eye: fenofibrate ↓DMO progression; consider if dyslipidaemia |
| **What is the ETDRS M1 grade?** | DMO on photography (thickening ≤1DD from fovea) → urgent referral |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **Bevacizumab unlicensed for DMO?** | YES — off-label; cost-effective; NICE allows if ranibizumab/aflibercept not suitable; same efficacy in trials |
| **Laser for CI-DMO?** | NO — **Anti-VEGF 1st line**; laser for non-centre DMO only |
| **OCT vs photography for DMO?** | OCT = gold standard (quantitative CST); photography = screening (M1 = refer) |

**Mnemonic: DMO-CI-AVE**
- **D**MO: thickening ≤1DD from fovea
- **M**1 grade = DMO on photo → refer
- **O**CT gold standard: CST >300-320µm
- **C**I-DMO = centre-involving ≤1DD → **Anti-VEGF 1st line**
- **I**ntravitreal: Ranibizumab 0.5mg, Aflibercept 2mg, Bevacizumab 1.25mg
- **L**oading: 3 monthly MUST
- **A**nti-VEGF → treat-and-extend OR PRN
- **T**reat-and-extend: extend 2-4wk if dry
- **P**RN: retreat if CST↑>50 or vision↓>5
- **E**xtend: most common regimen
- **O**zurdex (dexamethasone implant) 2nd line: watch IOP
- **N**on-centre DMO: laser
- **S**ystemic: HbA1c<53, BP<130/80, fenofibrate

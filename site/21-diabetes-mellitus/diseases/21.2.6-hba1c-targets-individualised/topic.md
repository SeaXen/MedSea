# HbA1c targets (individualised)

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Apply individualised HbA1c targets per ADA/EASD/NICE guidelines
- [ ] Recognise when to tighten or relax targets
- [ ] Counsel on HbA1c limitations (CKD, haemoglobinopathies)
- [ ] Apply targets in special populations (elderly, pregnancy, CKD)

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Standard target** | <53 mmol/mol (7.0%) |
| **Individualisation** | Based on age, comorbidities, hypoglycaemia risk, life expectancy |
| **Measurement** | NGSP/DCCT standardised; NGSP = IFCC × 0.0915 + 2.15 |

## 4. Classification / Staging / Grading

### HbA1c Targets by Guideline
| Guideline | Standard | Tight | Loose |
|-----------|----------|-------|-------|
| **ADA 2024** | <53 mmol/mol (7.0%) | <48 (6.5%) if safe | <58-64 (7.5-8.0%) if frail |
| **EASD/ADA 2023** | <53 | <48 if achievable | <58 (7.5%) if frail/elderly |
| **NICE NG28** | <48 (6.5%) | — | <58-64 if frail/elderly |

### Targets by Patient Profile
| Patient | Target | Rationale |
|---------|--------|-----------|
| **Newly diagnosed, young, no CVD** | <48 (6.5%) | Long life expectancy; prevent complications |
| **Standard** | <53 (7.0%) | Most adults |
| **Elderly/frail/comorbid** | <58-64 (7.5-8.0%) | Avoid hypoglycaemia; limited life expectancy |
| **CVD/CKD** | <53 (7.0%) | CVD benefit; avoid hypoglycaemia |
| **Pregnancy** | <48 (6.5%) pre-conception; <43 (6.1%) in pregnancy | Fetal protection |
| **CKD/ESRD** | Individualised; HbA1c unreliable | Use fructosamine/GA/CGM |

## 5. Diagnosis & Investigations
| Test | Frequency | Notes |
|------|-----------|-------|
| **HbA1c** | 3-monthly until target; then 6-monthly | NGSP/DCCT standardised |
| **FPG/SMBG** | Per regimen | Adjunct to HbA1c |
| **Fructosamine/GA** | If HbA1c unreliable | 2-3 week average |
| **CGM (TIR/GMI)** | If available | TIR >70% ≈ HbA1c <53 |

## 7. Management

### Target Adjustment Algorithm
```mermaid
flowchart TD
    A[New T2DM Diagnosis] --> B{Age/Comorbidities?}
    B -->|Young, no CVD, long LE| C[Target <48 (6.5%)]
    B -->|Standard adult| D[Target <53 (7.0%)]
    B -->|Elderly/frail/CKD/CVD| E[Target <58-64 (7.5-8.0%)]
    B -->|Recurrent severe hypo| E
    B -->|Limited LE| E
```

### Special Situations
| Situation | Target Adjustment |
|-----------|-------------------|
| **CKD G3b-G5** | HbA1c unreliable; use fructosamine/GA; target higher |
| **ESRD/Dialysis** | HbA1c unreliable; target <64; use GA/fructosamine |
| **Pregnancy** | Pre-conception <48; <43 in 1st tri; <48 in 2nd/3rd |
| **Haemoglobinopathy** | HbA1c unreliable; use fructosamine/GA/CGM |
| **Elderly >80/frail** | <64 (8.0%) prioritise avoiding hypo |

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **Standard target** | <53 mmol/mol (7.0%) |
| **Tight target** | <48 (6.5%) if achievable without hypo, short duration, no CVD |
| **Loose target** | <58-64 (7.5-8.0%) if elderly, frail, comorbidities, hypo risk |
| **CKD/ESRD** | HbA1c unreliable; use fructosamine/GA; target relaxed |
| **Pregnancy** | Pre-conception <48; pregnancy <43-48 |
| **Elderly/frail** | <58-64 to avoid hypoglycaemia |

## 9. Viva Questions
| Question | Expected Answer |
|----------|-----------------|
| **What is the standard HbA1c target for most adults with T2DM?** | **<53 mmol/mol (7.0%)** |
| **When do you target <48 mmol/mol (6.5%)?** | Newly diagnosed, young, no CVD, long life expectancy, no hypoglycaemia risk |
| **When do you relax targets to <58-64 (7.5-8.0%)?** | Elderly, frail, comorbidities, hypoglycaemia unawareness, limited life expectancy |
| **Why is HbA1c unreliable in CKD/ESRD?** | Falsely low due to ↓RBC lifespan, ESA use; use fructosamine/GA |
| **What is the pre-conception HbA1c target?** | <48 mmol/mol (6.5%) |
| **What is the HbA1c target in pregnancy?** | <48 (6.5%) generally; <43 (6.1%) if achievable without hypo |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **One target for all?** | NO - must individualise based on age, comorbidities, hypo risk |
| **Lower = better?** | NO - tight control in elderly increases mortality (ACCORD) |
| **HbA1c always reliable?** | NO - CKD, haemoglobinopathy, ESA, pregnancy, recent transfusion |

# Frailty and Individualised HbA1c Targets in the Elderly

> [!info]
> **Age alone ≠ target** — Use **frailty category** (fit / vulnerable / frail) to set HbA1c targets; avoid "one size fits all"; deintensify to prevent hypoglycaemia.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Classify older adults by frailty (CFS / Rockwood)
- [ ] Apply individualised HbA1c targets per category
- [ ] Identify drugs to deintensify/deprescribe
- [ ] Recognise hypoglycaemia risk factors in elderly
- [ ] Apply care home diabetes management principles

---

## 2. Frailty Classification (Clinical Frailty Scale — CFS)

| Category | CFS Score | Description | HbA1c Target |
|----------|-----------|-------------|--------------|
| **Fit** | 1–3 | Active, independent, comorbidities controlled | **<53 mmol/mol (7.0%)** |
| **Vulnerable / Mild frailty** | 4 | "Slowed up", needs help with heavy tasks | **<58 mmol/mol (7.5%)** |
| **Frail / Moderate-severe** | 5–7 | Needs help with ADLs, housebound | **<64 mmol/mol (8.0%)** |
| **Severely frail / End of life** | 8–9 | Completely dependent, terminal | **<75 mmol/mol (9.0%)** — symptom control only |

**CFS Assessment**: "How have you been in last 2 weeks?" — mobility, energy, activities, illnesses.

---

## 3. Individualised HbA1c Targets (ADA/NICE/IDF Consensus)

| Health Status | HbA1c Target | Rationale |
|---------------|--------------|-----------|
| **Healthy / Fit** (few comorbidities, intact cognition) | **<53 mmol/mol (7.0%)** | Long life expectancy; microvascular benefit |
| **Complex / Vulnerable** (multiple comorbidities, mild cognitive impairment) | **<58 mmol/mol (7.5%)** | Balance benefit vs hypoglycaemia risk |
| **Frail / Very complex** (ADL dependence, moderate-severe dementia) | **<64 mmol/mol (8.0%)** | Hypoglycaemia risk > microvascular benefit |
| **End-of-life / Palliative** | **<75 mmol/mol (9.0%)** | Symptom control only; avoid osmotic symptoms |

**Key**: **Avoid HbA1c <48 (6.5%) in frail elderly** — ↑ mortality (ACCORD legacy), ↑ falls, ↑ cognitive decline.

---

## 4. Deintensification / Deprescribing Algorithm

### When to Deintensify
| Trigger | Action |
|---------|--------|
| **HbA1c well below target** (e.g., <48 in frail) | Reduce/stop glucose-lowering drugs |
| **Recurrent hypoglycaemia** (Level 2/3) | Immediate deintensification |
| **Frailty progression** (CFS ↑) | Reassess targets, reduce therapy |
| **New dementia / cognitive decline** | Simplify regimen, avoid sulphonylureas/insulin if possible |
| **CKD progression** (eGFR <30) | Review metformin, sulfonylureas, insulin doses |
| **Polypharmacy / falls** | Reduce insulin/sulfonylurea burden |

### Drug-Specific Deintensification

| Drug | Deintensification Approach |
|------|----------------------------|
| **Sulfonylureas** | **First to stop** — high hypo risk, no CV benefit; taper over 2–4 weeks |
| **Insulin (basal)** | Reduce by 10–20% weekly; switch to once-daily if BD; consider stopping if HbA1c <58 |
| **Insulin (bolus)** | Stop first; retain basal if needed |
| **Metformin** | Continue if eGFR ≥30; reduce dose if eGFR 30–45; stop if <30 |
| **DPP-4 inhibitors** | Low hypo risk — can continue; reduce dose if eGFR <50 |
| **SGLT2 inhibitors** | Continue if eGFR ≥20 (CV/renal benefit); monitor volume status |
| **GLP-1 RA** | Continue if tolerated; reduce if weight loss excessive |

## 5. Hypoglycaemia Risk Factors in Elderly

| Factor | Mechanism |
|--------|-----------|
| **Renal impairment** (eGFR <45) | ↓ Insulin/glucose-lowering clearance |
| **Cognitive impairment / dementia** | Missed meals, erratic dosing, unable to recognise/treat hypo |
| **Polypharmacy** | Drug interactions, ↓ appetite, ↓ renal/hepatic clearance |
| **Malnutrition / sarcopenia** | ↓ Glycogen stores, ↓ gluconeogenesis |
| **Alcohol** | Impairs gluconeogenesis |
| **Beta-blockers** | Masks adrenergic symptoms (tremor, tachycardia) |
| **Sulfonylureas / insulin** | Direct hypo risk |
| **Autonomic neuropathy** | Impaired counter-regulation |

**Hypoglycaemia unawareness**: Common in elderly; CGM with alarms essential if on insulin/sulfonylurea.

---

## 6. Cognitive Impairment & Diabetes

| Issue | Management |
|-------|------------|
| **Missed doses** | Simplify to once-daily; use dosette box; carer administration |
| **Erratic eating** | Reduce/stop bolus insulin; basal only; regular carers for meals |
| **Unable to recognise hypo** | CGM with carer alerts; avoid sulfonylureas/insulin if possible |
| **Executive dysfunction** | Structured routines; visual aids; telemonitoring |
| **Advanced dementia** | **Comfort-focused** targets; stop glucose-lowering if HbA1c <64 |

---

## 7. Care Home Diabetes Management

| Principle | Action |
|-----------|--------|
| **Individualised targets** | Per frailty category (not blanket targets) |
| **Avoid sliding scale** | Use basal insulin + fixed bolus or basal only |
| **Hypoglycaemia protocol** | Written plan; carer training; glucagon available |
| **Medication review** | **6-monthly** (pharmacist-led); deprescribe aggressively |
| **Nutrition** | Regular meals/snacks; avoid "diabetic diet" (restrictive) |
| **Foot care** | Daily inspection; podiatry 3–6 monthly |
| **Screening** | Annual: HbA1c, renal, feet, eyes (if life expectancy >2y) |

---

## 8. End-of-Life Diabetes Care

| Situation | Approach |
|-----------|----------|
| **Last weeks of life** | Stop all glucose-lowering except insulin for symptom control |
| **Insulin** | Basal only once-daily; target 8–15 mmol/L (avoid osmotic symptoms) |
| **Steroids (dexamethasone)** | Expect hyperglycaemia; basal insulin + QDS correction |
| **Nausea/vomiting** | Reduce/stop insulin; treat symptoms |
| **Dying phase** | Stop insulin if not needed for comfort; capillary glucose only if symptomatic |

## 9. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **Frailty categorisation** | CFS 1–3 fit, 4 vulnerable, 5–7 frail, 8–9 severe/end-of-life |
| **HbA1c targets by frailty** | Fit <53; Vulnerable <58; Frail <64; End-of-life <75 mmol/mol |
| **Avoid HbA1c <48 in frail** | ↑ Mortality, falls, cognitive decline |
| **Deintensify first** | **Sulfonylureas** (highest hypo risk, no CV benefit) |
| **Insulin reduction** | 10–20% weekly; stop bolus first |
| **Cognitive impairment** | Simplify regimen; avoid bolus insulin; CGM with carer alerts |
| **Care homes** | Individual targets; avoid sliding scale; 6-monthly med review |
| **End-of-life** | Comfort targets 8–15 mmol/L; basal insulin only if needed |
| **Steroids + diabetes** | Basal insulin + QDS correction; anticipate hyperglycaemia |
| **SGLT2i/GLP-1 RA in elderly** | Continue if eGFR allows — CV/renal benefit outweighs risk |

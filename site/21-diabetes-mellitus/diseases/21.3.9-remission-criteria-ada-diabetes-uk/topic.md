## 4. Classification / Staging / Grading

### Remission Categories (ADA/EASD 2021)
| Category | Criteria |
|---------|----------|
| **Complete remission** | HbA1c <48 mmol/mol off ALL glucose-lowering meds ≥3 months |
| **Partial remission** | FPG 5.6-6.9 mmol/mol off meds ≥3 months (pre-diabetes range) |
| **Prolonged remission** | Complete remission sustained ≥5 years |

### Ineligible for Remission Label
| Scenario | Reason |
|----------|--------|
| **On any glucose-lowering med** | Including metformin |
| **HbA1c ≥48 mmol/mol** | Does not meet threshold |
| **<3 months off meds** | Insufficient duration |
| **Bariatric surgery <3mo** | Weight loss not stabilised |

## 5. Diagnosis & Investigations
| Test | Timing |
|------|--------|
| **HbA1c** | At 3 months off meds; then 3-6 monthly |
| **FPG** | If HbA1c unreliable (CKD, haemoglobinopathy) |
| **CGM** | TIR >70% supportive but not diagnostic |

## 6. Differential Diagnosis
| Scenario | Distinction |
|----------|-------------|
| **Partial remission** | FPG 5.6-6.9 off meds ≥3mo (pre-diabetes range) |
| **Good control on meds** | Not remission (on medication) |
| **Transient normoglycaemia** | <3 months off meds |

## 7. Management

### Confirming Remission
| Step | Action |
|------|--------|
| **1** | Confirm off ALL glucose-lowering medications ≥3 months |
| **2** | Measure HbA1c (or FPG if HbA1c unreliable) |
| **3** | Confirm HbA1c <48 mmol/mol (6.5%) |
| **4** | Document remission in records |
| **5** | Schedule 3-6 monthly monitoring |

### Relapse Definition
| Criteria | Action |
|----------|--------|
| **HbA1c ≥48 mmol/mol on 2 occasions** | Relapse confirmed; restart treatment |
| **FPG ≥7.0 on 2 occasions** | Relapse confirmed (if HbA1c unreliable) |
| **Restart medication** | Individualised (lifestyle + metformin 1st line) |

## 8. FCPS/MRCP High-Yield Summary
| Topic | Key Points |
|-------|------------|
| **Definition** | HbA1c <48 mmol/mol (6.5%) off ALL meds ≥3 months |
| **Categories** | Complete (<48 off meds), Partial (FPG 5.6-6.9), Prolonged (≥5y) |
| **Exclusions** | On any med, <3mo off meds, HbA1c ≥48 |
| **Relapse** | HbA1c ≥48 on 2 occasions -> restart treatment |
| **Monitoring** | 3-6 monthly HbA1c; counsel on weight maintenance |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **Remission = cure?** | NO - relapse possible; ongoing monitoring required |
| **Partial remission = diabetes?** | NO - pre-diabetes range (FPG 5.6-6.9); better than diabetes |
| **Metformin allowed in remission?** | NO - must be off ALL glucose-lowering meds |
| **HbA1c unreliable?** | Use FPG or CGM TIR >70% as supportive |

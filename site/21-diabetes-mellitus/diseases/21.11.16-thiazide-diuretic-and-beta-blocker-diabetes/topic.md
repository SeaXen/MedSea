# Thiazide diuretic and beta-blocker diabetes

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise thiazide and beta-blocker induced hyperglycaemia
- [ ] Explain mechanisms (hypokalaemia, ↓insulin secretion, masked hypoglycaemia)
- [ ] Apply risk mitigation strategies
- [ ] Differentiate from T2DM and other drug-induced DM

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Thiazides** | ↑Diabetes risk 20-30% (dose-dependent) |
| **Beta-blockers** | ↑Risk 20-25%; masks hypoglycaemia |
| **Mechanism (Thiazide)** | Hypokalaemia -> ↓insulin secretion; ↓insulin sensitivity |
| **Mechanism (Beta-blocker)** | ↓Insulin secretion; masks hypoglycaemia symptoms |

## 4. Classification / Staging / Grading

### Thiazide Risk by Dose
| Dose | Diabetes Risk |
|------|---------------|
| **Low (12.5mg)** | Low-moderate |
| **Standard (25mg)** | Moderate |
| **High (50mg)** | High |

### Beta-blocker Selectivity
| Type | Diabetes Risk |
|------|---------------|
| **Non-selective** (propranolol, nadolol) | Highest |
| **Cardio-selective** (atenolol, metoprolol, bisoprolol) | Lower |
| **Vasodilating** (carvedilol, nebivolol) | Lowest (may improve IR) |

## 5. Diagnosis & Investigations
| Test | Role |
|------|------|
| **Glucose/HbA1c** | Screen before/after initiation; monitor |
| **Potassium** | Check if on thiazides (hypokalaemia -> ↓insulin) |
| **Lipids** | Beta-blockers may worsen TG/HDL |

| **Antipsychotic DM** | Weight gain; specific agent history |

## 7. Management

### Thiazide-Induced Hyperglycaemia
| Strategy | Detail |
|----------|--------|
| **Dose reduction** | Low-dose (12.5-25mg) preferred |
| **Potassium** | Supplement if <4.0; K+-sparing combo |
| **Alternative** | ACEi/ARB/CCB if glucose intolerant |
| **Monitoring** | Glucose/K+ 4-6 weeks post-initiation |

### Beta-blocker Management
| Strategy | Detail |
|----------|--------|
| **Selectivity** | Prefer cardio-selective (bisoprolol, nebivolol, carvedilol) |
| **Vasodilating** | Carvedilol/nebivolol improve insulin sensitivity |
| **Monitoring** | Glucose/HbA1c 3-monthly if high risk |
| **Hypoglycaemia** | Warn: masks tremor/tachycardia; check glucose if unwell |

### Prevention & Mitigation
| Strategy | Detail |
|----------|--------|
| **Low-dose thiazide** | 12.5-25mg; avoid >25mg |
| **Cardio-selective BB** | Bisoprolol, metoprolol, nebivolol |
| **Vasodilating BB** | Carvedilol, nebivolol (↑insulin sensitivity) |
| **Monitor** | Glucose pre-treatment, 3/6/12mo, then annually |
| **Potassium** | Maintain >4.0 on thiazides |

## 9. Viva Questions
| Question | Expected Answer |
|----------|-----------------|
| **How do thiazides cause hyperglycaemia?** | Hypokalaemia -> ↓insulin secretion; ↓insulin sensitivity |
| **Which beta-blockers have lowest diabetes risk?** | Vasodilating (carvedilol, nebivolol) -> improve insulin sensitivity |
| **How do beta-blockers mask hypoglycaemia?** | Block adrenergic symptoms (tremor, tachycardia, sweating) |
| **What is the thiazide dose relationship?** | Dose-dependent: 12.5mg low, 25mg moderate, 50mg high risk |
| **What is the preferred beta-blocker in diabetes?** | Cardio-selective (bisoprolol, metoprolol) or vasodilating (carvedilol, nebivolol) |
| **Does diabetes resolve after stopping thiazide?** | Partially improves; may persist if underlying IR |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **All beta-blockers = same risk?** | NO - non-selective > cardio-selective > vasodilating (improves IR) |
| **Thiazide DM = T2DM?** | Drug-induced; often reversible with dose reduction/stop |
| **Beta-blocker in DM = contraindicated?** | NO - indicated for CAD/HF; choose cardio/vasodilating |

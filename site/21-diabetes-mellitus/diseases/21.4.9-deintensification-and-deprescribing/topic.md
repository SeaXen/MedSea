## 4. Classification / Staging / Grading

### Deprescribing Priority Order
| Priority | Drug Class | Reason |
|----------|------------|--------|
| **1 (STOP FIRST)** | **Sulfonylureas** | Highest hypoglycaemia risk; long half-life; renal accumulation |
| **2** | **Insulin (bolus/prandial)** | Complex; hypo risk; cognitive burden |
| **3** | **Insulin (basal)** | Reduce dose; simplify to once daily |
| **4** | **GLP-1 RA** | If weight loss/frailty worsening; GI intolerance |
| **4** | **SGLT2i** | If eGFR <20; volume depletion risk |
| **5 (KEEP)** | **Metformin** | If eGFR >=30; cardiorenal benefit |
| **5 (KEEP)** | **DPP-4i** | Weight neutral; low hypo; renal adjust (except linagliptin) |
| **5 (KEEP)** | **SGLT2i** | If eGFR >=20; cardiorenal benefit |
| **5 (KEEP)** | **ACEi/ARB/Statin** | Cardiovascular/renal protection |

## 7. Management

### Deprescribing Algorithm
```mermaid
flowchart TD
    A[Elderly DM on multiple agents] --> B{Assess: Frailty (Rockwood), HbA1c, Hypo history, Cognitive function}
    B --> C{HbA1c < target for frailty?}
    C -->|Yes (too tight)| D[Deintensify]
    C -->|No| E[Optimise current; consider intensification if fit]
    D --> F[STOP Sulfonylureas 1st]
    F --> G{Hypo resolved / HbA1c safe?}
    G -->|No| H[Reduce/stop bolus insulin; simplify basal]
    G -->|Yes| I[Monitor 3-monthly]
    H --> J{Hypo resolved?}
    J -->|No| K[Reduce basal insulin; consider once-daily]
    J -->|Yes| I
    I --> L[Monitor 3-monthly; CGM if available]
```

### Stepwise Deintensification
| Step | Action | Monitoring |
|------|--------|------------|
| **1** | **Stop sulfonylureas** (gliclazide, glimepiride, glibenclamide) | SMBG 4x/day x 2 weeks; watch for hypo resolution |
| **2** | **Stop prandial/bolus insulin** | Simplify to basal only; check pre-meal glucose |
| **3** | **Reduce basal insulin** (20-50% dose reduction) | Target higher fasting glucose per frailty target |
| **4** | **Simplify to once-daily basal** (glargine/degludec) | Reduce injection burden; carer-administered if needed |
| **5** | **Review GLP-1 RA/SGLT2i** | Stop if weight loss excessive, GI intolerance, eGFR < 20 |

### Simplification Strategies
| Strategy | Example |
|----------|---------|
| **Once-daily basal** | Glargine U-300 / Degludec OD |
| **Prefilled pens** | Reduce dosing errors |
| **Carer administration** | If cognitive impairment |
| **Blister packs** | Weekly organiser |
| **CGM with alerts** | Replace frequent SMBG |

## 9. Viva Questions
| Question | Expected Answer |
|----------|-----------------|
| **What is the first medication to stop in elderly diabetes?** | **Sulfonylureas** (highest hypoglycaemia risk) |
| **How do you simplify insulin in frail elderly?** | Stop bolus insulin; reduce basal 20-50%; switch to once-daily basal (glargine U-300/degludec) |
| **Which medications should be continued in frailty?** | Metformin (eGFR>=30), SGLT2i (eGFR>=20), DPP-4i, ACEi/ARB, statin |
| **When do you stop GLP-1 RA / SGLT2i in elderly?** | Excessive weight loss, GI intolerance, eGFR < 20, frailty worsening |
| **What is the HbA1c target for moderately frail elderly?** | < 64 mmol/mol (8.0%) |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **Stop all diabetes meds in frail?** | NO - keep cardiorenal protective (SGLT2i, ACEi/ARB, statin); stop SU/bolus insulin |
| **Deintensification = worse control?** | NO - targets relaxed per frailty; prevents hypo without harming outcomes |
| **Stop insulin completely?** | NO - reduce dose/simplify; basal insulin often still needed for glucose control |

**Mnemonic: DEINTENSIFY-DM**
- **D**eintensify: STOP sulfonylureas **FIRST** (hypo risk)
- **E**liminate: bolus insulin (complex, hypo risk)
- **I**nsulin basal: reduce 20-50%; simplify to OD
- **N**o more complex regimens: once-daily basal
- **T**argets: relax HbA1c per frailty (fit 53, mild 58, mod 64, sev 69)
- **E**liminate: GLP-1/SGLT2i if weight loss/frailty/eGFR<20
- **N**o sulfonylureas: glibenclamide/glimepiride highest risk
- **S**implify: once-daily basal; prefilled pens; carer admin
- **I**CGM: predictive alerts replace frequent SMBG
- **F**railty-based targets: fit 53, mild 58, mod 64, sev 69
- **Y**ield: monitor 2wk post-change; HbA1c 3-monthly
- **D**on't stop: metformin (eGFR>=30), SGLT2i (eGFR>=20), DPP-4i
- **M**etformin: half dose if eGFR 30-45
- **I**nsulin: reduce not stop (basal still needed)

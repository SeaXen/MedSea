# Phaeochromocytoma diabetes

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Recognise phaeochromocytoma as cause of secondary diabetes
- [ ] Explain mechanism (catecholamines -> glycogenolysis, ↓insulin)
- [ ] Apply diagnostic tests (plasma metanephrines, MIBG)
- [ ] Manage hyperglycaemia in phaeochromocytoma

## 2. Definition & Epidemiology
| Feature | Detail |
|--------|--------|
| **Definition** | Catecholamine-secreting tumour causing secondary diabetes |
| **Incidence** | 50% impaired glucose tolerance; 35% diabetes |
| **Mechanism** | Catecholamines -> ↑glycogenolysis, ↑gluconeogenesis, ↓insulin secretion |

## 5. Diagnosis & Investigations
| Test | Role |
|------|------|
| **Plasma free metanephrines** | **Gold standard** (sensitivity 99%) |
| **24h urinary metanephrines** | Confirmatory |
| **Clonidine suppression test** | If metanephrines equivocal |
| **CT/MRI abdomen** | Localisation (adrenal vs extra-adrenal) |
| **MIBG scintigraphy** | Functional imaging; extra-adrenal |
| **Genetic testing** | If syndromic (MEN2, VHL, NF1, SDHx) |

## 7. Management

### Acute Spell (Hypertensive Crisis + Hyperglycaemia)
| Step | Action |
|-------|--------|
| **1** | Phentolamine IV (alpha-blocker) for BP |
| **2** | Beta-blocker (ONLY after alpha-blockade) for tachycardia |
| **3** | Insulin infusion if severe hyperglycaemia |
| **4** | Fluids for volume expansion |

### Pre-operative (Definitive cure = surgery)
| Phase | Management |
|-------|------------|
| **Alpha-blockade** | Phenoxybenzamine (irreversible) 10-30mg/day 10-14 days pre-op |
| **Beta-blockade** | **After alpha** (propranolol/atenolol) if tachycardia |
| **Volume expansion** | High salt/fluids (prevents post-op hypotension) |
| **Glucose** | Insulin sliding scale; monitor labile glucose |

## 10. Confusions & Mnemonics
| Confusion | Clarification |
|-----------|---------------|
| **Phaeo DM = permanent?** | NO - resolves after tumour resection |
| **Beta-blocker first?** | NO - MUST alpha-blockade FIRST (unopposed alpha -> hypertensive crisis) |
| **All HTN = phaeo?** | NO - rule of 10s (10% extra-adrenal, 10% bilateral, 10% malignant, 10% familial) |

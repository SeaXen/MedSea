> [!tip]
> **Vaughan Williams classification guides toxicity**. **Class I (Na⁺ channel blockers)** = QRS widening, hypotension, seizures → **NaHCO₃**. **Class III (K⁺ channel blockers)** = QT prolongation, Torsades → **MgSO₄, pacing**. **Class II (β-blockers)** = see BB poisoning. **Class IV (CCBs)** = see CCB poisoning. **Amiodarone** = multi-class + iodine + pulmonary fibrosis. Key FCPS/MRCP: **NaHCO₃ for Class Ia/Ic QRS>100ms**; **MgSO₄ for Class III Torsades**; **avoid Class Ia/Ic in TCA/CCB/BB**; **amiodarone pulmonary toxicity**.

## 1. Learning Objectives
- Classify antiarrhythmics by Vaughan Williams and predict toxicity
- Manage Class I (Na⁺ channel) toxicity with NaHCO₃
- Manage Class III (K⁺ channel) toxicity with MgSO₄/pacing
- Recognize amiodarone-specific toxicity (pulmonary, thyroid, corneal)
- Differentiate from TCA, BB, CCB poisoning

## 2. Definition
Antiarrhythmic poisoning = toxicity from cardiac antiarrhythmic drugs causing **pro-arrhythmia, hemodynamic collapse, neurological, and organ-specific toxicity** based on Vaughan Williams class.

## 3. Core Physiology
### Vaughan Williams Classification & Toxicity
| Class | Mechanism | Agents | Key Toxicity |
|-------|-----------|--------|--------------|
| **Ia** | Na⁺ channel block (moderate), ↑ APD | Quinidine, Procainamide, Disopyramide | **QRS widening, QT↑, Torsades, hypotension, cinchonism (quinidine)** |
| **Ib** | Na⁺ channel block (weak), ↓ APD | **Lidocaine**, Mexiletine, Phenytoin | **CNS (seizures, confusion), hypotension** (less QRS effect) |
| **Ic** | Na⁺ channel block (strong), ↔ APD | Flecainide, Propafenone | **Marked QRS widening, hypotension, heart block, seizures** |
| **II** | β-blockade | Propranolol, Metoprolol, Esmolol, Sotalol* | **Bradycardia, hypotension, hypoglycemia, bronchospasm** |
| **III** | K⁺ channel block, ↑ APD | **Amiodarone**, Sotalol*, Dronedarone, Ibutilide, Dofetilide | **QT↑, Torsades, bradycardia, pulmonary (amiodarone), thyroid** |
| **IV** | Ca²⁺ channel block | Verapamil, Diltiazem | **Hypotension, bradycardia/heart block, hyperglycemia** |

* Sotalol = Class II + III

## 4. Clinical Features by Class

### Class Ia (Quinidine, Procainamide, Disopyramide)
- **Cinchoism** (quinidine): tinnitus, hearing loss, blurred vision, diplopia, confusion
- **QRS widening** (> 100 ms), **QT prolongation**, **Torsades de Pointes**
- **Hypotension** (vasodilation, myocardial depression)
- **Drug-induced lupus** (procainamide, hydralazine)
- **Agranulocytosis** (procainamide)

### Class Ib (Lidocaine, Mexiletine, Phenytoin)
- **CNS**: perioral numbness, tinnitus, confusion, **seizures**, coma
- **CV**: hypotension, bradycardia, heart block (high doses)
- **Phenytoin**: ataxia, nystagmus, gingival hyperplasia (chronic)

### Class Ic (Flecainide, Propafenone)
- **Marked QRS widening** (> 150 ms), **QT prolongation**
- **Severe hypotension**, cardiogenic shock
- **Heart block**, asystole
- **Seizures**, coma
- **Propafenone**: β-blockade adds bradycardia, bronchospasm

### Class III (Amiodarone, Sotalol, Dronedarone, Ibutilide, Dofetilide)
- **QT prolongation → Torsades de Pointes** (risk: female, hypokalemia, bradycardia, structural heart disease)
- **Bradycardia**, sinus node dysfunction, AV block
- **Amiodarone-specific** (chronic):
  - **Pulmonary fibrosis** (most serious) — cough, dyspnea, hypoxia
  - **Thyroid**: hypo- or hyperthyroidism (iodine load)
  - **Corneal microdeposits** (vortex keratopathy — asymptomatic)
  - **Hepatitis**, photosensitivity, blue-grey skin discoloration
  - **Neuropathy**, myopathy
- **Sotalol**: β-blockade + Class III → bradycardia, hypotension, Torsades

## 5. Differential Diagnosis
- **TCA poisoning**: similar QRS widening, but anticholinergic + seizures
- **BB/CCB poisoning**: bradycardia/hypotension without QRS widening (except propranolol/sotalol)
- **Digoxin**: bidirectional VT, hyperkalemia, Fab fragments
- **Electrolyte disorders**: hypokalemia, hypomagnesemia → QT prolongation

## 6. Investigations
- **ECG** — **serial**: QRS duration, QT interval, rhythm
- **Electrolytes**: K⁺, Mg²⁺, Ca²⁺ (critical for Torsades risk)
- **ABG/VBG** — pH, lactate
- **Drug levels**: lidocaine, procainamide (NAPA), amiodarone (rarely acute)
- **LFTs, TFTs, CXR** (amiodarone chronic)
- **Paracetamol level** (always)

## 7. Management

### 1. General Supportive Care
- **ABCDE**: continuous ECG monitoring, airway protection if seizures/coma
- **Correct electrolytes**: **K⁺ > 4.0 mmol/L, Mg²⁺ > 1.0 mmol/L** (critical for Torsades prevention)

### 2. Class Ia/Ic (Na⁺ Channel Blockade) → **Sodium Bicarbonate**
- **Indication**: **QRS > 100 ms**, hypotension, VT/VF
- **Dose**: **1-2 mEq/kg IV bolus**, target **pH 7.50-7.55**, QRS < 100 ms
- **Infusion**: 150 mEq in 1L D5W at 150-250 mL/hr to maintain pH
- **Avoid**: Class Ia/Ic antiarrhythmics (additive Na⁺ blockade)

### 3. Class III (K⁺ Channel Blockade) → **Magnesium + Pacing**
- **Torsades de Pointes**:
  - **MgSO₄ 2 g IV** over 10-20 min (repeat if needed)
  - **Overdrive pacing** (transvenous > transcutaneous) at 90-110 bpm
  - **Isoproterenol** (if pacing unavailable) — increases HR, shortens QT
  - **Avoid**: Class Ia/Ic, other QT-prolonging drugs
- **Bradycardia**: atropine → pacing if unresponsive

### 4. Class Ib (Lidocaine/Mexiletine/Phenytoin)
- **Seizures**: benzodiazepines (lorazepam/diazepam) → phenobarbital/propofol
- **Hypotension**: fluids → vasopressors
- **Phenytoin toxicity**: supportive (no specific antidote)

### 5. Amiodarone-Specific (Chronic/Overdose)
- **Pulmonary toxicity**: **stop amiodarone**, high-dose steroids (prednisolone 0.5-1 mg/kg), supportive
- **Thyroid dysfunction**: treat hypo-/hyperthyroidism per guidelines
- **Hepatitis**: stop drug, monitor LFTs
- **Bradycardia**: atropine, pacing if needed

### 6. Decontamination
- **Activated charcoal**: 1 g/kg if < 1-2h (sustained-release: consider WBI)
- **WBI**: sustained-release flecainide, propafenone, amiodarone

### 7. Enhanced Elimination
- **Not dialyzable** (high protein binding, large Vd, except procainamide/NAPA)
- **High-flux dialysis/hemoperfusion**: procainamide/NAPA (limited)
- **ILE (Intralipid)**: rescue for refractory cardiovascular collapse (lipophilic: flecainide, propafenone, lidocaine, amiodarone)

## 8. Complications
- Ventricular arrhythmias (Torsades, VF)
- Cardiogenic shock
- Seizures (CNS toxicity)
- Rhabdomyolysis (prolonged seizures/immobility)
- Pulmonary fibrosis (amiodarone)
- Thyroid dysfunction (amiodarone)

## 9. Prognosis
- **Class Ia/Ic**: good with early NaHCO₃; mortality if delayed
- **Class Ib**: good (short half-life); seizures self-limiting
- **Class III**: Torsades risk; amiodarone pulmonary fibrosis = poor if established
- **Amiodarone OD**: rare acute toxicity; chronic toxicity = major concern

## 10. FCPS/MRCP High-Yield Points
1. **Vaughan Williams class predicts toxicity**
2. **Class Ia/Ic**: QRS widening → **NaHCO₃** (1-2 mEq/kg, pH 7.50-7.55)
3. **Class III**: QT prolongation → **Torsades** → **MgSO₄ 2g IV + pacing**
4. **Class Ib**: seizures → benzos; **NO NaHCO₃ needed** (weak Na⁺ block)
5. **Amiodarone**: pulmonary fibrosis (stop + steroids), thyroid (iodine), corneal deposits
6. **Sotalol** = Class II + III → bradycardia + Torsades
7. **Procainamide** → lupus, agranulocytosis, NAPA metabolite
8. **Flecainide/propafenone** = marked QRS widening, shock
9. **Avoid Class Ia/Ic in TCA/BB/CCB** (additive Na⁺ block)
10. **Mg²⁺ > 1.0 mmol/L** for all Torsades risk

## 12. Common Confusions / Exam Traps
- **All Class I = NaHCO₃** → NO, **Class Ib (lidocaine) does NOT need NaHCO₃**
- **Class III = NaHCO₃** → NO, MgSO₄ + pacing
- **Amiodarone acute OD = pulmonary fibrosis** → NO, chronic only
- **Quinidine = cinchonism** (tinnitus, vision) but QRS widening also
- **Phenytoin for lidocaine seizures** → benzos first, phenytoin Class Ib additive
- **MgSO₄ for all ventricular arrhythmias** → only Torsades/polymorphic VT

## 13. Mnemonics
- **VAUGHAN WILLIAMS**: **I** = **Na** channel; **II** = **β**; **III** = **K** channel; **IV** = **Ca** channel
- **CLASS I TOXICITY**: **Ia/Ic** = **QRS** widening → **NaHCO₃**; **Ib** = **CNS seizures** → **Benzos**
- **CLASS III**: **K** channel block → **QT↑** → **Torsades** → **Mg + Pacing**
- **AMIODARONE**: **P**ulmonary fibrosis, **T**hyroid, **C**orneal, **H**epatic, **S**kin (blue-grey)
- **TORSADES**: **Mg**SO₄ 2g IV + **P**acing (overdrive)

## 📂 Chapter 1 Structure

### 1️⃣ Clinical Reasoning & Diagnostic Process
| Heading | Topic-Group | Disease-Level Topics |
|---------|-------------|----------------------|
| **1.1 Clinical Reasoning** | **Diagnostic Process** | History taking, Physical examination, Clinical hypothesis generation, Pattern recognition vs analytical reasoning, Diagnostic error types (premature closure, availability bias, anchoring) |
| | **Probabilistic Reasoning** | Pre-test probability, Likelihood ratios, Post-test probability, Fagan nomogram, Threshold model (test-treatment threshold) |
| | **Diagnostic Testing** | Sensitivity, Specificity, PPV, NPV, ROC curves, AUC, Choosing tests (SnNout/SpPin), Serial vs parallel testing |
| | **Clinical Decision Rules** | Wells score, Centor criteria, Ottawa rules, PERC rule, Canadian CT Head Rule, Rule validation (derivation/validation/impact) |
| | **Cognitive Biases** | Confirmation bias, Availability heuristic, Anchoring, Premature closure, Search satisficing, Overconfidence, Debiasing strategies |
| | **Diagnostic Error** | Types (no-fault, system, cognitive), Root cause analysis, M&M conferences, Second victim support |

### 1.2 Evidence-Based Medicine (EBM)
| Heading | Topic-Group | Disease-Level Topics |
|---------|-------------|----------------------|
| **1.2.1 Study Design** | **Hierarchy of Evidence** | Systematic reviews/MA, RCTs, Cohort, Case-control, Cross-sectional, Case series, Expert opinion |
| | **RCT Design** | Randomisation, Allocation concealment, Blinding, Intention-to-treat, Per-protocol, CONSORT, Cluster RCT, Adaptive designs |
| | **Observational Studies** | Cohort (prospective/retro), Case-control (matching, bias), Cross-sectional, Nested case-control |
| | **Systematic Reviews** | PRISMA, PICO, Search strategy, Risk of bias (RoB 2, ROBINS-I), Heterogeneity (I²), Publication bias (funnel plot), GRADE |
| | **Meta-Analysis** | Fixed vs random effects, Forest plots, Subgroup analysis, Sensitivity analysis, Network meta-analysis |
| **1.2.2 Critical Appraisal** | **Therapy** | RCT appraisal (validity, results, applicability), NNT, NNH, ARR, RRR, Fragility index |
| | **Diagnosis** | Diagnostic study appraisal, Sensitivity/specificity, LRs, Pre-test probability, Post-test probability |
| | **Prognosis** | Cohort appraisal, Hazard ratios, Kaplan-Meier, Cox regression, Competing risks |
| | **Harm/Etiology** | Case-control/cohort for harm, Attributable risk, Population attributable risk |
| | **Qualitative Research** | CASP, Thematic analysis, Framework analysis, Transferability |
| **1.2.3 Statistics for EBM** | **Common Metrics** | p-values, CI, Type I/II error, Power, Sample size, Parametric vs non-parametric |
| | **Regression** | Linear, Logistic, Cox, Poisson, Multivariable adjustment, Interaction, Confounding |
| | **Survival Analysis** | KM curves, Log-rank test, Cox PH model, Hazard ratios, Time-dependent covariates |
| | **Diagnostic Stats** | Sensitivity, Specificity, PPV, NPV, LRs, DOR, ROC/AUC, Youden index |
| **1.2.4 Applying Evidence** | **GRADE** | Quality of evidence (High/Moderate/Low/Very Low), Strength of recommendation (Strong/Conditional) |
| | **Guideline Development** | AGREE II, WHO handbook, NICE, SIGN, ESC, ADA/EASD, KDIGO |
| | **Shared Decision-Making** | Option grids, Decision aids, Risk communication, Values clarification |

## 🎯 FCPS/MRCP High-Yield "Must Know" Topics

### Tier 1: Core (Guaranteed to Appear)
- [ ] **WHO Analgesic Ladder** + Opioid Conversions (PO:SC 2:1, IV:SC 1:1)
- [ ] **Start Smart Then Focus** (AMS) + IVOST Criteria
- [ ] **Warfarin/DOAC Perioperative** Hold/Restart/Bridging
- [ ] **SGLT2 Inhibitors** STOP 3 Days Pre-op (Euglycaemic DKA)
- [ ] **Metformin** Hold Day of Surgery / 48h Contrast
- [ ] **ACEi/ARB** Hold 24h Major Elective
- [ ] **Beta-Blockers** Continue / Never Initiate Day of Surgery
- [ ] **Steroids >5mg>3w** → Stress Dose Hydrocortisone
- [ ] **PINCH Drugs** — Recognition & Prevention Strategies
- [ ] **TDM** — Hartford (Aminoglycosides), AUC (Vancomycin), Free Levels (Phenytoin, Valproate, Digoxin)
- [ ] **CYP3A4/P-gp Interactions** — Clarithromycin/Azoles/Rifampicin with CNIs, Statins, DOACs
- [ ] **MAOIs** — Stop 2w, Avoid Pethidine/Tramadol/Indirect Sympathomimetics

### Tier 2: High Probability
- [ ] **Antimicrobial Stewardship** — Empirical Regimens by Syndrome, Duration, De-escalation
- [ ] **C. difficile Prevention** — Clindamycin/Quinolones/Cephalosporins Highest Risk
- [ ] **Penicillin Allergy De-labelling** — 10% Label, <1% True IgE
- [ ] **Syringe Driver** — Compatibilities, Opioid Conversion, Renal Adjustments
- [ ] **Renal Opioids** — Avoid Morphine <30, Use Alfentanil/Fentanyl/Hydromorphone
- [ ] **Neuropathic Pain** — Amitriptyline/Duloxetine/Gabapentin/Pregabalin First-Line
- [ ] **Lithium** — Thiazide/ACEi/NSAID Interactions (Halve Dose), NDI, Hypothyroidism
- [ ] **Digoxin** — Amiodarone (Halve Dose), Sampling ≥6h, Free Level Indications
- [ ] **Phenytoin** — Zero-Order Kinetics, Free Level (Hypoalbuminaemia), Valproate Interaction
- [ ] **Carbamazepine** — Auto-Induction, HLA-B*15:02, Hyponatraemia
- [ ] **Valproate** — Pregnancy Contraindicated, Hepatotoxicity <2y, Lamotrigine Halve Dose
- [ ] **Tacrolimus/Cyclosporine** — C0 vs C2, CYP3A4/P-gp Interactions, CNI Toxicity
- [ ] **Sirolimus** — Wound Healing (Hold Peri-op), Pneumonitis, Hyperlipidaemia

### Tier 3: Differentiator (Vivas / SBA Nuance)
- [ ] **Drug Development** — Phase I-IV, CTD, PSUR/RMP/PASS, Generics vs Biosimilars, NICE ICER £20-30k/QALY
- [ ] **CD Schedules** — Sch 2 (Morphine, Diamorphine, Fentanyl, Oxycodone) = CD Rx + Register
- [ ] **Yellow Card / Black Triangle** — Report ALL ADRs for ▼ drugs (5 years)
- [ ] **Pharmacovigilance** — Signal Detection (ROR/PRR/IC), Risk Minimisation
- [ ] **Anticonvulsant TDM** — Phenytoin Zero-Order, Carbamazepine C2/Epoxide, Valproate Free Level
- [ ] **Immunosuppressant Conversion** — CNI to mTOR for Nephrotoxicity, CyA:Tac 5:1-7:1
- [ ] **Theophylline** — Smoking Doubles Clearance, Ciprofloxacin Halve Dose, Seizures Refractory
- [ ] **Antimicrobial Stewardship Biomarkers** — PCT <0.25 or ↓80-90% Stop Rule

## 📝 Exam Answer Templates (Viva/SAQ)

### Viva: "Describe the management of a patient on warfarin needing elective hip replacement"
1. **Risk Assessment**: Mechanical valve/AF CHA₂DS₂-VASc≥4/Recent VTE → High Thrombotic Risk
2. **Warfarin**: Hold 5 days pre-op (target INR <1.5)
3. **Bridging**: Therapeutic LMWH (Enoxaparin 1mg/kg BD) if high risk
4. **LMWH Timing**: Stop 24h pre-op (BD therapeutic)
5. **Post-op**: Restart Warfarin + LMWH 24h post-op when haemostasis secure
6. **Continue LMWH** until INR therapeutic × 2 days
7. **VTE Prophylaxis**: Mechanical + Pharmacological (Rivaroxaban 10mg OD 28-35d post THR)

### Viva: "Patient on tacrolimus starts clarithromycin for chest infection"
1. **Interaction**: Clarithromycin = Strong CYP3A4/P-gp Inhibitor → ↑ Tacrolimus 5-10x
2. **Immediate Action**: **Stop Clarithromycin** → Switch to **Azithromycin** (no CYP3A4 inhibition)
3. **If Clarithromycin Essential**: Reduce Tacrolimus dose 75-90%; Daily Levels; Monitor Renal/Neuro
4. **Monitoring**: Tacrolimus Level at 24h, 48h, 72h; Renal Function; Neurological Exam

### SAQ: "Syringe Driver Compatibility for Morphine + Midazolam + Antiemetic"
- **Morphine + Midazolam** = ✅ Compatible (WFI)
- **Add Levomepromazine** = ✅ Compatible (WFI) — broad antiemetic + sedative
- **Cyclizine >50mg/24h** = ❌ Precipitates — Use Levomepromazine instead
- **Hyoscine Hydrobromide** = ❌ — Use Glycopyrronium or Hyoscine Butylbromide
- **Alfentanil** = Use NaCl 0.9% (not WFI)

## 🎓 Final Notes for Exam Success

1. **Structure Answers**: Definition → Mechanism → Key Numbers → Clinical Application → Monitoring
2. **Use Numbers**: Doses, Ratios, Timeframes, Ranges (examiners love specifics)
3. **Contrast Tables**: Old vs New, Drug A vs Drug B, Indication A vs B
4. **Algorithms**: Stepwise approach (Hold → Convert → Monitor → Restart)
5. **Red Flags**: Know the "Never Events" (KCl on wards, Pethidine+MAOI, SGLT2i pre-op)
6. **Guideline Names**: NICE NG15/NG125/NG180, KDIGO, KDIGO, BST, WHO, ESC
7. **Cross-Link**: Show you see connections (e.g., CYP3A4 → CNIs + Statins + DOACs + Midazolam)

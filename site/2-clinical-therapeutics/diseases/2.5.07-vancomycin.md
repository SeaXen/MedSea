## 2. 2. 📊 AUC-Guided Dosing — **Current Standard (ASHP/IDSA 2020)**

| Parameter | **AUC/MIC-Guided (Preferred)** | **Traditional Trough-Only (Legacy)** |
|-----------|-------------------------------|--------------------------------------|
| **Target** | **AUC 400–600 mg·h/L** (assuming MIC ≤1 mg/L) | **Trough 15–20 mg/L** (severe infections) / **10–15 mg/L** (non-severe) |
| **Rationale** | AUC/MIC correlates with efficacy & nephrotoxicity | Trough >15 correlates with nephrotoxicity; poor AUC prediction |
| **Measurement** | **≥2 levels** (peak + trough OR 2 post-dose) → Bayesian software | Single trough (pre-dose) |
| **AUC:MIC Ratio Target** | **≥400** (MIC ≤1) | Not applicable |
| **Software** | **Bayesian (MWPharm, DoseMe, InsightRX, BestDose)** | Not needed |

> **Key Equations**:
> - **AUC₀₋₂₄ = (Dose / Cl) × (1 / (1 − e^(−kτ)))** — at steady state
> - **AUC/MIC ≥ 400** required for bactericidal activity (S. aureus MIC ≤1)
> - **Cl ≈ 0.00293 × CrCl + 0.014** (population estimate)

## 3. 3. 🧮 TDM Sampling Strategy

| Method | Levels Required | Timing | Best For |
|--------|----------------|--------|----------|
| **Bayesian (2-level)** | **Peak (1–2h post-end infusion) + Trough (pre-dose)** | Steady state (after 3–5 half-lives / ~24–48h) | **All patients — preferred** |
| **Bayesian (1-level + population)** | Trough only + population prior | Steady state | ICU CrRT (limited access) |
| **Trapezoidal (2-level)** | Peak (1–2h post) + Trough (pre-dose) | Steady state | If no Bayesian software |
| **Traditional (trough only)** | Trough (pre-dose) | Steady state | Legacy / resource-limited |

> **Steady state**: Vancomycin t½ ~6h (normal renal) → **3–5 half-lives = 18–30h** (check levels day 2–3)

## 4. 4. ⚖️ Renal Adjustment

| CrCl (mL/min) | Initial Dose (Intermittent) | Monitoring |
|---------------|----------------------------|------------|
| **>60** | 15–20 mg/kg q8–12h (max 2g/dose) | Levels at 24–48h |
| **40–60** | 15–20 mg/kg q12h | Levels at 24–48h |
| **30–40** | 15–20 mg/kg q12–18h | Levels at 24–48h |
| **20–30** | 15–20 mg/kg q18–24h | Levels at 24–48h |
| **10–20** | 10–15 mg/kg q24–48h | Levels at 48–72h |
| **<10 / Not on RRT** | **10–15 mg/kg q48–72h** | Levels at 48–72h |

### 1. Renal Replacement Therapy (RRT)
| Modality | Dose | Timing | Supplementation |
|----------|------|--------|----------------|
| **Intermittent HD** | **15–20 mg/kg post-HD** | After each session | **500–1000 mg** mid-week if long interval |
| **SLED / Extended HD** | 15–20 mg/kg post-session | After each session | As above |
| **CRRT (CVVH/CVVHDF)** | **10–15 mg/kg q24–48h** | Daily or q48h | Monitor levels; adjust by clearance |
| **PD** | 15–20 mg/kg q5–7d | Intraperitoneal or IV | Monitor levels |

> **Use actual body weight** unless obese (>120% IBW) → **adjusted body weight** (IBW + 0.4×(ABW-IBW))

## 5. 5. ☠️ Nephrotoxicity

| Parameter | Details |
|-----------|---------|
| **Definition** | ↑ SCr ≥0.5 mg/dL (≥44 μmol/L) OR ↑ 50% from baseline on ≥2 consecutive days |
| **Risk Factors** | **Trough >20 mg/L** (old), **AUC >650–700** (current), **Concomitant nephrotoxins** (aminoglycosides, piperacillin-tazobactam (↑ 2–3x risk!), contrast, NSAIDs, ACEi/ARB, amphotericin), **Critical illness, Sepsis, Age>65, Pre-existing CKD, Duration >7d** |
| **Vanco + Pip-Taz** | **↑ AKI risk 2–3x vs vanco alone or other β-lactams** — **"Vanc-Pip" nephrotoxicity** |
| **Monitoring** | **SCr q2–3d** (daily in ICU/critical); Baseline SCr before start |
| **Management** | **AUC-guided dosing keeps risk lowest**; If AKI: hold 24–48h, recheck AUC, ↓ dose, avoid concomitant nephrotoxins |

## 8. 8. 🤯 Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **AUC 400–600 vs Trough 15–20** | AUC-guided is current standard; trough 15–20 was old target for severe infections |
| **MIC >1 mg/L** | If MIC >1 (rare for MSSA/MRSA), target **AUC/MIC ≥400** (e.g., MIC=2 → AUC ≥800) — consult ID |
| **Bayesian vs Trapezoidal** | Bayesian uses population priors + levels → more accurate; Trapezoidal = direct AUC calculate from 2 points |
| **HD dosing** | **Post-HD** (not pre); supplements mid-week |
| **CRRT dosing** | **Low dose, frequent** (10–15 mg/kg q24–48h) — clearance is continuous but variable |

**Mnemonics:**
- **"AUC 400-600"** = **AUC 400–600** = efficacy + safety sweet spot
- **"VANCO-PiP = NEPHROTOXICITY"** = **Vanc + Pip-Taz** = 2–3x AKI risk
- **"BAYESIAN = PEAK + TROUGH"** = 2 levels for software
- **"STEADY STATE = DAY 2-3"** = t½ 6h → 4 half-lives = 24h
- **"HD = POST SESSION"** = dose after haemodialysis
- **"CRRT = LOW FREQUENT"** = 10-15 mg/kg q24-48h
- **"AUC >650 = NEPHROTOXICITY RISK"**

## 12. 12. 📝 Exam Answer Modes

### 1. Short Question (5 marks): *"Vancomycin TDM — current target, method, and nephrotoxicity risk factors."*
- **Target**: **AUC 400–600 mg·h/L** (AUC/MIC ≥400); **AUC >650–700 → nephrotoxicity**
- **Method**: **Bayesian** — **2 levels** (peak 1–2h post + trough pre-dose) at **steady state (day 2–3)**
- **Nephrotoxicity risks**: **AUC >650**, **Vanco + Pip-Taz (2–3x AKI)**, aminoglycosides, contrast, sepsis, age>65, CKD, duration >7d
- **Definition AKI**: SCr ↑≥0.5 mg/dL or ↑50% on 2 consecutive days

### 2. Viva (1 min): *"Patient on vancomycin + piperacillin-tazobactam for sepsis. Day 3: SCr rising. Why? Action?"*
- **Vanco + Pip-Taz = synergistic nephrotoxicity (2–3x risk)**
- **Stop Pip-Taz** if possible; switch to meropenem/ceftriaxone/cefepime
- **Recheck vancomycin AUC** — target 400–600
- **Hold vancomycin 24–48h** if AKI significant; resume at ↓ dose
- Avoid other nephrotoxins (contrast, NSAIDs, ACEi/ARB)

### 3. Ward Round (30 sec): *"Patient on HD started vancomycin for MRSA bacteraemia. Dosing?"*
- **15–20 mg/kg IV post-HD** (after each session)
- **500–1000 mg supplement** mid-week if long interdialytic interval
- Check level pre-next HD to guide supplemental dose

### 4. Last-Night Revision (1-liners):
- Vancomycin TDM = **AUC 400–600** (MIC≤1); AUC >650 = nephrotoxicity
- Bayesian: **Peak (1–2h) + Trough** at steady state (day 2–3)
- Vanco + Pip-Taz = **2–3x AKI** (avoid)
- HD = post-session dosing; CRRT = q24–48h
- Nephrotoxicity = SCr ↑0.5 or ↑50% on 2 days
- Obesity = adjusted BW (IBW + 0.4×(ABW-IBW))

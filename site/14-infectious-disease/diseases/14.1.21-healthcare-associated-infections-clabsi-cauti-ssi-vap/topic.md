# Healthcare-Associated Infections (HAI) — CLABSI, CAUTI, SSI, VAP

Related: [[Antimicrobial Prescribing Principles]], [[Antimicrobial Stewardship]], [[Infection in Immunocompromised Host (Transplant, Biologics)]], [[Sepsis and Septic Shock]], [[Clostridioides difficile Infection (CDI)]], [[Infection in Pregnancy and Neonates]], [[Urinary Tract Infections (Complicated, CA-UTI, Pyelonephritis)]], [[Hospital-Acquired and Ventilator-Associated Pneumonia (HAP-VAP)]]

> [!important]
> **HAI = infection acquired ≥48h after hospital admission (or ≤48h if healthcare-associated).** **CLABSI, CAUTI, VAP, SSI = ~80% of HAI.** **Prevention bundles = evidence-based, reduce HAI 30–70%.** **Surveillance = NHSN definitions; rate per 1000 device-days (CLABSI, CAUTI, VAP) or per 100 procedures (SSI).** **Antimicrobial stewardship + IPC = core strategies.**

## Learning Objectives
- Define HAI and classify by type (CLABSI, CAUTI, VAP, SSI, CDI)
- Apply diagnostic criteria (CDC/NHSN) for each HAI type
- Implement evidence-based prevention bundles (CLABSI, CAUTI, VAP, SSI)
- Select appropriate empirical therapy by HAI type and local epidemiology
- Implement surveillance and outbreak investigation
- Apply antimicrobial stewardship principles to HAI management

## Definitions & Surveillance
| HAI Type | Definition | Rate Denominator |
|----------|------------|------------------|
| **CLABSI** | **Central line-associated BSI** — laboratory-confirmed BSI in patient with central line ≥48h, not secondary to another site | **1000 central line-days** |
| **CAUTI** | **Catheter-associated UTI** — symptomatic UTI with indwelling catheter ≥48h | **1000 catheter-days** |
| **VAP** | **Ventilator-associated pneumonia** — pneumonia in patient on invasive ventilation ≥48h | **1000 ventilator-days** |
| **SSI** | **Surgical site infection** — infection at surgical site within 30d (90d if implant) | **100 procedures** |
| **CDI** | **Clostridioides difficile infection** — see CDI note | **10,000 patient-days** |

> [!key]
> **HAI = infection ≥48h after admission (not incubating at admission).** **CLABSI/CAUTI/VAP rates per 1000 device-days; SSI per 100 procedures.**

## Central Line-Associated BSI (CLABSI)
### Diagnosis (CDC/NHSN)
| Criterion | Details |
|-----------|---------|
| **Central line** | In place ≥48h before culture (or day of device removal) |
| **Laboratory-confirmed BSI** | **≥1 blood culture** with recognised pathogen **OR** ≥2 blood cultures with common skin commensal (CoNS, Corynebacterium, Bacillus, Micrococcus) + clinical signs (fever, hypotension) |
| **Not secondary** | No other site infection explaining BSI |

### Common Pathogens
| Pathogen | % | Key Feature |
|----------|---|-------------|
| **Coagulase-negative Staphylococci (CoNS)** | 30–40% | Most common; often contamination but true CLABSI if ≥2 cultures |
| **Staphylococcus aureus** | 15–20% | MSSA/MRSA; high mortality |
| **Enterococcus spp.** | 10–15% | VRE increasing |
| **Candida spp.** | 10–15% | C. albicans, C. glabrata, C. parapsilosis (biofilm) |
| **Gram-negative (Klebsiella, E. coli, Pseudomonas, Enterobacter)** | 15–20% | ESBL/CRE concern |
| **Other** | <5% | Bacillus, Corynebacterium, Stenotrophomonas |

### CLABSI Prevention Bundle (IHI/CDC)
| Component | Action |
|-----------|--------|
| **Hand hygiene** | Before/after line access; alcohol-based rub |
| **Maximal barrier precautions** | **Sterile gloves, gown, cap, mask, large drape** for insertion |
| **Chlorhexidine skin prep** | **2% CHG in 70% alcohol** (superior to povidone-iodine) |
| **Optimal site selection** | **Subclavian > Jugular > Femoral** (lowest infection risk) |
| **Daily review of line necessity** | **Remove as soon as no longer needed** |
| **Chlorhexidine dressings** | **CHG-impregnated sponge/dressing** for lines >7d |
| **Hub disinfection** | **Scrub hub 15s with 2% CHG/70% alcohol** before each access |
| **Avoid routine replacement** | Guidewire exchange NOT recommended for CLABSI prevention |

> [!tip]
> **Subclavian site = lowest CLABSI risk.** **CHG skin prep + maximal barrier = strongest evidence.** **Daily line necessity review = most impactful.** **CHG dressings for lines >7d reduce CLABSI 50%.**

### CLABSI Empirical Therapy
| Scenario | Regimen |
|----------|---------|
| **No MDR risk, stable** | Ceftriaxone 2g IV 24h + Vancomycin 15–20mg/kg 6h |
| **MDR risk (prior abx, ICU, known colonisation)** | Piperacillin-tazobactam 4.5g 6h OR Meropenem 1g 8h (+ Vancomycin) |
| **Suspected Candida** | Echinocandin (caspofungin/micafungin/anidulafungin) |
| **Line removal** | **Strongly recommended** for S. aureus, Candida, Pseudomonas, Bacillus, mycobacteria; consider salvage for CoNS |

## Catheter-Associated UTI (CAUTI)
### Diagnosis (CDC/NHSN)
| Criterion | Details |
|-----------|---------|
| **Indwelling catheter** | ≥48h (or day of removal) |
| **Symptoms** | Fever (>38°C), suprapubic pain, CVA tenderness, altered mental status, rigors |
| **Microbiology** | **≥10³ CFU/mL** of ≥1 organism from catheter specimen OR **≥10⁵ CFU/mL** from voided specimen |

### Common Pathogens
| Pathogen | % |
|----------|---|
| **E. coli** | 25–30% |
| **Klebsiella** | 15–20% |
| **Pseudomonas** | 10–15% |
| **Enterococcus** | 10–15% |
| **Candida** | 5–10% |
| **Proteus, Morganella, Providencia** | 10–15% |

### CAUTI Prevention Bundle
| Component | Action |
|-----------|--------|
| **Avoid unnecessary catheterisation** | Use alternatives (condom catheter, intermittent cath, nappies) |
| **Aseptic insertion** | Sterile technique, smallest appropriate catheter |
| **Closed drainage system** | Maintain unobstructed flow; no disconnections |
| **Catheter securement** | Prevent traction/urethral trauma |
| **Drainage bag below bladder** | Prevent reflux |
| **Daily necessity review** | **Nurse-driven protocol: remove ASAP** |
| **Periurethral cleaning** | Routine soap/water (no antiseptic benefit) |
| **Antibiotic prophylaxis at insertion** | **NOT routinely recommended** |
| **Catheter material** | Silver-alloy or hydrogel-coated if prolonged (>7d) anticipated |

> [!key]
> **Daily review of catheter necessity with nurse-driven removal protocol = single most effective intervention.** **Avoid unnecessary catheterisation = primary prevention.**

### CAUTI Management
| Step | Action |
|------|--------|
| **1. Remove/replace catheter** | If ≥2 days old; send tip for culture if removed |
| **2. Empirical antibiotics** | As per complicated UTI (see UTI note) |
| **3. Duration** | **7 days** if catheter removed; **10–14 days** if catheter must remain |
| **4. Catheter change** | **Replace after 48–72h of antibiotics** if catheter must stay |

## Ventilator-Associated Pneumonia (VAP)
### Diagnosis (CDC/NHSN — Clinical + Microbiological)
| Criterion | Details |
|-----------|---------|
| **Ventilation** | Invasive ventilation ≥48h |
| **Chest X-ray** | New/progressive infiltrate |
| **Clinical** | Fever/hypothermia, ↑ WBC, purulent secretions, ↑ O₂ requirement |
| **Microbiology** | **Qualitative/semi-quantitative culture** (endotracheal aspirate ≥10⁵ CFU/mL; BAL ≥10⁴ CFU/mL) |
| **Exclude** | Non-infectious causes (atelectasis, ARDS, pulmonary oedema) |

### Common Pathogens
| Timeframe | Pathogens |
|-----------|-----------|
| **Early VAP (≤5 days)** | S. aureus (MSSA), S. pneumoniae, H. influenzae, Enterobacteriaceae |
| **Late VAP (>5 days)** | **MRSA, Pseudomonas, Acinetobacter, Stenotrophomonas, Enterobacteriaceae (ESBL/CRE)** |

### VAP Prevention Bundle (IHI/CDC)
| Component | Action |
|-----------|--------|
| **Head of bed elevation** | **30–45°** (unless contraindicated) |
| **Daily sedation vacation** | **Daily sedation interruption + spontaneous breathing trial** |
| **Peptic ulcer prophylaxis** | PPI/H2 blocker (balance with C. difficile risk) |
| **DVT prophylaxis** | Pharmacological (unless contraindicated) |
| **Oral care** | **Chlorhexidine 0.12–2% oral rinse q6–12h** |
| **Subglottic secretion drainage** | **Endotracheal tube with subglottic suction port** |
| **Early extubation** | Daily assessment for readiness to extubate |
| **Hand hygiene + PPE** | Standard + contact for MDRO |

> [!tip]
> **Head of bed 30–45° + daily sedation vacation + SBT = strongest evidence.** **Subglottic secretion drainage reduces VAP 50%.**

### VAP Empirical Therapy
| Scenario | Regimen |
|----------|---------|
| **Early VAP, no MDR risk** | Ceftriaxone 2g 24h +/− Vancomycin (if MRSA risk) |
| **Late VAP / MDR risk** | Piperacillin-tazobactam 4.5g 6h OR Meropenem 1g 8h + Vancomycin 15–20mg/kg 6h |
| **MRSA risk** | Add Vancomycin (trough 15–20) OR Linezolid |
| **Pseudomonas risk** | Anti-pseudomonal β-lactam (Pip-tazo, Meropenem, Cefepime, Ceftazidime) +/− Aminoglycoside |
| **Duration** | **7 days** (short-course non-inferior to 14d); stop if culture negative |

## Surgical Site Infection (SSI)
### Classification (CDC)
| Type | Depth | Timeframe |
|------|-------|-----------|
| **Superficial Incisional** | Skin/subcutaneous | ≤30 days |
| **Deep Incisional** | Fascia/muscle | ≤30 days (90 days if implant) |
| **Organ/Space** | Organ/space manipulated | ≤30 days (90 days if implant) |

### Risk Factors
| Patient | Operative | Postoperative |
|---------|-----------|---------------|
| Diabetes, obesity, smoking, immunosuppression, malnutrition, age >60, prolonged pre-op stay | **Duration >T**, contaminated/dirty wound, emergency, implant, blood loss >1L, poor technique | Hypothermia, hyperglycaemia, hypoxaemia, haematoma, drain, transfusion |

### SSI Prevention Bundle (WHO/CDC)
| Phase | Intervention |
|-------|--------------|
| **Pre-operative** | **Shower with CHG night before + morning**; **No hair removal** (or clippers if necessary); **Antibiotic prophylaxis within 60 min before incision** (cefazolin 2g IV; add metronidazole if colorectal); **Glycaemic control** (perioperative glucose <180); **Normothermia** (>36°C); **Oxygenation** (FiO₂ 80% intraop + 2h post) |
| **Intra-operative** | **Skin prep: CHG 2% in alcohol**; **Sterile technique**; **Minimise OR traffic**; **Maintain normothermia**; **Glycaemic control** |
| **Post-operative** | **Continue antibiotic ≤24h** (no benefit beyond); **Glycaemic control** (first 48h); **Normothermia**; **Early mobilisation**; **Wound care** (sterile dressing 48h) |

### SSI Empirical Therapy
| Scenario | Regimen |
|----------|---------|
| **Superficial, MSSA/Strep** | Flucloxacillin 2g IV 6h / Cephalexin 500mg 6h |
| **Deep/Organ space, no MDRO** | Ceftriaxone 2g 24h + Metronidazole 500mg 8h |
| **MRSA risk** | Vancomycin 15–20mg/kg 6h +/− Ceftriaxone/Metronidazole |
| **Gram-negative / MDRO** | Piperacillin-tazobactam 4.5g 6h OR Meropenem 1g 8h |
| **Duration** | **5–7 days** (superficial); **7–14 days** (deep/organ space); **source control critical** (drainage, debridement) |

## Antimicrobial Stewardship in HAI
| Intervention | Description |
|--------------|-------------|
| **Formulary restriction** | Carbapenems, newer agents (CAV, MV, IR, plazomicin) require ID approval |
| **Prospective audit & feedback** | ID pharmacist/physician reviews at 48–72h; recommends de-escalation/stop |
| **IV→oral switch** | Protocol-driven at 48–72h if clinical stability |
| **Automatic stop orders** | Default durations (e.g., VAP 7d, CAUTI 7d, SSI 5–7d) with mandatory renewal |
| **Biomarker guidance** | **PCT-guided** antibiotic discontinuation in sepsis/VAP |
| **Penicillin allergy de-labelling** | Assessment + graded challenge → ↓ vancomycin/fluoroquinolone use |
| **Diagnostic stewardship** | **Do NOT culture asymptomatic patients** (catheter tips, urine, blood without clinical signs) |
| **Device removal** | **CLABSI: remove for S. aureus, Candida, Pseudomonas, Bacillus, mycobacteria**; **CAUTI: remove if possible**; **VAP: no routine line removal** |

## Outbreak Investigation
| Step | Action |
|------|--------|
| **1. Confirm outbreak** | Rate >expected baseline; molecular typing (WGS, PFGE) |
| **2. Case definition** | Person, place, time, clinical criteria |
| **3. Case finding** | Active surveillance; retrospective review |
| **4. Descriptive epidemiology** | Epidemic curve; person/place/time |
| **5. Hypothesis generation** | Common source vs person-to-person |
| **6. Analytical study** | Cohort/case-control; environmental sampling |
| **7. Control measures** | Isolation, cohorting, enhanced cleaning, device removal, antibiotic restriction |
| **8. Communication** | Staff, patients, public health, media |
| **9. Monitoring** | Continue surveillance until baseline reached |

## FCPS/MRCP High-Yield Points
- **HAI = ≥48h after admission; CLABSI, CAUTI, VAP, SSI = 80% of HAI**
- **CLABSI: CoNS #1; subclavian > jugular > femoral; CHG prep + maximal barrier + daily review + CHG dressings**
- **CAUTI: E. coli #1; daily necessity review = most effective prevention; remove ASAP**
- **VAP: early (≤5d) = MSSA, H. influenzae; late (>5d) = MRSA, Pseudomonas, ESBL; HOB 30–45° + daily SBT + subglottic drainage**
- **SSI: cefazolin prophylaxis ≤60min pre-incision; discontinue ≤24h post-op; glycaemic control + normothermia**
- **CLABSI treatment: line removal for S. aureus, Candida, Pseudomonas, Bacillus; CoNS consider salvage**
- **CAUTI: remove catheter if possible; 7d abx if removed, 10–14d if retained**
- **VAP duration: 7 days (short-course non-inferior)**
- **SSI prophylaxis: cefazolin 2g IV ≤60min pre-incision; stop ≤24h; glucose <180, normothermia**
- **Stewardship: 48–72h audit, IV→oral switch, PCT-guided, automatic stop orders, de-labelling**

## Common Viva Questions
1. **CLABSI prevention bundle components?** Hand hygiene, maximal barrier, CHG prep, subclavian site, daily review, CHG dressings, hub disinfection.
2. **CAUTI most effective prevention?** Daily catheter necessity review with nurse-driven removal protocol.
3. **VAP prevention — strongest evidence?** Head of bed 30–45° + daily sedation vacation + spontaneous breathing trial.
4. **SSI antibiotic prophylaxis timing?** Cefazolin 2g IV within 60 minutes before incision; discontinue within 24h.
5. **CLABSI — when to remove line?** S. aureus, Candida, Pseudomonas, Bacillus, mycobacteria; CoNS may salvage.
6. **VAP empirical therapy early vs late?** Early (≤5d): ceftriaxone +/− vancomycin; Late (>5d): pip-tazo/meropenem + vancomycin.
7. **SSI prophylaxis duration?** ≤24 hours post-op (no benefit beyond).
7. **VAP duration?** 7 days (short-course non-inferior to 14d).
8. **CLABSI most common pathogen?** Coagulase-negative Staphylococci (CoNS).
9. **Subglottic secretion drainage effect on VAP?** Reduces VAP by ~50%.
10. **SSI glucose target perioperatively?** <180 mg/dL (<10 mmol/L).

## Common Confusions / Exam Traps
| Confusion | Clarification |
|-----------|---------------|
| Routine central line replacement prevents CLABSI | **No benefit; increases infection risk; guidewire exchange NOT recommended** |
| CAUTI prophylaxis at insertion | **NOT routinely recommended** |
| VAP = any pneumonia on ventilator | **Must be ≥48h after intubation + new infiltrate + clinical + microbiological criteria** |
| SSI prophylaxis continue >24h | **No benefit; increases resistance/C. difficile** |
| CLABSI = any BSI with line | **Must be central line ≥48h + NOT secondary to another site** |
| VAP duration 14–21 days | **7 days short-course non-inferior** |
| Prophylactic antibiotics at catheter insertion | **Reduces CAUTI but selects resistance; NOT recommended** |
| CHG dressings for all central lines | **Only for lines expected >7 days** |
| Daily CHG bathing for all ICU | **Reduces CLABSI, MRSA, VRE; but not standard for all** |
| SSI = only superficial | **Deep incisional and organ/space also SSI (90d if implant)** |

## Summary
**Healthcare-associated infections (HAI)** are infections acquired **≥48 hours after hospital admission** (healthcare-associated). **Four major types account for ~80%**: **CLABSI, CAUTI, VAP, SSI**. **CLABSI**: central line ≥48h + laboratory-confirmed BSI not secondary; **pathogens**: CoNS (30–40%), S. aureus (15–20%), Enterococcus (10–15%), Candida (10–15%), Gram-negatives (15–20%); **Prevention bundle**: hand hygiene, maximal barrier precautions, **CHG 2% in 70% alcohol skin prep**, **subclavian site preferred**, daily line necessity review, **CHG-impregnated dressings for lines >7d**, **hub disinfection (CHG/alcohol 15s)**; **Treatment**: empirical **piperacillin-tazobactam/meropenem + vancomycin**; **remove line** for S. aureus, Candida, Pseudomonas, Bacillus, mycobacteria. **CAUTI**: indwelling catheter ≥48h + symptoms + ≥10³ CFU/mL; **pathogens**: E. coli (25–30%), Klebsiella, Pseudomonas, Enterococcus, Candida; **Prevention**: avoid unnecessary catheterisation, aseptic insertion, closed drainage system, securement, **daily necessity review with nurse-driven removal = most effective**, no routine antibiotic prophylaxis; **Management**: **remove catheter if possible**, empirical antibiotics per complicated UTI, **7 days if removed, 10–14 days if retained + change catheter at 48–72h**. **VAP**: invasive ventilation ≥48h + new/progressive infiltrate + clinical signs + microbiological criteria; **early (≤5d)**: MSSA, S. pneumoniae, H. influenzae; **late (>5d)**: MRSA, Pseudomonas, Acinetobacter, ESBL/CRE; **Prevention bundle**: **head of bed 30–45°**, **daily sedation vacation + spontaneous breathing trial**, **subglottic secretion drainage**, **chlorhexidine oral care**, stress ulcer/DVT prophylaxis; **Empirical therapy**: early — ceftriaxone ± vancomycin; late — piperacillin-tazobactam/meropenem + vancomycin; **duration 7 days**. **SSI**: infection at surgical site within 30 days (90 days if implant); **Prophylaxis**: **cefazolin 2g IV within 60 min before incision**, discontinue **≤24 hours post-op**; **glucose <180 mg/dL**, **normothermia >36°C**, FiO₂ 80% intraop + 2h post; **Empirical treatment** by depth and MDRO risk; **source control critical** (drainage, debridement). **Antimicrobial stewardship in HAI**: prospective audit at 48–72h with feedback, IV→oral switch protocols, PCT-guided duration, automatic stop orders, penicillin allergy de-labelling, diagnostic stewardship (no cultures without clinical indication), device removal over routine replacement. **Outbreak investigation**: confirm, define, find, describe, hypothesise, analyse, control, communicate, monitor.

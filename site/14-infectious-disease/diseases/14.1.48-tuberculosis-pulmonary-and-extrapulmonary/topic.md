> [!important]
> **TB = *Mycobacterium tuberculosis* complex.** **Pulmonary TB = most common (80%).** **Diagnosis: GeneXpert MTB/RIF Ultra (rapid, sensitivity ~85–95% smear+, 50–70% smear-), culture gold standard.** **Treatment: 2HRZE/4HR (6 months total) for drug-susceptible.** **MDR-TB: BPaLM (6 months) or 9–12 month regimen.** **TB/HIV: ART timing by CD4.** **Latent TB: treat high-risk.**

## 1. Learning Objectives
- Diagnose pulmonary TB: symptoms, CXR, GeneXpert, culture, LPA, sequencing
- Diagnose extrapulmonary TB: site-specific approaches, GeneXpert on non-respiratory samples
- Initiate standard 6-month regimen (2HRZE/4HR) with correct doses and monitoring
- Manage drug-resistant TB (RR/MDR/XDR): BPaLM, longer regimens, monitoring
- Manage TB/HIV coinfection: ART timing, IRIS, drug interactions
- Diagnose and treat latent TB infection (LTBI)
- Recognise and manage adverse drug reactions (hepatotoxicity, neuropathy, optic neuritis, hyperuricaemia)
- Apply infection control (airborne precautions, UV, ventilation)

## 2. Epidemiology & Transmission
| Aspect | Detail |
|--------|--------|
| **Global burden** | ~10.6M incident cases/year, 1.3M deaths (2022); leading infectious killer |
| **Transmission** | **Airborne** droplet nuclei (1–5μm) from sputum-positive pulmonary TB |
| **Infectivity** | Smear-positive > smear-negative > extrapulmonary (non-pulmonary rarely infectious) |
| **Risk factors** | HIV (20–30×), diabetes (2–3×), malnutrition, silicosis, TNF-α inhibitors, steroids, transplant, CKD, alcohol, smoking, prisoners, healthcare workers |
| **Latent TB (LTBI)** | ~25% world population infected; 5–10% lifetime reactivation risk (↑ with HIV, TNF-α inhibitors) |

## 3. Pathogenesis
1. **Inhalation** → alveolar macrophages
2. **Primary infection** → Ghon focus (subpleural) + hilar lymphadenopathy = **primary complex**
3. **Containment** → caseous granuloma (latent TB) OR **progression** → primary progressive TB
4. **Reactivation** (post-primary) → apical/posterior upper lobe / superior lower lobe (high O₂ tension)
5. **Haematogenous spread** → miliary TB, extrapulmonary sites (meninges, bone, kidney, etc.)

## 4. Clinical Features
### Pulmonary TB
| Symptom | Frequency |
|---------|-----------|
| **Cough >2 weeks** | 90%+ |
| **Fever** (often low-grade, evening) | 70–80% |
| **Night sweats** | 50–70% |
| **Weight loss / anorexia** | 70–90% |
| **Haemoptysis** | 20–30% |
| **Chest pain** (pleuritic) | 20–30% |
| **Dyspnoea** | If extensive disease/effusion |

### Extrapulmonary TB (EPTB) — Common Sites
| Site | Features |
|------|----------|
| **Lymph nodes** (cervical) | Painless, matted, firm, rubbery; may suppurate (cold abscess) |
| **Pleural** | Exudative effusion, lymphocytic, high ADA, often paucibacillary |
| **Meninges** (TBM) | Subacute, basilar exudates, cranial nerves, hydrocephalus — see TBM note |
| **Abdominal** | Ascites (high protein, lymphocytic), peritoneal nodules, ileocaecal mass, strictures |
| **Genitourinary** | Sterile pyuria, renal calculi, strictures, epididymal masses, infertility |
| **Bone/Joint** (Pott's spine) | Thoracic/lumbar spine, gibbus, paraspinal abscess, kyphosis, neuro deficit |
| **Pericardial** | Effusion, constriction, tamponade |
| **Miliary** | Diffuse 1–3mm nodules, high fever, hepatosplenomegaly, pancytopenia, choroidal tubercles |

## 5. Diagnosis
### Pulmonary TB
| Test | Sensitivity | Specificity | Role |
|------|-------------|-------------|------|
| **GeneXpert MTB/RIF Ultra** (sputum) | **85–95% smear+; 50–70% smear-** | >98% | **First-line rapid test**; detects RIF resistance |
| **Smear microscopy (ZN/fluorescence)** | 40–60% (culture+) | High | Rapid, cheap; cannot distinguish MTBC/NTM |
| **Culture (MGIT liquid)** | **Gold standard (100%)** | 100% | DST, species ID; 2–4 weeks |
| **LPA (Line Probe Assay)** | ~95% on culture; ~85% on smear+ sputum | High | Rapid DST (RIF, INH, FQ, SLI) |
| **Whole Genome Sequencing** | Definitive | Definitive | Comprehensive DST, transmission mapping |
| **CXR** | Variable | Low | Screening, supportive; upper lobe cavitation, miliary, effusion |
| **LAM Ag (urine)** | Low (CD4<100) | Moderate | HIV advanced disease only |

### Extrapulmonary TB
| Site | Best Diagnostic Approach |
|------|--------------------------|
| **Lymph node** | FNAC (granulomas, AFB), **GeneXpert on aspirate**, excisional biopsy |
| **Pleural effusion** | **ADA >40 U/L** (exudate, lymphocytic); GeneXpert on fluid (low yield); biopsy |
| **Meninges** | CSF GeneXpert, AFB, culture, ADA — see TBM note |
| **Abdominal** | Ascitic ADA, GeneXpert, laparoscopy/biopsy, CT |
| **Genitourinary** | Urine GeneXpert ×3 (early morning), IVU/CT, biopsy |
| **Bone/Joint** | MRI (gold standard), CT-guided biopsy, GeneXpert on pus |
| **Pericardial** | Echo, pericardial fluid ADA/GeneXpert, biopsy |
| **Miliary** | CXR (miliary pattern), CXR, blood culture, liver biopsy, choroidal tubercles |

> [!tip]
> **GeneXpert on non-respiratory samples: sensitivity lower than sputum but still valuable.** **Always send for culture + DST.** **ADA supportive in lymphocytic exudates (pleural, pericardial, ascitic).**

## 6. Standard Treatment — Drug-Susceptible TB (6 Months)
### Regimen
| Phase | Duration | Drugs | Doses (Daily) |
|-------|----------|-------|---------------|
| **Intensive** | **2 months** | **HRZE** | **H 5mg/kg (4–6), R 10mg/kg (8–12), Z 25mg/kg (20–30), E 15mg/kg (15–20)** |
| **Continuation** | **4 months** | **HR** | Same H, R doses |

> [!important]
> **Total 6 months (2HRZE/4HR).** **Daily dosing preferred.** **Intermittent (3×/week) ONLY in continuation phase if DOT assured.** **Pyridoxine 25–50mg/day with INH.** **Ethambutol: baseline visual acuity + colour vision; monthly monitoring.**

### Drug Doses (Daily, Weight-Based)
| Drug | Dose (mg/kg) | Max Dose | Key Toxicity |
|------|--------------|----------|--------------|
| **Isoniazid (H)** | 5 (4–6) | 300mg | **Peripheral neuropathy, hepatotoxicity** |
| **Rifampicin (R)** | 10 (8–12) | 600mg | **Hepatotoxicity, orange secretions, drug interactions (CYP450 induction)** |
| **Pyrazinamide (Z)** | 25 (20–30) | 2g | **Hepatotoxicity, hyperuricaemia/gout, GI upset** |
| **Ethambutol (E)** | 15 (15–20) | 1.2g | **Optic neuritis (colour vision, acuity)** |
| **Pyridoxine** | — | 25–50mg/day | Prevents INH neuropathy |

### Monitoring
| Parameter | Schedule |
|-----------|----------|
| **Sputum smear/culture** | End of intensive (2m), end of treatment (6m); if +ve at 2m → extend intensive to 3m |
| **LFTs (ALT, AST, bilirubin)** | Baseline, 2w, 4w, 8w, then monthly (more if abnormal) |
| **Visual acuity + colour vision (E)** | Baseline, monthly |
| **Uric acid (Z)** | Baseline, if symptomatic |
| **Neuropathy screen (H)** | Each visit |
| **Weight** | Monthly |

## 7. Treatment Response Criteria
| Timepoint | Expected Response | Action if Not Met |
|-----------|-------------------|-------------------|
| **2 months** | Smear/culture negative | Extend intensive phase to 3 months; review adherence, DST |
| **5 months** | Culture negative | Extend continuation phase to 7 months (total 9m) |
| **6 months** | Clinical/radiological cure | Declare treatment success |

> [!key]
> **Sputum conversion at 2 months = best predictor of cure.** **Failure = smear/culture +ve at 5m or later.**

## 8. Drug-Resistant TB
### Definitions
| Type | Definition |
|------|------------|
| **RR-TB** | Rifampicin-resistant (with or without other resistance) |
| **MDR-TB** | Resistant to **Rifampicin + Isoniazid** |
| **Pre-XDR-TB** | MDR + resistant to **any fluoroquinolone** |
| **XDR-TB** | MDR + resistant to **fluoroquinolone + at least one Group A drug** (bedaquiline, linezolid) |

### WHO 2022 Regimens (Preferred)
| Regimen | Composition | Duration | Indication |
|---------|-------------|----------|------------|
| **BPaLM** | **Bedaquiline + Pretomanid + Linezolid + Moxifloxacin** | **6 months** | **Preferred for RR/MDR-TB (no FQ resistance)** |
| **BPaL** | Bedaquiline + Pretomanid + Linezolid | 6 months | MDR-TB + FQ resistance (or intolerance) |
| **9–12 month all-oral** | Bedaquiline + Linezolid + Levofloxacin/Moxifloxacin + Clofazimine + Cycloserine/Terizidone + Pyrazinamide + Ethambutol | 9–12 months | If BPaLM contraindicated |

> [!important]
> **BPaLM = 6 months, all-oral, high success (~90%).** **Linezolid 600mg/day (reduce to 300mg if toxicity); monitor CBC, neuropathy, optic neuritis.** **Bedaquiline: ECG monitoring (QTc).** **Pretomanid: teratogenic (contraception mandatory).**

## 9. TB / HIV Co-infection
| CD4 Count | ART Timing |
|-----------|------------|
| **<50 cells/μL** | **Within 2 weeks** of TB treatment start |
| **50–100 cells/μL** | **Within 2–8 weeks** |
| **>100 cells/μL** | **Within 8 weeks** (can delay up to 12w if logistical) |

> [!important]
> **IRIS risk: highest with low CD4, early ART.** **Paradoxical TB-IRIS:** worsening TB symptoms/signs 2–12w post-ART; **Prednisolone 1.5mg/kg/day taper 2w** if moderate-severe.
> **Rifampicin + ART interactions:** **Dolutegravir 50mg BD (double dose)**; Efavirenz 600mg OK; **Avoid PIs** (rifampicin reduces levels); Nevirapine avoid.

### TB/HIV Key Points
- **TB treatment duration SAME (6 months)** for drug-susceptible TB
- **Cotrimoxazole prophylaxis** if CD4<350 or WHO stage 3/4
- **Pyridoxine mandatory** (INH neuropathy ↑ with HIV)
- **GeneXpert sensitivity higher in HIV** (higher bacillary load)

## 10. Latent TB Infection (LTBI)
### Who to Test & Treat
| High-Risk Groups | Test | Treat if Positive |
|------------------|------|-------------------|
| HIV+ | IGRA/TST | **Yes (regardless of BCG)** |
| TNF-α inhibitor candidates | IGRA/TST | **Yes** |
| Transplant candidates/recipients | IGRA/TST | **Yes** |
| Silicosis | IGRA/TST | **Yes** |
| Close contacts of smear+ TB | IGRA/TST | **Yes** (especially <5y, HIV+) |
| Healthcare workers (high TB burden) | Serial IGRA/TST | Consider |
| Prisoners, homeless, IVDU | IGRA/TST | Consider |

### LTBI Regimens
| Regimen | Duration | Notes |
|---------|----------|-------|
| **3HP** (Isoniazid 15mg/kg + Rifapentine 15mg/kg) **weekly ×12 weeks** | 3 months | **Preferred (high completion, less hepatotoxicity)**; DOT |
| **3HR** (Isoniazid + Rifampicin daily) | 3 months | Alternative |
| **4R** (Rifampicin daily) | 4 months | Preferred if INH contraindicated |
| **6H/9H** (Isoniazid daily) | 6/9 months | Older regimens; 9H if high risk |

> [!tip]
> **Exclude active TB BEFORE LTBI treatment** (symptom screen, CXR). **IGRA preferred over TST if BCG vaccinated.**

## 11. Adverse Drug Reactions — Major
| Drug | Reaction | Management |
|------|----------|------------|
| **Isoniazid** | Peripheral neuropathy | **Pyridoxine 25–50mg/day prophylaxis**; if established → pyridoxine 100mg/day |
| | Hepatotoxicity | See hepatotoxicity algorithm |
| | Seizures (rare) | Pyridoxine IV |
| **Rifampicin** | Hepatotoxicity | See hepatotoxicity algorithm |
| | Drug interactions (CYP450 induction) | Adjust co-medications (ART, anticoagulants, contraceptives, etc.) |
| | Flu-like syndrome, thrombocytopenia | Usually transient |
| **Pyrazinamide** | Hepatotoxicity | See hepatotoxicity algorithm |
| | Hyperuricaemia / gout | Allopurinol if symptomatic; usually asymptomatic |
| | GI intolerance | Take with food |
| **Ethambutol** | Optic neuritis (reversible if caught early) | **Baseline + monthly acuity/colour vision; STOP if deterioration** |
| **Bedaquiline** | QTc prolongation | ECG baseline, 2w, 4w, 8w, 12w; avoid with other QTc drugs |
| **Linezolid** | Myelosuppression (anaemia, thrombocytopenia) | CBC weekly ×4, then 2w; hold if Hb<8, plt<50 |
| | Peripheral/optic neuropathy | Monitor; reduce to 300mg/day if toxicity |
| **Clofazimine** | Skin discolouration (pink-brown), GI intolerance | Reversible; counseling |

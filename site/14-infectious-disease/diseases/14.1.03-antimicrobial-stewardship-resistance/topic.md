## 2. 📖 Antimicrobial Stewardship (AMS) — Core Interventions

### The "5 Core" AMS Interventions (CDC/WHO)

| Intervention | Description | Evidence of Impact |
|--------------|-------------|-------------------|
| **1. Prospective Audit & Feedback (PAF)** | ID Pharmacist/Physician reviews prescriptions → Real-time recommendations | ↓ ABX Use 20-30%, ↓ Resistance, ↓ Cost |
| **2. Formulary Restriction & Pre-Authorization** | Restricted Agents Require ID Approval (Carbapenems, Colistin, Newer Agents) | ↓ Inappropriate Use, ↓ Resistance Selection |
| **3. IV-to-Oral (IV→PO) Switch Protocol** | Automatic Switch Criteria at 48-72h (Clinical Stability, GI Function, Bioavailability >90%) | ↓ LOS, Cost, IV Complications, Line Days |
| **4. Automatic Stop Orders / Default Durations** | Default Duration (e.g., 5d CAP, 7d UTI, 4d IA) | Prevents Prolonged Unnecessary Therapy |
| **5. Prospective De-escalation Protocol** | Mandatory Review at 48-72h → Narrow/Stop/Stop Based on Cultures/Clinical | ↓ Broad-Spectrum Exposure, ↑ Targeted Therapy |

### Additional High-Impact Interventions

| Intervention | Description |
|--------------|-------------|
| **Rapid Diagnostics Integration** | MALDI-TOF, PCR, Syndromic Panels, T2Candida, Rapid AST → **Early Targeted Therapy** |
| **Handshake Stewardship** | Real-Time Bedside Review by ID Pharmacist/Physician (Non-Punitive) |
| **Dose Optimisation (PK/PD)** | Extended Infusion β-Lactams, AUC/MIC Vancomycin, Once-Daily Aminoglycosides |
| **Allergy De-labeling** | Structured Assessment → **>90% Penicillin Allergy Labels Removed** |
| **Outpatient Parenteral Antimicrobial Therapy (OPAT)** | Safe IV→Home Transition for Stable Patients |
| **Surgical Prophylaxis Optimisation** | **Right Drug, Right Dose, Right Timing (<60min), Right Duration (<24h)** |

## 3. 🦠 Resistance Mechanisms — Key Classes

### 1. β-Lactamases

| Class | Enzymes | Substrate Profile | Inhibitor Sensitivity | Clinical Significance |
|-------|---------|-------------------|----------------------|----------------------|
| **Class A (TEM, SHV, CTX-M, KPC)** | ESBL, KPC | Penicillins, Cephalosporins (3rd/4th Gen), Aztreonam; **KPC: Carbapenems** | **Clavulanate/Tazobactam** (ESBL), **Avibactam/Vaborbactam/Relebactam** (KPC) | **ESBL: 3rd Gen Ceph Resistant**; **KPC: Carbapenemase** |
| **Class B (Metallo-β-Lactamases: NDM, VIM, IMP)** | MBL | **All β-Lactams (Incl. Carbapenems)**, NOT Aztreonam | **EDTA (Chelator)**, **NOT Clavulanate** | **NDM-1 = Global Spread**, Carbapenem Resistance |
| **Class C (AmpC)** | Chromosomal (Inducible) / Plasmid | Cephalosporins (3rd/4th Gen), NOT Carbapenems | **Not Inhibited by Clavulanate** | **Inducible (Enterobacter, Citrobacter, Serratia)** |
| **Class D (OXA)** | OXA-48, OXA-23, OXA-58 | Carbapenems (Weak), Variable Cephalosporins | Variable | **OXA-48 = Carbapenemase (Enterobacterales)**, **OXA-23/58 = Acinetobacter** |

### 2. Other Resistance Mechanisms

| Mechanism | Examples | Affected Drugs |
|----------|----------|----------------|
| **Target Modification** | **mecA (PBP2a)** → MRSA; **gyrA/parC Mutations** → FQR; **rpoB Mutations** → Rifampicin Resistance | β-Lactams (MRSA), Fluoroquinolones, Rifampicin |
| **Efflux Pumps** | **AcrAB-TolC (Enterobacterales)**, **MexAB-OprM (Pseudomonas)**, **NorA (Staph)** | Multi-Drug (FQ, Tetracyclines, Macrolides, β-Lactams) |
| **Porin Loss** | **OmpK35/OmpK36 Loss (Klebsiella)**, **OprD Loss (Pseudomonas)** | Carbapenems, β-Lactams |
| **16S rRNA Methylases** | **ArmA, RmtA–RmtG** | **High-Level Aminoglycoside Resistance** |
| **Ribosomal Protection** | **Tet(M), Tet(O)** | **Tetracyclines** |

## 4. 🦠 Key MDROs — Definitions & Management

### Major MDROs

| Organism | Resistance Definition | Key Treatment Options | Infection Control |
|----------|----------------------|----------------------|-------------------|
| **MRSA** | **mecA/mecC** → PBP2a | **Vancomycin (AUC 400-600)**, Daptomycin, Linezolid, Ceftaroline, Ceftriaxone | Contact Precautions, Decolonisation (Mupirocin + Chlorhexidine) |
| **VRE** | **vanA/vanB** (Vancomycin Resistance) | **Linezolid, Daptomycin, Ceftaroline**, Tedizolid, Fidaxomicin (GI) | Contact Precautions, Cohorting |
| **ESBL** | **CTX-M** (Most Common), SHV, TEM | **Carbapenems (Meropenem/Ertapenem)** 1st Line | Contact Precautions, Cohorting |
| **CRE (KPC, NDM, OXA-48)** | **Carbapenemase Production** | **Ceftazidime-Avibactam, Meropenem-Vaborbactam, Imipenem-Relebactam**, Cefiderocol, Polymyxins, Tigecycline, Aminoglycosides, Fosfomycin | Strict Isolation, Cohorting, Dedicated Equipment |
| **CRAB (A. baumannii)** | **OXA-23/24/58, NDM** | **Sulbactam-Durlobactam**, **Polymyxins**, **Tigecycline**, **Minocycline**, **Cefiderocol** | Contact Precautions, Environmental Cleaning |
| **CRPA (P. aeruginosa)** | **Efflux + Porin Loss + Mutations** | **Ceftazidime-Avibactam, Ceftolozane-Tazobactam, Cefiderocol, Colistin, Fosfomycin** | Contact Precautions |
| **C. auris** | **Multidrug-Resistant Yeast** | **Echinocandins (1st Line)**, Liposomal Amphotericin B | **Contact Precautions**, **Environmental Disinfection (Bleach/UV)** |

> [!warning] **CRE = Carbapenemase Production (KPC, NDM, OXA-48, VIM, IMP)**. **CRAB = Carbapenem-Resistant A. baumannii**. **CRPA = Carbapenem-Resistant P. aeruginosa**.

## 5. 🔬 Surveillance & Metrics

### Key Surveillance Systems

| System | Scope | Key Metrics |
|--------|-------|-------------|
| **NHSN (CDC, USA)** | National HAI + AMR | **SIR (Standardized Infection Ratio)**, **AUR (Antimicrobial Use & Resistance)** |
| **EARS-Net (ECDC, Europe)** | European AMR Surveillance | **% Resistance by Country/Organism**, Trends |
| **GLASS (WHO Global)** | Global AMR Surveillance | **Priority Pathogens**, **National Action Plans** |
| **Local Hospital** | IPC + ASP Dashboard | **HAI Rates, SIR, DDD/1000 Patient-Days, % Compliance** |

### Key Metrics

| Metric | Definition | Target |
|--------|------------|--------|
| **SIR (Standardized Infection Ratio)** | Observed / Predicted HAI (Risk-Adjusted) | **<1.0** |
| **DDD/1000 Patient-Days** | **Defined Daily Doses** per 1000 Bed-Days | **↓ Trend** |
| **Days of Therapy (DOT)** | **Days of Antibiotic Therapy / 1000 Patient-Days** | **↓ Trend** |
| **% Compliance** (Bundle/Stewardship) | % Adherence to Protocols | **>90%** |
| **Time to Targeted Therapy** | Time from Culture to Targeted Rx | **<48-72h** |

## 6. 🎯 De-escalation & IV→PO Switch — Practical Protocols

### De-escalation Algorithm (48-72h)

```mermaid
flowchart TD
    A[48-72h Post-Empirical Therapy] --> B{**Culture Results Available?**}
    B -->|Yes| C{**Pathogen Identified?**}
    C -->|Yes| D[**Narrow to Narrowest Effective Agent**]
    C -->|No| E[**Reassess Diagnosis**, Consider Stop if Low Suspicion]
    D --> F{**Clinical Improvement?**}
    F -->|Yes| G[**IV→PO Switch Criteria Met?**]
    G -->|Yes| H[**Switch to Oral** (High Bioavailability Agent)]
    G -->|No| I[**Continue IV**, Reassess Daily]
    F -->|No| J[**Reassess Diagnosis/Source Control**]
```

### IV→PO Switch Criteria

| Criteria | Examples |
|----------|----------|
| **Clinical Improvement** | Afebrile 24-48h, ↓WBC/CRP, Haemodynamic Stability |
| **GI Function** | Tolerating Oral/NG, No Obstruction/Ileus |
| **Bioavailability >90%** | **Fluoroquinolones, Linezolid, Metronidazole, Doxycycline, Cotrimoxazole, Clindamycin** |
| **Bioavailability 50-90%** | **Amoxicillin, Co-amoxiclav, Cephalexin, Metronidazole** |
| **Avoid IV→PO** | **β-Lactams (Low Bioavailability), Glycopeptides, Aminoglycosides, Carbapenems** |

## 7. 🛡️ Infection Prevention & Control (IPC) for MDROs

| Measure | Application |
|---------|-------------|
| **Contact Precautions** | **Gown + Gloves** for All MDRO Contacts; **Single Room / Cohorting** |
| **Hand Hygiene** | **5 Moments**, **ABHR at Point of Care**, **Audit & Feedback** |
| **Environmental Cleaning** | **Daily + Terminal Cleaning**, **Bleach/Peracetic Acid for C. difficile/C. auris**, **UV/HPV for Terminal** |
| **Screening/Decolonisation** | **MRSA: Nares/Throat/Perineum Swabs → Mupirocin + Chlorhexidine**; **CRE: Rectal/Perineal Swabs** |
| **Antimicrobial Stewardship** | **Formulary Restriction**, **Audit & Feedback**, **IV→PO**, **Duration Limits** |
| **Surveillance** | **Active Surveillance Cultures** (Admission/Weekly ICU), **Environmental Cultures** |

## 9. ❓ Viva Questions

1. **What are the 5 core antimicrobial stewardship interventions?**
   - **Prospective Audit & Feedback, Formulary Restriction, IV→PO Switch, Automatic Stop Orders, De-escalation Protocol**.

2. **What are the main resistance mechanisms for β-Lactam antibiotics?**
   - **β-Lactamases (Class A: ESBL/KPC, B: MBL/NDM, C: AmpC, D: OXA)**, **Target Modification (mecA), Efflux, Porin Loss**.

3. **What are the key differences between ESBL, AmpC, and Carbapenemase producers?**
   - **ESBL: Inhibited by Clavulanate, 3rd/4th Gen Ceph Resistant**; **AmpC: Not Inhibited by Clavulanate, Inducible**; **Carbapenemase: Carbapenem Resistant (KPC, NDM, OXA-48)**.

4. **What is the first-line treatment for CRE (KPC-producing Enterobacterales)?**
   - **Ceftazidime-Avibactam, Meropenem-Vaborbactam, Imipenem-Relebactam, Cefiderocol** (Preferred over Polymyxins/Tigecycline).

5. **What is the definition of CRE?**
   - **Carbapenem-Resistant Enterobacterales** — **Carbapenemase Production (KPC, NDM, OXA-48, VIM, IMP)** OR **Non-Carbapenemase Mechanisms with Reduced Susceptibility**.

5. **What is the difference between ESBL and AmpC?**
   - **ESBL: Clavulanate-Inhibited, Plasmid-Mediated Usually**; **AmpC: Clavulanate-Resistant, Often Chromosomal/Inducible (Enterobacter, Citrobacter, Serratia)**.

6. **What are the 5 core AMS interventions?**
   - **Prospective Audit & Feedback, Formulary Restriction, IV→PO Switch, Automatic Stop Orders, De-escalation Protocol**.

6. **What is the definition of CRE?**
   - **Carbapenem-Resistant Enterobacterales** with **Carbapenemase Production (KPC, NDM, OXA-48, VIM, IMP)**.

7. **What is the first-line treatment for CRAB?**
   - **Sulbactam-Durlobactam** (New), **Polymyxin B/Colistin + Combination**, **Cefiderocol**, **Tigecycline/Minocycline**.

7. **What is the role of rapid diagnostics in antimicrobial stewardship?**
   - **MALDI-TOF (ID in Minutes), Syndromic PCR (Hours), T2Candida (3-5h), Rapid AST** → **Early Targeted Therapy, ↓ Broad-Spectrum Days**.

8. **How do you calculate SIR (Standardized Infection Ratio)?**
   - **SIR = Observed Infections / Predicted Infections** (Risk-Adjusted); **SIR <1 = Better Than Expected**.

9. **What are the key infection control measures for C. auris?**
   - **Contact Precautions (Gown/Gloves), Single Room/Cohort, Bleach/UV Disinfection, Screening Contacts, Echinocandins 1st Line**.

10. **What is the difference between ESBL and AmpC β-lactamases?**
    - **ESBL: Inhibited by Clavulanate, Usually Plasmid, Hydrolyses 3rd/4th Gen Ceph**; **AmpC: NOT Inhibited by Clavulanate, Often Chromosomal/Inducible, Hydrolyses Cephamycins**.

## 10. 🧠 Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **ESBL vs AmpC vs Carbapenemase** | **ESBL: Clavulanate-Inhibited, Ceph-R**; **AmpC: Clavulanate-Resistant, Inducible**; **Carbapenemase: Carbapenem-R (KPC, NDM, OXA)** |
| **CRE vs CRAB vs CRPA** | **CRE = Enterobacterales (KPC/NDM/OXA-48)**; **CRAB = Acinetobacter (OXA-23/58, NDM)**; **CRPA = Pseudomonas (Efflux+Porin+Mutation)** |
| **MRSA vs VRE** | **MRSA = mecA → PBP2a (β-Lactam R)**; **VRE = vanA/vanB → Vancomycin R** |
| **SIR vs Standardised Rate** | **SIR = Observed/Predicted (Risk-Adjusted)**; **Rate = Events/Time** |
| **De-escalation vs Switch** | **De-escalation = Narrow Spectrum**; **Switch = IV→PO (Same Drug Class)** |

| Mnemonic | Meaning |
|----------|---------|
| **"5 Core AMS = PAF, Formulary, IV→PO, Stop Orders, De-escalation"** | Core Interventions |
| **"ESBL = CTX-M, Clavulanate Inhibits"** | ESBL |
| **"AMPc = Not Inhibited by Clavulanate"** | AmpC |
| **"CRE = KPC/NDM/OXA-48"** | Carbapenemase Types |
| **"SIR = Observed/Predicted"** | Surveillance |
| **"IV→PO = Bioavailability >90%"** | Switch Criteria |
| **"CRE = Ceftazidime-Avibactam / Cefiderocol / Polymyxin"** | CRE Treatment |

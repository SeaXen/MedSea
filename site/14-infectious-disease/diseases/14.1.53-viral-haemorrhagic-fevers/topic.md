> [!important]
> **VHF = group of viral infections causing fever + bleeding diathesis + multi-organ failure.** **Ebola, Marburg, Lassa, CCHF, Dengue (severe), Yellow Fever.** **High case fatality (25-90%).** **Transmission: contact with infected body fluids (Ebola, Marburg), aerosols (Lassa), vectors (CCHF, Dengue, Yellow Fever).** **ISOLATE IMMEDIATELY on suspicion. PPE mandatory. Public health notification.**

## 1. Learning Objectives
- Classify VHF by virus family, vector, geography
- Recognise clinical syndrome: fever + bleeding + multi-organ involvement
- Apply immediate isolation and infection control measures
- Initiate supportive management and specific antivirals where available
- Apply contact tracing and infection control measures
- Recognise when to suspect VHF in returned traveller

## 3. Clinical Syndrome
| Feature | Typical Presentation |
|---------|----------------------|
| **Incubation** | 2–21 days (Ebola 2–21; Lassa 6–21; CCHF 1–13; Dengue 4–10) |
| **Prodrome** | Fever, myalgia, headache, fatigue, anorexia, nausea/vomiting |
| **Haemorrhagic Manifestations** | Petechiae, ecchymoses, mucosal bleeding (gums, nose), GI bleeding, haematuria, vaginal bleeding, injection site bleeding |
| **Organ Involvement** | Hepatitis (↑ AST/ALT), renal failure, encephalopathy, myocarditis, pancreatitis, ARDS, DIC, shock |
| **Laboratory** | Thrombocytopenia, leukopenia (Ebola, Lassa), ↑ AST/ALT, ↑ creatinine, prolonged PT/aPTT, fibrinogen ↓, D-dimer ↑ |

## 4. Key Viruses — Clinical Pearls

### Ebola Virus Disease (EVD)
| Aspect | Details |
|--------|---------|
| **Transmission** | Direct contact with blood/body fluids of symptomatic patient; fomites; bushmeat; sexual (persistent in semen) |
| **Clinical** | Sudden fever, severe headache, myalgia → vomiting, diarrhoea, abdominal pain → haemorrhage (day 5–7), multi-organ failure |
| **CFR** | **50–90%** (Zaire ebolavirus); Sudan ebolavirus ~50% |
| **Treatment** | **Inmazeb (atoltivimab/maftivimab/odesivimab) OR Ebanga (ansuvimab)** — monoclonal antibodies; supportive care |
| **Vaccine** | **rVSV-ZEBOV (Ervebo)** — single dose, ring vaccination |

### Marburg Virus Disease (MVD)
| Aspect | Details |
|--------|---------|
| **Transmission** | Similar to Ebola; bat exposure (caves/mines) |
| **Clinical** | Indistinguishable from Ebola; severe MACULOPAPULAR RASH (day 5) |
| **CFR** | **24–88%** |
| **Treatment** | Supportive; no licenced antiviral/vaccine (experimental monoclonal) |

### Lassa Fever
| Aspect | Details |
|--------|---------|
| **Transmission** | **Aerosol/excreta of Mastomys rats**; person-to-person (body fluids); nosocomial |
| **Clinical** | Insidious onset; fever, pharyngitis, retrosternal pain, **facial oedema**, pleural effusions, **sensorineural hearing loss** (common sequela), encephalopathy |
| **CFR** | **1–15%** (hospitalised); higher in pregnancy (3rd trimester) |
| **Treatment** | **Ribavirin IV** (early = better outcome); supportive; convalescent plasma (experimental) |
| **Vaccine** | None licenced (candidates in development) |

### Crimean-Congo Haemorrhagic Fever (CCHF)
| Aspect | Details |
|--------|---------|
| **Transmission** | **Hyalomma tick bite**; contact with infected livestock blood/tissues; nosocomial |
| **Clinical** | Sudden fever, myalgia, headache, dizziness → haemorrhage (day 3–5), hepatomegaly, confusion, multiorgan failure |
| **CFR** | **10–40%** |
| **Treatment** | **Ribavirin IV** (early = better outcome); supportive |
| **Vaccine** | None licenced (inactivated vaccine in limited use) |

### Dengue Haemorrhagic Fever / Dengue Shock Syndrome (DHF/DSS)
| Aspect | Details |
|--------|---------|
| **Transmission** | **Aedes aegypti/albopictus** mosquito |
| **Clinical** | **DHF**: fever, thrombocytopenia (<100k), haemorrhagic manifestations, plasma leakage (pleural effusion, ascites); **DSS**: shock, narrow pulse pressure |
| **CFR** | **<1%** with proper fluid management; **>20%** untreated |
| **Treatment** | **Fluid management** (crystalloids, colloids; avoid overload); no antiviral |
| **Vaccine** | **Dengvaxia** — ONLY for seropositive individuals (9–45y) |

## 5. Diagnosis
| Test | Utility |
|------|---------|
| **RT-PCR (blood)** | **Gold standard**; rapid (hours); available for Ebola, Marburg, Lassa, CCHF, Dengue, YF |
| **IgM/IgG ELISA** | Acute/convalescent; Lassa, CCHF, Dengue, YF |
| **Antigen Detection (Rapid Test)** | Dengue NS1; Ebola VP40 (field use) |
| **Virus Isolation** | BSL-4; definitive; not for acute management |
| **Routine Labs** | Thrombocytopenia, ↑ AST/ALT, prolonged PT, DIC screen |

> [!critical]
> **VHF SUSPICION → IMMEDIATE ISOLATION + PPE + PUBLIC HEALTH NOTIFICATION.** **Do NOT send samples routine lab — use dedicated VHF pathway.**

## 6. Management
### Immediate (Hour 0)
| Action | Details |
|--------|---------|
| **1. ISOLATE** | **Negative pressure room**; **PPE**: double gloves, impermeable gown, N95/FFP3, face shield, hood, boots; **trained buddy system** for donning/doffing |
| **2. NOTIFY** | **Public health authorities** immediately (statutory notification); hospital IPC team |
| **3. SUPPORTIVE** | **Fluid resuscitation** (crystalloids, avoid overload), **blood products** (platelets, FFP, cryoprecipitate for DIC), **vasopressors**, **renal replacement**, **mechanical ventilation** as needed |
| **4. SPECIFIC ANTIVIRALS** | **Ebola**: Inmazeb (atoltivimab/maftivimab/odesivimab) OR Ebanga (ansuvimab); **Lassa/CCHF**: **Ribavirin IV** (30mg/kg load → 16mg/kg q6h ×4d → 8mg/kg q8h ×6d); **Dengue**: Fluid management only; **Marburg**: Supportive only |

### Infection Control (IPC)
| Measure | Standard |
|---------|----------|
| **Isolation** | Negative pressure room; anteroom; dedicated equipment |
| **PPE** | **Double gloves**; impermeable gown; N95/FFP3; face shield; hood; boots; **buddy system** |
| **Waste** | Category A infectious; autoclave/incinerate |
| **Decontamination** | Hypochlorite 10,000ppm (1%) for surfaces; 0.5% chlorine for skin |
| **Contact Tracing** | **21 days** (Ebola/Marburg); **17 days** (Lassa); **14 days** (CCHF) |
| **Dead Body Management** | Trained team; minimal handling; sealed body bag; cremation/deep burial |

### Post-Exposure Prophylaxis (PEP)
| Virus | PEP |
|-------|-----|
| **Ebola** | **Monoclonal antibodies (Inmazeb/Ebanga) ± rVSV-ZEBOV vaccine** (ring vaccination) |
| **Lassa/CCHF** | **Ribavirin** (controversial; limited evidence) |
| **Dengue** | None (no PEPreventative vaccine for seronegative) |
| **Marburg** | None licenced |

## 7. VHF in Returned Traveller
| Scenario | Action |
|----------|--------|
| **Fever + travel to VHF-endemic area + bleeding/contact history** | **ISOLATE IMMEDIATELY**; notify IPC/public health; send blood for **RT-PCR (Ebola, Marburg, Lassa, CCHF, Dengue, YF)** |
| **HCW needlestick from VHF patient** | **Immediate washing**; baseline labs; **PEP per virus** (Ebola: vaccine ± mAb; Lassa/CCHF: ribavirin controversial); monitor 21 days |

## 8. FCPS/MRCP High-Yield Points
- **VHF = fever + bleeding + multi-organ failure; high CFR**
- **Ebola/Marburg = Filoviridae; contact fluids; CFR 50-90%; monoclonal antibodies (Inmazeb/Ebanga); rVSV-ZEBOV vaccine**
- **Lassa = Arenaviridae; Mastomys rat; aerosol; facial oedema, hearing loss; Ribavirin IV early**
- **CCHF = Nairoviridae; Hyalomma tick; livestock; Ribavirin IV early**
- **Dengue DHF/DSS = plasma leakage, thrombocytopenia; fluid management; Dengvaxia seropositive only**
- **ISOLATE IMMEDIATELY on suspicion; PPE; negative pressure; public health notification**
- **RT-PCR = gold standard; send via dedicated VHF pathway**
- **Ribavirin for Lassa/CCHF (early = better)**
- **Ebola: Inmazeb/Ebanga monoclonals; rVSV-ZEBOV vaccine**
- **No person-to-person for CCHF except nosocomial; Ebola/Marburg = body fluids; Lassa = aerosol/fluid**

## 9. Common Viva Questions
1. **First action on suspecting VHF?** Isolate immediately (negative pressure), PPE, notify public health.
2. **Which VHF has specific antiviral?** Lassa & CCHF = Ribavirin; Ebola = monoclonal antibodies (Inmazeb/Ebanga).
3. **Lassa fever characteristic features?** Facial oedema, pharyngitis, retrosternal pain, sensorineural hearing loss.
4. **CCHF transmission?** Hyalomma tick bite; livestock contact; nosocomial.
5. **Ebola treatment?** Inmazeb or Ebanga (monoclonal antibodies); supportive care; rVSV-ZEBOV vaccine for prevention.
6. **Dengue DHF vs DSS?** DHF = plasma leakage + haemorrhage; DSS = shock (narrow pulse pressure).
7. **First action on VHF suspicion?** ISOLATE (negative pressure), PPE, notify public health.
8. **Ribavirin use?** Lassa fever & CCHF (early = better outcome).
9. **Ebola vaccine?** rVSV-ZEBOV (Ervebo) single dose.
10. **VHF in pregnancy?** Lassa severe in 3rd trimester; Ebola high mortality; Ribavirin teratogenic (avoid in pregnancy).

## 10. Common Confusions / Exam Traps
| Confusion | Clarification |
|-----------|---------------|
| Ribavirin for all VHF | **Only Lassa & CCHF** (early); **NOT for Ebola/Marburg/Dengue** |
| Ebola = airborne | **Contact with body fluids**; NOT airborne (except lab aerosols) |
| Dengue vaccine for all | **Dengvaxia ONLY seropositive** (9-45y); seronegative = ↑ severe dengue |
| Ribavirin safe in pregnancy | **Teratogenic (Category X)** — avoid in pregnancy; Lassa in pregnancy = high mortality |
| Ebola = always haemorrhage | Only 50% develop overt bleeding; fever + GI symptoms common presentation |
| Marburg = same as Ebola | Clinically similar but distinct virus; maculopapular rash more prominent in Marburg |
| CCHF = only Africa | Also Middle East, Asia, Eastern Europe |
| Lassa = only rural | Urban transmission occurs (rat infestation); nosocomial outbreaks common |
| Vaccine for all VHF | Only Ebola (rVSV-ZEBOV), YF, Dengue (seropositive only) have licenced vaccines |
| VHF = always fatal | Supportive care + early specific therapy ↓ mortality significantly |

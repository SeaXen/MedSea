# Febrile Neutropenia (Neutropenic Sepsis)

Related: [[Sepsis and Septic Shock]], [[Central Line-Associated Bloodstream Infection]], [[Invasive Fungal Infections]], [[Antimicrobial Stewardship]], [[Bone Marrow Transplant Infections]]

> [!important]
> **Febrile Neutropenia (FN) = ONCOLOGIC EMERGENCY.** **Definition: ANC <0.5 ×10⁹/L (or <1.0 with predicted decline to <0.5) + Temperature ≥38.3°C (single) OR ≥38.0°C sustained ≥1h.** **Immediate IV antibiotics WITHIN 1 HOUR (piperacillin-tazobactam or meropenem or cefepime).** **Risk stratification: MASCC score (outpatient vs inpatient).** **Gram-neg coverage MANDATORY (Pseudomonas).** **Glycopeptide NOT routine initially.** **Duration: until ANC >0.5 ×10⁹/L + afebrile ≥48h (minimum 5–7d).** **Persistent fever: reassess at 48–72h, 4–7d, >7d (add antifungals).**

## Learning Objectives
- Define febrile neutropenia using ANC and temperature criteria
- Apply MASCC score for risk stratification (outpatient vs inpatient)
- Initiate empirical IV antibiotics WITHIN 1 HOUR
- Select appropriate empirical regimen (anti-pseudomonal β-lactam monotherapy preferred)
- Know when to add glycopeptide (vancomycin) and antifungal therapy
- Manage persistent fever (48–72h, 4–7d, >7d)
- Determine treatment duration and discontinuation criteria
- Implement infection prevention (prophylaxis, growth factors, central line care)

## Definition & Diagnosis
| Criterion | Threshold |
|-----------|-----------|
| **Absolute Neutrophil Count (ANC)** | **<0.5 ×10⁹/L** (or <1.0 ×10⁹/L with predicted decline to <0.5) |
| **Temperature** | **Single ≥38.3°C (101°F) OR Sustained ≥38.0°C (100.4°F) for ≥1 hour** |
| **Profound neutropenia** | ANC <0.1 ×10⁹/L |
| **Prolonged neutropenia** | Duration >7 days expected |

> [!critical]
> **FN = MEDICAL EMERGENCY.** **Antibiotics WITHIN 1 HOUR of presentation.** **Delay = ↑ mortality.** **Do NOT wait for cultures, imaging, or ANC confirmation if clinical suspicion.**

## Risk Stratification — MASCC Score
**Multinational Association for Supportive Care in Cancer (MASCC) Score** — identifies low-risk patients for outpatient/oral management

| Factor | Points |
|--------|--------|
| **Burden of illness** (no/minor symptoms = 5; moderate = 3; severe = 0) | 0–5 |
| **No hypotension** (SBP >90 mmHg) | 5 |
| **No COPD** | 4 |
| **Solid tumour / Haematologic malignancy with no previous fungal infection** | 4 |
| **No dehydration requiring parenteral fluids** | 3 |
| **Outpatient status at onset** | 3 |
| **Age <60 years** | 2 |

| **Total Score** | **Risk** | **Management** |
|-----------------|----------|----------------|
| **≥21** | **Low risk** | **Outpatient oral antibiotics possible** (if safe follow-up) |
| **<21** | **High risk** | **Inpatient IV antibiotics mandatory** |

> [!tip]
> **MASCC validated for solid tumours and lymphoma; less reliable for acute leukaemia / transplant.** **Clinical judgement > score.** **Low-risk criteria also: MASCC ≥21, no comorbidities, reliable follow-up, able to take oral, no hypotension.**

## Empirical Antibiotic Therapy (START WITHIN 1 HOUR)
### Monotherapy (Preferred — IDSA/ESMO/NICE)
| Agent | Dose | Spectrum |
|-------|------|----------|
| **Piperacillin-tazobactam** | **4.5g IV 6h** (extended infusion 4h preferred) | **Gram-neg (Pseudomonas) + Anaerobes + some Gram-pos + Enterococcus** |
| **Meropenem** | **1g IV 8h** (or 2g IV 8h if Pseudomonas bacteremia) | **Gram-neg (Pseudomonas, ESBL) + Anaerobes** (NO Enterococcus) |
| **Cefepime** | **2g IV 8h** (or 2g IV 12h) | **Gram-neg (Pseudomonas) + Gram-pos** (weaker anaerobes, NO Enterococcus) |

> [!key]
> **Anti-pseudomonal β-lactam MONOTHERAPY = standard.** **Carbapenem preferred if: ESBL risk, prior carbapenem use, severe sepsis, haemodynamic instability.** **Avoid cefepime if severe sepsis (mortality signal in meta-analyses).** **Pip-tazo = broadest (covers Enterococcus, anaerobes).**

### Combination Therapy (Only if Indicated)
| Indication | Add |
|------------|-----|
| **Suspected catheter-related infection** (tunnel infection, port pocket infection) | **Vancomycin 15–20mg/kg IV 6h** |
| **Known MRSA colonisation / prior MRSA infection** | **Vancomycin** |
| **Skin/soft tissue infection** | **Vancomycin** |
| **Haemodynamic instability / septic shock** | **Vancomycin** (some guidelines) |
| **Pneumonia (infiltrate on CXR)** | **Vancomycin** (if MRSA risk) +/− **Azithromycin/Levofloxacin** (atypical) |
| **Gram-positive bacteremia before ID** | **Vancomycin** |

> [!warning]
> **Vancomycin NOT routine initially.** **Add only if specific indication.** **If added, review at 48–72h — stop if cultures negative for Gram-positive.**

### Penicillin Allergy (Anaphylaxis)
| Regimen | Dose |
|---------|------|
| **Aztreonam 2g IV 8h + Vancomycin 15–20mg/kg IV 6h** | Covers Gram-neg (Pseudomonas) + Gram-pos (no anaerobes) |
| **OR Ciprofloxacin 400mg IV 12h + Vancomycin 15–20mg/kg IV 6h** | (Ciprofloxacin less reliable for Pseudomonas monotherapy) |

## Initial Workup (Do NOT Delay Antibiotics)
| Test | Timing |
|------|--------|
| **Blood cultures** | **×2 sets (peripheral + each lumen) BEFORE antibiotics** |
| **Urine culture** | If urinary symptoms |
| **Sputum culture** | If productive cough |
| **Stool culture / C. difficile PCR** | If diarrhoea |
| **CXR** | If respiratory symptoms (portable if unstable) |
| **Peripheral blood film** | Confirm neutropenia |
| **Lactate, CRP, PCT, LFTs, U&Es, Coagulation** | Baseline |
| **Central line cultures** | If line in situ (peripheral + each lumen) |

## Reassessment & Escalation
### At 48–72 Hours
| Scenario | Action |
|----------|--------|
| **Afebrile, clinically stable, cultures negative** | **Continue empirical monotherapy** |
| **Afebrile, positive cultures** | **De-escalate to targeted therapy** |
| **Persistent fever (≥38.3°C), haemodynamically stable** | **Reassess:** repeat cultures, CXR, CT if indicated; **continue empirical monotherapy** |
| **Persistent fever, haemodynamically UNSTABLE** | **Escalate:** broaden Gram-neg (e.g., Meropenem if on Pip-tazo), **ADD Vancomycin**, consider antifungal, image for source |

### At 4–7 Days (Persistent Fever)
| Action | Indication |
|--------|------------|
| **Empirical ANTIFUNGAL (Echinocandin)** | **Persistent fever ≥4–7 days** despite broad antibiotics, no identified source, **profound/prolonged neutropenia (ANC <0.1 ×10⁹/L >7d expected)** |
| **Preferred antifungal** | **Caspofungin 70mg load then 50mg IV OD** OR **Micafungin 100mg IV OD** OR **Anidulafungin 200mg load then 100mg IV OD** |
| **Alternative** | **Liposomal Amphotericin B 3mg/kg IV OD** (if echinocandin contraindicated) |
| **Avoid** | Fluconazole (poor mould coverage), Voriconazole (drug interactions) unless mould suspected |

### Beyond 7 Days (Prolonged Fever)
| Consider | Investigation |
|----------|---------------|
| **Invasive mould infection (Aspergillus, Mucorales)** | **CT chest/sinuses, Galactomannan (serum/BAL), β-D-glucan** |
| **Resistant bacteria (VRE, CRE, MDR Pseudomonas)** | Culture review, rectal swabs for MDRO screening |
| **Viral (HSV, VZV, CMV, HHV-6, respiratory viruses)** | PCR panels,-CSF if neuro symptoms |
| **Non-infectious (drug fever, transfusion reaction, malignancy)** | Drug review, tryptase, imaging |

## Treatment Duration & Discontinuation
| Criteria for STOPPING Antibiotics | Requirement |
|-----------------------------------|-------------|
| **ANC >0.5 ×10⁹/L** (recovering) | **AND** |
| **Afebrile ≥48 hours** | **AND** |
| **Minimum 5–7 days** of antibiotics | **AND** |
| **No active infection source** | **AND** |
| **Haemodynamically stable** | → **STOP** |

> [!key]
> **Do NOT continue antibiotics just because neutropenia persists if afebrile and stable.** **Antifungals: continue until ANC >0.5 ×10⁹/L if started empirically.** **If documented fungal infection: full course per pathogen.**

## Central Line Management in FN
| Scenario | Action |
|----------|--------|
| **Catheter-related bloodstream infection (CRBSI) suspected** | **Blood cultures: peripheral + each lumen**; if differential time to positivity >2h OR quantitative ratio >5:1 → CRBSI |
| **Tunnel infection / Port pocket infection** | **Remove line + antibiotics** |
| **Uncomplicated CRBSI (coagulase-negative Staph, no septic shock, no endocarditis)** | **Antibiotics through line + systemic; consider line salvage** |
| **S. aureus / Pseudomonas / Candida / Bacillus / Mycobacteria CRBSI** | **REMOVE LINE + antibiotics** |
| **Line removal** | **Guidewire exchange NOT recommended in FN** |

## Antimicrobial Prophylaxis in Neutropenia
| Prophylaxis | Indication | Agent |
|-------------|------------|-------|
| **Antibacterial (fluoroquinolone)** | **High-risk (ANC <0.1 ×10⁹/L expected >7d):** acute leukaemia induction, autologous/allogeneic transplant | **Ciprofloxacin 500mg PO 12h** (or Levofloxacin 500mg PO OD) |
| **Antifungal (mould-active)** | **High-risk prolonged neutropenia:** allogeneic transplant, AML induction, relapsed leukaemia | **Posaconazole 300mg PO OD** (delayed-release tablet) OR **Voriconazole** (TDG) |
| **Antiviral (HSV/VZV)** | **Seropositive for HSV/VZV** | **Acyclovir 400mg PO 12h** (or Valacyclovir) |
| **PJP prophylaxis** | **All allogeneic transplant, lymphopenia (CD4<200), prolonged steroids** | **TMP-SMX 160/800mg PO 3×/week** (or Dapsone/Aerosolised pentamidine) |
| **G-CSF (Growth factor)** | **Secondary prophylaxis** (prior FN) OR **Primary if high-risk regimen** (FN risk >20%) | **Filgrastim 5mcg/kg SC OD** OR **Pegfilgrastim 6mg SC single dose per cycle** |

> [!tip]
> **Ciprofloxacin prophylaxis reduces Gram-neg bacteremia but ↑ resistance.** **Posaconazole reduces invasive fungal infections in high-risk.** **G-CSF: secondary prophylaxis standard; primary prophylaxis for regimens with >20% FN risk.**

## Special Scenarios
| Scenario | Modification |
|----------|--------------|
| **Autologous stem cell transplant** | Similar to high-risk FN; fluoroquinolone prophylaxis standard |
| **Allogeneic stem cell transplant** | Higher fungal risk (GVHD, steroids); posaconazole prophylaxis; CMV monitoring |
| **CAR-T cell therapy** | Cytokine release syndrome (CRS) mimics FN; tocilizumab for CRS; antibiotics for FN criteria |
| **Typhlitis (neutropenic enterocolitis)** | CT: bowel wall thickening, fat stranding; Pip-tazo/Meropenem ± Vancomycin; surgery if perforation |
| **Perirectal abscess** | High risk in neutropenia; avoid digital rectal exam; CT/MRI; antibiotics + drainage if feasible |

## FCPS/MRCP High-Yield Points
- **FN definition: ANC <0.5 + Temp ≥38.3°C (single) or ≥38.0°C sustained 1h**
- **EMERGENCY: IV antibiotics WITHIN 1 HOUR**
- **Empirical MONOTHERAPY: Pip-tazo 4.5g 6h (preferred) OR Meropenem 1g 8h OR Cefepime 2g 8h**
- **Vancomycin NOT routine; add if: catheter infection, MRSA risk, SSTI, shock, pneumonia, Gram-positive bacteremia**
- **MASCC ≥21 = low risk → outpatient oral possible (ciprofloxacin + amoxicillin-clavulanate)**
- **Reassess 48–72h: persistent fever stable → continue; unstable → broaden + vancomycin + image**
- **Empirical antifungal (echinocandin) at 4–7d persistent fever + profound/prolonged neutropenia**
- **Duration: until ANC >0.5 + afebrile 48h + minimum 5–7d**
- **Prophylaxis: Ciprofloxacin (high-risk neutropenia), Posaconazole (mould-active, transplant/AML), Acyclovir (HSV+), TMP-SMX (PJP)**
- **CRBSI: S. aureus/Pseudomonas/Candida → remove line; CoNS → consider salvage**

## Common Viva Questions
1. **Definition of febrile neutropenia?** ANC <0.5 ×10⁹/L + Temp ≥38.3°C (single) or ≥38.0°C sustained 1 hour.
2. **Time to antibiotics?** Within 1 hour of presentation.
3. **Preferred empirical regimen?** Anti-pseudomonal β-lactam monotherapy: Pip-tazo 4.5g 6h OR Meropenem 1g 8h OR Cefepime 2g 8h.
4. **When do you add vancomycin?** Catheter infection, known MRSA, SSTI, haemodynamic instability, pneumonia, Gram-positive bacteremia.
5. **MASCC score ≥21 means?** Low risk → outpatient oral antibiotics possible.
6. **When to start empirical antifungal?** Persistent fever ≥4–7 days despite antibiotics + profound/prolonged neutropenia (ANC<0.1 >7d expected).
7. **Preferred empirical antifungal?** Echinocandin (Caspofungin/Micafungin/Anidulafungin).
8. **When to stop antibiotics?** ANC >0.5 + afebrile ≥48h + minimum 5–7 days + no active infection.
9. **CRBSI with S. aureus — line management?** REMOVE LINE + antibiotics.
10. **Prophylaxis for high-risk neutropenia (AML induction)?** Ciprofloxacin + Posaconazole + Acyclovir (if HSV+) + TMP-SMX (PJP).

## Common Confusions / Exam Traps
| Confusion | Clarification |
|-----------|---------------|
| Wait for ANC confirmation before antibiotics | **START antibiotics immediately on clinical suspicion; do NOT wait** |
| Vancomycin routine in all FN | **NOT routine; only with specific indications** |
| Dual Gram-neg coverage standard | **Monotherapy (anti-pseudomonal β-lactam) = standard** |
| Cefepime preferred in septic shock | **Avoid cefepime in severe sepsis/shock (mortality signal); use Meropenem or Pip-tazo** |
| MASCC ≥21 = automatic outpatient | **Clinical judgement required; need reliable follow-up, oral tolerance, no comorbidities** |
| Stop antibiotics when ANC still <0.5 | **Stop if afebrile 48h + stable + min 5–7d; do NOT wait for ANC recovery if stable** |
| Fluconazole for empirical antifungal | **Echinocandin preferred (mould coverage); fluconazole only for Candida** |
| Continue antifungals indefinitely | **Stop empirical antifungals when ANC >0.5; continue only if documented infection** |
| Guidewire exchange for CRBSI in FN | **NOT recommended in neutropenic patients** |
| Ciprofloxacin prophylaxis for all chemo | **Only high-risk (ANC<0.1 expected >7d): AML induction, transplant** |

## Summary
**Febrile neutropenia (FN)** = **ONCOLOGIC EMERGENCY.** **Definition:** Absolute Neutrophil Count (ANC) **<0.5 ×10⁹/L** (or <1.0 with predicted decline to <0.5) **AND** Temperature **≥38.3°C (single) OR ≥38.0°C sustained ≥1 hour.** **Immediate management:** **IV antibiotics WITHIN 1 HOUR.** Blood cultures ×2 sets (peripheral + each lumen) before antibiotics. **Empirical monotherapy (preferred):** Anti-pseudomonal β-lactam — **Piperacillin-tazobactam 4.5g IV 6h** (extended infusion preferred) **OR Meropenem 1g IV 8h OR Cefepime 2g IV 8h.** **Cefepime avoid in septic shock.** **Vancomycin NOT routine** — add only if: catheter-related infection, known MRSA, skin/soft tissue infection, haemodynamic instability, pneumonia, Gram-positive bacteremia. **Risk stratification:** **MASCC score ≥21 = low risk** → consider outpatient oral antibiotics (Ciprofloxacin + Amoxicillin-clavulanate) if reliable follow-up, oral tolerance, no comorbidities. **Reassessment:** **48–72h:** persistent fever + stable → continue monotherapy; **haemodynamically unstable** → broaden Gram-neg, ADD Vancomycin, image for source. **4–7 days:** persistent fever + profound/prolonged neutropenia (ANC <0.1 ×10⁹/L expected >7d) → **ADD empirical antifungal: Echinocandin** (Caspofungin 70/50mg, Micafungin 100mg, Anidulafungin 200/100mg IV OD). **>7 days:** consider invasive mould infection — CT chest/sinuses, Galactomannan, β-D-glucan. **Duration:** Continue until **ANC >0.5 ×10⁹/L AND afebrile ≥48h AND minimum 5–7 days** (stop if stable even if neutropenia persists). **Prophylaxis:** **Ciprofloxacin** (high-risk: ANC <0.1 expected >7d — AML induction, transplant); **Posaconazole** (mould-active — allogeneic transplant, AML induction); **Acyclovir** (HSV/VZV seropositive); **TMP-SMX** (PJP — allogeneic transplant, CD4<200, prolonged steroids); **G-CSF** (secondary prophylaxis, or primary if regimen FN risk >20%). **CRBSI:** *S. aureus, Pseudomonas, Candida, Bacillus, Mycobacteria* → **REMOVE LINE**; CoNS → consider salvage (antibiotics through line + systemic).

## Answer Key with Explanations
### MCQs
1. **B** — FN definition: ANC <0.5 ×10⁹/L + Temp ≥38.3°C (single) OR ≥38.0°C sustained ≥1 hour.
2. **B** — IV antibiotics within 1 hour of presentation (IDSA/ESMO/NICE).
3. **B** — Anti-pseudomonal β-lactam monotherapy: Pip-tazo, Meropenem, or Cefepime. Ceftriaxone NOT anti-pseudomonal.
4. **B** — MASCC ≥21 = low risk; outpatient oral antibiotics possible if clinically appropriate.
5. **D** — Vancomycin NOT routinely added; only with specific indications (catheter, MRSA, SSTI, shock, pneumonia, Gram+ bacteremia).
6. **B** — Empirical antifungal at 4–7 days persistent fever + profound/prolonged neutropenia (ANC <0.1 expected >7d).
7. **C** — Echinocandin (Caspofungin/Micafungin/Anidulafungin) preferred for empirical antifungal in neutropenia (mould coverage, fungicidal, no drug interactions).
8. **B** — Stop when ANC >0.5 ×10⁹/L AND afebrile ≥48h AND minimum 5–7 days antibiotics. Do NOT wait for ANC >1.0 if stable.
9. **C** — S. aureus CRBSI in neutropenia → REMOVE LINE + systemic antibiotics.
10. **B** — Ciprofloxacin prophylaxis for high-risk: ANC <0.1 ×10⁹/L expected >7d (AML induction, autologous/allogeneic transplant).

## 1. 🎯 Learning Objectives

- [ ] Identify **Transmission**: Rat urine → Contaminated water/soil → Skin/mucosal entry

- [ ] Recognise **Complications**: Pulmonary Haemorrhage (Anaemic, High Mortality), AKI, Myocarditis, Uveitis
- [ ] Apply **Prophylaxis**: Doxycycline 200mg Weekly for High-Risk Exposure

## 2. 📖 Definition & Epidemiology

| Feature | Details |
|---------|---------|
| **Causative Agent** | **Leptospira interrogans** (Spirochaete); **>250 Serovars** (Icterohaemorrhagiae, Canicola, Grippotyphosa, Pomona) |
| **Reservoir** | **Rats** (Primary), Cattle, Pigs, Dogs, Wildlife; **Asymptomatic Renal Carriage** → Urine Shedding |
| **Transmission** | **Direct/Indirect Contact** with Infected Urine → **Contaminated Water/Soil** → **Broken Skin/Mucosa/Conjunctiva** |
| **Risk Factors** | **Occupational** (Farmers, Sewer Workers, Vets, Abattoir), **Recreational** (Freshwater Swimming, Rafting, Caving), **Flooding** (Outbreaks) |
| **Geography** | **Tropical/Subtropical** (SE Asia, Caribbean, Oceania, Latin America); **Temperate** (Seasonal, Occupational) |
| **Incubation** | **5-14 Days** (Range 2-30 Days) |

> [!tip] **Leptospirosis = Rat Urine + Water Exposure + Fever + Jaundice/Renal Failure**. **Weil's Disease = Severe Icteric Form** (5-10% of cases).

---

## 3. 📖 Clinical Spectrum

| Form | Features | Proportion |
|--------|----------|------------|
| **Subclinical / Mild (Anicteric)** | Fever, Myalgia (Calf **Classic**), Headache, Conjunctival Suffusion, Rash | **~90%** |
| **Severe (Weil's Disease)** | **Jaundice**, **Renal Failure**, **Haemorrhage** (Pulmonary, GI, Adrenal), **Pulmonary Haemorrhage** (ARDS), Myocarditis, Aseptic Meningitis | **5-10%** |

### Weil's Disease Triad (Classic)

| Feature | Pathophysiology |
|---------|-----------------|
| **Jaundice** | Hepatocellular Dysfunction + Haemolysis; **Conjugated + Unconjugated Hyperbilirubinaemia** |
| **Renal Failure** | **Tubulointerstitial Nephritis** → ATN; **Non-oliguric** Common; Hypokalaemia |
| **Haemorrhage** | **Thrombocytopenia**, **Capillary Damage**, **Coagulopathy** (DIC-like); **Pulmonary Haemorrhage** → Major Cause of Death |

### Pulmonary Haemorrhage (Severe Complication)

| Feature | Details |
|---------|---------|
| **Incidence** | **10-50% of Severe Cases**; **Mortality 50-70%** |
| **Presentation** | **Haemoptysis**, Dyspnoea, Hypoxaemia, Bilateral Consolidation on CXR |
| **Pathology** | **Alveolar Capillary Damage** → Alveolar Haemorrhage; **Immune-Mediated** |
| **Management** | **High-Dose Methylprednisolone** (1g IV Daily ×3), **Ventilatory Support**, **RRT**, **Plasma Exchange** (Refractory) |

## 4. 🔬 Diagnostic Workup

```mermaid
flowchart TD
    A[Fever + Exposure History + Jaundice/Renal Failure] --> B[**Initial: CBC, LFT, Renal, Coag, CXR**]
    B --> C{**Key Findings**}
    C --> C1[**Leukocytosis + Neutrophilia**]
    C --> C2[**Thrombocytopenia** (Common)]
    C --> C3[**↑ Bilirubin (Conj+Unconj), ↑ ALT/AST (Mild)**]
    C --> C4[**↑ Creatinine, ↑ K+, Metabolic Acidosis**]
    C --> C4[**CXR: Bilateral Alveolar Infiltrates** (Pulm Haemorrhage)]
    C1 & C2 & C3 & C4 --> D[**Confirmatory Tests**]
    D --> D1[**MAT (Gold Standard)** - *Paired Sera 2-4w Apart, 4-fold Rise*]
    D --> D2[**IgM ELISA** - *Early (Day 5-7), Sensitivity 90%*]
    D3[**PCR (Blood/Urine)** - *Early (Day 1-5), High Sensitivity/Specificity*]
    D --> D4[**Dark Field Microscopy (Urine)** - *Motile Spirochaetes; Low Sensitivity*]
```

### Key Diagnostic Tests

| Test | Timing | Sensitivity/Specificity | Role |
|------|--------|------------------------|------|
| **MAT (Microscopic Agglutination Test)** | **Convalescent (Day 10-14)**; **Paired Sera** | **Gold Standard**; Serovar Identification | **Confirmatory** |
| **IgM ELISA** | **Day 5-7 Onwards** | **~90% Sens, 95% Spec** | **Early Diagnosis** |
| **PCR (Blood/Urine)** | **Day 1-10** (Before Antibodies) | **High Sens/Spec** | **Early / Confirmatory** |
| **Dark Field Microscopy** | **Fresh Urine** | **Low Sensitivity (40-50%)** | **Bedside / Rapid** |
| **Culture (Ellinghausen-McCullough-Johnson-Harris Medium)** | **Weeks** | **Low Sensitivity** | **Retrospective / Reference** |

> [!tip] **MAT = Gold Standard** but **Requires Paired Sera** (Acute + Convalescent 2-4 Weeks Apart). **IgM ELISA / PCR = Early Diagnosis**.

## 5. 💊 Management

### Antibiotics

| Severity | Regimen | Duration |
|----------|---------|----------|
| **Mild (Anicteric)** | **Doxycycline 100mg PO BD** | **7 Days** |
| **Severe (Weil's / ICU)** | **IV Penicillin G 1.5g IV Q6H** (or **Ceftriaxone 2g OD IV** / **Cefotaxime 2g Q6H**) | **7 Days** |
| **Penicillin Allergy** | **Doxycycline 100mg IV/PO BD** / **Ceftriaxone 2g OD IV** | **7 Days** |

> [!tip] **Doxycycline = First-Line for Mild/Outpatient**; **IV Penicillin/Ceftriaxone = Severe/ICU**. **Jarisch-Herxheimer Reaction** Common (Fever, Rigors, Hypotension 2-4h Post-First Dose) — **Supportive, Continue Antibiotics**.

### Supportive Care (Severe/Weil's)

| Complication | Management |
|--------------|------------|
| **Renal Failure** | **RRT** (Early Initiation); **K+ Management**, **Fluid Balance** |
| **Pulmonary Haemorrhage** | **High-Dose Methylprednisolone 1g IV Daily ×3-5d**; **Ventilatory Support** (Lung-Protective); **rFVIIa / Tranexamic Acid** (Refractory) |
| **Thrombocytopenia/Coagulopathy** | **Platelet Transfusion** (<20 or Bleeding); **FFP/Cryo** if DIC |
| **Myocarditis** | **Supportive**, **Inotropes** if Needed |
| **Electrolytes** | **Hypokalaemia** (Renal Tubular Dysfunction) → **Aggressive K+ Replacement** |

> [!warning] **Jarisch-Herxheimer Reaction**: Fever, Rigors, Hypotension, Worsening Rash **2-4h Post-First Dose** — **Supportive (Fluids, Vasopressors), Continue Antibiotics**, **Antipyretics**.

## 6. 🔬 Laboratory Findings

| Parameter | Typical Findings |
|-----------|------------------|
| **CBC** | Leukocytosis + Neutrophilia, **Thrombocytopenia** (Common), **Anaemia** (Haemolysis/Blood Loss) |
| **LFT** | **↑ Bilirubin (Conj+Unconj)**, **Mild ↑ AST/ALT** (Hepatocellular), **↑ ALP** (Cholestatic) |
| **Renal** | **↑ Creatinine, ↑ Urea**, **Hyperkalaemia** (Tubular Dysfunction), **Metabolic Acidosis** |
| **Coagulation** | **Prolonged PT/APTT**, **↓ Fibrinogen**, **↑ D-dimer** (DIC-like) |
| **CPK** | **Markedly Elevated** (Myositis) |
| **CSF** | **Lymphocytic Pleocytosis**, **Normal/Raised Protein**, **Normal Glucose** (Aseptic Meningitis) |

## 7. 🔄 Differential Diagnosis

| Condition | Differentiating Features |
|-----------|--------------------------|
| **Viral Hepatitis (A/E)** | **No Renal Failure/Pulmonary Haemorrhage**, **Marked Transaminitis** (>1000), **Anti-HAV IgM / Anti-HEV IgM** |
| **Malaria** | **Parasitaemia on Film**, **No Renal Failure** (Unless Severe), **No Pulm Haemorrhage** |
| **Septicemia (Gram -ve)** | **Blood Culture Positive**, **No Conjunctival Suffusion**, **No Myalgia (Calf)** |
| **Hantavirus (HFRS)** | **Similar Syndrome** (Renal Failure, Haemorrhage), **Hantaan/Seoul Virus**, **Exposure to Rodents** |
| **Sepsis (Other)** | **Blood Culture Positive**, **No Conjunctival Suffusion**, **No Calf Tenderness** |
| **Acute Viral Hepatitis (A/B/E)** | **ALT/AST >> Bilirubin**, **No Renal Failure**, **Serology Positive** |
| **Dengue** | **NS1 Ag+, Thrombocytopenia, Plasma Leakage**, **No Renal Failure/Pulm Haemorrhage (Usually)** |

> [!tip] **Conjunctival Suffusion (Non-Purulent Conjunctival Injection) = Highly Suggestive of Leptospirosis** (50-90%).

## 8. 🛡️ Prevention & Prophylaxis

| Strategy | Details |
|----------|---------|
| **Pre-Exposure Prophylaxis** | **Doxycycline 200mg PO Weekly** (Start 1-2 Days Before, Continue During Exposure) — **High-Risk Occupations/Activities** |
| **Post-Exposure** | **Doxycycline 100mg BD × 7 Days** if High-Risk Exposure Recognised |
| **Environmental** | **Rodent Control**, **Drainage**, **Protective Clothing/Footwear**, **Avoid Contaminated Water** |
| **Vaccination** | **Animal Vaccines Available** (Cattle, Dogs); **Human Vaccine** (Limited Serovar Coverage, Used in High-Risk Occupations in Some Countries) |

## 10. ❓ Viva Questions

1. **What is the classic triad of Weil's disease?**
   - **Jaundice + Renal Failure + Haemorrhage** (Pulmonary Haemorrhage is Major Cause of Death).

2. **What is the most characteristic physical finding in Leptospirosis?**
   - **Conjunctival Suffusion** (Non-Purulent Conjunctival Injection) — **Present in 50-90%**.

3. **What is the drug of choice for mild Leptospirosis?**
   - **Doxycycline 100mg PO BD × 7 Days**.

4. **What is the drug of choice for severe Leptospirosis (Weil's Disease)?**
   - **IV Penicillin G 1.5MU IV Q6H** (or **Ceftriaxone 2g OD IV / Cefotaxime 2g Q6H IV**).

5. **What is the Jarisch-Herxheimer Reaction and how is it managed?**
   - **Fever, Rigors, Hypotension 2-4h Post-First Dose** → **Supportive (Fluids, Vasopressors), Continue Antibiotics**.

6. **What is the gold standard diagnostic test for Leptospirosis?**
   - **Microscopic Agglutination Test (MAT)** on **Paired Sera (Acute + Convalescent 2-4 Weeks Apart)** showing **4-Fold Rise**.

7. **What is the earliest diagnostic test for Leptospirosis?**
   - **PCR (Blood/Urine)** — Detects DNA **Day 1-10**, Before Antibodies Appear.

7. **How is Pulmonary Haemorrhage in Leptospirosis managed?**
   - **High-Dose Methylprednisolone 1g IV Daily ×3-5 Days**, **Ventilatory Support**, **RRT**, **rFVIIa/Tranexamic Acid** if Refractory.

8. **What is the Prophylaxis for High-Risk Exposure?**
   - **Doxycycline 200mg PO Weekly** (Start 1-2 Days Before Exposure, Continue During Exposure).

9. **What is the typical CSF finding in Leptospiral Meningitis?**
   - **Lymphocytic Pleocytosis**, **Normal Glucose**, **Mildly Elevated Protein**.

10. **Differentiate Leptospirosis from Viral Hepatitis, Malaria, and Hantavirus.**
    - **Lepto: Conjunctival Suffusion + Calf Myalgia + Renal Failure + Pulm Haemorrhage**; **Hepatitis: ALT/AST >> Bilirubin, No Renal Failure**; **Malaria: Parasitaemia, Cyclical Fever**; **Hantavirus: Similar but No Conjunctival Suffusion, Rodent Exposure, Hantaan/Seoul Serology**.

## 11. 🧠 Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Leptospirosis vs Viral Hepatitis** | **Lepto: Renal Failure + Pulm Haemorrhage + Conjunctival Suffusion**; **Hepatitis: ALT/AST >> Bilirubin, No Renal Failure** |
| **Leptospirosis vs Malaria** | **Malaria: Parasitaemia on Film, Cyclical Fever, No Conjunctival Suffusion, No Pulm Haemorrhage** |
| **Leptospirosis vs Hantavirus (HFRS)** | **Both: Renal Failure + Haemorrhage**; **Hanta: No Conjunctival Suffusion, Hantaan/Seoul Serology, Rodent Exposure** |
| **Jarisch-Herxheimer vs Allergic Reaction** | **JH: 2-4h Post-Abx, Fever/Rigors/Hypotension, Self-Limiting**; **Allergy: Immediate, Urticaria, Angioedema, Bronchospasm** |
| **Doxycycline vs Penicillin** | **Doxy: Mild/Outpatient, PO**; **Penicillin/Ceftriaxone: Severe/ICU, IV** |

| Mnemonic | Meaning |
|----------|---------|
| **"Lepto = Rat Urine + Water + Weil's Triad"** | Transmission + Severe Form |
| **"Weil's = Jaundice + Kidney + Blood"** | Triad |
| **"Conjunctival Suffusion = Lepto Hallmark"** | Physical Sign |
| **"Doxy for Mild, Pen for Severe"** | Treatment Algorithm |
| **"JH Reaction = Fluids + Continue Abx"** | Jarisch-Herxheimer |

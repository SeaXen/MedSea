> [!important]
> **Immunocompromised host = defective innate/adaptive immunity → ↑ risk, atypical presentation, opportunistic pathogens.** **Risk stratification by deficit: neutropenia, T-cell deficiency, B-cell/humoral, complement, phagocyte, barrier.** **Transplant timeline: 0–1m (surgical, hospital flora), 1–6m (opportunistic: CMV, PCP, fungi), >6m (community + chronic rejection).** **Biologics: anti-TNF → TB, intracellular; anti-CD20 → HBV reactivation, PML; IL-17/23 → Candida, TB.**

## 1. Learning Objectives
- Classify immunodeficiency by immune deficit and risk profile
- Apply transplant timeline for infection risk (0–1m, 1–6m, >6m)
- Recognise biologic-specific infection risks (anti-TNF, anti-CD20, integrin inhibitors, JAK inhibitors)
- Select appropriate prophylaxis (PCP, fungal, viral, TB)
- Diagnose and manage key opportunistic infections (CMV, PCP, EBV-PTLD, BK virus, HHV-6)
- Apply vaccination principles in immunocompromised (timing, live vs inactivated)

## 2. Classification by Immune Deficit
| Deficit | Causes | Typical Infections |
|---------|--------|-------------------|
| **Neutropenia** | Chemo, HSCT, drugs (ganciclovir, valproate), aplastic anaemia | **Bacteria (Gram-neg, Staph), Candida, Aspergillus, Gram-neg sepsis** |
| **T-cell Deficiency** | HIV, transplant (CNI, mTOR, steroids), biologics (anti-TNF, alemtuzumab), chemo | **Viruses (CMV, EBV, BK, HHV-6, VZV), TB, fungi (Pneumocystis, Cryptococcus), Listeria, Nocardia, Toxoplasma** |
| **B-cell / Humoral** | Anti-CD20 (rituximab), BTK inhibitors, CLL, myeloma, hypogammaglobulinaemia | **Encapsulated bacteria (S. pneumoniae, H. influenzae), Giardia, enteroviruses, chronic sinopulmonary** |
| **Complement** | C1q–C9 deficiency, eculizumab, C1-INH deficiency | **Encapsulated bacteria (Neisseria, S. pneumoniae), immune complex disease** |
| **Phagocyte** | CGD, diabetes, steroids, dialysis, Burns | **Catalase-positive organisms (S. aureus, Burkholderia, Nocardia, Aspergillus)** |
| **Barrier Defect** | Burns, surgery, lines, drains, mucositis | **Skin flora (Staph, Strep), Candida, Gram-neg** |

## 3. Transplant Timeline — Infection Risk by Period
| Period | Timeframe | Key Risks | Typical Pathogens |
|--------|-----------|-----------|-------------------|
| **Early** | **0–1 month** | Surgical, hospital-acquired | **Wound infection, bacteremia (CoNS, Gram-neg), UTI, C. difficile, donor-derived** |
| **Intermediate** | **1–6 months** | **OPPORTUNISTIC** (peak) | **CMV (peak 1–4m), PCP, invasive fungi (Aspergillus, Candida), EBV-PTLD, BK virus, HHV-6, adenovirus, TB, Nocardia, Listeria** |
| **Late** | **>6 months** | Community-acquired + chronic rejection | **Community viruses, encapsulated bacteria, chronic viral (BK, CMV), chronic rejection** |
| **All periods** | Any time | **Net state of immunosuppression** | **Net state = baseline risk + current immunosuppression + exposures** |

> [!key]
> **CMV = MOST IMPORTANT opportunistic infection in transplant.** **Peak 1–4 months.** **PCP prophylaxis universal 6–12 months.** **Antifungal prophylaxis in high-risk (lung, liver, heart, re-transplant).**

## 4. Biologic-Specific Infection Risks
| Biologic Class | Target | Key Infection Risks | Screening/Monitoring |
|----------------|--------|---------------------|----------------------|
| **Anti-TNF** (infliximab, adalimumab, etanercept, certolizumab, golimumab) | TNF-α | **TB (reactivation), intracellular bacteria (Listeria, Salmonella, Nocardia), histoplasmosis, coccidioidomycosis, hepatitis B reactivation** | **TB screen (IGRA/TST) BEFORE; HBV screen; monitor for infection** |
| **Anti-IL-12/23** (ustekinumab) | IL-12/IL-23 | TB, fungal, viral (similar to anti-TNF but lower risk) | TB screen before |
| **Anti-IL-17** (secukinumab, ixekizumab) | IL-17A | **Candida (mucocutaneous), TB (rare)** | Monitor for Candida |
| **Anti-IL-23** (guselkumab, risankizumab) | IL-23 | TB, fungal (lower risk than anti-TNF) | TB screen before |
| **Anti-IL-6** (tocilizumab, sarilumab) | IL-6R | Bacterial, TB, diverticulitis, hepatotoxicity | TB screen; monitor LFTs, neutrophils |
| **Anti-CD20** (rituximab, ocrelizumab, ofatumumab) | CD20 (B-cells) | **HBV reactivation (HBsAg+ or anti-HBc+), PML (JCV), hypogammaglobulinaemia, encapsulated bacteria, VZV** | **HBV screen (HBsAg, anti-HBc) BEFORE; IgG levels; JCV index if natalizumab hx** |
| **Anti-CD52** (alemtuzumab) | CD52 (T/B/NK) | **Severe T-cell depletion → CMV, VZV, Listeria, Nocardia, TB, fungal, PML** | Intensive monitoring; prophylaxis (PCP, viral, fungal) |
| **Anti-integrin** (natalizumab, vedolizumab) | α4β1/α4β7 integrin | **PML (JCV) — natalizumab; vedolizumab gut-selective = lower PML risk** | **JCV antibody index (natalizumab); monitor** |
| **JAK Inhibitors** (tofacitinib, baricitinib, upadacitinib) | JAK1/2/3 | **Herpes zoster (VZV) ↑↑, TB, bacterial, fungal, CMV, EBV, lymphopenia** | VZV vaccination before; TB screen; monitor lymphocytes |
| **BTK Inhibitors** (ibrutinib, acalabrutinib) | BTK | **Bleeding, atrial fibrillation, infections (sinopulmonary, fungal), PML risk** | Monitor IgG, infections |
| **BCL-2 Inhibitor** (venetoclax) | BCL-2 | **TLS, neutropenia, infections** | TLS prophylaxis; monitor counts |

> [!tip]
> **TB screen (IGRA/TST) REQUIRED BEFORE starting anti-TNF, anti-IL-12/23, anti-IL-17, anti-IL-23, JAK inhibitors, alemtuzumab.** **HBV screen (HBsAg, anti-HBc) REQUIRED BEFORE anti-CD20, anti-TNF, JAK inhibitors.**

## 5. Key Opportunistic Infections in Transplant
### CMV Disease
| Aspect | Details |
|--------|---------|
| **Risk** | D+/R− (highest), lung/heart/intestinal transplant, lymphocyte-depleting induction |
| **Syndromes** | **Viraemia, syndrome (fever, leucopenia, hepatitis), tissue-invasive (pneumonitis, colitis, retinitis, encephalitis)** |
| **Diagnosis** | **Quantitative PCR (blood)** — viral load guides pre-emptive therapy |
| **Prophylaxis vs Pre-emptive** | **High-risk: prophylaxis (valganciclovir 900mg OD ×3–6m); Standard-risk: pre-emptive (weekly PCR ×3–6m)** |
| **Treatment** | **Valganciclovir 900mg BD (oral) OR Ganciclovir 5mg/kg IV 12h** ×2–3w → maintenance; **Foscarnet/Cidofovir** if resistant (UL97/UL54 mut) |
| **Resistance** | **UL97 mutation = ganciclovir/valganciclovir resistance; UL54 = cidofovir/foscarnet resistance** |

### PCP (Pneumocystis jirovecii Pneumonia)
| Aspect | Details |
|--------|---------|
| **Risk** | All transplant (T-cell suppression); HSCT; biologics (anti-TNF, alemtuzumab); HIV CD4<200 |
| **Prophylaxis** | **TMP-SMX 960mg PO daily (or 3×/wk)**; alternatives: dapsone, atovaquone, inhaled pentamidine |
| **Duration** | **≥6–12 months post-transplant**; lifelong if ongoing immunosuppression |
| **Treatment** | **High-dose TMP-SMX 15–20mg/kg TMP IV/PO q6h ×21d** + steroids if hypoxic |

### EBV-PTLD (Post-Transplant Lymphoproliferative Disorder)
| Aspect | Details |
|--------|---------|
| **Risk** | EBV D+/R−, intense immunosuppression (OKT3, ATG), paediatric, intestinal/lung transplant |
| **Spectrum** | **Early lesions → polymorphic PTLD → monomorphic PTLD (lymphoma)** |
| **Diagnosis** | **EBV PCR (blood)** — rising viral load; biopsy for histology |
| **Treatment** | **Reduce immunosuppression (RI) = cornerstone**; **Rituximab** (anti-CD20); **EBV-CTL**; chemo for monomorphic |

### BK Virus Nephropathy
| Aspect | Details |
|--------|---------|
| **Risk** | Kidney transplant; immunosuppression (Tacrolimus, MMF); ureteric stent |
| **Diagnosis** | **Quantitative PCR (blood, urine)** — decoy cells in urine; biopsy (SV40 staining) |
| **Treatment** | **Reduce immunosuppression (reduce/stop Tac/MMF)**; **Leflunomide, Cidofovir, IVIG** (adjunctive) |

## 6. Fungal Infections in Immunocompromised
| Fungus | High-Risk Host | Prophylaxis | Treatment |
|--------|----------------|-------------|-----------|
| **Aspergillus** | Lung transplant, HSCT, prolonged neutropenia, steroids | Posaconazole (HSCT, AML), voriconazole (lung) | **Voriconazole 1st-line**; isavuconazole; L-AmB |
| **Candida** | ICU, abdominal surgery, neutropenia, TPN, broad antibiotics | Fluconazole (selective), echinocandin (high-risk) | Echinocandin 1st-line; step-down fluconazole |
| **Mucormycosis** | DKA, neutropenia, deferoxamine, steroids, transplant | None standard | L-AmB + urgent surgery |
| **Pneumocystis** | All transplant, HIV CD4<200, biologics | **TMP-SMX** universal | High-dose TMP-SMX + steroids |
| **Cryptococcus** | HIV CD4<100, transplant, steroids | Fluconazole (targeted) | L-AmB + flucytosine → fluconazole |

## 7. Prophylaxis Protocols — Transplant (Example: Kidney)
| Pathogen | Agent | Duration | High-Risk Indications |
|----------|-------|----------|----------------------|
| **PCP** | TMP-SMX 960mg 3×/wk | 6–12 months | All (lifelong if ongoing immunosuppression) |
| **CMV** | Valganciclovir 900mg OD | 3–6 months (prophylaxis) OR pre-emptive | D+/R−, lung/heart, lymphocyte depletion |
| **Fungal** | Fluconazole 200mg OD / Posaconazole | 1–3 months | Lung, liver, heart, re-transplant, prolonged ICU |
| **VZV** | Acyclovir 400mg BD | 3–6 months | Seropositive recipients |
| **HBV** | Entecavir/TDF | Indefinite | HBsAg+ recipients |
| **UTI** | TMP-SMX / Nitrofurantoin | 3–6 months | Recurrent UTI, structural abnormality |

## 8. Vaccination in Immunocompromised
| Vaccine | Type | Timing | Notes |
|---------|------|--------|-------|
| **Inactivated (Flu, Pneumococcal, Hep A/B, HPV, Tdap, COVID, Rabies, IPV, Hib, MenACWY)** | Inactivated | **Anytime**; ideally **≥2 weeks BEFORE immunosuppression** | May have reduced response; give extra doses per schedule |
| **Live (MMR, Varicella, Zoster live, Yellow Fever, BCG, Oral Typhoid, Rotavirus, LAIV)** | Live | **≥4 weeks BEFORE immunosuppression**; **CONTRAINDICATED during** | **Except**: Zoster recombinant (Shingrix) = safe during; MMR/Varicella if CD4>200 in HIV |
| **Zoster Recombinant (Shingrix)** | Recombinant (non-live) | **Safe during immunosuppression** | **Preferred for immunocompromised ≥18y** |
| **Post-Transplant** | Inactivated: 3–6m post; Live: **≥2 years post** (if off immunosuppression) | Check local guidelines |

> [!warning]
> **Live vaccines CONTRAINDICATED during immunosuppression.** **Exception: Shingrix (recombinant) safe.** **HIV CD4>200: MMR/Varicella may be given.**

## 9. FCPS/MRCP High-Yield Points
- **Transplant timeline: 0–1m (surgical/hospital), 1–6m (opportunistic: CMV, PCP, fungi, EBV-PTLD), >6m (community)**
- **CMV = most important opportunistic infection; D+/R− highest risk; prophylaxis vs pre-emptive; valganciclovir/ganciclovir**
- **PCP prophylaxis: TMP-SMX universal 6–12m post-transplant; lifelong if ongoing immunosuppression**
- **Anti-TNF: TB reactivation (screen IGRA/TST BEFORE); histoplasma, coccidioides, HBV reactivation**
- **Anti-CD20 (rituximab): HBV reactivation (screen HBsAg/anti-HBc BEFORE); PML risk; hypogammaglobulinaemia**
- **JAK inhibitors: VZV (shingles) ↑↑; TB; vaccinate VZV before; screen TB**
- **Alemtuzumab: severe T-cell depletion; intense prophylaxis (PCP, viral, fungal)**
- **PCP prophylaxis: TMP-SMX 6–12m post-transplant; lifelong if ongoing immunosuppression**
- **Fungal prophylaxis: posaconazole/voriconazole in high-risk (lung, liver, heart, HSCT)**
- **Vaccination: live vaccines contraindicated during immunosuppression; Shingrix safe; give inactivated ≥2w before**

## 10. Common Viva Questions
1. **Transplant infection timeline?** 0–1m: surgical/hospital; 1–6m: opportunistic (CMV, PCP, fungi, EBV); >6m: community.
2. **CMV prophylaxis vs pre-emptive?** High-risk (D+/R−, lung/heart, lymphocyte depletion) = prophylaxis 3–6m; standard-risk = pre-emptive PCR-guided.
3. **Anti-TNF screening?** TB (IGRA/TST), HBV BEFORE starting.
3. **Anti-CD20 (rituximab) screening?** HBV (HBsAg, anti-HBc) BEFORE; monitor IgG, infections.
4. **JAK inhibitor infection risk?** VZV (shingles) ↑↑, TB, bacterial, fungal; vaccinate VZV before; monitor lymphocytes.
5. **PCP prophylaxis post-transplant?** TMP-SMX 960mg 3×/wk ×6–12m; lifelong if ongoing immunosuppression.
6. **CMV treatment?** Valganciclovir 900mg BD or ganciclovir 5mg/kg IV 12h; foscarnet/cidofovir if resistant.
7. **BK virus nephropathy management?** Reduce immunosuppression (Tac/MMF); leflunomide/cidofovir/IVIG adjunct.
7. **Anti-CD20 HBV reactivation?** Screen HBsAg/anti-HBc BEFORE; prophylactic antivirals if HBsAg+ or anti-HBc+.
8. **JAK inhibitor VZV risk?** Shingles ↑↑; vaccinate with Shingrix BEFORE starting.
9. **Alemtuzumab prophylaxis?** PCP (TMP-SMX), viral (acyclovir/valganciclovir), fungal (posaconazole) — intensive.

## 11. Common Confusions / Exam Traps
| Confusion | Clarification |
|-----------|---------------|
| CMV prophylaxis for all transplant | **Only high-risk (D+/R−, lung/heart, lymphocyte depletion); standard-risk = pre-emptive** |
| PCP prophylaxis only 6 months | **Continue lifelong if ongoing immunosuppression** |
| Anti-TNF safe if TB screen negative | **Screen IGRA/TST BEFORE; but can still reactivate latent TB (false neg) — educate patient** |
| Rituximab only affects B-cells | **Also causes hypogammaglobulinaemia → encapsulated bacteria; PML risk (JCV)** |
| JAK inhibitors = just oral DMARDs | **Significant VZV ↑↑, TB, infection risk; vaccinate Shingrix BEFORE** |
| Live vaccines always contraindicated | **Shingrix (recombinant) = SAFE during immunosuppression** |
| CMV treatment = valganciclovir only | **Resistance common (UL97/UL54); foscarnet/cidofovir for resistance** |
| PCP only in HIV | **Transplant, biologics, steroids, malignancy all at risk** |
| Fungal prophylaxis for all transplant | **Only high-risk (lung, liver, heart, re-tx, prolonged neutropenia, steroids)** |

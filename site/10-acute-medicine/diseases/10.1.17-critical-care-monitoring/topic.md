> [!important]
> Continuous monitoring is the foundation of critical care. Know what to monitor, why, and how to interpret.

## 1. Learning Objectives
- List essential monitoring in the critically ill
- Interpret basic haemodynamic parameters
- Know indications for invasive monitoring
- Recognise monitoring limitations

## 2. Standard Monitoring (All ICU Patients)
- **ECG** (continuous) — arrhythmia, ischaemia
- **SpO₂** (continuous) — oxygenation
- **NIBP** / **Arterial line** (continuous BP) — beat-to-beat, ABG access
- **Temperature** (continuous)
- **Urine output** (hourly via catheter) — gold standard for renal perfusion
- **Fluid balance** (strict daily)

## 3. Invasive Monitoring (Selected Patients)
| Parameter | Indication | Method |
|-----------|------------|--------|
| **CVP** | Fluid responsiveness, RV function | Central line |
| **Arterial BP** | Beat-to-beat, vasoactive drugs, frequent ABG | Arterial line |
| **Cardiac Output** | Unclear shock type, guided therapy | PiCCO, LiDCO, echocardiography |
| **ScvO₂ / SvO₂** | Tissue oxygen extraction | Central line / PA catheter |
| **ICP** | Severe TBI, raised ICP | EVD / bolt |

## 4. Echocardiography (Bedside / Focused)
- **FICE / FATE** — focused assessment in emergency
- Assesses: LV/RV function, volume status, tamponade, PE, valve pathology

## 5. Haemodynamic Targets (General)
- MAP ≥65 mmHg (individualise)
- ScvO₂ ≥70% or SvO₂ ≥65%
- Lactate clearance ≥10–20% per 2 hours
- Urine output ≥0.5 mL/kg/hr

## 6. Red Flags
- Rising lactate despite "normal" BP
- Falling urine output
- New arrhythmias
- Rising ventilator pressures

## 7. FCPS/MRCP High-Yield Points
- Urine output is the **most sensitive** real-time marker of renal perfusion
- Arterial line is essential if on vasoactive drugs
- CVP is a **poor** predictor of fluid responsiveness — use dynamic measures (SVV, PPV, PLR)

## 9. One-Page Revision Summary
**Monitor: ECG, SpO₂, BP, Temp, Urine, Fluid balance**
Invasive: arterial line, CVP, CO if needed

## 10. Must Know / Should Know / Nice to Know
### Must Know
- Standard ICU monitoring (ECG, SpO₂, BP, Temp, UO, fluid balance)
- Arterial line indications (vasoactives, frequent ABG, beat-to-beat)
- ScvO₂/SvO₂ targets
- Urine output = most sensitive perfusion marker
- CVP poor fluid responsiveness predictor

### Should Know
- Dynamic fluid responsiveness measures (SVV, PPV, PLR)
- Cardiac output monitoring methods (PiCCO, echo, PA catheter)
- ICP monitoring indications

### Nice to Know
- Advanced haemodynamic monitoring (pulse contour, transpulmonary thermodilution)
- Regional perfusion monitoring (NIRS, gastric tonometry)
- Continuous cardiac output via arterial waveform analysis

## 13. Flashcards
- Q: Best perfusion marker
  A: Urine output (hourly) — most sensitive real-time
- Q: Arterial line indications
  A: Vasoactives, frequent ABG, beat-to-beat BP
- Q: ScvO₂ target septic shock
  A: ≥70%
- Q: CVP limitation
  A: Poor fluid responsiveness predictor — use dynamic (PLR, SVV/PPV)
- Q: Lactate clearance
  A: ≥10–20% per 2 hours
- Q: SvO₂ normal
  A: 65–75%
- Q: Dynamic fluid responsiveness
  A: PLR, SVV/PPV, mini-fluid challenge
- Q: FICE/FATE echo
  A: LV/RV function, volume, tamponade, PE

## 14. Answer Key with Explanations
**MCQ 1**: B — Urine output hourly = most sensitive real-time.
**MCQ 2**: B — Arterial line mandatory for vasoactives.
**MCQ 3**: C — ScvO₂ ≥70% in septic shock.
**MCQ 4**: B — CVP reflects pressure, not preload responsiveness.
**MCQ 5**: C — PLR is dynamic (gold standard).
**MCQ 6**: B — ≥10–20% per 2 hours.
**MCQ 7**: B — SvO₂ 65–75%.
**MCQ 8**: B — PiCCO = CO + EVLW + GEDV.
**MCQ 9**: C — UO ≥0.5 mL/kg/hr.
**MCQ 10**: A — FICE/FATE = focused assessment.

**SBA 1**: B — Rising lactate + low UO = inadequate perfusion despite "normal" MAP.
**SBA 2**: B — Arterial line mandatory on vasoactives.
**SBA 3**: A — SVV >13% = fluid responsive (if conditions met).
**SBA 4**: B — Low ScvO₂ = inadequate DO₂ (anaemia, low CO, high VO₂).
**SBA 5**: B — GCS ≤8 + abnormal CT = ICP monitor indication.
**SBA 6**: B — Assess responsiveness first, don't blindly fluid load.
**SBA 7**: B — PA catheter = PA pressure, PCWP, CO, SvO₂.
**SBA 8**: A — NIRS = regional tissue oxygenation.
**SBA 9**: B — High ScvO₂ + high lactate = cytopathic hypoxia (mitochondrial failure).
**SBA 10**: D — Coronary angiography not an echo view.
**SBA 11**: B — Neutral/slightly negative balance after resuscitation phase.

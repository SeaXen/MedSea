> [!important]
> Vasopressors = increase SVR/BP. Inotropes = increase contractility/CO. Some drugs do both. Choose based on shock physiology.

## 1. Learning Objectives
- Differentiate vasopressors, inotropes, and inodilators
- Know receptor profiles and clinical effects
- Choose first-line agent by shock type
- Know dosing, titration, and complications

## 2. Drug Summary Table
| Drug | Receptors | Primary Effect | Typical Dose | Main Use |
|------|-----------|----------------|--------------|----------|
| **Noradrenaline** | α₁ > β₁ | Vasoconstriction + mild inotropy | 0.01–0.5 µg/kg/min | **First-line septic / distributive** |
| **Adrenaline** | α₁ = β₁ = β₂ | Vasoconstriction + inotropy + chronotropy | 0.01–0.5 µg/kg/min | Septic shock refractory, anaphylaxis |
| **Dobutamine** | β₁ > β₂ | Inotropy + vasodilatation | 2.5–20 µg/kg/min | **Cardiogenic shock** (low output) |
| **Milrinone** | PDE III inhibitor | Inotropy + vasodilatation | 0.25–0.75 µg/kg/min | Cardiogenic shock, weaning support |
| **Vasopressin** | V₁ receptors | Vasoconstriction (non-catecholamine) | 0.01–0.04 units/min | Vasodilatory shock, catecholamine-sparing |
| **Phenylephrine** | α₁ pure | Vasoconstriction only | 0.5–5 µg/kg/min | Neurogenic shock, obstetrics |

## 3. Choice by Shock Type
| Shock Type | First-Line | Second-Line / Add-On |
|------------|------------|----------------------|
| **Septic / Distributive** | Noradrenaline | Vasopressin (add), adrenaline |
| **Cardiogenic** | Dobutamine | Noradrenaline (if hypotensive), milrinone |
| **Obstructive** | Noradrenaline | (Treat obstruction primarily) |
| **Anaphylactic** | **Adrenaline IM** | Noradrenaline IV if refractory |
| **Neurogenic** | Noradrenaline | Phenylephrine (if bradycardic) |

## 4. Key Practical Points
- **Arterial line mandatory** before starting (except adrenaline IM in anaphylaxis)
- **Central line preferred** for noradrenaline/adrenaline (extravasation necrosis)
- **Dobutamine causes hypotension** (vasodilatation) — often needs noradrenaline combo
- **Wean gradually** when MAP stable, lactate normalising, urine output adequate

## 5. Red Flags
- Peripheral extravasation → phentolamine infiltration
- Tachyarrhythmias (especially with adrenaline/dobutamine)
- limb ischaemia (vasoconstriction)

## 6. FCPS/MRCP High-Yield Points
- Noradrenaline = **first-line in septic shock**
- Dobutamine = **first-line in cardiogenic shock** (add noradrenaline if hypotensive)
- Vasopressin spares catecholamines
- Milrinone = avoid in hypotension (pure vasodilator + inotrope)

## 8. One-Page Revision Summary
**Noradrenaline** = septic/distributive (first-line)
**Dobutamine** = cardiogenic (first-line)
**Adrenaline** = anaphylaxis, refractory septic
**Vasopressin** = catecholamine-sparing add-on
**Milrinone** = cardiogenic (if noradrenaline/dobutamine inadequate)

## 9. Must Know / Should Know / Nice to Know
### Must Know
- Noradrenaline first-line septic/distributive
- Dobutamine first-line cardiogenic
- Adrenaline for anaphylaxis/refractory septic
- Vasopressin add-on (catecholamine-sparing)
- Arterial line mandatory (except adrenaline IM anaphylaxis)
- Central line preferred for noradrenaline/adrenaline

### Should Know
- Dobutamine causes hypotension (vasodilatation) → often needs noradrenaline combo
- Milrinone = avoid in hypotension (pure vasodilator + inotrope)
- Phenylephrine for neurogenic shock (if bradycardic)
- Weaning criteria: MAP stable, lactate normalising, UO adequate

### Nice to Know
- Angiotensin II for refractory vasoplegia
- Levosimendan (Ca²⁺ sensitiser) alternative in cardiogenic
- Dopamine — no longer recommended (arrhythmia risk)
- Fenoldopam, nitroglycerin, nitroprusside for afterload reduction

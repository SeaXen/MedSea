# Bronchiolitis and Obliterative Bronchiolitis

Related: [[Bronchiectasis]], [[COPD]], [[ILD/Hypersensitivity pneumonitis|Hypersensitivity pneumonitis]], [[Transplant/Bronchiolitis obliterans syndrome|Bronchiolitis obliterans syndrome]], [[Airway Diseases/Other obstructive and small-airway disorders|Other obstructive and small-airway disorders]]

> [!important]
> **Bronchiolitis** = inflammation of bronchioles (<2mm). **Obliterative bronchiolitis (OB)** = fibrotic obliteration of bronchiolar lumen → **fixed airflow obstruction**. **Post-infectious** (adenovirus, RSV), **post-transplant (BOS)**, **connective tissue disease**, **inhalational injury**, **drug-induced**. **Key FCPS/MRCP**: distinction from asthma/COPD, HRCT mosaic attenuation, constrictive vs proliferative, post-transplant BOS, management.

## 1. Learning Objectives
- Define bronchiolitis vs obliterative bronchiolitis (constrictive vs proliferative)
- Identify aetiologies (post-infectious, post-transplant, CTD, inhalational, drugs)
- Apply HRCT findings (mosaic attenuation, air trapping, tree-in-bud)
- Differentiate from asthma, COPD, bronchiectasis, HP
- Manage based on aetiology (steroids, immunosuppression, transplant)

## 2. Definitions
| Term | Definition |
|------|------------|
| **Bronchiolitis** | Inflammation of **bronchioles** (airways <2mm diameter, no cartilage) |
| **Acute bronchiolitis** | Viral (RSV, adenovirus) in infants — usually self-limiting |
| **Obliterative bronchiolitis (OB)** | **Fibrotic obliteration** of bronchiolar lumen → **fixed, irreversible airflow obstruction** |
| **Constrictive bronchiolitis (CB)** | **Concentric fibrosis** → luminal narrowing (most common OB pattern) |
| **Proliferative bronchiolitis (PB)** | **Intraluminal polyps** (Masson bodies) organising pneumonia pattern (COP/BOOP) |

> **FCPS/MRCP tip**: **OB = constrictive (fixed obstruction) vs PB = proliferative (organising pneumonia/COP)**. **Post-transplant OB = Bronchiolitis Obliterans Syndrome (BOS)**.

## 3. Classification by Aetiology

| Category | Causes | Key Features |
|----------|--------|--------------|
| **Post-infectious** | Adenovirus (esp. serotype 21), RSV, influenza, parainfluenza, mycoplasma, measles | Children; often unilateral (Swyer-James/MacLeod syndrome); bronchiectasis |
| **Post-transplant** | **Lung (BOS)**, HSCT (GVHD), solid organ | BOS = leading cause of late mortality post-lung transplant |
| **Connective Tissue Disease** | **RA**, Sjögren's, SLE, systemic sclerosis, MCTD | Often with ILD (NSIP/UIP); CB common |
| **Inhalational / Toxic** | **Nitrogen dioxide** (silo filler's disease), chlorine, ammonia, sulfur dioxide, welding fumes, popcorn worker's lung (diacetyl) | Acute exposure → delayed OB (2-6 weeks) |
| **Drug-Induced** | Penicillamine, gold, sulfasalazine, carbamazepine, 5-FU, bleomycin | Subacute onset; may improve on withdrawal |
| **Idiopathic** | **Cryptogenic constrictive bronchiolitis** | Rare; diagnosis of exclusion |
| **Post-radiation** | Thoracic RT | Fibrotic airway narrowing |

## 4. Pathological Patterns

| Pattern | Histology | Causes |
|---------|-----------|--------|
| **Constrictive (CB)** | **Concentric fibrosis** → luminal narrowing/obliteration; **no intraluminal polyps** | Post-infectious, post-transplant (BOS), CTD, inhalational, drugs, idiopathic |
| **Proliferative (PB) / BOOP / COP** | **Intraluminal buds of granulation tissue (Masson bodies)** extending into alveoli | Cryptogenic organising pneumonia (COP), CTD, post-infectious, drugs |

> **Key distinction**: **CB = fixed obstruction (irreversible)**; **PB = organising pneumonia (responsive to steroids)**.

## 5. Clinical Features

| Presentation | Typical Features |
|--------------|------------------|
| **Dyspnoea** | Progressive, exertional (often out of proportion to spirometry) |
| **Cough** | Dry or minimally productive |
| **Wheeze** | May be present (fixed wheeze) |
| **Exercise limitation** | Disproportionate to FEV₁ |
| **Crackles** | Early inspiratory (if associated ILD) |
| **Signs of cause** | CTD features, transplant history, exposure history |

### Syndrome-Specific
| Syndrome | Key Features |
|----------|--------------|
| **Post-infectious (Swyer-James/MacLeod)** | Unilateral hyperlucent lung, bronchiectasis, recurrent infections; often post-adenovirus |
| **Post-transplant BOS** | Progressive FEV₁ decline >20% from baseline; FEV₁/FVC <0.7; no other cause |
| **CTD-associated** | RA/Sjögren's/SLE; often with NSIP/UIP pattern ILD |
| **Inhalational (diacetyl)** | "Popcorn worker's lung" — fixed obstruction, popcorn factory workers |
| **Penicillamine/Gold** | Rheumatoid arthritis treatment; subacute onset |

## 6. Investigations

### Pulmonary Function Tests
| Pattern | Findings |
|---------|----------|
| **Spirometry** | **Irreversible obstructive** (FEV₁/FVC <0.7, no reversibility) |
| **Lung volumes** | **Air trapping**: ↑ RV, ↑ RV/TLC |
| **DLCO** | Usually **preserved** (unless concomitant ILD/emphysema) |
| **Flow-volume loop** | **Concave expiratory limb** (fixed obstruction); plateau |

### HRCT (Diagnostic Gold Standard)
| Finding | Significance |
|---------|--------------|
| **Mosaic attenuation** | **Patchy lucency** (air trapping) + dense areas = **hallmark** |
| **Expiratory air trapping** | **Lucent areas persist on expiration** (key diagnostic) |
| **Tree-in-bud** | Centrilobular nodules (if infectious/inflammatory component) |
| **Bronchial wall thickening** | Non-specific |
| **Bronchiectasis** | May coexist (post-infectious) |
| **Centrilobular emphysema** | Differential (smoking) |

> **HRCT Protocol**: **Inspiratory + Expiratory** slices — air trapping only visible on expiration!

### Bronchoscopy
- Usually **normal** airways (small airway disease)
- **BAL**: neutrophilia (if neutrophilic inflammation); exclude infection
- **Transbronchial biopsy**: limited yield (small airways); may show fibrosis

### Pulmonary Function — Serial Monitoring
- **BOS grading** (post-transplant): FEV₁ decline from baseline
  - BOS 0-p: FEV₁ 81-90% + FEF₂₅-₇₅ <75%
  - BOS 1: FEV₁ 66-80%
  - BOS 2: FEV₁ 51-65%
  - BOS 3: FEV₁ ≤50%

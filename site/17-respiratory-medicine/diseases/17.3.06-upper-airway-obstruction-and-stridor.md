# Upper airway obstruction and stridor

> [!important]
> **Stridor** is a **medical emergency** — indicates ≥50% airway narrowing. Causes: **anaphylaxis, foreign body, laryngeal oedema, tumour, croup, vocal cord dysfunction**. Inspiratory stridor = extrathoracic (laryngeal). Treatment depends on cause — **IM adrenaline** for anaphylaxis, **bronchoscopy** for foreign body, **intubation/tracheostomy** for severe.

Related: [[Asthma]], [[Foreign body aspiration in adults]], [[Vocal cord dysfunction and inducible laryngeal obstruction|VCD]], [[Wheeze and stridor approach]]

> [!tip] **FCPS/MRCP pearl**: **Stridor = upper airway emergency**. **Inspiratory** = extrathoracic (laryngeal). **Expiratory** = intrathoracic (trachea). **Biphasic** = fixed (subglottic stenosis, tumour). Always exclude **anaphylaxis** (IM adrenaline), **foreign body** (bronchoscopy), **tumour** (laryngoscopy + CT), **croup** (dexamethasone).

## 1. Definition

**Stridor** = monophonic harsh sound from **upper airway obstruction** (larynx, trachea).

### Types
- **Inspiratory stridor**: extrathoracic (laryngeal, subglottic) — narrows on inspiration
- **Expiratory stridor**: intrathoracic (tracheal) — narrows on expiration
- **Biphasic stridor**: fixed obstruction (subglottic stenosis, tracheal tumour)

## 2. Aetiology

### Acute
| Cause | Notes |
|-------|-------|
| **Anaphylaxis** | Acute, with urticaria, hypotension |
| **Foreign body** | Sudden, often children, unilateral wheeze |
| **Angioedema** (hereditary, ACE-i) | Facial swelling |
| **Croup** | Children 6 months–3 years, barking cough, viral |
| **Epiglottitis** | Children, toxic, drooling, sitting up |
| **Laryngospasm** | Post-extubation, post-thyroidectomy |
| **Bacterial tracheitis** | Severe, post-viral |
| **Retropharyngeal abscess** | Children, fever, drooling |
| **Diphtheria** | Unvaccinated, membrane |

### Subacute / chronic
| Cause | Notes |
|-------|-------|
| **Laryngeal tumour** | Smoker, progressive hoarseness, weight loss |
| **Bilateral vocal cord paralysis** | Post-thyroidectomy |
| **Subglottic stenosis** | Post-intubation/tracheostomy |
| **Tracheal stenosis** | Post-tracheostomy, tumour |
| **Laryngeal papillomatosis** | Children, HPV |
| **Tracheomalacia** | Floppy trachea, dynamic |
| **Goitre** | Compression |
| **TBM** | Mediastinal mass |
| **Lymphoma** | Mediastinal nodes |
| **VCD** | Functional, mimics asthma |

## 3. Pathophysiology

### Mechanism
- **Air flow through narrowed airway** → turbulence → stridor
- **Critical** (≥50% narrowing)
- **Inspiratory stridor**: extrathoracic (negative intraluminal pressure during inspiration narrows extrathoracic airway)
- **Expiratory stridor**: intrathoracic (positive intrathoracic pressure compresses intrathoracic airway)
- **Biphasic**: fixed (cartilage)

### Acute severe obstruction
- **Complete obstruction** → stridor → aphonia → respiratory arrest
- **Time-critical** (minutes to hours)

## 4. Clinical Features

### Symptoms
- **Noisy breathing** (stridor, wheeze)
- **Dyspnoea**
- **Hoarseness** (laryngeal)
- **Cough** (croupy, barking)
- **Drooling, dysphagia** (epiglottitis, abscess)
- **Anaphylaxis symptoms** (urticaria, angioedema)

### Signs
- **Stridor** (auscultate over neck/chest)
- **Drooling, tripod position** (epiglottitis)
- **Hoarseness**
- **Use of accessory muscles**
- **Tracheal tug, recession**
- **Cyanosis** (severe)
- **Tachycardia, tachypnoea**

## 5. Diagnosis

### Investigations
| Test | Purpose |
|------|---------|
| **ABG** | Hypoxaemia, hypercapnia (severe) |
| **CXR** (inspiration + expiration) | Foreign body, mass, mediastinal |
| **Neck X-ray (AP)** | Subglottic narrowing (steeple sign in croup) |
| **CT neck/chest** | Mass, abscess, foreign body |
| **Bronchoscopy** | Foreign body, tumour, biopsy |
| **Laryngoscopy** | Epiglottitis, vocal cords |
| **Spirometry (flow-volume loop)** | Extrathoracic vs intrathoracic obstruction |
| **Allergy tests** | Anaphylaxis triggers |

### Flow-volume loop
- **Variable extrathoracic**: flattening of **inspiratory** limb
- **Variable intrathoracic**: flattening of **expiratory** limb
- **Fixed**: flattening of both limbs

## 6. Management

### General
- **A–B–C** assessment
- **Position**: sitting up (avoid supine if epiglottitis)
- **O₂** (humidified)
- **Avoid distressing** the patient
- **Senior/anaesthesia/ENT** early
- **Keep child with parent**

### Specific
| Cause | Rx |
|-------|-----|
| **Anaphylaxis** | **IM adrenaline 0.5 mg (1:1000)** + IV fluids + antihistamine + steroids + intubate if needed |
| **Croup** | Dexamethasone (oral) + nebulised adrenaline if severe |
| **Epiglottitis** | **Anaesthesia + intubation** (in OR) + ceftriaxone + dexamethasone |
| **Foreign body** | **Bronchoscopic removal** (rigid preferred in children) |
| **Laryngeal oedema** | Adrenaline (nebulised), steroids, intubate |
| **Laryngeal tumour** | ENT evaluation, biopsy, oncology |
| **Bilateral VC paralysis** | Tracheostomy |
| **Subglottic stenosis** | Dilatation, stenting |
| **VCD** | Speech therapy, reassurance |

### Don'ts
- **Don't examine throat** (epiglottitis) — may cause complete obstruction
- **Don't lie child down** (epiglottitis)
- **Don't delay intubation** in impending obstruction

## 8. MCQs (10)

1. Inspiratory stridor indicates:
   A. **Extrathoracic (laryngeal) obstruction**
   B. Intrathoracic
   C. Lower airway
   D. None
   E. Both
   **Answer: A** — Extrathoracic.

2. Expiratory stridor indicates:
   A. Extrathoracic
   B. **Intrathoracic (tracheal)**
   C. Lower airway
   D. None
   E. Both
   **Answer: B** — Intrathoracic.

3. Biphasic stridor suggests:
   A. Asthma
   B. **Fixed (subglottic stenosis, tracheal tumour)**
   C. COPD
   D. None
   E. Pneumonia
   **Answer: B** — Fixed.

4. Anaphylaxis with stridor. First:
   A. Steroids
   B. **IM adrenaline 0.5 mg (1:1000)**
   C. Antihistamine
   D. Wait
   E. None
   **Answer: B** — IM adrenaline.

5. The "steeple sign" on neck X-ray suggests:
   A. **Croup**
   B. Epiglottitis
   C. Foreign body
   D. None
   E. Asthma
   **Answer: A** — Croup.

6. The "thumb sign" on lateral neck X-ray suggests:
   A. Croup
   B. **Epiglottitis (swollen epiglottis)**
   C. Foreign body
   D. None
   E. Asthma
   **Answer: B** — Epiglottitis.

7. The most common cause of stridor in adults:
   A. Croup
   B. **Laryngeal tumour (smoker) or laryngeal oedema**
   C. Epiglottitis
   D. None
   E. Foreign body
   **Answer: B** — Adult.

8. Foreign body in airway. Best:
   A. Antibiotics
   B. **Bronchoscopy (rigid preferred)**
   C. Steroids
   D. Wait
   E. None
   **Answer: B** — Bronchoscopy.

9. A 3-year-old with barking cough and stridor. Most likely:
   A. **Croup (viral)**
   B. Epiglottitis
   C. Asthma
   D. Foreign body
   E. None
   **Answer: A** — Croup.

10. The most common cause of post-extubation stridor:
    A. Croup
    B. **Laryngeal oedema**
    C. Anaphylaxis
    D. None
    E. Foreign body
    **Answer: B** — Laryngeal oedema.

## 9. SBA Questions (10)

1. The most useful investigation for chronic stridor is:
   A. **Laryngoscopy +/− bronchoscopy + CT**
   B. CXR
   C. Spirogram
   D. ABG
   E. None
   **Answer: A** — Direct visualisation.

2. A 60-year-old smoker with hoarseness and stridor. Best:
   A. Antibiotics
   B. **Laryngoscopy + CT (laryngeal cancer workup)**
   C. Wait
   D. Steroids
   E. None
   **Answer: B** — Laryngeal cancer.

3. Acute epiglottitis is treated with:
   A. Steroids alone
   B. **Anaesthesia + intubation + ceftriaxone**
   C. Antibiotics alone
   D. Wait
   E. None
   **Answer: B** — Airway + abx.

4. The most common cause of stridor in a child is:
   A. **Croup**
   B. Epiglottitis
   C. Foreign body
   D. None
   E. Tumour
   **Answer: A** — Croup.

5. The first-line treatment for severe croup:
   A. Antibiotics
   B. **Dexamethasone (oral) + nebulised adrenaline if severe**
   C. Theophylline
   D. Wait
   E. None
   **Answer: B** — Dexamethasone ± adrenaline.

6. Don'ts in suspected epiglottitis:
   A. Steroids
   B. **Examine throat (may cause complete obstruction)**
   C. Antibiotics
   D. None
   E. Wait
   **Answer: B** — Don't examine throat.

7. The most useful imaging for foreign body:
   A. **CXR (inspiration + expiration) ± CT**
   B. ABG
   C. Spirogram
   D. None
   E. Echo
   **Answer: A** — CXR ± CT.

8. The flow-volume loop in fixed obstruction shows:
   A. **Flattening of both inspiratory and expiratory limbs**
   B. Normal
   C. Restrictive
   D. None
   E. Variable
   **Answer: A** — Both.

9. Anaphylaxis with airway oedema requires:
   A. Wait
   B. **IM adrenaline + intubation (if needed)**
   C. Steroids
   D. Antibiotics
   E. None
   **Answer: B** — Adrenaline + airway.

10. A 50-year-old with bilateral vocal cord paralysis causes:
    A. Hoarseness
    B. **Stridor + respiratory failure**
    C. None
    D. Asthma
    E. Wheeze
    **Answer: B** — Stridor + RF.

## 12. Summary

Stridor = ≥50% airway narrowing; emergency. Inspiratory = extrathoracic (laryngeal); expiratory = intrathoracic (trachea); biphasic = fixed. Anaphylaxis: IM adrenaline. Croup: dexamethasone. Epiglottitis: anaesthesia + intubation + ceftriaxone. Foreign body: bronchoscopy. Don't examine throat in epiglottitis.

> [!tip] **FCPS/MRCP Priority: HIGH**
> **Bladder cancer = Urothelial carcinoma (90%)**; **NMIBC (Ta/Tis/T1) = 75%** vs **MIBC (T2-4) = 25%**. **NMIBC**: TURBT → Risk stratification (EAU/EORTC) → **Low/Intermediate: Single TURBT ± Chemo instillation**; **High: BCG induction + Maintenance** (SWOG 8507). **BCG-unresponsive** → **ICI** (Pembro/Nivo) or **Enfortumab vedotin** or **Radical Cystectomy**. **MIBC**: **Neoadjuvant Cisplatin chemo (GC/DD-MVAC)** → **Radical Cystectomy + PLND** (+ Adjuvant Nivo if high risk). **Metastatic**: **Cisplatin-eligible: GC/DD-MVAC** → **Maintenance Avelumab (JAVELIN Bladder 100)**; **Cisplatin-ineligible: ICI (Pembro/Atezo) or Gem/Carbo** → **2L: Enfortumab vedotin (EV-301), Erdafitinib (FGFR3), Sacituzumab govitecan**.

## 5. 5. Staging & Classification

| System | Detail |
|--------|--------|
| **TNM 8th Edition** | **Ta**: Non-invasive papillary; **Tis**: CIS (flat high-grade); **T1**: Lamina propria; **T2**: Muscularis propria (T2a inner half, T2b outer half); **T3**: Perivesical fat (T3a micro, T3b macro); **T4**: Adjacent organs (prostate, uterus, vagina, pelvic wall) |
| **N Stage** | N1: Single regional node; N2: Multiple regional; N3: Common iliac nodes |
| **M Stage** | M1a: Non-regional nodes; M1b: Distant mets |
| **Grade (WHO 2016/2022)** | **Low-grade** (PUNLMP → LGUC); **High-grade** (HGUC) — **Replaces 1973 Grade 1/2/3** |
| **Risk Stratification (EAU/EORTC)** | **Low**: Single, primary, Ta, LG, <3cm, no CIS
**Intermediate**: Not low, not high (multiple, recurrent, >3cm, Ta LG, T1 LG)
**High**: **Any T1 HG, Any CIS, Any Ta HG, Multiple/large/recurrent HG** |

## 6. 6. Diagnosis & Investigations

| Investigation | Role | Key Findings |
|---------------|------|--------------|
| **Cystoscopy** | **Gold standard** — Visualisation, TURBT | White light + **Narrow Band Imaging (NBI) / Blue Light (Cysview)** for CIS detection |
| **TURBT** | **Diagnosis + Staging + Treatment** | **Must include detrusor muscle** (except Ta LG); Re-TURBT if T1 or incomplete |
| **Urine Cytology** | Adjunct for CIS/High-grade | Sensitivity: CIS 80%, HG 50%, LG <10% |
| **Urine Biomarkers** | NMP22, UroVysion FISH, Cxbladder | Not standard; Adjunct for surveillance |
| **CT Urogram / MRI** | Staging (MIBC/Metastatic), Upper tract | **CTU**: Excretory phase for urothelial enhancement; **MRI**: Local staging if CT equivocal |
| **Chest CT** | Pulmonary mets | Standard for MIBC/Metastatic |
| **Bone Scan / PET-CT** | Symptomatic bone; PET for equivocal | Not routine for NMIBC |
| **Biomarkers (Metastatic)** | **FGFR3 mut/fusion**, **PD-L1**, **TMB**, **MSI**, **HER2** | Guide targeted/ICI therapy |

| Condition | Distinguishing Features |

## 11. 11. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **NMIBC vs MIBC — T1 is NMIBC** | **T1 = Lamina propria** (NMIBC); **T2 = Muscularis propria** (MIBC) — Critical distinction |
| **BCG Induction vs Maintenance** | **Induction**: 6 weekly; **Maintenance**: 3 weekly at 3,6,12,18,24,30,36mo (SWOG 8507) — **Total 27** |
| **BCG-Unresponsive vs BCG-Relapsing** | **Unresponsive** = High-risk recurrence after **adequate** BCG → Cystectomy/ICI; **Relapsing** = Recurrence after **inadequate** BCG → Can retry BCG |
| **GC vs DD-MVAC** | **GC**: q3w, less toxic, **preferred**; **DD-MVAC**: q2w + G-CSF, higher pCR, more toxic |
| **Adjuvant Nivo vs ICI in Metastatic** | **Adjuvant Nivo**: Post-cystectomy for **ypT1-4/ypN+** (CheckMate 274); **Metastatic ICI**: 1L cisplatin-ineligible or Maintenance Avelumab |
| **FGFR3 mut vs FGFR3 fusion** | Both sensitive to **Erdafitinib**; Fusion more common in upper tract |

**Mnemonic: BLADDER**
- **B**CG: Induction 6wk → **Maintenance 3wk at 3,6,12,18,24,30,36mo**
- **L**ow/Int/High: EAU/EORTC risk tables
- **A**dequate BCG: ≥5 ind + ≥2 maint (or ≥2 ind + ≥1 maint)
- **D**etrusor muscle in TURBT (mandatory for T1)
- **D**D-MVAC vs GC: **GC preferred** neoadjuvant
- **E**nfortumab: **Nectin-4 ADC** (2L post-PD-1+Plat)
- **R**adical Cystectomy: Extended PLND + Diversion

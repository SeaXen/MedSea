## 2. 2. PSA Testing — Informed Choice Programme

| Aspect | Detail |
|--------|--------|
| **Status** | **Informed Choice Programme** (Men >50 can request PSA) |
| **Test** | **Serum PSA** (Age-specific cutoffs) |
| **Delivery** | **GP Discussion → Informed Decision → PSA if Wanted** |
| **Not Population Programme** | **No Automatic Invitation** (Unlike Breast/Cervical/Colorectal/Lung) |

### 1. Age-Specific PSA Cutoffs (NICE)
| Age (Years) | PSA Cutoff (ng/mL) |
|-------------|-------------------|
| **<50** | **<2.5** |
| **50-59** | **<3.0** |
| **60-69** | **<4.0** |
| **70-79** | **<5.0** |
| **≥80** | **<6.5** |

> **PSA > Age-Specific Cutoff → Urgent Urology Referral** (2-week wait target)

---

## 3. 3. Key Trial Evidence

### 1. ERSPC (European Randomized Study of Screening for Prostate Cancer)
| Aspect | Detail |
|--------|--------|
| **Design** | RCT, **182,000** men, **PSA Screening vs Control** |
| **Follow-up** | **13 years** (Extended to 16yr) |
| **Result** | **20% Prostate Cancer Mortality Reduction** (RR 0.80) |
| **NNS** | **570** (Number Needed to Screen) |
| **NNT** | **18** (Number Needed to Treat to prevent 1 death) |
| **Overdiagnosis** | **~50%** of screen-detected cancers |

### 2. PLCO (Prostate, Lung, Colorectal, Ovarian) Trial
| Aspect | Detail |
|--------|--------|
| **Design** | RCT, **76,000** men, **Annual PSA + DRE vs Usual Care** |
| **Follow-up** | **13 years** |
| **Result** | **No Mortality Benefit** (HR 1.09, NS) |
| **Contamination** | **~40-50% Control Arm Had PSA Testing** |
| **Key Difference** | **Lower Contamination in ERSPC** → Clearer Benefit |

## 4. 4. MRI Pathway (mpMRI + PI-RADS)

### 1. Standard Pathway (NICE NG131)
```mermaid
flowchart TD
    A[PSA > Age-Specific Cutoff] --> B[**mpMRI (PI-RADS)**]
    B --> C{PI-RADS Score}
    C -->|**1-2**| D[**No Biopsy** (Consider Repeat PSA in 6-12mo)]
    C -->|**3**| E[**Discuss Biopsy** (Risk Stratification, Patient Choice)]
    C -->|**4-5**| F[**Targeted Biopsy** (MRI-US Fusion or Cognitive)]
    F --> G[**Diagnosis & Staging**]
```

### 2. PI-RADS v2.1 Scoring
| Score | Likelihood of csPCa | Management |
|--------|---------------------|------------|
| **1** | **Very Low** (<5%) | **No Biopsy**, Repeat PSA/mpMRI |
| **2** | **Low** (<10%) | **No Biopsy**, Repeat PSA/mpMRI |
| **3** | **Intermediate** (10-20%) | **Discuss Biopsy** (Shared Decision) |
| **4** | **High** (40-70%) | **Targeted Biopsy Recommended** |
| **5** | **Very High** (>90%) | **Targeted Biopsy Strongly Recommended** |

## 5. 5. ERSPC vs PLCO — Key Differences

| Aspect | **ERSPC** | **PLCO** |
|--------|-----------|----------|
| **Design** | **PSA every 4yr** (Core age 55-69) | **Annual PSA + DRE** (Age 55-74) |
| **Control Arm** | **No Screening** | **Usual Care (40-50% Contamination)** |
| **Mortality Result** | **20% Reduction (RR 0.80)** | **No Benefit (HR 1.09)** |
| **Contamination** | **Low (~15%)** | **High (40-50%)** |
| **Overdiagnosis** | **~50%** | **High** |

> **Key**: **ERSPC Benefit Driven by Low Contamination**; **PLCO Null Result Likely Due to Contamination**

---

## 6. 6. Overdiagnosis & Lead Time Bias

| Concept | Explanation |
|---------|-------------|
| **Overdiagnosis** | **Detection of cancers that would never cause symptoms/death** (e.g., Gleason 6, Low Volume) |
| **Estimate** | **~50% of screen-detected cancers** (ERSPC) |
| **Lead Time Bias** | **Earlier diagnosis artificially inflates survival time** without changing time of death |
| **Length Time Bias** | **Slow-growing tumours more likely detected** → Better prognosis in screened group |
| **Gleason 6** | **Grade Group 1** — **Never Metastasises**, **Overdiagnosis** if detected by screening |

## 9. 9. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **ERSPC vs PLCO** | **ERSPC: 20% Mortality ↓, Low Contamination**; **PLCO: No Benefit, High Contamination (40-50%)** |
| **mpMRI vs TRUS Biopsy** | **mpMRI First**: PI-RADS Guides Targeted Biopsy; **TRUS Biopsy: Random, Misses Anterior Cancers** |
| **PI-RADS 3 vs 4** | **3: Intermediate (Discuss)**; **4: High (Targeted Biopsy Recommended)** |
| **Overdiagnosis vs False Positive** | **Overdiagnosis**: True Cancer, Indolent; **False Positive**: No Cancer, Test +ve |
| **PSA Velocity** | **Not Recommended for Screening** (NICE), **Used for Risk Stratification Post-Dx** |
| **PSA Density** | **PSA/Prostate Volume**, **>0.15 ng/mL/cm³ → Higher Risk**, **Used Post-mpMRI** |

**Mnemonic: PSA-SCREENING**
- **P**SA: **Informed Choice** (Not Population Programme)
- **S**creening Age: **>50 Can Request**
- **A**ge Cutoffs: **<50: 2.5, 50-59: 3.0, 60-69: 4.0, 70-79: 5.0, ≥80: 6.5**
- **S**creening Trials: **ERSPC 20% ↓ (Low Contam), PLCO No Benefit (High Contam)**
- **C**hoosing mpMRI: **PI-RADS 1-2 No Bx, 3 Discuss, 4-5 Bx**
- **R**isk Factors: **Age, FH, Black Ethnicity, BRCA2**
- **E**arly Detection: **mpMRI Pathway (PI-RADS) Before Biopsy**
- **E**vidence: **ERSPC 20% ↓ (NNS 570, NNT 18)**
- **E**arly Detection: **Overdiagnosis ~50% (Gleason 6)**
- **N**o Population Programme: **Informed Choice Only**
- **G**leason 6: **Grade Group 1**, Never Metastasises, Active Surveillance
- **A**t Risk: **Black Men, FH, BRCA2**
- **N**on-Screening: **PSA Velocity Not Recommended**
- **Diagnosis Pathway**: **PSA → mpMRI (PI-RADS) → Targeted Bx**

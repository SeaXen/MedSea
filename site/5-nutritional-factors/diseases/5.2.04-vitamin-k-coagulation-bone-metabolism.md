## 2. 2. Definitions / Key Concepts

| Term | Definition |
|------|------------|
| **Phylloquinone (K1)** | Plant form (green leafy veg, plant oils); major dietary source (~90% intake); side chain: phytyl (saturated) |
| **Menaquinones (K2, MK-n)** | Bacterial/animal forms; side chain: unsaturated isoprenoid (n repeats); MK-4 (animal tissues), MK-7 (natto, fermented foods) |
| **Menadione (K3)** | Synthetic naphthoquinone; no side chain; water-soluble; toxic (haemolysis, kernicterus) — not used therapeutically |
| **γ-Glutamyl Carboxylase (GGCX)** | Vitamin K-dependent enzyme; carboxylates Glu → γ-carboxyglutamate (Gla) on target proteins; requires CO₂, O₂, vitamin K hydroquinone |
| **Gla Residue** | γ-carboxyglutamic acid; binds Ca²⁺ via dicarboxylic acid; essential for protein function (clotting, bone, vascular) |
| **VKORC1** | Vitamin K epoxide reductase complex subunit 1; recycles vitamin K epoxide → quinone → hydroquinone (active cofactor); warfarin target |
| **Vitamin K Cycle** | KH₂ (hydroquinone) → GGCX (Glu→Gla) → KO (epoxide) → VKORC1 (KO→KH₂) → repeat |
| **Phytomenadione** | Synthetic K1 analogue; used for prophylaxis/treatment (IM, IV, PO); Konakion MM (mixed micellar) for IV |
| **HDN (Haemorrhagic Disease of Newborn)** | Classical (day 1–7), Late (2–12w, exclusively breastfed, cholestasis); prevented by 1 mg IM phytomenadione at birth |
| **Osteocalcin (Bone Gla Protein, BGP)** | Osteoblast product; γ-carboxylated (3 Gla); binds hydroxyapatite; serum = bone formation marker; uncarboxylated (ucOC) = vit K status |
| **Matrix Gla Protein (MGP)** | Vascular smooth muscle/chondrocyte product; γ-carboxylated (5 Gla); inhibits vascular calcification (binds Ca crystals); MGP-knockout → arterial calcification |
| **Protein C & S** | Vitamin K-dependent anticoagulants; Protein C (activated by thrombomodulin-thrombin) → inactivates Va/VIIa; Protein S = cofactor |
| **PT/INR** | Prothrombin time; extrinsic pathway (factor VII); most sensitive to vit K deficiency (t½ 6h); INR standardised |
| **aPTT** | Activated partial thromboplastin time; intrinsic pathway; normal in early vit K deficiency, prolonged if severe |
| **PIVKA-II** | Protein Induced by Vitamin K Absence-II (uncarboxylated prothrombin); sensitive marker of vit K deficiency |
| **Superwarfarins** | Brodifacoum, bromadiolone, difenacoum; long-acting rodenticides; inhibit VKORC1 irreversibly; half-life days–weeks; treatment: high-dose phytomenadione weeks–months |
| **Coumarin Resistance** | VKORC1 mutation (e.g., VKORC1 -1639G>A) → reduced warfarin sensitivity; requires higher dose |

## 3. 3. Core Content

### 1. Section 1: Forms, Sources & Absorption
**Vitamin K1 (Phylloquinone):** Green leafy vegetables (spinach, kale, broccoli, Brussels sprouts), plant oils (soybean, canola). Contributes ~90% dietary intake. Bioavailability: 5–10% from vegetables (matrix-bound), higher with fat/mechanical disruption. Absorption: mixed micelles → enterocyte → chylomicron.
**Vitamin K2 (Menaquinones MK-n):** Bacterial synthesis (gut microbiota: MK-6 to MK-13), animal tissues (MK-4 from menadione in feed), fermented foods (natto = MK-7, cheese = MK-8/MK-9). MK-4 in meat, dairy, eggs. MK-7 (natto) — long half-life (72h vs K1 1–2h), better extrahepatic distribution.
**Menadione (K3):** Synthetic, water-soluble; converts to MK-4 in tissues; toxic (haemolysis, kernicterus in neonates) — NOT used clinically.
**Absorption:** Requires bile salts, pancreatic lipase; chylomicron secretion; lymphatic transport. K1: hepatic uptake via chylomicron remnant; MK-7: longer circulation, extrahepatic delivery.

### 2. Section 2: γ-Glutamyl Carboxylase & Vitamin K Cycle
**Reaction:** Glu (target protein) + CO₂ + O₂ + KH₂ (vitamin K hydroquinone) → Gla + KO (vitamin K epoxide) + H₂O.
**Enzyme:** γ-glutamyl carboxylase (GGCX); ER membrane-bound; processive (multiple Glu per substrate).
**Vitamin K Cycle:**
1. **KH₂ (hydroquinone)** = active cofactor for GGCX
2. **GGCX** uses KH₂ → carboxylates Glu → Gla → produces **KO (epoxide)**
3. **VKORC1** reduces KO → **K (quinone)** → **KH₂ (hydroquinone)** using NAD(P)H
4. **Warfarin** inhibits VKORC1 → KO accumulates → KH₂ depleted → ↓γ-carboxylation

**Target proteins (17 human Gla-proteins):**
- **Coagulation:** Factors II (prothrombin, 10 Gla), VII (10 Gla), IX (12 Gla), X (11 Gla)
- **Anticoagulation:** Protein C (9 Gla), Protein S (11 Gla), Protein Z
- **Bone:** Osteocalcin (BGP, 3 Gla), Matrix Gla Protein (MGP, 5 Gla)
- **Other:** Gas6, PRGP1/2, TMG3/4, periostin

### 3. Section 3: Warfarin Pharmacology
**Mechanism:** Competitive inhibitor of VKORC1 (binds epoxide binding site); prevents KO → KH₂ recycling.
**Onset:** Delayed (factor half-lives): Factor VII t½ 6h (PT ↑ first, 12–24h), IX t½ 24h, X t½ 40h, II t½ 60h; full effect 3–5 days.
**Monitoring:** INR (target therapeutic range: 2.0–3.0 most indications, 2.5–3.5 mechanical mitral valve).
**Dosing:** Genetic (CYP2C9 *2/*3 poor metaboliser, VKORC1 -1639G>A low expression), age, weight, diet, interacting drugs.
**Reversal:** Vitamin K1 (phytomenadione) 1–10 mg IV/PO (IV: faster, anaphylaxis risk <0.1%); PCC (4-factor) for urgent; FFP if PCC unavailable.
**Interactions:** 
- **↑INR:** Antibiotics (↓gut flora K2), amiodarone, metronidazole, TMP-SMX azoles, PPIs, cranberry, NSAIDs
- **↓INR:** Enzyme inducers (rifampicin, carbamazepine, phenytoin, barbiturates, St John's wort), vitamin K-rich foods

### 4. Section 4: Vitamin K Deficiency
**Adults (rare):**
- **Malabsorption:** Coeliac, Crohn's, CF, biliary obstruction, short bowel, bariatric
- **Antibiotics:** Broad-spectrum (3rd gen cephalosporins, carbapenems) → ↓gut flora MK synthesis
- **Warfarin:** Intentional iatrogenic deficiency
- **TPN without vitamin K:** Weeks without supplementation
- **Severe malnutrition**

**Neonates (HDN):**
- **Physiological:** Low placental transfer, sterile gut, low breast milk K1 (1–4 µg/L vs formula 50–100 µg/L)
- **Classical HDN:** Day 1–7; bleeding (umbilical, GI, circumcision, intracranial)
- **Late HDN:** 2–12 weeks; exclusively breastfed; cholestasis, malabsorption, antibiotics; **intracranial haemorrhage 30–50%**
- **Prevention:** **1 mg phytomenadione IM at birth** (universal); oral alternative: 2 mg at birth, 2 mg day 4–7, 2 mg week 4 (less reliable)

**Clinical features:** Bleeding (ecchymosis, haematuria, GI bleed, intracranial), ↑PT/INR (early), ↑aPTT (late), normal platelet count, normal fibrinogen, ↑PIVKA-II.

### 5. Section 5: Treatment of Deficiency
| Scenario | Phytomenadione Dose | Route | Notes |
|----------|---------------------|-------|-------|
| **HDN Prophylaxis** | 1 mg | IM (single) | Gold standard; Konakion MM paediatric |
| **HDN Prophylaxis (Oral)** | 2 mg ×3 | PO (birth, day 4–7, week 4) | If parents refuse IM; lower compliance |
| **Bleeding HDN / Late HDN** | 1 mg | IV/PO (IV faster) | Repeat if INR not corrected; FFP/PCC if life-threatening |
| **Warfarin Overdose (No Bleeding, INR 5–9)** | 1–2 mg | PO | Hold warfarin; restart lower dose when INR therapeutic |
| **Warfarin Overdose (Bleeding / INR >9)** | 5–10 mg | IV (slow) + PCC/FFP | PCC preferred (4-factor); monitor INR q6h |
| **Malabsorption / Cholestasis** | 10–20 mg | PO/IM weekly | Water-miscible preparation preferred |
| **Antibiotic-associated** | 5–10 mg | PO/IV | Stop antibiotic if possible; supplement during course |

### 6. Section 6: Bone & Vascular Metabolism
**Osteocalcin (BGP):** Osteoblast product; 3 Gla residues; γ-carboxylation → binds hydroxyapatite (Ca²⁺); **serum osteocalcin = bone formation marker**; uncarboxylated osteocalcin (ucOC) ↑ with vit K deficiency; total osteocalcin reflects turnover.
**Matrix Gla Protein (MGP):** Vascular smooth muscle cells, chondrocytes; 5 Gla; **potent inhibitor of vascular calcification** (binds Ca-phosphate crystals, inhibits BMP-2); MGP-knockout mice → severe arterial calcification, death by 6w.
**Vitamin K & Bone Health:** Observational: low K intake → ↑fracture risk, ↓BMD; RCTs: K1 1–5 mg/day or MK-4 45 mg/day (Japan) → ↓fracture (hip/vertebral); MK-7 (natto) 180 µg/day → ↓ucOC, ↑BMD.
**Vascular Calcification:** MGP requires γ-carboxylation to inhibit calcification; warfarin → ↓MGP carboxylation → ↑vascular calcification (animal models; human observational).
**MK-7 vs K1:** MK-7 (natto) longer half-life, better extrahepatic distribution (bone, vessel), more potent at carboxylating osteocalcin/MGP at lower doses.

### 7. Section 7: Special Forms & Analogues
| Compound | Use | Notes |
|----------|-----|-------|
| **Phytomenadione (K1)** | Standard prophylaxis/treatment | IM, IV (slow), PO; Konakion MM (mixed micellar) for IV |
| **Menadione (K3)** | NOT used | Haemolysis, kernicterus, GSH depletion |
| **Menaquinone-4 (MK-4)** | Osteoporosis (Japan, 45 mg/day) | High dose; short half-life |

## 6. 6. Common Confusions / Exam Traps

| Trap | Correction |
|------|------------|
| aPTT rises first in vit K deficiency | **PT/INR rises FIRST** (factor VII t½ 6h); aPTT late (factors IX, X, II) |
| Factor II (prothrombin) half-life is shortest | **Factor VII t½ 6h is shortest**; Factor II t½ 60h longest |
| IM vit K safe on warfarin | **IM CONTRAINDICATED** (haematoma risk); use IV/PO |
| Warfarin works immediately | **Delayed 3–5 days** (factor half-lives); bridge with heparin if urgent |
| HDN only in first week | **Late HDN: 2–12 weeks** (exclusive breastfeeding + cholestasis/malabsorption); ICH risk |
| Osteocalcin = vit K status marker | **Total osteocalcin = bone formation; ucOC/total OC = vit K status** |
| MK-7 = K1 | **MK-7 (natto) longer half-life (72h vs 1–2h), extrahepatic, better bone carboxylation** |
| Superwarfarin = warfarin dose | **Superwarfarin: massive K1 (50–100 mg/day) for WEEKS–MONTHS**; VKORC1 irreversible inhibition |
| Coumarin resistance = non-compliance | **VKORC1 mutation (-1639G>A) → true resistance**; check genotype |
| Vitamin K reverses warfarin instantly | **IV K1: 6–12h for factor synthesis; PCC for immediate reversal** |

## 7. 7. Mnemonics

- **K forms:** **K1 = PLANTS** (phylloquinone), **K2 = BACTERIA/ANIMALS** (menaquinones MK-n), **K3 = SYNTHETIC TOXIC** (menadione)

- **VKORC1:** **RECYCLES** KO → KH₂; **WARFARIN INHIBITS** VKORC1
- **Coagulation factors:** **1972** = **1**0 Gla (X), **9** Gla (VII), **7** (IX), **2** (II) — wait: **II=10, VII=10, IX=12, X=11** → **Ten-Ten-Twelve-Eleven**

- **Factor half-lives:** **VI**I **S**IX **T**EN **T**WO = **7, 6, 24, 40, 60** (VII 6h, IX 24h, X 40h, II 60h) — shortest VII
- **HDN:** **CLASSICAL = Day 1–7**; **LATE = 2–12 weeks** (breastfed + cholestasis) → **ICH**
- **PT first:** **P**T = **P**rothrombin (factor VII) **T**est = extrinsic; **S**hortest t½
- **Warfarin reversal:** **PCC = I**mmediate (factors); **K1 = S**ustained (days); **Bleeding = B**OTH
- **Bone:** **Osteocalcin = B**one formation marker; **ucOC = u**ncarboxylated = **vit K status**
- **MGP:** **M**atrix **G**la **P**rotein = **M**ajor **G**uard against **P**athologic calcification
- **Superwarfarin:** **BRODIFACOUM** = **B**ig dose K1, **R**eally long (weeks-months), **O**ral/IV, **D**aily, **I**NR monitor, **F**or **A**ntidote, **C**ontinue, **O**n and **U**ntil normal, **M**onths

### 1. Long Answer / Essay (20 min)

- Forms: K1 (phylloquinone, plants), K2 (menaquinones MK-n, bacteria/animal), K3 (menadione, toxic)

- MK-7: longer half-life, extrahepatic distribution, better osteocalcin/MGP carboxylation

- **Physiology:** Low placental transfer, sterile gut, low breast milk K1 (1–4 µg/L)
- **Classical HDN:** Day 1–7; bleeding (umbilical, GI, circumcision, intracranial); prevented by IM prophylaxis
- **Late HDN:** 2–12 weeks; exclusively breastfed + cholestasis/malabsorption/antibiotics; **ICH 30–50% mortality**
- **Prophylaxis:** **1 mg phytomenadione IM at birth** (universal); Oral alternative: 2 mg birth, day 4–7, week 4 (less reliable)
- **Treatment:** 1 mg phytomenadione IV/PO; repeat if INR not corrected; FFP/PCC if life-threatening bleeding

### 3. Viva Answer (3 min)
**Q:** "Why does PT/INR rise before aPTT in vitamin K deficiency?"
"A: **Factor VII has the shortest half-life (6 hours)** among vitamin K-dependent factors. PT tests the extrinsic pathway (factor VII). aPTT tests intrinsic pathway (factors VIII, IX, XI, XII) — factor IX t½ 24h, factor X t½ 40h, factor II t½ 60h. Therefore PT/INR prolongs within 12–24h of warfarin/deficiency, aPTT only later when factors IX, X, II fall."

### 4. Ward Case Discussion (5 min)
**Case:** 4w exclusively breastfed infant, biliary atresia (acholic stools, jaundice), sudden seizure, bulging fontanelle, INR 4.5.
"Diagnosis: **Late HDN** (breastfed + cholestasis → fat malabsorption → vit K deficiency → ICH). **Immediate: Phytomenadione 1 mg IV** (IM contraindicated in coagulopathy); **cranial CT** for ICH; **neurosurgery** if surgical lesion; **FFP/PCC** if active bleeding; **water-miscible vit K 5 mg PO daily** ongoing; **biliary diversion surgery** (Kasai) for definitive treatment."

### 5. Last-Night-Before-Exam Sheet (1 min)
- **Forms:** K1 (phylloquinone, plants), K2 (menaquinones MK-n, bacteria/animal), K3 (toxic)
- **Cycle:** KH₂ → GGCX (Glu→Gla) → KO → VKORC1 (recycles) → KH₂; Warfarin inhibits VKORC1
- **Targets:** II, VII, IX, X (coag); C, S, Z (anticoag); Osteocalcin, MGP (bone/vascular)
- **Factor t½:** VII 6h (PT first), IX 24h, X 40h, II 60h
- **HDN:** Classical day 1–7; Late 2–12w breastfed+cholestasis → ICH; Prophylaxis 1 mg IM birth
- **Warfarin reversal:** Bleed = PCC + K1 5–10 mg IV; Minor = K1 1–2 mg PO
- **Bone:** Osteocalcin (formation), ucOC (vit K status), MGP (vascular calcification inhibitor)
- **MK-7 (natto):** t½ 72h, extrahepatic, better bone/vessel carboxylation
- **Superwarfarin:** K1 50–100 mg/day weeks–months; Coumarin resistance: VKORC1 mutation

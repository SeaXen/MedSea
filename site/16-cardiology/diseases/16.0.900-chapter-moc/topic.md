## Key Algorithms & Scoring Systems (Memorise)

### ACS
| System | Use Case | Key Thresholds |
|--------|----------|----------------|
| **GRACE Score** | ACS 6-month mortality | >140 = high risk (early invasive) |
| **TIMI Score** (UA/NSTEMI) | 14-day adverse events | ≥5 = high risk |
| **HEART Score** | Chest pain risk stratification | 0-3 low, 4-6 moderate, ≥7 high |
| **TIMI Score** (STEMI) | 30-day mortality | ≥5 = high risk |

### Heart Failure
| System | Use Case | Key Thresholds |
|--------|----------|----------------|
| **NYHA Class** | Functional capacity | I-IV |
| **ACC/AHA Stages** | Disease progression | A (at risk) → D (refractory) |
| **Seattle Heart Failure Model** | 1-3 year survival | Prediction |
| **MAGGIC** | HF mortality | <10 low, 10-20 moderate, >20 high |
| **HFA-PEFF / H2FPEF** | HFpEF diagnosis | ≥5 = definite |
| **SCAI Shock Stages** | Cardiogenic shock severity | A (at risk) → E (extremis) |

### Valvular
| System | Use Case | Key Thresholds |
|--------|----------|----------------|
| **AS Severity** | AS grading | Peak V >4 m/s, MG >40, AVA <1.0 cm² |
| **AR Severity** | AR grading | VC >0.6 cm, RV >60%, PHT <200 ms |
| **MS Severity** | MS grading | MVA <1.5 cm² (severe), <1.0 (very severe) |
| **MR Severity** | MR grading | VC >0.7 cm, EROA >0.4 cm², RV >60 mL |

### Arrhythmias
| System | Use Case | Key Thresholds |
|--------|----------|----------------|
| **CHA2DS2-VASc** | AF stroke risk | ≥2 (M) / ≥3 (F) = anticoagulate |
| **HAS-BLED** | Bleeding risk on anticoag | ≥3 = high risk |
| **HCM Risk-SCD** | HCM sudden death risk | ≥6% 5-yr = ICD |
| **ESC Risk Table (PH)** | PAH prognosis | Low/Intermediate/High |
| **SCAI Shock Stages** | Cardiogenic shock | A (at risk) - E (extremis) |

### Infective Endocarditis
| System | Use Case | Key Thresholds |
|--------|----------|----------------|
| **Duke Criteria** | IE diagnosis | 2 Major / 1 Major+3 Minor / 5 Minor = Definite |

## High-Yield Viva Topics (Definitions, Doses, Durations)

| Topic | Expected Answer Points |
|-------|------------------------|
| **STEMI vs NSTEMI ECG** | ST elevation ≥1mm (2mm V2-3) in ≥2 contiguous leads vs ST depression/T-wave inversion |
| **STEMI Management** | Primary PCI <90 min, fibrinolysis if PCI >120 min, DAPT 12 months |
| **NSTEMI Management** | Early invasive ≤24h (high risk), DAPT 12 months, GDMT |
| **DAPT Duration** | ACS: 12 months; Stable CAD post-PCI: 6 months; High bleeding risk: 3-6 months |
| **HFrEF GDMT** | ARNI/ACEi/ARB + Beta-blocker + MRA + SGLT2i (quadruple therapy) |
| **HFpEF Diagnosis** | HFA-PEFF or H2FPEF score ≥5; LVEF ≥50%; raised NPs; structural/functional abnormalities |
| **HFpEF Treatment** | SGLT2i (empagliflozin/dapagliflozin); diuretics; treat comorbidities |
| **AS Management** | Symptomatic/severe: TAVI vs SAVR (heart team); Asymptomatic severe: surveillance |
| **MS Management** | PMVL if Pliant valve (Wilkins score <8); Surgery if not suitable |
| **MR Management** | Primary: Surgery (repair > replacement); Secondary: GDMT + TEER (MitraClip) |
| **AF Stroke Prevention** | CHA2DS2-VASc ≥2 (M) / ≥3 (F): DOAC preferred over warfarin |
| **AF Rate vs Rhythm** | Rate control 1st line; Rhythm if symptomatic despite rate control |
| **AF Ablation** | Symptomatic paroxysmal AF refractory to ≥1 AAD; Early AF <1 yr |
| **VT/VF Management** | Unstable: Immediate cardioversion; Stable: IV amiodarone/procainamide; ICD for 2° prevention |
| **ICD Indications** | Primary: HFrEF LVEF ≤35% + GDMT 3mo; HCM risk >6%; Secondary: Survived VT/VF |
| **CRT Indications** | LVEF ≤35%, sinus, LBBB QRS ≥150ms, NYHA II-IV on GDMT |
| **Infective Endocarditis** | Duke criteria; Blood cultures ×3; Echo (TTE→TOE); Surgery for HF/abscess/emboli |
| **Acute Pericarditis** | ≥2 of: chest pain, rub, ECG changes, effusion; Rx: NSAIDs + colchicine |
| **Cardiac Tamponade** | Beck's triad + echo signs (RA/RV collapse); Emergency pericardiocentesis |
| **Constrictive vs Restrictive** | Constrictive: Kussmaul sign, pericardial thickening, ventricular interdependence; Restrictive: no pericardial thickening, elevated E/e' |
| **Aortic Dissection** | Stanford A (ascending) = Surgery; Stanford B (descending) = Medical (BP control) |
| **HCM** | Asymmetric septal hypertrophy; LVOTO; Risk-SCD score >6% = ICD; Mavacamten for LVOTO |
| **Cardiac Amyloid** | ATTR vs AL; PYP scan; Tafamidis (ATTR); Chemo (AL); CMR LGE pattern |
| **ARVC** | Task Force Criteria; PKP2 mutations; Exercise restriction; ICD; Ablation |
| **Pulmonary Hypertension** | RHC: mPAP >20, PAWP ≤15, PVR >2 WU; Group 1 PAH: ERA + PDE5i ± prostacyclin |
| **CTEPH** | V/Q scan mismatched defects; PEA surgery; Riociguat if inoperable |
| **Aortic Dissection** | Stanford A = Surgery; Stanford B = Medical (SBP 100-120, HR <60) |
| **Takotsubo** | Apical ballooning; Post-stress; Troponin mild ↑; RV: apical akinesis with basal hyperkinesis |
| **Takotsubo Management** | Supportive; Beta-blocker/ACEi if LV dysfunction; Avoid catecholamines |

# Pulmonary Valve Stenosis / Regurgitation (ACHD)

## 1. Pulmonary Stenosis
- **Valvar** (most common, usually congenital)
- **Subvalvular** (RVOT obstruction, e.g. post-TOF repair)
- **Supravalvular** (PA stenosis, often in syndromes)
- Symptoms: dyspnea, fatigue; severe can cause RV failure
- Echo: peak gradient >64 mmHg = severe
- **Treatment**:
  - **Balloon valvuloplasty** for valvar PS (peak gradient >50 mmHg)
  - **Surgical valvotomy** if balloon not feasible
  - **TPVR** (Melody, Sapien) for dysplastic valve or conduit

## 2. Pulmonary Regurgitation
- **Most common cause: post-TOF repair** (transannular patch)
- Long asymptomatic period
- Progressive RV dilation → TR → RV failure
- Symptoms: exercise intolerance, AF, VT
- **Echo + CMR** (gold standard for PR fraction)
- **Treatment**: TPVR or surgical PVR before irreversible RV dysfunction

## 3. Follow-up
- **Lifelong** cardiology follow-up
- **Annual echo** for moderate/severe
- **CMR** every 2–3 years for severe PR/RV volume
- Endocarditis prophylaxis if prosthetic valve

> **Pearls** — **PS** = treat with **balloon valvuloplasty** if severe. **PR** = treat with **TPVR** before RV dysfunction irreversible. **CMR** is gold standard. Lifelong follow-up.

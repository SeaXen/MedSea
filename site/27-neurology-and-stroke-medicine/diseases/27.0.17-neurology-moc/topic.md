## High-Yield FCPS/MRCP Priority Matrix

### Tier 1: Core (Must Know — 80% of marks)
| Topic | Key Focus |
|-------|-----------|
| **Stroke/TIA** (Ch26) | FAST, ABCD2, thrombolysis/thrombectomy, secondary prevention |
| **Epilepsy** | Classification, status epilepticus, ASM mechanisms/monitoring, pregnancy |
| **Multiple Sclerosis** | McDonald criteria, DMT algorithm, NMO/MOG differentiation |
| **Parkinson's Disease** | Diagnostic criteria, motor/non-motor, levodopa, advanced therapies |
| **Dementia (AD, FTD, DLB, VaD)** | Diagnostic criteria, biomarkers, differentiation, management |
| **MND/ALS** | El Escorial, red flags, riluzole/edaravone, MDT |
| **Guillain-Barré** | Variants, treatment (IVIG/plasma exchange), prognosis |
| **Myasthenia Gravis** | Antibodies, ice pack test, thymectomy, crisis management |
| **CNS Infections** | Bacterial/viral/TB/fungal meningitis, HSV encephalitis, LP interpretation |
| **Coma** | GCS/FOUR, structural vs metabolic, brainstem death criteria |
| **Headache Red Flags** | Thunderclap, SNOOP4, IIH, temporal arteritis |

### Tier 2: High Probability (Should Know — 15%)
- Atypical parkinsonism (MSA, PSP, CBD, DLB)
- Tremor, dystonia, chorea classification
- Peripheral neuropathy approach (diabetic, CIDP, hereditary, vasculitic)
- Inflammatory myopathies (DM, PM, IBM)
- Spinal cord compression (metastatic, DCM)
- Autoimmune encephalitis (anti-NMDA, LGI1, CASPR2)
- Functional neurological disorders (positive signs)
- Sleep disorders (narcolepsy, OSA, RBD)
- Neuro-ophthalmology (optic neuritis, IIH, CN palsies)
- Genetic disorders (Huntington, Friedreich, mitochondrial)

### Tier 3: Differentiator (Nice to Know — 5%)
- Rare MS variants (tumefactive, Marburg, Balo)
- Rare dementias (PCA, CBD, semantic variant PPA)
- Congenital myasthenic syndromes
- Mitochondrial disorders (MELAS, MERRF, LHON)
- Leukodystrophies
- Paraneoplastic syndromes (antibody panels)
- Rare movement disorders (WKS, NBIA)
- Channelopathies

## Key Algorithms & Scoring Systems (Memorise)

| System | Domain | Key Thresholds |
|--------|--------|----------------|
| **FAST / ABCD2** | TIA/Stroke | ABCD2 ≥4 = high risk |
| **NIHSS / mRS** | Stroke severity/outcome | NIHSS ≥16 severe |
| **GCS / FOUR Score** | Coma | GCS ≤8 = intubation |
| **McDonald 2017** | MS diagnosis | DIS + DIT |
| **El Escorial** | ALS diagnosis | UMN + LMN in ≥3 regions |
| **Movement Disorder Society** | PD diagnosis | Bradykinesia + rigidity/tremor |
| **NIA-AA / IWG** | AD diagnosis | Amyloid + tau + neurodegeneration |
| **SCORTEN** | SJS/TEN | ≥3 = 35% mortality |
| **APACHE II / SOFA** | ICU/Encephalopathy | Organ failure scoring |
| **MMSE / MoCA** | Cognitive screen | MoCA <26 = impairment |
| **IASP / Neuropathic Pain** | Neuropathic pain | DN4, LANSS |
| **Epworth Sleepiness Scale** | Narcolepsy/OSA | >10 = excessive sleepiness |

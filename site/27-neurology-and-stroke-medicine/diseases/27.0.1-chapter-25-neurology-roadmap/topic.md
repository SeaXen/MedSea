## Build Process Per Batch

### Pre-Session
1. Run chapter verification: `python3 chapter_verify.py "/mnt/tb/Medicine/Neurology" 218`
2. Identify next 10 Tier 1 topics from hierarchy
3. Open template: `/mnt/tb/Medicine/Templates/Neurology Topic Template.md`

### During Session (Per Topic)
1. Copy template → rename to topic filename
2. Fill frontmatter (topic, section, references)
3. Populate all 14 sections + MCQs/SBAs/Flashcards/Answer Key
4. Set `status: full-fcps-mrcp-note`
5. Update parent hub file completion counter
6. Update MOC completion tracker

### Post-Session
1. Git commit: `git add . && git commit -m "Neurology: Batch N - [topic list]"`
2. Run verification script
3. Update Roadmap current counts
4. Push to remote

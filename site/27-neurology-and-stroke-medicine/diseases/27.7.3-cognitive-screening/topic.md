# Cognitive Screening

> [!tip] **Cognitive screening is the gateway to dementia diagnosis.** Choose tools by **purpose** (screening vs detailed), **setting** (bedside vs clinic), and **population** (educated, MCI, language, literacy).

> [!tip] **NICE NG97 (2018) recommends:** MMSE or MoCA for initial assessment; if brief, use 6-CIT or GPCOG; ACE-III or similar for detailed testing; 4AT for delirium.

## 1. Definition / Epidemiology

**Cognitive screening** = brief structured assessment to detect cognitive impairment. **Comprehensive neuropsychological testing** = detailed hour-long battery administered by psychologist for subtyping and MCI.

| Test | Time | Use | Cut-off |
|------|------|-----|---------|
| **MMSE** | 5-10 min | Dementia screening, monitoring | <24/30 |
| **MoCA** | 10 min | MCI, FTD, vascular, executive | <26/30 |
| **ACE-III** | 15-20 min | Comprehensive, subtyping | 88 (sensitivity 100%) / 82 (specificity 100%) |
| **AMTS** | 5 min | Bedside, primary care | ≤6/10 |
| **6-CIT** | 2-3 min | Brief screening, ED | >7/28 |
| **4AT** | 2 min | **Delirium** detection | ≥4 |
| **GPCOG** | 5 min | Primary care | <9/15 |
| **Clock Drawing** | 1-2 min | Visuospatial, executive | Multiple scoring |
| **Mini-Cog** | 3 min | Combined recall + clock | 0-2 abnormal |

## 2. MMSE — Components and Scoring

| Domain | Score | Tests |
|--------|-------|-------|
| Orientation (time/place) | 10 | Day, date, month, year, season, country, city, district, hospital, floor |
| Registration | 3 | Name 3 objects, repeat |
| Attention/Concentration | 5 | Serial 7s OR spell WORLD backwards |
| Recall | 3 | Recall 3 objects at 5 min |
| Language | 9 | Naming (2), repetition (1), 3-stage command (3), reading (1), writing (1) |
| Visuospatial | 1 | Copy pentagon (intersecting) |

**MMSE Limitations:** Poor executive function testing, ceiling effect in highly educated, floor effect in severe dementia, age/education bias (always record years of education), copyright restrictions.

**MMSE Severity (Alzheimer):**
- **Mild:** MMSE 21-26
- **Moderate:** MMSE 10-20
- **Severe:** MMSE <10

## 3. MoCA — Components and Scoring

| Domain | Points |
|--------|--------|
| Visuospatial/Executive | 5 (trail, cube, clock) |
| Naming | 3 (animals) |
| Attention | 6 (digits, vigilance, serial 7) |
| Language | 3 (repetition, fluency) |
| Abstraction | 2 (similarities) |
| Recall | 5 (delayed) |
| Orientation | 6 |
| **Total** | **30** |

**MoCA cut-offs:** <26/30 = abnormal; <br>**Add 1 point if ≤12 years education.** Better than MMSE for: **MCI, executive dysfunction, FTD, vascular dementia, Parkinson's dementia**.

## 4. Addenbrooke's Cognitive Examination (ACE-III)

**5 domains, 100 points:**
- **Attention** (18): Orientation, registration, concentration
- **Memory** (26): Recall, anterograde, retrograde
- **Verbal Fluency** (14): Letter (P), Category (animals)
- **Language** (26): Naming, comprehension, repetition, reading
- **Visuospatial** (16): Clock, copying, perception

**Advantages:** Better than MMSE for **FTD** (verbal fluency + language); ACE-III-R 2020 updated; **free to use**.

**Subtyping clues:**
- **AD:** Memory + orientation poor
- **FTD:** Fluency + language poor (memory preserved)
- **DLB:** Visuospatial + attention poor
- **VaD:** Executive + attention poor (subcortical pattern)

## 5. Other Brief Tests

**4AT (Delirium):**
- Alertness (0-4) + AMT4 (0-2) + Attention (0-2) + Acute change (0-4)
- **≥4 = probable delirium; 1-3 = cognitive impairment**

**AMTS (Hodkinson 1972):** Age, time to nearest hour, address recall (3 items, 5 min), year, place, recognition of 2 people. **≤6/10 abnormal.**

**6-CIT (Six-Item Cognitive Impairment Test):** Orientation, attention, memory. Score 0-28; **>7/28 abnormal**.

**GPCOG (General Practitioner Assessment of Cognition):** Patient + informant. **Step 1 (patient):** 9 items, time orientation, clock, recall, news; **<9 = cognitive impairment → Step 2 (informant)**.

**Mini-Cog:** 3-word recall + clock drawing. **0-2 of 3 words recalled = abnormal.**

**Clock Drawing Test:** Circle, numbers, hands at 11:10. Multiple scoring systems (Sunderland, Shulman). **Sensitive to visuospatial, executive, semantic impairment**.

## 6. Choosing the Right Test

| Clinical Scenario | Best Tool |
|-------------------|-----------|
| **GP screening** | GPCOG, Mini-Cog, 6-CIT |
| **Hospital admission** | 4AT (delirium) + AMTS |
| **Memory clinic (initial)** | MoCA or ACE-III |
| **MCI detection** | MoCA (better sensitivity) |
| **FTD** | ACE-III (verbal fluency, language) |
| **Vascular/Executive** | MoCA (trail, abstraction) |
| **DLB** | ACE-III (visuospatial) |
| **Severe dementia** | Severe MMSE, severe MoCA |
| **Illiterate / low education** | Rowland Universal Dementia Assessment Scale (RUDAS) |
| **Non-English** | RUDAS, culturally adapted ACE |
| **Acute (delirium)** | 4AT (NICE) |
| **Brief ward** | AMTS, 6-CIT |

## 7. Investigations After Positive Screen

1. **Bloods:** TSH, B12, folate, FBC, U&E, LFT, glucose, Ca, syphilis (VDRL), HIV
2. **MRI Brain:** Atrophy pattern, vascular changes, NPH, mass
3. **CT Head** (if MRI contraindicated or acute presentation)
4. **CSF biomarkers** (NICE NG97): Aβ42 ↓, total-tau ↑, p-tau ↑ (AD)
5. **FDG-PET / Amyloid PET** (specialist)
6. **DAT-SPECT** (DLB vs AD)
7. **EEG** (if seizures, CJD, delirium)

## 8. Key Points for FCPS/MRCP

- **MCI:** Objective cognitive decline but **preserved activities of daily living** (ADLs)
- **MMSE copyright:** Must pay; **MoCA/ACE-III free**
- **MMSE age/education bias:** Adjust for age and ≥12 years education (MoCA +1 point)
- **Floor effect of MMSE at <10:** Cannot distinguish severe from very severe
- **MoCA better than MMSE for:** MCI, executive, FTD, vascular
- **4AT for delirium** (NICE recommended, not MMSE)
- **Clock Drawing Test** = excellent visuospatial screen (1-2 min)
- **Verbal fluency** (animals in 1 min) = sensitive to FTD, AD, delirium
- **Recall at 5 min** = most sensitive single item for AD
- **Serial 7s vs WORLD:** Either acceptable; WORLD for low numeracy
- **ACE-III:** Better than MMSE for **subtyping**; 100 points; free
- **Cultural/language issues:** RUDAS, Rowland Universal Dementia Assessment Scale (6 items, /30)

## 9. Diagnostic Pathways (NICE NG97)

**Primary care:** GPCOG or 6-CIT → if positive → bloods + CT → refer memory clinic
**Memory clinic:** ACE-III or MoCA → bloods + MRI → specialist opinion → CSF/PET if needed
**Acute hospital:** 4AT (delirium screen) → if positive → medical workup

## 10. Differential Diagnosis & Pitfalls

| Pitfall | How to Address |
|---------|----------------|
| Sensory impairment | Ensure glasses/hearing aid used |
| Acute illness | Rescreen after recovery (delirium) |
| Depression | Consider Geriatric Depression Scale (GDS) |
| Low education | Use MoCA with +1 correction; RUDAS |
| Language/cultural | RUDAS, translated MoCA |
| Anxiety/stress | Repeat testing on another day |
| Test anxiety | Reassure, allow rest |

## Viva Questions
1. **Q:** Components and scoring of MMSE. **A:** Orientation 10, Registration 3, Attention 5 (serial 7s or WORLD), Recall 3, Language 9, Visuospatial 1. Total 30; <24 abnormal.
2. **Q:** When is MoCA preferred over MMSE? **A:** MCI detection, FTD, vascular dementia, executive dysfunction, Parkinson's dementia.
3. **Q:** What is the 4AT test? **A:** 4-item (Alertness, AMT4, Attention, Acute change); 0 normal; 1-3 cognitive impairment; **≥4 delirium** (NICE).
4. **Q:** Components of ACE-III. **A:** Attention 18, Memory 26, Fluency 14, Language 26, Visuospatial 16. Total 100.
5. **Q:** MMSE vs MoCA — which is free? **A:** MoCA and ACE-III are **free**; MMSE is **copyrighted** (cost).
6. **Q:** Best brief test for FTD? **A:** ACE-III (verbal fluency + language) or MoCA (executive).
7. **Q:** MCI definition. **A:** Objective cognitive decline on testing with **preserved activities of daily living** (ADLs) and not meeting dementia criteria.
8. **Q:** What is the clock drawing test useful for? **A:** Visuospatial, executive, semantic memory — sensitive to dementia subtypes. Multiple scoring (Sunderland, Shulman).
9. **Q:** Best cognitive test for low education/illiterate? **A:** **RUDAS** (Rowland Universal Dementia Assessment Scale) or Mini-Cog.
10. **Q:** What does NICE NG97 recommend for initial cognitive assessment? **A:** MMSE or MoCA for initial; 4AT for delirium; ACE-III or similar for detailed.

## Common Confusions / Exam Traps
| Confusion | Clarification |
|-----------|---------------|
| MMSE vs MoCA scoring | MMSE <24; MoCA <26 (with +1 for ≤12yr education) |
| MMSE copyright | Yes — pay to use; MoCA/ACE free |
| 4AT vs MMSE | 4AT for **delirium**; MMSE for **dementia** |
| MCI vs dementia | MCI = objective decline + **preserved ADLs** |
| Test of executive function | Trail B (MoCA), clock, verbal fluency |
| Severe MMSE | Floor effect at <10 |
| Recall timing | 5 min (AD sensitive) |
| RUDAS | Rowland Universal Dementia Assessment Scale |

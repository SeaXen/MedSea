# EEG & Epileptiform Disorders

> [!tip] **EEG = function of cortical neurons**, not structure. Normal EEG ≠ no epilepsy (yield ~50% in interictal routine EEG). Always **clinical context + EEG**, never EEG alone.

## Learning Objectives
- [ ] Describe EEG electrodes, montages, normal rhythms
- [ ] Identify common normal variants vs epileptiform discharges
- [ ] Recognise characteristic EEG patterns (spike-wave, PLEDs, GPDs, triphasic)
- [ ] Choose appropriate EEG type (routine, ambulatory, video, cEEG)
- [ ] Apply Salzburg criteria for NCSE
- [ ] Interpret EEG in encephalopathy, dementia, brain death

---

## 1. Definition / Classification

### Definition
**Electroencephalogram (EEG):** Non-invasive recording of cortical electrical activity via scalp electrodes (10-20 international system). Reflects summated post-synaptic potentials of cortical pyramidal neurons.

### Types of EEG
| Type | Duration | Best For |
|------|----------|----------|
| **Routine EEG** | 20-30 min | First seizure, interictal screening |
| **Sleep EEG** | 30-60 min (incl. sleep) | Activates generalised discharges |
| **Sleep-deprived EEG** | 30-60 min post-sleep deprivation | ↑ yield in suspected generalised epilepsy |
| **Ambulatory EEG** | 24-72h, portable | Frequent events, characterisation |
| **Video-EEG (telemetry)** | Hours to days | Pre-surgical; PNES vs epileptic |
| **Continuous EEG (cEEG)** | ≥24h in ICU | NCSE, RSE, post-cardiac arrest, coma |

## 2. Normal EEG Rhythms

| Rhythm | Frequency | Location | State |
|--------|-----------|----------|-------|
| **Delta (δ)** | 0.5-4 Hz | Diffuse, frontal | Deep sleep, encephalopathy |
| **Theta (θ)** | 4-8 Hz | Diffuse, temporal | Drowsy |
| **Alpha (α)** | 8-13 Hz | Posterior (occipital) | Awake, eyes closed |
| **Beta (β)** | >13 Hz | Frontal, central | Alert, eyes open; sedatives |
| **Mu rhythm** | 8-13 Hz | Central (C3/C4) | Sensorimotor cortex; suppressed by movement/thought |

### Normal Variants (NOT epileptiform)
- **Lambda waves:** Sharp transients in occipital during saccadic eye movements (scanning)
- **Positive occipital sharp transients (POSTs):** In sleep
- **Vertex sharp waves:** Stage 2 sleep (central, sharp)
- **K-complexes:** Sleep (universal)
- **Benign epileptiform transients of sleep (BETS / small sharp spikes)**
- **14 & 6 Hz positive bursts:** Adolescents, drowsy

## 3. Epileptiform Discharges & Patterns

### Interictal Patterns
| Pattern | Significance |
|---------|-------------|
| **Spike** | <70 ms, epileptiform if focal/with slow wave |
| **Sharp wave** | 70-200 ms |
| **Spike-and-slow-wave (SSW)** | Generalised 3 Hz = absence; focal = focal epilepsy |
| **Polyspike** | Generalised = JME, myoclonic |
| **Hypsarrhythmia** | Chaotic high-voltage, infants — West syndrome |

### Ictal Patterns
| Pattern | Significance |
|---------|-------------|
| **Rhythmic theta evolving to spike-wave** | Temporal lobe seizure |
| **3 Hz generalised SSW** | Absence seizure |
| **Generalised polyspike-and-wave** | Myoclonic / generalised tonic-clonic |
| **Focal rhythmic alpha evolving** | Focal aware seizure (aura) |

### Specific Epilepsy Syndromes — EEG Features
| Syndrome | EEG Hallmark |
|----------|-------------|
| **Childhood absence epilepsy** | 3 Hz generalised SSW, provoked by hyperventilation |
| **JME** | Generalised 4-6 Hz polyspike-and-wave |
| **West syndrome** | Hypsarrhythmia |
| **Lennox-Gastaut** | Slow (<2.5 Hz) SSW + generalised fast activity |
| **Benign Rolandic epilepsy (BECTS)** | Centro-temporal spikes (CTS), activated by sleep |
| **Panayiotopoulos syndrome** | Occipital spikes, autonomic seizures |

---

## 4. Common Pathological EEG Patterns

| Pattern | Context |
|---------|---------|
| **Generalised periodic discharges (GPDs)** | Metabolic (hepatic, uremic), post-anoxia |
| **Lateralised periodic discharges (LPDs/PLEDs)** | Acute stroke, HSV encephalitis, tumour |
| **Bilateral independent PLEDs (BIPLEDs)** | Severe encephalitis, anoxia |
| **Triphasic waves** | Hepatic/uremic encephalopathy; CJD |
| **Periodic sharp wave complexes (PSWCs)** | CJD (1 Hz) |
| **SIRPIDs** | Stimulus-induced rhythmic, periodic, or ictal discharges (ICU) |
| **Burst-suppression** | Coma, deep anaesthesia, hypothermia, severe anoxia |
| **Electrocerebral inactivity (ECI)** | Brain death confirmation |

---

## 5. EEG in Encephalopathy

| Condition | EEG Pattern |
|-----------|-------------|
| **Hepatic encephalopathy** | Triphasic waves (1-2 Hz, frontal) |
| **Uremic encephalopathy** | Slowing + triphasic |
| **Hypoglycaemia** | Slowing, then spike-wave if severe |
| **Hashimoto's encephalopathy** | Slowing + occasional epileptiform |
| **CJD** | 1 Hz periodic sharp wave complexes |
| **SSPE** | Periodic complexes (every 4-15 sec) |
| **Anti-NMDA encephalitis** | Extreme delta brush |
| **Limbic encephalitis** | Focal temporal slowing/discharges |
| **Post-anoxia** | Burst-suppression, GPDs, suppressed background |

## 6. Non-Convulsive Status Epilepticus (NCSE) — Salzburg Criteria

### Definite NCSE (any one):
1. **Generalised periodic discharges (GPDs) >2.5 Hz** + clinical improvement with IV ASM
2. **Focal rhythmic activity >1 Hz** + evolving or clinical correlates
3. **Absence status:** generalised 2.5-4 Hz spike-wave + clinical features

### Probable NCSE:
- EEG patterns suggestive + encephalopathy + **response to IV ASM trial**

### Possible NCSE:
- EEG patterns only, no clinical correlation

### When to Suspect NCSE
- Prolonged unexplained confusion / coma (post-seizure, post-cardiac arrest, ICU)
- Fluctuating mental status with subtle motor signs (eyelid flutter, myoclonus, nystagmus)
- Failure to wake after sedation/seizure

---

## 7. Indications for EEG

### Definite Indications
| Indication | EEG Type |
|------------|----------|
| **First unprovoked seizure** | Routine (± sleep-deprived if normal) |
| **Suspected epilepsy / classification** | Routine + sleep-deprived |
| **Suspected NCSE** | **cEEG** (urgent) |
| **Refractory status epilepticus** | **cEEG** (continuous) |
| **Post-cardiac arrest** | **cEEG** (≥24h, per guidelines) |
| **Suspected PNES** | Video-EEG (gold standard) |
| **Pre-surgical epilepsy workup** | Video-EEG, ictal SPECT, MEG |
| **Brain death confirmation** | Standard EEG (showing ECI) |
| **Unexplained encephalopathy** | Routine, may need cEEG |

### Adjunctive Activations
- **Hyperventilation** (3 min) — provokes absence seizures
- **Photic stimulation** — photosensitivity (1-3% epilepsy, JME)
- **Sleep / sleep deprivation** — ↑ yield for focal/generalised

---

## 8. Continuous EEG (cEEG) in ICU

### Indications
- **NCSE suspected / confirmed** (Salzburg criteria)
- **Refractory / super-refractory SE** (titrate anaesthetics to burst suppression)
- **Post-cardiac arrest** (prognostication + seizure detection)
- **Coma of unknown cause**
- **Intracranial haemorrhage / TBI** with altered mental status
- **Subarachnoid haemorrhage** (vasospasm + seizures)
- **Periodic EEG patterns** (LPDs, GPDs)

### Monitoring
- Quantitative EEG trends (alpha variability, asymmetry, rhythmicity)
- Burst-suppression ratio
- Seizure burden

---

## 9. EEG in Brain Death

### Requirements
- Minimum 8 scalp electrodes + reference
- Sensitivity ↑ (2 µV/mm), low filter 0.5 Hz
- 30-min recording (2x if no reactivity)
- Exclude confounders: hypothermia (<32°C), sedatives, metabolic, severe hypotension

### Findings
- **Electrocerebral inactivity (ECI):** <2 µV activity over 30 min
- No reactivity to stimuli
- EEG does NOT assess brainstem function (use clinical + SSEPs)

---

## 10. EEG in Dementia

| Dementia | EEG Pattern |
|----------|-------------|
| **Alzheimer's** | Diffuse slowing (theta, delta) in advanced disease; usually normal early |
| **DLB** | Diffuse slowing, frontal intermittent rhythmic delta activity (FIRDA) |
| **CJD** | **Periodic sharp wave complexes (1 Hz)** — highly suggestive |
| **HIV-associated** | Diffuse slowing |
| **SSPE (late)** | Periodic complexes |

---

## 11. Pitfalls & Artefacts

| Artefact | Mimics |
|----------|--------|
| **Eye blinks** | Frontal slow/sharp waves |
| **Chewing** | Rhythmic 4-6 Hz temporal |
| **ECG** | Periodic generalised spike |
| **Electrode pop** | Focal spike |
| **Sweat** | Slow baseline drift |
| **IV drip** | 50 Hz interference |
| **Photomyoclonic** | Photoparoxysmal response |

> Always interpret EEG **with knowledge of patient state and artefact** — use simultaneous video.

## 12. FCPS/MRCP High-Yield Summary

| Category | Key Points |
|----------|------------|
| **Normal rhythms** | α = posterior, awake, eyes closed; β = frontal, alert; μ = central |
| **Epileptiform** | Spike, sharp wave, polyspike, spike-wave |
| **3 Hz SSW** | Childhood absence epilepsy (provoked by hyperventilation) |
| **Hypsarrhythmia** | Infantile spasms (West syndrome) |
| **Centro-temporal spikes** | BECTS (benign Rolandic) — sleep-activated |
| **Triphasic waves** | Hepatic / metabolic encephalopathy |
| **PSWCs (1 Hz)** | CJD |
| **PLEDs** | Stroke, HSV encephalitis, tumour |
| **Salzburg NCSE** | GPDs >2.5Hz + clinical improvement after IV ASM |
| **First seizure** | Routine EEG within 24-48h; sleep-deprived if normal |
| **cEEG** | Mandatory in NCSE, RSE, post-cardiac arrest |
| **Brain death EEG** | Electrocerebral inactivity (<2 µV, 30 min) excluding confounders |

---

## 13. Special Situations

| Situation | Consideration |
|-----------|---------------|
| **Pregnancy** | EEG safe; avoid sleep deprivation if risk of falls |
| **Paediatric** | Shorter sessions; sedation rarely needed; sleep EEG useful |
| **Elderly** | More slowing is normal; comorbidities confuse interpretation |
| **ICU / ventilated** | cEEG mandatory for NCSE; sedation confounds |
| **Post-cardiac arrest** | cEEG ≥24h; reactivity predicts outcome |
| **Brain death** | Exclude sedatives, hypothermia, hypotension |

## Viva Questions

1. **Q:** What is the difference between spike and sharp wave?
   **A:** Spike <70 ms; sharp wave 70-200 ms.
2. **Q:** What is the EEG hallmark of absence seizures?
   **A:** 3 Hz generalised spike-and-slow-wave.
3. **Q:** What are PLEDs and what do they suggest?
   **A:** Periodic lateralised epileptiform discharges = acute focal injury (stroke, HSV encephalitis, tumour).
4. **Q:** Define Salzburg criteria for definite NCSE.
   **A:** GPDs >2.5 Hz + clinical improvement with IV ASM OR focal rhythmic >1 Hz + clinical OR absence status with 2.5-4 Hz SSW.
5. **Q:** What does burst suppression indicate?
   **A:** Severe encephalopathy, deep anaesthesia, post-anoxia, or pharmacologically induced (RSE target).

## Common Confusions

| Confusion | Clarification |
|-----------|---------------|
| Normal EEG ≠ no epilepsy | Routine yield ~50%; sleep-deprived / ambulatory improves |
| EEG localisation ≠ seizure onset zone | Irritative zone may extend beyond; video-EEG + MRI for surgery |
| Triphasic waves = hepatic only | Also uremic, hypercapnic, post-anoxia |
| PSWCs = CJD only | SSPE also periodic (different intervals) |
| Brain death EEG ≠ brainstem death | EEG = cortical; brain death requires brainstem tests |

---

## Mnemonics

1. **α EYES CLOSED, β EYES OPEN** — alpha posterior with eyes closed; beta anterior when alert
2. **3 Hz = ABSENCE** — generalised 3 Hz spike-wave
3. **HYPS = WEST** — hypsarrhythmia in West syndrome
4. **CTS = BECTS** — centro-temporal spikes in benign Rolandic epilepsy
5. **PLEDs = LATERAL** — periodic lateralised = lateralised lesion
6. **ECI = ELECTROCEREBRAL INACTIVITY** — brain death EEG

## SBA Questions (10)

1. **Scenario:** A 70-year-old man post-cardiac arrest is comatose at 24h. Subtle facial twitching noted. Next best step?
   **A:** Empirical phenytoin **B:** cEEG **C:** CT head **D:** MRI
   **Ans:** B — Post-cardiac arrest: high risk of NCSE. cEEG required to detect and titrate treatment.

2. **Scenario:** A 30-year-old woman with generalised tonic-clonic seizures. EEG shows generalised 4-6 Hz polyspike-and-wave. Diagnosis?
   **A:** Absence epilepsy **B:** JME **C:** West syndrome **D:** BECTS
   **Ans:** B — JME (Juvenile Myoclonic Epilepsy) shows generalised 4-6 Hz polyspike-wave; onset in adolescence.

3. **Scenario:** A 60-year-old with rapidly progressive dementia and myoclonus. EEG shows 1 Hz periodic sharp wave complexes. Diagnosis?
   **A:** AD **B:** CJD **C:** DLB **D:** VaD
   **Ans:** B — 1 Hz PSWCs + rapidly progressive dementia = CJD.

4. **Scenario:** A child with infantile spasms. EEG shows chaotic high-voltage slow waves with multifocal spikes. Pattern called?
   **A:** Burst suppression **B:** Hypsarrhythmia **C:** Slow spike-wave **D:** FIRDA
   **Ans:** B — Hypsarrhythmia = West syndrome (infantile spasms).

5. **Scenario:** A 50-year-old with new-onset seizures. CT shows hypodensity in left temporal lobe. EEG shows periodic lateralised discharges in left temporal region. Best diagnosis?
   **A:** Stroke **B:** HSV encephalitis (until proven otherwise) **C:** Tumour **D:** GBS
   **Ans:** B — PLEDs + temporal = HSV encephalitis until ruled out (LP for HSV PCR, start aciclovir).

6. **Scenario:** A 25-year-old with "seizures" lasting 30+ minutes, asynchronous, with eye closure and pelvic thrusting. Routine EEG normal. Best test?
   **A:** MRI brain **B:** Video-EEG telemetry **C:** CT **D:** SPECT
   **Ans:** B — Features suggest PNES; video-EEG = gold standard (no EEG correlate during event).

7. **Scenario:** A patient with status epilepticus on midazolam. Target on cEEG is:
   **A:** 50% burst suppression **B:** 90-95% burst suppression **C:** Isoelectric EEG **D:** Normal background
   **Ans:** B — 90-95% burst suppression (~95% interburst interval) is the standard anaesthetic titration target.

8. **Scenario:** A patient in coma after drug overdose. EEG shows ECI. Before declaring brain death, must exclude:
   **A:** Renal failure **B:** Severe hypothermia, sedatives, hypotension **C:** Diabetes **D:** Hypertension
   **Ans:** B — Confounders (sedatives, hypothermia <32°C, severe hypotension, severe metabolic) must be excluded before diagnosing ECI/brain death.

9. **Scenario:** A 12-year-old with nocturnal seizures, EEG shows bilateral centrotemporal spikes activated by sleep. Diagnosis?
   **A:** BECTS (benign Rolandic epilepsy) **B:** JME **C:** Absence epilepsy **D:** Temporal lobe epilepsy
   **Ans:** A — BECTS: centrotemporal spikes, sleep-activated, often nocturnal orofacial seizures.

10. **Scenario:** Patient with hepatic encephalopathy. EEG shows characteristic pattern. Pattern is?
    **A:** PSWCs **B:** Triphasic waves **C:** Hypsarrhythmia **D:** PLEDs
    **Ans:** B — Hepatic encephalopathy classically shows triphasic waves (frontal, 1-2 Hz).

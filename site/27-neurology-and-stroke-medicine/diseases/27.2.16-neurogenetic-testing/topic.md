# Neurogenetic Testing

> [!tip] **Test smart**: only order a genetic test when result will change management or family planning. Always **genetic counselling BEFORE testing** for predictive/huntingtin/dominant disorders.

## Learning Objectives
- [ ] Describe types of genetic tests (single gene, panel, WES, WGS, mtDNA, CGH array)
- [ ] Choose appropriate test based on phenotype
- [ ] Interpret variants (pathogenic, VUS, benign)
- [ ] Apply ACMG variant classification
- [ ] Provide pre-test and post-test genetic counselling
- [ ] Recognise ethical issues (incidental findings, predictive testing)

---

## 1. Definition / Classification

### Definition
**Genetic testing** in neurology identifies pathogenic DNA variants (single nucleotide, copy number, repeat expansion, mitochondrial) responsible for or predisposing to neurological disease. Choice depends on:
- Clinical phenotype (single gene vs panel vs exome/genome)
- Inheritance pattern
- Family history
- Need for predictive/presymptomatic testing

### Inheritance Patterns
| Pattern | Risk to Offspring | Examples |
|---------|-------------------|----------|
| **Autosomal dominant (AD)** | 50% | Huntington's, DYT1 dystonia, NF1, CMT1 |
| **Autosomal recessive (AR)** | 25% (sibs) | Wilson's (ATP7B), Friedreich's ataxia, Duchenne carriers |
| **X-linked** | Variable (50% male, 50% carrier daughters) | DMD, Kennedy's, Fragile X |
| **Mitochondrial (maternal)** | All offspring (variable expression) | MELAS, MERRF, LHON, NARP |
| **Trinucleotide repeat** | Anticipation | Huntington's, Fragile X, Myotonic dystrophy, Friedreich's |
| **De novo** | Variable | DMD, many |

---

## 2. Types of Genetic Tests

| Test | Detects | Best For |
|------|---------|----------|
| **Sanger sequencing** | Single gene, point mutations | Confirmatory (known familial mutation) |
| **Targeted panel** | Multiple genes for phenotype | Epilepsy, ataxia, neuropathy, muscular dystrophy |
| **WES (whole exome sequencing)** | All exons (~20,000 genes) | Diagnostic odyssey |
| **WGS (whole genome sequencing)** | All DNA (exons + introns + regulatory) | Comprehensive; structural variants |
| **CGH array / MLPA** | Copy number variants (deletions, duplications) | DMD (exon deletions), CMT1A duplication |
| **Repeat-primed PCR / Southern blot** | Trinucleotide repeat expansions | Huntington's, Fragile X, Myotonic dystrophy, Friedreich's |
| **mtDNA sequencing** | Mitochondrial genome | MELAS, MERRF, LHON |
| **Karyotype** | Large chromosomal abnormalities | Trisomies, translocations |
| **FISH** | Specific microdeletions | 22q11 (DiGeorge), 15q11 (Angelman/Prader-Willi) |
| **Non-invasive prenatal testing (NIPT)** | Cell-free fetal DNA | Screening |

## 4. ACMG Variant Classification

| Class | Interpretation | Action |
|-------|---------------|--------|
| **Pathogenic** | Cause disease | Report; medical management |
| **Likely pathogenic** | 90% likely pathogenic | Report; consider management |
| **VUS (variant of uncertain significance)** | Insufficient evidence | Do NOT change management; re-evaluate |
| **Likely benign** | 90% likely benign | Usually not reported |
| **Benign** | Not disease-causing | Not reported |

### Key Terms
- **Penetrance:** % of carriers who develop disease (e.g., LRRK2 G2019S ~30-70%)
- **Expressivity:** Severity variation in carriers
- **Anticipation:** Earlier onset / worse disease in successive generations (trinucleotide repeats)
- **Variable expressivity:** Different manifestations in same family
- **Pleiotropy:** One gene → multiple organ systems affected (e.g., NF1)

## 5. Choosing a Genetic Test

### Algorithm
```
Phenotype → Hypothesis (single gene? multiple? chromosomal?)
  ↓
Clear single gene → Sanger (confirm) or targeted panel
  ↓
Differential of genes → Targeted panel (epilepsy, ataxia, CMT, dystrophy, leukodystrophy)
  ↓
Unclear / diagnostic odyssey → WES → WGS
  ↓
Repeat expansion suspected (CAG, CTG, GAA, CGG, etc.) → Repeat-primed PCR
  ↓
Mitochondrial suspected → mtDNA sequencing ± nuclear mitochondrial panel
  ↓
Deletions/duplications suspected (DMD, CMT1A) → MLPA / CGH array
```

### Specific Scenarios
| Presentation | First Test |
|--------------|-----------|
| **Huntingting's** | HTT CAG repeat (PCR) |
| **Myotonic dystrophy** | DMPK CTG repeat |
| **FSHD** | D4Z4 repeat analysis |
| **DMD/BMD** | MLPA for deletions (60-70%); sequencing for point mutations |
| **Friedreich's** | FXN GAA repeat |
| **Dravet** | SCN1A panel |
| **LHON** | 3 primary mtDNA mutations (G11778A, G3460A, T14484C) |
| **CMT** | PMP22 duplication first (CMT1A); then panel |
| **Focal epilepsy** | Consider SCN1A (GEFS+), LG1, etc. |
| **Familial hemiplegic migraine** | CACNA1A, ATP1A2, SCN1A |
| **Wilson's** | ATP7B (single gene) |

## 6. Genetic Counselling — Pre-Test

### Indications for Counselling
- **Predictive/presymptomatic testing** (Huntington's, BRCA)
- **Reproductive decisions** (carrier, PGD, prenatal)
- **Family cascade testing**
- **VUS interpretation**
- **Psychosocial support**

### Key Points to Cover
- Inheritance pattern and risks
- Penetrance, variable expressivity, anticipation
- Limitations of testing (VUS, no test 100% sensitive)
- Implications for insurance, employment, mental health
- Family communication
- **Pre-symptomatic Huntington's testing protocol:** ≥2 counselling sessions, ≥1 month wait, psychological assessment, support person present

---

## 7. Genetic Counselling — Post-Test

### Positive / Pathogenic
- Discuss implications for patient and family
- Cascade testing offer to relatives
- Reproductive options (PND, PGD, gamete donation)
- Surveillance / prevention (if available, e.g., SCAs, NF1)
- Long-term support

### Negative
- Doesn't rule out other genes / clinical diagnosis remains
- Limitations of test
- Re-evaluation if phenotype changes

### VUS
- **Do NOT change management based on VUS alone**
- Periodic re-evaluation as evidence emerges
- Consider functional studies
- Family segregation studies

---

## 8. Special Considerations

### Predictive Testing (Asymptomatic)
- **Huntington's:** Strict protocol (HDSA / UK Huntington's consortium guidelines)
- Genetic counselling BEFORE and AFTER
- ≥1 month wait between counselling sessions
- Psychological assessment
- Adult-onset; not in minors unless intervention possible

### Pregnancy / Reproductive
- **PND (prenatal diagnosis):** Chorionic villus / amniocentesis
- **PGD (preimplantation):** IVF + embryo selection
- **Donor gametes**
- **Adoption**
- **No pregnancy / natural conception** (after counselling)

### Insurance / Employment (UK)
- **Code of practice:** Genetic testing cannot be requested by insurers for standard cover
- Higher insurance limits may require disclosure
- Employment: GINA-equivalent legislation (US); varies globally

### Incidental Findings (WES/WGS)
- ACMG "actionable" gene list (59 genes, updated periodically)
- Patient choice: opt-in or opt-out
- Examples: BRCA1/2, Lynch, RET, familial hypercholesterolaemia

---

## 9. Ethical Issues

| Issue | Consideration |
|-------|---------------|
| **Predictive testing in minors** | Generally not offered unless preventive intervention available in childhood |
| **Adoption** | Access to biological family history may be limited |
| **Maternity / paternity** | Surprises possible (non-paternity) — pre-test counselling |
| **VUS** | Don't change management |
| **Direct-to-consumer tests** | Variable quality; clinical correlation essential |
| **Research findings** | Consent for feedback (or not) |

## 11. Special Situations

| Situation | Consideration |
|-----------|---------------|
| **Pregnancy** | Pre-conception counselling; PGD/PND options |
| **Children** | Predictive testing generally not offered for adult-onset |
| **Adoption** | May lack family history; whole-genome approaches useful |
| **Consanguinity** | ↑ autosomal recessive disorders; targeted panels |
| **Founder mutations** | Higher pre-test probability in some populations (e.g., Tay-Sachs, BRCA) |
| **Triplet repeats** | Anticipation; consider in families with progressive disease |

## Viva Questions

1. **Q:** What test for Huntington's?
   **A:** PCR for CAG repeat expansion in HTT gene on chromosome 4.
2. **Q:** What test for Duchenne muscular dystrophy?
   **A:** MLPA first (detects deletions in 60-70%); sequencing for point mutations.
3. **Q:** What is anticipation?
   **A:** Earlier onset and/or worse disease in successive generations (trinucleotide repeats).
4. **Q:** What is the most common CMT?
   **A:** CMT1A (PMP22 duplication).
5. **Q:** How do you manage a VUS?
   **A:** Do NOT change clinical management; re-evaluate as evidence evolves; consider segregation.

## Common Confusions

| Confusion | Clarification |
|-----------|---------------|
| VUS = pathogenic | VUS = uncertain; do not change management |
| Negative test = no disease | Depends on coverage; WES misses some, mtDNA heteroplasmy varies |
| All repeat disorders = HD | Multiple (DM1, FXN, FMR1, AR, etc.) |
| Predictive testing in minors | Generally not offered (except where preventive intervention exists) |
| One panel for everything | Different phenotypes → different panels |

## Mnemonics

1. **CMT1A = PMP22** — peripheral myelin protein 22 duplication
2. **DMD = DELETIONS (60-70%)** — MLPA first
3. **HD = CAG REPEAT** — PCR for HTT
4. **DM1 = CTG DMPK** — myotonia, frontal balding, cataracts
5. **LHON = MT G11778A** — males, painless vision loss
6. **MELAS = MT A3243G** — mitochondrial encephalopathy, lactic acidosis, stroke

   **A:** CGG repeat in FMR1 **B:** CAG repeat in HTT **C:** GAA repeat in FXN **D:** CTG repeat in DMPK
   **Ans:** B

2. **Q:** The most common genetic cause of CMT is:
   **A:** MPZ mutation **B:** GJB1 mutation **C:** PMP22 duplication (CMT1A) **D:** MFN2 mutation

   **A:** Whole genome sequencing **B:** MLPA for deletions/duplications **C:** Karyotype **D:** Repeat-primed PCR

4. **Q:** MELAS is caused by:
   **A:** mtDNA G11778A **B:** mtDNA A3243G **C:** HTT CAG expansion **D:** SCN1A mutation

5. **Q:** VUS in ACMG classification means:
   **A:** Variant is benign **B:** Variant is pathogenic **C:** Insufficient evidence — do NOT change management **D:** Always report

6. **Q:** Which is NOT trinucleotide repeat disorder?
   **A:** Huntington's **B:** Myotonic dystrophy **C:** Friedreich's ataxia **D:** CADASIL

   **A:** SCN1A mutation **B:** SCN4A **C:** KCNQ2 **D:** LGI1
   **Ans:** A — Avoid sodium-channel blockers (worsen seizures).

9. **Q:** Wilson's disease gene:
   **A:** HTT **B:** ATP7B **C:** NOTCH3 **D:** DMD
   **Ans:** B — Copper-transporting ATPase.

10. **Q:** Predictive testing for Huntington's requires:
    **A:** Single visit **B:** Genetic counselling before AND after, ≥1 month wait, psychological assessment **C:** No counselling needed if adult **D:** Family consent

## SBA Questions (10)

1. **Scenario:** A 35-year-old with progressive chorea, dementia, family history of similar illness. Next test?
   **A:** MRI brain **B:** HTT CAG repeat PCR **C:** Wilson's disease screen **D:** LP
   **Ans:** B — Clinical picture + family history suggests HD.

2. **Scenario:** A 4-year-old boy with proximal weakness, calf pseudohypertrophy, CK 20,000. Genetic test?
   **A:** SCN1A panel **B:** MLPA for DMD gene deletions (first-line) **C:** mtDNA sequencing **D:** Karyotype
   **Ans:** B — DMD phenotype; MLPA first (deletions in 60-70%).

3. **Scenario:** A 25-year-old with ataxia, hypertrophic cardiomyopathy, scoliosis, diabetes. Repeat expansion?
   **A:** CAG (HTT) **B:** GAA (FXN) — Friedreich's ataxia **C:** CTG (DMPK) **D:** CGG (FMR1)
   **Ans:** B

4. **Scenario:** Patient reports 23andMe result showing "increased risk" for Alzheimer's (APOE ε4). Action?
   **A:** Confirm with clinical testing, interpret cautiously (low predictive value) **B:** Diagnose Alzheimer's **C:** Start donepezil **D:** Repeat MRI
   **Ans:** A — Direct-to-consumer tests not diagnostic; APOE ε4 is risk factor, not deterministic.

5. **Scenario:** Mother of a boy with DMD wants carrier testing. Result: heterozygous deletion in DMD. Implications?
   **A:** She's a carrier; 50% sons affected, 50% daughters carriers **B:** She's not a carrier **C:** Diagnosis in her is confirmed **D:** No further testing needed
   **Ans:** A — DMD X-linked; female carriers have 50% affected sons.

6. **Scenario:** A 50-year-old with painless progressive bilateral visual loss. mtDNA test: G11778A. Diagnosis?
   **A:** MELAS **B:** MERRF **C:** LHON **D:** Kearns-Sayre
   **Ans:** C — LHON, primary mutation G11778A.

7. **Scenario:** Mother with myotonic dystrophy wants to know risk to child. Information?
   **A:** AD; 50% risk; ANTICIPATION (CTG expansion worse in offspring, esp. congenital form)
   **A:** **Anticipation** — earlier onset, worse disease in successive generations. **AD inheritance**, **50% risk**.
   **Ans:** B

8. **Scenario:** Genetic test reveals BRCA2 variant, classified as VUS. Management?
   **A:** Bilateral mastectomy **B:** Do NOT change management; re-evaluate periodically **C:** Treat as pathogenic **D:** MRI breast annually
   **Ans:** B — VUS = do not act on it.

9. **Scenario:** Young man with stroke, MRI shows confluent white matter hyperintensities, family history of stroke and migraine. Skin biopsy?
   **A:** Muscle biopsy **B:** NOTCH3 immunohistochemistry on skin biopsy (CADASIL)
   **C:** Nerve biopsy **D:** Brain biopsy
   **Ans:** B — CADASIL diagnosis with NOTCH3 staining (vascular smooth muscle).

10. **Scenario:** Adult asymptomatic at-risk relative of Huntington's patient requests predictive test. Protocol?
    **A:** Test immediately **B:** Genetic counselling before, ≥1 month wait, then test, then post-test counselling **C:** No protocol needed **D:** Family consent first
    **Ans:** B — Strict protocol for predictive HD testing.

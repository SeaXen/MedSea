# Syphilis Serology Interpretation (VDRL/RPR, TPHA/FTA-ABS)

Related: [[Syphilis (Primary, Secondary, Latent, Tertiary)]], [[Genital Ulcer Disease Syndrome (GUD)]], [[Congenital Syphilis]], [[Neurosyphilis]], [[HIV Infection and AIDS]]

> [!important]
> **Two-test algorithm**: **Non-treponemal (RPR/VDRL)** = quantitative, screens + monitors activity. **Treponemal (TPHA/FTA-ABS/EIA)** = qualitative, confirms exposure. **Traditional**: RPR screen → TPHA confirm. **Reverse**: EIA/CIA screen → RPR quantify.

## Learning Objectives
- Differentiate non-treponemal vs treponemal tests
- Interpret all serological patterns (active, treated, false positive, prozone)
- Apply traditional vs reverse algorithms
- Monitor treatment response (4-fold titre drop)
- Interpret syphilis in pregnancy, HIV, and congenital settings

## Test Classification

| Category | Tests | What Detects | Quantitative? | Use |
|----------|-------|--------------|---------------|-----|
| **Non-treponemal (NTT)** | **RPR** (Rapid Plasma Reagin), **VDRL** (Venereal Disease Research Lab) | **Anti-lipoidal antibodies** (cardiolipin-cholesterol-lecithin) | **YES** (titre = disease activity) | **Screening, monitoring, activity** |
| **Treponemal (TT)** | **TPHA** (T. pallidum Haemagglutination), **FTA-ABS**, **EIA/CIA**, **TPPA**, **MHA-TP** | **Anti-T. pallidum specific antibodies** | NO (reactive/non-reactive) | **Confirmatory, lifelong positivity** |

### Key Differences
| Feature | Non-treponemal (RPR/VDRL) | Treponemal (TPHA/FTA-ABS) |
|---------|---------------------------|---------------------------|
| **Antigen** | Cardiolipin-cholesterol-lecithin | T. pallidum specific proteins |
| **Quantitative** | Yes (titre) | No (reactive/non-reactive) |
| **Sensitivity (early primary)** | 70–80% | 85–95% |
| **Sensitivity (late latency)** | May revert negative | Remains positive for life |
| **Specificity** | Lower (false + in pregnancy, autoimmune, infections) | High (>99%) |
| **After treatment** | **Titre drops** (4-fold = cure) | **Remains reactive** (lifelong) |
| **Cost/Time** | Cheap, rapid | More expensive, longer |

## Testing Algorithms

### Traditional Algorithm (Low Prevalence / Resource-Limited)
```
Screen: RPR/VDRL
    │
    ├─ Non-reactive → No syphilis (unless early primary / prozone)
    └─ Reactive → Confirmatory TPHA/FTA-ABS
                │
                ├─ Positive → **Syphilis confirmed** (quantify RPR for activity)
                └─ Negative → False positive RPR (unless early primary / prozone)
```

### Reverse Algorithm (Automated / High-Throughput / High Prevalence)
```
Screen: Treponemal EIA/CIA/CLIA (high sensitivity)
    │
    ├─ Non-reactive → No syphilis
    └─ Reactive → Reflex RPR/VDRL (quantitative)
                │
                ├─ Reactive RPR → **Active/treated syphilis** (RPR titre = activity)
                └─ Non-reactive RPR → **Treated / late latent / early primary / prozone**
                              │
                              ↓
                        Confirmatory TPHA/FTA-ABS (if not done)
                │
                ├─ Positive → Treated / late latent / prozone
                └─ Negative → False positive EIA (rare)
```

## Interpretation Framework

| RPR/VDRL | TPHA/FTA-ABS | Interpretation |
|----------|--------------|----------------|
| Negative | Negative | **No syphilis** (or very early primary <2w) |
| Positive | Positive | **Active/treated syphilis** — quantify RPR for activity |
| Negative | Positive | **Treated / Late latent / Prozone / Early primary** |
| Positive | Negative | **False positive RPR** (or early primary / prozone) |

### Prozone Effect (Hook Effect)
- **Mechanism**: Excess antibody prevents lattice formation → false negative RPR/VDRL
- **Suspect**: High clinical suspicion + negative RPR + positive TPHA + high clinical risk
- **Resolution**: **Dilute serum** (1:10, 1:100, 1:1000) → retest RPR → titre appears
- **Common**: Secondary syphilis, early latent, high-titre cases

## Interpretation by Clinical Stage

| Stage | RPR/VDRL | TPHA/FTA-ABS | Notes |
|-------|----------|--------------|-------|
| **Early Primary** | Negative/low (70–80% +) | Positive (85–95%) | RPR may be negative; repeat in 2w |
| **Secondary** | **High titre** (1:32–1:1024) | Positive | Peak reactivity |
| **Early Latent (<1yr)** | Positive (moderate-high) | Positive | Infectious |
| **Late Latent (≥1yr)** | Low/negative | Positive | Non-infectious (except vertical) |
| **Tertiary (Gumma/Cardio/Neuro)** | Variable (often low/neg) | Positive | RPR may be negative |
| **Post-treatment** | **Declining** → non-reactive at 12–24m | **Remains positive** (lifelong) | 4-fold drop at 6–12m = cure |
| **Re-infection** | **Rising** (4-fold increase) | Remains positive | Compare to baseline |

## Treatment Response Monitoring (RPR Titre)

### Serological Cure Criteria
| Stage | Timepoint for 4-fold (2 dilution) Drop | Adequate Response |
|-------|----------------------------------------|-------------------|
| **Primary / Secondary / Early Latent** | **6 months** | 4-fold drop at 6m = cure |
| **Late Latent / Tertiary (non-neuro)** | **12 months** | 4-fold drop at 12m |
| **Neurosyphilis** | **6, 12, 24 months** (CSF VDRL + cell count/protein) | CSF normalisation |

### Treatment Failure Criteria
| Scenario | Action |
|----------|--------|
| **No 4-fold drop** by 6m (early) / 12m (late) | **Re-treat**; CSF exam if neuro possible |
| **4-fold rise** (from nadir) | **Re-infection** or **treatment failure** → re-treat |
| **Persistent high titre** >6m (early) / >12m (late) | Evaluate for re-infection / treatment failure / CSF exam |

## Special Populations

### Pregnancy
| Test | Frequency | Action |
|------|-----------|--------|
| **RPR + TPHA** | Booking, 28w, delivery (high prevalence) | Treat stage-appropriate |
| **Penicillin allergy** | **Desensitisation mandatory** | Doxycycline contraindicated; azithromycin insufficient |

### HIV Co-infection
| Feature | Difference |
|---------|------------|
| **RPR titres** | May be lower; false negatives more common |
| **Prozone effect** | More common |
| **TPHA** | May be delayed/weaker |
| **Neurosyphilis risk** | **Higher** — **CSF exam recommended for ALL** |
| **Treatment response** | Slower serological decline; lower threshold for re-treatment |

### Congenital Syphilis (Infant)
| Test | Interpretation |
|------|---------------|
| **Infant RPR (quantitative)** | **≥4-fold > maternal** = probable congenital |
| **Infant TPHA/FTA-ABS** | Passive transfer of maternal IgG (positive until 15–18m) |
| **Infant IgM (FTA-ABS 19S-IgM)** | **Positive = congenital infection** (doesn't cross placenta) |
| **CSF VDRL** | Reactive = neurosyphilis |
| **Long bones X-ray** | Metaphyseal lucencies, periostitis |
| **Management** | If mother adequately treated >4w before delivery + infant RPR ≤ maternal → observation; else full workup + treatment |

### False Positive RPR (Biological False Positive - BFP)
| Cause | Frequency |
|-------|-----------|
| **Pregnancy** | Common |
| **Autoimmune** (SLE, APS) | Common |
| **Acute infections** (viral, bacterial, TB, malaria) | Common |
| **Vaccinations** | Transient |
| **Malignancy** | Occasional |
| **Drugs** (procainamide, hydralazine) | Rare |
| **Age >65** | Higher baseline |

> **BFP**: RPR positive, **TPHA negative**, usually low titre (<1:8), transient

## Neurosyphilis CSF Interpretation
| CSF Finding | Interpretation |
|-------------|---------------|
| **WBC >5/µL (lymphocytic)** | Abnormal |
| **Protein >0.45 g/L (>45 mg/dL)** | Abnormal |
| **CSF VDRL (reactive)** | **Diagnostic** (specific but insensitive) |
| **CSF FTA-ABS** | Sensitive but not specific (blood contamination) |
| **CSF PCR (T. pallidum)** | High sensitivity; emerging |

> **CSF VDRL reactive + clinical picture = neurosyphilis** (VDRL specific but insensitive)
> **CSF VDRL non-reactive** does NOT exclude neurosyphilis

## Rapid Point-of-Care Tests
| Test | Target | Use Case |
|------|--------|----------|
| **Dual RPR/TPHA cassette** | Combined | Field screening (antenatal, outreach) |
| **Syphilis RST (Rapid Syphilis Test)** | TP only | Screening (antenatal, key populations) |
| **Syphilis + HIV combo** | TP + HIV Ab | Integrated screening |

> **POC = screening only** — **all reactive must be confirmed** by lab-based quantitative RPR + TPHA

## FCPS/MRCP High-Yield Points
- **RPR/VDRL** = quantitative, activity; **TPHA/FTA-ABS** = confirmatory, lifelong
- **Traditional**: RPR screen → TPHA confirm; **Reverse**: EIA screen → RPR quantify
- **Prozone**: High antibody → false negative RPR → dilute serum
- **Interpretation**: RPR+ TPHA+ = active; RPR- TPHA+ = treated/late; RPR+ TPHA- = false positive
- **Treatment response**: 4-fold RPR drop at 6m (early) / 12m (late) = cure
- **Re-infection**: 4-fold rise from nadir
- **Neurosyphilis**: CSF VDRL reactive = diagnostic; FTA-ABS sensitive not specific
- **Pregnancy**: Screen booking/28w/delivery; penicillin only cure (desensitise if allergic)
- **HIV**: Lower RPR titres; prozone more common; CSF exam ALL; slower decline

## Common Confusions / Exam Traps
- **RPR negative, TPHA positive** = treated/late latent/prozone/early primary (NOT active disease)
- **RPR positive, TPHA negative** = false positive (pregnancy, autoimmune, viral) OR early primary OR prozone
- **Prozone** = false negative RPR in high titre → dilute serum
- **RPR titre drop** = 4-fold = 2 dilutions (e.g., 1:64 → 1:16)
- **TPHA stays positive for life** — does not monitor treatment
- **RPR monitors treatment** — TPHA does not
- **Congenital syphilis** = infant RPR >4x maternal + IgM positive
- **POC tests** = screening only; ALL reactive need lab confirmation

## Mnemonics
- **RPR vs TPHA**: **RPR = Q**uantitative (activity); **TPHA = Q**ualitative (confirmation)
- **Algorithm**: **T**raditional = **R**PR → **T**PHA; **R**everse = **E**IA → **R**PR
- **Interpretation**: **R+ T+** = Active; **R- T+** = Treated/Late; **R+ T-** = False +/Prozone/Early
- **Prozone**: **D**ilute **S**erum → **R**e-test = **DSR**
- **Cure**: **4-fold drop** = 2 dilutions (e.g., 1:64 → 1:16)
- **Re-infection**: **4-fold rise** from nadir

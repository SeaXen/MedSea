# Renal Osteodystrophy (CKD-Mineral and Bone Disorder - CKD-MBD)

> [!info]
> **CKD-MBD = Systemic Disorder of Mineral & Bone Metabolism** due to CKD. **Universal in CKD G4-5D.** **Triad: Abnormal Ca/PO₄/PTH, Bone Turnover Abnormalities, Vascular Calcification.** **KDIGO 2017 Targets: PTH 2-9x ULN, Ca 2.1-2.5, PO₄ 0.8-1.5, Ca×P <4.5.**

## 2. Pathophysiology — The CKD-MBD Vicious Cycle

```
CKD (↓ GFR)
    │
    ├─→ **Phosphate Retention** (↓ Filtered Load)
    │       │
    │       ├─→ **FGF23 ↑** (Osteocyte Response) → ↓ 1α-Hydroxylase → **↓ 1,25-(OH)₂D**
    │       │       │
    │       │       └─→ **↓ Intestinal Ca²⁺ Absorption** → **Hypocalcaemia**
    │       │
    │       └─→ **Direct PTH Stimulation** (Phosphate → PTH Gene Expression)
    │
    └─→ **Hypocalcaemia** (↓ Ca²⁺ Absorption + ↓ Vit D) → **PTH ↑↑** (Secondary HPT)
```

### Key Mediators
| Mediator | Change in CKD | Effect |
|----------|---------------|--------|
| **Phosphate** | ↑↑ (Retention) | Direct PTH Stim; FGF23 ↑ |
| **FGF23** | ↑↑↑ (Early Rise) | ↓ 1α-Hydroxylase; ↓ 1,25-(OH)₂D |
| **1,25-(OH)₂D (Calcitriol)** | ↓↓↓ | ↓ Intestinal Ca²⁺ Absorption → Hypocalcaemia |
| **Calcium** | ↓ | ↑ PTH Secretion |
| **PTH** | **↑↑↑** (Secondary HPT) | Bone Resorption; Phosphate Excretion |

## 2. Stages of CKD-MBD (KDIGO)

| CKD Stage | GFR (mL/min) | Phosphate | FGF23 | 1,25-(OH)₂D | PTH | Bone Turnover |
|-----------|--------------|-----------|-------|-------------|-----|---------------|
| **G3a (45-59)** | Mild | Normal | ↑ | Normal/↓ | ↑ (1-2x ULN) | Normal/High |
| **G3b (30-44)** | Moderate | ↑ | ↑↑ | ↓ | ↑↑ (2-5x ULN) | High |
| **G4 (15-29)** | Severe | ↑↑ | ↑↑↑ | ↓↓ | ↑↑↑ (5-10x ULN) | High/Adynamic |
| **G5 (<15) / Dialysis** | Failure | ↑↑↑ | ↑↑↑↑ | Absent | ↑↑↑↑ (10-50x ULN) | Adynamic/Mixed |

## 2. KDIGO 2017 Targets (CKD G3-5D)

| Parameter | Target (CKD 3-5D) | Target (Dialysis) |
|-----------|-------------------|-------------------|
| **PTH** | **2-9x ULN** | **2-9x ULN** |
| **Corrected Calcium** | **2.1-2.5 mmol/L** | **2.1-2.5 mmol/L** |
| **Phosphate** | **0.8-1.5 mmol/L** (Stage 3-4) | **1.1-1.5 mmol/L** (Dialysis) |
| **Ca × P Product** | **<4.5 mmol²/L²** | **<4.5 mmol²/L²** |
| **25-OH Vitamin D** | **≥30 ng/mL (75 nmol/L)** | **≥30 ng/mL** |
| **Alkaline Phosphatase** | Monitor Trends | Monitor Trends |

> **Note**: PTH Target is Wide (2-9x ULN) — Avoid Over-Suppression (Adynamic Bone Disease).

## 2. Management Algorithm (KDIGO 2017) — Stepwise

```
CKD-MBD Diagnosed (CKD G3-5, Abnormal Ca/Phos/PTH)
         │
         ▼
STEP 1: **PHOSPHATE BINDERS** (With Meals)
         │
         ├── **Calcium-Based** (Carbonate/Acetate) — **1st Line if Ca²⁺ Normal/Low**
         │       Dose: 0.5-1g with Meals; Max 1.5-2g Elemental Ca/day
         │       Watch: Hypercalcaemia, Vascular Calcification
         │
         ├── **Non-Calcium-Based** (Sevelamer, Lanthanum, Sucroferric Oxyhydroxide)
         │       **1st Line if Ca²⁺ High / Vascular Calcification / Adynamic Bone**
         │       Sevelamer 800-2400mg TDS with Meals
         │
         └── **Ferric Citrate** (Newer; Also Treats Anaemia)
                 Dose: 1g TDS with Meals
                 │
         ▼
STEP 2: **VITAMIN D ANALOGUES** (If PTH Still >9x ULN after Phosphate Control)
         │
         ├── **Alfacalcidol (1α-OH D3)** 0.25-1µg/day (or 1µg Alternate Days)
         ├── **Calcitriol (1,25-(OH)₂D)** 0.25-1µg/day (or 1µg Alternate Days)
         ├── **Paricalcitol** (Selective VDR Agonist) 1-2µg 3x/week (Post-HD)
         │       Advantage: Less Hypercalcaemia/Hyperphosphataemia
         │
         ▼
STEP 3: **CALCIMIMETICS** (If PTH Still >9x ULN Despite Above)
         │
         ├── **Cinacalcet** 30-180mg/day (BD) — CaSR Agonist
         │       Mechanism: ↓ PTH Secretion; Effect: ↓ PTH, ↓ Ca, ↓ Phos
         │       Adverse: Hypocalcaemia, Nausea, QTc Prolongation
         │       Advantage: **↓ PTH, ↓ Ca, ↓ Phos** (Unique Triple Effect)
         │
         └── **Etelcalcetide** (IV, Post-HD) 5-15mg 3x/week (HD Only)
```

## 2. Phosphate Binder Selection Algorithm

| Patient Profile | 1st Line Binder | Rationale |
|-----------------|----------------|-----------|
| **Normal Ca²⁺, No Vascular Calcification, eGFR ≥30** | **Calcium Carbonate/Acetate** | Cheap, Ca²⁺ Supplement |
| **Hypercalcaemia / Vascular Calcification / Adynamic Bone** | **Non-Calcium (Sevelamer / Lanthanum / Sucroferric / Ferric Citrate)** | Avoids Ca²⁺ Load |
| **Dialysis + Hyperphosphataemia** | **Sevelamer / Lanthanum / Ferric Citrate** | Ferric Citrate → Also Treats Anaemia |

> **Key Principle**: **Avoid Calcium-Based Binders** if Ca²⁺ High or Vascular Calcification Present.

## 2. Vitamin D Analogues in CKD

| Agent | Indication | Dose | Advantage |
|-------|-----------|------|-----------|
| **Alfacalcidol (1α-OH D3)** | CKD G3-5 | 0.25-1µg/day | Rapid 1α-Hydroxylation; Reliable |
| **Calcitriol (1,25-(OH)₂D₃)** | CKD G4-5, Dialysis | 0.25-1µg/day | Active Form; Short t½ |
| **Paricalcitol** | CKD G4-5D / Dialysis | 1-2µg 3x/week (Post-HD) | **Selective VDR** → Less Hypercalcaemia/Phosphataemia |
| **Doxercalciferol** | CKD G4-5 | 10-30µg TIW | 1α-OH D2 Analogue |

> **Key**: **Nutritional D2/D3 (Cholecalciferol/Ergocalciferol) INSUFFICIENT in CKD G4-5/Dialysis** — Requires Renal 1α-Hydroxylation.

## 2. Cinacalcet (Calcimimetic)

| Feature | Details |
|---------|---------|
| **Mechanism** | **CaSR Agonist** → ↓ PTH Secretion |
| **Dose** | 30-180mg/day (BD); Start 30mg BD, Titrate q2-4wk |
| **Effect** | **↓ PTH, ↓ Ca, ↓ Phos** (Unique Triple Effect) |
| **Adverse Effects** | **Hypocalcaemia**, Nausea, QTc Prolongation |
| **Monitoring** | Ca²⁺ q1-2wk (Start); PTH q1-3mo; QTc ECG |
| **Etelcalcetide** (IV, HD Only) | 5-15mg 3x/week Post-HD; No Oral Absorption Issues |

## 3. CKD-MBD Management Algorithm (KDIGO 2017)

```
CKD G3-5 / Dialysis + Abnormal Ca/Phos/PTH
         │
         ▼
STEP 1: DIETARY PHOSPHATE RESTRICTION (800-1000 mg/day)
         │
         ▼
STEP 2: PHOSPHATE BINDERS (With Meals)
         │       (See Selection Algorithm Above)
         │
         ▼
STEP 3: VITAMIN D ANALOGUES (If PTH >9x ULN Despite Phosphate Control)
         │       Alfacalcidol / Calcitriol / Paricalcitol
         │
         ▼
STEP 4: CALCIMIMETICS (If PTH >9x ULN Despite Above)
         │       Cinacalcet (PO) / Etelcalcetide (IV HD)
         │
         ▼
STEP 5: PARATHYROIDECTOMY (Refractory SHPT)
         │       Indications: PTH >800-1000 pg/mL; Ca >2.75; Phos >1.8; Calciphylaxis
         │       Procedure: Subtotal (3.5-Gland) + Autotransplant
```

## 3. Adynamic Bone Disease — Over-Suppression

| Feature | Recognition | Management |
|---------|-------------|--------------|
| **Clue** | PTH **<2x ULN** (or <100-150 pg/mL) + Low ALP + Low Bone Turnover | **Stop/Reduce Vitamin D Analogues & Calcimimetics** |
| **Risk** | **Fracture ↑** (Low Bone Turnover), ↑ Vascular Calcification | PTH Target **2-9x ULN** (Not Lower) |
| **Diagnosis** | **Bone Biopsy (Gold Standard)**; Low PTH + Low ALP | PTH Target **2-9x ULN** (Not Lower) |

---

## 3. Calciphylaxis (Calcific Uraemic Arteriolopathy)

| Feature | Management |
|---------|-----------|
| **Diagnosis** | Painful Violaceous Skin Lesions → Necrosis/Ulceration; Biopsy (Medial Calcification) |
| **Risk Factors** | Ca×P >4.5, Obesity, Diabetes, Warfarin, High PTH, High Calcium/Phosphate |
| **Emergency Management** | **Urgent Parathyroidectomy** (If Severe SHPT) + Intensive Wound Care |
| **Medical** | **Sodium Thiosulfate** 25g IV 3x/week (Post-HD); **Cinacalcet** (↓ PTH, Ca, Phos); **Bisphosphonates** (IV Zoledronic) |
| **Wound Care** | Multidisciplinary (Nephrology, Dermatology, Surgery, Plastics) |
| **Mortality** | **50-80%** |

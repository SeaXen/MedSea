# Lipid Disorders (Dyslipidaemia)

> [!info]
> **Dyslipidaemia = Abnormal Lipid Profile** → Atherosclerotic Cardiovascular Disease (ASCVD) Risk. **ESC/EAS 2019 / NICE Guidelines** Guide Targets. **Statin 1st Line**; **Ezetimibe/PCSK9i** Add-On. **Familial Hypercholesterolaemia (FH)** = Autosomal Dominant LDLR/APOB/PCSK9 Mutations.

## 2. Classification & Lipid Profile Interpretation

### Fredrickson/WHO Classification
| Type | Lipid Abnormality | Typical Pattern |
|------|-------------------|-----------------|
| **Type IIa** | **Isolated ↑ LDL-C** | Familial Hypercholesterolaemia (FH) |
| **Type IIb** | **↑ LDL-C + ↑ TG** | Combined Hyperlipidaemia |
| **Type III** | ↑ IDL (β-VLDL) | Dysbetalipoproteinaemia (ApoE2/E2) |
| **Type IV** | **↑ TG** (VLDL), Normal LDL, ↓ HDL | Metabolic Syndrome, Obesity, T2DM |
| **Type V** | **↑↑ TG** (Chylomicrons + VLDL) | Severe Hypertriglyceridaemia (Pancreatitis Risk) |

### Standard Lipid Profile (Fasting 8-12h)
| Parameter | Optimal / Target | High Risk / Action |
|-----------|------------------|-------------------|
| **Total Cholesterol** | <5.0 mmol/L | >5.0 |
| **LDL-C** | **<1.8 (Very High Risk)** / <2.6 (High Risk) / <3.0 (Moderate) | Per Risk Category |
| **HDL-C** | **>1.0 (M) / >1.2 (F)** | <1.0 (M) / <1.2 (F) |
| **Triglycerides** | **<1.7 mmol/L** | 1.7-2.3 (Borderline); >2.3 (High) |
| **Non-HDL-C** | **<2.6 (Very High Risk)** / <3.4 (High Risk) | Secondary Target |
| **ApoB** | <80 mg/dL (Very High Risk) | Alternative to LDL-C |
| **TC/HDL Ratio** | <4.0 | >5.0 (High Risk) |

## 2. Cardiovascular Risk Categories & LDL-C Targets (ESC/EAS 2019)

| Risk Category | Criteria | **LDL-C Target** | **LDL-C Reduction** |
|---------------|----------|------------------|---------------------|
| **Very High Risk** | Established ASCVD, Diabetes + Target Organ Damage, CKD eGFR<30, FH | **<1.4 mmol/L (55 mg/dL)** | **≥50% Reduction** |
| **High Risk** | Diabetes (no target organ), CKD eGFR 30-59, 10yr CVD Risk ≥10%, Markedly Elevated Single Risk Factor | **<1.8 mmol/L (70 mg/dL)** | **≥50% Reduction** |
| **Moderate Risk** | 10yr CVD Risk 5-10% | **<2.6 mmol/L (100 mg/dL)** | — |
| **Low Risk** | 10yr CVD Risk <5% | **<3.0 mmol/L (115 mg/dL)** | — |

> **Non-HDL-C Target** = LDL-C Target + 0.8 mmol/L (30 mg/dL)

## 2. Statin Therapy — Intensity & Selection

| Intensity | Statins & Doses | Expected LDL-C Reduction | Indication |
|-----------|-----------------|--------------------------|------------|
| **High-Intensity** | **Atorvastatin 40-80mg** / **Rosuvastatin 20-40mg** | **≥50%** | **Very High / High Risk** (1st Line) |
| **Moderate-Intensity** | Atorvastatin 10-20mg / Rosuvastatin 5-10mg / Simvastatin 20-40mg | 30-49% | Moderate Risk / Intolerance to High-Intensity |
| **Low-Intensity** | Simvastatin 10mg / Pravastatin 10-20mg / Fluvastatin 20-40mg | <30% | Low Risk / Elderly / Drug Interactions |

### Statin Choice by Clinical Scenario
| Scenario | Preferred Statin |
|----------|------------------|
| **High CVD Risk / FH** | **Atorvastatin 40-80mg** / **Rosuvastatin 20-40mg** |
| **CKD (eGFR <30)** | **Atorvastatin** (No Dose Adjustment) / **Rosuvastatin** (Max 10mg) |
| **Statin Intolerance** | Try Alternative Statin → Ezetimibe → PCSK9i → Bempedoic Acid |
| **Drug Interactions** (CYP3A4 Inhibitors) | **Rosuvastatin** / **Pravastatin** (Minimal CYP3A4) |
| **Asian Ethnicity** | **Rosuvastatin** (Lower Dose 5-10mg Often Sufficient) |

> **Check Baseline**: LFTs, CK (If >5x ULN → Do Not Start), Renal Function.

## 2. Add-On Therapy (If Target Not Met on Max Tolerated Statin)

| Add-On Agent | Dose | LDL-C Reduction | Indication |
|--------------|------|-----------------|------------|
| **Ezetimibe** | **10mg OD** | **15-20%** | **1st Add-On** (If Statin Max Tolerated / Intolerant) |
| **PCSK9 Inhibitor** | **Evolocumab 140mg SC q2wk** / **Alirocumab 75-150mg SC q2wk** | **50-60%** | **2nd Add-On** (Very High Risk, FH, Statin Intolerant) |
| **Bempedoic Acid** | **180mg OD** | **15-25%** | **Statin Intolerant** (No Myopathy Risk) |
| **Fibrates** (Fenofibrate 145mg) | 145mg OD | TG ↓30-50%, HDL ↑10-15% | **If TG >2.3 mmol/L** (Combined with Statin) |
| **Omega-3 Fatty Acids** | 2-4g/day (EPA/DHA) | TG ↓20-30% | Adjunct for Severe HyperTG |

### Treatment Algorithm (ESC/EAS)

```
MAX TOLERATED STATIN
         │
         ├── LDL-C AT TARGET → CONTINUE
         │
         └── LDL-C ABOVE TARGET
                 │
                 ├── ADD EZETIMIBE 10mg OD
                 │       ├── AT TARGET → CONTINUE
                 │       └── NOT AT TARGET (Very High Risk / FH)
                 │               → ADD PCSK9 INHIBITOR
                 │                       (Evolocumab / Alirocumab)
                 │
                 ├── IF STATIN INTOLERANT
                 │       → EZETIMIBE 1st → BEMPEODOIC ACID → PCSK9 INHIBITOR
                 │
                 └── TG >2.3 mmol/L (Despite Statin ± Ezetimibe)
                         → ADD FENOFIBRATE 145mg OD / OMEGA-3 2-4g/d
```

## 3. Familial Hypercholesterolaemia (FH)

### Genetics & Classification
| Type | Genetics | Frequency | LDL-C Level | Clinical Features |
|------|---------|-----------|-------------|-------------------|
| **Heterozygous FH (HeFH)** | **LDLR (85-90%)**, **APOB**, **PCSK9** | **1:250** | **>4.9 mmol/L (Adults)**, **>4.0 (Children)** | Tendon Xanthomas, Corneal Arcus, Premature CVD |
| **Homozygous FH (HoFH)** | **2 Mutant Alleles** (LDLR/APOB/PCSK9) | **1:300,000** | **>13 mmol/L** | **Planar Xanthomas**, Aortic Valve Disease, Childhood CVD |

### Diagnostic Criteria
| System | Criteria |
|----------|-----------|
| **Dutch Lipid Clinic Network (DLCN)** | **Points-Based**: Family History, Personal History, LDL-C, Tendon Xanthomas, Corneal Arcus, DNA Testing. **≥8 = Definite FH**; 6-8 = Probable; 3-5 = Possible |
| **Simon Broome** | **Definite FH**: LDL-C >4.9 (Adult) + Tendon Xanthomas OR DNA Mutation; **Possible FH**: LDL-C >4.9 + Family/Personal History |

### Management of FH
| Scenario | Management |
|---------|------------|
| **HeFH** | **High-Intensity Statin + Ezetimibe** → **PCSK9 Inhibitor** (If Target Not Met) |
| **HoFH** | **Statin + Ezetimibe + PCSK9i** → **Lipid Apheresis** (Regular, Lifelong) → **Liver Transplant** (Rare) |
| **Cascade Screening** | 1st-Degree Relatives → LDL-C + Genetic Testing (If Mutation Known) |
| **Children** | Statin from Age 8-10 (If LDL-C >4.0); PCSK9i from Age 12 (If Needed) |

## 3. Severe Hypertriglyceridaemia (TG >10 mmol/L / 880 mg/dL)

| Severity | TG Level | Risk | Management |
|--------|----------|------|------------|
| **Moderate** | 2.3-10 mmol/L | CVD Risk | Fibrates, Omega-3, Optimise Statin, Treat Secondary Causes |
| **Severe** | **>10 mmol/L** | **Acute Pancreatitis** | **Emergency**: Insulin Infusion, Heparin, Plasmapheresis, Fibrates, Omega-3 |

### Aetiology of Severe HyperTG
| Primary (Genetic) | Secondary (Acquired) |
|---------------------|----------------------|
| **Familial Chylomicronaemia** (LPL, ApoC-II, ApoA-V, GPIHBP1) | **Uncontrolled Diabetes** |
| **Familial Combined Hyperlipidaemia** | **Alcohol Excess** |
| | **Obesity / Metabolic Syndrome** |
| | **Hypothyroidism** |
| | **Drugs** (Oestrogen, Tamoxifen, Retinoids, Protease Inhibitors, Beta-Blockers, Thiazides) |
| | **Nephrotic Syndrome** |
| | **Pregnancy** |

### Emergency Management (TG >10 mmol/L / Pancreatitis)
| Step | Action |
|-----|--------|
| **1. IV Insulin** | **Insulin Infusion 0.1U/kg/h** + **D5W** (Promotes LPL Activity) |
| **2. Heparin** | **50-100U/kg IV Bolus** → **10-20U/kg/h** (Releases LPL) |
| **3. Plasmapheresis** | **If TG >20 mmol/L** or **Refractory Pancreatitis** |
| **4. Fibrates** | Fenofibrate 145mg OD (After Acute Phase) |
| **5. Omega-3** | 2-4g/day EPA/DHA (Adjunct) |
| **6. Treat Underlying** | Glycaemic Control, Alcohol Cessation, Drug Review |

## 3. Statin Adverse Effects & Monitoring

| Adverse Effect | Incidence | Management |
|----------------|-----------|------------|
| **Myopathy** (CK >5x ULN + Symptoms) | <0.1% (Rhabdo <0.01%) | **Stop Statin**; Check CK; Rechallenge Lower Dose/Alternative |
| **Asymptomatic CK Rise** | 0.5-2% (CK 5-10x ULN) | Continue if <5x ULN; Reduce Dose if 5-10x; Stop if >10x |
| **New-Onset Diabetes** | Slight ↑ Risk (High-Dose) | Monitor HbA1c; Benefit >> Risk in High CVD Risk |
| **Hepatotoxicity** | ALT/AST >3x ULN | Reduce/Stop if >3x ULN Persistent |
| **Drug Interactions** | CYP3A4 Inhibitors (Clarithromycin, Azoles, Grapefruit) | Avoid Simvastatin/Lovastatin >20mg; Use Rosuvastatin/Pravastatin |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Primary vs Secondary Prevention Targets** | Very High Risk (ASCVD) = <1.4; High Risk = <1.8; Moderate = <2.6 |
| **Statin Intolerance** | Try 3 Different Statins → Ezetimibe → PCSK9i → Bempedoic Acid |
| **PCSK9i vs Ezetimibe** | PCSK9i = -50-60% LDL (Injectable); Ezetimibe = -15-20% (Oral, 1st Add-On) |
| **Fibrate + Statin** | Only if TG >2.3; Fenofibrate Preferred (Less CYP Interaction) |
| **Bempedoic Acid** | ATP-Citrate Lyase Inhibitor; **No Myopathy Risk**; Statin-Intolerant |
| **HoFH** | LDL >13; Planar Xanthomas; **Lipid Apheresis** + PCSK9i + Liver Transplant |

| **Statin + Fibrate** | Only If TG >2.3; Fenofibrate Preferred (Less Interaction) |
| **Bempedoic Acid** | ATP-Citrate Lyase Inhibitor; **No Myopathy**; Statin-Intolerant |

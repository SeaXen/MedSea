# Hypophosphataemia

> [!info]
> **Hypophosphataemia = Serum PO₄ <0.8 mmol/L.** Common but Often Asymptomatic Until Severe. **Key Contexts: Refeeding Syndrome, DKA Treatment, Respiratory Alkalosis, Hungry Bone Syndrome.** **Severe (<0.3) = Life-Threatening** (Muscle Weakness, Rhabdomyolysis, Hemolysis, Respiratory Failure).

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Classify causes: redistribution (acute) vs true depletion (chronic)
- [ ] Recognise clinical consequences: muscle weakness, rhabdomyolysis, haemolysis, respiratory failure
- [ ] Apply IV phosphate replacement thresholds (severe <0.3)
- [ ] Monitor for complications (hypocalcaemia, hypokalaemia, hyperphosphataemia)
- [ ] Recognise refeeding syndrome and hungry bone syndrome

## 2. Aetiology

### Acute Redistribution (Transient, Normal Total Body Phosphate)
| Context | Mechanism |
|---------|-----------|
| **Refeeding Syndrome** | Insulin Surge → Intracellular Shift of PO₄/K⁺/Mg²⁺ |
| **DKA Treatment** | Insulin → Intracellular Shift of PO₄/K⁺/Mg²⁺ |
| **Respiratory Alkalosis** | Alkalosis → Intracellular Shift |
| **Alkalosis (Metabolic)** | Alkalosis → Intracellular Shift |
| **Hungry Bone Syndrome** (Post-PTX/Thyroidectomy) | Rapid Bone Remineralisation Sequesters PO₄/Ca²⁺/Mg²⁺ |
| **Acute Leukaemia / Tumour Lysis** | Massive Cellular Proliferation Sequesters PO₄ |

### True Depletion (Chronic, Depleted Total Body Phosphate)
| Cause | Mechanism |
|------|-----------|
| **Malabsorption** (Coeliac, Short Bowel, Bariatric) | ↓ Intestinal Absorption |
| **Chronic Alcoholism** | Poor Intake + Renal Losses + Malabsorption |
| **Antacids** (Al/ Mg/ Ca-Containing) | Bind Dietary Phosphate |
| **Vitamin D Deficiency** | ↓ Intestinal PO₄ Absorption |
| **Renal Losses** (Fanconi, X-Linked Hypophosphataemic Rickets, Oncogenic Osteomalacia, Diuretics) | Renal Phosphate Wasting |
| **Hyperparathyroidism** (Primary/Secondary) | PTH → ↓ Proximal Reabsorption |

## 2. Clinical Features

| System | Features |
|-------|----------|
| **Musculoskeletal** | **Proximal Myopathy** (Weakness, Difficulty Rising), Rhabdomyolysis (Severe <0.3), Osteomalacia (Chronic) |
| **Haematological** | **Haemolytic Anaemia** (RBC 2,3-DPG Depletion → Rigid Cells), Leukocyte Dysfunction, Platelet Dysfunction |
| **Respiratory** | **Respiratory Failure** (Diaphragm Weakness), Difficulty Weaning from Ventilator |
| **Cardiac** | Cardiomyopathy, Arrhythmias, Decreased Contractility |
| **Neurological** | Confusion, Seizures (Severe), Coma |
| **Renal** | Proximal Tubular Dysfunction (Fanconi-Like) |

### Severity Grading
| Severity | Phosphate (mmol/L) | Clinical Features |
|---------------|-------------------|-------------------|
| **Mild** | 0.6-0.8 | Asymptomatic or Mild Fatigue |
| **Moderate** | 0.3-0.6 | Myopathy, Weakness, Bone Pain |
| **Severe** | **<0.3** | **Life-Threatening**: Rhabdomyolysis, Haemolysis, Respiratory Failure, Cardiac Failure, Seizures, Encephalopathy |

## 3. Diagnosis & Investigations

| Test | Normal | Hypophosphataemia | Interpretation |
|--------|--------|-------------------|----------------|
| **Serum PO₄** | 0.8-1.5 mmol/L | **<0.8** | Confirm Diagnosis |
| **Serum Ca²⁺** | 2.15-2.55 | Low/Normal | Check for Associated Hypocalcaemia |
| **Serum Mg²⁺** | 0.7-1.1 | Low/Normal | Often Associated Hypomagnesaemia |
| **Serum ALP** | 30-130 U/L | **High** (If Osteomalacia/Rickets) | Bone Turnover Marker |
| **PTH** | 1.6-6.9 pmol/L | Normal/High | Check for Hyperparathyroidism |
| **25-OH Vitamin D** | >50 nmol/L | Low | Vitamin D Deficiency |
| **Urine PO₄** | 13-42 mmol/24h | High (If Renal Losses) | FEP = (Urine PO₄ × Serum Creatinine) / (Serum PO₄ × Urine Creatinine) >5% = Renal Wasting |
| **FGF23** | <100 RU/mL | High | Oncogenic Osteomalacia, XLH, ADHR |

## 2. Aetiological Classification

### Redistribution (Acute, Normal Total Body Stores)
| Context | Mechanism | Key Features |
|---------|-----------|--------------|
| **Refeeding Syndrome** | Insulin → PO₄/K⁺/Mg²⁺ Shift Into Cells | High Risk: BMI<16, Weight Loss>15%, NBM>10d, Alcoholism |
| **DKA Treatment** | Insulin + Glucose → Intracellular Shift | Monitor q6h Post-Insulin |
| **Respiratory Alkalosis** | Alkalosis → Intracellular Shift | Hyperventilation, Sepsis, Salicylates |
| **Hungry Bone Syndrome** | Post-PTX/Thyroidectomy Bone Remineralisation | Post-Op Day 1-3; Monitor Ca²⁺, PO₄, Mg²⁺ q6-12h |

### True Depletion (Chronic, Depleted Stores)
| Cause | Mechanism | Key Features |
|-------|-----------|--------------|
| **Malabsorption** | Coeliac, Crohn's, Short Bowel, Bariatric | Low 25-OH Vit D, Steatorrhoea |
| **Alcoholism** | Poor Intake + Renal Losses + Malnutrition | Hypomagnesaemia Often Coexists |
| **Antacids** (Al/Mg/Ca-Based) | Bind Luminal Phosphate | Chronic Use |
| **Vitamin D Deficiency** | ↓ Intestinal Absorption | Low 25-OH D, ↑ ALP, ↑ PTH |
| **Renal Phosphate Wasting** | Fanconi, XLH, Oncogenic Osteomalacia, Diuretics | FGF23 High (XLH, Oncogenic), FEP >5% |

| Severity | PO₄ (mmol/L) | Clinical Manifestations |
|----------|--------------|------------------------|
| **Mild** | 0.6-0.8 | Often Asymptomatic; Fatigue, Anorexia |

## 3. Management

### IV Phosphate Replacement (Severe <0.3 or Symptomatic Moderate)
| Parameter | Protocol |
|-----------|----------|
| **Indication** | **PO₄ <0.3 mmol/L** OR Symptomatic Moderate (0.3-0.6 with Symptoms) |
| **Preparation** | **Sodium/Potassium Phosphate** (e.g., Phosphate Sandoz, Neutraphos) |
| **Dose** | **15-30 mmol IV over 6-12 Hours** (Dilute in 500-1000mL 5% Dextrose/0.9% NaCl) |
| **Max Rate** | 0.2 mmol/kg/h (Max 15 mmol/h) |
| **Monitoring** | **Ca²⁺, Mg²⁺, K⁺, Creatinine q6h**; ECG Continuous; PO₄ q6h |

### Oral Phosphate Replacement (Mild/Moderate / Chronic)
| Preparation | Dose |
|-------------|------|
| **Phosphate Sandoz** (Na/K Phosphate) | 1-2 Tablets TDS (Each ~16 mmol PO₄) |
| **Neutraphos** | 1-2 Packets QID (Each 8 mmol PO₄) |
| **Jouvence** (Organic Phosphate) | 1-2 Ampoules TDS |
| **Dietary** | Dairy, Meat, Fish, Eggs, Nuts, Legumes |

---

## 3. Monitoring During Replacement

| Parameter | Frequency | Target / Precaution |
|------------|-----------|---------------------|
| **Serum PO₄** | q6h (IV); Daily (PO) | Normalise >0.8; **Avoid >1.5 (Hyperphosphataemia)** |
| **Serum Ca²⁺** | q6h (IV); Daily (PO) | **Monitor for Hypocalcaemia** (PO₄ Binds Ca²⁺) |
| **Serum Mg²⁺** | q6-12h | Correct if Low (Commonly Coexists) |
| **Serum K⁺** | q6-12h | Often Low (Associated) |
| **Creatinine / U&Es** | q12-24h | Renal Impairment → Reduce Dose / Avoid |
| **Urine Output** | Hourly | Ensure Adequate for Excretion |

## 3. Special Contexts

### Refeeding Syndrome
| Feature | Management |
|---------|------------|
| **At Risk** | BMI <16, Weight Loss >15%, NBM >10 Days, Alcoholism, Anorexia |
| **Mechanism** | Insulin Surge → K⁺/Mg²⁺/PO₄ Shift Into Cells |
| **Prophylaxis** | **Thiamine 100mg IV/PO Before Feeding** + **IV Phosphate 15-30mmol over 24h** + **IV Magnesium** + **IV Potassium** |
| **Feeding** | Start 10 kcal/kg/d; Increase Slowly Over 5-7 Days |
| **Monitoring** | **PO₄, K⁺, Mg²⁺, Ca²⁺ q6h × 72h**; ECG Continuous |

### Hungry Bone Syndrome (Post-PTX/Thyroidectomy)
| Feature | Management |
|---------|------------|
| **Mechanism** | Rapid Bone Remineralisation Sequesters Ca²⁺/PO₄/Mg²⁺ |
| **Onset** | Post-Op Day 1-3 |
| **Monitoring** | **Ca²⁺, PO₄, Mg²⁺ q6-12h × 7-10 Days** |
| **Replacement** | **IV Ca Gluconate + IV PO₄ + IV Mg²⁺** (Aggressive, Protocoled) |
| **Duration** | 1-3 Weeks (Until Bone Remineralisation Complete) |

### Oncogenic Osteomalacia
| Feature | Details |
|---------|---------|
| **Mechanism** | Tumour Secretes FGF23 → Renal Phosphate Wasting + ↓ 1,25-(OH)₂D |
| **Tumours** | Mesenchymal (Haemangiopericytoma, Fibrous Tumours) |
| **Biochemical** | **↓ PO₄, ↑ FGF23, FEP >5%, Normal PTH, Low 1,25-(OH)₂D** |
| **Management** | **Tumour Resection (Curative)**; Phosphate + Calcitriol Pre-Op |

### X-Linked Hypophosphataemic Rickets (XLH)
| Feature | Details |
|---------|---------|
| **Gene** | **PHEX** (X-Linked Dominant) |
| **Mechanism** | ↓ PHEX → ↑ FGF23 → Renal Phosphate Wasting + ↓ 1,25-(OH)₂D |
| **Clinical** | Rickets (Childhood), Osteomalacia (Adult), Short Stature, Dental Abscesses |
| **Management** | **Burosumab** (Anti-FGF23 mAb) 0.8mg/kg SC q2wk (Child) / 1mg/kg q4wk (Adult); Phosphate + Calcitriol (Traditional) |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Redistribution vs Depletion** | Redistribution = Acute (Refeeding, DKA, Alkalosis); Depletion = Chronic (Malabsorption, Antacids, Renal Losses) |
| **Hypophosphataemia in DKA** | Insulin Drives PO₄ Into Cells; Monitor q6h Post-Insulin |
| **IV PO₄ Monitoring** | **Always Monitor Ca²⁺** (PO₄ Binds Ca²⁺ → Hypocalcaemia) |
| **Refeeding Algorithm** | **Thiamine 100mg FIRST** → Slow Feeding (10 kcal/kg/d) + PO₄/K⁺/Mg²⁺/Thiamine |
| **Hungry Bone** | Post-PTX → Bone Remineralisation → **Ca/PO₄/Mg Sequestration** |
| **Oncogenic Osteomalacia** | FGF23-Secreting Tumour → **Resection Curative** |
| **XLH Treatment** | **Burosumab (Anti-FGF23)** 1st Line (Replaces Phosphate + Calcitriol) |
| **IV PO₄ Max Rate** | 0.2 mmol/kg/h (Max 15 mmol/h) |
| **IV PO₄ Precaution** | **Hypocalcaemia Risk** (PO₄ Binds Ca²⁺); **Renal Impairment → Reduce/Avoid** |
| **FGF23** | Key Mediator (XLH, Oncogenic, ADHR): ↑ FGF23 → ↓ Renal PO₄ Reabsorption |

# Haemochromatosis (Iron Overload)

> [!info]
> **Hereditary Haemochromatosis = Autosomal Recessive Iron Overload Disorder**. **HFE C282Y Homozygosity = 90% of Cases**. **Secondary Iron Overload** (Transfusional, Ineffective Erythropoiesis). **Venesection (Phlebotomy) = Gold Standard Treatment**. **Triad: Cirrhosis, Diabetes, Skin Pigmentation**.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Diagnose HFE haemochromatosis using TSAT, Ferritin, and Genetic Testing
- [ ] Differentiate primary (HFE) from secondary iron overload
- [ ] Apply venesection protocol (induction → maintenance)
- [ ] Recognise organ complications: cirrhosis, diabetes, cardiomyopathy, hypogonadism, arthropathy
- [ ] Apply screening and surveillance for family members

## 2. Classification

| Type | Gene | Frequency | Features |
|------|------|-----------|----------|
| **Type 1 (HFE)** | **HFE** (C282Y Homozygous 90%; H63D Compound Het) | **90%** | Adult-Onset; Northern European Ancestry |
| **Type 2A (Juvenile)** | **HJV** (Hepcidin Antimicrobial Peptide) | Rare | Severe, Early-Onset (20); Cardiac/Endocrine |
| **Type 2B (Juvenile)** | **HAMP** (Hepcidin) | Rare | Similar to Type 2A |
| **Type 3** | **TFR2** (Transferrin Receptor 2) | Rare | Intermediate Severity |
| **Type 4 (Ferroportin Disease)** | **SLC40A1** (Ferroportin) | Rare | **Autosomal Dominant**; Macrophage Iron Retention (High Ferritin, Normal/High TSAT) |
| **Secondary** | Transfusional, Ineffective Erythropoiesis (Thalassaemia, Sideroblastic) | Common | Acquired; No Genetic Defect |

---

## 2. Pathophysiology

| Step | Mechanism |
|------|-----------|
| **1. HFE Mutation (C282Y)** | ↓ HFE-αβ2M Interaction → ↓ Hepcidin Expression |
| **2. Low Hepcidin** | ↓ Hepcidin → ↑ Ferroportin Activity (Enterocytes, Macrophages) |
| **3. Increased Iron Absorption** | ↑ Dietary Iron Absorption (No Feedback Inhibition) |
| **4. Iron Overload** | Progressive Deposition: Liver → Pancreas → Heart → Joints → Skin → Pituitary → Gonads |
| **5. Organ Damage** | Oxidative Stress (Fenton Reaction) → Fibrosis, Cell Death, Dysfunction |

### Hepcidin-Ferroportin Axis
| Component | Normal Function | Haemochromatosis |
|---------|----------------|------------------|
| **Hepcidin** | Hormone from Liver → Binds Ferroportin → Degradation | **Low** (HFE Mutation → ↓ Hepcidin Transcription) |
| **Ferroportin** | Iron Exporter (Enterocytes, Macrophages, Hepatocytes) | **Unregulated** (No Hepcidin → Ferroportin Stable on Membrane) |
| **Result** | Iron Excretion = Absorption | **Positive Iron Balance** → Accumulation |

## 2. Clinical Presentation

### Classic Triad (Late Presentation)
| Organ | Manifestation |
|------|---------------|
| **Liver** | **Cirrhosis** (Most Common), Hepatomegaly, HCC (20-30% with Cirrhosis) |
| **Pancreas** | **Diabetes Mellitus** ("Bronze Diabetes") — 30-60% |
| **Skin** | **Bronze/Grey Pigmentation** (Melanin + Iron), Slate-Grey Colour |

### Systemic Manifestations
| System | Manifestations |
|-------|----------------|
| **Cardiac** | **Cardiomyopathy** (Dilated/Restrictive), Heart Failure, Arrhythmias, Conduction Defects |
| **Endocrine** | **Hypogonadotrophic Hypogonadism** (Impotence, Loss of Libido, Amenorrhoea), Thyroid Dysfunction |
| **Joints** | **Arthropathy** (2nd/3rd MCP Joints — "Iron Hand"), Chondrocalcinosis (Pseudogout) |
| **Liver** | Hepatomegaly, **Cirrhosis (50-70%)**, **HCC** (20-30% if Cirrhosis) |
| **Skin** | **Bronze/Grey Pigmentation** (Face, Hands, Genitals) |

### Age of Onset
| Presentation | Typical Age |
|-------------|-------------|
| **Biochemical Iron Overload** | 20-30 Years |
| **Symptomatic Disease** | **40-60 Years** (Male Earlier Than Female) |
| **Cirrhosis/Diabetes** | 50-70 Years |

## 2. Diagnosis

### Biochemical Screening
| Test | Normal | Haemochromatosis |
|------|--------|------------------|
| **Transferrin Saturation (TSAT)** | **<45%** | **>45%** (Early Marker); **>55%** Strongly Suggestive |
| **Serum Ferritin** | **30-300 µg/L (M)** / **15-200 µg/L (F)** | **>300 (M) / >200 (F) µg/L** (Iron Overload) |
| **Liver Function Tests** | Normal | ↑ ALT/AST (If Hepatitis/Cirrhosis) |
| **Glucose** | Normal | **Impaired Fasting Glucose / Diabetes** |
| **Hormones** | Normal | ↓ Testosterone/Oestradiol, ↑ LH/FSH (Hypogonadotrophic Hypogonadism) |

### Genetic Testing (Confirmatory)
| Genotype | Frequency | Significance |
|----------|-----------|--------------|
| **C282Y Homozygous** | **90%** of Clinical Haemochromatosis | **Diagnostic** (Penetrance ~50% Biochemical, ~10-30% Clinical) |
| **C282Y/H63D Compound Heterozygous** | ~5% | Mild Iron Overload (Lower Penetrance) |
| **H63D Homozygous** | Rare | Usually No Significant Iron Overload |
| **Non-HFE** (HJV, HAMP, TFR2, SLC40A1) | Rare | Juvenile, TFR2, Ferroportin Disease |

### Liver Iron Quantification
| Method | Indication | Normal / Thresholds |
|--------|-----------|---------------------|
| **MRI Liver (FerriScan / R2*)** | Non-Invasive Quantification | **HIC >1.9 mg/g (Mild), >3.8 (Moderate), >7.0 (Severe) mg/g DW** |
| **Liver Biopsy** | If Non-HFE / Ferritin >1000 / Diagnostic Uncertainty | **HIC >1.9 (Mild), >3.8 (Moderate), >7.0 (Severe) mg/g DW** Hepatic Iron Index (HII) = HIC / Age |

## 3. Management — Venesection (Phlebotomy)

### Induction Phase
| Parameter | Protocol |
|-----------|----------|
| **Volume** | **400-500 mL** Whole Blood (≈200-250 mg Iron per Unit) |
| **Frequency** | **Weekly / Biweekly** |
| **Target** | **Ferritin 50-100 µg/L**; **TSAT <50%** |
| **Duration** | **Induction: 1-2 Years** (Weekly → Biweekly → Monthly) |

### Maintenance Phase
| Parameter | Protocol |
|-----------|----------|
| **Volume** | **400-500 mL** |
| **Frequency** | **Every 3-4 Months** (Lifelong) |
| **Target** | **Ferritin 50-100 µg/L**; **TSAT <50%** |
| **Monitoring** | **Ferritin, TSAT, Hb q3-4mo** |

### Iron Removal Calculation
| Parameter | Value |
|---------|-------|
| **Iron per Unit Blood** | **200-250 mg** (1 Unit = 450-500mL; Hb ~14g/dL → 200-250 mg Fe) |
| **Total Body Iron Excess** | **Men: 5-10g**; **Women: 3-5g** |
| **Time to De-Iron** | **1-2 Years Weekly Phlebotomy** |

## 3. Contraindications / Alternatives to Venesection

| Contraindication | Alternative |
|------------------|-------------|
| **Anaemia** (Hb <11 g/dL) | **Chelation** (Deferasirox / Deferiprone / Deferoxamine) |
| **Cardiac Failure** (Low Output) | **Chelation** |
| **Poor Venous Access** | **Chelation** |
| **Patient Refusal** | **Chelation** |

### Chelation Therapy (Second-Line)
| Agent | Dose | Monitoring |
|--------|------|------------|
| **Deferasirox** | **10-20 mg/kg/day OD** | Renal Function, LFTs, GI SE, Auditory/Ophthalmic |
| **Deferiprone** | **75 mg/kg/day TDS** | **Agranulocytosis** (Weekly FBC), Arthralgia, Zinc Deficiency |
| **Deferoxamine** | **30-50 mg/kg/day SC/IV (8-12h)** | Infusion Site Reactions, Ocular/Auditory Toxicity |

---

## 3. Organ Complication Management

| Complication | Management |
|--------------|------------|
| **Cirrhosis** | Standard Care; **HCC Surveillance** (US q6mo if Cirrhosis) |
| **Diabetes** | **Insulin** (Often Required); Standard DM Management |
| **Cardiomyopathy** | Standard HF Rx (ACEi, β-Blocker, MRA, SGLT2i, Loop Diuretic) |
| **Hypogonadism** | **Testosterone / Oestrogen Replacement** (If Symptomatic) |
| **Arthropathy** | NSAIDs, Joint Replacement (If Severe) |
| **HCC** | **US q6mo** (If Cirrhosis); AFP Monitoring |

## 3. Screening & Surveillance

### Cascade Screening (Family Members)
| Relative | Testing |
|----------|---------|
| **1st-Degree** (Siblings, Parents, Children) | **HFE Genotyping** + **TSAT + Ferritin** |
| **If C282Y Homozygous** | **TSAT >45% + Ferritin >300 (M)/200 (F)** = Biochemical Iron Overload |
| **Children** | Test at Age 18 (Unless Symptomatic Earlier) |

### Surveillance (Post-Diagnosis)
| Parameter | Frequency |
|-----------|-----------|
| **Ferritin / TSAT** | q3-4mo (Maintenance) |
| **Liver Function** | q6-12mo |
| **Glucose / HbA1c** | q6-12mo |
| **Echocardiogram** | Annual (If Cardiomyopathy Suspected) |
| **HCC Surveillance** | **US q6mo** (If Cirrhosis) + AFP q6mo |
| **Bone Density** | DEXA q2-3yr (Osteoporosis Risk) |
| **Endocrine** | Annual (Testosterone/Oestradiol, TSH, Prolactin) |

---

## 3. Special Populations

### Pregnancy
| Aspect | Management |
|--------|------------|
| **Venesection** | **Continue if Ferritin >50** (Target 50-100 µg/L); Avoid if Anaemic |
| **Iron Supplementation** | **Contraindicated** (Excess Iron Harmful) |
| **Monitoring** | Ferritin, Hb, TSAT qTrimester |
| **Genetic Counselling** | Offer to Partner; Foetal Risk if Both Parents Carriers |

### Neonatal/Childhood
| Aspect | Management |
|--------|------------|
| **Screening** | Not Routine (Adult-Onset); Test at 18 If Family History |
| **Juvenile Haemochromatosis** | HJV/HAMP Mutations (Types 2A/2B); Severe, Early-Onset; Cardio/Endocrine Dominant |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Ferritin vs TSAT** | **TSAT = Early Marker** (Rises First); Ferritin = Storage (Also Acute Phase Reactant) |
| **HFE C282Y vs H63D** | C282Y Homozygous = 90% Clinical HH; H63D Compound Het = Mild, Low Penetrance |
| **Ferritin in Inflammation** | **Ferritin = Acute Phase Reactant** (↑ in Infection, Liver Disease, Malignancy) → TSAT More Specific |
| **Type 4 (Ferroportin)** | **AD Inheritance**; High Ferritin, Normal/High TSAT; Macrophage Iron Retention |
| **Secondary Iron Overload** | Transfusional (Thalassaemia, SCD), Ineffective Erythropoiesis (Sideroblastic) |
| **Venesection Target** | **Ferritin 50-100 µg/L; TSAT <50%** |
| **Iron per Unit** | **200-250 mg Fe** (1 Unit = 450-500mL Whole Blood) |
| **Cardiac Iron** | **MRI T2* (Cardiac Iron)**; Chelation if T2* <20ms |
| **Liver Iron** | **MRI R2* (FerriScan)** or **Biopsy HIC** |
| **Type 4 Ferroportin** | **AD**; High Ferritin, Normal/High TSAT; Macrophage Iron Retention |
| **Secondary Iron Overload** | Transfusion, Ineffective Erythropoiesis |
| **Venesection Contraindicated** | Anaemia, Cardiac Failure, Poor Access → Chelation |
| **C282Y Penetrance** | ~50% Biochemical; ~10-30% Clinical Disease |
| **Homozygous C282Y** | **90% of HH**; Test Family Members (Siblings 25% Risk) |
| **MRI Liver Iron** | **FerriScan (R2*)**; HIC >1.9 Mild, >3.8 Mod, >7.0 Severe mg/g DW |

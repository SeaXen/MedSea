# Amyloidosis

> [!info]
> **Amyloidosis = Extracellular Deposition of Misfolded Protein Fibrils** (β-Pleated Sheets) → Organ Dysfunction. **Classification by Precursor Protein**: AL (Light Chain), AA (Serum Amyloid A), ATTR (Transthyretin), Others. **Diagnosis = Tissue Biopsy + Congo Red (Apple-Green Birefringence) + Typing**.

---

## 2. Classification by Precursor Protein

| Type | Precursor Protein | Source | Associated Conditions |
|------|-------------------|--------|----------------------|
| **AL** (Primary) | **Immunoglobulin Light Chain** (κ/λ) | Plasma Cell Dyscrasia | **Multiple Myeloma**, MGUS, Waldenström, B-Cell Lymphoma |
| **AA** (Secondary) | **Serum Amyloid A (SAA)** | Acute Phase Reactant | **Chronic Inflammation**: RA, IBD, TB, FMF, Bronchiectasis |
| **ATTRv** (Hereditary) | **Transthyretin (TTR) Mutant** | Genetic (TTR Mutation) | **Familial Amyloid Polyneuropathy (FAP)**, Cardiomyopathy |
| **ATTRwt** (Wild-Type) | **Transthyretin (Wild-Type)** | Age-Related | **Senile Systemic Amyloidosis** (Elderly Males; Cardiac) |
| **ALECT2** | Leukocyte Chemotactic Factor 2 | Unknown | Renal (Hispanic/Asian) |
| **ApoAI/II** | Apolipoprotein A-I/II | Genetic | Renal, Hepatic, Neuropathy |
| **Fibrinogen** | Fibrinogen α-Chain | Genetic | Renal |
| **Lysozyme** | Lysozyme | Genetic | Renal, GI, Hepatic |

## 2. AL Amyloidosis (Primary / Light Chain)

### Epidemiology & Pathogenesis
| Feature | Details |
|---------|---------|
| **Incidence** | ~9-12/million/year; Median Age 65 |
| **Pathogenesis** | **Clonal Plasma Cells** → Monoclonal Light Chain (κ > λ) → Misfolding → Fibril Deposition |
| **Association** | **10-15% Multiple Myeloma**; 30-40% MGUS; Can Occur in Isolation |

### Clinical Manifestations
| Organ | Features |
|-------|----------|
| **Renal** | **Nephrotic Syndrome** (Proteinuria >3.5g/d), CKD, End-Stage Renal Disease |
| **Cardiac** | **Restrictive Cardiomyopathy**, HFpEF, Arrhythmias, Conduction Disease, Low Voltage ECG |
| **Neurological** | Peripheral Neuropathy (Sensory > Motor), Carpal Tunnel Syndrome, Autonomic Neuropathy |
| **Hepatic** | Hepatomegaly, Cholestasis, Elevated ALP |
| **GI** | Macroglossia, GI Motility Disorders, Bleeding (Amyloid Vascular Fragility) |
| **Soft Tissue** | **Periorbital Purpura**, Shoulder Pad Sign, Subcutaneous Nodules |
| **Haematological** | **Factor X Deficiency** (Adsorption to Amyloid), Coagulopathy |

### Diagnosis
| Test | Finding |
|------|---------|
| **Tissue Biopsy + Congo Red** | **Apple-Green Birefringence** (Gold Standard) |
| **Mass Spectrometry / IHC** | **AL-κ or AL-λ** (Typing) |
| **Serum Free Light Chains (FLC)** | **Abnormal κ/λ Ratio** (Abnormal Ratio; Normal Total) |
| **Bone Marrow Biopsy** | **Clonal Plasma Cells** (% Plasma Cells) |
| **Cardiac MRI / Echo** | Late Gadolinium Enhancement, Granular Sparkling |

### Management
| Therapy | Details |
|---------|---------|
| **First-Line** | **Borortezomib-Based** (CyBorD, Dara-CyBorD) — Target Plasma Cells |
| **Autologous SCT** | **Eligible**: Age <70, Good PS, ≤2 Organ Involvement, Cardiac Stage I-II |
| **Supportive** | Diuretics, ACEi/ARB (Cautious), Compression Stockings, Anticoagulation (If Factor X Deficiency) |
| **Emerging** | Daratumumab (Anti-CD38), CAR-T, Bispecific Antibodies |

### Prognosis
| Stage (Mayo 2004) | Median Survival |
|-------------------|-----------------|
| **Stage I** (Cardiac Biomarkers Normal) | **>10 Years** |
| **Stage II** (One Elevated) | ~5 Years |
| **Stage III** (Both Elevated) | ~2 Years |
| **Stage IV** (Three Elevated) | <1 Year |

## 2. AA Amyloidosis (Secondary / Reactive)

### Pathogenesis
| Feature | Details |
|---------|---------|
| **Precursor** | **Serum Amyloid A (SAA)** — Acute Phase Reactant (IL-6 Driven) |
| **Trigger** | **Chronic Inflammation** → Sustained High SAA → Fibril Formation |
| **Associated Conditions** | **RA, JIA, AS, PsA, IBD, FMF, TB, Bronchiectasis, Chronic Osteomyelitis, Castleman Disease** |

### Clinical Manifestations
| Organ | Features |
|-------|----------|
| **Renal** | **Nephrotic Syndrome**, CKD (Most Common Presentation) |
| **Hepatic** | Hepatomegaly, Splenomegaly |
| **Cardiac** | Rare (Unlike AL) |
| **Neurological** | Rare |

### Management
| Strategy | Details |
|----------|---------|
| **Primary** | **Treat Underlying Inflammation** (TNF Inhibitors, IL-1/6 Blockers, DMARDs) |
| **FMF** | **Colchicine** (Prophylaxis + Treatment) |
| **Supportive** | Diuretics, ACEi/ARB (Careful), Renal Replacement if ESRD |
| **Prognosis** | **Better than AL** if Inflammation Controlled |

## 2. ATTR Amyloidosis (Transthyretin)

### ATTRv (Hereditary / Variant)
| Feature | Details |
|---------|---------|
| **Genetics** | **>130 TTR Mutations** (V30M Most Common in Portugal/Japan; T60A in UK/Ireland) |
| **Inheritance** | **Autosomal Dominant** (Penetrance Variable) |
| **Phenotypes** | **FAP** (Neuropathy + Autonomic), **Cardiomyopathy**, **Mixed** |

### ATTRwt (Wild-Type / Senile Systemic)
| Feature | Details |
|---------|---------|
| **Demographics** | **Elderly Males** (>70 Years) |
| **Presentation** | **Cardiomyopathy** (HFpEF, Conduction Disease), Carpal Tunnel (Precedes by Years) |
| **Genetics** | **Wild-Type TTR** (No Mutation) |

### Diagnosis
| Test | Finding |
|------|---------|
| **Tissue Biopsy + Congo Red** | Apple-Green Birefringence |
| **Mass Spec / IHC** | **ATTR** (Transthyretin) |
| **PYP/DPD Scintigraphy** | **Grade 1-3 Cardiac Uptake** (Non-Invasive Diagnosis for ATTR) |
| **Genetic Testing** | **TTR Mutation** (ATTRv); Wild-Type (ATTRwt) |
| **Cardiac MRI / Echo** | Late Gadolinium Enhancement, Granular Sparkling, Apical Sparing |

### Management
| Type | Therapy |
|------|---------|
| **ATTRv / ATTRwt (Cardiac)** | **Tafamidis 61mg OD** (1st Line; TTR Stabiliser — ATTR-ACT Trial), **Diflunisal** (Neuropathy; Off-Label) |
| **ATTRv (Neuropathy)** | **Gene Silencers**: **Patisiran** (siRNA IV q3wk), **Inotersen** (ASO SC Weekly), **Vutrisiran** (siRNA SC q3mo) |
| **ATTRv** | **Liver Transplant** (Curative for Variant Production) + Cardiac Tx if Needed |
| **Supportive** | Heart Failure Rx (Diuretics, Avoid β-Blockers Initially in ATTR), Pacemaker/ICD, Autonomic Neuropathy Rx |

## 3. Diagnostic Pathway

```
SUSPECTED AMYLOIDOSIS (Organ Dysfunction + Proteinuria/Cardiomyopathy/Neuropathy)
         │
         ▼
TISSUE BIOPSY + CONGO RED STAIN
         │
         ├── **APPLE-GREEN BIREFRINGENCE** → **AMYLOID CONFIRMED**
         │       │
         │       ▼
         │   TYPING (Mass Spectrometry / IHC)
         │       │
         │       ├── **AL (κ/λ)** → Serum FLC, Bone Marrow, Plasma Cell Disorder Workup
         │       ├── **AA** → Inflammatory Workup (ESR/CRP, Autoimmune Screen, Infection Screen)
         │       ├── **ATTR** → PYP Scan, Genetic Testing (TTR), Family History
         │       └── **OTHER** → Genetic Testing (ALECT2, ApoAI, Fibrinogen, Lysozyme)
         │
         └── **NEGATIVE** → Consider Other Diagnoses
```

## 3. Treatment Summary by Type

| Type | First-Line | Second-Line / Advanced |
|------|------------|------------------------|
| **AL** | **Borotetomib-Based (CyBorD/Dara-CyBorD)** → **Auto-SCT** (If Eligible) | Daratumumab, CAR-T, Clinical Trials |
| **AA** | **Treat Underlying Inflammation** (TNF/IL-1/6 Inhibitors, Colchicine for FMF) | — |
| **ATTRv (Neuro)** | **Patisiran / Inotersen / Vutrisiran** (Gene Silencers) | Liver Transplant (+ Cardiac) |

| **ATTRwt** | **Tafamidis** | Gene Silencers, Heart Transplant |
| **ALL** | **Supportive**: Diuretics, ACEi/ARB (Cautious), Compression, Anticoagulation (Factor X), Palliative Care |

## 8. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **AL vs AA** | AL = Light Chain (Plasma Cell); AA = SAA (Inflammation) |
| **ATTRv vs ATTRwt** | ATTRv = Genetic (TTR Mutation); ATTRwt = Age-Related (Wild-Type) |
| **AL vs ATTR Cardiac** | AL = Restrictive CM + Low Voltage ECG + Factor X Deficiency; ATTR = Thick Walls + Apical Sparing + Carpal Tunnel |
| **Typing** | **Mass Spec / IHC on Biopsy** (Not Serum); AL = κ/λ; AA = SAA; ATTR = TTR |
| **PYP Scan** | **ATTR Cardiac = Grade 1-3**; AL = Grade 0 |
| **AL vs AA Renal** | Both = Nephrotic; AL = FLC Ratio Abnormal; AA = High CRP/ESR + Chronic Inflammation |
| **ATTRv vs ATTRwt Carpal Tunnel** | ATTRv = Earlier/Younger; ATTRwt = Elderly Males, Years Before Cardiac |
| **Tafamidis** | **TTR Stabiliser** (1st Line ATTR Cardiac); Not for AL/AA |
| **Gene Silencers** | Patisiran/Inotersen/Vutrisiran = **ATTRv Neuropathy** (Not Cardiac 1st Line) |
| **Liver Transplant** | **Curative for ATTRv** (Stops Variant TTR Production) |
| **Congo Red** | **Apple-Green Birefringence** = Amyloid (Gold Standard) |
| **Free Light Chains** | AL = **Abnormal κ/λ Ratio**; Total FLC May Be Normal |
| **AL Factor X Deficiency** | Adsorption to Amyloid → Bleeding Risk |
| **AA Trigger** | Chronic Inflammation (RA, FMF, TB, IBD, Bronchiectasis) |

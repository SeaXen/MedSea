# Graves Disease

> [!info]
> **Graves Disease = Autoimmune Hyperthyroidism** driven by **TSH Receptor Stimulating Antibodies (TRAb)**. **Commonest cause of hyperthyroidism (70-80%)**. Triad: Hyperthyroidism + Ophthalmopathy + Dermopathy. **Autoimmune, Female Predominance (5-10:1), Peak 30-50y**.

## 2. Pathophysiology

| Component | Details |
|-----------|---------|
| **Autoantibody** | **TRAb (TSH Receptor Antibodies)** — **Stimulating Type** (Mimics TSH) |
| **Target** | **TSH Receptor (TSH-R)** on Thyroid Follicular Cells |
| **Mechanism** | **Constitutive Activation** → ↑ cAMP → ↑ T4/T3 Synthesis/Secretion + Thyroid Growth |
| **Other Antibodies** | **TPOAb** (80-90%), **TgAb** (50-70%) — Support Autoimmune Aetiology |
| **Genetics** | **HLA-DR3** (Strongest Association); **CTLA-4, PTPN22, FCRL3, CD40** |
| **Trigger** | Stress, Smoking, Infection, Postpartum, Iodine Excess, Immune Checkpoint Inhibitors |

## 3. Clinical Features

### Thyrotoxicosis (General)
| Feature | Frequency |
|---------|-----------|
| Weight Loss + ↑ Appetite | >90% |
| Heat Intolerance, Sweating | >90% |
| Tachycardia, Palpitations | >90% |
| Tremor, Anxiety, Irritability | >80% |
| Diarrhoea, ↑ Bowel Frequency | 50-60% |
| Menstrual Irregularities | Common (Women) |
| Proximal Myopathy | 30-40% |

### Graves-Specific Features (The Triad)
| Feature | Frequency | Key Details |
|---------|-----------|-------------|
| **Hyperthyroidism** | 100% | Biochemical Thyrotoxicosis |
| **Ophthalmopathy** | **25-50%** | Proptosis, Lid Lag, Diplopia, Optic Neuropathy |
| **Dermopathy (Pretibial Myxoedema)** | **1-5%** | Bilateral Pretibial, Non-pitting, Plaques/Nodules |
| **Acropachy** | **<1%** | Digital Clubbing, Periosteal Reaction, Fingertip Swelling |

### Ophthalmopathy (Graves Eye Disease) — Grading (EUGOGO)
| Grade | Features | Management |
|-------|----------|------------|
| **Mild** | Minor Lid Retraction/Lag, Mild Proptosis (<2mm), Mild Soft Tissue Signs | Observation, Lubricants, Selenium 100µg BD |
| **Moderate** | Proptosis 2-4mm, Lid Swelling, Diplopia (Intermittent), Moderate Soft Tissue Involvement | **Steroids (IV Methylprednisolone → Oral)**; Orbital RT if Steroid Fails |
| **Severe** | Proptosis >4mm, **Sight-Threatening** (Optic Neuropathy, Corneal Ulcer), Severe Diplopia | **Urgent IV Methylprednisolone (500-1000mg × 3-6 doses) → Oral Taper** + **Orbital RT**; **Emergency Decompression** if Optic Neuropathy |

## 6. Investigations

| Test | Expected in Graves | Interpretation |
|------|-------------------|----------------|
| **TSH** | **Suppressed (<0.01 mIU/L)** | Confirm Thyrotoxicosis |
| **fT4 / fT3** | **Elevated (fT3 > fT4 often)** | Confirm Thyrotoxicosis |
| **TRAb** | **Positive (>95%)** | **Diagnostic of Graves** |
| **TPOAb / TgAb** | Positive (80-90% / 50-70%) | Supports Autoimmune |
| **RAI Uptake** | **Diffuse ↑ (Homogeneous)** | Confirms Graves; Differentiates |
| **Thyroid US** | Diffuse Enlargement, Hypervascular (Colour Doppler) | Supports Graves |
| **CT/MRI Orbits** | If Moderate-Severe Ophthalmopathy | Muscle Enlargement, Fat Expansion |

**Diagnosis of Graves**: **TSH Suppressed + fT4/fT3 ↑ + TRAb Positive** = **Diagnostic** (No RAI Scan Needed if TRAb+)

## 7. Management — Algorithm

```
Graves Disease Diagnosed
         │
         ├── PATIENT FACTORS
         │       ├── Age <18 → Carbimazole (Long Course)
         │       ├── Elderly / Cardiac Disease → RAI or Surgery
         │       ├── Large Goitre (>80g) / Compressive → Surgery
         │       ├── Active Ophthalmopathy → Avoid RAI; Surgery Preferred
         │       ├── Pregnancy → PTU 1st Tri → Switch Carbimazole
         │       └── Patient Preference / Access
         │
         ├── 1ST LINE: ANTITHYROID DRUGS (ATDs)
         │       Carbimazole 15-40mg/day (Divided BD/TDS)
         │       Target: 12-18 Months Course
         │       Remission Rate: 40-50%
         │       Relapse → RAI or Surgery
         │
         ├── DEFINITIVE THERAPY (If Relapse / Contraindication to ATDs)
         │       ├── RAI (¹³¹I) — **Contraindicated in Active Ophthalmopathy**
         │       │       Dose: 400-600 MBq (Calibrated)
         │       │       Hypothyroidism >80% at 1yr
         │       │
         │       └── SURGERY (Total/Near-Total Thyroidectomy)
         │               Large Goitre (>80g) / Compressive / RAI Contraindicated
         │               Pre-op: Carbimazole → Euthyroid → Add Lugol's Iodine 5-7 Days
         │
         └── BLOCK & REPLACE (If Relapse on ATDs)
                 Carbimazole 20-40mg/day + Levothyroxine 100-150µg/day
```

## 8. Antithyroid Drugs (ATDs) — Comparison

| Parameter | **Carbimazole** | **Propylthiouracil (PTU)** |
|-----------|-----------------|----------------------------|
| **Mechanism** | Inhibits TPO → Blocks Iodination + Coupling | Inhibits TPO + **Peripheral T4→T3 Conversion** |
| **Dose (Induction)** | 15-40mg/day (Divided BD/TDS) | 100-200mg TDS |
| **Dose (Maintenance)** | 5-15mg/day | 50-150mg/day |
| **Onset** | 3-6 Weeks | 3-6 Weeks |
| **Half-life** | ~6 Hours | ~1-2 Hours |
| **Crosses Placenta** | **Yes** (Higher Teratogenicity — Aplasia Cutis) | **Yes** (Lower Teratogenicity) |
| **Preferred in Pregnancy** | **2nd/3rd Trimester** | **1st Trimester** |
| **Major Side Effects** | Agranulocytosis (0.1-0.5%), Hepatitis, Rash, Arthralgia | **Hepatotoxicity (Rare, Severe)**, Agranulocytosis, Vasculitis |
| **Lactation** | Compatible (Low Milk Transfer) | Compatible (Minimal Transfer) |

---

## 9. Block & Replace Regimen

| Indication | Regimen |
|------------|---------|
| **Relapse on ATDs** | **Carbimazole 20-40mg/day + Levothyroxine 100-150µg/day** |
| **Non-Adherence to Titration** | Fixed Dose ATD + Levothyroxine |
| **Advantage** | Stable Thyroid Status; No Dose Adjustment |
| **Disadvantage** | Higher ATD Dose → More Side Effects; No Physiological Feedback |

---

## 10. Radioactive Iodine (RAI) — Definitive Therapy

| Aspect | Details |
|--------|---------|
| **Agent** | **Iodine-131 (¹³¹I)** — Beta Emitter (t½ 8 Days) |
| **Dose** | **400-600 MBq** (Calibrated to Thyroid Uptake/Size) |
| **Mechanism** | β-Radiation → Thyroid Follicular Cell Destruction |
| **Onset** | 6-12 Weeks (Hypothyroidism) |
| **Outcome** | **>80% Permanent Hypothyroidism at 1 Year** |
| **Contraindications** | **Pregnancy, Breastfeeding, Active Ophthalmopathy** |
| **Relative Contraindications** | Age <15 (Relative), Active Ophthalmopathy (Relative), Inability to Follow Radiation Safety |
| **Preparation** | Stop ATDs 3-5 Days Prior; Low Iodine Diet 2 Weeks |
| **Post-RAI** | **Hypothyroidism >80% at 1yr** → Lifelong Levothyroxine |
| **Cancer Risk** | Small ↑ Solid Cancer Risk (Dose-Dependent); Not Leukaemia |

## 11. Surgery (Thyroidectomy)

| Indication | Procedure |
|------------|-----------|
| **Large Goitre / Compressive Symptoms** | **Total / Near-Total Thyroidectomy** |
| **RAI Contraindicated / Failed** | Total / Near-Total Thyroidectomy |
| **Suspicion of Malignancy** | Total Thyroidectomy ± Central LND |
| **Patient Preference / Rapid Control** | Total Thyroidectomy |
| **Active Ophthalmopathy** | **Surgery Preferred Over RAI** |

**Pre-op Preparation**:
1. Achieve Euthyroidism with ATDs
2. **Lugol's Iodine** 5-7 Days Pre-op (10-20 Drops TDS) → ↓ Vascularity
3. **Calcium/Vit D** Baseline (Parathyroid Protection)

## 12. Graves Ophthalmopathy — Management Algorithm

```
Graves Ophthalmopathy (Confirmed)
         │
         ├── MILD (EUGOGO)
         │       Lubricants, Selenium 100µg BD, Smoking Cessation, Elevate Head
         │       Monitor q3-6mo
         │
         ├── MODERATE
         │       **IV Methylprednisolone 500mg Weekly ×6 → 250mg Weekly ×6**
         │       → Then Oral Prednisolone Taper (4-6 Months)
         │       If Steroid-Resistant → **Orbital Radiotherapy (20Gy/10fx)**
         │
         ├── SEVERE / SIGHT-THREATENING
         │       **URGENT IV Methylprednisolone 500-1000mg × 3-6 Doses**
         │       → Oral Prednisolone Taper (4-6mo)
         │       **Emergency Orbital Decompression** if Optic Neuropathy
         │       Orbital RT (20Gy/10fx) if Steroid-Resistant
         │
         └── REFRACTORY / RECURRENT
                 Rituximab (Anti-CD20) / Tocilizumab (IL-6R) / Teprotumumab (IGF-1R Ab)
```

### Key Ophthalmopathy Points
| Feature | Details |
|---------|---------|
| **Smoking** | **↑ Risk 2-3x; Worsens Prognosis** — **Cessation Mandatory** |
| **Selenium** | 100µg BD → Reduces Progression (Mild Disease) |
| **IV Methylprednisolone** | 500-1000mg Weekly × 6 → 250mg Weekly × 6 (Standard EUGOGO) |
| **Orbital RT** | 20Gy/10 Fractions; If Steroid-Resistant |
| **Orbital Decompression** | Sight-Threatening; Post-Rehab (Squint/Ptosis Surgery Later) |
| **Teprotumumab** | IGF-1R Antibody; FDA-Approved (Significant Proptosis Reduction) |

## 13. Graves in Pregnancy

| Trimester | Management |
|-----------|------------|
| **Pre-conception** | Achieve Euthyroidism; TRAb <5 IU/L Ideal; Optimise ATD |
| **1st Trimester** | **PTU 50-150mg/day** (Lower Teratogenicity) |
| **2nd/3rd Trimester** | **Switch to Carbimazole** (PTU Hepatotoxicity Risk) |
| **ATD Dose** | **Lowest Effective Dose**; Target fT4 Upper Normal Range |
| **TRAb Monitoring** | Each Trimester; **TRAb >5 IU/L → ↑ Neonatal Thyrotoxicosis Risk** |
| **Fetal Surveillance** | **TRAb >5 IU/L → Fetal US (Goitre, Tachycardia, Hydrops)** |
| **Delivery** | Vaginal Preferred; Neonate Monitored for Thyrotoxicosis (if TRAb High) |
| **Breastfeeding** | **Carbimazole/PTU Compatible** (Low Milk Transfer) |

## 14. Thyroid Storm (Graves) — Emergency

| Criteria (Burch-Wartofsky) | Points |
|----------------------------|--------|
| Temperature (°C) | 37.2-37.7: 5; 37.8-38.3: 10; 38.4-38.8: 15; 38.9-39.4: 20; 39.5-39.9: 25; ≥40: 30 |
| CNS | Absent: 0; Mild: 10; Moderate: 20; Severe (Seizure/Coma): 30 |
| GI | Absent: 0; Moderate: 10; Severe: 20 |
| Cardiac | HR 90-109: 5; 110-119: 10; 120-129: 15; ≥130: 20; AF: 10; HF: 10 |
| Precipitant | Absent: 0; Present: 10 |

**Score ≥45 = Diagnostic**; 25-44 = Impending; <25 = Unlikely

### Emergency Protocol
| Step | Drug | Dose |
|------|------|-------|
| **1. ATD** | **PTU 600mg Load → 200mg Q6H** (Preferred) | Blocks Peripheral T4→T3 Conversion |
| **2. β-Blocker** | **Propranolol 40-80mg Q6H** (IV if Needed) | Controls Adrenergic Symptoms |
| **3. Iodine** | **Lugol's Iodine 8 Drops Q6H** (1hr After PTU) | Blocks Hormone Release |
| **4. Steroids** | **Hydrocortisone 100mg IV q8h** | Blocks Peripheral Conversion |
| **4. Supportive** | Fluids, Cooling, Antipyretics, Anticoagulation (AF), Inotropic Support |

## 16. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Graves vs Toxic Nodule** | TRAb + Diffuse Uptake vs Focal Uptake (Hot Nodule) |
| **RAI vs Surgery for Eyes** | RAI **Worsens** Ophthalmopathy; Surgery Preferred if Active Eye Disease |
| **PTU vs Carbimazole** | PTU = 1st Trimester + Storm; Carbimazole = Standard 1st Line |
| **Block & Replace** | Used for Relapse/Non-Adherence; Higher ATD Dose → More Side Effects |
| **TRAb vs TPOAb** | TRAb = Pathogenic (Stimulates); TPOAb = Marker Only (Not Pathogenic) |
| **Ophthalmopathy Timing** | Can Precede, Coincide, or Follow Thyrotoxicosis (Independent Course) |
| **Carbimazole Teratogenicity** | Aplasia Cutis, Choanal Atresia — Avoid 1st Trimester |

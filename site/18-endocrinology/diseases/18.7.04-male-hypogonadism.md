# Male Hypogonadism

> [!info]
> **Male Hypogonadism = Testosterone Deficiency** with Impaired Spermatogenesis. **Primary (Hypergonadotropic)** = Testicular Failure. **Secondary (Hypogonadotropic)** = Pituitary/Hypothalamic Failure. **Late-Onset (Age-Related)** Increasingly Recognised.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Differentiate primary vs secondary hypogonadism using LH/FSH
- [ ] Diagnose using testosterone, LH, FSH, and clinical features
- [ ] Prescribe testosterone replacement therapy (formulations, monitoring)
- [ ] Address fertility preservation (hCG, rhFSH)
- [ ] Manage special situations (obesity, elderly, cardiovascular disease)

---

## 2. Classification & Aetiology

| Type | LH/FSH | Testosterone | Common Causes |
|--------|--------|--------------|---------------|
| **Primary (Hypergonadotropic)** | **↑↑** | **Low** | Klinefelter (47,XXY), Mumps Orchitis, Chemo/Radiotherapy, Autoimmune, Trauma, Torsion, Genetic (SRY, NR5A1), HIV |
| **Secondary (Hypogonadotropic)** | **↓/Normal** | **Low** | Pituitary Adenoma, Hyperprolactinaemia, Kallmann Syndrome, Functional (Obesity, Stress, Exercise), Chronic Disease, Drugs (Opioids, Glucocorticoids) |
| **Mixed / Functional** | Variable | Low | Ageing, Obesity, Alcohol, Liver Disease, Chronic Illness |

---

## 3. Clinical Presentation

| Domain | Features |
|--------|----------|
| **Sexual** | Low Libido, Erectile Dysfunction, Infertility, Reduced Ejaculate Volume |
| **Physical** | Reduced Beard/Body Hair, Gynaecomastia, Reduced Muscle Mass, Increased Visceral Fat, Decreased Bone Density |
| **General** | Fatigue, Low Mood, Poor Concentration, Hot Flushes, Sleep Disturbance, Anaemia |
| **Metabolic** | Insulin Resistance, ↑ Visceral Fat, Dyslipidaemia, Metabolic Syndrome |

---

## 4. Diagnosis — Biochemical Differentiation

### Key Investigations
| Test | Primary Hypogonadism | Secondary Hypogonadism | Normal |
|------|---------------------|------------------------|--------|
| **Total Testosterone** | **Low** (<12 nmol/L / 350 ng/dL) | **Low** | Normal |
| **Free Testosterone** | Low | Low | Normal |
| **LH** | **↑↑** | **Low / Inappropriately Normal** | Normal |
| **FSH** | **↑↑** | **Low / Inappropriately Normal** | Normal |
| **SHBG** | Normal/High | Normal/Low | Normal |

### Diagnostic Criteria
| Parameter | Threshold |
|-----------|-----------|
| **Total Testosterone** | **<12 nmol/L (350 ng/dL)** on 2 Occasions (Morning, Fasting) |
| **Free Testosterone** | **<225 pmol/L (65 pg/mL)** (If SHBG Abnormal) |
| **LH/FSH** | Interpret with Testosterone (See Table Above) |

---

## 5. Investigations — First-Line

| Test | Purpose |
|------|---------|
| **Total Testosterone (09:00, Fasting)** | **×2 Occasions** (Confirm Low) |
| **LH, FSH** | **Differentiate Primary vs Secondary** |
| **Prolactin** | Exclude Hyperprolactinaemia (Prolactinoma) |
| **SHBG** | Calculate Free Testosterone |
| **TSH, fT4** | Exclude Thyroid Disease |
| **FBC, U&E, LFT, Glucose, Lipids** | General Health |
| **Bone Density (DEXA)** | If Osteoporosis Risk |
| **Semen Analysis** | If Fertility Concern |

## 6. Aetiology — Primary vs Secondary

### Primary Hypogonadism (Testicular Failure)
| Cause | Key Features |
|------|-------------|
| **Klinefelter Syndrome (47,XXY)** | **Commonest Genetic**; Tall, Small Firm Testes, Gynaecomastia, Azoospermia |
| **Mumps Orchitis** | Post-Pubertal; Unilateral/Bilateral; ↑ FSH > LH |
| **Chemotherapy / Radiotherapy** | Alkylating Agents (Cyclophosphamide), Pelvic RT |
| **Autoimmune** | Polyglandular Syndromes (APS-1, APS-2), Anti-Testicular Antibodies |
| **Genetic** | SRY, NR5A1, DAX1, ATRX Mutations |
| **Toxins** | Alcohol, Heavy Metals, Pesticides |
| **Trauma / Torsion** | Bilateral Trauma / Untreated Torsion |

### Secondary Hypogonadism (Pituitary/Hypothalamic)
| Cause | Key Features |
|------|-------------|
| **Pituitary Adenoma** | Macroadenoma; Hyperprolactinaemia; Mass Effect |
| **Hyperprolactinaemia** | Prolactinoma + Stalk Effect |
| **Kallmann Syndrome** | Anosmia + Hypogonadotropic Hypogonadism; ANOS1/KAL1 Gene |
| **Functional (Weight/Stress/Exercise)** | Obesity, Anorexia, Excessive Exercise, Chronic Illness |
| **Drug-Induced** | Opioids, Glucocorticoids, GnRH Analogues, Chemotherapy |
| **Pituitary Adenoma** | Macroadenoma (Compression); Hyperprolactinaemia |
| **Craniopharyngioma** | Suprasellar; Visual Field Defects; Panhypopituitarism |

## 7. Investigation Algorithm

```
Low Testosterone (<12 nmol/L) on 2 Occasions
         │
         ▼
MEASURE LH & FSH
         │
         ├── LH/FSH HIGH → PRIMARY (TESTICULAR FAILURE)
         │       Karyotype (Klinefelter), Mumps History, Chemo/Radiotherapy Hx
         │       Genetic Testing if Young / Family History
         │
         └── LH/FSH LOW / NORMAL → SECONDARY (PITUITARY / HYPOTHALAMIC)
                 │
                 ├── PROLACTIN ↑ → PROLACTINOMA / DRUGS (Antipsychotics, Metoclopramide)
                 ├── MRI PITUITARY → ADENOMA / CRANIOPHARYNGIOMA / EMPTY SELLA
                 ├── MRI BRAIN (Hypothalamus) → KALLMANN / TUMOUR / INFILTRATION
                 └── FUNCTIONAL → OBESITY / STRESS / EXERCISE / DRUGS
```

## 8. Testosterone Replacement Therapy (TRT)

### Indications
| Criteria | Details |
|--------|---------|
| **Symptomatic** | Low Libido, ED, Fatigue, Low Mood, Reduced Muscle Mass |
| **Biochemical** | **Total T <12 nmol/L (350 ng/dL) on 2 Occasions** + Symptoms |
| **Fertility Desire** | **Contraindicated** if Active Fertility Desired (Use hCG/rFSH Instead) |

### Contraindications (Absolute)
| Condition | Reason |
|----------|--------|
| **Prostate Cancer** | Androgen-Dependent |
| **Breast Cancer** | Rare in Men |
| **Severe BPH (IPSS >19)** | Risk of Urinary Retention |
| **Uncontrolled Heart Failure** | Fluid Retention Risk |
| **Polycythaemia (Hct >54%)** | Thrombosis Risk |
| **Active Desire for Fertility** | TRT Suppresses Spermatogenesis |

### Relative Caution
| Condition | Management |
|---------|------------|
| **Age >65** | Start Low Dose; Monitor PSA, Hct, Lipids |
| **Sleep Apnoea** | Monitor (May Worsen) |
| **Cardiovascular Disease** | Individualised (Recent Trials: Neutral) |
| **Polycythaemia** | Monitor Hct; Venesection if >54% |

## 9. Testosterone Formulations

| Formulation | Dose | Frequency | Advantages | Disadvantages |
|-------------|------|-------------|------------|---------------|
| **Transdermal Gel** | 5g (50mg) Daily | **First-Line** (Physiological, Steady Levels) | Skin Transfer Risk, Daily Application |
| **IM Testosterone Undecanoate (Nebido)** | 1000mg (4mL) | q10-14 Weeks | **Long-Acting**, No Peaks/Troughs | Requires IM Injection; Delayed Onset |
| **IM Testosterone Enantate** | 250mg | q3-4 Weeks | Cheap | Peaks/Troughs; Injection Site Pain |
| **Testosterone Propionate** | 25-50mg | q2-3 Days | Short-Acting | Frequent Injections |
| **Oral Testosterone Undecanoate** | 40-80mg BID | With Meals | No Injections | Variable Absorption, Hepatotoxicity Risk |
| **Subcutaneous Pellets** | 600-1200mg | Every 4-6 Months | Long-Lasting | Surgical Insertion; Expulsion Risk |

## 11. Fertility Preservation / Induction

| Goal | Treatment | Protocol |
|------|---------|----------|
| **Fertility Preservation (Pre-TRT)** | **Sperm Cryopreservation** | Before Starting TRT |
| **Fertility Induction (Secondary Hypogonadism)** | **hCG** (Mimics LH) | 1500-5000 IU 2-3x/week; Add **rFSH** if No Response |
| | | **+ Recombinant FSH** | 75-150 IU 3x/week (If No Response to hCG Alone) |
| **Fertility with TRT** | **Contraindicated** (Suppresses Spermatogenesis) | Use hCG/rFSH Instead |

## 12. Special Populations

### Late-Onset Hypogonadism (LOH / "Andropause")
| Feature | Details |
|---------|---------|
| **Definition** | Age-Related Testosterone Decline + Symptoms |
| **Diagnosis** | T <12 nmol/L + Symptoms + Exclude Other Causes |
| **TRT Benefit** | Modest (Libido, Mood, Body Composition); **Not Universally Recommended** |
| **Risks** | ↑ Hct, ↑ PSA, CV Events (Controversial) |

### Obesity-Related Hypogonadism
| Feature | Management |
|---------|-----------|
| **Mechanism** | Aromatase ↑ → Oestrogen ↑ → LH/FSH Suppression; Inflammatory Cytokines |
| **First-Line** | **Weight Loss (5-10%)** → Restores Testosterone in 50% |
| **TRT** | Only if Persistent Symptoms After Weight Loss |

### Advanced Age (>75)
| Consideration | Approach |
|--------------|----------|
| **Frailty** | Lower Starting Dose (Gel 25mg/day) |
| **Cognitive Impairment** | Caution (Testosterone May Improve Mood/Cognition) |
| **Cardiovascular Risk** | Individualised Decision |

## 15. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Primary vs Secondary** | Primary = Testicular Failure → **LH/FSH ↑↑**; Secondary = Pituitary Failure → **LH/FSH ↓/Normal** |
| **Total vs Free Testosterone** | Free T Better if SHBG Abnormal (Obesity, Ageing, Liver Disease) |
| **TRT & Fertility** | TRT **Suppresses** Spermatogenesis; **Use hCG/rFSH** if Fertility Desired |
| **hCG vs rFSH** | hCG = LH Mimic; rFSH = Direct FSH Action; Combo if hCG Alone Fails |
| **Klinefelter** | 47,XXY; Small Firm Testes, Gynaecomastia, Azoospermia; High FSH/LH |
| **Testosterone Threshold** | **<12 nmol/L (350 ng/dL)** = Low |
| **PSA on TRT** | Monitor Annually (>40yr); Stop if PSA ↑ >1.5x Baseline or >4 ng/mL |
| **Polycythaemia** | Common Side Effect; **Hct >54% → Venesection / Dose Reduction / Stop** |
| **hCG vs Testosterone** | hCG **Preserves** Fertility; Testosterone **Suppresses** Spermatogenesis |
| **Testosterone Gel** | **Transfer Risk** to Partner/Children; Wash Hands, Cover Area |

# Liver Transplantation

> [!tip] **FCPS/MRCP Priority: HIGH**
> **Liver Transplantation = Definitive treatment for end-stage liver disease** — Indications (MELD ≥15, ALF, HCC Milan, metabolic), contraindications (absolute/relative), allocation (MELD-based), pre-transplant assessment, post-transplant complications (rejection, infection, recurrence), immunosuppression protocols.

---

## Learning Objectives
By the end of this note you should be able to:
- [ ] List **indications for liver transplant** (MELD ≥15, ALF, HCC Milan, metabolic)
- [ ] Apply **absolute and relative contraindications**
- [ ] Understand **MELD-based allocation** and **exception points** (HCC, HPS, PoPH)
- [ ] Perform **pre-transplant assessment** (cardiac, pulmonary, renal, malignancy, psychosocial)
- [ ] Manage **post-transplant complications**: vascular, biliary, rejection, infection, recurrence
- [ ] Select **immunosuppression regimen** (CNI-based, mTOR, steroid-free)

## 1. Indications for Liver Transplant

### MELD-Based (Chronic Liver Disease)
| Indication | Criteria |
|------------|----------|
| **Decompensated Cirrhosis** | **MELD ≥15** (or MELD-Na ≥15) |
| **Recurrent Decompensation** | ≥2 episodes of ascites/variceal bleed/HE in 6 months |
| **Refractory Ascites** | Requiring LVP ≥q2weeks despite diuretics |
| **Recurrent SBP** | ≥2 episodes despite prophylaxis |
| **Hepatorenal Syndrome** | HRS-AKI unresponsive to terlipressin |
| **Hepatic Encephalopathy** | Recurrent Grade II-IV despite maximal therapy |
| **Portal Hypertensive Complication** | Gastric variceal bleed, portal hypertensive gastropathy bleed |

### Acute Liver Failure (Super-Urgent)
| Indication | Criteria |
|------------|----------|
| **King's College Criteria** | Met (pH <7.3 or PT>100s + Cr>300 + Grade III/IV HE) |
| **CLIF-C ACLF Grade 3** | ≥3 organ failures |
| **Lactic Acidosis** | Lactate >3.5 mmol/L post-resuscitation |
| **Aetiology-specific**: Wilson disease, AFLP, AIH presenting as ALF | Any |

### HCC (Milan / Up-to-Seven Criteria)

| Criteria | Eligible |
|----------|----------|
| **Milan** | Single ≤5cm **OR** ≤3 nodules ≤3cm, **no vascular invasion**, **no extrahepatic spread** |
| **Up-to-Seven** | **Number of tumours + largest diameter (cm) ≤7** (expanded in some centres) |

> **Exception Points**: HCC within Milan = **MELD 22** (UK); increases every 3 months

### Cholestatic / Metabolic / Genetic
| Indication | Specific Criteria |
|------------|-------------------|
| **PBC / PSC** | UDCA failure (PBC), recurrent cholangitis/unresectable CCA (PSC), MELD ≥15 |
| **Autoimmune Hepatitis** | Steroid/azathioprine failure, decompensation, or cirrhosis |
| **Wilson Disease** | **ALF** OR **decompensated cirrhosis** (neuro deterioration despite chelation) |
| **Alpha-1 Antitrypsin Deficiency** | Decompensated cirrhosis |
| **Haemochromatosis** | Decompensated cirrhosis |
| **Cystic Fibrosis** | Decompensated cirrhosis + severe lung disease (combined transplant) |
| **Familial Amyloid Polyneuropathy** | Standard indication |

---

## 2. Contraindications

| Type | Contraindication |
|------|------------------|
| **Absolute** | **Active untreated sepsis**, **Extrahepatic malignancy** (except cured >5yr), **Active substance abuse** (alcohol <6mo abstinence), **Severe cardiopulmonary disease** (not correctable), **Uncontrolled sepsis**, **HIV with AIDS-defining illness**, **Anatomic anomaly** (precluding transplant), **Non-compliance** (documented) |
| **Relative** | **Age >70** (individualised), **Portal vein thrombosis** (grade determines feasibility), **HIV (controlled, CD4>200)**, **Obesity BMI>40**, **Severe malnutrition**, **Active smoking**, **Psychiatric instability**, **Limited social support**, **Age >65** (for some centres) |

> **Alcohol-related liver disease**: **6-month abstinence rule** (UK) + addiction psychiatry assessment + social work

## 3. MELD-Based Allocation (UK)

### MELD Score Calculation
**MELD = 3.78 × ln(bilirubin mg/dL) + 11.2 × ln(INR) + 9.57 × ln(creatinine mg/dL) + 6.43**

**MELD-Na = MELD + 1.32 × (137 - Na) - [0.033 × MELD × (137 - Na)]** (Na 125-137)

| Parameter | Cap |
|-----------|-----|
| **Bilirubin** | No cap |
| **INR** | No cap |
| **Creatinine** | **Capped at 4.0 mg/dL** (if on dialysis = 4.0) |
| **Sodium** | Capped 125-137 mmol/L |

### Exception Points (UK)
| Condition | Exception MELD |
|-----------|----------------|
| **HCC within Milan** | **22** (increases by 3 points every 3 months) |
| **HPS (PaO₂ <60mmHg)** | **22** |
| **PoPH (mPAP >35 on RHC)** | **22** |
| **Familial Amyloid Polyneuropathy** | 22 |
| **Primary Oxaluria** | 22 |
| **Recurrent Cholangitis (PSC)** | Case-by-case |

## 3. Pre-Transplant Assessment

| System | Assessment |
|--------|------------|
| **Cardiac** | **Echocardiography** (EF, wall motion), **Stress test/Dobutamine echo** if >50 or risk factors, **Coronary angiography** if positive stress test / high risk |
| **Pulmonary** | **Chest X-ray**, **Spirometry**, **ABG**, **CT Chest** (if suspicion), **Right Heart Catheter** if PoPH suspected |
| **Renal** | **eGFR**, **Urine protein**, **Ultrasound** (native kidneys); consider **Combined liver-kidney** if eGFR<30 or dialysis >8 weeks |
| **Malignancy** | **CT CAP**, **Colonoscopy** (if >50 or risk factors), **Mammography** (women), **PSA** (men), **Dermatology review** (PSC) |
| **Infection** | **HBV/HIV/HCV/CMV/EBV/VZV/HSV serology**, **TB screening** (Quantiferon/TST), **Strongyloides** (if endemic travel) |
| **Bone Health** | **DEXA scan**, Calcium/Vit D if osteoporotic |
| **Nutritional** | **SGA/MUST**, **Sarcopenia assessment** (CT L3 psoas index) |
| **Psychosocial** | **Psychiatry review** (alcohol, mental health), **Social work** (support, housing, finances), **Compliance history** |
| **Vaccinations** | **Seasonal flu, Pneumococcal (PCV23), Hepatitis A/B** (pre-transplant); **Live vaccines CONTRAINDICATED post-transplant** |

## 4. Post-Transplant Complications

### Timeline
| Period | Complications |
|--------|---------------|
| **Early (0-30 days)** | Primary non-function, **Hepatic artery thrombosis (HAT)**, **Portal vein thrombosis**, **Biliary leak/stricture**, **Rejection (acute cellular)**, **Infection** (bacterial, fungal, CMV), **Renal dysfunction** |
| **Intermediate (1-12 months)** | **Acute cellular rejection**, **Chronic rejection**, **CMV/EBV infection**, **Biliary strictures**, **Recurrent disease** (HCV, HBV, AIH, PBC, PSC), **De novo malignancy** (PTLD, skin) |
| **Late (>1 year)** | **Chronic rejection**, **Metabolic syndrome** (diabetes, HTN, dyslipidaemia), **Renal failure** (CNI nephrotoxicity), **Cardiovascular disease**, **De novo malignancy** (skin, PTLD, solid tumours), **Recurrent disease** |

---

## 4. Key Complications Detail

### Hepatic Artery Thrombosis (HAT) — **Most Devastating Early Complication**
| Feature | Detail |
|---------|--------|
| **Incidence** | **3-8%** (higher in living donor/paediatric) |
| **Presentation** | **Sudden ALT/AST rise >1000**, biliary leak/stricture (liver infarcts), sepsis, fever |
| **Diagnosis** | **Doppler US (absent HA flow)**, **CTA/MR angiography** |
| **Management** | **Emergent re-exploration / thrombectomy** (if <24h); **Retransplant** if failed/liver infarct |

### Biliary Complications
| Type | Incidence | Management |
|------|-----------|------------|
| **Biliary Leak** | **10-20%** (early) | **ERCP + stent** (primary); surgery if failed |
| **Anastomotic Stricture** | **5-15%** (late) | **ERCP + dilation/stenting** (balloon); surgical revision if failed |
| **Non-anastomotic (Ischemic)** | **5-10%** (HAT-related, ABO-incompatible) | **Multiple strictures**; ERCP dilation/stenting; **Retransplant** if diffuse |

### Rejection

| Type | Timing | Histology | Treatment |
|------|--------|-----------|-----------|
| **Acute Cellular Rejection (ACR)** | **1-12 months** (peak 2-8wks) | Portal inflammation, bile duct damage, endothelialitis | **Methylprednisolone 500mg-1g IV ×3d** → oral taper; **Optimize CNI** |
| **Chronic Rejection** | **>1 year** | **Ductopenia**, arteriopathy, fibrosis | **Optimize CNI**, **Switch to mTOR**, **Retransplant** if ductopenia >50% |

### Recurrent Disease

| Disease | Recurrence Rate | Surveillance | Treatment |
|---------|-----------------|--------------|-----------|
| **HCV** | 100% (if pre-tx viraemic) | **SVR with DAA** (post-tx) | **DAA (SOF/VEL/GLE/PIB)** — cure >95% |
| **HBV** | **<5%** (with prophylaxis) | HBsAg, HBV DNA q3-6mo | **HBIG (short-term) + TDF/ENT** |
| **AIH** | **20-30%** | LFTs, IgG, autoantibodies q6mo | **Steroids + Azathioprine** (restart) |
| **PBC** | **20-30%** (10-yr) | ALP, AMA, IgM q6-12mo | **UDCA 13-15mg/kg** |
| **PSC** | **20-30%** | MRCP + CA19-9 q12mo; IBD surveillance | **ERCP for dominant strictures** |
| **HCC** | **10-15%** (if within Milan) | CT/MRI q6mo + AFP | **Resection/Ablation/TACE** if recurrence |
| **NASH** | **30-40%** (10-yr) | LFTs, FibroScan q12mo | **Weight loss, Pioglitazone/GLP-1** |

## 4. Immunosuppression Protocols

### Standard Triple Therapy (Most Common)

| Drug | Induction | Maintenance | Key Monitoring |
|------|-----------|-------------|----------------|
| **Tacrolimus (CNI)** | 0.1-0.2 mg/kg/day IV → 0.05-0.1 mg/kg/day PO BID | **Target trough 5-10 ng/mL** (early), 3-7 ng/mL (late) | **Trough level**, renal, glucose, neuro, Mg, K |
| **Mycophenolate (MMF)** | 500mg BD IV/PO → 1g BD | **1-2g/day** (adjust for GI/renal) | FBC, LFT, GI symptoms |
| **Prednisolone** | 500mg IV → 20mg OD | **Taper to 5mg OD by 3-6 months**; **Steroid-free** by 6-12mo if low risk | BP, glucose, bone, cataracts, infections |

### Alternative Regimens
| Regimen | Indication |
|---------|------------|
| **CNI-free** (mTOR + MMF) | CNI nephrotoxicity, recurrent HCC (mTOR anti-tumour) |
| **Steroid-free** (Tacrolimus + MMF) | Diabetic, obese, high infection risk |
| **mTOR-based (Sirolimus/Everolimus)** | **Recurrent HCC** (anti-tumour), CNI nephrotoxicity, renal protection |
| **Belatacept** (experimental) | CNI-free, EBV+ recipients only |

### CNI Trough Targets

| Period | Tacrolimus Target |
|--------|-------------------|
| **0-1 month** | 10-15 ng/mL |
| **1-3 months** | 8-12 ng/mL |
| **3-12 months** | 5-10 ng/mL |
| **>12 months** | 3-7 ng/mL |

## 4. Outcomes & Prognosis

| Scenario | 30-Day Mortality | 90-Day Mortality |
|----------|------------------|------------------|
| **HRS-AKI (treated)** | ~30-40% | ~50-60% |
| **HRS-AKI (untreated)** | >80% | >90% |
| **HRS-CKD** | Lower than AKI | Depends on liver function |

> **Liver transplant = definitive cure** for both types

## 5. FCPS/MRCP High-Yield Summary

| Topic | Key Points |
|-------|------------|
| **Indications** | **MELD ≥15**, **ALF (King's College)**, **HCC (Milan/Up-to-7)**, **PBC/PSC failure**, **Wilson/Alpha-1-AT**, **Metabolic (ALF)** |
| **Contraindications** | **Absolute**: Sepsis, cancer, alcohol <6mo; **Relative**: Age>70, PVT, HIV, BMI>40 |
| **MELD Allocation** | **MELD = 3.78×ln(bili) + 11.2×ln(INR) + 9.57×ln(Cr) + 6.43**; **Creatinine cap 4.0**; **Exception: HCC MELD 22** |
| **Exception Points** | **HCC Milan: MELD 22**; **HPS PaO₂<60**: 22; **PoPH mPAP>35**: 22 |
| **Pre-Tx Assessment** | Cardiac (Stress), Pulmonary (RHC), Renal (Combined LK if eGFR<30), Cancer Screen, Infection, Psychosocial |
| **HAT** | **3-8%**, sudden ALT/AST >1000, absent HA on Doppler → **Emergent re-exploration / retransplant** |
| **Biliary** | Leak: **ERCP stent**; Stricture: **ERCP dilation**; Ischemic: retransplant |
| **Rejection** | **ACR**: Steroids + Optimise CNI; **Chronic**: Ductopenia >50% → retransplant |
| **Recurrence** | **HCV: DAA cure**; **HBV: HBIG+TDF/ENT**; **AIH: Steroids+Aza**; **PBC: UDCA**; **PSC: ERCP+TX**; **HCC: Milan low risk** |
| **Immunosuppression** | **Tacrolimus** (Target 5-10→3-7) + **MMF 1-2g** + **Pred taper → steroid-free** |
| **mTOR (Sirolimus/Everolimus)** | **Recurrent HCC**, CNI nephrotoxicity, renal protection |
| **CNI Nephrotoxicity** | **Minimise CNI, Switch to mTOR, Belatacept (experimental)** |

## 6. Viva Questions (MRCP PACES / FCPS)

| Question | Expected Answer |
|----------|-----------------|
| **Liver Transplant Indications — MELD, ALF, HCC, Metabolic?** | **MELD ≥15**, **ALF (King's College)**, **HCC (Milan/Up-to-7)**, **PBC/PSC UDCA fail**, **Wilson/A1AT**, **Metabolic (ALF)**. |
| **Absolute Contraindications — Name 5?** | **Active sepsis, Extrahepatic malignancy, Active substance abuse (<6mo abstinence), Severe cardiopulmonary disease, Uncorrectable anatomic anomaly**. |
| **MELD Score — Formula, Creatinine Cap?** | **3.78×ln(bili) + 11.2×ln(INR) + 9.57×ln(Cr) + 6.43**; **Creatinine capped at 4.0 mg/dL**; Dialysis = 4.0. |
| **Exception Points for HCC?** | **Milan criteria → MELD 22** (increases +3 every 3 months); **Up-to-7** also eligible in some regions. |
| **Pre-Transplant Cardiac Assessment?** | **Echo (EF, wall motion) + Stress test/Dobutamine echo if >50/risk factors** → Angio if positive. |
| **Hepatic Artery Thrombosis — Incidence, Presentation, Management?** | **3-8%**, **Sudden ALT/AST >1000**, **Absent HA on Doppler** → **Emergent re-exploration / Retransplant**. |
| **Biliary Complications — Leak vs Stricture?** | **Leak (10-20%): ERCP stent**; **Anastomotic stricture (5-15%): ERCP dilation/stent**; **Non-anastomotic (ischemic): Multiple, retransplant**. |
| **Acute Cellular Rejection — Histology, Treatment?** | **Portal inflammation, bile duct damage, endotheliitis** → **Methylprednisolone 500mg-1g IV ×3d + Optimise CNI**. |
| **Chronic Rejection — Histology, Management?** | **Ductopenia, arteriopathy, fibrosis** → **Optimise CNI, Switch mTOR, Retransplant if ductopenia >50%**. |
| **HCV Recurrence — Management?** | **DAA (SOF/VEL/GLE/PIB) → SVR >95%**; no interferon needed. |
| **HBV Recurrence — Prophylaxis & Management?** | **HBIG (short-term) + TDF/ENT lifelong**; HBsAg/HBV DNA q3-6mo. |
| **Immunosuppression — Standard Triple, Tacrolimus Trough Targets?** | **Tacrolimus + MMF + Prednisolone**; **Tacrolimus trough: 10-15 (0-1m), 8-12 (1-3m), 5-10 (3-12m), 3-7 (>12m)**. |
| **mTOR Inhibitor — Indications?** | **Recurrent HCC** (anti-tumour), **CNI nephrotoxicity** (renal sparing), **Renal protection**. |
| **CNI Nephrotoxicity — Management?** | **Minimise CNI, Switch to mTOR, Belatacept (experimental CNI-free)**. |

## 6. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **MELD vs MELD-Na** | **MELD-Na** = MELD + sodium adjustment; **used for allocation**; Na capped 125-137 |
| **HCC Exception Points** | **Milan = 22** (not automatic transplant, but priority); increases +3/3months |
| **HAT vs PVT** | **HAT**: Sudden AST/ALT >1000, absent HA on Doppler → **Emergent surgery**; **PVT**: Gradual, portal flow reversal, managed with anticoagulation/TIPS |
| **ACR vs Chronic Rejection** | **ACR**: Reversible, steroid-responsive, portal inflammation; **Chronic**: Irreversible ductopenia/arteriopathy, retransplant |
| **HCV vs HBV Recurrence** | **HCV**: Universal without DAA, **DAA cure >95%**; **HBV**: Prophylaxis 95% effective, HBIG+TDF/ENT |
| **mTOR in HCC** | **Sirolimus/Everolimus**: Anti-tumour (mTOR pathway), used for recurrent HCC / CNI nephrotoxicity |
| **Living Donor vs Deceased** | Living: lower HAT risk, shorter wait, but donor risk; Deceased: standard, longer wait |
| **Combined Liver-Kidney** | Indicated if **eGFR<30** or **dialysis >8 weeks** (irreversible renal failure) |

**Mnemonic: LIVER-TRANSPLANT**
- **L**iver Indications: **MELD≥15, ALF (King's College), HCC Milan, Metabolic (Wilson, A1AT)**
- **I**nclusions/Exclusions: **Absolute (Sepsis, Cancer, Alcohol<6mo), Relative (Age>70, PVT, HIV)**
- **V**ascular: **HAT 3-8% → Emergent Re-exploration/Retransplant**
- **E**xception Points: **HCC Milan=22, HPS=22, PoPH=22**
- **R**enal: **Combined LK if eGFR<30 or Dialysis>8wks**
- **T**ransplant Assessment: **Cardiac (Echo/Stress), Pulmonary (RHC), Cancer Screen, Psychosocial**
- **R**ejection: **ACR (Steroids+Optimise CNI), Chronic (Ductopenia→Retransplant)**
- **A**llocation: **MELD-Na (Cr cap 4.0), Exception HCC=22**
- **N**ephrotoxicity: **CNI→mTOR Switch, Belatacept Experimental**
- **S**teroids: **Taper to 5mg by 6mo, Steroid-Free if Low Risk**
- **P**ost-Tx Complications: **HAT, Biliary, Rejection, Infection, Recurrence, Malignancy**
- **L**iver Recurrence: **HCV=DAA Cure, HBV=HBIG+TDF, AIH=Steroids+Aza, HCC=Milan Low**
- **A**ntibodies: **Donor-Specific (DSA) → AMR Risk**
- **N**ew Oncology: **PTLD (EBV), Skin Ca (SCC), De Novo**
- **T**umor HCC: **Milan Criteria (Single≤5cm or ≤3≤3cm)**

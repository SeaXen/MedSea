## HVPG Measurement: Step-by-Step

```mermaid
flowchart TD
    A[Patient Preparation] --> B[Right IJV Access (Preferred)]
    B --> C[Seldinger Technique: Sheath 7-8 Fr]
    C --> D[Balloon Catheter (7-8 Fr) Advancement]
    D --> E[FLUOROSCOPY: Position in Right Hepatic Vein]
    E --> F[Baseline: FHVP Measurement (Balloon Deflated)]
    F --> G[Balloon Inflation: Wedged Position]
    G --> H[WHVP Measurement (Balloon Inflated)]
    H --> I[HVPG = WHVP - FHVP]
    I --> J[Repeat x3 for Mean]
    J --> K[Balloon Deflation, Catheter Removal]
    K --> L[Compression/Haemostasis]
```

## Detailed Procedure

### 1. Patient Preparation
| Step | Detail |
|------|--------|
| **Consent** | Informed Consent (Risks: Bleeding, Arrhythmia, Vessel Injury) |
| **Fasting** | 4-6 Hours |
| **Coagulation** | INR <1.5, Platelets >50k (Correct if Needed) |
| **Monitoring** | ECG, SpO₂, BP, Continuous During Procedure |
| **Antibiotics** | Not Routine (Unless High Risk) |
| **Sedation** | Midazolam ± Fentanyl (If Needed) |

### 2. Vascular Access
| Parameter | Detail |
|-----------|--------|
| **Site** | **Right Internal Jugular Vein** (Preferred — Straight Course to IVC) |
| **Alternative** | Left IJV, Femoral Vein, Subclavian (Less Preferred) |
| **Technique** | Ultrasound-Guided Seldinger Technique |
| **Sheath** | 7-8 Fr Introducer Sheath |

### 3. Catheter Positioning
| Step | Detail |
|-------|--------|
| **Catheter** | 7-8 Fr Balloon Occlusion Catheter (e.g., Fogarty) |
| **Advancement** | Through Sheath → IVC → Right Hepatic Vein |
| **Fluoroscopy** | **Essential** — Confirm Position in Right Hepatic Vein (Not Middle/Left) |
| **Position Check** | Catheter Tip 2-3 cm into Right Hepatic Vein |
| **Balloon** | Deflated for FHVP; Inflated (1-2 ml) for WHVP |

## Pressure Measurements

### FHVP (Free Hepatic Venous Pressure)
| Detail | Value |
|--------|-------|
| **Balloon** | Deflated |
| **Position** | Free in Right Hepatic Vein / IVC Junction |
| **Transducer** | Zeroed at Mid-Axillary Line (Phlebostatic Axis) |
| **Waveform** | Respiratory Variation (a, c, v Waves) |
| **Mean FHVP** | Average of 3-5 Cardiac Cycles (End-Expiration) |

### WHVP (Wedged Hepatic Venous Pressure)
| Detail | Value |
|--------|-------|
| **Balloon** | Inflated (1-2 ml Air/Saline) |
| **Position** | Wedged in Terminal Hepatic Venule (Sinusoids) |
| **Waveform** | **Damped** (Loss of a, c, v Waves) — Confirms Wedging |
| **Mean WHVP** | Average of 3-5 Cardiac Cycles (End-Expiration) |

> **Key**: **Loss of Phasic Waveform = Successful Wedging**

---

## HVPG Calculation

```
HVPG = Mean WHVP - Mean FHVP
```

- **Units**: mmHg
- **Normal**: **1-5 mmHg**
- **Reproducibility**: Mean of **3 Consecutive Measurements** (Accept if Variation <1 mmHg)

| HVPG Value | Interpretation |
|------------|----------------|
| **1-5 mmHg** | **Normal** |
| **6-9 mmHg** | Portal Hypertension (Not Clinically Significant) |
| **≥10 mmHg** | **Clinically Significant Portal Hypertension (CSPH)** |
| **≥12 mmHg** | **Variceal Development Risk** |
| **≥16 mmHg** | High Bleeding Risk |
| **≥20 mmHg** | Very High Mortality Risk |

---

## Technical Pitfalls & Troubleshooting

| Pitfall | Cause | Solution |
|---------|-------|----------|
| **WHVP = FHVP (No Gradient)** | Catheter Not Wedged (In Large Vein) | Advance Further; Re-inflate Balloon |
| **Damped FHVP** | Catheter Against Vessel Wall | Withdraw Slightly; Reposition |
| **WHVP > Portal Pressure** | Incomplete Wedging / Collateral Flow | Re-wedge; Check Position |
| **Arrhythmia** | Catheter Tip in Right Atrium | Withdraw to Hepatic Vein |
| **Catheter Migration** | Respiratory Movement | Secure Catheter; Re-measure |

## Interpretation in Clinical Context

### 1. Cirrhosis (Sinusoidal PH)
| Measurement | Expected |
|-----------|----------|
| **FHVP** | Normal/Slightly ↑ |
| **WHVP** | **Markedly ↑** (Reflects Sinusoidal Pressure) |
| **HVPG** | **≥10 mmHg** (CSPH) |

### 2. Portal Vein Thrombosis (Pre-sinusoidal)
| Measurement | Expected |
|------------|----------|
| **FHVP** | Normal |
| **WHVP** | **= FHVP** (Normal) |
| **HVPG** | **Normal** (Despite Portal HTN) |

### 3. Budd-Chiari Syndrome (Post-sinusoidal)
| Measurement | Expected |
|------------|----------|
| **FHVP** | **Markedly ↑** |
| **WHVP** | **≈ FHVP** (Both High) |
| **HVPG** | Normal/Near Normal (But Both Pressures High) |
| **IVC Pressure** | **Markedly ↑** |

### 4. Cardiac Causes (Constrictive Pericarditis, TR)
| Measurement | Expected |
|------------|----------|
| **FHVP** | **Markedly ↑** |
| **WHVP** | **≈ FHVP** |
| **HVPG** | Normal/Mild ↑ |
| **IVC Pressure** | **Markedly ↑** |

## HVPG in Clinical Decision-Making

| Clinical Question | HVPG Threshold | Action |
|-------------------|----------------|--------|
| **CSPH Diagnosis** | **≥10 mmHg** | Confirms CSPH |
| **Variceal Risk** | **≥12 mmHg** | Screen/Prophylaxis |
| **NSBB Response** | **<12 mmHg** or **↓>20%** | Continue NSBB |
| **Bleeding Risk** | **≥16 mmHg** (High), **≥20** (Very High) | Urgent Intervention |
| **TIPS Indication (Refractory)** | **≥12** (With Complications) | Consider TIPS |
| **Surgery Risk** | **≥10** | ↑ Perioperative Risk |
| **TIPS Function** | **<12** (Post-TIPS) | Patent Shunt |

---

## Non-Invasive Alternatives (Baveno VII)

| Parameter | Cut-off | CSPH Rule-Out |
|-----------|---------|---------------|
| **LSM (FibroScan)** | **<20 kPa** | **<20 kPa** |
| **Platelet Count** | **>150 ×10⁹/L** | **>150** |
| **Combined (Baveno VII)** | **LSM <20 kPa + Platelets >150** | **NPV 97% for CSPH** |

> **If Baveno VII Criteria Met → No HVPG/Endoscopy Needed**

## FCPS/MRCP High-Yield Summary

| Concept | Key Points |
|---------|------------|
| **Procedure** | Right IJV → Balloon Catheter → Right Hepatic Vein → WHVP - FHVP |
| **FHVP** | Free Hepatic Venous Pressure (Balloon Deflated) |
| **WHVP** | Wedged Hepatic Venous Pressure (Balloon Inflated — Damped Waveform) |
| **HVPG** | **WHVP - FHVP** (Normal 1-5 mmHg) |
| **CSPH** | **≥10 mmHg** |
| **Variceal Risk** | **≥12 mmHg** |
| **NSBB Response** | **HVPG <12 mmHg OR ↓>20%** |
| **Bleeding Risk** | **≥16 mmHg** (High), **≥20 mmHg** (Very High) |
| **Pre-sinusoidal** | HVPG Normal (WHVP = FHVP) — PVT, Schisto |
| **Post-sinusoidal** | HVPG Normal BUT FHVP/WHVP High — Budd-Chiari |
| **Non-Invasive** | **Baveno VII: LSM <20 kPa + Platelets >150 = No Endoscopy** |

## Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| WHVP vs FHVP | **FHVP = Free (Balloon Deflated)**; **WHVP = Wedged (Balloon Inflated)** |
| Wedging Confirmation | **Damped Waveform** (Loss of a, c, v Waves) |
| CSPH Threshold | **≥10 mmHg** |
| Variceal Risk | **≥12 mmHg** |
| Pre-sinusoidal HVPG | **Normal** (WHVP = FHVP) — Obstruction Pre-Sinusoids |
| Sinusoidal PH | **HVPG ↑↑** (WHVP > FHVP) — Cirrhosis |
| Post-sinusoidal | **HVPG Normal** BUT **FHVP/WHVP Both High** — Budd-Chiari |
| Baveno VII | **LSM <20 kPa + Platelets >150 = No Endoscopy** |
| NSBB Response | **HVPG <12 OR ↓20%** |

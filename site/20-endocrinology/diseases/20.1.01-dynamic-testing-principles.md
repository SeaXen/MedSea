# Dynamic Testing Principles

> [!info]
> **Dynamic hormone testing uses stimulation or suppression to reveal endocrine reserve, autonomy, or resistance that baseline measurements cannot.** Mastering protocols, cut-offs, and interpretation is core FCPS/MRCP knowledge.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Select the correct dynamic test for a given clinical question
- [ ] Apply standard protocols (dose, timing, sampling)
- [ ] Interpret results using validated cut-offs
- [ ] Identify false positives/negatives and confounding factors
- [ ] Explain the physiological basis of each test

---

## 2. Principles of Dynamic Testing

| Test Type | Physiology | When to Use |
|-----------|------------|-------------|
| **Stimulation** | Provocative agent → Hormone release → Tests **reserve/capacity** | Suspected **deficiency** (e.g., GH, cortisol, gonadotropins) |
| **Suppression** | Inhibitory agent → Hormone suppression → Tests **autonomy** | Suspected **excess/autonomy** (e.g., Cushing, acromegaly, autonomous nodules) |
| **Provocation** | Physiological stressor → Hormone response | Physiological reserve testing (e.g., hypoglycaemia for GH/cortisol) |
| **Replacement/Withdrawal** | Hormone given/withdrawn → Observe axis response | Diagnostic clarification (e.g., thyroxine suppression historical) |

## 3. Standard Stimulation Tests

### 1. Synacthen (ACTH) Test — Adrenal Reserve
| Parameter | Standard Test (250µg) | Low-Dose Test (1µg) |
|-----------|----------------------|---------------------|
| **Dose** | 250µg tetracosactide IM/IV | 1µg tetracosactide IV |
| **Indication** | General screening (high sensitivity) | Secondary AI (more physiological) |
| **Sampling** | Baseline, 30min, 60min | Baseline, 20min, 30min, 60min |
| **Normal Response** | **Cortisol ≥500-550 nmol/L (18-20 µg/dL)** at 30/60min | Same cut-off |
| **Adrenal Insufficiency** | **Cortisol <400-500 nmol/L** at 30/60min | Same cut-off |
| **Limitations** | Supraphysiological; may miss partial AI | More sensitive for secondary AI; technically demanding |

**Interpretation**:
- **Cortisol <100 nmol/L at baseline + <400 at 30/60min** = Adrenal Insufficiency
- **Cortisol >500 nmol/L at 30/60min** = Normal adrenal reserve
- **Indeterminate (100-500 nmol/L)** = Repeat or low-dose test

**Contraindications**: Known severe AI (give hydrocortisone first), acute illness.

### 2. GH Stimulation Tests — GH Deficiency

| Test | Protocol | Cut-off (Peak GH) | Indication |
|------|----------|-------------------|------------|
| **Insulin Tolerance Test (ITT)** | IV Insulin 0.1 U/kg → Hypoglycaemia <2.2 mmol/L | **<3 µg/L** (adults); **<5 µg/L** (children) | **Gold standard**; Tests GH + cortisol simultaneously |
| **Glucagon** | Glucagon 1mg IM/IV → Sampling 0, 60, 90, 120, 150, 180min | **<3 µg/L** (adults) | Safer than ITT; Preferred if ITT contraindicated |
| **Arginine** | Arginine 0.5g/kg IV over 30min | **<0.4 µg/L** (children) | Alternative |
| **Clonidine** | Clonidine 0.15mg/m² (max 0.25mg) PO | **<6.8 µg/L** | Historical |
| **GHRH + Arginine** | GHRH 1µg/kg + Arginine | **<9 µg/L** (adults) | Most sensitive for adult GHD |

**ITT Protocol — Gold Standard**:
1. **Fast overnight**, IV access, ECG monitoring
2. **Baseline**: Glucose, GH, Cortisol
3. **Insulin 0.1 U/kg IV bolus**
4. **Sampling**: q15-30min for 2h (Glucose, GH, Cortisol)
5. **Target**: Glucose <2.2 mmol/L (symptomatic hypoglycaemia)
5. **Recovery**: IV Glucose + monitors

**Contraindications for ITT**: IHD, Cerebrovascular disease, Epilepsy, Severe hypopituitarism (cortisol <100), Age >60, Pregnancy.

### 3. Gonadotropin Stimulation Tests

| Test | Protocol | Interpretation |
|------|----------|----------------|
| **GnRH Test** | GnRH 100µg IV/SC → LH/FSH at 0, 20, 40, 60, 90, 120min | **Prepubertal**: FSH > LH; **Puberty**: LH > FSH; **PCOS**: LH > FSH (2:1); **Hypogonadotropic**: Blunted |
| **hCG Stimulation (Male)** | hCG 1500-5000 IU IM × 3-5 days → Testosterone at 24h post-last | **Testosterone rise >1.5x baseline** = Leydig cell response; **No rise** = Primary testicular failure |
| **Clomiphene Challenge** | Clomiphene 100mg/day × 5 days → FSH/LH/Oestradiol day 3 & day 10 | **Ovarian reserve**: ΔOestradiol > 400 pmol/L = Good reserve |

---

### 4. Thyroid Axis Stimulation

| Test | Protocol | Interpretation |
|------|----------|----------------|
| **TRH Stimulation** | TRH 200-500µg IV → TSH at 0, 20, 40, 60, 90, 120min | **Hyperthyroidism**: Blunted TSH; **Primary Hypo**: Exaggerated/prolonged TSH; **Central Hypo**: Blunted/delayed; **TSH-oma**: Paradoxical rise |
| **Recombinant Human TSH (rhTSH)** | rhTSH 0.9mg IM × 2 days → Radioiodine uptake / Thyroglobulin | Thyroid remnant ablation prep; Diagnostic scanning |

---

### 5. Adrenal Steroid Stimulation — CAH Diagnosis

| Test | Protocol | Interpretation |
|------|----------|----------------|
| **ACTH Stimulation (21-OH Def)** | Synacthen 250µg → 17-OH Progesterone, 11-Deoxycortisol, Androstenedione at 0, 30, 60min | **17-OHP >30 nmol/L (1000 ng/dL)** = 21-OH deficiency; >100 = Severe |
| **ACTH Stimulation (11β-OH Def)** | Synacthen 250µg → 11-Deoxycortisol, DOC at 0, 60min | **11-Deoxycortisol ↑↑** |
| **ACTH Stimulation (17α-OH Def)** | Synacthen 250µg → 17-OH Pregnenolone, 17-OH Progesterone | Both ↑↑ |

## 4. Standard Suppression Tests

### 1. Dexamethasone Suppression Tests — Cushing Syndrome

#### Low-Dose DST (LDDST) — Screening
| Protocol | Overnight (1mg at 23:00 → Cortisol 09:00) OR 48h (0.5mg q6h × 8 doses → Cortisol 6h post-last) |
|----------|------------------------------------------------|
| **Cut-off** | **Cortisol <50 nmol/L (1.8 µg/dL)** = Normal suppression |
| **Positive (Non-suppression)** | Cortisol ≥50 nmol/L → Proceed to HDDST |

#### High-Dose DST (HDDST) — Differential Diagnosis
| Protocol | 48h: 2mg q6h × 8 doses (or overnight 8mg at 23:00) |
|----------|---------------------------------------------------|
| **Cushing Disease (Pituitary ACTH)** | **Cortisol suppresses >50%** (or <50% of baseline) |
| **Ectopic ACTH / Adrenal Cushing** | **No suppression** (<50% suppression) |
| **False Positives** | Obesity, Depression, Alcohol, Drugs (Phenytoin, Carbamazepine, Rifampicin) → Induce CYP3A4 → Faster dexamethasone clearance |

#### DST Interpretation Summary
| Result | Interpretation |
|--------|----------------|
| **LDDST Suppressed (<50)** | Normal (No Cushing) |
| **LDDST Not Suppressed + HDDST Suppressed** | **Cushing Disease** (Pituitary ACTH) |
| **LDDST Not Suppressed + HDDST Not Suppressed** | **Ectopic ACTH** or **Adrenal Cushing** → Check ACTH level |

---

### 2. OGTT for GH Suppression — Acromegaly

| Protocol | 75g Glucose PO → GH at 0, 30, 60, 90, 120min |
|----------|-----------------------------------------------|
| **Normal** | **GH suppressed <1 µg/L** at 60-120min |
| **Acromegaly** | **GH not suppressed ≥1 µg/L** (gold standard) |
| **Partial Suppression** | GH falls but >1 µg/L = Active acromegaly |

---

### 3. Thyroid Hormone Suppression (Historical)

| Test | Protocol | Replaced By |
|------|----------|-------------|
| **T3 Suppression Test** | T3 25µg q12h × 10 days → RAI uptake | **TRH Stimulation** / **TRAb** / **RAI Scan** |
| **T4 Suppression** | T4 100µg/day × 7 days | Same |

---

### 4. Fludrocortisone Suppression — Primary Aldosteronism

| Protocol | Fludrocortisone 0.1mg q6h × 4 days + High salt diet → Aldosterone, Renin, K+ |
|----------|-----------------------------------------------------------------------------|
| **Positive** | Aldosterone **not suppressed** (remains >150 pmol/L) |

---

### 5. Saline Infusion Test — Primary Aldosteronism

| Protocol | 2L 0.9% NaCl IV over 4h → Aldosterone, Renin |
|----------|---------------------------------------------|
| **Positive** | Aldosterone **>140 pmol/L** (or >280 pmol/L per some guidelines) after infusion |

## 5. Provocative / Physiological Stress Tests

### 1. Insulin Tolerance Test (ITT) — Gold Standard for GH + Cortisol Reserve
| Aspect | Details |
|--------|---------|
| **Physiology** | Hypoglycaemia → Stress → CRH → ACTH → Cortisol + GHRH → GH |
| **Protocol** | Insulin 0.1 U/kg IV → Target glucose <2.2 mmol/L |
| **Cortisol Cut-off** | **Peak >500-550 nmol/L** = Normal |
| **GH Cut-off** | **Peak >3 µg/L** (adults) = Normal |
| **Contraindications** | IHD, CVD, Epilepsy, Severe hypopituitarism, Age >60, Pregnancy |

### 2. Water Deprivation Test — Diabetes Insipidus
| Phase | Protocol |
|-------|----------|
| **Fluid Deprivation** | NPO + Fluid restriction → Weight, Serum/Urine Osmolality q1-2h until ↑ >5% weight loss or Urine Osm > Plasma Osm |
| **Desmopressin Challenge** | DDAVP 2µg SC/IV → Urine Osmolality at 60min |
| **Central DI** | Urine Osm **↑ >50%** after DDAVP |
| **Nephrogenic DI** | **No response** to DDAVP |
| **Primary Polydipsia** | Urine Osm **concentrates during deprivation** (no need for DDAVP) |

### 3. Solute Infusion Test — Aldosteronism
See Fludrocortisone Suppression / Saline Infusion above.

## 7. Specialised Dynamic Tests

### 1. Secretin Stimulation — Gastrinoma (ZES)
| Protocol | Secretin 2 U/kg IV → Gastrin at 0, 2, 5, 10, 15, 30min |
|----------|--------------------------------------------------------|
| **Positive** | **Gastrin rise >120 pg/mL** (or >200 pg/mL from baseline) |
| **MEN1 Association** | 20-25% of gastrinomas |

### 2. Calcium Stimulation (SACI) — Insulinoma Localisation
| Protocol | Selective arterial calcium injection → Hepatic venous insulin sampling |
|----------|------------------------------------------------------------------------|
| **Positive** | **Insulin step-up >2x baseline** in hepatic vein draining tumour region |

### 3. Arginine Stimulation — Glucagonoma/GH
| Protocol | Arginine 5g IV (children) / 30g IV (adults) → Glucagon/GH |
|----------|----------------------------------------------------------|
| **Glucagonoma** | Glucagon rise >2x baseline |

### 4. Lupron/GnRH Agonist Stimulation — Gonadotropin-independent Precocious Puberty
| Protocol | Leuprolide 0.2mg/kg SC → LH/FSH at 1-2h |
|----------|----------------------------------------|
| **Positive** | LH >5 IU/L (pubertal response) |

## 8. Common Pitfalls & Confounders

| Test | Pitfall | Correction |
|------|---------|------------|
| **DST** | **Drugs inducing CYP3A4** (Phenytoin, Carbamazepine, Rifampicin, St John's Wort) → Faster dexamethasone clearance → **False non-suppression** | Check medications; Use CYP3A4-independent confirmatory tests (late-night salivary cortisol, 24h UFC) |
| **DST** | **Obesity, Depression, Alcohol** → False non-suppression | Confirm with 24h UFC + Late-night salivary cortisol |
| **DST** | **Pregnancy, OCP** → ↑ CBG → ↑ Total cortisol (free normal) → False non-suppression if total cortisol used | Use **free cortisol** (salivary) or 24h UFC |
| **Synacthen** | **Early secondary AI** may pass 250µg test | Use **Low-dose (1µg) Synacthen** |
| **ITT** | **Counter-regulatory failure** in long-standing hypopituitarism | Use Glucagon test instead |
| **GH Stimulation** | **Obesity** → Blunted GH response (false positive GHD) | Use BMI-adjusted cut-offs; Confirm with IGF-1 |
| **TRH Test** | **Dopamine agonists, Dopamine antagonists** affect TSH | Stop drugs 48h prior |
| **Water Deprivation** | **Primary polydipsia** can mimic partial DI | Desmopressin response + Clinical context |
| **Synacthen** | **Oestrogen/OCP** → ↑ CBG → ↑ Total cortisol (free normal) | Measure **Free cortisol** (salivary) or 24h UFC |

## 9. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **Synacthen standard dose** | 250µg IM/IV; Cortisol ≥500-550 nmol/L at 30/60min = Normal |
| **Synacthen low-dose** | 1µg; More sensitive for secondary AI; Technically demanding |
| **ITT** | **Gold standard** for GH + cortisol; Insulin 0.1 U/kg → Glucose <2.2; Contraindicated: IHD, CVD, Epilepsy, Age >60 |
| **GH cut-off (ITT)** | **Peak GH <3 µg/L** = GHD (adults); <5 µg/L (children) |
| **LDDST** | 1mg overnight or 0.5mg q6h × 48h; **Cortisol <50 nmol/L = Normal suppression** |
| **HDDST** | 2mg q6h × 48h; **>50% suppression = Cushing Disease**; No suppression = Ectopic/Adrenal |
| **OGTT for Acromegaly** | 75g glucose; **GH <1 µg/L at 60-120min = Normal**; ≥1 = Acromegaly |
| **ITT Contraindications** | IHD, CVD, Epilepsy, Severe hypopituitarism, Age >60, Pregnancy |
| **Water Deprivation** | Central DI: DDAVP → Urine Osm ↑ >50%; Nephrogenic: No response; Polydipsia: Concentrates |
| **Water Deprivation Stop** | Weight loss >5% or Urine Osm > Plasma Osm or Haemodynamic instability |
| **Synacthen for CAH** | 17-OHP >30 nmol/L (1000 ng/dL) = 21-OH deficiency |
| **Drugs affecting DST** | Enzyme inducers (Phenytoin, Carbamazepine, Rifampicin) → False non-suppression |
| **DST in Pregnancy/OCP** | ↑ CBG → ↑ Total cortisol; Use **Salivary cortisol** or **24h UFC** |
| **GH Stimulation in Obesity** | Blunted GH → False positive GHD; Use BMI-adjusted cut-offs + IGF-1 |

## 10. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **LDDST vs HDDST** | LDDST = **Screening** (is it Cushing?); HDDST = **Differential** (Pituitary vs Ectopic/Adrenal) |
| **Synacthen 250µg vs 1µg** | 250µg = Standard screening; 1µg = More sensitive for secondary/early AI |
| **GH Stimulation** | ITT = Gold standard but dangerous; Glucagon = Safer alternative; GHRH+Arg = Most sensitive for adults |
| **DST Failure** | Most common cause = **Drugs (enzyme inducers)**; Obesity/Depression/Alcohol also |
| **Cortisol Cut-offs** | DST: <50 nmol/L; Synacthen: ≥500-550 nmol/L; ITT: >500-550 nmol/L; Morning: 200-700 nmol/L |
| **ITT vs Glucagon** | ITT tests cortisol + GH simultaneously; Glucagon safer but only GH well-validated |
| **OGTT for Acromegaly** | **Only validated suppression test for acromegaly**; GH <1 µg/L = Normal |

**Mnemonic: DYNAMIC TESTS BY AXIS**
- **HPA**: Synacthen, ITT, DST (LD/HD), Water Deprivation
- **HPT**: TRH stim, TSH suppression (historical)
- **HPG**: GnRH, hCG stim, Clomiphene challenge
- **GH/IGF-1**: ITT, Glucagon, Arginine, GHRH+Arg, OGTT (suppression)
- **Adrenal/Steroid**: Synacthen (CAH), DST, Saline/Fludro suppression (Conn)
- **Insulinoma**: 72h Fast, Calcium stimulation (SACI)
- **Gastrinoma**: Secretin stimulation
- **DI**: Water Deprivation + DDAVP
- **Insulinoma**: 72h Fast, SACI

## Algorithm Quick-Reference Table

| Clinical Scenario | First-Line Algorithm | Key Investigations | FCPS/MRCP Focus |
|-------------------|----------------------|-------------------|-----------------|
| **Polyuria/Polydipsia** | ADH axis → Water deprivation → Desmopressin test | Serum/urine osmolality, ADH, MRI pituitary | Central vs Nephrogenic DI, SIADH |
| **Hypokalaemia + Hypertension** | Aldosterone/Renin → CT adrenal → AVS | Aldosterone, renin, ratio, CT adrenal, AVS | Primary vs secondary hyperaldosteronism |
| **Hypercalcaemia** | PTH → PTHrP → Vit D → Bone scan | PTH, calcium, phosphate, PTHrP, 1,25 Vit D, bone scan | Primary HPT vs malignancy vs granulomatous |
| **Cushing Syndrome** | 1mg DST → Late-night salivary cortisol → ACTH | 24h UFC, DST, ACTH, CRH test, IPSS, imaging | Cushing disease vs ectopic vs adrenal |
| **Adrenal Insufficiency** | 0900 cortisol → Synacthen test → ACTH | Cortisol, ACTH, electrolytes, renin, aldosterone | Primary vs secondary vs tertiary |
| **Hyperthyroidism** | TSH → fT4/fT3 → TRAb → RAI uptake | TSH, fT4, fT3, TRAb, radioiodine scan | Graves vs toxic nodules vs thyroiditis |
| **Hypothyroidism** | TSH → fT4 → Anti-TPO → US | TSH, fT4, anti-TPO, US thyroid | Primary vs central, Hashimoto |
| **Pituitary Adenoma** | Hormone panel → MRI pituitary ± visual fields | Prolactin, GH/IGF-1, ACTH, TSH, FSH/LH, cortisol, MRI | Micro vs macro, functioning vs non-functioning |
| **Diabetes Insipidus** | Water deprivation → Desmopressin → ADH | Serum/urine osmolality, ADH, MRI | Central vs nephrogenic vs primary polydipsia |
| **Acromegaly** | IGF-1 → OGTT GH suppression → MRI | IGF-1, GH, OGTT suppression, MRI pituitary | Active vs controlled, complications |
| **Prolactinoma** | Prolactin → MRI → Visual fields | Prolactin, MRI, visual fields, macroprolactin | Micro vs macro, dopamine agonist response |
| **Phaeochromocytoma** | Plasma metanephrines → CT/MRI → MIBG | Metanephrines, CT/MRI/MIBG, genetic testing | Sporadic vs syndromic (MEN2, VHL, NF1) |
| **Primary Aldosteronism** | ARR → CT adrenal → AVS | Aldosterone, renin, CT, adrenal venous sampling | Unilateral vs bilateral, surgery vs medical |
| **Thyroid Nodule** | TSH → US (TI-RADS) → FNA (Bethesda) | TSH, US, FNA, molecular markers | Malignancy risk stratification |
| **Male Hypogonadism** | LH/FSH/Testosterone → Prolactin → MRI if secondary | Testosterone, LH, FSH, prolactin, MRI | Primary vs secondary, fertility |
| **PCOS** | Rotterdam criteria → Androgens → US | Testosterone, SHBG, LH/FSH, US ovaries, glucose | Diagnosis, metabolic risk, fertility |

## Algorithm Summary Table (16 Core Pathways)

| # | Pathway | Key Decision Points | First-Line Treatment |
|---|---------|---------------------|----------------------|
| 1 | **Hypokalaemia + HTN** → ARR → CT → AVS | ARR >70 (or ARR >30 + aldosterone >400) | Unilateral: adrenalectomy; Bilateral: spironolactone/eplerenone |
| 2 | **Hypercalcaemia** → PTH → imaging | PTH high = primary HPT; PTH low = malignancy/granuloma | Hydration, bisphosphonates, cinacalcet, surgery if primary HPT |
| 3 | **Cushing Syndrome** → 24h UFC/DST → ACTH → imaging/IPSS | ACTH-dependent vs independent | Transsphenoidal surgery (Cushing disease); adrenalectomy (adrenal); resection (ectopic) |
| 4 | **Adrenal Insufficiency** → 0900 cortisol → Synacthen | Cortisol <100 = likely AI; >500 = excludes; 100-500 = synacthen | Immediate hydrocortisone 100mg IV; maintenance hydrocortisone 15-20mg + fludrocortisone |
| 5 | **Adrenal Crisis** | ABCDE + IV fluids + hydrocortisone 100mg IV stat | Hydrocortisone 100mg IV q6h + aggressive fluids + glucose |
| 6 | **Thyroid Storm** | Burch-Wartofsky score ≥45 | PTU (load 600mg then 200mg q6h) + propranolol + iodine (after PTU) + steroids + fluids |
| 7 | **Myxoedema Coma** | Hypothermia, altered mental status, hypoventilation | IV thyroxine (load 300-500mcg) + hydrocortisone + supportive |
| 8 | **Hyperthyroidism** | TSH low → fT4/fT3 high → TRAb/RAI scan | Carbimazole 15-40mg (or PTU in pregnancy) ± beta-blocker; RAI/surgery definitive |
| 9 | **Hypothyroidism** | TSH high, fT4 low → anti-TPO | Levothyroxine 1.6mcg/kg (start low/go slow in elderly/cardiac) |
| 10 | **Pituitary Adenoma** | Hormone panel → MRI → visual fields | Microprolactinoma: cabergoline; Others: transsphenoidal surgery |
| 11 | **Diabetes Insipidus** | Water deprivation → desmopressin response | Central: desmopressin; Nephrogenic: thiazide + low salt + NSAID |
| 12 | **SIADH** | Euvolaemic hyponatraemia, urine osmolality >100, urine Na+ >20 | Fluid restriction (<1L), demeclocycline, tolvaptan (severe) |
| 13 | **Acromegaly** | IGF-1 high → OGTT GH not suppressed | Surgery (transsphenoidal) → somatostatin analogues → pegvisomant |
| 14 | **Prolactinoma** | Prolactin >1000 + pituitary MRI | Cabergoline 0.25mg BD titrate; surgery if resistant/intolerant |
| 15 | **Phaeochromocytoma** | Metanephrines → CT/MRI/MIBG → genetic | Alpha-blockade (phenoxybenzamine) ≥2 weeks pre-op + beta-blockade after α-blockade |
| 16 | **Primary Aldosteronism** | ARR → CT adrenal → AVS | Unilateral: laparoscopic adrenalectomy; Bilateral: MR antagonist |

# Adrenal Cushing (ACTH-Independent Cushing Syndrome)

> [!info]
> **Adrenal Cushing = ACTH-Independent Cortisol Excess from Adrenal Tumour/Hyperplasia.** Accounts for **10-15% of Endogenous Cushing**. **ACTH Suppressed (<10 pg/mL)**. Subtypes: Adrenal Adenoma (Commonest), Adrenal Carcinoma, PPNAD, AIMAH.

---

## 1. Learning Objectives
By the end of this note you should be able to:
- [ ] Differentiate adrenal Cushing from ACTH-dependent causes using ACTH and HDDST
- [ ] Distinguish adrenal adenoma from carcinoma clinically and radiologically
- [ ] Recognise rare causes: PPNAD, AIMAH, macronodular hyperplasia
- [ ] Outline management algorithm (surgery vs medical vs bilateral adrenalectomy)
- [ ] Apply incidentaloma workup algorithm

---

## 2. Classification of ACTH-Independent Cushing

| Subtype | Frequency | Key Features |
|---------|-----------|--------------|
| **Adrenal Adenoma** | **Most Common (50-60% of Adrenal Cushing)** | Unilateral, Benign, Cortisol-Secreting |
| **Adrenal Carcinoma** | **5-10%** | Large (>6cm), Heterogeneous, Invasion, Metastases |
| **Primary Pigmented Nodular Adrenal Disease (PPNAD)** | Rare | Bilateral Micronodules, Carney Complex, ACTH Suppressed |
| **ACTH-Independent Macronodular Adrenal Hyperplasia (AIMAH)** | Rare | Bilateral Macronodules, Aberrant Receptors (GIP, LH, b-AR) |
| **McCune-Albright Syndrome** | Rare | Fibrous Dysplasia, Café-au-lait, GnRH-Independent Precocious Puberty |

---

## 3. Clinical Presentation — Adrenal vs ACTH-Dependent Cushing

| Feature | **Adrenal Cushing** | **ACTH-Dependent (Cushing Disease / Ectopic)** |
|---------|---------------------|--------------------------------------------------|
| **ACTH** | **Suppressed (<10 pg/mL)** | Normal / Elevated (10-50 pg/mL in Disease; >200 in Ectopic) |
| **HDDST (2mg q6h × 48h)** | **No Suppression** (Same as Ectopic) | **>50% Suppression = Cushing Disease; No Suppression = Ectopic** |
| **Cortisol** | High | High |
| **Hypokalaemia** | Mild/Moderate | Severe in Ectopic |
| **Hyperpigmentation** | **Absent** (ACTH Low) | **Present** in Ectopic (High ACTH) |
| **Onset** | Insidious | Variable |
| **Aetiology** | Adrenal Tumour/Hyperplasia | Pituitary / Ectopic / Adrenal |

## 4. Adrenal Adenoma — Most Common Adrenal Cushing

### Clinical & Biochemical Profile
| Feature | Details |
|---------|--------|
| **Prevalence** | **50-60% of Adrenal Cushing** |
| **Size** | Usually **2-5 cm** (Solitary, Unilateral) |
| **Laterality** | Unilateral (Right = Left) |
| **Cortisol Secretion** | Autonomous, ACTH-Independent |
| **ACTH** | **Suppressed (<10 pg/mL)** |
| **DHEAS** | Normal or Slightly Elevated |
| **Androgens** | Normal (Unless Mixed Secretion) |

### CT Adrenal Findings
| Feature | Adenoma | Carcinoma |
|---------|---------|-----------|
| **Size** | **<6 cm** (Usually 2-5 cm) | **>6 cm** (Often >10 cm) |
| **Attenuation (Non-Contrast)** | **<10 HU** (Lipid-Rich) | **>20 HU** (Often >30-40 HU) |
| **Washout** | **>60% Absolute / >40% Relative** (Rapid) | **<60% Absolute / <40% Relative** (Delayed) |
| **Enhancement** | Homogeneous | Heterogeneous, Necrosis, Calcification |
| **Margins** | Smooth, Well-Defined | Irregular, Infiltrative, Invasion |

---

## 5. Adrenal Carcinoma — Rare But Aggressive

### Key Features
| Feature | Details |
|---------|---------|
| **Incidence** | **5-10% of Adrenal Cushing**; 1-2/million/year |
| **Size** | **>6 cm** (Mean 10-13 cm) |
| **Malignant Features** | **Invasion** (Vena Cava, Kidney, Liver), **Metastases** (Lung, Liver, Bone, Lymph Nodes) |
| **Hormonal Profile** | **Cortisol + Androgens (DHEAS, Testosterone ↑)**; Mixed Secretion Common |
| **Hormone Excess** | Often **Mixed** (Cortisol + Androgens ± Oestrogens) |
| **Biochemical Clues** | **DHEAS ↑↑**, Testosterone ↑, 17-OHP ↑ |

### Imaging Red Flags
| Finding | Significance |
|---------|--------------|
| **Size >6 cm** | High Suspicion |
| **Heterogeneous Enhancement** | Necrosis, Haemorrhage |
| **Irregular Margins / Invasion** | Vena Cava, Kidney, Liver |
| **Lymphadenopathy** | Regional Nodes |
| **Distant Metastases** | Lung, Liver, Bone, Lymph Nodes |

### Weiss Score (Histopathological) — >3 = Carcinoma
| Criterion | Points |
|-----------|--------|
| **High Nuclear Grade** | 1 |
| **Mitotic Rate >5/50 HPF** | 1 |
| **Atypical Mitoses** | 1 |
| **Clear Cells <25%** | 1 |
| **Necrosis** | 1 |
| **Venous Invasion** | 1 |
| **Sinusoidal Invasion** | 1 |
| **Capsular Invasion** | 1 |
| **Total >3** = **Carcinoma** | |

## 6. Adrenal Carcinoma — Management

### Surgical Management
| Stage | Procedure |
|-------|-----------|
| **Localized (T1-2, N0, M0)** | **Radical Adrenalectomy** (Open Preferred) + **Lymphadenectomy** |
| **Locally Advanced (T3-4, N0, M0)** | **Radical Adrenalectomy + En Bloc Resection** (IVC, Kidney, Liver) + Lymphadenectomy |
| **Metastatic (M1)** | **Palliative Resection** (Symptom Control) + **Mitotane** |

### Adjuvant Therapy
| Agent | Indication | Dose |
|-------|-----------|------|
| **Mitotane (o,p'-DDD)** | **Adjuvant (All Stages)** / **Metastatic** | **2-6 g/day** (Titrate to 14-20 mg/L Serum Level) |
| **EDP Chemotherapy** (Etoposide, Doxorubicin, Cisplatin) | **Metastatic / Unresectable** | Standard Regimens |
| **Radiotherapy** | Palliation (Bone Mets) | Palliative Doses |

### Mitotane Monitoring
| Parameter | Frequency | Target |
|-----------|-----------|--------|
| **Mitotane Level** | q4-6wk | **14-20 mg/L** (Therapeutic) |
| **LFTs, Cortisol** | q4-6wk | Adrenal Insufficiency Monitoring |
| **Lipid Profile** | q3mo | Hypercholesterolaemia Common |

## 7. Rare Causes of ACTH-Independent Cushing

### Primary Pigmented Nodular Adrenal Disease (PPNAD)
| Feature | Details |
|---------|---------|
| **Pathology** | Bilateral Micronodules (<1cm), **Pigmented** (Lipofuscin) |
| **Syndrome** | **Carney Complex** (PRKAR1A Mutation) — Spotty Pigmentation, Myxomas, Schwannomas |
| **Hormonal** | Cortisol Secretion; **ACTH Suppressed**; May Have Periodic Cushing |
| **Genetics** | **PRKAR1A** (Chromosome 17q24); Autosomal Dominant |
| **Management** | **Bilateral Adrenalectomy** (Curative); Pre-op Steroid Cover |

### ACTH-Independent Macronodular Adrenal Hyperplasia (AIMAH)
| Feature | Details |
|---------|---------|
| **Pathology** | **Bilateral Macronodules (>1cm)**; Adrenal Hyperplasia |
| **Mechanism** | **Aberrant Membrane Receptors** (Ectopic Hormone Response) |
| **Common Receptors** | **GIP** (Post-Prandial Cortisol Spike), **LH/hCG**, **β-AR**, **Vasopressin**, **Serotonin** |
| **Genetics** | **ARMC5** (Tumour Suppressor, Chromosome 16) — 25-50% Familial |
| **Management** | **Bilateral Adrenalectomy** (Curative); Pre-op Steroid Cover; Receptor Studies |

### McCune-Albright Syndrome
| Feature | Details |
|---------|---------|
| **Triad** | **Fibrous Dysplasia** (Bone), **Café-au-lait Spots** (Jagged Edges), **Endocrine Hyperfunction** |
| **Endocrine** | **Precocious Puberty** (GnRH-Independent), **Hyperthyroidism**, **Cushing (Adrenal Hyperplasia)**, **Acromegaly** |
| **Genetics** | **GNAS Mutation** (Post-Zygotic, Mosaic) — Activating Gsα |
| **Management** | Adrenal: Surgery/Medical; Others: Disease-Specific |

## 8. Adrenal Incidentaloma — Workup Algorithm

```
Adrenal Incidentaloma on Imaging
         │
         ▼
HORMONAL WORKUP (All Three Required)
         │
         ├── 1mg DST (Cortisol <50 nmol/L = Excluded)
         ├── Plasma Free Metanephrines (Normal = Excluded)
         └── Aldosterone/Renin Ratio (Normal = Excluded)
         │
         ├── ALL NORMAL → NON-FUNCTIONING
         │       │
         │       ├── SIZE <4cm + BENIGN FEATURES (<10 HU, Washout >60%) → SURVEILLANCE
         │       │       CT 6-12mo → Yearly ×2 → Hormonal Recheck 1yr
         │       │
         │       └── SIZE ≥4cm OR SUSPICIOUS (>10 HU, Washout <40%, Heterogeneous) → SURGERY
         │
         └── ANY ABNORMAL → FUNCTIONING → SPECIFIC MANAGEMENT
                 ├── Cushing → Surgery (Adrenalectomy)
                 ├── Phaeo → Alpha-Blockade → Surgery
                 └── Conn → AVS → Surgery (Unilateral) / Medical (Bilateral)
```

### Surveillance Protocol (Non-Functioning, <4cm)
| Timepoint | Action |
|-----------|--------|
| **6-12 Months** | CT Adrenal (1mm, Contrast) |
| **Yearly ×2** | CT + Hormonal Recheck |
| **Then** | If Stable → Every 2-3 Years; Hormones Yearly |

## 9. Exam Pearls (FCPS/MRCP)

| Topic | Key Point |
|-------|-----------|
| **Adrenal Cushing ACTH** | **Suppressed (<10 pg/mL)** |
| **Adrenal Cushing HDDST** | **No Suppression** (Same as Ectopic ACTH) |
| **Adrenal Adenoma vs Carcinoma** | Adenoma: <6cm, <10 HU, Washout >60%; Carcinoma: >6cm, >20 HU, Invasion |
| **Adrenal Carcinoma** | >6cm, Heterogeneous, Invasion, Metastases, ↑ DHEAS/Testosterone |
| **Adrenal Incidentaloma Workup** | 1mg DST + Metanephrines + Aldosterone/Renin |
| **Non-Functioning <4cm** | Surveillance (CT 6-12mo → Yearly ×2 → 2-3yrly) |
| **Non-Functioning ≥4cm** | Surgery (Laparoscopic Adrenalectomy) |
| **Adrenal Cushing Surgery** | **Laparoscopic Adrenalectomy** (Curative for Adenoma) |
| **Adrenal Carcinoma** | >6cm, Heterogeneous, >20 HU, Invasion, ↑ DHEAS/Testosterone |
| **PPNAD** | Carney Complex, PRKAR1A, Bilateral Micronodules, ACTH Suppressed |
| **AIMAH** | Bilateral Macronodules, Aberrant Receptors (GIP, LH), ARMC5 Mutation |
| **McCune-Albright** | Fibrous Dysplasia, Café-au-lait, Precocious Puberty, GNAS Mutation |
| **Incidentaloma Workup** | 1mg DST + Metanephrines + ARR — All 3 Required |
| **Adrenal Adenoma CT** | <10 HU, Washout >60% = Lipid-Rich Adenoma |

## 10. Confusions & Mnemonics

| Confusion | Clarification |
|-----------|---------------|
| **Adrenal Cushing vs Cushing Disease** | Adrenal: ACTH Suppressed, No HDDST Suppression; Cushing Disease: ACTH Normal/High, HDDST Suppresses |
| **Adrenal Cushing vs Ectopic ACTH** | Both: ACTH-Independent (Suppressed ACTH) vs ACTH-Dependent (High ACTH); HDDST No Suppression Both |
| **Adenoma vs Carcinoma Size** | <6cm vs >6cm; <10 HU vs >20 HU; Washout >60% vs <60% |
| **Adrenal Incidentaloma** | Must Check ALL 3: DST, Metanephrines, ARR |
| **PPNAD** | Carney Complex, Bilateral Micronodules, PRKAR1A |
| **AIMAH** | Bilateral Macronodules, Aberrant Receptors (GIP, LH), ARMC5 |
| **Adrenal Carcinoma Surgery** | Open Radical + Lymphadenectomy + Mitotane Adjuvant |
| **Adrenal Incidentaloma Size** | <4cm = Observe; ≥4cm = Surgery (If Benign Features Absent) |

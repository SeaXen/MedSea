## High-Yield Summary Table

| Disorder | Clinical Key | Investigation | 1st Line Management | Key Feature |
|----------|--------------|---------------|---------------------|-------------|
| **Androgenetic alopecia (M)** | Bitemporal recession, vertex thinning | Clinical, trichoscopy | Minoxidil 5% + Finasteride 1mg | Pattern, genetic |
| **Androgenetic alopecia (F)** | Diffuse central thinning, preserved frontal | Clinical, trichoscopy, hormones | Minoxidil 2-5% ± Anti-androgen | Ludwig pattern |
| **Alopecia areata** | Patchy non-scarring, exclamation hairs | Trichoscopy (yellow dots), biopsy | TCS potent / Intralesional triamcinolone | Autoimmune, atopy, nails |
| **Telogen effluvium** | Diffuse shedding 3m post-trigger | Pull test +ve, trichoscopy | Reversible, address trigger | Post-partum, stress, drugs |
| **Scarring alopecia** | Permanent loss, follicular ostia loss | Biopsy (lymphocytic vs neutrophilic) | LPP: TCS/TCI, hydroxychloroquine, doxycycline | LPP, FFA, FD, CCCA |
| **Tinea capitis** | Scaly patches + alopecia, kerion | KOH + culture (Trichophyton/Microsporum) | **Griseofulvin 6-8w** / Terbinafine 4w | Kerion = inflammatory |
| **Onychomycosis** | Thick, discoloured, dystrophic nail | KOH + culture (dermatophyte) | **Terbinafine 3-6m** / Itraconazole pulse | Distal lateral subungual |
| **Nail psoriasis** | Pitting, onycholysis, oil spots, subungual hyperkeratosis | Clinical, biopsy if needed | TCS/TCI intralesional, systemic if severe | 80% psoriasis have nails |

## Key Algorithms

### Alopecia Diagnostic Approach
```mermaid
flowchart TD
    A[Hair Loss] --> B{Scarring?}
    B -->|No (follicular ostia present)| C{Pattern}
    C -->|Patchy| D[Alopecia Areata: TCS/IL steroid, JAKi if severe]
    C -->|Diffuse| E[Telogen Effluvium: Trigger 3m ago, reversible]
    C -->|Patterned (M)| F[Androgenetic M: Minoxidil + Finasteride]
    C -->|Patterned (F)| G[Androgenetic F: Minoxidil ± Anti-androgen]
    B -->|Yes (follicular ostia lost)| H[Scarring Alopecia: Biopsy]
    H --> I{Lymphocytic}
    I -->|Yes| J[LPP/FFA/CCCA: Hydroxychloroquine, TCS/TCI, doxy]
    I -->|No| K[Neutrophilic: FD: Dapsone, antibiotics]
```

### Nail Disorder Approach
```mermaid
flowchart TD
    A[Nail Abnormality] --> B{All 20 nails?}
    B -->|Yes| C[Systemic: Clubbing, Beau lines, Half-and-half, Terry, Koilonychia]
    B -->|No (few nails)| D{Inflammation?}
    D -->|Yes| E[Paronychia: Acute (S.aureus) vs Chronic (Candida/irritant)]
    D -->|No| F{Morphology}
    F -->|Pitting/Oil spots| G[Nail Psoriasis]
    F -->|Thick/Dystrophic| H[Onychomycosis: KOH + Culture → Terbinafine]
    F -->|Longitudinal bands| I[Melanonychia: Dermoscopy → Biopsy if Hutchinson]
    F -->|Thin/Brittle| J[Trauma, Age, Nutritional, Lichen Planus]
```

## FCPS/MRCP Viva Topics (High-Yield)

1. **Androgenetic alopecia** - patterns (Hamilton-Norwood M, Ludwig F), minoxidil mechanism, finasteride 5α-reductase inhibitor, DHT
2. **Alopecia areata** - patchy/totalis/universalis/ophiasis, exclamation hairs, yellow dots trichoscopy, atopy association, JAK inhibitors (baricitinib, ritlecitinib)
3. **Telogen effluvium** - trigger 3 months prior, pull test positive, self-limiting 6-9 months
4. **Scarring alopecias** - LPP (lichen planopilaris), FFA (frontal fibrosing), FD (folliculitis decalvans), CCCA (central centrifugal) - lymphocytic vs neutrophilic
5. **Tinea capitis** - griseofulvin still 1st line in kids, terbinafine alternative, kerion = inflammatory, Woods lamp (Microsporum fluoresces)
6. **Onychomycosis** - DLSO most common, terbinafine 3-6m fingernails/toenails, itraconazole pulse, confirm with KOH/culture before treating
7. **Nail psoriasis** - pitting, onycholysis, oil spots, subungual hyperkeratosis, nail matrix vs nail bed signs
8. **Nail in systemic disease** - clubbing (cyanotic heart/lung/GI), Beau lines (acute illness), koilonychia (iron deficiency), half-and-half (renal), Terry (liver), Mees lines (arsenic/chemo)
9. **Trichoscopy** - yellow dots (AA), comma hairs (tinea), black dots (AA), hair shaft disorders, vellus hairs (AGA)
10. **Hirsutism vs hypertrichosis** - Ferriman-Gallwey score,PCOS, drug-induced, androgen-secreting tumours

## Mnemonics

- **Alopecia types:** `SCAR` = **S**carring (LPP, FFA, FD, CCCA) vs **C**atagen? No - **SCAR** = **S**carring = **C**icatricial = **A**lopecia = **R**eversible? No - Use: **SCARRING** = **S**evere, **C**icatricial, **A**lopecia, **R**arely **R**egrows, **I**nflammatory, **N**eeds **G**rouping (lymphocytic/neutrophilic)
- **Nail signs systemic:** `CLUB TERRY` = **C**lubbing (heart/lung/GI), **L**eukonychia (trauma/arsenic), **U**npleasant (Beau lines = illness), **B**rittle (age/nutrition), **T**erry (liver), **E**xclusion (half-and-half = renal), **R**idges (age/lichen), **R**ed (melanonychia), **Y**ellow (onychomycosis)
- **Tinea capitis treatment:** `GRISEO` = **Griseofulvin** 6-8 weeks (still 1st line paeds) / **TERB**inafine 2-4 weeks alternative
- **Alopecia areata severity:** `PATCH → TOTALIS → UNIVERSALIS` = Increasing extent, decreasing regrowth potential

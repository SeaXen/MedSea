- Rare genodermatoses (Darier, Hailey-Hailey, porokeratosis)
- Complex immunocompromise scenarios

## Key Algorithms & Scoring Systems (Memorise)

| System | Use Case | Key Thresholds |
|--------|----------|----------------|
| **PASI** | Psoriasis severity | <10 mild, 10-20 moderate, >20 severe |
| **SCORTEN** | SJS/TEN mortality | ≥3 = high mortality (>35%) |
| **RegiSCAR** | DRESS diagnosis | Score ≥5 = definite |
| **EUROSCAR** | AGEP diagnosis | Score ≥8 = confirmed |
| **ABCDE** | Melanoma screening | Asymmetry, Border, Colour, Diameter, Evolution |
| **Breslow thickness** | Melanoma prognosis | ≤1mm, 1-2mm, 2-4mm, >4mm |
| **Clark level** | Melanoma microstaging | I-V (dermis invasion depth) |
| **UAS7** | Urticaria activity | Daily scoring ×7 days |
| **EASI** | Atopic dermatitis severity | 0-72 scale |
| **SCORAD** | Atopic dermatitis severity | Objective + subjective |
| **CASPAR** | Psoriatic arthritis | ≥3 points = PsA |
| **DAPSA** | PsA disease activity | Remission <4, LDA 4-14, MDA 15-28, HDA >28 |
| **DLQI** | Quality of life impact | 0-30 (higher = worse) |
| **HLA associations** | Drug reactions | HLA-B*58:01 (allopurinol), B*15:02 (carbamazepine), B*57:01 (abacavir) |
| **LRINEC** | Necrotising fasciitis | ≥6 = high suspicion |
